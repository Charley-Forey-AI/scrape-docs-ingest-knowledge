# Set Up Tracking Codes

Set up codes used to track benefits, activities, and
 classifications in the Human Resources module.
Use only the code types that you need. Most options require that you enter a
 code and description. Options that require additional setup are listed on the [Human Resources - Getting Started](/en/spectrum/spectrum/human-resources/human-resource-module/human-resources---getting-started) page.

1. Select Human Resources > Maintenance.

1. Select the code type to set up. For example, Recurring Benefit Codes.

1. Click New.

1. Enter a Code and Description.

1. Click OK.

1. Set up additional codes as needed.

1. Click Cancel to exit the setup
 window. The new code displays in the grid.

1. To run a report of all codes of this type, click
 Listing.
