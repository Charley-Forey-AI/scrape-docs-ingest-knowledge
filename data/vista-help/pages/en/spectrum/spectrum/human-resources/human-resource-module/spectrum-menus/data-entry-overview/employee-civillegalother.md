# Employee Civil/Legal/Other

Use the Employee Civil/Legal/Other screen to review
 employee information for civil, legal, and other similar activities.
Use the header portion of the screen to narrow your inquiry results. If this screen is
 accessed in context to an employee, all events for the employee code will display in the
 listbox. If this screen is accessed for a non-employee specific format, use the search box
 to narrow your inquiry results.
