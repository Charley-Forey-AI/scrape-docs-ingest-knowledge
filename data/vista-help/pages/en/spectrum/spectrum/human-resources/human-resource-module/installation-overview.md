# Installation Overview

Before you set up installation options, ensure that the
 checkbox for the Human Resources module has been selected on the System Administration > Installation > Company > Modules tab. Be sure to save any changes made on this tab.
Note: The system administrator should have completed this step
 already.
To set up Human Resources, select System Administration > Installation > Human Resources. The screen has five tabs that allow you to set up different options, such
 as numbering schemes and or security, within the system.

- Click the [Human Resources Installation - Properties](/en/spectrum/spectrum/human-resources/human-resource-module/installation-overview/human-resources-installation---properties) tab to define numbering schemes and set up options for user-defined fields.

- Click the [Human Resources Installation - Reporting](/en/spectrum/spectrum/human-resources/human-resource-module/installation-overview/human-resources-installation---reporting) tab if your company participates in FMLA/OSHA; enter all of the necessary information to set up tracking, including the average number of employees and hours worked last year.

- Click the [Human Resources Installation - Security](/en/spectrum/spectrum/human-resources/human-resource-module/installation-overview/human-resources-installation---security) tab to define Human Resources security schemes. It is a good idea to consider which employees will have access to Human Resources prior to completing Human Resources security setup.

- Click the [Human Resources Installation - Operators](/en/spectrum/spectrum/human-resources/human-resource-module/installation-overview/human-resources-installation---operators) tab to assign employee security schemes.

- Click the [Human Resources Installation - User-Defined Fields](/en/spectrum/spectrum/human-resources/human-resource-module/installation-overview/human-resources-installation---user-defined-fields) tab to define parameters for user-defined fields.
Important: When you complete the Human Resources Installation
 process for the first time, the software will prompt you to create Human Resources
 job titles from existing job titles in the Payroll employee master file. Click OK to
 create job titles from Payroll.
