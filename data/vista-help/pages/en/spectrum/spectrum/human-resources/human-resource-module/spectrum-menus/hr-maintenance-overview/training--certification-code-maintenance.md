# Training / Certification Code Maintenance

Use the Training/Certification Code
 Maintenance screen to track and maintain training / certification designation
 codes.
Examples are codes to identify various levels of schooling, including High School diploma,
 Associates Degree, Trade Certification, Bachelor's Degree, and so forth. It can also be
 used to track continuing education programs encouraged or required by the company.
