# HR Maintenance Overview

Use the Human Resources >Maintenance menu to manage
 the codes used to track benefits, activities, and classifications relevant to Human
 Resources. From the Site Map screen, click Human Resources, and then click
 Maintenance.
The following code types are available. Click a code to learn how
 to set it up in the system:
