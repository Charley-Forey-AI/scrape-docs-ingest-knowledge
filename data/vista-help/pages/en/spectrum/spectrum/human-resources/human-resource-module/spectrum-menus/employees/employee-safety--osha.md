# Employee Safety / OSHA

Use this screen to manage Safety/OSHA incident information.
 If this screen is accessed in context to an employee, all items for the employee code will
 display in the listbox. If this screen is accessed for a non-employee specific format, use the
 search box to narrow your inquiry results.
