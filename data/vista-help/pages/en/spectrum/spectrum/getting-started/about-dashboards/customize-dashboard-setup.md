# Customize Dashboard Setup

Learn how to customize the main landing screen you see when logging in.
You can customize your Dashboard by selecting from a variety of apps that can be used for things like managing jobs and viewing alerts to potential issues. Some of the kinds of apps that are available include RSS feeds, Menu lists, pie charts (such as cash flow), and graphs.
Your operator profile, which is indicated in the Operator Maintenance screen, dictates which apps you have access to.
Logging into the Dashboard does not require a Spectrum license; however, accessing a Spectrum screen via any one of the apps does.

1. Select the Dashboard tab.

1. Select the Settings button to open the Dashboard Builder screen. This screen allows you to drag and drop apps and define the dashboard layout.

1. Use the Search field and Go button to conduct a manual search on app titles.

1. Use the Category filter to show only apps matching a selected category. By default, all categories are included.

1. Use the Type filter to show only apps matching that type: types are the 'widgets' such as RSS feeds, Menu lists, Charts, and Favorites.

1. Select the number of Matches to show per screen. The default is 6 for smallest screen resolution, but it can go up to 12. This setting is used the next time you log on.

1. To rearrange apps, drag and drop them to different locations.

1. To delete an app from the dashboard, drag it to the trash can.
