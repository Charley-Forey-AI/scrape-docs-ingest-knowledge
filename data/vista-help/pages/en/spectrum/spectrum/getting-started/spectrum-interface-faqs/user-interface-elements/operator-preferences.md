# Operator Preferences

The Operator Preferences screen provides a convenient place where you can update your preferred system settings.
To access this screen, select the Preferences tab located between the Company drop-down list and the Help tab in the Spectrum bar.
If needed, refer to [Operator Preferences - Field Descriptions](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/operator-preferences/operator-preferences---field-descriptions).
