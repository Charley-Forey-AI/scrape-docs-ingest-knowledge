# About Dashboards in Trimble Analytics

Dashboards in Trimble Analytics provide in-depth
 and interactive visualizations of your Spectrum data, enabling you
 to make data-driven decisions for your enterprise.
From each Dashboard in Trimble Analytics, you can filter the data
 by various parameters, modify how the data is displayed, and export the data as an .xlsx
 or .csv file.
Dashboards retain any security settings configured for a user in Spectrum.
For instructions on setting up and working with Dashboards, see [Dashboards](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:Analytics_000004:Analytics:Analytics) in the
 Trimble Analytics Help.
Note: Users must first be using single
 sign-on (SSO) in order to access Dashboards. For details on enabling single sign-on,
 see [Single Sign-On](/en/spectrum/spectrum/getting-started/single-sign-on).
