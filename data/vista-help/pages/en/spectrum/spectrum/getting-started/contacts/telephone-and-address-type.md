# Telephone and Address Type

Use this page to set up a standardized list of validated types of phone numbers and addresses associated with contacts. Any contact may be assigned an unlimited number of phone and address types. This helps you identify various ways to communicate with that contact.
Example:
You enter a phone type labeled "Direct line." If a user adds a new contact telephone number in the Vendor Contacts page, the "Direct line" phone type displays in the drop-down menu of the Description field.
