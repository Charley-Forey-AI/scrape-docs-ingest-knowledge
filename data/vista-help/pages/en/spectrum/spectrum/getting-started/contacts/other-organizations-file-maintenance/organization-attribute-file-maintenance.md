# Organization Attribute File Maintenance

Use this page to define a standardized and validated list of classifications associated with organizations. Any given organization (vendor, customer, or other organization) may be assigned an unlimited number of Attributes that help identify contacts associated with particular types of organizations.

- Architectural

- Electrical

- Demolition

- Subcontract

- Plumbing

- Clean-up

- Owner

- Mechanical

-  Landscaping

-  Engineering

-  Marine

- Compliance

- Supplier

- Bridges

- Catering

- Corporate

-  Transportation

- Publicity

The organization attributes display alphabetically in the List box. Click the Organization type button to filter organization types: vendors, customers, or other organizations. Spectrum allows you to delete attributes assigned to other organizations. See the Attributes field in [Other Organizations File Maintenance](/en/spectrum/spectrum/getting-started/contacts/other-organizations-file-maintenance) for more information.
