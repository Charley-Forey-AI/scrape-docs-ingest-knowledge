# About Dashboards

The Dashboard is your Spectrum landing page, and it only
 takes a few minutes to set up a custom Dashboard comprised of apps.
Apps are programs that run
 on the Dashboard. These can be charts, lists, RSS feeds, to-do lists, and more that are
 most meaningful to you. For example, you can use them to manage jobs or alert you to
 potential issues. For more information on specific apps, see [Dashboard App Descriptions](/en/spectrum/spectrum/getting-started/about-dashboards/dashboard-app-descriptions).
The Dashboard uses visual analytics to provide information, while providing the opportunity to drill into the associated screens when you want additional information.Note: Use of the Dashboard does not require a Spectrum license. A license is occupied only if you use one of the Dashboard apps to access a screen in Spectrum, which requires logging in.
