# Export W-2 File and Nelco State Tax Filing

The steps to export a W-2 to use with Nelco.
[Print W-3 Forms](/en/spectrum/spectrum/year-end/year-end-for-u.s.-payrolls/processing/year-end-processing-screen-u.s./print-w-3-forms)
Trimble Viewpoint partners with Nelco to support W-2 filing To make it easier to file state W-2 forms, Viewpoint has partnered with Nelco. The Spectrum interface includes an exported CSV file which you can upload to the Nelco site by via the Nelco CSV button on this screen.
You can then choose from a menu of Nelco services, including electronic filing and delivery of forms to employees. Nelco supports foreign employees in the export file if the province, postal code, and country codes are not blank.
Tip: After creating the W2REPORT file, use a text editor such as Notepad to confirm the W-2 information. To further review the contents of the electronic file, the Social Security Administration offers AccuWage Software.
Important: When using Nelco to file W-2s, the State ID must be entered in the exact format specified in the [attached document](https://trimble.deploy.heretto.com/v4/deployments/mVyHSMAo9PbkfqcxUBWF/object/a0c381e0-8ea8-11ec-9179-02420ae60f1a?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6InZpZXdwb2ludCIsImh0dHBzOi8vam9yc2VrLmNvbS9lemQvb2JqZWN0X3V1aWQiOiJhMGMzODFlMC04ZWE4LTExZWMtOTE3OS0wMjQyMGFlNjBmMWEiLCJleHAiOjE3NzUzMjc2NjcsImp0aSI6ImFlODk3OGUzNDJmNDQzNTNhYzJlOGQ4ZWQ0OTRmMjUyIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZF9maWxlc2V0IjoiOE1GRmpJZEw3ZEZLQmJiVGwxTXMifQ.PtnNdg257ukJrGc6j2-s-Zblbrm2tVDXDGjNLZK-XOA&response-content-disposition=filename%3D%22Valid_State_ID_Formats.pdf%22). Verify the State ID prior to building W-2s.
Note:For the State of Iowa onlyYou will be required to provide your Iowa Business eFile Number to Nelco for proper electronic processing. To do this, export the W-2 file for the State of Iowa only, open the file in a text editor, and then add the eFile Number into the IA_BEN column.

1. Complete and review the screen selections, paying special attention to the tax year selection.

1. Select the Nelco CSV button. The Export to Nelco CSV Format information window displays.

1. Select the hyperlink to learn more about Nelco Solutions, or select Continue if you are already sure you wish to proceed with the export.

1. A second information window displays letting you know if the export was successful and listing any exceptions (records containing errors) if they exist.

1. Open the [Nelco partner site](https://spectrum.nelcoportal.com/import/gwximport). If you haven't already created an account, select the Create Account link. Here you'll be prompted to enter your general info (name, address, email, etc.) about your company.

1. Select the Import Data link and select the 'Federal Information Reporting' Form Type and select the File to Import.

1. Select the Import button.

1. On the Forms In Progress screen, choose the filing services you wish to use: Federal E-filing, State Filing, Recipient Mail & Online Retrieval. These services can be selected as a bundle or individually.

1. Make sure all the employees for your companies are selected on this screen and select the Next button to proceed.

1. If you have selected the 'Recipient Mail & Online Retrieval' service, enter recipient email addresses.

1. Review the recipients in your cart and select the Next button.

1. On the Payment Details screen, enter credit card information.

1. Select the Pay & Submit button to complete processing.

Proceed as directed at [Year End Processing Screen (U.S.)](/en/spectrum/spectrum/year-end/year-end-for-u.s.-payrolls/processing/year-end-processing-screen-u.s.).
