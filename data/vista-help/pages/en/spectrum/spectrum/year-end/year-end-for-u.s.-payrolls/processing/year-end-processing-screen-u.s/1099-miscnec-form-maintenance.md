# 1099-MISC/NEC Form Maintenance

The 1099-MISC/NEC Form Maintenance
 screen displays a list of all the vendors currently receiving a 1099.
If doing year-end processing, [Build 1099-MISC/NEC Forms](/en/spectrum/spectrum/year-end/year-end-for-u.s.-payrolls/processing/year-end-processing-screen-u.s./build-1099-miscnec-forms).
You can use this
 screen to update and print 1099-MISC/NEC forms for vendors.
This screen provides protection based on the vendor status.

- Not Used - Entry is prevented.

- Inactive - A warning displays when entry is attempted.

If doing year-end processing, [Print 1099-MISC/NEC Forms](/en/spectrum/spectrum/year-end/year-end-for-u.s.-payrolls/processing/year-end-processing-screen-u.s./print-1099-miscnec-forms).
