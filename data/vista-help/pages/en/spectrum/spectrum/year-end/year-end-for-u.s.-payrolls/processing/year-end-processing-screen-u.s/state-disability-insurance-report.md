# State Disability Insurance Report

The State Disability Insurance Report screen prints disability insurance information.
This report prints in sequence first by state, then by employee within each state. Choose specific dates and states to obtain the exact report desired.
Note: In the current company, if the Entity Tracking checkbox is selected in Enterprise Installation > Edit Company Settings, the Employer and Employer 'Percent' labels and rates do not display in the heading of the page.
