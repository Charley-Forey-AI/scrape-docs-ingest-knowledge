# Year-End for Canada Payrolls

Preparing for and completing year-end processing of Canada payrolls also prepares you for commencing next year's payroll processing.

## Prior to year-end processing

We recommend that you familiarize yourself with the entire year-end process prior to beginning your year-end processing.

- View the Release Notes. These will be published soon. Once they are published, a link will be provided here.

- Explore [Year-End Training for Customers with Canada Payrolls](https://learn.trimble.com/learn/catalog/view/101?ctldoc-catalog-0=field45-_Spectrum) at Trimble Learn.

Tip:Viewpoint provides a data file that automatically updates the Tax Tables for in Spectrum.

- You may download this file from the [Viewpoint Customer Portal](https://support.viewpoint.com/s/spectrum).

- The file includes all known changes to electronic tax tables for Federal and all provinces.

- We update this file throughout the year to reflect changes issued by each entity.
