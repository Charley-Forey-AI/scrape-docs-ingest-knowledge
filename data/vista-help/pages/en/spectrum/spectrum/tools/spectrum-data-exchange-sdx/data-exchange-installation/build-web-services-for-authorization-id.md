# Build Web Services for Authorization ID

To add Data Exchange to the Authorized ID, from the New
 Data Exchange Authorization window, click the Build button and the Build Data Exchange for
 Authorization ID page displays. The available Web services display in the left pane, and the
 authorized Web services already assigned to the current Authorization ID display in the right
 pane.
Use the Select
 and Deselect buttons to establish
 a list of services. Once the Data Exchange are defined for the Authorization ID and
 updated, the main Authorization page displays with the Options button available.
Note:
 Some web services displayed on the Build Web Services screen are
 created for 3rd party Vendor Use only and the Description shown states that
 information.
