# Copy File to New Document

After scanning or attaching an image file, you can then
 copy this file to another document.

1. In the Thumbnail Viewer portion of the Info Pane, use the up an down arrow buttons to navigate to the file you want to copy and select it.

1. Left-click on the selected image file. A list of possible actions displays; select the Copy to New Document option.

1. A Create New Document dialog box displays. Enter the Document ID that you want to assign to the file being copied and click OK to save the file.

1. The new document will display as a new tab in the Slider Bar.Thumbnail Viewer options
Locating the Current Image Directory

1. On the Spectrum Site Map screen, click Document Imaging  >  Search Documents.

1. Enter the specifications for the files you want to locate (for example, the Cabinet and Drawer names), and click the Search Documents button.

1. When the search results are listed, find the image you are interested in and make a note of the corresponding Cabinet and Drawer. Exit.

1. Open the Document Imaging  >  Cabinet File Maintenance screen. Sort the columns to locate the Cabinet and Drawer combination you just noted in step 3.

1. Highlight the row that contains the Cabinet and Drawer you are interested in and scroll to see the Effective path, Path basis, and Override path.
