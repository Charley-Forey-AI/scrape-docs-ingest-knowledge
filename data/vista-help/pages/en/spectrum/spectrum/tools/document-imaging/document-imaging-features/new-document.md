# New Document

This screen allows you to add a new document and / or
 create a new folder. You can store images within multiple folders, where applicable. Use this
 screen to save the same document to as many folders as desired. For example, while adding an
 A/P invoice for equipment repair, the image can be stored in both the vendor's invoice folder
 and within the applicable equipment folder.
