# Search Documents

This screen allows you to look for stored image files after
 entering some search criteria.

- After completing the initial Search Images window, the search
 results will display in a list box. When you select a transaction, the associated
 image will automatically display in the Slider Bar.

- When Images are accessed from the Info Bar, a specific cabinet and folder
 will default as the search values and the search results and Slider Bar will display
 automatically (without opening the Search Images window first).

- A New
 button will display if your operator has Document Imaging security level 3
 (Read/Write with annotations), even if your operator does not have security for the
 New Document feature.
Note:
 Note Regarding Archived Records: Because archived records are
 stored by the archive date (year and period), be aware that when searching
 archived records that the archive date may not be the same as the record date; for
 example, if you archived the Year End Financial Statement (dated 12/31/2014) on
 3/12/2015, the archived record is stored in folder 2015, period 3.
