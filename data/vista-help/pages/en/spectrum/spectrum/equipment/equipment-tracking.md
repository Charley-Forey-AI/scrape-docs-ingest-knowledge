# Equipment Tracking

The Equipment Tracking module is used to charge company equipment, small tools and inventory to a job. Use this module to start and stop chargeable standby billing that depends on location. Online inquiries are available both in summary and detail formats and various billings are supported, including both weekly and partial periods. This module integrates with Equipment Control, Job Cost, Inventory Control, and General Ledger. In addition, Equipment Tracking is integrated with Document Imaging to scan and retrieve equipment documents.
To set up Equipment Tracking, follow these steps:

1. Configure default settings and enter rate brackets for hourly,
 daily, weekly or monthly rates. (see Equipment Tracking Installation below).

1. Set up transaction codes, and apply standby rates and operating
 rates to equipment. (see Wquipment Tracking Maintenance below).

1. Issue and return equipment and enter recurring equipment charge
 transactions. (see Equipment Tracking Data Entry below).

Refer to the following sections for details.

## Equipment Tracking Installation

Go to System Administration > Installation > Equipment Tracking to set up the Equipment Tracking module. Options include default
 properties, rate brackets for hourly, daily, weekly or monthly rates, default phase and
 transaction codes, and user-defined fields.

- Complete the [Equipment Tracking Installation](/en/spectrum/spectrum/equipment/equipment-tracking/equipment-tracking-installation) screens.

## Equipment Tracking Maintenance

Use the Maintenance screens to set up transaction codes, and apply
 standby rates and operating rates to equipment. Some of these screens have been
 pre-configured in the default company, but you can add additional items for your
 company's equipment as needed.

- Use the [Transaction Code Maintenance](/en/spectrum/spectrum/equipment/equipment-tracking/spectrum-menus/maintenance-overview/transaction-code-maintenance) screen to classify a transaction as billable, and set the G/L debit account
 code for transactions.

- Use the Equipment Charge Rates screen to add standby rates for a specific
 piece of equipment.

- Use the Job-Specific Equipment Charge Rates screen to add operating rates
 for a specific piece of equipment.

## Equipment Tracking Data Entry

Use the Data Entry screens to issue and return equipment, and enter
 recurring equipment transactions for current jobs.

- Use the [Deployment Log](/en/spectrum/spectrum/equipment/equipment-tracking/spectrum-menus/data-entry-overview/deployment-log) screen to issue and return equipment.

- Use the [Recurring Equipment Charge Processing](/en/spectrum/spectrum/equipment/equipment-tracking/spectrum-menus/data-entry-overview/recurring-equipment-charge-processing) screen to enter recurring equipment transactions for
 current jobs.

## Equipment Control Reports
