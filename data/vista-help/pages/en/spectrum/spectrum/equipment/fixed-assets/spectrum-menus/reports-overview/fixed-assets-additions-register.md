# Fixed Assets Additions Register

The Fixed Assets Additions Register prints a list of all
 assets added to the system since the last Additions Update was performed.
After the report has printed, the operator is provided the opportunity to purge the
 additions.
Important: This report should be retained as a permanent
 record of the company.
After the Fixed Assets Additions Register has printed, the
 Update screen appears.
