# Department Code File Maintenance

Department Code File Maintenance is used to maintain a file
 of department codes and descriptions.
 When specific assets are entered into the system using [Fixed Assets File Maintenance](/en/spectrum/spectrum/equipment/fixed-assets/spectrum-menus/maintenance-overview/fixed-assets-file-maintenance), one of the codes
 entered here will be assigned.
