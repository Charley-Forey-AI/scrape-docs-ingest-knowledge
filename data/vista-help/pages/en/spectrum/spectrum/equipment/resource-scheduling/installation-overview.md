# Installation Overview

From the System Administration menu, select Installation
 > Company.
Note that the System Administrator should have previously completed this screen.
 Double-check that the checkbox for Resource Scheduling module has been selected. After the
 Company Installation screen has been completed, click OK to save the information and return to
 the Installation list.
