# Meter Reading Transaction Entry

The Enter Equipment Meter Reading screen is used
 periodically to enter meter readings.
This screen is particularly useful for equipment usage that is entered by the hour, rather
 than by meter reading. Meter readings entered in Transaction Entry also display in this
 screen.
Warning messages display when a meter reading is entered with a higher than normal (expected) meter usage value for the period. In determining the normal usage value, the software first looks at the equipment type and meter type as set up in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen, and secondarily looks at the normal values stored in the [Equipment Control Installation – Properties](/en/spectrum/spectrum/equipment/equipment-control/equipment-control-installation-overview/equipment-control-installation--properties) screen.
Figures entered on this screen will need to be updated using Transaction Report. Once the update is run, these meter readings can be viewed in Equipment.
Tip: Because the meter reading can be used to determine when
 to schedule preventive maintenance, those users with the Preventive Maintenance module will
 want to make the most of this feature.
Enter a Batch code and then refer to one of the related topics below based on what you'd like to do next.
