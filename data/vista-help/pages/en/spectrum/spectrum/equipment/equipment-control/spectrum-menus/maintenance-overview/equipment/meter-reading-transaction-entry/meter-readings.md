# Meter Readings

Use this window to enter meter readings while recording
 equipment usage (EU) transactions and equipment cost (EC) transactions.
This window can also be used when recording stand-by time, although in this case, the hours
 are not computed from the change-in-meter.
After you complete your entries, the software will automatically create up to four meter transactions for the specified equipment. If the unposted meter reading already exists, the update will adjust the meter reading if changed. When multiple readings exist on the same day, the log entry date is used and readings display in log time order. If you need to delete a meter reading for any reason, you must use the [Meter Reading Transaction Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry) screen.
Important: If you are entering a base meter (beginning
 entry) for meters numbers 1, 2, 3, or 4, or if you are recording the replacement of a
 broken meter, you must use the Base or Replace buttons to access these entry screens. Do
 not use Meter Readings for base or replace entries.
