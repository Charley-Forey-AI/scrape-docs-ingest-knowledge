# Equipment

The Equipment screen is used to add, change, and delete
 equipment information.
The Equipment Type File Maintenance screen must be set up before equipment may be added on
 this screen.

- If you have security rights to add equipment, click the New button to open the [New Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/quick-equipment-setup-maintenance/new-equipment) window. Otherwise, in the Search field, enter a code or press Enter on your keyboard, select an equipment code, and then click the Edit button to open the selected employee in the Equipment Main Properties screen.

- The Update Location button allows you to enter the current location of a selected piece of equipment. You must have permissions to add or edit equipment in order to use this feature.

- Use the [Info
 Bar](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/info-bar/info-bar-faqs) on the left side of the screen to access other equipment-related
 screens such as Compliance, Warranty, Statistics, PM Schedule, Meter Readings,
 Ownership Costs, and more.
