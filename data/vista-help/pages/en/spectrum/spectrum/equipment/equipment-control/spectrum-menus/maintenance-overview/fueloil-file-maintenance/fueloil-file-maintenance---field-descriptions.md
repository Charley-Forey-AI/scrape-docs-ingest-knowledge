# Fuel/Oil File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Fuel/oil code
Enter a unique code to identify the fuel/oil. The codes established here may be used in the Equipment to indicate the kind of fuel and oil that are needed for specific pieces of equipment.

Properties

Description
Enter the fuel/oil description (for example, Halvoline, Regular, Unleaded, Diesel, and so forth).

Type
Select the fuel/oil type from the drop-down menu.

Unit of measure
Select the fuel/oil unit of measure from the drop-down menu (for example, Gallons, Quarts, or Liters).This list will vary based on the lexicon specified in the Company Installation screen.

Price
Enter the current fuel/oil price per unit of measure.
