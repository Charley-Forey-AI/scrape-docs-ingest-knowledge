# Yard File Maintenance

The Yard File Maintenance screen is used to identify the
 yard in which the equipment is currently located.
Equipment can be tracked to a specific yard and/or location within a
 yard. This allows you to print an equipment listing by yard/location, which helps in
 tracking inventory. Standards are set up in the Equipment screen. The codes set up in this
 screen are used in the Equipment Tracking module to transfer equipment from yard/location
 to another.
