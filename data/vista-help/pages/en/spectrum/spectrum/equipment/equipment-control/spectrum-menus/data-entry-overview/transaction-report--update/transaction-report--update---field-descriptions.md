# Transaction Report / Update - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Batch code
Enter the batch code for this transaction report, or type "ALL" to print all batches or press Enter to accept the default operator code. Double-click in this field to select from a list of valid batch codes.
 If you accept the default of ALL, the report will wait until all operators have finished entering their transactions prior to previewing or printing the report.

From/to transaction date
Enter the date range of the transactions you wish to include on this report.

Source
Select the source (or sources) or press Enter to print this report for all sources.
Select the meter readings checkbox only if meter readings, which have been entered (through Enter Equipment Meter Reading and/or Equipment Transaction Entry), should be updated to the Equipment Master File.

Usage/standby sort by

Meter report

My Reports button
To learn more about how to use the My Reports features, click here.

Meter Reading button
Click this button to open the global [Meter Readings](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry/meter-readings) window.
