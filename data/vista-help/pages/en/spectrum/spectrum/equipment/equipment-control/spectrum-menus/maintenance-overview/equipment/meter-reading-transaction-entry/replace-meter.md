# Replace Meter

This window allows you to track equipment usage without
 interruption when a meter has been broken and needs to be replaced.
Use this window to record the initial meter reading of the replacement meter. Fields in the
 Old meter section allow you to
 record the last reading of the old, broken meter.

- The base reading of the new meter is not interpreted as the usage of the equipment; instead, the base reading permits the calculation of the period-to-date and life-to-date usage when the initial meter readings are not calibrated to zero.

- After you click OK to save your entries, a meter transaction for the new base meter reading is created. If the Enter final reading for old meter checkbox is selected, a meter transaction for the final 'old meter' reading will also be created.
Important: Do not use this screen for establishing the original
 meter (base reading). When adding a new equipment code to Spectrum use the New Base
 Meter Reading Entry screen to record the new meter reading.
