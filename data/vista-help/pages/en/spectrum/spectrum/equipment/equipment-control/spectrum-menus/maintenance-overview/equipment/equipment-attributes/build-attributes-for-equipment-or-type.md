# Build Attributes for Equipment (or Type)

Use this window to assign attribute codes to the specified
 equipment code or equipment type.
The software will determine whether to show all codes or the
 type-specific list based on the Assign
 attribute codes for this equipment type checkbox setting in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen.

- When a list of all the codes available in this company
 displays, the column header in the left pane will say Attribute.

- When a type-specific list displays, the column header in the
 right pane will say Standard
 attribute.

How to use this screen:

1. Select the attribute you want to apply to the current
 equipment from the left pane and click the Select button to move the attribute
 to the pane on the right . Multiple attributes may be selected at once by holding
 down the Ctrl key while
 clicking on your choices.

1. The attributes (if any) that appear in the right pane
 upon opening this window are the attributes previously-assigned to the specified
 equipment. To remove an attribute prior to performing the update, select the
 attribute from the right pane and click Deselect.

1. Once the list of attributes that you want to add to
 this equipment is finalized in the right pane, select OK to run the update. If any of the
 attributes includes a picture, the picture path will also be added to the equipment's
 attribute record when the update is run.
