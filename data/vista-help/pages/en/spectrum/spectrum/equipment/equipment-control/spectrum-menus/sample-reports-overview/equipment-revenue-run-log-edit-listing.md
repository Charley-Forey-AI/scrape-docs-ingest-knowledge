# Equipment Revenue Run Log Edit Listing

This report sorts primarily by Equipment Usage (EU), then No Charge, then by Errors Found. Records with errors include a message line describing the nature of the error.
Note: All No charge entries display in the No Charge section, even if errors would
 exist for the record had the entry be set to chargeable (for example, due to an invalid
 equipment code).
