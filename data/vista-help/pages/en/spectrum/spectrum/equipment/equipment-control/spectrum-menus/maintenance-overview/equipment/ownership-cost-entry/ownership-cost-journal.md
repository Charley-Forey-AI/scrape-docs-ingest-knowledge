# Ownership Cost Journal

The Ownership Cost Journal screen is used to print a list of all equipment for which this information was calculated using Build Ownership Cost Transactions.
Review the report for accuracy and, if correct, proceed with the Ownership Cost Journal. Otherwise, make appropriate adjustments through the Ownership Cost Entry screen, then reprint this list.
Important: This report should be retained as a permanent
 audit record of the company.
