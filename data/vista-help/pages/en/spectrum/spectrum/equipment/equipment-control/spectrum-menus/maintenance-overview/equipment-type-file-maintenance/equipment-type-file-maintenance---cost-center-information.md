# Equipment Type File Maintenance - Cost Center Information

If cost centers are being used in the current company, when
 adding or changing an equipment type code, Spectrum allows entry of a G/L account code only if
 you have permission to assign that code.
The system compares the G/L account's list of shared cost centers with cost centers in
 your operator's assigned cost center scheme, and if there are no common cost centers,
 then that G/L account cannot be assigned.
Once a G/L account code has been assigned to an equipment type (by an authorized operator), then that G/L account will be used by all operators for processing involving that equipment type code.
