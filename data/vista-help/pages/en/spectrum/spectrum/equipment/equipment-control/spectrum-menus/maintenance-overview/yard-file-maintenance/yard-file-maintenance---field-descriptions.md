# Yard File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Yard code
Enter the yard code, or press F4 to select from a list of valid yard codes.

Description
Enter a yard description, or press Enter to accept the corresponding default yard description.

Latitude Longitude
These fields are optional, but allow you to enter precise location coordinates.
