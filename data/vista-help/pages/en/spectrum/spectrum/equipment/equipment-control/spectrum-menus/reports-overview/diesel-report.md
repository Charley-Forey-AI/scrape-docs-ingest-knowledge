# Diesel Report

The Diesel Report provides information on diesel entries
 for on-road and off-road equipment.
There are several parameters that can be specified to print the exact report needed.
By default, this report is sorted by equipment code (numerically, then alphabetically). This report can also be sorted by equipment type.
Note: The unit of measure types (for example, Gallons vs. Liters) on
 this report will vary based on the lexicon specified in the Company Installation
 screen.
