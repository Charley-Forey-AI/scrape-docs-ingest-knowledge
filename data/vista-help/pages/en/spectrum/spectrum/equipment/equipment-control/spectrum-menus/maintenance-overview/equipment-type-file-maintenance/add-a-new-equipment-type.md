# Add a New Equipment Type

Follow these steps to add a new equipment type.

1. Navigate to Equipment Control > Maintenance > Equipment Type File Maintenance.

1. Select New .

1. Enter the Equipment type code, up to two characters.

1. Complete the Properties tab.

- Enter the Description, up to 30 characters.

- Select the Class.

- If this type of equipment can be attached to another type for the purposes of equipment usage entry, select the Allow this type of equipment to be attached to other equipment codes? checkbox.

- If this type of equipment will be managed using the Resource Scheduling module, select the Utilize move scheduling for this type of equipment? checkbox.

- If you want to track this tool or equipment using Hilti ON!Track, select the Hilti OnTrack? checkbox. For more information, see [Create a Hilti Equipment Type](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/set-up-the-hilti-ontrack-integration-in-spectrum/create-a-hilti-equipment-type).

- If you want Spectrum to provide warnings when the actual usage exceeds the average weekly usage, enter the Normal hours/week, Normal miles/week  and Normal kilometers/week accordingly.

- Under Require type-specific codes in equipment maintenance, select each checkbox according to the lists you want to create for this equipment type. Each checkbox you select enables its own tab.

1. Complete the G/L Codes tab.

- For Depreciation, enter the Expense (debit) G/L code and Contra (credit) G/L code. The expense code must be set to direct equipment cost and the credit must be set to no direct cost.

- For License, enter the Expense (debit) G/L code and Contra (credit) G/L code. The expense code must be set to direct equipment cost and the credit must be set to no direct cost.

- For Insurance, enter the Expense (debit) G/L code and Contra (credit)  G/L code. The expense code must be set to direct equipment cost and the credit must be set to no direct cost.

1. If applicable, complete the Weight Classes tab.

- Click New.

- Select the appropriate Weight class code.

1. If applicable, complete the Standard Attributes tab.

- Click New.

- Select the proper Attribute code.

- To automatically add this attribute to a new piece of equipment, select the Add this attribute automatically to new equipment? checkbox.

- If you want to add an image, click the Picture button.

1. If applicable, complete the Standard Components tab.

- Click New.

- Select the appropriate Component code.

- To automatically add this item to a new piece of equipment, select the Add this component automatically to new equipment? checkbox.

- If you want to add an image, click the Picture button.

1. If applicable, complete the Standard Statistics tab.

- Click New.

- Select the appropriate Statistic code.

- Modify the Standard instructions, Standard norm, Standard minimum, and Standard maximum as needed.

- As needed, enter User-Defined fields.
