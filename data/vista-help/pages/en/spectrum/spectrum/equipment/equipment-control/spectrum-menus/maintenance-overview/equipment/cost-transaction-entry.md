# Cost Transaction Entry

Use this screen when you are entering cost transactions,
 including EC, AP, and PR transaction types.
Revenue transactions should be entered using the [Revenue Transaction Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/data-entry-overview/revenue-transaction-entry) screen.
After completing the Transaction date and Batch fields, click one of the available tabs (EC, AP, or PR).
