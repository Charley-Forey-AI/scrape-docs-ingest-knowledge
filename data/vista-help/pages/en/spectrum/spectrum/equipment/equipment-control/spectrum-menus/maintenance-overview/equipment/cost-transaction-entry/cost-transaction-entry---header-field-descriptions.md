# Cost Transaction Entry - Header Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Fields/Buttons
Descriptions

Transaction date
The current date displays, or select the desired
 transaction date using the calendar displayed in this window, or use the
 blue arrows to search for and select the desired date. The transaction date
 must be within the range of the minimum and maximum Equipment Control
 processing dates. This is the date that will be used in Equipment Control
 History, General Ledger, and, if applicable, Job Cost and Time + Material.


Batch
The current batch defaults. Press Enter to accept this
 batch, or enter another batch code to identify this group of
 transactions.

New/Edit/Delete buttons
These buttons display after you have selected a
 transaction type tab. Use this action buttons to add, edit, or remove
 transactions.

Meter Reading button
Click this button to enter [Meter Readings](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry/meter-readings) while recording
 equipment cost (EC) transactions.
