# Revenue Transaction Entry - Header Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Transaction date
The current date displays, or select the desired transaction date using the calendar displayed in this window, or use the blue arrows to search for and select the desired date. The transaction date must be within the range of the minimum and maximum Equipment Control processing dates. This is the date that will be used in Equipment Control History, General Ledger, and, if applicable, Job Cost and Time + Material.

Batch
The current batch defaults. Press Enter to accept this batch, or enter another batch code to identify this group of transactions.

Update History button
Click this button to display the [Transaction Report / Update](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/data-entry-overview/transaction-report--update) starting screen. This screen allows the selection of a range of transaction dates to print and update.

Update G/L button
Click this button to display the [G/L Detail Report](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/cost-transaction-entry/gl-detail-report) starting screen. The G/L Detail Report screen is used to print and review equipment revenue information prior to updating it to the General Ledger.

New/Edit/Delete buttons
These buttons display after you have selected a transaction type tab. Use this action buttons to add, edit, or remove transactions.

Meter Reading button
Click this button to enter [Meter Readings](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry/meter-readings) while recording equipment cost transactions.
