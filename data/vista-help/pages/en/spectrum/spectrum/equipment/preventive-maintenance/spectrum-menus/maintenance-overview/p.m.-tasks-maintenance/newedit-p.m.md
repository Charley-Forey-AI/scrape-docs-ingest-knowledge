# New/Edit P.M. Task

Use this window to add or modify task information.

## Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Task code
Task code is the Key field for
 the task file. This code distinguishes each task record from all other
 task records in the file. When maintenance is scheduled, these tasks will
 be used to indicate the work to be done on each specific piece of
 equipment.A note about using numeric coding:
 Because many users prefer alphabetic or combination alpha and numeric
 coding, this code is not a numeric-only field. Because of this, the
 code left-justifies when entered. Users preferring a strictly numeric
 coding scheme should be advised to use leading zeros when adding codes
 in order to produce desired sort result on screens and reports.
 Without leading zeros, '1', '10', '100' will appear before '2', '20',
 '200'. Instead, codes should be set up '001', '002', and so forth. If
 more than 1000 codes are anticipated, enter '0001', '0002', and so
 forth.

Description
Type a full description of the
 task.
Frequency method

Select one of the frequency methods listed below.
 Changing the frequency in this screen does not affect entries already
 recorded in Data Entry.

Meter reading basis
(M) Maintenance due
 based on Meter reading only (for example, every 3000 miles)

- Elapsed days

(ED) Maintenance due
 after Elapsed Days only (for example, every 90 days)
Meter or elapsed days
(MED) Maintenance due
 based on Meter or Elapsed Days, whichever comes first

- Fixed date

(FD) Maintenance due at
 a Fixed Date each year (for example, 09/15)
Meter
This field is only available when
 the frequency method is Meter reading basis or Meter or elapsed days.
 Enter the default meter number when the maintenance will be due. When
 this task is assigned to specific pieces of equipment through [Equipment P.M. Setup](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/equipment-p.m.-setup), this
 information will default. In Equipment P.M. Setup, when the actual
 maintenance schedule is different, this default can be
 overwritten.
Number of days
This field is only available when
 the frequency method is Elapsed Days or Meter or elapsed days.
 Enter the default number of days when the maintenance will be due. When
 this task is assigned to specific pieces of equipment through Equipment
 P.M. Setup, this information will default. In Equipment P.M. Setup, when
 the actual maintenance schedule is different, this default can be
 overwritten.
Fixed date
This field is only available when
 the frequency method is Fixed Date. Enter the default fixed date for this
 task in which maintenance will be due. When this task is assigned to
 specific pieces of equipment through Equipment P.M. Setup, this
 information will default. In Equipment P.M. Setup, when the actual
 maintenance schedule is different, this default can be
 overwritten.
Labor

Labor hours
Enter the default number of labor
 hours required for this task. When this task is assigned to specific
 pieces of equipment through Maintenance Schedule Entry, this information
 will default. In Equipment P.M. Setup, when the
 amount of time required to complete this task differs from the normal
 number of hours entered here, this default can be
 overwritten.

Labor rate
Enter the default labor rate
 required for this task. When this task is assigned to specific pieces of
 equipment through Equipment P.M. Setup, this information will default. In Equipment P.M. Setup, when the labor rate paid
 to complete this task differs from the normal rate entered here, this
 default can be overwritten.
