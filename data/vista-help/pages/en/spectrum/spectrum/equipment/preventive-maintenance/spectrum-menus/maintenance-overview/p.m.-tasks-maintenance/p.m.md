# P.M. Tasks Maintenance - Field Descriptions

Use the table below when completing the fields on this
 screen.
Fields/buttons
Descriptions

Click New or Edit on a selected task to
 open the [New/Edit P.M. Task](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/p.m.-tasks-maintenance/newedit-p.m.-task)
 window.
List box
The List box displays tasks that have
 been entered or modified in the New/Edit P.M. Task window. Once entered, they
 can be selected in this screen as tasks to be performed on specific pieces of
 equipment.
