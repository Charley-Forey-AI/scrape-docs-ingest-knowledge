# Equipment Work Order Cost Analysis

Use this screen to conveniently review total costs by cost category type. You can view actual cost details and cost category sub-totals.
Note: This screen displays records for equipment with or
 without an associated work order, as long as there are costs. Image files cannot be
 attached to these equipment-only records.
