# Batch P.M. Service Entry

Use this screen to indicate that preventive maintenance
 tasks have been performed.
Note: Duplicate entries are not allowed.
If the Canadian lexicon designation (en-ca) is selected in Company
 Installation, labels on all meter types will display using the preferred Canadian
 measurements (for example, "kilometers" instead of "miles").
