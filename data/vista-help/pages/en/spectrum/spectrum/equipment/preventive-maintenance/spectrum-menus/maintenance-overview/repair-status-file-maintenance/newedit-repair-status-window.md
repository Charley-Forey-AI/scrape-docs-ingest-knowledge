# New/Edit Repair Status window

Use this screen to set up and maintain a list of repair
 status codes.
Repair status codes are located on the [Repair Status File Maintenance](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/repair-status-file-maintenance) screen.
Tip: Use the New Repair Status window to add new equipment
 repair status codes. Use the Edit Repair Status window to make changes to existing
 equipment repair status codes. As entries are completed in these windows, the data will
 display on the Repair Status File Maintenance starting screen.
Field
Description

Repair status code
Enter up to 10 characters for the new
 repair status code.
Description
Enter up to 50 characters for a brief
 description of the repair status code.In the Edit window,
 make any required changes to the existing repair status code's
 description.
Note: If a long
 description is entered in this field, the description may appear truncated
 in areas of the software where space is limited.

Flag color
Use the drop-down arrow to select a
 flag color to use for the repair status code selected. Select <blank> if
 you do not want to add a flag.
Use custom icon (where applicable) in place of standard icon and color
If you select this checkbox, the
 Select Custom Icon window displays. Use this window to select a custom icon to
 associate with the repair status code.
