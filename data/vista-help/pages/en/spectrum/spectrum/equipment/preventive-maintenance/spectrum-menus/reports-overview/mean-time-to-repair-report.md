# Mean Time To Repair Report

The Mean Time To Repair Report is a management report that
 shows how long it takes to repair the equipment.
This will be measured by computing how many days an equipment work order is open, that is, the
 date the work order was created compared with the date the work order is finally repaired
 and the equipment is ready for use again. This report helps determine the cost of a typical
 failure and the effectiveness of maintenance improvements.
