# New/Edit Priority window

Use this screen to set up and maintain a list of priority
 codes.
Priority codes are listed on the [Priority File Maintenance](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/priority-file-maintenance) screen.
Tip: Use the New Priority window to add new priority codes.
 Use the Edit Priority window to make changes to existing equipment priority codes. As
 entries are completed in these windows, the data will display on the Priority File
 Maintenance starting screen.
Field
Description

Priority code
Enter up to 10 characters for the
 priority code.
Priority description
Enter up to 50 characters for a brief
 description of the priority code.In the Edit window, make
 any required changes to the existing priority code's description.
Note: If a long description is entered in this
 field, the description may appear truncated in areas of the software where
 space is limited.

Flag color
Use the drop-down arrow to select a
 flag color to use for the priority code selected. Select <blank> if you
 do not want to add a flag.
Use custom icon (where applicable) in place of standard icon and color
If you select this checkbox, the
 Select Custom Icon window displays. Use this window to select a custom icon to
 associate with the priority code.
