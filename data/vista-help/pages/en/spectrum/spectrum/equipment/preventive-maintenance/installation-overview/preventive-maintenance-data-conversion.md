# Preventive Maintenance Data Conversion

Below is a summary of the steps needed to complete installation. It is important that these steps be performed in the order listed.
Conversion Summary

- Complete the Preventive Maintenance Installation screen. Install the Equipment
 Control module, if not already installed, following instructions in the Equipment
 Control Help file and the Getting Started guide.

- Set the processing date to the conversion date using Processing Date Change.

- Set the processing dates to the new period using Processing Date Change.

- Enter the equipment types, if necessary. These codes were set up
 during Equipment Control conversion, using Equipment Type File Maintenance.

- Print the equipment type file listing and review for accuracy.

- Print the fuel/oil file listing and review for accuracy.

- Enter equipment information, if necessary, using Equipment File
 Maintenance.

- Print the Equipment File Listing and review for accuracy.

- Enter the individual Preventive Maintenance tasks in P.M. Tasks
 Maintenance.

- Print the Task File Listing and check for accuracy.

- Enter a category for parts using Category File Maintenance.

- Enter parts used for equipment repairs in the Inventory Item File
 Maintenance screen.

- Print an Inventory Item File Listing and check for accuracy.

- Enter the schedule at conversion using Maintenance Schedule Entry.

- Print a P.M. Tasks Maintenance Listing and check for accuracy. If
 some pieces of equipment have similar maintenance needs, they may be copied using Copy
 P.M. Schedule.

- Enter outstanding equipment work orders as of the conversion date
 using Equipment Work Order Entry.
