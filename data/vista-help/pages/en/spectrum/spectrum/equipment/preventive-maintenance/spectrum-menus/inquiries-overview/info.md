# Info

This window displays detailed information about the maintenance of the piece of equipment selected on the Maintenance Schedule Inquiry screen.
Field
Description

Frequency
The frequency with which the selected
 task is to be performed displays. This information defaults from the P.M. Tasks
 Maintenance screen.
Last done
The date the meter reading or
 maintenance was last performed displays. No entry is allowed.
Service date
A description of the required task
 displays. No entry is allowed.
Labor hours
The default number of labor hours
 required for this task display. No entry is allowed.
Labor rate
The default labor rate required for
 this task displays. No entry is allowed.
