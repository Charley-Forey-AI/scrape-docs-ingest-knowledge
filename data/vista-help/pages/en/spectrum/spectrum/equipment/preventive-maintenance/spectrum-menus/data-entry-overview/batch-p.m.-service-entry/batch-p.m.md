# Batch P.M. Service Entry - Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Batch
Your operator code defaults in this
 field, but you can change it to a different operator code or date stamp. You
 can also search for a current batch code by clicking the search arrow. You
 cannot enter ALL.
New/Edit/Delete
Add, modify, or delete a batch
 entry.
Grid

Equipment
The Equipment code entered here must
 have been previously set up in Equipment Control > Equipment File Maintenance.
Task
Enter or search for the task code,
 which must have been previously set up in [P.M. Tasks Maintenance](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/p.m.-tasks-maintenance).
P.M. date
The current preventive maintenance
 processing date defaults. You can enter or search for a new date in this
 required field.
Meter #1 reading
Enter the meter reading at the time of
 this service.
Um
The unit of measure (hours, miles, and
 so on) displays.
Personel
Enter the name of the person who
 performed this service.
Parts
Click the arrow to open the [P.M. Service - Parts Used](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/batch-p.m.-service-entry/p.m.-service---parts-used) window.
Summary comment
Enter a brief comment pertaining to
 this entry.
Notes
Click the arrow to open the [P.M. Service Performed Notes](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/batch-p.m.-service-entry/p.m.-service-performed-notes) and view or
 enter notes about this service.
Equipment description
The descriptions of the
 equipment and tasks entered in the first two columns display.
