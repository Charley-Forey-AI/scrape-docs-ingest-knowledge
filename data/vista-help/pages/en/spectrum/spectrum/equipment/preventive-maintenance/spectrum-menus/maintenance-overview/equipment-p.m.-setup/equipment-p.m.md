# Equipment P.M. Setup - Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Equipment code
Enter the equipment code for this
 maintenance schedule. The code must be for a piece of equipment previously set
 up in Equipment Control > Equipment File Maintenance.
Task Description
The task code to add to the schedule
 must have been defined in P.M. Tasks
 Maintenance.
Frequency method
Select one of the following:

- Meter reading basis (M)

- Elapsed Days (ED)

- Meter or elapsed days (MED)

- Fixed date (FD)

Elapsed meter
If the frequency method is M or MED,
 enter the meter number.
Elapsed days
If the frequency method is ED or MED,
 enter the number of days that have elapsed.
Fixed date
If the frequency method is FD, enter
 the month and day in MM/DD format.
Instructions
Click in this field to open the [P.M. Instructions](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule/p.m.-instructions) window. Yes indicates that notes are
 present.
Parts list
Click in this field to open the [Standard Parts List](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule/standard-parts-list)
 window. Yes indicates
 that parts are present.
Std labor hours Std labor rate
Enter or modify the standard labor
 hours and rates.
Last P.M. date
Enter or search for the last date of
 preventive maintenance.
Last meter #1 reading
Enter or modify the meter reading the
 last time this maintenance task was performed.
Printed?
This checkbox is used in conjunction
 with the [P.M. Schedule Report](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/reports-overview/p.m.-schedule-report), which lists the maintenance tasks to be performed
 on a particular piece of equipment. On the report start screen, the Unprinted option allows you
 to select only those tasks that have not been printed. It is assumed that
 printing the report indicates that the task has been assigned to a mechanic or
 technician. By selecting tasks that have not been printed, the P.M. Schedule
 Report effectively displays the tasks that have previously been assigned.The Printed? checkbox is automatically selected after this task
 has been previewed on the P.M. Schedule Report. This prevents you from
 assigning the same task twice.
This checkbox is
 cleared when the P.M. Service Performed Update is run.
Repeat entry for additional lines. When all maintenance
 tasks have been defined for this piece of equipment, click
 OK.
