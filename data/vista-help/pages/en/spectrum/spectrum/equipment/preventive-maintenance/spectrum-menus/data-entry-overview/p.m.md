# P.M. Processing Date Change

P/M Processing Date Change is used to change the date under which this module operates. The current date, fiscal year, periods, period end dates and previous period end dates may be set for default or changed.
Fields
Descriptions

Current settings

Processing date
The current processing date displays.
 Enter the new processing date by typing over the displayed date. To leave the
 processing date without making any changes, click Cancel.If the new date is before the Minimum Processing Date or after the Maximum
 Processing Date established during the installation procedures, the
 following error message displays:
DATE ENTERED IS
 NOT WITHIN THE CURRENT MINIMUM AND MAXIMUM DATE RANGE.
