# Installation Overview

The following is a summary of how to complete installation.
Click the Admin button on the toolbar of any menu; the Installation menu will appear. On the Installation menu, double-click Company Installation. Note that the System Administrator should have previously completed this screen.
Ensure that the checkbox for Preventive Maintenance has been checked.
After the Company Installation screen has been completed, click OK to save the information and return to the Installation list.
