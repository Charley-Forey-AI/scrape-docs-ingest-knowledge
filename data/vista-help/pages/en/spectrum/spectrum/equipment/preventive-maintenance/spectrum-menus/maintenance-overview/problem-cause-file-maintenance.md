# Problem Cause File Maintenance

Use this screen to set up and maintain a list of problem
 causes to be used in the New/Edit Equipment Work Order window.
The codes defined on this screen represent standard types of problem
 causes that require investigation or equipment repair.
Problem causes can also be useful for inquiries and reports.
Fields
Descriptions

New/Edit/Delete
Click New or Edit on a selected line to
 open the [New/Edit Problem Cause Window](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/problem-cause-file-maintenance/newedit-problem-cause-window)
 window.
