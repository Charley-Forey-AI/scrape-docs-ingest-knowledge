# Copy P.M. Schedule

This screen is used to ease scheduling of maintenance on similar pieces of equipment. After a maintenance schedule has been entered for one piece of equipment, that information can be copied to other equipment.
Fields
Descriptions

Copy from equipment code
The Equipment code entered here must
 have been previously set up in Equipment Control > Equipment File Maintenance. For more information on that screen, click on the Spectrum Help
 link from Equipment File Maintenance.Enter or search for the code of the piece of equipment to
 copy the schedule to.

Properties

Year & make
The year and make, model,
 type of equipment, and last job code and description display from
 Equipment File Maintenance.

Copy to

Copy to code or type
Select Code to copy the specified
 schedule to a specific piece/group of equipment by specifying the equipment
 codes. Select Type to copy the schedule to a specific
 equipment type (for example, all Backhoes).

Copy to equipment code/type
The code or type entered here must have
 been previously set up in the Equipment File Maintenance
 screen.If you selected Code above, enter the code of
 the equipment to copy this schedule to. If you selected Type, enter the type
 of equipment to copy this schedule to.
