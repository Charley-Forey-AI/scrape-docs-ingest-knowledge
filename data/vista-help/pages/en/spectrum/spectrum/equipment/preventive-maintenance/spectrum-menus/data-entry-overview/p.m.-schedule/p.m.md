# P.M. Instructions

Use this window to view or edit task notes for the selected
 piece of equipment.
The NotePad will not display if you do not have security authorization
 for the [P.M. Schedule](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule) screen.
You can access this window from P.M. Schedule and [Equipment P.M. Setup](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/equipment-p.m.-setup).
