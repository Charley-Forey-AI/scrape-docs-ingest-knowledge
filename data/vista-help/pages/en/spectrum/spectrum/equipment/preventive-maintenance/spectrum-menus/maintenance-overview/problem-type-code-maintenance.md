# Problem Type Code Maintenance

Use this screen to set up and maintain a list of problem
 types to be used in the New/Edit Equipment Work Order window. The codes defined on this screen
 represent standard types of problems that require investigation or equipment
 repair.
Problem types can also be useful for inquiries and reports.
Fields
Descriptions

New/Edit/Delete
Click New or Edit on a selected line to
 open the [New/Edit Problem Type Code window](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/problem-type-code-maintenance/newedit-problem-type-code-window)
 window.
