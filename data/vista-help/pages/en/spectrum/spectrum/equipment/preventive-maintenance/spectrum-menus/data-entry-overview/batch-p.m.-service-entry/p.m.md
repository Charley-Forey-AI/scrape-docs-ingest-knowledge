# P.M. Service Performed Notes

Use this window to enter or review any notes documenting what steps were taken to complete the performed service. The NotePad will not display if you do not have security authorization.
You can access this window from [P.M. Schedule](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule), [Batch P.M. Service Entry](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/batch-p.m.-service-entry),and [Equipment P.M. Service History](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/inquiries-overview/equipment-p.m.-service-history).
Note: If two users are editing the same service note at the
 same time, the note will be saved by the last user to click OK.
