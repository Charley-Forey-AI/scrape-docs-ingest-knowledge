# Equipment Tracking Installation

Whenever a module is added in Spectrum, two steps are required: the module must be specified in the Company Installation screen and the module-specific installation must be completed.

## Part I: Select the Equipment Tracking module in Company Installation

1. On the Site Map, select System Administration > Installation > Company.

1. Select the Equipment Tracking? checkbox.

## Part II : Complete the Equipment Tracking Installation

1. Go to System Administration > Installation > Equipment Tracking.

1. Complete the Installation settings, using the Help if you need specific information for a field.
