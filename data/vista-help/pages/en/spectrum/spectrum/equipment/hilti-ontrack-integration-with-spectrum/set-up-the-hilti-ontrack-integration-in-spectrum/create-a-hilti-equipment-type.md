# Create a Hilti Equipment Type

Create one or more Hilti-specific equipment types that are
 used in Spectrum to identify the tools
 being tracked by Hilti.
Set up the integration
 between Hilti and Spectrum first. For
 guidance, see [Enable Hilti ON!Track in Equipment Tracking Installation](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/set-up-the-hilti-ontrack-integration-in-spectrum/enable-hilti-ontrack-in-equipment-tracking-installation).

1. In Spectrum, go to Equipment Control > Maintenance > Equipment Type.

1. Select New.

1. In the equipment Type field, enter a code you
 want to use for Hilti equipment tracking.

1. Under Properties, add a description to indicate
 that this is for Hilti equipment tracking, and select the Hilti OnTrack?
 checkbox.Checking this checkbox is necessary for the integration to
 work properly. Equipment in Spectrum with a Hilti-specific equipment type will be
 tracked and managed by Hilti ON!Track, *not* by Spectrum's standard equipment tracking.
If you have not enabled the integration, this Hilti
 checkbox will not display.
Note: You cannot change or edit the equipment type for equipment that is currently issued on the Spectrum Deployment Log. You will see the following error message:ERROR - Equipment type cannot be changed when asset is currently issued on the Deployment Log.

1. Select Save.

For more information about setting up equipment types
 in Spectrum, see [Add a New Equipment Type](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance/add-a-new-equipment-type).
