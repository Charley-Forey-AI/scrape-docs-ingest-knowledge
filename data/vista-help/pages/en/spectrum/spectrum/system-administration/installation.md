# Installation

In order for new users to begin entering company information and installing the appropriate modules on the system, it is necessary to access the Installation Menu.

## Module Setup

Module setup precedes data conversion/migration and begins with the module's Installation screen. To begin installing any module, go to System Administration > Installation.

1. Before installing any module, begin with Company Installation. Enter the basic company information in the Properties tab. If needed, refer to [Company Installation - Properties tab](/en/spectrum/spectrum/system-administration/installation/company-installation---properties-tab).

1. Install individual modules.The online Help has information for installing each module. After opening any module listed under System Administration > Installation, select the Help button to open the relevant page.

1. Begin data conversion. Refer to the detailed conversion checklists for more information on getting started.

## Security

An inherent part of installing, using, and maintaining your software is setting up security and maintaining those settings. To get started, see [Security](/en/spectrum/spectrum/system-administration/security).
