# Set Standard Security Options

Use this screen to reset Spectrum security to the original, standard settings. This is especially helpful if your company has previously customized the security settings but needs to return to the standard settings for function levels 1-9 for any reason.
Note: Because function security settings are shared across all Spectrum companies, this update resets security levels for all companies. Be sure this is what you want before continuing.

To run the utility:

1. On the Site Map screen, open a new tab and select System Administration, then choose the Utilities option.

1. Select Set Standard Security Options.

1. Select the Delete function-specific operator security overrides checkbox.

1. Select Continue to revert to standard settings.

All security settings for all companies are reverted to their original settings.
