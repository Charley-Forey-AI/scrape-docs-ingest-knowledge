# Company Installation - Properties tab

The Company Installation page is used to set up the name
 and address information for a company or division.
This is the first maintenance page used to set up a new company. It may
 also be used after initial installation when, for example, the company's address changes,
 or additional modules are purchased for the Spectrum system. After using this page to set
 up a new company, new users should proceed immediately to the General Ledger Installation
 page. It is important to complete the General Ledger Installation page even if G/L module
 is not installed on the system.
After the Company Installation page has been completed, click the OK
 button to save the information and return to the Installation list.
Fields/Buttons
 Descriptions

Company Info
 At the Company field, the previously entered company/division code
 displays. No entry is allowed. Note that company codes should not contain
 any special characters (ampersands, slashes, commas, dashes, etc.).
Complete the remaining fields for this company, including
 the company Name
 field, Address
 fields that will be used by Spectrum for certain reports, such as the A/R
 statement and the Payroll W-2; and the Phone field.

Suppress quotation marks around e-mail addresses?
 By default, Spectrum formats outgoing messages using
 single quotes around the address, inside the angle braces. However, this
 formatting may be problematic for some e-mail servers. This checkbox allows
 you to specify the e-mail format that will work in conjunction with your
 particular e-mail server.

Show contacts for current company only?
 Select this checkbox to limit what contacts show up for
 this company. This limits what company (or companies) contacts show up, and
 is based on its association with a vendor code, employee code, job code, or
 side code in a given company.

Sync this company with Team
 If you are using the Viewpoint Team project management
 application, select this checkbox if you wish to synchronize new companies
 you create in Spectrum to Team.
