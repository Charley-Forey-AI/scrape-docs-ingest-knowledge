# Installing Modules

1. On the Site Map page, click System Administration  >  Installation  >  Company. The CompanyInstallation page displays.

1. Complete the Company Installation page and click the Modules tab.

1. Specify the modules you wish to install by selecting the appropriate checkboxes.

1. Click OK twice to save your changes and return to the Menu.
