# Sample Company Installation Options

The following examples illustrate the order in which the
 conversion process takes place in two different companies. In the first example, the
 conversion process is relatively simple and can be done all at once. In the second example,
 the company has a more extensive set of modules installed and must perform the conversion in
 two stages.

## Example A:

A company needs to install seven modules, including the basic five
 modules. The conversion process begins with file maintenance. Next, the data conversion
 takes place (note that the General Ledger conversion happens after the other four of the
 basic five modules have been converted). The table numbering shows the conversion
 process in the order it would be performed.
