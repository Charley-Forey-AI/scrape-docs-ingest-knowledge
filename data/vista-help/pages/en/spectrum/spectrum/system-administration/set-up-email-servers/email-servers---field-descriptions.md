# Email Servers - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldsDescriptions
Server nameEnter the name of the email server. Additional fields display after this field is completed.
Server typeSelect a server type. After making your selection, the remaining fields may be automatically set to the standard values pertaining to the selected server type.
Does this server require authentication?Select if connections to the server require a username/password to send with each request to the server.
Default from addressThe email address used when sending email messages from Spectrum. This is only used if the user has not set up their own email address in Operator Maintenance or Operator Preferences.
 Outgoing mail settings
HostnameThe hostname used with outgoing SMTP messages.
PortThe port for use with the outgoing hostname.
 Incoming mail settings
HostnameThe hostname used when connecting to the IMAP or POP3 server to download messages.
PortThe port for use with the incoming hostname.
SSL required?Select to indicate that the connections to the incoming server need to use SSL.
ProtocolSelect a mail protocol option: IMAP or POP3.
