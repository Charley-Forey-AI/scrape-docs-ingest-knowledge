# Installation Overview

Click the Admin button on the toolbar of any menu; the Installation list will appear. From the
 Installation list, select
 Company Installation. Note that the System
 Administrator should have previously completed this screen.
Ensure that the flag for this specific module has been checked and
 click OK to save the
 information and return to the Installation list. Next, select Inventory Control from the
 Installation list. When the installation screen is selected for the first time, the
 software will beep and a dialog box stating "CONTROL RECORD HAS NOT BEEN CREATED OK TO
 PROCEED" will display in the lower left corner on the screen. Once this installation screen
 has been completed and the information saved, the message will not appear again.
Spectrum provides an online help file for almost all the fields within
 this screen. Double click at the desired prompt to obtain detailed instructions for
 completing the fields and screens.
