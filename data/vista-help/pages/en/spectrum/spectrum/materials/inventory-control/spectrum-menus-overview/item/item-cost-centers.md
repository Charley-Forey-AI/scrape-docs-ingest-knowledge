# Item Cost Centers

Use the Item Cost Centers screen to define the cost centers that will include the selected inventory item code. This screen is available only if cost centers are enabled for the current company.
Highlight cost centers and then use the Select and Deselect buttons to move cost centers to and from the Available <cost center >  codes list and the Assigned <cost center >  codes list.
