# Physical Inventory Variance Report

Perform this procedure to select a sort option.
This report can be sorted by Warehouse or Item code.

1. On the Physical Inventory
 Reports screen, click the Selections button adjacent to the
 Variance report option.


1. Select a sort option and click OK.
