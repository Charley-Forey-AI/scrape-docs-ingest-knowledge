# Inventory Control Data Conversion

Learn how to complete the inventory control data
 installation.
Conversion
 Summary

1. Complete the Inventory Control Installation screen.

1. Set the processing dates to the new period for Inventory Control using I/C Processing Date Change.

1. Enter all warehouse codes using Warehouse File Maintenance.

1. Print warehouse codes and check for accuracy.

1. Enter G/L departments using G/L Department File Maintenance.

1. Print G/L departments and check for accuracy.

1. Enter all inventory categories (for example, Lumber, Nails, Plumbing Supplies, Rebar, and so forth) using Category File Maintenance.

1. Print inventory categories and check for accuracy.

1. Enter all standard discounts, if applicable, using Discounts File Maintenance. Steps 8 and 9 are only applicable if the Order Processing module is used.

1. Print standard discounts and check for accuracy. Steps 8 and 9 are only applicable if the Order Processing module is used.

1. Enter item location codes, if applicable, using Location File Maintenance.

1. Print item location codes and check for accuracy.

1. Enter area codes, if applicable, using Area Code File Maintenance.

1. Print area codes and check for accuracy.

1. Enter cost adjustment codes, if applicable, using Cost Adjustment Source Maintenance.

1. Print cost adjustment codes and review for accuracy.

1. Enter inventory item file information using Inventory Item File Maintenance. At the bottom of this screen, set up each warehouse where the item may be held. If the Trade Service Price Update will be used, it is not necessary to set up items since the import can automatically create the item records during the update.

1. Print inventory item file information and review for accuracy using Inventory Item File Listing and Item/Warehouse Detail Listing.

1. Enter mix components, if applicable, using Mix Maintenance.

1. If Order Processing module is present on your system: perform Part I of the Order Processing Conversion Worksheet before continuing with the Inventory Control Conversion. If Order Processing is not present, proceed immediately to step 22.

1. Enter quantity breaks, if applicable, using Quantity Breaks File Maintenance. Print and review the quantity break entries.

1. Enter customer special pricing, if applicable, using Customer Special Price Maintenance. Print and review the special pricing entries.

1. Set the processing date to the conversion date using Processing Date Change.

1. Confirm that the Inventory Controlreceipts checkbox is not selected in the General Ledger Installation screen.

1. Enter the balance forward quantities and costs for all inventory items in stock at the date of conversion. This information may be entered in batch format, as needed, using Receipts Entry.

1. Print the Inventory Receiving Report. Review and correct entries as necessary.

1. Print and update the inventory transactions using Inventory Transaction Report. Retain the printout.

1. Post to General Ledger using the Inventory G/L Summary Update,which follows the Inventory Transaction Update. Retain the printout.

1. Print and retain the summary Items-On-hand Valuation Report.

1. In the Post to G/L section of the General Ledger Installation – Properties tab, select the Inventory Control adjustments and Inventory Control receipts checkboxes.

1. If necessary, enter a journal entry to set the Inventory Asset account equal to the total from the Items-On Hand Valuation Report as of the conversion date by use of a Journal Entry. Update the journal entry.
