# Inventory Price Listing

The Inventory Price Listing screen is used to print a list
 of inventory items and the prices established for each item.
This report provides pricing information
 from Inventory Item File Maintenance. The report may be printed with or without standard
 cost information depending upon the intended use of the printout. A variety of selections
 are provided, such as category, item code, and ABC class code.
