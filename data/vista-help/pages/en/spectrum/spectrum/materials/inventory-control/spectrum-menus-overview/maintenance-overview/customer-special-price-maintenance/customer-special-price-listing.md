# Customer Special Price Listing

This screen prints a list of customer special price entries
 and related information entered during Customer Special Price Maintenance.
 This report contains only cost
 adjustment entries, not all cost entries. It is also possible to select only certain
 customers or items, or to specify a certain customer type when printing this report.
Note: This screen is relevant
 only if the Order Processing or Materials Management module is installed.
