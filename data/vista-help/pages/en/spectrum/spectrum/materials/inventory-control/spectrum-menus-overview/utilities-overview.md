# Utilities Overview

The Utilities menu is used to change item codes, copy mixes, copy master file information across companies, copy items to new warehouse, set up items in new warehouse, clear inventory quantities, perform a quick item delete, perform a quick item category delete, delete items in one warehouse, and average costs for items with negative buckets.
Important: Functions contained on the Utilities menu are normally reserved for use by supervisory and administrative personnel only.
