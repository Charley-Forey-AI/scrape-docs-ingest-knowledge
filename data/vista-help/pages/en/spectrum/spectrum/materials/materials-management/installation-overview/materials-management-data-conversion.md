# Materials Management Data Conversion

It is important to complete the Materials Management Data
 Conversion in the proper steps.

1. Complete Materials Management Installation. Also install Accounts Receivable
 and Order Processing, if not already installed.

1. Enter data format information using Scale Data Format Maintenance.


1. Preview the data formats and check for accuracy.


1. Enter warehouses using Inventory Control Warehouse File Maintenance
 or Price Update.

1. Enter plant/pit file information using Plant/Pit File Maintenance.

1. Preview the plant/pit information and review for
 accuracy.

1. Enter warehouse / G/L department codes using Warehouse GL Department
 Maintenance.

1. Preview the warehouse G/L department information and
 check for accuracy.

1. Enter hauler information, if applicable, using
 Hauler File Maintenance.


1. Preview the truck information and review for accuracy.


1. Enter truck information, if applicable, using
 Truck File Maintenance.


1. Preview the hauler information and check for accuracy.


1. Enter freight adjustment information, if applicable,
 using Freight Adjustment File
 Maintenance.

1. Preview the freight adjustment information and check
 for accuracy.

1. Enter item translations, if applicable, using Item
 Code Translation File
 Maintenance.

1. Preview the item code translation file information and
 check for accuracy.

1. Enter job/company information, if applicable, using
 Job / Company File
 Maintenance.

1. Preview the job/company file information and check for
 accuracy.

1. Enter area codes for job tickets, if applicable, using
 Inventory Control Area Code File
 Maintenance.

1. Preview the area code information and check for
 accuracy.

1. Enter jobs with special set up requirements, if
 applicable, using Job Contract File
 Maintenance.

1. Preview the Job Contract information and check for
 accuracy.

1. Enter customers with special set up requirements, if
 applicable, using Customer Material
 Contract Maintenance.Note: If using cost centers, click the Setup button and
 complete the associated Cost Center Setup screen. The Cost Center setup is
 Optional for all cost centers. A confirmation window displays prior to updating
 your changes.

1. Preview the Customer Contract information and check
 for accuracy.

1. Set the processing dates to the new period for
 Materials Management using Processing
 Date Change.
