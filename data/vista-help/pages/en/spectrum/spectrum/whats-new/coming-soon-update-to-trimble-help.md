# Coming Soon! Update to Trimble Help

To continue adding Help for even more products, and eventually additional resources such as knowledge base articles, we're updating the Trimble Help site in the weeks to come.
Right away, you'll notice a new layout and a new look and feel. Some updates we think you'll appreciate:

- The Trimble Help Assistant can provide quick answers to questions about your Trimble products. Access the assistant by selecting the following icon in the bottom right corner of the page.

- When viewing a page with multiple headings, the right-hand table of contents is now collapsed by default. Select the following icon to access it.

- The language selector is now available in the upper right-hand corner of the page. Note that translated Help is available for some but not all products.

- A feedback option at the bottom of all Help pages allows you to submit a feedback form directly to the Trimble Help team.

- An option to sign in using your Trimble ID. This lays the foundation for us to provide enhanced functionality in the future, tailored to Trimble users.

Once the initial update is complete, any links to the Help will continue to work as they do today.
For a preview of the update, you can visit the Trimble Help beta site at [trimblehelp.trimble.com](http://trimblehelp.trimble.com)
