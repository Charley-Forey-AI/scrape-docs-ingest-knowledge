# Service Contract Covered Equipment

Use this tab to create and manage any equipment related to
 a service contract.
From the Service Contracts Info
 Bar, select PropertiesCovered Equipment.
Fields/Buttons
Descriptions

Switch button
Click the Switch icon to conveniently
 open a different record without going through the Spectrum menu system.
New Equipment button
Click this button to open the [New/Edit Contract Equipment - Properties](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contracts/newedit-contract-equipment---properties)
 window.
Site code

Contract

Search
When first accessing the screen, all
 Active equipment automatically displays.
Status button
Use this button to define the list of
 records that display on the screen by selecting one or more record status
 types.
Add button
Click this button to open the Add
 Covered Equipment for Contract window. Here you can select from a list of All
 Site Equipment and use the Select and Deselect buttons to manage the list of Covered Equipment.
Edit button
Select a record and click this button
 to open the [New/Edit Contract Equipment - Properties](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contracts/newedit-contract-equipment---properties) window.
Remove button
Select a record and click this button
 to remove the equipment from the current contract. Any related component
 overrides will also be removed.
Components button
Click this button to open the Override
 Components for Covered Equipment window.

- All components for a piece of equipment are
 automatically covered when the equipment is added to a contract, unless
 components are specifically listed.

- When a piece of equipment has components, and no
 overrides are present, the window states 'All components are covered.'
