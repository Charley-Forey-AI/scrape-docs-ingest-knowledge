# Service Contract Work Order Revenue

The Service Contract Work Order Revenue window shows revenue breakdown for work orders referencing the selected contract.
The amounts in this window do not include unposted revenue for work order, and do not include revenue history that does not reference the selected contract. To view all activity associated with the work order, open the View Work Order window to see other work order transactions.
