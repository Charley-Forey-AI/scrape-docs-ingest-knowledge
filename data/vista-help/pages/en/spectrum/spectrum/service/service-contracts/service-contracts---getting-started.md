# Service Contracts - Getting Started

Access Service Contracts options from the Site Map and Service Contracts Info Bar. This module is
 integrated with Work Order and Accounts Receivable.

## Service Contracts Installation

Use the System Administration > Installation > Service Contracts menu to set up the Service Contracts module. Options include default
 properties (such as work order dispatch status and priority) and user-defined fields.
 See [Installation Overview](/en/spectrum/spectrum/service/service-contracts/installation-overview)
 for details.

## Service Contracts Maintenance

Use the Service Contracts > Maintenance menu to create and manage service contracts. In order to create a new
 service contract, you must first set up maintenance codes for contract types (for
 example, full maintenance, inspection, and warranty) and labor billing rates.
 Optionally, you can set up contract-specific labor billing rates in the Service
 Contracts module. See [Maintenance Overview](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview) for details.

- [Contract Type File Maintenance](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/contract-type-file-maintenance), which govern the billing source (work order or service contract) and establish
 the [About the Default Earned Revenue Basis for a Service
 Contract](/en/spectrum/spectrum/service/service-contracts/service-contracts---getting-started/about-the-default-earned-revenue-basis-for-a-service-contract) for a contract.

- If applicable, create a list of [Service Contract Description Maintenance](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contract-description-maintenance)
 that you can re-use across contracts.

- [Create Service Contracts](/en/spectrum/spectrum/service/service-contracts/service-contracts---getting-started/create-service-contracts). Follow the Info Bar links for the Service Contract module to complete
 contract setup. Note: Click the Status button in the upper right
 of the Service Contracts screen to filter or view contracts based on their status:
 Proposal, Active, Inactive, or Expired Proposal.

- [Service Contract Main Properties](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/service-contracts/service-contract-main-properties), including general contract, maintenance work
 order, notes, and exception information.

- If applicable, [Service Contract Covered Equipment](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/service-contracts/service-contract-covered-equipment), [New Contract Equipment - Components](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contracts/new-contract-equipment---components), and associated [New/Edit Contract Components - Warranty](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contracts/newedit-contract-components---warranty).

- If applicable, set up contract-specific [labor
 billing rates](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/labor-billing-rates-maintenance). These rates apply to labor performed on maintenance
 work orders outside of the scope of the service contract. If multiple labor billing rates exist, the system uses
 the [Labor Billing Rate Hierarchy](/en/spectrum/spectrum/service/work-order/spectrum-hierarchies/labor-billing-rate-hierarchy) (contract, site, customer, and
 standard rate) to determine the appropriate billing rates.

- [Create Service Contract Billings](/en/spectrum/spectrum/service/service-contracts/service-contracts---getting-started/create-service-contract-billings). You can calculate billing amounts manually or have the
 system calculate them automatically.

- [Service Contract Visits](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/service-contracts/service-contract-visits) for a contract.

- Review the [Service Contract Main Properties](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/service-contracts/service-contract-main-properties)
 for details about earned revenue, work order revenue, total revenue, total cost, and
 gross profit.

- [Service Contract Form](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/service-contracts/service-contract-form).

## Service Contracts Data Entry

Use the Service Contracts >  Data Entry menu to create and manage work orders, prepare proposals, schedule
 contract visits, print schedules, and bill service contracts. See [Data Entry Overview](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/data-entry-overview) for
 details.

- [Create Work Orders](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/data-entry-overview/create-work-orders).

- [Process Contract Billing](/en/spectrum/spectrum/service/service-contracts/service-contracts---getting-started/process-contract-billing).

- If applicable, [Earned Revenue Processing](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/data-entry-overview/earned-revenue-processing) for
 a period.
