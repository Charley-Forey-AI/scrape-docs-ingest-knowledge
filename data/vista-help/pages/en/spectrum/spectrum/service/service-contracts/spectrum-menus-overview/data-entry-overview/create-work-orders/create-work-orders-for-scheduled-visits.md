# Create Work Orders for Scheduled Visits

When you create work orders from the Service Contracts
 module, the following apply.

- A maintenance work order format is automatically used when work orders are created
 from the Service Contracts module. These work orders are flagged as maintenance work
 orders and the detail created in the work order notes is extensive.

- Equipment is labeled with the model number, serial number, location, and so forth;
 the tasks to be completed are also created with the corresponding task parts or
 equipment parts required (if specified in the Service Contracts Installation). The
 maintenance work order that is printed gives the worker all the information that is
 required to work on the equipment. The format is different than the standard work
 order print, which offers a notes area to the right of each task to be performed. A
 pre-printed form (same as the plain-paper format) may be used.

-  This screen will set the 'Alternate address' flag and Bill-to code based on the
 Service Contracts setting.

- When the scheduled contract visit information is updated to the Work
 Order module, associated Task or Subtask notes print on the Crystal-Link Work Order
 Print. When you add subtasks, the notes and descriptions from Task File
 Maintenance will appear in the Notepad window
 that displays from the Work Order Entry > Note Topics window.

- If a dispatch or priority code is specified in Service Contract Installation, all
 new work orders created from the Service Contract module will be assigned this
 status and priority by default.

- When the Work Order installation checkbox options for 'Automatically build task
 detail or labor/material/other charges' is selected, the technician's billing code
 will also be added to the Work Order Labor Transaction Detail Table as part of this
 update.

- When tasks are added to the new work order, set the Sell price/unit from the Service
 Contract Equipment Table.

Important: This report should be retained as a
 permanent audit record of the company.

Note: Service contract task-related notes that are updated in the Create Work Order for
 Scheduled Visits screen display in the Work Ordered section of the Work Order form.
 Task-related notes not entered in these screens display in the Work Notes section of
 the Work Order form.
