# Renewal & Expires Report

This screen is used to obtain a report listing Service
 Contracts that expire during a specified time period.
This information may be useful in contacting customers for renewal. There are several
 parameters that may be specified to print the exact report needed.
Use the fields to create the report that will be most useful to you.
 After specifying your report criteria, if you wish to save your entries, click the
 Save Current button in the Sidebar.
