# New/Edit Contract Components - Warranty

Use this tab to view labor and material warranty
 information for an individual component associated with this service contract.
Note: After the initial warranty (manufacturer 1) has
 expired, the next warranty (manufacturer 2) commences, followed by the third warranty.
 Fields/Buttons
Descriptions

Labor manufacturer 1-3

Code
Enter the customer code affiliated
 with the labor warranty manufacturer, or double-click on this field to
 select from a list of available customer codes. If the customer code you
 select is not in use, then entry will not be allowed.
Warranty period
Enter the warranty period (in years)
 affiliated with the labor warranty manufacturer.
Expiration date
Enter the warranty expiration date,
 or double-click on this field to select a date.
Standard hours

Standard rate
Enter the standard labor rate for
 this contract equipment component.
Material manufacturer 1-3

Code
Enter the customer code affiliated
 with the material warranty manufacturer, or double-click on this field to
 select from a list of available customer codes. If the customer code you
 select is not in use, then entry will not be allowed.
Warranty period
Enter the warranty period (in years)
 affiliated with the material warranty manufacturer.
Expiration date
Enter the warranty expiration date,
 or double-click on this field to select a date.
Material level
Enter the material level on a scale
 of 0 to 5, with 0 indicating that there is no cost associated with the
 material.
