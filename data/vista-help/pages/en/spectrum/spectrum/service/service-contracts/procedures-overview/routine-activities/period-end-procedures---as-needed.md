# Period End Procedures - As Needed

You should exercise extreme caution when removing records from the system. Ensure that a successful backup is performed prior to selecting any of these options.

- Purge closed service contracts using the Closed Service Contracts Purge.

- Purge Proposed contracts that will never be changed to Accepted
 using Service Contract Proposal
 Purge.

- Delete contract systems as necessary using Contract System / Schedule Purge.
