# Lead Source Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Button

Lead source code
Enter a unique code to identify this
 lead source. A search window is available for viewing existing lead source
 codes.
Description
Enter a description of this lead source
 code. Examples might include yellow pages, direct mailing, contractor referral,
 or satisfied customer.
Type
Enter the type for this lead source
 code.
Effective from / to
Enter the beginning and end dates for
 this lead source, for example, the first and last days that an advertisement
 ran.
Dollar budget
Enter the budget allocated for this
 particular lead source.
Notes
Enter any notes pertaining to this lead
 source.
Status
Click to indicate whether this lead
 source is currently Active or Inactive.
