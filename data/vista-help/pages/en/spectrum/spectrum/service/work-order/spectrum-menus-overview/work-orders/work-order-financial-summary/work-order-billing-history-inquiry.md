# Work Order Billing History Inquiry

The Work Order Billing History Inquiry window will display billing summary information and a history of transactions associated with the particular Bill # for Labor hours, Material transactions, Other charge transactions and Flat-rate tasks.
Note: This window is not applicable for reverse
 billings.
Field
Description

Work order
The work order number of the selected
 transaction will display in this field.
Bill #
The Bill # for the work order is
 displayed in this field.
Customer
The customer info for this work order
 displays in this field.
Summary
This section of the window contains
 summary billing totals for Labor hours, Material, Other charges and Flat rate
 tasks. The tabs beneath this section will contain more detailed billing
 information for these billing transactions.
Labor Hours
Detailed billing information for labor
 hours are available on this tab. All labor transactions for the work order that
 have been transferred to A/R will be listed on this tab. Select an item from
 this list, and click the View button to open the View Labor Hours History
 window.
Material
Detailed billing information for
 material charges are available on this tab. All material transactions for the
 work order that have been transferred to A/R will be listed on this tab. Select
 an item from this list, and click the View button to open the View Material History window.
Other Charges
Detailed billing information for other
 charges are available on this tab. All other charge transactions for the work
 order that have been transferred to A/R will be listed on this tab. Select an
 item from this list, and click the View button to open the View Other Charges History
 window.
Flat Rate Tasks
Detailed billing information for flat
 rate tasks are available on this tab. Select an item from this list, and click
 the View button to open the View
 Task History window.
