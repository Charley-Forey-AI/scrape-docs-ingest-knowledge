# Work Order Labor for Job work orders

If a valid job work order was selected, the grid will
 display labor transactions for the job work order.
 This entry screen is used to enter labor hours for a specific technician for Job work
 orders. It also allows you to record equipment information when entering employee time,
 even if the Service Contract module is not present.
If desired, labor hours entered here can be updated to Pre-Time Card Entry in the Payroll module so that duplicate entry is not necessary. The ability to transfer to Payroll is controlled through the Update Labor Hours prompt on the Work Order Installation screen.
Note: You can quickly navigate between Work Order screens
 via the Info Bar links, or use the hot-key equivalents on the command bar.
