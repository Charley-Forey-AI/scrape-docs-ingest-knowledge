# Service Request

Use the Service Request window as the starting point for
 entering service requests from customers.
Search for an existing site or create a service request for a new site. The tabs available
 on this screen allow you to view work order history; information about equipment, service
 contracts, and purchase orders related to this site; and the customer's credit limit,
 terms, current balance, and payment history.
