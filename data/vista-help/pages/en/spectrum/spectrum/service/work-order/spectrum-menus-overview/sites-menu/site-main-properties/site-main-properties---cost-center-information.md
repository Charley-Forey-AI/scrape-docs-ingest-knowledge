# Site Main Properties - Cost Center Information

If the cost center feature is enabled in the Enterprise installation screen for this company, this window includes a Cost center field in the Work order defaults section. This field associates the site with a particular cost center. When adding a new site address, the Cost center field must be completed, and the entry must be a valid active cost center for the operator adding the site address. At the Customer field, the software validates that the customer is compatible with the work site cost center specified.
