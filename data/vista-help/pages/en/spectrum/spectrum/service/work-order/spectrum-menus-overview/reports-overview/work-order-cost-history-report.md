# Work Order Cost History Report

The Work Order Cost History Report provides information on
 cost transactions associated with work orders.
The summary format of this report will provide a listing of cost totals by work order, site
 or G/L date. The detail format will show individual cost transactions.
When void payroll checks and A/P credit memos appear on this report, values will appear as negative numbers.
The G/L code assigned to labor transactions from the payroll update is the 'pay extension' G/L account, but the transaction total includes payroll burden which may have been posted to another G/L account(s).
When the 'Include unposted labor transactions' option is selected, direct work order add-ons for pre-time card entries will be included on this report.
If the 'Payroll overhead' option is selected for the work order, unposted labor cost transactions will be calculated and then increased by the specified overhead percentage or rate/hour amount.
