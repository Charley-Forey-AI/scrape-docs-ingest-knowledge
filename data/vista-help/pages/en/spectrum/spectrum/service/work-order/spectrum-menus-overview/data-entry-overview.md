# Data Entry Overview

Use the Work Order > Data Entry menu to create, schedule, and print work orders, and create proposals.
 Information from proposals that are accepted by a customer can be automatically transferred to
 a work order, eliminating the need for duplicate entry. On the Site Map screen, click Work Order, and then click the Data Entry menu.
