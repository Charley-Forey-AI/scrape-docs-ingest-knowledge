# Technician Maintenance - Properties tab

Use this tab to enter information about the technician,
 including pay rates, technician status, and address information.
Field
Description

Name
Upon entry of an employee code, the
 employee's name will display from Employee Maintenance. Press Enter to accept
 the employee name; if the wrong employee name displays, press the Cancel icon
 back to the top of the screen and enter the correct employee code. The employee's department displays beneath this
 field.

Category
Enter a valid labor category code for
 this employee. This code must have been previously set up in Labor Category
 Maintenance. A search window is provided for viewing valid codes.
Burden method
Entering a burden method in this field
 establishes the default for burden calculations for this specific technician,
 rather than for the entire category. A technician may have a burden method that
 differs from other people within the same category. Note: Burden method is only used on site work orders.


Percentage / Hourly rate
If a burden method was selected above,
 enter the regular, overtime, or double time burden percentage or hourly rate
 specific to this technician.
Zones
This field is used with the Dispatch
 Schedule and Dispatch Board screens' zone selections to determine which
 technicians are displayed on the dispatch screen. Enter ALL to indicate that
 this technician works in all zones.
Status
Select the technician's status in this
 field. If the status is Active, then the Dispatch Schedule or Dispatch Board screens
 will display information about this technician's current work order. If the
 status is Inactive,
 then the Dispatch Schedule or Dispatch Board screens will display the "Dispatch
 desc" information instead of the current work order information.
Dispatch description
Enter the description to appear on the
 Basic Dispatch Schedule, Advanced Dispatch Schedule, or Dispatch Board screens
 or during work order entry when this technician's status is 'Inactive'. Two
 lines are available for entering the description. Note that any description
 text entered in the Dispatch Schedule or Dispatch Board screens will display
 here automatically.
Communication
The technician's phone number, and
 email address defaults from the employee's file in Payroll.
