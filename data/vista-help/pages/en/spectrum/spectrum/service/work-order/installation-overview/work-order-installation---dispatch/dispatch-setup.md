# Dispatch Setup

Click the Description Setup button on the Work Order Installation – Dispatch tab to display this window. Use this window to designate which fields are displayed in the long and short work order descriptions on the Dispatch Board and Dispatch Schedule.
Field
Description

Field
The field name displays. No entry is
 allowed.
Max length
The field length displays. No entry is
 allowed.
Short
Enter the order (up to 10 characters)
 this field will appear in the short description.
Long
Enter the order (up to 30 characters)
 this field will appear in the long description.
Prompt
Enter a prompt to identify this field.
 Entry in this field is not required.
