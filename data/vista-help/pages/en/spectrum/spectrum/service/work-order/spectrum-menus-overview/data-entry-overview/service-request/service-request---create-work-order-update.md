# Service Request - Create work order update

Click the Create W.O. button to initiate the update to create a new work order. The
 update will perform the following steps:

1. Add new customer (if work order is for new customer and new site). The New Customer Code window will open immediately after the record is added, unless this feature is disabled in the Service Request Operator Preferences window. When you close this window, the update will proceed to the next step.

1. Add new site (if work order is for new site). The New Site window will open immediately after the record is added, unless this feature is disabled in the 'Service Request Operator Preferences' window. When you close this window, the update will proceed to the next step.

1. Get work order number. If the Work Order Installation option for 'Auto sequenced work order number' is selected, the next work order number will be assigned automatically and the update will continue. If this option is not selected, or the 'Always prompt to assign new work order #' option in the Service Request Operator Preferences window is selected, you will be prompted to enter a work order number. The software will verify that the specified work order is not already present before proceeding.

1. Add work order.

1. Launch email confirmation. This only applies if the service request originated from an email dropped into the 'Find Site for Service Request' window, and this feature is enabled in the Service Request Operator Preferences window.

1. Open a New Work Order window. If you do not have access to Work Order Entry, the View Work Order window will open instead.
