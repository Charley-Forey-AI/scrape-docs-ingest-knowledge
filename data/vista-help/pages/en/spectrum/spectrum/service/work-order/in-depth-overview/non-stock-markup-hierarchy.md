# Non-Stock Markup Hierarchy

When non-stock markups are being used, the software
 determines the non-stock value using the hierarchy shown below.
Regarding the calculation of markups from cost, be aware that when computing the price, the
 software marks up the non-stock cost and any sales and use tax. Protection is included to
 make sure any sales tax included in the extension of purchase order lines is only included
 once.
