# Work Order Tasks

Use the Work Order Tasks screen to
 enter tasks for the work order. To access this screen, select Work Order Info Bar > Activity > Tasks.
Work Order tasks can be both flat rate and standard. In addition, each
 task can have up to five levels of subtasks, so you can make changes that will filter up to
 the main task. When a task's price is updated, a window displays that allows you to
 immediately update the main tasks. Work Order tasks and Service Contract tasks are not
 maintained as two separate lists. Service Contract tasks fully integrate with Work Order
 and are automatically added to the generated work orders.
Even though tasks are automatically included in the next billing after
 they are added, it is possible to add transactions referring to previously-billed tasks
 when adding or changing entries in Labor Hours Entry, Material Entry and Other Charges
 Entry.
Work Order Tasks - How to Use This Screen
The Task notes and any notes on a Subtask attached to a task will print
 on the work order, whether the work order is created in Work Order or Service Contracts.
 When you add subtasks, the notes and descriptions from Work Order > Task File Maintenance will appear in the Notepad window that displays from
 the Work Order Entry > Note Topics window.
When viewing notes previously entered for the proposed work order in
 Notepad, the notes will display first according to task followed
 by the task's subtask notes, if any exist. Any text added to an existing task will appear
 at the end of the existing task note.
All notes and task descriptions will display alphabetically in this
 Notepad window, regardless of the order in which they are added.
 For example, if you enter task B100, followed by task 100, and lastly A100, the order of the tasks will be:

- 100

- A100

- B100

All work order tasks can be both flat rate and standard; for example,
 you can have a flat rate task and a standard subtask. Each task can have up to five levels
 of subtasks, which means you can make changes that will filter up to the main task. When a
 task's price is updated, a window displays that allows you to immediately update the main
 tasks. The Pricing type column
 will display 'Flat rate' when the 'Flat rate' checkbox in Task File
 Maintenance is selected, and 'Time + Material' otherwise.
When the Work Order installation checkbox
 options for 'Automatically build task detail or labor/material/other charges' is selected,
 the software will create detail transactions from setup in the Task File
 Maintenance screen, and the price setting of the work order (flat rate or
 T&M) and the task type (flat rate or non-flat rate) determines how the transactions are
 priced. The technician's billing code will also be added to the Work Order Labor
 Transaction Detail Table as part of this update.Updates will additionally store Equipment
 and Contract codes assigned to the task in the Work Order Task Table in the new transaction
 detail records.
Note: You can quickly navigate between Work
 Order screens via the Info Bar links, or use the
 hot-key equivalents on the command bar.
