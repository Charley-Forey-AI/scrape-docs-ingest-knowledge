# Manufacturer File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Manufacturer
Enter a valid customer code
 corresponding to this site, indicating the customer for whom the work is to be
 done. The code must have been previously defined in Customer File
 Maintenance.
Description
Enter a description for this
 manufacturer.
Standard billing rate
Enter a standard billing rate for this
 manufacturer.
Print alternate address on invoice
Select this box to use an alternate
 address for billing invoices and enter the address below.
Bill-to code
Enter the bill-to code to use for the
 selected customer site identification code. Bill-to codes are set up in Accounts Receivable > Alternate Bill-To Address Maintenance.
