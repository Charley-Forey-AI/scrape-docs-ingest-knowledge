# Technician Maintenance - Time Off tab

Use this tab to enter time off information logged for the
 selected technician.
Field
Description

Start date
Enter the date on which the leave will
 begin.
End date
Enter the date on which the leave will
 end.
All day event?
Select this checkbox if the time off
 will take the entire day(s).
Start time
If the leave will only include part of
 the day, enter the time of day the leave will begin. If the All day event check
 box is selected, this field is disabled.
End time
If the leave will only include part of
 the day, enter the time of day the leave will end. If the All day event checkbox is
 selected, this field is disabled.
Description
Enter a brief description for the time
 off.
