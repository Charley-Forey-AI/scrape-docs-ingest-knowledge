# Site Payroll Setup

Use the Site Payroll Setup screen to define additional properties for worker's compensation and wage codes for the selected site.
Field
Description

Defaults

Worker's compensation
Enter the worker's compensation code
 for this work site. This code must have been previously defined in Worker's
 Compensation Base Code Maintenance in Payroll.
Work state
Use the drop-down lists to
 select the tax codes for the work site. Up to 10 characters are
 allowed.
Work country

Work local

Rate level
Enter the wage rate level for this work
 site.
Pay group
Enter the pay group for this work
 site.Note: This field will only be
 available if the pay groups option is selected in the Payroll Installation
 screen.

Wage code grid

Wage code
Enter the wage code to be used for this
 site. This code must have been previously defined in Wage Code File Maintenance
 in the Payroll module.Note: The wage code
 grid will be hidden entirely if wage codes are not enabled in the Payroll
 Installation screen.

Union
Enter the union code to be used for
 this site. This code must have been previously defined in Union File
 Maintenance in the Payroll module.
Wage description
The full wage description displays from
 the wage code file.
Union name
The union name for this site will
 display in this field.
Craft
The craft name for this site will
 display in this field.
