# Sites - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screenfor this company, the Site Main Properties screen includes a Cost center field at the top of the screen. This field associates the site with a particular cost center. When adding a new site address, the cost center field must be completed, and the entry must be a valid active cost center for the operator adding the site address.
When editing, viewing, or deleting a site address, Spectrum compares the site's assigned cost center with cost centers in the operator's assigned scheme; if the site's cost center is not present, then that site address code is not accessible. Those sites that are not allowed will not display in the list box on the main screen.
