# Update Work Orders to Invoice Entry

This update will transfer any work order labor, company
 equipment costs, materials and other cost data to Accounts Receivable.
A/R invoices will be created for all Job and Site work orders selected here, and multiple
 contracts can be entered on one work order. Operators can generate an unlimited number of
 invoices, but when there is no detail billing information present for the work order this
 update will simply change the Work Order status to 'Complete'.
When this screen is first opened, the Build List of Work Orders to Bill window will open to specify which work orders to update.
When adding a new material transaction record for a work order, the current Work Order processing date will be assigned as the 'Transaction Create Date'.
Note: Only work orders that have already been printed can be
 updated; unprinted work orders will not transfer to Accounts Receivable.
