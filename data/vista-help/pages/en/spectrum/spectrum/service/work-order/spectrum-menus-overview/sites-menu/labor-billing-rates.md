# Labor Billing Rates

Use this screen to review default labor billing rates for
 the selected site or service contract.
All billing codes will display, along with their particular rates based on the current
 setup in Labor Billing Rates Maintenance, Site Labor Billing Rates Maintenance, Customer
 Labor Billing Rates, and Contract Labor Billing Rates Maintenance.
Field/Button
Description

Site code
Enter a site code to review site labor
 billing rates.
Contract
If this screen is accessed from Service
 Contracts, enter a contract # to review contract labor billing rates.
Customer
The customer code and name for the
 selected site will display in this field. The total number of sites referencing
 this customer code will display to the right of the name.
Standard pricing
The standard pricing rate for the
 customer code displays in this field, if available.
Rate hierarchy
Use this section to add new Site and
 Customer billing rates. If this screen is accessed from Service Contracts, you
 will also have an option to add Contract billing rates. You must have security
 authorization to Site Labor Billing Rates Maintenance, Customer Labor Billing
 Rates Maintenance, and/or Contract Labor Billing Rates Maintenance to add these
 rates. Standard rates cannot be added on this screen.Specify a billing code and rates for Regular, Overtime, and Double time
 pay.

Edit Delete
Select an existing billing code from
 the grid to edit billing rates or delete the selected row. You must have
 security authorization to Site Labor Billing Rates Maintenance, Customer Labor
 Billing Rates Maintenance and/or Contract Labor Billing Rates Maintenance to
 edit these rates. Standard rates cannot be modified on this screen.
