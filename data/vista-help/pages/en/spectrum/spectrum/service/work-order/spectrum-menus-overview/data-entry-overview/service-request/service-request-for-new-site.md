# Service Request for New Site

The Service Request for New site window will allow you to create a new site and new customer, or create a new site for an existing customer.
The Credit Check tab and Alerts section of this window are only available once an 'Existing' site customer is selected.
Note: When a new customer will have a different billing address than the
 site address, open the Operator Preferences window and select the 'New site only: Always
 auto-open new customer window' checkbox, but do not save as the default. This allows the
 customer window to open just once.
