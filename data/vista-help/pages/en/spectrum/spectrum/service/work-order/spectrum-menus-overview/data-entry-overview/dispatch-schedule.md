# Dispatch Schedule

Use the Dispatch Schedule to assign technicians to work
 orders. Drag and drop work orders from the list shown in the left pane to the schedule shown
 in the right pane. You can be update the dispatch schedule at any time.
Watch the following video to learn more about the features available on
 the Dispatch Schedule:

Note: If unfinished work orders with a scheduled date
 earlier than the current work order processing date are included on the schedule, and the
 'Automatically roll the scheduled date of past-due work orders to new date' option is
 selected in Work Order Installation, a dialog box will automatically open to allow the
 Operator to roll-over the scheduled date to a new date.
