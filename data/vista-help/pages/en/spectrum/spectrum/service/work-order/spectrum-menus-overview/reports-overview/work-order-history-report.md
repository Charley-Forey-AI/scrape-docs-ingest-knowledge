# Work Order History Report

The Work Order History Report will print a history of labor, materials, other charges, or all three as desired. This information prints for selected technician work orders, sites, jobs, phases, dates, and material items.
This report presents transaction information from work orders showing billed amounts, not cost amounts. The transactions are listed across billings, when multiple invoices have been generated for the given work order.
Note: Regarding Historical Site Information, the current site
 description is saved when a cost is first posted, and this site description displays on the
 Work Order History Report. Any previous site descriptions for the same site may be viewed
 in the WO_COST_HISTORY_MC table using a custom Crystal Report.
