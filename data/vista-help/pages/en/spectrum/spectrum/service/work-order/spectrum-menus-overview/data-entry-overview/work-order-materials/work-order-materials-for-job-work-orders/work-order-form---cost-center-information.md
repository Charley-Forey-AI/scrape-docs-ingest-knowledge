# Work Order Form - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, a Cost group field displays beneath the Technician field. Spectrum only allows entry of a cost group that the operator has security authorization to access. The software compares the cost centers authorized for the operator with the cost center associated to the work order (or job); if the operator is not authorized to access the cost center, then that work order is not included.
For Site work orders, the cost group selection evaluates the cost center assigned to the work order. For Job work orders, the Cost group selection evaluates the cost center assigned to the job, and the work order will only be included when the work order (or job) cost center is included in the requested Cost group range.
Cost center overrides are applied in this screen to determine whether the operator has access to the work order.
