# Work Order Data Conversion

Complete the following steps to complete the maintenance
 portion of the work order data conversion.

1. Complete the Work Order
 Installation screen.

1. Click the Admin button at the
 bottom of the screen to access the System Administration menu.


1. On the System Administration
 menu, double-click Processing Date Change and set up the
 Work Order portion of the Processing Date
 Change screen. Click OK to return to the
 System Administration menu.

1. On the Site Map screen, click
 to select the Work Order module.

1. Click the Data Entry menu
 button. This is where most daily work such as work order entry and dispatch
 scheduling will occur.

1. Enter outstanding work orders as of the conversion
 date using Work Order Entry.

1. Complete the "Work Order" portion of the
 Processing Date Change screen, setting the minimum and
 maximum dates as desired, and "current" to the actual calendar date.

1. Within the Work Order module,
 click to select the Maintenance menu. This is where all
 routine file maintenance will occur.

1. Enter the labor billing rates using Labor
 Billing Rates File Maintenance. Print the labor billing rates and
 check for accuracy.

1. Enter the standard phases using W/O
 Standard Phase File Maintenance. Print the standard phases and review
 for accuracy.

1. Enter the labor categories using Labor
 Category File Maintenance. Print the labor categories and review for
 accuracy.

1. Enter technicians and assign them to categories using
 Technician File Maintenance. Print the technicians and
 review for accuracy.

1. Enter the general ledger departments using
 G/L Department File Maintenance. Print the G/L departments
 and review for accuracy.

1. Enter the work order types using Work Order
 Type File Maintenance. Print the work order types and review for
 accuracy.

1. Enter the site addresses, if desired, using
 Work Site Address File Maintenance. Print the work order
 site addresses and review for accuracy.

1. Enter the standard work order messages, if applicable,
 using Work Order Message File Maintenance. Print the work
 order messages and review for accuracy.

1. Enter the material item codes using
 Material Item Code File Maintenance. Do not use this menu
 selection if the Inventory Control module or Price
 Update is used. Print the work order material items and review for
 accuracy.

1. Enter zip code zones and zone descriptions using
 Zip Code Zone File Maintenance.

1. Enter work order dispatch status codes using
 Dispatch Status File Maintenance. Set the status color
 codes for Open, Hold, and
 Finished work. Print the dispatch status codes and review
 for accuracy.

1. Enter other costs (as opposed to labor and materials)
 using Other Cost File Maintenance. Print the other costs and
 review for accuracy.

1. Enter work order tasks that will be billed on a
 flat-rate pricing basis, using Task File Maintenance. Print
 flat-rate tasks and review for accuracy.

1. Enter lead sources using Lead Source File
 Maintenance. Print lead sources and review for accuracy.

1. Select priority color coding that will be used on the
 Dispatch Board and in Work Order
 Entry using Priority Maintenance. Print the
 priorities and review for accuracy.

1. Select priority color coding that will be used on the
 Dispatch Board and in Work Order
 Entry using Technician Status Maintenance.
 Print the technician statuses and review for accuracy.
