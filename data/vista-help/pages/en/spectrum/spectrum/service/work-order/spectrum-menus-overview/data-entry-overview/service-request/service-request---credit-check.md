# Service Request - Credit Check

Use the Credit Check tab to view financial information about the customer assigned to the selected site, including current balance and aging information, plus billing and payment history over the past month, year and prior year. This tab also provides a summary display of credit info, including credit limits and standard terms.
In the case where no customer code is specified in Site File Maintenance, the Customer field will be replaced with 'No site customer code' and the Edit Cust, View Cust and More Details buttons will be disabled. If there is a customer code referenced for the existing site but it is invalid, the Customer field will be replaced with 'Invalid site customer code' and the Edit Cust, View Cust and More Details buttons will be disabled.
