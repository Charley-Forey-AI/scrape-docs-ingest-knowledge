# Build Standard Makes and Models

This update will create records in Make File Maintenance
 and Model File Maintenance from equipment set up for sites.
This update will add an entry to the maintenance file for each existing make, and will add
 a make-and-model record to the maintenance file for models. The update will also add makes
 and models of equipment components set up in Site File Maintenance. Make-and-model records
 already existing in Make File Maintenance and Model File Maintenance will not be
 changed.
Note: If the Service Contract module is installed, this update will also
 add makes and models of any proposed equipment specified on Service Contract
 proposals.
