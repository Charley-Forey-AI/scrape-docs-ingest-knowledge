# Service Request - Service Contracts

Use the Service Contracts tab to view service contract
 details and any associated images set up for the selected site.
You can also assign an existing contract to the new work order. If the Service Contract
 module is not installed, this tab is disabled.
Note: If you are tracking equipment, use the [Service Request - Equipment](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request/service-request---equipment) tab to assign the work
 order to a service contract.
Button
Description

Assign to WO
Click this button to assign the
 selected service contract to the new work order. Once a service contract is
 assigned to the work order, a green checkmark will appear next to the contract
 #.Note: If a contract or task is already
 assigned to this service request, this button will be
 disabled.

Unassign
Click this button to remove the service
 contract assigned to the selected work order.Note: If a contract or task is already assigned to this
 service request, this button will be disabled.

Service contract
Click this link to open the [Service Contract Main Properties](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/service-contracts/service-contract-main-properties) screen to set up and manage general contract, maintenance work order, notes
 and exception information related to the service contract.Note: If the Operator does not have security to the Service
 Contract module, this link will be hidden.
