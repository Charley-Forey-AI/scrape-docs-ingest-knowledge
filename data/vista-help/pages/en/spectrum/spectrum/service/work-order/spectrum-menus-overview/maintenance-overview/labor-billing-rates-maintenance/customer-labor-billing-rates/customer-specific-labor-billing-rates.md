# Customer Specific Labor Billing Rates

This screen is used to define customer-specific billing
 rates for a selected customer.
For example, B1 might have been previously defined as the Standard Billing Rate of $100 per
 hour ($100 regular time, $150 overtime). However, relatives of the owner get a discount, so
 they are billed only $70 per hour. Or perhaps a certain customer is so difficult that
 technicians will only go there if paid double. In these situations, this screen is used to
 re-define B1 rates for the customer in question. Later, when work orders are entered, the
 person entering the work orders does not need to be aware of any special circumstances for
 the correct dollar amount to be automatically billed.
Field
Description

Customer code
Enter the customer code or use the
 window to search for a particular customer. If you select a customer with a
 status set to Inactive or Not
 used, a warning will appear.
Bill code
Enter the billing code, a unique code
 to identify a billing rate. The billing code must exist in Labor Billing Rates
 Maintenance before it can be entered on this screen.

Regular
Enter the regular billing rate, or

Enter the overtime billing rate, or
 press Enter to accept
 the default from Labor Billing Rates Maintenance or enter another billing

Double time
Enter the double time billing rate, or
 press Enter to accept
 the default from Labor Billing Rates Maintenance or enter another billing
