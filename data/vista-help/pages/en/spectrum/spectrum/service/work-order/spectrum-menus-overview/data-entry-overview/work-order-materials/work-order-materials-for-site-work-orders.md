# Work Order Materials for Site Work Orders

If a valid site work order was selected, the listbox will
 display material transactions for the site work order.
The entry window will contain sections for transaction details, customer
 billing and warranty information. Work orders with a status of 'Proposed' or 'Completed'
 will be view-only.
This screen will conditionally display Transactions available to bill in the
 upper right hand corner of the screen if there are 'available to bill' transactions in
 purchase order, unapproved invoice, unposted invoice and/or Work Order cost history for the
 selected work order. If available, click the Material button to open the [Bill Available Material Transactions](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-materials/work-order-materials-for-site-work-orders/bill-available-material-transactions) window.
When adding a new material transaction record for a work order, the
 current Work Order processing date will be assigned as the 'Transaction Create Date'.
