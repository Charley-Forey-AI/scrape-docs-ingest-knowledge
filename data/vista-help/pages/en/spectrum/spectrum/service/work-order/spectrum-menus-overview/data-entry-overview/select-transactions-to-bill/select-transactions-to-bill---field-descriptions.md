# Select Transactions to Bill - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field/Button
Description

Total invoice extension
The sum of the 'Selected to Invoice'
 amounts from all tabs displays in this field.
Available
The sum of the 'Billing extension'
 records for all displayed transactions on the corresponding tab displays in
 this field.
Selected to invoice
The sum of the 'Billing extension'
 records that are currently selected on the corresponding tab (equal to 'Total'
 minus 'Unselected').
Buttons

Select All
Press this button to select all of the
 detail lines on the tab for transfer.
Select
Click this button to mark the selected
 detail line for transfer.
Deselect All
Click this button to deselect a detail
 line.
Deselect
Click this button to deselect all
 detail lines.
