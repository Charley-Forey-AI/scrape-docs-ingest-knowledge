# Site Equipment

The Site Equipment screen is used to manage equipment
 assigned to the site.
If the Service Contract module is not installed, you can use this screen to add systems to
 the site that are covered by the service agreement. Entry is not allowed if the Service
 Contract module is installed and the work site is set up with one or more a service
 contracts.
Total billed, Total cost, and Gross profit amounts from the listbox will display in the header.
Note: If this screen is not accessed in context to a
 particular site, all sites for the equipment code will display in the listbox. If this
 screen is accessed for a non-site specific format, use the search box to narrow your
 inquiry results.
