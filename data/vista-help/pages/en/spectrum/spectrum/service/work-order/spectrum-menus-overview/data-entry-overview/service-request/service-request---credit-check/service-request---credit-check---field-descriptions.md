# Service Request - Credit Check - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Customer
The customer code from Site
 File Maintenance will display in this field.
Outstanding

Current
This field displays the total
 outstanding items that are not past due, excluding retention. This value is the
 same as the 'Total balance' listed above minus the aging values listed below.
1-30 days past due
The total outstanding
 balance of items that are past due 1-30, 31-60, 61-90 and 91+ days are
 displayed in these fields. These same values appear on the Customer Detail Inquiry > Analysis tab.
31-60 days past due

61-90 days past due

91+ days past due

Total current due
Display the total outstanding balance
 for the customer, regardless of invoice date.
Retention balance
The total outstanding retention balance
 displays in this field. Retention credit memos reduce this amount.
Total balance due
The total outstanding non-retention
 balance displays in this field.
Credit info

Credit limit
This field will show the credit limit
 values assigned in Customer File Maintenance. If no
 credit limit is specified, this field will be blank.
Average # of days to pay
This field will display the same
 'Average days to pay' value as shown in Customer Detail
 Inquiry.
Terms
This field will display the terms code
 and description assigned in Customer File Maintenance.
Date established
This field displays the established
 date from Customer File Maintenance.
Last bill date
This field displays the 'last billed'
 date from Customer File Maintenance.
Last paid date
This field displays the 'last paid'
 date from Customer File Maintenance.
