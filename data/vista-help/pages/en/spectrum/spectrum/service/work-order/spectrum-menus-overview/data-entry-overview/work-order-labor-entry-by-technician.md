# Work Order Labor Entry by Technician

Use the Work Order Labor Entry by Technician screen to
 enter labor time across work orders for a specified technician. This screen displays all open
 labor entries (both site and job) for the technician.

- Begin by selecting a Technician in the header portion of the screen. The open labor
 records for the selected technician display.

- Select a work order from the grid and the details of the work
 order in context display in the [Work Orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-orders) pane.
