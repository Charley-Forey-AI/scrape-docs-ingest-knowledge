# Work Order Standard Phase Maintenance

This screen is used to manage standard phases for the
 purpose of job-cost material, labor and other cost entries.
Whenever a new job is automatically created through Work Order Entry, the costs posted to
 these phases will create both the Phase File Maintenance entry and (optionally) a Job Cost
 Transaction Entry.
