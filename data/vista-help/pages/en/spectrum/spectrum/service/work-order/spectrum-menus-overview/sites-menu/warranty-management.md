# Warranty Management

The Warranty Management screen will display a list of all
 component records across all equipment and sites.
Use this screen to manage warranty information for all components. Warranty details are
 conveniently stored at the component level of the site, and warranties can be set up
 without the need to create a contract first. In addition, this screen allows the operator
 to issue proposals for potential extended service contracts.
Note: If this screen is accessed in context to a site, the
 site code will display in the header instead of the Search field.
