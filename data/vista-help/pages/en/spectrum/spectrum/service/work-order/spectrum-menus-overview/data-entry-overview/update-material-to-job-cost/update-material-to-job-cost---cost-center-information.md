# Update Material to Job Cost - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, this update creates records in the Job Cost Transaction Entry screen. The cost center is determined by the phase cost center; if that is blank, then Spectrum uses the job cost center. Both the debit and credit cost centers on the transaction file will be assigned to this cost center.
