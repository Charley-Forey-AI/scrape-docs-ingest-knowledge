# Site Contacts

Use the Site Contacts screen to manage contacts for the
 selected site, and all contacts for the associated customer.
Field/Button
Description

Site contacts
All active contacts associated with the
 current site will appear in this listbox.
Add Contact
Click this button to search for a
 contact to add to the site. If this is the first contact assigned to the site,
 it will automatically be set as primary.
Edit
Click this button to edit the selected
 site contact.
Set as Primary
Click this button to designate the site
 contact as the main point of communication, and a green check mark will display
 adjacent to the contact's name.
Remove
Click this button if you wish to
 disassociate the contact from this site.
E-Mail
Click this button to send an email to
 the selected site contact.
Down arrow
Click this button to add the
 highlighted site contact to the customer.
Up arrow
Click this button to add the
 highlighted customer contact to the site.
All contacts for this customer
All active contacts associated with the
 current customer will appear in the listbox.
