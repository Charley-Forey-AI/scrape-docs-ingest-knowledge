# Zip Code Zone Maintenance

The Zip Code Zone Maintenance screen is used to delineate
 zones for the service area covered.
Data on this screen displays in the sequence by zip code. Once updated, entries made in
 this screen will automatically be reflected in Site Main Properties. Sites with zip codes
 not specified on this screen will not be changed.
