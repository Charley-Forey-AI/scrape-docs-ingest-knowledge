# Assign Tasks to Equipment - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Search partial
Enter a work task code, or leave this
 field blank to display all tasks.
View Task setup
Click this button to open the View Task
 window. Note: If tasks are not already set up
 here, they can be added individually in Work Order Entry.

Select
Click this button to assign the
 selected task(s) to the selected equipment.
Deselect
Click this button to remove the
 previously assigned task(s) from the selected equipment.
Edit W.O. Task
Click this button to edit the selected
 work order task. Use this sub-window to edit the contract code, current
 quantity, and current sell price per unit.
View Task Setup
Click this button to open the
 View Task window.
 Note: If tasks are not already set up
 here, they can be added individually in Work Order Entry.

View Contract
Click this button to open the
 View Service
 Contract window to view contract details for the selected piece
 of equipment. If no contract is associated with the equipment, an error message
 displays indicating no contract is assigned to the record. Note: This button will be disabled if more than one piece of
 equipment is selected.
