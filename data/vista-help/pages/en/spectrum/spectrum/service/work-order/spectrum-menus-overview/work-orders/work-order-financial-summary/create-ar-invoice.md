# Create A/R Invoice

Use the Create A/R Invoice window to transfer any work order labor, materials, and other cost data to Accounts Receivable.

- To access this window, click the Bill Now button at the top of the
 [Work Order Financial Summary](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/work-orders/work-order-financial-summary) screen.

- This screen is specific to the work order selected on the Work
 Order Entry screen; additional work orders cannot be transferred from this screen.

After you transfer the work order to Accounts Receivable, the invoice is created immediately. If a bill-to code has been assigned to the selected work order site in Work Order Site Address Maintenance, this code will be transferred to the invoice in Accounts Receivable. If there is a salesperson code assigned to the work order, this code will be recorded on the A/R invoice during the transfer.
