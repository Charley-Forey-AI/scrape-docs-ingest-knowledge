# Site Labor Billing Rates Maintenance - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, this screen will only permit the operator to add or view sites they have permission to access. The system compares the cost center specified in Site File Maintenance with cost centers in the operator's assigned scheme; if the cost center is not present, that site will not be shown.
