# Edit All Selected Transactions

This update will reset the checkbox settings and
 corresponding invoice extensions of labor transactions currently highlighted in the listbox.
Choose Yes to set the referenced
 checkboxes to selected, choose No to clear the referenced boxes in all selected material records, or
 choose Do not change current
 setting to leave the option as is.
When Bill customer or Bill manufacturer is selected, the update will recalculate the corresponding stored extension using the currently-stored billing rate to compute the new invoice extension. If these are not selected, or if the billing rate is 0.00, the invoice extension will be rest to 0.00.
Note: No changes will be made to records that have already
 been transferred to A/R, regardless of whether they are highlighted or not.
