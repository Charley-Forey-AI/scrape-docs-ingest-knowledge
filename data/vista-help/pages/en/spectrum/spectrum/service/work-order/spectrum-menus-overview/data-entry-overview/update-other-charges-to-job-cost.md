# Update Other Charges to Job Cost

This update will transfer material data entered through the
 Work Order Other Charges screen to the Job Cost module, creating entries in the Job Cost
 Transaction File.
Before running this update, two things should occur:

- The Other Charges Report should be carefully checked for accuracy


- Job Cost Transactions should be updated in the Job Cost module


After this update has transferred other cost data to Job Cost, those
 transactions should be updated.
Important: This screen is only available if the 'Allow job
 work orders?' checkbox is selected in Work Order Installation.
