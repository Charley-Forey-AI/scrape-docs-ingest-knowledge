# Maintenance Overview

Use the Work Order > Maintenance menu to set up the work order codes, rates, and categories necessary for
 tracking and billing service calls. From the Site Map screen, click
 Work Order, and then click the Maintenance
 menu. The following options are available. Click an option to learn how to set it up in the
 system. Items marked with an asterisk (*) are required.
