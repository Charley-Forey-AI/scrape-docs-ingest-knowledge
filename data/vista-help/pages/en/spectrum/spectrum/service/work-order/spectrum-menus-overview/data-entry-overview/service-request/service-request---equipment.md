# Service Request - Equipment

Use the Equipment tab to view equipment details for a site, as well as any work orders that include activity for the selected equipment.
You can also assign tasks to equipment. When one or more tasks is assigned to the selected piece of equipment, a green checkmark will appear next to the equipment description in the listbox. Contract codes assigned to equipment tasks on this tab will take precedence over any general contract assignments made on the Service Contracts tab.
When the Work Order installation checkbox options for 'Automatically build task detail or labor/material/other charges' is selected, the technician's billing code will be added to the Work Order Labor Transaction Detail Table as part of this update.
Note: If you are not tracking equipment, use the [Service Request - Service Contracts](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request/service-request---service-contracts) tab to assign
 the work order to a contract.
