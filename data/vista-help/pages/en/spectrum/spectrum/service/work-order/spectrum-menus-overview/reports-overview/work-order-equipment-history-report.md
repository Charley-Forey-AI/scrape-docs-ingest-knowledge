# Work Order Equipment History Report

The Work Order Equipment History Report is used to show billing and actual cost information by equipment. This report will only include equipment that had activity during the selected date range.
The Summary format will show billings, costs and profitability information totaled by equipment. The Billing detail format will itemize billing transaction details by equipment. The Actual cost detail format will list labor, material and other charges associated with each piece of equipment. This report will also offer the ability to include un-invoiced billing amounts and unposted labor and material costs if desired.
When the 'Include unposted labor costs' option is selected, direct work order add-ons for pre-time card entries will be included on this report.
If the 'Payroll overhead' option is selected for the work order, unposted labor cost transactions will be calculated and then increased by the specified overhead percentage or rate/hour amount.
