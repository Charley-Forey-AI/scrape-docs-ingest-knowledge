# Reopen Completed Work Order - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, Spectrum only allows entry of a work order that the operator has security authorization to access. The software compares the cost center assigned to the work order with cost centers authorized for the operator; if the work order is not authorized, then entry is not allowed. In addition, the Search Completed Work Orders window only shows work orders that the operator has security authorization to access.
Cost center overrides are applied in this screen to determine whether the operator has access to the work order.
