# New/Edit Site - User-Defined

Use the Site User-Defined Fields screen to assign
 user-defined fields to the site.
User-defined categories are set up in [WO Site User-Defined Fields Maintenance](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/wo-site-user-defined-fields-maintenance).
Field/Button
Description

Prompt
The field label displays. No entry is
 allowed. To assign a value to the selected user-defined prompt, click the
 Edit button and
 then complete the Vendor User-Defined Field Edit window.
Current Setting
Valid user entries for this
 user-defined field display.
Edit
Click this button to edit the selected
 user-defined field.
