# Site Work Order History

Use the Site Work Order History screen to view work orders
 assigned to a specific site, or for all sites.
The listbox will be ordered from newest to oldest order date, and can be filtered by Status
 Type: Proposed, Unfinished, Finished and/or Complete.
Only site work orders will display on this screen. To view job work orders, click the Cost history link to open the [Work Order Cost History](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/inquiries-overview/work-order-cost-history) screen.
