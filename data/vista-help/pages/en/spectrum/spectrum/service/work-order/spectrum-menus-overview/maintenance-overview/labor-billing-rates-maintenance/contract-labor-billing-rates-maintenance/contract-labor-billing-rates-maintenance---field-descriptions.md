# Contract Labor Billing Rates Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Site
Enter a site code in this field. A
 warning will display if an inactive site is entered.
Contract
Enter a contract # for the previously
 selected site in this field. A warning will display if an inactive contract is
 entered.
Billing code
Enter a billing code in this
 field.
Regular
Enter a regular labor billing rate in
 this field.
Overtime
Enter an overtime labor billing rate in
 this field.
Double time
Enter a double-time labor billing rate
 in this field.
