# Truck Restocking Alerts

This dashboard app will display items set up in the
 warehouse assigned to the current technician that are at or below the reorder point. This app
 will only list items for this warehouse needing to be replenished.
Hover over an item to show the reorder point, as specified in Item
 Warehouse Details. Double-click on an item to open the [Truck Stock Manager](/en/spectrum/spectrum/service/field-tech/truck-stock-manager) screen.
Note: If the current operator is not a Work Order technician
 with an assigned warehouse (truck), or if all items in the warehouse are fully stocked,
 this app will be empty.
