# Work Order Main

The Field Tech Work Order screen opens in context of whichever work order is selected in any of the Field Tech dashboard apps.
This screen displays information about the work order from the Work Order Main Properties and Site Main Properties screens. Use this screen to quickly clock in or clock out technicians, and/or change the status of the work order.
If you have the Document Imaging module installed, click the slider bar to view any images associated with the work order.
