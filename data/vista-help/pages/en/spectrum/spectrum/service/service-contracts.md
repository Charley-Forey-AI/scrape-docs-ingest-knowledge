# Service Contracts

The Service Contracts module allows you to manage and control
 service contract revenues and costs. Once a proposal is entered, the system creates the
 contract, work orders, and all billings.
Important: In order to use Service Contracts, you must
 have implemented the Work Order module.
To set up Service Contracts, follow these steps:

- Configure default Service Contracts settings.

- Define service contract maintenance codes, and generate service
 contracts.

- Create work orders and A/R invoices, and process earned revenue.


Refer to the[Service
 Contracts - Getting Started](/en/spectrum/spectrum/service/service-contracts/service-contracts---getting-started) for details.
