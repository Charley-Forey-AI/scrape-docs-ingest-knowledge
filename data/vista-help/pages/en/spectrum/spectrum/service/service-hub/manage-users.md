# Manage Users

The Manage Users page allows you to invite, delete, and search for users, as well as modify user permissions across Service Hub.
You can manage Service Hub users in either business context. From the customer business context, Customer Admins can select Manage Users from the left navigation menu to manage users within their company's customer account. From the ERP Administrator business context, ERP Admins can select a company from the list of Activated Companies to manage users within the company's customer account.
The Manage Users page features a table displaying all Service Hub users, both active and pending. It also displays key user information, as defined below. NameDisplays the name affiliated with the user. This field cannot be modified.EmailDisplays the email affiliated with the user. This field cannot be modified.Site AccessDisplays which sites a user can view. For more information, see [Change a User's Site Access](/en/spectrum/spectrum/service/service-hub/manage-users/change-a-users-site-access).RoleDisplays the role assigned to the user. For more information, see [User Roles](/en/spectrum/spectrum/service/service-hub/manage-users/user-roles).StatusDisplay the status of the user. These are the user statuses.

- Active: The user has accepted their invitation and their account is operational.

- Pending: The user has received their invitation but has not accepted it.

- Expired: The user's invitation has timed out because they did not accept it within 30 days.
