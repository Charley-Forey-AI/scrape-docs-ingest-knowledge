# Create a Service Request

Your service customers can make service requests directly
 through Service Hub Work Orders
 page.
You must be a Customer Admin or Site Admin to complete these
 steps.
To create a service request, the service customer
 completes the following steps.

1. From the Work Orders page, select Create Service Request.

1. Enter the following details:

- Site Name

- Site Address

- Site Contact

- Site Contact Email

- Send a copy of this request to site contact?

- Site Contact Phone Number (optional)

- Description of the Issue

- The Work Order Number of the Previous Request
 (optional)

- Equipment (optional)

- Purchase Order Number (optional)

Note: If your company has enabled custom fields, they also
 display here. For more information, see [Add Custom Fields to the Service Request Form](/en/spectrum/spectrum/service/service-hub/getting-started/change-company-settings/add-custom-fields-to-the-service-request-form).

1. Select Send Service
 Request.

An email is sent to the
 email address defined in the portal's service request settings.
The recipient of the
 service request must manually enter the service request into Spectrum.For
 more information about work orders in Spectrum, see [Work
 Orders](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:spectrum_000026).
