# Service Hub - Prior Releases

Review past enhancements to Service Hub.

## Service Hub 2025 R8

This release adds a new Company Administrator role for customers
 with multiple companies. It also adds a paid invoices filter to the Invoices
 page.
New Company Administrator
 Role
This release introduces a new role, Company Admins. They have the
 same permissions as ERP Admins but for a single company.
For more information, see [User Roles](/en/spectrum/spectrum/service/service-hub/manage-users/user-roles) and [Invite a Company Admin](/en/spectrum/spectrum/service/service-hub/manage-users/invite-an-admin#task-6c822fad-0dbb-4016-bdd8-cc3960036b65--en).
Paid Invoices Filter
ERP users and customer users can now view paid invoices on the
 Invoices page by selecting the Paid filter.
For more information, see [Invoices](/en/spectrum/spectrum/service/service-hub/invoices).

## Service Hub 2025 R3

This release adds several new features for Spectrum users.
Manage Users' Site Access
Customer admins can now limit new and existing users' site access
 to selected sites.
For more information, see [Change a User's Site Access](/en/spectrum/spectrum/service/service-hub/manage-users/change-a-users-site-access)
 and [Invite a User](/en/spectrum/spectrum/service/service-hub/manage-users/invite-a-user).
Copy the Site Contact on Service
 Requests
When creating a service request, users can now send a copy of it
 to the site contact.
For more information, see [Create a Service Request](/en/spectrum/spectrum/service/service-hub/work-orders/create-a-service-request).

## Service Hub 2025 R2

This release adds several requested enhancements for Spectrum users.
Invoices Now Show Service
 Contracts
Invoices now dynamically display either a work order or service
 contract field depending on the type of work.
 New Service Request Form
 Fields
Users can now optionally add equipment and purchase orders details
 to their service requests. For more information, see [Create a Service Request](/en/spectrum/spectrum/service/service-hub/work-orders/create-a-service-request).
 Suggestion Box
Service Hub now
 has a suggestion box for users to directly submit feedback. Select the lightbulb
 icon  at the top right of the ERP Administrator context to
 add your idea to the suggestion box.
