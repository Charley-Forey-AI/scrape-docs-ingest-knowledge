# Change a User's Role

Follow these steps to change a user's role in Service Hub.
You must be an ERP Admin, Company Admin, or Customer Admin to complete these steps.

1. Navigate to the Manage Users page.

1. Use the search bar to find the appropriate user by name or email.Tip: Use the  Filters to further refine your search by Role
 or Status.

1. Select the cell in the Site Access column for the
 user.

1. Select the new user role that you want to assign
 to the user.

A pop-up message displays to confirm that the user was
 updated successfully.
