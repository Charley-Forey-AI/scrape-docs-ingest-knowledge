# Add Custom Fields to the Service Request Form

Follow these steps to add custom fields to a company's Service
 Request form.
You must be an ERP Admin or Company Admin to complete these
 steps.

1. If you administrate multiple companies, select the
 company whose Service Request form you want to edit from the list of My
 Activated Companies.

1. Select the pencil icon  next to Service Request
 Settings.

1. Select Add New Field.Note: You can add up to four custom fields.

1. In the Required column, check the box if you want to
 mark the field as required.

1. In the Field Description column, enter the name you want
 to display for the field.

1. If you need to delete a field, select the ellipses icon  then select Delete
 Field.

1. Select Update.

The custom fields display on all new Service Request
 forms.
