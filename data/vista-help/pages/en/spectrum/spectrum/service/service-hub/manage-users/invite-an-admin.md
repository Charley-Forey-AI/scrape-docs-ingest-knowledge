# Invite an Admin

You can invite two different kinds of administrators: ERP
 Admins and Company Admins.
For more information about both roles, see [User Roles](/en/spectrum/spectrum/service/service-hub/manage-users/user-roles).

## Invite an ERP Admin

Follow these steps to invite an ERP Admin in Service Hub.
You must be an ERP
 Admin to complete these steps.

1. In the ERP Administration page, select
 Add Administrators.The Add ERP Administrator page opens.

1. Enter the new administrator's details in the
 First Name,
 Last Name, and
 Email
 fields.Note: This email will be used to
 invite the user to the portal.

1. Select Invite and Close to send the
 invitation and return to the Company Settings page. Alternatively, select
 Invite and Add New
 to send the invitation and invite a new contact.

The contacts are sent
 an email invitation to join your customer account. They have 30 days to accept the
 invitation before it expires and a new one must be issued.
To reinvite a
 contact, navigate to the ERP Administration page and select the ellipses icon
  next to the user you want to reinvite.
 Select Resend
 Invitation.

## Invite a Company Admin

Follow these steps to invite a Company Admin in Service Hub.
You must be an ERP Admin or Company Admin to complete these steps.

1. In the ERP Administration page, select the
 appropriate company from the list.The Company Settings page opens.

1. Select Add Administrators.

1. Enter the new administrator's details in the
 First Name,
 Last Name, and
 Email
 fields.Note: This email will be used to
 invite the user to the portal.

1. Select Invite and Close to send the
 invitation and return to the Company Settings page. Alternatively, select
 Invite and Add New
 to send the invitation and invite a new contact.

The contacts are sent
 an email invitation to join your customer account. They have 30 days to accept the
 invitation before it expires and a new one must be issued.
To reinvite a
 contact, navigate to the Company Settings page and select the ellipses icon  next to the user you want to reinvite.
 Select Resend
 Invitation.
