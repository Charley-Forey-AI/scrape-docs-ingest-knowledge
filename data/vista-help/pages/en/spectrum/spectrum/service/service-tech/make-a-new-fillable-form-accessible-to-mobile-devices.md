# Make a New Fillable Form Accessible to Mobile Devices

In order for Service Tech Mobile users to access fillable forms, the forms must be stored in a certain and specific location in the Document Imaging (DI) module in Spectrum.
The intended fillable form (PDF file) must already exist and must reside in a location you can browse to.
Each PDF form requires a Spectrum record to contain it.To create the record and attach a fillable PDF form to it:

1. Go to Document Imaging > New Document.

1. Create a new DI record which will hold the form. The location you create the record dictates whether the form is site specific or company wide.

1. If the Form should be accessible to all users:

1. In the Cabinet field, select SERVICE TECH.

1. In the Drawer field, select COMPANY FORMS.

1. Entry in the Folder field is required, but it doesn't matter what you enter as a value.

1. If the Form should be accessed only by users assigned to a given site:

1. In the Cabinet field, select WO.

1. In the Drawer field, select SITE FORMS.

1. In the Folder field, select the folder pertaining to the site.

1. In the Name field, enter a name for the document.

1. Select Create.

1. Upload the PDF file to the new DI record.

1. Select the left-pointing arrow to expand the slider bar.

1. Select Attach, then Choose File.

1. Browse to and select the file.

1. Select Upload File.

1. Select Save.

The PDF form is now accessible to the relevant Service Tech Mobile app users.
If needed, you can [view or update an existing form](/en/spectrum/spectrum/service/service-tech/view-or-update-a-fillable-form).
