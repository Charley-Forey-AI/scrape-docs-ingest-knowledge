# End of Support for Service Tech Mobile on Windows Mobile Devices

Information regarding the end of support for Service Tech Mobile on Windows Mobile devices. If you use Windows Mobile Devices to access the STM application, please send this notification to your Systems Administrator.
This is notice of a significant change that will affect your usage of the Service Tech Mobile (STM) application on any Windows Mobile Device.
After March 5, 2025 you can no longer download the STM application on Windows Mobile Devices, and essential parts of your STM application may not work as expected.
So after careful consideration in adherence to our software and services support policies, we ended support for Windows Mobile Devices for the STM application, effective March 5th, 2025.

## Why is this happening?

Microsoft ended support for the Windows Subsystem for Android (WSA) on March 5, 2025. WSA is essential for users to run Android apps on Windows 11, and its discontinuation means the Amazon Appstore will no longer be supported on Windows mobile devices.
The Service Tech Mobile application is reliant on Amazon Appstore to run as expected on Windows Mobile Devices. Microsoft ending support for WSA means if something goes wrong with your STM application on a Windows Mobile Device, our support and development experts will not be able to help you.
The continued success of your business is important to us. The decision to discontinue support for Windows Mobile Devices for the Service Tech Mobile application is driven by our dedication to ensuring the security and efficiency of your construction projects.

## Recommended Actions

To ensure the continuity of your operations in-line with your product experience, we strongly recommend you make provisions to stop using Windows Mobile Devices to access or run your Service Tech Mobile application.
We understand that this transition may require significant adjustments, and we are committed to providing assistance throughout the process. If you need to discuss these changes with us, please contact your success Manager.
