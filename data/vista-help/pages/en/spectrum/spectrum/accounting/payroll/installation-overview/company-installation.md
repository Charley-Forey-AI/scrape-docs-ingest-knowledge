# Company Installation

Click Site Map > System Administration > Installation > Company.
Note that this screen should have been previously completed by the System
 Administrator.
Ensure that the box for this specific module has been checked.
After the Company Installation screen has been completed, click Save to save the information and return
 to the Installation list.
