# Create or Edit a Deduction or Add-on Code

Follow these steps to create a new deduction or add-on
 code.
Deductions are amounts withheld from a paycheck. Add-ons are amounts added to a paycheck.

1. On the Site Map, go to Payroll > Maintenance > Deduction/Add-on Code.

1. Select New and enter
 the deduction or add-on code (2-10 characters) you are adding.

1. In the Descriptions field, enter a description up to 15 characters.

1. Select the type of code you are adding: Deduction or Add-on.

1. In the Calculation method field, select the calculation method you want to apply to the new code, and then press Enter.

1. Proceed through the Properties tab to designate
 additional code properties.Note: Depending upon the calculation method you have selected,
 certain Properties fields may or may not be
 applicable.

1. Once you have completed entry for all Properties information, select the
 Tax Effects tab and
 enter the applicable tax information for the new code.Note: If the new code has no tax effects and is classified as
 an Add-on, proceed to the Add-on tab and then to the
 Limits tab. If the new code has no tax effects and is
 classified as a Deduction, proceed to the Limits tab and
 then to the A/P Invoice tab. For more information on the
 entering information in the fields contained within the
 Deduction/Add-on Code Maintenance screen, select [Deduction / Add-on Code Maintenance](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/deduction--add-on-code-maintenance).

1. Once you have entered all the necessary information on all applicable tabs, select OK twice to return to the Site Map.
