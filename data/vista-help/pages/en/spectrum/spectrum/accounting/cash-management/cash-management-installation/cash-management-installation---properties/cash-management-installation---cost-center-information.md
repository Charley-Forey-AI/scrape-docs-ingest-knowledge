# Cash Management Installation - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, Spectrum offers the ability to set up separate inter-post and
 inter-company G/L accounts.
The Cost center inter-posting G/L account
 code field will display on this screen, directly below the Direct checks dollar limit field.
 Companies can define a single G/L account for each other company with accounting ties to
 the current company using the Inter-Company G/L account code field and button.
