# Time + Material Installation - Markups

Use this tab to select the markup settings for the Time + Material module. These settings can be changed as needed at any time.
These markups are applied to the price/cost levels previously specified
 in this screen for equipment, material, and labor. Entries in the [Job-Specific Markups](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-markups/time--material-installation---markups/job-specific-markups) window are
 originally based on defaults established in the Time + Material Installation screen, but
 these defaults are not used when calculating. When calculating, Spectrum looks to the
 amounts specified in this window and the [Cost Type Markup File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/cost-type-markup-file-maintenance) screen.
Fields
Descriptions

Line
Line numbers are assigned and maintained by the system.

Ct
Enter or search for the cost type to be marked up. The cost type must be a valid code already set up in Cost Type File Maintenance.

Description
The cost type description displays. You can edit the description.

Type
Use the drop-down menu to select the type of markup: Percentage or Rate per hour.

Basis
This field works in conjunction with the Type field to allow you to establish different markups to various pay types. For example, cost plus contracts may specify a different markup percent for regular payroll hours versus overtime hours.
From the drop-down list, select the labor transactions that will be used when calculating markup rates. Transaction options include:

- All transactions,

- Regular payroll,

- Overtime payroll,

- Double time payroll, and

- Other Payroll transactions.

Amount
Enter the markup amount (either a flat rate or the percentage).
