# Time + Material - Run Pre-Billing Report

Run the Pre-Billing Report on a regular basis (at least
 monthly) to identify which Time + Material transactions have or have not been selected for
 billing based on a specified date range.

1. Select Time + Material > Reports > Pre-Billing.Note: For field descriptions and a sample report, see [Pre-Billing Report](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/reports-overview/pre-billing-report).

1. Enter selections for Job,
 Division, Job status,
 Phase, and Cost type, or accept the
 defaults.

1. Select the report date range:

- Use From transaction date and Through transaction date to select the first and last transaction dates to include.

- Use From PR work date and To PR work
 date if you want to include payroll transactions only.

1. Under Billing status, select
 one of the following:

- All - View all Time + Material
 transactions not yet billed.

- Selected - View only Time + Material
 transactions already selected for billing.

- Unselected - View only Time +
 Material transactions not yet selected for billing.

1. Choose sorting and other options.

1. Click Preview. The system
 generates a report showing transaction amounts plus any markup.
