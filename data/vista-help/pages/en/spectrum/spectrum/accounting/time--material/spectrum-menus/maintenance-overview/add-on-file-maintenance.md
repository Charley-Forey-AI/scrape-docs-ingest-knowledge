# Add-On File Maintenance

Use the Add-On File Maintenance screen to set up a file of standard add-ons. Specific add-ons may later be selected from this file and applied to a Time + Material or Cost Plus job.
Click New or Edit to add or modify the information displayed in the list box. The [New/Edit Add on File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/add-on-file-maintenance/newedit-add-on-file-maintenance) window displays.
