# New/Edit Equipment Billing Rates

Use this window to add or edit Billing codes and up to five
 pay rate levels.
Fields
Descriptions

Billing code
Enter or search for the Time + Material equipment billing
 rate code. The billing code can be up to 10 characters.
This code will then be available when entering equipment
 usage. Equipment usage can be entered in either the Payroll or Equipment
 Control modules. Billing codes can also be used for Billing Transaction
 Entry.

Description
Enter the time + material equipment billing rate
 description.

Rates

Level
Enter the equipment billing rate for level 1 through 5 in
 each of the three rate type columns: Regular, Overtime, and Double time. The
 level number refers to the labor level number that is entered in the
 T+M Job Billing Setup screen.

Stand-by rates
Enter stand-by equipment billing rates for Hourly, Daily, Weekly and Monthly periods. These
 rates will only be posted to Equipment Control > Revenue Entry.
