# Time + Material - Set Up T+M Equipment Billing Rates

Optionally, set up T+M billing rates for equipment
 usage.
For example, you may want to charge different rates for the
 use of backhoes, medium-sized bulldozers, and other equipment. Assign these billing codes
 to equipment in the Equipment Control module. These rates override default billing rates
 set up in the Equipment Billing Rates Maintenance screen. The system automatically applies
 these rates when equipment usage hours are entered against a Time + Material job.

1. Select Time + Material > Maintenance > Equipment Billing Rates.For field descriptions, see [New/Edit Equipment Billing Rates](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/equipment-billing-rates-maintenance/newedit-equipment-billing-rates).

1. Click New to create a new line
 entry. To modify an existing code, select the code in the grid, and click
 Edit.

1. Enter a Billing code and
 Description.

1. Enter equipment billing rate levels 1 through 5 in
 each of the three rate type columns: Regular,
 Overtime, and Double time.

1. If applicable, set stand-by equipment billing rates
 for Hourly, Daily,
 Weekly, and Monthly periods.

1. When you are finished, click
 OK. The Equipment Billing Rates grid
 displays the rates set up for each billing code.

1. [Time + Material - Assign Equipment Billing Codes to
 Equipment](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---assign-equipment-billing-codes-to-equipment).
