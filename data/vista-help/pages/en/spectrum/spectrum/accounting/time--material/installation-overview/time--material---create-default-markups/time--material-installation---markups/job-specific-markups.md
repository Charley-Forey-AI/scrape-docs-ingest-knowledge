# Job-Specific Markups

These markups are applied to the price/cost levels
 previously specified in this screen for equipment, material, and labor.
Entries in this window are originally based on defaults established in
 the Time + Material Installation screen, but these defaults are not used when calculating.
 When calculating, Spectrum looks to the amounts specified in this window and the [Cost Type Markup File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/cost-type-markup-file-maintenance) screen.
In this window you may enter the same Markup cost type multiple times if more than one type of markup is needed for a particular cost type. This is especially handy for cost plus projects that call for different markup percentages for material, labor, and so forth. They can also be used to mark up materials, subcontract and other charges on a T+M bill.
You can add, edit, or delete markups as necessary.
