# Reverse T+M Billing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job Customer
Enter the job number of the billing to be reversed. The customer code and description display.

Billing #
Enter the billing number to be reversed. You can also select from a list of available bill numbers from the Posted Bills window. If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Change request #
If the billing was updated to a change request, the change request # will display and the remaining fields will be hidden. In their place, an informational section with the Original C/R amount sent to the A/R change request will display below.

G/L date
The invoice general ledger date displays.

A/R invoice batch
Enter the batch code identifying the Accounts Receivable invoice to be reversed. The Billing summary totals will display in the section below.
