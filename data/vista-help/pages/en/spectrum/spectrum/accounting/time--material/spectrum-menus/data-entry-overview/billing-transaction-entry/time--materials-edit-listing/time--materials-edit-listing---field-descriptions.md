# Time + Materials Edit Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Dates
Enter the first and last transaction dates you want to view on the report.

Batch code
Enter a batch code to separate your transactions from others for data entry and update purposes.

Transaction types
Select the transaction type(s) you want to view:

- Accounts Payable

- Equipment

- Inventory

- Job Cost

- Payroll
