# Change Equipment Billing Rates

This screen is used to change equipment billing rates on a specific bill only after that bill has been selected. If it is necessary to change the rate on line items that have not been selected, the items must first be selected; make the changes through this screen, then deselect the transactions.
Fields
Descriptions

Job
Enter the Time + Material job number. Based on the number entered, the full job description will display.

Billing #
Enter the billing number to change, or press Enter for the current billing number. If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Selections

Equipment code
Press Enter for ALL equipment, click the arrow to search, or enter the equipment code to change billing rates for a specific piece of equipment. Based on the code entered, the full equipment description will display, as well as the equipment type and type description.

Equipment type
Press Enter for ALL types, click the arrow to search, or enter the type code to change the billing rates for a specific type of equipment. Based on the code entered, the full description will display.

Old rates

From billing code
Enter the billing code to be changed. Click the arrow to select from a list of available billing codes.

- Regular

- Overtime

- Double time

Regular, overtime, and double time rates display.

New rates

To billing code
To keep the same rates but change billing codes, enter the new billing code, or click the arrow to search for valid billing codes.
 Otherwise, to change rates but keep the same billing code, repeat the code entered above in the From Billing Code field, then enter the new rates.

- Regular

- Overtime

- Double time

Regular, overtime, and double time rates display.
