# Reverse T+M Billing - VAT processing information

If the 'Utilize value added tax (VAT) tracking?' checkbox is selected in General Ledger Installation, VAT tax information will be included when creating the A/R credit memo (reverse T=M invoice) for a billing that had a VAT tax code assigned to it.
