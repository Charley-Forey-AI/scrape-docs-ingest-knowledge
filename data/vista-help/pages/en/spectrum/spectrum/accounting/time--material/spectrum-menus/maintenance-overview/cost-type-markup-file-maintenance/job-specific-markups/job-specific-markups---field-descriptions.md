# Job-Specific Markups - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Cost type
Add or edit the marked-up cost type. After selecting the Cost type, the description displays, but it can still be edited.

Description

Type
Select the type of markup for this job: Percentage or Rate per hour.

Basis
This field works in conjunction with the Type field to allow you to establish different markups to various pay types. For example, cost plus contracts may specify a different markup percent for regular payroll hours versus overtime hours.
From the drop-down list, select the labor transactions that will be used when calculating markup rates. Transaction options include:

-  A - All transactions

- R - Regular payroll

- O - Overtime payroll

- D - Double time payroll

- P - Other payroll

Amount
The markup amount displays (either a flat rate or the percentage). You can edit the amount for this job.
