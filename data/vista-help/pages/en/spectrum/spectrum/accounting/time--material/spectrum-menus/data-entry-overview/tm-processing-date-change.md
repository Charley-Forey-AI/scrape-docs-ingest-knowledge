# T/M Processing Date Change

This screen is used to view or change the Processing date.
Fields
Descriptions

Current settings

Processing date
The left column displays the current Time + Materials processing information.
 The information is re-displayed on the calendar on the right. Enter the new
 processing date by typing over the displayed date, or click the date on the
 calendar. To leave the processing date as it is displayed, click the
 Reset button to restore the present setting. Click
 Cancel to exit the screen without saving the
 changes. If the new date is before the minimum processing date or after the
 maximum processing date established during the Installation procedures, the
 following error message displays: DATE ENTERED IS NOT WITHIN THE current
 MINIMUM AND MAXIMUM DATE RANGE.

Fiscal year
The fiscal year in which the requested date is located displays.

Period
The period number in which the requested date is located (number 1-13, depending on the General Ledger settings) displays.

Period start date
The start date of the period displays.

Period end date
The ending date of the period in which the requested date is located displays.

Allowable entry range

Minimum date
The first processing date displays. No entry is allowed. This date is set by
 the system administrator in the Admin > Installation > Processing Date Change screen.

Maximum date
The last processing date displays. No entry is allowed. This date is set by
 the system administrator in the Admin > Installation > Processing Date Change screen.
