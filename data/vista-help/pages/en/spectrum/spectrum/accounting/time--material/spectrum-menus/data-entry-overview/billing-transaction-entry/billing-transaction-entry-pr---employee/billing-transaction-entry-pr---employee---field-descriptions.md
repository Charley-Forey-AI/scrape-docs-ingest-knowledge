# Billing Transaction Entry: PR - Employee - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Description

Employee code
Enter or search for a valid Employee code in this field. Based on the code
 entered, the employee name will display from the Employee
 Maintenance.

Name

Job
Enter or search for the job code in this required field. Based on the number entered, the full job description will display from Jobs.

Phase
Enter or search for the phase code in this required field. Phases containing up to 20 characters can display in this column.
Note: Data entry is prevented if the phase status is set to Complete. A warning
 displays if the status is set to Inactive.

Ct
Enter or search for the Cost type code in this required field. Then enter the check number.

Check #

Pay type
Enter the pay type: Regular,
 Overtime or Double time.
 Entry in this field is required and will determine the Billing rate for this
 transaction.

Total hours/qty
Enter the hours or quantity for this employee pay line.

Billing rate
Enter the billing rate for this employee. If the employee has a billing code
 set up in Payroll > Employee File
 Maintenance, the associated Billing rate will be used here; otherwise,
 the Billing rate will be blank.

Total amount
The dollar total for this pay line (Total hours multiplied by Billing rate)
 displays. Note that this field is linked to the Billing rate. When changes
 are made in this field, changes will also be made to the Billing
 rate field.

Remark
Enter any additional information you would like to include that is relevant to this transaction.

Job name
This information displays after you enter data in the Job and Phase columns.

Phase description
