# Time + Material - Create Job-Specific Add-Ons

As needed, create special add-ons to use on a job by job
 basis.
For example, you may want an insurance add-on of 20% or a
 general maintenance add-on of 17%. Add these amounts to the [Time + Material - Create Default Add-Ons](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-add-ons) established
 for Time + Material jobs, or use them in place of default add-ons.

1. Select Time + Material > Maintenance > Add-ons.For field descriptions, see [New/Edit Add on File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/add-on-file-maintenance/newedit-add-on-file-maintenance).

1. Click New to create a new line
 entry.

1. Enter an Add-on code and
 Description.

1. Enter the Add-on percent.

1. When you are finished, click
 OK.
