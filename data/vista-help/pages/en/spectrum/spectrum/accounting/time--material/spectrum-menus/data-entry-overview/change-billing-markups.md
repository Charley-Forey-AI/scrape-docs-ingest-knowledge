# Change Billing Markups

This screen is used to change markups for a specific bill only after that bill has been selected. If it is necessary to change the rate on line items that have not been selected, the items must first be selected; make the changes through this screen, then deselect the transactions.
Fields
Descriptions

Job
Enter the Time + Material job number. Based on the number entered, the full job description will display.

Billing #
Enter the billing number to change, or press Enter for the current billing number. If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Selections

Cost type
Enter the cost type for which markups are to be changed, or click the arrow to select from a list of available cost types.

Standard type
The markup type displays: Percentage of extension, or hourly Rate (see the complete list of types below). If the selected cost type does not currently have a markup specified, the screen will indicate it is a New Markup.

- Percentage (all transactions)

- Rate per hour (all transactions)

- Percentage (regular payroll)

- Rate per hour (regular payroll)

- Percentage (overtime payroll)

- Rate per hour (overtime payroll)

- Percentage (double time payroll)

- Rate per hour (double time payroll)

- Percentage (other payroll)

- Rate per hour (other payroll)

Standard amount
The standard markup displays.

Change to

New type
Select the new markup type from the drop-down menu: Percentage of extension or hourly Rate per hour.

New basis
Select the new markup basis from the drop-down menu:

- All transactions

- Regular payroll

- Overtime payroll

- Double time payroll

- Other payroll

New amount
Enter the new markup rate or percentage.
