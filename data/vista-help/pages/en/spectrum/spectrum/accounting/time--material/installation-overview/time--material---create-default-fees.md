# Time + Material - Create Default Fees

If your organization applies standard fees to time and
 material jobs, set these fees as defaults on the Time +Material Installation -Fees
 tab.
The system applies
 these defaults to all T+M jobs; however, you can customize these settings per job as
 needed.

1. Select System Administration >  Installation >  Time + Material.

1. Click the Fees tab.For field descriptions, see [Time + Material Installation - Fees](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-fees/time--material-installation---fees).

1. Click New to create a new line entry.

1. Enter a Description for the fee.

1. Select the Type of fee: Flat fee or Percent.

- If you select Flat fee, the cursor skips to the Amount
 column.

- If you select Percent, the cursor advances to the Base on
 column.

1. For fees that are a percentage, use the Base on column
 to select the basis for the fee:

- C (Individual cost type): Base this fee on a specific cost
 for the job. For example, labor. If you select C, specify a Cost type in the
 Add-On Cost Types window. You can select multiple cost types.

- S (Subtotal of all cost types): Base this fee on all costs
 for the job.

- T (Total including add-ons): Base this fee on all costs plus
 the add-ons included in the job.

1. In Amount, enter either a percentage or flat rate for
 the fee.

1. If this fee is taxable, select the Tax checkbox.


1. When you are finished, click Save.
