# New/Edit Job Labor Billing Rates

Use this window to add or edit job labor Billing codes and up to five pay rate levels.
Fields
Descriptions

Job
Enter a job number. The job description will display, as well as all fields linked to the job.

Customer
Based on the job number entered, the Customer code and name display.

Billing code
Enter or search for the labor Billing rate code. The billing code can be up to 10 characters.
This code will then be available when entering payroll hours. An employee will be paid the pay rate indicated in the payroll module, but the job will be billed the rate corresponding to this billing code. Billing codes can also be used for Billing Transaction Entry.

Description
Enter a description for this Time + Material labor billing rate.

Markup type
Entry in this field allows you to set up standard billing rates with the labor burden calculated separately as a labor markup. This Markup is stored in the Time + Material system, which also allows you to either print the totals combined or with the markup printing separately.
Enter the markup type:

- Percentage

- Rate per hour

- None
Note: This setup screen is not applicable for pay type markup.
 Markups for specific pay types are recorded in T+M Job Billing Setup,
 across bill codes. Any markups assigned in this screen are calculated
 in addition to any markups specified in T+M Job Billing Setup.

Rates

- Level

- Regular

- Overtime

- Double time

- Markup

Enter the job labor billing rate for levels 1 through 5 in each of the four rate type columns: Regular, Overtime, Double time, and Markup. The level number refers to the job labor level number that is entered in the T+M Job Billing Setup screen.
Note: If None is selected from the Markup type drop-down menu, then
 the Markup column is not applicable.
