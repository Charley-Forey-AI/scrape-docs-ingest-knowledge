# Cost Type Billing History Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Selections

Job
Enter the job numbers you want to display on the report.

Bill #
Enter the number of the bill to print, or press Enter to print the largest posted bill number stored in the billing history file for each job.
When a specific job is selected, enter ALL to print the history report with all posted billings for the job, or double-click to select from a list of posted bill numbers.
 If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Print job notes?
Select the checkbox to print job descriptions.

Format

Print markups?
Select whether or not to display detail with markups.

Report type

- Summary

- Detail by phase

- Detail by phase with group subtotals

Select the report type you want to view.
