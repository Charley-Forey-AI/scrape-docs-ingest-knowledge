# Cost Type Markup File Maintenance

Use this screen to set up a file of standard markups that
 will be used for all Time + Material jobs, provided that the Markup checkbox is selected in Job Phases > Properties. These markups are for default purposes only.
Entries are originally based on defaults established in the
 Time + Material Installation screen, but these defaults are not
 used when calculating. When calculating, Spectrum looks to the amounts specified here and
 in the [Job-Specific Markups](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/cost-type-markup-file-maintenance/job-specific-markups) window.
Important: If markups have been entered in T+M
 Job Billing Setup, those figures will be used.
Fields
Description

Cost type
Enter or search for a Cost type for this default markup code; only one entry
 per cost type is allowed for standard markups. This must be a valid cost
 type code previously defined in Cost Type File
 Maintenance.

Markup description
Enter this cost type's markup description. The cost type description will default from the job cost system, but can be overwritten. Examples of cost types are material, subcontracts, or other.

Markup type
Select the type of markup calculation for this cost type:

- Percentage

- Rate per hour

- None

Markup basis
This field works in conjunction with the Markup type
 field to allow you to establish different markups to various pay types. For
 example, cost plus contracts may specify a different markup percent for
 regular payroll hours versus overtime hours.

- All transactions

- Regular payroll

- Overtime payroll

- Double time payroll

- Other payroll

Amount
Enter this cost type's default percentage or hourly rate.
