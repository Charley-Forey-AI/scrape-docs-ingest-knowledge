# Time + Material - Create Cost Type Markups

As needed, create cost type markups to use on a job by job
 basis.
For example, you may want to mark up materials by 25% for a
 specific job or labor by 50%. Add these amounts to the [Time + Material - Create Default Markups](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-markups) established
 for T+M jobs (Time + Material Installation - Markups tab), or use them in place of default
 markups.

1. Select Time + Material > Maintenance > Cost Type Markup.For field descriptions, see [Cost Type Markup File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/cost-type-markup-file-maintenance).

1. Click New to create a new line
 entry.

1. Enter a Cost type code and
 Markup Description.

1. Select the Markup Type:
 Percentage, Rate per hour, or
 None.

1. Select the Markup Basis. This
 field indicates how you want the system to calculate this markup:

-  Based on All transactions (typically used for
 materials)

- Based on a specific pay type, such as Regular, Overtime,
 Double time, or Other payroll (typically used for labor)
Note: For details on how the system calculates markups for
 labor rates, see [Calculate Markups Based on Pay Type](/en/spectrum/spectrum/accounting/time--material/procedures-overview/in-depth-overview/calculate-markups-based-on-pay-type).

1. In Amount, enter either a
 percentage or flat rate for the markup.

1. When you are finished, click
 OK. The Cost Type Markup grid displays
 the markups set for each cost type.
