# Phase Summary Billing Form - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Selections

Job
Enter or search for the job numbers you want to view.

Billing #
Enter the number of the bill to display on the report, or press Enter to preview the largest posted bill number stored in the billing history file for each job. When a specific job is selected, enter ALL to print the history report with all posted billings for the job, or double-click to select from a list of posted bill numbers. If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Print job notes?
Select the checkbox if detailed job descriptions are to be printed. The detail job descriptions are printed from Job Notes.

Format

- Detail with markup shown

- Detail with no markup display

Select whether or not to display details with markups.

Report type

- Summary

- Summary with group subtotals

- Detail

- Detail with group subtotals

Select the report type you want to view.
