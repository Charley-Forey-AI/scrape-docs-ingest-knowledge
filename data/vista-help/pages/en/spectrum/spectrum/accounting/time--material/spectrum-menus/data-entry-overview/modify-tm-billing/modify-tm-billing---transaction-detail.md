# Modify T+M Billing - Transaction Detail

Use this tab to view and edit billing transaction details.
Fields
Descriptions

Selections
Click this button to open the Selections window. Use this window to limit transactions displaying on this tab by specifying phase, cost type, transaction dates, and transaction types.

Phase Ct
The associated phase (up to 20 characters) and cost type display.

Tran date
Click the arrow to change the transaction date.

Type
The billing source type displays: AP, EQ, GL, IC, JC, or PR.

Description (name)
You can edit the name associated with the transaction. The first 20 characters display.

Hours/qty
The total hours (or units, if applicable) display. If you edit this field, the Extension and Bill amount will also change.

Pay type
This column displays one of the following pay types for the Payroll type transactions:
R (Regular)
O (Overtime)
D (Double time)

Billing rate
The Billing rate per hour (or unit) displays. If you edit this field, the Extension and Bill amount will also change.

Extension
The billing extension (total bill less markup) displays. If you edit this field, then the Bill amount will also change.

Markup
The Markup displays. If you edit this field, then the Bill amount will also change.

Bill amount
The dollar total for each line (Extension plus Markup) displays.

Taxable?
 The checkbox is selected if this transaction is taxable.
The information that displays varies based on the Type (AP, EQ, JC, PR, GL, or IC).

Message
Enter any additional comments regarding each line transaction.

Code Reference P.O. #
You can edit the code, reference, and purchase order number for this transaction.

Work date
The date on which the equipment was used displays.

Check date
The Check date displays.

Billing code Phase description
The billing code and phase description for the transaction display. The billing code can be up to 10 characters.

Sub-job
If this is a sub-job billed from a master job, the sub-job number and description will display in this column.
