# Billings-in-Progress Summary window

This window is used to review and deselect transactions on a billing selection before it is processed.
Fields/Buttons
Descriptions

Buttons

Detail Billing
 Click to view the [Detail Billing Selection](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/detail-billing-selection) screen for the highlighted job in a new tab.

Remove Billing
Click to remove the job for this billing cycle

List box

Job
A list of job numbers associated with the billing selection displays. Eighteen jobs can display in the list box before scrolling further.

Billing #
A list of bill numbers associated with the billing selection displays.

Transaction count
The number of transactions completed for the corresponding bill number displays.

Transaction $
The total transaction amount for the corresponding bill number displays.
