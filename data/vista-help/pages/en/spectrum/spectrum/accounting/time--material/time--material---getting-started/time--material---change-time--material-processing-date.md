# Time + Material - Change Time + Material Processing Date

Use the T/M Processing Date Change screen to specify the
 date range for processing T+M transactions. Changing the processing date requires System
 Administrator access.

1. Select Time + Material > Data Entry > Processing Date Change.

1. Select a new processing date using the calendar.For field descriptions, see [T/M Processing Date Change](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/tm-processing-date-change). For more information on processing dates, see Change Processing Dates in Spectrum Help.
