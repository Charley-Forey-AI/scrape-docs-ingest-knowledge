# Override Fees

Use this window to view or modify fees.
Field
Description

Description
Enter the description for this default fee.

Type
Enter the type of fee: Flat or Percentage.
If Flat fee is entered, the system skips to the Amount column. If Percentage is entered, the system advances to the next field.

Base
Enter C to base the fee on individual cost types, S to base the fee on the subtotal of all cost types, or T to base the fee on the total (including any add-ons). If S or T is entered, the system skips to the Amount column. If C is entered, the system advances to the next field.

Cost Types
Enter the Cost type(s) to be included in this fee.

Amount
Enter the amount of the flat fee or percentage on which to base the fee.

Extension
The extension amount displays. The maximum value can be up to $99,999,999.99

Taxable?
Select this checkbox if this add-on line is taxable.
