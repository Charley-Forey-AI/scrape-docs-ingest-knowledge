# Add-on History Window

Use this screen to view add-on history details.
Fields
Descriptions

Add-on
The add-on code and description display.

Description

Cost types
The Cost type(s) to be included in this add-on display.

Add-on %
The add-on percent displays.

Extension
The system has automatically calculated the extension.

Taxable?
Yes displays if this add-on is taxable, No displays if this add-on is not taxable.
