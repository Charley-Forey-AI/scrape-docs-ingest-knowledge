# Time + Material

Use the Time + Material module to calculate and prepare
 billings for Time + Material jobs. Access Time + Material options from the Site Map and the
 Jobs Info Bar. The module is integrated with Payroll, Job Cost, Inventory Control, Accounts
 Payable, Document Imaging, and Equipment Control, as well as the General Ledger and Accounts
 Receivable, to accumulate all billing amounts automatically.
To set up and use Time + Material, follow these steps:

- Configure default settings for T+M jobs ([T+M Installation](#ID-0000008d--en__ID-0000009d)).

- If applicable, define job-specific rates, markups, or add-ons ([T+M Maintenance](#ID-0000008d--en__ID-000000a9)).

- Process T+M transactions, and generate bills ([T+M Data Entry](#ID-0000008d--en__ID-000000bc)).

Refer to the following sections for details.

## T+M Installation

Use the System Administration >  Installation >  Time + Material menu to set defaults for Time + Material jobs and create any standard markups,
 add-ons, or fees that your organization uses. See [Installation Overview](/en/spectrum/spectrum/accounting/time--material/installation-overview) for
 additional information.

- [Time + Material - Set Default T+M Properties](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---set-default-tm-properties).

- If applicable, create standard [Time + Material - Create Default Markups](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-markups), [Time + Material - Create Default Add-Ons](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-add-ons), or [Time + Material - Create Default Fees](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-fees).

## T+M Maintenance

Optionally, use the Time + Material >  Maintenance menu to define billing rates for specific jobs, and, if applicable, create
 special rates, markups, or add-ons that you can add to jobs as needed. If you do not set
 up special rates, the system uses standard employee rates. If you do not set up
 job-specific markups or add-ons, the system uses the default settings from the Time +
 Material Installation screen. See [Maintenance Overview](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview) for
 additional information.

- Create default [Time + Material - Set Up T+M Labor Billing Rates](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---set-up-tm-labor-billing-rates) and [Time + Material - Set Up T+M Equipment Billing Rates](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---set-up-tm-equipment-billing-rates) rates.

- Assign T+M billing codes to [Time + Material - Assign Labor Billing Codes to Employees](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---assign-labor-billing-codes-to-employees) and [Time + Material - Assign Equipment Billing Codes to
 Equipment](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---assign-equipment-billing-codes-to-equipment).

- Create [Time + Material - Create Cost Type Markups](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---create-cost-type-markups) and [Time + Material - Create Job-Specific Add-Ons](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---create-job-specific-add-ons).

- [Time + Material - Customize Billing Setup, Labor Rates, and
 Equipment Rates for a Job](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---customize-billing-setup-labor-rates-and-equipment-rates-for-a-job).

## T+M Data Entry

Use the Time + Material  >  Data Entry menu to view and edit T+M job transactions, select transactions for billing,
 generate bills, and transfer T+M data to Accounts Receivable. Note that the system
 processes Time + Material data automatically, with no changes to the workflow for your
 Payroll or Accounts Payable clerks. The following steps provide the standard data entry
 workflow. For a list of all options available from the Data Entry menu, see [Data Entry Overview](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview).

- [Time + Material - Run Pre-Billing Report](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---run-pre-billing-report) (identify unbilled transactions).

- [Time + Material - Update Pre-Time Cards for Billing](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---update-pre-time-cards-for-billing).

- If applicable, [Time + Material - Enter Billing Transactions](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---enter-billing-transactions).

- [Time + Material - Select Transactions for T+M Billing](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---select-transactions-for-tm-billing).

- If necessary, [Time + Material - Modify Billing](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---modify-billing) (use this option before data has been transferred to
 Accounts Receivable).

- [Time + Material - Generate Billing Forms](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---generate-billing-forms).

- [Time + Material - Update Time + Material Billings](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---update-time--material-billings) (transfer data to Accounts Receivable).
