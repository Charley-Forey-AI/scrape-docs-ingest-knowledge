# Vendor Recurring Invoice Entry

This screen is used to enter recurring invoices and credit memos (for example, rent, insurance, or loan payments). Entries are only one time; thereafter, invoices are routinely generated by Spectrum.
Upon completion of this screen, the A/P Recurring Invoice Detail window automatically displays. This window is used to enter line transactions for the recurring invoice.
