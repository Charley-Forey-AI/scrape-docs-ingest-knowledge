# Check Reconciliation File Listing - Cost Center Information

(If the cost center feature is enabled in the Enterprise Installation screen, when printing the reconciliation listing Spectrum includes items only if you have permission to access them. Spectrum compares the cost center assigned in the check with cost centers in your assigned cost center scheme, and if the cost center of the check is not included, then the check will not appear. Spectrum also compares the vendor's list of shared cost centers with cost centers in your assigned cost center scheme, and if there are no common cost centers, then checks for that vendor will also not appear.
