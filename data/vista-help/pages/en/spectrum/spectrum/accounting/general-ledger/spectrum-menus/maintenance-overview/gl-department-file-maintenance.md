# G/L Department File Maintenance

Use the G/L Department File Maintenance screen to set up and maintain departments such as service, construction, and product sales; or departments based on geographic regions such as Washington, Oregon, and Idaho offices.
Companies electing to not departmentalize may set up assets, liabilities, capital, revenue, direct cost, expense, and so forth, based on groupings of accounts in the chart of accounts. Each department is entered with a unique code and description to identify the department.
