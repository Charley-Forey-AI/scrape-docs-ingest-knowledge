# Warning - G/L Account Code Entered Has Inactive Status.

If an inactive equipment code is entered, this message displays. Click OK to continue entering data, if desired.
To change the equipment's status, click Equipment Control  >  Maintenance  >  Equipment to open the Equipment File Maintenance screen. In the Status field, enter a different status code, or double-click on this field or press F4 to select from a list of available equipment status codes.
