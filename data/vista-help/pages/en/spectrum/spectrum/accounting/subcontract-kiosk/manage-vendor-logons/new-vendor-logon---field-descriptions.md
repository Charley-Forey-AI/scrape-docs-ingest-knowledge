# New Vendor Logon - Field Descriptions

Use the table below for reference when completing the fields in the New Vendor Logons screen.
Fields/ButtonsDescriptions
VendorEnter a valid, active vendor code or press F4 to select one.
Contact nameEnter the contact name for the vendor. If an exact match isn't found, the Search Vendor Contacts window opens.
Spectrum logon IDDefault value is the Contact's name. The system verifies the logon ID is not already in use. Make changes as needed.
PasswordAppears above the grid if you have security access to the Operator Maintenance screen. Choose a contact and select this button to open the screen.
Launch e-mail containing logon information?Select to cause the application to send an email after you select Continue. The message will use the vendor's email address and include the Spectrum Logon ID and password.
ContinueSelect to finish creating the new user logon. When starting the next record, the system auto-repeats the vendor code.
