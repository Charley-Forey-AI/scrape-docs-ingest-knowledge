# My Subcontract Manager - Field Descriptions

Use the table below for reference when completing the fields in the My Subcontract Manager screen.
Fields/ButtonsDescriptions
SubcontractThe selected subcontract defaults, or if you accessed this screen from the Site Map, select the drop-down arrow to open the Search My Subcontracts window where you can select from a list of subcontracts for the current operator in the current company.
Switch to Another Subcontract buttonDisplays only if other subcontracts are available.
JobThe job code and description associated with the subcontract display. No changes are allowed.
VendorThe vendor code and description associated with the subcontract display. No changes are allowed.
Subcontract progress
Total contract amountThe total contract amount displays, excluding both sales tax and VAT. This value is the equivalent to the revised contract amount.
Invoiced to-dateThe 'Total amount' of posted invoices for this subcontract, excluding both sales tax and VAT.
Unapproved invoice (review in progress)The 'Total amount' of unapproved and unposted invoices for this subcontract, excluding both sales tax and VAT.
Billing entry (in progress)The 'Total amount' of the billing transaction in progress for this subcontract, excluding both sales tax and VAT. Because My Subcontract Billing Entry does not support multiple billings for the same subcontract at the same time, there will be, at most, one 'Billing Entry' header record.
Remaining to billThe 'Total contract amount' minus 'Invoiced to-date' minus 'Unapproved invoice' minus 'Billing entry'.
Activity Report buttonSelect to display the [Subcontract Activity Details Report - Sample Report](/en/spectrum/spectrum/accounting/subcontract-kiosk/my-subcontract-manager/subcontract-activity-details-report---sample-report).
Unapproved Billing Report buttonAvailable only when there is at least one progress billing that is unapproved or unposted. When selected, the [Unapproved Billing Report Sample Report](/en/spectrum/spectrum/accounting/subcontract-kiosk/my-subcontract-billing-entry/unapproved-billing-report-sample-report) is generated showing the detail by billing for those unapproved or unposted items.
Enter Billing Amounts buttonMay read, 'Continue Billing Entry' if the amount is a non-zero amount. Select to display the [My Subcontract Billing Entry](/en/spectrum/spectrum/accounting/subcontract-kiosk/my-subcontract-billing-entry) screen where you can record completed work.
Subcontract snapshotThe fields in this section provide quick reference information about the
 selected subcontract. Contract totals originate from the [Subcontract Financial Summary](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/subcontracts/subcontract-financial-summary) page.
Warnings and alertsThis section displays all the [Document Tracking Item Maintenance](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/maintenance-overview/document-tracking-item-maintenance) warning messages for the selected subcontract for Action Type #7
 (Display warning message in Subcontract Kiosk). Alerts for sub-tier
 vendors will also appear in this list, below the alerts for the
 selected subcontract. Select the [Submit Documentation](/en/spectrum/spectrum/accounting/subcontract-kiosk/my-subcontract-manager/submit-documentation)
 button to add documents and notes to the selected open tracking
 item.
