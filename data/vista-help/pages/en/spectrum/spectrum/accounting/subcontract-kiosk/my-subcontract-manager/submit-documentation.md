# Submit Documentation

Use this screen to add documents and notes to the specific
 open tracking item for the subcontract.
 Documents are dragged and dropped into the Document Imaging pane, and then submitted to
 the subcontract. The status of the tracking item will then be switched to 'In review' to
 facilitate evaluation in Job Compliance Tracking. At that point, the compliance entry can
 either be approved and switched to 'Closed' status, or switched back to 'Open' status to
 reappear in the Subcontract Kiosk open items list.
Field
Description

Submit
Once documents are attached to this tracking item, click this button to submit to the subcontract.

Subcontract
The subcontract, job, and vendor name display in these fields.

Job

Vendor

For sub-tier vendor
If there is a sub-tier vendor associated with the subcontract, the sub-tier code and description displays in these fields.

Description

Comment
Any comments stored in the Compliance History Table for this tracking item will display in this box, but can be overwritten as desired.
