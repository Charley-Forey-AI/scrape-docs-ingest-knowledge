# Installation Overview

Click the Admin button on the toolbar of any menu; the Installation list will appear. From the Installation list, select Company Installation. Note that this screen should have been previously completed by your System Administrator.
Ensure that the flag for this specific module has been checked. After the Company Installation screen has been completed, click OK to save the information and return to the Installation list.
