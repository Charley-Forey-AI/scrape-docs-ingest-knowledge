# Purchase Order Data Conversion

Below is a summary of the steps needed to complete
 installation. It is important that these steps be performed in the order listed.

1. Complete the Purchase OrderInstallation page.

1. Set the P/O processing date to the conversion date using Processing Date Change.

1. If the Accounts Payable module is not present, enter vendors using Vendor File Maintenance. If vendors have already been entered through Accounts Payable, proceed to step 5. Vendors may be entered through either Accounts Payable or Purchase Order; it is not necessary to do duplicate entry.

1. Print a listing of vendors and check for accuracy.

1. Enter all shipping terms (for example, Prepaid, COD, and so forth) using Shipping Terms Maintenance.

1. Print a listing of shipping terms and check for accuracy.

1. Enter ship via information (for example, Company truck, UPS ground, UPS RED, FedEx, and so forth) using Ship Via File Maintenance.

1. Print a listing of ship via codes and check for accuracy.

1. Enter special shipping instructions which are routinely used using Special Instructions Maintenance. One line of instruction may be tied to each purchase order.

1. Print a listing of special instruction codes and check for accuracy.

1. Enter remarks which are routinely used in P.O. Remarks Maintenance. One line of remarks may be tied to each purchase order.

1. Print a listing of standard P.O. remarks and review for accuracy.

1. If the Inventory Control module is not present, enter item codes using Purchase Order Item Code Maintenance. If items have already been entered through Inventory Control, proceed to step 15.

1. Print a listing of item codes and check for accuracy.

1. Enter any special discount or costs allowed by vendors using Vendor Special Cost Maintenance.

1. Print a listing of vendor special costs and check for accuracy.

1. Enter all purchase orders outstanding as-of the conversion date using Purchase Order Entry.

1. Set the P/O processing dates to the current date using Processing Date Change.
