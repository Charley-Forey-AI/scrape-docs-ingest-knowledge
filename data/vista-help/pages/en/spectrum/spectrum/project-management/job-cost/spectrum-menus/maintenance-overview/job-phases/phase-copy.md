# Phase Copy

The Phase Copy screen has two different uses.
Use the Phase Copy screen to:

- Copy phases to a new job

- Auto create wage codes

## Copy Phases to a New Job

You can copy the phase setup from an existing job to a new job without affecting any phases that you've already set up for the new job. Before you use this feature, the new job must be set up in the appropriate company.
At the same time that you're copying over phases to the new job, you can also copy estimates and phase setup. To set the projected costs, quantity, and hours on the new job equal to the current estimated cost and current estimated hours on an existing job:

1. In the Job Cost Installation > Properties tab, select the Set projected costs, quantity, and hours to current estimate when adding phases checkbox

1. In the Phase Copy window, select the Include estimates checkbox.

## Auto Create Wage Codes

While this doesn't create any new new phases, all combinations of the wage codes for the defined union code will be added to the Job Payroll Setup page automatically. This simplifies setting up the various wage and union code combinations needed for the new job.
To quickly add all wage code / union code combinations to the Job Payroll Setup page:

1. Enter the union codes that you want wage codes added by setting up all wage codes based on the union codes you've entered in Job Payroll Setup for the new job.

1. Enter that job number as both the old job and new job.
