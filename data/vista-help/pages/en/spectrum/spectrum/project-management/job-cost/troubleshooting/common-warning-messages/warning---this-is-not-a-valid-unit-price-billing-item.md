# Warning - This is not a Valid Unit Price Billing Item

This message indicates that one of the following setup errors has occurred:

- The billing item entered in Job Phases has a price type that is not set to unit price. Either change the price type to unit price or enter a new billing item.

- The billing item entered does not exit in Contracts for the default customer. The default customer is defined as the customer code entered on the job itself.

- There is not a customer code defined in Job - Other.

- The customer code entered on the job does not have a contract set up for it
 over in Accounts Receivable > Contracts.
