# Job Cost Installation - Security

Use this tab to select default security settings for the Job Cost module. These settings can be changed as needed at any time.
The Job Cost and Project Management modules offer a comprehensive job-based operator security feature. This feature can restrict access so that a given project manager, for example, can only look at the jobs that he or she is responsible for. Most inquiry screens allow for viewing information one job at a time. The inquiry screen will validate the job immediately upon entry. If the job does not match the operator's security, a warning message will display and access will be denied. For most reports, the security validation occurs while the report is compiling. If an operator runs a report for a job that he or she does not have security access to, then the job will not be included on the report and a warning message will display while the report is running.
