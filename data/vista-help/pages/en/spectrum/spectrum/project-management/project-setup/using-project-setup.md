# Using Project Setup

You can use Project Setup to link a
 non-Spectrum estimating program to the Spectrum Construction Software.

1. Create the transfer file and save it. Make sure to note the file name and path.

1. Make sure the [Project Setup Installation](/en/spectrum/spectrum/project-management/project-setup/project-setup-installation-overview/project-setup-installation) screen is set up to use your estimating package.

1. In the [Standard Phase File Maintenance](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/standard-phase-file-maintenance) screen, set up the standard phases.

1. Import the estimate info Spectrum Project
 Setup using the [Job Setup](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup) > Import screen.

1. Check / adjust the job in the screen.

1. Use the [Update Project Setup (Base Contract)](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/update-project-setup-base-contract) screen to move the job into the Job Cost module. If applicable, the job will be set up in Time + Materials. Contract and billing information for the job can be set up in Accounts Receivable, if applicable.

1. Use the [Create Historical Costs](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/create-historical-costs) screen to send information back to your estimating package.

1. Make a daily backup.
