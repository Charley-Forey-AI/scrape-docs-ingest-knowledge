# Standard Phase File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Import phase code
Enter the phase code, as it is set up
 in the estimating software. The import phase must be an exact match, including
 all hyphens, spaces, and more.
Phase code
The Spectrum phase is recorded without
 the extra characters.
Phase description
If this field is left blank, the phase
 description set up in the estimating software will be used. If a different
 phase description is desired, enter that description.
Unit of measure
If this field is left blank, the unit
 of measure specified in the estimating transfer file will be used. If a
 different unit of measure should be associated with this phase, enter that unit
 of measure.
Worker's compensation code
If this field is left blank, the
 worker's compensation code specified in the estimating transfer file will be
 used. If a different worker's compensation code should be associated with this
 phase, enter that code. A search window is also available to lookup worker's
 compensation codes if necessary.
