# Copy Standard Phases

This update will add (set up) phases in the Project Setup > Standard Phase File Maintenance based on information already recorded in the Job Cost > Standard Phase Category. If the phase is already set up in the Project Setup > Standard Phase File Maintenance, it will be updated accordingly.
Fields
Descriptions

Phase category
Enter the phase category, or
 double-click on this field to select from a list of available phase categories,
 or press Enter to
 leave the field blank and accept the company-wide default.
Phases to update
Decide if you want to update all of the
 Job Cost standard phases, or only existing Project Setup phases.
