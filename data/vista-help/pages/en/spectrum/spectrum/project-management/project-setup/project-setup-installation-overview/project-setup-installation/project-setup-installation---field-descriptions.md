After you choose an estimating software, additional entry fields display, and these vary based on the estimating software. Likewise, the fields in the Job Import screen vary based on the estimating software being used.
Properties
Default quantity cost typeEnter the Spectrum cost type code corresponding to the estimate quantity information. After you press Enter, the full cost type description displays from the Cost Type File Maintenance screen. Entry in this field is optional; it simply assigns a default that can be overridden.
User-defined cost typesEnter the cost type to be used during the Create Historical Costs update.
Estimating systemEnter the name of the estimating system used, for example, EXCEL SPREADSHEET.
