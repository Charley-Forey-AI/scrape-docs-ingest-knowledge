# Project Setup Installation

This Project Setup module is used to link a non-Spectrum estimating program to the Spectrum Construction Software.
When using the Project Setup Installation screen,
 you initially must specify the third-party software that your company is using for
 estimating. Depending on which estimating package you select from the Estimating method drop-down menu, the
 Spectrum system will complete the remainder of the installation screen's fields for further
 entry.
Standard Project Setup installation is utilized by those users who are starting from scratch. In other words, this is for users that aren't converting from a prior estimating system such as Accubid, Grantlin, Timberline, and so forth.
On the Site
 Map screen, click Admin  > Installation  >  Project Setup to access the Project Setup
 Installation screen.
