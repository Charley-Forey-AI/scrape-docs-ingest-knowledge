# Create Historical Costs - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Job
Enter the number of the Spectrum job to
 be transferred back into the estimating software, or press Enter to accept ALL jobs.
Priority quantity cost type
Enter the priority quantity cost type,
 or press Enter to
 accept the system default from the Project Setup Installation screen.
Other Cost Types button
Click this button to view additional
 cost types. The cost types shown may be changed in the Project Setup
 Installation and Job Cost Installation screens.
Labor cost type
Enter up to five labor cost types, or
 press F4 to select
 from a list of valid cost types. This field defaults based on the Labor Cost type field in Job
 Cost Installation.
Area code field
Enter the number (0-10) of the
 user-defined job field for the area code. This field is not applicable for the
 HCSS historical costs file.

- Select Import to set the phase code based on the phase code
