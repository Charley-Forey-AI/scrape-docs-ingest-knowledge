# Job Setup - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Job
If the estimate is not present, enter
 an imported Spectrum job number or double-click in this field to select from a
 list of valid job codes. Click on the job code to clear the current job and
 estimate and re-enter.To be valid, the Spectrum job must
 have first been imported into Job Import. Entry in this field is required.


Estimate type
Use the drop-down menu to select the
 estimate type: Base
 contract or Change
 request.Estimates for base contracts will
 continue to be recorded in Job Cost > Phases. Estimates for change requests will be updated to Accounts Receivable > Change Request Log / Entry.
If you select Change request, the
 Reference field
 displays. In the Reference field, enter an
 eight-digit alphanumeric reference for the change request estimate or select
 from a list of valid change requests.
Note: Only
 users with the proper security settings can add change requests on-the-fly
 by clicking the Maintenance button in the Change Requests For
 Selected Contract window. The Maintenance button does
 not display for users with insufficient security.

Reference
Enter an eight-digit alphanumeric
 reference for the change request estimate or select from a list of valid change
 requests. This field does not display if the estimate type is Base contract.
Total estimated cost
The total estimated dollar amount for
 the job displays. Estimates for base contracts are recorded in Job Cost > Phases. Estimates for change request are recorded in Accounts Receivable > Change Request Entry. Click the available drop-down to open the Estimated
 Cost Breakdown Inquiry window where you can see costs by cost
 type sequentially with their associated descriptions. No entry is allowed.
Switch button
Click this button to access another job
 estimate.
Buttons

Import
Click this button to open the
 appropriate Import starting screen based on the current Project
 Setup Installation screen settings. The system will perform a
 'quick save' of the current grid line before proceeding.
Subcontracts
Click this button to display the [Subcontract Setup](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/subcontract-setup) screen.
Billing Detail
Click this button to display the [Billing Detail Setup - Unit Price Format](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/billing-detail-setup---unit-price-format) screen.
Listing
Click this button to display the [Job Setup Report](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/job-setup-report) screen.
Update
Click this button to proceed to the
 appropriate [Update Project Setup (Base Contract)](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/update-project-setup-base-contract) starting screen, based on the current Project Setup
 Installation settings. The system will perform a 'quick save' of the current
 grid line before proceeding.
Click this button to display the
 More Project Setup Information window. Here you can
 see details for this job, including address, estimator, sell price, sales tax,
 contract amount, and total estimated cost. If you change the fields in the
 Job info section, the job will be updated immediately
 (for base estimate types). Note: Fields in
 the Financials section are for informational purposes
 only. Data entered here does not appear elsewhere in
 Spectrum.
