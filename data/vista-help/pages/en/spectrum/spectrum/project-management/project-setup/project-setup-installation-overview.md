# Project Setup Installation Overview

Click the Admin button on the toolbar of any menu; the Installation menu will appear.
 From the Installation list, select Company Installation. Note that the System Administrator
 should have previously completed this screen.
Ensure that the checkbox for this specific module has been selected.
After the Company Installation screen has been completed, click OK to save the information and return to the Installation menu.
