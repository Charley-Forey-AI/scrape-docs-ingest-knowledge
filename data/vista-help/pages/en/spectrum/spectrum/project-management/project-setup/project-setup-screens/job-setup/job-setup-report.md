# Job Setup Report

This screen is used to print the Phase Setup, Subcontract
 Setup or Billing Detail Setup information.
The phase report will print in the following order: assigned costs, assigned cost totals,
 unassigned costs, unassigned cost totals and grand totals.
Once you have selected a job number and report type, click Preview.
