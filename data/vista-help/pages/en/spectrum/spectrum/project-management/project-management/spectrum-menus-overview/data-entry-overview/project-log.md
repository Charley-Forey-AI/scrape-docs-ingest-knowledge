# Project Log

Use this screen to maintain accurate and up-to-date records
 of communication on a project.
The Project Log displays change requests, RFIs, daily logs, email correspondence, and other
 documents sent and received during a project.
Some columns can be edited. While some fields do not display in the Log Entry screen based on category setup, you will not be restricted from entering data into these fields in this grid. Data recorded in these fields will remain hidden in [Log Entry](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/log-entry), but will display on the [Print Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/category-maintenance/print-form).
Tips

- Group records by selected columns.

- Drag a row to associate it with another record.

- Double-click to open the record in Change Request Entry or Log
 Entry.

- If you have new/edit security, you can edit rows that are not
 Change Requests.

Import from Inbox
You can move email related to a project into the project log. Use subfolders in your email account to determine which email messages will be imported into Spectrum. Each folder will have an associated job and category.
For this feature to work, your email credentials must be set up and stored. Contact your administrator for further information.
