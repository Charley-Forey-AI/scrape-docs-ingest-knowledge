# Related Documents

The HTML Forms provide a simplified method for generating a
 PDF that includes not only the 'Document images' associated with the particular item, but also
 the HTML Forms for selected 'Related documents' (plus the 'Document images' attached to those
 other documents).
For example, the PDF for an RFI Form can now include its images, plus related MEMO's,
 CORRESPONDENCE and other related HTML documents and their respective images.
The Include Related Documents window displays after you click the Form button on the [Project Log](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/project-log) screen. You can select or deselect other HTML documents and their images to be part of the PDF form.
View example:
A user selects to print an RFI (#456 with four images), and chooses
 three related documents to be included using the Add Item button (two ISSUEs and a MEMO,
 each of which also has one or two images), the PDF would contain the following in this
 order:

1. RFI #456 form

1. RFI #456's first image

1. RFI #456's second image

1. RFI #456's third image

1. RFI #456's fourth image

1. ISSUE #82 form

1. ISSUE #82's image

1. ISSUE #109 form

1. ISSUE #109's first image

1. ISSUE #109's second image

1. MEMO #7 form

1. MEMO #7's image
