# My Training and Certifications

This app provides a list of all Human Resources Training/Certification records for the current operator. The app shows the 'expiration date' and description of the particular training or certification code. In the case where there is no 'expiration date', the app will substitute the 'required date' or end date', if available. The list displays in date sequence, with the latest date at the top (and undated training at the very bottom).
Note: This app will only display Human Resources information if the
 current operator is linked through the Contact Master Table to an employee code in the
 current company. This app is a snapshot app and does not allow you to drill in for further
 detail.
