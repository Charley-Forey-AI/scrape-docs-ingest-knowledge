# Insurance Code Maintenance - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field/Button
Description

Insurance code Description
Enter the new insurance code and a brief description.


Rate Table
Click this button to enter insurance group rates for the
 associated group code. These rates should be obtained from your insurance
 company.

Properties

Insurance type
Enter an insurance type for this insurance code.

Group number
Enter the insurance group number for the insurance code
 you are adding.

Scheme
Enter the insurance scheme for this current scheme. This
 field is optional.

Note
Enter any notes for the new insurance code.

Contact

Contact Address
Enter contact information and address for the new
 insurance code.
