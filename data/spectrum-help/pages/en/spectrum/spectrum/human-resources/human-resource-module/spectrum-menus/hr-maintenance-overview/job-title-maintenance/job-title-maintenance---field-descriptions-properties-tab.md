# Job Title Maintenance - Field Descriptions: Properties tab

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Job title
Enter the new job title.

Details

Job group
Enter the job group associated with the new job title.

EEO class
Enter the EEO classification code associated with the new job title.

Occupation
Enter the occupation code associated with the new job title.

Trade
Enter the trade code associated with the new job title.

Department
Enter the department code associated with the new job title.

Union
Enter the union code or pay group code (depending on your Payroll Installation screen settings) associated with the new job title. Up to 10 characters are allowed.

Wage code
Enter the wage code associated with the new job title. Up to 10 characters are allowed.

Score multipliers

Interview
Enter the interview score multiplier to use with the selected job title. This number is used to calculate and subsequently rate an employee's interview score for the job title he or she holds within the company.

Qualifications
Enter the qualifications score multiplier to use with the selected job title. This number is used to calculate and subsequently rate an employee's qualifications for the job title he or she holds within the company.

Posting
Click this button to display the Spectrum notepad, which can be used to enter any notes pertaining to the posting of the selected job title. Information entered here will be saved as a .txt file.

Summary

Created
The time and date on which the job title was created displays.

Operator
The operator who created job title was created displays.

Modified
The time and date on which the job title was last modified displays.

Operator
If you are adding a new job title, this field is not applicable.

Total openings
The total number of openings available for the selected job title displays.
Click the associated drill-down button to view the All Openings for Job Title
 window, which displays detailed job opening information for the selected job
 title.

Current openings
The total number of openings currently available for the selected job title displays. If multiple openings are selected in Job Openings, the number here will be calculated based on the remaining number of openings for open jobs. If multiple opening is not selected, this field will default to one (1).
Click the associated drill-down button to view the Current Openings for Job
 Title window, which displays detailed job opening information for the
 selected job title.

Employees
The total number of employees currently employed under the selected job title displays.
Click the associated drill-down button to view the Employees with Job Title
 window, which displays detailed job opening information for the selected job
 title.
