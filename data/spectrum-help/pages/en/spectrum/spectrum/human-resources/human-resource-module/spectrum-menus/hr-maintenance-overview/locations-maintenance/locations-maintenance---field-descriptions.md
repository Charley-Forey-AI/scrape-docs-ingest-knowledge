# Locations Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Code
Enter the location code in this field.

Description
Enter the location description in this field.

OSHA Establishment
If there is an OSHA establishment associated with this location, enter the code here. If this field is left blank, the Main company will serve as the default OSHA establishment.

Headquarters for EEO veteran reporting?
Select this checkbox if this location should appear on the Veteran Employment Report as the 'headquarters' location. This option will be hidden if the 'Report by location' checkbox in Human Resources Installation is not selected, or the 'Non-US payroll processing' checkbox in Payroll Installation is selected.

Location
Enter the address of the location in these fields.

Status
Select a status for the location:

- Active

- Not used
