# Archiving Safety/OSHA Incidents

Export images and store them for later retrieval.
This procedure applies only to Document Imaging users.To export your Safety/OSHA images and attached files:

1. Navigate to Document Imaging  >  Search Documents.

1. In selection criteria, type Employee in the cabinet name.

1. In the drawer field, type OSHA.

1. To search for a specific employee, enter their employee code in the Folder field. You can also use SuperSelect options to search for multiple employees, as needed.

1. Select OK to filter the results in the Search Documents screen.

1. Verify that these are the images you would like to export and select Listing .

1. Select Export and enter the folder name where you want to store the information.

The information is archived in your chosen location.
When needed, copy or distribute the information found in the folder name entered above.
