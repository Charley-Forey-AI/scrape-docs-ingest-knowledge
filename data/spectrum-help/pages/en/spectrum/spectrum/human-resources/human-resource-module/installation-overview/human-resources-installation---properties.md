# Human Resources Installation - Properties

Use this tab to define numbering schemes, select default wizards, and to set up options for user-defined fields.

Field/Button
Descriptions

Next item tracking code number
 Sequential item tracking numbers are automatically assigned to items as they are entered to the software. Enter the item tracking number that the software should assign the first item. When tracking company items that are issued to employees, the software will use this number to track items. This number can be up to 10 digits long.

Default security scheme
 The Human Resources default security scheme displays.

Next safety / OSHA log number
 Sequential safety/OSHA log numbers are automatically assigned to incidents as they are added to the software. Enter the next log number you want to use for safety/OSHA tracking. When creating safety/OSHA log files, the software will use this number to track incidents. This number can be up to 10 digits long.

Validate job titles?
 Select this checkbox if you want to require that a valid job title be assigned to each employee in the Payroll  >  Employees and Human Resources  >  Employees screens.
 If you do not want to require that a valid job title be assigned to each employee in Payroll (so that any job title can be selected in Employees, not just a valid job title), then leave this checkbox clear.
Note: If this is the first time that this screen is being
 completed for the current company and you select this checkbox, the software
 will offer the option to create job titles in the job title master file
 based upon existing job titles stored in the employee master file when
 exiting the screen.
