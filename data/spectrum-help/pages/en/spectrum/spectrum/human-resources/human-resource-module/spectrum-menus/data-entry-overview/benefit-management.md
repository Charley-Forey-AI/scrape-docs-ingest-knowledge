# Benefit Management

Use the Benefit Management screen to look up those
 employees who are currently set up for selected benefits and to manage recurring Payroll
 add-ons and deductions.
You can also use this screen to edit rates and limits. Any edited rates or limits will
 automatically update to Payroll upon exiting the screen. Employees set up for benefits for
 which they do not qualify for will be shown in red.
When this screen is first accessed, specify employee selection criteria in the Build Employee List for Benefit Management window.
If the add-on calculation method is set to one of the 'by time card' options, the 'Limit' columns will be disabled for these methods, with the exception of the 'Fixed Amount by Time Card' option.
