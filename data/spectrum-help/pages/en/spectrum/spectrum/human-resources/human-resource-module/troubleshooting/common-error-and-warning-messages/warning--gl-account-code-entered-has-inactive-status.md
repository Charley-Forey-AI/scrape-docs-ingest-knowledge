# Warning – G/L Account Code Entered Has Inactive Status

If an inactive General Ledger account code is entered, this
 message displays. Click OK to continue entering data, if
 desired.
To change the General Ledger account code's status, click GeneralLedger > Maintenance > MasterFile to open the G/L Master File Maintenance screen. In
 the Status section, select either Active or
 Inactive.
