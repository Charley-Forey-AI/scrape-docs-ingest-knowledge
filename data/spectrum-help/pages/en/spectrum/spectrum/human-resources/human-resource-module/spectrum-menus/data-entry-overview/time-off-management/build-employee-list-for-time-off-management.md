# Build Employee List for Time Off Management

Use the table below for reference when specifying employee
 selection criteria in this window.
After entering the benefit selection criteria, all employees that meet your selection
 criteria will display. Current hours will be calculated from the Payroll Time Off Bank Log
 Table. Any changes made in this screen will be committed once the operator clicks the
 Update button.
Field/Button
Description

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you
 can enter a cost center entity here in order to display entity-specific
 hours bank values. Leave this field blank to use the main company
 entity.

Employee Status code Department Supervisor Location
Enter the employee code, status code, department code, supervisor code, and location code to look up on this screen, or press Enter to include ALL.

Hire date
Specify the first and last hire dates to include in the update, or press Enter to accept the default.

Default balance revisions

Calculate vacation bank?
Select this checkbox if you are updating vacation hours. Enter the amount you are adding to the selected employee(s) vacation hours.

Calculate holiday bank?
Select this checkbox if you are updating holiday hours. Enter the amount you are adding to the selected employee(s) holiday hours.

Calculate sick bank?
Select this checkbox if you are updating sick leave hours. Enter the amount you are adding to the selected employee(s) sick leave hours.
