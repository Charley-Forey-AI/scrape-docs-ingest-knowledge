# Insurance Type Maintenance

Use the Insurance Type Maintenance screen to add and
 maintain insurance type codes, such as medical groups and dental groups, in the system. For
 example, you can set up insurance type codes to track various medical insurance providers
 offered through your company. These insurance types are then used to help define the insurance
 code subsets in Insurance Code Maintenance.
