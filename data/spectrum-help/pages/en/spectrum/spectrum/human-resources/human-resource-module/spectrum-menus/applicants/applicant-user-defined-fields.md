# Applicant User-Defined Fields

Use this screen to assign user-defined fields to the new applicant. All user-defined fields are optional. User-defined field data is stored in a single database table, but it is stored in individual records within the table.
Field/Button
Description

Field Status
Click this button to select a status for the selected user-defined field: Currently set or Unused.

Prompt
The user-defined field prompt will display in this field.

Current setting
The user-defined field's status will display in this field.
