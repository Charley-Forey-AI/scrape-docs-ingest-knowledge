# Item Requisition Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Tracking code Owner Issued to Location
Enter the specific tracking code, owner code, employee code, and location code, or press Enter to include ALL on the report.

Transaction types
Select which transaction types to include on the report: issued transactions and/or returned transactions.

Due date
Enter the item due dates the report is to begin and end with, or leave these fields blank to begin with the first date and end with the current Payroll processing date.

Date issued
Enter the item issue dates the report is to begin and end with, or leave these fields blank to begin with the first date and end with the current Payroll processing date.

Date returned
Enter the item return dates the report is to begin and end with, or leave these fields blank to begin with the first date and end with the current Payroll processing date.

Sort by
Click to select a sort option for the report. Options include sorting by Tracking code or by Employee code.
