# Incident User-Defined Fields Listing- Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Company
Enter the company code for which you want to print the listing of user-defined incidents, or press Enter to accept the default of ALL.

Include alpha fields
Select this checkbox to include incident user-defined alpha fields on the listing; otherwise, leave this checkbox clear.

Include date fields
Select this checkbox to include incident user-defined date fields on the listing; otherwise, leave this checkbox clear.

Include numeric fields
Select this checkbox to include incident user-defined numeric fields on the listing; otherwise, leave this checkbox clear.
