# Tracking Auto-Deposit Information

Learn how to set up auto deposit information for a selected
 employee.

1. On the Site Map screen, click Human Resources  >  Employees  >  Employees.

1. Enter the Employee code and select the Auto Deposit link on the Info Bar.

1. Select the Allocation method: Percent or Fixed amount.

1. In the Bank account code field, enter the checking or savings account to which funds should be deposited and press Enter.

1. In the Account type field, use the drop-down menu to select Checking or Savings.

1. In the ABA # field, enter the routing number for the employee's bank.

1. In the Status field, use the drop-down menu to select the employee's current employment status: (Y)es active, (N)o inactive, or Pre-notification.

1. In the Rate field, enter either the percent of the employee's net pay or the total dollar amount that should be deposited to his/her account depending on the Allocation method you selected in step 3.

1. Repeat steps 4-8 for each additional account to which funds should be automatically deposited for the employee.
