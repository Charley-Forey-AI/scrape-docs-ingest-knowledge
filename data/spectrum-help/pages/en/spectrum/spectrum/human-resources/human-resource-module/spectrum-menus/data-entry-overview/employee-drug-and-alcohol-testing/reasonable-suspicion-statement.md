# Reasonable Suspicion Statement

Use this window to enter information about the employee's behavior that led the company to find reasonable suspicion for drug or alcohol use. Some of the fields and the window's title will vary slightly depending on if you have selected to add a statement about suspicion of drug, alcohol, or other substance abuse.
Field/Button
Description

Observation
Enter the date, time, and location where the suspicious behavior was first noted by the witness(es).

Presence of
If the employee was found using alcohol at work, then select the Alcohol checkbox. Otherwise, if the test is for drug use, select Drugs or Drug Paraphernalia, depending on which item was found. (Select both if drugs and paraphernalia were found.)

Appearance
Select all those checkboxes that correspond with the employee's appearance at the time the suspicious behavior was witnessed.

Behavior
Select all those checkboxes that correspond with the employee's speech behavior and awareness at the time the suspicious behavior was witnessed.

Motor skills
Select all those checkboxes that correspond with the employee's motor skills at the time the suspicious behavior was witnessed.

Other observed actions or behavior
Enter any other suspicious actions or behaviors witnessed that were not noted above.

Witnessed by
Enter the name and title of the witness(es), and the date and time on which they witnessed the suspicious behavior.
