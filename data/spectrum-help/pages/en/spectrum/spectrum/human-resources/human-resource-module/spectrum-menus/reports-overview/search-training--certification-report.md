# Search Training / Certification Report

Use this screen to print a report of training/certification skill history. Use the Selections section of the screen to select who you want to appear on the report according to employee code, job title, department, date, and so forth. Use the Qualification monitor section either to view employees with all skills specified in Skill selection, or to view employees who do not currently qualify for the skills specified. In Skill selection, select up to four training/certification codes and corresponding levels that you want to appear on the report.
Note: If multi-company reporting is enabled for the current operator, the Company code field must be completed before proceeding with report selections. If multi-company reporting is not enabled for the current operator, the Company code field will not display.
