# Authorize Recurring Benefits - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, the entity specified in the Build Employee List to Authorize Benefits window displays here. No changes are allowed.

Employee
The employee code and name display in these columns.

Benefit code
The benefit code and description displays in these columns. Up to 10 characters are allowed.

Eligible?
The benefit eligibility status is displayed in this column.
Clicking the lookup icon to display the Eligibility Inquiry window. Use this window to look up detailed benefit information for a selected code.

Status
Select an eligibility status code.

Rate
Enter the amount or percentage amount for this employee add-on or deduction.

Monthly limit
Enter the monthly dollar amount limit for this add-on or deduction, if applicable.

Annual limit
Enter the annual dollar amount limit for this add-on or deduction, if applicable.

Calculation method
The calculation method type displays in this column.

Formula
The add-on/deduction formula code displays in this column.

Variable
Up to three variable codes display in these columns.

Related code
The code related to the benefit code displays in this column.

Department
The employee's department code and description displays in this column.

Employee status
The employee's status code and description displays in this column.
