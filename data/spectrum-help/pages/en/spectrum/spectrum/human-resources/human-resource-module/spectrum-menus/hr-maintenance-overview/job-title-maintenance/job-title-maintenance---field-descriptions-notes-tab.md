# Job Title Maintenance - Field Descriptions: Notes tab

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Topic
A list of note topics displays in this column. Note topics are set up in the Job Title Note Topics Maintenance screen.

Text
Enter and format any text for the selected note topic in this column.
