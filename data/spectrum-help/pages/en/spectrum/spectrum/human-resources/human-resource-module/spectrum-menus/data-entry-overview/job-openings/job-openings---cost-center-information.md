# Job Openings - Cost Center Information

If cost centers are set to 'Yes' in the current company, specify a Cost Center on this screen to use during the EEO Employment Report compilation when the user selects the 'Applicant' report type. When specified, the cost center will also restrict operator access to disallowed job openings. If no cost center is specified, the user will be able to access and edit all job openings in the list box.
