# Training Levels window

Use this window to assign skill levels to training/certification codes. When adding levels, you can assign values of 1-999; leading zeros should not be used because they will not be recognized elsewhere in the software.
Field/Button
Description

Code
The training/certification code and description selected for validation displays in the header.

Level
The validation levels assigned to the training/certification code displays in this column. When adding levels, you can assign values of 1-999; leading zeros should not be used because they will not be recognized elsewhere in the software.

Description
The description associated with the corresponding validation level displays in this column. To modify any of the validation levels that appear in the list, click the Edit, New, or Delete buttons.

Purge
Click this button to permanently purge a validation level from the selected training/certification code. This button must be used to delete a validation level that has been applied to an employee during routine Human Resources processing.
