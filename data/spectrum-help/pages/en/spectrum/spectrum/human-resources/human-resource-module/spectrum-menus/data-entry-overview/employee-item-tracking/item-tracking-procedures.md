# Item Tracking Procedures

Item Tracking Management provides a process to assign and track items currently assigned to employees. Before you begin tracking items using, there is some setup required.

- Set up item types using [Item Type Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/item-type-maintenance). Item types are codes that represent broad categories of company property you want to track. These are general classifications such as laptops, backhoes or lifts.

- Set up item owners using [Item Owner Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/item-owner-maintenance). The item owner is an optional entry during tracking and may be used to track the group or division that owns the item.
