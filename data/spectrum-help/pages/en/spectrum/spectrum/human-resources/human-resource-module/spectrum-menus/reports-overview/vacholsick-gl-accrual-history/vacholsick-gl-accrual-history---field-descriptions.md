# Vac/Hol/Sick G/L Accrual History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you can enter a cost center entity here in order to calculate entity-specific accrual details.

Employee Employee status Home department VHS accrual codes G/L account code
Enter the employee code, employee status code, home department code, vacation/holiday/sick code, and the G/L account code, or press Enter to include ALL on the report.

Accrual date
Enter the date for which the vacation, holiday, and sick balance should be determined, or press Enter to accept the current Payroll processing date default.
Note: This date must fall within the Payroll minimum/maximum date
 range.

Transaction date
Enter the accrual transaction date that will be assigned to the General Ledger (for the selected period), or press Enter to accept the current Payroll processing date default.
Note: This date must fall within the Payroll minimum/maximum date
 range.

Vacation accrual? Holiday accrual? Sick accrual?
Select the corresponding checkbox if you want to include vacation hours, holiday hours, and sick hours on the accrual.

Sort by
Select the sort option for the report.

- G/L account code – this format sorts and subtotals the report based on accrual type (vacation, holiday, and sick pay)

- Employee code – this format sorts and subtotals the report based on employee code
