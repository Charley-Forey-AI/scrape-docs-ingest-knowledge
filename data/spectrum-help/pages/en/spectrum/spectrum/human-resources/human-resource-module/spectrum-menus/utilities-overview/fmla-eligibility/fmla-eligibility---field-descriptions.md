# FMLA Eligibility - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you can enter a cost center entity here in order to display entity-specific FMLA eligibility.

Employee code Social security number Department Supervisor Location Status
Enter the employee code, social security number, department code, supervisor code, location code, and employee status code to include on the report, or press Enter to include ALL codes on the report.

Hire date
If you want to perform the FMLA update based on employment dates, enter the earliest and last hire dates to update, or press Enter to accept the current Payroll processing date.

Minimum days since hire date
Enter the minimum number of days since an employee's hire date that must pass before the employee is eligible for FMLA leave.

Calculate as of date
Enter the earliest and last dates to use when calculating FMLA eligibility, or press Enter to accept the current processing date.

Minimum hours in preceding 12 months
Enter a minimum number of hours worked in the preceding 12-month period

Period end date
If you entered a date in the Minimum hours in preceding 12 months field, enter the beginning and ending date of the 12-month period to calculate the hours worked.

Exclude pay types
If certain pay types should be excluded from this update to avoid reading those pay type records from the pay history file enter the pay type codes, or press Enter to accept the software default. Pay types that can be excluded include:

- Regular

- Overtime

- Doubletime

- Vacation

- Holiday

- Sick

- ER Equipment Regular

- EO Equipment Overtime

- ED Equipment Doubletime

- SR Special rate

- Unpaid time
