# Build Employee List for Rate Management - Field Descriptions

Use the table below for reference when specifying employee
 selection criteria in this window. After entering the benefit selection criteria, all
 employees that meet your selection criteria will display. If you wish to change the selection
 criteria, click the Build button to
 re-open the Build Employee List for Rate Management window. Any changes made in this screen
 will be committed once the operator clicks the Update button.
Field/Button
Description

Employee code
Employee name
Department
Supervisor
Job title
Occupation
Trade
Enter the employee code, employee name, department code, supervisor code, job title code, occupation code, and trade code you want to look up on this screen, or press Enter to include ALL.

Default rate revision settings

Effective date
Enter an effective date for rate revisions.

Calculate new pay rate for selected employees?
Select this checkbox to have Spectrum automatically calculate the pay rate change(s) for the selected employees, based on a percentage or fixed amount rate.

Pay types to include
Select the employee pay type you want Spectrum to display in the detail
 portion of the screen: Hourly or Other. If Hourly is selected, then only those employees who
 are hourly will appear in the detail portion of the screen. If Other is
 selected, then all employees who are salaried will display in the detail
 portion of the screen.

Other options
Select the corresponding checkbox to display only those employees with an Active status, Journeyman work class, Apprentice work class, or Trainee work class.
