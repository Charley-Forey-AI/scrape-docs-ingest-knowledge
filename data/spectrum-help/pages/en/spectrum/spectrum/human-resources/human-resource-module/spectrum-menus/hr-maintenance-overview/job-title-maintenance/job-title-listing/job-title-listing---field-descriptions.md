# Job Title Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Job title
Enter the job title code for which you want to view a listing, or press Enter to accept the software default of ALL.

Job group
Enter the job group code for which you want to view a listing, or press Enter to accept the software default of ALL.

Include qualification information?
Select this checkbox if you want to include qualification information on the listing.

Include pay level information?
Select this checkbox if you want to include pay level information on the listing.
