# Safety / OSHA Report

Use this screen to print a report of safety / OSHA
 incidents. The fields available on this screen enable you to define which safety / OSHA
 incidents you want to include on the report.
Note:
 If multi-company reporting is enabled for the current operator, the
 Company code field must be completed before proceeding with report selections. If
 multi-company reporting is not enabled for the current operator, the Company code field
 will not display.
