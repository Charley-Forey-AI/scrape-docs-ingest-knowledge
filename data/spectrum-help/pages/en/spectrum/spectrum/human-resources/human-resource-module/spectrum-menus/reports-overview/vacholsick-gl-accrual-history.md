# Vac/Hol/Sick G/L Accrual History

Use the Vac/Hol/Sick G/L Accrual History screen to print vacation, holiday, and sick benefit accrual information for an employee or group of selected employees. Transactions appearing on this report were created in the Vac/Hol/Sick G/L Accrual/Update and Reverse Vac/Hol/Sick G/L Accrual screens.
Note: If multi-company reporting is enabled for the current
 operator, the Company code field must be completed before proceeding with report
 selections. If multi-company reporting is not enabled for the current operator, the Company
 code field will not display.
