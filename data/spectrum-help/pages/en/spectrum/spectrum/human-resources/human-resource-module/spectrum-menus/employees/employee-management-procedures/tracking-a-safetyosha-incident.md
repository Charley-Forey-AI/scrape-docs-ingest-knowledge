# Tracking a Safety/OSHA Incident

Learn how to track a safety/OSHA incident.

1. On the Site Map screen, select Human Resources  >  Employees  >  Safety / OSHA.

1. Enter an Employee code and select the Edit button.

1. Complete all the necessary information on the Properties, Description, Treatment, and Job Info tabs.

1. Click OK until you return to the Site Map screen.
