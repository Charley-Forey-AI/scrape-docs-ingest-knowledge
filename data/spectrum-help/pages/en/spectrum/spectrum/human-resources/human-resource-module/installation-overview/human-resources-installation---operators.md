# Human Resources Installation - Operators

Use this tab to assign security schemes to operators. When an operator is assigned a security scheme, this setting will take precedence above any other company-defined security. If an operator is not assigned a security scheme, then the company default scheme will be used instead.
Important: If a default company scheme doesn't exist and the
 operator is not assigned a Human Resources security scheme, then the operator will have
 access to ALL of the employees in both the Employees and Applicants screens. To protect pay
 rate information, it is important to make sure security settings are established for all
 operators.
Field/Button
Description

Operator
Enter the operator code you want to add.

Security scheme
Enter the security scheme you want to apply to the selected operator.

Allow multiple company reporting for this operator?
Select this checkbox if the selected operator will be set up for multi-company reporting.
