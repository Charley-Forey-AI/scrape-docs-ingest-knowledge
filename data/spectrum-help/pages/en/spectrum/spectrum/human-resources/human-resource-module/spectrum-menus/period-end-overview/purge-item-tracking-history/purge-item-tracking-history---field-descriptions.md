# Purge Item Tracking History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Tracking item code
Enter the specific tracking code to purge, or press Enter to accept the software default and include ALL of the tracking codes from the software in the purge.

Remove old items

Purge 'Retired' items and all associated history?
Select this checkbox to purge all retired items and associated history. When this checkbox is selected, the rest of the fields on this screen will be disabled.

Remove history

Transaction date
Enter the first and last transaction dates to include in the purge. To begin with the first date and end with the current Payroll processing date, leave these fields blank by pressing Enter.

Save last history record?
Select this checkbox if you want to save the last history record for the selected tracking item(s) so that it is not included in the purge. For example, the name of the most recent employee to whom the item is issued will be saved if this checkbox is selected. Likewise, if the item has a status of 'Retired' at the time of the purge this information will be stored in the software. Otherwise, if you do not want to retain a last history record for the tracking item(s), leave this checkbox clear.

Include 'Available' items?
Select this checkbox to include items with a status code of 'Available' in the purge.

Include 'Issued' items?
Select this checkbox to include items with a status code of 'Issued' in the purge.

Include 'Retired' items?
Select this checkbox to include items with a status code of 'Retired' in the purge.
