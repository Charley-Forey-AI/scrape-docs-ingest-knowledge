# Search Training / Certification Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee code Job title Department Location Supervisor
Enter the employee code, job title, department code, location code, and supervisor code to include on the report, or press Enter to accept the software default of ALL.

As of date
The As of date evaluates whether an employee has a particular skill or certification based on date. Enter the date within the fiscal calendar to determine whether the employee qualifies for skills and certifications with required start dates, end dates, or expiration dates. An employee is qualified for a skill based on how these dates are set up in the [Training / Certification Code Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/training--certification-code-maintenance) screen:

- If the skill code expiration date is set to yes, then the employee's expiration date must be equal to or later than the As of date to be qualified.

- If the skill code start date is set to yes, then the employee's start date must be equal to or earlier than the As of date to be qualified.

- If the skill code end date is set to yes, then the employee's end date must be equal to or later than the As of date to be qualified.

Example: If a skill requires a start date of 4/23/10 and an end date of 10/31/10, and the As of date is 6/23/10, then the employee is qualified for the skill or certification. However, if the skill's end date is 5/23/10, then the employee is not qualified for the skill or certification because the end date is earlier than the As of date.

Active status only?
Select this checkbox to include only those employees with an Active status on the report.

Qualification monitor
Select whether to view all employees with the selected skills specified below, or employees that currently lack the selected skills.

Skill selection
Enter up to four training/certification codes you want to filter for the report. For each Skill field that you enter information, enter the corresponding minimum level for this skill to help narrow the information that will appear on the report. If you do not want to narrow the information that will appear on your report by training/certification skill level, leave these fields blank.
