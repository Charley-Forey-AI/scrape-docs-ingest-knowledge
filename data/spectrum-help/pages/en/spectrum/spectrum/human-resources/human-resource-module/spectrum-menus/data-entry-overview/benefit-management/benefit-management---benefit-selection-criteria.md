# Benefit Management - Benefit Selection Criteria

After entering the benefit selection criteria, all employees that meet your selection criteria and with a status of Pending, review again next time will display. Additionally, those codes that match codes specified in the Recurring Benefits File Maintenance screen will display, based on the same eligibility requirements associated with the benefit code(s) selected. Use the drop-down menu in the Status column to make changes to the employee benefit status information. The selected Eligibility as of date will display in the screen header, along with the Entity (if applicable).
Field/Button
Description

Entity
This field only displays if cost centers are set to Yes or Pending in the current company and if the 'Entity tracking?' checkbox is selected in Enterprise Installation.
Leave this field blank if you want to use the main company entity, or select an entity from the available drop-down list to find eligible benefits by entity.

Employee code Department Benefit code Benefit status
Enter the employee code, payroll department code, benefit code (up to 10 characters), and benefit status code you want to look up on this screen, or press Enter to include ALL.

Eligibility as of
Enter the eligibility date for benefits, or press Enter to accept the current Payroll processing date default.

Active status only?
Select this checkbox to display only those employees with an Active status for benefit adjustments.

Automatically adjust rate and limits for selected benefit code?
Select this checkbox to automatically adjust rate and time limits for a selected benefit code. This checkbox is available only if a specific benefit code is designated in the Benefit code field.

Select by salary range?
Select this checkbox to select employees based on salary range.

Date
Enter from and through dates to filter the salary range. The first day of the current year and the current payroll processing date will default , unless an earlier date than the first day of this year was entered in the From date field.

Earnings
Enter the minimum and maximum amount of earnings during the specified date range to include when selecting employees for display on the detail.

Select by employee age?
Select this checkbox to select employees based on age range.

As of date
Enter an as of date to compute employee age. The current payroll processing date defaults.

Age
Enter the minimum and maximum employee ages to include on the detail, computed using the date entered in the As of date field above.
