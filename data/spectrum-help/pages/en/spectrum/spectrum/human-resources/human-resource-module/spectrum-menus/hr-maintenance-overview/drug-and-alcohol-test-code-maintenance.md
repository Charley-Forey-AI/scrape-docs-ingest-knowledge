# Drug and Alcohol Test Code Maintenance

Use the Drug and Alcohol Test Code Maintenance screen to
 add and drug and alcohol test codes to the Spectrum system.
Drug and alcohol test codes are used to track drug and alcohol test types that are
 administered to employees for any reason (regardless of whether the testing is performed
 because of suspicious behavior or as a part of routine testing in accordance with company
 drug testing policy). Additionally, codes set up here are used to track drug and alcohol
 testing for all commercial vehicle drivers and other workers who are required to perform
 drug and alcohol testing in order to meet the Department of Transportation's testing
 requirements.
Note: You cannot edit the actual test code, only its description and type. To change an actual test code, select the incorrect test code and click the Delete button. Once the incorrect test code has been deleted, click the New button to add the correct test code to the software.
