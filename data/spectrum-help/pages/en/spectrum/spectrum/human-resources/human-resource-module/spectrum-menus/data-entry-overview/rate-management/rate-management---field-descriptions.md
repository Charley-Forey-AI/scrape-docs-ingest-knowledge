# Rate Management - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Total employee rate changes
The number of rows with a non-zero amount change will display in this field.

Old annual pay
The annual hourly pay totals display in this field.

Net change
The net change totals display in this field. The percent change displays to the right of this field.

New annual pay
The new annual pay totals display in this field.

Detail

Employee
The selected employee code and name display in these columns.

Old hourly/annual rate
If the 'Hourly' pay types option was selected in the Build Employee List for
 Rate Management window, the old hourly rate will display in this field.
If the 'Other' pay types option was selected in the Build Employee List for
 Rate Management window, the old annual rate will display in this field.

% change
Enter the percent rate change in this field.

Amount change
Enter the amount change in this field.

Net pay rate
Enter the net pay rate for the selected employee.

Effective date
The effective date for the rate change will default from the Build Employee
 List for Rate Management window, but can be changed as needed.

Pay type
The selected employee's assigned pay type will display in this field.

Frequency
The selected employees assigned frequency description will display in this field.

Department
The selected employee's department code and description display in this field.

Supervisor
The selected employee's supervisor code and description display in this field.

Job title
The selected employee's job title description displays in this field.

Occupation
The selected employee's occupation description displays in this field.

Trade
The selected employee's trade description displays in this field.

Status
The selected employee's status code and description display in this field.
