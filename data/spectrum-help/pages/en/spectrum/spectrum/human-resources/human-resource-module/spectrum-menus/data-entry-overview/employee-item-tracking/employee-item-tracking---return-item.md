# Employee Item Tracking - Return Item

Use this window to document the return of a company-issued item by an employee. Once returned, the item's status will be changed to 'Available'.
Field/Button
Description

Item
The item name displays.

Employee
The employee to whom the item is issued displays.

Location
Enter the location to which the item is being returned.

Return date
Enter the date on which the item was returned.

Due date
The date the item was due displays.

Notes
Enter any notes pertaining to the item being returned, for example, if the
 item has been damaged in any way.
