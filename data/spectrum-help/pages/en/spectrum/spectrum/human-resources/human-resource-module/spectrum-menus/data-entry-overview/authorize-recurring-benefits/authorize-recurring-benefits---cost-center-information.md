# Authorize Recurring Benefits - Cost Center Information

If cost center entity tracking is enabled in Enterprise Installation, the Build Employee List to Authorize Benefits window will include an 'Entity' selection field. Once specified, the software will proceed to build the list for that entity, and employees authorized for the entity, only.
