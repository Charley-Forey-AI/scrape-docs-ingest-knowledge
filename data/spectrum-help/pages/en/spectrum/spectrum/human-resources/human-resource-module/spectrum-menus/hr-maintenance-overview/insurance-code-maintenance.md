# Insurance Code Maintenance

Use the Insurance Code Maintenance screen to add and maintain insurance codes in Spectrum for selected insurance companies. For example, you can set up codes to track medical, dental, life, supplemental, and other insurance codes provided through your company. Insurance codes defined here are subsets of the insurance types, which are defined in Insurance Type Maintenance.
