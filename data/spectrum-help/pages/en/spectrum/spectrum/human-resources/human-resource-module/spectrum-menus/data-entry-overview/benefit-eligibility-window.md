# Benefit Eligibility window

This window shows detailed information that explains how benefit eligibility is configured in Spectrum. Entry is not allowed in this window. Changes can be made to the information that appears in this window by accessing the Recurring Benefits File Maintenance screen on the Maintenance menu.
If there are any matching codes associated with the selected benefit, the Match Code For window will first appear, prompting you to select the specific payroll deduction code for which you wish to view the eligibility information. When this window displays, click to select the code you want to view, and then click the Eligibility button. The Benefit Eligibility window will then appear.
Field/Button
Description

Benefit code
The benefit code displays based on the code selected on the main screen.

Code type
The benefit code type displays.

Benefit start date
The benefit start date displays.

Matching code
The matching code, if any was selected, displays.

Home department
The code of the home department displays.

Employee status
The employee status code associated with the selected benefit displays.

Pay type
The pay type associated with the selected benefit code displays.

Casual
This field indicates whether or not this employee is casual (Yes or No).

Minimum standard hours
The minimum number of employee hours allowed for this benefit code to take effect displays.

Minimum days since hire date
The minimum number of days since hire required for this benefit code to take effect displays.

Minimum hours since hire date
The minimum number of hours worked since hire date required for this code to take effect displays.

User-defined elgibility
Any user-defined eligibility information displays.
