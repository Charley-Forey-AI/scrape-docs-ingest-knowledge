# Employee Civil/Legal/Other Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee code
Enter the employee code, or press Enter to accept the default.

Department
Enter the department code, or press Enter to accept the default.

Activity code
Enter the civil/legal/other activity code, or press Enter to accept the default.

Date
Enter the beginning and ending dates for the civil/legal/other activities you want to appear on this report, or press Enter to accept the current Payroll processing date defaults.

Notes
Type SuperSelect options to select specific notes that may have been entered into the Notes field in the Employee Civil/Legal/Other Activities Entry screen. Entry in this field is optional.

Sort by
Select how you want the report sorted, either by Employee code or Date.
