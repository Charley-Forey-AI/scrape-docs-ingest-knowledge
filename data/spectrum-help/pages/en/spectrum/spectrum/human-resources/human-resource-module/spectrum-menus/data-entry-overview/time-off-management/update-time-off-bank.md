# Update Time Off Bank

Use the Update Time Off Bank screen to print a listing of global hour updates made in the Time Off Management screen. Select a 'Sort by' option and enter a 'Reason' for the update to appear on the report.
If cost centers and entity tracking are enabled in Enterprise Installation, this report will print the company name or entity name, depending on the data entered in the Build Employee List for Time Off Management window.
