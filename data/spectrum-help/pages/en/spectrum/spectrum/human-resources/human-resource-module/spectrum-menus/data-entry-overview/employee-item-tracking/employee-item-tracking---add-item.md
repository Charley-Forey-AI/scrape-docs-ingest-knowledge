# Employee Item Tracking - Add Item

Use this window to add items that can be issued to employees.
Field/Button
Description

Tracking item
Enter an item tracking code for the item you are adding, or press Enter to accept the default.
Note: Spectrum assigns default sequential numbers to tracking
 items as they are added to Spectrum in this field; if you do not want to use
 this numbering system, simply enter the tracking code you want to use
 instead.

Description
The description defaults based on the item type you have selected. Press Enter to accept the default, or enter a new description if necessary.

Details

Item type
Enter the item type in this field. Item types are defined in the Item Type
 Maintenance screen.

Owner
Enter the owner of the item in this field. Owners are defined in the Item
 Owner Maintenance screen.

Location
Enter the location of the item in this field.

Purchase information

Acquired date
Enter the date on which this item was purchased.

Cost
Enter the cost of the item. If you do not know this information, this field can be left blank.

Vendor
Enter the vendor for the item. If you do not know this information, this field can be left blank.

Notes
Enter any notes pertaining to the item. If you don't have any notes to add, leave this field blank.

Current standing

Status
The item's status displays in this field: Available,
 Issued, or Retired. If the status is
 Issued, this field will be display only.

Due date
The date the item is due to be returned displays. This field displays only if
 the Status field is set to
 Issued.

Issued to
The person who was issued the item displays. This field displays only if the
 Status field is set to
 Issued.

Manufacturer

Make
Enter the make, model, serial #, and product # of the item. If you do not know this information, these fields can be left blank.

Model

Serial #

Product #

Warranty

Parts
Enter the warranty for the item's parts, if one exists, and any labor associated with the item. If you do not know this information, these fields can be left blank.

Labor

History

Use this tab to view the tracking item's history details, including all requisitions and returns.
Note: If the payroll year-to-date Period End Clear is run and
 includes an employee who has an item checked out, the item's status will be
 changed to Deactivated. Additionally, the
 Tracking
 History > History tab for the item will show the issuance of the item to the
 deleted employee as the item's final transaction.
