# Human Resources Installation - Security

 With the Human Resources
 Installation screen, you can set up a Human Resources security scheme that
 allows an operator to access Employees, but restricts them from viewing any pay rate
 information.
It is important to take the time and design your security scheme prior to using the Human
 Resources module. Traditional Spectrum security enables you to lock a specified user out of
 a particular screen altogether. As there are many screens that use tabs (for example, the
 Human Resources Installation screen you are currently working
 in), security schemes must be developed using employee-based security.
Field/Button
Description

Scheme
 Enter the name for the scheme you are adding (up to 10 characters in length),
 for example, "LOW".
Note: If you are editing a scheme, the scheme name that appears
 here cannot be modified.

Description
 A list of existing security scheme descriptions displays in the list box. To edit a description, click to highlight the scheme and then click the Edit button.

Employee/applicant code
 If you want to implement employee-based security, enter the employee/applicant code for whom you are establishing Human Resources security, or press Enter to accept the software default of ALL.

Supervisor
 If you want to implement employee-based security, enter the supervisor code of the employee for whom you are establishing Human Resources security, or press Enter to accept the software default of ALL.

Department
If you want to implement employee-based security, enter the department code of the employee for whom you are establishing Human Resources security, or press Enter to accept the software default of ALL.

Location
 If you want to implement employee-based security, enter the location code of the employee for whom you are establishing Human Resources security, or press Enter to accept the software default of ALL.

Alpha code
 If you want to implement employee-based security, enter the alpha reference code of the employee for whom you are establishing Human Resources security, or press Enter to accept the software default of ALL.
