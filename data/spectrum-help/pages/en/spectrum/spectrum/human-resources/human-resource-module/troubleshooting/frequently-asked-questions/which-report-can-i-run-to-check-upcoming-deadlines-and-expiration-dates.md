# Which report can I run to check upcoming deadlines and expiration dates?

Spectrum provides a convenient Tickler
 Report that is used to track deadlines and upcoming issues and requirements.
 For example, you can run this report to see if an employee's CPR certification has expired, or
 when the next round of employee reviews is due.
