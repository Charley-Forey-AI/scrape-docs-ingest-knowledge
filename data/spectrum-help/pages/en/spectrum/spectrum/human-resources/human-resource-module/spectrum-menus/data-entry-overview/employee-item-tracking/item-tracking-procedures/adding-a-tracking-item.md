# Adding a Tracking Item

Learn how to add a company-owned item to Human Resources.
Before company-owned items can be tracked, you must first
 make sure item type codes and item owner codes are established using the Human Resources Maintenance
 menu.

1. On the Site Map screen, click Human Resources > Data Entry > Item Trackingto display the Employee Item Tracking
 screen.

1. Click the New button to display
 the New Tracking Item window.

1. At the Tracking item field, the
 software will automatically assign a tracking code number to the item you are adding.
 Press Enter to accept this code, or enter a different tracking
 code if necessary.

1. At the Item type field, enter
 the item type code for the tracking item, or press F4 to
 select from a list of available item type codes.Note: Item type codes are established in [Item Type Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/item-type-maintenance).

1. At the Description field, the
 item type description defaults based on the item type you entered. Press
 Enter to accept this default or enter a new description if
 necessary.

1. At the Owner field, enter the
 item owner code for the tracking item, or press F4 to select
 from a list of available item owner codes.Note: Item owner codes are established in [Item Owner Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/item-owner-maintenance).

1. At the Location field, the employee's location defaults. If the employee
 doesn't have a defined location, then the item's location will be used instead. Press
 Enter to accept the
 default or enter a different location.This is an optional field.

1. At the Notes field, enter any
 special notes pertaining to the item you are adding.

1. At the Purchase information, Manufacturer, and Warranty sections of the window, add the item's product
 information.
 Any of the fields contained in these sections can be left
 blank.


1. Once you have completed these fields, click OK.Once you have added a tracking item to Human Resources, you can
 then issue the item to an employee, which the software will track.
