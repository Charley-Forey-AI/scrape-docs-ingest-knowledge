# Safety / OSHA Report - Field descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Incident number
Enter the incident number you want to report, or press Enter to include ALL incidents on the report.

Employee code
Enter the employee code to print, or press Enter to include ALL employees.

Incident date
Enter the incident dates to begin and end this report with, or press Enter to leave this field blank and display report results from the beginning and end with the current Payroll processing date.

Supervisor Department Location Job Claim type Claim status Injury type Body part Injury cause
Enter the supervisor code, department code, location code, job code, claim type code, claim status code, injury type code, body part code, and injury cause code you want to include on the report, or press Enter to include ALL available codes on the report.

Active status only?
Select this checkbox to display only those employees with an Active employment status on the report.

Work-Related
To include work-related safety/OSHA incidents on the report, accept the default of All, or select Work-related only or Non-work-related only.

OSHA Eligible
Accept the default of All, or select OSHA Eligible only or Non-eligible only incidents to include on the report.

FMLA
Accept the default of All,or select FMLA only or Non-FMLA only incidents to include on the report.
