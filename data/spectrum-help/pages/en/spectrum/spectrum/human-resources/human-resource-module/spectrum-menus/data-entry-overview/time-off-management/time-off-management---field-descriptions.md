# Time Off Management - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee
The selected employee code and name display in these columns.
If the cost center feature is enabled in the Enterprise
 Installation screen, the employee list displays only
 employees associated with cost centers that the operator is authorized to
 access.

Hire date
The employee's hire date displays in this column.

Current vacation hours
The employee's current vacation balance displays in this column.
If the Set to zero
 anniversary? checkbox was selected on the Payroll > Time Off Code Maintenance screen, the vacation balance will be calculated for the
 current year only based on the employee's anniversary date.

New vacation balance
Enter the new vacation balance for the employee in this field.

Current holiday hours
The employee's current holiday balance displays in this column.
If the Set to zero
 anniversary? checkbox was selected on the Payroll > Time Off Code Maintenance screen, the holiday balance will be calculated for the
 current year only based on the employee's anniversary date.

New holiday balance
Enter the new holiday balance for the employee in this field.

Current sick hours
The employee's current sick leave balance displays in this column.
If the Set to zero
 anniversary? checkbox was selected on the Payroll > Time Off Code Maintenance screen, the sick balance will be calculated for the current
 year only based on the employee's anniversary date.

New sick balance
Enter the new sick leave balance for the employee in this field.

Department
The employee's department code and description display in this field.

Supervisor
The employee's supervisor code and description display in this field.

Location
The employee's location code and description display in this field.

Status
The employee's status code and description display in this field.
