# Employee Dependents - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

Employee code
Enter an employee code.

Type
Select the type of depending you are adding: Spouse, Child, or
 Other.

Name
Enter the name of the employee's dependent.

Gender
Select the dependent's gender: Female or Male.

Date of birth
Enter the birthday of the employee's dependent. If you
 type this date as six digits (DD/MM/YY), Spectrum will default the year to a
 past date. For example, if you type "02/02/50," the year will default to
 1950 instead of 2050. If you enter a birth date earlier or later than the
 expected range, a warning message will display.

Social security #
Enter the dependent's social security number.

Note
Enter any other pertinent information about the dependent
 being added.
