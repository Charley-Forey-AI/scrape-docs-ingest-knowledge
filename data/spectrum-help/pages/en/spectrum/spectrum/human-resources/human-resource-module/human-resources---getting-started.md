# Human Resources - Getting Started

Access Human Resources options from the Site Map or the Employee Info Bar. This module is integrated with Spectrum Document Imaging, General Ledger, and Payroll.
To set up Human Resources, follow these steps:

- Configure default settings ([Human Resources Installation](#ID-0000009d--en__ID-000000b7)).

- As needed, create codes to track benefits and employee activities ([Human Resources Maintenance](#ID-0000009d--en__ID-000000bd)).

- Manage and track benefits and employee activities ([Human Resources Data Entry](#ID-0000009d--en__ID-000000de)).

Refer to the following sections for details:

## Human Resources Installation

Use the System Administration > Installation > Human Resources menu to set up the Human Resources module. Options include general properties, reporting (FMLA, OSHA, and EEO), group-level security, operator-level security, and user-defined fields.

- [Installation Overview](/en/spectrum/spectrum/human-resources/human-resource-module/installation-overview)

## Human Resources Maintenance

Use the Human Resources > Maintenance menu as needed to create codes used to track benefits, activities, and classifications relevant to Human Resources. For example, you can set up codes for recurring benefits, work absences, OSHA or FMLA incidents, and training and certification, to name a few. Most options require that you enter a code and description to complete setup. Options that require additional steps have their own procedure listed below. For a list of all options available from the Maintenance menu, see [HR Maintenance Overview](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview).

- [Set Up Tracking Codes](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/set-up-tracking-codes) (options that require only a code and description).

- Set up [Insurance Scheme Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/insurance-scheme-maintenance) and [Insurance Code Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/insurance-code-maintenance).

- See [Setting Up a Job Group](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/setting-up-a-job-group) and [Setting Up a Job Title](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/setting-up-a-job-title).

- [Locations Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/locations-maintenance)

- [OSHA Establishment Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/osha-establishment-maintenance)

- [Recurring Benefit Code Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/recurring-benefit-code-maintenance)

- [Training / Certification Code Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/training--certification-code-maintenance)

## Human Resources Data Entry

Use the Human Resources > Data Entry menu to manage and track the benefits, activities, and classifications provided to employees. For example, track and authorize 401(k) benefits; make global pay rate changes; or manage job openings at your company.
