# Employee Drug and Alcohol Testing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee
The selected employee code and name will display in this field. When adding a new record, enter a new employee code.
If the cost center feature is enabled in the Enterprise Installation screen, the employee list displays only employees associated with cost centers that the operator is authorized to access.

Test code
When adding a new record, enter the test code in this field.

Status
The status of the test displays.

Reason
Select a reason for the drug and alcohol test from the drop-down menu: Follow-up, Other, Post-accident, Pre-employment, Random, or Reasonable suspicion.
If Reasonable suspicion is selected, click the Statement button to open the [Reasonable Suspicion Statement](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/data-entry-overview/employee-drug-and-alcohol-testing/reasonable-suspicion-statement) window to enter information about the employee's behavior that led the company to find reasonable suspicion for drug or alcohol use.

Entry date
Today's date displays. Enter a new date, if necessary.

Required by
Enter the date by which the test is required.

Notification
Enter the date on which the employee was notified of the test.

Scheduled date
Enter the date for which the test is scheduled.

Test date
Enter the date on which the test was actually taken by the employee.

Test time
Enter the time at which the employee took the test.

Facility
Enter the facility where the test was administered.

Administered by
Enter the name of the person who administered the test.

Result
Enter the result of the drug or alcohol test: Inconclusive, Negative, Positive, or Other.

Follow up?
Specify whether follow-up testing is required.

Follow-up deadline
If follow-up testing is required, enter the follow-up deadline date.

Notes
Enter any notes pertaining to the selected test as needed.
