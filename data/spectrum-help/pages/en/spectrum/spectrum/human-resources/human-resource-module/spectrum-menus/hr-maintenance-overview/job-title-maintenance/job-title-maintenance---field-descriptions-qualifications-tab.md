# Job Title Maintenance - Field Descriptions: Qualifications tab

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Training
Enter the training code you are applying to the job title.

Description
The description displays based on the training/certification code you have selected to apply to the job title.

Is the training/certification required?
Select this checkbox if the training/certification is required for the selected job.

Minimum level
Enter the minimum level of training/certification required to perform the selected job title. This field is available only if the Is the training/certification checkbox is selected.

Relevance
Enter a numeric value representing the training/certification's relevance to the selected job. This field is available for all training/certification skills, regardless of whether or not they are flagged as required. This allows you to track skill relevance points for applicants who meet the training/certification requirements, even if their skill score is "0".
