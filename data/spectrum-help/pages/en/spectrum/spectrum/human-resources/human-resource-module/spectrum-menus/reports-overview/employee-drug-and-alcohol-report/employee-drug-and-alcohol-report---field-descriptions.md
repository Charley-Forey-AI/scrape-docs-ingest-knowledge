# Employee Drug and Alcohol Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee code Department Test code Report status Reason
Enter the employee code, department code, test code, report status code, and reason codes to print, or press Enter to include ALL on the report.

Required by date
Enter the earliest and last required by dates to include on the report. All required testing performed between the dates entered in these fields will appear on the report. To begin with the first date and end with the current Payroll processing date, leave these fields blank.

Scheduled date
Enter the beginning and latest scheduled testing dates to include on the report. All scheduled testing occurring between the dates entered in these fields will appear on the report. To begin with the first date and end with the current Payroll processing date, leave these fields blank.

Test date
Enter the earliest and last drug and alcohol test dates to include on the report. All testing performed between the dates entered in these fields will appear on the report. To begin with the first date and end with the current Payroll processing date, leave these fields blank.

Follow-up date
Enter the earliest and last required follow-up test dates to include on the report. All follow-up testing performed between the dates entered in these fields will appear on the report. To begin with the first date and end with the current Payroll processing date, leave these fields blank.

Include suspicion report?
Select this checkbox if you want to include the suspicion statement, if one is available, on the report.

Active status only?
Select this checkbox to include only those employees with an Active status on the report.
