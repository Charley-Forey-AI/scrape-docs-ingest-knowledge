# Employee Civil/Legal/Other - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/ButtonDescription
CodeEnter an employee or activity code in this field.
Total hoursThe sum of values currently displayed in the Hours column will display in this field.
Total costThe sum of values currently displayed in the Total cost column will display in this field.
Detail
EmployeeA list of employee codes displays in this column.

- When adding a new activity, the current employee code will default.

- When this screen is accessed in context to an employee, the current employee code displays in every row.

If the cost center feature is enabled in the Enterprise Installation screen,
 the employee list displays only employees associated with cost centers that
 the operator is authorized to access.
NameThe selected employee name displays in this column.
DateSpecify an activity date.
Activity codeSelect the civil/legal/other activity code.

- When adding a new activity, the current activity code will default.

- When this screen is accessed in context to an activity code, the current activity code displays in every row.

DescriptionThe civil/legal/other activity code will display in this field.
HoursEnter the total number of hours for the activity in this field.
Total costEnter the total cost for the activity in this field.
NoteEnter any notes for the activity in this field.
