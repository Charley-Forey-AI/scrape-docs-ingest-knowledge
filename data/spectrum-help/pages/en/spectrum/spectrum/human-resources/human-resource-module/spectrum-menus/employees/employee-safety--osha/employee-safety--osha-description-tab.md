# Employee Safety / OSHA Description tab

Use this window to add a description of a selected incident number.
Field/Button
Description

OSHA description
Enter a description of the OSHA incident, or click the Build button to build the description from OSHA codes for illness/injury type, body part, and injury cause.

What was the employee doing just before the incident occurred?
Enter a description of the employee's pre-incident activities.

What happened?
Enter a description of the incident.

What was the injury or illness?
Enter a description of the injury or illness .

What object or substance directly harmed the employee?
Enter a description of the object or substance that harmed the employee.
