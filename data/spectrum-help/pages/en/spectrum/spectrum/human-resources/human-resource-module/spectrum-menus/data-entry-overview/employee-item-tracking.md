# Employee Item Tracking

Use this screen to view a detailed inventory of tracked
 items as well as requisition history. To view history information for an item, highlight the
 item and select the History tab.
Note: If this screen is accessed in context to an employee,
 all items for the employee code will display in the listbox. If this screen is accessed for
 a non-employee specific format, use the search box to narrow your inquiry results.
