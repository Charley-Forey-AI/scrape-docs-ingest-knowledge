# Authorize Recurring Benefits

Use this screen to set up recurring Payroll deduction and
 add-on codes for employees as they become eligible for benefits.
This screen displays only those employees that are eligible for a
 benefit as of the date specified in the screen header, but who have not yet been set up for
 it. When an employee meets all minimum conditions defined for a benefit code on the
 Recurring Benefits File Maintenance screen, including the date defined in the Benefit start
 date field, the employee and the conditional benefit will appear in the list box where the
 status can then be changed using the drop-down list under the Status column.
This screen can also be used to activate benefits that have been
 previously entered to Spectrum. Benefits that the employee is not eligible for or that have
 been declined previously will not appear on this screen. You should review this screen on a
 regular basis to see which employees have become eligible for various benefits.
This screen also displays any matching codes specified in Recurring
 Benefits File Maintenance. These codes will only appear if the associated benefit code
 displays in this column of the detail portion of this screen (meaning this code has a
 status of 'pending'). The matching code will only appear once even if more than one benefit
 code lists it as a matching code. If the benefit code has already been approved or
 declined, then the matching code will not appear.
When this screen is first accessed specify a Department, Eligibility as of date, and
 Active status to filter the
 employees that display on this screen. The selected Eligibility as of date will display
 above the listbox.
If the add-on calculation method is set to one of the 'by time card'
 options, the four 'Limit' columns will be disabled for these methods, with the exception of
 the 'Fixed Amount by Time Card' option.
Note: During the initial set up of Human Resources, this
 screen will contain all of your current employees. Once they have been authorized, the
 screen will only include those employees who have just become eligible for benefits.
