# Triggers Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Trigger type
Select from the following trigger types to include on the report:

- All

- Alpha

- Date

- Numeric

- Paycheck numeric

Note: The option you select here will determine which trigger options will be available in the Search Triggers window, which can be accessed from the Trigger field (below).

Tracking item
Select the desired tracking item to include on the report, or press Enter to accept the software default of ALL.

Trigger
Select the desired trigger to include on the report, or press Enter to accept the software default of ALL.
Note: This field is not available if All is selected as the Trigger type option.

Action
Select the desired trigger action to include on the report, or press Enter to select the software default of ALL.

Message
Enter the trigger warning message to include on the report, or press Enter to select the software default of ALL. Entry in this field is not required.

This trigger applies to
Use this option to include either triggers that apply to All employees, or triggers that apply to Employees of a specific supervisor on the report. If you select Employees of a specific supervisor, the software will prompt you for a supervisor code.

Expiration
Use this option to include either triggers with No expiration or triggers that are Discontinued on a certain date on the report. If you choose to discontinue the trigger on a specific date, two additional fields will display to allow you to set the date range for the trigger.
