# Benefit Management Report

Use the Benefit Management Report screen to print a report of employee benefit information. This report can be printed for a select employee or for a group of employees.
Note: If multi-company reporting is enabled for the current
 operator, the Company code field must be completed before proceeding with report
 selections. If multi-company reporting is not enabled for the current operator, the Company
 code field will not display.
