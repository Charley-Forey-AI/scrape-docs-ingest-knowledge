# Employee Safety / OSHA Other Info tab

Use this tab to add other job information pertaining to a selected incident number.
Field/Button
Description

Other parties
This section of the tab is used to add or review the details related to any other parties involved in the safety/OSHA incident. If the party involved is a company employee, select the 'Employee' checkbox and the rest of the details (SSN, License #, employer, contact information, etc.) will default in.

Property/vehicles
This section of the tab is used to add or review the details related to any property or vehicles involved in the safety/OSHA incident. Select the property type (Vehicle, Equipment, Building, Materials, and Other) and item #, and if the associated details have already been entered elsewhere in the system, they will default.
