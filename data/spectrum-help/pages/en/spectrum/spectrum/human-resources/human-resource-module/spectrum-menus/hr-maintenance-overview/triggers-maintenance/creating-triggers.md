# Creating Triggers

Triggers are powerful, customizable and highly versatile
 tools used to track events and then notify designated supervisors as these events occur with
 warnings, email notifications and updates to various data fields. For example, triggers can be
 used to warn a supervisor that an employee's vacation balance is at zero hours, that an
 employee's driver's license is expiring, or that an important employee field has not been
 completed during setup.
