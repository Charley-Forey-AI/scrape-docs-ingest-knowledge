# OSHA Forms - Multi-Company Reporting

If multi-company reporting is enabled for the current operator, the Company code field must be completed before proceeding with report selections. To complete this field, enter a valid company code. If multi-company reporting is not enabled for the current operator, the Company code field will not display.
