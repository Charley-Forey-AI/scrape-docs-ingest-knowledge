# Employee Change Log

Use this screen to view time-stamped changes that have been made to employee records for the current employee in the current company. You can also see the operator who changed the employee data.
A Selections window allows you to specify the date range and specific fields to include in the list box display. By default, the date range will be one year, and all fields will be selected.
This inquiry does not include entity-specific filtering, changes made to
 employee 'attributes', or Safety/OSHA incidents. For these items, please see the System Administration > Security > Audit Log feature.
Tip: Use the Save Current button to
 store your current inquiry selections for future use.
