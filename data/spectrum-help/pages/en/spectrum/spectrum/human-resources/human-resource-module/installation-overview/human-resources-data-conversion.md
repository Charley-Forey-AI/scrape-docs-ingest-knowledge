# Human Resources Data Conversion

Learn how to complete the Human Resources data conversion
 installation.

1. Complete the Human Resources
 Installation screen.

1. Set up driver's license class information using
 Driver's License Class Maintenance. Print the Driver's
 License Class Listing and review for accuracy.

1. Set up training and certification maintenance codes
 using Training/Certification Maintenance. Print the
 Training/Certification Listing and review for accuracy.

1. Set up triggers using Triggers
 Maintenance. Print the Triggers Listing and review for accuracy.

1. Set up employee location codes using
 Locations Maintenance. Print the Locations Listing and
 review for accuracy.

1. Set up recurring employee benefits using
 Recurring Benefit File Maintenance. Print the Benefits
 Listing and review for accuracy.

1. Set up item type codes for company-owned items using
 Item Type Maintenance. Print the Item Type Listing and
 review for accuracy.

1. Set up item owner codes for use when issuing
 company-owned items using Item Owner Maintenance. Print the
 Item Owner Listing and review for accuracy.

1. Set up job group codes using Job Group
 Maintenance. Print the Job Group Listing and review for
 accuracy.

1. Set up job title codes using Job Title
 Maintenance. Print the Job Title Listing and review for
 accuracy.

1. Set up insurance types using Insurance Type
 Maintenance. Print the Insurance Type Listing and review for
 accuracy.

1. Set up insurance schemes using Insurance
 Scheme Maintenance. Print the Insurance Scheme Listing and review for
 accuracy.

1. Set up insurance codes using Insurance Code
 Maintenance. Print the Insurance Codes Listing and review for
 accuracy.

1. Set up civil/legal/other activity codes using
 Civil/Legal/Other Activities Maintenance. Print the
 Civil/Legal/Other Activities Listing and review for accuracy.

1. Set up drug and alcohol test codes using
 Drug and Alcohol Test Codes Maintenance. Print the Drug and
 Alcohol Test Codes Listing and review for accuracy.

1. Set up approval routing codes using
 Approval Routing Code Maintenance. Print the Approval
 Routing Code Listing and review for accuracy.

1. Set up insurance claim type codes using the Claim Codes Maintenance > Type tab. Print the Claim Type Listing and review for accuracy. Set up
 insurance claim status codes using the Claim Codes Maintenance > Status tab. Print the Claim Status Listing and review for accuracy.

1. Set up injury/illness type codes using the Injury/Illness Codes Maintenance > Type tab. Print the Injury/Illness Type Listing and review for accuracy.
 Set up injury/illness body part codes using the Injury/Illness Codes Maintenance > Body Part tab. Print the Injury/Illness Body Part Listing and review for
 accuracy. Set up injury/illness cause codes using the Injury/Illness Codes Maintenance > Cause tab. Print the Injury/Illness Cause Listing and review for
 accuracy.

1. Set up EEO race classification codes using
 EEO Race Classification Maintenance. Print the EEO Race
 Classification Listing and review for accuracy.

1. Set up all of the employee data for employees in your
 company using Employees.

1. Set up benefits for eligible employees using
 Benefit Management.

1. Authorize employees for recurring benefits as they
 become eligible using Authorize Recurring Benefits.

1. Enter item tracking data for all items owned by the
 company using Item Tracking Management.

1. Set up all job openings currently available in the
 company using Job Openings.

1. Set up employment applicants using
 Applicants.
