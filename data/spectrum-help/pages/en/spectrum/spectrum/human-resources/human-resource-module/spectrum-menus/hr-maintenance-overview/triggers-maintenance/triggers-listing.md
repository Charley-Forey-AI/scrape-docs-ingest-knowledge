# Triggers Listing

Use this screen to designate the types of triggers you want printed on the listing. Trigger listings can include all trigger types, or only designated trigger types such as alpha or date triggers. Additionally, triggers for the listing can be selected based on employee supervisor, tracking item, and expiration date.
