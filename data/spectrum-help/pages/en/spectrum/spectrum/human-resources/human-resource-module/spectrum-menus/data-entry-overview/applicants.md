# Applicants

Use the Applicants screen to track applicants who are applying for positions within your company.

- Click the New button to open the New Applicant window to quickly set up a new applicant. Once set up, the applicant's details can be added in the [Applicant Main Properties](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/applicants/applicant-main-properties) window.

- Click the E-Mail button to launch an e-mail message to the selected applicant.

- Click the Hire Now
 button to open the [Employee Main Properties](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-main-properties) window for the
 selected applicant and enter new hire information.
