# Triggers Maintenance

The Triggers Maintenance screen displays Human Resources
 triggers and their properties. Triggers can be set up to display warnings when certain events
 occur. Triggers might be used to warn a supervisor that an employee's vacation balance has
 gone to zero, when an employee driver license has expired, or when employee fields haven't
 been completed during setup.
