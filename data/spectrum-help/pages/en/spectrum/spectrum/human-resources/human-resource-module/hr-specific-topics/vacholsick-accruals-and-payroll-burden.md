# Vac/Hol/Sick Accruals and Payroll Burden

Whether or not you are use Actual Payroll Burden, the Vac/Hol/Sick G/L Accrual Report/Update starting screen will prompt for an estimated employer burden percentage. As the accrual itself is an estimation, the percentage burden method is used in the calculations. The rate of zero indicates that no burden should be applied.
The Payroll burden field will default from the Job Cost Installation screen. If you are using Actual Payroll Burden, this field will default from the Pre-time card burden % prompt.
An additional prompt will appear for companies using Pay Groups asking if you would like to use the employee's specific burden rate. If the employee doesn't have a burden rate defined in Employee Maintenance, the rate on the start screen will be used.
