# Set Up Vacation, Holiday, and Sick Accruals

Learn how to start set up accruals for employee vacation, holiday, and sick hours.

## Part A

1. On the Site Map screen, select Payroll > Maintenance > Hours Bank Code to display the Hours Bank Code Maintenance screen.

1. Enter information for vacation, holiday, and sick leave benefit accruals and select OK.

1. Select the Listing button and print the associated listing and review for accuracy.

1. On the Payroll Maintenance menu, select Department Expense to open the Department Expense Maintenance screen.

1. Select the department code being used when tracking vacation, holiday, and sick accruals in the General Ledger and select the Edit button.

1. Select the V/H/S Accrual tab and enter the accrual G/L information. The debit account is an expense account and the credit account is a balance sheet accrual. These fields require entry of a valid non-direct G/L account code, regardless of the department's direct cost or equipment status.

1. Select OK to save.

1. On the Site Map screen, select Human Resources > Employees to display the Employees screen.

1. Enter the first employee code for whom you want to enter vacation, holiday, and sick accrual balance information and select the Time Off Bank link on the Info Bar.

1. Select Edit and enter any vacation, holiday, and /or sick hours for the employee.

1. Select Save to return to the Employees screen.

1. Repeat for each additional employee.

1. Select the Master Report link on the Info Bar to review hourly bank totals for this employee.

## Part B

Prior to Part C, you need to make a temporary setting change.

1. On the Site Map, select System Administration > Installation > General Ledger.

1. On the General Ledger Installation > Properties tab, make sure that the Human Resources transactions checkbox is not selected and then select Save.

## Part C

Confirm the Accrual date and Transaction date fields are set to the conversion date.

1. On the Site Map screen, select Human Resources > Data Entry > Vac/Hol/Sick G/L Accrual.

1. Select Preview and review the Vacation, Holiday and Sick Accrual Report.

1. On the Vac/Hol/Sick G/L Accrual Update screen, select Continue.

## Part D

Now that Part C is complete, you can reverse the setting change you made earlier.

1. On the Site Map, select System Administration > Installation > General Ledger.

1. On the General Ledger Installation > Properties tab, select the Human Resources checkbox again and then select Save.
