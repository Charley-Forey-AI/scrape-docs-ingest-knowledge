# Triggers Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Tracking item
Click this button to enter which item you want to track. There are 34 different tracking item types available. For a complete list of available tracking items, click [Creating Triggers - Tracking Items](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/triggers-maintenance/creating-triggers/creating-triggers---tracking-items).

Trigger
Click this button to enter the trigger values, which will define what
 activates the trigger from the Add Trigger Wizard
 screen. Depending on the tracking item, the software may instruct you to
 further refine your requirements. For example, you can activate the trigger
 before or after the tracking date. The employee's anniversary date can also
 act as a trigger.
Additional fields will display in the Values section
 adjacent to this button depending on the type of trigger you select.
 Complete these fields before proceeding to the next step in adding a
 trigger.

Action
Click this button to assign an action to the trigger. Once a trigger is activated, you must define what it is supposed to do. For example, you might set up warnings that will display during Time Card Entry or during Check Calculation or you might designate that the trigger send an email alerting someone (or multiple people), notifying them that the trigger has been met. When creating a trigger that includes more than one email address, be sure to separate the addresses with a semi-colon.

- If you do elect to notify supervisors via email, the software will automatically notify the supervisor via email when a triggers is initiated. The email will detail the employee who caused the trigger to activate in addition to the warning message assigned to the trigger.

-  If more than one employee activates a trigger, the trigger's information will be consolidated into one email and will then be sent to the supervisor. For example, if 10 employees activate five specific triggers during a pay cycle, only one email will be created per trigger. So even though the triggers were activated a total of 50 times (5 triggers x 10 employees = 50), the software will only send out one email for each trigger, each emailing containing all 10 names of the employees who activated the trigger.

- When you click the Action button, a field display that
 is conditional based on your selection in the
 Values section of this window.

Supervisor
Click this button to select the supervisor for whom this trigger applies. Triggers can be created for only one supervisor, or for ALL.

Expiration date
At a certain point in time, you can choose to deactivate the trigger. Click this button to set the expiration date for this trigger, as needed.

Warning message
Enter the actual warning message text that will be sent to the person identified in the email notification.

Description
The description of the current trigger value displays. This field only displays text when you edit the trigger you are currently adding.

Values
When you click a trigger button in the Fields section of the window, a question or series of questions related to the trigger button display in this section.
Important: If you create an action that requires you to enter an
 email address and you want to enter multiple email addresses, be sure to
 separate the addresses with a semi-colon.
