# Employee Drug and Alcohol Testing

Use the Employee Drug and Alcohol Testing screen to review
 and enter employee information for drug and alcohol testing.
If this screen is accessed in context to an employee, all tests for the
 employee code will display in the listbox. If this screen is accessed for a non-employee
 specific format, use the search box to narrow your inquiry results.
