# Veteran Employment Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Entity
If cost centers are set to 'Yes' in the current company and 'entity tracking' is enabled, enter an Entity code to print by entity-specific contractors or subcontractors who were paid during the specified date range. If the 'Report by location' option is selected in Human Resources Installation, this report will also print by 'Establishment' for companies with multiple locations.

Twelve month period ending
Enter the period ending date. The date entered must fall between July 1 and August 31.

Contact name Contact telephone Contact email
Enter your company's contact name, telephone, and email address.

Type of organization
Select the type of organization to include on the report: Prime contractor and/or Subcontractor. These settings will default from human Resources Installation, unless a saved selection is present.
