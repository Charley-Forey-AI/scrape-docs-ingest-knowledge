# Employee Benefits

Use this screen to view, maintain, add, and organize benefit entries for the selected employee. If the Payroll cycle is currently in progress, you will not be allowed to edit benefit information until the cycle has been completed. All employees who are set up for recurring benefits for which they do not qualify will display in red.
If the add-on calculation method is set to one of the 'by time card' options, the 'Limit' columns will be disabled for these methods, with the exception of the 'Fixed Amount by Time Card' option.
