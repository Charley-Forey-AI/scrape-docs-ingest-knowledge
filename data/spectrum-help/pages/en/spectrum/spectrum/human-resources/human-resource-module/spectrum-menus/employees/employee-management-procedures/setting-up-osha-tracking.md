# Setting Up OSHA Tracking

Learn how to set up OSHA tracking.

1. On the Site Map screen, click Human Resources > Data Entry > Safety/ OSHA Tracking.

1. Click the New button.

1. At the Employee code field,
 enter the employee code for the OSHA incident you are tracking.

1. Complete the Properties,Description, Treatment, and
 Other Info tabs. Click OK.

1. Repeat for each additional OSHA incident.
