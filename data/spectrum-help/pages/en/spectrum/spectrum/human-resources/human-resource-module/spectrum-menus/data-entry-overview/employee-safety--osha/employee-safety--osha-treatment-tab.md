# Employee Safety / OSHA Treatment tab

Use this tab to add treatment information about a selected
 incident number.
Field/Button
Description

Was treatment given away from the work site?
If the employee was hospitalized overnight as an
 in-patient, select this checkbox.

Was employee treated in an emergency room?
If the employee was treated in the facility's emergency
 room, select this checkbox.

Was employee hospitalized overnight as an in-patient?
If the employee was hospitalized overnight as an
 in-patient, select this checkbox.

Physician
Enter the name of the physician who treated the
 employee.

Treatment location
Enter the name, address, and phone number of the facility
 where the employee was treated.
