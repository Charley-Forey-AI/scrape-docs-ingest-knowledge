# Incident User-Defined Fields Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Type
Use the drop-down menu to select the field type: Alpha, Date, or Numeric.

Prompt
Enter the title for this field (for example, DUE DATE).

Company
Enter the code for the company that uses this user-defined field, or press Enter to accept the current company code.

Mask
Enter the user-defined field mask (only applicable for Amount fields), or press Enter to accept the current field mask.
The default display mask is "ZZZZZZZ.ZZ-", but you can enter any combination of "Z", "9", and "," as long as the total length of the mask is equal to 10 characters and no more than one decimal point is used. The 11th character is always the minus (-) sign, which you can either include at the end of the display mask or let the software automatically add it in.
 Examples of valid masks are: "ZZZZ.9999-" and "ZZ,ZZZ,ZZZ-".
This display mask will be used as a default only.

Validated?
This option is only available for Alpha user-defined fields. Select this checkbox if the specified user-defined field has validation codes. If this user-defined field does not have validation codes, leave this checkbox clear.
 If you select this checkbox, then click the Valid Codes button to define which codes are valid for the selected user-defined field.

Status
Select the status for this user-defined field: Active or Not used.
