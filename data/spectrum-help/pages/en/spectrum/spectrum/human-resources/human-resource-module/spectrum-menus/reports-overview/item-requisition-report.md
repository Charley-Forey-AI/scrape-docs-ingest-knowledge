# Item Requisition Report

Use the Item Requisition Report screen to print a listing of item tracking requisitions. Use the fields in this screen to determine the range of information you want to display on the printed report.
Note: Note : If multi-company reporting is enabled for the current operator, the Company code field must be completed before proceeding with report selections. If multi-company reporting is not enabled for the current operator, the Company code field will not display.
