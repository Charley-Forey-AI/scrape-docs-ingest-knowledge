# Job Openings

Use this screen to enter, track, and manage job openings within your company. As a part of setting up a job opening in your company, Spectrum provides a number of tabs on which to enter a variety of information related to posting and filling a job opening including qualifications, salary, contact, applicants, and associated costs. Once the job opening is established, these tabs can be revisited to edit job opening information as needed.
Multiple openings can be created for a job opening so that multiple hires can be tracked for the same position without having to track each hire using a separate job opening. For example, you may want to hire two carpenters, with the same job description and requirements. Rather than setting up two job openings, you can simply set up one job opening with two available openings.
