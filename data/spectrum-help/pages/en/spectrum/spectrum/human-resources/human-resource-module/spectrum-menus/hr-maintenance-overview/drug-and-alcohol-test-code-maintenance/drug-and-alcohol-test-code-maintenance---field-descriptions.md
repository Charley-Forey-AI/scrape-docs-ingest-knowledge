# Drug and Alcohol Test Code Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Code
Enter a drug and alcohol test code.

Description
Enter the description for the drug/alcohol test code you are adding.

Type
Select the type of drug or alcohol test to associate with the selected code. Options include Drug, Alcohol, or Other.
