# Employee Drug and Alcohol Report

Use the Employee Drug and Alcohol Report screen to print a
 listing of items that are past due for return by an employee or group of employees. Use the
 fields in this screen to determine the range of information you want to display on the printed
 report.
Note:
 If multi-company reporting is enabled for the current operator, the
 Company code field must be
 completed before proceeding with report selections. If multi-company reporting is not
 enabled for the current operator, the Company code field will not display.
