# Employee FMLA - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee code
Enter an employee code in this field. If this screen is accessed in context to an employee, the employee code will default in this field.

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you
 can enter a cost center entity here in order to manage FMLA eligibility and
 leave details by entity.
This screen will display entity-specific FMLA eligibility and other details for the employee, even though the employee may currently be 'inactive' for payment from that entity.

Eligible for FMLA?
If the selected employee is eligible for FMLA benefits, this checkbox will be selected and the following fields will be available for entry.

Totals for last 12 months

Weeks used
Enter the number of weeks of FMLA leave this employee has taken during the last 12 months, if applicable. Partial weeks may be entered by using decimal places (up to two spaces). If this employee has not used any FMLA leave, leave this field clear.
Either whole or partial weeks of FMLA leave can be used. If the employee has used a partial week of FMLA leave, the number displays as a decimal number for more accurate FMLA tracking. For example, if an employee takes 4 days of FMLA leave, the number of days used displays as .8.

Weeks remaining
The remaining number of weeks of FMLA leave this employee has displays.
 Spectrum calculates the amount in this field based on the number entered in
 the Weeks available per 12-month period field, which is located on the Human Resources Installation > FMLA/OSHA screen. If the employee has used more than the allotted
 number of FMLA time, the number of weeks remaining displays as a negative
 number.

Calculate as of
Enter the date used to calculate the remaining weeks of FMLA leave available to this employee.
Note: This calculation is incremented on a week by week basis; day
 to day calculations are not performed. Instead, the software will round any
 extra days taken up to the next week. For example, if an employee is on
 leave for two weeks and two days, the report will show that three weeks
 leave has been taken.

Start FMLA leave

Currently on FMLA leave?
If the employee is currently on FMLA leave, select this checkbox and enter the following fields.

Actual start date
If this employee is currently on FMLA leave, enter the date the employee went on leave.

Expected end date
If this employee is currently on FMLA leave, enter the last date the employee will be on leave.

Attach Incident
Click this button to attach an unassigned incident to this employee.

Clear Incident
Click this button to clear the incident log for the selected FMLA record.

End FMLA leave

End current FMLA leave?
If an employee's leave period has ended, select this checkbox and enter the following fields.

Actual end date
Enter the date when the employee returned to work.

Weeks used
Enter the number of FMLA weeks (whole or partial) that the employee used.
 Either whole or partial weeks of FMLA leave can be used to track FMLA leave. If the employee has used a partial week of FMLA leave, the number displays as a decimal number for more accurate FMLA tracking. For example, if an employee takes 4 days of FMLA leave, the number of days used displays as .8.
When you click OK, the FMLA history file is updated with the amount of FMLA leave used by the selected employee.

FMLA history

Incident
Click this button to view the selected incident history.
