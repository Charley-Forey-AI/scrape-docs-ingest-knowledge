# Employee Item Tracking - Issue Item

Use this window to issue an item to an employee.
Field/Button
Description

Item
Enter the item code in this field. Only items with a status of 'Available' can be issued.
If this window is accessed for a non-employee specific format, the item code on the highlighted row will default, but can be changed.

Employee
If this window is accessed in context to an employee, the employee code will default in this field.
If this window is accessed for a non-employee specific format, the employee code on the highlighted row will default, but can be changed.

Location
Enter the location for the item.

Requisition date
Enter the date on which the item was requested.

Due date
Enter the due date for the item.

Notes
Enter any notes pertaining to the item being issues. For example, if the item
 is damaged in any way.
