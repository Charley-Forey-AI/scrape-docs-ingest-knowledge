# Item Type Maintenance

Use the Item Type Maintenance screen to maintain item type codes in the Spectrum system. Item type codes represent broad categories of company property that is issued to employees, such as computer peripherals, company credit cards, cell phones, backhoes, lifts--almost anything you can think of.
