# Employee Reviews - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee code
The selected employee code displays in this field.

Description
Enter a description for the review. For example, if you are entering a a twice yearly performance review, enter BI-ANNUAL EMPLOYEE REVIEW.

Date required
Enter the date the review is required.

Comment
Enter any comment for the review.

Status
Select a status for the review: In progress or Completed

Date completed
Enter the date the review was completed.
