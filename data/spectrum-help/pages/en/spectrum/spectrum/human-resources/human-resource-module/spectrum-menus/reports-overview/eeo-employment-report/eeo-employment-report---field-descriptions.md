# EEO Employment Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/ButtonDescription
EntityIf cost centers are set to 'Yes' in the current company and 'entity tracking' is enabled, enter an Entity code to print by entity-specific employees based on payment history during the specified date range and employee characteristics. If the 'Report by location' option is selected in Human Resources Installation, this report also prints by 'Establishment' for companies with multiple locations.
Report typeSelect who you want to appear on the report: Employees or Applicants.
Employee selectionsThese fields apply only to Employees. If you have selected the Applicants option in the Report type option box, these fields don't display.

Period end dateEnter the earliest and last period end dates to include on this report, or press Enter to accept the current Payroll processing date.
Previous year dateEnter the earliest and latest previous year's dates to include on this report, or press Enter to accept the previous year's default end date.
Hire dateEnter the earliest and latest employee hiring dates to include on this report, or press Enter to leave these fields blank and include all hire dates.
The hire date specified here does not apply to the previous year totals; all employees are included on the previous year totals regardless of hire date.

Applicant selectionsThese fields apply only to Applicants. If you have selected the Employees option in the Report type option box, these fields don't display.

Post dateEnter the earliest and latest job posting dates to include on this report.
Start dateEnter the earliest and latest applicant starting dates to include on this report.
