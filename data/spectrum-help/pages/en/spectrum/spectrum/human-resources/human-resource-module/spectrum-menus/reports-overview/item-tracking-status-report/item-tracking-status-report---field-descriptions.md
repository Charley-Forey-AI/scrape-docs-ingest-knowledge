# Item Tracking Status Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Tracking code Owner Item type Vendor Location
Enter the tracking codes, owner codes, item type codes, vendor codes, and location codes you want to include on the listing, or press Enter to include ALL on the report.

Status
Select the corresponding checkbox if you want to include those items that are currently available on the listing, those items that are currently issued to employees on the listing, and /or those items that are currently retired (unavailable for issue) on the listing.

Issued to
Enter the specific employee code to print, or press Enter to include ALL of the employee codes in Spectrum on the report.

Acquire date
Enter the earliest and last item acquisition dates to include on the report, or press Enter to include items with future posting and/or closing dates.

Due date
Enter the earliest and last item due dates to include on this report, or press Enter to print all dates.

Retire date
Enter the earliest and last item retire dates to include on this report, or press Enter to include items with future posting and/or closing dates.
