# Tickler Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee code
Status
Department
Supervisor
Location
Enter the employee code, employee status, department code, supervisor code, and location code, or press Enter to include ALL the employee codes.

Item selections

Item date
Enter the earliest and latest dates for which employees were issued items to include on the report, or press Enter to accept the current Payroll processing date.

Include...
Select whether to include the following items for the employee(s) on the report:

- Birthday

- Latest hire date anniversary

- Driver's license expiration date

- I9 expiration date

- Last review date

- Next review date

- FMLA start date

- FMLA expected end date

- User-defined field dates

- Training expiration dates

- Training required dates

- Insurance expiration dates
