# Procedures As Required

View the procedures as required in the HR module.

- Add user-defined fields as needed using Employee User-Defined
 Fields Maintenance.

- Set up triggers as needed using [Triggers Maintenance](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/triggers-maintenance).

- Use the Employees screen to set up employee-specific
 information pertaining to insurance, dependents, training, reviews, medical, and so
 forth.Note: All of the codes used to set up this
 employee-specific data must first be set up using the screens available on the
 Human Resources Maintenance menu.

- Use [Job Openings](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/data-entry-overview/job-openings) and [Applicants](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/data-entry-overview/applicants) to
 manage hiring and termination data for the company. These screens are available from
 the Human Resources Data Entry menu.

- Use Safety/OSHA Tracking to enter and track safety
 and OSHA-related incidents. These incidents can also be handled using the
 Employees screen.

- Use Rate Management to adjust employee pay
 rates.

- Use Item Tracking to issue and track company-owned
 items, such as computers and computer peripherals, that are issued to
 employees.

- Track vacation, holiday, and sick leave accruals using
 Vac/Hol/Sick G/L Accrual/Update. Accrual information
 entered and updated using this screen will post to the general ledger. If an accrual
 is performed in error, reverse the accrual using Reverse Vac/Hol/Sick G/L
 Accrual.

- Manage employee benefits using Benefit Management.
 Integrated with this screen is the Authorize Recurring
 Benefits screen, which allows you to manage recurring benefits for
 employees as they become eligible.

- When G/L accrual history information is no longer needed, use the
 Purge G/L Accrual History screen to remove this
 information.

- When item tracking information is no longer needed, use the Purge
 Item Tracking History screen to remove this information.

- When employee detail for a select employee is no longer needed, use the
 Purge Employee History screen to remove this
 information.
