# Employee Training / Certifications

Use this screen to manage skills and certification
 information on select employees and applicants. If this screen is accessed in context to an
 employee, all items for the employee code will display in the listbox. If this screen is
 accessed for a non-employee specific format, use the search box to narrow your inquiry
 results.
