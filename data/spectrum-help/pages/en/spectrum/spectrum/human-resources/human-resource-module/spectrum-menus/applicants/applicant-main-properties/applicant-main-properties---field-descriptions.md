# Applicant Main Properties - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/ButtonDescription
Applicant codeThe selected applicant code displays. The applicant name displays to the right of this field.
Alpha referenceEnter the employee's alpha reference code, or press Enter to accept the current alpha reference code. (This field defaults from the Payroll module.)
Applicant statusEnter a status code, or press F4 to select a code from the Search Status Codes window. Once selected, the associated status type displays below. Note: Applicants are not considered active until they are assigned to at least one active job opening.

Employment info
OccupationEnter the applicant's occupation, or press F4 to select a valid location code.
TradeEnter the applicant's trade, or press F4 to select a valid trade code
EEO classificationEnter the EEO (Equal Employment Opportunity) classification code for this applicant, or press F4 to select from a list of valid EEO classification codes.
Work classEnter the applicant's work class, or use the drop-down menu to select from a list of valid work classes.

LocationEnter the applicant's location, or press F4 to select a valid location code.
Contact info
Personal telephoneEnter the applicant's telephone number.
Telephone memoEnter any notes related to the applicant's telephone number in this field.
FaxEnter the applicant's fax number, if applicable.
Mobile phoneEnter the applicant's mobile phone number.
E-mailEnter the applicant's email address, if applicable.
Governmental reporting
Social security #Enter the applicant's social security number.
 If Payroll reporting is set to 'Canada' in the Installation screen, the number entered here will be formatted to comply with Canadian mask guidelines, including a single space between the 3rd and 4th digits and a single space between the 6th and 7th digits.

RaceUse the drop-down menu to select the applicant's race from the list of defined race types. Race options on the drop-down menu are defined in Human Resources EEO Classification Maintenance.
SexEnter the applicant's gender in this field.
I-9 statusEnter the applicant's I-9 status, or use the drop-down menu to select from a list of valid I-9 statuses.
I-9 expirationEnter the applicant's I-9 expiration date, if applicable.
Mailing addressEnter the applicant's mailing address.
Applied for
Add OpeningSelect the job opening number for which the candidate has applied, or select New to create a new job opening.
Edit StatusSelect to open the Applicant Hire Status window for the selected applicant.
RemoveSelect to remove the selected applicant.
