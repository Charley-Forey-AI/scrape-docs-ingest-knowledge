# Employee FMLA Report

Use this screen to select the FMLA information you want to print on the Employee FMLA Report.
If multi-company reporting is enabled for the current operator, the
 Company code field must be completed before proceeding with
 report selections. If multi-company reporting is not enabled for the current operator, the
 Company code field will not display.
Additionally, if the Employees are eligible for family and
 medical leave under FMLA checkbox is clear in the Human Resources Installation > FMLA/OSHA screen, then the following message will display: THIS COMPANY HAS
 BEEN DESIGNATED INELIGIBLE FOR FMLA IN THE HUMAN RESOURCES INSTALLATION
 SCREEN. To resolve this issue, you must adjust settings on the Human Resources Installation > Reporting tab.
