# Error - The FMLA Period Option Is Not Set

The FMLA Period Option Is Not Set. Please Complete Data in
 HR Installation Screen.
This error displays if the fields on the Human Resources Installation – Reporting tab are
 left blank.
To remove this error message, on the Site Map screen, select System Administration > Installation > Human Resources, click the Reporting tab and complete the following fields:
