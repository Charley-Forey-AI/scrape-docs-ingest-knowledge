# Purge G/L Accrual History

Use the Purge G/L Accrual History screen to permanently delete vacation, holiday, and sick G/L accrual history from the software for a specific date range. Be sure to print the G/L Accrual Report prior to proceeding with the purge.
CAUTION: Exercise extreme caution
 when performing purges.
