# OSHA Establishment Maintenance - Field Descriptions

A reference for completing the fields on this screen.
Field/ButtonDescription
Establishment code
Name
Address
Enter a code, name, and address for the new OSHA establishment.
Properties
Industry descriptionEnter an industry description for the establishment.
NAICS codeEnter a six-digit NAICS code.
Avg # employees
Total hours last year
Enter the average number of people that the company employs during the year, and average number of total hours worked by all employees in a year. Note: These two values are automatically calculated from Payroll history if specified in the Reporting tab of the Human Resources Installation screen.

Add codeSelect to add location codes to this establishment. If the location code is already in use by another establishment, you will be prompted to switch to the new location.
RemoveSelect to remove a location code from this establishment.
