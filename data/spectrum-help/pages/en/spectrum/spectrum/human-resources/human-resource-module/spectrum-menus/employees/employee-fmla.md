# Employee FMLA

Use the Employee FMLA screen to
 track the selected employee's Family Medical Leave Act eligibility. You can also track the
 employee's FMLA leave dates, if applicable.
If the Employees are eligible for family and medical leave under FMLA
 checkbox is clear in the Human Resources Installation > FMLA/OSHA screen, then the following message will display:
ERROR – THE FMLA PERIOD OPTION IS NOT SET. PLEASE COMPLETE DATA IN HR INSTALLATION SCREEN
To resolve this issue, you must adjust settings on the Human Resources Installation > FMLA/OSHA screen.
