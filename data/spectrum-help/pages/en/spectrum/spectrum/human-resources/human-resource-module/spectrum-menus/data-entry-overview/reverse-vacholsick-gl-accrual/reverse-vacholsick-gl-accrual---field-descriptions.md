# Reverse Vac/Hol/Sick G/L Accrual - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Employee Employee status Home department VHS accrual codes
Enter the employee code, employee status code, home department code, and the vacation/holiday/sick code, or press Enter to include ALL on the report.

Reverse accrual date
Enter the accrual date when the Vac/Hol/Sick G/L Accrual/Update was run, or press Enter to accept the current Payroll processing date default.
Note: This date must fall within the Payroll minimum/maximum date
 range.

Vacation accrual? Holiday accrual? Sick accrual?
Select the corresponding checkbox if you want to include vacation hours, holiday hours, and sick hours on the reversal.

Sort by
Select the sort option for the report.

- G/L account code – this format sorts and subtotals the report based on accrual type (vacation, holiday, and sick pay)

- Employee code – this format sorts and subtotals the report based on employee code
