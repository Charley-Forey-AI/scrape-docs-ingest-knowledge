# Benefit Management Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you
 can enter a cost center entity here in order to display entity-specific
 balance information. Leave this field blank to use the main company
 entity.
Multi-company reporting is not applicable when 'Entity tracking' is used. The report will only include records for employees in the current company.

Employee code
Employee status
Home department
Benefit code
Benefit status
Enter the employee code, employee status code, home department code, benefit code, and benefit status code you want to appear on the report, or press Enter to include ALL on the report.

Include deductions?
Select this checkbox if you want to include deductions on this report.

Include add-ons?
Select this checkbox if you want to include add-on information on this report.

Salary range

Select by salary range?
Select this checkbox if you want to want to select information to appear on the report based on salary range.

Date
Enter the date from which the salary range information should begin and end. To begin with the first day of the current year (based on the current Payroll processing date) and end with the current Payroll processing date leave this field blank by pressing Enter.
 This and the following fields are only available if the Select by salary range checkbox is selected.

Minimum earnings
Enter the minimum annual earnings amount that you want to determine which benefits will appear on the report.

Maximum earnings
Enter the maximum annual earnings amount that you want to determine which benefits will appear on the report.

Age range

Select by employee age?
Select this checkbox if you want to want to select information to appear on the report based on employee age.

As of date
The date entered in this field is compared to the employee birth date to determine whether or not the employee should appear on the report. Press Enter to use the current Payroll processing date.
 This and the following fields are only available if the Select by employee age checkbox is selected.

Minimum age
Enter the minimum age employee must be to appear on the report.

Maximum age
Enter the maximum age that you want to determine which benefits will appear on the report.
