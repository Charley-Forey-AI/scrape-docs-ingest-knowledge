# Applicant Main Properties - Cost Center Information

If cost centers are set to 'Yes' in the current company, the cost center will restrict operator access to disallowed job openings. The job openings that the applicant has applied for will still appear in the listbox, but the operator will not be allowed to edit status openings they do not have access to.
