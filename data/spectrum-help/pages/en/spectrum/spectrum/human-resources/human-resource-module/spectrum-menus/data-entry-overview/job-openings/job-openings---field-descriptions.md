# Job Openings - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Opening #
Enter a job opening number for the opening you are adding, or press Enter to accept the system default.

Job title
Enter a job title or press F4 to select a job title. Job titles are set up in Job Title Maintenance.

Notes
Click this button to open the Job Opening Notes sub-window. Use this window to add any notes related to the job opening.

Properties

Supervisor
Enter the hiring supervisor's code, or press F4 to select a supervisor.

Job group
The job group number displays based on the job title you have selected. Press Enter to accept the default code, or press F4 to select a different code than the one assigned the selected job title.

Occupation
The occupation code displays based on the job title you have selected. Press Enter to accept the default code or press F4 to select a different code than the one assigned the selected job title.

Trade
The trade code displays based on the job title you have selected. Press Enter to accept the default code, or press F4 to select a different code than the one assigned the selected job title.

EEO classification
The EEO classification code displays based on the job title you have selected. Press Enter to accept the default code, or press F4 to select a different code than the one assigned the selected job title.

Location
Enter the job opening's location, or press F4 to select a location.

Department
The department code displays based on the job title you have selected. Press Enter to accept the default code, or press F4 to select a different code than the one assigned the selected job title.

Union
The union code or pay group displays based on the job title you have selected.
 (Union and pay group settings are established on the Payroll
 Installation screen.) Press Enter to accept the
 default code, or press F4 to select a different code than the one assigned the
 selected job title. An alert message will display if the union code entered
 has an Inactive status, but you will be allowed to continue entry.

Wage code
The wage code displays based on the job title you have selected. Press Enter to accept the default code, or press F4 to select a different code than the one assigned the selected job title.
Note: This field will be hidden if the Payroll Installation > Default rate table is not set to 'Wage codes/union'.

Status

Status
Select the job opening's status from the drop-down menu: Open, Closed, Hold, or Hire in Progress.

Milestones

Posted
The current Payroll processing date defaults. Press Enter to accept this default, or enter a different job posted date as needed.

Open date Closing date Start date
Enter the job opening date, closing date, and job start date, or press
 F4 to select
 dates from the Date Change window.

Applications

New
Click this button to open the [Applicants](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/data-entry-overview/applicants) window.

Edit Status
Click this button to open the Applicant Hire Status window for the selected row. Use this window to change the hire status for the selected applicant.

Remove
Click this button to remove the selected applicant.

E-Mail
Click this button to launch a new email message to the selected applicant.

Hire Now
Click this button to open the [Employee Main Properties](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-main-properties) window
 for the selected applicant and enter new hire information.

Applicant main properties
Click this link to open the [Applicant Main Properties](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/applicants/applicant-main-properties) window for the selected applicant.
