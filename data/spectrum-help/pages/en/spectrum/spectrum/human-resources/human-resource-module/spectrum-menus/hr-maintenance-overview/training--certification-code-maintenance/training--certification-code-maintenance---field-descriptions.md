# Training / Certification Code Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Code
Enter the training / certification code, or press Enter to accept the current training / certification code. If you are editing an existing code, changes are not allowed in this field.

Description
Enter a description for the training / certification code.

Track levels?
Select this checkbox if this training / certification has multiple levels that
 must be completed by the employee, for example, an HVAC certification
 program that has three levels.

Validated?
Select this checkbox if the selected training/certification code should have validated training levels. When this checkbox is selected, the [Training Levels window](/en/spectrum/spectrum/human-resources/human-resource-module/spectrum-menus/hr-maintenance-overview/training--certification-code-maintenance/training-levels-window) window will automatically display prompting you to enter skill levels for validation.

Require starting date?
Select this checkbox if you want to track the start date for this training / certification.

Require ending date?
Select this checkbox if this training / certification has a required completion date.

Require need-by date?
Select this checkbox if this training / certification has a need-by expiration date.

Require expiration date?
Select this checkbox if this training / certification has an expiration date.

Price
Enter the training / certification price.

Employee Reimbursement?
Select this checkbox if employees can be reimbursed for this training.

Deduction/add-on code
If the Employee reimbursement checkbox is selected, enter the deduction/add-on code for this training/certification, or press Enter to accept the current deduction/add-on code. Up to 10 characters are allowed. Once this code is associated with the employee, they will be reimbursed automatically through the payroll system.

Status
Select the current status of this training / certification: Active or Not used.
