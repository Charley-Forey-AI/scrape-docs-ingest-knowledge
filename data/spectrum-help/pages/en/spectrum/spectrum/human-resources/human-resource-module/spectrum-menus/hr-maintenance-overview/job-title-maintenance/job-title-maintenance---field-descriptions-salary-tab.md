# Job Title Maintenance - Field Descriptions: Salary tab

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Defaults

Pay type
Use the drop-down menu to select a pay type associated with the selected job
 title: Hourly, Salary,
 Overtime Salary, or Commission.

Frequency
Use the drop-down menu to select a pay frequency associated with the selected
 job title: Daily, Weekly,
 Bi-weekly, Semi-monthly,
 Monthly, Quarterly, or
 Annually.

Standard hours
Enter the standard number of hours associated with the selected pay frequency. For example, you might enter "40" if the selected job title is paid on a weekly basis.

Salary rate
Enter the salary for the given pay period. This field is not available if the selected pay type is Hourly.

Hourly rate
Enter the hourly rate for the given pay period. Press F4 to access the Viewpoint
 Online Calculator.

Annual rate
The system calculates and displays the annual rate.

Salary range
Enter a salary range for the job title.

Other
Enter any additional salary information that pertains to the job title.

Pay levels

Pay level
Enter a new pay level code in this field.

Description
Enter a description for the new pay level code you are adding.

Salary rate
Enter a salary rate for the new pay level in this field. This field is
 available only if the selected pay type is Salary,
 Overtime Salary, or Commission.

Hourly rate
Enter a hourly rate for the new pay level in this field. This field is available only if the selected pay type is Hourly.

Annual rate
The system calculates and displays the annual rate in this field.
