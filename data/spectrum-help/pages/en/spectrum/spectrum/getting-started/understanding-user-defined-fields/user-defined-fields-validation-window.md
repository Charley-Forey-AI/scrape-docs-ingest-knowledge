# User-Defined Fields Validation window

This window is available for alpha user-defined fields that
 have validation codes. Use this window to add and edit valid field values and their
 descriptions.
If an alpha user-defined field has validation codes, only specified
 values are allowed for that field; for example, a user-defined field may require a Yes or
 No entry. If an alpha user-defined field does not require a limited value (for example, an
 emergency contact's name), that field should not have validation codes.
