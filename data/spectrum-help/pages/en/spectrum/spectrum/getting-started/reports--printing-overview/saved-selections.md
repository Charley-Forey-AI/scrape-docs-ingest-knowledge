# Saved Selections

Located in the Saved Selections bar, click the Save Current
 button after you have made your screen selections if you think you want to save them for
 future use. A window opens that allows you to enter a name for the "template" that you are
 saving, and you can set this template as your default. Additional settings allow you to share
 this template with other operators or to assign it as a system-wide default (thereby ensuring
 that everyone is seeing the same items).
