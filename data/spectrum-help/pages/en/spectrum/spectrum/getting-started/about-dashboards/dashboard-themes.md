# Dashboard Themes

You can customize the appearance of your Spectrum Dashboard
 using multi-colored themes.

1. On the app you wish to change, select the Edit icon.

1. Select the new color theme and interval that will determine when the changes take effect.

1. Select Save to implement your changes.
The order in which the chart uses the data determines the color used. When the Default Theme is selected, the colors are used in this order:

1. Red

1. Dark Green

1. Blue

1. Light Green

1. Yellow

1. Dark Blue

When there are more than six data points, the colors will start over from the top of the list.
The two main colors are used in the name. For example, the 'Blue and Gray' theme also includes green and the 'Blue and Green' also includes yellow.
