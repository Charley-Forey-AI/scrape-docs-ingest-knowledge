# Keyboard Shortcuts

In addition to function keys, there are several other keyboard shortcuts available in Spectrum. These keystrokes allow you to operate the software even when a mouse is not present on the software.
To access options shown on the pull-down menus, hold down the Alt key and press the letter of the desired option.This is the same approach one would use with any Microsoft Windows product. Other keyboard shortcuts include:
Shortcut
Action

 Ctrl+X
Cut - Copies the contents of the current selection within the current input field to the clipboard, then deletes the selection.

Shift+Delete
Cut - Copies the contents of the current selection within the current input field to the clipboard., then deletes the selection.

 Ctrl+C
Copy - Copies the contents of the current selection within the current input field to the clipboard.

Ctrl+Insert
Copy - Copies the contents of the current selection within the current input field to the clipboard.

 Ctrl+V
Paste - Copies the contents of clipboard to the current insertion point within the current input field.

Shift+Insert
Paste - Copies the contents of clipboard to the current insertion point within the current input field.

Ctrl+Home
Top - Repositions cursor to the beginning of the grid.

Ctrl+End
Bottom - Repositions cursor to the end of the grid.

Ctrl+F5
Insert - Causes records to be placed at the current position with the file.

Ctrl+Break
PROIV Interrupt Key - This exits the process and should not be used unless instructed to do so by the Spectrum Support staff.

Period+Enter
Saves all the information on the current screen.

/+Enter
Escape - Reverts changes and then exits. This is another way to CANCEL.

ESC
Escape - Reverts changes and then exits. This is another way to CANCEL.
