# Build a Menu

Create a list of menu shortcuts for display in the Navigation Bar.

1. Log in to Spectrum.The Dashboard and Navigation Bar
 display.

1. In the Navigation Bar, select Edit to open the Menu
 Builder screen.

1. Choose one:

- Select Add Menu to create a
 personal menu that will display on the Navigation Bar.

- Select Add Public Menu to add a
 pre-defined menu to the Navigation Bar.

- Select a Spectrum menu function to have it
 display in your Favorites list.

1. Select Spectrum functions from the left pane.
 Select to Show All
 (default), show Menus, or
 show Functions, and then
 navigate to select the specific functions you want to include in your personal menu
 or bookmark as a Favorite.

1. Drag a function from the left pane to any location on
 the 'canvas'. Functions can be rearranged on the canvas by simply dragging and
 dropping them to different locations.

1. Select Save to exit the Menu Builder screen and see your new menus display
 on the bottom of the Navigation Bar. Favorites display nearer the top of the
 Navigation Bar.

1. Return to the Menu Builder screen any time
 you want to rearrange your menu lists or delete. To delete a function
 link, or an entire menu, simply select it and drag it to the Delete
 icon (trash can) above the canvas. menu items.

If you wish to share menus with other operators, you must
 create a profile that is associated with the menus using the Menus button in the
 Profile Maintenance screen. Next, assign the
 profile to other operators in the Operator
 Maintenance screen.
