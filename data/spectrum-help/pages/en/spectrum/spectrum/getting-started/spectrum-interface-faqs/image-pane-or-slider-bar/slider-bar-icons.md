# Slider Bar Icons

Use the table below as a quick reference for the Slider Bar
 icons. You can also hover your mouse over the icons in the software to see descriptions of the
 icon functions.
Icons/Buttons
Descriptions

Transaction Header Information

Open or close the Slider Bar.

Open (or "pop out") the image in a new browser tab. The
 viewer will remain synchronized with its parent Spectrum tab and the last
 transaction will stay visible.
If you attempt to view a file type that is not supported
 by the Slider Bar (for example, an .xls file), then the image file will
 automatically open via the browser.

The count of the total documents attached to the
 transaction displays.

- Hover over the wrench icon on the transaction
 header to see the transaction's Cabinet, Drawer, and Folder
 designation.

- Click the wrench icon to open the Modify
 Transaction Information window where you can change the transaction
 description and enter keywords.

The description for the current transaction displays.
 Click the drop-down arrow to view and select from a list of additional
 documents that display alphabetically by description.
If you select a file type that is not supported by the
 Slider Bar (for example, an .xls file), then the image file will open via
 the browser.

Use this icon to mark an image as 'Private'. When set to
 Private, the padlock will display as red and locked. When white and
 unlocked, the image is set to Public (this is the default setting).
Image status changes are not permanent: you may change
 the status from Public to Private and back to Public, and more.
By designating an image as Private, you exclude certain
 images from being sent to your customers, including:

- AR Change Request Proposal Form

- PJ HTML Forms

- T+M Detail Billing Form

- T+M Detail Billing History Report

- Attachments when emailing non-HTML PJ Forms


Also, Private images will be excluded in the Distribute
 feature in the following areas, although you still have the option to
 include 'Private' images as desired:

- DI Search Documents Listing

- Job Cost History Report
 Note: The Private status is stored on the image
 file itself, NOT the Document Imaging transaction.

Action Buttons

Scan
Click this button to open your scanning software
 (scanning is not supported from the browser viewer). After completing the
 scan, you will need to confirm the scan and the file will be saved as either
 a PDF or TIF file (as designated in the Document Imaging Installation -
 Properties screen.
Note: You must have security level 3 to use this
 button.

Attach
Click this button to open a File Upload dialog box. Type
 a patch and file name, or browse to find the file you want to attach. A
 confirmation window lets you know when the file has attached successfully
 and you can enter a document description or accept the transaction
 description.
Note: You must have security level 3 to use this button.

Batch
Click this button to open the Batch Documents window and
 select a different batch. Only batches assigned to the current operator will
 display.

- If a batch is selected, it will display in the
 Slider Bar in a new batch.

- Only one batch may be open at a time.

- When the last screen is removed from the batch
 document, the document will be deleted and the Batch tab will close.

- If the 'new browser tab' option is used, the batch
 is not opened in the new browser; only the current transaction is
 opened.

Manipulation Icons (only available with level 3
 security)

Click to print the current image. Select a screen range
 and grayscale.

Click to export the image file. You will be prompted to
 pick an output format (PDF, TIF, or original), and then click the Export button.

When viewing multi-screen files, this tells you which
 screen you are currently looking at.

Screen up / screen down.

Toggles view between thumbnail and full image.

Zoom in / Zoom out.

This Zoom Menu icon offers a drop-down list of 4
 options:

1. Rubber band Zoom - lets you draw around a portion of
 the image to zoom.

1. Fit to Window - lets you display images to match
 the window size.

1. Fit to Height - lets you display the full vertical
 height of images.

1. Fit to Width - lets you display the full horizontal
 width of images.

Click to rotate the image 90 degrees right or left.

Annotation Icons (only available with level 2 or 3
 security)

Click this icon and drag and release it to draw an arrow
 on the image file.

Click this icon and drag and release to highlight an area
 on the image file.

1. Click this icon to draw a note on the image file.

1. Right-click on the note and select Edit Text to type
 the note.

1. Click the green check mark to save the new note.
 Note: To delete a note, you need to right-click on
 it and select Delete.

1. Click this icon and select Add New Text to
 create a note on the image file.

1. Right-click on the not, select a font color and
 size, and then click Edit Text to enter new text.

1. Click the blue check mark to save the new text.

1. To close the edit window, click outside of the
 window.
 Note: To delete a note, right-click on the note and
 select Delete.
