# New, Edit, View, and Delete buttons

Use these action buttons to keep your data current. Primary actions will display as a larger, green button on the toolbar, while secondary actions will display as smaller, gray buttons.
ButtonsDescriptions
NewSelect to create a new record or transaction. For list boxes, this button typically appears on the command bar; for grid screens it's located next to the Edit button.
EditChoose and transaction and select Edit to make changes to an existing record or transaction. For list box screens, you can also make changes by simply double-clicking on the list box, pressing the Enter key in the list box, or pressing the Tab key in the list box.
ViewSelect to view detailed information about a line of data, but not to make changes.
DeleteChoose a record and select this button to delete it.
InsertSelect to create a line directly above the line your cursor is currently in. This button appears only in screens where the order of the detail records is meaningful.
ExpandSelect to toggle the grid display between single-line view and multiple lines with additional description fields per record. This button appears only in screens that don't have the option to scroll.
