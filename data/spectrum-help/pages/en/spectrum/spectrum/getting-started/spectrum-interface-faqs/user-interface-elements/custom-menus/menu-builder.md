# Menu Builder

To access the Menu Builder, on the Navigation Bar, click the Edit option.
The Menu Builder screen is used to create a list of menu shortcuts that will be displayed in the Navigation Bar, which is visible from any screen in the system. Administrators can use the Menu Builder to set up public menus--menus that are available to everyone in the system but that can only be modified by the Administrator. Operators can add, delete, or move public menus, but not the items listed in the menus.
The Menu Builder differs from the Favorites feature in that it allows you to create personal menus, or groupings, of shortcuts that you reference most often. For example, if there are certain features you only use once-a-month, you might create a "Monthly Items" menu and include links to those functions here. The menus you create will display on the Navigation Bar.
