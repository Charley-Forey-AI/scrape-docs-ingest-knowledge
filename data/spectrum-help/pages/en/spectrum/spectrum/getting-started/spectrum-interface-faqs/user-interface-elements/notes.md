# Notes

The Notes windows found throughout the software allow you
 to enter unlimited descriptions and comments.
Features include:

- Easy entry directly from the screen (no need to drill into another
 window), even from touch-screen devices

- No length restrictions

- Automatic wrapping

- Spell check


- Formatting options

- Copy and Paste

- Date stamping

- Search functionality

- Tool tips to explain toolbar options

- Security controls for managing note topics, including when notes
 can be seen in more than one company. In the note topic window, when switching from
 all topics to a single company, any existing notes for the topic will no longer
 display in other companies. Existing notes in the current company will display
 unchanged.
