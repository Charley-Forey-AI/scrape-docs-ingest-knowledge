# Custom Menus

Use the Site Map Customization screen to customize the Spectrum Site Map by extending any menu to include additional functions.
To open the Site Map Customization screen, go to System Administration > Security > Site Map Customization.
Custom menus allow you to customize the
 Spectrum Site Map by extending any menu to include
 additional functions, such as:

- Links to external websites

- Custom functions – provide access to custom functions created by
 Viewpoint's Special Projects Group

- Menus – add submenus to any location (for example, to group
 similar functions)

- Crystal Reports – place any crystal report on the menu, and when
 selected the report generates automatically

- File Download – place any file on the menu to
 download it when selected

- Email – enter a default recipient so that when selected, the
 default email client opens to send an email to that person
