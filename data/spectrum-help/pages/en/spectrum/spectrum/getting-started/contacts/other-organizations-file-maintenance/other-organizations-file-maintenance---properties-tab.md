# Other Organizations File Maintenance - Properties tab

This tab includes general information about the
 non-vendor/non-customer organization.
Fields/Buttons
Descriptions

Name
Enter a name for the organization, such as a trade name
 or government entity.

Remarks
Enter any remarks that you want to make available to
 others when this organization is referenced.

Attributes
Click to open the Build Contact Attributes List and
 assign a list of Attributes to this organization.
Highlight an Attribute in the All Organization Attributes
 column on the left, and click Select to add it to the current organization's
 attributes.
You can add and delete organization attributes in [Organization Attribute File Maintenance](/en/spectrum/spectrum/getting-started/contacts/other-organizations-file-maintenance/organization-attribute-file-maintenance).
Note:
 Spectrum allows you to delete attributes assigned to
 other organizations. If you Deselect a deleted attribute in the right box, it will
 not display in the left box.


Status
Assign a status to this organization using the available
 drop-down selections.

Primary communication

Telephone
Extension
Fax
Address
Enter the phone number and extension, the fax number, and
 the mailing address for this organization. At the Address field, a New
 Address window displays where you can enter address details that then
 display as the default address when you click OK.
