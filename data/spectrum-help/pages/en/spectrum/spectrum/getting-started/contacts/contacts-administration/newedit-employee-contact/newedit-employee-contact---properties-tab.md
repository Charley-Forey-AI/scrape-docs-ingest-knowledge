# New/Edit Employee Contact - Properties tab

A reference when completing the fields in this screen.
Fields
Descriptions

Contact name
On the Properties tab, you can maintain additional phone numbers and send email. Select one or more statuses to display.

Status

Title

Remarks

Attributes
The employee contact's assigned attributes display. You can create the contact attributes list in [Contact Attribute File Maintenance](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/contact-attribute-file-maintenance).

E-mail

Authorized/Not authorized to access Spectrum
Employee contacts have access to Spectrum if they have an assigned operator code, which displays to the right.
If Employee contacts do not have an assigned operator code, they do not have access to Spectrum.

Telephone
The list box displays the Employee contact's phone information.
The green check mark indicates the primary phone. To choose a different primary phone, highlight the row, and click Set as Primary.
Click New, Edit, or Delete to maintain more phone numbers.
Spectrum does not allow you to edit or delete the employee contact's work extension or mobile phone.
