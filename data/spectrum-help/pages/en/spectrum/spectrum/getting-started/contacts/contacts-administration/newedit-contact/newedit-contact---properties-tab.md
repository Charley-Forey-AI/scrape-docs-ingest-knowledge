# New/Edit Contact - Properties tab

Use the table below for reference when completing the
 fields in this screen.
Fields
Descriptions

Contact name
Click the drop-down arrow to edit the contact name using
 the Contact Name Detail window.

Status
Select either an Active or Inactive status for this
 contact.

Title
Enter the contact's job title and any notes in these
 optional fields.

Remarks

Attributes
Click to open the Build Contact Attributes List and
 define attributes for the contact.
Highlight an Attribute in the All Contact Attributes
 column on the left, and click Select to add it to the current contact's attributes.
You can add and delete the contact attributes in [Contact Attribute File Maintenance](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/contact-attribute-file-maintenance).
Note:
 Spectrum allows you to delete attributes assigned to
 contacts. If you Deselect a deleted attribute in the right box, it will
 not display in the left box.

E-mail
Enter or change the contact's email address. Click the
 email icon to open a message addressed to this contact. Click More to enter up to two
 additional email addresses.

Authorized/Not authorized to access Spectrum
Contacts have access to Spectrum if they have an assigned
 operator code, which displays to the right.
If contacts do not have an assigned operator code, they
 do not have access to Spectrum.

Default as a job contact on contracts / subcontracts?
Select to add this contact to the Job Contact List in
 Accounts Payable Subcontracts or Accounts Receivable Contract entry.

Telephone
The list box displays the contact's phone information.
 The green check mark indicates the primary phone. You can add a new phone
 number, edit, delete, or set a primary contact.
