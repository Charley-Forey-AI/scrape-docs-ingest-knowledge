# Contact Attribute File Maintenance

Use this page to define a standardized and validated list
 of classifications associated with contacts. Any given contact may be assigned an unlimited
 number of Attributes that help identify contacts associated with particular roles or
 responsibilities.

- Office Manager

- Estimator

- Billing Admin

- Payables Admin

- Project Manager

- Superintendent

- Field Admin

- Salesperson

- Purchasing Manager

- Shipping

- Architect

- Inspector

- CPA

- AGC

- Subcontractor

- Owner

- Executive

- Design Team

The contact attributes display alphabetically in the List box. You can
 add or delete contact attributes on this page. You can also delete attributes assigned to
 contacts. See the Attributes
 field in [New/Edit Contact](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/newedit-contact) for more
 information.
