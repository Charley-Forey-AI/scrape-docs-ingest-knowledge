# Other Organizations File Maintenance - Contacts tab

The list box shows the individuals associated with the current organization.
Fields/ButtonsDescriptions
NewSelect to open the [New/Edit Contact](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/newedit-contact) window.
Add ContactSelect to open the Search Contacts window, where you can add a new contact by selecting New below the Search bar.
EditSelect to open the [New/Edit Contact](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/newedit-contact) window.
Set as PrimarySelect the organization contact you want to mark as primary and then select the Set as Primary button. A green checkmark indicates this is a primary contact.
RemoveSelect to remove the contact.
StatusSelect to display organizations with contacts that are active, inactive, or no longer affiliated with the organization.
