# New/Edit Employee Contact - Jobs tab

Use the table below for reference when completing the fields in this screen.
Fields/Buttons
Descriptions

Add Job
This tab displays jobs that the Employee contact is associated with. You must have security authorization to add or remove jobs. Highlight the row and then click one of the buttons to add or remove a job.

Remove

Job Status
Select one or more statuses to display.

Grid
The table sorts by Job company, then by job code. You can only add or remove jobs that are valid in the current company. All of the columns are display-only, except for Job code (below).

Job code
Enter a valid job code. Click the drop-down arrow to search jobs. The rest of the columns are display-only.
