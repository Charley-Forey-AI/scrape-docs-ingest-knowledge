# Contacts Administration - Field Descriptions

Use the table below for reference when completing the fields in this screen.
Fields/ButtonsDescriptions
NewSelect to create a new contact.
Search BarEnter the item you want to search for. Select the drop-down arrow to open the [Advanced Contact Search Options](/en/spectrum/spectrum/getting-started/contacts/advanced-contact-search-options) window. Select Go to display data in the List box.
StatusSelect to change the status you want to display in the list box.
Selection Display PanelThe panel to the left allows you to see what is currently selected in the Advanced Contact Search Options window. You can save settings as a template for later searches.

- Template description: Select to select a search template. The search results automatically display. To create a new template, enter your search terms in the Search bar above, then select Save. You can provide this search template to all users.

- Tool icon: Select to edit the template name or delete the template. You can select to provide this template to all users.

- Modify: Select to open the [Advanced Contact Search Options](/en/spectrum/spectrum/getting-started/contacts/advanced-contact-search-options) window and change the current template.

- Save: Select to create a new template for the current selection. You can select to provide this template to all users.

- Display box: The box beneath the Modify and Save buttons reflects the selections on the [Advanced Contact Search Options](/en/spectrum/spectrum/getting-started/contacts/advanced-contact-search-options) window. The specific attributes you select display in this screen.

EditSelect the contact record that you want to edit. The [New/Edit Contact](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/newedit-contact) screen displays.
InactivateYou can set multiple contacts to an inactive status. The list box reappears after you select this button. If you leave the Status button at the Active default, the inactive contacts do not display.
You cannot inactivate an employee contact in this screen, but you can in the Payroll module.

DeleteYou can select multiple contacts to delete from Spectrum. A confirmation box displays. Select Continue to confirm the contact's deletion from vendor, customer, and job contact lists. The list box reappears.
You cannot delete an Employee contact in this screen, but you can in the Payroll module.

E-mailSelect at least one contact to send an email message to.
Logon? columnIndicates whether the contact has a Spectrum logon (and if present, whether it is Active or Inactive). Spectrum logons are required for non-Spectrum users to access the Subcontract Kiosk.
