# Images and Operator Security

When adding or opening documents in Spectrum, the software
 determines the operator's security level in this order:

1. The system looks at the operator drawer override,
 and if one exists, uses this security level.

1. The system looks at the operator cabinet override
 and if one exists uses this security level.

1. If no operator overrides exist, the software uses
 the default operator security level.
