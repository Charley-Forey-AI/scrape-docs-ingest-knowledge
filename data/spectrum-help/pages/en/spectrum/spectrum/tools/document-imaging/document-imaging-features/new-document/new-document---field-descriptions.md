# New Document - Field Descriptions

Use the table below for assistance when filling out fields on this screen.
Fields/Buttons
Descriptions

Cabinet
Enter a cabinet code, or select from a list of valid cabinets. You must have security access level 3 for both the cabinet and drawer entered on this screen.
The following cabinets are pre-defined:

- Archive Reports

- Cash Management

- Components

- Contracts

- Customer

- Employee

- Equipment

- Fixed Asset

- GL

- Insurance

- Items

- Job

- Projects

- Scale

- Site

- Subcontracts

- Titles

- Vendor

- WO
In addition, you can set up your own cabinets in the [Cabinet File Maintenance](/en/spectrum/spectrum/tools/document-imaging/document-imaging-features/cabinet-file-maintenance) screen.

Drawer
Enter a drawer, or select from a list of valid cabinets. You must have security access level 3 for both the cabinet and drawer entered on this screen.

Folder
If the Validate folders for this cabinet checkbox was not selected in the Cabinet File Maintenance screen, then New Documents will accept any input at the folder field and the document will get "filed" under this folder name. Otherwise you must enter a folder name that has been set up in the Cabinet File Maintenance screen.

Name

Keywords
Enter the keywords that will be used help you to retrieve this document at a later date. For example, you might want to include the type of document (for example, INVOICE) and the date of the document (for example, SEPTEMBER) and any other helpful information such as the company name, and so forth.

Create button
When you click the Create button, the Slider Bar opens and you will see buttons that will allow you to Attach a file, or look at (unassigned) files. File types supported by Document Imaging include: JPG, TIF, PDF, PNG, GIF.
Note: You must assign a name to the file before you attempt to create the image or you will receive an error message.
