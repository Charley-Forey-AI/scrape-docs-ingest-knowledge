# Upload a File

You can upload a file to the Document Imaging (DI) module without associating it to an already-existing record.
See this example video

1. Go to Document Imaging > New Document.

1. In the Cabinet field, select an existing one or create a new one.

1. In the Drawer field, select the module pertinent to the file.

1. In the Folder field, enter a folder name where you wish to store the file, or select from a list of existing folders.

1. In the Name field, enter a name to help identify the file.

1. In the Keywords field, enter any keywords that might help to locate and identify the file at a later date. This could be the document date, the type of document, and so forth.

1. Select Create.The system creates the record which will house your upload.

1. To attach the file in question to this new record, expand the Slider Bar and do one of the following:

- Drag and drop the file into the pane.

- Select Attach and navigate to and choose the file.

The file image appears within the expanded Slider bar pane.

1. Select Save.
