# Add a New Cabinet

1. Click the New button and enter a new cabinet name.

1. Click Continue.

1. In the New Cabinet window, enter an Override path (either a mapped drive volume or a UNC name). The override path is the location where the image files are stored and it will override the default path. Note that this is the path as it appears to the workstation.

1. Optional: If you want to create a list of folders that will be valid for this cabinet, select the Validate folder for this cabinet? checkbox.

1. Click the New button to create a list of the operators who can access this cabinet (and the valid folders, if applicable). Enter an Operator ID; the associated Logon ID will default.

1. Enter the Override security level for the drawer (levels 0-3) and click OK.
