# Change Document Path - Field Descriptions

Use the table below for assistance when filling out fields on this screen.
Fields/Buttons
Descriptions

Select images

Cabinet
Drawer
Folder
Enter or select the cabinet, drawer, and folder of the image files you want to update with a new directory path.

From/to date
Enter a date range of the image files you want to update with a new directory path.

Select 'from' current path
When this selection is set to Partial path, all selected documents in this directory and any subdirectories will be assigned to the new path.
When this selection is set to Specific path, a New path can be entered and all the selected documents in this directory (but NOT the subdirectories) will be assigned the new path.

Update 'to' new partial path
Enter the new directory path for the document; up to 200 characters are allowed.
Note: In order to move the image directory designation, the System Administrator or authorized operator must cut and paste the image files to the new directory using Windows Explorer or a similar file utility.

Buttons

Preview
Click this button to preview the Change Path Listing so that you can review it before proceeding to the Change Document Path Update screen. On the update screen, click Continue to complete the update.

Export
Click this button to distribute Spectrum reports to others.
