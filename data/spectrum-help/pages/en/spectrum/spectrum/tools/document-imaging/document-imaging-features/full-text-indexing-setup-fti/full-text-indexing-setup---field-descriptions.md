# Full-Text Indexing Setup - Field Descriptions

Use this table for reference when completing the setup
 screen.
Fields/buttons
Descriptions

Enable OCR service in all companies?
Select this checkbox to allow full-text indexing of
 scanned documents. Spectrum Full-Text Indexing (also known as Optical
 Character Recognition or OCR Indexing) provides you with the ability to
 search documents using any typed text on the scanned document--not just
 keywords that were entered at the time the image was added to the
 software.

Total documents processed
This is the current number of documents in the
 full-text index .

Documents pending OCR
This is the number of documents that have yet to be
 indexed.

Total errors
This is the total number of OCR errors.

Schedule button
Click this button to open the Service Schedule window
 and establish a timeframe for when you would like to run the OCR indexing
 process, including which days of the week and what times it will run. By
 default the schedule will not run on weekdays between 7 a.m. and 7
 p.m.
View example

Retry button
Select a document from the list and click this button
 to try indexing it again.

View button
Select a document from the list and click this button
 to preview the image file in the Slider Bar.
