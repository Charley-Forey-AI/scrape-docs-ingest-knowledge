# Purge Documents

This is a starting screen for an update that will delete
 the links and any associated image files from a directory. After selecting the file(s) you
 wish to purge, a listing report will print.
When you click the Preview button, the report listing will only show the Cabinet, Drawer,
 Folder, and Description; no file-specific info will display on the report. Be sure to
 review this report carefully before proceeding to the [Purge Documents - Verify](/en/spectrum/spectrum/tools/document-imaging/document-imaging-features/purge-documents/purge-documents---verify) screen.
