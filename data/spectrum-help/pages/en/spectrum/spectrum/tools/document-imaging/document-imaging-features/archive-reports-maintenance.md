# Archive Reports Maintenance

The archiving process allows you to specify descriptions
 and keywords for reports and then save the reports into a selected cabinet, drawer, and
 folder. This process allows for easy report retrieval at a later date.

- Archived reports are stored by company, regardless of cabinet or
 drawer, to assure that the Document Imaging > Search Documents feature only displays those documents generated from the current
 company.

- Because archived records are stored by the archive date (year and
 period), be aware that when searching archived records that the archive date may not
 be the same as the record date; for example, if you archived the Year End Financial
 Statement (dated 12/31/2012) on 3/12/2013, the archived record is stored in folder
 2013, period 3.

- You must be granted access to archive reports in the System Administration > Security > Operator Maintenance screen in order to take advantage of this feature.

- Administrators may also use the Archive Reports Maintenance screen
 to flag any reports that may not be archived for security reasons.

- Settings in this screen are company-specific.

- Use the [Copy Archive Report Parameters](/en/spectrum/spectrum/tools/document-imaging/document-imaging-features/archive-reports-maintenance/copy-archive-report-parameters) to copy override settings to another company.

- By default, the list of reports displays sorted primarily by
 cabinet, then by drawer, folder, and screen code. The Status button can be used to filter
 the items that display in the list box.
