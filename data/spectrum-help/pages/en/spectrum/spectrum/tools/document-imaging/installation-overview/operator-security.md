# Operator Security

Select the operator whose settings you wish to change, then click the Settings button in the lower left of the screen (if it is not enabled, press Enter to move to the Operator settings section of the screen). Use this window to edit the specified operator's authorization levels.
Note: When operators have limited security access in
 Document Imaging (such as permission to only view images), and they call up an attachment;
 if, for example, that attachment is an Excel spreadsheet, any modifications the operators
 are allowed to perform (based on network permissions) will be saved regardless of the
 Document Imaging security limitations.
