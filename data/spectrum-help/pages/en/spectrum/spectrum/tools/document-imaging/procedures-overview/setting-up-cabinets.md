# Setting Up Cabinets

Follow these steps to create new image file
 cabinets.

1. Go to Document Imaging > Cabinets.The Cabinet File Maintenance window opens.

1. In the Company field, enter a
 company or type ALL to make this cabinet available in all
 companies.

1. In the Cabinet field, type a
 cabinet name or select and edit one from the available drop-down list box.

1. In the Description field, enter
 a description to help identify this cabinet.

1. The Path field is an optional
 field that allows you to set up different cabinet image paths for different companies
 (for example, the CONTRACTS cabinet in company XYZ may have a different directory
 path than the CONTRACTS cabinet in company ABC). Enter the directory path where the
 cabinet images should be stored. If you leave this field blank, the default image
 directory path as set up in the Document Imaging Installation
 screen will be used.

1. Select the Validate folders for this cabinet checkbox
 if you want the software to ensure that a particular folder is valid for this cabinet
 when new documents are added. If you choose to select this checkbox, you will need to
 use the Valid Folders button to add valid folders for this
 cabinet.

1. Select the Operator Security
 button and press F5 to grant (or restrict) individual operator
 access to this cabinet.

- 0 - No
 access

- 1 - Read only access

- 2 - Annotate access

- 3 - Write access

Note: If you choose not to complete
 this window, no one, including yourself, will be able to use the cabinet you just
 created.

1. Select OK to save your new
 cabinet setup and return to the Site Map screen.

You must create or assign a drawer to this new cabinet in order to use the cabinet.
