# Attaching Documents

Learn how to attach documents.

1. On the Site Map screen, navigate to the module and
 screen where you want to attach an image. For example: Job Cost  >  Maintenance  >  Jobs, and enter a job number.


1. Depending where you are in the software, the Slider Bar will display automatically or you can access it from the Info Bar by selecting Documentation  >  Images.

1. Select New to open the New Document screen and enter the transaction details, then select Create.

1. In the Slider Bar, select the Attach button and select the image file you want to associate with the transaction you just created , or you can drag-and-drop an image from your desktop into the Slider Bar. If you want to attach an image from an email, first drag the image file to your desktop and then drag it from your desktop into the Slider Bar.

1. Select Save to complete the process of associated the image file with the transaction.
