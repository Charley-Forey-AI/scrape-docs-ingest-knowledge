# Sharing Search Results

Spectrum allows you to email image file search results.

1. Go to Document Imaging > Search Documents.

1. Enter your search criteria and select Search Documents.

1. Select the Listing button to
 generate a report of the search results.

1. In the Print Report window,
 select Distribute to create an executable report
 that bundles all of the images related to the search results into a
 mini-application.

1. In the Export Report and Image Files for
 Distribution window, select Continue.

1. When the Download Distribution
 File window displays, select either Download Now or Keep File on Server.

The compressed file (.zip file) has now been saved to
 your desktop or hard drive.
Attach these items to an email. Include all the files from the directory in the
 email or the image links will be disabled.
