# Add a Batch of Unassigned Documents

1. Select an operator code from the available drop-down list.

1. Click the New or Edit buttons and the New (or Edit) Batch of Unassigned Documents window displays. Use this window when scanning in any number of screens into a new batch.

1. After the fields have been completed, the transaction description will be used to identify the transaction in the Slider Bar. At this point, you may upload image files in the Slider Bar using the Attach button.Fields/Buttons
Descriptions

Batch type
Select a batch type. Your selection will determine the defaults available in the Batch properties section of this screen.

Batch properties

Description
After the batch type has been selected, this field will default to say "BATCH TYPE - current date, current time, and the current operator code."

Cabinet and Drawer
The following pre-defined cabinet / drawer combinations will default based on the Batch type selection:

- Vendor / Unassigned Invoice

- Employee / Unassigned Time Cards

- Batch / Other

Changes are allowed. Use the available drop-down arrow if you wish to select a different cabinet or drawer.

Folder
This field is hard-coded to say "UNASSIGNED BATCH."

Current Operator
This optional field allows you to assign this batch of images to a different operator using the available drop-down list. If you choose to leave this field blank, the batch will remain assigned to your operator.
In order for operators to use documents in the batch, they will need DI security level 3 (including override for Unassigned Invoices Drawer) in the company where the Batch is created as well as the company where the batch is being used.

Remarks button
This button only displays in the Edit Batch window. It allows you to enter any notes pertaining to the selected image batch and the reason for the changes being made.
