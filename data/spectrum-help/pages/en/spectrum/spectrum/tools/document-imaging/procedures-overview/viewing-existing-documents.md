# Viewing Existing Documents

To view an existing document that has already been scanned,
 just follow these steps.

1. Find the transaction in Spectrum that you want to
 view. For example, if you are drilling down to an A/P invoice, go to
 Invoice Entry and enter the invoice number.

1. If the Slider Bar has a number greater than zero in the transaction header, then an image is associated with this screen. Click the transaction name a drop-down list of associated files displays.

1. Click on one of the records from the list. Now, you are ready to view, print, annotate, edit, or otherwise work with the image.
