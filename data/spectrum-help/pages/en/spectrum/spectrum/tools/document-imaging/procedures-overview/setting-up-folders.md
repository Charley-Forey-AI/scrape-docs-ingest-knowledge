# Setting Up Folders

How and where you can set up image folders depends on whether
 or not you want the software to validate the image folders for the cabinet where they will
 be stored.
The option to validate folders is provided in the Document Imaging  >  Cabinet File
 Maintenance screen. If the Validate
 folders for this cabinet? checkbox is selected, all documents must
 go into a pre-defined folder. If cleared, folders may be entered without
 validation. Normally, this option should be selected because this will provide
 control over folder names.
Important: For the
 standard cabinets, folders are always validated.
If you select the Validate folders for this cabinet checkbox in
 the Cabinet File Maintenance screen:

1. Click the Valid Folders
 button.

1. In theValid Folders window,
 click New and type a
 name for the new folder.

1. Type a Description to help identify the new
 folder.

1. Press Enter to create
 additional folders or click OK to save your changes and
 return to the Cabinet File Maintenance screen.If you do NOT select the Validate folders for this cabinet
 checkbox in the Cabinet File Maintenance screen:

1. On the Site Map screen,
 click Document Imaging and then select the New
 Document menu option.

1. Complete the Cabinet and
 Drawer fields using the available drop-down list
 boxes.

1. In the Folder field, type
 the name for the folder you wish to create.

1. Complete the rest of the screen click
 OK to save your changes.
