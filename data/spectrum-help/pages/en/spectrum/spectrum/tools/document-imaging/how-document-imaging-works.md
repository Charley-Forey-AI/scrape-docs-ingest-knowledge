# How Document Imaging Works

Document Imaging uses AJAX technology (Asynchronous Java script and XML).
This is the same technology used by Google to do Google Maps and display thumbnail pictures. Unlike the older HTML technology, it allows the web client to communicate with the server without having to send a whole screen at a time, so you are able to begin looking at your images and documents without having to wait for the whole file to be transmitted.
