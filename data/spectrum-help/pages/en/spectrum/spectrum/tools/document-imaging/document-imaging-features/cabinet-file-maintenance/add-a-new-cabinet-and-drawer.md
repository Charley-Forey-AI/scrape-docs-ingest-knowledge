# Add a New Cabinet and Drawer

1. Click the New button and enter a new cabinet name and drawer name. If a cabinet row is highlighted when you click New, the new cabinet name will default from the highlighted row, but it may be overwritten.

1. Click Continue.

1. In the New Drawer window, enter an Override path (either a mapped drive volume or a UNC name). The override path is the location where the image files are stored and it will override the default path. Note that this is the path as it appears to the workstation.

1. Click New to create a list of the operators who can access this cabinet/drawer combination and enter an Operator ID; the associated Logon ID will default.

1. Enter the Override security level for the drawer (levels 0-3) and click OK. More infoSelect the security level you want to assign to future operators when viewing or editing images. These are as follows: 0. None. The user cannot view or change image files. The Slider Bar does not display. 1. Read only. The user can only view image files. The Slider Bar displays but the Annotation and Page Manipulation options are not available. 2. Read with Annotations. The user may make annotations on files. The Slider Bar displays but the Page Manipulation options are not available. 3. Read / Write with Annotations. The user can add, change, delete, or annotate image files. The Slider Bar displays along with the Scan, Attach, and Batch buttons. The Annotation and Page Manipulations options are also available. Only the System Administrator should change a user's default security level.
