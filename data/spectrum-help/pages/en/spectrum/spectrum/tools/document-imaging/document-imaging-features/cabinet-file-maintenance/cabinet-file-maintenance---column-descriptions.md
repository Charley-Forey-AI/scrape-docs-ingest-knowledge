# Cabinet File Maintenance - Column Descriptions

Columns/Buttons
Descriptions

Row type
Specifies either "Cabinet" or "Drawer."

Cabinet name

Drawer name

Opr override
This column says "Yes" if any operator in the current
 company has an override security record for the cabinet or drawer in this
 row.

Effective path
This column shows the image storage path based on the
 following hierarchies.

- For Cabinet rows:

1. Shows the Override path, if
 specified

1. Shows the Standard path from
 Document Tracking Installation

- For Drawer rows:

1. Shows the Override path, if
 specified

1. Shows the associated cabinet's Override path, if
 specified

1. Shows the Standard path from
 Document Tracking Installation

Path basis
This column further describes where the path shown in the
 Effective path
 column is from.

Override path
The first 60 characters of the cabinet or drawer override
 path display, if an override path has been specified.

Type
This column displays Standard if the cabinet or drawer is
 one of the pre-defined items that ships with the software, or it displays
 Custom if the cabinet or drawer has been set up by your System Administrator
 to better accommodate your company's needs.
