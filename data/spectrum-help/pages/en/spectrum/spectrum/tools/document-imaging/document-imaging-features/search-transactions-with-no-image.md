# Search Transactions with No Image

Use this screen to search for and display a list of all
 transactions within a selected range that do not currently have an image attached.
Once the list displays you can conveniently attach images to
 the transactions using the Documenting Imaging Slider Bar. As images are attached to
 transactions, those rows will continue to appear in the list box for review (and to add
 additional documents if desired) until you select to 'Refresh' the list.

1. On the Site Map, select the
 Document Imaging  >  Transactions with No
 Image option.

1. The Search Transactions window displays. Complete some of the search
 criteria fields.

1. Click OK and a list of transactions will display, along with the Document
 Imaging Slider Bar.

1. Select a transaction in the list box and click
 Attach in the Slider
 Bar. Once attached, images are committed but remain in the list box until it is
 manually refreshed so you can review the image, add annotations, attach more images
 to the same transaction.

1. Click Refresh to update the list box so that only those transactions
 without attached images display.[How Document Imaging Works](/en/spectrum/spectrum/tools/document-imaging/how-document-imaging-works)
