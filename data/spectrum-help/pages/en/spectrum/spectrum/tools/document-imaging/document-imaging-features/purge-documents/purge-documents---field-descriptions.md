# Purge Documents - Field Descriptions

Use the table below for assistance when filling out fields on this screen.
Fields/Buttons
Descriptions

Cabinet
Enter or select a cabinet name.
The following cabinets are pre-defined:

- Archive Reports

- Cash Management

- Components

- Contracts

- Customer

- Employee

- Equipment

- Fixed Asset

- GL

- Insurance

- Items

- Job

- PM

- Projects

- Scale

- Site

- Subcontracts

- TM

- Titles

- Vendor

- Work Order

All cabinets and drawers are company-specific.

Drawer
Enter or select a drawer.

Folder
Enter or select a valid folder.

Created by
Enter an operator code.

From/to create date
Enter a creation date range. All image files created within this range and meeting other selection criteria will be purged.

Image description
Enter part of the document name you would like the software to search for. For example, if the document name was "INVOICE 524," you could simply type INV.

Contains the words
Click the drop-down arrow to display the Advanced Full Text Search window where you can further narrow down your search criteria for the records you want to include in the purge. In this window you can enter specific words and phrases that you want to include or exclude, and you can limit the number of records the search will yield.

Keywords
Enter one or more of the keywords you would like the software to search for.

Include records containing
Select to search on all of the keywords or any of the keywords entered.

Select path

Specific path only?
Specific path
This checkbox and field are useful when you want to remove image files that are pointing to a non-existent physical file (for example, when an image was stored on a local drive on a workstation that is no longer in use).
In the rare case where a transaction has images stored in different paths, if one image does not match the specific path selection, the image will not be purged.

Buttons

Preview
Click this button to preview the Purge Listing so that you can review it before proceeding to the DI Purge Documents - Verify screen. Once everything looks okay, click Continue to complete the update.

Export
Click this button to distribute Spectrum reports to others.

My Reports
Click this button to copy Spectrum reports so that you can customize them by adding or editing titles, files names, security levels, and the order in which the reports display.
