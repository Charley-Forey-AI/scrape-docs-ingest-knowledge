# Batch Documents

Use this screen when you want to attach files in batches
 for entry into Spectrum at a later time. Your System Administrator can designate whether
 individual operators can only view batches assigned to themselves, or whether they can view
 others' batches as well (operators who can only view their own queue will not have access to
 the Operator field; their own ID
 automatically defaults).
In conjunction with this feature, the following pre-defined cabinet /
 drawer combinations are included with the software:

- Vendor / Unassigned Invoice

- Employee / Unassigned Time Cards

- Batch / Other
