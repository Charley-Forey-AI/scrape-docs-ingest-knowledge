# List of Crystal Reports that Offer Image Links

The table below shows a list of the Spectrum reports that offer links to any associated images when the reports are previewed. Other important features of these reports include:

- Image links always display whenever an image is available.

- A single link is provided per transaction, even if multiple images are present.

- You must be logged into Spectrum for the Document Imaging links to open images.

- Image links will open in a new browser tab for your system default browser. Set your favored browser in the browsers Options or from your system Default Programs. (Start > Default Programs > Set your default programs).

- If you are using Spectrum in a browser other than your system default browser, then the image link may generate an error message.Note: If using Google® Chrome® browser, the Chrome .pdf viewer does not support hyperlinks so the Document Imaging images cannot be viewed.
