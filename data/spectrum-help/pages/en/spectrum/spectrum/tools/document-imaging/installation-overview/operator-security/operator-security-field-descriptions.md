# Operator Security Field Descriptions

View the Operator Security field descriptions when completing
 fields on this screen.
Fields/Buttons
Descriptions

Default security level
Select the security level you want to assign to
 this operator when viewing or editing images. For operators set up
 before the Document Imaging system was installed, the default level
 will be set to 1 - Read only.

Allow this operator to archive reports?
Select this checkbox if you want this operator to have the ability to
 save previewed reports; otherwise leave this checkbox clear.
 Archived documents are stored by company, regardless of cabinet or
 drawer, to assure that Document Imaging > Search Documents displays only those documents generated from the
 current company.

Allow this operator to access other operators'
 unassigned lists?
Select this checkbox to grant this operator
 access to all unassigned document batches (in other words, including
 those batches assigned to other operators).

Security overrides
The list of security overrides for the document
 images displays.

Edit/New/Delete buttons
Use these buttons to enter a new, or make changes
 to an existing, security override. The New Operator Security
 Override window allows you to select a cabinet/drawer combination
 and then apply a operator-based security level to this setup.
 Security levels range from None (allowing no
 access) to Read/write
 with annotations.

Restore button
Click this button to reset the selections and
 search results to the original defaults (usually blank or ALL).
