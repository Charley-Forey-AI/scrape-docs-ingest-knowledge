# Using Search Documents

The steps provided here outline how to select the search
 criteria you want to use when trying to locate an image file.

1. On the Site Map screen, click
 Document Imaging > Search Documents.

1. Enter the Cabinet
 associated with the image you are looking for. For example, if you are looking
 for an employee timecard, select the Employee cabinet.

1. Enter the Drawer associated
 with the image you are looking for. Continuing with the example in step 1,
 select the Timecards drawer.

1. In the Folder field, select
 the employee whose timecard you want to view.

1. Optional step: If you know who scanned the
 image or when the file was scanned into the software, you may choose to complete
 the date fields. For example, if you know that the timecard you are looking for
 was scanned sometime in February, the date fields would be very useful.

1. Leave the Description and
 Keyword fields blank unless you know the naming
 convention used for the file you are seeking. If you do enter keywords, in the
 Include records containing section, select All of these keywords or Any of these
 keywords.

1. Click the OK button to
 review your search results.

1. If you're not seeing the results you were
 expecting, click the Search button to open the Search Images
 window and enter new search criteria.
