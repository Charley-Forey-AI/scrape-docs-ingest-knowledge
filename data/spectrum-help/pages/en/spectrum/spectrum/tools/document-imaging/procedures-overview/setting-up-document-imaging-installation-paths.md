# Setting Up Document Imaging Installation Paths

Before you can begin using Spectrum Document Imaging
 transactions, you need to set up the installation parameters. This tells Spectrum
 where the image files will be stored and whether any special operator security is
 designated for specific file cabinets.

1. On the Site Map screen, click
 System
 Administration  >
 Installation  >
 Document Imaging.

1. In the Path
 field on the Properties screen, enter the path for the
 folder that will serve as the default image storage
 directory. In the simplest case, you can put all image files
 for all cabinets and all companies in one network directory.
 However, if you put various types of documents into one
 directory, you will not have the ability to control access
 to these documents using network directory permissions.
 Also, if you have many image files, you will end up with a
 very large number of files in this directory.

1. Complete the remaining fields on
 the [Document Imaging Installation - Properties](/en/spectrum/spectrum/tools/document-imaging/installation-overview/document-imaging-installation---properties) screen.
