# Archive Reports Maintenance - Field Descriptions

Click the New button and use the New Archive Report Parameters window to add default
 cabinet, drawer, folder, description and keywords for the report specified in the Screen code field, and use the Edit
 Archive Report Parameters window to make changes to existing defaults.
Fields/Buttons
Descriptions

Screen code
Enter the screen code, or select from a list of
 screen functions.
When editing report parameters, the screen code
 defaults from the record you selected from the Archive Reports
 Maintenance screen and no changes are allowed.

Default settings

Default cabinet
Select the cabinet to archive this report to for
 this company.

Default drawer
Select the drawer to archive this report to for
 this company.The default drawer is the two-character module
 abbreviation (for example, AR = Accounts Receivable).

Default folder
The Default folder is the Spectrum function name,
 similar to the Screen
 code, but with the period replaced with an
 underscore.

Default description
Enter a description for the archive report in this
 field. The default description is the full Spectrum screen name.

Default keywords
Enter any keywords for the archive report that
 will be helpful in retrieving it at a later date.

Prevent overrides
These checkboxes allow you to prevent
 unauthorized activity on a report-by-report basis.

Save status
Designating a report as Disallowed means
 it cannot be archived. The save status assignments made here are
 also used in conjunction with the toggle button on the main Archive
 Report Parameters screen to filter the list of records that
 displays.

Buttons

Restore Defaults
Select a record in the list box and click this
 button to return the default cabinet, drawer, folder, description,
 and keywords to the original default values. By default, all
 archived reports are stored in a cabinet titled Archived
 Reports.

Copy
Select a report and click this button to display a
 [Copy Archive Report Parameters](/en/spectrum/spectrum/tools/document-imaging/document-imaging-features/archive-reports-maintenance/copy-archive-report-parameters) that allows you to change the defaults
 for selected report in the destination company.

Status
Use the Status filter
 button to determine which records will display in the list of
 reports.
