# Attaching Images

Spectrum allows you to attach most types of images
 (documents, spreadsheets, .PDFs, .TIFs, and so forth) to Spectrum records.
In order to attach files, the operator must be set up with
 the appropriate security access in the [Document Imaging Installation - Security](/en/spectrum/spectrum/tools/document-imaging/installation-overview/document-imaging-installation---security).

1. On the Site Map screen,
 navigate to the module and screen where you want to attach an image. For example:
 Job Cost >  Maintenance >  Jobs, and enter a job number.

1. Depending where you are in the software, the
 Slider Bar will display
 automatically or you can access it from the Info Bar by clicking Documentation >  Images.

1. Click New to open the New
 Document screen and enter the transaction details, then click
 Create.

1. In the Slider Bar, click the Attach button and select the image file you want to associate with
 the transaction you just created , or you can drag-and-drop an image from your
 desktop into the Slider
 Bar. If you want to attach an image from an email, first drag the image
 file to your desktop and then drag it from your desktop into the Slider Bar.

1. Click Save to complete the process of associated the image file with the transaction.
