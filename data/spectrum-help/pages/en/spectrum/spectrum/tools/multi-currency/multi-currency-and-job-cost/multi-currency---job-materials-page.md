# Multi-Currency - Job Materials Page

The Job Materials page will display Purchase Orders in the vendor currency which may be different than the reporting currency. This means that there will be different currencies all displayed on the page.
When a purchase order is created in a non-reporting currency, it will be noted in the 'Currency' column of the screen. This is also the way that the Subcontracts page presents multi-currency subcontracts.
The key point here is that this page is truly a Purchase Order and Inventory Control screen that happens to be accessed in Job Cost. The values here will display using the items' currency.
Tip: Use the 'Currency' column to sort the Account listbox. To group accounts with the same currency together, sort the Accounts listbox by 'Currency' code. Remember that you can also drag the column over to the left so that it is easier to view.
