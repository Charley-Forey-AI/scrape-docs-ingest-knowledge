# Multi-Currency and Job Cost

Job Cost will track projects using the reporting currency.
 Open commitments will be translated into the reporting currency using the current FX
 rate.
Reporting Currency: All Job Cost inquiries and reports shall use the
 reporting currency.

## Processing Rules

- Transactions originating in Job Cost will continue to be in the
 reporting currency.

- Transactions posting to Job Cost from Accounts Receivable,
 General Ledger and Payroll will be recorded using the reporting currency, which is
 the currency of these source modules as well.

- Transactions posting to Job Cost from A/P and Cash Management
 will be recorded using the reporting currency, even if the source transaction is
 tracked in that other module in a different currency.

- Open commitment transactions from purchase orders will be
 translated into the reporting currency using the current exchange rate.

- Unposted transactions posting to Job Cost will be translated
 into the reporting currency using the current exchange rate.
