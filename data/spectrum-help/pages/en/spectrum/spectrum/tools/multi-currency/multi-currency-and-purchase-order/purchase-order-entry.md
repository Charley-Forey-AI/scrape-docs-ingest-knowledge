# Purchase Order Entry

The vendor will determine the currency used on the purchase
 order. The vendor currency code will display in Financials section as well as on the Vendor
 Info Bar.
Important: The items standard cost will default into
 the detail line using the reporting currency. This means that the user will have to
 change the value into the vendor’s currency in order to record the agreed upon
 purchase cost of the item.

## Import Purchase Orders

When Multi-Currency module is present, the Import Purchase Orders
 option will store the vendor’s override currency in the Purchase Order Header Table
 as if it was added in the entry screen.
