# Setup a New Currency Code

You can setup a new currency code by selecting
 New and going to the New Currency Code
 window.

1. On the Currency Code Maintenance screen, click New. The New Currency Code window displays.

1. Enter a new 3-character Currency code and Description.Example:

1. Complete the Installation override G/L account fields. These accounts will be used in place of the defaults on the installation screens. If a field is left blank the transaction will use the default installation G/L code. When a transaction uses a non-reporting currency, the system will replace the standard installation G/L accounts with these G/L accounts. At the end of each accounting period, the Controller will revalue these currencies using the month-end Foreign Exchange (FX) rate. Controllers can then make a reversing J/E to revalue the amounts.
