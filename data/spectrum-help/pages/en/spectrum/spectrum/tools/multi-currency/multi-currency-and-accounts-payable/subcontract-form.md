# Subcontract Form

The vendor currency will display on the 'Total' row when
 different from the reporting currency. The currency code will not display when the
 subcontract is written in the reporting currency.

## Recording Subcontract Changes when Vendor Currency is not Reporting
 Currency

When creating a Change Request that will result in a subcontract
 change, it will be necessary to enter the 'Estimate amount' in the reporting
 currency and the 'Sub CO amount' field in the vendor's currency.
This will allow Job Cost, Change Orders and Change Requests to
 process using the reporting currency, while providing the ability to pay the
 subcontractor using their currency.
