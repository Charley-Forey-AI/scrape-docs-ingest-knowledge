# Void Check Entry

The Void Checks process functions exactly the same with or
 without Multi-Currency enabled. The system will void the transaction exactly as it went in,
 regardless of currency.
When processing void checks, the software will automatically reverse
 any gain or loss from currency fluctuations posted to the General Ledger when the check
 was originally paid.
All invoices associated with the now voided check will be open and
 available for payment.
