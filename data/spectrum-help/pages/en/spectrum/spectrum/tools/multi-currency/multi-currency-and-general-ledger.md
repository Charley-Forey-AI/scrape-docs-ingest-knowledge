# Multi-Currency and General Ledger

General Ledger will track financial information using the reporting currency.
Reporting Currency: All General Ledger inquiries and reports shall use the reporting currency.
VAT Processing: The system will continue to work the way VAT works today. No changes are needed. If applicable, create tax jurisdiction VAT codes with the appropriate G/L codes. (For example, in England, set up VAT-GB, and in the Eurozone, VAT-EURO.)
