# Enter Time - Unit Quantities

The Unit Quantities page allows you to update your phase unit quantity using a mobile device.

- In order for a job's phases to display in Unit Quantities in the mobile app, the Spectrum application must be set up with quantity estimates on those phases.

- In order to enter past quantities, Unit Quantities must be turned on in Company Admin. Tap the plus symbol on the right-hand side to change the displayed date.

Your estimated quantity budgets for each phase code are visible on the mobile device via the main Unit Quantities page. The "Add" total under each phase code displays any entered quantities for the date selected, job-to-date quantity, and the resulting balance.

To add unit quantities:

1. Select the phase code you want to update.

1. Tap the Additional Qty line and enter the quantity completed for this transaction.

1. (Optional) Attach a note to your quantity entry.

1. Save your entry by tapping the disk icon in the top-right corner.
