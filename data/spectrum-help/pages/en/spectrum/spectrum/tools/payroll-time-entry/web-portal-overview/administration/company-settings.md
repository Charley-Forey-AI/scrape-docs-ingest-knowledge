# Company Settings

The Company Settings screen configures the parameters and default settings used when setting up new jobs.
The Customer ID and Customer PIN values are registered by Viewpoint during the system setup.

- Web Service URL: This is the URL to connect to your company's web server via Spectrum Data Exchange (SDX).

- User: Web service user account used by SDX.

- Password: Web service password used by SDX.

- Use: Option to use either Spectrum Divisions or Cost Centers if Spectrum Enterprise Management is being used.

- Record Equipment Time: If checked, Spectrum Equipment module will be synched and used for entering equipment time.

- Record Job Quantities: If checked, Job Cost Quantity records can be entered via mobile device for a selected phase.

- Disable Add Employees from Device: Select this checkbox to disable the option to add employees from the mobile app.

- Disable Add Equipment from Device: Select this checkbox to disable the option to add equipment from the mobile app.

- Disable Edit Timecard When Batched: Select this checkbox to prevent timecards from being edited or deleted once the batch has been posted to Spectrum. Clearing the batch makes the records available to re-submit and could result in duplicate timecards in Spectrum.

- Enable Signature Approval: Select this checkbox to activate new menu option on web and mobile devices to collect signatures. Text entered on the Signature Agreement window will display on the Time Approval page in the mobile app.

- Cost Types: Specify cost types for equipment and labor. Up to three characters are allowed.

- Default Job Time Reporting Options: These are defaults used when setting up a new job and can be overridden once the job has been setup.

- Assign Job Employees to Project Manager

- Allow Wage Code Override: Allows mobile user to modify the wage code to any of those available on the selected job.

- Allow Overtime/Doubletime: Allows mobile user to select if labor time should be recorded as Overtime or Doubletime from the mobile device

- Time Rounding (Minutes): Set the company's time rounding preferences (for example, 1/4 of an hour or 1/6 of an hour).

- Current Payperiod Only: Select this option to limit time card entry to the current payperiod only.

- Don't Copy Wage Code or Pay Level: When copying timecards from one employee to another, select this option to prevent copying the wage code and pay level from the original employee.

- Enter Start/Stop Time: This is your default time tracking and can be set to collect total shift hours only or you can set the system to collect start, stop, and downtime (meal break).
