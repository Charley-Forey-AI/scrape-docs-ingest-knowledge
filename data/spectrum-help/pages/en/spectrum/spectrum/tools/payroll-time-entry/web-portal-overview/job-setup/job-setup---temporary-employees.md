# Job Setup - Temporary Employees

To add Temporary Employees to a job, check the box in the
 bottom left corner of the "Add/Remove" Employee dialogue box. This will allow you to enter the
 last name and last four digits of the employee's social security number. This will create a
 temp number and allow time to be entered for the employee from mobile devices. Once the
 employee is entered in Spectrum the web portal will be updated with the new Spectrum employee
 number by matching the last name and SSN.
