# Generate Web Portal Reports

Generate and download preconfigured resource, timecard, exception, and custom reports with the Reports tab in the Payroll Time Entry (PTE) web portal.
These instructions assume you have already configured the PTE web portal and the PTE mobile app, and have set up jobs in the PTE web portal.
See this video example:

1. In the PTE web portal, select the Reports tab.The Category and Report drop-down menus display.

1. Select an option in the Category drop-down menu.

- Resources - Summarizes hours by phase code.

- Timecards - Offers equipment timecards and employee timecards organized by company, phase, or job.

- Exceptions - Highlights employee hours that exceed a user-defined threshold such as flagging an employee who has entered more than 10 work hours in a day.

- Custom - Allows you to select a custom report which you have already configured.

1. In the Report drop-down menu, select a report.

1. Select the Load button.Parameters display on the right side of the screen.

1. To filter data, select report parameters.

1. Select the Refresh button to generate a report.A dynamic report displays.

1. To download the report, select the cloud icon.Export format options display.

1. Select an export format.

The report downloads.
