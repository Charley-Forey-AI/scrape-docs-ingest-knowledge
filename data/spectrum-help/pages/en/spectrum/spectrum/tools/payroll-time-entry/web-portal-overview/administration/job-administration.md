# Job Administration

Use this screen if you need to set a master or sub-job to
 'Active'. This would be necessary when the job status has been marked 'Closed' or 'Inactive'
 in Spectrum, and then changed back to 'Active'.
The Managed
 checkbox allows you to see the jobs that are currently set up in Job Setup and assigned to
 one or more project managers.
Find the job in the list and make sure the Active checkbox is selected so that the
 master and sub-job information will sync properly.
