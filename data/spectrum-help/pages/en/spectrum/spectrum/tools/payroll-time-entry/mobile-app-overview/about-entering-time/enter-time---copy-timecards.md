# Enter Time - Copy Timecards

Many times when entering labor information, you'll want to enter the same information for a group of employees. There are two ways to copy timecards: Copy Labor and Copy Days.

## Copy Labor

This is a two-step process, first you enter time for an employee for the day and save the timecard. Second, click the Copy Labor icon to open the Copy Labor screen. Select the employee you wish to copy from; a green check mark displays next to the selected employee. Next, tap on one or more of the employees listed in the scrollable area at the bottom of the screen. Once all the employees you wish to clone from the selected timecard are checked, click the Save icon and those employees will now have the same values for that date. You can still edit those timecards before syncing to the cloud.

## Copy Days

Many times you have jobs that utilize the same crew performing the same tasks and span multiple days. Tap the Copy Days icon and you are presented with a list of previously-synced employees for a particular day. Simply select the day you wish to clone from and the day you wish to clone to by tapping on the dates and selecting the desired values. Next, tap on each employee you wish to copy from one day to the target date and click the Save icon. The employees now have the same values set for the new date. Again, you may still update any of the cloned timecards should you wish before syncing to the cloud.
