# Manage Jobs

Use the Manage Jobs page to add or remove employees and set defaults at the job level.
Before entering the Manage Jobs page, first select the job you want to work with. It's not possible to navigate from job to job from this page.
Note: Check for and delete Timecards before removing an employee.All un-synced Timecard records will sync to the portal the next time an app sync is performed, even those of an employee you deleted prior to the sync.
