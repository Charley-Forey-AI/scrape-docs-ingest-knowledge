# Cloud Sync Configuration

The Cloud Sync Conguration screen contains tabs for
 selecting companies for your tenant, appropriate equipment statuses, cost categories and
 add-ons for your companies and then sync these selections from Spectrum to the PTE
 server.

- Use the Companies tab to select Job, Equipment and Employee companies.
 Click the Import Pay
 Periods button if you have selected a company that uses the
 Spectrum pay calendar.

- Use the Settings tab to select the Equipment status, Equipment cost
 category and add-ons for the selected companies. This is a good way to filter out
 items you don't want to see in the PTE mobile app.

- After you have selected your companies and settings, go to the
 Console tab and click
 the Request Full Sync
 button to initiate a sync from Spectrum to the PTE server. Sync results will
 display below.
