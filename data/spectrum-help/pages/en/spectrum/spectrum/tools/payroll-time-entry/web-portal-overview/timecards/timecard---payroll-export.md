# Timecard - Payroll Export

The Payroll Export tab allows you to create batches and post
 records into Spectrum Pre-Time Card Entry.

1. Simply select either a Job or Payperiod, then
 select the Employee Payroll Company to determine which employees will be
 selected for export.

1. Enter a batch code and click Generate Payroll File
 or Generate Add-On File to create records for import.

1. A file named "ExportXXX.csv" (where XXX is the
 Company Code) will be generated in your local Downloads directory on your
 computer.

1. Run the export (steps 1-3) for each payroll
 company you with to import into Spectrum Pre-Time Card Entry.

The user security role of either PTE Admin, Company
 Admin, or Department Manager will determine which records will be created.Once records are created they will be stamped with the
 "Batch ID" used during the file creation, so these records will not be picked up on
 a subsequent batch creation. If you need to clear a batch so it can be re-created
 for the records previously exported, you can highlight the export history batch and
 select Clear Export on the
 bottom left of the screen.
