# User Security Report

Use this report to print a list of operator security information for selected company codes, including users and table categories.
For each field, enter the value that you want to print the corresponding security codes for. Each of these fields provides SuperSelect options (/, *,?, #).
Fields/ButtonsDescriptions
User or groupChoose User if you want to grant one person access to Info-Link, or Group if you want to grant access to a selection of users.
Company/division codeEnter the company/division code.
Database userEnter the database user.
Table categoryEnter the file categories.
Report typeSelect the report format.

- Detail - table and view permissions by user

- Summary - permissions by table/view category for each user.
