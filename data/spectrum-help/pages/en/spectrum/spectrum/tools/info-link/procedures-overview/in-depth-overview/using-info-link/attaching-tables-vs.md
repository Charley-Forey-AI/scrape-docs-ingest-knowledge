# Attaching Tables vs. Importing Tables

Normally when using Info-Link, it is recommended that you attach rather than import Spectrum tables. This way, you will always be working with the actual up-to-the-minute information, rather than a copy of the table. The method you will use to attach a Spectrum table will vary with the front-end application you are using. See the section on "Using Info-Link with Microsoft Access" in the Getting Started manual for instructions on attaching tables with Access.
