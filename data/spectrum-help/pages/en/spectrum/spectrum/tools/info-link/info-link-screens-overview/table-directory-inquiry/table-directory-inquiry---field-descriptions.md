# Table Directory Inquiry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
ModuleEnter the two-character module abbreviation (for example, ET for Equipment Tracking), press Enter to see all modules, or use the drop-down list to view all available module descriptions.
Search table nameEnter a table or view name (or partial name) or press Enter to display all object names.
Code, Name and ModuleThe list box displays a list of table and view names, the object type, and the two-character module abbreviation (for example, ET for Equipment Tracking) associated with each object.
TypeSelect to filter the object type display. Objects may be tables or views, or both.
DetailsSelect to display the Search Table Columns window. Use this window to view detailed table column information including Column name, Column Type, Size, and Description. The Description column tells you what the column is used for in the application.
ListingSelect to open the Table Directory Listing screen.
