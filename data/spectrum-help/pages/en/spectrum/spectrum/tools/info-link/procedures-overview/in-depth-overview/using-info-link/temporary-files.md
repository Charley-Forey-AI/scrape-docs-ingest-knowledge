# Temporary Files

Certain Spectrum files are denoted with the word TEMP in the table description, for example, IM_TEMP_EXCESS_INV_REP. The TEMP or TMP designation indicates that it is a work file that is erased and revised automatically when a given Spectrum report or update is run. These files do not store permanent information. For example, the work file IM_TEMP_EXCESS_INV_REP is automatically erased and revised each time the Excess Inventory Report is printed.
