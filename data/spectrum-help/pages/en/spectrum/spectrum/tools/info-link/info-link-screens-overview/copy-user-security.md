# Copy User Security

Use this screen to quickly set up users that you intend to give the same or similar security access by copying the setup for an existing operator to a new operator.
This feature creates a new operator with the same Info-Link security.Note: This feature works only to create new user records. If the record already exists, you may opt to delete it in User Security Maintenance and then recreate it using this screen.
Many fields in this screen are disabled if you are copying from a group.FieldDescription
Source

- From company code: Enter the company/division code of the operator from which you wish to copy.

- From database user: Enter the ID code for the operator that you want to copy the setup from, or use the dropdown list to select from a list of valid ID codes.

- Password: Enter the password for the operator that you want to copy. All Info-Link user passwords are encrypted and whenever records are added or modified, the records automatically store the password.

Destination

- To company code: Enter the company/division code of the operator that you want to create.

- To database user: Enter the operator that you want to create, or use the dropdown list to select from a list of existing database users.

- Password: Enter the password for the operator that you want to create.

ContinueSelect to proceed with your choices.
CancelSelect to exit the screen without saving your changes.
