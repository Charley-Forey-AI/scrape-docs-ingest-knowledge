# Spectrum Crystal-Link Reports Files

A table of all Crystal-Link reports.
Each Crystal-Link report program is stored as a file with the extension "RPT". These files are typically stored in the same directory as the Spectrum program and data files.
For example, if your main Spectrum directory on the network is F:\SPECTRUM, the Crystal-Link report files are stored in F:\SPECTRUM\REPORTS.
If you are using Remote-Link, you may store the Crystal RPT files on your local directory to improve performance over the wide-area network. Your system administrator can set this up in the operator security area of Spectrum.
