# Testing Info-Link Setup

You can test the Info-Link setup on your
 workstation.
Use Microsoft® Excel to test your info-link setup.

1. On the workstation, launch Excel with a blank
 spreadsheet.

1. Go to the Data menu, select Import External Data, and then
 select New Database Query.
 The Choose Data Source window will display all of the data sources set up on the
 workstation.

1. Select the Spectrum data source (normally
 "Spectrum").

1. The SQL Server Login dialog box
 will display, requiring you to enter in your user name and password. Enter the user
 name and password that was created for you using Info-Link.

1. After a successful login, the Query Wizard
 – Choose Columns box will display a list of all of the tables and
 views in the database that you have access to.

1. If the correct tables and views display, you know you
 have correctly set up your Info-Link security. You can cancel the query and close
 Excel.Similar steps can be followed to test Info-Link from other
 applications, such as Microsoft Access or Crystal Reports.
