# Reports and Info-Link

You do not need to buy Crystal Reports and Info-Link to run the standard Crystal-Link reports. However, if you want to customize Crystal-Link reports you do need to install Crystal Reports Professional and Spectrum Info-Link.
Crystal-Link reports can be shared with other Spectrum users, as long as the custom reports were created for the same Spectrum version.
