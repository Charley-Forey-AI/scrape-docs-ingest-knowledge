# Logging on to Info-Link

Company / Operator Codes Entered In Name Text Box
When accessing Spectrum tables using Info-Link, the operator will need to enter the company code, operator code, and password. To comply with Microsoft standards, in all capital letters, the user combines the 3-character company code and the 3-character operator code into a single 6-character name. For example, if the company is ABC and the operator code is JJS, the operator would enter ABCJJS in the Name text box. Press the TAB key to move to the Password text box, not the Enter key. If you press the ENTER key, you will receive a security error message. Type your password in the Password text box, and then click the OK button.
To access Spectrum tables in Word, from Insert, select Database, then select Get Data, and then select MS Query. You may also access Spectrum tables in Word another way. From Tools, select Mail Merge, select Create, select Data Source, select Open, and then select MS Query. To access Spectrum tables in new versions of Lotus, from Tools, select Data Base, and then select Connectto External Data.
