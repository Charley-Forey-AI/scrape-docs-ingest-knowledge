# Crystal-Link

Crystal-Link is software developed by Viewpoint that seamlessly connects Crystal Reports with Spectrum. A Crystal-Link report is accessed from Spectrum and uses its tables to create a Crystal Report. There are many benefits to using Crystal-Link reports, including:

- Print-time features such as previewing, exporting, printing selective pages, group-tree navigation, and search

- Improved appearance using Windows fonts and graphics

- More information per page than older reports

- Ability to customize and create new report formats using Crystal Reports
