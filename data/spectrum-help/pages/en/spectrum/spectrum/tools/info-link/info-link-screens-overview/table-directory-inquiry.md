# Table Directory Inquiry

Use this screen to view and print database table names and column documentation for database tables.
A list of table and column names can be helpful if you want to familiarize yourself with the information that can be found within the system or if you want to create your own Crystal reports.
Access this screen from the Info-Link menu or the Enterprise Management > Inquiries menu.
