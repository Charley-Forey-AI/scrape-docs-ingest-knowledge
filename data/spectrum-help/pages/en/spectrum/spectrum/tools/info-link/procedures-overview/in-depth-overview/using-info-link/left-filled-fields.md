# Left-Filled Fields

Some Spectrum fields have leading spaces inserted to make the fields right justified. Examples of this include the job number, vendor code and employee code. These fields usually have one or more leading spaces. For example, job 100 will actually be job bbbbbb100, where the letter "b" represents blanks. Therefore, if you want to select only records for job 100, you need to include the spaces or use appropriate wild card characters.
