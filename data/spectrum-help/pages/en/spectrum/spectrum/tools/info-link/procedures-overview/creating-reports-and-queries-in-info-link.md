# Creating Reports and Queries in Info-Link

Learn how to create reports and queries using
 Info-Link.

1. Design the desired report or query on paper.

1. Determine which Spectrum tables and columns are needed. If more than one table is used, determine the relationship between the tables (for example, which columns are used to join the tables (see [Joining Tables](/en/spectrum/spectrum/tools/info-link/procedures-overview/in-depth-overview/using-info-link/joining-tables)). It is also helpful to print the table documentation available in Info-Link (see [Table Directory Inquiry](/en/spectrum/spectrum/tools/info-link/info-link-screens-overview/table-directory-inquiry)).

1. Run the desired front-end Windows application (for example, Microsoft Excel) and select the command needed to initiate a data query.

1. Select the desired Spectrum tables and columns using the data query program.
