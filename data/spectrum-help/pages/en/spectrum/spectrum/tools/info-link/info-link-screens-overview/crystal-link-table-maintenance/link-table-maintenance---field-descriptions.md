# Link Table Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Search table nameEnter a table or view name, or press Enter to select from a list of available names.
New buttonSelect to enter a new table or view name to the list.
Type buttonThe object type displays. Objects may be tables or views or both.
Details buttonSelect a table or view from the grid and select this button to display the Search Table Columns window. Here you can see the column names, column types, column size, and description. No entry is allowed.
