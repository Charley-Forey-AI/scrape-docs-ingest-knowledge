# Setting up Info-Link

Instructions for setting up Info-Link.
Consider reviewing [Introduction to Info-Link](/en/spectrum/spectrum/tools/info-link/introduction-to-info-link).
Before Info-Link can be used for the first time, the System Administrator must set up security information for each Info-Link user.

1. Add security category, IL, to the Operator Maintenance.

1. In the Modules window of the Company Installation screen, select the Info-Link checkbox.

1. Run Info-Link Setup on individual computers. Refer to the instructions for performing this procedure in the Installation Guide.

1. Add operators and table categories for each Info-Link user in User Security Maintenance.

1. Add tables for each table category in [Table Security Maintenance](/en/spectrum/spectrum/tools/info-link/info-link-screens-overview/table-security-maintenance).

Users may access any tables for which they have security authorization.
