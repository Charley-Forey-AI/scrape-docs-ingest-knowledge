# Understanding Cost Centers

Cost centers serve two main purposes within
 Spectrum.

- Cost centers provide the security, selection, and validation
 features that are inherent when using multiple Spectrum companies, while still
 providing the flexibility of sharing data and reporting that is not supported when
 using multiple companies. For example, information can be stored in one
 company--allowing shared vendors, customers, and so forth. While each data entry
 transaction would still belong to a single cost center, it will be viewable across
 selected cost centers, depending on the operator's cost center security.

- SCost centers include longer G/L account codes. Every G/L account
 used in the system is posted to General Ledger with a cost center assigned to it.
