# Warehouse Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Warehouse code
Type the warehouse code (or codes) associated with the assets that you want to include on this screen. Alternatively, you can double-click in this field to select from a list of available warehouse codes, or press Enter to accept the system default of ALL.

Current cost center
Type the cost center code currently in use, double-click on this field to select from a list of available cost centers, or press Enter to accept the system default of ALL.

New cost center
From the drop-down list, select the new cost center code that you want to
 assign to the specified warehouse, or press Enter to select the
 default of <no
 change> and view a list of associated cost centers.

Grid columns that allow entry

New cost center
Select a warehouse and then use the drop-down list box to assign a cost center
 to the warehouse. If you choose not to assign a cost center, leave this
 field set to <no
 change>.
