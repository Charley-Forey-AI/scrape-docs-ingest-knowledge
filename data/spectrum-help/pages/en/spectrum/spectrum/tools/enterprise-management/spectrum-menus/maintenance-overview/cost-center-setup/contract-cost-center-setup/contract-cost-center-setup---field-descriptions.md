# Contract Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Job number
Type the number of the job to be included on this screen, double-click on this field to select from a list of available job numbers, or press Enter to accept the system default of ALL.

Customer code
Type the customer code (or codes) to be included on this screen. Alternatively, you can double-click on this field to select from a list of available customer codes, or press Enter to accept the system default of ALL.

Current cost center
Type the cost center code currently in use, double-click on this field to select from a list of available cost centers, or press Enter to accept the system default of ALL.

New cost center
From the drop-down list, select the new cost center code that you want to
 assign to the specified contract or contracts, or press Enter to select the
 default of <no change> and view a list of
 contracts and the associated cost centers.

Grid columns that allow entry

New cost center
Select a job contract and then use the drop-down list box to assign a cost
 center to the contract. If you choose not to assign a cost center, leave
 this field set to <no change>.
