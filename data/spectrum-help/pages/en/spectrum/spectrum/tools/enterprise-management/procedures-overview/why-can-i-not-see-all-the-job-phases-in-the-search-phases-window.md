# Why can I not see all the job phases in the Search Phases
 window?

If cost centers are set to Pending
 in Enterprise Installation for your company, all phases display in the Search Phases window
 regardless of the operator's cost center security.
However, once you set the company cost center to Yes, the search results return only the
 codes for which the operator has security access.
