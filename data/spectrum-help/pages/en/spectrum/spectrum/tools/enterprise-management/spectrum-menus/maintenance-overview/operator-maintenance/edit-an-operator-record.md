# Edit an Operator Record

Use the Edit Operator window to make changes to operators' default settings.
You can review operator information using the Operator Listing. For more information, see [View a List of Operators](/en/spectrum/spectrum/system-administration/installation/view-a-list-of-operators).
To edit an existing operator:

1. Go to System Administration > Security > Operator Maintenance.

1. Choose one:

- Press Enter and select the record from the list of operator and select the Edit button.

- Enter the operator's Logon ID in the Search field and press Enter.

The Edit Operator window opens.

1. Make the necessary changes to operator's default settings. If needed, refer to [Edit Operator Screen - Field and Tab Descriptions](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/maintenance-overview/operator-maintenance/edit-an-operator-record/edit-operator-screen---field-and-tab-descriptions).
