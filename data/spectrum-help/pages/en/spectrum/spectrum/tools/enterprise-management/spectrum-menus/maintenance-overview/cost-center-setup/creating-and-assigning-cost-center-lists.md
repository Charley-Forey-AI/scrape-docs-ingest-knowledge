# Creating and Assigning Cost Center Lists

Learn how to create and assign cost center
 lists.
Follow these steps to learn how to create and assign cost
 center lists.

1. On the Site Map screen, click
 the Enterprise Management option.

1. On the Maintenance menu, click
 the Cost Center Setup option.

1. On the [Cost Center Setup](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/maintenance-overview/cost-center-setup) screen,
 click one of the modules in the list and then click the Edit
 setup link to access the associated Cost Center
 Setup screen.

1. Select the record you want to assign a cost center
 list to.Note: Only screens that allow cost center sharing or overrides
 allow you to specify a cost center list (Bank Account, Customer, Deduction/Add-on, Department, G/L Department, Inventory, Location, Vendor, and Master File cost center setup
 screens).

1. In the Current cost center field, select a cost center.

1. In the New cost center field,
 select Entered list from the available options.

1. In the Build Shared Cost Center
 Codes window, in the Available cost centers
 box, click on a cost center Code that you want to add to the list you are creating,
 and then click the Assign button to move the cost center to
 the Assigned cost centers box.

1. When you are done adding cost centers to the
 Assigned cost centers box, click OK
 to save your changes and return to the main screen.Tip: You can return to the
 Build window by clicking on the Build
 List button.

1. Select the Overwrite existing cost center
 codes? checkbox only if you want to overwrite existing codes in the
 destination company; otherwise leave this checkbox clear.

1. Click Go. A second screen displays a list of codes, descriptions, [Old Cost Center Sharing](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/maintenance-overview/cost-center-setup/creating-and-assigning-cost-center-lists/old-cost-center-sharing)
 and [Build Shared Cost Center Codes](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/maintenance-overview/cost-center-setup/creating-and-assigning-cost-center-lists/build-shared-cost-center-codes).

1. Click Continue to apply your changes to the selected record(s).
