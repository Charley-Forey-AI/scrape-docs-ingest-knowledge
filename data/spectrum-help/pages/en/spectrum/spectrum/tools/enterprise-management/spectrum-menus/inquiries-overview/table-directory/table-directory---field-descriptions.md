# Table Directory - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
FieldsDescriptions
ModuleEnter the two-character module abbreviation (for example, PR for Payroll), press Enter to see all modules, or double-click in this field to view all available module descriptions.
Table/view nameEnter a partial table or view name, press Enter for ALL, or double-click in this field to view all available tables.
Find buttonSelect to open the Find Table/View window and re-select modules and tables/views.
Reset buttonSelect this button in the Find Table/View window to return the field selections to the original defaults (ALL).
Detail buttonSelect to display the Search Table Columns window. Use this window to view table column information including Type, Column name, Array Length or Size, and Display code.
ListingSelect to go to the [Table Directory Listing](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/reports-overview/table-directory-listing) screen.
