# Master File Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

G/L code
Type the G/L code (or codes) associated with the assets that you want to
 include on this screen. Alternatively, you can double-click in this field to
 select from a list of available General Ledger codes, or press
 Enter to accept the software default of ALL.

Current Cost Center
Type the cost center code currently in use, or double-click on this field to select from a list of available cost centers.

New Cost Center
From the drop-down list, select the new cost center code that you want to
 assign to the specified department, or press Enter to
 select the default of <no
 change> and view a list of departments and the associated
 cost centers.

Overwrite existing cost center?
Select this checkbox only if you want to overwrite existing codes in the destination company; otherwise leave this checkbox clear.

Grid columns that allow entry

New cost center sharing
 Select a G/L code and then use the drop-down list box to assign cost centers
 to the G/L codes. Options include <no change>,
 Entered list, or Available in
 all.
 If Entered list is selected, click the New
 cost center sharing button to create a cost center list from
 a directory of valid cost centers.
