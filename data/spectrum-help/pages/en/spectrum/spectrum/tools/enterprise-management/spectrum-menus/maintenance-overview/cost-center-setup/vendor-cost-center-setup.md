# Vendor Cost Center Setup

Use this screen to apply new cost center codes to vendors that currently use cost centers. This screen allows a list of specified cost centers (or all cost centers) to be assigned to each record that displays in the grid.
Note: The term "cost center" is user-defined, so the field names will
 display the user-defined names in Spectrum after your cost centers and cost groups are set
 up.
