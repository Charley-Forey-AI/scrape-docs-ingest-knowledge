# Build Shared Cost Center Codes

The New cost center sharing button displays after the header fields have been completed and is only available for records that have previously been assigned a list of cost centers or when Entered list is selected in the New cost center sharing grid column.
Select a grid record and click the New Cost Center Sharing button. Use this window to create a list of valid cost centers that will be assigned to the specified record.
