# Cost Center Scheme - Overrides Tab

Use the New Scheme window and the Edit Scheme window to create or amend the cost center scheme overrides, which will be used during data entry only (including updates, listings, and file imports). Changes made here determine which records an operator can enter in for each assigned cost center.
Select a table and click the Override button to add or change an override for the table. Override options include No override, All cost centers, Cost group only or Cost center list (which allows you to specify cost centers). These schemes are used in place of the normal security schemes, and allow you the flexibility to see a special list of cost centers. For example, you could enter invoices for jobs in cost centers Seattle and Portland, but you would only be allowed to perform time card entry for Seattle employees.
When a cost center override is specified, the operator then has exclusive access to the specified cost center, instead of access to the original cost center data. Example An operator has access to cost center 101 with an override for work orders for cost center 201. This operator is only allowed to see work orders for cost center 201. In order to have access to work orders with cost center 101 and 201, the operator would need to have 101 added to the work order overrides.
