# Equipment Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Equipment code
Type the unique code to identify each piece of equipment to be included on this screen, or double-click on this field to select from a list of available equipment codes, or press Enter to accept the system default of ALL.

Current cost center
Type the cost center code currently in use, double-click on this field to select from a list of available cost centers, or press Enter to accept the system default of ALL.

New cost center
From the drop-down list, select the new cost center code that you want to
 assign to the specified equipment, or press Enter to select the
 default of <no
 change> and view a list of equipment and the associated
 cost centers.
