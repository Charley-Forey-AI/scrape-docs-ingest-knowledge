# Copy Cost Centers

Use this screen to set up cost centers across companies quickly and easily. You can only copy cost centers from and to companies that you have security access to, and if any cost groups or cost center schemes are included, then all the associated cost center and cost group codes will also be copied.
If the "Copy from" cost center has the same code as a cost center or cost group in the "Copy to" company, the copy will only proceed if the Overwrite existing codes checkbox is selected.
