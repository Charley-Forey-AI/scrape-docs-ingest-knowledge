# Cost Center Scheme Maintenance

Use this screen to establish cost center schemes that can aid in the set up and ongoing maintenance of the cost centers available to each operator if the cost center feature is not turned on in the Enterprise Installation screen, this screen will not display. After a cost center scheme is created, it can be applied to an individual operator's company setup in Operator Maintenance screen ( located on the Site Map - Security menu). For example, you might create a scheme that allows for full access to cost center information, or only access to a specific region or division.
