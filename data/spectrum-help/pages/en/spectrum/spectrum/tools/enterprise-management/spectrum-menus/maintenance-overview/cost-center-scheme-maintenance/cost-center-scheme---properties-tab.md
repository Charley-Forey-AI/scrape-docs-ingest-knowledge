# Cost Center Scheme - Properties Tab

Use the table below for assistance when completing the fields on this tab.
Fields
Descriptions

Scheme
Type or edit the cost center scheme name. Schemes are alphanumeric and can be up to 10 characters long. You cannot set up a scheme named ALL.
Tip: Set up a scheme named FULL for use by operators that will have access to all cost centers.

Description
Type or edit the scheme description. Descriptions are also alphanumeric and may be up to 40 characters long.

Report default
Type the cost group you want to use as the default for report and inquiry cost group selections, or double-click to select from a list of all available cost groups.
This option is helpful when a certain cost group is typically selected for reporting by users assigned to this scheme.

Cost center selection
Select a method that will be used to verify the validity of a cost center when using this scheme.

- Select All if you want to use all cost centers.

- Select Cost group entry if you want to use a cost group only. If you choose this option, a Validation box displays where you can specify the cost group you want to use with this scheme.

- Select Cost center list if you want to enter a list of cost centers. If you choose this option, a Validation box displays where you can create (or edit) a list of cost centers that will be associated with this scheme.

Validation
If the Cost group entry option is selected in the Cost center selection field, a Cost group field displays here to allow you to designate the group you want to associate with this scheme. Type the cost group code or double-click to select from a list of available cost groups.
