# Cost Group Maintenance

Use this screen to create and maintain a list of cost center groups; if the cost center feature is not turned on in the Enterprise Installation screen, this screen will not display. Cost center groups can consist of a collection of cost centers or cost center groups, but not a combination of both. Each cost group codes are unique and therefore the same code may not be used as both a cost center and cost group. Furthermore, each cost group can only have one parent.
