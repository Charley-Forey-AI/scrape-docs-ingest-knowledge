# Checking for Unassigned Records

It is recommended that managers follow the check for
 unassigned record steps on a regular basis to find any records that might need cost center
 maintenance. For example, if you use Quick-Link or PO Import to add inventory items on a
 routine basis, or if you occasionally use the copy vendors utility, some records might be
 added without the necessary cost center information.

1. On the Site Map screen, click the Enterprise Management option.

1. On the Maintenance tab, click the Cost Center Setup option.

1. On the Cost Center Setup screen, review the Unassigned column. The numbers in this column are associated with Required setup types. This number tells you how many records in each file have yet to be assigned cost centers.

1. Select a module and click the Edit button to assign a cost center to the appropriate record.
