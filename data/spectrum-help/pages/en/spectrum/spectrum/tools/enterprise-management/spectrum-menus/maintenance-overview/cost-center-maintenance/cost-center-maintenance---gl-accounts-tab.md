# Cost Center Maintenance - G/L Accounts Tab

This tab displays a list of all the G/L accounts that are
 currently set up in the module installation screens. Any G/L accounts set up here will
 override the company setting.
Before accounts can be set up here, they must be made available for use
 in the General Ledger > G/L Master File Maintenance screen by selecting an account code and clicking the Edit button and then clicking the
 Cost Center Sharing tab. Select the Available in all checkbox to validate the
 account for use with cost centers.
Fields
Descriptions

Module
The modules where the associated G/L account codes are
 setup displays. No entry is allowed.

Description
Abbreviated G/L account code descriptions display (as
 they appear in the associated module installation screens). No entry is
 allowed.

G/L override
Use this column to set up any G/L account overrides
 (entries made here will override the company settings). Double-click in the
 column to open the Search G/L Accounts window to
 select from a list of available codes. After making your selection, the code
 will be validated.
Before accounts overrides can be established, the account
 codes must be made available for use in the General Ledger > G/L Master File Maintenance screen by selecting an account code and clicking the
 Edit button and then clicking the Cost
 Center Sharing tab. Select the Available in all checkbox to
 validate the account for use with cost centers.

Account description
The G/L account code descriptions associated with the G/L
 account overrides display.
