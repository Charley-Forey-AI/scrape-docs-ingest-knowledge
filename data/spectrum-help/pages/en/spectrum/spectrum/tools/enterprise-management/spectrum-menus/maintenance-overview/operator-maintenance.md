# Operator Maintenance

Use this screen to set up and maintain Spectrum operators and their default settings.
To open this screen, go to System Administration > Security > Operator Maintenance.
The number of tabs that display depends on other settings in the system. For example, the Jobs tab displays only if the Utilize job-based security checkbox is selected on the Job Cost Installation - Security tab.
Information you enter here overrides the general scheme settings on the Enterprise Installation screen for the individual operator in this company only.
These buttons appear when you open the screen.ButtonDescription
NewCreate new user record.
ListingSelect to open the Operator Listing report launcher in a new tab.
SearchPress Enter to see a list of all the operators who are set up in the current company, or use the drop-down arrow to narrow your search.
Status TypeFilter the operator list based on current status.
