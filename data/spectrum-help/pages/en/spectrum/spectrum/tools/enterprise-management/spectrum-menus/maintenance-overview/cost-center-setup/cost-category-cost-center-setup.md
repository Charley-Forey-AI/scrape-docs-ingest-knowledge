# Cost Category Cost Center Setup

Use this screen to apply new cost center codes to cost categories that currently use cost centers. This screen allows only one cost center to be assigned to each record that displays in the list box.
Note: The term "cost center" is user-defined, so the field names will display the user-defined names in Spectrum after your cost centers and cost groups are set up.
