# Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Module
A list of all the modules that use cost centers displays. Modules that are not installed do not display. Select a module file and click the Edit setup link to open the corresponding cost center setup screen.

Description
A list of the maintenance file descriptions displays.

Setup type
A list displays telling you whether the cost center setup is Required (all assigned cost centers require input) or Shared (shared cost centers allow one or more cost centers to be assigned, and the default is ALL cost centers).
If the setup type is Required, then a number will display in the corresponding Unassigned column. This number tells you how many records in each file have yet to be assigned cost centers.

Unassigned
The numbers in this column are associated with Required setup types. This number tells you how many records in each file have yet to be assigned cost centers. Select a module and click the Edit setup link to assign a cost center.

Edit Setup linkSelect a module file and click this link to make any necessary changes to the cost center assignment. Each module file links to a different screen, and security access is based on the security access for that module.
For example, you cannot access a job record if the job cost center is already assigned to a cost center that you do not have security permission for via your cost center scheme assignment. Likewise, you can only assign a record to cost centers you have permission to access.
