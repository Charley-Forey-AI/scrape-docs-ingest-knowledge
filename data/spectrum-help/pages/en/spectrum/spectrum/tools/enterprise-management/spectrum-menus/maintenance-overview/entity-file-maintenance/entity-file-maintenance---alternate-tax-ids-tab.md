# Entity File Maintenance - Alternate Tax IDs tab

Use the table below for reference when completing the New Entity - Alternate Tax IDs tab fields.
Fields
Descriptions

Accounts payable
This optional field will be used during 1099-MISC year end processing and in Canadian T5018 forms.

Code
Enter a tax code or press F4 to open the Search Tax Codes window and select a valid tax code.

Payroll tax table description
The Payroll tax table description defaults after entering a tax code.

Tax ID number
Enter a tax ID number. This field is required.

Electronic filing # If the selected tax code is filed electronically, enter the
 electronic filing # in this field.
SUTA tax ID numberEnter your SUTA tax ID number in this field, if
 applicable.
