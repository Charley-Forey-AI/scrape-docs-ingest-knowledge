# Cost Category Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Equipment cost categories

Cost category code
Type the number of the cost category or categories to be included on this
 screen, click on the drop-down arrow to select from a list of available cost
 category codes, or press Enter to accept the software
 default of ALL.

Current cost center
Type the cost center code currently in use, or click on the drop-down arrow to select from a list of available cost centers.

New cost center
Click the drop-down arrow and select the new cost center code that you want to
 assign to the specified category or categories, or press
 Enter to select the default of <no
 change> and view a list of categories and the associated
 cost centers.

Grid columns that allow entry

New cost center
Use the drop-down list box to assign a new cost center to the selected cost category.
