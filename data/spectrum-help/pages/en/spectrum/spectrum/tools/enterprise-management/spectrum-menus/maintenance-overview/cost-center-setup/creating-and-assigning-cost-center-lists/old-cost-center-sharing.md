# Old Cost Center Sharing

This button displays after the header fields have been completed and is only available for records that have previously been assigned a list of cost centers. Records that are available in all cost centers or no cost centers will not use this window.
Select a record and click this button to display the Old Cost Center Sharing window and review an existing list of cost centers assigned to the specified record.
