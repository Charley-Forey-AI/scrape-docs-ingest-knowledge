# Work Site Address Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Site ID
Type the site ID to be included on this screen. Alternatively, you can double-click on this field to select from a list of available work order job site IDs, or press Enter to accept the system default of ALL.

Current cost center
Type the cost center code currently in use, double-click on this field to select from a list of available cost centers, or press Enter to accept the system default of ALL.

New cost center
From the drop-down list, select the new cost center code that you want to assign to the specified site ID, or press Enter to select the default of <no change> and view a list of associated cost centers.

Grid columns that allow entry

New cost center
Select a work order job site and then use the drop-down list box to assign a cost center to the site. If you choose not to assign a cost center, leave this field set to <no change>.
