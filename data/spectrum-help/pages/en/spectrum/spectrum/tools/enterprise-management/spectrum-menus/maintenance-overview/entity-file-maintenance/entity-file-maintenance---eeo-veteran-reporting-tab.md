# Entity File Maintenance - EEO Veteran Reporting tab

Use the table below for reference when completing the New Entity - EEO Veteran Reporting tab fields. This tab will be hidden if the 'Non-US payroll processing' checkbox has been selected in Payroll Installation in the current company, or the 'Report by location' checkbox is not selected in Human Resources Installation, or if Human Resources module is not present for the current company.
Fields
Descriptions

Company number NAICS code DUNS ID Employer ID County
Enter the company number, North American Industry Classification System (NAICS) code, Dun and Bradstreet (DUNS) identification number, Employer Identification number, and county where the company resides to include on the Veteran Employment Report.
