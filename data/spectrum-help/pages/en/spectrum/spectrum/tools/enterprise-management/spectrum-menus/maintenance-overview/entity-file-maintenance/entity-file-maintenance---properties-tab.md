# Entity File Maintenance - Properties tab

Use the table below for reference when completing the New
 Entity - Properties tab fields.
Fields
Descriptions

Name
The entity name entered here will replace the name for
 Company Installation on selected external forms and reports.

Address
Telephone
This address and phone number will replace the Company
 Installation settings on selected external forms and reports.

Remark
Text in this field will appear in the Search Entities
 window.

Entity logo
If you choose to upload a logo file here, it will be used
 on certain forms, in place of the Company logo designated in Company
 Installation.
Use the Upload button to copy the company logo image file to the
 proper location so that it will display on your Crystal Reports. Use the
 Remove button
 to delete the logo file.
