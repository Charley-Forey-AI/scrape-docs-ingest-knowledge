# Assign Cost Centers to Maintenance Items

Learn how to assign cost centers to maintenance
 items.

1. Navigate to Enterprise Management  >  Maintenance  >  Cost Center Setup. This screen lists the maintenance items that require cost centers. The Setup type column lists whether or not these elements are Required (one cost center only) or Shared (one or more cost centers). The Unassigned column lists the number of records that have not yet been assigned to a cost center.

1. Select the module that you would like to assign and click the Edit button.

1. On the setup screen for the selected module, set up the cost center details and click Save.

1. Repeat steps 2 –3 for each module. Once you have completed this process, verify that all elements have cost centers assigned.
