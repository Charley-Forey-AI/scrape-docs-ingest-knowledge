# Job Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Job number
Type the number of the job to be included on this screen, double-click on this field to select from a list of available job numbers, or press Enter to accept the system default of ALL.

Current cost center
Enter the number of the cost center currently in use to be included on this screen, double-click on this field to select from a list of available cost center numbers, or press Enter to accept the system default of ALL.

New cost center
Enter the number of the new cost center to be included on this screen, use the
 drop-down menu to select from a list of available cost centers, or select
 <no
 change> and view jobs whose cost centers have not
 changed.
