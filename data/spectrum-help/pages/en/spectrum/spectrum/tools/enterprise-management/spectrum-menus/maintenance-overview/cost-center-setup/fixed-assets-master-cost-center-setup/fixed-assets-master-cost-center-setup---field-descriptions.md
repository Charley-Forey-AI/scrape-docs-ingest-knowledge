# Fixed Assets Master Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

G/L control code
Type the G/L control code assigned to this asset to limit the search results
 on the Fixed Assets Cost Center Setup screen.
 Alternatively, you can double-click in this field to select from a list of
 valid control codes, or press Enter to accept the system default of ALL. This entry must
 be a valid control code previously assigned in G/L Control Code
 Maintenance.

Asset number
Type the unique asset number (or numbers) to be included on this screen. Alternatively, you can double-click in this field to select from a list of available asset numbers, or press Enter to accept the system default of ALL.

Current cost center
Type the cost center code currently in use, double-click on this field to select from a list of available cost centers, or press Enter to accept the system default of ALL.

New cost center
From the drop-down list, select the new cost center code that you want to
 assign to the specified asset, or press Enter to select the
 default of <no
 change> and view a list of assets and the associated cost
 centers.

Grid columns that allow entry

New cost center
Select an asset and then use the drop-down list box to assign a cost center to
 the asset. If you choose not to assign a cost center, leave this field set
 to <no
 change>.
