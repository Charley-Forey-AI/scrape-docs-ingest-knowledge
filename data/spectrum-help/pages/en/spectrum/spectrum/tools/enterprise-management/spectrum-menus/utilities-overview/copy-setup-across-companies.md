# Copy Setup Across Companies

Learn how to set up new codes in a specified
 company.
Income tax tables, worker's compensation base codes,
 vacation/holiday/sick control codes, EEO classification codes, and/or formulas will be
 added based on current setup in this company. Codes will be copied based on the
 selections made on this screen and the software will verify that any General Ledger
 account codes are valid in the destination company.
You must have the appropriate security access to Payroll in the
 destination company in order to use this feature.

1. In the Copy from section, select the checkboxes corresponding to the items you wish to copy across companies.

1. In the Copy to section, enter a destination company and decide if you want to overwrite any existing codes with the codes from the current company.

1. Click Continue to proceed with the copy process.
