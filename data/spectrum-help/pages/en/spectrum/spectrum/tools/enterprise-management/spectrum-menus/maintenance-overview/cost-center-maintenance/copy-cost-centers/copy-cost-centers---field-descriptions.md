# Copy Cost Centers - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields
Descriptions

Copy from

Company
The company you are currently logged in to displays is the company the cost center information will be copied from; no entry is allowed. To change companies, exit this screen and click the company drop-down list on the Command Bar to select a different company.

Cost center
Type the cost center you want to copy the setup from, or select from a list of available cost centers.
Select the Include cost group? checkbox if you want to copy all referenced cost centers in the copy.

Cost group
Type the cost group you want to copy the setup from, or select from a list of available cost groups
Select the Include cost center schemes? checkbox if you want to include all referenced cost groups and cost centers in the copy.

Scheme
Type the cost center scheme you want to copy the setup from, select from a list of available cost center schemes.

Copy to

Company
Type the company code you want to copy the cost center setup to, or select from a list of available companies.

Overwrite existing codes?
Select this checkbox only if you want to overwrite existing codes in the destination company.
