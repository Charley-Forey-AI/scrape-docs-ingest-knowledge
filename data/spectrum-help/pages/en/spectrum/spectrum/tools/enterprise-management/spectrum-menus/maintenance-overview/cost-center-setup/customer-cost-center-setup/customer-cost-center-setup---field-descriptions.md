# Customer Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the
 fields on this screen.
Fields/Buttons
Descriptions

Customer code
Type the customer code (or codes) to be included on this
 screen. Alternatively, use the drop-down arrow to select from a list of
 available customer codes,

Current cost center
Type the cost center code currently in use, use the
 drop-down arrow to select from a list of available cost centers, or press
 Enter to accept
 the system default of ALL.

New cost center
Click the drop-down list to make a selection. If
 Entered list is
 selected, click the New cost
 center sharing button to create a cost center list from a
 directory of valid cost centers. Press Enter to select the
 default of <no
 change> and view a list of current cost center assignments.


Overwrite existing cost
 center?
Select this checkbox only if you want to overwrite
 existing codes in the destination company; otherwise leave this checkbox
 clear.

Grid columns that allow entry

New cost center sharing
Select a customer code and then use the drop-down list
 box to assign cost centers to the code. Options include <no change>,
 Entered list, or Available in
 all. If Entered list is selected, click the [Build Shared Cost Center Codes](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/maintenance-overview/cost-center-setup/creating-and-assigning-cost-center-lists/build-shared-cost-center-codes)
 button to create a cost center list from a directory of valid cost
 centers.
