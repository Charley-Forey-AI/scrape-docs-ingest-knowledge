# Build Shared Cost Center Codes

Use this window to create a list of valid cost centers that
 will be assigned to the specified item code.
Select a code and click the Build List button to display the
 Build Shared Cost Center Codes window.Note: The Build List button is only available for
 codes that have previously been assigned a list of cost centers or when Entered list is
 selected in the New cost center field.

1. In the Available cost centers
 pane, click on a cost center code that you want to add to the list you are creating,
 and then click the Assign button to move the cost center to
 the Assigned cost centers pane.

1. When you are done adding cost centers to the
 Assigned cost centers pane, click OK
 to save your changes and return to the main screen.
