# Entity File Maintenance - Custom CCLBL Codes tab

Use the table below for reference when completing the New Entity - Custom CCLBL Codes tab fields. This tab shows a list of the cost centers assigned to the current Entity.
Fields
Descriptions

Add code button
Click this button to open the Add Custom CCLBL to Entity window and enter the cost center you want to associate with the current entity code.

Remove button
Select a cost center code and then click this button to remove the entity code from the cost center.
