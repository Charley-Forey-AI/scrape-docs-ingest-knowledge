# Copy Deduction & Add-on Codes

Learn how to set up deduction and add-on codes across
 companies while verifying that the General Ledger account codes and vendor codes are valid in
 the destination company.

1. In the Copy from section, the current company defaults. In the Code field, enter the deduction or add-on codes you wish to copy from, or use the available drop-down to select from a list of codes.

1. In the Copy to section, enter a destination company and code. Then decide if
 you want to overwrite existing codes, include tax effects, and include HR benefit
 setup (if you have the Human Resources module).

1. Click Continue to complete the code copy process.
