# Create Cost Groups

Learn how to create cost groups.

1. Navigate to Enterprise Management  >  Maintenance  >  Cost Group Maintenance.

1. Click the Newbutton to define which cost centers are to roll up into a group. You can also create a group of other Cost Groups, as needed.
