# Department Expense Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Department code
Type the payroll department code (or codes) you want to include in the search results on the Department Expense Cost Center Setup screen. Alternatively, you can double-click in this field to select from a list of valid department codes, or press Enter to accept the system default of ALL.
Note: Because many users prefer alphabetic or combination alpha and numeric
 coding, this code is not a numeric-only field. Because of this, the code
 left-justifies when entered.

Current cost center
Type the cost center code currently in use, double-click on this field to select from a list of available cost centers, or press Enter to accept the system default of ALL.

New cost center
Click the drop-down list to make a selection. If Entered list is selected,
 click the New cost center
 sharing button to create a cost center list from a directory
 of valid cost centers. Press Enter to select the default of <no change> and view a list
 of departments and their current cost center assignments.

Overwrite existing <cost center>?
Select this checkbox only if you want to overwrite existing codes in the destination company; otherwise leave this checkbox clear.

Grid columns that allow
 entry
New cost center sharing
Select a vendor code and then use the drop-down list box to assign cost centers to the vendor. If Entered list is selected, click the New cost center sharing button to create a cost center list from a directory of valid cost centers.
