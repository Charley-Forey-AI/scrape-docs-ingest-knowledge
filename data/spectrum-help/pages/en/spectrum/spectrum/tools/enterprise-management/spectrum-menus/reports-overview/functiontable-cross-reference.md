# Function/Table Cross Reference

Use this screen to print a listing of all files used by a specified function.
You can designate a partial function name or double-click in this field to select the desired menu option. Formats are available for printing files associated with the specified function. The report can also be printed to include all functions related to it. The report is available in two formats: Summary (which lists each file once per function) and Detail (which lists every time the file is used).
