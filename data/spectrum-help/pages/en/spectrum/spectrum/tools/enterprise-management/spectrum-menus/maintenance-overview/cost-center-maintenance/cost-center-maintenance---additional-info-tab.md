# Cost Center Maintenance - Additional Info Tab

Use this tab to enter or change any supplemental information relating to a cost center, including contact and shipping information. All of the fields on this tab are optional.
Fields
Descriptions

General

Main contact
Type the name of the main contact person for this cost center.

Fax number
Enter the contact's fax number, if available.

ID
Type the identification code or name for this cost center.

Web address
Type the website address (URL) for this cost center; for example: www.dcg.com.

Email address
Type the email address of the contact person (or you can enter a general email address; for example, for the customer service department).

Shipping address

Address
Enter any primary and secondary address information for this cost center. The Address 2 line can be used to enter any suite numbers or post office boxes.

Other info

User-defined fields
Type any user-defined information for this cost center. The main purpose of the user-defined fields is to allow you to add fields to the major files throughout the software for reporting purposes without the need for customization.
