# Entity File Maintenance - Pay Cycle Calendar tab

Use the table below for reference when completing the New Entity - Pay cycle Calendar tab fields. This tab allows you to manage the Pay Cycle Calendar to support entity companies.
Fields
Descriptions

Build Calendar
Click this button to open the Build Calendar window. Use this window to specify pay frequency, the first period ending date, and number of days to check date to define the Pay Cycle Calendar.
