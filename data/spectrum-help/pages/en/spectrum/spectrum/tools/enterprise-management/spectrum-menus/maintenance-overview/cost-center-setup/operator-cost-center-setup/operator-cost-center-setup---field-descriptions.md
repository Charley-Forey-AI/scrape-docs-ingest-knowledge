# Operator Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields
Descriptions

Logon ID
Type the logon ID of the operator or operators you want to assign or change
 cost center schemes for. Alternatively, you can double-click on this field
 to select from a list of available logon IDs, or press
 Enter to accept the software default of ALL
 operators.

Current cost center scheme
Type the cost center scheme currently in use, double-click on this field to
 select from a list of available cost center schemes, or press
 Enter to accept the software default of ALL.

New cost center scheme
From the drop-down list, select the new cost center scheme that you want to
 assign to the specified operator or operators, or select <no change> to view a
 list of operators and their current scheme assignments.
