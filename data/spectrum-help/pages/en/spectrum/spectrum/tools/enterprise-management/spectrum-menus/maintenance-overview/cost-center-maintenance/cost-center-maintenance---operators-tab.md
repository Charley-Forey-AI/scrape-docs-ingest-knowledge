# Cost Center Maintenance - Operators Tab

A display-only list of operators (users) associated with the specified cost center.
Operators cannot be added to a cost center using this tab.
To add operators to a cost center, see [Create a New Operator Record](/en/spectrum/spectrum/tools/enterprise-management/spectrum-menus/maintenance-overview/operator-maintenance/create-a-new-operator-record).
