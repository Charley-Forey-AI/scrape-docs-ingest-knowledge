# Cost Group Maintenance - Field Descriptions

Use the table below for assistance when completing the
 fields on this screen.
Fields
Descriptions

Cost group
Type or edit the cost group code. Codes are alphanumeric
 and can be up to 10 characters long.
Note:
 "ALL" is not a valid cost group code name.


Description
Type or edit the unique cost group description.

Member type
Select the cost group type. Cost groups can consist of
 cost centers or other cost groups, but not both. Once a selection has been
 made and centers or groups have been added to the list, no changes can be
 made to this field.
