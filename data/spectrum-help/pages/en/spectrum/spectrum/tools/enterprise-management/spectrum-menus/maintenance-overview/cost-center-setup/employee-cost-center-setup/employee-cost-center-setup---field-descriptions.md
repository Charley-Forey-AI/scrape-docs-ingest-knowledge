# Employee Cost Center Setup - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields/Buttons
Descriptions

Employee code
Type the employee code (or codes) to be included on this screen. Alternatively, you can double-click on this field to select from a list of available employee codes, or press Enter to accept the system default of ALL.

Current cost center
Enter the number of the cost center currently in use to be included on this screen, double-click on this field to select from a list of available cost center numbers, or press Enter to accept the system default of ALL.

New cost center
From the drop-down list, select the new cost center code that you want to assign to the specified employee or employees, or press Enter to select the default of <no change> and view a list of employees and their current cost center assignments.

Grid columns that allow entry

New cost center
Select an employee and then use the drop-down list box to assign a cost center to the employee. If you choose not to assign a cost center, leave this field set to <no change>.
