# Cost Center Setup

This screen provides a list of all the file maintenance
 screens that use cost centers. If the cost center feature is not turned on in the Enterprise
 Installation screen, this screen will not display.
Select a module and file and click the Edit setup link. The associated
 cost center setup screen displays. Only those modules that are installed are included on
 this list, and you will only be able to see those screens that you have security to access.
 These same setup screens are available via cost center buttons on the associated module
 maintenance screens if cost centers are enabled (for example, Vendors, Employees, etc.).
 The module setup screens include common features, including a Selections button and a new
 cost center entry field, and a list box that allows you to review all affected records. The
 Operator Maintenance setup screen allows you to assign cost center schemes to operators.
Note: The term "cost center" is user-defined, so the field
 names will display the user-defined names in Spectrum after your cost centers and cost
 groups are set up.
