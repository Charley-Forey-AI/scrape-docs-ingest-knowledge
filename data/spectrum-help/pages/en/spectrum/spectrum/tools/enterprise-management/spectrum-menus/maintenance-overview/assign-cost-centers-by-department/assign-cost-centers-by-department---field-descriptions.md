# Assign Cost Centers by Department - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Fields
Descriptions

Department
A list of G/L departments displays; no entry is allowed. Use the Cost center field to assign one cost center to each G/L department in the company.

Cost center
Use the available drop-down list to assign one cost center to each G/L department in the company. The software verifies that the cost center you selected is valid in the current company.
If cost centers had previously been assigned in this utility screen to one or more departments in this company, then these cost centers will default automatically. Each time you exit this screen, the assigned cost centers are save regardless of whether the update is actually performed or not. This save feature ensures consistency should the update need be performed more than once, and to allow cost centers to be defined even before your company switches from 'Pending' to 'Yes' in the Cost Center Settings window.

Description
The description corresponding to the cost center displays. No entry is allowed.
