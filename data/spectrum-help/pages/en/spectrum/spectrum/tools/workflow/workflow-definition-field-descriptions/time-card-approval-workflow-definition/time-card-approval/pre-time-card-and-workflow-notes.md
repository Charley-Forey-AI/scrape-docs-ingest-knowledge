# Pre-Time Card and Workflow Notes

Use this window to add, edit or delete the pre-time card
 lines that are applicable for the current workflow.
When accessing time cards for a Workflow for 'approval by job', the list
 box will provide the same columns as in Pre-Time Card by Job, and otherwise, when accessing
 a Workflow for 'approval by employee', the list box will have columns like in Pre-Time Card
 Entry. A history section is also provided, and if an entry line was rejected, you can
 review the reason for the rejection, if one was provided.
This window is accessed from the main [Time Card Approval](/en/spectrum/spectrum/tools/workflow/workflow-definition-field-descriptions/time-card-approval-workflow-definition/time-card-approval) screen using the Pre-Time Cards/Notes button, which acts
 like an Edit button when a record is selected.
Note: Once an entry line has been approved, it cannot be
 edited so it will not display in this window.
