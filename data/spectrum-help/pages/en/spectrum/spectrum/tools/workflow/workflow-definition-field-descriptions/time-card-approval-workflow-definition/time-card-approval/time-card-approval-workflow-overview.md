# Time Card Approval Workflow Overview

This window provides at-a-glance summary information about
 Time Card Approval workflow items that have been 'Rejected' or are 'In Review' or are
 'Approved' and ready for transfer to Time Card Entry.
The window also offers the ability to select a single reviewers list, to allow override
 operators the ability to easily approve on behalf of an absent reviewer.

- This window always displays automatically for [Override Operators](/en/spectrum/spectrum/tools/workflow/workflow-definition-field-descriptions/time-card-approval-workflow-definition/override-operators) when the Time Card Approval screen is accessed.

- Click View All Rejected to open the Time Card Approval screen displaying only Rejected items. From here you can make changes or return the items for review.

- Click View All In Review to open the Time Card Approval screen displaying only items awaiting action (Approve/Reject, or make changes).

- Click See Reviewers to select an override operator. It is possible that a Workflow may be assigned to 'no reviewer'. In this case it is possible to select this group of lost time cards in the 'Select Reviewer' window.

- This window is also available via an Overview button on the main Time Card Approval screen. In the unusual instance where the operator is included in the Workflow Group assigned to handle 'Rejected' time cards AND that operator is also a Reviewer that currently has one or more sets of time cards to review, the screen first displays the items 'In Review', but then allows that operator to switch back and forth to the 'Rejected' items.
