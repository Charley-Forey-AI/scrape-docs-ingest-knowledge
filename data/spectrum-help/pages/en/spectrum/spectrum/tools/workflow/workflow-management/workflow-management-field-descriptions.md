# Workflow Management Field Descriptions

Use the table below for reference when completing fields on this screen.
Field/Button
Description

Selections
Click this button to specify an Assigned to contact and/or Workflow type to limit the display of records in the grid.

Reassign Step
Click this button to reassign the selected task to a different contact, if you have permission for the workflow type.

Launch Step
Click this button to open the selected workflow task function in a new Spectrum tab, if you have permission to access that screen.

End Workflow
Click this button to end the selected workflow without performing the remaining steps, and change the status to 'Active'.

Refresh
Click this button to refresh grid results based on current selections and date/time.

Status
Click this button to filter the records in the grid by Overdue, Due within 24 hours, and Due beyond 24 hours.

Summary
The summary bar displays a grand total of the numerical columns on the grid.

Group by
Select a column from this drop-down list to sort the grid results, or drag one or more of the column headings to this bar to sort.
