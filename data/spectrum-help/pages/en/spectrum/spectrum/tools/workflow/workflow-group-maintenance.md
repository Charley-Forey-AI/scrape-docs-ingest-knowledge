# Workflow Group Maintenance

Use this screen to define a workflow group code, description, and a list of contacts who are a member of the group.
Field
Description

Group code
Enter a unique workflow group code and description.

Add Contact
Click this button to add a contact to this workflow group. Click the Remove button to remove a contact from this group.
