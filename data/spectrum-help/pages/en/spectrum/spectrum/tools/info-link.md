# Info-Link

The Spectrum Info-Link module allows a non-technical person to set up and maintain security for the Spectrum database from within Spectrum, without having to use SQL or other database commands. This makes it easier to control which users have access to specific tables or views in the database.
Note: For security purposes, all passwords for Info-Link
 users are encrypted so that they cannot be read from the database.
The Spectrum database resides in a SQL database (for example, Microsoft SQL Server). Using Info-Link, this database can be accessed from many programs outside of Spectrum, including Crystal Reports, Microsoft Excel, Microsoft Access, and many others.
