# About Trimble Assistant - Beta

Trimble Assistant is a chat agent you can use to ask specific questions about the tasks you can perform in the application.
To start a chat, select the Trimble Assistant icon  in the navigation bar.
Important:Trimble Assistant is trained only on product-related content on the Trimble Help website. The assistant does not have access to any of your company's data or any data within the application database.

## Tips for Chatting With Trimble Assistant

- Trimble Assistant is still learning. To improve results, ask specific questions. If you do not get a helpful answer the first time, try rephrasing your question and asking it again.

- Trimble Assistant is looking for feedback. Select the Rate button in the lower right of the chat window to rank and comment on the answer to your question. Your feedback will be used to improve the assistant and documentation as needed.

- Trimble Assistant is enabled by default. To turn off the assistant in Spectrum, go to System Administration > Installation >  System, and clear the Enable Trimble Assistant checkbox.

## Trimble Assistant FAQs

Can Trimble Assistant perform tasks within the product?No. Trimble Assistant cannot complete tasks in your product. However, it can provide detailed instructions on how you can complete a given task.
Can I ask Trimble Assistant context-specific questions? For example, what is my vendor balance?No. Trimble Assistant can answer only product-related questions using content in the Help.
What should I do if I receive the following error message: Trimble Assistant isn't available right now. Please contact your administrator to enable it.Contact your system administrator who can enable Trimble Assistant for you by going to the System Administration > Installation >  System screen and selecting the Enable Trimble Assistant? checkbox.
