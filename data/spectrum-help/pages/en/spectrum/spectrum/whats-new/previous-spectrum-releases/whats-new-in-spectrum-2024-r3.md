# What's New in Spectrum 2024 R3

With this version, you can confidently proceed with your year-end processing. Spectrum 2024 R3 is the year-end release which contains the required changes for processing forms like W2s, 1099s, and those for ACA. This release also contains updates, time-saving enhancements, and fixes to customer-reported issues.

## Year-end Programming Changes

US regulatory updates for tax year 2024
We've updated Spectrum so that the pertinent reports and electronic files are IRS-compliant, as well as functional with our partners, like Nelco and Aatrix.
Before the first payroll for 2025, we will provide Tax Tables in a separate file that you will need to load.
Canada regulatory updates for tax year 2024
We've updated the pertinent reports and electronic files to be CRA-compliant.
Before the first payroll for 2025, we will provide Tax Tables in a separate file that you will need to load.
Important:If your Spectrum instance is on premises

- On-premises customers must be on 2024 R2 prior to installing this year-end release.

- This is the last release for on-premises customers; as previously announced, support ends 03/31/2025.

- Please contact your account manager for information on moving your instance to the cloud.

## T4 Build Captures Dental Plan Selection

Spectrum now pulls the benefit code from the Employer dental field in the Employee Main Properties screen when you run the T4 Build.

## Updated T619 Transmittal

The [CRA issued changes](https://www.canada.ca/en/revenue-agency/services/e-services/filing-information-returns-electronically-t4-t5-other-types-returns-overview/t619-next-year.html) to the electronic transmittal record used to submit T4s. We've made the necessary changes so that your T619 file will be accepted.
The changes issued by the CRA consisted of some entries being removed, and others were added. To match the transmittal changes, we've also aligned the fields in the Export T4 File screen.

## W-2 Box 12 Updates

Box 12 of the Form W-2 is dedicated to Retirement Codes. :

- When building W-2s, you now have an additional eight Box 12 positions - for a total of 12 Retirement Alpha Codes.

- We've added the new code II (not to be confused with 11 or LL) issued this year by the IRS. You can enter this new in any one of the Box 12 fields. According to the IRS, code II should be used report Medicaid waiver payments not included in Form W-2 box 1.

## W-2 Box 14 Updates

Box 14 of the Form W-2 is dedicated to Other descriptions. You now have an additional six Box 14 positions - for a total of nine Other description fields when building W-2s.

## New Report Option in Aatrix

The [California Pay Data Reporting](https://calcivilrights.ca.gov/wp-content/uploads/sites/32/2021/01/CA-Pay-Data-Reporting-User-Guide.pdf) user guides indicates that CA employers of 100 or more employees must report certain pay and hours-worked data to the Department of Fair Employment and Housing (DFEH).
In Payroll > Reports > Aatrix Report Filing, the CA Pay Data report now appears in the dropdown list of available reports.

## Spectrum 2024 R3 Bug Fixes

This release includes fixes to customer-reported issues. To see the list:

1. Log in to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues?tabset-d4f10=2) of the Viewpoint Customer Portal.

1. Make your selections in the product and module filter options.

1. In the Fixed in Version field, enter 2024 R3.

1. Select Apply Filters.
