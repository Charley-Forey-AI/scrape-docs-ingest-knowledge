# What's New in Spectrum 2023 R2

Spectrum 2023 R2 contains updates, time-saving enhancements, and fixes to customer-reported issues.

## Improve Your Tool and Asset Management Using Hilti ON!Track

Better manage your equipment and tools and cut costs using the Hilti ON!Track integration with Spectrum. This integration unites Hilti's asset tracking technology with Spectrum's robust accounting capabilities, helping your company keep track of your assets and accurately bill tool usage to jobs.
Note: This integration is available only to Spectrum cloud customers on the Trimble Construction One platform.
Once you properly configure the integration and enter tool data in Spectrum, Hilti ON!Track automatically tracks and records all asset movements. This data then flows back into Spectrum, where you can review requisitions and create transactions to charge tool usage to jobs.
For additional information, see [Hilti ON!Track Integration with
 Spectrum](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum).

## Faster Access to Trimble Work Center

If your Spectrum license is part of a Trimble Construction One contract, accessing the Trimble Work Center is now easier than ever.
You now have the option to launch Trimble Work Center when logging into Spectrum with single sign-on.
Not enabled by default, system administrators can turn on this option for users by selecting the Enable Trimble Work Center checkbox in the System Installation screen.

## More Easily Match Received Items

You can now opt to have purchase order costing information appear on the Packing List Quantity Entry screen, with new Unit cost and Extension columns showing amounts derived from costs given in the PO.
The Unit cost displays when you access the screen, and once you enter the received quantity, the Extension populates.
The addition of this new cost information helps you more easily match the items you've received with the quantities on the purchase order.
To enable this functionality, select the Display received dollar amount in P.O. packing list entry checkbox in the Purchase Order Installation screen.

## Better Control of Subcontract Records

Spectrum now prevents accidental deletion of subcontracts with billed amounts.
When you attempt to delete a subcontract, Spectrum checks whether it has any billed amounts. If any exist, Spectrum prevents the subcontract from being deleted.

## Simplified Fixed Asset Entry

This version of Spectrum doesn't require entering a zero value in the Salvage Value field when setting up a new fixed asset in the Fixed Assets File Maintenance screen.
Your application now treats a blank field value as zero, eliminating the required keystroke and aligning your user experience with what you might expect.

## Support for Stronger Passwords

The character limit in the Password field of the E-mail Credentials screen is expanded from a 30-character limit to a limit of 80 characters, enabling much more secure passwords.
This expansion supports authentication levels required by some email service applications.
To open the E-mail Credentials screen and to change your email password, select the Credentials button in the Edit Operator screen.

## Easier Email Reset

If an Operator in Spectrum is a single sign-on (SSO) user but is not able to log in due to an email mismatch, System Administrators can now help the user without having to contact Support for assistance.
Now, after the System Administrator changes the operator's email in the Spectrum Operator Maintenance screen, they can re-migrate the user. For more information about SSO, see [Single Sign-On](/en/spectrum/spectrum/getting-started/single-sign-on).

## Service Tech Mobile Credit Card Processing is Coming Soon

A future release of Service Tech Mobile (STM) will include the ability for field technicians to collect credit card payments using their mobile device.
There are two fields in Spectrum related to this new mobile feature, both in the Service Tech Installation screen:

- Enable mobile payment collection - once the feature is released in the mobile application, select this checkbox to enable this functionality for your organization.

- Paid by credit Card - use this drop-down field to assign a default status code to be assigned when a service technician collects a credit card payment payment using their mobile device.

Even when enabled at the organizational level, administrators control which technicians can collect payment in the field; the feature appears in the mobile app only to technicians who have been granted the appropriate function level security in Spectrum.

## Spectrum 2023 R2 Bug Fixes

This release provides fixes to over 30 customer-reported issues. To see the list:

1. Log in to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues?tabset-d4f10=2) of the Viewpoint Customer Portal.

1. Make your selections in the product and module filter options.

1. In the Fixed in Version field, enter 2023 R2.

1. Select Apply Filters.
