# Service Tech Mobile - Prior Releases

Review enhancements from prior years.

## Service Tech Mobile 2023 R4 (October 2023)

Fillable PDFs - Technicians can now fill out, save, and update site-specific and company-wide forms.
Using their mobile devices, technicians can search from the fillable PDFs they have security access to. Upon saving, the app uploads the PDF's changes to Spectrum's Document Imaging module.
Office admins can use their choice of PDF editing software (for example, Adobe Acrobat) to create these forms.Note: The mobile app supports these field types:

- text input

- checkbox

- drop down list

- option button

Once created, and to make these forms accessible to mobile device users, office admins upload them to certain document Cabinets and Drawers in Spectrum's Document Imaging module. For instructions, see [Make a New Fillable Form Accessible to Mobile Devices](/en/spectrum/spectrum/service/service-tech/make-a-new-fillable-form-accessible-to-mobile-devices).
Faster billing process - When Technicians assign a unit cost to non-stock items in site-based work orders, they can present a correct price without delay. The app now automatically provides the unit price, calculated according to the item's pricing settings in Spectrum.
In-app user assistance - The app's hamburger menu now has a Guides button which takes the mobile app user to a menu of several short in-app guides. Technicians can access these guides for quick reminders for completing common tasks.

## Service Tech Mobile 2023 R3 (June 2023)

Faster time entry - When users add new labor lines in Service Tech from a work order, the default date for each new line is now the current day, greatly reducing the number of taps.Note: When you are entering time from the timekeeping page, the default date behavior is the same as always - the default date is the date you select.

Easier time entry review - The STM app now displays the site or job name on the timekeeping page, making it easier for technicians to review their time entries. The site/job field displays in the the labor card for the work orders in the timekeeping page. Even sites and jobs which are inactive or no longer used appear in this way.
Use the Android app on a Windows device - The Android version of STM is now compatible with Microsoft Windows 11 and published at the Amazon App store. Using a Windows subsystem for Android, you can now install STM on a Windows 11 machine. This is useful for technicians who have laptops or Surface tablets and want to use those devices to access Service Tech. To do this, please download the Amazon App Store from the Microsoft Store on your Windows 11 device..
Dedicated search bar - The Open Assignments and Other Assignments pages now each have a search bar dedicated to making it easy to find these commonly needed labels.

- site name

- work summary

- customer name

- WO number

- contact name

Any subsequent action you take dictates whether the application retains your search results:

- If you navigate to the work order detail page, your search results are retained so that they are visible if you navigate back to the Open/Other Assignments page.

- If you use the hamburger menu to navigate away from the Open/Other Assignments page, your search results are not retained. Once you return, you can run the search again if needed.

The Report Problem link in the hamburger menu has been removed. To submit a case to Customer Support, follow the standard method of logging a support case in the Viewpoint Customer Portal. When creating your case, provide as much detail as possible about the issue. If you would like to view and/or include the app log in your support case, go to About > Export Logs.

## Service Tech Mobile 2023 R2 (March 2023)

With our primary goal of making your work easier, this release provides improved error messaging to speed troubleshooting your configuration settings. The Service Tech Mobile app is available to download 24 to 48 hours following the release date.
A Necessary Update - Because this release brings a required version, shortly after it's available, all users which have not not already updated to the new version will be logged out of the app and will need to update in order to log back in.
To avoid user interruptions, we recommend all mobile device users update to this version as soon as it becomes available in their respective app store.
Improved Error Message - We've improved the error messaging related to certain configuration settings which now provide more helpful information when troubleshooting.
Ongoing Azure Migrations - No action is required on your part and you'll receive a notification in advance of your organization's move to Azure.
Stabilization - This release fixes a number of customer-reported bugs, and brings performance optimizations and security improvements.

## Service Tech Mobile 2023 R1 (January 2023)

Improvements to synchronization result in performance enhancements. These improvements also pave the way for your organization's move to Azure.Note: For the move to Azure, no action is required on your part and you'll receive a notification in advance of the move.
As always, this release includes bug fixes, as well as incremental updates which reflect our commitment to security.

## November 2022 (Service Tech Mobile 2022 R3)

This release brings enhancements to the Signature Page and closer alignment with user expectations when creating work orders.
Pricing Visibility Options - For technicians who have visibility to Work Order pricing in the mobile app, a new toggle button hides or displays the pricing, allowing both Admins and Technicians more flexibility in how and when pricing should be shown to the signer.Note: To change pricing visibility, go to System Administration > Security > Function Security > Service Tech > Service Tech Signature Toggle Pricing.

Multiple Recipients of Signed Work Orders - Once the site contact signs a Work Order, the Technician now has the option to email the PDF to more than one recipient.
Better Alignment between Service Tech Mobile and Spectrum - In response to customer requests, we've aligned the cost center logic when creating work orders to follow the type first, and then go to the site, rather than following only the site. This way it matches the behavior of the Spectrum application and enables more consistent reporting.
As with each release, we've included security enhancements, performance improvements, and bug fixes.

## April 2022 (Service Tech Mobile 2022 R1)

This release allows for more flexibility for technicians to upload and share files, as well as ensuring correct cost information is associated with labor on job-based work orders.
In summary:

- You can now attach documents from your device to work orders.

- The app now supports multiple file formats.

- You now have the option to enter phase/cost type information when clocking in on job-based work orders.

- You can now share files from Service Tech Mobile's Documents and Images folder.

Additionally, we fixed several customer-reported synchronization bugs.

## September 2021 (Service Tech Mobile 2021 R3)

- Tasks. The Tasks page lists the tasks by associated site equipment. Tasks which have not been linked to any site equipment are listed as "Site Tasks". Select the Add button on this page to create a new task.

- Custom Items. Custom items are the work order user-defined fields. The Custom Items page displays the number of user-defined fields which have data.

- [This link was removed in the the 2023 R3 release.]Report a Problem. If you encounter a problem while using the mobile app in the field, select the hamburger menu and select the Report Problem link to initiate an email report. Provide as much detail as possible about the issue; your notes and the app log will be sent to the Spectrum development team to research the issue.

Note: With this release, Service Tech Mobile reached feature parity with the Service Tech web application. If you are still using the web application, refer to [Migrate to Service Tech Mobile](/en/spectrum/spectrum/service/service-tech/migrate-to-service-tech-mobile) for the steps you need to take to start using Service Tech Mobile.

## May 2021 (Service Tech Mobile 2021 R2)

The following features were added in this release:

- Work Order History - Work Order History is accessed from the Work Order Detail page via the History button. It displays the last 20 work orders for the site, and you can drill down to a full synopsis of the work order with details. We've included the tech's phone number to make it easier to get experienced help.

- Site Equipment - The Site Equipment list is accessed via the Site Equipment button on the work order detail page: The screen lists the equipment for the site as entered into Spectrum. At the bottom of the equipment detail page is a new Service Tech feature: Equipment History. Tap through to see the last ten work orders associated with this machine, and then tap a work order to see the details for that work order. To associate equipment with a work order, so that it shows up in Equipment History you need to allocate a line of labor, or material, or other charge to the item of Site Equipment, or associate a Task with that equipment.

- Company Equipment - Company Equipment is accessed via the Company Equipment button on the Work Order detail page. The Company Equipment detail page has an Add + link. When pushed you will be presented with a list of company equipment. Select the equipment you want to add to the work order, then tap Add Hrs. This equipment can be multi selected and hours added in one step.

## March 2021 (Service Tech Mobile 2021 R1)

The following features were added in this release:

- New Work Order - You can create a new work order from within your Open Assignments by selecting a Site, specifying the Work Order Type and then adding a Work Summary.

- Other Charges - Adding other charges to a work order on the mobile app is even easier than using the web application. Tap the Other Charges button, tap Add, and then select the charges you want to add to the work order.

With the addition of these new features we are coming closer to retiring the Service Tech web application. Once Service Tech Mobile reaches feature parity with Service Tech Web, we will retire the web app and offer only the mobile app. Until then, both the web and mobile versions of Service Tech share the same data.

## November 2020 (Service Tech Mobile 2020 R5)

The following features were added in this release:

- Materials - The Material Page has similar functionality to what is available in Service Tech Web, but has been enhanced to allow for quicker data entry. Once you select your warehouse or truck, you can enter multiple items without needing to re-enter your warehouse or search term after each entry.

- Purchase Orders - The Purchase Order page has been enhanced to request a proposed purchase order by entering the request directly in the PO notes.

- Signature - The Signature page has been enhanced to add Materials and Purchase Orders. Purchase Orders show both the notes and the items which have been added to the PO from the office. Pricing can also be displayed or not, based on the tech's functional security settings.

## September 2020 (Service Tech Mobile 2020 R4)

The following new features were added in this release:

- Signature - The Signature page was added to review and edit the work completed notes, review and edit labor, optionally show pricing, review and edit name and email, capture a signature, and create a PDF of the work order that is emailed to the customer.

- Open/Other Assignments - When the "Is Assignment Done" switch is selected, the assignment tile slides off the screen. This was done to make it easier to understand that it is no longer part of the Open or Other Assignments list.

- Display Pricing - Pricing can be displayed on a tech-by-tech basis using the Field Tech Function Security settings in Spectrum. Technicians who have security settings below those established by function security will not be able to display pricing.

- Add Logo to PDF - Your company or departmental logo can be added to the PDF via Cost Center Entities.

## June 2020 (Service Tech Web 2020 R1)

Included in this release of Service Tech Web:

- Support for new Cost Center hierarchies released in Spectrum 2020 R1: The new Work Order cost center hierarchy has been enhanced so that if cost centers are on the Work Order Type, Spectrum looks to see if the Work Order header has a cost center assigned to it. When there is no cost center on the Work Order Type, Spectrum will look to the service contract for a cost center. When there is no cost center on the service contract, Spectrum will use the cost center assigned to the site.

- Security enhancements

## May 2020 (Service Tech Mobile 2020 R3)

This release included performance improvements and bug fixes.

## April 2020 (Service Tech Mobile 2020 R2)

There are four new features added to the mobile app in this release:

- Timekeeping - Use this page to manage time for the current or previous pay period.

- Support for Job Based Work Orders - When adding labor for a job-based work order, you can now specify a phase for the labor time.

- Add Technician to a Work Order - On the Labor screen you can also add one or more technicians to the work order.

- Support for Flat Rate Work Orders - For work orders that have been designated as flat rate, you can now add labor without adding to billing hours.

## March 2019 (ver 2.9)

- Work Order - The Customer field can now be changed in a work order. This is useful when you want to bill a different division within a company for the work performed on the work order.

## December 2018 (ver. 2.8)

- Dashboard - Technicians can now be assigned to the same work order multiple times. The dashboard will list the multiple assignments.

## June 2018 (ver. 2.7)

- Signatures - The Signatures screen will now allow you to enter multiple email addresses.

## February 2018 (ver. 2.6)

- Timekeeping- The Timekeeping screen has been enhanced to display Non-Work Order Time entries. Work Order Time and Non Work Order Time totals will be dynamically updated on this screen's header as new time entries are added.

- Tasks - Security has been added to the Tasks screen to allow Service Managers to control which technicians can adjust task pricing. If you do not have security to adjust task prices, the Price field will be read-only.

## January 2018 (ver. 2.5)

- Dashboard - For site work orders, you can now change your default billing code if you need to add a different billing code for drive time, weekend time, after hours work, and more. This billing code can also be changed while working offline.

- Tasks- If the corresponding work order has already been billed, the Tasks screen will now display 'Billed' in the Done column.

- Labor - Columns on the main dashboard can now be sorted by selecting the column heading. Work orders that have already been clocked in will automatically display at the top of the dashboard.
