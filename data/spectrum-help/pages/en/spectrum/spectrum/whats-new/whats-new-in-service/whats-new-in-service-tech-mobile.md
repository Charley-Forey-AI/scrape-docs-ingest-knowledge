# What's New in Service Tech Mobile

Review the latest enhancements to your Service Tech Mobile application.

## Service Tech Mobile 2025 R2 (July 2025)

This release includes the second of two critical updates to the application's underlying framework. We have transitioned to Microsoft's latest .NET Multi-platform App UI (MAUI) framework. This update ensures the long-term stability and compatibility of the Service Tech Mobile app with evolving platform requirements.
While this update may not introduce immediate, visible feature enhancements, it is essential for maintaining the app's functionality and security in the future. This change also lays the groundwork for improved performance, and future feature development.
Required: Uninstall the STM app from your device before updating
Due to the significant underlying changes in this update, a clean installation is required. You will need to uninstall your existing app before installing the newly released app.
If you have already installed the new version, uninstall it and install it fresh from the app store.

## Service Tech Mobile 2025 R1 (March 2025)

This release includes the first of two critical updates to the application's underlying framework.
Required: Uninstall your Apple version of the STM app before updating

## Service Tech Mobile 2024 R1 (June 2024)

Credit Card Payment Collection
This release of Service Tech Mobile (STM) includes the ability for field technicians to collect credit card payments using a mobile device, at the moment of service.
To make this possible, you'll need to establish an account with Stripe®, which provides you with required entries in the Spectrum application.
There are several new fields in Spectrum related to this new mobile feature, all in the Service Tech Installation screen:

- Paid by credit Card - use the drop-down field to assign a default dispatch status code to be assigned when a service technician collects a credit card payment payment using their mobile device.

- Enable mobile payment collection - select this checkbox to activate this functionality for your organization.

- Account ID - enter the required information you obtain from Stripe.

Even when enabled at the organizational level, administrators control which technicians can collect payment in the field; the feature appears in the mobile app only to technicians who have been granted the appropriate function level security in the Function Security Maintenance screen.

- [Set Up Service Tech Mobile Credit Card Payment Collection](/en/spectrum/spectrum/service/service-tech/set-up-service-tech-mobile-credit-card-payment-collection)

- [Collect Credit Card Payments using a Mobile Device](/en/spectrum/spectrum/service/service-tech/mobile-overview/collect-credit-card-payments-using-a-mobile-device)

Easier Equipment Management
Whether you add, edit, or delete site equipment for an existing work order, the app now automatically applies your changes to all the work orders pertaining to the same site.
