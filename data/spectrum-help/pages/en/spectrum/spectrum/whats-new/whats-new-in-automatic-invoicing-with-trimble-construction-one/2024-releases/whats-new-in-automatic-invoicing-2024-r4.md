# What's New in Automatic Invoicing 2024 R4

This release adds a filter to the Inbox to display all unprocessed invoices. It also adds a link on the Invoices tab to unprocessed items in the Inbox.

## Link and Filter to View Unprocessed Items in the Inbox

With this release, you can select a link in the upper right-hand corner of the Invoices page to view all unprocessed items in your Inbox.

Also, on the Inbox page, you can select All Unprocessed to view all unprocessed items in the Inbox.
