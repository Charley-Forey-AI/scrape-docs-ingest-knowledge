# What's New in Automatic Invoicing 2025 R10

With this release, for Spectrum users running version 2025 R2 or later, invoice
 details now include a field for Purchase Order Number, which is automatically
 entered if an invoice includes a purchase order number, and a Notes field, which you
 can use to manually enter notes.

## New Fields in Invoice Details

This release adds the following fields to the Invoice Details section of the
 Process page: the Purchase Order Number
 field, which is populated automatically; and the
 Notes field, which you can use to
 manually enter notes. For details on processing invoices, see [Process Invoices](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/automatic-invoicing-for-trimble-construction-one-users/process-invoices). Note: The Purchase
 Order Number and Notes fields are only available if you are
 running Spectrum version 2025 R2 or
 later.
