# What's New in Automatic Invoicing 2023 R5

This release of Automatic Invoicing includes a larger PDF viewer, the ability to view email messages from the Inbox in addition to attachments, a date range filter for both the Inbox and Completed pages, and an enhancement to the Process step to help you ensure you've selected the correct vendor.

## View Email Messages from the Inbox

With this release, in addition to accessing email attachments, you can now open the email body for any email forwarded to your Inbox by selecting it in the Message column.

## Increased PDF Viewer Size

This release increases the size of the PDF viewer to make it easier to read full-page invoices. The increased PDF viewer is available when opening a PDF from the Upload step, the Inbox, or the Completed page.

## Date Range Filter for Inbox

With this release, you can now filter the emails displayed in the Inbox by date. A drop-down menu in the upper right-hand corner of the Inbox page (to the right of the trash icon) provides a number of date range options you can use to filter the list of emails.

## Date Range Filter for Completed Invoices

On the Completed page, you can now filter the list of completed invoices by date. A drop-down menu in the upper right-hand corner of the page provides a number of date range options you can use to filter the list completed invoices.

## Vendor Address Displayed in Process Step

From the Process page, once you select a vendor, the address of the selected vendor now displays below the Vendor field to help you ensure that you selected the correct vendor.
