# What's New in Automatic Invoicing 2025 R7

With this release, when reviewing invoice details, you can use
 the arrow keys to switch between invoices. In
 addition, for Spectrum, this release adds an invoice setting option to set the sales tax
 to zero by default.

## Navigation with Arrow Keys

Now, when reviewing invoice details, you can use the arrow keys to
 switch between invoices more easily.

## Invoice Setting to Always Set the Sales Tax to Zero

This release adds a new option on the Invoice Settings page to
 always set the Sales Tax amount to zero. When this setting is enabled, the Sales Tax
 field will default to 0 for all invoices. This setting does not change the invoice
 total. For details, see [Edit Invoice Settings](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/automatic-invoicing-for-trimble-construction-one-users/edit-invoice-settings).
