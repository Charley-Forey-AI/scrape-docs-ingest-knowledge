# What's New in Automatic Invoicing 2023 R3

This release of Automatic Invoicing improves the accuracy of the Vendor field when processing an
 invoice.

## Improved Accuracy of the Vendor Field

With this release, when processing an invoice, Automatic Invoicing now checks the address
 on the invoice against the primary address in the ERP to auto-populate the Vendor
 field more accurately. This means, even when there are multiple vendors with a
 similar or the same name, Automatic Invoicing can provide accurate results.
