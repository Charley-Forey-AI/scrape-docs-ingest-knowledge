# Set up Email Servers

Use the E-mail Servers screen to set up user email credentials and system default credentials.
This setup is necessary in order to email Spectrum reports directly from the Preview window.

1. Go to System Administration > Security > E-mail Servers.

1. Select New.

1. Enter a name for the email server.Additional fields appear.

1. If needed, refer to [Email Servers - Field Descriptions](/en/spectrum/spectrum/system-administration/set-up-email-servers/email-servers---field-descriptions) as you complete the remaining fields.
