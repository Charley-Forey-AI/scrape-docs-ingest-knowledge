# Import Item Prices from Text File - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldDescription
Read fromSelect Book price to read the book price; Price 1 to read price 1, or Price 2 to read price 2.
Write toSelect  List price to update the list price; otherwise, press Enter to accept the system default and update the Standard cost. If List price is entered, the system displays the following additional window:
Price Update – Price Level Selection:
Enter the default price level (1-5).
