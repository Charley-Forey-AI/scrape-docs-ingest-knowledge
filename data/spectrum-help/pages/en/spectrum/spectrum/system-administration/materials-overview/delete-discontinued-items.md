# Delete Discontinued Items

This page may be used to remove discontinued and/or not
 used items from the item file.
This feature will read the Inventory Items. Items that have
 the Discontinued checkbox selected are eligible for deletion. The
 discontinued "date" will then be read and compared with the date entered on this page.
 Items discontinued on or before the date entered here will be completely removed from the
 system. Active items will not be removed.Important: This
 feature is only available to users with the full Inventory Control module. It is
 recommended that a backup be made before this feature is used.

1. In the Delete through discontinued
 date field, enter the date through which discontinued items should be
 deleted from the system.

1. Select whether to include items with a Discontinued and/or Not used status.

1. Click Continue to proceed with the update, or select the
 Cancel option button to cancel the update and exit the
 page.
