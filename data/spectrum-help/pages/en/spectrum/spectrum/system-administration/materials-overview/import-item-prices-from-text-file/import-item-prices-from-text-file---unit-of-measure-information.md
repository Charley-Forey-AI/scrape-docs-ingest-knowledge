# Import Item Prices from Text File - Unit of Measure
 Information

- If E is
 entered as the unit of measure, the program sets the buy/sell units of measure to 'E'
 and the price per factor to 1.

- If C is
 entered as the unit of measure, the program sets the buy/sell units of measure to 'C'
 and the price per factor to 100.

- If M is
 entered as the unit of measure, the program sets the buy/sell units of measure to 'M'
 and the price per factor to 1000.

It IS necessary to have quotation marks around the unit of measure, for
 example "E" or "C."
