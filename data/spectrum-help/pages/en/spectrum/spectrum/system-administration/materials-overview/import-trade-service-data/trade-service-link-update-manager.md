# Trade Service Link Update Manager

Trade Service is a program that has been integrated with Spectrum to allow you to import an inventory item directly into the system. Items can be imported individually or in batches, and can be brought into the system through a variety of entry pages throughout Spectrum.
When items are imported from Trade Service into these entry pages, they can be brought in as either stock items and set up in the inventory file at that time automatically, or as non-stock items for one-time use. Stock items use the category file to determine whether an item mask is specified. If a mask is specified, the new item is created in Spectrum using the next available item code for that category. If no mask is indicated, then the system adds the item using the EDP number as the new item code. This screen also uses this method.
