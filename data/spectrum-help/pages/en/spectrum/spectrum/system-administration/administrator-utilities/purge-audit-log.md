# Purge Audit Log

Use the Purge Audit Log feature to delete records stored in the audit log through a specified date.
Important: This feature is assigned security category PA, Level 8. Use caution when running this or any other purge because purged records cannot be retrieved.
The Security  > Audit Log contains details about every add, change, and deletion in many database tables (AP, AR, CM, EC, FA, GL, HR, Inventory, JC, PR and SC). As a result, the audit log file can become very large.
