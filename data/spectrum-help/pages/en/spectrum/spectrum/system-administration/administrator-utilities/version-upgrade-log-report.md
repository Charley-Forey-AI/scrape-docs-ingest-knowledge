This screen is used when installing a Spectrum version upgrade with assistance from Viewpoint Customer Support.
Details on how to complete this page are provided in the security manual. Your System Administrator is responsible for
 setup procedures, so any changes should be discussed with them.
If additional copies of the security manual are needed, the
 System Administrator should contact Viewpoint Support through the [Viewpoint Customer Portal](https://support.viewpoint.com/s/).
