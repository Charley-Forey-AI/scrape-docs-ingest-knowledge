# Function/Table Cross Reference Report

The Function/Table Cross Reference Report provides a listing of all files used by a specified function.
To find a function, you can enter a partial function name in the search or use the drop-down list to select a menu option.
Formats are available for printing files associated with the specified function.The report can also be printed to include all functions related to it.
