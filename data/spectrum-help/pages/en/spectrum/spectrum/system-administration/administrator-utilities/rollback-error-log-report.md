# Rollback Error Log Report

To help troubleshoot user issues, use this screen to print a log of rollback messages for a
 specified date range.
Important: Use this page only under the direction
 of Viewpoint Support personnel.
Each SQL error given to an operator is recorded in a log file.
This report displays these errors, along with any comments
 the operator enters in the Rollback Error
 Tracking window.
