# Purge Operator Event Log

Use this tool to delete records stored in the Operator Event Log for the operators, companies, and dates specified.
The fields that appear on this screen are the same as those that appear on the Operator Event Log screen. See [Operator Event Log - Field Descriptions](/en/spectrum/spectrum/system-administration/administrator-utilities/operator-event-log/operator-event-log---field-descriptions).
