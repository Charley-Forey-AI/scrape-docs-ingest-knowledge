# Crystal Report Inquiry

Use this page to identify Crystal Report names associated with standard Spectrum starting pages.
This feature can be helpful when multiple Crystal Reports are available from the same starting page and you need to locate a specific one.

- If you leave the Search for field blank, all reports display.

- You can sort the columns.

- Select the Restore button to reset the columns to their original sort order (by module).
