# Operator Event Log

View a report of user activity in the Spectrum application.
Use this screen to view a report of

- logon attempts

- menu choices selected in the application

- logoff instances

This report may be run for one or multiple operators and can be sorted by date or by operator.
To open this screen, go to System Administration > Security > Operator Event Log.
