# Create a New Company

Create a destination company which will receive the data you copy from your live company.
If you have previously created a Test Company and want to re-use it, instead of creating a new company, you can purge it of all records. See [Purge Company Data](/en/spectrum/spectrum/system-administration/administrator-utilities/create-a-test-company/purge-company-data).
Creating a test company is a good way to test processes and functions outside your live company. In this case, the process includes copying data from your live company into the test/destination company.
To create a new company:

1. Go to System Administration > Security > Operator Maintenance.

1. From the list of available operators, locate and select your operator record.The Edit Operator window opens.

1. Select the Companies tab.

1. Select New.

1. In the New company field, enter a three-character alphanumeric code that you want to use for your company.Note: Don't use dashes, ampersands, slashes, or other special characters.

1. Select OK.The New Company window closes.

This company is the destination company which will receive the data you copy from your live company. See [Copy Company Data](/en/spectrum/spectrum/system-administration/administrator-utilities/create-a-test-company/copy-company-data).
