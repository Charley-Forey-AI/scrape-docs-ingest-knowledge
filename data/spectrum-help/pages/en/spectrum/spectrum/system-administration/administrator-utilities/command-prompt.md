# Command Prompt

The Spectrum command prompt allows you to look up system functions.
This screen is not needed for normal operations. If you've opened this screen by accident, press Escape to return to the Site Map screen.
Access the Command Prompt by pressing Ctrl + Break.
If you select Continue, a prompt window displays and allows you to enter a Spectrum function name (if known) to go to a specific screen, or you can enter PA to return to the Site Map screen. A checkbox is available if you want the software to automatically convert your entry to uppercase.
