# Purge Data Exchange Event Log

To access the various purge features available from the Data Exchange, go to System Administration > Utilities > Purge Data Exchange Event Log.
The Purge Data Exchange Event Log screen offers these purge options:

- Service Event Purge – Deletes the historical details on the individual services that have accessed the database.

- Authorization ID Event Purge – Deletes the current details for the Authorization ID.

Each report has associated selection criteria and sort options available. Once you have defined the criteria, select Continue on the purge screen to remove the records.
