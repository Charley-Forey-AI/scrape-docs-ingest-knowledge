# Default Security Classifications

Spectrum provides a pre-defined screen level security configuration.
The default settings are not fixed. Each organization defines their own operator-level security.
If a particular function is assigned security level 6, operators with security level of 1 through 5 are not permitted to use the function while operators with levels of 6 through 9 are permitted access.
After module security level access is determined, operators can be assigned security levels to control access to the modules.
Function TypeDefault Security LevelAccess
Inquiries and Reports2Inquiry screens, report selections, and certain listings.
Data Entry4Most data entry functions and a limited group of file maintenance screens; access to all level 2 screens and reports.
Senior Data Entry6All file maintenance and data entry screens, plus the use of certain period-end functions; access to all level 2 and level 4 screens and reports.
Supervisory8All level 6 functions; access to installation screens, purges, and other global utilities.
System Administrator9Performing upgrades, applying security settings, and other sensitive administrative activities.
