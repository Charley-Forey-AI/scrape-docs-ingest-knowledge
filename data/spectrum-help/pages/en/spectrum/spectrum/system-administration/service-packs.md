# Service Packs

Spectrum contains a function that validates service pack
 names.
When a vpx name is entered during an upgrade, Spectrum will verify the
 most recent service pack number. If Spectrum determines that an incorrect service pack is
 being loaded, the function will terminate the import. If you experience a service pack
 termination, please contact Spectrum Support by visiting the [Viewpoint Customer Portal](https://support.viewpoint.com/s/).
