# View a List of Operators

You can use the Operator Listing screen for many things. One is to view a list of operators.
Use the Operator Listing screen when you want to print or preview a specific list of operators.
Leave the default selections unchanged to preview or print a list all operators. If you want to narrow your search to specific operators, apply selections in the search filters.

1. Go to System Administration > Security > Operator Maintenance.

1. Select Listing.The Operator Listing screen appears.

1. If you want to narrow the results, change the values in the drop down fields.Example: If you want to view active operators only, with a specific default company assigned:

1. In the Default Company drop-down, choose the specific company.

1. In the Operator status section, choose Active.

1. Select Preview.The report appears in a new tab. Depending on your selections, it may take a few minutes.

Choose from several options which may be shown at the top of the preview window, depending on your settings:

- Email - the E-mail Report screen opens with the report attached.

- Download (button) - downloads the report to your default location.

- Download (icon) - choose where you want to save the downloaded report.

- Print (icon) - send the report to a printer or save as a PDF.
