# Company Installation - Modules Tab

Select the checkbox for each of the modules to be used in the system and that you want to appear on the main Site Map.
Clear checkboxes for all modules that are not included in your software purchase, or if the company or division you are setting up should not have access to it.
If you select the checkbox of a module that is not installed:

- it will appear in the module list

- users will not have access to any of the functions

- Spectrum may work incorrectly during operations in modules you do have

During operations in a given company, if you do not want to access a certain installed module, clear the checkbox for the module. For example, if you are setting up a company that does not require access to the Payroll module, you can eliminate both potential access and additional file maintenance by clearing the Payroll checkbox.
