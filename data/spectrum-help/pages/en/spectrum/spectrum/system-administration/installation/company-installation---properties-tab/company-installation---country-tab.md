# Company Installation - Country tab

Use this tab to set localization options for the current
 company.
Fields
 Descriptions

Lexicon
 From the drop-down list, select a lexicon option. This
 allows you to rename certain labels in the software to match the
 nomenclature used by your company. For example, some users may not use the
 term 'Phase' for parts of the job, instead calling them 'Stage', 'Task',
 'Category' or 'Milestone'. Likewise, the lexicon feature supports different
 labels due to language / country – such as 'hold back' and 'province'
 instead of 'retention' and 'state' in Canada.
Any custom lexicon names created in the Custom Label
 screen are also listed. The default value for all companies is
 <blank>, which will mean 'use browser setting' for the current
 operator.
Note:
 Setting the language code at the company level enables
 cases where a single environment may have both Canadian and US companies.
 It also allows for different custom labels to be used in different
 companies from one environment.


Utilize value added tax (VA) tracking
 Select this checkbox to enable VAT processing in the
 current company. When this checkbox is un-selected, the software will not
 offer the VAT Code Maintenance screen on the Site Map and will not display
 'VAT' fields in Customer or Vendor invoices or elsewhere.

Default tax code
 Use the available drop-down list to select the tax code
 that will then default in on customer and vendor invoices.
