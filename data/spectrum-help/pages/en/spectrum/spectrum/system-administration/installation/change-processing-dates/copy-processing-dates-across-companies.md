# Copy Processing Dates Across Companies

Use this page to copy the processing dates across specified companies for certain modules. This feature allows you to close periods easily without having to change to each company and revise the minimum/maximum dates.
After completing the fields in this page, click Continue to proceed with the copy. During the copy process, the system validates the minimum and maximum date settings in the source company to ensure they are contained within the unlimited fiscal calendar of the destination company.
Fields
Descriptions

(Copy from) Module
Enter the module abbreviation (for example: AR, AP, GL) you want to copy from, or use the available drop-down arrow to select a module.

(Copy from) Company
The current company displays. The update will be based on the settings in this company. To change the current company you need to cancel out of this page and select a different company using the drop-down list on the Command Bar from the Site Map page.

(Copy to) Company
Enter the company you want to copy to, or use the available drop-down arrow to select a module.
