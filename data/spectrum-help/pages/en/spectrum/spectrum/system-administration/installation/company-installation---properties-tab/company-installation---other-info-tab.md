# Company Installation - Other Info Tab

Use this page to enter or look up optional information
 about the company and any user-defined fields. These fields are available when formatting
 electronic Payroll files, writing Crystal-Link reports, and more.
FieldsDescriptions
Main contact
Fax number
Company ID
Web address
E-mail address
Complete the company's general information, including the company's primary contact, fax, and email.
The company ID will be used when generating the default file name for Payroll > Electronic File Table Import/Export.
The web address may be the company's Internet or Intranet address.
Spectrum verifies the validity of any e-mail addresses entered by making sure:

- one @ sign is included in the e-mail address.

- The e-mail address structure is recipient's name@domain name (for example, JohnDoe@XYZ.com).

Shipping addressEnter the company's mailing address and shipping telephone number.
User-defined 1-5Enter any user-defined information that you want to be available for this company.
