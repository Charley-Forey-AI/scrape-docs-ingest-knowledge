# Work Order Functions

The Service Tech Mobile has entry options for Labor, Notes, and Documents & Images.

## Labor Entry

The Labor Entry screen starts with a Work Order labor report. In the example above we can see:

- This Work order has 4 hours of labor, but 5.5 hours will be billed.

- Mark Schwartz worked on two days between September 19th and January 6th for a total of 3.5 hours.

- Kathleen Barnett worked only on September 19th.

When you select Mark's records, you will first see the most recent day, which in this case is January 6th. Select the calendar to review other days.

## Notes

The Notes screen uses an icon to indicate note topics which already have text in them.

## Docs & Images

Documents download only after you navigate to the Docs & Images screen while connected to the internet. The Docs & Images screen displays thumbnails of the images.

- To attach images from your mobile device, select Add > Gallery, locate and tap the image you want to add, and then tap Select > Save.Note: Repeat for each image you want to attach.

- To make changes to a given image, select Edit Description.
