# View or Update a Fillable Form

If you want to view which fillable forms are accessible to Service Tech Mobile app users, or need to update a form currently in use, use Spectrum's Document Imaging (DI) module.
You can view all the forms which are accessible from the Service Tech Mobile app, and you can update them as needed.Note: Only the latest version of a given form is accessible by Service Tech Mobile app users.
Use the search function to view a list, and choose from the results the one(s) you want to update.

1. Go to Document Imaging > Search Documents.

1. Determine if the form in question is accessible in all work orders or only in specific ones.

1. If the form is a company-wide form:

1. In the Cabinet field, select SERVICE TECH.

1. In the Drawer field, select COMPANY FORMS.

1. If the form is specific to a specific work order site:

1. In the Cabinet field, select WO.

1. In the Drawer field, select SITE FORMS.

1. In the Folder field, select the folder pertaining to the work order site.

1. Select OK.All the form names for your chosen context appear in the grid.

1. Select a record in the grid.

1. Select the left-pointing arrow to expand the slider bar.The newest version of the form pertaining to the record appears in the slide bar pane.

1. To update the record with a new form version:

1. Select Attach, then Choose File.

1. Browse to and select the form.

1. Select Upload File.

1. Select Save.The new version the form is now accessible by Service Tech Mobile app users, and the prior version is no longer accessible.
