# Create a Dispatch Status - Paid by Credit Card

Set up a dispatch status which the Service Tech Mobile app should assign to the work order when the work order has been paid by credit card.

1. Go to Work Order > Maintenance > Dispatch Status.

1. Select New.

1. In the Dispatch status field, enter a status.

1. In the Description field, enter a value which will indicate this work order has been paid by credit card, for example, `Pd by CC` or similar.

1. Set the Work order status to Finished.Important: This status is required so that when a work order is paid by credit card, it immediately becomes inaccessible in the mobile app.

1. Select OK.

When the service tech collects a credit card payment on a work order, the mobile app will assign your chosen Dispatch status and set the work order status to Finished.
