# Service Contract Cost History Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Site Contract number Customer code Contract type Work order type Salesperson code
Use these fields to create the report
 that will be most useful to you. After specifying your report criteria, if you
 wish to save your entries, click the Save Current button
 in the Sidebar.
From/to cost date
Enter the G/L date range that you wish
 to include on the report.
Equipment subtotals? Active contracts only?
Use these checkboxes to include more
 details about the subtotals or to limit the report to Active contracts only.
Include unposted labor costs?
When the selected Calculation method is 'Actual
 cost', select this checkbox to include any unposted labor transactions for
 each selected contract.If both 'Include unposted labor
 costs' and 'Include unposted material costs' are selected, unposted other
 charge transactions will also be included on the report. The unposted costs
 of Hourly, %Bill and %Cost other charge transactions will be assigned to the
 service contract specified in the Work Order header. If no service contract
 is specified, these unposted costs will not be calculated.

Include unposted material costs?
When the selected Calculation method is 'Actual
 cost', select this checkbox to include any unposted material transactions for
 each selected contract.If both 'Include unposted labor
 costs' and 'Include unposted material costs' are selected, unposted other
 charge transactions will also be included on the report. The unposted costs
 of Hourly, %Bill and %Cost other charge transactions will be assigned to the
 service contract specified in the Work Order header. If no service contract
 is specified, these unposted costs will not be calculated.

Calculation method
Select a calculation method for this
 report:

- Actual
 Cost shows costs with a G/L date within the selected date
 range.

- Standard Cost shows all costs associated with invoices
 that have a G/L date within the selected date range.
