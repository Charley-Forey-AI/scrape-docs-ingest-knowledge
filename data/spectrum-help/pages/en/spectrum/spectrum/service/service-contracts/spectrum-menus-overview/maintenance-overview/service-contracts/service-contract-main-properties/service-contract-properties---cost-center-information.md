# Service Contract Properties - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, the Cost center field will be available for entry on this tab.
The cost center must be valid and active in the current company, and the operator must
 have authorization to this cost center. The cost center must also be valid for the G/L
 codes specified in Contract Type File Maintenance, and the cost center must be valid for
 the customer code assigned to the contract.
If the service contract cost center is blank, the cost center of the associated site will be used and all validations will be checked against the site cost center.
