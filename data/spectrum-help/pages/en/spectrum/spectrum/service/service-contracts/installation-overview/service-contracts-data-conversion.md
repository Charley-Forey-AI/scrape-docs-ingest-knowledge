# Service Contracts Data Conversion

Below is a summary of the steps needed to complete
 installation. The steps on this list correspond to the Conversion Worksheet found in the
 User's Guide. It is important that these steps be performed in the order listed.

1. Complete the Service Contracts Installation
 screen.

1. Set the processing dates to the new period for Service
 Contracts using Processing Date
 Change.

1. Enter the contract types using Contract Type File Maintenance.


1. Print the contract types and check for accuracy.


1. Enter the service contract tasks using Work Order >  Task >  File Maintenance.

1. Print the tasks and check for accuracy.

1. Enter the zip code zones using Work Order >  Zip Code Zone
 Maintenance.

1. Enter the desired service contract items and
 descriptions using Service Contract
 Description.

1. Complete the Service Contracts portion of the
 Processing Date Change
 screen.

1. Enter the outstanding service contracts as of the
 conversion date using Service
 Contracts.

1. Print the outstanding service contracts and check for
 accuracy.

1. Enter equipment file maintenance for all outstanding
 service contracts using Service
 Contracts.

1. Print the systems and check for accuracy.

1. Enter the system components using Component File Maintenance.

1. Print the components and check for accuracy.

1. Schedule site visits for service contracts in progress
 using Schedule Contract
 Visits.
