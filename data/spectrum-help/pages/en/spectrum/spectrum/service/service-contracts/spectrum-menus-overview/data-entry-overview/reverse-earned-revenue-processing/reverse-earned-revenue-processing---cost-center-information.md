# Reverse Earned Revenue Processing - Cost Center Information

If cost centers are being used, then the cost center
 assigned to the site of the contract is used in the Reverse Earned Revenue Processing screen.
The software includes the contract in the temporary table only if you have permission to
 access that contract. No cost center inter-post entries are generated during this
 reversal, since matching amounts are always posted to a Debit and Credit account for
 each contract.
