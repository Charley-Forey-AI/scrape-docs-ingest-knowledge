# Service Contract Visits

Use this screen to manage scheduled visits for the selected
 service contract. From the Service Contracts
 Info Bar, select PropertiesVisits.
The variance columns in the list box calculates values from other columns:

- Variance
 hours: Variance represents the difference between the actual hours and
 budget hours.

- Variance hours
 %: Percent variance is calculated as the quotient of the variance and the
 budget hours.

- Variance
 cost: Variance represents the difference between the actual cost and
 budget cost.

- Variance cost
 %: Percent variance is calculated as the quotient of the variance and the
 budget cost.

Fields/Buttons
Descriptions

Site code
Enter a site code.
Contract
Enter a service contract for the
 selected site.
Total contract visits
Click the Edit button to change the
 number of Total contract
 visits for this contract. The number of Currently scheduled visits
 will display to the right of this field.
Total contract amount Total billed-to-date
For service contracts set to bill from
 the work order, summary billing information will display in the header of this
 screen. The Total contract
 amount is the sum of the flat rate task price, and the
 Total
 billed-to-date is the work order revenue generated across all
 visits for this contract.
Buttons

View visits
Click this button to view the selected
 visit. When editing visits, this button will no longer be available, but if the
 selected visit has already been uploaded to the work order, it will be
 view-only.
Copy visit
Click this button to copy a scheduled
 visit to the contract by an incremental amount. For each task added to the new
 visit, the unit price stored in the visit will be multiplied by the quantity of
 the task and summed across all tasks to arrive at the Total price of the
 visit.
