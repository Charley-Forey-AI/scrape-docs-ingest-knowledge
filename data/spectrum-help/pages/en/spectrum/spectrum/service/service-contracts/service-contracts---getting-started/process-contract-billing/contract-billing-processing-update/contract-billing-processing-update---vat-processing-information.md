# Contract Billing Processing Update - VAT Processing Information

If the 'Utilize value added tax (VAT) tracking?' checkbox is selected in General Ledger Installation, select the Assign VAT tax code checkbox and specify a Tax code: Goods & Services Tax (GST) or Harmonized Sales Tax (HST) for the invoice.
