# Installation Overview

Before you set up installation options, ensure that the
 checkbox for the Service Contracts module has been selected on the System Administration >  Installation >  Company - Modules tab. Be sure to save any changes that you make on this tab.
Note: The System Administrator should have already completed this
 screen.
To set up Service Contracts, select  System AdministrationInstallationService Contracts. The screen has
 two tabs that you can use to set defaults for different service contract options, including
 auto-sequencing, sorting, work order dispatch status and priority, and user-defined fields.
When the installation screen is selected for the first time, the
 software will beep and a dialog box stating "CONTROL RECORD HAS NOT BEEN CREATED OK TO
 PROCEED" will display on the screen. Once this Installation screen has been completed and
 the information saved, the message will not appear again.
