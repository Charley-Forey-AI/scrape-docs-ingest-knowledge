# Create Work Orders for Scheduled Visits - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Site Contract Customer Contract type
Use these fields to create the report
 that will be most useful to you. After specifying your report criteria, if you
 wish to save your entries, click the Save Current button
 in the Sidebar.
Cost group
Enter a cost group code to include work
 orders based on cost center. This field will only be available if cost centers
 are enabled for the current company.
From/to scheduled date
Enter the scheduled date range that you
 wish to include on the report.
New work order

Assign dispatch status
Select a dispatch status code for the
 new work order.
Assign requested technician?
Select this checkbox to automatically
 schedule the requested technician for the contract site to the new maintenance
 work order.Note: If this checkbox is
 selected but the requested technician is not in the site file or not
 eligible for scheduling, the update will simply set the scheduled date for
 the new work order.

Assign scheduled date of the visits?
Select this checkbox to automatically
 assign the scheduled date from the contract setup to the new maintenance work
 order.
