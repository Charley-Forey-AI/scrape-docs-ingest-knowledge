# Add a Service Contract Description

You can add a service contract description.

1. From the Site Map, select Service Contracts >  Maintenance >  Contract Description.

1. Click New.

1. The New Service Contract Description window opens. Enter a unique code to be specified whenever a service contract should receive this description.

1. Enter a full Description for the code and click OK to save the new description.
