# Create Work Orders

Create work orders based on scheduled service contract
 visits.
Options specified on
 the Service Contract Installation screen determine what is recorded in the new work order.
 The system cannot create work orders for inactive contracts but can create work orders for
 service contracts effective on a future date.
To create a work order:

1. Go to Service Contracts > Data Entry > Create Work Orders.

1. Enter a Site code.

1. Use From
 scheduled date and To
 scheduled date to enter a date range for service visits.The system
 searches for all open contracts scheduled for visits in the selected time frame.

1. Select the dispatch status for the new work order. For
 example, Open.

1. To automatically schedule the preferred technician for
 the contract site, select Assign
 requested technician.

1. To automatically schedule a service date on the work
 order, select Assign scheduled date of
 the visit. Leave this checkbox blank to have the dispatcher manage
 scheduling.

1. Select Preview to review the Create Scheduled Work Orders Pre-Update Register.This report shows
 details of the work orders that the system will create.

1. After you confirm work order information shown on the
 Pre-Update Register, select the Create
 Scheduled Work Orders tab (displays automatically when you preview the
 Pre-Update Register).

1. To schedule service contract scheduled visits and
 create work orders based on the Pre-Update Register, select Continue.Important: Retain the Post-Update Register as a
 permanent audit record of the company.

The system generates a
 Post-Update Register. This report shows service visits that were updated.
