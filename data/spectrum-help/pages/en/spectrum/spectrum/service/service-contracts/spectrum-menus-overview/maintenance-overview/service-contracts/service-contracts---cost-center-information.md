# Service Contracts - Cost Center Information

If cost centers are being used, when adding or changing the
 customer code, the contract cost center must be valid for the customer before you will be
 allowed to proceed.
The contract cost center must also exist and be valid for the contract type G/L account.
 The list box will contain a Cost center column. This column will contain the code +
 description + source in parentheses for each contract. This column will only display
 cost centers that you are permitted to access.
