# Service Contracts - Field Descriptions

Use the table below for reference when completing the fields on this screen.
ButtonsDescriptions
Edit/DeleteUse these action buttons to keep your data current by making changes or removing obsolete records.
E-MailSelect to open a new email message window and send an email message to a selected recipient. Note: An email program (such as Microsoft® Outlook) must be actively running in order for the E-mail button to open the email window.

FormSelect a record, then select this button to print the selected service contract.
RenewSelect a record, then select this button to open the [Renew Service Contract](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contracts/renew-service-contract) window. This update copies contract properties, equipment, notes, and exceptions set up for the expired contract to the new contract.
CopySelect a record, then select this button to open the [Copy from Master Service Contract](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/service-contracts/copy-from-master-service-contract) window to create a contract from an existing one.
