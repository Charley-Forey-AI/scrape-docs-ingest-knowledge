# Service Contract Visits Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Site Contract number Contract type
Use these fields to create the report
 that will be most useful to you. After specifying your report criteria, if you
 wish to save your entries, click the Save Current button
 in the Sidebar.
From/to scheduled date
Enter the scheduled date range that you
 wish to include on the report.
Include visits already updated to Work Order?
Select this checkbox to include all
 visits, regardless of update status. If this checkbox is not selected,
 scheduled visits that have already been transferred to Work Order will be
 deselected.
Include equipment part notes?
Select this checkbox to include
 equipment part notes on the report as a sub-section, just below the
 equipment/task line. Equipment notes will not repeat on the report if there are
 multiple tasks listed for the same equipment code on the same visit, and will
 appear after the last occurrence of the equipment code.For the equipment parts notes to show on this report the note topic "PARTS"
 needs to be added to WO Equipment Note Topics Maintenance. In Service
 Contract Covered Equipment: Equipment Notes, add a note to the PARTS topic
 and select the "Include equipment part notes?" checkbox.

Include task part notes?
Select this checkbox to include task
 part notes on the report as a sub-section, just below the equipment/task
 line.
Contract status
Select to include Proposed, Active, or
 Inactive contracts (or any combination of these three status types) on the
 report.
