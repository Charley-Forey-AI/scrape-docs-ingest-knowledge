# Year-End Procedures

You can use the Year-End procedures to backup data and
 reset for a new year.

- Follow all the Period End Procedures.

- Make a backup and label it year-end.

- Reset the beginning start date for week one in the new year using
 Service Contract Calendar
 Maintenance.
