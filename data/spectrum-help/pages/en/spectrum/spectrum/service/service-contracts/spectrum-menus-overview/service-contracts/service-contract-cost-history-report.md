# Service Contract Cost History Report

This screen is used to print past cost information.
This report displays the work orders that were created from Service Contracts, which allows
 you to view the work order numbers that were created for the specified contracts.

- When the 'Include unposted labor costs' option is selected, service
 contract add-ons for pre-time card entries will be included on this report.

- If the 'Payroll overhead' option is selected for the work order,
 unposted labor cost transactions will be calculated and then increased by the specified
 overhead percentage or rate/hour amount.

- Company equipment costs will be included on the report for ER
 transactions.
