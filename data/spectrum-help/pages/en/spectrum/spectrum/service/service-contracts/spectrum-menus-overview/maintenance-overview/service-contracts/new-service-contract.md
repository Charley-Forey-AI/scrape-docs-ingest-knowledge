# New Service Contract

This window opens when setting up a brand new service
 contract. Refer to the field descriptions in the table below when you are completing the
 fields in this window. Click OK to
 save and commit the service contract.
Fields
Descriptions

Site
Enter a site code or click the
 drop-down arrow to select a site ID.
Proposal?
Select this checkbox if this is a
 proposed service contract.
Contract #
Enter a contract number. The system
 will verify that the code does not already exist. If the Service Contract
 Installation for 'Use auto-sequenced service contract numbers?' checkbox is
 selected, then when you leave the 'Contract #' field blank, the next available
 contract number will default automatically. If a non-sequenced code is desired,
 the simply type the alternate code here.
Description
Enter a description or click the
 drop-down arrow to select a description.
Contract type
Enter a contract type of click the
 drop-down arrow to select a contract type, as set up in [Contract Type File Maintenance](/en/spectrum/spectrum/service/service-contracts/spectrum-menus-overview/maintenance-overview/contract-type-file-maintenance)
Work order type
Enter a work order type or use the
 drop-down arrow to select a type.
Customer
The customer defaults from the site
 information.
