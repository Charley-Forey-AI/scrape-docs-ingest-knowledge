# Contract Type File Maintenance

Use the Contract Type File Maintenance screen to manage
 types of service contracts (for example, Full Maintenance, Inspection, Warranty, or Labor only),
 including default earned revenue calculation information and deferred revenue G/L accounts.
From the Site Map, select
 Service Contracts > Maintenance > Contracts/Contract.
Contract types are used when service contracts are set up. Click New to add a new contract type or Edit to make changes to an existing contract type. Once a G/L account code has been assigned to a contract type, then that G/L account will be used by all operators for contract billing and earned revenue processing.
Note: Consult your accountant when assigning G/L account
 codes to record contract billings (invoices) and accrued service contract deferred revenue
 in your company.
