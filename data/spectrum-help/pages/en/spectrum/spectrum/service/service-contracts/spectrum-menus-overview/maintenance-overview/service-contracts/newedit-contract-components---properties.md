# New/Edit Contract Components - Properties

Use this tab to add general information about components associated with a service contract. If you are adding a new component, once you select an existing component code the rest of the fields will be enabled.
Fields/Buttons
Descriptions

Identification

Description
Enter a description for the contract
 component in this field.
Install date

Make
Enter a valid make code for the
 contract component, or use the drop-down arrow to search for available
 makes.
Model
Enter a valid model code for the
 contract component, or use the drop-down arrow to search for available
 makes.
Serial number
Enter a serial number for the component
 in this field.
Current info

Location
Enter the site location for the
 component in this field.
Contract
The designated contract code for the
 service proposal displays in this field. If you are
 creating a new service contract, enter a new contract code in this field to
 enable the rest of the fields on the Properties tab.

Contract component comment
This field will either display any
 existing comments recorded for this component, or allow you to enter new
 comments.
Warranty notes
Enter any warranty notes you want to
 include for this component entry.
Notes button
Clicking this button will move the
 cursor to the Contract component
 comment field. See [Notes](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/notes) for more
 information.
