# Service Contract Profitability Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Format

Summary
When the Summary format is selected,
 this box will be available. Select whether to view Contract-to-date or
 Life-to-date
 information on the report.

- The Contract-to-date option will show revenue, cost and
 profit information for the contract Start/End period currently specified
 in Service Contracts.

- The Life-to-date option will show total revenue, cost and
 profit through the specified period, including amounts preceding the
 current contract start date. This option is useful for companies that
 maintain a single contract and periodically change the Start and End
 dates each time the contract is renewed.

Detail
When the Detail format is selected,
 this box will be available. Select whether to Include year-to-date and/or
 Include
 life-to-date information on the report.

- The
 Include year-to-date option will show revenue, cost and
 profit information for the specified fiscal year.

- The
 Include life-to-date option will show total revenue, cost
 and profit through the specified period, including amounts preceding the
 current contract start date.

Selections

Site Contract Customer Contract type Salesperson
Use these fields to specify the records
 you want to print, or double-click on these fields to select from a list of
 valid options.When this screen is accessed using the
 Service Contract Info Bar, the current site code and contract code default
 in context from the Info Bar.

Through period-end date
Enter a period-end date for the report
 in this field. The corresponding Fiscal year and Period will display to the right of this field.This field will default to the current year and period,
 based on the current Service Contract processing date.

Active contracts only?
Check this box to include only active
 contracts on the report.
Sort by
Select how to sort this report:

- Site

- Contract type

- Salesperson

Calculation method
The Actual cost method is
 recommended for companies using 'direct work order cost' accounts and Service
 Contract earned revenue processing to manage deferred revenue. When this method
 is selected it is also possible to elect to include certain unposted
 transactions in order to review current profitability information.The Standard cost, as billed method is recommended for companies
 that track standard costs in Work Order instead of 'actual cost' and treat
 contract billings as earned revenue. When this method is selected the
 'Earned revenue' will be set automatically equal to the billed
 amount.

Unposted transactions

- When the Include un-invoiced work order
 revenue checkbox is selected, include the cost of
 un-invoiced work order transactions for the selected contract.

- When the Include unposted labor
 costs checkbox is selected, include the cost of unposted
 labor transactions for each selected contract. Company equipment costs
 will be included in labor actual costs.

- When the Include unposted material
 costs checkbox is selected, include the cost of estimated
 and unposted material transactions for each selected contract.

- If both Include unposted labor
 costs and Include unposted material costs are selected, unposted
 other charge transactions for each selected contract will also be
 included on the report. The unposted costs of Hourly, %Bill and %Cost
 other charge transactions will be assigned to the service contract
 specified in the Work Order header. If no service contract is specified,
 these unposted costs will not be calculated.
