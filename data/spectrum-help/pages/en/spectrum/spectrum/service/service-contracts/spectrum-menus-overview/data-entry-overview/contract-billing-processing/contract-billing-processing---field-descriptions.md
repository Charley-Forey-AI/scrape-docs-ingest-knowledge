# Contract Billing Processing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Site Contract Customer
Use these fields to create the report
 that will be most useful to you. After specifying your report criteria, if you
 wish to save your entries, click the Save Current button
 in the Sidebar.
Cost group
Enter a cost group code to select
 departments based on cost center. This field will only be available if cost
 centers are enabled for the current company.If the cost
 center is blank on a given service contract, the cost center from the work
 order site will be displayed in the 'Cost center' column on the
 report.

From/to date
Enter a date range for the records you
 want to include on this report. The dates selected must be within the Service
 Contract minimum/maximum processing date range.
G/L date
The current Service Contracts
 processing date defaults but may be overwritten.
Invoice date
The current Service Contracts
 processing date displays. Press Enter to accept this default, enter a different invoice
 date.
