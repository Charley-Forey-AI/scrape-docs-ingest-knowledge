# Print Contract Equipment Labels - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Print options
Select the print options you want.

1. To print an alignment mask:

- Load the labels into the printer.

- Select Alignment mask.
 This prints XXXs on the labels where information will be printed.

- Check whether the XXXs are aligned correctly.
 If the alignment is incorrect, adjust the printer and then print
 the alignment mask again.

- When alignment of the XXXs is correct, select
 Label
 print.

Site
Enter the site ID you want to print on
 the report, or press Enter to accept the system default of ALL.
Contract
Enter the contract to print, or press
 Enter to accept
 the system default of AL.These labels can be placed on
 the customer's system. They contain your company name and telephone number
 as entered on the installation screen. Customers who call for service may
 read the site, contract, and system information off of the
 label.
