# Service Contract File Listing - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Format
Select the report output format.

Selections

Site
When this screen is accessed using the
 Site Info Bar, the current site code will default in context from the Info
 Bar.
Contract
When this screen is accessed using the
 Service Contract Info Bar, the current site code and contract code will default
 in context from the Info Bar.
Equipment
This field will be enabled if either of
 the equipment formats are selected above.
Include checkboxes
Select the checkboxes for the notes
 and details you want to include on this report. Some of these options will
 change depending on which report format you selected above.
Contract status
Select to include Proposed, Active, or
 Inactive contracts (or any combination of these three status types) on the
 report.
Equipment status
Select the status of the equipment you
 want listed on the report.
