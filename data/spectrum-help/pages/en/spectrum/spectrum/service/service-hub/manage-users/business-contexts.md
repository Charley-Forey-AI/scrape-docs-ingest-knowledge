# Business Contexts

An ERP Admin or Company Admin whose Trimble ID is associated with additional business
 contexts can access different pages of Service Hub from each context.
Note: To switch business contexts after you have logged in, select your user profile in the upper right-hand corner, and then select Change Context.

## Business Context Definitions

Customer UserCustomer Users are the end users of Service Hub. Their permissions
 depend on which user role they are assigned. For a side-by-side comparison
 of user role permissions, see [User Roles](/en/spectrum/spectrum/service/service-hub/manage-users/user-roles).This
 business context is the default for the following user roles:

- Customer Admin

- Finance User

- Site Admin

- Portal User

ERP AdministratorThe ERP Admin or Company Admin is a Service Provider
 team member whose responsibility is to configure the portal for the Service
 Customers. These administrators have the exclusive ability to configure the
 Stripe payment portal, invite new admins, and define service request email
 recipients. They can also onboard Customer Users.This
 business context is the default for the following user roles:

- ERP Admin

- Company Admin
