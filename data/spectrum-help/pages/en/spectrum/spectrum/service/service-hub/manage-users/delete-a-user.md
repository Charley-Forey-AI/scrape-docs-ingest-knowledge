# Delete a User

Follow these steps to permanently delete a Service Hub user.
You must be an ERP Admin, Company Admin, or Customer Admin to complete these steps.

1. Navigate to the Manage Users page.

1. Use the search bar to find the appropriate user by name or email.

1. Select the ellipses icon  next to the user you want to
 delete.

1. Select Delete User.

1. Select Delete User again to
 confirm.A pop-up message displays to confirm that the user was deleted
 successfully.

The user is deleted.
