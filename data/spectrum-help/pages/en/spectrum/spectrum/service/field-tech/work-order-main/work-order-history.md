# Work Order History

Use this screen to view all other work orders, past and present, for the current site work order.
Work orders are sorted by finished date, but you can sort them by any column heading just by clicking it.

- Click New Work Order to generate a new work order for the current site or job. Once added, you will be able to call up the work order in Field Tech to add notes or other information.

- Click View to open the highlighted work order in the Field Tech Work Order Main screen.
