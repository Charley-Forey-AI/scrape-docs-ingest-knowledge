# Work Order Main - Field Descriptions

Use the table below when completing the fields on this screen.
Field/ButtonDescription
Work orderDisplays the selected work order number. The summary text from the Work Order Main Properties screen appears below this field.
Clock In/OutNote: This button doesn't appear if the work order status is 'Proposed' or 'Complete'.
Click to clock in or clock out of a job. The system takes the following actions, based in part upon the settings in Field Tech Installation:

- Adds new labor entries for the technician assigned to this work order

- Adds the technician's billing codes

- Records the time stamp

- Calculates hours worked

- Updates the dispatch status

StatusDisplays the assigned dispatch code. Click to select a different status.Note: If the status is 'Complete' or no dispatch status codes exist in the current company, the Select New Status window is not available.

Site alertDisplay only. If the Display in work order checkbox is selected in the Site Main Properties screen, the alert text displays in this field.

Address These fields are display only.Values come from the Work Order Main Properties screen.
