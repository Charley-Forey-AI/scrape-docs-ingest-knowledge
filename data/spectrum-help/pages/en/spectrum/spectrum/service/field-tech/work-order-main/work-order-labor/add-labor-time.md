# Add Labor Time

Use this window to manually add or edit labor hours and billing codes for a technician, and override any default labor entries.
Note: You can always add more labor time to a work order,
 even one that is billed. Time cannot be entered onto work orders with a Proposed
 status.
Field
Description

Technician
The technician will default to the current operator, but can be overridden.

Pay type
Select a pay type for the labor hours entry:

- Regular

- Special rate

- Overtime

- Double time

Work date
Enter a work date for the labor hours entry.

Time in Time out Hours worked Billing hours
Enter a time in and time out for the labor hours entry. Spectrum will automatically calculate the hours worked and billing hours.

Phase Cost type
Specify a phase and cost type. Spectrum will verify that the phase and cost type are valid for the current job.
Note: If this is a site work order, these fields are hidden.
