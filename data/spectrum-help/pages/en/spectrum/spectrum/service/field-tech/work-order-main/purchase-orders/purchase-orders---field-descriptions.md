# Purchase Orders - Field Descriptions

Use the table below when completing the fields on this screen.
Field/Button
Description

Vendor
Enter a vendor code for the proposed purchase order.

Purchase for
Select whether the purchase order is for the current work order, or the technician's (truck) warehouse. This option will only be available if the current technician has an assigned warehouse.

Internal notes
Enter any notes related to this purchase order. These notes will display in the Internal Notes window of the resulting purchase order.
