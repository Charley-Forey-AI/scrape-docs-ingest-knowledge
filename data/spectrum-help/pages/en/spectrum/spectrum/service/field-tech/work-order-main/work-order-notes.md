# Work Order Notes

This screen displays when you click the Notes link on the
 Info Bar. Use this screen to view, add, and edit notes related to the selected work order.
The Note Topics window provides a Notepad window in which virtually unlimited notes may be
 entered. To include standard text on the work order, click the insert standard text icon
 (image\Std_comments.gif) on the Notepad window and select the standard text to be included
 on the work order. These notes will print on the work order.
