# Service/Field Tech Installation

Use this screen to set up clock-in/clock-out dispatch statuses, select a Crystal finished work order form, decide whether to roll up technician's labor hours to the next quarter hour or half hour, or exclude certain dispatch status codes.
Note: Fields on this screen apply to Field Tech
 dashboard apps, Service Tech, and the Service Tech Mobile app. See the field descriptions
 below for any product specific requirements.
