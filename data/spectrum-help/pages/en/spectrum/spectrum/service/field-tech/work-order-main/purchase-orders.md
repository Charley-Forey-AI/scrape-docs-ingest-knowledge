# Purchase Orders

Use this screen to view all purchase orders for the current work order, providing updated quantity ordered and quantity received for each item on order. In addition, use this screen to initiate a proposed purchase order, provided the operator has security authorization to add new purchase orders.
Note: If the selected work order is a job work order, the
 listbox will be empty and the New
 P.O. button will be hidden.
