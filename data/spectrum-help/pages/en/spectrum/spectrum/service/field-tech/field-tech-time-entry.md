# Field Tech Time Entry

The Field Tech Time Entry screen will show all transactions for the selected work date, and allow the technician to add labor time or edit existing entries that are not already processed.
Note: If the selected work order is currently assigned a
 dispatch status that is flagged for exclusion in Field Tech Installation, you will be
 unable to edit or delete the entry.
