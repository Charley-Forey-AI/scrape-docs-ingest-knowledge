# My Open Work Orders

This dashboard app displays all open work orders assigned
 to the current technician.
Open work orders are defined having one of these dispatch codes:

- Open

- Assigned

- Arrived

- None

- Any work order with no dispatch code
 assigned and which is not yet complete.

Work orders are sorted in order of scheduled date, scheduled time,
 work order number, and site name. If the work order has no scheduled date, it is sorted by priority description in place of date.
Note: Work orders with a dispatch status that is flagged for
 exclusion in Field Tech Installation are hidden.
Click on a work order to open the [Work Order Main](/en/spectrum/spectrum/service/field-tech/work-order-main) page in a new tab.
