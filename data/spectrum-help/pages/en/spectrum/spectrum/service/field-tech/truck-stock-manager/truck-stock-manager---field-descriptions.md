# Truck Stock Manager - Field Descriptions

Use the table below when completing the fields on this screen.
Field/Button
Description

New P.O.
If authorized, click this button to enter a new purchase order.

Truck warehouse
The name of the (truck) warehouse assigned to the technician displays in this field.

Item code Item description EDP part number Primary vendor Vendor part #
The selected item code and description, EDP part #, vendor and part # assigned to the item will display in these fields.

Available quantity
The available quantity for this item will display in this field. This is the same quantity that appears in the Item Warehouse Details screen.

Pending adjustment
All unposted inventory adjustments for this item and truck warehouse display in this field.

Net truck stock
The sum of the available quantity and pending adjustments displays in this field.

Reorder point Quantity on order Last received
The selected item's reorder point, quantity on order, and last received date from the Item Warehouse Details screen displays in these fields.
