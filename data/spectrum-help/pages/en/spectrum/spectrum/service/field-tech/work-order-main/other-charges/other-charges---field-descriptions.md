# Other Charges - Field Descriptions

Use the table below when completing the fields on this screen.
Field
Description

Other charge
Select an other charge item code. The item description will display to the right of this column.

Billed?
For site work orders, this column will display 'Yes' if the other charge record has already been invoiced (that is, part of the 'Billed-to-date' total in Work Order Summary). Otherwise, this field will be blank if part of 'Current billing' or 'Unselected' totals.
For job work orders, this column will display 'Yes' if the work order is completed. Otherwise, this field will be blank if the work order is still open.

Phase Cost type
If this is a job work order, enter a phase code and cost type. The Phase description will display to the right of this column. If an existing other charge record has already been added to this work order, the software will default the phase code from that record. The software will validate that the phase + cost type is valid for the current job. Completed phases will be disallowed.
Note: These fields will be hidden if this is a site work order.
