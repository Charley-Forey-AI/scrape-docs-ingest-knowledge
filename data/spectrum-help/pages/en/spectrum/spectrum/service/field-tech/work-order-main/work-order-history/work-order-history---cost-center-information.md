# Work Order History - Cost Center Information

If the cost center feature is enabled in the Enterprise
 installation screen for this company, the software includes activity only if the operator has
 permission to access the cost center assigned to the work order. The software compares the
 cost center assigned to the work order with cost centers in the operator's assigned scheme; if
 the cost center is not included, the work order is not displayed.
