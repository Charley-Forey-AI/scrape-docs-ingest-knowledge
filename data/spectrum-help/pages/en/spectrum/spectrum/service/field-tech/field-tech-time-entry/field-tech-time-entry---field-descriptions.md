# Field Tech Time Entry - Field Descriptions

Use the table below when completing the fields on this screen.
Field/Button
Description

Technician
The selected technician will display in this field. When a new entry is created in this screen, the technician's billing code will be stored in the Work Order Transaction Detail record.

Work date
Enter a work date for the labor hours entry.

Total work order hours Other labor hours
The total Work Order hours for this technician, non-Work Order hours currently recorded in Pre-Time Card Entry, and the sum of all hours display in the header.

- Click the Work Order button to open the [Work Order Main](/en/spectrum/spectrum/service/field-tech/work-order-main) screen in context to the selected work order.

- If other labor hours are available and the Employee Kiosk module is available,
 click the Other
 Hours button to open the My Time
 Entry screen in context to the operator.

Work order
Enter a work order number. Proposed or Completed work orders, and work orders with an excluded dispatch status will not be included.

Pay type
Select a pay type for the labor hours entry:

- Regular

- Special rate

- Overtime

- Double time

Time in / out Hours worked Billing hours
Enter a time in and time out for the labor hours entry. Spectrum will automatically calculate the hours worked and billing hours.

Phase Cost type
If this is a Job work order, specify a phase and cost type. Spectrum will verify that the phase and cost type are valid for the current job.
Note: If this is a site work order, these fields will be
 hidden.
