# Work Order Labor - Assigned Technicians

Use the table below to complete the fields for clocking in and adding technicians.
Button
Description

Clock In
Click this button to add new labor entries, add a billing code, and change the dispatch status for the selected technician. The clock in time will be the current system time.
Note: If the work order status is 'Proposed' or 'Complete', this button is disabled.

Add Tech
Click this button to add new technicians to the work order. If the operator does not have security authorization to add new technicians, this field will be hidden.
Note: If the work order status is 'Complete', this button is disabled.

Summary hours
The quoted hours, total hours worked and total billing hours on the current work order will display to the right of this listbox.

Refresh
Click this button to recalculate summary hour totals.
