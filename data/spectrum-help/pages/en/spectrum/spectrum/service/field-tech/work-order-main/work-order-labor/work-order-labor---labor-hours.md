# Work Order Labor - Labor Hours

Use the table below to complete the fields for clocking out and adding and editing labor for the selected technician.
Button
Description

Clock Out
Click this button to clock out and change the dispatch status for the selected technician. The clock out time will be the current system time. The system will calculate the hours worked by comparing the clock in time with the clock out time for the selected technician.
Note: If the work order status is 'Proposed' or 'Complete', all buttons in the Labor Hours section are disabled.

New
Click this button to open the [Add Labor Time](/en/spectrum/spectrum/service/field-tech/work-order-main/work-order-labor/add-labor-time) window to manually add labor hours for a technician.

Edit
Click this button to open the [Add Labor Time](/en/spectrum/spectrum/service/field-tech/work-order-main/work-order-labor/add-labor-time) window to edit labor hours for a technician, or override default settings.

Delete
Click this button to delete the selected labor hours transaction.
