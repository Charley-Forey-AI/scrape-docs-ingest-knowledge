# Labor Transaction Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order
Enter a work order number to print, or
 press Enter to print
 labor entered for all work orders.
Technician
Enter a technician code to print, or
 press Enter to print
 labor entered for all technicians.
Labor work date
Enter the first and last labor work
 dates to be included on the report, or press Enter to begin with the first
 entry on file and end with the current Work Order processing date.
Work order completed date
Enter the first and last actual Work
 order completed dates to be included on the report, or press Enter to begin with the first
 entry on file and end with the current Work Order processing date.
Print tax codes and billing codes?
Select this checkbox if tax codes and
 labor billing codes should be printed on the edit list. Otherwise, clear this
 checkbox if codes need not be included on the edit list.
Include un-invoiced billing transactions?
Select this checkbox to include
 un-invoiced billing transactions on the report. By default this checkbox will
 be selected.
Include billing history transactions?
Select this checkbox to include
 billing history transactions on the report. By default this checkbox will be
 selected.
Report format
Indicate whether the edit list should
 print in sequence by work order number, or in sequence by technician.
Sort
Indicate whether technician hours
 should print in sequence by entry, in sequence by phase, or in sequence by pay
 type.
