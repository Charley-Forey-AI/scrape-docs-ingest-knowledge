# Labor Billing Rates Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Billing Code
Enter a unique code to identify this
 billing rate. These billing codes will later be assigned to various phases and
 labor categories, providing a default billing rate for specific types of work.
Description
Enter a unique description for this
 billing code.
Rates
Enter hourly billing rates. There are
 three columns of data. The first is the regular billing rate, the second is the
 overtime billing rate, and the third is the double time billing rate. Repeat
 for levels 2 through 5. The labor level number will be
 specified on the work order, then point to the appropriate billing rate from
 this table. For example, it may be appropriate to set up a billing code for
 an apprentice electrician (billing code EL), with five different billing
 levels that could be used for new construction customers, service contract
 work, government work, preferred customers, and so forth. Each one of those
 customers would have a different labor billing level that would be
 designated during Work Order Entry.
