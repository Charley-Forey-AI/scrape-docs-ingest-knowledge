# Subtasks

Use the Subtasks window to view, add, and edit subtasks associated with the selected task. Subtasks can only be created when there are no labor, material or other costs set up on the main task. Subtask notes will appear on the Crystal-Link Work Order Form.
If the main task is billed as T+M, then all subtasks must also be T+M. However, if the main task is billed using a flat rate, then the subtasks can be T+M and/or Flat Rate.
