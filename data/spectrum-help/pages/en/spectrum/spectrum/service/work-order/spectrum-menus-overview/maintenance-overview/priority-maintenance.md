# Priority Maintenance

Use the Priority Maintenance screen to establish
 user-defined priority codes, which can be applied to work orders.
There are sixteen colors to choose from when setting up priority codes. The description and
 color code you choose will appear on the Dispatch Board, Advanced Dispatch Schedule, and
 Work Order Entry.
Note: When Priority is selected as a sorting preference on the [Work Order Installation - Dispatch](/en/spectrum/spectrum/service/work-order/installation-overview/work-order-installation---dispatch) tab, then Spectrum will sort work orders based on the codes set up on this screen. It is
 recommended that careful consideration be given to defining each code so that the
 presentation will flow from high to low of 01 to 99 (numerals are sorted before alphabetic
 characters).
