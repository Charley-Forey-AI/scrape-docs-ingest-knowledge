# Work Order Labor

Use the Work Order Labor screen to enter labor hours on the selected work order. When a work order number is entered, the software will determine if it is a site or job work order:

- If a valid site work order was selected, the listbox will display labor
 transactions for the [site work order](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-labor).

- If a valid job work order was selected, the listbox will display labor transactions for the [Work Order Labor for Job work orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-labor/work-order-labor-for-job-work-orders).
