# Work Order Installation - Dispatch

Use this tab to select the dispatch settings for the Work Order module. These settings can be changed as needed at any time.
Field/Button
Description

Description Setup
Click this button to open the [Dispatch Setup](/en/spectrum/spectrum/service/work-order/installation-overview/work-order-installation---dispatch/dispatch-setup) window. Use this window
 to designate which fields are displayed in the long and short work order
 descriptions on the Dispatch Board and Dispatch Schedule.
Work order receiving

Auto-set dispatch status and priority of fully received work orders?
As items are received on purchase
 order, the associated work order information can be modified automatically.
 Select this checkbox if you are receiving or closing a purchase order (and the
 corresponding work order is fully received) in order to update the dispatch
 status and priority automatically.
Default dispatch status
If the Auto-set dispatch button is
 selected, enter the desired dispatch status code to set fully received work
 orders to.
Default priority
If the Auto-set dispatch button is
 selected, enter the desired priority code.
Daily dispatch schedule roll-over

Automatically roll the 'scheduled date' of past-due work orders to new date?
Select this checkbox to specify
 whether to perform daily roll-over of past-due work orders. If selected, the
 software will prompt the initial Operator to access the Dispatch Schedule to
 reset the 'scheduled date' of all unfinished work orders with an earlier
 scheduled date to the new date. If the initial Operator elects to not roll-over
 the dates, the next Operator to access the Dispatch Schedule will be prompted
 to reset the dates. This allows the update to be deferred to the primary
 dispatcher if another Operator logs in first.
