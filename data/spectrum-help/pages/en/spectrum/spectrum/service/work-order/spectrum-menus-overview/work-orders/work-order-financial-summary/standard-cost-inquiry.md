# Standard Cost Inquiry

The view-only Standard Cost Inquiry window will compute and display profitability figures.
This will be useful for those using the 'cost of goods sold' method instead of 'actual
 cost' accounting method. Even when the 'actual cost' method is used, this window may
 provide useful profitability information when comparing actual costs with expected standard
 costs.
Field
Description

Total sale
The total due from the customer net of
 sales tax displays.
Labor cost
The sum of the cost associated with the
 Labor Hours By Entry Work Order screen displays. When a union or pay group is
 specified in Labor Hours Entry by Technician or Labor Hours Entry by Work
 Order, the labor cost is then based on the pay rates entered in the following
 Payroll screens: Employee File Maintenance, Wage Code File Maintenance, Union
 File Maintenance, Non-Union Pay Group File Maintenance, depending on
 Installation settings.
Material cost
The material cost associated with the
 date entered on the Material Entry screen displays. This amount is the use tax
 (if any was calculated on the line item) plus the actual purchase price for the
 Work Order until it is updated or completed. After the work order is updated to
 Accounts Receivable, this field displays the actual material cost from
 Inventory Control.
Other cost
If applicable, the total of any other
 costs associated with the date entered on the Other Cost Entry screen displays.
Total standard cost
The total cost associated with the work
 order displays.
Profit
The total profit for this work order
 displays. The profit equals the total cost subtracted from the total sale.
% Profit
The total profit divided by the total
 sale displays.
