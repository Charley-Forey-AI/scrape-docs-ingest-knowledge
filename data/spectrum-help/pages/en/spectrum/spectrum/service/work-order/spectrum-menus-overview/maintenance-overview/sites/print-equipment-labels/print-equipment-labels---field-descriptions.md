# Print Equipment Labels - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Print options
Select the print option you want:
 Label print or
 Alignment mask

1. To print an alignment mask:

1. Load the labels into the printer.

1. Select Alignment mask.
 This prints XXXs on the labels where information will be printed.

1. Check whether the XXXs are aligned correctly.
 If the alignment is incorrect, adjust the printer and then print
 the alignment mask again.

1. When alignment of the XXXs is correct, select
 Label
 print.

Site ID
Enter the site ID to print, or press
 Enter to accept
 the software default of ALL.
Equipment status
Select the equipment status for the
 labels you want to print: Active and/or Inactive
