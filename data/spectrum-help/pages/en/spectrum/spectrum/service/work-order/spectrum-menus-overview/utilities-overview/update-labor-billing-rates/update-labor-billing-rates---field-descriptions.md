# Update Labor Billing Rates - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Update rates for
Specify which labor billing rate tables
 to change:

- Standard rates

- Customer-specific rates

- Site-specific rates

- Contract-specific rates

Billing code
Enter a billing code in this field, or
 enter ALL to update all standard billing rates.
Customer
Enter a customer ID, or enter ALL to
 update all customer-specific rates.  This field will be
 unavailable unless the Customer-specific rates checkbox above is selected.


Site
Enter a site ID, or enter ALL to update
 all site-specific rates. This field will be unavailable
 unless the Site-specific
 rates and/or Contract-specific rates checkboxes above are selected.


Contract
Enter a service contract ID, or enter
 ALL to update all contract-specific rates. This field
 will be unavailable unless the Contract-specific rates checkbox above is selected. This
 field will be hidden if the Service Contract module is not present.


Customer user-defined
Enter a user-defined validation code,
 or enter ALL to accept the default of all customer records. This field will be unavailable if the Customer-specific rates
 checkbox is not selected, or if there are no alpha customer user-defined
 fields available.

Site user-defined
Enter a user-defined site validation
 code, or enter ALL to accept the default of all site records. This field will be unavailable if the Site-specific and/or
 Customer-specific
 rates checkboxes are not selected, or if there are no alpha
 site user-defined fields available.

Change labor rates by
Specify which adjustment method to use
 for labor rates:

- Percent adjustment

- Dollar amount

Rate change percent
When Percent adjustment is
 selected, specify the desired change for Regular, Overtime and Double-time
 labor rates as a percentage amount. These fields will permit entry of positive
 or negative percentages, or enter zero (0) for no change. When Dollar
 amount is selected, specify the desired change for Regular,
 Overtime and Double-time labor rates in dollar amounts. These fields will
 permit entry of positive or negative dollar amounts, or enter zero (0) for
 no change.
