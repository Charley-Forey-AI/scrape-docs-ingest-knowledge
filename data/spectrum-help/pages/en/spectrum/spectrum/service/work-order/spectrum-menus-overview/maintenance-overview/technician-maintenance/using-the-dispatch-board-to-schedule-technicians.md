# Using the Dispatch Board to Schedule Technicians

You can use the dispatch board to schedule
 technicians.

1. On the Site Map screen, click
 System Administration > Installation > Work Order > Dispatch Setup > Description Setup.

1. Select those items you want to display on the Dispatch data entry screens and click OK to save your choices.

1. On the Site Map, click
 Work Order > Data Entry > Dispatch.

1. Notice that the current work order is highlighted with a red line appearing underneath it. Make sure you have the correct work order (the one you want to assign to a technician) selected by clicking in the white area beneath the work order number.

1. Place your cursor in the white area of the selected work order.

1. Right-click once and hold down the mouse button as you
 drag the work order to the technician you want to assign the work order to and
 release the mouse button.Note: You cannot assign work orders to a technician whose
 status is currently "unavailable." Technician statuses are set up in the
 Technician File Maintenance screen. [Technician Maintenance](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/technician-maintenance)

1. The Dispatch Board will reorganize and display the assigned work order under the technician's name and in order of priority (in other words, high priority work orders appear at the top of the screen).See [Dispatch Board](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/dispatch-board).
