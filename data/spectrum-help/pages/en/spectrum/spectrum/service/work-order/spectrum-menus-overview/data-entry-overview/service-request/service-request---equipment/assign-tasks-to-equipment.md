# Assign Tasks to Equipment

The Assign Tasks to Equipment window is used to build a task list for equipment when adding to the new work order. When assigning a task to a single piece of equipment, this window will allow the Operator to edit task quantity or adjust the price. If there is an active service contract for the selected equipment, the software will also automatically assign the contract to the task entry, but this can be edited if necessary.
The left side of this window will display tasks set up for the current company, and the right side will display tasks that have been selected for the selected equipment. These tasks will be added to the work order automatically when this window is closed. The window title will contain the equipment code if an individual piece of equipment is selected. If more than one piece of equipment is selected, the window title will say 'MULTIPLE' in place of the individual equipment code.
Note: If tasks are not already set up here, they can be
 added individually in Work Order Entry.
