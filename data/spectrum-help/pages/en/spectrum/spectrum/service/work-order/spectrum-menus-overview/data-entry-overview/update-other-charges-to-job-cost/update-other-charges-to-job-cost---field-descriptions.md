# Update Other Charges to Job Cost - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Select Materials

Work order
Enter the work order number to update,
 or press Enter to
 update materials entered for all work orders. Running
 this update for non-job work orders has no effect: nothing will be updated
 to Job Cost Transaction Entry.

Ordered date
Enter the first and last entry dates to
 be included in the update, or press Enter to begin with the first
 entry on file and end with the current Job Cost processing date.
Actual complete
Enter the first and last actual
 completion dates to be included in the update, or press Enter to begin with the first
 entry on file and end with the current Job Cost processing date.
Create job cost transactions

Job cost batch
Enter batch code to be used for these
 Job Cost transactions, or press Enter to use the operator logon as the batch code.
Transaction date
Enter the date for these Job Cost
 transactions, or press Enter to use the current Job Cost processing date.
