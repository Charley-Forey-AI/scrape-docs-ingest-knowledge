# Reopen Completed Work Order

Use the Reopen Completed Work Order window to reopen the selected completed site work order and assign a new dispatch status, priority and markup code.
After selecting the work order and assigning the New dispatch status, Priority code and Markup code, click Continue to reopen the work order.
