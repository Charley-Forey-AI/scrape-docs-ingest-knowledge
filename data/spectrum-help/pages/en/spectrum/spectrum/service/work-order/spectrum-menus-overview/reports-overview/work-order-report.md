# Work Order Report

The Work Order Report provides a printout of work orders.
A variety of selection criteria are available, which allows this report
 to be used in a number of different ways. In addition to printing work orders for a
 specific job, site, or customer, it is possible to print work orders for a specified
 technician. Alternately, work orders of a specific type (for example, all residential, all
 commercial, and so forth) can be printed, or work orders can be printed based on
 lead-source.
