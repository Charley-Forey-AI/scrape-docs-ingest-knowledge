# Work Orders

Use the Work Orders screen to enter site or job work
 orders. When you specify the job or site during work order entry, the system automatically
 uses the physical address found in the corresponding Jobs or Sites file.
Work orders can be assigned a status of 'Proposed.' A proposed work
 order cannot have labor, material, or other charge transactions entered against it, and
 Accounts Receivable billing cannot be generated. Proposed status will be available for both
 site and job work orders.
Click the New
 button to open the New Work Order
 window. Specify a work order #, site or job code, work order type, and customer code in
 this window. After this initial information has been entered, the Work Order Main
 Properties screen automatically opens, allowing entry of tax and pricing information. This
 screen also provides fields in which pricing structure (either fixed price, or time and
 materials), price quoted for the work, and estimated hours are all specified. If the
 customer requests a specific technician, this can be recorded on this screen.
If you are accessing a work order that has already been completed, this
 screen will automatically display in view mode. It is not possible to delete a completed
 work order. Likewise, it is not possible to delete a work order that includes Labor Hours
 Entry transactions that have been transferred to payroll.
Note: If a service contract work order is entered, this
 screen will automatically reset the work order type to match the service contract work
 order type specified in Service Contract File Maintenance.
