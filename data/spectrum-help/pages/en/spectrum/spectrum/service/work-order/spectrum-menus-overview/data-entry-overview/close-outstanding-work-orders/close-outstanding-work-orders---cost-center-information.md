# Close Outstanding Work Orders - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, this screen only displays work orders that the operator has security authorization to access. The software will compare the cost center assigned to the work order with cost centers authorized for the operator, and if the work order is not authorized it will not appear. The Build List of Work Orders to Close window will offer a Cost group field when cost centers are set to 'Yes' for this company.
Cost center overrides will apply in this screen to determine whether the Operator has access to the work order.
