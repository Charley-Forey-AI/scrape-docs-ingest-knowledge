# Work Order Other Charges for Site Work Orders

If a valid site work order was selected, the listbox will
 display other charge transactions for the site work order.
This screen is used to conveniently track billing amounts other than
 labor and material on the work order. To track other charges associated with a work order,
 specify one or more other cost codes defined in Other Charge Maintenance. For all methods
 other than 'fixed price', the sell price for Other Charge entries are adjusted
 automatically as additional labor, material, and other costs are recorded for the work
 order. Other charges for the work order will be recalculated when this screen is
 completed.
This screen will conditionally display Transactions available to bill in the
 upper right hand corner of the screen if there are journal entries in history transactions
 for the selected work order. If available, click the History button to open the [Bill Other Charges from Cost History](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-materials/work-order-materials-for-site-work-orders/bill-other-charges-from-cost-history) window.
Note: Other charge transactions will always be billed to the
 customer, not the manufacturer.
