# Bill Labor from Pre-Time Cards

This screen will display a list of all pre-time card detail lines for the current work order that have not already been transferred to the Labor Entry screen for billing.
After the desired pre-time card detail lines have been selected, click the Continue button to initiate the update. This update will create billing transactions for this work order, based on the operator's selections. The software will verify that a valid non-job G/L department code is associated with the work order type assigned to the work order. If the G/L department is blank in the work order header, or the assigned department is a 'Direct cost' department, the software will require the operator to enter a valid G/L department to continue. If an error is made while selecting transactions to bill, click the Deselect button to remove unwanted billings. This will un-flag transactions so that they may be re-selected to bill later, if desired.
