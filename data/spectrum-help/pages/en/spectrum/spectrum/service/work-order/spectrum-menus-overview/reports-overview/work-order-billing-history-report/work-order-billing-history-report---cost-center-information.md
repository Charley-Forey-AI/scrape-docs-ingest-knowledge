# Work Order Billing History Report - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, a Cost group field displays beneath the Contract field. When a cost group or cost center is specified on this screen, the report only shows billings for work orders assigned to cost centers in that group. Spectrum includes billings for the work order on the report only if the operator has permission to access the cost center assigned to the work order. Spectrum verifies that the operator has permission to access the selected cost center before proceeding.
