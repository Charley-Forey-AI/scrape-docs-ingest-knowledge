# Dispatch Schedule - Selections window - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Selections

Work order zone
Press Enter to schedule work orders
 in all zones that are assigned to this work order, or double-click in this
 field to select from a list of available zones.
Dispatch status
Press Enter to schedule work orders
 of all dispatch status codes that are assigned to this work order, or
 double-click in this field to select from a list of available dispatch codes.
Priority
Press Enter to schedule work orders
 of all priority codes that are assigned to this work order.
Work order type
Press Enter to schedule work orders
 of all types that are assigned to this work order, or double-click in this
 field to select from a list of available work order types.
Customer
Press Enter to schedule work orders
 for all customers that are assigned to this work order, or double-click in this
 field to select from a list of available customer codes.
Site
Press Enter to schedule work orders
 for all sites that are assigned to this work order, or double-click in this
 field to select from a list of available work sites.
Job
Press Enter to schedule work orders
 for all jobs that are assigned to this work order, or double-click in this
 field to select from a list of available jobs.
From date
Specify the earliest date to display
 work orders on the Dispatch Schedule.
Through today plus
Specify the default number of days from
 the dispatch date to display on the Dispatch Schedule screen.
Contract work orders
Select a view option for assigned
 contract work orders on the dispatch board:

- Show all work orders

- Maintenance visits only

- Exclude maintenance visits Note: This option will be hidden if the Service
 Contract module is not present

Exclude off hours

Work start time Work end time
Specify a work start time and end time
 for the Dispatch Schedule.
Hide Saturdays? Hide Sundays?
Select these checkboxes to exclude
 Saturdays and Sundays from the Dispatch Schedule. When selected, this feature
 will exclude weekend columns and not calculate projected hours across 'hidden'
 dates and times.
Assigned W.O. display

Assigned W.O. display
Select whether to display assigned work
 orders by site name or work order number. Job description will display for job
 work orders instead of site name.
Technicians

All technicians
All technicians set up in the current
 company display in this listbox. Select technicians to appear on the Dispatch
 Schedule from this column, and click the Select  button to transfer to
 the Selected column.Note: If the Include all checkbox is
 selected, all technicians for the current company will be included by
 default.

Selected
Selected technicians to appear on the
 Dispatch Schedule display in this column. Click the < Deselect button to
 remove any selected technicians from the Dispatch Schedule.
