# Work Order Zone Maintenance

Use the Work Order Zone Maintenance screen to enter a description of work order zones. This screen is accessed by clicking the Zone button on the Zip Code Zone Maintenance screen.
Field
Description

Zone
Enter the zone number.
Description
Enter a description for this
 zone.
Flag color
Select the flag color to associate with
 the selected zone number.
Use custom icon in place of standard icon and color?
Select this checkbox if a custom icon
 should be used in place of a standard icon and color for the associated zone.
 If you select this checkbox, the Custom icon field displays the Select Icon button. Click
 this button to select a custom icon to associate with the zone.
