# Site Equipment - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Equipment
Enter a new equipment code in this
 field. If you are editing an existing piece of equipment, the selected
 equipment code will display in this field.
Equipment Notes
Click this button to add any notes
 related to the site. If there are any existing notes, a 'Yes' will appear next
 to the button name.
Identification

Description
Enter a description for this piece of
 equipment.
Equipment type
This field will open the Equipment
 Types search window where you can select a pre-defined equipment type.
Year installed
Enter the year the equipment was
 installed.
Make
If the Service Contract module is not
 present on your system, enter the equipment make. Otherwise, this information
 will display from Service Contract setup.
Model
If the Service Contract module is not
 present on your system, enter the equipment model. Otherwise, this information
 will display from Service Contract setup.
Serial number
Enter the equipment serial number. Up
 to 50 characters are allowed in this field.
Current info

Location
Enter the equipment's specific on-site
 location.
Equipment status
Select an Active or Inactive equipment
 status in this field. This will filter the list of equipment in the equipment
 listbox.
Contract
When the Service Contract module is
 present, the Contract field will display the currently-active contract for this
 piece of equipment. If there is more than one active contract the field will
 display 'Multiple'. If there are no active contracts for this piece of
 equipment, the field will be accessible to look up and select a contract. Note: If the Service Contract module is not present, this
 field will be hidden.

Contract comment
When a single contract is selected,
 this field will either display any existing comments recorded for this
 contract, or allow you to enter new comments.
Components

New/Edit
Click this button to open the [New Components](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/warranty-management/new-components) window.
Delete
Click this button to delete the
 selected component.
View
If the Equipment window was opened in
 view mode, the View
 button will be the only available option for the Components.
