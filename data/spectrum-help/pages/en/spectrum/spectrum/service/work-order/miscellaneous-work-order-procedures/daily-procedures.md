# Daily Procedures

You can utilize different procedures in Spectrum for daily
 tasks.

- Enter (or change) and print work orders using theWork Orders
 screen.

- Print dispatch tickets using Work Order Form.

- Dispatch technicians and prioritize work orders using Dispatch
 Schedule or Dispatch Board.

- Enter labor hours using the Work Order Labor screen.

- Enter materials using the Work Order Materials screen.

- Enter other costs using the Work Order Other Charges screen.

- Once work has been finished and labor, material, and other costs
 are entered, the work order is transferred to accounts receivable using theWork Order
 Transferred to AR screen.
