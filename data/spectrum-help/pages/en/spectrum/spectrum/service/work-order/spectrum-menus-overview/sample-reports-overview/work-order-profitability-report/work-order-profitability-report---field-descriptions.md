# Work Order Profitability Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Work order Customer Site Technician Work order type Lead source Salesperson Work order zone
Enter the work order number, customer
 code, site code, technician code, work order type, lead source, salesperson,
 and work order zone to look up cost information for, or press Enter to include all work
 orders.
Through period-end
Enter the period-end date in this
 field. The corresponding fiscal year and period will display to the right of
 this field.
Include un-invoiced billing revenue?
Select this checkbox to include
 un-invoiced billing revenue on the report.
Include unposted labor costs?
Select this checkbox to include
 unposted labor costs on the report.
Include unposted material costs?
Select this checkbox to include
 unposted material costs on the report.
Include contract-specific transactions?
Select this checkbox to include
 contract-specific transactions on the report.
Actual cost formats
Select the report format you want to
 print:

- Profitability

- Over/under billing

- Revenue and cost

- Budget variance

Click the Standard Cost Formats
 button to open the [Standard Cost Profitability Report](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/sites-menu/work-order-profitability-report/standard-cost-profitability-report) instead.

Work order status as of period-end date
Select which work order statuses to
 include on the report:

- Unfinished

- Finished

- Completed

Enter the fist and last dates to be printed, or
 press Enter to
 accept the current Work Order processing date. For example, for work orders
 completed between July 1 and July 12, 2020, type 070120 in the From actual complete date
 field, and 071220 in the Thru
 actual complete date field.

Sort by
Click to specify the sequence in which
 this report should sort data:

- Work order

- Site/work order

- Actual complete date

- Work order type

- Lead technician

- Lead technician/work order type

-  Lead source

- Salesperson

Price types to include
Select the specific price type(s) you
 wish to include on the report:

- Both Flat rate and T+M

- Flat rate

-  T+M only
