# Dispatch Status Maintenance - Field Descriptions

Use the table below for reference when completing the fields on the screen.
Field/ButtonDescription
Dispatch statusEnter a valid work order dispatch status code, or window to view a list of existing status codes.
DescriptionEnter a description for this dispatch status code.
Work Order statusSelect the status that you want associated with the dispatch status code.Note: Work orders with a status of Finished can be posted to A/R to create an invoice. Once work orders have been posted to A/R, the status will change to Completed.

Flag colorSelect a flag color to use for this dispatch status. If you do not want to add a flag to this dispatch, select blank from the drop-down menu. The following table lists dispatch codes and their associated description, status, and flag color.
Flags appear on the Dispatch Board for easy identification of the dispatch status associated with each work order.

Use custom icon in place of standard icon and color?Select this checkbox if a custom icon should be used in place of a standard icon and color for the selected dispatch status code. If you select this checkbox, the Custom icon field displays the Select Icon button.
Custom iconSelect the Select Icon button to choose a custom icon to associate with the dispatch status code.
