# Frequently Asked Questions

You can reference the frequently asked questions for Work
 Orders.
How can we save time and make sure our material billings are
 complete?
Answer: You can automatically record materials on work orders from information on associated purchase orders. Work Order and Purchase Order have been fully-integrated to provide the ability to automatically transfer material purchases for site work orders from Purchase Order into the Work Order material file.
What is the Labor Update Exception Report used
 for?
The Exception Report will print/preview when it finds an employee that
 already has nine check sequences. The Work Order Labor Update's purpose is to transfer work
 order labor to Pre-Time Card Entry. If an employee already has nine check sequences, there
 is no room for these labor lines to transfer over. When this event occurs, the Exception
 Report displays.
