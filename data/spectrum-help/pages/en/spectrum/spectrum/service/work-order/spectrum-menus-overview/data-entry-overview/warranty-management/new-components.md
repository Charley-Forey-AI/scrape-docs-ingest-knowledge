# New Components

Use this window to add or edit component details, contract information and comments, and add any warranty notes.
Field/Button
Description

Site
The site ID and name will display in
 this field.
Equipment
Enter a new equipment code in this
 field. If you are editing an existing piece of equipment, the selected
 equipment code will display in this field.
Component
Enter a new component number in this
 field.
Warranty Notes
Click this button to add any notes
 related to warranty. If there are any existing notes, a 'Yes' will appear next
 to the button name.
Properties tab

Identification

Description
Enter a description for this component.
Install date
Enter an install date for the
 component.
Make
If the Service Contract module is not
 present on your system, enter the component make. Otherwise, this information
 will display from Service Contract setup.
Model
If the Service Contract module is not
 present on your system, enter the component model. Otherwise, this information
 will display from Service Contract setup.
Serial number
Enter the component serial number.
Current info

Location
Enter the component's specific on-site location.

Contract
When the Service Contract module is
 present, the Contract field will display the currently-active contract for this
 component. If there is more than one active contract the field will display
 'Multiple'. If there are no active contracts for this component, the field will
 be accessible to look up and select a contract. Note: If the Service Contract module is not present the
 Contract fields will not be visible.

Contract component comment
When a single component is selected,
 this field will either display any existing comments recorded for this
 contract, or allow you to enter new comments.
Warranty tab

Labor manufacturer

Code
Enter the code affiliated with the
 manufacturer warranty. If the code you select is not in use, then entry will
 not be allowed. Note: After the initial
 warranty (Labor manufacturer 1) has expired , the next warranty (Labor
 manufacturer 2) commences, followed by the third warranty.

Warranty period
Enter the warranty period in days, from
 the installation period forward.
Expiration date
Enter the warranty expiration date.
Standard hours
Enter the standard number of hours of
 labor required to repair the component.
Standard rate
Enter the standard rate charged (per
 hour) for component repair.
Material manufacturer

Code
Enter the code affiliated with the
 manufacturer warranty. If the code you select is not in use, then entry will
 not be allowed. Note: After the initial
 warranty (Material manufacturer 1) has expired , the next warranty (Material
 manufacturer 2) commences, followed by the third warranty.

Warranty period
Enter the warranty period in days, from
 the installation period forward.
Expiration date
Enter the warranty expiration date.
Material level
Enter the material level on a scale of
 0 to 5, with 0 indicating that there is no cost associated with the material.
