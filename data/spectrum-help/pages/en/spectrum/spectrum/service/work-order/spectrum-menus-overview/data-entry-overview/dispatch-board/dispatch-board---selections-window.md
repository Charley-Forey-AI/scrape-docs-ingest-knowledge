# Dispatch Board - Selections window

Use this window to select which information displays on the Dispatch Board. Default display is determined by settings in the Work Order Installation screen, unless operator-specific defaults have been previously set up.
Once you've made your selections, click the Save Current button on the Info Bar to create a new template for the current selections. You can specify whether this template will default when the Dispatch Board is opened.
