# Work Order Materials

Use the Work Order Materials screen to enter new or edit existing material transactions. When a work order number is entered, the software will determine if it is a site or job work order:

- If a valid site work order was selected, the listbox will display material transactions for the [Work Order Materials for Site Work Orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-materials/work-order-materials-for-site-work-orders).

- If a valid job work order was selected, the listbox will display material transactions for the [Work Order Materials for Job Work Orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-materials/work-order-materials-for-job-work-orders).
Note: You can quickly navigate between Work Order screens via the
 Info Bar links, or use the hot-key equivalents on the command bar.
