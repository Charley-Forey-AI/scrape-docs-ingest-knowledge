# Service Request - Header Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Site
Enter a site code for the service
 request. If the site code is valid, service request processing will continue.
 If the site code is invalid, the search window will open to allow you to refine
 your search.To create a service request for a new site,
 click the [Service Request for New Site](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request/service-request-for-new-site) button.

Name
The selected site code name display in
 this field.
Address
The selected site's
 address and phone number display in these fields.
Site phone

Contact email
The contact email address specified for
 the selected site displays in this field. Click the email button to the right
 of this field to send a new email message to the selected site contact.
Alerts
Applicable alerts for the customer and
 site are displayed in this section of the window. These alerts include:

- Credit limit issues when the customer associated
 with the selected site is at, above or near their credit limit.

- Collections issues when the customer associated with
 the selected site has a positive overdue Accounts Receivable balance.
 Retention is not considered a past due receivable.

- Site issues previously specified in Site File
 Maintenance.

- Service contracts present for the selected site that
 are either currently active, set up for a future date or expired within
 the past year.

- Outstanding purchase orders with quantity due for
 work order(s) for the selected site. This alert will not appear when
 there are no open purchase orders for the site.

- Customer-specific work order warnings previously
 specified in Customer File Maintenance.

- Site and customer status alerts when the site or
 customer is set to a status of 'Inactive' or 'Not used'.

Each alert offers a hyperlink to view more
 information pertaining to the alert.
