# Labor Transaction Report

The Labor Transaction Report is a printed report of labor
 hours entered through the Work Order Labor screen.
The report should be carefully checked for accuracy before hours are updated to Pre-Time
 Card Entry in the Payroll module.
Options are given for selecting specific work orders, technicians, or dates. The report can be grouped either by work order or by technician, and can be sequenced by pay type, phase, or entry date for each technician.
