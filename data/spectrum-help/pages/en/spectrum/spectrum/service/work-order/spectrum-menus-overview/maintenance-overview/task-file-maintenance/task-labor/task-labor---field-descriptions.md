# Task Labor - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Category
Enter a valid work order labor category
 code. This must have been previously defined in Labor Category File
 Maintenance.
Description
The description of this category
 displays from the Labor Category file.
Hours
Enter the standard number of hours that
 technicians in this labor category will spend on this task.
Pay rate
Enter the standard hourly labor rate
 for technicians in this labor category.
Burden
Enter the standard hourly payroll
 burden for technicians in this labor category. The burden amount defaults in
 Add mode based on settings in the Labor Category Maintenance screen for burden
 calculation. This burden figure is also re-defaulted if the hours or pay rate
 for this line is changed.
Budget
The labor category hourly budget amount
 is automatically calculated by using the following equation: Hours x Pay Rate + Burden = Budget
This calculated amount can be overridden if desired.


Total hours
All labor totals are
 calculated and displayed in this section.
