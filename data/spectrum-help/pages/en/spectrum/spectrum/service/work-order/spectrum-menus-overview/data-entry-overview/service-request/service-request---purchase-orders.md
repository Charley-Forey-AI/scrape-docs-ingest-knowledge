# Service Request - Purchase Orders

Use the Purchase Orders tab to view information about
 purchases related to work orders for the selected site.
View detailed information for the purchase order itself or for the underlying work order.
 You can edit an existing work order from this tab, in order to change the dispatch status,
 priority, add notes or make any other entries.
