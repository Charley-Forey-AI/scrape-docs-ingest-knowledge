# Service Request for New Site - New Work

The New Work tab provides the ability to enter key information about the service request. Information entered on this tab will appear in the new work order. All fields on this tab can be edited later within Work Order Entry.
Field
Description

Customer info

Contact name
This field will default to the contact
 name specified on the New Site tab, but the Operator can enter a different
 contact for the new work order. However, if the contact name was left blank on
 the New Site tab, the contact name entered on this tab will automatically be
 recorded as the site contact.
Other phone
This field will default to the other
 phone number specified on the New Site tab, but the Operator can change this
 phone number for the new work order. However, if the Other phone was left blank
 on the New Site tab, the phone number entered on this tab will automatically be
 recorded as the site other phone if different than the site phone.
Customer request
Enter a request code(s) in this field
 to automatically generate customer request text, or click the drill-down button
 to open the Search Customer Request Comments window and select request code(s).
 You can also enter new customer request text in this
 field, by itself or in addition to existing request code(s). To do this
 enter the desired request code(s) followed by a space and then any
 additional customer request text.

Customer P.O.
Enter a customer P.O. for the service
 request in this field, if applicable.
Other info

Dispatch status
Enter a dispatch status code for the
 work order in this field.  If the Operator has access to
 Dispatch Status Maintenance, use the Dispatch Statuses window to add or edit
 dispatch codes.

Priority
Enter a priority code for the work
 order in this field.  If the Operator has access to
 Priority Maintenance, use the Priorities window to add or edit priority
 codes.

Case type
Enter a case type code for the service
 request in this field.  If the Operator has access to
 Case Type Maintenance, use the Case Types window to add or edit case types.


Quote
Enter the total amount quoted for the
 work order in this field.
Hours
Enter the total projected hours for the
 entire work order visit in this field.
Requested date / time
Enter the requested date and time for
 the work order.
Work ordered
Enter a description of the work ordered
 for the service request in this box. Messages originally
 created in Work Order Message File Maintenance screen will display in this
 window, and can be edited as needed.
