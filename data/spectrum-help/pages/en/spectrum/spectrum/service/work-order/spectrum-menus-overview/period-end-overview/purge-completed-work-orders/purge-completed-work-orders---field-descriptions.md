# Purge Completed Work Orders - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldDescription
Work orderPress Enter to delete all work orders
 completed within a specified date range.
From actual complete dateEnter the first date to be deleted, or
 leave this field blank to begin with the first completed work order on file.
For example, to delete work orders completed between July 1 and July 12, 2025,
 type 070125 at the From date field and type 071225 at the To date
 prompt.

Thru actual complete dateEnter the last date to be deleted.
For
 example, to delete work orders completed between July 1 and July 12, 2025, type
 070125 at the From date field and type 071225 at the To date prompt.
