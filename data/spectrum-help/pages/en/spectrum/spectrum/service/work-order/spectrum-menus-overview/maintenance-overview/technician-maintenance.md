# Technician Maintenance

Use the Technician Maintenance screen to assign technicians
 to labor categories and define the burden calculation method. This screen also displays the
 logon status for each technician.
When technicians are assigned to categories, the category must have been
 previously designated in Labor Category Maintenance and the technician should have been
 previously designated as an employee in the payroll module's Employee Maintenance. For Work
 Order users that do not have the Payroll module installed on their system, the "employee
 name" to be used on reports comes from this Technician File Maintenance, instead of from
 Payroll records. If you don't want to create a technician record for an employee, enter
 work order labor entries using the employee's code from the Payroll module.
Employees who are already in the Payroll module but are not set up in
 this screen can be used on work orders.
The burden method entered on this screen is the default method for
 non-job costing burden for this technician.
