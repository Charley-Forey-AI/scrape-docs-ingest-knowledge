# Case Type Maintenance

Use the Case Type Maintenance screen to set up case types
 that can be used to classify work orders.
While Work Order types are generally used to classify residential versus
 commercial work orders, case types are used to further classify work orders into categories
 such as maintenance, installation, and repair.
Case types can be validated using settings on the Work Order
 Installation – Properties tab.
