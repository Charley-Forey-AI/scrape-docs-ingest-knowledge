# Bill Other Charges from Cost History

The Bill Other Charges from Cost History window displays a list of all G/L journal entry detail lines for the current work order that have not already been transferred into the Work Order Other Charges screen for billing.
When this window is first opened, the Selected column will be blank. The operator will be able to highlight one or more of the G/L journal entry detail lines and click the Select button to transfer. The G/L department will be assigned based on the code assigned in the work order header.

- If the G/L department is blank, or the assigned department is a
 'direct cost' department, the operator will be prompted to enter a valid G/L department
 before continuing with the update.

- The Sell
 price column will display the price recorded in the G/L journal entry for
 this cost history record. The displayed price will be stored in the Other Charges
 Transactions record when the line is selected and updated, and may be edited after the
 billing transactions are created.

- If an error is made while selecting transactions to bill, click the
 Deselect button to remove
 unwanted billings. This will un-flag transactions so that they may be re-selected to
 bill later, if desired.

- After the desired G/L journal entry detail lines have been
 selected, click the Continue
 button to initiate the update. This update will create billing transactions for this
 work order, based on the operator's selections.
