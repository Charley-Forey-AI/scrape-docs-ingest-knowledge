# Service Request Purchase Orders - Cost Center Information

All applicable purchase order lines will display regardless of cost center assignment and Operator security, since the software has already verified that the Operator has authorization for the selected site. However, if the cost center feature is enabled in the Enterprise Installation screen and the Operator does not have access to a selected work order, Operator will be unable to edit or view the selected work order.
