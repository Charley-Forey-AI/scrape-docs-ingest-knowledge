# Labor Category File Listing

The Labor Category File Listing screen will print information contained in the Work Order Labor Category file. The user has the opportunity to print a specific category or a specific billing code. The listing will print the description of this particular category of technicians along with the billing code assigned to those technicians, as well as the burden calculation method for regular, overtime, and double time.
