# Work Order Equipment History Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Equipment Equipment type Equipment make Equipment model Site Site zone Customer Work order Work order type Contract
Enter the equipment code, equipment
 type code, equipment make, equipment model, site code, work order zone,
 customer code, work order type, work order code, work order zone, and contract
 number to print on the report, or press Enter to include all on the
 report.
G/L date
The software will compare the selected
 date range with the G/L date associated with each actual cost history record
 when computing 'Actual cost'. When computing 'Total billed', the software will
 compare the selected date range with the G/L date associated with each billing
 history record.
Include un-invoiced billing revenue?
When this checkbox is selected the
 billing totals will include 'Current billing' transactions. The Bill # for
 these items will be blank.
Include unposted labor costs?
When this checkbox is selected,
 unposted labor transactions for each selected piece of equipment will be
 included in the actual cost totals. 'Available for billing' pre-time entries
 and time card entries are also included on the report when this checkbox is
 selected. Note: If both 'Include unposted
 labor costs' and 'Include unposted material costs' are selected, unposted
 other charge transaction costs will also be included on the report for each
 selected piece of equipment.

Include unposted material costs?
When this checkbox is selected,
 estimated and unposted material transactions for each selected piece of
 equipment will be included in the actual cost totals. 'Available for billing'
 A/P invoice entries will be included on the report when this checkbox is
 selected. Note: If both 'Include unposted
 labor costs' and 'Include unposted material costs' are selected, unposted
 other charge transaction costs will also be included on the report for each
 selected piece of equipment.

Include billing and cost not assigned to equipment?
When this checkbox is selected the
 report will include a section for 'No equipment', listing billing and cost
 amounts that were entered without referencing an equipment code. The unposted
 cost of Hourly, %Bill and %Cost Other Charge transactions will be included in
 the 'No equipment' section.
Format
Select a report format:

- Summary

- Billing detail

- Actual cost detail

Sort by
Select a sort option for the report:


- Site

- Equipment type

Report type
Specify whether to sort and subtotal by
 Equipment code or
 Equipment code and
 Component. When the Equipment code and
 Component option is selected, the report will include sub-sections (where
 applicable) labeled 'No component' in order to display transactions that
 were assigned to an equipment code but not a particular component.


Current work order status
Select one or more of the following
 status types to include on the report:

- Unfinished

- Finished

- Complete
