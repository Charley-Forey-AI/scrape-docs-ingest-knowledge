# Dispatch Schedule - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

New W.O.
Click this button to open the [Service Request](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request) screen to start a new
 work order.
Selections
Click this button to open the [Dispatch Schedule - Selections window](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/dispatch-schedule/dispatch-schedule---selections-window)
 window to select which information displays on the Dispatch Schedule.
Go to Date
Click this button to change the
 dispatch date.
Through MM/DD/YY
Click this button to change the work
 order processing date. The dispatch date specified above plus the number of
 days specified in the 'through today plus' field on the Dispatch
 Schedule Selections window will default and display on this
 button.
Refresh
Click this button to refresh the
 dispatch schedule display after making any changes. The time of the last
 refresh will display on this button.
Current work order
The selected work order will display in
 this box. The work order #, address, phone, and any work order notes will
 display in this box. Right-click in this box to:

- Email technician(s)

- Assign a new dispatch status code

- Assign a new priority code to the work order

- Create a new assignment for a technician

Technicians assigned to this work order will display
 to the right of this box.
If the selected work
 order is for a customer with an overdue A/R balance, a Collections Monitor link
 will display to the right of this field. Click on this link to open the
 Customer Financial Summary (if
 authorized).

Unassigned work orders
Work orders not yet assigned to a
 technician display in this listbox. Results can be grouped by column headings
 or other specified types. This list can also be printed and exported to
 Excel.Unassigned work orders can be dragged from this
 list and assigned to a technician in the grid. Technician assignment will be
 validated based on the Validate work order type and Validate case type
 fields on the Work Order Installation > Properties tab. The technician will be assigned as the next technician
 on the current work order.

Assigned
A list of all the technicians displays
 in this grid. The assigned work orders scheduled for each technician (if any)
 appear in the corresponding row on the grid. Select a work order number in the
 grid to make it the Current work
 order in the header.Change the unit of
 time that displays in the grid by changing the Zoom to option in the
 footer of the grid. The schedule can also be printed or saved as a PDF
 file.
Changes made to the grid will be synchronized
 to Work Order > Work Orders > Main Properties > Technician.

- Double-click on a technician to open the Technician Inquiry
 window. Use this window to view status and dispatch details, and current
 work orders assigned to the selected technician.

- Right-click on an assignment to e-mail the
 technician, assign a new dispatch status, assign a new priority to the
 work order, or create a new assignment.

- To re-assign the work order to a different
 technician, drag the work order to the technician's row. Technician
 assignment will be validated based on the Validate work order type and
 Validate case type fields on the Work Order Installation > Properties tab.

- Copy an assignment by dragging a work order to a
 different technician while holding the Shift key.

- To add additional assignments for the same
 technician, hold the Ctrl key and drag the work order to a different position
 on the same row.

- To re-assign the start time for a work order, drag
 the work order to a different position on the same row. To adjust the
 amount of time needed for a work order, drag the right-hand or left-hand
 edge of the work order to expand or contract the time.

- Overlapping work orders will display in pink,
 indicating an error.

- To remove a work order assignment, drag the work
 order off the grid.
