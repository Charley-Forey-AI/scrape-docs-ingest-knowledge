# Deciding Which Dispatch Screen to Use

Operators who prefer looking at technician names when scheduling work orders generally prefer the Dispatch Board. People who prefer viewing work order information in real-time and like viewing times/dates when scheduling work orders generally prefer the Dispatch Schedule.
