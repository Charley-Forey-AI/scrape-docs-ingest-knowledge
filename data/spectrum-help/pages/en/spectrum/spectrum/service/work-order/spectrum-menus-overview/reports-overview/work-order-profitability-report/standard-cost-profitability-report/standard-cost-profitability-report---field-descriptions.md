# Standard Cost Profitability Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order Customer Job/site Technician Work order type Lead source Salesperson
Enter the work order number, customer
 code, job/site number, technician code, work order type, lead source, and
 salesperson to look up cost information for, or press Enter to include all work
 orders.
Actual complete date
Enter the fist and last dates to be
 printed, or press Enter to accept the current Work Order processing date. For
 example, for work orders completed between July 1 and July 12, 2010, type
 070110 in the From actual
 complete date field and 071209 in the Thru actual complete date
 field.
Price types to include
Select the specific price type(s) you
 wish to include on the report:

- Both Flat rate and T+M

- Flat rate

-  T+M only

Standard cost formats
Select the report format you want to
 print:

- Profitability report

- Budget variance report

- Detail profitability report

- Detail budget variance report

Click the Actual Cost Formats button
 to open the [Work Order Profitability Report](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/reports-overview/work-order-profitability-report) instead.

Sort by
Click to specify the sequence in which
 this report should sort data:

- Work order

- Actual complete date

- Work order type

- Technician

- Technician, work order type

- Lead source

- Salesperson
