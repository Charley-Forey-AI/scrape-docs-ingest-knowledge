# Procedures As Required

Learn how to perform common procedures in Work
 Order.

- Enter sales tax information using Sales Tax Code Maintenance in Accounts Receivable, as appropriate.

- Enter new customer terms code information using Terms Code File Maintenance in Accounts Receivable, as appropriate.

- Modify work order labor billing rates using Labor Billing Rates Maintenance, as required.

- Modify work order labor categories using Labor Category File Maintenance, as required.

- Modify work order employees using Technician File Maintenance, as required.

- Enter General Ledger account destinations for work orders using G/L Department File Maintenance.

- Link work order types to G/L departments using Work Order Type File Maintenance.

- Maintain key information about the work site for non-job cost work orders in Work Site Address File Maintenance.

- Enter standard messages using Work Order Message File Maintenance, as appropriate.

- Modify material item codes if new material has been added to the Work Order module using Work Order Material Item Code File Maintenance. This is applicable only if Inventory Control is not installed.

- Enter and group zip code and zip code zones in Zip Code Zone Maintenance.

- Enter user-defined status codes that correspond to a Work Order status in Dispatch Status Maintenance.

- Enter other costs using Other Cost Maintenance as required.

- Enter flat-rate pricing tasks using Task File Maintenance as required.

- Enter lead sources using Lead Source Maintenance as required.

- Define bitmap colors and associated descriptions to attach to each priority code in Priority Maintenance.

- Define bitmap colors and associated descriptions to attach to each technician's status in Technician Status Maintenance.
