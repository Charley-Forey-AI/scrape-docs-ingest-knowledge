# Site File Listing - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

This icon lets you know SuperSelect options are
 available for this field

Format
Select the report format you want
 to print: Summary,
 Detail or Equipment.  When the Equipment format is
 selected, the Equipment section below is enabled.

Site id
Enter the site ID code(s) to
 include on the report, or press Enter to print all
 sites.
Include directions?
Select this checkbox if you want
 to include directions entered on the Directions tab of the New Work Site
 window. Otherwise, leave this checkbox clear. If
 the Equipment format is selected, this option will be disabled.


Equipment
Enter the equipment code(s) to
 include on the report, or press Enter to include all
 equipment on the report.
Include equipment notes?
Select this checkbox to include
 any notes associated with the selected equipment on the report.
Show components?
Select this checkbox if you wish
 to generate the Detail Site Equipment Listing report and include
 components associated with the selected equipment.
Include component notes?
If you have selected Show
 Components above, select this checkbox to include any notes associated
 with the selected components.
Include warranty details?
If you have selected Show
 Components above, select this checkbox to include any warranty details
 associated with the selected components.
Equipment status
Select the status of equipment to
 include on the report: Active and/or Inactive
Site status
Enter the work site status to
 include on the report: Active, Inactive and/or Not Used
Sort by
Select to sort the listing by
 Site or by
 Customer.
