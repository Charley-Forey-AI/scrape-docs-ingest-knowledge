# Work Order Tasks - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Task
Enter the work task. Work task
 descriptions are also provided in this window. Note: You must enter the task / system combination in Service Contracts > Data Entry > Schedule Contract Visits before completing the Task and System fields.

Pricing type
This view-only field will display 'Flat
 rate' when the Flat rate checkbox in Task
 File Maintenance is selected, or 'Time + material' if the task
 is not a flat rate tax.
Task

Equipment
Enter a piece of equipment to assign to
 the task in this field. The equipment code must be valid for the site to
 continue. This field will only be available for Site work orders, and is not
 applicable for Job work orders.When the task code is
 entered in the task list in the Edit Work Order
 window, the selected equipment code will serve as a default when adding new
 transactions in Labor Hours Entry,
 Material Entry, and Other Charges
 Entry.
This field is also used when
 tracking revenue by equipment if the task is assigned a non-zero flat rate
 price.

Contract
Enter a contract to assign to the task
 in this field. This field will only be available for Site work orders, and is
 not applicable for Job work orders. This field will be hidden if the Service
 Contract module is not present. If a contract is
 specified in the work order header, this contract # will default if valid
 for the equipment code, and the operator will be disallowed from entering a
 different contract number. If a contract is not specified in the work order
 header, the operator will be allowed to enter any combination of contracts,
 or leave this field blank. If the Work Order
 Installation option for Require equipment code for
 work order transactions is not selected, the operator will be
 permitted to enter a valid contract # without assigning an equipment code.

When a contract is entered in the task list in the
 Edit Work Order window, the selected contract will
 serve as a default when adding new transactions in Labor Hours
 Entry, Material Entry, and
 Other Charges Entry.
 If
 there is a single current contract referencing the selected piece of
 equipment, this contract # will default. If there are multiple current
 contracts referencing the selected piece of equipment, if the contract is
 specified on the Other info tab and it references the
 selected piece of equipment, this contract # will default. If there are
 multiple current contracts referencing the selected piece of equipment, and
 no contract is specified for the work order as a whole, the first current
 contract #found will default.
 This field is also
 used when tracking revenue by contract if the task is assigned a non-zero
 flat rate price.

Quantity
Enter or change the task's quantity.
Sell price/unit
Enter a sell price per unit for the
 task item.
Extension
The corresponding extension displays.
 No entry is allowed.
Budget financials

Labor cost
The portion of the task's price
 attributed to labor displays. No entry is allowed.
Material cost
The portion of the task's price
 attributed to materials displays. No entry is allowed.
Other costs
The portion of the task's price
 attributed to other costs displays. No entry is allowed.
Subtask costs
The portion of the task's price
 attributed to subtasks displays. No entry is allowed.
Total cost budget (estimate)
The total budgeted cost of the task
 displays. No entry is allowed.
Overhead
The portion of the task's price
 attributed to overhead displays. No entry is allowed.
Profit
The portion of the task's price
 attributed to profit displays. No entry is allowed.
Subtotal (including overhead and profit)
The sum of labor, materials, and other
 costs displays as the price's subtotal. No entry is allowed.
Flat price
The bill rate to be paid by the
 customer displays. No entry is allowed.
Sell price/unit
The sell price per unit entered above
 will display in this field.
