# Work Order Standard Phase Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Phase
Enter a job cost phase code.
Description
Enter a description for this job phase
 code. This description will default into Work Order Entry, but can be changed
 at that time, if desired.
Billing code
Enter the labor billing code for this
 phase. This code must have been previously set up in Labor Billing Rates
 Maintenance. Once a billing code has been assigned to a phase, this billing
 code will default in Labor Hours Entry whenever the phase is entered. Entry in this field is not required, and will be used for
 default purposes only. Double-click in this field to select from a list of
 available billing codes.
