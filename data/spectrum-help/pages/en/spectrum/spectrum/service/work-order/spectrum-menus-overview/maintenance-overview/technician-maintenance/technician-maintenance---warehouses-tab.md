# Technician Maintenance - Warehouses tab

Use this tab to enter warehouse information for Service
 Techs.
Note: These warehouses are for Service Tech usage
 only.

- Select a Truck warehouse code for the field technician to use on service calls. This warehouse would typically be the inventory of items available in their service vehicle.

- Select the Technician has access to all warehouses checkbox to give the field technician access to all common warehouses.

- If the checkbox above is not selected and the field technician has access to more than one common warehouse, add or remove warehouses below. These warehouses cannot be the same as the Truck warehouse selected above.
