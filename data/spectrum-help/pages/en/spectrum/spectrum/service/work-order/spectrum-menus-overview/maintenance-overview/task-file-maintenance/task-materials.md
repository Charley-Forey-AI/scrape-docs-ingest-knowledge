# Task Materials

Use the Task Materials window to enter materials allocated for this task and their corresponding costs. When entry is complete, the total dollar amount for materials will display on the main Work Site Address Maintenance screen.
Field
Description

Item code
Enter a valid item code. If the
 Inventory Control or Price Update modules are present, this code must have been
 previously set up there. Otherwise, this code must have been previously defined
 in Work Order's Material Item Code Maintenance. A search window is provided for
 viewing item codes. When an item code is entered,
 Spectrum recognized the item's flat rate conversion factor and will factor
 this into any material calculations. For example, if the item's conversion
 factor is "12" and you add "24" units of the item in the Quantity field of
 this window, Spectrum will multiply the item cost by "24".

Description
The description of this item displays
 from the item file.
Quantity
Enter the standard quantity of this
 item to be used for this task. When an item code is
 selected in the Item code field, Spectrum recognized the item's flat rate
 conversion factor and will factor this into any material calculations. For
 example, if the item's conversion factor is "12" and you add "24" units of
 the item in the Quantity field of this window, Spectrum will multiply the
 item cost by "24".

Um
Enter the unit of measure for this
 item. The unit of measure displays from the item file, but can be overridden if
 desired.
Unit cost
The unit cost displays from the item
 file, but can be overridden if desired.
Budget
The budget amount for this material
 item is automatically calculated as the product of "quantity" and "unit cost"
 divided by the inventory price-per-factor. This calculated amount can be
 overridden, if desired.
Total budget
The sum of all budget lines
 displays.
