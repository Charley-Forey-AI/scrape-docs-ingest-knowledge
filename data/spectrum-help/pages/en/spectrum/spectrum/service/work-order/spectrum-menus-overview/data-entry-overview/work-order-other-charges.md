# Work Order Other Charges

 Use the Work Order Other Charges screen to manage any
 other charges associated with a work order.
When a work order number is entered, the software will determine if it is a site or job
 work order:

- If a valid site work order was selected, the listbox will display other charges transactions for the [Work Order Other Charges for Site Work Orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-other-charges/work-order-other-charges-for-site-work-orders).

- If a valid job work order was selected, the listbox will display other charges transactions for the [Work Order Other Charges for Job Work Orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-other-charges/work-order-other-charges-for-job-work-orders).
Note: You can quickly navigate between Work Order screens via the
 Info Bar links, or use the hot-key equivalents on the command bar.
