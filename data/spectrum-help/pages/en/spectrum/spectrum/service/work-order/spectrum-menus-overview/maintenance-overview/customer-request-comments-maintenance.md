# Customer Request Comments Maintenance

This allows call takers to enter standard descriptive text
 quickly when entering service requests.
Use this screen to set up request codes for standard customer comment
 text included during [Service Request - New Work](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request/service-request---new-work) entry.
Note: As comments are ordered by request code in Service
 Request entry, Service Managers may wish to assign the primary request a code that
 alphabetically or numerically places it near the top of the list in order to allow multiple
 customer requests to appear in a logical order.
Field
Description

Request code
Enter a request code that does not
 already exist for the current company. This field is must-enter.
Customer request comment
Enter any default customer request
 comments in this field.
