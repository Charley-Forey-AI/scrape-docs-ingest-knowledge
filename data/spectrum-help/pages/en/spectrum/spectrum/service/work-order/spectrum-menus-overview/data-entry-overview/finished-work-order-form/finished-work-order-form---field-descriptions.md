#  Finished Work Order Form - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order Site / job Customer Type Zone
Enter the work order number, site or
 job code, customer code, work order type code, and work order zone to print, or
 press Enter to print
 ALL.
Finished date
Enter a starting and ending date for
 finished work orders you want to include on the report in this field. The date
 selection will only apply to history transactions, not current activity.
Include technician listing?
Select this checkbox if you want to
 include the technician listing section on the report.
Include current transactions?
When this checkbox is selected the
 records that have not yet been updated to A/R will appear on this report.
Use standard cost?
Select this checkbox to use standard
 costs when printing profit summary. If this option is selected, the Labor Cost, Material Cost and
 Other Cost totals
 on the report will be the same as those on the Standard Cost Inquiry window.
 This format will also show sales and costs for the work order as a whole. This field will be selected by default if the 'Default
 calculation method' in Work Order Installation is set to 'Standard cost'.

This field will be hidden if the Work Order
 Installation for 'Print profit on finished work order form' is not
 selected.

Include invoiced transactions?
When this checkbox is selected the
 records that have been updated to A/R will appear on this report. If this
 option is cleared, the rest of the fields in the History section will be
 unavailable.
Bill #
This field will be available when
 invoice transactions are included. Enter the billing number, or press
 Enter to print all
 work order billings.
Invoice date
Enter a starting and ending date for
 invoices you want to include on the report in this field. The date selection
 will only apply to history transactions, not current activity.
Format
Choose a format for the report:
 Summary or
 Detail. The
 Summary format will consolidate all transactions across billings, and the
 Detail format will print separate forms by billing.
Profitability
Select whether to include a profit box
 on the finished work order form. This option will be hidden if the user does
 not have security authorization to it.
