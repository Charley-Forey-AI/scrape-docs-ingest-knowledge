# Finished Work Order Form - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, a Cost group field will display beneath the To Finished Date field. The software will compare the cost centers authorized for the operator with the cost center associated to the work order (or job), and if the operator is not authorized to access the cost center that work order will not be included.
For Site work orders the cost group selection will evaluate the cost center assigned to the work order. For Job work orders the Cost group selection will evaluate the cost center assigned to the job, and the work order will only be included when the work order (or job) cost center is included in the requested Cost group range.
Cost center overrides will apply in this screen to determine whether the operator has access to the work order.
