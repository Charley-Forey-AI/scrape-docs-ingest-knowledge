 report.
Note: The Transaction Amount of labor entries posted from Payroll includes payroll
 burden. The burden component is shown in the Payroll Burden column to facilitate balancing with General Ledger. The
 difference between these columns represents the pay extension of that time card line. The
 Unit Cost column is computed
 by dividing the Transaction Amount by the number of Hours.
