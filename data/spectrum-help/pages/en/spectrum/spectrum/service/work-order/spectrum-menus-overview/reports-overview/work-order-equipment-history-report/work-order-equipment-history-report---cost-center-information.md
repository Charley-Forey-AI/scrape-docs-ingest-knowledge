# Work Order Equipment History Report - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, Spectrum includes activity only if the operator has permission to access the cost center assigned to the site. Spectrum compares the cost center assigned to the site with the cost centers in the operator's assigned scheme, and if the cost center is not included, then equipment records for that site will not be included.
