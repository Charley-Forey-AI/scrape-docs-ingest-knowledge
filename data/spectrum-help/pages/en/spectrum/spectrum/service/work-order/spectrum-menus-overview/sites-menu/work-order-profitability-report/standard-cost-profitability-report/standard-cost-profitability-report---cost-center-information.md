# Standard Cost Profitability Report - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen for this company, Spectrum includes a work order on the report only if the
 operator has permission to access the cost center assigned to the work order.
The software compares the cost center assigned to the work order with cost centers in
 the operator's assigned scheme; if the cost center is not included, then the work order
 is not shown.
