# Reverse Work Order Billing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Billing to be reversed

Work order
Enter a completed work order # to
 reverse.
Bill #
Enter a billing # to reverse. This
 window will only contain billings that have not been reversed. The Bill # is
 not needed for job work orders.
Job-Site
The job code/site and name displays
 based on the work order selected. This field will be labeled Site if the
 Work Order Installation option for Allow job work orders is not
 selected.
Invoice
The invoice number displays in this
 field.
G/L date
The original work order G/L date
 defaults if it falls within the minimum/maximum Accounts Receivable processing
 date. Otherwise, the current Accounts Receivable processing date defaults.
 Press Enter to accept
 this default date, or enter a different date.
Invoice date
The original work order posting date
 defaults if it falls within the minimum/maximum Accounts Receivable processing
 date. Otherwise, the current Accounts Receivable processing date defaults.
 Press Enter to accept
 this default date, or enter a different date.
New credit memo

A/R invoice batch
The current operator logon code
 defaults in this field, but may be overwritten as needed. Double-click this
 field to select from a list of available A/R batch codes.
New G/L date
Enter a new G/L date for the work
 order. This date must be within the current Work Order minimum/maximum date
 range.
New invoice date
Enter a new invoice date for the work
 order. This date must be within the current Work Order minimum/maximum date
 range.Note: If the Invoice date must be in G/L
 period option is selected in Accounts Receivable
 Installation, this date must be within the same fiscal period
 as the G/L date entered above.

Dispatch status
Enter a dispatch status code to assign
 to the work order. Entry is required in this field.
Priority
The priority code will default based on
 the dispatch status, but the operator can change the priority.
Markup code
Enter a markup code assigned to the
 work order. This field is applicable only for site work orders, and will not
 appear if the selected work order is for a job.
