# Work Order Billing History Report

The Work Order Billing History Report will provide a
 listing of invoices and credit memos by work order, site or G/L date.
This report can be used as an audit report, showing billing (and
 reversal) history to substantiate site revenue, invoice by invoice. Because no cost
 information appears on this report, it could also be provided to the work order customer
 when billing questions arise.
This report provides a historical record of amounts updated to Accounts
 Receivable. Any editing performed in Accounts Receivable is not reflected on this report.
 Warranty invoices are included on this report. When credit memos appear on the report,
 values will be shown as negative numbers. This report will only include equipment billing
 hours, and the Bill customer billing setting will be ignored when determining what
 transactions are displayed on this report.
