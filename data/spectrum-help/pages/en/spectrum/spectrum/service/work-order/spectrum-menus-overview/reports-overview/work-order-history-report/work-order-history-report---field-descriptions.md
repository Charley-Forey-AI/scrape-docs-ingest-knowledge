# Work Order History Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order Technician Item code Work order type Site/job Phase
Enter the work order number, technician
 code, item code, work order type, site/job code, and phase code to include on
 the report, or press Enter to print work orders for all sources.
Labor date
Click to specify the type of date to be
 used in printing work order history:

- Ordered

- Estimated complete

- Actual complete

- Data entry

Enter the beginning date to be included on this
 report, or press Enter to begin with the FIRST. The date entered here is
 then compared with work orders for inclusion on the report, but the
 particular date that the software looks at is based on entry in the date
 type field.

Labor sort
Click to specify the sequence in which
 this report should sort data. Options are:

- Work order number, Ordered date

- Work order number, Phase

- Work order number, Technician, Ordered date

- Technician, Ordered date

- Technician, Phase

- Technician, Pay type

History of
Click to specify the history to be
 included on this report:

- Labor,

- Materials

- Other charges

- All

Items to include
Click to specify the work orders to be
 included on this report:

- History

- Open

- History and open
