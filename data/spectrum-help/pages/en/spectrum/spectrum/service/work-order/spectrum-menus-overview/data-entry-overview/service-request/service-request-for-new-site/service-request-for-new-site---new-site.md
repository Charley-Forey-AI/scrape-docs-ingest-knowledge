# Service Request for New Site - New Site

The New Site tab is used to set up a new site for a service request. This tab allows you to enter basic site information to create a new customer.
Field
Description

Site info
The fields in the Site info section of
 this window will be blank when a new site is set up.
Alerts
The alerts section of this window will
 be empty, unless an 'Existing' site customer is selected. Once an 'Existing'
 site customer is selected, the following applicable alerts are displayed:

- Credit limit issues when the customer associated
 with the selected site is at, above or near their credit limit.

- Collections issues when the customer has a positive
 overdue Accounts Receivable balance. Retention is not considered a past
 due receivable.

-  Site and customer status alerts when the site or
 customer is set to a status of 'Inactive' or 'Not used'.

 Each alert offers a hyperlink to view more
 information pertaining to the alert.

New site info

Site code
Enter a new site code in this field.
 Spectrum will verify that this site code is not already present in the current
 company. If the Use auto-sequenced site
 numbers checkbox is selected in Work Order Installation, if
 this field is left blank the next available site code will default
 automatically and the cursor will advance to the next field.

Site name
Enter a site name for the new site.
 This field is optional.
Address
Enter the Address, City, State and Zip
 code for the new site. These fields are optional.
Site phone
Enter a primary site phone number and
 extension for the new site. These fields are optional.
Other phone
Enter a secondary site phone number and
 extension for the new site. These fields are optional. Use these fields to
 store the contact's phone number, if different from the primary site phone
 number. This phone number entered here will default on
 the New Work tab.

Contact
Enter a primary contact for the site in
 this field. This may be the person who is making the service request, or
 another site contact. This field is optional. The contact
 name entered here will default on the New Work tab.

E-mail
Enter an email for the site. This field
 is optional, but entry of a site email address is encouraged as this address
 will be utilized when using the 'Quick Print' feature in Work Order Entry to
 print the Work Order Proposal and Finished Work Order Form.
Lead source
Enter a lead source code for this site
 in this field. If the 'Require lead source in work order entry' checkbox is
 selected in Work Order Installation, this field will be required.
Site defaults

W.O. type
Enter a work order type for the new
 site. This field is required.
Customer info

Site customer
Select whether to create a New Customer
 or use an Existing customer. The rest of the fields in this section will vary
 based upon your choice. Note: The Credit Check tab and Alerts section of this window are only
 available once an 'Existing' site customer is selected.

Code
When 'New customer' is selected and the
 Accounts Receivable Installation option for 'Use automatic customer numbers' is
 selected, this field will display 'AUTOMATIC' and the next sequential customer
 number will be assigned automatically when the new customer is added. If this
 installation option is not selected, the previously selected Customer code will
 default in this field, but can be changed by the Operator. The software will
 verify that the customer code entered does not already exist for the current
 company. When 'Existing' is selected, the Operator must
 enter a valid code in the current company. Once entered, the Alerts section
 of this window will show applicable alerts for the selected customer and the
 Credit Check tab will be enabled.

Customer name
When 'New customer' is selected, this
 field will default to the site name. This field is optional. When 'Existing' is selected, this field will display the
 stored name from Customer File Maintenance. No changes are allowed.


Terms
When 'New customer' is selected, enter
 a terms code. This field is required. When 'Existing' is
 selected, this field will display the stored terms code from Customer File
 Maintenance.

Salesperson
When 'New customer' is selected, enter
 a salesperson code in this field. This field is optional. When 'Existing' is selected, this field will display the stored salesperson
 code from Customer File Maintenance.

Tax code
When 'New customer' is selected, enter
 a tax code in this field. This field is required. When
 'Existing' is selected, this field will display the stored tax code from
 Customer File Maintenance.

Taxable
When 'New customer' is selected, choose
 whether the site customer is taxable. Leave this field blank to equal 'No
 default'. When 'Existing' is selected, this field will
 display the stored taxable setting from Customer File Maintenance. No
 changes are allowed.

Credit limit
When 'New customer' is selected, enter
 a credit limit in this field. When 'Existing' is
 selected, this field will display the stored credit limit from Customer File
 Maintenance.

Material level
When 'New customer' is selected, enter
 a material level from 1-5 in this field. When 'Existing'
 is selected, this field will display the stored material level from Customer
 File Maintenance.

Labor level
When 'New customer' is selected, enter
 a labor level from 1-5 in this field. When 'Existing' is
 selected, this field will display the stored labor level from Customer File
 Maintenance.
