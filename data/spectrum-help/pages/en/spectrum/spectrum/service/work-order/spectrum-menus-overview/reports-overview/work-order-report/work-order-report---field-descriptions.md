# Work Order Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order Customer Job Site Technician Type Lead source
Enter the work order number, customer
 code, job code, site code, technician code, type code, and lead source code to
 include on the report, or press Enter to print work orders for all sources.
Date
Specify the type of date to be used in
 printing work orders: Ordered,
 Estimated complete, Assigned, Actual complete, or Finished. The date type
 selected is then compared with work orders for inclusion on the report. To print a specific range of dates, enter the first and
 last dates to be printed.

Status
Specify the status of the work orders
 to be displayed: All, Proposed,
 Open, On Hold, Assigned, Arrived, Hold, Completed, or Finished work orders.
Items to include
Specify the print status of work orders
 to be included on this report: only work orders that have been previously
 printed, only work orders that have not yet printed, or all work orders,
 regardless of print status.
Work order reference
Specify which work orders are to be
 printed: job work orders, site work orders, or both.
Sort by
Specify the sequence in which work
 orders are to be printed. This report can be grouped by work order number, by
 technician (lead technician), by ordered date, or by the estimated completion
 date. Sorting by lead technician is particularly useful if open work orders are
 being scheduled and techs have been entered in the technician field on the Work
 Order Entry screen; this report can show which work is assigned for every
 technician.
