# Task Other Charges

Use the Task Other Charges window to enter other costs pertaining to this task. For example, trip fees would be entered here. When entry is complete, the total dollar amount for other costs will display on the main Work Site Address Maintenance screen.
Field
Description

Item code
Enter a valid other item code for this
 task. This code must have been previously defined in Other Cost
 Maintenance.
Description
The description displays from the Other
 Item Code file.
Rate type
The rate type displays from the Other
 Item Code file.
Hours/amt
The standard number of hours that
 technicians in this labor category will spend on this task displays in this
 field.
Rate
The standard rate defaults from the
 Other Item Code file, but can be overridden if desired.
Budget
The budget amount for this other cost
 is automatically calculated but can be overridden if desired.
Total budget
This figure displays based on entry of
 data in the window that opens when the Other button is clicked.
