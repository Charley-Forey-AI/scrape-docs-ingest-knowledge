# Labor Category File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Category
Enter a unique category code in this
 field. These categories can later be used to group technicians together for
 purposes of assigning them to the same billing code and burden calculation
 method. Note that these category codes must be set up prior to adding
 technicians in Technician File Maintenance.
Description
Enter a description for this group of
 technicians.
Billing code
Enter the labor billing code (from
 Labor Billing Rates Maintenance) to be used for this category of technicians.
Burden method
Click to indicate the appropriate
 burden method: Percentage,
 Hourly, or None. During processing, if labor hours
 are entered the option chosen here will determine how site work orders will
 be burdened: either by a percentage of labor dollars, a flat hourly rate, or
 no charge. In addition to specifying the burden method, it will also be
 necessary to specify rates.

Burden rates

Regular
If burden method is Percentage, enter
 the burden percentage (regular time). If burden method is Hourly, enter the
 hourly burden rate (regular time).
Overtime
If burden method is Percentage, enter
 the burden percentage (overtime). If burden method is Hourly, enter the hourly
 burden rate (overtime).
Double time
If burden method is Percentage, enter
 the burden percentage (double time). If burden method is Hourly, enter the
 hourly burden rate (double time).
