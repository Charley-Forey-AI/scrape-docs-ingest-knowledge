# Work Order Materials for Site work orders - Re-price Material Transactions

To re-price records from the Material Transactions list,
 select a record (or records), and click the Re-price button.
This update recalculates the customer pricing for all selected material records, based on
 the current setup for inventory price levels, customer special pricing, quantity discounts,
 promotional specials and material markup rates as of the specified effective date.
Note: If the Inventory Control module is not present, the
 update will recalculate the customer pricing based on the current setup for material price
 levels and material markup rates only.
