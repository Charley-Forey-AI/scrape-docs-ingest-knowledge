# Finished Work Order Form - Sample Report

The Total cost field shown on this report is equal to the
 work order material cost and is computed using the use tax amount (if there is any use tax
 calculated) plus the purchase price.
If the Operator code has been added to the Work Order Operator Security Table, and the
 'Hide cost on the finished work order form' checkbox is selected, the cost and profit
 information will not appear on this report.
