# Task Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Task
Enter the task code(s) you wish to
 print, or press Enter
 to print all tasks.
Print notes?
Select this checkbox if you want to
 include the task notes on the listing report.
Report format
Select to print the listing in either
 Summary or
  Detail
 format.
