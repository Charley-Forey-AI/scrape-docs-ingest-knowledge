# Edit All Selected Transactions

This update will reset the checkbox settings and corresponding invoice extensions of the material transactions currently highlighted in the list box.
Note: No changes will be made to records where the "Bill
 manufacturer for this material " checkbox is not applicable, that is, because no component is
 specified for the selected transaction.
