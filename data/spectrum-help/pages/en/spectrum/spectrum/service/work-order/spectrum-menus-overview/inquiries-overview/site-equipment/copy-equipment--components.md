# Copy Equipment / Components

 This update will copy the selected equipment to a designated site. Options are provided to copy equipment directly onto a contract and to include components and warranty details, if applicable.
Field
Description

Copy from equipment

From site
This view-only field shows the site
 location of the selected equipment.
From equipment
This view-only field shows the selected
 equipment.
Copy to equipment

To site
Select the same site (default) or
 another existing site from the Search Sites window. If this window is being
 opened from the Service Contract module, this site field cannot be changed.
To equipment
Select an existing equipment code, or
 create a new equipment code.
Equipment type
The Equipment type field will default
 based on the equipment you selected in the To equipment field.
Installation date
If you are creating a new equipment
 code you will need to enter an installation date, otherwise the existing
 installation date for the equipment code will default in this field.
Include components?
Select this checkbox to include any
 components associated with this equipment code. Click the
 Details button
 to open the Component
 Details window.

Include warranty details
If you have chosen to include
 components, select this checkbox to include warranty details associated with
 the components.
Assign equipment to a service contract?
Select this checkbox if you want to
 assign this equipment code to a service contract, and specify the contract
 number in the Contract field. If you are in the process
 of adding new site equipment, the Contract field is
 disabled. This field only displays when you are editing and existing
 contract.
