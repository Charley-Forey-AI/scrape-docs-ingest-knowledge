# Work Order Type Maintenance

Use the Work Order Type Maintenance screen to manage work
 order types and link them to G/L departments.
Work order types are entered during Work Order Entry. The work order type you select
 determines the General Ledger account to use for posting revenue associated with the work
 order (and subsequent accounts receivable invoices).
