# Dispatch Board - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

New W.O.
Click this button to open the [Service Request](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request) screen to start a new
 work order.
Selections
Click this button to open the [Dispatch Board - Selections window](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/dispatch-board/dispatch-board---selections-window) window
 to select which information displays on the Dispatch Board. Once you've made your selections, click the Save Current button on the
 Info Bar to create a new template for the current selections. You can
 specify whether this template will default when the Dispatch Board is
 opened.

Through MM/DD/YY
Click this button to change the work
 order processing date. The current date plus the number of days specified on
 the Dispatch Board Selections window will default and display on this
 button.
Refresh
Click this button to refresh the
 dispatch board display after making any changes. The time of the last refresh
 will display on this button.
Current work order
The selected work order will display in
 this box. The work order #, address, phone, and any work order notes will
 display in this box. Right-click in this box to:

- Assign or unassign technician(s)

- Email technician(s)

- Assign a new dispatch status code

- Assign a new priority code to the work order

If the selected work order is for a customer with an
 overdue A/R balance, a Collections Monitor link will display to the right of this
 field. Click on this link to open the Customer Financial Summary (if
 authorized).

Technician hot box
Drag-and-drop technicians from the Info
 Bar or Assigned section to this hot box to assign to the current work order.

- If this is the first technician assigned to the work
 order, the technician will automatically be assigned as 'Lead'.

- If this is the second technician assigned to the
 work order, the technician will automatically be assigned as 'Second'.

- Further technicians will be assigned as the next
 technician.

Unassigned work orders
Work orders not yet assigned to a
 technician will display in this listbox. Results can be grouped by column
 headings or other specified types. This list can also be printed and exported
 to Excel.
Assigned
Work orders assigned to technicians
 will display in this section, grouped by technician. If multiple technicians
 are assigned to a work order, that same work order will appear in each
 technician's column. Each technician column will display
 the technician code followed by the total hours scheduled and number of work
 orders assigned in the header. If the technician is inactive or already
 scheduled out, the label will simply say "Out". The assigned work order
 number, dispatch status and priority icons, site/job description, and total
 projected hours for the entire site visit will display for each work order
 assigned to the technician.

- Click the arrow keys at the top of this list to
 scroll through technicians, or click the Find Tech button to
 select and display a specific technician.

- Double-click on the column title to open the
 Technician
 Inquiry window. Use this window to view status and
 dispatch details, and current work orders assigned to the selected
 technician.

- Click the work order number to make that the current
 work order.

- To re-assign a role on a work order, drag the work
 order number to another column. The work order will be removed from the
 original technician's list and assigned to the new technician.
