# Work Order Form - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order
Enter the work order number(s) to
 print, or press Enter
 to print all work order numbers. Note: Any
 notes included on the selected work order(s) will print; however, if any of
 the notes exceed 50 pages, they will not appear on the report and a warning
 will display.

Site / job
If Work Order Installation is set to
 'Allow job work orders' this field will be labeled Site/job. If not, the field
 will be labeled Site.  Enter the site number to print, or
 press Enter to
 print work orders for all sites.

Customer Type Zone
Enter the customer order(s), work order
 type(s), and work order zone(s) to print, or press Enter to print ALL.
Technician
Enter a technician, leave blank for
 unassigned projects, or press Enter to print all the technicians.
Entered date
To print a specific range of dates,
 enter the first and last entered dates to be printed. Entry in these fields
 references the date each work order was originally added to the software.
Include directions?
Select this checkbox to include
 driving directions to the work order site. These directions originate from the
 Site Notes for the selected site.
Include technician detail?
Select this checkbox to include
 technician detail on the work order.
Include maintenance work orders only?
Select this checkbox to include only
 maintenance work orders.
Print status
Select the current status of work
 orders that are to be printed. Options are to print only those work orders for
 which this will be a first printing, only those that were previously printed,
 or all work orders regardless of print status.
Status
Select the status of work orders to
 print: Proposed
 and/or Open
 (unfinished)
