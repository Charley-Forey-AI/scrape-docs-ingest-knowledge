# Dispatch Status Maintenance

Use the Dispatch Status Maintenance screen to establish user-defined status codes, which may then correspond to a Work Order status.
The Dispatch Status can be used as a factor in sorting work orders on the dispatch board. The dispatch status is ranked by code (A, B, C, O, etc.), not by the attached Work Order status. The description and color code display you choose appears on the [Dispatch Board](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/dispatch-board), [Dispatch Schedule](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/dispatch-schedule), and [Work Orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-orders). There are 16 colors to choose from when setting up dispatch codes.Note: When Dispatch status is selected as a sorting preference on the [Work Order Installation - Dispatch](/en/spectrum/spectrum/service/work-order/installation-overview/work-order-installation---dispatch) tab, Spectrum sorts work orders based on the codes set up on this screen, starting from 01 to 99 (numerals are sorted before alphabetic characters). In this way, you can define each code so that the presentation flows in a way which is useful to you.

When the dispatch status is changed in Work Order Entry, then the Work Order status will also be updated on that work order. If the Work Order Status field isn't set to anything, the status will not be changed on the work order when that dispatch status code is used.
No matter where a work order's dispatch status is changed, the application automatically updates the work order status and sets any corresponding date and time stamps for that work order.
