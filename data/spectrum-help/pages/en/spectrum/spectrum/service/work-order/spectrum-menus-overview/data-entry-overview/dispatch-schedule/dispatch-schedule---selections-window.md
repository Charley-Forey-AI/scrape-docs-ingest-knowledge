# Dispatch Schedule - Selections window

Use this window to select which information displays on the Dispatch Schedule. Default display is determined by settings in the Work Order Installation screen unless operator-specific defaults have been previously set up.
Click the Save Current button to save the selections specified on this window for future use in the SAVED SELECTIONS portion of the Info Bar.
