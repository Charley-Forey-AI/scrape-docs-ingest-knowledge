# Update Work Orders to Invoice Entry - Field Descriptions

After the list has been built, all selected work orders will display in the grid. Work orders with a status of 'Proposed' or 'Completed' will always be de-selected.
The following fields will be available for entry in the grid:
Field
Description

Terms
The terms code will default from the
 selected work order, but can be changed. This field is must-enter.
Sales tax code
The sales tax code will default from
 the selected work order, but can be changed. If this code is changed, the
 Invoice total
 will be recalculated. This field is must-enter.
G/L date
Enter the G/L date. This date must
 be within the current Work Order minimum/maximum date range.
Invoice date
Enter the invoice date. This date
 must be within the current Work Order minimum/maximum date range.Note: If the Invoice date must be in G/L
 period option is selected in Accounts Receivable
 Installation, this date must be within the same fiscal period as the G/L
 date entered above.

Update?
Select this checkbox to include
 this work order in the update.
Close?
Select this checkbox to close the
 work order after the update is complete.
