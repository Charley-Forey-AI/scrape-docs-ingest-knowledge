# Material Transaction Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order
Enter the work order number(s) to
 include on the report, or press Enter to print materials entered for all work orders.
Ordered date
Enter the first and last entry dates to
 be included on the edit list, or press Enter to begin with the first
 entry on file and end with the current Work Order processing date.
Actual complete date
Enter the first ands last actual
 completion dates to be included on the edit list, or press Enter to begin with the first
 entry on file and end with the current Work Order processing date.
Include un-invoiced billing transactions?
Select this checkbox to un-invoiced
 billing transactions on the report. By default this checkbox will be selected.
Include billing history transactions?
Select this checkbox to billing
 history transactions on the report. By default this checkbox will be selected.
