# Dispatch Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Zone
Enter the zone you want to appear on
 the report, double-click in this field to select from a list of available zones
 saved in the system, or press Enter to accept the system default of ALL.
Dispatch status
Enter the dispatch status you want to
 appear on the report, double-click in this field to select from a list of
 available dispatch status numbers saved in the system, or press Enter to accept the system
 default of ALL.
Priority
Enter the priority code you want to
 appear on the report, double-click in this field to select from a list of
 available priority codes saved in the system, or press Enter to accept the system
 default of ALL.
Technician
Enter the name of the technician you
 want to appear on the report, double-click in this field to select from a list
 of available technician names saved in the system, or press Enter to accept the system
 default of ALL.
Job
Enter the job number you want to appear
 on the report, double-click in this field to select from a list of available
 job numbers saved in the system, or press Enter to accept the system
 default of ALL. Note: To include only
 job-based work orders on the report, make sure the Site field is left
 clear.

Site
Enter the work site address you want to
 appear on the report, double-click in this field to select from a list of
 available work site addresses saved in the system, or press Enter to accept the system
 default of ALL. Note: To include only
 site-based work orders on the report, make sure the Job field is left
 clear.

Customer
Enter the customer code you want to
 appear on the report, double-click in this field to select from a list of
 available customer codes saved in the system, or press Enter to accept the system
 default of ALL.
Type
Enter the work order type you want to
 appear on the report, double-click in this field to select from a list of
 available work order types saved in the system, or press Enter to accept the system
 default of ALL.
Work order #
Enter the work order number you want to
 appear on the report, double-click in this field to select from a list of
 available work order numbers saved in the system, or press Enter to accept the system
 default of ALL.
Print unassigned WO's?
Select this checkbox if you want to
 print unassigned work orders on the report.
Print short description?
Select this checkbox if you want to
 print the work order short descriptions on the report.
Print long description?
Select this checkbox if you want to
 print the work order long descriptions on the report.
Date selection
From the Date selection list, select
 the type of date you want to use to select data to appear the listing:

- Technician scheduled start date

- Entered date

- Requested date

- Scheduled date

- Assigned date

- Arrived date

From / To date
Enter the beginning and end dates to be
 included on this report, or press Enter to begin with the minimum Work Order processing date and
 end with the maximum Work Order processing date. The date specified in these
 fields will be used to display dispatch entries based on the type of date you
 selected in the Date selection field.
Sort by
Select from the following sorting
 options:

- Technician / scheduled start

- Technician / work order #

- Work order #
