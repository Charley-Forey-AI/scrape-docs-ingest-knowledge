# Technician Maintenance - WO Types tab

Use this tab to change work order type approval status for the selected technician.
Field
Description

WO type code
A list of work order type codes
 displays.
Description
A description of the corresponding work
 order type displays.
Approved?
Select this checkbox to indicate that
 the selected technician is approved for the corresponding work order
 type.
