# Dispatch Board

The Dispatch Board displays a list of both unassigned work
 orders, and assigned work orders sorted by technician.
This screen will offer drag-and-drop capability to assign technicians to
 a work order, or re-assign a work order from one technician to another. In addition, work
 order dispatch status codes and priority codes can be easily assigned from this screen.
Note: The Dispatch Board is optimized for a screen
 resolution of 1024 x 1920. If this is not enough to display the whole board, try turning
 off toolbars in the browser, entering full screen mode, or increasing the vertical
 resolution to 1080 or more.
