# Work Order Form

The Work Order Form is used to print a group of work orders and provides options for selecting specific work orders, customers, jobs, and types desired.
If the Service Contracts module is installed, there is an additional
 maintenance work order print format available. A maintenance work order format is
 automatically used when work orders are created from the Service Contracts module. These
 work orders are flagged as maintenance work orders; the detail created in the work order
 notes window is extensive. Site equipment is now labeled with the model number, serial
 number, location, etc, and the task to be completed are also created with the corresponding
 task parts required (if specified in the Service Contract
 Installation). The maintenance work order that is printed gives the worker
 all the information that is required to work on the equipment. This format is different
 than the standard work order print.
This screen will be launched automatically from the Work Orders or Work Order Main Properties screens when the Forms button is clicked. If a work order is selected in context, the software will determine the status of the work order. If the work order has a status of 'Finished' or 'Completed' the [Finished Work Order Form](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/finished-work-order-form) will open instead.

## Information Flow

If projected hours are entered in Schedule Contract
 Visits, then the Dispatch work order printed for this service contract
 will include these projected hours. This information will print above the
 quote $ field. If no projected hours exist for a work order,
 then zeros will print in its place. If the Service Contracts module is not installed,
 then the work order is not affected.
Activity, Work Ordered, and Work Completed notes will print on the work order. The alternate bill-to name and address will appear in the Bill To section of the printed work order based on the settings in the Work Order Header Table.
Note: Service contract task-related notes that are updated in the Service Contracts > Data Entry > Contract Work Order Processing screen or the Service Contracts > Utilities > Contract Work Order Processing screen display in the Work Ordered section of the
 Work Order form. Task-related notes not entered in these
 screens display in the Work Notes section of the Work
 Order form. This screen is not used to print completed work orders;
 instead, use the [Finished Work Order Form](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/finished-work-order-form)
 screen.
