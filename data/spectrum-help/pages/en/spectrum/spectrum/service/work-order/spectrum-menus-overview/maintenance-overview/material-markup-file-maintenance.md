# Material Markup File Maintenance

Use the Material Markup File Maintenance screen to set up
 and maintain a list of markup tables that allow you to define cost-basis break levels and
 associated markup dollar amounts or percentages.
Material markup codes are used in Work Order Installation, Customer File Maintenance and
 Site File Maintenance to establish markup amounts. The default amounts will then default
 into new work orders.
Markup amounts are calculated as follows:

- When calculating stock (inventory) transactions markup price, the software marks up the price using price level, quantity discount, promotional pricing, and more.

- When calculating non-stock transactions markup price, the software marks up the cost based on 'unit cost' (before sales tax).
