# Update Labor Hours to Payroll - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Work order
Enter a work order number, or press
 Enter to update
 labor entered for all work orders.
Work order type
Enter a work order type, or press
 Enter to update
 labor entered for all work order types.
G/L department
Enter a G/L department, or press
 Enter to update
 labor entered for all G/L departments.
Technician
Enter a technician, or press Enter to update labor entered
 for all technicians.
Union
Enter a union code, or press Enter to update labor entered
 for all union codes.
Wage code
Enter a wage code, or press Enter to update labor entered
 for all wage codes.
Labor category
Enter a work order labor category, or
 press Enter to update
 labor entered for all labor categories.
Batch code
Enter the batch code to be used for
 these transactions in Payroll Pre-Time Card Entry, or press Enter to use the operator
 logon as the batch code.
From labor work date / To labor work date
Enter the first and last labor work
 dates to be included in the update, or press Enter to begin with the first
 work date on file and end with the current payroll processing date.
From work order completed date / To work order completed date
Enter the first and last work order
 completed dates to be included in the update, or press Enter to begin with the first
 completed date on file and end with the current payroll processing date.
Cost group
If the cost center feature is enabled
 in the Enterprise Installation screen for this company, select a Cost center or
 Cost group to include in the update. If a cost center or cost group is
 specified, this update will only transfer labor to Pre-Time Card Entry from
 work orders assigned a cost center within the selection range. The update uses
 the work order cost center for site work orders, or the job/phase cost center
 for job work orders as the cost center assigned to the time card line.Note: If the cost center feature is not enabled
 for this company, this field will be hidden.
