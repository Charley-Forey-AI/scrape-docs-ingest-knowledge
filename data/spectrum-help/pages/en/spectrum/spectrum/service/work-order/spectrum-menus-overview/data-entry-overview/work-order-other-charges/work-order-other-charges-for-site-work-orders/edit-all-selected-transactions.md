# Edit All Selected Transactions

This update will reset the checkbox settings and corresponding invoice extensions of the other charge transactions currently highlighted in the listbox.
Field
Description

Task complete?
Choose Yes, complete to set the
 checkbox to selected, No, not
 complete to clear the box in all selected other charge
 records, or Do not change current
 setting.
