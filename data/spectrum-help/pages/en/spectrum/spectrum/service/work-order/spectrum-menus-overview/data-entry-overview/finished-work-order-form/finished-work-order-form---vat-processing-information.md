# Finished Work Order Form - VAT processing information

If the 'Utilize value added tax (VAT) tracking?' checkbox
 is selected in General Ledger installation, select the Assign VAT tax code? checkbox to include
 VAT totals, and specify which Tax
 code to use: Goods & Services Tax (GST) or Harmonized Sales Tax
 (HST).
