# Work Order Standard Phase Listing

Use the Work Order Standard Phase Listing to print a
 listing of the job cost phase, the description, and the labor billing code, as set up in Work
 Order Standard Phase Maintenance.
There are options for selecting a specific group of phases, a specific billing code, or for
 printing the entire listing, in which case all phases and all billing codes that are set up
 in Work Order Standard Phase Maintenance will print.
