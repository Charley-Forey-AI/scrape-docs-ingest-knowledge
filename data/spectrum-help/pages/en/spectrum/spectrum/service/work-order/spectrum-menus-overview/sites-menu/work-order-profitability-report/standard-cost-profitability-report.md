# Standard Cost Profitability Report

The Standard Cost Profitability Report provides information
 on completed work orders, showing the total sales, total cost, profit/loss amount, and the
 profit percentage for every work order which prints.
Work orders flagged as 'Proposed' will automatically be deselected.
 This report will open by default when the Work Order Installation
 default calculation method is set to 'Standard cost'.
The operator may define the sequence in which the report is to print,
 either by work order number or type, technician, actual complete date, lead source or
 salesperson. If a technician is specified here the report will only include work orders
 where that technician is entered in the Work Order header, and the technician must be
 either the Lead or 2nd technician (this is specified when the technician is assigned to
 the work order).
For labor transactions, the standard calculation uses the estimated
 burden rate which is found in the employee record. If there is no burden rate specified
 for the employee, and a 'Payroll burden percent' is entered on the Installation screen,
 this percent will be used instead.
