# Work Order

Work Order provides a complete billing and tracking system
 for service calls.
Enter work orders as calls are taken, print dispatch tickets or work orders on the
 spot, and record detailed work descriptions or recommended future work. Billing is
 automatically transferred to accounts receivable. Access Work Order options from the
 Site Map and the Work Order Info Bar. This module is integrated with Service Contracts,
 Inventory Control, Accounts Receivable, General Ledger, Purchase Order, Payroll, Job
 Cost and Document Imaging.

To set up and use Work Order, follow these steps:

1. Configure default Work Order settings.

1. Define work order codes, rates, and categories.

1. Enter service requests; create and schedule work orders.

Refer to the following sections for details.

## Work Order Installation

Use the System Administration > Installation > Work Order menu to set up the Work Order module. Options include default properties
 (such as numbering, warnings, price type, and cost types) and settings for labor (labor
 entries, cost history, and billing), material (task entry and billing), printing
 (indicate what to print on the work order), and dispatch (work order receiving). See for
 additional information.

## Work Order Maintenance

Use the Work Order > Maintenance menu to set up the work order codes, rates, and categories required for
 tracking and billing service calls. For a list of all options available from the
 Maintenance menu, see [Maintenance Overview](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview).

1. [Create a dispatch status codes. ](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/dispatch-status-maintenance)

1. [Set up priority codes. ](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/priority-maintenance)

1. [Set up labor billing rates. ](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/labor-billing-rates-maintenance)

1. [Set up labor categories. ](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/labor-category-file-maintenance)

1. [Set up material markup tables. ](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/material-markup-file-maintenance)

1. [Set up technicians.](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/technician-maintenance)

1. [Create case types.](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/case-type-maintenance)

1. [Create work order types.](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/maintenance-overview/work-order-type-maintenance)

1. If applicable, [add work sites](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/sites-menu/sites) for non-job cost work orders.

## Work Order Data Entry

Use the Work Order > Data Entry menu to create, schedule, and print work orders. Work order data can come
 from different sources. For example, labor hours entered in Payroll or purchase orders
 recorded in the Purchase Order module. If transactions sent from another module are
 available to bill, you can select them during work order entry. Work orders may be
 billed one at a time or in groups. See for additional information.

1. [Enter service requests](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/service-request) for existing or new work
 sites.

1. [Create work orders](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-orders) for a job or site.

1. Assign technicians to work orders on the [Dispatch Schedule](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/dispatch-schedule).

1. If applicble, enter labor, material, purchase order items, or
 other charges during work order entry, and bill any available transactions entered
 in other modules.

- [Add labor hours to a work order](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-labor), or [enter labor hours by technician](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-labor-entry-by-technician).

- [Add materials to a work order](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-materials).

- [Add other work order charges ](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/work-order-other-charges)(not labor or
 materials). For example, transportation.

- [Add purchase order items](/en/spectrum/spectrum/accounting/purchase-order/spectrum-menus/data-entry-overview/purchase-orders).

1. As needed, [add notes](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/notes) to the work order. On the Work Order Notes screen, be sure
 to choose the appropriate notes option:

- Notes added under WORK COMPLETED appear on
 the customer invoice and in Service Tech.

- Notes added under WORK ORDERED are for
 internal use only.

1. [Assign standard or flat rate tasks](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/work-orders/work-order-tasks) to a work
 order.

1. Bill work orders. You bill work orders one at a time or in a
 group:

- To bill a single work order:

- [Review invoice financial
 information](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/work-orders/work-order-financial-summary).

- [Create the Accounts Receivable invoice](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/work-orders/work-order-financial-summary/create-ar-invoice) from the Financial
 Summary screen.

- To bill a group of work orders, [update work orders to invoice entry](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/update-work-orders-to-invoice-entry).

1. If you entered labor, material, or other charges during work
 order entry, transfer that data back to the relevant module:

- [Update labor hours to payroll.](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/update-labor-hours-to-payroll)

- [Update material to job cost.](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/update-material-to-job-cost)

- [Update other charges to job cost.](/en/spectrum/spectrum/service/work-order/spectrum-menus-overview/data-entry-overview/update-other-charges-to-job-cost)
