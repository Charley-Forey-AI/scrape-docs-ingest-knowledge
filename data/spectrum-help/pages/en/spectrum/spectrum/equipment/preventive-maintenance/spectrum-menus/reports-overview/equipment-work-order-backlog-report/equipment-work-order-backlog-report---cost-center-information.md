# Equipment Work Order Backlog Report - Cost Center
 Information

If the cost center feature is enabled in the Enterprise
 Installation screen, the Cost Group
 field displays in the Selections section. You can enter a valid cost group or cost center
 code. Spectrum will verify that you have permission to access that cost center before
 proceeding.
The software compares the cost center assigned to the equipment with
 the cost centers in the operator's assigned scheme, and if the cost center is not
 included then that piece of equipment is not shown on the report. In addition, the
 software will filter which equipment is included on the report using the operator's cost
 center security scheme.
