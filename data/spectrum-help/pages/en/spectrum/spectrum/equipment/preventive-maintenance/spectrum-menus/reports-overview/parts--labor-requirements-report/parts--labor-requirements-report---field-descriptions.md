# Parts & Labor Requirements Report - Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Selections

Equipment code
The Equipment code must have been
 previously set up in Equipment File Maintenance.
Equipment type
The equipment type must have been
 previously set up in the Equipment Type File Maintenance screen.
Task code
This code must have been set up in the
 P.M. Tasks Maintenance screen.
Forecast through date
Enter the due date through which
 maintenance tasks should be included.
Report format

- Parts

- Labor

- Both

Select Parts to display the parts
 required to perform scheduled tasks. Select Labor to display labor
 requirements for the scheduled tasks. In both cases, costs associated with the
 tasks will display.
Report type

- Complete listing

- Exception report

A complete listing includes all
 maintenance tasks, regardless of when those tasks should be performed.An exceptions report includes maintenance tasks that need
 to be completed.

Minimum percent due
Enter the minimum percent due for
 including maintenance tasks. Tasks with a percentage due of 100% are due for
 maintenance immediately (or may be overdue for maintenance), while tasks with a
 maintenance percentage of 0% have recently been completed. Therefore, running
 the report for tasks with a percentage due of 50% would yield results showing
 any tasks that are more than half way to the next service date.
