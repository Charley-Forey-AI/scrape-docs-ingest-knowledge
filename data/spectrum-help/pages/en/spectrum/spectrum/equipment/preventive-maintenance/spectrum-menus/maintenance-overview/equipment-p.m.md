# Equipment P.M. Setup

This feature can be used to enter the specific tasks
 necessary to maintain the company's equipment.
Before scheduling work, the equipment must have been entered in
 Equipment File Maintenance and the tasks must have been entered in P.M. Tasks Maintenance.
 This feature is used to indicate the tasks that are to be performed on each piece of
 equipment.
