# New/Edit Problem Type Code window

Use this screen to set up and maintain a list of problem
 type codes.
Problem type codes are located on the [Problem Type Code Maintenance](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/problem-type-code-maintenance) screen.
Tip: Use the New Problem Type Code window to add new
 equipment repair status codes. Use the Edit Problem Type Code window to make changes to
 existing equipment repair status codes. As entries are completed, the data will display on
 the Problem Type Code Maintenance starting screen.
Field
Description

Problem type
Enter up to 10 characters for a new
 problem type code (for example, brakes).
Problem description
Enter up to 50 characters for a brief
 description of the new problem type code.In the Edit
 window, make any required changes to the existing problem type code's
 description.
Note: If a long
 description is entered in this field, the description may appear truncated
 in areas of the software where space is limited.
