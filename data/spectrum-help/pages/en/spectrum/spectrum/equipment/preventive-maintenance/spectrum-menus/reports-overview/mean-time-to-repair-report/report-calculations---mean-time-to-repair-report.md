# Report Calculations - Mean Time To Repair Report

The following tables describe how the Mean Time To Repair Report is calculated.
Note: This report will only include completed or finished
 work orders. No open work orders will be included.
Field
Description

Problem type, Equipment code or Equipment type
Depending on how the report was sorted,
 the Problem type, Equipment code or Equipment type will display in this
 column.
Description
The corresponding description displays
 in this column.
WO count
The number of work orders included in
 the calculation displays in this column. If the 'Sort by' option is 'Problem
 type', this is the number of work orders assigned to this type.
Days to repair: Total days
The total repair days is calculated as
 the number of days between the 'work returned' date of the work order and the
 'work ordered' date.
Days to repair: Mean days to repair
The mean repair days is calculated as
 the total repair days divided by the number of work orders.Note: If the 'returned date' is blank, the report will use the 'finished
 date' to calculate the mean number of days.

Actual hours: Total hours
The actual total hours is calculated by
 summing actual hours from the work order detail.
Actual hours: Mean hours to repair
The mean actual hours is computed by
 summing the total actual hours to repair of all work orders included in the
 count, divided by the count.
