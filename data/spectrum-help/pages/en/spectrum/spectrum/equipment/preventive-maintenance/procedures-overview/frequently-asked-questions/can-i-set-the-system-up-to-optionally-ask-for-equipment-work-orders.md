# Can I set the system up to optionally ask for equipment work orders?

You can define which codes require or allow a work order.
Yes.
To set this up, go to Equipment Control >  Maintenance > Cost Category File Maintenance.
Options include the following:

- None – never allow entry of a work order

- Optional – allow entry, but not required

- Required – require entry of a work order
