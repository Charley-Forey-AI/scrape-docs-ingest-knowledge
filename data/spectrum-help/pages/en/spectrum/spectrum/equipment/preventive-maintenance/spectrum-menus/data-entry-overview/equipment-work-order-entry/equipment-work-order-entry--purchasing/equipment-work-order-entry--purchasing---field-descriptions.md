Use the table below when completing the fields on this screen.
Fields
Descriptions

New/Edit/View buttonsClick New to add a new purchase

The View button displays if you do not have security
 authorization to add or edit purchase orders. These buttons are unavailable
 if you do not have authorization to Purchase Order Entry.
