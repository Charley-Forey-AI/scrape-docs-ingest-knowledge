# Equipment Work Order Entry - User-Defined - Field
 Descriptions

Use the table below when completing the fields on this
 screen.
Fields
Descriptions

Prompt
The user-defined field
 prompt displays.Valid entries display in the
 Current setting column.
Click the column header to arrange the data in ascending order; click the
 column header again to arrange the text in descending order. Use your mouse
 to determine the width of this column by sliding the column title's side
 borders.

Current setting

Hide unused
Click this toggle button to hide all
 unused user-defined fields that appear in the list box for the selected job. Note: If you give a value to a user-defined field and
 that value is later removed or if you press Enter when defining a
 prompt's value (thereby leaving the prompt's value blank), the
 user-defined field will display when you click the Hide Unused
 button.

Edit
Click this button to edit the selected
 user-defined field.
