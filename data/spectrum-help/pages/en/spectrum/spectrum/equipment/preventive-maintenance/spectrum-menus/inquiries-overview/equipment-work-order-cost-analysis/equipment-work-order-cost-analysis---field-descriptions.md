# Equipment Work Order Cost Analysis - Field Descriptions

Use the table below when completing the fields on this screen.
Fields/Buttons
Descriptions

Status Type
Click to display a combination of
 active, inactive, or retired pieces of equipment in the List box.
Cost Details
Click to open the Work Order Detail
 Cost Inquiry window. Use this window to view costs broken out by cost category
 for the selected work order(s). You can review costs by cost category,
 particularly when the Cost Category Type amounts on the main screen are outside
 of the expected parameters. Drill down to view equipment cost history records
 and the original A/P invoices and Payroll time cards.
View work order
Click on this link to edit the selected
 work order in the [Equipment Work Order Entry](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/equipment-work-order-entry) screen.Note: This
 link will not display if you do not have security authorization for this
 screen.
