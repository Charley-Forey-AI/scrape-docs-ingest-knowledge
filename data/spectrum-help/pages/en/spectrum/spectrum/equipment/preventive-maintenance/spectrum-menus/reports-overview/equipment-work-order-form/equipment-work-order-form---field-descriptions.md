# Equipment Work Order Form - Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Selections

Equipment code Equipment type
Enter an Equipment code and equipment
 type to include on the Work Order Form.
Yard Area Work order Priority Mechanic
Enter the yard, location code, worker
 order number, priority code, and mechanic's code to include on the
 report.
Include work ordered detail?
Select to include the detailed
 information about the work ordered.
Include active warranty information?
Select to include the equipment's
 active warranty information.
Include open P.O. items on order?
Select to include the open purchase
 order items.Note: This checkbox is only
 available if the Purchase Order module is installed for the current
 company.

Include parts used list?
Select to include a list of parts used
 during work on the equipment.
Include work order notes?
Select to include the notes recorded
 from the Equipment Work Order Entry > Notes tab.
 Include P.M. tasks due for service?
Select this checkbox to display any
 maintenance that is due for service on the work order, including service notes
 and parts lists associated with Preventive Maintenance tasks.
 Minimum % due
This field is available when the
 Include P.M. tasks due for
 service? checkbox is selected. Enter a percentage amount for
 tasks to include on the work order. For example, enter 100% to include tasks
 that are due now or overdue.
Include list of other open work orders for same equipment code?
Select this checkbox to include a list
 of all other work orders associated with the selected equipment code.
Print status

First print only? Reprint?
Select whether to print equipment work
 order forms that have never been printed (First print only) or have
 been printed before (Reprint). To print all equipment work order forms, select both
 checkboxes.
Work order status

Include open work orders? Include finished work orders? Include closed work orders?
Select to include work orders with a
 status of open, finished, and/or closed.
