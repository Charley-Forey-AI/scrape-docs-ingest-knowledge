# Equipment Work Order Entry- Properties

Preventive Maintenance equipment work orders provide
 project tracking for equipment repairs and a detailed historical repair for the equipment and
 its components.
Use the Properties tab to record the summary information about the equipment work order,
 such as the priority code, the need-by date, the mechanic's name, and so forth. The
 information on this tab displays in the Equipment Work Order Entry list box columns as
 read-only text. If a scheduled P.M. task is more than 90% due, a convenient alert message
 displays on this screen. This message only displays when the work order status is set to
 Open or Finished.
Use the Edit Equipment Work Order window to review and update any
 information that was previously entered in the New Equipment Work Order window.
