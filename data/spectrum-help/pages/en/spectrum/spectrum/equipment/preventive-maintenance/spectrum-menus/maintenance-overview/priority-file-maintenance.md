# Priority File Maintenance

Use this screen to establish user-defined priority codes,
 which can be used to track the priority of the work order in the New/Edit Equipment Work Order
 window and in Batch P.M. Service Entry.
There are sixteen colors to choose from when setting up priority codes.
Fields
Descriptions

New/Edit/Delete
Click New or Edit on a selected line to
 open the [New/Edit Priority window](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/priority-file-maintenance/newedit-priority-window)
 window.
