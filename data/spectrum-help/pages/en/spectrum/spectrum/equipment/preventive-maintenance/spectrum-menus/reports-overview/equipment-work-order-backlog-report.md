# Equipment Work Order Backlog Report

The Equipment Work Order Backlog Report is a management
 report that shows how many repair orders the equipment maintenance shop did not return to
 service by the required date specified on the work order.
This is measured by computing how many days an equipment work order is returned to service
 later than the required date.
This report will print a list of all currently-open equipment work orders in priority sequence, or a list of historical backlog delays in returning equipment to service. A backlog is defined as work which has not been completed by the specified 'required by date'. If no 'required by date' was specified for the work order, the work order will be included in the backlog.
