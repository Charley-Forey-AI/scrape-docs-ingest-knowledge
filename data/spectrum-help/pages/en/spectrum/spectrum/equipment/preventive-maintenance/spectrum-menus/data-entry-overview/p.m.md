# P.M. Schedule

Use this window to view scheduled and completed preventive maintenance tasks for the selected piece of equipment.
This window displays one line for each p.m. task that has been set up in [Equipment P.M. Setup](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/equipment-p.m.-setup). When each task is complete, then a green check mark displays in the first column next to the applicable task. An alert message will display in the header of this window if any scheduled P.M. task is more than 90% due. This alert will only display when the work order status is Open or Finished.
Note: This screen is view-only if you do not have security
 authorization.
