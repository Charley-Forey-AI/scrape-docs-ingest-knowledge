# Equipment Work Orders

Use this screen to view, add, or search for equipment orders and edit
 them in the Equipment Work Order Entry screen. You can also view equipment code data
 associated with a specified work order in the [Info Bar](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/info-bar/info-bar-faqs).
Note: The New, Edit, and Delete buttons are unavailable if
 you do not have security authorization for the Equipment Work Order Entry screen.
