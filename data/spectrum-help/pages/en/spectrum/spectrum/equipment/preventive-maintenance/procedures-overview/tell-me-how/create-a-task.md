# Create a Task

The best way to deal with maintenance needs is to plan
 ahead for them. Complete the following steps to create a task, which will allow you to stay
 ahead of any equipment maintenance needs your company may encounter.

- On the Site
 Map, click Preventive Maintenance  >  Maintenance  >  P.M. Tasks. The P.M Tasks
 Maintenance screen displays.

- Click New
 to display the New P.M Task
 window. In the Task
 code field, create a name for the maintenance task (for example, CHEFAN),
 up to 10 characters.

- In the Description
 field, enter a brief description of the task (for example, CHECK FAN BELTS),
 up to 30 characters.

- In the Frequency
 methodsection, select how often you want this task to be performed.

- The M
 option means the task's frequency will be based solely on the equipment's meter reading.
 If you select this option, enter the default meter reading amount for this task in the
 Meter field.

- The ED
 option means the task's frequency will be based on elapsed days only (for
 example, every 90 days). If you select this option, enter the default number of elapsed
 days for this task in the Number of days
 field.

- The MED
 option means the task's frequency will be based on meter reading or the elapsed days,
 which ever occurs first (for example, 5 years or 50,000 miles). If you select this
 option, enter the default meter reading amount for this task in the Meter field and enter the default
 number of elapsed days for this task in the Number of days field.

- The FD
 option means the task's frequency will be based on a fixed calendar date (for example,
 June 18). Enter the default date for this task in the Fixed day field.

- In the Labor hours
 field, enter the approximate number of hours needed by the technician to
 complete this task.

- In the Labor rate
 field, enter the rate the technician is paid to complete this task. Note: This field is typically left blank unless the task is
 associated with an equipment work order.

- Click OK
 until you return to the P.M.
 Tasks Maintenance screen. Then click the check mark in the upper right
 corner to save and close.
