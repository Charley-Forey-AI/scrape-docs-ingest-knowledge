# P.M. Tasks Performed Information - Field Descriptions

Use this window to input details regarding the completion
 of the selected task.
This information will display on the [P.M. Schedule](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule) screen.
Field
Description

Batch code
Enter the applicable batch code or
 press Enter to accept
 the default of the current operator code.
Service person
Enter the name of the person who
 performed the completed task.
P.M. date
The current Preventive Maintenance
 processing date is the default. You can enter a new date or search for the date
 that the selected task was completed.
Meter # 1
This field will default to the latest
 meter reading on or before the specified P.M. date, but you can enter a
 different meter value. This field is required if meter # 1 is defined for this
 equipment code.
Comment
Enter a note of up to 30 characters
 that pertains to the completed task.
Warehouse
These fields are required,
 and the entries will display on the [P.M. Service - Parts Used](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/batch-p.m.-service-entry/p.m.-service---parts-used) window for all tasks selected to be completed. Entries made in this window will override entries in
 Inventory Control Installation, Inventory Item, or partial entries in the
 [Standard Parts List](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule/standard-parts-list) window.
 For example, if the completion process includes two tasks (one with
 warehouse/cost category and one without) then entries made in this window
 will override both.
Note: These fields
 display only if the Update
 Inventory Control checkbox is selected on the Preventive
 Maintenance Installation screen, and there are parts detail records for the
 selected task that do not have a valid warehouse.
