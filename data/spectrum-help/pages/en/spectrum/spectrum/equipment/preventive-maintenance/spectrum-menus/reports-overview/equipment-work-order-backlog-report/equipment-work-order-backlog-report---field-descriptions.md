Format

- Open backlog

- Summary historical backlog

- Detail historical backlog

Select to display the report in open
 backlog, summary, or detail historical backlog format.
Include work orders with no returned-to-service date?
This checkbox becomes available if you
 select the Summary or Detail historical backlog format above. Select this checkbox to include work orders that do not
 have the specific date returned-to-service recorded in the work
 order.

Include only work orders returned to service date later than required?
This checkbox becomes available if you
 select the Summary or Detail historical backlog format above. Select this checkbox to include exceptions (work orders
 returned to service later than required) on the report.
