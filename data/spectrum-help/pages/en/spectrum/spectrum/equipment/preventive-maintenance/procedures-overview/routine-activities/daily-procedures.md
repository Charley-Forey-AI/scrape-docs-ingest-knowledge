# Daily Procedures

Use the following for daily Preventive Measure procedures.

- Print a maintenance schedule using P.M. Tasks Maintenance Listing.

- Enter completed maintenance once equipment has been serviced using
 Batch P.M. Service Entry.

- Enter equipment work orders using Equipment Work Order Entry.

- Print and review the P.M. Service Performed Register. Update if
 approved.
