# What is a "Memo" type of transaction on an equipment work order?

This is a way to record parts used on the equipment repair.
Notice that no costs of the material are charged to the repair, rather this is a way to
 document what was used. Another strategy for using the Memo type is to define the standard
 parts used on this type of repair.
