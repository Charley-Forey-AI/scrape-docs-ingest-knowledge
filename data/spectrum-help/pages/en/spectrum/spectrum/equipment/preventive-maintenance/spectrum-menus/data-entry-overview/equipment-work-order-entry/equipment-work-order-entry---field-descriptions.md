# Equipment Work Order Entry - Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Work order
Enter a new work order number or search
 for an existing one. You must have security authorization to add or modify the
 work order.
Equipment code
Enter or search for a valid Equipment
 code. The equipment code displays if it is associated with an existing work
 order.
Alert
An alert message displays if a
 scheduled P.M. task is more than 90% due. This message only displays when the
 work order status is set to Open or Finished.
Cost History
Click to view the Equipment Control > Equipment Cost History Detail Inquiry screen. This window displays all cost transactions incurred
 to-date on the work order, A/P invoice ,and Payroll time card. For more
 information, click on the Spectrum Help link from this screen.
