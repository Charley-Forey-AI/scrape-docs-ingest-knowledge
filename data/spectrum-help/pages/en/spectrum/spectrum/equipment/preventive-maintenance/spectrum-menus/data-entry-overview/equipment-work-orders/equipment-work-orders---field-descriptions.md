# Equipment Work Orders - Field Descriptions

Use the table below when completing the fields on this screen.
Buttons/Fields
Descriptions

Buttons

Status
Click to select open, finished, and/or
 closed work orders.
New
Click to open the New Work Order
 window. In this window, press Enter to accept the next
 work order number. In the Equipment field, enter or search for the Equipment code.
 Click OK to open
 the [Equipment Work Order Entry](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/equipment-work-order-entry)
 screen.

Edit
Click to modify the selected work order
 in the Equipment Work Order Entry screen.
Delete
Click to remove the selected work
 order.
Form
Click to display the [Equipment Work Order Form](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/reports-overview/equipment-work-order-form) window for the selected equipment work order. Note: This button is unavailable if you do not have security
 authorization.

List box

The List box displays equipment work
 orders that have been entered in Equipment Work Order Entry. Select a record to
 view the associated equipment code in the Info Bar.
