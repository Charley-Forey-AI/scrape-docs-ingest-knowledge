# User-Defined Fields Maintenance Overview

Use the user-defined screens to view, edit, add, and delete
 Preventive Maintenance user-defined fields.
The fields in these screens will be modified based on the settings on the Preventive
 Maintenance Installation - User-Defined Fields tab.
