# Report Calculations - Equipment Work Order Backlog Report

The following tables describe how the Equipment Work Order Backlog Report is calculated. These apply to historical report formats only.
Field
Description

W.O. count and On-time count
The number of work orders within the
 selection criteria will display in these columns.
Mean days early
The total number of days early among
 all work orders included in the associated count, divided by the count are
 displayed in this column. This number is shown as a negative value.
Mean days late
The total number of days late among all
 work orders included in the associated count, divided by the count are
 displayed in this column. This number is shown as a positive value.
Mean backlog days
The net backlog among all work orders
 (Total days late - Total days early), divided by the work order count.
Returned to Service - Days Early and Returned to Service - Days Late
The returned to service days are
 calculated by subtracting the 'date required' from the 'date returned'. If the
 value is positive, the amount is shown in the 'Days late' column. If the amount
 is negative, the amount is shown in the 'Days early' column.
