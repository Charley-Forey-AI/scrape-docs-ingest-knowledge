# Equipment Work Order Cost Report - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, when running the Equipment Work Order Cost Report, Spectrum includes the
 equipment on the report only if you have permission to access the piece of equipment.
The system compares the cost center assigned to the equipment with cost centers in your
 operator's assigned cost center scheme, and if the cost center is not included, then
 that piece of equipment is not shown on the report. The Cost group field on this screen will
 default to the 'Report default' of the operator's cost center scheme, if specified, and
 otherwise it will default to ALL.
Spectrum filters which equipment is included on the report using the
 operator's cost center security scheme.
