# New/Edit Problem Cause Window

Use this screen to set up and maintain a list of problem
 cause codes.
Problem cause codes are located on the [Problem Cause File Maintenance](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/problem-cause-file-maintenance) screen.
Tip: Use the New Problem Cause window to add new equipment
 repair status codes. Use the Edit Problem Cause window to make changes to existing
 equipment repair status codes. As entries are completed, the data will display on the
 Problem Cause File Maintenance starting screen.
Field
Description

Problem cause
Enter up to 10 characters for a new
 problem cause code (for example, weather).
Problem description
Enter up to 50 characters for a brief
 description of the new problem cause code. In the Edit
 window, make any required changes to the existing problem cause code's
 description.
Note: If a long
 description is entered in this field, the description may appear truncated
 in areas of the software where space is limited.
