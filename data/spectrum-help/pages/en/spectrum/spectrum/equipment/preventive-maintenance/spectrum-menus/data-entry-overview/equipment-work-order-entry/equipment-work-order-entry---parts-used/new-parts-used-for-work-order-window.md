# New Parts Used for Work Order Window

Use this tab to store information regarding the parts used to complete the selected equipment work order.
Once this window is complete and the operator clicks OK, the details entered in this window display on the [Equipment Work Order Entry - Parts Used](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/equipment-work-order-entry/equipment-work-order-entry---parts-used) tab. Memo type entries are of particular benefit if the Equipment shop does not maintain a perpetual inventory (tied to a G/L inventory asset account). Memo entries can also serve as a scratch pad for the mechanic to note parts used in a field-by-field entry.
