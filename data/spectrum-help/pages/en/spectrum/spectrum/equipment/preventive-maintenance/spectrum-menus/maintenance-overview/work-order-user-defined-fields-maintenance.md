# Work Order User-Defined Fields Maintenance

Use this screen to set up virtually unlimited
 contract-specific user-defined fields related to theequipment work order. In addition, this
 screen is used to view, edit, and delete Accounts Receivable customer user-defined fields.

- If the Allow setup of
 company-specific fields checkbox is selected on the Preventive
 Maintenance - User-Defined Fields tab, company-specific user-defined fields can be set
 up for the current company.

- If the Allow setup of
 non-company-specific fields (company ALL) checkbox is selected on the
 Preventive Maintenance - User-Defined Fields tab, users are permitted to set up
 user-defined fields for all companies.

-  If the Show
 non-company-specific fields in maintenance work order entry (company ALL)
 checkbox is selected on the Preventive Maintenance - User-Defined Fields
 tab, user-defined fields for all companies can be added and edited. Click the New or Edit
 button to add or modify the selected [user-defined fields](/en/spectrum/spectrum/getting-started/understanding-user-defined-fields/newedit-user-defined-field).
Work Order User-Defined Fields
 Listing
Use this screen to print a listing of
 work order-related user-defined fields for a specified range of companies. This
 listing can include work order-related user-defined alpha fields, date fields,
 numeric fields and/or validation codes for the selected field type(s) and selected
 companies.
