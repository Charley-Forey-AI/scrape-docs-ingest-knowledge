# Work Ordered - Properties - Field Descriptions

Use the table below when completing the fields on this screen.
Header
Fields
Descriptions

Work order
The work order number,
 equipment code, and description default from the [Equipment Work Order Entry- Properties](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/equipment-work-order-entry/equipment-work-order-entry--properties) window.
Equipment

Properties tab
Fields
Descriptions

Summary

Component group
Enter a component group, or
 double-click on this field or press F4 to open the Search
 Component Group window and select from a list of available component groups.
 To assign a component group, select the group you want and then click
 OK.The Search Component Group window also allows you to
 add new groups by clicking the New button, and you can
 edit or delete groups using the corresponding buttons.

Warranty-related
Use the drop-down menu to select
 whether this equipment repair is covered under the equipment's warranty.
 Select from the following options:

- Yes Select this option if the required work is
 covered by the selected piece of equipment's current warranty.

- Possible Select this option if only some of the
 warranties for the selected piece of equipment have expired.

- No, expired Select this option if all warranties
 for the selected piece of equipment have expired.

- No, not applicable Select this option if no
 warranties are present for the selected piece of equipment.

Repair status
Enter the applicable repair status
 code for the selected equipment work order, or double-click on this field or
 press F4 to select
 from a list of available repair status codes. Repair
 status codes are set up and maintained in the [Repair Status File Maintenance](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/repair-status-file-maintenance) screen.

Work ordered note
Enter a brief description (up to 250
 characters) of the work that was ordered for the selected piece of
 equipment.
Reference

Component reference
Enter a component reference code, or
 double-click on this field or press F4 to select from a list
 of available component reference codes. For convenience, the Search
 Component window only displays components for the component group specified
 above. If no component group was specified above, then all components will
 display in the window (based on search criteria in the window header). Even
 though the search window is filtering on component group, the operator will
 be permitted to enter any valid component, for reference purposes.Component reference codes are set up and maintained in
 the Equipment Control > Equipment Component Maintenance screen. Please refer to the Equipment Control Help file
 for more information regarding this topic.

Sequence
Enter a sequence code associated
 with the component group. Entry in this field is optional.This field serves as a reference for cases when it is
 useful to indicated a particular component within the component
 group.

Estimated hours
Enter the estimated number of hours
 required to complete this equipment work order. Entry in this field is
 optional.
Actual hours
Enter the actual number of hours
 used to complete this equipment work order. Entry in this field is
 optional.
Status

- Open

- Pending

- Closed

Select whether this equipment work
 order is currently Open, Pending, or Closed.
