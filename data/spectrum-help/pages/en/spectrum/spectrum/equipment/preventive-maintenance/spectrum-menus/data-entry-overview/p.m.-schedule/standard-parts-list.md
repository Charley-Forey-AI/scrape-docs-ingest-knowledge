# Standard Parts List

Use this window to add or edit the estimated inventory
 items and quantity required for this task.
This information displays on the Parts & Labor Requirements
 report. You can access this window from [P.M. Schedule](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/data-entry-overview/p.m.-schedule) and [Equipment P.M. Setup](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/maintenance-overview/equipment-p.m.-setup).
Parts entered here must have been previously set up in Inventory Control > Inventory Items.
Note: The [New, Edit, View, and Delete buttons](/en/spectrum/spectrum/materials/order-processing/spectrum-menus/maintenance-overview/customer-ship-to-addresses/customer-ship-to-addresses---field-descriptions/new-edit-view-and-delete-buttons)buttons will not display if you do not have security
 authorization.
