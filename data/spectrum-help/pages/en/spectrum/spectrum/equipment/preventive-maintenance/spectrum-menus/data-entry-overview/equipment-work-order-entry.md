# Equipment Work Order Entry

Use this screen to enter equipment work orders. Preventive
 Maintenance provides the ability to track maintenance and repair costs by project.
For example, it is possible to identify and track all equipment costs,
 such as for a major engine overhaul. The system provides an option for tracking a work
 order for each maintenance/repair, based on Preventive Maintenance installation. If the
 company is using equipment work order tracking, the system will prompt for work orders
 throughout Spectrum. Whenever an operator records an equipment cost (Accounts Payable,
 Payroll, Equipment Control, Accounts Receivable or Purchase Order), the system will prompt
 for the work order number. The work order number is stored in the Equipment Control history
 file for later reference in the Preventive Maintenance module, in order to review the total
 cost of that work order for that piece of equipment. An [Equipment Work Order Cost Analysis](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/inquiries-overview/equipment-work-order-cost-analysis)
 screen and report is provided, as is an [Equipment P.M. Service History](/en/spectrum/spectrum/equipment/preventive-maintenance/spectrum-menus/inquiries-overview/equipment-p.m.-service-history)
 screen and report.
If the Canadian lexicon designation (en-ca) is selected in Company Installation, labels on all meter types will display using the preferred Canadian measurements (for example, "kilometers" instead of "miles").
