# Equipment Work Order Cost Report - Field Descriptions

Use the table below when completing the fields on this screen.
Fields
Descriptions

Selections

Equipment code Equipment type
Enter an Equipment code and equipment
 type to include on the Work Order Form.
Yard Area
Enter the yard code and associated
 equipment location.
Work order Problem type Problem cause Priority
Enter a work order number and the
 associated problem type, problem cause, and priority code to include on the
 report.
Equipment status
Enter the equipment status codes
 associated with the work order's equipment to include on the report.
Cost group
The Cost group field will default
 to the 'Report default' of the operator's cost center scheme, if specified;
 otherwise it will default to ALL.Note: This field displays only when the Enterprise Installation Use cost centers radio
 group is set to Yes for this company.

From finished date To finished date
Enter the first and last work completed
 dates to include on the report. Use date shortcuts, or click the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)

From cost history date To cost history date
Enter the first and last cost history

 icon to calculate a different date.
Include work ordered info?
Select this checkbox to include detail
 work order task information on the report.When this check
 box is selected, the component group, description, and associated work
 ordered notes will display on the Summary and History Detail report formats.
 For the Component Group report format, this checkbox controls whether the
 work ordered note associated with the component group is included in the
 report.

Include costs not assigned to work order?
Select this checkbox to include a
 summary line for non-work order costs for each equipment code on the
 report.

Work order status

- Include open work orders?

- Include finished work orders?

- Include closed work orders?

Select to include information about
 open, closed, and/or closed work orders on the report.
Equipment status types

- Include active equipment?

- Include inactive equipment?

- Include retired equipment?

Select to include information about
 active, inactive, and/or retired equipment on the report. If the Equipment Tracking module is present, the first two checkboxes are
 automatically selected.
