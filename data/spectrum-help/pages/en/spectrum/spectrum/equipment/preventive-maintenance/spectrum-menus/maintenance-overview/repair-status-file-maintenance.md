# Repair Status File Maintenance

Use this screen to establish user-defined repair status
 codes, which can be applied to work orders to track the status of repairs.
There are sixteen colors to choose from when setting up priority codes. The description and
 color code you choose will appear on the New/Edit Equipment Work Order window and Equipment
 Work Order form. Repair status codes can also be used for selection purposes when running
 certain reports.
