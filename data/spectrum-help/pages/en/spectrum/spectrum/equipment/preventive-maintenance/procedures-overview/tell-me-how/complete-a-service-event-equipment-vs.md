# Complete a Service Event (Equipment vs. Rental)

When completing a service event in Preventive Maintenance,
 it is important to make sure all tasks are set up properly.
Tasks are originally set up in the P.M. Task Maintenance screen and
 then regular equipment maintenance work is scheduled using the Scheduled P.M.
 Maintenance screen, based on the frequency code from the task screen.
After setting up a task, the process for completing a service event
 is to first proceed to the Reports menu to print out a listing of tasks to complete.
 Next, go to the Data Entry menu to enter the maintenance performed.
The following procedure is for tasks that have already been set up,
 but need to be completed and updated.
Note: If you select Unprinted, the print status will
 change to Printed (an 'X' will display) in the Equipment P.M. Setup screen after the
 report has been run, unless the item is scheduled for maintenance outside the effective
 date range.

1. On the Site Map, click Preventive Maintenance  >  Reports  >  P.M.Schedule.

1. Complete the Selections portion of the screen.

1. At the Print status
 section, select Both.


1. Complete the Report type and Requirement reports sections and click Preview and then click the check mark on the right corner to save return to the site map.

1. On the site map, click Preventive Maintenance  >  Data Entry  >  P.M. Service Performed
 Register.

1. Enter the appropriate Batch code. If the Equipment
 Work Order Entry screen was used, the system will assign work order numbers (only if
 the work orders have a completed status). If work order numbers were not used, the
 report will print OTHER in the work order portion.

1. Click to Preview . Once run, the register will update information in the maintenance schedule, service history, Equipment Control meter history, and Inventory Control transaction files.
