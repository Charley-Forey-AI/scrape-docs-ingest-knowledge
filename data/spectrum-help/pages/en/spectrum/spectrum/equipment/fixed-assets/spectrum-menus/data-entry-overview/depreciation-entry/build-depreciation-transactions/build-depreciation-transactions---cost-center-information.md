# Build Depreciation Transactions - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen in this company, Spectrum verifies that the fixed asset has a valid cost center, assigned to either the fixed asset or department file maintenance, that can be used to post to the General Ledger. If no cost center is assigned, then the fixed asset will not be included in the calculation and Spectrum will display a warning message after the calculation.
