# MACRS Table Maintenance - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields
Descriptions

Description
Enter or edit a description of the MACRS table being entered.

Properties

Property type
Enter the property type of the asset. The different types are any combination of the following IRS defined property type numbers: 3, 5, 10, 15, 18, or 19. If more than one type is being entered, separate the numbers with a comma and do not enter spaces.

Recovery period (years)
Enter the number of years over which the depreciation may be dispersed.

From/to service
Enter the date range during which assets may be depreciated using this table.

Month placed in service
Use the Action buttons to maintain.

Year
The depreciation year displays. No entry is required.

1-12
Enter the percentage allowed for the years displayed. For property types 15, 18, or 19 only, a percentage must be entered for each of the 12 months of the year that the asset will be depreciated, based on the percent allowed for the month when the asset was placed in service.
