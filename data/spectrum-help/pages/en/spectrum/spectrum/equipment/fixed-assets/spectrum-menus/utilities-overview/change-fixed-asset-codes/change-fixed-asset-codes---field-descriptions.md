# Change Fixed Asset Codes - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields/Buttons
Descriptions

Old asset

G/L control code
Asset number
Sequence #

1. Enter the existing G/L control code for the asset you wish to change. Following entry of the control code, the full corresponding description displays.

1. Enter the existing asset number.

1. Enter an existing sequence number,.

New asset

G/L control code
Asset number
Sequence #

1. Enter the G/L control code you wish to assign to the new asset. This must be a
 valid control code previously assigned in G/L Control Code
 Maintenance. Following entry of the control code, the
 full corresponding description displays.

1. Enter a unique code/number to be used in identifying the new asset.

1. Enter a sequence number from 1 to 9. Enter 1 for assets being entered for the first time. Sequence numbers are changed only when the depreciation method changes. The new sequence information should be a duplicate of the old, showing a different appreciation method and sequence.
