# Update Depreciation to Master - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields
Descriptions

G/L accounts updated to
The General Ledger accounts to which this information will be updated display. No entry is allowed.

Depreciation date
Press Enter to accept the default date based on the date specified during the Build Depreciation Transactions. Otherwise, enter the depreciation calculation date, or press Enter to accept the default of the date the depreciation was built.
Make sure the depreciation has been calculated correctly before proceeding with the update.
