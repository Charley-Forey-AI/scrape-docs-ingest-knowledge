# Depreciation Entry - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields/Buttons
Descriptions

G/L control
The G/L control code of the asset to be edited and the G/L control code description display.

Asset # Seq Asset description
The asset number and asset description associated with the depreciation entry display.

Book depr - current Tax depr - current Other depr - current
Enter the desired current depreciation amount for the appropriate depreciation type, or press Enter to accept the displayed pre computed amount.
After the build is run, you can add amounts, or edit pre computed amounts.

Equipment
The equipment code and associated description displays. No entry is allowed.
