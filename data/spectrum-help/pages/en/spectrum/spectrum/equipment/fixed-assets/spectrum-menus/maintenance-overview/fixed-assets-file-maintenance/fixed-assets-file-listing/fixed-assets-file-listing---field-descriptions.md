# Fixed Assets File Listing - Field Descriptions

Use the table below for assistance when entering fields on
 this screen.
Fields
Descriptions

G/L control code
Enter the G/L control code associated with the assets you
 want to include on this report, or use the drop-down to select from a list
 of valid G/L control codes.

Department
Enter the department code associated with the assets you
 want to include on this report, or use the drop-down to select from a list
 of valid department codes.

From / to date capitalized
Enter a date range to include assets placed in service
 during this range on the report.

Asset number
Enter the asset number or numbers you want to include on
 this report.

Status
Select to generate the report for active or inactive
 assets.

Sort by
Select a report sort option.
