# Fixed Assets Deletions Register

The Fixed Assets Deletions Register prints a list of all
 assets deleted from the system since the last Deletions Update was performed.
After the report has printed, the operator is provided the opportunity to purge the
 deletions.
Important: This report should be retained as a permanent
 record of the company.
After the Fixed Assets Deletions Register prints, the
 Update screen appears.
