# G/L Control Code Maintenance - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields
Descriptions

G/L control code
Enter a two-character code to be used as a control code, or select from a list of valid control codes. This code is used for posting book figures to the General Ledger. When specific assets are entered into the system, one of the codes here will be assigned.
Note: The G/L Control Code Maintenance
 screen has validates General Ledger account codes to ensure that they are
 not Direct Cost type.

Description

Asset account code
Enter the General Ledger asset account for assets assigned with this control code, or select from a list of valid account codes.

Contra account code
Enter the General Ledger account to be used as a contra asset account for assets assigned with this control code, or select from a list of valid account codes. Spectrum will credit this G/L account when the Fixed Assets G/L Summary Update is performed.

Expense account code
Enter the General Ledger account to be used as the expense account for assets assigned with this control code, or select from a list of valid account codes. Spectrum will debit this G/L account when the Fixed Assets G/L Summary Update is performed.

Equipment expense account code
Enter the General Ledger account to be used as the expense account for assets assigned with this control code, or select from a list of valid account codes. This field is used during the monthly Fixed Assets update to Equipment Control to record depreciation expense when an asset is tied to a piece of equipment.
