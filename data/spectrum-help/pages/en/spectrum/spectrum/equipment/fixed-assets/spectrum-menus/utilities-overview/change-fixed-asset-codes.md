# Change Fixed Asset Codes

This screen is used to change Fixed Asset codes throughout the system, including all history files. Spectrum will verify that the new code is (a) different than the old code, (b) does not already exist in Fixed Assets File Maintenance, and (c) is not already recorded in the code change utility table.
