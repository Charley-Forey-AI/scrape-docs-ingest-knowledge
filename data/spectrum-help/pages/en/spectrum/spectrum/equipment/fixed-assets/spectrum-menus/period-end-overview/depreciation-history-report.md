# Depreciation History Report

Depreciation History Report prints the depreciation history
 of each asset.
After the report has printed, the operator is provided the opportunity to purge the history
 records.
Important: This report should be retained as a permanent
 record of the company.
Enter the first depreciation date range to be
 included on the report, select a sort option, and click Preview.
