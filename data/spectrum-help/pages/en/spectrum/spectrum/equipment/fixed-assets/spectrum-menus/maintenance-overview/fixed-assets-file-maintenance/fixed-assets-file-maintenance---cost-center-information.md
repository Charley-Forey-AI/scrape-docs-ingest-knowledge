# Fixed Assets File Maintenance - Cost Center Information

If cost centers are being used, a Cost Center field displays beneath the Sequence # field. This field associates the fixed asset with a particular cost center. Enter the cost center that you want to apply to the specified asset or double-click in this field to select from a list of available assets.
In addition, a Setup button displays allowing you to view the Fixed Assets Cost Center Setup screen. For more information about this screen, please refer to the Enterprise Management Help file.
When a new asset is added in this screen, Spectrum verifies that this cost center is valid for your operator. Spectrum compares the asset's cost center with cost centers in your operator's assigned cost center scheme, and if the cost center is not present, then that cost center cannot be assigned.
When adding or changing the cost center, Spectrum validates that the cost center entered is valid for all G/L accounts entered for the G/L account code in G/L Control Code Maintenance.
When editing, viewing, or deleting a fixed asset, Spectrum compares the cost center assigned to the asset with cost centers in your operator's assigned cost center scheme, and if the cost center is not present, then that asset is not shown.
Note: The Vendor field is simply a memo field, so the vendor cost center does not have to match the cost center assigned to the fixed asset.
