# Fixed Assets Register

This screen displays after the Fixed Assets Deletions
 Register or Fixed Assets Additions Register prints.
 Do not continue until the report has printed correctly. Once it has printed
 and is accurate, click Continue. This function purges all detail
 items from the deletions register file.
Important: Extreme caution should be
 exercised when performing purges.
