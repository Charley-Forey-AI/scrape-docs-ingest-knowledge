# Change Fixed Asset Codes Update

The Change Fixed Asset Codes Update
 screen is used to update the fixed asset information entered in Change Fixed Asset
 Codes Maintenance. You should have a copy of the Change Fixed Asset Codes
 Listing before performing this update.
This update will perform code changes throughout the system. It is a
 productivity tool to assist you in making code changes on a wide scale. Be advised that any
 custom files, pre-release and new files, and non-updated data files may not have been
 included in this automated process. Therefore, confirm that updates have been performed
 prior to using this function and review records carefully, including installation screens,
 immediately following completion of the program.Important: This update
 should never be performed while others are logged into Spectrum. Confirm that this
 workstation is the only one currently logged into Spectrum prior to proceeding with the
 update.
