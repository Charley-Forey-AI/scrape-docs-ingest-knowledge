# Depreciation Entry

Depreciation Entry is used to change any depreciation information that was created using the Build Depreciation Transactions. Changes are made in the lower portion of the screen.
The system will not allow depreciation entry after the General Ledger update has been run until you run Build Depreciation Transactions again.
