# Changing Fixed Asset Codes

Learn how to change existing Fixed Asset codes.

1. On the Site Map screen, click Fixed Assets  >  Utilities  >  Change Fixed Asset
 Codes.

1. Click New.

1. In the Old asset section enter the existing G/L control code, Asset number and
 Sequence #. If you do
 not know these, press F4 to
 search available options.

1. In the New Asset section select a new G/L control code. This must be a
 valid control code previously assigned in G/L Control Code Maintenance.

1. Enter a unique code/number in the Asset number field.

1. Enter a sequence number from 1 to 9 in the Sequence # field. Enter 1 for
 assets being entered for the first time. Sequence numbers are changed only when the
 depreciation method changes.

1. Click the Listing button to access the Change Fixed Asset Codes Listing
 screen and click Preview to
 verify your code change.

1. After previewing or printing the report, the system
 will return you to the Change Fixed Asset Codes screen, and the Code Change Warning
 will appear. Read the warning and click OK.

1. The Change Fixed Asset Codes Update screen will appear. Click Continue.Maintenance screens are usually updated as part of the Change
 Fixed Asset Codes Update. Nevertheless, it is a good idea to review the affected
 codes in the Change Fixed Asset Codes Maintenance screen after the update to
 ensure they were properly updated.
