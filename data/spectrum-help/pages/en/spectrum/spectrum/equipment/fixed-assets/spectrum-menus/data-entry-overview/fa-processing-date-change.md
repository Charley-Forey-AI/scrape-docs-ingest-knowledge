# F/A Processing Date Change

The F/A Processing Date Change screen is used to change the date under which this module operates. The processing date must be set separately within each application being run on the system.

1. The current processing information displays in the Current settings section. You may
 enter a new processing date in the field or click the date on the calendar. An error
 message displays if the new date is before the minimum processing date or after the
 maximum processing date established during installation by the system administrator
 using the Processing Date Change screen.

1. Click Save to retain your changes.
