# Depreciation Method Evaluation Report - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields
Descriptions

Asset value
Enter the current value of the asset to be depreciated.

Salvage value
Enter the salvage value of the asset to be depreciated.

Life in months
Enter the asset life in months for the asset.

Capitalization date
Enter the capitalization date for the asset.

Start date
Enter the date that the depreciation schedule is to begin.

Depreciation method
Select a depreciation method to use for this report:

Declining balance %
MACRS table ID

- If Declining balanced is the selected depreciation method, enter the percent to be used in the depreciation calculation.

- If MACRS is the selected depreciation method, enter the ID of the Accelerated Cost Recovery System table to be used in the depreciation calculation, or press F4 to select from a list of valid MACRS codes.
