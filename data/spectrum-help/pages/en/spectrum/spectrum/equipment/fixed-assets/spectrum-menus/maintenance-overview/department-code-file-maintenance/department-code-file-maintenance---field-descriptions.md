# Department Code File Maintenance - Field Descriptions

Use the table below for assistance when entering fields on this screen.
Fields
Descriptions

Department code
Enter a code for the department being defined, or press F4 to select from a list of valid department codes. If the General Ledger module is installed, use the same department codes as used in G/L.

Description
Enter a description of the department, if desired.
