# Build Depreciation Transactions

Build Depreciation Transactions is used to calculate the depreciation for assets as of a specific date. Simply enter the date from which depreciation is to be calculated and click Continue.
The following reflects the proper procedure used to pass depreciation data from the Fixed Assets module to the Equipment Control module:

1. Access the Fixed Assets module and process the depreciation calculations for the same
 period using Build Depreciation Transactions through update.

1. Continue the monthly processing in Equipment Control using DLI Entry through
 update.
