# Period End Procedures

- Execute the Build Depreciation Transactions for Book, Tax, and/or Other.

- Print the Depreciation History Report and check for accuracy.

- If adjustments or corrections are necessary, use Depreciation Entry. Reprint the edit listing and proof again, as necessary.

- When the information appearing on the report is correct, update the computed depreciation to the master files by executing Update Depreciation to Master.

- If Fixed Assets information is to be posted to the General Ledger, execute the Fixed Assets G/L Summary Update.

- Print and save [Depreciation Registers](/en/spectrum/spectrum/equipment/fixed-assets/spectrum-menus/reports-overview/depreciation-registers) for Book, Tax, and Other Depreciation.
Important: Make a backup!
