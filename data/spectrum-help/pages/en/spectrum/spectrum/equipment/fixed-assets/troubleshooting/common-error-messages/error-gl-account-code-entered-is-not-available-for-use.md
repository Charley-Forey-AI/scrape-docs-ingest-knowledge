# Error: G/L Account Code Entered Is Not Available For Use.

If a General Ledger account code with the status of Not used is entered, this message displays. This error message disallows further data entry.
To change the General Ledger account code's status, click General Ledger > Maintenance > Master File to open the G/L Master File Maintenance screen and in the Status section, select either Active or Inactive.
