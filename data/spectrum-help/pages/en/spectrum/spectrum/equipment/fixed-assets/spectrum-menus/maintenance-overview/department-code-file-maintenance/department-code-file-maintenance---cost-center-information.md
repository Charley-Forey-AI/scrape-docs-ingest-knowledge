# Department Code File Maintenance - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, a required Cost center field displays beneath the Department code field. This field associates the department with a particular cost center. In addition, a Setup button displays allowing you to access the Department Code Cost Center Maintenance screen. For more information about this screen, please refer to the Enterprise Management Help file.
When a new department is added in this screen, Spectrum verifies that this cost center is valid for your operator. Spectrum compares the department's cost center with cost centers in your operator's assigned cost center scheme, and if the cost center is not present, then that cost center cannot be assigned.
When editing, viewing, or deleting a department, Spectrum compares the cost center assigned to the department with cost centers in your operator's assigned cost center scheme, and if the cost center is not present, then that department is not shown.
