# Build Unassigned Equipment Moves window

Use this window to create a new unassigned move record for
 the equipment's destination job, if there are no existing move records for the selected piece
 of equipment for that day.
The window also creates an unassigned moved record for the equipment's origination job, if
 there are no existing move records for the assignment end date.
Once this build is complete, the Unassigned Moves list box on the Schedule Equipment Moves screen will show the newly-built and any other unassigned moves.
