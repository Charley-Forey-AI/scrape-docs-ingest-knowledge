# Work Schedule Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

General selections

From / To date
Enter the starting and ending dates to
 include on the report, or press Enter to accept the system default of the earliest date on file
 and the system default of the current Resource Scheduling processing date.
Job selections

Job
Enter the company code (if applicable)
 and job number to include on the report, or press Enter to accept the default
 of ALL.
Division
Enter the code associated with the job
 division to include on the report, or press Enter to accept the default
 of ALL.
Crew
Enter the code associated with the job
 crew to include on the report, or press Enter to accept the default
 of ALL.
Equipment selections

Equipment
Enter the code associated with the
 piece of equipment to include on the report, or press Enter to accept the default
 of ALL. If the Operator only has one authorized equipment
 company in Resource Scheduling, this company will default; otherwise, the
 current company will default.

Type
Enter the code associated with the job
 to include on the report, or press Enter to accept the default
 of ALL.
Division
Enter the code associated with the
 equipment division to include on the report, or press Enter to accept the default
 of ALL.
Employee selections

Employee
Enter the code associated with the
 employee to include on the report, or press Enter to accept the default
 of ALL.
Department
Enter the code associated with the
 employee department to include on the report, or press Enter to accept the default
 of ALL.
Area
Enter the code associated with the
 employee location to include on the report, or press Enter to accept the default
 of ALL.
Other selections

New page for each job?
Select this checkbox to create fresh
 report page for each job that is included on the report; otherwise, leave this
 checkbox clear. This checkbox is available only if the
 Job option is
 selected in the Sort
 by field.

Include a Gantt chart?
Select this checkbox to print a Gantt
 chart listing the assigned job by date for all the resources at the end of the
 report; otherwise, leave this checkbox clear.  If the
 Sort by field
 is set to Resource, this checkbox is automatically selected.


Sort by
Select whether to organize the report
 by Job, Resource, or
 Date.  If you select the Date option, the report
 will sort by date, resource, and then job. If you select the Resource option, the
 report will sort by resource code and then by date, and then job.
