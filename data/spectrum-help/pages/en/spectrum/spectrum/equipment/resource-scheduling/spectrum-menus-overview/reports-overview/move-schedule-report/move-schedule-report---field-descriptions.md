# Move Schedule Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

General selections

From / To
Enter the starting date to include on
 the report, or press Enter to accept the system default of the earliest date on
 file.
Needed by
Enter the date the equipment is
 required at the new location, or press Enter to accept the system
 default of the current Resource Scheduling processing date.
Equipment
Enter the company code (if applicable)
 and code associated with the piece of equipment to include on the report, or
 press Enter to accept
 the default of ALL. If the Operator only has one
 authorized equipment company in Resource Scheduling, this company will
 default; otherwise, the current company will default.

Mover code
Enter the company code (if applicable)
 and code associated with the party responsible for moving the equipment to
 include on the report, or press Enter to accept the default of ALL.
Move from

Job
Enter the company code (if applicable)
 and job number associated with equipment's original location, or press
 Enter to accept
 the default of ALL.
Vendor
Enter the vendor code associated with
 equipment's original location, or press Enter to accept the default
 of ALL.
Yard
Enter the yard code associated with
 equipment's original location, or press Enter to accept the default
 of ALL.
Area
Enter the code associated with
 equipment's original location, or press Enter to accept the default
 of ALL.
Move to

Job
Enter the company code (if applicable)
 and job number associated with equipment's destination, or press Enter to accept the default
 of ALL.
Vendor
Enter the vendor code associated with
 equipment's destination, or press Enter to accept the default of ALL.
Yard
Enter the yard code associated with
 equipment's destination, or press Enter to accept the default of ALL.
Area
Enter the code associated with
 equipment's destination, or press Enter to accept the default of ALL.
Sort by
Select whether to organize the report
 by Mover, Move date,
 or Equipment.

- If you select the Mover option, the
 report will be sorted by mover code and then date and equipment code.

- If you select the Move date option, the
 report data will be organized by date and then equipment and mover.

- If you select the Equipment option, the
 report will be sort by equipment code and then by date and mover.
