# Move Schedule Report

The Move Schedule Report lists the scheduled equipment
 moves, sorted by move date, equipment or mover (default).
The report draws its data from the Schedule Equipment Moves screen. This screen is
 protected by company-based security and only those operators with the required security
 settings can access this screen's information.
