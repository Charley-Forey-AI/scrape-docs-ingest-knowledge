# Purge Schedule History

Use this screen to purge all employee and equipment history
 through a specified date.
After the corresponding report is either printed or previewed, the Purge Schedule History
 update screen displays.
This screen is protected by company-based security and only those operators with the required security settings can access this screen's information.
