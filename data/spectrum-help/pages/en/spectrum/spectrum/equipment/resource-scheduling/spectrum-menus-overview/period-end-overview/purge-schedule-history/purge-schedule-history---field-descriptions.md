# Purge Schedule History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldDescription
Employee codeEnter the code associated with the
 party responsible for moving the equipment to include in the purge, or press
 Enter  to accept
 the default of ALL.If the Operator only has one
 authorized equipment company in Resource Scheduling, this company will
 default; otherwise, the current company will default.

Equipment codeEnter the company code (if applicable)
 and equipment code associated with the schedule history to include in the
 purge, or press Enter
  to accept the default of ALL.
Assigned jobEnter the company code (if applicable)
 and job code associated with the schedule history to include in the purge, or
 press Enter  to accept
 the default of ALL.
Purge through dateEnter the last date to include in the
 purge. If this field is left blank, then no records will be purged from the
 system.
