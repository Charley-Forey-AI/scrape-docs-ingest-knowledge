# Add Operators to Resource Scheduling

Follow these steps to give operators access to jobs and equipment for specified companies while in the Resource Scheduling module.

- On the Site
 Map, click System
 Administration.

- Click the Installation tab, then click Resource Scheduling.

- Press Enter to bypass the first two fields and select the Allow multi-company for scheduled jobs and
 equipment checkbox.

- When the list box area displays, click the New button at the bottom of the list
 box. The New Operator window displays.

- In the Operator field, click the drop-down arrow to display the Search
 Operators window. Select an operator and click OK.

- Select the Multi-company jobs and Multi-company equipment checkboxes as appropriate.

- Clear a checkbox to deny the operator access to jobs or
 equipment,

- Select the checkboxes so that you can then specify the
 companies the operator can access.

- Click OK
 to save your changes. The operator should now display in the list box section of the
 Resource Scheduling Installation screen.
