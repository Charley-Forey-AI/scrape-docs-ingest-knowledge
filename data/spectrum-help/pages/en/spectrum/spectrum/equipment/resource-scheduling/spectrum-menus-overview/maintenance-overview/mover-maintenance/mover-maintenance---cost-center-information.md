# Mover Maintenance - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, the list box will only display movers if the current operator has
 security for the cost center assigned to that employee, or one of the cost centers assigned to
 that vendor.
No overrides apply from the operator's cost center scheme.
The operator cannot enter an employee or vendor in the New/Edit Mover
 window unless s/he has security for the cost center assigned to that employee, or one of
 the cost centers assigned to that vendor. The search windows for employee or vendor will
 also only display codes that the operator has cost center security for.
