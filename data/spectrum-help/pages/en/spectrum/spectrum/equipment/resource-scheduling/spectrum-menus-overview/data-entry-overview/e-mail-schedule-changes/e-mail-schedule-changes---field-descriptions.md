# E-mail Schedule Changes - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

General selections

Include only items for which email has not yet been sent?
Select this checkbox to send an e-mail
 message only for items for which an e-mail has not already been sent. This checkbox's setting will determine if all records
 are available for selection or only scheduled employees and moves that have
 not yet been sent are available.

Hide email addresses of other receipients?
Select this checkbox to protect the
 privacy of the e-mail recipients by hiding their e-mail addresses.  If you select this checkbox, the recipient's e-mail
 information will display in the e-mail application's BCC (blind carbon copy)
 field instead of the To field.
Note: Some versions of Outlook may not display the BCC field, even if the field
 is populated by Spectrum. If you select this checkbox, the e-mail addresses
 are hidden, regardless if Outlook displays the BCC field.

Start date / move date
Enter a date for the scheduled
 employees that is based on the start date for the assignment or enter a date
 for scheduled moves that is based on the move date or press Enter to accept the
 default of the current Resource Scheduling processing date.
Assigned job
Left-click on any assignment bar to
 display the selected job's address, as well as the start and end dates. Right-click on any assignment bar in the scheduling grid
 to open a menu that offers the following options:

- Edit Select this option to open the Edit Assignment
 window. Use this window to alter the resource's job assignment details,
 such as the start date, end date, crew number, and more.

- Color Select this option to open the Edit Job Color
 window. Use this window to alter the color scheme associated with the
 selected job.

- Unassign Select this option to release the selected piece
 of equipment or the employee from the job.

Employees

Include scheduled employees?
Select this checkbox to include
 scheduled employees in the e-mail notification process. For employees assigned to a job, the e-mail subject line will read:
 Work assignment for <job
 start date. The automated message includes the job number,
 job site address, and job start time. Additional text can be manually
 entered in the e-mail window as needed.

Employee
Enter the employee code to include in
 the e-mail notification process, or press Enter to accept the system
 default of ALL. Note: This field is available
 only if the Include scheduled employees checkbox is selected.


Crews

Include crew leaders?
Select this checkbox to include crew
 leaders in the e-mail notification process. For crew
 leaders assigned to a job, the e-mail subject line will read: Crew assignment for <job start
 date. The automated message includes the job number, job site
 address, list of crew employees assigned to the job, and the job's start
 time. Additional text can be manually entered in the e-mail window as
 needed.

Assigned crew
Enter crew code to include in the
 e-mail notification process, or press Enter to accept the system
 default of ALL. Note: This field is available
 only if the Include crew leaders checkbox is selected.

Crew leader
Enter crew leader code to include in
 the e-mail notification process, or press Enter to accept the system
 default of ALL. Note: This field is available
 only if the Include crew leaders checkbox is selected.

Movers

Include equipment moves by employers?
Select this checkbox to include
 equipment moves to be performed by company employees in the e-mail notification
 process.
Include equipment moves by vendors?
Select this checkbox to include
 equipment moves to be performed by vendors in the e-mail notification
 process.
Mover
Enter the mover code to include in the
 e-mail notification process, or press Enter to accept the system
 default of ALL. Note: This field is available
 only if the Include equipment
 moves by employees checkbox and/or the Include equipment moves by
 vendor checkbox is selected.
