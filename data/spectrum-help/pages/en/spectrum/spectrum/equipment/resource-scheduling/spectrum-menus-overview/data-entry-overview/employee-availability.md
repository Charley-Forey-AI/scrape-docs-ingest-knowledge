# Employee Availability

This screen provides a convenient method for recording
 up-to-the-minute employee availability.
Assign the status codes from the [Schedule Status File Maintenance](/en/spectrum/spectrum/equipment/resource-scheduling/spectrum-menus-overview/maintenance-overview/schedule-status-file-maintenance)
 screen to employees. For example, you can enter current "Time Off" information when an
 employee calls in sick, or you can record vacation time or any other reasons that the
 employee is not available for assignment.
