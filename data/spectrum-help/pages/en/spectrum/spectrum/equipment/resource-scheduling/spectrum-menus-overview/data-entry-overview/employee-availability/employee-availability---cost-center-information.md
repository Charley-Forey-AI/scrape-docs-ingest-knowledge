# Employee Availability - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, then this screen will only display employees if you have security access
 for the cost center assigned to those employees.
You cost center scheme override setting for Employee Override is used to validate the
 cost centers.
