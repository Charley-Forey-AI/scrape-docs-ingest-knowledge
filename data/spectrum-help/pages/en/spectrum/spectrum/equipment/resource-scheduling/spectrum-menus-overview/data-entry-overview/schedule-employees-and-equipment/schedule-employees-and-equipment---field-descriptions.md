# Schedule Employees and Equipment - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Resource type
Select a resource type: Employees or Equipment.
Resource code
Enter the employee or equipment
 code.
Assign to job
Enter the company and job code the
 resource is being assigned to.
Start date
Enter the date the employee or piece of
 equipment is scheduled to begin the selected assignment, or press Enter to accept the default
 of the current Resource Scheduling processing date.  This
 field is used as a memo for the purposes of e-mailing and planning resource
 moves. It is not meant to determine whether a resource is available or
 scheduled for a specific time.
Note: Scheduling is
 tracked in full-day increments.

End date
Enter the date that the assignment is
 scheduled to finish.
Start time
Enter the time that the assignment is
 expected to begin or press Enter to accept the time that defaults from the Default start time field on
 Resource Scheduling Installation screen. This field is
 used as a memo for the purposes of e-mailing and planning resource moves. It
 is not meant to determine whether a resource is available or scheduled for a
 specific time.

Crew number
Enter the code associated with the job
 crew to include on the report, double-click on this field or press F4 to select
 from a list of available crew codes, use the SuperSelect options, or press
 Enter to accept the default of ALL.
Notes
Enter any notes that pertain to the
 resource or assignment. The text entered in this field displays on the Work
 Schedule Report.
