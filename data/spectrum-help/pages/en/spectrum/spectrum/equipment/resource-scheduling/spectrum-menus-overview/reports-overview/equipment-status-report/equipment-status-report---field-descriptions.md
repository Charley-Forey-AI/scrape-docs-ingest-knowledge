# Equipment Status Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Equipment
Enter the company code (if applicable)
 and code associated with the piece of equipment to include on the report, or
 press Enter to accept
 the default of ALL.If the Operator only has one
 authorized equipment company in Resource Scheduling, this company will
 default; otherwise, the current company will default.

Division
Enter the code associated with the
 equipment division to include on the report, or press Enter to accept the default
 of ALL.
Type
Enter the code associated with the job
 to include on the report, or press Enter to accept the default
 of ALL.
Status
Enter the code associated with the
 equipment status to include on the report, or press Enter to accept the default
 of ALL.
Include unassigned equipment?
Select this checkbox to included
 pieces of equipment that are currently not assigned to any jobs on the
 report.
Include assigned equipment?
Select this checkbox to included
 pieces of equipment that are currently assigned to any jobs on the
 report.
Job
Enter the company code (if applicable)
 and job number to include on the report, or press Enter to accept the default
 of ALL.
Effective date
Enter the effective date for this piece
 of equipment , or press Enter to accept the default of the current Resource Scheduling
 processing date. Equipment not assigned to a job as of
 the effective date will be assumed to reside at its home yard.

Sort by
Select whether to organize the report
 by Equipment, Current
 location, or Status.

- If you select the Equipment option, the
 report will sort by equipment company and then by equipment code.

- If you select the Location option, the
 report will sort by job company / job code, and then by equipment company
 / equipment code.

- If you select the Status option, the
 report will sort by status code and then by equipment company, equipment
 code and job company / job code.
