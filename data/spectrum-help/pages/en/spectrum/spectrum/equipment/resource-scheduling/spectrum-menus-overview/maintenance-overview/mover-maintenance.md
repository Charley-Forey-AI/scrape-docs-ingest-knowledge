# Mover Maintenance

Use this screen to create and manage a list of equipment
 movers for employees or vendors.

- Click the New button to begin the setup process.

- Select a Mover
 type (Employee or Vendor), and then enter an Employee or Vendor code, and click Continue.

- The status and contact info for the selected employee or vendor
 code will display in the Edit
 Mover window.

- Enter a Maximum
 load for this mover. A mover's maximum load is the greatest amount of
 weight the selected mover can safely and legally move. The maximum load is used during
 Schedule Equipment Moves processing in conjunction with the gross weight (stored in
 Equipment File Maintenance) to help determine whether the mover should be assigned to
 the mover. If this field is left blank, no weight-related warning messages display in
 Schedule Equipment Moves.Note: The gross weight for equipment is
 defined on the Equipment Control  >  Equipment File Maintenance  >  New/Edit Equipment - Details
 tab.

- Enter any Notes (up to 250 characters) regarding the mover (for example, No
 out-of-state work accepted).

- Click OK
 to save the mover.
