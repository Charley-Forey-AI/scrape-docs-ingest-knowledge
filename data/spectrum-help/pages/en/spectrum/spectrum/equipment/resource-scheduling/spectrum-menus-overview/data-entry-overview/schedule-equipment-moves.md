# Schedule Equipment Moves

It is common for pieces of equipment to be moved from job
 to job as various needs arise; use this screen to schedule unassigned moves to movers and to
 review and edit any scheduled moves.
The Schedule Employees and Equipment screen is protected by company-based security and only
 those operators with the required security settings can access this screen's
 information.
Click the New Move button to initiate a new move, or use the listboxes on the main screen to quickly schedule a move. The Unassigned moves listbox will display a complete list of equipment to be moved, and the Movers listbox will display a complete list of available movers already set up in Mover Maintenance. Click the Details button to display details related to assigned moves. From here you can edit and delete equipment move assignments. Click the Build button to open the [Build Unassigned Equipment Moves window](/en/spectrum/spectrum/equipment/resource-scheduling/spectrum-menus-overview/data-entry-overview/schedule-equipment-moves/build-unassigned-equipment-moves-window) window to create a new unassigned move record for the equipment's destination job.
An exception icon () displays in the Unassigned Moves listbox and the Assigned Moves listbox if one of the following criteria are met:

- If the move-to job does not match the job on the move assignment

- If the assignment date is earlier than the "move to" date

- If the move's gross weight exceeds the mover's maximum load amount
