# Schedule Status File Maintenance

Use this screen to establish a list of status codes that
 can be designated as Available or Unavailable.
These status codes can then be assigned to employees in the Employee Availability screen
 and used for selection criteria in the scheduling screens.
Field
Description

Status
Enter a scheduling status code, for
 example, FIELD or VACATION.
Description
Enter a description for the status
 code, for example, EMPLOYEE SICK DAY or AT JOBSITE - UNAVAILABLE
Type
Select whether this scheduling status
 is Available or
 Unavailable.
