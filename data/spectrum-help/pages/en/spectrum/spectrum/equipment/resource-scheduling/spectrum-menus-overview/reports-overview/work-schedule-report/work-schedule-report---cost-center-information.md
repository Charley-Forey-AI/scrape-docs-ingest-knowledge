# Work Schedule Report - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, the job will not display unless the operator has security to the job's
 cost center (no overrides apply).
Only Job cost center security is used for this report. Employee and equipment cost
 center will not be checked here.
