# Equipment Control

The Equipment Control module tracks all costs and revenue
 for each piece of equipment.
Usage and maintenance are entered during payroll timecard entry, or
 directly in the equipment control system. This module is integrated with the Accounts
 Payable, Job Cost, Payroll, Accounts Receivable, Work Order, Preventive Maintenance, Time +
 Material Billing, Fixed Assets, Purchase Order, Equipment Tracking and General Ledger
 modules. In addition, Equipment Control is integrated with Document Imaging to scan and
 retrieve equipment documents. Hover over the image below to view the Equipment Control
 integration flowchart.
To set up Equipment Control, follow these steps:

1. Configure default settings ([Equipment Control Installation](#ID-000001af--en__ID-000000bf)).

1. Set up equipment, equipment type codes, status codes, cost
 category codes, fuel and oil codes, yard codes, and equipment charge rates ([Equipment Control Maintenance](#ID-000001af--en__ID-000000c5)).

1.  Enter cost transactions, revenue transactions recurring
 transactions for equipment, meter readings, and ownership costs ([Equipment Control Data Entry](#ID-000001af--en__ID-000000e3)).

Refer to the following sections for details.

## Equipment Control Installation

Go to Site Map > System Administration > Installation > Equipment Control to set up the Equipment Control module. Options include default
 properties, default G/L codes, cost category types, and user-defined fields.

- Complete the [Equipment Control Installation Overview](/en/spectrum/spectrum/equipment/equipment-control/equipment-control-installation-overview) screens.

## Equipment Control Maintenance

Use the Maintenance screens to set up your equipment (type codes,
 status codes, cost category codes, fuel and oil codes, locations, yard codes, equipment
 charge rates). Some of these screens have been pre-configured in the default company,
 but you can add additional items for your company's equipment as needed. Print and
 retain the associated listing reports when you have completed entry in these
 screens.

- Use the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen to set up equipment information such as serial
 numbers, license numbers, weight and default settings.

- Enter equipment status file codes for equipment tracking
 purposes using the [Equipment Status File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-status-file-maintenance) screen.

- Group equipment types together using the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen.

- Set up codes that group equipment cost transactions together
 using the [Cost Category File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/cost-category-file-maintenance) screen.

- Enter the yard code in which the equipment is currently located
 using the [Yard File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/yard-file-maintenance) screen.

- Enter fuel and oil codes and current purchase price using the
 [Fuel/Oil File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/fueloil-file-maintenance) screen. The information will be used when doing fuel
 entries or transaction entries in Transaction Entry.

- Enter the physical location of equipment using the [Equipment Area File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-area-file-maintenance) screen.

- Enter job or rental rates for a specific piece of equipment on
 the [Equipment Charge Rates](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-charge-rates) screen.

- You can also copy existing equipment information to create a
 new equipment record using the [Quick Equipment Setup Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/quick-equipment-setup-maintenance) screen.

## Equipment Control Data Entry

Use the Data Entry screens to enter equipment cost transactions,
 revenue and recurring transactions, and ownership costs.

- Enter cost transactions for equipment cost, AP vendor cost and
 Payroll cost transaction types in the [Cost Transaction Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/cost-transaction-entry)
 screen.

- Enter revenue transactions for EU (usage and cost) and ES
 (standby) transaction types using the [Revenue Transaction Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/data-entry-overview/revenue-transaction-entry)
 screen.

- Enter recurring transactions for EU and ES transaction types
 using the [Recurring Transaction Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/data-entry-overview/recurring-transaction-entry) screen.

- Enter equipment ownership costs on the [Ownership Cost Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/ownership-cost-entry)
 screen.

- Use the [Import Equipment Costs from Text File](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/cost-transaction-entry/import-equipment-costs-from-text-file)
 screen to create unposted equipment transactions from a comma-delimited text file.


- Calculate depreciation, license and insurance costs on the
 [Build Ownership Cost Transactions](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/data-entry-overview/transaction-report--update/build-ownership-cost-transactions) screen. The [Ownership Cost Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/ownership-cost-entry) screen
 is used to change or modify any information computed by the system using the Build
 Ownership Cost Transactions screen.

- Use the [Equipment Meters](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-meters) screen to set up four separate meter readings for a
 piece of equipment.

- Enter meter readings on the [Meter Reading Transaction Entry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry) screen.
