# Edit Hilti Transaction

Use this window to resolve any errors appearing in the Approve
 Hilti Requisitions screen.
Records with errors are invalid and are indicated by a warning icon in the Validation Status column.
The system ignores invalid records/rows when creating transactions, so if a row that has a warning is one you want to use to create a transaction, you'll need to resolve all invalid field values. To do this, select the row you want to edit and click Edit.
