# Deployment Log Report - Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Fields
Descriptions

Equipment
This field defaults to ALL or you can use the available
 drop-down arrow to select a different equipment code.

Equipment type
This field defaults to ALL or you can use the available
 drop-down arrow to select a different equipment type.

Equipment division
This field defaults to ALL or you can use the available
 drop-down arrow to select a different equipment division (if the 'Validate
 equipment division with G/L department?' option is selected in Equipment
 Control Installation).

Equipment status
This field defaults to ALL or you can use the available
 drop-down arrow to select a different equipment status code.

From/to date
Enter the date range that you want to use for this
 report.

Status types

Include active equipment?
Include inactive equipment?
Include retired equipment?
Use these options to further refine the report results by
 equipment status.
By default, only Active equipment records will be
 included.
