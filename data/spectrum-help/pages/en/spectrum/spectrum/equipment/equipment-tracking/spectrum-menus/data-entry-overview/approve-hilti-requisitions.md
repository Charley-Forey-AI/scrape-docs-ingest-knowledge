# Approve Hilti Requisitions

Equipment tracking data from Hilti ON!Track flows into the Approve Hilti Requisitions screen.
When you open this screen, Spectrum validates the asset tracking entries coming
 from the Hilti ON!Track app for the tools you've set up
 in Spectrum, and
 marks each with a validation status of either Valid or Error.

- Edit: opens the [Edit Hilti Transaction](/en/spectrum/spectrum/equipment/equipment-tracking/spectrum-menus/data-entry-overview/approve-hilti-requisitions/edit-hilti-transaction) window for the selected
 row, where you can update equipment data. If an
 entry has an error, you can edit the field value(s)
 to resolve the error.

- Delete: deletes the selected
 record.

- Refresh: refreshes the screen to
 reflect any changes made to entries.

- Create
 Transactions: processes all eligible
 records in chronological order and, if the records
 for a given piece of Hilti equipment are error-free,
 creates a new Equipment Revenue Transaction entry.
 The equipment usage cost is then charged to the job
 using the charge rates set up previously for the
 equipment.

For more information, see [Review Hilti Requisitions and Charge Equipment Use to a Job](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/review-hilti-requisitions-and-charge-equipment-use-to-a-job).
