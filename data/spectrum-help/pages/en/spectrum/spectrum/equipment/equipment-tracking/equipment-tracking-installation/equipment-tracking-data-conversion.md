# Equipment Tracking Data Conversion

 View the summary of the steps needed to complete the
 Equipment Tracking Data Conversion installation.
Note: It is important that these steps be completed in the order listed.

1. Complete the Equipment Tracking Installation screen. Install Inventory
 Control and/or Equipment Control, if not already installed.

1. Set the processing date to the conversion date using Processing Date
 Change.

1. Set the processing dates to the new period for Equipment Tracking using
 Processing Date Change.

## For Companies using Equipment Control:

1. Enter the equipment types, if necessary, using Equipment Type File
 Maintenance. These codes are normally set up during Equipment
 Control conversion.

1. Print the Equipment Type File Listing and check for accuracy.

1. Enter fuel/oil codes, if necessary, using Fuel /Oil File Maintenance.
 These codes are normally set up during Equipment Control conversion.

1. Print Fuel/Oil Code Listing and check for accuracy.

1. Enter equipment file information, if necessary, using Equipment.
 These codes are normally set up during Equipment Control conversion.

1. Print the Equipment File Listing and check for accuracy.

## For Companies using Inventory Control:

1. Enter warehouses, if necessary, using Warehouse File Maintenance.
 These codes are normally set up during Inventory Control conversion.

1. Print the Warehouse File Listing and check for accuracy.

1. Enter categories, if necessary, using Category File Maintenance.
 These codes are normally set up during Inventory Control conversion.

1. Print the Category File Listing and check for accuracy.

1. If Inventory is present, enter G/L departments using G/L Department File
 Maintenance. These codes are normally set up during Inventory
 Control conversion.

1. Enter item file information, if necessary, using Inventory Item File
 Maintenance. These codes are normally set up during Inventory
 Control conversion.

1. Print Inventory Item File Listing and check for accuracy.

1. Enter transaction codes using Transaction Code
 Maintenance.

1. Print Transaction Code Listing and review for accuracy.

1. Enter transactions for recurring charges for equipment already assigned to
 jobs at the conversion date using Equipment Tracking
 Requisition.

1. Print and update the transaction report.

1. Delete the unposted Equipment Tracking transactions created above, if these
 charges have already been applied to jobs prior to the conversion date.
