# Recurring Equipment Charge Processing - Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Field
Description

Through last billing date
Enter the last billing date through which to print the
 report, or press Enter to accept the current Equipment Tracking processing
 date.

Transaction date
Enter the transaction date through which to print, or
 press Enter to
 accept the current Equipment Tracking processing date. The date in this
 field must be within the Equipment Tracking minimum and maximum processing
 date range. If you enter a date that is outside of the minimum/maximum date
 parameters, the following error displays: Date entered is not in range of the
 processing dates.

Batch code
Enter the batch code, or press Enter to accept the system
 default code of the operator code.
