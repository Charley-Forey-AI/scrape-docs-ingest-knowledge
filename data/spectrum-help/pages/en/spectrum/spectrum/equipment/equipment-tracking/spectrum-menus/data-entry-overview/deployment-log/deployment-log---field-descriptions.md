# Deployment Log - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field/Buttons
Descriptions

Selections
Click this button to filter the records that display in the grid. For example, you might filter by equipment code, or by equipment currently requisitioned to both the job and site.

Issue
Click this button to open the [Equipment Tracking Requisition](/en/spectrum/spectrum/equipment/equipment-tracking/spectrum-menus/data-entry-overview/equipment-tracking-requisition)
 screen to initiate the process of issuing one or more equipment codes to a
 job. Any rows that are selected in the Deployment Log
 screen will be added to this new requisition.

Return
Click this button to open the Return Issued Equipment window to return one or more equipment codes from a job. The update will generate return requisitions for each selected job (and will contain all returning equipment codes for that particular job), and will also update the current location.
The billing rate assigned to the equipment (and any attachments) requisition
 will default from the stand-by rates defined on the Time + Material > Job Equipment Billing Rate Maintenance screen, based on Hourly, Daily, Weekly or Monthly rates. If
 there is no rate defined there, Spectrum will look to the Equipment Billing
 Rate Maintenance table for the stand-by billing rate. If no rate is defined
 there, the billing rate is zero.

Update Location
Click this button to open the Update Current Equipment Location window to record a new current location for one or more equipment codes from a yard, job, or site without changing the requisition assignment or availability.

Refresh
Click this button to rebuild the grid list.

Group by
Use the available drop-down list to organize the grid columns or drag one or more column captions onto this bar to group records in the grid. Once this checkbox is selected, the grid details will update accordingly.

Columns

Equipment
The equipment code displays in this column

Current location
This is the yard, job, or site code and description, and company if the yard, job, or site is in a company other than the current company.

Issued to
The job or site and description entered in Equipment Tracking Requisition Entry.

Phase
The phase and description entered in Equipment Tracking Requisition Entry.

Ct
The cost type and description entered in Equipment Tracking Requisition Entry.

Requisition
The requisition number, if currently issued.

Date/time
The date/time entered in Equipment Tracking Requisition Entry. This column is blank if the equipment is available.

Contact name
The Contact entered in Equipment Tracking Requisition Entry. This column is blank if the equipment is available.

Transaction code
The transaction code and description entered in Equipment Tracking Requisition Entry. This column is blank if the equipment is available.

Message
Displays the text originally entered in Equipment Tracking Requisition Entry. This column is blank if the equipment is available.

Division
The division code assigned to the equipment code.

Equipment type
The type assigned to the equipment code.

Year, Make, Model, Serial/VIN, Equipment status, Status type, and
 Count
Display from Equipment.

Count
This is the sum of the number of records grouped, if applicable.
