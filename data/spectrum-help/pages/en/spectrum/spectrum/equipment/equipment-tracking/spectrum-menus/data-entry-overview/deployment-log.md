# Deployment Log

Use the Deployment Log screen to manage equipment assignments. This screen will list equipment and the job where it is currently issued, plus available (un-requisitioned) equipment. The grid also shows current location and other details.
Key features include:

- Select the columns you want to display in the grid. Click the tree grid icon  and select the items you want to display in the grid. Click OK to save your changes.

- Group records by selected columns.

- Drag a row to associate it with another record.

- Print or export to Excel. Click the  icon and select the columns you want to export. Then click OK. A window displays prompting you to select the spreadsheet software you want to use. Select the "Do this automatically for files like this from now on" checkbox if you want to avoid this prompt in the future.

- Issue, return, or redeploy a group of equipment codes to another job directly from the grid.

- Manage the current equipment location.

Export grid data
Several Spectrum report starting screens off the ability to export files. Users need only to enter the file NAME in the 'Export file' field. The file path is not necessary because the file save location depends on the browser settings.
