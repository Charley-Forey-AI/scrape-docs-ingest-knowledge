# Requisition History Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Requisition
This field defaults to ALL, or you can use the available drop-down arrow to select a different requisition.
If this report is accessed from the Equipment Tracking
 Requisition screen, the requisition # will default from that
 screen.

Job
This field defaults to ALL or you can use the available drop-down arrow to select a different job.

Site
This field defaults to ALL or you can use the available drop-down arrow to select a different site

Equipment
This field defaults to ALL or you can use the available drop-down arrow to select a different equipment code.

Equipment type
This field defaults to ALL or you can use the available drop-down arrow to select a different equipment type.

From/to date
Enter the date range that you want to use for this report.

Page break between requisitions?
Select this checkbox to add a page break between each requisition.

Requisition types

Issue?
Return?
Use these options to further refine the report results by requisition status type. By default, only Issued requisitions will be included.
