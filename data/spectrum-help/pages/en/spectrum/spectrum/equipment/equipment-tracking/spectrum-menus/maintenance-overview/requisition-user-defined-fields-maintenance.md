# Requisition User-Defined Fields Maintenance

Use this screen to set up virtually unlimited requisition-specific user-defined fields. In addition, this screen is used to view, edit, and delete Accounts Receivable customer user-defined fields.
The fields in this screen default based on the settings on the System Administration > Installation > Accounts Receivable > User-Defined Fields tab.
