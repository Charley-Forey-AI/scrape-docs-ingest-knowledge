# Transfer Tools Using Hilti ON!Track

Use Hilti ON!Track to
 transfer tools and equipment to different jobs and locations.
Before you start
 transferring tools using Hilti ON!Track, make sure you have entered the tool's
 Equipment Code from
 Spectrum in the Hilti ON!Track asset Inventory Number field. For more
 information about linking equipment in both applications, see [Enter Tools in Spectrum to be Tracked by Hilti](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/enter-tools-in-spectrum-to-be-tracked-by-hilti).

1. Log in to Hilti ON!Track.

1. Choose to transfer equipment assets to a location,
 storage asset, or employee.For information about Hilti ON!Track and transferring equipment assets, reference
 the [Hilti ON!Track Help](https://www.hilti.com/kb-search?continue=true).

1. Assign or change the employee responsible for each asset.

1. Set transfer and return dates for each
 asset.

See the following video for more information about
 transferring and tracking assets using Hilti ON!Track.

Return to Spectrum to review the Hilti equipment
 requisitions. See [Review Hilti Requisitions and Charge Equipment Use to a Job](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/review-hilti-requisitions-and-charge-equipment-use-to-a-job).
