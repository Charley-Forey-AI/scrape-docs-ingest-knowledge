# Enter Tools in Spectrum to be Tracked by Hilti

Add new tools and equipment in Spectrum, then link them to an asset in
 Hilti ON!Track.
Prior to linking assets
 in Hilti, make sure you have properly enabled the integration and created a
 Hilti-specific equipment type code in Spectrum. For details, see [Set Up the Hilti ON!Track Integration in Spectrum](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/set-up-the-hilti-ontrack-integration-in-spectrum).Important: In general, it is
 best to create new equipment records when entering tools you want to manage
 using Hilti ON!Track. However, in some
 cases, you may want to use existing equipment being tracked by Spectrum and move these to Hilti
 tracking. To do this, see [Switch Existing Equipment to be Tracked by Hilti](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/enter-tools-in-spectrum-to-be-tracked-by-hilti/switch-existing-equipment-to-be-tracked-by-hilti) for requirements and guidance.

1. Add new tools and equipment in Spectrum. You can do this either
 through Quick Equipment
 Setup or the standard equipment entry process.

1. On the Equipment Main Properties screen,
 select New.

1. In the New Equipment window, fill out all
 fields. Select the Quick Equipment Setup checkbox if you'd like to
 create the record that way. For Type, enter the
 Hilti-specific Equipment Type code you created in the integration
 setup.The minimum required fields for tracking this
 asset in Hilti include: Equipment code,
 Description, Year, Make, Model, Type, Division, and
 Status.Note: To later be able to create a transaction,
 the equipment Status cannot be set to Inactive or
 Retired.

1. Select OK.

1. Optionally, you can fill out additional
 fields in the Equipment Main Properties screen for the new tool you just
 added.

1. Open the Charge Rates screen and enter the
 amounts you want to charge to the job for use of the tool.These charge rates will be used later in
 calculating job costs.

1. To add the new equipment record, select
 Save.


Note: Equipment Attachments supported in Spectrum are not supported in Hilti ON!Track. Each piece of equipment has its usage tracked independently.

For more information on adding equipment in Spectrum, see [New Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/quick-equipment-setup-maintenance/new-equipment).

1. In Hilti ON!Track, add the Equipment Code from Spectrum to the Hilti
 asset to link the tool / equipment in both applications.

1. Note or copy the Equipment Code  from
 the tool you entered in Spectrum.

1. In Hilti ON!Track, go to Edit Asset.

1. Under Enter Asset Details, in the Inventory
 Number field, enter the Spectrum Equipment
 Code.This equipment in Spectrum is now linked to the associated asset
 in Hilti.

After linking both applications, tool and equipment
 data from Spectrum will flow into
 Hilti ON!Track, where it will be
 tracked.
After entering your
 new tool and equipment records and linking them in Hilti, you can [Transfer Tools Using Hilti ON!Track](/en/spectrum/spectrum/equipment/hilti-ontrack-integration-with-spectrum/transfer-tools-using-hilti-ontrack).
