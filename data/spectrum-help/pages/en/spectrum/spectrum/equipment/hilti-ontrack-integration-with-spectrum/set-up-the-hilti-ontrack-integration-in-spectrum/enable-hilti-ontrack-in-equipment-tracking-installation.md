# Enable Hilti ON!Track in Equipment Tracking Installation

Enabling this integration allows you to manage tools and
 equipment in Spectrum using Hilti ON!Track.
You can still manage
 equipment using Spectrum's standard
 equipment tracking capabilities. You can even leverage the equipment tracking
 functionalities of both Hilti ON!Track and Spectrum, provided that equipment is not
 duplicated on both records (this will result in errors).

1. In Spectrum, go to System
 Administration > Installation > Equipment Tracking
 Installation.

1. On the Properties tab, under Integrations, select the Enable Hilti
 OnTrack equipment/tool management? checkbox. For more information about this checkbox, see [Equipment Tracking Installation - Properties](/en/spectrum/spectrum/equipment/equipment-tracking/equipment-tracking-installation/equipment-tracking-installation---properties).

1. On the Defaults tab, fill in the Hilti integration
 default fields. Note: These fields only display
 after you enable the integration in the previous step.The information you enter in these fields is
 required for properly identifying and managing
 tools.

1. Enter a Phase. Enter this one
 phase to track all of your Hilti tools across all of your jobs.

1. Choose a Cost type you will
 use to track your tools.

1. Select a Transaction code that
 issues the tool to a job.

1. Select a Default billing
 period.

1. Select Save.
