# Renew Equipment License - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Create Renewal

Document #, License #, Permit #
The field name that displays here is dependent on the compliance code type: Permit, License, Inspection, Violation, or Other. Enter the appropriate number (document number, permit number, etc.) here.

Registration #, Issuer name
The field name that displays here is dependent on the compliance code type: Permit, License, Inspection, Violation, or Other. Enter the registration number (for permits and licenses) or issuer name (for inspections and violations).

Issue date
Enter the date that this equipment compliance record was issued, or use the available drop-down list to select a date.

> Shortcut Key
Date Automatically Entered

T
Today's date.

H
The last day of the current month.

M
The first day of the current month.

+
Tomorrow's date.

-
Yesterday's date.

Expiration date
This field defaults to the standard number of days after the issue date, or you can manually enter the date that this equipment compliance record is scheduled to expire.

Follow-up date
If this equipment compliance record will require follow-up at a later date, enter that date here.

Amount
Enter the amount paid for this inspection, license, permit, violation, and more (for example, if you paid $1200.00 for this permit, type 1200 and press Enter).

Comments
Enter any additional comments that might be helpful to others when reviewing this equipment compliance record.
