# Equipment Utilization Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Equipment code
Enter the equipment code of the item you want to run the report for, or use the Search Equipment window to select an equipment code. If your operator has security authorization for the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen, a Maintenance button that allows you to create equipment codes on the fly will be available from the Search Equipment window.

Equipment type
Enter the equipment type code of the item you want to run the report for, or use the Search Equipment Types window to select an equipment type. If your operator has security authorization for the[Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen, a Maintenance button that allows you to create equipment types on the fly will be available from the Search Equipment Types window.

Equipment status
 Enter the status of the equipment you want to run the report for, or use the Search Equipment Status Codes window to select an equipment type.

From/to fiscal year
Enter the fiscal date range of records to be included on the report.

Meter types
Indicate the types of equipment meter for which the report should be printed, or press Enter to print all meter types. Equipment without a specified meter type is included automatically.

Status types
Select which equipment status types to include on the report.

- If the Equipment Tracking module is present, then the Include active equipment and Include inactive equipment checkboxes are selected by default.

-  If the module is not present, then only the Include active
 equipment checkbox is selected by default.

- Any combination of options is permitted.
