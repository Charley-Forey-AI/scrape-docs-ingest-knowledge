# Weight Class File Maintenance

Use this screen to create and maintain a list of standard
 weight classifications.
Weight class codes allow equipment managers to compare equipment codes within the same
 weight class (even when looking at different makes or models), and the codes may be used as
 selection criteria on various inquiry and report screens.

- Click New to display the New Weight Class window and add new weight classes.

- Select a record and click Edit to open the Edit Weight Class window and make changes to the specified record.
