# Equipment Attributes

Use this screen to create and maintain a list of standard
 attributes.
An attribute can be thought of as an option that a piece of equipment
 possesses. For example, a dozer (equipment) might have several different attributes: a
 removable blade, a GPS unit, a scoop, and more. Standard attributes should be pre-defined by the
 equipment manager in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen.
When a new piece of equipment is added, standard attributes are added
 automatically when the equipment is saved based on the equipment type assigned to the new
 code. If the Assign attribute codes for this
 equipment type checkbox is selected in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) - Properties tab
 for the assigned type, then any additional attributes for that type that are marked to be
 added automatically in the Standard Attributes tab will be added too.

- Click New
 to display the New Equipment Attributes window and add new attributes.

- Click Build to open the [Build Attributes for Equipment (or Type)](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-attributes/build-attributes-for-equipment-or-type) window.

- Select a record and click Edit to open the Edit Equipment
 Attributes window and make changes to the specified record.
