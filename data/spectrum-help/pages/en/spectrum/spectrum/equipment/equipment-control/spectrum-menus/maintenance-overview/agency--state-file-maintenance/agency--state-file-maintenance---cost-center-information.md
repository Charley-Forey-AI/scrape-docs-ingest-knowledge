# Agency / State File Maintenance - Cost Center Information

If cost centers are being used in the current company, the
 Agency / State File Maintenance screen will validate whether the operator has security
 authorization to enter a disallowed vendor code but disallowed vendors will appear in this
 screen if already assigned.
To determine whether the operator may add a vendor code, the software compares cost
 centers assigned to the vendor with cost centers in the operator's assigned cost center
 scheme, and if there are no matches, then that vendor code will not be allowed.
