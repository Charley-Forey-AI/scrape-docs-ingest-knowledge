# Ownership Cost Entry

This screen allows adding equipment codes directly into the
 screen.
The Ownership Costs screen is used to change or modify any information
 computed by the system using [Build Ownership Cost Transactions](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/data-entry-overview/transaction-report--update/build-ownership-cost-transactions). When modifying or adding equipment codes, the system will check
 that entered G/L codes are valid. This feature is especially useful for recording
 transactions for brand-new equipment not properly set up in advance of the Build Ownership
 Cost Transactions update.
If depreciation for a new piece of equipment did not figure during Build
 Ownership Cost Transactions, it is possible that the depreciation amount per year or the
 purchase cost has not been recorded in the Equipment screen.
