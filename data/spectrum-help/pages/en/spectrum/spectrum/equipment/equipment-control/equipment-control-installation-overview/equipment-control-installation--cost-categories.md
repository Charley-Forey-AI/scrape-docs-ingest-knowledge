# Equipment Control Installation – Cost Categories

Use this tab to set default cost category information in
 the Equipment Control module.
Fields
Descriptions

Cost category type 4-7
 description:

- The system is prompting for the four user-defined
 cost category types. Cost category types 1, 2, and 3 are reserved by
 Spectrum for FUEL, LUBE &
 OIL, and TIRES, respectively.

- The remaining four cost category types can be
 specified based on your particular operations. Examples of possible cost
 category types might include "labor", "labor burden", "parts &
 materials", "repairs & maintenance", "outside repairs",
 "maintenance", and "other costs". "Labor" and "labor burden" may be
 combined under one type or may be separated, as desired. Likewise,
 "maintenance" may be combined with or tracked independent of "repairs",
 and "in-house repairs" may be isolated from "outside repairs" in some
 operations. If you choose to use custom labels, these need to be added to
 a custom lexicon in System
 Administration >  Security >  Custom
 Labels.

- These seven cost category types are used
 specifically on the PM Work Order Cost Analysis
 screen and Equipment Cost Report, which prints the
 types 1-7 horizontally across the page.

- "Depreciation", "license" and "insurance" costs are
 tracked separately from these seven cost category types; therefore, it is
 unnecessary to reserve any of these user-defined cost category types for
 this purpose.

Um
Following each cost category type description, the system
 prompts for the unit of measure (u/m) for each type. Examples of possible
 units of measure include "hours" and "each". This field may be left blank if
 a unit of measure is non-applicable, such as with "other costs". Entry in
 this field will be used on certain screens and reports.
