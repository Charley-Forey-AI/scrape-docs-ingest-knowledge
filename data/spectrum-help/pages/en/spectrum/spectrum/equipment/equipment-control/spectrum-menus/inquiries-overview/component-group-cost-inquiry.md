# Component Group Cost Inquiry

Use this screen to review costs by component groups.
The columns in this screen display the sum of costs by equipment code
 for the associated cost categories for month-to-date, quarter-to-date, year-to-date and
 life-to date. Drill down to view actual cost detail for associated cost categories,
 scheduled P.M. for associated components, and component group information stored in the
 equipment work order detail table.
