# Tire File Maintenance

Use this screen to setup and maintain a list of standard
 tires for use in Equipment.
Tire information may also be used for selection purposes on the Preventive Maintenance
 Inquiry and on some reports.
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.

- Click New to display the New Tire window and add new tires. See field descriptions below.

- Select a record and click Edit to open the Edit Tire window and make changes to the specified record. See field descriptions below.
