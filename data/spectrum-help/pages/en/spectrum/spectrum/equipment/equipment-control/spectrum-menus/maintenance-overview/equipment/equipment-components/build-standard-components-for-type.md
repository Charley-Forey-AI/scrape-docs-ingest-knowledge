# Build Standard Components for Type

Use this window to assign components codes to the specified
 equipment code or equipment type.
The software will determine whether to show all codes or the
 type-specific list based on the Assign
 component codes for this equipment type checkbox setting in the Equipment
 Type File Maintenance screen.

- When a list of all the codes available in this company
 displays, the column header in the left pane will say Component.

- When a type-specific list displays, the column header in the
 right pane will say Standard
 component.

How to use this screen:

1. Select the component you want to apply to the current equipment from the left pane and click the Select button to move the component to the pane on the right . Multiple components may be selected at once by holding down the Ctrl key while clicking on your choices, or you can select all of the component codes at once by clicking the Select All button. By default, all available components will be displayed on the left if the source is [Component File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/component-file-maintenance).

1. The components (if any) that appear in the right pane upon opening this window are the components previously-assigned to the specified equipment type. To remove a component prior to performing the update, select the component from the right pane and click Deselect. Likewise, you can disassociate all of the current components for this equipment by clicking the Deselect All button.

1. Once the list of components that you want to add to this equipment is finalized in the right pane, select the Yes, proceed with update option to run the update. Associated user-defined fields are also stored for the equipment when the update is run.
