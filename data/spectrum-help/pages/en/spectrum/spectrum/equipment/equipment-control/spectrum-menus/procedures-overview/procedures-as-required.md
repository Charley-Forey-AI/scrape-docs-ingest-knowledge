# Procedures As Required

Below are procedures for equipment control.

- Establish an equipment master record as needed for each new piece of equipment using Equipment.

- Add equipment types as appropriate using Equipment Type File Maintenance.

- Enter cost categories as appropriate using Cost Category File Maintenance.

- As appropriate, enter information about new fuel and oil, or record changes to fuel/oil rates using Fuel / Oil File Maintenance.

- If specific jobs have rates other than the customary charges, enter job-specific equipment rates, as appropriate, using Job-Specific Equipment Charge Rates.

- Enter additional status codes, as appropriate, using Equipment Status File Maintenance.

- Enter additional Equipment Areas, as appropriate, using Equipment Area File Maintenance.

- Enter additional yard location codes, as appropriate, using Yard File Maintenance.

- Use the equipment inquiry screens to review information about designated equipment or jobs.
