# User-Defined Fields Purge

Use this window to purge selected user-defined
 fields.
 Click the column header to arrange the data by ascending order; click the column header
 again to arrange the text in descending order. Use your mouse to determine the width of
 this column by sliding the column title's side borders.
Fields
Descriptions

Comp
The code for the company that uses this field displays. No entry is allowed.

Type
The user-defined field type (Alpha, Numeric, or Date) displays. No entry is allowed.

Prompt
The user-defined field's title (for example, Manufacturer) displays. No entry is allowed.

Purge
Click this button to purge the selected user-defined field.
