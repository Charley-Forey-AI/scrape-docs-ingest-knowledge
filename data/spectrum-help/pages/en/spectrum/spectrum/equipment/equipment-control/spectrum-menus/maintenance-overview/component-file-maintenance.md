# Component File Maintenance

Use this screen to create and maintain a list of equipment
 component types and user-defined fields.
For use in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance), [Equipment Components](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-components), and [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screens. These components are
 also useful for selection purposes on inquiries and reports.
Click the Group Codes button to open the [Component Group File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/component-group-file-maintenance) screen where you can classify components and associate cost categories with groups of components.
