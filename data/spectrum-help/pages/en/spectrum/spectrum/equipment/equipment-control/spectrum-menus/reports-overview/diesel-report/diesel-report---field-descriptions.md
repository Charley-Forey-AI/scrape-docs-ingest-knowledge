# Diesel Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Equipment code
Enter the equipment code of the item you want to run the report for, or use the Search Equipment window (F4) to select an equipment code. If your operator has security authorization for the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen, a Maintenance button that allows you to create equipment codes on the fly will be available from the Search Equipment window.

Equipment type
Use this field to help define your report results. Enter the equipment type code of the item you want to run the report for, or use the Search Equipment Types window (F4) to select an equipment type. If your operator has security authorization for the[Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen, a Maintenance button that allows you to create equipment types on the fly will be available from the Search Equipment Types window.

Equipment status
Use this field to help define your report results. Enter the status of the equipment you want to run the report for, or use the Search Equipment Status Codes window (F4) to select an equipment type.

From/to transaction date
Use the From and To transaction date fields to specify the date range of the equipment usage that you want to include on this report.

Include equipment with no activity?
Clear the checkbox if you want to only include equipment with activity.

Sort by
Select Equipment code to sort only by equipment code, regardless of type. Select Equipment type to sort this report by equipment type, followed by equipment code within each type.

Operating basis
By default, the operating basis is set to Equipment usage hours for this report unless a report filter is being used. If you select the Meter history option instead, then the Meter types section will be enabled to allow you to select the meter readings you wish to include on the report.

Meter types
These option boxes only display if the Operating basis is set to Meter history. Select the meter types that you want to include on the report. By default, all meters are selected unless a report filter is being used.

Road usage
Select to include only on-road, only off-road, or all types of equipment.

Fuel type
Select to include only diesel, only gasoline, or both types of equipment.

Status types
Select which equipment status types to include on the report.

- If the Equipment Tracking module is present, then the Include active equipment and Include inactive equipment checkboxes are selected by default.

- If the module is not present, then only the Include active
 equipment checkbox is selected by default.

- Any combination of the options is permitted.
