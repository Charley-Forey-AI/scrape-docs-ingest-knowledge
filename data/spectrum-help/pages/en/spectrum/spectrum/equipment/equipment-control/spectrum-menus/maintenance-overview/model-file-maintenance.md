# Model File Maintenance

Use this screen to set up and maintain equipment models,
 component models, and tire models while associating the codes with the appropriate equipment
 types.
Once this list is established, the make codes are also available for
 selection in the[Equipment Main Properties](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-main-properties) tab,
 for convenience when entering new equipment.
If you only have lookup only security permission for this screen,
 the Edit, New and Delete buttons will be disabled.

- Click New to display the New Model window and add new records.

- Select a record and click Edit to open the Edit Model window
 and make changes to the specified record.
