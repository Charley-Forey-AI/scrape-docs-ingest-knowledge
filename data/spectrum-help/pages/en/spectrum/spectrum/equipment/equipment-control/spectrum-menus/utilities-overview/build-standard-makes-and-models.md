# Build Standard Makes and Models

Use this screen to automatically create records in the Make
 File Maintenance and Model File Maintenance screens from equipment, components, tires,
 engines, transmissions, and power axles that have already been set up in the Equipment and
 Component File Maintenance screens.
 In the case where the make is already recorded, but the model field is blank, this update will add a record to the standard list of makes but no new models will be added. Likewise, in the case where the model is already recorded but the make field is blank(For example, when the model information was added in Equipment, but the make information was left blank.), then a new model is not added to the standard list of models.
Click Continue to create records in Make File Maintenance and Model File Maintenance. Otherwise, select Cancel to exit the update screen without copying any information.
For more information, see [Make File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/make-file-maintenance), [Model File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/model-file-maintenance), [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) and [Component File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/component-file-maintenance).
