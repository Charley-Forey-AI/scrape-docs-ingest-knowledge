# Daily Procedures

Below are daily procedures for equipment
 control.

- Enter equipment control transactions for costs using Transaction Entry.

- Enter equipment control transactions for equipment usage costs using Transaction Entry.

- Print and review the Transaction Report. Update if approved using E/C Transaction Update.

- Print and review the G/L Detail Report. Update if approved using E/C G/L Detail Update.

- Record meter readings using Enter Equipment Meter Reading.

- MAKE A DAILY BACKUP
