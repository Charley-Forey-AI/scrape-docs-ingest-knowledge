# Ownership Cost Entry - Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Fields / Buttons
Descriptions

Fiscal year
The two-digit year for this entry displays from the
 latest setting in Build Ownership Cost Transactions. Press Enter to accept
 this default or enter a different year.

Period
The two-digit period for this entry displays from the
 latest entry in Build Ownership Cost Transactions. Press Enter to accept
 this default or enter a different period.

New/Edit/Delete buttons
User these action buttons to add records, make changes to
 existing records, or delete records.

Equipment code
The equipment codes display. Enter or arrow to move the
 cursor to the desired code, then TAB, or enter the desired equipment code
 directly into the field.

Description
The equipment descriptions display.

Transaction type
Select the transaction type from the drop-down menu:
 Depreciation, License, or Insurance.

Amount
Press Enter to accept the existing amount, or enter the
 desired transaction amount.

G/L debit
Press Enter to accept the existing G/L account code, or
 enter the desired G/L debit account for this transaction. The debit account
 description will display. Select the Equipment cost option in the G/L Master
 File Maintenance Direct cost menu in order to display this information.
Note: Data entry is prevented if the General
 Ledger account code status is Not Used. A warning message displays if the
 General Ledger account code status is Inactive.

G/L credit
Press Enter to accept the existing G/L account code, or
 enter the desired G/L credit account for this transaction. The credit
 account description will display. Select the No direct cost option in the
 G/L Master File Maintenance Direct cost menu in order to display this
 information.
Note: Data entry is prevented if the General
 Ledger account code status is Not Used. A warning message displays if the
 General Ledger account code status is Inactive.

YTD cost
The year-to-date cost of ownership displays.

LTD cost
The life-to-date cost of ownership displays.
