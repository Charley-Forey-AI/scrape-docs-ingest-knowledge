# Warrantor File Maintenance

Use this screen to set up and maintain a list of vendors or
 others who provide warranties on equipment, components, and tires.
These provider codes can then be assigned to equipment in the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen. You also have the option to
 associate a vendor with the warrantor, at which point the address and contact information
 display automatically. Warrantor information may also be used for selection purposes on
 some inquiries and reports.
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.
