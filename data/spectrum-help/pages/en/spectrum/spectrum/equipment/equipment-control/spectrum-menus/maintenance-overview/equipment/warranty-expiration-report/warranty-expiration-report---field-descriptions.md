# Warranty Expiration Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Equipment code
Enter the equipment code of the item you want to run the report for, or use the Search Equipment window (F4) to select an equipment code. If your operator has security authorization for the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen, a Maintenance button that allows you to create equipment codes on the fly will be available from the Search Equipment window.

Equipment type
Enter the equipment type code of the item you want to run the report for, or use the Search Equipment Types window (F4) to select an equipment type. If your operator has security authorization for the[Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen, a Maintenance button that allows you to create equipment types on the fly will be available from the Search Equipment Types window.

Warranty type
Indicate the types of warranties for which the report should be printed.

Warrantor
Enter the name of the name of the vendor who provides warranties on equipment, components, and tires to include on this report. Warrantor information is set up and maintained in the [Warrantor File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/warrantor-file-maintenance) screen.

Component
Enter the component code to include on this report. Component information is set up and maintained in the [Component File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/component-file-maintenance) screen.

Tire
Enter the tire code to include on this report.Tire information is set up and maintained in the [Tire File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/tire-file-maintenance) screen.

Yard
Enter the yard code to include on this report.Yard information is set up and maintained in the [Yard File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/yard-file-maintenance) screen.

Area
Enter the code of the location within an equipment storage yard to include on this report. Location information is set up and maintained in the [Equipment Area File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-area-file-maintenance) screen.

Equipment status
 Enter the status of the equipment you want to run the report for, or use the Search Equipment Status Codes window (F4) to select an equipment type.

From/to expiration date
Enter the expiration date range for records to be included on the report.

Include warranty comments?
Select this checkbox to include any comments pertaining to the warranties on the report.

Sort by

Status types
Select which equipment status types to include on the report.

- If the Equipment Tracking module is present, then the Include active equipment and Include inactive equipment checkboxes are selected by default.

-  If the module is not present, then only the Include active
 equipment checkbox is selected by default.

- Any combination of options is permitted.

Expiration type
Select which warranties to include on the report based on the expiration calculation method. Select one of the following options:

- All warranties Select this option to include all warranties on the report, regardless of the warranty calculation method.

- Date expirations Select this option to include only warranties that expire based on date.

- Meter expirations Select this option to include only warranties that expire based on meter readings.

Meter types
Indicate the types of equipment meter for which the report should be printed, or press Enter to print all meter types. Equipment without a specified meter type is included automatically.
