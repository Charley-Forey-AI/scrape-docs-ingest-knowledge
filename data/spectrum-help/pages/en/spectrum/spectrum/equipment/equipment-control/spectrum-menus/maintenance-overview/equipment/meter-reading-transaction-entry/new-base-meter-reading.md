# New Base Meter Reading

Use this window to record initial 'base' meter readings
 when new equipment is added or when an additional meter on an existing piece of equipment
 needs to be tracked. After you click OK to save your changes, the software will create a meter transaction for the
 new base meter reading.
If you are setting a base reading for a Replacement Meter (replacing a
 broken meter) you will want to record that reading in the [Replace Meter](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry/replace-meter) screen. Replace Meter allows you to add
 the old meter (broken) totals along with the new beginning base reading (for the new meter)
 so your life-to-date totals and your Preventative Maintenance Schedule will be correct
 after the replacement.
Important: The New Base Meter Reading
 screen is used only when establishing the first meter reading for a meter after a new
 equipment code is entered in Equipment – Or – if a new meter (#2, 3, or 4) is activated on
 the equipment file.
