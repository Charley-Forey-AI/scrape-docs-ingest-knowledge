# Equipment Meter Reading History Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Equipment
Enter the equipment code associated with the meter history records you would like to review, or double-click to open the Search Equipment window and select an equipment code. I
f your operator has security authorization to the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen, the Search Equipment window will include a Maintenance button that allows you to enter equipment on the fly.

Equipment Type
Enter the equipment type (for example, bulldozer) associated with the meter history records you would like to review, or double-click to open the Search Equipment Types window and select a type code.
If your operator has security authorization to the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen, the Search Equipment Types window will include a Maintenance button that allows you to enter equipment on the fly.

Equipment status
Enter the equipment status(for example, Active or Out of Service) associated with the meter history records you would like to review, or double-click to open the Search Status Codes window and select a status code.

From date / To date
Enter the date range for which you would like to review historical meter reading records, or use the Date calculator windows to select a date range. The To date defaults to the current Equipment Control processing date unless a saved filter is set as the default.
Tip: If you select a Date calculator and then create a report
 filter, the Date calculator will be saved as part of the report filter
 settings.

Meter types
Select the checkboxes for the meter types that you want to include on this report. At least one checkbox must be selected, and multiple selections are allowed.

Status types
Select the checkboxes for the status types that you want to include on this report. At least one checkbox must be selected, and multiple selections are allowed.
The option to Include inactive equipment is selected by default only if the Equipment Tracking module is present.
