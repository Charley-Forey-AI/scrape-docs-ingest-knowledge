# Equipment Type Budget Costs - Field Descriptions

Use the table below as a reference when editing existing equipment types and weight classes in the Budget Cost Maintenance window.
Fields
Descriptions

Properties

Target annual work hours
Enter the number of targeted work hours. Each time this field is changed, the system recalculates all related values in this screen (for example, ownership costs).
The maximum number you can enter here is 9999 for once piece of equipment of the type specified.
Note: Costs are divided by the number of units.

Set rental charge rates from budget?

Rental markup %
Enter a markup value between 0 and 999.99%.

Depreciation
License
Insurance
Property taxes
Interest
Other
When both 'Target annual working hours' and these fields are non-zero, the hourly budget costs display to the right of the field (hourly rate= ownership cost/target hours).

Operating cost (hourly budget) box

Add Code button
Use this button to create a new cost category code.

Edit button
Select a code and click this button to make changes to the cost category. Changes to the description are not allowed.

Remove button
Select a record and click this button to delete it.

Summary Info

Ownership cost
This is the total cost/hour across all six ownership costs. Annual ownership costs are also provided.

Operating cost
This is the total cost/hour across all cost category types. Annual operating costs are also provided.

Total budget cost
The sum of ownership + operating annual cost, and the sum of the ownership + operating cost/hour display.
