# Equipment Charge Rate Workbook - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Overhead total
Enter the amount for allocation across the fleet selected.Note: This amount can be calculated from the G/L Detail
 Account Inquiry. To do this, you need to enter the same date range on
 this screen, and specify the G/L account codes that make up the total
 equipment overhead cost. This amount will be allocated across the fleet
 when loaded into the Grantlun workbook.

Selections

Equipment Equipment type Equipment status
Complete these fields manually or double-click in them to display the available search windows and make your selections.

From/to fiscal year and Period
Enter the fiscal year and period date range for the charge rates you will be creating. These date fields provide flexibility when, for example, you want to set your charge rates on a seasonal basis.

Cost mapping

Cost category type
The cost categories shown in this section are labeled with the type
 descriptions from the Equipment Control Installation
 screen. Use the drop-down lists in the show in worksheet column to select a
 column heading that will be mapped to the cost category on the Grantlun
 workbook. You can map two or more Spectrum cost category types to the same
 worksheet column.

Include meter types
Select the meter types that you want to include in the charge rate spreadsheet. Any combination of these may be selected.
