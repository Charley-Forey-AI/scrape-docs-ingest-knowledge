# New/Edit Quick Equipment Setup - General

Use the General tab to establish which information from the source equipment code should be copied to the new equipment code, including the equipment's general information and rates.
Fields
Descriptions

Group code
Enter the equipment group code to be copied, or select from a list of valid group codes.

Description
Enter the group description to be copied.

Equipment mask
Enter the equipment mask to be copied. For example, SMXXX if equipment in this group should be assigned SM001, SM002, SM003 and so forth.

Next equipment number
The next available equipment sequence number displays. Press Enter to accept the default.

Copy from equipment
Enter the equipment from which to copy, or select from a list of valid equipment codes.
