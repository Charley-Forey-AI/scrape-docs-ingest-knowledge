# Equipment Cost Report

The Equipment Cost Report provides month-to-date,
 year-to-date, and life-to-date costs for depreciation, license, insurance, and each of the
 cost category types defined on the Equipment Control Installation screen.
There are several parameters that can be specified to report on the exact equipment needed.
 Hours shown on these reports include all types of hours: Full, Operating and Standby time.
This report is available both in summary and in detail.
