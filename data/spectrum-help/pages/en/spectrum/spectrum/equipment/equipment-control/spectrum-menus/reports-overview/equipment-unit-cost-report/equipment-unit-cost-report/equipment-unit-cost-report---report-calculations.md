# Equipment Unit Cost Report - Report Calculations

Use the descriptions below for report
 calculations.
Column Name
Calculation Method

Elapsed meter
This tab displays the change in meter reading between the last reading in the Meter Reading History table for the current month and for the last meter reading dated prior to the beginning of the month for each of the four meters.

- If the starting and ending meter readings have different meter types then the first record with a matching meter to the latest-current month reading will be used as the beginning value.

- If the starting and ending meter readings have a base entry within the date range, then the software will compute the change in meter before and after the meter replacement.

Fuel, Lube & Oil, Tires, Depreciation, License, and Insurance columns
These values are calculated by adding the actual equipment costs for the assigned cost categories during the selected date range and then dividing the sum by the elapsed meter value (see above). Values are shown to four decimal places.

Total Cost
These values are calculated by adding all actual equipment costs during the selected date range.
