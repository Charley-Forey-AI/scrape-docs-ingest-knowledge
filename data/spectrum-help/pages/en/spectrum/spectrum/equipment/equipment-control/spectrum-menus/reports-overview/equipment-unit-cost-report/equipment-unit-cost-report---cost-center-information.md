# Equipment Unit Cost Report - Cost Center Information

If cost centers are being used in the current company, the
 software will compare the cost group assigned to the equipment with the cost centers the
 operator is permitted to access (operator scheme overrides do not apply).
If an equipment code is included on the report, activity for all cost categories of
 authorized pieces of equipment is included on the report.
