# Equipment Attachments - Field Descriptions

Use the fields described in the table below when creating new or editing existing equipment attachments.
Fields
Descriptions

Equipment
The primary equipment code that you
 want to associate with an attachment displays. No changes are allowed.
Attached equipment
Enter the equipment code that you wish
 to associate with the primary equipment, or select an equipment code from the
 available drop-down list. The software also validates the equipment code to
 make sure it is an attachable type based on the Allow this type of equipment to be
 attached to other equipment codes setting in the Equipment Type
 File Maintenance screen. If you are in Edit mode, this
 field will be disabled.

Comments
Enter any comments pertaining to the
 attached equipment. Comments made here display in the confirmation window of
 the Transaction Entry and Recurring Transaction Entry screens when the primary
 equipment code is added.
Attachment status
Select an attachment status: Active or Inactive. This status setting
 determines whether any data entry transactions will be generated for the
 attached equipment codes. For example, usage transaction lines are not created
 when an attachment is Inactive, but the equipment code can remain attached even
 if it is not currently in use. Once this status is
 assigned, you can use the Active and Inactive buttons on the main screen to customize your
 viewable list of attachments.
