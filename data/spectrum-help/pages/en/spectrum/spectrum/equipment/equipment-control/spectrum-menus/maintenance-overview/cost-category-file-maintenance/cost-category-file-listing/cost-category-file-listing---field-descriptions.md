# Cost Category File Listing - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Cost category
Enter the cost category, or press Enter to accept the system default of ALL.

Component group
Enter the component group, or press Enter to accept the system default of ALL.

Category types
Select the cost category type(s) to be printed. Note that the first three categories obtain information from the Equipment Control Installation screen, and the last four categories are set up as user-defined fields in the Equipment Control Installation screen.

Status
Select to display Active or Inactive categories on the list.
