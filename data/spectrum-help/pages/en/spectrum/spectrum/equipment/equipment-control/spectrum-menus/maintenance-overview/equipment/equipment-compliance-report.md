# Equipment Compliance Report

Use this screen to create a report showing equipment
 compliance records, across equipment.
Three formats of this report are available using the following information: compliance
 history, compliance expiration dates, or compliance records that need follow-up. The date
 selection fields that display will vary slightly, based on the report format chosen. In
 addition, if the Follow-up format
 is selected, the Include closed
 items checkbox will be disabled because this formation always includes open
 items only.
