# G/L Detail Report - Cost Center Information

If cost centers are being used in the current company, then
 when printing and updating equipment G/L transactions, then the cost center associated with
 each transaction detail line is shown on the report.
All unposted General Ledger transaction entries are included regardless of cost center
 setup.
