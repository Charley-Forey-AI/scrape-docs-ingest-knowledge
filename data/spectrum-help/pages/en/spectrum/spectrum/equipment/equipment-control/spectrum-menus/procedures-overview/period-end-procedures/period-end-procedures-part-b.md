# Period End Procedures: Part B

Follow the steps below for Part B Period End
 Procedures.

1. Process recurring Equipment Control Transactions Entry using Recurring Equipment Transaction Entry.

1. Process recurring entries using Recurring Equipment Transaction Report and update.

1. Run the transaction report and update costs using Transaction Report and update.

1. Process Equipment Control using G/L Detail Report and Update.

1. Periodically, use Processing Date Change to set minimum and maximum processing dates.
