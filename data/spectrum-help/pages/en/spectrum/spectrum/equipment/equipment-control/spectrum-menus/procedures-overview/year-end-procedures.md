# Year End Procedures

This update can be run more than once and must be run any
 time that changes are made to the previous year.

- Complete all monthly period end procedures.

- Make a backup and label it year-end.

- Run the Opening Forward Balance Update to set the beginning balances for a new fiscal year from the previous year's data.
