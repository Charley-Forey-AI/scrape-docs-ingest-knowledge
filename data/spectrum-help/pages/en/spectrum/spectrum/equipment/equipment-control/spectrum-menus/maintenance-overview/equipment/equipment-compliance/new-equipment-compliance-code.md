# New Equipment Compliance Code

The New Equipment Compliance and Edit Equipment Compliance
 windows are used to enter new equipment compliance code associations or make changes to
 existing compliance code associations.
Imaging capability allows you to attach pictures or other scanned documents (such as
 permits or licenses) for easy retrieval.
The fields that display in the Properties section of this window will vary slightly, based on the compliance code type. For example, for permits, Permit # and Registration # fields will display, and for licenses, License # and Registration # fields display, and inspections will provide Document # and Issuer name fields. In addition, the Closure section only displays when editing existing equipment compliance codes, not when setting up new code associations.
Fields
Descriptions

Equipment
The selected equipment code displays in this field.

Agency/State
Enter the Agency / State code for the organization that you want to associate with this equipment compliance code, or select an Agency / State from the available drop-down list.
Click the icon adjacent to this field to open a lookup window if you wish to review the Agency / State Contact Information, as set up in the [Agency / State File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/agency--state-file-maintenance) screen, before making your selection.

Compliance code
Enter the compliance code that you want to associate with this piece of equipment, or select a code from the available drop-down list. Compliance codes are originally set up in the [Compliance Code File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/compliance-code-file-maintenance) screen.

Properties

Document #, License #, Permit #
The field name that displays here is dependent on the compliance code type: Permit, License, Inspection, Violation, or Other. Enter the appropriate number (document number, permit number, etc.) here.

Registration #, Issuer name
The field name that displays here is dependent on the compliance code type: Permit, License, Inspection, Violation, or Other. Enter the registration number (for permits and licenses) or issuer name (for inspections and violations).

Issue date
Enter the date that this equipment compliance record was issued, or use the available drop-down list to select a date.

> Shortcut Key
Date Automatically Entered

T
Today's date.

H
The last day of the current month.

M
The first day of the current month.

+
Tomorrow's date.

-
Yesterday's date.

Expiration date
This field defaults to the standard number of days after the issue date, or you can manually enter the date that this equipment compliance record is scheduled to expire.

Follow-up date
If this equipment compliance record will require follow-up at a later date, enter that date here.

Comments
Enter any additional comments that might be helpful to others when reviewing this equipment compliance record.

Reference details

Job Employee
For reference purposes, enter the job and employee associated with this equipment compliance code (this information is especially useful for Inspections and Violations).

Amount
Enter the amount paid for this inspection, license, permit, violation, and more (for example, if you paid $1200.00 for this permit, type 1200 and press Enter).

Closure

Closed?
Select this checkbox to close this equipment compliance record, regardless of expiration dates (if applicable).
Note: If this compliance code was set to track expiration dates in the
 New/Edit Compliance Code window, then the Expiration type field will
 be enabled .

Authorized by
Enter the name of the person or organization responsible for closing this equipment compliance record.

Close date
Enter the date that this equipment compliance record was closed.

Expiration type
If the compliance code that has been closed originally was set to track expirations in the Compliance Code File Maintenance screen, then this field is enabled and you must select an expiration type from the available drop-down list. Options include Renewed or Discontinued (which is the default selection), and these expiration types will then display in the list box on the main Equipment Compliance screen.
