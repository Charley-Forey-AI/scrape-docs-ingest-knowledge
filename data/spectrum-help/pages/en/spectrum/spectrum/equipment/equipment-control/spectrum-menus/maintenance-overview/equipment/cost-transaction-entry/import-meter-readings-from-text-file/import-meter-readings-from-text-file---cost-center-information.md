# Import Meter Readings from Text File - Cost Center Information

If cost centers are being used in the current company,
 validation will take place while importing meter reading transactions.
Spectrum will allow the Operator to import a meter reading for a piece of equipment only
 if the Operator has permission to access that equipment in the specified company.
 Spectrum will compare the equipment's cost center with cost centers in the Operator's
 assigned cost center scheme, and if the cost center is not present, the import of meter
 readings for that piece of equipment will be disallowed and the records will be sent to
 the Import Meter Reading Error Listing.
