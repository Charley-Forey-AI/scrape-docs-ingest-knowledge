# Make File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Make
Enter the equipment make; up to 20 characters are allowed. The makes entered here can then be used in the Advanced Search Options window to narrow search results.

Equipment type
Enter the equipment type code or select from the available drop-down list. Up to four characters are allowed, and entry in this field is optional. Once a type is entered, the longer equipment type description displays. Information entered here may be used in the Advanced Search Options window to narrow search results.
