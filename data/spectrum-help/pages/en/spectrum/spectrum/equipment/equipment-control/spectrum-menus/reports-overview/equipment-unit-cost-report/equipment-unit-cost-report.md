# Equipment Unit Cost Report

If the equipment history contains a cost record with a cost
 category that is not valid, the cost of that transaction will appear in the Totals columns, but the amount will not show
 up anywhere else on this report.
Please review the [Equipment Unit Cost Report - Report Calculations](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/reports-overview/equipment-unit-cost-report/equipment-unit-cost-report/equipment-unit-cost-report---report-calculations) following the report for calculation information.
 Note Regarding Unit Costs:
All unit costs in Spectrum are calculated as the total cost / hours or meter reading, and this number is rounded. Therefore, if you try to multiply the unit cost times the hours, this number will NOT equal the total costs.
