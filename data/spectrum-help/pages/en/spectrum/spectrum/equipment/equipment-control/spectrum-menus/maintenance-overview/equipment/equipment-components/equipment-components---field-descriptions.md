# Equipment Components - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Equipment

Component
Use this field to assign a component to the selected piece of equipment.

- If the Assign component codes for this equipment type checkbox is selected in the Require type-specific codes section of [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance), then the software will only allow users to enter a component code that is authorized for this type of equipment. Likewise, the Search Components window will only display codes that are assigned to this type.

- If the Assign component codes for this equipment type checkbox is not selected, then any valid component code may be selected and the Search Components window will show all codes that are set up in this company.

While it is not necessary to add components to equipment, when a component code is added, the software assures that it is a valid code and determines whether it must be authorized for the particular equipment type.

Properties

Year
Enter the year this component was made.

Make
Enter the component make or select a make from the Search Makes window. Makes are set up in the [Make File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/make-file-maintenance) screen.

Model
Enter the component model or select a model from the Search Models window. Models are set up in the [Model File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/model-file-maintenance) screen.

Serial # / VIN
Enter the component serial or VIN number.

Purchase date
Enter the date that this component was purchased or click the drop-down arrow to select a date.

In-service date
Enter the date that this component was initially placed into service, or click the drop-down arrow to select a date.

Inactive date
Enter the date that this component was pulled from service and became inactive, or click the drop-down arrow to select a date. If a date is entered in this field, the Status field will automatically default to Inactive.

Meter reference
Enter the meter reference, if applicable. Meter information is set up in Equipment Meters, and provides users with tracking capability. For example, a meter reading might be set up for mileage or usage hours, to allow users to track mileage to and from the job, or to tally hours while working at the jobsite.

Status
This field defaults to Active when adding a new equipment component. If you enter a date in the Inactive date field, the status will automatically default to Inactive.
