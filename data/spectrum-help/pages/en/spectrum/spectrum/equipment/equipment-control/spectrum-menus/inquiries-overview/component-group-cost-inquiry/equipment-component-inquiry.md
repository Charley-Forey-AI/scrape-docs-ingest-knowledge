# Equipment Component Inquiry

Use this window to view information about equipment components that comprise the selected component group totals.
Select a record and click the View button to open the Equipment Component window. Use this window to review the specific details for the selected piece of equipment, such as year, make, model, VIN, purchase date, status, and so forth. Changes are not allowed in this window.
