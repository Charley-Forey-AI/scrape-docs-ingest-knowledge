# Equipment Warranty - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Properties

Warranty type
Select the warranty type from the available drop-down list. The list of valid warranty types is set up and maintained in the [Warranty Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/warranty-type-file-maintenance) screen.

Warrantor
Select the warrantor from the available drop-down list. A list of warrantors is set up and maintained in the [Warrantor File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/warrantor-file-maintenance) screen.

Start date
Enter the first day that the warranty went into effect for this piece of equipment, or select a date.

Expiration date
Enter the last day that the warranty will be in effect for this piece of equipment, or select a date.

Start meter
Enter the initial meter value/reading, if applicable.

Expiration meter
Enter the Expiration meter value associated with the warranty, if applicable. Once an entry is made in this field, the associated Meter # field will be enabled and you must enter a meter number.

Meter #
If an Expiration meter value has been entered, the software will attempt to default the meter# (1-4) automatically from either the equipment component or equipment tire tables (but will allow you to assign a different meter number). Meter numbers are originally set up in the [Equipment Meters](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-meters) tab (for example, Meter #1 may track mileage). After the meter number is entered, the associated description displays to the right of the field.

Comments
Type any comments that you would like to include with these warranty details.

Warrantor name
Address 1 Address 2 City State Zip

E-mail
The warranty provider information defaults based on the warranty type and warrantor entered for this piece of equipment. Warranty provider information is maintained in the [Warrantor File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/warrantor-file-maintenance) screen, and may be dependent on the vendor file.

Notes tab
Use the Notes tab screen to add and maintain (equipment notes and standard messages; two topics (Notes and Filters) are pre-defined. The Filters notes entered on the Components screen are also available from this screen.
