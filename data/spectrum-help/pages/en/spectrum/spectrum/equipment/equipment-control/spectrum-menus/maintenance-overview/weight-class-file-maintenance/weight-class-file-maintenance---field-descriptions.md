# Weight Class File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Weight class
Enter a weight class code. There are no industry standards for these codes; they are user-defined and should make sense to others in your company (for example, for heavy equipment over twenty tons, you might use H20T). Entry in this field is required.

Description
Enter a short description for the weight class code. Entry in this field is required.

U/m
Enter the unit of measurement for the weight class (for example, pounds, tons, or kilograms). Entry in this field is required.

Minimum weight
Enter the minimum weight associated with this code. This field must be completed if a maximum weight it entered. When adding new weight classes, the minimum weight will default from the maximum weight of the previous entry (in the same session).

Maximum weight
Enter the maximum weight associated with this code. This field may be left blank, but in this case, the system will include all weights greater than the minimum in this class.
