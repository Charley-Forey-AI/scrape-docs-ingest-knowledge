# Agency / State File Maintenance - Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Fields
Descriptions

Code
Create or edit an Agency / State code. For example, you
 might choose to use acronyms or a numbering system, or some other standard
 naming convention.

Description
Enter or edit the description for this Agency / State
 code.

Properties

Type
Select an agency type from the available drop-down list.
 This is a must-enter field.

Vendor code
Enter a vendor code to associate it with this agency /
 state code, if applicable. This field is optional, but if a code is
 specified here, the Contact information for the vendor will default below. A
 Vendor Inquiry button next to this field opens the Vendor Detail Inquiry
 window for a more detailed look at this vendor's information.

Comment
Enter any notes pertinent to this agency / state code or
 vendor code.

Contact

Address, phone, email, and more.
If a vendor was entered in the Vendor code field, the
 contact information for that vendor will default here. If no vendor was
 specified, enter the contact information for this agency / state code,
 including the mailing address, phone, and e-mail address. A Send e-mail
 button is available for convenience if you want to send a message to a
 specified contact person regarding a license, permit, inspection, and more.
