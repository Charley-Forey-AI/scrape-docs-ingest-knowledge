# New/Edit Statistic

Use the New Statistic Code screen to create new codes, and
 use the Edit Statistics Code screen to make changes to an existing statistic code.
The Properties tab (shown here) prompts for a statistic code, and allows
 you to enter a description and some default instructions on how to obtain the statistic
 test result. The instructions entered here will then default during setup in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) and [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screens.
If an alpha-numeric test result type is specified, the Validation section will be disabled, and instead a Validate alph-numeric entries? checkbox will display. If this checkbox is selected, the Valid Results button will be enabled.
Fields
Descriptions

Statistic
Enter or select a statistic code.

Properties

Result format
Select the result format that you want to associate with this statistic code. The fields that display in the Validation section will depend on the selection you make here.

Standard instructions
Type the default instructions on how to obtain the statistic test result. The instructions entered here will then default during setup in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) and [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screens.

Standard norm
Enter the standard value for this statistic code (for example, the standard tire PSI number). This entry will be used for comparison purposes, and will validate and display based on the mask entered above.

Standard minimum
Enter the default minimum value for this statistic code. This entry will be used for comparison purposes, and numbers below this value will result in a "Fail" test result.

Standard maximum
Enter the default maximum value for this statistic code. This entry will be used for comparison purposes, and numbers above this value will result in a "Fail" test result.
