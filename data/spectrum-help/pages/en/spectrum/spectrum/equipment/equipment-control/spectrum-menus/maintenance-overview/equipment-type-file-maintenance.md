# Equipment Type File Maintenance

The Equipment Type File Maintenance screen is used to group equipment types together, for example, bulldozers, dump trucks, and other equipment.
Equipment types must be defined in this screen prior to setting up the Equipment screen.
Aside from using this screen to group equipment types together, you can also use this screen to record the debit and credit accounts for depreciation, license, and insurance transactions for each type.
If you want, you can use different depreciation expense accounts for each type of equipment. Debit fields require 'Equipment cost' accounts, and credit fields require 'Non-direct cost' accounts.
Equipment type will also be used for certain selections on some of the Equipment Control reports.
