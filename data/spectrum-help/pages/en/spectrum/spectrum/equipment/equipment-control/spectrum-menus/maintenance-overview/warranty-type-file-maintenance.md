# Warranty Type File Maintenance

Use this screen to define and maintain a list of standard
 warranty types (for example, manufacturer, extended, or parts-only).
These warranty types can be applied in the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen. Warranty type information may also be used for selection
 purposes on some inquiries and reports.
