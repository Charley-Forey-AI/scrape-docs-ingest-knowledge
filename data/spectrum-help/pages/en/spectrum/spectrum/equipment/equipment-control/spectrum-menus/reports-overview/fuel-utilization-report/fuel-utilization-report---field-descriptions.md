# Fuel Utilization Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Equipment
Enter the equipment code of the item you want to run the report for, or use
 the Search Equipment window (F4) to select an
 equipment code. If your operator has security authorization for the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screen,
 a Maintenance
 button that allows you to create equipment codes on the fly will be
 available from the Search Equipment window.

Equipment type
Enter the equipment type code of the item you want to run the report for, or
 use the Search Equipment Types window (F4) to select
 an equipment type. If your operator has security authorization for the[Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen, a Maintenance button that
 allows you to create equipment types on the fly will be available from the
 Search Equipment Types window.

Equipment status
 Enter the status of the equipment you want to run the report for, or use the
 Search Equipment Status Codes window (F4) to
 select an equipment type.

Job
Enter the job code(s) that you would like to include on this report or
 double-click to access the Search Jobs window and
 select the job.
Note: If your operator has security access for the Job Cost > Job File Maintenance screen, a Maintenance button will
 display in the Search Jobs window to allow you to add
 a job on the fly.

Fuel truck

- Select the fuel truck(s) that you would like to include on this report or
 double-click to access the Search Fuel Trucks
 window and select the truck.

- Your selection will limit the information displayed on the report based on the truck associated with the transaction.

- The fuel truck corresponds to the Device ID recorded in the Equipment Service System; the device type and serial number are also displayed.

From/to transaction date
Use the From and To transaction date fields to specify the date range of the equipment usage that you want to include on this report.

Sort by
Select to sort the report data by job, equipment, or fuel truck. If the fuel truck option is selected, the Fuel truck selection box will be enabled, thereby allowing you to further refine your report data by selecting to show all fuel pumped, unallocated fuel only, or equipment-specific fuel only.

Format
Select to print this report in Summary or Detail format. Examples of the difference between these two formats can be seen in the sample reports.

Road usage

- Select to include on-road or off-road equipment (or both) on the report. If both of these checkboxes are clear, the report will be blank.

- Road usage is defined in the [Equipment Ownership Information](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-ownership-information) tab.

Fuel truck
 If the fuel truck option is selected in the Sort by section, these additional selections will be enabled, thereby allowing you to further refine your report data by selecting to show all fuel pumped, unallocated fuel only, or equipment-specific fuel only.
Note Regarding Unallocated Fuel Transactions:
Unallocated fuel transactions are non-company specific. In the unusual case where equipment is maintained in multiple Spectrum companies, it is possible that the same unallocated fuel transactions would display on the Fuel Utilization Report in both companies. For this reason, it is recommended that you only select the Unallocated fuel only option when you are printing in the main equipment company.
