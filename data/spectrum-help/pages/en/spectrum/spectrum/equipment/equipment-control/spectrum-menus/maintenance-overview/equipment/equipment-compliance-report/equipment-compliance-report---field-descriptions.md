# Equipment Compliance Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Report Format

-  If the Compliance history or Expiration formats are selected, date selections are provided for From/To issue date and From/To expiration date (From dates default as clear, and To dates default the last day of last year in the fiscal calendar).

- If the Follow-up format is selected, date selections are provided for From/To follow-up date. The From follow-up date defaults to the current Equipment Control processing date and the To follow-up date defaults to 30 days beyond the From date (unless a saved filter is present). In addition, the Include closed items checkbox is disabled for this format because it always includes only items that are not close (and items will display in follow-up date order, from oldest to future dates).

- If the Expiration or Follow-up format is selected, a Contact line displays on the report, directly below the Comment line (if comments are present).

Sort by
Select to sort the report data alphabetically by Equipment, Agency / state, or Compliance code. Your selection will determine the section headers that will sub-divide the report.

Selections

Equipment Equipment Type Agency / state Compliance code Equipment status Job Employee
Use the fields in this section to customize the resulting list of equipment compliance records that will display on the report. Drop-down lists and SuperSelect options are available at each field for easy entry from existing records as set up in the [Compliance Code File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/compliance-code-file-maintenance) screen and[Equipment Compliance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-compliance) screen.

Include closed items
Select this checkbox if you want to include Closed items on this report. If selected, the Status column on the report displays the word 'Closed' for these items, and if the expiration type was being tracked, the Closed items may further specify Renewed or Discontinued items, based on the setting in the [New Equipment Compliance Code](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-compliance/new-equipment-compliance-code) screen.

Dates

From / to dates
Different date selection fields will display in this section, based on the report format selected:

-  If the Compliance history or Expiration formats are selected, date selections are provided for From/To issue date and From/To expiration date (From dates default as clear, and To dates default the last day of last year in the fiscal calendar).

- If the Follow-up format is selected, date selections are provided for From/To follow-up date. The From follow-up date defaults to the current Equipment Control processing date and the To follow-up date defaults to 30 days beyond the From date (unless a saved filter is present).

Agency / state type
Select the agency/state types that you wish to include on this report. The agency / state organizations responsible for issuing the equipment compliance types (licenses, permits, etc.) that are being tracked. For example, you might want to generate a report that strictly shows any local (agency) inspections (compliance type).

Compliance type
Select the checkboxes corresponding to the equipment compliance types that you want to include on this report. Choose to include licenses, permits, inspections, violations, or other types (you may select any combination of these compliance types).

Status types
Select which equipment status types to include on the report.

- If the Equipment Tracking module is present, then the Include active equipment and Include inactive equipment checkboxes are selected by default.

-  If the module is not present, then only the Include active
 equipment checkbox is selected by default.

- Any combination of options is permitted.
