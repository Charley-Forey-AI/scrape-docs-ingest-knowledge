# Cost Category File Listing - Cost Center Information

If cost centers are being used in the current company, when
 running the Cost Category File Listing, Spectrum includes the cost category on the listing
 only if the you have permission to access the equipment cost category.
The system compares the cost center assigned to the cost category with cost centers in
 your operator's assigned cost center scheme, and if the cost center is not included,
 then that equipment cost category is not shown on the listing.
