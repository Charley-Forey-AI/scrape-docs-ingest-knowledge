# Component Group Cost Inquiry - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Date
The current Equipment Control processing date displays. To run the inquiry based on a different date, enter the desired date, or double-click to select a date from the Date Change window.

Search
 Search allows you to type your search words and press Enter or click Go to display the search results. You only need to enter a partial value for the selection you are searching.

Buttons

Report
Click this button to display the [Equipment Cost History Report](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-cost-history-report) starting screen. Use this report to review a list of all the equipment costs incurred, sorted by piece of equipment.

Status Type
Select the checkboxes for the equipment status type(s) you want to review on this inquiry: Active, Inactive, or Retired.

Cost Categories
Click this button to display the [Summary Cost Inquiry by Cost Category - Year-to-date](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/inquiries-overview/component-group-cost-inquiry/summary-cost-inquiry-by-cost-category---year-to-date) tab.

Components
Click this button to display the [Equipment Component Inquiry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/inquiries-overview/component-group-cost-inquiry/equipment-component-inquiry) screen.

W.O. Details
Click this button to display the [Component Group / Detail Work Order Inquiry](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/inquiries-overview/component-group-cost-inquiry/component-group--detail-work-order-inquiry) window.
