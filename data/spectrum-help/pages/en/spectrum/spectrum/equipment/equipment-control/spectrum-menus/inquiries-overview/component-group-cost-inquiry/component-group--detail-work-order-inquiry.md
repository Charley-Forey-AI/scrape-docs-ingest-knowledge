# Component Group / Detail Work Order Inquiry

Use this window to view information about work orders that comprise the selected component group totals.
Fields
Descriptions

Component group
The component group code and description associated with the selected work order display. No entry is allowed.

Equipment code
The equipment code and description associated with the selected work order display. No entry is allowed.

View W.O.
Click this button to view the associated work order in Lookup Mode.
