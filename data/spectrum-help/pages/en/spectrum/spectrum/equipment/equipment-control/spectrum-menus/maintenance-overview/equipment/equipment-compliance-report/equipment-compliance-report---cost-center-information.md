# Equipment Compliance Report - Cost Center Information

If cost centers are being used in the current company, the
 Equipment Compliance Report will validate whether the operator has security authorization to
 access the piece of equipment as in Equipment Compliance.
When set to 'Pending', no cost center filtering is applied. Like the
 inquiry screen, all reference data will appear for authorized equipment.
If the cost centers are set to 'Yes' and entities are enabled for the
 current company, the company name in the report title will be replaced with entity
 name.
In the Cost
 group field (below the Equipment status field), the operator can filter which equipment is
 included on the report when a single cost center or cost group is specified on the
 starting screen.
Enter a valid Cost
 group or Cost
 center code, and then press Enter to accept ALL or the specified Report default of the operator's
 cost center scheme. After a code is entered, the associated description displays to the
 right.
If cost center entities are being used, this report displays the
 Entity name in the heading of the report whenever all cost centers included in the
 selected 'Cost group' for the report are assigned to the same entity.
