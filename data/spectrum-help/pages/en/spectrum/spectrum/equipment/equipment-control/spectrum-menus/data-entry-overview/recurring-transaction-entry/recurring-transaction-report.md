# Recurring Transaction Report

The Recurring Transaction Report screen is used to print a
 report of information entered in Recurring Transaction Entry.
After printing, the Recurring Transaction Update screen appears. This screen updates the
 equipment transaction files with the recurring transaction information. Use this screen to
 proceed with the update only after the Recurring Transaction Report has printed accurately.
 This report should be kept as a permanent record.
Note: While Inactive batches may display on this report, Inactive
 batches are NEVER updated. Similarly, if you select to see Inactive batches on the report
 and Active batches exist, the Active batches will be included in the update.
