# Compliance Code File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Code:
Enter an alphanumeric compliance code. This is a user-defined code that will be assigned to equipment for tracking purposes, for example, inspections, permits, licenses, and more.

Description:
Enter a long description for the corresponding compliance code. Up to 30 characters are allowed.

Type:
From the available drop-down list, assign a type to the compliance code that you are setting up. Options include :License, Inspection, Permit, Violation, and Other.

Track expiration?
Select this checkbox if you want to track the expiration date for this compliance code (for example, if a permit is only good for a limited time). Selecting this checkbox and entering a number of days in the next field will allow the system to automatically calculate the expiration date for you when an issued date is entered in the Equipment - Compliance tab.
Tip: For one-time use permits, leave this checkbox clear.

Standard number of days from issue to expiration:
This field is only enabled if the Track expiration checkbox is selected. If enabled, enter the number of days that this compliance code is good for before it will expire if not renewed. If you select the Track expiration checkbox and leave this field blank, the software will prompt you to manually enter an expiration date for this code in the Equipment - Compliance tab; if you enter a number here, the system will automatically calculate the expiration date for you (based on the date of issue).
