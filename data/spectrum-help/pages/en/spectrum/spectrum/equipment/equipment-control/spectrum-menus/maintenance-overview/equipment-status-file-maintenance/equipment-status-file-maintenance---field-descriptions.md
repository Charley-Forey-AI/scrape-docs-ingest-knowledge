# Equipment Status File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Status code
Enter the status code, or select from a list of valid status codes.

Description
Enter a description of the equipment status, or press Enter to accept the corresponding default description.

Status type
Select the equipment status type from the drop-down menu: Active, Inactive, or Retired. A status of Retired indicates the equipment is permanently disabled, which offers protection throughout Spectrum from improper equipment tracking.
