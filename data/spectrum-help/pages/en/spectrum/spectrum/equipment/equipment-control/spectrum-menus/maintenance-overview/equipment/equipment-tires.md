# Equipment Tires

The New Equipment Tire and Edit Equipment Tire windows are
 used to enter new equipment tires or make changes to existing tire records.
This window allows you to easily track multiple like tires for the
 equipment, including tires that are later retired and replaced.
If you have access to the Meter Reading Transaction Entry screen and a
 Mount date is specified within
 the current processing date range, a Meter button conditionally displays. Use this button to access the [Meter Readings](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/meter-reading-transaction-entry/meter-readings) window and enter
 the mount date meter reading.

- Click New to display the New Equipment Tire window and create a new
 statistic.

- Select a record and click Edit to display the Edit Equipment
 Tire window make changes to an existing statistic.
