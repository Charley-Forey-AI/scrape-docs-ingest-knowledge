# G/L Detail Update - Cost Center Information

If cost centers are being used in the current company, when
 printing and updating equipment G/L transactions, then the cost center associated with each
 transaction detail line is shown on the report (both inter-post and inter-company G/L accounts
 are assigned when the report is compiled).
All unposted General Ledger transaction entries are included regardless of cost center
 setup.
