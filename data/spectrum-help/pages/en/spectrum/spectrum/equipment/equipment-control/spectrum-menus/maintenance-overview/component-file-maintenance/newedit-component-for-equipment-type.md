# New/Edit Component for Equipment Type

The New Component for Equipment Type and Edit Component for
 Equipment Type windows are used to associate or change components associated with equipment
 types.
When a new component is added, the user-defined fields default from the
 [Component File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/component-file-maintenance) screen (if any
 exist).
Fields
Descriptions

Component
Select an equipment component from the available drop-down list, for example, a front power axle. Entry in this field is required.

Add this component automatically to new equipment?
Select this checkbox if you want this component to be added automatically when a new piece of equipment (assigned to this equipment type) is added.

Hide Unused
Click the Hide Unused button to hide all unused user-defined fields that appear in the list box for the selected component. The green indicator on the button will display when unused fields are hidden.
Note: If you give a value to a user-defined field and that value
 is later removed or if you press Enter when defining a prompt's value
 (thereby leaving the prompt's value blank), the user-defined field will
 display when you click the Hide Unused button.

Edit
Click the Edit button to edit the selected user-defined field.
