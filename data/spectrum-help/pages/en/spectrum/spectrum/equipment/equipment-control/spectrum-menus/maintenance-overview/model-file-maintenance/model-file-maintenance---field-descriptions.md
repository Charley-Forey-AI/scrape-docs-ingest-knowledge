# Model File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Make
Enter the equipment make or use the available drop-down list to select a make. Only valid codes already set up in the [Make File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/make-file-maintenance) screen are allowed.

Model
Enter the equipment model. Up to twenty characters are allowed.

Equipment type
The equipment type defaults to the type specified for the make, or you can enter a valid equipment type code or select one from the available drop-down list.
