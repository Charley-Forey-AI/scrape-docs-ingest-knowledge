# Cost Category File Maintenance - Cost Center Information

If cost centers are being used in the current company, the
 Cost Category File Maintenance screen includes a Cost center field.
This field is typically left blank unless individual cost categories are associated with
 certain cost centers across equipment codes.
Cost categories assigned to a cost center that you are not permitted to access do not display. The system compares the cost center assigned to each cost category with a cost center in your operator's assigned cost center scheme, and if the cost center is not included, then that equipment cost category is not shown.
