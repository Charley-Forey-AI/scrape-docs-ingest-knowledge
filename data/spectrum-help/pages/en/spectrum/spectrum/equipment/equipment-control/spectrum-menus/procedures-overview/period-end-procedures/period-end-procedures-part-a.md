# Period End Procedures: Part A

Follow the steps below for Part A Period End
 Procedures.

1. Calculate depreciation, license, and insurance costs using Build Ownership Cost Transactions.

1. Enter, as necessary, depreciation, license, and insurance costs using DLI Entry.

1. Print and review the Depreciation, License and Insurance Edit List.

1. Update depreciation, license, and insurance costs using Ownership Cost Journal after approving the edit list.

1. Print and review the Depreciation, License and Insurance G/L Report. Update if approved using DLI G/L Update.
