# Transfer Component to Other Equipment - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Transfer from equipment

Equipment
Enter the equipment you want to transfer the component from, or select the equipment using the available search window. You may select equipment that has a status ofInactive or Not used, in order to salvage useful components from a retired piece of equipment.

Component
Enter the component code that you wish to transfer, so select a component from the available search window. You may select a component with a status of Inactive; for example, in order to salvage a component from a retired piece of equipment. Retired components may not be transferred.

Sequence
After a component is specified, the sequence number automatically defaults or you may select a different one.

Transfer date
Enter the date you wish to transfer the component from the original equipment. This date defaults to the current Equipment Control processing date.

Transfer to equipment

Equipment
Enter the equipment code that you want to transfer the component to, or select the equipment from the available search window. This equipment code may have a status of Active or Inactive, but not Retired.

Meter reference
This field is only enabled if the specified component has a meter reference, allowing you to enter a new meter reference number (1-4) and transfer the meter readings/history with the component. For example, if an engine component had an hours meter and this meter was tracked as meter #3 in the original equipment, then it would be helpful to move the meter reading to the new equipment . Occasionally, a component may be associated with a meter reading that cannot be moved with the component, in which case this field would remain blank.
Protection is provided to make sure that meter readings are not merged into the destination equipment by validating that the meter specified is not currently being used.
