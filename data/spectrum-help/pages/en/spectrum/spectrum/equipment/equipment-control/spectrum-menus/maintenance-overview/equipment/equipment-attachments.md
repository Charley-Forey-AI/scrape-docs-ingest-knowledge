# Equipment Attachments

Use this screen to maintain a list of the equipment pieces
 that are regularly used in conjunction with the selected (primary) equipment.
An example of an attachment would be a tractor-trailer (wherein the
 tractor is the primary equipment and the trailer is the secondary equipment). When the
 equipment usage is recorded, two entries are needed because both pieces of equipment are
 present on the job. The attachments screen allows you to record the usage needs for both
 equipment codes without having to remember to indicate the secondary usage--thereby
 assuring that usage and job charges are properly recognized.
In the Payroll > Time Card Entry screens, when a transaction for pay types ER, EO, ED, or EU is added, the
 software determines whether the equipment code assigned to the time card entry has any
 attachments. If so, additional EU (Equipment Usage) time cards will be generated
 automatically, based on a setting in the Equipment Control Installation screen.
The first column on this screen displays an icon indicating an exception
 in the specific case where the attached equipment is currently set to Retired status, and the Transaction Entry
 and Recurring Transaction Entry screens will not generate transaction lines for this
 equipment.

- Click Add
 to select the piece of equipment that you wish to associate with the primary equipment.

- If you click Edit to make changes to an existing attachment, you may only edit the
 Comments and Attachment status fields.

- Click the Status button to determine which types of records will display in the
 list box.
