# Equipment File Listing - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Equipment code
Enter a unique code to identify each piece of equipment, or press F4 or double-click on this field to select from a list of available equipment codes. It is recommended that the code be set up in a manner that reflects the equipment's name because the equipment file can be sorted on this variable. It may be a straight numeric code or an alphanumeric code. Whatever coding method is used, consistency is highly recommended.
The equipment code will be used throughout Spectrum to refer to a specific piece of equipment.

Equipment type Division Yard code Area Equipment status
Enter the specific equipment parameters to narrow your report results, or press Enter to accept the system default and print ALL records.

Include equipment notes? Include components? Include compliance records? Include warranties? Include statistics?
Customize the information that displays on the listing report by selecting only the appropriate checkboxes (any combination of checkboxes is allowed). Options include equipment notes, components, compliance records, warranties, and statistical data. Any associated comments, for example, compliance comments, will also display on the listing (for statistics, the instructions and examination notes will display).

Purchase types

Include owned equipment? Include leased equipment? Include rented equipment?
Select the equipment types that you want to include on the report. Options include owned, leased, and rented equipment (or any combination of these three equipment types).

Format
Select a format for the report: Summary or Detail

Status types

Include active equipment?
Include inactive equipment?
Include retired equipment?
This field works in conjunction with the Equipment status field to limit the equipment codes shown on the listing. For example, if you were to select all codes starting with A (A* ), plus selects the checkboxes for Active and Inactive status types, then the listing would all omit 'retired' equipment even if the status code starts with the letter A. In other words, the resulting data is filtered using the SuperSelect options and the Status type selections.

Sort by
Select whether to sort the report by Equipment code or Yard code.
