# Equipment Meter Reading History Report - Cost Center Information

If cost centers are enabled in this company, a cost group
 field will display.
Enter the cost group code associated with the meter history records you would like to
 review or double-click to display the Search Cost Centers/Cost Groups window and select
 a cost group. The software will verify that your operator has security authorization for
 the cost group entered, and only equipment that your operator is allowed to access will
 display.
