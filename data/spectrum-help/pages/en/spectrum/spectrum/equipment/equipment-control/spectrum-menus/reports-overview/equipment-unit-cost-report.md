# Equipment Unit Cost Report

Use this report to review equipment costs based on actual
 elapsed meter readings for a specified date range.
You can also evaluate the costs per unit by cost category type and total operations,
 comparing the operational unit costs among equipment codes.
Note: Note Regarding Unit Costs: All unit costs in Spectrum are
 calculated as the total cost / hours or meter reading, and this number is rounded.
 Therefore, if you try to multiply the unit cost times the hours, this number will NOT equal
 the total costs.
