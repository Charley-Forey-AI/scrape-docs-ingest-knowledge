# Compliance Code File Maintenance

Use this screen to set up and maintain a list of standard
 compliance codes (licenses, permits, inspections, etc.).
For use in the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) and
 [Equipment Compliance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-compliance)
 screens. You can also track compliance expiration information. Compliance codes can also be
 used for selection purposes in some of the inquiry and report screens.
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.
