# Tire File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Code
Enter a tire code.

Description
Enter a short description for the tire code.

Make
Enter the name of the tire manufacturer, or select from a list of available makes.

Model / Size
Enter the module number or size of the tire (provided by the manufacturer), or select from a list of available models.

Standard PSI
Enter the recommended air pressure for this tire model (provided by the manufacturer).

Detail description
Enter a long description for this tire code.
