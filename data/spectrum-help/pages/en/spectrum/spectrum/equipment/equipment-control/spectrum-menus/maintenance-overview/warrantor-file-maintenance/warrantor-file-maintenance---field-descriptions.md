# Warrantor File Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Warrantor
Enter the warrantor code. The codes created here can then be used in the Advanced Search Options window to narrow search results.

Description
Enter a description for the warrantor code (up to 30 characters are allowed). Warrantor descriptions can be used in the Advanced Search Options window to narrow search results.

Vendor
Enter or select a vendor code if you wish to associate one with the warrantor code. If you do choose to associate a vendor with a warrantor code, the Properties section of this window will automatically be completed with information from the vendor file. In addition, once associated with a warrantor code, the vendor can be used in the Advanced Search Options window to narrow search results in the Warrantor File Maintenance screen.

Warrantor name
Enter the name of the company that will be providing the warranty. This field will be completed automatically if a vendor code has been associated with this warrantor code, otherwise you may manually enter the warrantor name. Warrantor names can also be used in the Advanced Search Options window to narrow search results.

Properties

Address, Phone, Email, and more.
If a vendor is associated with the warrantor code, this section will automatically be completed with information from the vendor file. If no vendor has been entered, you may complete these fields (address and contact information) manually.
