# Purge Unposted EC Transactions

Use this update to delete a batch of unposted equipment cost, revenue transactions and meter readings for the current company.
Warning: If used inappropriately data loss will occur and
 will result in having to enter transactions once again into the system. These is no way to
 reverse this update once it is performed.
