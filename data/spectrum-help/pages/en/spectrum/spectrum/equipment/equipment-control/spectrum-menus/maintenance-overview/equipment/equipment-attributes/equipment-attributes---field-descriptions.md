# Equipment Attributes - Field Descriptions

Use the fields described in the table below when creating new or editing existing equipment attributes.
Fields
Descriptions

Attribute
Enter an equipment attribute for this piece of equipment or select an attribute from the available drop-down list; for example, a blower or a grader. Entry in this field is required.

- If the Assign attribute codes for this equipment typecheckbox
 is selected in the Equipment Type File Maintenance screen, then the
 software will only allow you to enter an attribute code that is
 authorized for this equipment type (and the Search Attributes window
 (F4) will
 only display codes that are assigned to this type).

- If the
 Assign attribute codes for this equipment type checkbox
 is NOT selected in the Equipment Type File Maintenance screen, then any
 valid attribute code may be selected and the Search Attributes window
 (F4) will
 show all codes set up in the current company.

Comments
Enter any additional comments about this equipment attribute. Entry in this field is optional.
