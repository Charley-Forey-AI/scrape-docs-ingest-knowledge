# Change Equipment Codes - Field Descriptions

Use this table for reference when completing the fields on this screen.
Buttons
Descriptions

New
Click this button to display the Assign New Equipment
 Code window and complete the following fields:

- Old equipment code: Enter the existing equipment code you want to change, or select from a list of available equipment codes.

- New equipment code:
 Enter the new code for the piece of equipment entered in the
 Old equipment code field.

Listing
Click this button to display the Equipment Code Change
 Listing screen. Use this screen to print a listing of the old
 and new equipment codes. [Equipment Code Change Listing](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/utilities-overview/change-equipment-codes/equipment-code-change-listing)

Update
Click this button to display the Change Equipment Codes
 Update screen. Use this screen to update one or more
 equipment code changes entered in the Change Equipment
 Codes screen. Select Continue.
