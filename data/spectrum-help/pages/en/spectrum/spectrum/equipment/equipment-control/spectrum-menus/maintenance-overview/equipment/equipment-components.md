# Equipment Components

Use this screen to create and maintain a list of standard
 components.
A component should be thought of as an equipment part. A component could
 not operate by itself, but it might be critical to the operation of a piece of equipment,
 (for example, an engine, power axle, transmission, or loader fork). Components should be
 pre-defined by the equipment manager in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen.
When a new piece of equipment is added, components will be added
 automatically when the equipment is saved based on the equipment type assigned to the new
 code. if the Assign component codes for this
 equipment type checkbox is selected in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) screen for the
 assigned equipment type, then any standard components for that type are flagged to
 Add this component automatically to new
 equipment will be added too.
After a component list is established, these codes are available for
 warranty tracking and statistical tests and readings.

- Click New to display the New Equipment Component window and create a new
 code.

- Click Build to display the [Build Standard Components for Type](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-components/build-standard-components-for-type) window.


- Click Status to determine which types of records will display in the list
 box.
