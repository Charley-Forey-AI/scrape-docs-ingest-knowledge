# Make File Maintenance

Use this screen to set up and maintain equipment makes,
 component makes, and tire makes while associating the codes with the appropriate equipment
 types.
Once this list is established, the make codes are also available for
 selection in the[Equipment Main Properties](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-main-properties) tab,
 for convenience when entering new equipment.
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.

- Click New to display the New Make window and add new records.

- Select a record and click Edit to open the Edit Make window and make changes to the specified record.

- Click Build to open the [Build Standard Makes and Models](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/utilities-overview/build-standard-makes-and-models) screen. This utility allows you to create records in Make File Maintenance and Model File Maintenance from equipment, components, and tires set up in Equipment File Maintenance and Component File Maintenance.
