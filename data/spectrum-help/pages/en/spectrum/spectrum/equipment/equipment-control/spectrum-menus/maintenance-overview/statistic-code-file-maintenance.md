# Statistic Code File Maintenance

This allows you to track statistical information pertaining
 to a specific piece of equipment, as of a certain date.
Use this screen to create and maintain a list of statistical codes for
 use in [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment), [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance), and Statistics
 Entry. For example, after a tire inspection, you might record that as of 11/12/11, the tire
 pressure on a dump truck was within the default psi measurements, and the tread wear was at
 50%. Statistical measurements set up in this screen are assigned a pre-defined result
 format (for example, Pass/Fail, Date, or Percentage), and this screen can be used to record
 an unlimited number of test results (which can be sorted in a variety of ways by clicking
 on any of the column headers).
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.

- Click New to display the New Statistic window and add new statistics.

- Select a record and click Edit to open the Edit Statistics window and make changes to the specified record.
