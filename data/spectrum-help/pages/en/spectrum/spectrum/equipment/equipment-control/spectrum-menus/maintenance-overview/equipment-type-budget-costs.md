# Equipment Type Budget Costs

Use this screen to assemble budget costs by equipment type
 and weight class in order to record annual estimated ownership costs.
Record Depreciation, License, Insurance, Property taxes, Interest and Other costs; the
 'target annual working hours' field is used to compute a cost/hour for each of these
 ownership costs.You can also record estimated costs/hour by cost category in order to
 develop an estimated 'operating cost/hour' for that type of equipment.
When you first open this screen, the list box shows one row for every equipment type with a weight class. This includes a row for each type with a blank weight class only when there are no weight classes set up for that type.
Click the Update button to open the [Update Charge Rates from Budget Cost](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-budget-costs/update-charge-rates-from-budget-cost) screen. This screen allows you to store the hourly budget values from the new Budget Cost maintenance screen into equipment codes assigned to the particular equipment type and weight class.
