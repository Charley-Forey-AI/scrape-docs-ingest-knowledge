# Fuel/Oil File Maintenance

The Fuel/Oil File Maintenance screen is used to define fuel
 and oil codes, which include diesel gasoline, unleaded, regular, super-unleaded, or oil with
 various weights.
Each oil or fuel type can then be referred to the Equipment.
This screen is used to record the current purchase prices for fuel and oil. The information will be used when doing fuel entries or transaction entries in Transaction Entry.
