# Quick Equipment Setup Maintenance

The Quick Equipment Setup Maintenance screen is used to
 copy equipment information from existing files into new files, speeding the information entry
 process.
This screen enables you to set up a standard piece of equipment (for
 example, dozers) and then have an equipment group that references that standard setup.
 Additionally, this screen allows you to also define which fields will be copied from
 Equipment (for example, model, type, filters, and so forth) and the equipment code mask
 with the next sequential number so you will have a numbering scheme based on an equipment
 group (for example, DZ001 for the first dozer).
To use the Quick Equipment Setup Maintenance feature when adding new
 equipment, select the Use quick equipment
 setup checkbox in the [New Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/quick-equipment-setup-maintenance/new-equipment) window. The equipment groups will display, and you can then
 select which group is being added. The system will fill in the rest of the information
 automatically, and you are given the option to copy all notes from the source equipment to
 the new equipment.

- Click New to display the [New/Edit Quick Equipment Setup - General](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/quick-equipment-setup-maintenance/newedit-quick-equipment-setup---general) and
 create a new code.

- Select a record and click Edit to display the [New/Edit Quick Equipment Setup - General](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/quick-equipment-setup-maintenance/newedit-quick-equipment-setup---general)make
 changes to an existing code.
