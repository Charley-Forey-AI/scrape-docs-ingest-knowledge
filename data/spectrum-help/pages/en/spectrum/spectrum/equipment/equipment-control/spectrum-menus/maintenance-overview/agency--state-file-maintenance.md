# Agency / State File Maintenance

Use this screen to manage a list of government agencies
 that issue licenses, permits, inspections, and more.
For use in the [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) and
 [Equipment Compliance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment/equipment-compliance)
 screens. Agency / State codes can also be used for selection purposes in some of the
 inquiry and report screens.
 If an e-mail address has been entered for the selected agency / state code, click the convenient E-mail button to write and send a message to the designated contact person (for example, if you have a question about your permit approval).
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.

- Click New to display the New Agency / State window and add new records.

- Select a record and click Edit to open the Edit Agency / State window and make changes to the specified record.
