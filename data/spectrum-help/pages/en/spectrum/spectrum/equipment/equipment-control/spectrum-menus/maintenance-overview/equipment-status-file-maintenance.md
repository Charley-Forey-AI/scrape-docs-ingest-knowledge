# Equipment Status File Maintenance

The Equipment Status File Maintenance screen is used to
 record and view status file codes for equipment tracking purposes.
At a minimum, it is recommended that Active, Inactive, and Retired status file codes be set
 up. Additional status codes may be created for inactive type: Sold, Outside Repairs, Shop,
 and Lost. Each new type of status would still be flagged as either Active, Inactive, or
 Retired. These codes are used extensively in the Equipment Tracking module.
