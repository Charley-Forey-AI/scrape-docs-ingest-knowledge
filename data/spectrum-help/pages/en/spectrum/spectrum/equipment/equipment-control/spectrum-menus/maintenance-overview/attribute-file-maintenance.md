# Attribute File Maintenance

These attributes are also useful for selection purposes on
 inquiries and reports.
Use this screen to create and maintain a list of standard attributes for
 use in the [Equipment Type File Maintenance](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment-type-file-maintenance) and
 [Equipment](/en/spectrum/spectrum/equipment/equipment-control/spectrum-menus/maintenance-overview/equipment) screens.
If you only have lookup only security permission for this screen, the Edit, New and Delete buttons will be disabled.

- Click New to display the New Attribute window and add new attributes. See field descriptions below.

- Select a record and click Edit to open the Edit Attribute window and make changes to the specified record. See field descriptions below.
