# Equipment Specifications - Field Descriptions

Use this table for reference when completing the fields on this screen.

Model
Enter the year, make, and model of the engine currently installed on this piece of equipment. At the Make and Model fields, you can use the convenient drop-down lists to select from available codes as set up in the Make File Maintenance screen and the Model File Maintenance screen, respectively. Entries in these fields may be up to 20 characters long.

Entries in these fields may be up to 20 characters long.
