# Recurring Transaction Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Batch frequency
Enter a specific frequency code, or press Enter to print ALL batch frequencies. A lookup window is available to view valid frequency codes.

Batch code
Enter the batch code, or press Enter to accept the defaulting operator code.

Transaction date
Enter a transaction date to print, or press Enter to accept the system default.

Equipment usage? Equipment stand-by?
Select the transaction source code: Equipment usage or Equipment standby, or press Enter to accept the system default of ALL.

Sort by

Recurring transaction status
By default, only Active recurring transactions are included on this report. You may select to also include Inactive recurring transactions, or you can deselect the Active checkbox and only include Inactive recurring transactions. (At least one checkbox must be selected.)
Only Active recurring transactions can be updated.
