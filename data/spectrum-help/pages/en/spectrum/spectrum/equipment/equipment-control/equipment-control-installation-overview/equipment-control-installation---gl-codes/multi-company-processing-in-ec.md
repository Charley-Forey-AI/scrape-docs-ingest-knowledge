# Multi-company processing in EC

Spectrum provides the ability to track the accounting
 records of multiple companies.
The multi-company feature allows a business to record
 revenue and expenses in separate companies, while combining that information on a single
 report whenever financial statements are printed. This feature also allows one company to
 issue checks while the expense for those checks is charged to a separate company. When a business decides to use the Spectrum multi-company feature,
 the following setup is required:

1. Create the desired companies

1. Complete the Company Installation screen - once for each company

1. Complete the installation screens for all modules present on the system; note that this will need to be done multiple times - once for each company

1. Identical G/L account code masks and financial calendars must be set up for all companies.

1. The multi-company checkbox should be selected for Equipment Control, and an inter-company equipment account code designated.

1. Set up the chart of accounts in every company created. Be sure to include the inter-company accounts specified on the installation screens when setting up the chart of accounts. Don't forget that it is possible to copy the chart of accounts from one company to another, easing file maintenance in the General Ledger.

1.  Set up equipment in every company that will record the equipment as a capital item. When using the multi-company feature, it may be necessary to enter a single piece of equipment in more than one company.
