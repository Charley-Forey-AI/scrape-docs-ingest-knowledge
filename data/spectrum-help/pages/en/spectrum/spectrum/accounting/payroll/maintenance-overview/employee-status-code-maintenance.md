# Employee Status Code Maintenance

The Employee Status Code Maintenance screen is used to edit
 existing status codes or to add additional status codes.
The status code must be one character long, and the corresponding description can be up to
 ten characters long.
Fields
Descriptions

Status code
Enter the status code you want to edit or double-click on this field select an existing code from the Search Status Codes window.

Descriptions
Edit the status code description or, if adding a new code, enter a new status code description.

Status Type
Select the status type you want to associate with the status code.
