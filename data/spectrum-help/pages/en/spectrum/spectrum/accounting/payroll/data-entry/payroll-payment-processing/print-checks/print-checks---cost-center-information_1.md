# Print Checks - Cost Center Information

If cost centers are being used in this company, Spectrum verifies that you have authorization for all cost centers included in the current payroll cycle; if not, then the checks will not be printed.
If your scheme includes override settings for employee entries in Cost Center Scheme Maintenance, Spectrum validates the cost centers assigned to the time cards and employee checks based on these overrides. The override cost center(s) supersede the cost center list defined for the scheme in general.
The Check Print starting screen also has from / to selections by cost center – these are only available when reprinting. These cost centers will select on the employee's assigned cost center, and only include checks for employees with cost centers within the entered range.
