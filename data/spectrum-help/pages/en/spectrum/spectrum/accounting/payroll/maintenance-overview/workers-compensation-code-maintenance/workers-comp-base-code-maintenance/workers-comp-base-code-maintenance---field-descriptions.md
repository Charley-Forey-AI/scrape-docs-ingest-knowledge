# Worker's Comp Base Code Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Base code
Enter the worker's compensation code; this base code will be used in conjunction with a state code in [Worker's Compensation Code Maintenance](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/workers-compensation-code-maintenance).
Note: Because many users prefer alphabetic or combination alpha
 and numeric coding, this code is not a numeric-only field. Because of this,
 the code left-justifies when entered. Users preferring a strictly numeric
 coding scheme should be advised to use leading zeros when adding codes in
 order to produce desired sort results on screens and reports. Without
 leading zeros, 1, 10, and 100 will appear before 2, 20, and 200. Instead,
 codes should be set up 001, 002, and so forth.

Descriptions
Enter the worker's compensation class code description.
