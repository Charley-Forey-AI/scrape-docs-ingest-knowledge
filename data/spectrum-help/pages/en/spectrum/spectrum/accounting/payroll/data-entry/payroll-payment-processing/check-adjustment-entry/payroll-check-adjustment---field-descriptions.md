# Payroll Check Adjustment - Field Descriptions

Use the table below for reference when completing the fields on the Payroll Check Adjustment or Payroll Cheque Adjustment screen.
To open this screen, go to Payroll > Data Entry > Payment Processing > Check/Cheque Adjustments.

Fields/ButtonsDescriptions
Employee codeEnter the employee code.
Check sequencePress Enter to accept the default sequence number, or enter the desired sequence number.
The first check during any check cycle is sequence number one. If additional checks are written to an individual, the second check is sequence #2, the third is sequence #3, and so forth. The check type displays to the right of this field.

Payment methodSelect the payment method (Auto deposit or Check) you want for this check adjustment. If the employee you have selected is not set up to receive auto-deposits, these options aren't available.
FICASelect to open the FICA window.
Use this window to refactor the employee's Social Security and Medicare withholdings or employer paid taxes.

CPP Select to open the CPP window.
Use this window to refactor the employee's Social insurance withholdings or employer paid taxes.

Income TaxSelect to open the Income Tax Detail window. Use this window to enter the state deductions for residence, work state, county, and local income tax.
Disability (SDI)Select to open the Disability Detail window. Use this window to enter the state deductions for residence, work state, county, and local disability insurance tax. State Disability Insurance (SDI) and Resident Worker's Compensation amounts will be listed separately on this report
Disability (EI) Employment Insurance (EI) amounts are listed on this report.
Deductions/Add-onsSelect to open the Deductions/Add-ons for Check window. Use this window to enter deduction and add-on amounts for this employee.
Check #The check number associated with the void or manual check.
HoursThe employee's hours display in an itemized list.
Other EarningsThe employee's other earnings, if any, display in an itemized list.
TaxesEnter the employee's federal income tax deduction, or press Enter to accept the system default.
The rest of the tax totals in this section are display-only. If changes are needed, use the buttons to open separate windows and make them.

DeductionsThe itemized deduction amounts display as entered in Time Card Entry, Union File Maintenance, Employee Recurring Deductions/Add-ons, and the Deductions window of this screen.
Add-onsThe itemized add-on amounts display as entered in Time Card Entry, Union File Maintenance, and the Employee Recurring Deductions/Add-ons screens.
TotalsThe total hours, earnings recorded for this pay check, other earnings (commissions, retro pay, etc), taxes deducted from this pay check, total deductions, add-ons, and total net check amount.
