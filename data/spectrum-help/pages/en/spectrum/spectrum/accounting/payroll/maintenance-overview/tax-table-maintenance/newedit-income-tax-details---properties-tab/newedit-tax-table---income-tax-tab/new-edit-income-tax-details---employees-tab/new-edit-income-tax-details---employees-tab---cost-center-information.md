# New Edit Income Tax Details - Employees tab - Cost Center Information

Please note that when cost centers are being used, it is
 possible that both the employee list and company list could appear empty, like the filing
 status is not in use.
However, you would still be disallowed from deleting the filing status because one or
 more active employees that are assigned to disallowed cost centers exist.
