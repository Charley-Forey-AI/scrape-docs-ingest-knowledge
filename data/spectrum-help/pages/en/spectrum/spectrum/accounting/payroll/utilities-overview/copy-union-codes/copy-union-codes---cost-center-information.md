# Copy Union Codes - Cost Center Information

If cost centers are being used in this company, when
 performing the Union File Copy update, Spectrum® will copy the union only if you have
 permission to access the union's G/L accounts.
Spectrum compares each account's list of shared cost centers with cost centers in your
 operator's assigned cost center scheme, and if there are no common cost centers, then
 the union is not copied. Likewise, Spectrum also verifies that you have permission to
 access the union's deduction/add-on codes and if not, the union is not copied.
