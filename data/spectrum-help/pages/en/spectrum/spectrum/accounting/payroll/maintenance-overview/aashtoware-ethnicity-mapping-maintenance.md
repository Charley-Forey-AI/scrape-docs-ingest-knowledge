# AASHTOWare Ethnicity Mapping Maintenance

Use this screen to maintain state codes that are used to track an employee's ethnicity when using Electronic Certified Payroll reporting provided by AASTHOWare.
When the AASHTOWare format is selected on the [Certified Payroll Report](/en/spectrum/spectrum/accounting/payroll/payroll-reports/certified-payroll-report), information entered on this screen will be exported to an XML file for processing.
Field
Description

Job state
Select a job state. This field is required.

Race
Select a race. This field is required.

State ethnicity code
Enter the ethnicity code for the selected race. Some states use a 1 character code, while others use 4. This field is required.
