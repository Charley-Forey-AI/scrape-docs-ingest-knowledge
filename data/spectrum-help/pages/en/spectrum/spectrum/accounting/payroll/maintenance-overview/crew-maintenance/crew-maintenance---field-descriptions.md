# Crew Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Crew number
Enter the crew number for the new crew.

Crew name
Enter the name of the crew.

Crew leader
Enter the employee code of the crew's leader.

Last activity
Enter the date on which the crew last had activity.

Status
Select the status of the crew: Active, Inactive, or Not used.
