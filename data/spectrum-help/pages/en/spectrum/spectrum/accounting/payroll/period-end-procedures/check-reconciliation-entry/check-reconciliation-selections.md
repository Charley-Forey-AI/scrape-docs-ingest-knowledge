# Check Reconciliation Selections

The software allows the user to enter the specific check
 numbers to be reconciled.
This lookup window allows the user to enter these numbers during the reconciliation
 process.
Fields
Descriptions

Check number
Enter the check number or use the SuperSelect options, or press Enter to accept the software default of ALL.

From transaction date
Enter the transaction date, or press Enter to leave blank and accept from the first date.

To transaction date
Enter the ending transaction date, or press Enter to accept the current payroll processing date default.

Transaction type
Enter a transaction type, or press Enter to accept the software default of ALL.
Transaction types are Regular, Manual, Void, Deposit, and Adjustment.

Options
Select the appropriate option to leave the checks unchanged, to auto-reconcile all selected checks, or to un-reconcile all selected checks.
