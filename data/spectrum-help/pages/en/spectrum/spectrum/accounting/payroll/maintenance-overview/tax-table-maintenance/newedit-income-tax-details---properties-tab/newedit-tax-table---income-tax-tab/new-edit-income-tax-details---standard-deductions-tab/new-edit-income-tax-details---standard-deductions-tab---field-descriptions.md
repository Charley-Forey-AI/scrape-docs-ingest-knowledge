# New Edit Income Tax Details - Standard Deductions tab - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Standard deductions
Select a standard deduction type. The additional fields that display will vary based on your selection in this field.

Annual deduction amount
This field displays when one of the 'Amount' options is selected as the standard deduction type. Enter the annual standard deduction amount.

Deduction percentage
Enter the standard deduction percentage rate by which the calculation will be figured.

Minimum annual deduction
Enter the minimum standard deduction. The system calculates the standard deduction based on the percentage entered. That amount is compared to the minimum standard deduction entered here. If the standard deduction computed by the system is less than the minimum standard deduction, the minimum standard deduction is used for the deduction.

Maximum annual deduction
Enter the maximum standard deduction. The standard deduction calculated by the system is compared to the maximum standard deduction entered. If the standard deduction computed by the system is greater than the maximum standard deduction, the maximum standard deduction is used for the calculation. If the maximum is zero, no computation is done.

Deduction adjustment amount
This applies to only a few state taxing authorities. If it applies to the state being entered, enter the deduction adjustment amount. If it does not apply, leave this field blank.

Apply deduction adjustment
Use the available drop-down list to select when the deduction adjustment should be applied (before or after calculating).

Variable deduction code
Use the available drop-down list to select a tax adjustment code for the variable deduction amount. Only tax adjustment codes designated as standard deductions are available for selection.
