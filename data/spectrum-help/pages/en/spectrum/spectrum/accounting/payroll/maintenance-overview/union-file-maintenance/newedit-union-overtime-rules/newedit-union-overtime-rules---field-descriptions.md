# New/Edit Union Overtime Rules - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Auto-overtime for this union?
Select this checkbox to set up union-specific auto-overtime rules. Once this checkbox is selected, the other fields on this screen will be available.
In addition, if this checkbox is selected, the software displays a message letting you know that overrides set up in this page will supersede Federal and State overtimes rules.

This union overrides job rules?
This checkbox defaults to selected and the overtime hierarchy is:
 Union>Job>State>Federal.
When this checkbox is clear, the overtime hierarchy is:
Job>Union>State>Federal.

Daily overtime starts at
Specify the number of hours (per day) after which point overtime will be calculated. This rate is applied during Auto-Overtime Updates if this union code is assigned to the time card line.

Weekly overtime starts at
Specify the number of hours (per week) after which point overtime will be calculated. This rate is applied during Auto-Overtime Updates if this union code is assigned to the time card line.

Saturdays
Use the available drop-down lists to select how to apply the Daily and Weekly thresholds. Options include setting the rules to the same rules used by weekdays, setting all Saturday and Sunday hours to overtime, or setting all Saturday and Sunday hours to double-time.
