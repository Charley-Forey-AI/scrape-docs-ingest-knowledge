# Void an Auto Deposit

When the Payroll Administrator needs to void out an Auto
 Deposit Check, there are two options.
The best practice is to use the [Replacement Check Entry](/en/spectrum/spectrum/accounting/payroll/data-entry/replacement-check-entry) feature. Not only will this "void" out the transaction, but you also have the option to then print a manual check that you can issue to the employee.
In the event that the transaction needs to be voided and reversed, void the auto deposit check in Pre-Time Card or Time Card Entry.
Either way, you don't need to make any changes to Cash Management.
