# Add on Report - Cost Center Information

When cost center entities are enabled in Enterprise
 Installation, the entity name will display on the report in place of the company name when an
 entity is stored for the check cost center of this pay cycle.
Moreover, the 'Limit' and 'Balance' values will be entity-specific.
