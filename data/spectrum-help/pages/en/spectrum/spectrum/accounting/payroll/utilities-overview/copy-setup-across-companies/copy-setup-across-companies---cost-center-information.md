# Copy Setup Across Companies - Cost Center Information

If cost centers are being used in the destination company,
 then when performing the Copy Setup Across Companies update, Spectrum will copy the tax table
 only if you have permission to access the tax table's G/L accounts in that other company.
Spectrum compares each account's list of shared cost centers with cost centers in your
 operator's assigned cost center scheme in the destination company, and if there are no
 common cost centers, then the tax table is not copied.
