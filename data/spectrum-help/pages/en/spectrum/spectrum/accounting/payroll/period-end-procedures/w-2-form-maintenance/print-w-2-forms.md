# Print W-2 Forms

Use the Print W-2 Forms screen to print W-2s as part of year-end processing.
Perform the [Year End Update](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/year-end-update).
If W-2s need to be grouped by state when printing, use the tax drop-down list to select the applicable state.
Tip:

- Before you load W-2 forms into the printer, print a few of them on plain paper to confirm the correct printer destination and alignment.

- Export your final W-2 print to .PDF format and save it in a read-only, year-specific folder on your server or workstation. This will allow you to reprint the W-2 forms in the future, if necessary.

This screen is compatible for contractors who use custom forms. When this screen is linked to a custom form, a new icon will appear in the lower left corner of the screen to indicate the presence of the custom form. If no custom form is set up for a particular form, then the icon will not appear on the starting screen.
