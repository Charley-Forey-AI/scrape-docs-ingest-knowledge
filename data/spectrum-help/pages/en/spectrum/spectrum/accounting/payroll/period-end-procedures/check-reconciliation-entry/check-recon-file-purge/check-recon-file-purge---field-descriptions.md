# Check Recon File Purge - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Purge Date
Choose the type of date from which the purge should select: Check date or Statement date.

Through transaction date
Enter the date through which to purge, or press F4 or double-click on this field to display the Date Change window. All reconciled items on file up to and including this date will be removed from the software.

Proceed with update?
Click Continue to proceed directly to the update or click Cancel to cancel and exit the screen.
