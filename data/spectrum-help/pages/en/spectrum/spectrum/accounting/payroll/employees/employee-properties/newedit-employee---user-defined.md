# New/Edit Employee - User-Defined

This screen displays any applicable user-defined fields.
If no fields have been set, an error message displays reminding you to set the fields
 first.
Note: Click the Switch button at any time to display a
 different employee.
Fields/Buttons
Descriptions

Prompt
The user-defined field's title
 displays. No entry is allowed. Click the column header to
 arrange the data by ascending order; click the column header again to
 arrange the text in descending order. Use your mouse to determine the width
 of this column by sliding the column title's side borders.

Current setting
The current setting for each
 user-defined field displays. No entry is allowed. If the Current Setting field
 is blank, then that user-defined field is not in use for the current
 employee.Click the column header to arrange the data
 by ascending order; click the column header again to arrange the text in
 descending order. Use your mouse to determine the width of this column by
 sliding the column title's side borders.
