# Voiding a Paycheck

Once a paycheck has been entered, it may be necessary to void it if mistakes were made.
The void process automatically records time card lines for voided checks by retrieving the time card history of the check being voided.
Note: You cannot void an auto deposit paycheck using the Time Card Entry screen. If you need to void an auto deposit paycheck, it can only be processed through the Payroll > Data Entry > Replacement Check screen.
To void a non-auto deposit paycheck:

1. On the Site Map screen, go to Payroll  >  Data Entry  >  Payment Processing.

1. Select the Time Card Entry button.Note: If the Time Card Entry button is disabled, you must set a new payroll cycle using the Set New Cycle button. Once a new cycle has been set, you can enter and void paychecks as necessary until payroll checks are once again printed and the cycle is updated.

1. Enter the employee code for the paycheck you are voiding. If you do not know this code, double-click in the field to select from a list of valid employee codes.

1. Press Enter.

1. At the Check sequence field, press Enter.

1. At the Check type field, use the drop-down list to select the Void option, and then press Enter.

1. At the Check # field, type the check number for the Payroll check you are voiding, and then press Enter.Note: If you do not know the check number you are voiding, review the Payroll inquiry for the employee to locate the check number. If you attempt to void a check that has a check date from a previous year, there may be payroll tax ramifications.
The associated line transaction information displays.

1. Select Save to complete the process.

1. On the Payroll Payment Processing menu, select Check Calculation.

1. Select Proceed with update and then OK.Important: Perform the Check Print function of this cycle, even if the voided item is the only entry in the cycle.
