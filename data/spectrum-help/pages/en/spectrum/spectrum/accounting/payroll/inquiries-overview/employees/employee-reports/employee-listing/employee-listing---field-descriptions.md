# Employee Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Employee
Press Enter to obtain a report containing ALL employees. Otherwise, enter any combination of employee codes that the report is to include.

Supervisor
Press Enter to obtain a report containing employees of ALL supervisor codes. Otherwise, enter the supervisor code that the report is to include.

Department
Press Enter to obtain a report containing employees of ALL departments. Otherwise, enter the department code that the report is to include.

Employee status
Press Enter to obtain a report containing ALL employee status types. Otherwise, enter any combination of status codes that the report is to include.

Include employee notes?
Select this checkbox to print employee notes, or leave the checkbox clear to omit remarks from the report. An unlimited number of notes can be printed on the listing.
Note: If any notes have been entered for this employee in the
 Human Resources module, they will be merged with any notes entered for the
 employee in Payroll.

Sort by
To print the report in Alphabetical order, press Enter, or select to print the report in Employee Code order.

Pay type
Press Enter to obtain a report containing employees of ALL pay types. Otherwise, select any combination of pay types that the report is to include: Commission, Hourly, Overtime salary, and Salary.

Report format
Press Enter to print the report in Detail, or select to print the report in Summary format.

Pay frequency
Press Enter to obtain a report containing employees of ALL pay frequencies. Otherwise, select any combination of pay frequencies that the report is to include: Daily, Weekly, Bi-weekly, Semi-monthly, Monthly, Quarterly, or Annually.
