# Non-Union Pay Group Properties - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Pay group
The pay group code entered in the main screen defaults.

Descriptions
The description automatically displays based on the code entered. Changes are allowed.

Burden
Enter the burden percentage you want to assign to this pay group. This burden
 percentage is used instead of the job percentage when the job is set up to
 NOT use the actual payroll (in the Job File Maintenance > Properties > Burden window)
The burden amount refers to the employer's additional payroll costs (FICA, FUTA, SUTA, worker's compensation costs and other burden components).
For more information on pay groups burden hierarchy, click [Pay Groups Burden Hierarchy](/en/spectrum/spectrum/accounting/payroll/default-hierarchies-overview/pay-groups-burden-hierarchy).

Minimum pay
Enter the minimum pay percentage you want to assign to this pay group.

Default pay rates and codes

Level and Title
The software automatically generates the level ID numbers and you assign a title to these levels.

Regular
Overtime
Double time
Billing code
Enter the various pay rates for each pay level, or press Enter to accept the default information.

Billing code
Enter the time and material billing code for each pay level. This is only used if the Time + Material module is installed and a Time + Material job is used.
