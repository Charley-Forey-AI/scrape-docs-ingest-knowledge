# P/R Processing Date Change

P/R Processing Date Change is used to change the date under
 which this module operates.
If the desired processing date is outside the Minimum and Maximum processing dates
 previously set, refer to the Processing Date Change discussion in the User's Guide.
Once you exit this screen, the effective rates update runs automatically.
