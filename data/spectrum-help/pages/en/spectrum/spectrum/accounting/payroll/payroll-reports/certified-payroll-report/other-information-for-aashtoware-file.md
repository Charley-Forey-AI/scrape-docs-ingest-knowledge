# Other Information for AASHTOWare File

When creating the AASHTOWare format, on the Certified
 Payroll Report starting screen, enter the 'Job number', 'Pay period' and 'Work week 'ending
 date plus the fields in this window.
All other start screen fields are ignored.
For more information, see [Certified Payroll Report](/en/spectrum/spectrum/accounting/payroll/payroll-reports/certified-payroll-report).
Fields
Descriptions

Contract ID
Enter your Contractor's Vendor ID number from the state.

License type
Leave <BLANK> as this field is not used with the AASHTOWare format.

License #
Leave <BLANK> as this field is not used with the AASHTOWare format.

Insurance #
Leave <BLANK> as this field is not used with the AASHTOWare format.

Contracting agency
Leave <BLANK> as this field is not used with the AASHTOWare format.

Enter the email address of the contact at your company.
