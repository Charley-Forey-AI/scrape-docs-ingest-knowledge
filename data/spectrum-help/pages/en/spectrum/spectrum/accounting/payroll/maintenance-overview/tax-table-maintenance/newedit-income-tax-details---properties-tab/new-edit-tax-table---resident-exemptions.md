# New Edit Tax Table - Resident Exemptions

This window is used to record tax exemptions for resident
 and work state, county, and local municipalities.
Exemption settings in this window are shared among all Spectrum companies.

- Tax entries set up in the Employees screen at the employee level override entries in this screen.

- The Payroll Check Calculation screen looks to this screen to determine if the resident location should be taxed versus the work location. The temporary work state, county, and local exemption defaults will also print on the Tax Table Listing.

- Click the New or Edit buttons to display the Residents Exemptions window. Field descriptions are provided in the table below.

- This tab will not display if the tax table is set to 'US'.
