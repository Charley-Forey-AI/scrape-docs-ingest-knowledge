# Procedures As Required

Follow the steps below for procedures as required in Payroll.

- As new employees are added, or as existing employee information changes, use Employees to make any changes or additions.

- If deduction plans change, or new ones are added, use Deduction / Add-on Code Maintenance to make changes or additions.

- Make changes or additions to department information using Department Expense Maintenance.

- Keep supervisor information up-to-date using Supervisor Code > Maintenance.

- Make required changes to Worker's Compensation > Code Maintenance.

- Vacation, Holiday, and Sick Leave information should be kept current using Vac, Hol, and Sick Control Maintenance.

- Run the Seniority Report as required.

- Run the New Hire/Terminated Report as required.

- Run the Birthday List as required.

- Periodically, according to the needs of your company, use Processing Date Change to reset minimum and maximum processing dates.
