# Save Auto Deposit Files

The controls for saving / downloading the auto deposit file
 are in the browser settings.
For example, in the screen shot below you see that the downloads are set to 'always
 prompt'. The 'save as' dialog box will always appear in the folder defined in the folder
 listed as the 'save file to' setting. In the screen shot below the folder is set to TEMP.
 The default location for downloads is a folder called DOWNLOADS.

- Google Chrome: Customize / Controls button > Settings > Show Advanced Settings > Downloads

- Fire Fox Browser Setting: Tools > Options > General Tab > Downloads
