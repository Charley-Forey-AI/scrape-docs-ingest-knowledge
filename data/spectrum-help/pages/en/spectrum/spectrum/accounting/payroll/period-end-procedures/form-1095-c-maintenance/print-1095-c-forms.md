# Print 1095-C Forms

Use this report to print 1095-C forms for year-end filing in the United States.
Note: Set your Operator Preferences (accessed from the Preferences
 button) for the number of records to display on a page - otherwise you may receive an alert
 if the list is too large to display.
FieldDescription
Employee codeEnter an employee code to include on the report, or leave this field blank to include ALL.
EntityIf Entity tracking is enabled for the current company, enter the entity company to print, or leave this field blank for the main company entity.
YearEnter the year to print.
Sort bySelect how to sort the report results:

- Employee alpha

- Social security number
