# Payroll Trial Balance Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you can enter a cost center entity here in order to display entity-specific balance information. Leave this field blank to use the main company entity.

Employee code
Enter an employee code or double-click on this field to display the associated Search Employees window.

Report type
Select Exceptions only to print an exceptions report, or click All employees to print a complete report.

Pay type
Select the checkboxes for the employee pay types that you want to include on this report: Hourly, Salary, Overtime salary, or Commission.

Status
Select the checkboxes for the employee status types that you want to include on this report. Options include: Active, Inactive, Terminated, or Deceased.

Frequency
Select the checkboxes for the employee pay frequencies that you want to include on this report: Daily, Weekly, Bi weekly, Semi monthly, Monthly, Quarterly, or Annually.
