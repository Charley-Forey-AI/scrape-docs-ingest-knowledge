# Employer Pension Contribution Report

Use this report to show total gross wages and hours for each non-cash Add-on Code included on the report (as selected using the "Pension Codes" button).
The leftmost set of figures is for all jobs, including but not limited to Davis-Bacon and Prevailing Wage jobs. The Adjustment column is relevant in cases where the gross amount used to calculate the Add-on is subject to adjustment (as set in Deduction / Add-on Code Maintenance). Four versions of this report are available, including Add-on Detail, Add-on Summary, Employee Detail, and Employee Summary. View report display hierarchy
This report displays results in the following order:

1. Add-on code

1. Employee by add-on code,

1. Employee for total gross,

1. Gross adjustments (the adjustments are for non-Davis-Bacon or Prevailing wage),

1. Adjusted gross,

1. Total hours (regular, overtime, double time),

1. Total Contribution,

1. Davis-Bacon Gross,

1. Davis-Bacon hours (regular, overtime, double time),

1. Davis-Bacon Contribution,

1. Prevailing Wage Gross,

1. Prevailing Wage hours (regular, overtime, double time),

1. Prevailing Wage Contribution
