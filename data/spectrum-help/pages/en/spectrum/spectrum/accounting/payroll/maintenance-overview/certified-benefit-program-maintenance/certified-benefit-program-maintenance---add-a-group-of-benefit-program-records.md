# Certified Benefit Program Maintenance - Add a Group of Benefit Program Records

Follow the steps below to add a group or benefit program
 records.

1. Click the Build button to open the Build Program Types for Job State window.

1. Specify the Job state and File format you want to use:

- eCPR - this XML format is used in California

- AASHTOWare - this XML format is from the American Association of State Highway and Trasnportation Officals (AASHTO) and is used in Minnesota, Ohio and Virginia.

- LCPtracker - this format is for clients performing work for jurisdictions using LCPtracker Labor Compliance Software.
