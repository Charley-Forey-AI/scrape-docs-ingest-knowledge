# Worker's Compensation Rate Per $100 Report - Sample Report

The "Subject to earnings" column is designed to report
 earnings that have not exceeded the limit defined in the PR Maintenance Worker's
 Compensation Code.
Many states have limits to the worker's compensation earnings that are
 used to calculate the tax.
If your state does not have a limit, "Subject to" is the same as total
 earnings. The "Covered earnings" is the column to review (covered earnings * rate =
 premium).
