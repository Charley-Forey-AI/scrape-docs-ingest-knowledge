# Wage and Tax History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Employee
Tax code
Employee status
Enter an employee code, tax code, and employee status to include on the report, or press Enter to include ALL records.
Check dateEnter the first and last check dates to include on this report, or press Enter to begin with the first check date and end with the current Payroll processing date.
Period end dateEnter the first and last period end dates to include on this report, or press Enter to begin with the first period end date and end with the current Payroll processing date.
Sort byChoose whether to sort by Employee code or by Tax code.
Report formatChoose whether to print the report in Detail or Summary format.

Cost groupDisplays only if cost centers are enabled in the current company, and the field label may differ if you have a user-defined cost group label specified in Enterprise Installation.
Works as a filter that allows you to include only a portion of the records your operator has security for, thereby allowing reports to be produced for a segment of employees.
The application does NOT read for authorization of individual cost centers contained within the specified cost group to determine permission for the check cost center.
