# Retro Pay

The Automatic Retro Pay screen is used to specify the work
 dates that apply to the retro pay (RP) pay type. After selecting these dates, the software
 will automatically create new 'RP' pay lines for those time card history lines.
You can then apply the retro pay back to the original job and phase distribution.
If you prefer to enter this information manually, you can cancel out of
 this screen. Even if you use the automatic window to pay the retro pay for the last two pay
 periods, you can still use the standard RP pay type to enter in the last few days of work.

## Note regarding Related Taxes

If related tax codes have been set up for the underlying time card record, the related
 tax codes will automatically be created when the record is written from this screen. If
 the underlying time card record does not have related codes, the retro time card records
 will not either.
