# Job/Wage Code Summary Report - Cost Center Information

If cost centers are being used in this company, when printing
 the Job / Wage Code Summary Reports, Spectrum includes activity only if you have permission
 to access the job. Spectrum compares the cost center assigned to the job with cost centers
 in your operator's assigned cost center scheme, and if the cost center is not included, then
 the job will not be shown.
In addition, Spectrum assures that your operator has permission to
 view the corresponding phase information; otherwise the phase's activity is not shown.
 Lastly, Spectrum includes the employees in the report totals only if you have permission
 to access the employee.
