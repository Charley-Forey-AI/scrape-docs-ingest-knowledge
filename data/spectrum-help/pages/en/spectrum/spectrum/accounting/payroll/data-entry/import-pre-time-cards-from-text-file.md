# Import Pre-Time Cards from Text File

Use this screen to transfer time card information from
 comma-delimited text files into the pre-time card entry screen.
For more information, see [Pre-Time Card Entry](/en/spectrum/spectrum/accounting/payroll/data-entry/pre-time-card-entry).
Each import line is then validated and processed; it is important that
 the import lines use the proper codes or data may be compromised. The Import Pre-Time Cards
 from Text File screen sets the pay rate based on the union code, wage code, non-union pay
 group, pay rate level and Payroll Installation settings, using the same criteria as when
 time is added directly into the Pre-Time Card Entry screen.
