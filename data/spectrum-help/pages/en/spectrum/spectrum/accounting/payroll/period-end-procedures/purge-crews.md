# Purge Crews

This update will delete crew codes for the current company.
This is not a typical process and should only be used in extreme situations. These is no
 way to reverse this update once it is performed. It is not recommended that the user
 perform this purge more than once every couple of years.
Records in the following tables will be purged:

- Pre-Time Card History

- Time Card History

- Job Quantity Complete History

- Crew Maintenance
