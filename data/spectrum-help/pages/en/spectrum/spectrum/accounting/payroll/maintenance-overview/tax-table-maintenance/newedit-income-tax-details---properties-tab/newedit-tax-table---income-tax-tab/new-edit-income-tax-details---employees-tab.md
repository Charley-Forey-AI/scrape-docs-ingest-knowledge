# New Edit Income Tax Details - Employees tab

Use this window to view the list of employees in the
 current company who are currently assigned to the selected filing status and a list of the
 other companies in the system that have one or more employees who are currently assigned this
 filing status.
The employees may be assigned this status for either permanent or temporary work.
This tab will only display employees with the selected tax code as
 their main tax code. Employees with this tax code as a related tax code will not appear
 here.
