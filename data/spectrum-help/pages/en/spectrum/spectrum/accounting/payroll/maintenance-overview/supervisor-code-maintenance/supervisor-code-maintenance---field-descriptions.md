# Supervisor Code Maintenance - Field Descriptions

Use the table below as a reference when completing or editing fields in the Supervisor window.
Fields
Descriptions

Supervisor code

Name
Enter the name of the supervisor, or of the supervisor's position (for example, job supervisor, office manager, union construction, mason supervisor, and so forth).

Department
Enter the department code of this supervisor. Following entry of a valid department code, the department description will display.

Employee code
Enter a valid employee code.
