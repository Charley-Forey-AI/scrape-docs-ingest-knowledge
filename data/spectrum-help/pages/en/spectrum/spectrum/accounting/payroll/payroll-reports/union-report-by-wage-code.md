# Union Report by Wage Code

Use this report to view detailed information broken out by
 Union + Wage codes.
The report will list each employee's earnings, deductions, add-ons and fringe benefits,
 plus a summary of fringe benefits and payments for that Union + Wage code, essentially
 providing a series of sub union reports separating activity wage code by wage code. This
 report provides the ability to review union costs at a more granular level within the union
 reporting.
