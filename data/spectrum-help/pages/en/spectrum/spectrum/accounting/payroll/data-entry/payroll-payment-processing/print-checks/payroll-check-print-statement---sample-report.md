# Payroll Check Print (Statement) - Sample Report

Checks will show the entity name (from the Entity Master Table) in place of the 'Company name' when an 'Entity' code is stored for the 'Check cost center' of this pay cycle.
Note: If the "Display accumulated balance on paycheck?" option is
 selected in the [New/Edit Deduction/Add-on Code - Properties](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/deduction--add-on-code-maintenance/create-or-edit-a-deduction-or-add-on-code/newedit-deductionadd-on-code---properties) screen, the accumulated balance will display instead of the Year
 To Date" amounts. The column title on the form will still say "Year To Date."
Vacation and Sick time balances show the hours earned, not the hours used.
