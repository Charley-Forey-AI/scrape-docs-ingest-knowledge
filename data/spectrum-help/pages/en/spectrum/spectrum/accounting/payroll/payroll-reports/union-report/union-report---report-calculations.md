# Union Report - Report Calculations

Use the table below for reference when completing the
 fields on this screen.
Column Name
Calculations

Regular Hours
This column only includes pay types (R)egular, (O)vertime, and (D)ouble time.

Total Hours
This column is an hourly count that is affected by the fringe method in order to arrive at the fringe amount shown on the report. This column is NOT just a simple sum of the hours.
