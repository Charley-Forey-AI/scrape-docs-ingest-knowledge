# Build W-2 Forms - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Selections
EmployeeEnter the employee code, use the SuperSelect options, or double-click on this field to select from a list of available employee codes.
From/to check dateSpecify a check date range for the build.
Social security limitThe value will default from the current year's tax table. Press Enter to accept this amount for the W-2s. If your U.S. tax table has already been set to the next year's limits, enter the correct dollar amount.
Retirement planUse the drop-down menu to select the pension plan indicator.

- Y – Yes, check retirement on all W-2s If you select Y, Spectrum places an X in this field of the W-2 form for all employees.

- N – No, leave retirement blank on all If you select N, Spectrum leaves this field on the W-2 form blank for all employees.

- K – Check only for 401k participants If you select K, Spectrum places an X in this field for employees that have a non-zero 401(k) deduction total. Note: During the W-2 Form Build, if you select K – Check only for 401(k) participants, Spectrum selects the employee's W-2 Box 13 Retirement plan checkbox when either type D or type S amounts are recorded in Box 12 of the W-2.

Write 1 state and locality per W-2?Select the checkbox to write one state and locality per W-2 form, or press Enter to leave the checkbox clear and accept the system default which will write two states and localities per W-2 form. An advantage of having one state per form is the ability to print W-2s selectively by state. An advantage of having two states per form is that fewer forms are needed to produce W-2s.
Box 10 - Dependent Care
Deductions buttonSelect to display the Dependent Care Deductions window where you can enter a valid deduction code for dependent care or select an available code from the Payroll Deductions window. You may enter an unlimited number of lines here.
Add-ons buttonSelect to display the Dependent Care Add-ons window where you can enter a valid add-on code for dependent care or select an available code from the Payroll Add-ons window. You may enter an unlimited number of lines here.
Box 12 - Retirement
Alpha codesEnter the alpha code for the Box 12 field (for example, D for 401(k), C for group term life insurance in excess of $50,000, W for Affordable Care Act coverage amount, and so forth). Needs at your company may vary; please refer to IRS Box 12 filing instructions, state filing requirements and your CPA for specific uses relative to your operation.
If a type D or type S amount is recorded in this field and the K – Check only for 401(k) participants retirement option is selected in the Retirement plan field, Spectrum automatically selects the Box 13 checkbox on the W-2 Form Maintenance screen.
Note for Qualified Equity Grants

- Employers with employees who have qualified equity grants must report the amount included in gross income under section 83(i) for the calendar year using code GG.

- Employers must also report the aggregate amount of income which employees elect to defer under section 83(i) for the calendar year using code HH.

Note for Oregon Statewide Transit Tax:
The state of Oregon requires both the wages and the tax to be reported for each employee as part of the Statewide Transit Tax. To do this, enter code OR-STT in Box 14 and then link this code to an existing tax table in the Other Deductions window below by entering a plus sign (+) and the Oregon tax table code (for example, +OR).
