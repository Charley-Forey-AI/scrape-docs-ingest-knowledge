# File Layout Window

The file layout should be a comma delimited text file. For more information on file layout specifics, [File Layout Specifics](/en/spectrum/spectrum/accounting/payroll/data-entry/import-pre-time-cards-from-text-file/file-layout-specifics).
Note: If cost centers are being used, an optional cost center field will display on the File Layout list.
