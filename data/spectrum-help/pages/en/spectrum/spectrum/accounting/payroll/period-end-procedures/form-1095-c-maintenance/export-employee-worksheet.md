# Export Employee Worksheet

A list of all employees with current-year earnings from Payroll, including employees' dependents when coverage is provided by a self-insured employer.
FieldDescription
EntityIf Entity tracking is enabled for the current company, enter the entity company to export, or leave this field blank for the main company entity.
YearEnter the year to export.
Employer provided self-insured coverage?Select this checkbox if the employee is covered under a policy that is self-insured by the employer.
Health coverage statusSelect which coverage statuses to include in the export: Full time, Part time, Variable hour and/or Unassigned.
File nameEnter a file name for the exported file. Add a .csv extension to the file name to create the Employee Worksheet for processing in Nelco.
