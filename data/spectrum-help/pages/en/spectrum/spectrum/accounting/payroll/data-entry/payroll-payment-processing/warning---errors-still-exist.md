# Warning - Errors still exist

This dialog box displays if errors still exist in Pre-Time Card Import Errors. Click OK to continue without including the records that need to be corrected, or click Cancel to return to the main screen without saving if you wish to correct the errors before proceeding.
Employee error records
Click the View button to display the employee error records currently being held in the Pre-Time Card Import Errors screen.

Quantity error records
Click the View button to display the quantity error records currently being held in the Pre-Time Card Import Errors screen.
