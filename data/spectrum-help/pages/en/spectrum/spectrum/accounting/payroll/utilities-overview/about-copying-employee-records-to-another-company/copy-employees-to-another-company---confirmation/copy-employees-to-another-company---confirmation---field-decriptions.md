# Copy Employees to Another Company - Confirmation - Field Decriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Copy from company
The current company code displays, and the current company name defaults from the Company Installation screen. No changes are allowed.

Copy to company
The company code specified in the Copy Employees to Another Company starting screen and the corresponding company name default. No changes are allowed.

Include employee history?
Select this checkbox if you wish to include the employee earnings history, tax history, and time card history in the employee copy to the destination company .

Flag these employees for removal from this company following review?
Select this checkbox if you want to mark these employees for later removal from the current company. This checkbox determines whether or not these employees will display in the Purge Copied Employees starting screen.
 If employees are being copied to a test company, you should clear this checkbox to prevent employees from being deleted in error from this company. Also, if the employee history is not included in the copy to the destination company, then Spectrum will prevent the employee from being purged.
