# Tax Table Maintenance - Cost Center Information

If cost centers are being used in this company, then when
 adding or changing a tax table, Spectrum will allow entry of a G/L account code only if you
 have permission to assign that code.
Spectrum compares the G/L account's list of shared cost
 centers with cost centers in your operator's assigned cost center scheme, and if there are
 no common cost centers, then that G/L account cannot be assigned.Once
 a G/L account code has been assigned to a tax table (by an authorized operator), then
 that G/L account will be used by all operators for processing involving that tax table
 code.
