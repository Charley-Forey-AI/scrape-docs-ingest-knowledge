# P/R G/L Distribution Report - Cost Center Information

If the Enterprise Installation option for cost centers is set to Yes in this company and a cost group or cost center is specified, then the report shows only G/L activity assigned to that cost group/cost center, provided you have permission to access the specified cost center.
Likewise, Spectrum includes employee activity and G/L transaction activity on the report only if you have permission to access the employee cost center and the cost center assigned to the G/L entry during the Payroll Update. When a single cost group or cost center is specified on the starting screen, Spectrum will only include lines with activity for the selected cost center(s).
The G/L Distribution Report shows the cost center associated with each G/L transaction line on the report.
