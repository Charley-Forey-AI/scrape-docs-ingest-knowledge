# Delete Employee from Current Company

Use this screen to purge employee records from the current
 company.
Important: This utility flags the selected employee for
 deletion. This is not a typical process and should only be used in extreme situations.
 Please contact Viewpoint Support prior to usage.
After entering an employee code, click Continue to be taken to the Purge Copied Employees page. You will then be presented with the final option to purge the employee from the current company.
Employee records will be purged in the current company in the following screens:

- Employee Earnings History

- Wage and Tax History

- Worker's Compensation History

- Time Card History

- Employee Master

- HR Safety Incidents

- HR Drug, Alcohol and Other History

This screen will not be available if a payroll cycle is in progress. Do not purge employee records that have earnings in the current year and need to have a W-2 or T4 issued.
If this screen is used inappropriately, data loss will occur. There is no way to reverse this update once it is performed.
