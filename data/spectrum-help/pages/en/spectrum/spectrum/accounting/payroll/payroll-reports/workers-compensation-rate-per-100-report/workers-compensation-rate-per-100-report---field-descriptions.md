# Worker's Compensation Rate Per $100 Report - Field
 Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Worker's compensation code
Enter the worker's compensation code, or press Enter to include ALL
 records on the report.

State code
Enter the state code, or press Enter to include ALL
 records on the report.

Check date
Enter the first and last check dates to be included on
 this report, or leave these fields blank to start with the earliest date and
 end with the current Payroll processing date.

Sort by
Select to sort the report by State code or Worker's compensation
 code

Format
Select a format for the report: Detail or Summary

- The Detail format provides detailed information for
 each worker's compensation code.

- The Summary format is used for easier submittal to
 the state. This format will simply show totals by worker's
 compensation code and state, and will suppress the employee amounts
 and G/L distribution.

Check cost center

Cost group
This field only displays if cost centers are enabled in
 the current company, and the field label may differ if you have a
 user-defined cost group label specified in Enterprise Installation.
This field works as a filter that allows you to include
 only a portion of the records your operator has security for, thereby
 allowing reports to be produced for a segment of employees.
The software does NOT read for authorization of
 individual cost centers contained within the specified cost group to
 determine permission for the check cost center.
