# Purge Time Card History

 This update will delete employee time card history records
 for the current company.
This is not a typical process and should only be used in extreme situations. These is no
 way to reverse this update once it is performed.
Records in the following tables will be purged:

- Employee Time Card History
Selected time card history records are purged based on the parameters you enter in this screen. Before performing this purge, print a time card history report and keep it as a permanent record.
Important: Caution should always be used when performing
 purges.
