# Period End Clear

This function clears items from the payroll file for each
 employee's earnings for the period specified (month or quarter).
Quarter-end clear will clear both the quarter and month. Running the period end clear will
 update the employee totals in the Employee Financial Summary screen, so it's best to do
 this on a monthly basis to keep these totals up-to-date.
