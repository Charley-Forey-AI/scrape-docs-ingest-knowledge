# Deduction / Add-on Code Listing - Canadian lexicon information

If the Canadian lexicon designation (en-ca) is selected in
 Company Installation, some labels on this listing will display using the preferred Canadian
 spelling or reference.
For example, "Other" instead of "Unemployment" and "CPP" instead of
 "Fica," and more.
