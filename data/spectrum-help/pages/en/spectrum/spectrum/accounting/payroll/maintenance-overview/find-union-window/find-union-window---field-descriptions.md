# Find Union window - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Union code
Enter the union code to include in the search, or double-click on this field to select from a list of available union codes, or press Enter to search ALL union codes.

Descriptions
Enter the union description to include in the search, or press Enter to search ALL union descriptions.

Craft
Enter the union craft to include in the search, or press Enter to search ALL union crafts.

Deduction/add-on
Enter the deduction/add-on code to include in the search, or double-click on this field to select from a list of available codes using the Payroll Deductions window, or press Enter to leave this field blank.

Fringe
Enter the fringe code to include in the search, select a code from the drop-down list, or press Enter to leave this field blank.
