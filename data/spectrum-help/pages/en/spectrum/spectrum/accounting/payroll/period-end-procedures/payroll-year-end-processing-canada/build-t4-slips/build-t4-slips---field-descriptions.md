# Build T4 Slips - Field Descriptions

Reference the table below when completing the fields on this screen.
Fields
Descriptions

Employee
Enter the employee code, use the SuperSelect options, or double-click on this field to select from a list of available employee codes.

From/to cheque date
Specify a check date range for the update.

Default RPP number (box 50)
This number will be stored in Box 50 if Box 20 is non-zero. The Payroll Manager will then perform any necessary editing in T4 Slip Maintenance if multiple retirement plans are provided, or when modifying the retirement amounts to report.

Override button
Union contractors have multiple RPP numbers for their employees. Click this
 button to open the Override Registered Pension Plan (RPP) Number window and
 enter an override RPP for the union. During the build process, the system
 will use this override number on the T4 form. If no override exists, the
 employee's home union RPP will be used.

Add-ons and deductions to include on T4 Slip (Settings default from last update performed.)

Use these boxes to enter codes and amounts that relate to employment commissions, taxable allowances and benefits, deductible amounts and other entries if they apply.
The boxes are not pre-numbered as they are in the top part of the slip. Instead, the Payroll Administrator enters the codes and amounts that apply to the employee.
The software will calculate up to six Other Information codes for one employee's T4 slip. In the event that additional codes are needed, the Payroll Administrator will need to create an additional slip and add the appropriate information. This is expected to be a rare situation.
Important: This does not mean that only six codes can be defined
 on the Build start screen. The limit of six only
 applies to one employee's T4 Slip.
When building the T4 Slips, the Payroll Administrator will identify which add-on and deduction codes to include as well as which box they should be part of.
The following codes shall be assigned here:

- Box 44: Union Dues (deductions only)

- Box 46: Charitable Contributions (deductions only)

- Box 14: Registered Retirement Savings Plans (add-ons and deductions)

- Box 20: Pension Plan (deductions only). The Build T4 Update includes the total of box 20 in box 14 (Employment Income) for each form.

- Box 52: Pension Adjustment (add-ons and deductions)

- Other Information Code

Contact the Canadian Revenue Agency for more information on Other Information Codes.
In the event that there more than one code assigned to one box, the values will be summarized on the employee's T4 slip.
Select the appropriate checkbox to assign an add-on or deduction code to a specific box on the slip. Enter the code number when the add-on or deduction code relates to an "Other Information" code. In the example below, code 11 represents Other Information Code 40.
Employees who work in more than one province or territory will receive multiple T4 slips. See the Appendix at the end of this document for how certain items are allocated to each T4 slip.
