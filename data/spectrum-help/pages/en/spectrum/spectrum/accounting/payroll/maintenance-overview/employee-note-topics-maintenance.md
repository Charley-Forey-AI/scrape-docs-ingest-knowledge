# Employee Note Topics Maintenance

Use this screen to manage subcontract note topics.
Topics entered here will be available in the Employee Notes screen.

- No length restrictions

- Automatic wrapping

- Spell check

- Date stamping

[Learn more about Notes features](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/notes)

## Enter a New note topic

- Click the New button

- In the Topic name field, type a label for the topic. For example, "Safety."

- At the Shared Topic prompt, select to allow this topic in all companies or the current company only.

- Click OK to save your entry. The topic you entered will now display in the Notes screens throughout the software.

## Edit an existing note topic

- Select an existing note topic and click the Edit button.

- At the Shared Topic prompt, select to allow this topic in all companies or the current company only.

- Click OK to save your changes. The topic you entered will now display in the Notes screens throughout the software.
