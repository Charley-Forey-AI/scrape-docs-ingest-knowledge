# New / Edit Electronic Master Table

Use the New Electronic Master Table window to enter a new master table code and a list of tables that will comprise this master table or select an existing master table code and click the Edit button to make changes to the table list comprising this master table.
When new master tables are added to this list, they display at the bottom of the list, and the tables are run in the order in which they appear.
