# New/Edit Deduction/Add-on Code - Tax Effects - Field Descriptions

A reference for completing the fields on this screen.
Fields/ButtonsDescriptions
Tax codeEnter the state or federal income tax table code.
DescriptionEnter the full description for the tax code. For example, for tax code "US", an appropriate description might be "United States".
Income taxSelect if this deduction or add-on will change the amount of state or federal income tax.
EmployeeSelect if this deduction or add-on will change the amount of taxable wages (as reflected on FICA or SDI) for the employee.
EmployerSelect if this deduction or add-on will change the amount of taxable wages (as reflected on FICA or SDI) for the employer.
UnemploymentSelect if this deduction or add-on will change the amount of state or Federal, state, county or local unemployment taxes.
Worker's compSelect if this add-on will change the amount of worker's compensation.
When a worker's comp code is selected here, the corresponding worker's compensation 'covered earnings' will be adjusted (raised) when the add-on is recorded in the time card file and the rate per $100 computation will be performed during Check Calculation.
