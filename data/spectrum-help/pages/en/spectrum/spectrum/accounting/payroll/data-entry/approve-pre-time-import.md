# Approve Pre-Time Import

This screen allows you to approve time cards from SDX Web
 Services.
The screen displays all imported transactions, and offers options to 'Edit', 'Delete' and
 'Approve' selected time cards. You can also select to 'Approve All' entries. When approved,
 the selected time cards will be transferred to Pre-Time Card Entry for further processing.
The system will validate that related codes are not already entered as State, County or
 Local tax codes. Related codes cannot be entered directly on the time card record.

- Invalid codes will generate an error and move the pre-time card to the Pre-Time Card
 Import Errors screen.

- All records that are validated and approved will automatically attach related codes to
 the pre-time card.

Note: Records imported using the Import Pre-Time Cards from
 Text file feature are excluded from this screen. Instead, valid records will go directly to
 Pre-Time Card Entry, and records with errors go directly to Pre-Time Card Import
 Errors.
