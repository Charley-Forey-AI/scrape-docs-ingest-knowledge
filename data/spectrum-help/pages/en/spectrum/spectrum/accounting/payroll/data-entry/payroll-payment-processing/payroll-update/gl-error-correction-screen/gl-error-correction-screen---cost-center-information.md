# G/L Error Correction Screen - Cost Center Information

If cost centers are being used in this company, Spectrum
 validates each record before posting to General Ledger.
It will check that the cost center assigned to the G/L account for that line is
 valid.
This screen also allows the cost center to be modified. Spectrum validates that the cost center is valid for your operator, and that it is an allowed cost center for that G/L account. When changes are made directly in this screen, protection ensures that debits always equal credits.
