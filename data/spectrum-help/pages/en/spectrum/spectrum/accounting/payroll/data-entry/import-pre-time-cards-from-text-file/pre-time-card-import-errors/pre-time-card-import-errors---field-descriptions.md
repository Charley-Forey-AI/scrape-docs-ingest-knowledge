# Pre-Time Card Import Errors - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Batch code
Enter the batch code, or press Enter to accept the default of ALL.

Edit
Click this button to make corrections to the selected pre-time card line.

Delete
Click this button to delete the selected pre-time card
 line.

Delete AllClick this button to delete all pre-time card records with
 errors.
Recheck All
Click this button to move any valid records (including any recent changes) to Pre-Time Cards.
