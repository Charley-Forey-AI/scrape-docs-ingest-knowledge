# Manual Check Log Print - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Check
Enter a check number, ALL, or use the SuperSelect options.

From date
Enter the first date to print, or press F4 or double-click on this field to select a date from the Date Change window.

To date
Enter the last date to print, press Enter to select the default of the current Payroll processing date, or press F4 or double-click on this field to select a date from the Date Change window.
