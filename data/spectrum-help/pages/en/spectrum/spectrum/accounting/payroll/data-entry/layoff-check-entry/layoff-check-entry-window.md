# Layoff Check Entry Window

Use this window to add new layoff check lines or to edit existing layoff check lines.
As entries are completed in this window, they display in the columns on the Layoff Check Entry starting screen.
If errors occur during data entry, a warning message displays and a View Error Messages button appears on the lower right of the window. Select this button to open the Error For Current Record window and review the specific error message for the problematic field. To make corrections to the field, double-click the error message line or select OK to return to the Layoff Check Entry window and make your changes there.
A Meter button displays if you enter an equipment pay type and valid equipment code, and have security access to the Equipment Control > Meter Reading Transaction Entry screen. Select this button to open the [Meter Readings](/en/spectrum/spectrum/accounting/payroll/data-entry/layoff-check-entry/layoff-check-entry-window/meter-readings) window and enter meter readings for the specified equipment.
