# Certified Payroll Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Selections
Job numberEnter the job number to print, or press Enter to include all jobs.
Pay periodEnter the date of the pay period to be printed, or press Enter to accept the default.
Work week datesEnter the last date of the work week, or press Enter to accept the default. Based on the work week ending date, the beginning date and corresponding weekday display.
Include jobs with no activity?Select this checkbox to include active, certified jobs with no activity.
This option defaults to selecting the checkbox because contractors often need to print the Certified Payroll Report for all active, certified jobs, even if they have no activity for the current week. This will just produce a Certified Payroll Report with headings and cover screen (if indicated). It will also print the Pay No. on the report.

Include job totals?Select this checkbox to include job totals on the report.
Show employee address?Select this checkbox to include the employee address on the Certified Payroll Report. By default this checkbox will not be selected.
Show entire social security number?Select this checkbox to display the full social security number; otherwise leave it clear. This checkbox defaults to clear unless a saved filter is present.
Identify workers using employee code?If the AASHTOWare or LCPtracker file formats are selected, select this checkbox to use the employee codes on the report to identify workers instead of social security number.
FormatSelect whether to print Standard, Expanded, or Expanded with employer union fringe version of the Certified Payroll Report. If you select to include the employer union fringe, a second section will display.
You also have the option to generate export files containing certified report information conforming to eCPR, AASHTOWare and LCPtracker standards.

Employer union fringeThis option box displays when the Expanded with employer union fringe format of the report has been selected. Choose to produce this report in Detail or Summary format. The Detail format shows the breakdown of the employer fringe, including the regular, overtime, double time, and total hours for each fringe.
Cover page and file options
Print cover screenCheck this box to accept the system default and print a cover screen for the report; otherwise, leave blank to print the report without a cover screen. If this checkbox is selected, the Cover Sheet Setup window displays.
Other File InfoThis button displays if the 'Format' selection field is set to eCPR, AASHTOWare or LCPtracker file formats.
Select to enter additional information required by the state you are reporting in. This information will be included in the export file.

- [Other Information for eCPR File](/en/spectrum/spectrum/accounting/payroll/payroll-reports/certified-payroll-report/other-information-for-ecpr-file)

- [Other Information for AASHTOWare File](/en/spectrum/spectrum/accounting/payroll/payroll-reports/certified-payroll-report/other-information-for-aashtoware-file)

Cover screen dateEnter the date that is to appear on the cover screen, if desired; otherwise, press Enter to accept the system current date default.
Signatory partyEnter the name of the person responsible for receiving the Certified Payroll Report.
Signatory titleEnter the title of this person, if desired.
Fringe paid to plan?Select this checkbox to indicate if this fringe is to be paid to a plan.
Fringe paid in cash?Select this checkbox to indicate if this fringe is to be paid in cash.
Described deductionsEnter descriptions for the deductions, if desired.
ExceptionEnter any report exceptions, if desired.
ExplanationEnter any exception explanations, if desired.
Remarks when no activityWhen the Include jobs with no activity checkbox is selected, enter any remarks in this field. These remarks will appear in the 'Remarks' box on the Certified Payroll cover page for *all jobs without any activity* for the week.
