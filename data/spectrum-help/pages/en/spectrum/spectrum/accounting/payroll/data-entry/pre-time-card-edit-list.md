# Pre-Time Card Edit List

After Pre Time Card Entry is complete, the Pre-Time Card
 Edit List can be printed and checked for accuracy.
Make any necessary corrections and reprint the report.
This report offers a drill-down to Document Imaging that displays when the Preview button is clicked to access the reports. For easy identification, items with an associated Document Imaging display are shown in blue in the Crystal-Link Preview window.
If the Show pre-time card pay rates? checkbox in the Payroll Installation is not selected, the rate and extension columns on this report will be blank.
Important: If some employees appear to be missing from the
 listing, check the work dates entered to make sure the missing employees were not
 inadvertently omitted because of the date range specified for the listing by the
 user.
