# Export T4 File - Field Descriptions

Reference the table below when completing the fields on this screen.
FieldsDescriptions
YearEnter a four-digit year.
Federal tax codeDefault value is the Default federal tax code for new employee found in [Payroll Installation - Properties](/en/spectrum/spectrum/accounting/payroll/installation-overview/payroll-installation---properties). You may enter a different payroll tax code.
EntityNote: Displays only if cost centers are being used and entity tracking is also enabled in the current company.
Enter or select the Entity code. The software will verify that the current operator has security authorization for the selected entity before proceeding. If this field is left <blank>, the main company entity is used.
The system will include all employee forms within the specified entity and meeting the rest of the starting screen selections.

Contact information
Contact name
Telephone
Telephone extension
Contact email
Enter a contact name and additional contact information in case of a problem or error.
Transmitter information
Submission referenceEnter a unique number each time the file is generated for the specified year.
Transmitter account typeSelect your business number format.
Transmitter account numberEnter the entity's CRA Account Number. Remains disabled until you've selected an account type. If you leave this field blank, the Transmitter rep ID field is required.
Transmitter rep IDEnter the unique identifier assigned to the individual submitting the electronic file. If you leave this field blank, the Transmitter account number field is required.
Transmitter nameEnter the MM number used to identify the entity submitting electronic information returns.
CountryThe 3-character country code, for example, CAN.
Social insurance numbers
Proprietor #1 SINOptional.
Enter one or more proprietor's Social Insurance Number.
