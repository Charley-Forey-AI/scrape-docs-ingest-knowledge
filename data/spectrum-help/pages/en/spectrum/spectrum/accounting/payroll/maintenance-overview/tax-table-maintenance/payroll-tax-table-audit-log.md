# Payroll Tax Table Audit Log

Use this screen to create the Audit Log report for tax table changes only.
For all users and within your chosen date range, this report includes all inserts, updates, and deletions for these tables:

- PR_TAX_TABLE

- PR_INCOME_TAX_ADJUSTMENT_MC

- PR_INCOME_TAX_TABLE_MC

Field
 Description

Tax codeSpecify the tax code to include on the report, or press Enter to include ALL on the report.
From date
To date
Specify a date range for the report. The 'To date' defaults to the current payroll processing date, but can be changed.
