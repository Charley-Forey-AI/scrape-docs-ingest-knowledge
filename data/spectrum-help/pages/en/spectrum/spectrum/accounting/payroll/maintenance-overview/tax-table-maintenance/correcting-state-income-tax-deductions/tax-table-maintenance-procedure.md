# Tax Table Maintenance Procedure

You can make corrections to the Tax Table Maintenance
 numbers.

1. On the Site Map screen, click Payroll  >  Maintenance  >  Tax Tables.

1. Correct the tax table and/or the employee's tax file so future calculations are correct.

1. In Tax Table Maintenance create a new dummy tax state called, for example, XX or ZZ. Note that there are no taxes, deductions, exemptions, unemployment, and so forth. Also note that the Tax Exclusion Maintenance window has excluded all pay types from SIT, SDI, and SUTA.
