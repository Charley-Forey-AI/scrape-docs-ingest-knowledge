# ACA Client Communication Test Instructions

An export that uses approved sample data along with key organizational information.
The IRS requires all employers who are going to file ACA information electronically to perform a one-time communications test.

1. From the Payroll > Period End > Form 1095-C Maintenance screen, select the Export Forms button.

1. On the Export Setup - Identification window, enter a signatory title and date for the export.

1. Enter a contact name and phone, and transmission control code for the export.

1. Select the Test button. A dialog box displays alerting you to the fact that two files (a data file and a manifest file) will be created for submission to the IRS.

1. Select Continue.

1. Download the data file (Sample_1094C_Forms.xml) and manifest file (Sample_1094C_Manifest.xml) from the browser.

1. Transmit the two files to the IRS using your UI portal.
