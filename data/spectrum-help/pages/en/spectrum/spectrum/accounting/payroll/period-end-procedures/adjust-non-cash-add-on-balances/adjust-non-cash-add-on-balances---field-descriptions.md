# Adjust Non-Cash Add-on Balances - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Employee code
Enter the employee code for the employee whose year-to-date balances for non-cash add-ons need to be adjusted.

Entity
If cost centers and entity tracking is enabled in Enterprise Installation, you can enter a cost center entity here in order to adjust entity-specific balances.

Detail
Click this button to open the [Employee Check Detail](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/adjust-non-cash-add-on-balances/employee-check-detail) window and make any changes to the Year-to-date, Davis-Bacon, or Prevailing wage amounts.

Payroll add-on
A list of all of the non-cash add-ons for this company displays.

Descriptions
The add-on description displays in this field.

Calculation method
The calculation method used for this add-on displays.

Current rate
The current add-on rate for the employee displays from the Employee Recurring Deductions/Add-ons screen, if applicable.

Year-to-date balance
The current year-to-date balance for the employee displays, if applicable.

Davis-Bacon balance
The current Davis-Bacon balance for the employee displays, if applicable.

Prevailing wage balance
The current prevailing wage balance for the employee displays, if applicable.

Example #1 Adding Visibility of Employer Paid Benefits:
The company provides long-term disability for its employees. This policy has been paid for through the normal Accounts Payable processes. Management has decided to display this benefit on the payroll check. This function can be used to enter the year-to-date amount paid for each employee. Going forward, an add-on (not paid to the employee) will accrue this expense.
Notice that no journal entry is required as this was already booked through Accounts Payable.
Example #2 Employee Moves to Another Company
When an employee moves from one Spectrum company to another, it is nice to be able to print the year-to-date 401(k) employer matching funds or other non-cash add-on information on the paycheck. This function will allow the entry of these balances directly into the new company.
It may be necessary to record a journal entry for accrual purposes.
