# Rebuild Employee File

This rebuild uses tables throughout the system and should
 only be performed when all other users are logged out.
Confirm that this workstation is the only one currently logged into Spectrum prior to
 proceeding.
Important: This screen requires a level 9 of security
 category ER (Error Recovery) to access.
