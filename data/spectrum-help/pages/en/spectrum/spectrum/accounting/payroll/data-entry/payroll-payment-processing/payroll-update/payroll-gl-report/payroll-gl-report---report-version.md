# Payroll G/L Report - Report version

Press Enter to accept the software default and print a Summary report, or select to print a Detailed report, or Both types of reports.
The G/L date that appears on the report(s) is either the period end date or the check date, depending on the default in Payroll Installation. The reports will also have subtotals for period totals and company totals, with a new screen break between company codes.
