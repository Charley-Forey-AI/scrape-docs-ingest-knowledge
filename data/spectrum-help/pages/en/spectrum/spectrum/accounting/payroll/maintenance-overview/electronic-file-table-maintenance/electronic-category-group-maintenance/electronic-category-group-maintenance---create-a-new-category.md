# Electronic Category Group Maintenance - Create a new category

Follow the steps below to create a new
 category.

1. Click the New button to display the New Category Group window.

1. Enter a new Group code and corresponding Descriptions.

1. At the Code field, enter an add-on or deduction code or use the available drop-down arrow to select a code. The code description and type default.

1. Repeat step #3 until all the appropriate deductions and add-ons have been associated with the new category code.

1. Click OK.
