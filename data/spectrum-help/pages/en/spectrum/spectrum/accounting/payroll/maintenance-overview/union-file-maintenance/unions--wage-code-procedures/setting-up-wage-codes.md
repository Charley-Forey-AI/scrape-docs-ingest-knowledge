# Setting Up Wage Codes

For jobs that are covered by the Davis-Bacon Act,
 contractors are typically required to pay workers according to prevailing wage rates defined
 by the law.
This complicates the payroll process by requiring
 contractors to pay different rates, often depending on the type of work that is being done.
 Spectrum allows wage codes to be set up and determined automatically by the software during
 Payroll Time Card Entry. For a discussion of the most efficient setup of wage/union codes to
 suit your specific needs; it is recommended that you schedule training with the
 Viewpoint Training Coordinator.

1. On the Site Map screen, click Payroll  >  Maintenance  >  Wage Code.

1. In the Wage code field, enter the wage code, or press F4 or double-click on this field to select from a list of available wage codes.

1. In the Union code field, enter the union code, or press F4 or double-click on this field to select from a list of available union codes.

1. Complete the Abbreviated wage description and Full wage description fields.

1. In the Worker's compensation code field, enter the worker's compensation code if the worker's compensation will be the same for all employees in this wage code, press F4 or double-click on this field to select from a list of valid worker's compensation codes, or leave the field blank.

1. Enter the regular, overtime, double time, and bill codes for each wage code
 level. You can press F4 or
 double-click on any of these fields to select from a list of available rates.Note: When the cursor is in the Level 1 Regular field, you can
 press F4 or double-click on this field to open the
 Rates window. If you enter the base rate for each pay
 level, the software will automatically calculate the fringe. When all pay levels
 have been entered, click OK. Then, tab through the
 Regular, Overtime, and
 Double time fields; the software will automatically
 calculate the amounts.

1. When all level rates and bill codes are entered, click Save until you return to the Site Map screen.

1. On the Site Map screen, click Payroll > Maintenance > Employees. The Employees screen displays.

1. In the Search field, enter the employee code, or press F4 or double-click in this field to select from a list of available employee codes.

1.  In the Employee Main
 Properties screen, in the Wage code field, enter the wage
 code, or press F4 or
 double-click on this field to select from a list of available wage codes.

1. Select the employee you want to apply the wage code to and click Edit.

1. In the Wage code field, enter the wage code, or press F4 or double-click on this field to select from a list of available wage codes.

1. Click Save to apply your changes and return to the Site Map screen.
