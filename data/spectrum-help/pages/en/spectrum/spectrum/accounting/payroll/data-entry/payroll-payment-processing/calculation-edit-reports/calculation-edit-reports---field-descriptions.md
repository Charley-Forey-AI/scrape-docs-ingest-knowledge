# Calculation Edit Reports - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Report Name
Descriptions

State Wage and Tax Listing
The Wage and Tax Listing is used to print the employee's state wage and tax information. This report is designed to provide income tax, workers' compensation, SDI/FICA, and unemployment figures. It will detail earnings subject to each tax and the tax amounts computed, which is useful for Federal, county and local municipalities, as well as single-state operations. This report is available both in summary and detail, and it may be sorted both by employee and by tax code. [Wage and Tax Listing Selections window](/en/spectrum/spectrum/accounting/payroll/data-entry/payroll-payment-processing/calculation-edit-reports/wage-and-tax-listing-selections-window)

Pre-Check Register
The Pre-Check Register prints a list of all checks as they will appear when the checks are printed. It also lists all manual and void checks being processed. Each item is specifically identified by the employee code and check sequence number. [Pre-Check Register Selections Window](/en/spectrum/spectrum/accounting/payroll/data-entry/payroll-payment-processing/calculation-edit-reports/pre-check-register-selections-window)

Deduction Report
Use this report to print a list of all the deductions applied to employee paychecks for this pay period.

Add-on Report
Use this report to print a list of all the add-ons applied to employee paychecks for this pay period.
