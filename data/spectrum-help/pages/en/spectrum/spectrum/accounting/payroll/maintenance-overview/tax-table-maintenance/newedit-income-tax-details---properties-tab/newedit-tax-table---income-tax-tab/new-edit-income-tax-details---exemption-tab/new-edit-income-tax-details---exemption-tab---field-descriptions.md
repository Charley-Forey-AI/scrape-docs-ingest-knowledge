# New Edit Income Tax Details - Exemption tab - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Tax exemptions
Select a tax exemption basis. The additional fields that display will vary based on your selection in this field.

Annual exemption amount
Enter an annual exemption amount. This will be applied to each category (Single, Married, etc.)

Separate personal exemptions for dependents?
Select the checkbox if the dependent's exemptions are a different amount than the employee's personal exemptions. Leave the checkbox clear if the dependent's exemptions are the same amount as the employee's.

Annual amount for dependents
If the dependent exemption amount is different from the amount for the employee, enter that amount here. It is the same for each category, and only needs to be entered once. If entry in the separate personal exemptions checkbox was not selected, this field will be skipped
