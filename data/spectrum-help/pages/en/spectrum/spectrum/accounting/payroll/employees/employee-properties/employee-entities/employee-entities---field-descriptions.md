# Employee Entities - Field Descriptions

Use the table below for reference when completing the
 fields in the Employee Entities screen.
Fields/Buttons
Descriptions

New button
Click this button to open the Authorize
 New Entity window. Here you can select a different entity from the Search
 Entities window and determine if payments can be made from this entity. Note: Entity tracking is enabled in the Enterprise
 Installation screen and entities are set up and maintained in the Entity
 File Maintenance screen in the Enterprise Management module.

Modify button
The 'main company entity' authorization
 displays in the upper right corner of the page, and the employee will either be
 "authorized" or "not authorized." A Modify button opens the Main
 Company Entity Settings window where employee information can be edited, and
 includes the ability to enter an optional override cost center for G/L debit
 cost center posting.
Set Status button
Click this button to open the Set Pay
 Cycle Authorization Status window and determine whether the current employee
 can be included in pay cycles for the particular entity by selecting the
 "Authorized for payment from this entity?"
Remove button
Select a record and click this button
 to delete the employee entity. You will receive an error message if the entity
 has current year earnings, time cards, layoff checks, pre-time cards,
 and more.
Deductions/Add-ons button
Click this button to open the
 Entity-Specific Recurring
 Deductions/Add-ons window where you can apply deduction and
 add-on codes for this employee for the particular entity.Click the Status button to specify
 which deduction and add-on codes display in this window. This will help when
 uploading deduction and add-on codes at year-end.
An optional cost center field is also provided. If specified, this cost
 center will be used during the Payroll Update to General Ledger in place of
 the employee's cost center (from [Employee Main Properties](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-main-properties)).


Time Off Bank button
Click this button to open the
 Entity-Specific Time Off
 Bank window and complete the vacation, holiday, and sick accrual
 codes to be applied for this employee when included in pay cycles for this
 entity. This window will work the same as the [Employee Time Off Bank](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-time-off-bank) screen.
Columns

Entity
A list of all current employee entities
 displays.
Description
The entity descriptions display.

Ded/add setup?
Yes displays when there is at
 least one deduction/add-on record for this employee and entity.
Time off setup?
Yes displays when there is at
 least one accrual (vac/hol/sick) on record for this employee and
 entity.
Pay cycle status
This will say Authorized if the employee
 and entity are not Inactive.
