# Mapping the AASHTOWare System to Spectrum

The following explains where the information set up on the spreadsheet is also set up in Spectrum.

1. Contractor's Vendor ID: This is your Vendor ID number in the CRL website. Enter your ID into
the Contractor ID field within the Other XML Info button.

1. Contract #: This is the Agency's identification for the work and is a six digit contract number
found on the CRL website. Enter it in the Contract # field in Job Main Properties.

1. Fringes Paid in Cash or to Plan: This information is entered on the Certified Payroll Report's
start screen. Select the appropriate checkboxes to indicate how fringes are paid.

1. Benefit Program Information: This information is entered on the Properties tab of the Certified
Benefit Program Maintenance screen.

1. Benefit Program Name

1. Benefit Program Type: Select one of the pre-populated types

- Apprentice Fund

- Health/Welfare

- Pension

- Vacation/Holiday

- Other 1

- Other 2

1. Benefit Account Number

1. Benefit Program Classification

1. Contact Person

1. Contact Phone

1. Employee Information: This information comes from the employee's record (see next section).
Salaried employees are converted to a rate per hour automatically here. Spectrum does not
track whether or not the employee information has changed on the XML file.

1. Craft Code and Labor Code: The Craft Code comes from the Certified craft code on the union.
The Labor Code comes from the Certified labor code on the wage code.

1. Apprentice ID and Apprentice Wage %: These are set up on the Employee Pay Rates page.

1. Classification Fringe Benefit Hourly Rate Credits: Fringes, add-ons and deductions that have
been mapped to a Benefit Program Type will default to this section of the file automatically.
Provided you have set up the Certified Benefit Program Maintenance screen, no additional Spectrum setup is required.

1. Exceptions: This information is entered on the Certified Payroll Report's start screen.

1. Other Deductions: automatically populates the other add-ons and deductions not
already mapped to a benefit program here. No setup is required here.

## Employee Information Tab

The information found on this tab is found on the Employee pages in Spectrum.

- Employee Main Properties: Contains First, Middle and Last Name

- Employee Personal Information: Contains Mailing and Domicile Addresses, Race, and Sex. The
Race field in Spectrum is properly translated into the AASHTOWare Ethnicity code format.

- Employee Tax Set up: Contains the employee's Social Security Number.

- Employee Pay Rates: Contains the Apprentice ID and Apprentice Wage % fields

## Craft Tab

Each state or agency will have their own craft codes for you to use.

## Labor Tab

Each state or agency will have their own labor codes for you to use.
