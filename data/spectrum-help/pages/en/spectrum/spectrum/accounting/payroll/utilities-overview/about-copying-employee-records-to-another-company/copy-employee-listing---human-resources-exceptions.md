# Copy Employee Listing - Human Resources Exceptions

If any errors are encountered for the Human Resources setup
 in the destination company, the software will generate an exceptions report, as shown below.
This report lists the employee code, the employee name, and all the fields that were not
 matched. The employees listed on this report will not be copied into the destination
 company.
