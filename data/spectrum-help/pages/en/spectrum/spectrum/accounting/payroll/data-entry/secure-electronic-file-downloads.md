# Secure Electronic File Downloads

If the 'Use secure download page for electronic files'
 option was selected on the Installation screen, ACH files processed by A/P or Payroll clerks
 will appear on this screen in encrypted format.
Controllers or those with a security level of 8 or higher can use this screen to download
 decrypted ACH files for processing. Once the files have been processed by your financial
 institution, return to this screen and delete the files.
