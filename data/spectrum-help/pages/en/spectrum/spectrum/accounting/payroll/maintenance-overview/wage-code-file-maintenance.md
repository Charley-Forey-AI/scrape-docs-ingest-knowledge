# Wage Code File Maintenance

This screen is primarily used in conjunction with Equipment
 Control, where more than nine wage rates might be necessary for a single union, or the same
 work is performed by the employee under the jurisdiction of multiple union locals.
It can also be used to set up prevailing wages; in this situation, each level would record
 the prevailing rates in effect at the start of the job.
Security has been included so that if you attempt to delete a wage code that has already been posted to Time Card History you will receive an error message. In addition, if you delete an unused wage code, any revision records that were associated with the code will also be deleted.
Use the Status link to determine which records display. Please refer to the [Union & Wage Codes](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/union-file-maintenance/unions--wage-code-procedures/union--wage-codes) topic before setting up union and wage codes.
