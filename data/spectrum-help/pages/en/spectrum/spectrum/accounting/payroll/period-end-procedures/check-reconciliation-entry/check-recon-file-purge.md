# Check Recon File Purge

The Check Recon File Purge screen first prints a list of
 items to be removed and then removes items from the check reconciliation file that have been
 marked as reconciled.
All reconciled checks, up to and including the date entered, will be removed. Open items
 will not be purged, regardless of date.
