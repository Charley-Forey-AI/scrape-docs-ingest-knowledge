# New/Edit Union - Benefit Tiers tab

Use this tab to add benefit tiers to the selected union code. Benefit tiers allow users to define overrides for union deductions and add-ons by wage code on a date-sensitive basis, instead of having to create multiple union codes to track different fringe packages.
Field
Description

Benefit tier
Enter a unique benefit tier code.

Description
Enter a description for the benefit tier.
