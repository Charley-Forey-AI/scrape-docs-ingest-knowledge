# Check Reconciliation Entry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Statement date
Enter the date of the bank statement, or press F4 to display the Date Change window to select the date.

List checks
From the drop-down menu, select to include all checks or only current and open checks.

Selections button
Clicking this button displays the [Check Reconciliation Selections](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/check-reconciliation-entry/check-reconciliation-selections) where you can enter the specific check numbers to be reconciled.

OK?
If the displayed information is to be reconciled, select the checkbox. This increases the dollar amount in the "total" field by the amount of this transaction. The displayed "total" may be compared with the bank's statement.

Current total
This field is maintained by the system and displays a running total of all transactions as they are reconciled.

Listing button
Click this button to go to the [Check Recon File Listing](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/check-reconciliation-entry/check-recon-file-listing) screen.

Maintenance button
Click this button to go to the [Check Reconciliation Maintenance](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/check-reconciliation-entry/check-reconciliation-maintenance) screen.

Purge button
Click this button to go to the [Check Recon File Purge](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/check-reconciliation-entry/check-recon-file-purge) screen.
