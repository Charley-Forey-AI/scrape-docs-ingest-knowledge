# Labor Distribution History Report

The purpose of this report is to assist in calculating tax
 liability by payroll department.
The report's FICA, FUI and other amounts are calculated as a percentage of the check
 totals.
Field
Description

Department
Enter the payroll department code to include on the report, or press Enter to include ALL departments on the report.

From/to check date
Enter the check date range for the report, or press Enter to start with the earliest processing date and end with the current processing date.

Cost group
If cost centers are enabled for the current company, specify the cost group to include on the report and whether to sort the report by department or cost group. This report will only include cost centers that the operator has security to access.
