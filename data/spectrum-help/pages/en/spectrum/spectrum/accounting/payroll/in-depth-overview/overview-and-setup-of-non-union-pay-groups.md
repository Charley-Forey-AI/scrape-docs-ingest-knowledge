# Overview and Setup of Non-Union Pay Groups

Pay groups provide a means to set up a series of default
 pay rates for non-union employees.
Features include the ability to set different (estimated) burden rates by employee, job, or
 phase. This is a wage table that works for both prevailing wage and non-prevailing wage
 jobs.
Before setting up pay groups, be aware of the following:

- Only one pay group code can be assigned to a job. This means that a limited number of rates can be assigned to a pay group.

- Actual Payroll burden can be used with pay groups.

- Like Spectrum's union code setup, Deductions/Add-ons can be assigned to a job.

General Setup for Non-Union Pay Groups:

- On the Payroll Installation screen, select the Pay groups option in the Default rate table section. While it depends on the your company's policy, most will also select the Higher of the two rates option in the Default pay rate section.

- In the Payroll  >  Maintenance  >  Non-Union Pay Group screen, [Setting Up Pay Groups](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/non-union-pay-group-maintenance/pay-group-procedures/setting-up-pay-groups).

- If applicable, set up [Overriding Employee and Phase Codes](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/non-union-pay-group-maintenance/pay-group-procedures/overriding-employee-and-phase-codes).

- [Adding Pay Groups to a Job](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/non-union-pay-group-maintenance/pay-group-procedures/adding-pay-groups-to-a-job).
Note: Note to Wage Code Users: Similar to the wage code
 information, the pay group code information is entered in the job's Properties
 window.

- [Adding Pay Groups to an Employee](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/non-union-pay-group-maintenance/pay-group-procedures/adding-pay-groups-to-an-employee).
