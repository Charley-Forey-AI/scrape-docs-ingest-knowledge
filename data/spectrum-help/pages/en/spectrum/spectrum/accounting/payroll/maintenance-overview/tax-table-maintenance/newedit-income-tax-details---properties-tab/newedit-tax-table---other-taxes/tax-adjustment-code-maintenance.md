# Tax Adjustment Code Maintenance

This screen allows you to define 'adjustments' to the
 income tax calculation, and is especially helpful for customers who have complex calculations.
You can specify the tax basis (gross earnings or income tax amount), whether the adjustment
 increases or decreases tax withholding, and the graduated tax table adjustment. Any new
 adjustment codes will then be shared among all Spectrum companies.
To get started, click the New button to add a new code or select a record and click the Edit button to make changes. Use the table below for reference when completing the fields in the New Tax Adjustment Code window.

## Ontario Provincial Tax Reduction

Ontario has a special pre-defined rule for low earning employees. This tax is based on two published factors and the number of Dependents recorded on the [Employee Personal Information](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-personal-information) page. A window displaying the Calculation Details will display the tax formula; this formula will be maintained by our Support Department as Canada updates it.
