# Wage Code Maintenance

Assign Benefit Tier code to the wage code. In this manner, the employee's wage and benefit tier default into the time entry screens.
Note:Important: For existing clients, we recommend that
 you sunset the 'old' fringes at the end of the period. When transitioning from the
 'old' method to this new functionality, you must configure the 'old' fringes to go
 to a zero rate at month end. Then start the 'new' fringes as of the first day of
 the next month. Failure to do this will result in the Union Report's 'Check Total'
 being inaccurate for the transition time frame.
