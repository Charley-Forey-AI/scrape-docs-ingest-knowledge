# New/Edit Union - Fringes tab

Use this tab to add union fringe rates to the selected union code. Fringes may be computed in a variety of ways and are part of the employer burden, which is passed to the Job Cost module and to General Ledger.
Important: When you want to change a fringe rate and/or effective
 date it is necessary to add a new rate because the old rate and dates are still there
 for the previous month's entries to calculate against.
Note: For companies tracking union fringe benefits using add-on
 codes and non-union companies, this tab can be hidden in the Payroll Installation
 screen.
