# Mapping the eCPR system to Spectrum

The following explains where the information set up on the iForm is also set up in Spectrum.

1. Contractor (FEIN): the 'Tax ID number' from Tax Table Maintenance or Entity File Maintenance for the US code populates here.

1. License Type: entered into the 'License type' field on the 'Other XML Info' window within the Certified Payroll Report.

1. License Number: entered into the 'License #' field on the 'Other XML Info' window within the Certified Payroll Report.

1. Contractor Name: defaults from Company Installation or Entity File Maintenance.

1. Contractor (PWCR): entered into the 'Contractor ID' field on the 'Other XML Info' window within the Certified Payroll Report.

1. Address Information: defaults from Company Installation or Entity File Maintenance.

1. Insurance Number: entered into the 'Insurance #' field on the 'Other XML Info' window within the Certified Payroll Report.

1. Contractor Email: entered into the 'Contact email' field on the 'Other XML Info' window within the Certified Payroll Report.

1. DIR Project ID: entered into the 'Certified project ID' field on Job Payroll Setup.

1. Contract With: entered into the 'Contracting agency' field on the 'Other XML Info' window within the Certified Payroll Report.

- Items A through P come from the Certified Payroll Report.

- Items 1 through 8 are defined on the Certified Benefit Program Maintenance screen.

- Item 9 – 'Other' contains any other add-ons and deductions not defined on the Certified Benefit Program Maintenance screen.
