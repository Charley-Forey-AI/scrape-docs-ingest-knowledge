# New Hire / Terminated Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Employee Supervisor Department Status
Enter the Employee, Supervisor, Department, and Employee Status, or press Enter to include all on the report.

Date
Enter the dates which this report is to begin and end with, or press Enter to accept the current payroll processing date.

Report type
Select New Hire to print the Employee New Hire Report, or select Terminated to print the Employee Termination Report.
