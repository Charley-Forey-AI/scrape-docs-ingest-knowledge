# Employee Pay Group Rates - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields/Buttons
Descriptions

Pay group
Enter the pay groups number, or press
 F4 to select from
 a list of applicable pay groups. The pay group description displays to the
 right of this field.
Title
Enter the employee's title when
 performing work in this pay group, or press Enter to accept the default
 title.
Regular
Enter the employee regular pay rate for
 work with this group.
Overtime
Enter the employee overtime pay rate
 for work with this group.
Double time
Enter the employee double time pay rate
 for work with this group.
Burden
Enter the burden percentage rate for
 this pay group. The burden percentage refers to the employer's additional
 payroll costs (FICA, FUTA, SUTA, worker's compensation costs and other burden
 components).
Fringe
Enter the fringe override amount, if
 applicable. You can only enter a fringe override here when there is a
 prevailing wage add-on code entered in the Pension or Cash fringe benefit
 fields of the Prevailing Wage window. The fringe rate
 associated with the first pay level will default from the Prevailing Wage
 window, this fringe amount can be overwritten manually (if the fringe is
 blank then the fringe for this override rate will be zero <$0>). In
 addition, fringe rules and the certified status can be disabled by phase,
 employee, or employee phase.
For more information
 on the fringe rate hierarchy and Prevailing Wage processing, refer to the
 [Prevailing Wage and Davis-Bacon Processing](/en/spectrum/spectrum/accounting/payroll/in-depth-overview/summary-of-prevailing-wage-benefits/prevailing-wage-and-davis-bacon-processing) topic.

Bill code
Enter the time and material billing
 code for this pay group, or press F4 or double-click on this field to select from a list of
 available codes. This is only used if the Time + Material module is installed
 and a Time + Material job is used. The bill code description will display to
 the right of this field.
Disable DB/PW?
If Davis-Bacon and prevailing wages are
 not applicable, select this checkbox to post the time card line as a private
 job and to ensure the employee will be paid based on the base pay of the pay
 group (no fringe, prevailing wage, prevailing wage adjustment or pension would
 apply). Any company 401(k) employee and employer amounts will be calculated, as
 they would be for a private job.
Disable cert?
Select this checkbox to designate the
 time card line (that uses this pay group) as non-certified. This checkbox can
 be selected even if the job indicates the line is certified.
