# Find Wage Code Window - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Wage code
Enter the wage code to include in the search, use the SuperSelect option, press F4or double-click on this field to select from a list of available wage codes, or press Enter to search ALL union codes.

Union code
Enter the union code to include in the search, use the SuperSelect option, press F4or double-click on this field to select from a list of available union codes, or press Enter to search ALL union codes.

Short desc.
Enter the short wage description to include in the search, use the SuperSelect option, or press Enter to search ALL union descriptions.

Full desc.
Enter the full wage description to include in the search, use the SuperSelect option, or press Enter to search ALL union descriptions.

Worker's comp.
Enter the worker's compensation code to include in the search, use the SuperSelect option, press F4or double-click on this field to select from a list of available worker's compensation codes, or press Enter to search ALL worker's compensation codes.
