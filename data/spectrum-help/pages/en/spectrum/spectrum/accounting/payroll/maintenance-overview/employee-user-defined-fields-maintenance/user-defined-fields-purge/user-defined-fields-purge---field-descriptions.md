# User-Defined Fields Purge - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Comp
The code for the company that uses the user-defined field displays. No entry is allowed. Click the column header to arrange the data by ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by slicing the column title's side borders.

Type
The user-defined field's type (Amount, Date, or Numeric) displays. No entry is allowed. Click the column header to arrange the data by ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by slicing the column title's side borders.

Prompt
The user-defined field's title (for example, DUE DATE) displays. No entry is allowed. Click the column header to arrange the data by ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by slicing the column title's side borders.

Purge
Click this button to purge the selected user-defined field.
