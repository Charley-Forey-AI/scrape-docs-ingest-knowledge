# Deductions For Check - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Code
Enter a deduction code. The corresponding deduction amount displays, but may be overridden, if desired.

Descriptions
The description of the deduction code displays.

Amount
The deduction amount for this check displays. Press Enter to accept the software default or manually enter a different amount.

Source
The source of the deduction information displays. Source options include: Employee Deduction Maintenance, Union File Maintenance, Time Card Entry.
