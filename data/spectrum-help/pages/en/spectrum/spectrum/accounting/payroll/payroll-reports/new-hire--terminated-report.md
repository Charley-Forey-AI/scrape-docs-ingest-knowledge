# New Hire / Terminated Report

The New Hire / Terminated Report prints a listing of employees in employee code order, or a listing of the terminated and deceased employees.
If either the hire date or the rehire date fields is within the selected range on the starting screen, this employee will be included on the Employee New Hire Report. An asterisk appearing in front of the date means that more than one record was found in the date range period. If both dates fall within the selected range, the later date appears in the Hire Date column.
