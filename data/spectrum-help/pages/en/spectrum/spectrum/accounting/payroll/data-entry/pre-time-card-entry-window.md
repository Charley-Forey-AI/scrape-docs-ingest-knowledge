# Pre-Time Card Entry Window

Use this window to add new pre-time card lines or to edit existing pre-time card lines.
As entries are completed in this window, they appear in the columns on the Pre-Time Card Entry starting screen.

- If you are editing existing pre-time card lines, some of the key fields are disabled (for example, Employee, Pay type, and when applicable, Job). Similarly, some fields are disabled based on the department selected (for example, if equipment is selected, the Job, Phase, and Cost type fields aren't relevant).

- If errors occur during data entry, a warning message display and a View Error Messages button appears on the lower right of the window. Select this button to open the Error For Current Record window and review all the current errors for this record. To make corrections, double-click an error message line or select OK to return to the Pre-Time Card Entry window and make your changes there.

## Note regarding Equipment Usage pay types

When a transaction for pay type ER, EO, ED, or EU is added, the software evaluates whether the equipment code assigned to the time card entry has any active attachments. If so, additional EU time cards are generated automatically or the [Attached Equipment window](/en/spectrum/spectrum/accounting/payroll/data-entry/pre-time-card-entry-by-job/attached-equipment-window) displays (based on the Automatically add transactions for attached equipment setting in the Equipment Control Installation > Properties screen).
Note regarding 'Direct work order cost' departments:
If the Department is a 'Direct work order cost' department, the following fields appear in place of the existing fields in the screen below. If the time card line originated in Work Order > Labor Hours Entry, or the time card has already been selected for billing in Work Order > Labor Hours Entry, these fields are display-only.
