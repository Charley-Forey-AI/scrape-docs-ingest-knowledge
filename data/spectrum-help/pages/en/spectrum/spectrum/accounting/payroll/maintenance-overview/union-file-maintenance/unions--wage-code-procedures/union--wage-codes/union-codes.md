# Union Codes

Union codes may be set up to track the work performed by
 various union employees.
When pay rates are entered in the Union File Maintenance screen, these pay rates will
 default during Pre-Time Card Entry and Time Card Entry. The standard union code system
 gives contractors up to nine pay levels for paying union employees.

The Union Report provides a listing of hours worked by each employee in a union, as well as a breakdown of fringe benefits by union. One option for printing this report will group unions according to the first three characters of the union code. Thus, for example, if information for all Washington carpenter's unions will be desired, the various local carpenter's unions may be set up with codes of WAC01, WAC02, WAC03, and so forth.
