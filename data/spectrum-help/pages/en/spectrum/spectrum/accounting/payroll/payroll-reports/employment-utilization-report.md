# Employment Utilization Report

The Employment Utilization Report, which is required on
 some public works jobs, provides a breakdown by ethnic group of the employees working on a
 particular job.
The report prints by job by EEO Classification. The ethnic groups include African American,
 Hispanic, Asian, and Native American (Alaskan/American Indian). The report also provides a
 breakdown by gender, but if a gender was not specified in the Other tab of the Employees
 screen, the report will treat the employee as female.
The report provides the option to include certain text information in the heading and the base of the report. Modeled after the federal form layout, this report provides the user with the ability to designate covered areas, goals for minorities and females, address and telephone number of the contractor, and so forth.
