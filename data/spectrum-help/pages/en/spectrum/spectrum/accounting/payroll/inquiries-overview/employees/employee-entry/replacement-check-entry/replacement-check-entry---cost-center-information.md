# Replacement Check Entry - Cost Center Information

If cost centers are being used in this company, then when
 entering a replacement check for an employee, Spectrum allows entry only if you have
 permission to access that employee.
As you request each employee, Spectrum compares the cost center assigned to the employee
 with cost centers in your operator's assigned cost center scheme, and if the cost center
 is not included, then replacement check entries for that employee will not be allowed.
If your scheme includes override settings for employee entries in Cost Center Scheme Maintenance, this screen will validate the cost center assigned to the employee based on these overrides. The override cost center(s) supersede the cost center list defined for the scheme in general. Spectrum will compare the cost center assigned in the entry screen detail with employee override cost centers in your operator's assigned cost center scheme, and if the cost center is not included, then replacement check entry for the employee will not be allowed.
If the operator is authorized to access the employee, then any check for that employee can be replaced and all time card detail, deduction, and add-ons for that check will be available for view. The replacement check will use the same cash cost center and bank account as the original check.
Cost center entities
If entity tracking is enabled in Enterprise Installation, the Replacement Check screen will read for the entity code from the original check when printing the Manual replacement check.
