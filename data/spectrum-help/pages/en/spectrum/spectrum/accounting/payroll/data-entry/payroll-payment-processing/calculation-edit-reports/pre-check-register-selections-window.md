# Pre-Check Register Selections Window

Click the Selections button and the Pre-Check Register Selections window appears. Select the appropriate sorting (by employee code or alphabetically) and detail options (include tax code detail or deduction/add-on detail) that you want to include on the register.
Pre-Check Register prints a list of all checks as they will appear when the checks are printed through the check-printing program. It also lists all manual and void checks being processed. Each item is specifically identified by the employee code and check sequence number.
