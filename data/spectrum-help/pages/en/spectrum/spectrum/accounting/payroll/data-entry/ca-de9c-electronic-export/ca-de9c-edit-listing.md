# CA DE9C Edit Listing

After the file name (MMREF) is downloaded, the 'California DE9C Edit Listing' start screen displays.

- The Preview button displays the report which is provided for your records as a reference only.

- The Export and My Reports buttons have the same functionality as standard report listing screens.
