# Employee Personal Information -Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Personal telephone Telephone memo Fax Work mobile
 phone Work extension Work e-mail Personal e-mail Contact remark
Enter the customer's personal
 telephone, any notes about the number, fax number, mobile work phone, work
 extension, work e-mail address, personal e-mail address, and up to 250
 characters in the contact remarks section. Information
 entered in these fields also displays on the Contacts Administration screen.

Note: If the employee contact record
 is shared between two companies, the contact remark will display in both
 companies. If you change the text in one company, it will change in the
 other.

Mailing address
Enter the employee's mailing address in
 this section. This address will print on the employee's paychecks.
Domicile address
Enter the employee's physical address
 in this section. This address is particularly useful on the Certified Payroll
 Report when the employee's mailing address is a P.O. Box.
Authorization to access Spectrum
You can select or search for the
 Operator code to associate a Spectrum operator with the selected employee. This
 way the operator will see any jobs associated with the employee on the
 dashboard.

- An employee is linked to a particular operator code
 through entry of the 'employee contact' in Operator Maintenance. When the
 operator code is not accessible here (for security reasons), contact your
 Administrator to set this up.

- If the current operator has security access for
 Operator Maintenance, the operator code currently stored in the Contacts
 master table for this employee displays, and changes are allowed.

 If you remove an operator code with an active logon
 status, the code will not be linked to this employee, but it still remains
 authorized to access Spectrum.

Driver's license
Enter the employee's driver's license number, expiration date,
 licensing state, and class in this section.
 Driver's license information will be validated if the Human Resources module is
 installed.
Emergency
Enter an emergency contact, telephone,
 and relation to the
 employee in this section.
Governmental reporting
Select the Include on certified report
 checkbox if this employee will appear on any Certified Payroll Report. Select the employee's race, sex, country
 of citizenship, I-9
 status and expiration date, security status, and veteran status (if
 applicable).
Veteran Status
 Codes:

- (D) Disabled Veteran

- (A) Armed Forces Service Medal Veteran

- (M) Disabled Armed Forces Service Medal Veteran

- (O) Other Protected Veteran

- (P) Disabled Other Protected Veteran

- (N) Non-Veteran

Select the Recently separated veteran
 checkbox if the employee is a recently separated veteran. A recently
 separated veteran is defined as any veteran who served on active duty in the
 U.S. military, ground, naval or air service during the three-year period
 beginning on the date of the selected veteran's discharge or release from
 active duty.
Select the Disabled checkbox if the
 employee is disabled, and enter a brief description of the
 employee's disability (for example, HEARING IMPAIRED).

Other personal information
Enter the employee's date of birth, place of birth, marital
 status, number of dependents, spouse's name and date of birth (if
 applicable).
Termination
If the employee is being terminated,
 after entering the Latest
 termination date, you may enter a brief Reason for the termination
 and you may assign a Rehire rating (on a scale of 0-10, with 10 being most
 favorable toward rehiring).
