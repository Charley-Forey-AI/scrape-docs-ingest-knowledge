# Pre-Time Card Import Errors - Cost Center Information

For records sent to the error screen due to the import operator having insufficient security for the employee, the Pre-Time Card Import window provides an error message clarifying where the error occurred.
When rechecking a time card entry line, you must have security for the employee's cost center, and the cost center assigned to the pre-time card line in order to post the record to Pre-Time Card Entry. When rechecking, if any records that would be updated are flagged as having an import operator cost center security error, the following prompt displays: "Update record(s) with errors due to import operator's cost center security settings now?"
The Cost center field will be display-only.
