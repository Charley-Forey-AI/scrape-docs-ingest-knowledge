# Union Copy Between Companies - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

From company
The current company code from which the union code(s) will be copied is displayed. No changes are allowed.

To company
Enter the company code to which the union code(s) will be copied.

Union code
Enter the union code to copy, use the SuperSelect options, or double-click on this field or press F4 to select from a list of available union codes, or press Enter to accept the software default of ALL.

For new unions, copy all rate history entries?
Select this checkbox to include all rate history information (for deductions, fringe benefits, and pay rates) in the new union, regardless of how many changes there have been since inception.
If this checkbox is left clear, then the new union is set up with the deduction and fringe codes from the source company, but all past rate and status information is excluded, except for the latest rate.

For existing unions, append new rate history entries?
Select the checkbox to add any new deductions, fringe benefits, and pay rates that did not previously exist for the destination union. Likewise, for existing deductions, fringe benefits, and pay rates, any rate history records dated on or later than the current Payroll processing date of the destination company will be copied to the destination company union. The rate history of existing unions remains intact.
Leave the checkbox clear to update only new unions to the destination company (no changes will be made to any existing unions).
