# New / Edit Electronic Master Table - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Master table code
Enter a brief name for this master table.

Descriptions
Enter a longer description for this master table. Up to 80 characters are allowed.

Memo
Enter any messages that might pertain to this master table. Up to 250 characters are allowed.

New button
Use this button to open the New Electronic Table window and select a table that you want to add to this master table.

Delete button
Select a table code from the list and click Delete to remove it from the master table.

Move Up / Move Down
Use these buttons to change the sequence of the tables. The software then resets the numeric sequence numbers accordingly.
