# Pay Cycle Calendar - Field Descriptions

FieldDescriptions
PeriodThe calendar period displays.
End dateThe end date for this period displays.
If the end date matches the period end date on the Set Payroll Cycle screen, the update will write to an entity-specific calendar.

Check dateThe check date for this period displays.
