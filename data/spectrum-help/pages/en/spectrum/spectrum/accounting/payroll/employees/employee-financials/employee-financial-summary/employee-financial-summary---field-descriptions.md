# Employee Financial Summary - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Other Entity Financials
When cost center entities are being
 used. click this button to drill into entity-specific details, while the main
 screen continues to display the current year financial information for the main
 company entity.
Gross earnings
The employee's month-to-date,
 quarter-to-date, and year-to-date gross earnings display in this field. Click
 the drill-down button to open the Gross Wages by Tax Code window.
Federal income tax
The employee's month-to-date,
 quarter-to-date, and year-to-date federal income tax deductions display in this
 field.
Social security
The employee's month-to-date,
 quarter-to-date, and year-to-date Social security deductions display in this
 field.
Medicare
The employee's month-to-date,
 quarter-to-date, and year-to-date Medicare deductions display in this field.
State income tax
The employee's month-to-date,
 quarter-to-date, and year-to-date state income tax deductions for the state of
 residence display in this field. Click the drill-down button to open the
 State Income Tax by Tax
 Code window.
State disability (SDI)
The employee's month-to-date,
 quarter-to-date, and year-to-date state disability deductions for the state of
 residence display in this field. Click the drill-down button to open the
 Disability Tax by Tax
 Code window.
Worker's compensation
The employee's month-to-date,
 quarter-to-date, and year-to-date worker's compensation deductions display in
 this field. View the [Worker's Compensation Hierarchy](/en/spectrum/spectrum/accounting/payroll/default-hierarchies-overview/workers-compensation-hierarchy).

County taxes
The employee's month-to-date,
 quarter-to-date, and year-to-date county income tax deductions display in this
 field. Click the drill-down button to open the County Income Tax by Tax Code
 window.
Local taxes
The employee's month-to-date,
 quarter-to-date, and year-to-date local income tax deductions display in this
 field. Click the drill-down button to open the Local Income Tax by Tax Code
 window.
Deductions
The employee's month-to-date,
 quarter-to-date, and year-to-date total deductions displays in this field.
 Click the drill-down button to open the Deductions by Code
 window.
Add-ons
The employee's month-to-date,
 quarter-to-date, and year-to-date total add-ons displays in this field. Click
 the drill-down button to open the Add-ons by Code window.
Net pay
The employee's month-to-date,
 quarter-to-date, and year-to-date net income display in this field.
Hours worked
The employee's month-to-date,
 quarter-to-date, and year-to-date hours worked display in this field. Note: Work hours = Regular + Overtime + Double
 time + Special rate

Other hours
The employee's month-to-date,
 quarter-to-date, and year-to-date other hours display in this field. Note: Other hours = Vacation + Holiday + Sick
