# Employee Main Properties

Use this screen to edit existing employee details.
After entries are completed on this screen, the data displays in the columns on the Employees starting screen the next time it is opened.

- If the Human Resources module is installed, some field information will default from the Human Resources module. However, the field-level information may be changed in this window without altering the Human Resources files.

- If a new hire was created with Trimble Construction One Onboarding, select the HR Management button to open up Onboarding in a new tab to make any changes to employee details and documents. Check out the [Using the Dashboard to Manage New Hires](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-main-properties/using-the-dashboard-to-manage-new-hires) topic to learn more about configuring Spectrum for onboarding.
