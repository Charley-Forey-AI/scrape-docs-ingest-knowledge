# Export W-2 File and Nelco State Tax Filing - Cost Center Information

If cost centers are being used in the current company, the
 software assures that the operator has security authorization to access all employees to be
 included in the export file.
In order to perform the 'Build' or to generate the 'Export File', the operator must have
 full security.
When 'entity tracking' is enabled, the screens and reports throughout W-2 Processing will hide W-2 records for entities the operator is not allowed to access.
