# Electronic Master File Maintenance

Use this screen to set up and maintain a list of electronic master files, grouping electronic tables.
This feature is especially helpful for those users who must file electronically in multiple tax jurisdictions because it allows you to create a concatenated list of file formats which include the header and footer information required by many agencies.
