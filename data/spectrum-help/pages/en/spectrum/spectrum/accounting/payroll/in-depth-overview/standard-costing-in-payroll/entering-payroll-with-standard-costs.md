# Entering Payroll with Standard Costs

Labor time entry and processing will remain the same as it
 is when using actual costs. Pre‐Time Card Entry and Time Card Entry will continue to use the
 actual labor rate.
When posted, the Payroll module will continue to report all information using actual
 costs.
