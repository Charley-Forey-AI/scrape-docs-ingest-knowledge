# Note

Spectrum® automatically stores Payroll add-on amounts as
 negative numbers (but they display as positive amounts on reports).
Therefore, if you would rather your add-ons be stored as absolute or positive values,
 you will need to create a formula in the Electronic File Formula Maintenance screen that
 will reverse the negative amounts and make them positive.
