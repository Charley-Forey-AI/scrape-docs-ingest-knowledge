# Employee Personal Information

Use this screen to enter additional details for the selected customer including contact, driver's license, and government reporting information.
If the Human Resources module is installed, some field information will default from the Human Resources module. However, the field-level information may be changed in this window without altering the Human Resources files.
Note: Click the Switch button at any time to display a
 different employee.
