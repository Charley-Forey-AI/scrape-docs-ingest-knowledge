# Print T4 Slips

The employer submits this form to the Canada Revenue Agency.
If doing year-end processing, see [Year End Update](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/year-end-update).
Important: For each employee, the Crystal Report prints two slips, and then it prints the T4 instructions. The CRA recommends that you print both on one sheet of paper.

If doing year-end processing, see [Export T4 File](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/payroll-year-end-processing-canada/t4-slip-maintenance/export-t4-file).
