# W-2 Form Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldsDescriptions
EntityDisplays only if cost centers are being used and entity tracking is also enabled in the current company.
Enter or select an Entity code. The software verifies you have security authorization for the selected entity before proceeding.

Employee codeEnter the employee code to print, or press Enter to print all employee codes.
Manually added W-2s (those that were not system-generated during the Build W-2 Forms) require an employee code, even if it's dummy one.

State IDEnter the state ID or press F4 to select a date from the Payroll Income Tax Table window.
Totals only?Select the checkbox to print totals only (for balancing purposes).Note: When this report is run for totals only, additional report
 information displays, including all of the employee's state income tax,
 wages, and tips information.
