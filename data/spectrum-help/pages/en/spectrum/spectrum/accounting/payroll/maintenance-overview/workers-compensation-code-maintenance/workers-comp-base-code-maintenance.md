# Worker's Comp Base Code Maintenance

Worker's Comp Base Code Maintenance is used to establish
 the codes, which will be used to track worker's compensation information.
Combined with Worker's Comp Code Maintenance, this feature allows the same code to be used
 for a specific type of work, regardless of the work's location; this is particularly
 helpful on multi-state payrolls.
Once you have completed this screen, click the Listing button to print a listing of the information entered in this screen, if applicable.
