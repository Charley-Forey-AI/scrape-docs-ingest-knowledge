# New Edit Tax Table - Overtime Rules

Use this tab to designate auto overtime rules by
 jurisdiction.
The Federal setup will be used unless a State override is present. State rules should only
 be set up when regulations in that locale supersede Federal standards. The Auto-Overtime
 Update first looks for State settings, and if not found, the Federal rules are used. One
 exception: if the current company processes Canadian payroll, then the update reads the Job
 assigned to the time card to determine whether it is a Federal job and if so, it will read
 the Federal rules, ignoring any Provincial settings.
Note: The calculations will be based on Federal and State setup; any
 'overtime' entries recorded for County or Local jurisdictions will be disregarded.
