# Kiosk Clock Hours Report

Use this screen to print a report for employees where the 'Track daily working hours' installation was enabled in the Employee Kiosk module.
Field
Description

Employee
Enter the employee code(s) you wish to include on the report, or press Enter to include ALL employees with daily working hours.

Department
Enter the department code(s) you wish to include on the report, or press Enter to include all departments.

Work date
Specify the starting and ending work dates to include on the report.

Suppress records where calculated hours match actual hours?
Select this checkbox if you wish to suppress records where the daily working hours match actual working hours.

Suppress records without hours logged?
Select this checkbox if you wish to suppress records where no daily work hours were logged.
