# Write Automatic Deposits - Cost Center Information

If cost centers are being used in this company, Spectrum
 verifies that you have authorization for ALL the cost centers included in the current payroll
 cycle.
Spectrum compares the cost centers entered in the set pay cycle screen with cost centers
 in your operator's assigned cost center scheme, and if any cost center is not included,
 then the file will not be created.
If your scheme includes employee override entries in Cost Center Scheme Maintenance, Spectrum will validate the cost centers assigned to the time cards and employee checks based on these overrides. The override cost center(s) supersede the cost center list defined for the scheme in general.
If you do not have security to run the check print an error message displays.
