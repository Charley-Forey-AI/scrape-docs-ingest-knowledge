# Create or Edit a Payroll Department

Steps to create a new payroll department code or modify an existing one.

1. On the Site Map screen, select Payroll > Maintenance > Department Expense.

1. In the Department code field, enter a new department code, or double-click on this field to select an existing department code to modify.

1. Using [New/Edit Department - Properties - Field Descriptions](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/department-expense-maintenance/newedit-department---properties/newedit-department---properties---field-descriptions) as a reference, complete the remaining fields.

1. Select OK to save.
