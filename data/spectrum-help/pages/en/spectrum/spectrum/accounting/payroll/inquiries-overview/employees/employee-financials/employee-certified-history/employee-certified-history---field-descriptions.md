# Employee Certified History - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields/Buttons
Descriptions

Employee code
Enter an employee code, or double-click
 on this field to select from a list of valid employee codes.
Entry button
Click this button to open the [Time Card History Inquiry](/en/spectrum/spectrum/accounting/payroll/inquiries-overview/employee-earnings-history/time-card-history-inquiry)
 window for the selected check. Use this window to edit certified information in
 the time card detail history lines for this transaction.
Check button
Click this button to open the [Payroll Check History Inquiry](/en/spectrum/spectrum/accounting/payroll/inquiries-overview/employee-earnings-history/payroll-check-history-inquiry) window for the selected check. Use this window to
 review the selected employee's check information.
