# Copy Setup Across Companies - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Copy from
Select the checkboxes for those files you wish to copy from the current company to another company. Clear any files that should not be copied to the new company.

Company
Enter the code for the company you want to copy the new payroll codes to, or press Enter to accept the default of ALL.

Overwrite existing payroll codes?
Select this checkbox if you want to add new codes and overwrite your existing payroll codes. Otherwise, accept the default and leave the checkbox clear if you want to add only those codes that are not set up yet and keep your existing payroll codes.
