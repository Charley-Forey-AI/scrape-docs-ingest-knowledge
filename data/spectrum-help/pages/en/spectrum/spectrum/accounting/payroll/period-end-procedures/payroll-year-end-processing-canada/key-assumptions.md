# Key Assumptions

You can refer to the key assumtions for
 payroll.

## No Confidential Payroll Support

The Canadian Revenue Agency (CRA) permits employers to submit multiple electronic files for the same Tax ID number. For this reason, the CP module will not support the creation of one "unified" file. Instead, the Payroll Administrator will submit multiple electronic T4 files to the CRA.

## No Amendment or Cancel Electronic File

The software will not support the creation of an electronic file that amends or deletes a previously submitted record. Instead, the Payroll Administrator will submit this information directly to the CRA's website. While the CRA supports these types of modifications via electronic file, we are going to follow the way we handle these changes in the U.S. as a manual process.

## No Cost Center Security

As with W-2 processing, cost center security is not applicable to T4 processing. The idea is that if you have security access to perform these functions, then you have security to see the information.

## All Canadian Companies Required to File Electronically

The software will not support the T4 Summary form. Instead, the Payroll Administrator will file electronically. As the CRA requires all employers with more than 50 T4 slips to file electronically, this shouldn't be unexpected by our clients.
