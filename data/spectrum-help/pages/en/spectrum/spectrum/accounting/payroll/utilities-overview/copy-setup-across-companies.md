# Copy Setup Across Companies

Use this screen to set up new income tax tables,
 jurisdiction and related tax codes, worker's compensation base codes, vacation/sick/holiday
 control codes, EEO classification codes and formulas by copying them from the current company
 and adding them to another company.
Only those files that are selected will copy, and the software will confirm that the
 corresponding G/L account codes are valid when writing to the destination company.
This screen is only available if you have Enterprise Management security access (security category EM). Likewise, new payroll codes can only be set up in those companies where you have security access to add codes manually into the corresponding maintenance screens (Tax Table Maintenance; Worker's Compensation Code Maintenance; Vac, Hol, and Sick Control Maintenance; EEO Classification Maintenance; and Formula File Maintenance).
