# Job Distribution Report

This report is used to review for unexpected large
 variances between the actual cost of payroll and the standard cost.
The [Job Distribution Report Selections Window](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/wage-code-file-maintenance/job-distribution-report-selections-window) can be run using standard costs when the detail version is selected.
