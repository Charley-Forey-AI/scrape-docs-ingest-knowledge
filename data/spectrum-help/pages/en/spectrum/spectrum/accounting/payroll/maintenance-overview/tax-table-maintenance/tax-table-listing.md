# Tax Table Listing

The Tax Table Listing report prints the tax table
 information entered in Tax Table Maintenance.
One screen prints for each tax table. Selection options to include income tax adjustments,
 other taxes, pay type exclusions, resident exemptions and related taxes may be used to
 specify the exact listing needed.
