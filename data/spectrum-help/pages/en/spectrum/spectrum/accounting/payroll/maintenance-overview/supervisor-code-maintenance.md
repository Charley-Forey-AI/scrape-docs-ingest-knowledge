# Supervisor Code Maintenance

Use this screen to assign a unique code to each
 supervisor's position, then enter associated information such as department and employee
 code.
Department codes must be entered in [Department Expense Maintenance](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/department-expense-maintenance) before entry of
 supervisor codes. Type a partial entry in the Search field or click the New
 Search button to look up an open item by code, name, department, employee,
 and more.
