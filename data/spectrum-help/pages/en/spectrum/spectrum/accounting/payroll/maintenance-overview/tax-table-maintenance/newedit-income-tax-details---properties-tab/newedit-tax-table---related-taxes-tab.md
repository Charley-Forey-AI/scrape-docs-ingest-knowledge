# New/Edit Tax Table - Related Taxes tab

Use this tab to define and manage additional tax codes. Additional tax codes added here will automatically be added to time card records when this tax code is used.
FieldDescription
Tax instructionsEnter instructions for the additional tax codes in this field.
Related code 1-5Select up to five additional tax codes for the selected tax table. The description of the tax code will display to the right of this field.Note: Entry of federal tax codes and duplicate tax codes is not allowed.
