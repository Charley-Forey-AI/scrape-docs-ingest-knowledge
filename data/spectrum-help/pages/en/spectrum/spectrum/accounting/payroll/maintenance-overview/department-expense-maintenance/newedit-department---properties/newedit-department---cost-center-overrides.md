# New/Edit Department - Cost Center Overrides

Use this tab to review the cost centers that will be valid for this department in Time Card Entry.
If a cost center is not listed here, then entry of time card lines for the department/cost center combination will not be permitted. This tab also provides the ability to assign override G/L accounts to specific cost centers for this department code. All authorized cost centers for the department code display, regardless of operator security.
Note: This tab is only available if the cost center feature is set to Yes in the Enterprise Installation screen.
