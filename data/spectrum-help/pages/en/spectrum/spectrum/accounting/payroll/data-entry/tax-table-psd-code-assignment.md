# Tax Table PSD Code Assignment

Use this screen to assign Pennsylvania Subdivision Codes (PSD) to Spectrum tax tables. PSD codes are necessary for a Pennsylvania employee working in a different city than their resident city, where tax rates may be different.
Enter a PSD code for the selected Tax table code, and select whether this code is for a Resident or Non-Resident. Non-Resident codes will appear as 'Resident Exmptions' on the Tax Table Maintenance screen. When this screen is closed, the build will process automatically. Tax table codes which have a PSD code assigned to them are exported to the custom history table and are available for printing on the PSD History Report.

- Click the Listing button to print a list of all PSD codes and tax percentages.

- Click the TC Loc Tax Code Assign button to change the local tax code for time card lines when an indirect department code is entered. This option will only be available when a Time Card Entry is in progress. The 'Work Local Tax Code' assignment is based on the User-Defined field setting for the Employee. If the User-Defined field does not have a valid tax code or is blank, the custom routine will not change the local tax code on the time card line.

- Click the Build PSD Hist button to export records from Payroll history for custom reporting.

- Click the PSD Hist Report to export all PSD records for Payroll history to an Excel spreadsheet to meet agency reporting requirements.

- Click the PSD Hist Maint button to view payroll records processed without tax or PSD codes. Use this screen to add a Location PSD code.
