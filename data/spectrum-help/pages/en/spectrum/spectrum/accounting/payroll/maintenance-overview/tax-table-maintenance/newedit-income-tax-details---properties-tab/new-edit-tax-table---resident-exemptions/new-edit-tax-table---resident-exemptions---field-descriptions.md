# New Edit Tax Table - Resident Exemptions - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Resident tax table
Enter the code to identify the resident state, county, or local tax table code. Be sure to refer to the corresponding work state, county, or local below. For example, if a county is recorded here, be sure to list the county codes below.

Work tax table
Enter the work state, county, or local tax code. This line will describe the taxability of employees working in this state, county, or local place while residing in the state, county, or local place identified at the top of the screen.

Income tax
Row 1: This row is used to determine how to tax the work state for income tax. Select the Exempt checkbox if wages earned while working in the state, county, or local place specified on this line but residing in the state, county, or local place identified at the top of this screen are exempt from state, county, or local income taxes.
Row 2: This row is used to determine how to tax the resident state income tax when working in the corresponding work state. Select the Exempt checkbox if wages earned while residing in the state, county, or local place specified on this line but residing in the state, county, or local place identified at the top of this screen are exempt from state, county, or local income taxes.

Unemployment
Row 1: This row is used to determine how to tax the work state for unemployment tax. Select the Exempt checkbox if wages earned while working in the state, county, or local place specified on this line but residing in the state, county, or local place identified at the top of this screen are exempt from state, county, or local income taxes.
Row 2: This row is used to determine how to tax the resident state unemployment tax when working in the corresponding work state. Select the Exempt checkbox if wages earned while residing in the state, county, or local place specified on this line but residing in the state, county, or local place identified at the top of this screen are exempt from state, county, or local income taxes.

Resident taxability
Row 1: This row is used to determine how to tax the work state for disability tax. Select the Exempt checkbox if wages earned while working in the state, county, or local place specified on this line but residing in the state, county, or local place identified at the top of this screen are exempt from state, county, or local income taxes.
 Row 2: This row is used to determine how to tax the resident state disability tax when working in the corresponding work state. Select the Exempt checkbox if wages earned while residing in the state, county, or local place specified on this line but residing in the state, county, or local place identified at the top of this screen are exempt from state, county, or local income taxes.
