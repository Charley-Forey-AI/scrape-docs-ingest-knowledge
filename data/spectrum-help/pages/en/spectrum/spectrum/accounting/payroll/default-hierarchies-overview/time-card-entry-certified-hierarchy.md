# Time Card Entry Certified Hierarchy

When determining if an entry is certified or not, the
 software first checks to see if the job or phase is certified.
The hierarchy below shows how the Certified checkbox defaults in the [Time Card Entry window](/en/spectrum/spectrum/accounting/payroll/employees/employee-entry/time-card-entry-window) window.
 Next, the software checks the employee file. Lastly, the software checks the Payroll
 Installation screen to see if pay groups are being used, and if so, the software checks a
 number of other places to see if the certified setting has been disabled.
