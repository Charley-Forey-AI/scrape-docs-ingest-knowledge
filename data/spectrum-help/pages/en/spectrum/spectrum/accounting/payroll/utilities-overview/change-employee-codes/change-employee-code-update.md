# Change Employee Code Update

This screen is used to update the changes entered in Change
 Employee Codes.
The user should first obtain an accurate data backup and a printout of the employee codes
 to be changed using Employee Code Change Listing.
Important: Read the following
 carefully:This update will perform code changes throughout the system. It is a
 productivity tool to assist the user in making code changes on a wide scale. Be
 advised that any custom files, pre-release and new files, and non-updated data
 files may not have been included in this automated process. Therefore, confirm
 that updates have been performed prior to using this feature. Review records
 carefully, including installation screens, immediately following completion of
 the program.
This update should never be performed while others are logged in to
 Spectrum. Confirm that this workstation is the only one currently in use prior
 to proceeding with the update.
Make sure you have an accurate data backup before proceeding. This update
 will change employee codes system wide according to employee code change
 maintenance information. You should have a copy of the employee code change
 listing before proceeding with this update.
