# Employer Pension Contribution Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Add-on codeEnter the add-on code you wish to include on the Employer Pension Contribution Report. Use the drop-down arrow to display the Payroll Add-ons window and select, or press Enter to accept the default of ALL codes recorded in the Pension Codes window.
Employee codeEnter the employee code you wish to include on the Employer Pension Contribution Report, or use the available drop-down arrow to display the Search Employees window.
From/to check dateEnter the check date range that you wish to include on the report, or use the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator) to select dates.
From/to period end dateEnter the period end date range that you wish to include on the report, or use
 the date calculator.
Report typeSelect the type of report you wish to print.
Pension CodesSelect to open the Add-on Pension Maintenance window.
This window displays a list of all the available non-cash add-on codes defined as pension. Review the list and then clear any pension checkboxes that you do NOT want to include on the Employer Pension Contribution Report. Select OK to save your changes and return to the Employer Pension Contribution Report.
