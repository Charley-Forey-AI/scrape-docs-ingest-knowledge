# New Edit Income Tax Details - Other Taxes tab

Use this window to designate income tax adjustments based
 on other taxes.
The system calculates the taxes in the order shown on the screen (Federal, FICA, Resident
 state, and so forth). Once the calculation for each tax type has been completed, it may be
 deducted from each subsequent tax before the final calculation is performed.
For each tax type shown (Federal, FICA, resident state, work state, county) select:

- No - If the pre-computed tax amount is Not deducted from the base.

- Before calculation - If the pre-computed tax amount should be deducted from the base Before tax calculations.

- After calculation - If the pre-computed tax amount should be deducted from the base After tax calculations.

Select the Deduct non-resident income tax from resident tax amount? checkbox if you want to reduce the resident income tax deduction by the amount of the "work" location income tax. The resident income tax will not be set to less than $0.00.
Tax adjustments - Click to add or remove tax adjustment codes, or click the Maintenance link to open the [Tax Adjustment Code Maintenance](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/tax-table-maintenance/newedit-income-tax-details---properties-tab/newedit-tax-table---other-taxes/tax-adjustment-code-maintenance) window where you can set up adjustment codes for income tax calculations. When a code is added, it is automatically assigned the next higher sequence number. If the sequence in which the adjustments are computed needs to be altered, simply remove and re-add the codes in the necessary order.
