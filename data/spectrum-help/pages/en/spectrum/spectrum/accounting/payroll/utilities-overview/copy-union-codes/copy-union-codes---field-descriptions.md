# Copy Union Codes - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

From union code
Enter the union code from which you wish to copy file information. Entry in this field is required. An alert message will display if the union code entered has an Inactive status, but you will be allowed to continue entry.

To union code:
Enter the union code to which you wish to copy file information. Entry in this field is required.
Note: Sometimes it is helpful if the new union code has the same
 first three characters as the old union. For example, from ELE01 to ELE02.If the To union code does not yet exist, a new union
 and any active wage codes will be created effective the current
 processing date less one day.If the To union
 code already exists, the software displays the following
 message:
"The target union code already exists. Only new wage codes, deductions, add-ons, benefit tiers, fringe benefits and pay rate information will be added to the target union. Auto-overtime settings will be reset. The union name, status and craft will not be updated."

Copy rate history?
Select this checkbox to copy rate history; otherwise, leave this checkbox clear.
