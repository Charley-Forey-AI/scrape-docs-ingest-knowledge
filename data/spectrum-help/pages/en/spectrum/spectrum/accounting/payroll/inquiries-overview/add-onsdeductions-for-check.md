# Add-ons/Deductions For Check

This screen provides at-a-glance information about an employee's add-ons or deductions, including descriptions and amounts.
Field
Descriptions

Employee
The employee code and name display. No entry is allowed.

Check
The check number displays. No entry is allowed.

Code
The add-on/deduction codes associated with this check display. No entry is allowed.

Descriptions
The corresponding code description displays. No entry is allowed.

Amount
The add-on/deduction amounts for this check display. No entry is allowed.
