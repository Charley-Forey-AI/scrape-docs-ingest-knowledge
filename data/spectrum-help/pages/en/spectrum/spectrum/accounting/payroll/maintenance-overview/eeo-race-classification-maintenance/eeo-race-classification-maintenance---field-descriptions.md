# EEO Race Classification Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Edit button
Click this button to edit an existing EEO race classification description.

Standard button
Click this button to display the Standard EEO Race Classification window. Use this window to copy the standard EEO race classifications for use in the current company.

Short description
The short description you have selected to edit displays. Press Enter to accept the current description or type a new description.

Report description
The report description you have selected to edit displays. Press Enter to accept the current description or type a new description.
