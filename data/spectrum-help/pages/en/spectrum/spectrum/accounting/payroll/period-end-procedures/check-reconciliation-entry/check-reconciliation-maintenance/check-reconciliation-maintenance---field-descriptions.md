# Check Reconciliation Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Check/reference number
If the transaction to be maintained is a check, enter the check number. The check information will display in the remaining fields on the screen.
If the transaction is an adjustment or a deposit, enter the reference number.

Transaction type
Select the transaction type code for the transaction.

Properties

Employee code/reference
If the transaction type is a check, the employee code displays. Double-click on this field to display the Search Employees window and select from a list of valid employee codes.
If the transaction type is an adjustment or deposit, enter the reference number.

Employee name/description
If the transaction type is a check, the employee name displays.
If the transaction type is an adjustment or deposit, enter a description of the transaction.

Transaction date

Statement date

Amount
Enter the dollar amount of the transaction. Do not enter negative numbers except for adjustments or void checks.

Cleared amount
Enter the dollar amount that cleared the bank.
