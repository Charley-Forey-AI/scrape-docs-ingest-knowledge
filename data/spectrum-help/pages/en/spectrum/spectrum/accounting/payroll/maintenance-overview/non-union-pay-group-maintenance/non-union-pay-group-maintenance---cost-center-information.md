# Non-Union Pay Group Maintenance - Cost Center Information

When adding, editing or deleting a deduction or add-on
 code, Spectrum verifies that the operator has permission to access the deduction/add-on code.
Spectrum compares the deduction's list of shared cost centers with cost centers in your
 operator's assigned cost center scheme, and if there are no common cost centers, then
 entry of that deduction/add-on code is disallowed.
Once a deduction/add-on code has been assigned to a pay group (by an authorized operator), then that deduction/add-on code will be used by all operators for processing involving that non-union pay group code.
When adding or changing an employee for the non-union pay group, Spectrum will allow entry of the employee code only if you have permission to access that employee. Spectrum compares the cost center assigned to the employee with cost centers in your operator's assigned cost center scheme, and if the cost center is not included, then that employee is not displayed on the screen and cannot be added. Likewise, when adding, editing, or deleting a deduction or add-on code, Spectrum verifies that the operator has permission to access the deduction/add-on code.
