# State Disability Insurance Report - Canadian Lexion
 Information

If the Canadian lexicon designation (en-ca) is selected in
 Company Installation, the name of this report will display as Employment Insurance Report,
 and some labels will also be changed to the Canadian lexicon.
For exanple, "Province" instead of "State", and "cheque" instead of
 "check".
