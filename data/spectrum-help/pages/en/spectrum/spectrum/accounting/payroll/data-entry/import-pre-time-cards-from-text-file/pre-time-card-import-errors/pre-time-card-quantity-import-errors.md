# Pre-Time Card Quantity Import Errors

Use this summary screen to make any necessary corrections
 before proceeding to import the information into Pre-Time Card Entry.
By placing the cursor on a current record and selecting it, you can see what corrections
 (if any) need to be made by looking at the Error column. This column also tells you exactly which fields in the
 current record need to be corrected.
This screen will auto-copy imported quantities to designated cost types upon
 approval of time cards sent to this screen during the import update. If no errors are
 found, the record will be transferred to Pre-Time Card Entry and
 additional pre-time card quantity records will be generated based on the settings in Job Cost Installation > Cost Types. If errors are found, the record will not be transferred to Pre-Time Card
 Entry and no records will be generated for other cost types.
