# Payroll Update - Cost Center Information - Error Correction

When the Enterprise Installation option for cost centers is
 set to Yes in this company, Spectrum
 will validate each record before posting to General Ledger. It will check that the cost center
 assigned to the G/L account for that line is valid.
The error window will also allow the cost center to be modified. It will validate that
 the cost center is valid for the operator, and that it is an allowed cost center for
 that G/L account.
