# New/Edit Union Overtime Rules

Use this tab to set up union-specific Auto-overtime rules.
 The default hierarchy for Union overtime rules is as follows.

1. Union rules (or optionally Job-then-Union), if defined

1. Job rules, if defined

1. State rules, if defined

1.  Federal tax code of employee.
