# Copy Non-Union Pay Group Codes - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

From pay group
Enter the pay group code from which you wish to copy information. Entry in this field is required.

Pay group
Enter the new pay group code to create. Entry in this field is required.

Descriptions
The corresponding pay group description displays; this may be overwritten.

Copy employee specific detail
Select this checkbox to copy employee phase/detail rates information from the existing Non-Union Pay Group Maintenance record to the newly created group.

Copy phase override detail?
Select this checkbox if you want to copy over the phase overrides. Phase override is copied from the Phase Rate Override window of the Non-Union Group File Maintenance screen.
