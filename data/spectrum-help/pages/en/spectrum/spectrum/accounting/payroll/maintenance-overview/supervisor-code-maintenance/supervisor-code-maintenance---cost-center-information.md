# Supervisor Code Maintenance - Cost Center Information

If cost centers are being used, then when adding or
 changing a supervisor code, Spectrum will allow entry of an employee code only if you have
 permission to assign that code.
Spectrum compares the employee's assigned cost center with cost centers in your
 operator's assigned cost center scheme, and if the cost center is not present, then that
 employee cannot be assigned.
Once an employee code has been assigned to a supervisor code (by an authorized operator), the assigned supervisory employee will be used by all operators for processing involving that supervisor code.
