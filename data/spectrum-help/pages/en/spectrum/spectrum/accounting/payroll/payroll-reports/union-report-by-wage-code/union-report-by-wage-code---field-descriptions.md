# Union Report by Wage Code - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

Union
Enter the union code to include on the report, or press Enter to print ALL union codes.

Date type
Select whether to print the report by Pay period-end date or Work date.

From/to date
Depending on the date type selected above, enter the first and last period end or work dates to include on the report. The current Payroll processing date defaults.

Show entire social security number?
Select this checkbox to display the full social security number. This checkbox defaults to clear unless a saved filter is present.

Sort by
Select to sort employees by Alpha code or Employee code within each union included in this report.
