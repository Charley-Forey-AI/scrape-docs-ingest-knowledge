# G/L Error Correction Screen

This screen displays automatically if any errors are
 encountered during the printing of the Payroll G/L Report.
Use this screen to make any necessary corrections before proceeding with the report
 printing and update. The Errors for current
 line box will help you to locate the errors that need correcting, and
 clicking on the Expand button
 displays the year and period associated with the transaction date.
Note: This screen only applies when posting items to G/L. If
 the 'Payroll' checkbox in General Ledger Installation is cleared, this window will not
 open, and the update will proceed without posting to G/L.
