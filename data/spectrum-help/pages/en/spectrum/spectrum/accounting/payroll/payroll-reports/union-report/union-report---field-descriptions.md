# Union Report - Field Descriptions

A reference for completing the fields on this screen.
Fields/ButtonsDescriptions
Selections
UnionEnter the union code to include on the report, or press Enter to include all union codes.
Date typeSelect whether to print the report by Pay period-end date or Work date.
From/to dateDepending on the date type selected above, enter the first and last period end or work dates to include on the report. The default value is the current Payroll processing date.
Show entire social security number?Select if you want the full social security number to appear in the report.
Report type
Detail Select to print the report using the basic detail format.
Extended detailSelect to print the report in expanded format.
Extended detail with employee fringe detailSelect to print detailed fringe benefit information by employee.
Summary by employeeSummarizes fringe benefits and deductions/add-ons by employee. The report totals provide the fringe benefits and deductions/add-ons for all unions included on the report.
Summary by deduction/add-onThis Summary report groups union deductions and add-ons separately (in alphabetical order), and provides the union totals amounts. The values displayed on this report format are taken from the Deductions and Add-ons columns as seen on the basic Summary by employee report format.
Sort byChoose Employee code or Alpha code within each union included in this report.
StatusChoose which unions should should be included in the report based on its current status.
Check cost center
Cost group
This section appears only if cost centers are being used in the current company. Enter a valid cost group code or cost center code to filter data.
If you entered a cost center or cost group in the starting screen, amounts from the check are included only if the pay check cost center is the same as the cost center on the starting screen. This report conditionally prints the entity name.
