# Pre-Check Register Selections Window - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Employee code
Enter the employee code to include on the Payroll Pre-Check Register, use the SuperSelect options, press F4 or double-click on this field to select from a list of available employee codes, or press Enter to accept the software default of ALL.

Sort by
Select to organize the Payroll Pre-Check Register by Employee code or Alpha reference.

Display tax code detail?
Select this checkbox to include detailed information regarding the selected employee's tax code, if applicable; otherwise, leave this checkbox clear.

Display deduction and add-on detail?
Select this checkbox to include detailed information regarding the selected employee's Payroll deduction and add-ons, if applicable; otherwise, leave this checkbox clear.
