# P/R G/L Distribution Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

G/L account Employee code Check number Company code
Enter a General Ledger account code, employee code, check number, and company code, or press Enter to accept the system default of ALL.

Date
Enter the date with which to begin and end this report, or press Enter to leave this field blank and print from the beginning and end with the current payroll processing date default.

Format
Select whether to print the report using the Summary or Detail format.

Date type
Press Enter to accept the system default to print by G/L date, or select Check or Period end date.

Check type
Select the checkbox(es) for any combination of desired check types (Manual, Regular or Void).
