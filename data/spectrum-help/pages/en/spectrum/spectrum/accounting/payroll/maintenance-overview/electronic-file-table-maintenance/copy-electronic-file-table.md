# Copy Electronic File Table

Use this screen to copy information from one file table to another.
Field
Descriptions

From table code
Enter the table name you want to copy the information from, or double-click on this field to select from a list of available tables. Master table codes, as set up in Electronic Master File Maintenance, cannot be entered in this field.

To table code
Enter the table name you want to copy the information to. Master table codes, as set up in Electronic Master File Maintenance, cannot be entered in this field.

Proceed with update?
Select Continue to continue or Cancel to exit without saving your changes.
