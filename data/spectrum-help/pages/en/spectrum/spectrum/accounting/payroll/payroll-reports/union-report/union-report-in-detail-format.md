# Union Report in Detail Format

The fringe benefit Rate amount shown on this report is the
 rate as of the current Payroll processing date (system processing date).
If you wish to print an older rate, you will need to change the
 processing date in the [P/R Processing Date Change](/en/spectrum/spectrum/accounting/payroll/data-entry/pr-processing-date-change) screen to the period ending date of the report. Print the report
 and then change the processing date back to the current date.
