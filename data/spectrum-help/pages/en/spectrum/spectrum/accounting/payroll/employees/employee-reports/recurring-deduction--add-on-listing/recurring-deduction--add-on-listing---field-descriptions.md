# Recurring Deduction / Add-on Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldDescriptions
Employee
Deduction/add-on
Employee status
Enter an employee code, deduction/add-on code, and employee status to include on the report, or press Enter to include ALL records.
EntityIf cost centers and entity tracking is enabled in Enterprise Installation, you can enter a cost center entity here in order to display entity-specific recurring deductions and add-ons information on the listing.
Code statusSelect the deduction / add-on status type(s) you want to include in this report, or press Enter to accept the default of ALL.
