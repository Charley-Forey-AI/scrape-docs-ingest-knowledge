# Average Hours Report - Field Descriptions

Use the table below for reference when completing the report
 starting screen selections.
Fields
 Descriptions

Selections

Employee
 Enter or select one or more employee codes.
This field defaults to ALL unless a saved
 selection is present.

Department
 Enter or select the department(s) you want to
 include on this report.
This field defaults to ALL unless a saved
 selection is present.

From/to work date
 Specify a date range for the report.
Hours worked during this window are
 totaled and then averaged by employee and pay type.

Pay types
 Select the pay types you want to include on
 the report.
By default, all are selected
 except for JX and Unpaid.

Health coverage status

FT, PT, Variable hour, Unassigned
 Select the types of employees you want to
 include on the report based on their health coverage status (as
 set up in the [Employee Main Properties](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-main-properties) screen).


 Employee status type
Active, Inactive, Terminated, Deceased
 Select the types of employees you want to
 include on the report based on their employment status (as set
 up in the [Employee Main Properties](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-main-properties) screen).
