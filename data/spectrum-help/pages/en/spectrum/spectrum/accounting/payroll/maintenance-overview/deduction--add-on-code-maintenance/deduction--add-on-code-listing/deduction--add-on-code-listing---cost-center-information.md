# Deduction / Add-on Code Listing - Cost Center Information

If cost centers are being used, this screen includes a
 selection for a cost group.
When a cost group or cost center is specified, then the report will show only
 deductions/add-ons assigned to that cost group/cost center. This screen also includes an
 option to include cost center security detail on the listing.

- If you specify a cost center, you must have permission to access that cost center before proceeding.

- When running the Deduction/Add-on Code Listing including cost center detail, Spectrum will list all cost centers for the code.
