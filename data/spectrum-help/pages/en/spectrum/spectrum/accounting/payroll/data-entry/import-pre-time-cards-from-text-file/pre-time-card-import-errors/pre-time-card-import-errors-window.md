# Pre-Time Card Import Errors Window

Use this window to make corrections to, or delete, erroneous pre-time card entries that were discovered during import.
Some fields will be disabled based on the department selected (for example, if equipment is selected, the Job, Phase, and Cost type fields will be irrelevant).
Note regarding 'Direct work order cost' departments:
If the Department is a 'Direct work order cost' department, the
 following fields will appear in place of the existing fields in the screen below. If the
 time card line originated in Work Order > Labor Hours Entry, or the time card has already been selected for billing in Work Order > Labor Hours Entry, these fields will be display-only.

## Make Corrections to an Erroneous Pre-Time Card Entry

1. Highlight a line and click the Edit button to open the Pre-Time Card Import Errors window. Descriptions of the fields that display in this window are the same as the fields found in the [Time Card Entry window](/en/spectrum/spectrum/accounting/payroll/employees/employee-entry/time-card-entry-window).

1. Errors for this record will be marked as 'Invalid' or with a red circle with an 'X' through it. Click the View Error Messages button to open the Errors for Current Record window and review all the current errors, and the fields associated with the errors, for this entry.

1. Click OK to return to the Pre-Time Card Import Errors window and make changes in the appropriate fields.
