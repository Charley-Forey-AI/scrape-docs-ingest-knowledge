# Union File Maintenance - Cost Center Information

If cost centers are being used in this company, then when
 adding or changing a union code, Spectrum will allow entry of a G/L account code only if you
 have permission to assign that code.
Spectrum compares the G/L account's list of shared cost centers with cost centers in
 your operator's assigned cost center scheme, and if there are no common cost centers,
 then that G/L account cannot be assigned. Likewise, when adding, editing, or deleting a
 deduction or add-on code, Spectrum verifies that you have permission to access the
 deduction/add-on code.
Once a G/L account code and a deduction/add-on code have been assigned to a union code (by an authorized operator), that G/L account and deduction/add-on code will be used by all operators for processing involving that union code.
