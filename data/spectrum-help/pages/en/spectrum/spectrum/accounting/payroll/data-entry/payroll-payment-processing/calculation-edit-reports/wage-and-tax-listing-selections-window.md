# Wage and Tax Listing Selections window

Use this window to select which information will display on the Wage and Tax Listing and the format in which the information will display.
Field
Descriptions

Employee code
Enter the employee code to include on the Wage and Tax Listing, double-click on this field to select from a list of available employee codes, or press Enter to accept the software default of ALL.

Tax code
Enter the tax code to include on the Wage and Tax Listing, double-click on this field to select from a list of available tax codes, or press Enter to accept the software default of ALL.

Sort by
Select to organize the Wage and Tax Listing by Employee code or Tax code.

Report version
Select to print the Detail or Summary format of the Wage and Tax Listing.
