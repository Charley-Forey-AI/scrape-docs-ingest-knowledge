# Time Card Entry window

Use this window to add new time card lines.
As entries are completed in this window, they will display in the columns on the Time Card
 Entry starting screen.
If you are editing existing time card lines, some of the key fields will
 be disabled (for example, Employee, Pay
 type, Company
 code, and when applicable, Job).
 Similarly, some fields will be disabled based on the department selected (for example, if
 equipment is selected, the Job,
 Phase, and Cost type fields will be irrelevant).
If errors occur during data entry, a warning message will display and a
 View Error Messages button
 will display on the lower right of the window. Click this button to open the
 Error For Current Record window and review any current errors for
 this record. To make corrections, simply double-click an error message line or click
 OK to return to the
 Time Card Entry window and make your changes there. A Meter button displays if you enter an
 equipment pay type and valid equipment code, and have security access to the Equipment Control > Meter Reading Transaction Entry screen. Click this button to open the [Meter Readings](/en/spectrum/spectrum/accounting/payroll/data-entry/layoff-check-entry/layoff-check-entry-window/meter-readings) window and enter meter readings for the specified equipment.

## Note Regarding Equipment Usage Pay Types

When a transaction for pay type ER, EO, ED, or EU is added, the software will
 evaluate whether the equipment code assigned to the time card entry has any active
 attachments. If so, additional EU time cards will be generated automatically or the
 [Attached Equipment window](/en/spectrum/spectrum/accounting/payroll/data-entry/pre-time-card-entry-by-job/attached-equipment-window) will display (based on
 the Automatically add transactions for
 attached equipment setting in the Equipment Control Installation -
 Properties screen).
