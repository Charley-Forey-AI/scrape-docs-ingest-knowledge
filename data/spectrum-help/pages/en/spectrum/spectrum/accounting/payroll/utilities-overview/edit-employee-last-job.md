# Edit Employee Last Job

Use this screen to edit the employee's last job when you
 need to print checks in job number order.
Spectrum automatically maintains this field as part of the Payroll Update process. However,
 there is a flag in the screen for designating whether to suppress updating the last job
 during the check calculation process for certain employees. This allows you to suppress
 this update and put in a location like "office," then the selected employee's checks will
 be sorted with the office personnel instead of by job.
Important: When this screen is first opened, the Build
 Employee List window will open first to specify employee, department, last job, and
 employee status to include on this screen.
