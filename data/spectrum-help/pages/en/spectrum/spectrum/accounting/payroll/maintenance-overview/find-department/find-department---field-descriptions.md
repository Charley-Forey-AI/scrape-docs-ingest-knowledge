# Find Department - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Department code
Use this field to search on the department code, or double-click in this field
 to select from a list of available codes. To accept the default of ALL,
 press Enter.Tip: If you only know the first characters of the
 alpha-numeric code you want to search on, enter these characters and then
 enter an *
 (asterisks).

Descriptions
Use this field to search on the department description. To accept the default
 of ALL, press Enter.Tip: If you only know the first
 characters of the alpha-numeric description you want to search on, enter
 these characters and then enter an *
 (asterisks).

Worker's compensation code
Use this field to search on the worker's compensation code, or double-click in
 this field to select from a list of available codes. To accept the default
 of ALL, press Enter.Tip: If you only know the first
 characters of the alpha-numeric code you want to search on, enter these
 characters and then enter an *
 (asterisks).

Direct cost
Select the checkboxes to include only departments where the Direct cost = Non-direct cost, Job cost, Equipment cost or Work order cost (or any combination of these four checkboxes) in your search results.

Reset
Click this button to return all the fields in this window to their original defaults (which include ALL departments and all direct cost options).
