# New/Edit Deduction/Add-on Code - Properties - Cost Center Information

When adding or changing a deduction/add-on code, Spectrum
 allows entry of a G/L account code only if you have permission to assign that code. Spectrum
 compares the G/L account's list of shared cost centers with cost centers in your operator's
 assigned cost center scheme, and if there are no common cost centers, then that G/L account
 cannot be assigned.
Once a G/L account code has been assigned to a deduction/add-on code (by an authorized
 operator), then that G/L account will be used by all operators for processing involving
 that deduction/add-on code.
