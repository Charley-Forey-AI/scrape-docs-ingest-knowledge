# Check Adjustment Entry

This screen should be used to enter deductions whenever a manual Payroll check is entered.
Once checks have been calculated in [Check Calculation](/en/spectrum/spectrum/accounting/payroll/data-entry/payroll-payment-processing/check-calculation), use this screen to enter any required adjustments. This allows you to make any necessary changes to the calculated federal income tax deduction, FICA, SDI, worker's compensation deductions, and other deductions.
If a deduction or add-on code used by another code for adjusting the gross pay
 (as set up in the Deduction/Add-on Code Maintenance > Gross Pay Adjustment window) is changed in the Deductions for Check or
 Add-ons for Check windows during the payroll cycle, an
 Update Dependent Codes window appears before the check
 calculation results display. The Update Dependent Codes window
 allows you to change the code amounts or you can click Cancel to retain the original code
 amounts.
Note: Changes cannot be made to a check if it has been voided and the original check is still in the history file.
Check Adjustment will apply limits to union add-ons and deductions based on setup in the [New/Edit Deduction/Add-on Code - Limits](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/deduction--add-on-code-maintenance/create-or-edit-a-deduction-or-add-on-code/newedit-deductionadd-on-code---limits) tab.
In the rare case where you might need to treat the taxable/non-taxable attributes of the various earnings codes independently (for example, you have a FICA-exempt bonus time card line and a FICA--taxable deduction), you may need to use an add-on to make sure the taxes calculate correctly.
