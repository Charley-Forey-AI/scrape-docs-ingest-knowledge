# Quantity Complete Update

Use this update to update records from the Payroll Pre-Time Card Quantity Transaction Table (PR_PRE_TC_QTY_ENTRY_MC) to Job Cost Quantity History.
Fields
Descriptions

Job
Enter the job code or codes you want to include in the update.

Batch code
The batch code will default from the batch code in context on the Entry screen. Enter a different batch or job number corresponding to the pre-time card information to be printed, or leave this field blank to print ALL batches and jobs entered in Quantity Complete Entry.

From/to work date
Enter the work date range that you want to include in the update.
