# Employee Earnings History Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Employee Resident state tax code Employee status Bank account
Enter the employee code, employee's resident state, employee status, and bank account number, or press Enter to include ALL on the report.
Note: The Bank account field is only available if the Cash
 Management module is installed.

Check date
Enter the start and end dates of the checks to include on this report, or press Enter to begin with the first check and end with the current Payroll processing date.

Period end date
Enter the period end start and end dates to include on this report, or press Enter to begin with the first period end date and end with the current Payroll processing date.

Format
Select a format for the report:

- Summary

- Detail

- Employee summary
