# Note

If you want all of your employees to have the same check
 date, regardless of their employee status (Terminated, Inactive, Deceased, or Active), you
 must first open the Electronic File Table Maintenance screen and select the check date
 (PR.EHDATE (1)) and then set the Period
 field to Current.
Then in the Build Electronic Data File screen you need to set the 'to' and 'from' check
 dates equal to each other (and corresponding to the current check period). After you run
 the build, any employees with an "outdated" check date will be updated to the current check
 period and will appear in the electronic file you create or print.
