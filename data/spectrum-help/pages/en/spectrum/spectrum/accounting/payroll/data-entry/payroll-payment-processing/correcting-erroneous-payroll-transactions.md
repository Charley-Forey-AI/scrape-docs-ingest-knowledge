# Correcting Erroneous Payroll Transactions

If an employee enters the wrong job, phase, or number of
 hours on his/her time card and the error is discovered after the payroll cycle is complete,
 perform the following steps to correct the erroneous transaction(s).
There are two methods you can use to correct incorrect Payroll transactions.

- Use the first option if you do not have access to the Job Cost module. This method corrects/updates the Job Cost, General Ledger, and Payroll modules.

- Use the second option if you have access to the Job Cost module. This method only corrects/updates the Job Cost module.
