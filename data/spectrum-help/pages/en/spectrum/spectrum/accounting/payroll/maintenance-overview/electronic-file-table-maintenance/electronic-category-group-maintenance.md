# Electronic Category Group Maintenance

Use this screen to set up any categories for deductions or add-ons; for example, you may want to build a file using only the 401(k) deductions. Categories can be used to filter the results or formulas.
In order to use categories, the variable must come from the Payroll Deduction/Add-on Code Master file (PR.VDCTL) or Check History Deductions / Add-ons (PR.EHVD).
