# User-Defined Fields Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Company
Enter the company code to print, use the SuperSelect options, press F4 or double-click on this field to select from a list of available company codes, or press Enter to accept the software default of ALL.

Include alpha fields?
Select this checkbox to include alpha user-defined fields in the report; otherwise, leave this checkbox clear.

Include date fields?
Select this checkbox to include date user-defined fields in the report; otherwise, leave this checkbox clear.

Include numeric amounts?
Select this checkbox to include amount user-defined fields in the report; otherwise, leave this checkbox clear.

Print the validation codes?
Select this checkbox to include validation codes in the report; otherwise, leave this checkbox clear.
