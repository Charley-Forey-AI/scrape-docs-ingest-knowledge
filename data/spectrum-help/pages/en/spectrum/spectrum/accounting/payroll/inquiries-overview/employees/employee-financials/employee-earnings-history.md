# Employee Earnings History

The Employee Earning History Inquiry screen presents all
 earnings, taxes and deductions for the selected employee on a check-by-check basis.
If no checks have been issued for the employee, the list box portion of the screen will be
 blank. The screen displays information in check number order.
Clicking any one of the column heading buttons allows you to sort the
 line transaction information accordingly. For example, clicking the Check column will organize the line
 transactions according to check number.
Note: Click the Switch button at any time to display a
 different employee.
