# Export T4 File

Use this screen to compile Canadian T4 Tax information and create the XML file for year-end electronic filing.
If doing year-end processing, see [Print T4 Slips](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/payroll-year-end-processing-canada/print-t4-slips).
Please see the [CRA's document](http://www.cra-arc.gc.ca/E/pub/tg/rc4120/)
 "Employers' Guide Filing the T4 Slip and Summary" for information on how to complete
 this screen.

If doing year-end processing, proceed as directed at [Year End Processing Screen](/en/spectrum/spectrum/year-end/year-end-for-canada-payrolls/processing/year-end-processing-screen-can).
