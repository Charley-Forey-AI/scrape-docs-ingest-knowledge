# Time Card History Inquiry

Use this window to view certified information in the time card detail history lines for this transaction. The columns in this window are both moveable and sortable, to allow you to customize the display.

- Click the Employee images link to view the selected employee time card.

- Click the Job Images button to view the selected job time card.

- Click the `Approval history` button to view Workflow History for a selected time card. This button only displays if the Workflow module is serialized and the Time Card Approval workflow is being used.

- Click the Related Tax Codes button to view all of the related tax codes set up for the selected time card line.
