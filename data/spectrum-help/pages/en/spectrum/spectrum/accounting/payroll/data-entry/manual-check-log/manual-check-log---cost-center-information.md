# Manual Check Log - Cost Center Information

If cost centers are being used in this company, then when
 recording manual checks, Spectrum permits the employee to be entered only if you have
 permission to access the employee.
You may access all checks and edit records, but you cannot modify the employee code for
 a check if you do not have cost center security for that employee.
If your operator's scheme includes override settings for employee entries in Cost Center Scheme Maintenance, this screen validates the cost center assigned to the employee-based on these overrides. The override cost center(s) supersede the cost center list defined for the scheme in general.
