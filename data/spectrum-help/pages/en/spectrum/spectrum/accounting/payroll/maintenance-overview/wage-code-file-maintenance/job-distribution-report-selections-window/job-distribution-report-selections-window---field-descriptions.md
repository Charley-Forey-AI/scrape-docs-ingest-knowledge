# Job Distribution Report Selections Window - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Job number
Enter the job number, or press F4 or double-click on this field to select from a list of
 available jobs, or use the SuperSelect option, or press Enter to accept the
 default selection of ALL.

Page break between jobs?
Enter Yes for screen breaks between jobs. Otherwise, enter No for no screen breaks between jobs.

Report version
Click to select Summary to print the summary version of the report, or click to select Detail to print the detailed version of the report.

Display standard cost variance on report?
Select this checkbox if you selected the Post to Job Cost using standard labor
 rates? option on the Payroll Installation > Defaults screen and want to include standard labor costs on the
 Standard Cost Variance Report.
For more information about standard job rates, refer to the [Standard Costing in Payroll](/en/spectrum/spectrum/accounting/payroll/in-depth-overview/standard-costing-in-payroll) in-depth article.
