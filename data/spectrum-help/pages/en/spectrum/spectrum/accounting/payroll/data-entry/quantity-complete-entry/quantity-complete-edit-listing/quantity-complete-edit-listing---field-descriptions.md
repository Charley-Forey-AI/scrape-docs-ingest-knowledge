# Quantity Complete Edit Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job
Enter the job code or codes you want to include on the listing.

Batch code
Enter the batch or job number corresponding to the pre-time card information to be printed. Press Enter to print ALL batches and jobs entered in Quantity Complete Entry.

From/to work date
Enter the work date range that you want to include on this listing.
