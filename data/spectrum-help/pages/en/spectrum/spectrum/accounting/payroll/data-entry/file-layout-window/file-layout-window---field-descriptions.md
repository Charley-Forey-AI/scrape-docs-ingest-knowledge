# File Layout Window - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Descriptions
Each file's description displays. No entry is allowed.
Click OK to exit the File Layout window.

Type
If A displays, the file type is alphanumeric; if N displays, the file type is numeric. No entry is allowed.
Click OK to exit the File Layout window.

Max Len
The maximum length of the file displays. No entry is allowed.
Click OK to exit the File Layout window.
