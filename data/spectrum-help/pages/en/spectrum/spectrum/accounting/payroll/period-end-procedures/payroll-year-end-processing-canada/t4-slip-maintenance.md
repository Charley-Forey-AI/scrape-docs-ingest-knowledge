# T4 Slip Maintenance

This screen only displays if the Payroll Installation
 option for 'Payroll reporting' is set to Canada.
If doing year-end processing, first [Build T4 Slips](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/payroll-year-end-processing-canada/build-t4-slips).
Use this screen to enter T4 Tax information.
The Payroll Administrator will build the T4 information, will make any needed modifications, and then will print out the T4 slips to give to the employee. Spectrum does not provide a T4 Summary form; instead, all companies with Canadian payroll must file electronically to the CRA.

If doing year-end processing, see [Tax Table Maintenance](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/tax-table-maintenance).
