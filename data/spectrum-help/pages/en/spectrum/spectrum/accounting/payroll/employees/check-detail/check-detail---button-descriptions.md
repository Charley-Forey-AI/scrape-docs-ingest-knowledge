# Check Detail - Button Descriptions

Use the table below for reference for button functions.
Buttons
Descriptions

Earnings detail
Click this button to display the Earnings Detail window. This window provides you with at-a-glance information pertaining to the employee's related tax codes and descriptions, and gross earnings.

FICA
Click this button to display the FICA window. The employee's Social Security and Medical aid deductions display

SIT
Click this button to display the SIT window. Use this window to review the employee's state income tax.

SDI
Click this button to display the SDI window. Use this window to review the employee's state disability deductions.

County
Click this button to display the County window. Use this window to review the employee's county income tax.

Local
Click this button to display the Local window. Use this window to review the employee's local income tax.

Worker's comp
Click this button to display the Worker's Compensation window. The employee's worker's compensation class, state, description and deduction amounts display.
