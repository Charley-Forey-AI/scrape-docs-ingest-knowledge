# Payroll Trial Balance Report

The Payroll Trial Balance Report is used to print the trial
 balance. Employee code and status, as well as pay type and frequency may be specified to
 obtain the report desired.
This trial balance tests for errors in the year-to-date payroll information by checking
 employee totals against the payroll earnings history.
Once you have completed this screen, click the Print or Preview button to review the information, if applicable.
