# Pay Cycle Calendar Maintenance

This screen may only be used if the Payroll Installation screen's Utilize pay cycle calendar checkbox is selected.
Use this screen to view and/or edit a list of regularly occurring payments, including period end date and check date. The information in this screen may be verified against information entered in Set Payroll Cycle.
Fields
Descriptions

Period
The calendar period displays. No entry is allowed.

End date
Enter the end date for this period or double-click on this field to select a date from the Date Change window.

Check date
Enter the check date for this period or double-click on this field to select a date from the Date Change window.

Build Calendar
The [Build Calendar Window](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/pay-cycle-calendar-maintenance/build-calendar-window) is used to set calendar defaults.
