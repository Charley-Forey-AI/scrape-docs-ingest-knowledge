# Certified Benefit Program Maintenance - Add Benefit Programs

Click the Edit button to add benefit programs to the selected state.

1. On the Properties tab enter a Program name, Account number, Classification and Contact information. this tab only applies to the AAASHTOWARE export file.

1. On the Fringes tab select union fringes to assign to this benefit program.

1. On the Adds/Deducts tab select add-ons/deductions to add to this benefit program.
