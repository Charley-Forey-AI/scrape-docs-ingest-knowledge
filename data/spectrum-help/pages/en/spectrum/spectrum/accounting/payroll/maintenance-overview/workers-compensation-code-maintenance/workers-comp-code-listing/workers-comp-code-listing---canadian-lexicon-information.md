# Worker's Comp Code Listing - Canadian lexicon information

If the Canadian lexicon designation (en-ca) is selected in
 Company Installation, some labels on all formats of this report will display using the
 preferred Canadian spelling or reference.
For example, "province" instead of "state" and "Province wide" instead
 of "Statewide," and more.
