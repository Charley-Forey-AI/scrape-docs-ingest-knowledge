# Payroll Trial Balance Report - Canadian lexicon information

If the Canadian lexicon designation (en-ca) is selected in
 Company Installation, some labels on this report will display using the preferred Canadian
 spelling or reference.
For example, "province" instead of "state" and "CPP" instead of "Fica,"
 and more.
