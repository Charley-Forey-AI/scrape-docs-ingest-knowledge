# To add a note to a topic

Follow the steps below to add a note to a
 topic.

1. Complete the Employee code field.

1. A list of note topics displays. If no topics have been
 set up yet, only a Remarks item will display. Topics that already have notes assigned
 will have a green dot next to them.

1. Highlight a note topic and click the Edit button. The text formatting
 toolbar displays.

1. Place your cursor directly under the formatting
 toolbar and type your notes.

1. Apply any formatting changes using the toolbar To
 access the font styles and sizes, use the drop-down arrow, and click Save. Notice that a green dot will
 now display next to the note topic if there wasn't one before.
