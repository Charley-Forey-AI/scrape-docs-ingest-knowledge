# Monthly Reporting

Follow the steps below for monthly reporting in
 Payroll.

- Preview a copy of the Payroll Register and save it.

- If needed, print the Union Report and save it.

- If needed, print the Employment Utilization Report.

- If the user is using monthly limits for deductions, run the Period End Clear, selected for Month End. Be sure that a backup has been made before running this purge for the first time.
