# Payroll Installation – Defaults Tab

Using benefit tiers are optional in Spectrum. We recommend that new converting clients use them, but this is not required. Existing clients can continue to use a combination of their existing setup along with benefit tiers as desired.
Clients implementing Spectrum will want to hide the "old" fringe method
 of setting up unions. This is done on the Payroll Installation > Defaults tab. Select the Hide fringes
 tab in Union File Maintenance checkbox. It will be unselected by
 default.
Selecting the checkbox does not delete existing union fringe setup; instead, it only hides this tab from within this company.
