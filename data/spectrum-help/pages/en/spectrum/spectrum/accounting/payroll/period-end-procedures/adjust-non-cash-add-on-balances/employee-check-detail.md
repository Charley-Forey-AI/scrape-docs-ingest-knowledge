# Employee Check Detail

Use this window to correct Payroll records. Only checks
 that are within the current minimum/ maximum Payroll processing date range
 display.
If the minimum date is in a previous year, then only checks dated in the year of the
 maximum date will display. Select a check (or checks) from the available list and adjust
 the applied amounts.
Fields/Buttons
Descriptions

Check #
The employee's check numbers (for the specified Payroll processing date range) display.

Period end date
The period end date corresponding to the employee's check displays.

Check date
The associated check date displays.

Check type
The associated check type displays.

Gross earnings
The employee's gross earnings for the corresponding check number display.

Add-on amount
Enter the employee's add-on amount, Davis-Bacon allocation amount, and employee's prevailing wage allocation for this check. For checks that already include the specified non-cash add-on, the current amounts will be shown, but you may overwrite the amounts.
An error check will be performed on these lines once the amount is changed, to confirm that when the original check was issued, the add-on code was processed as a non-cash item. If the code has been already been paid on a check, no changes will be allowed for this line.
