# Build Calendar Window

A window is available at this prompt to set calendar
 defaults.
Make the appropriate selection and select Yes to save calendar defaults.
Field
Descriptions

Pay frequency
Select Weekly, Bi-weekly, Semi-monthly, or Monthly.

First period ending date
Enter the first period ending date or press Enter to select the default of first of the year, or press F4 or double-click on this field to select a date from the Date Change window.

Number of days to check date
Enter the number of days between the period end date and the check date. For example, if the period end date is Saturday and checks are issued on Tuesday, enter 3.

Proceed with update
Select Yes, build new calendar to create new calendar, or select No, return to view current calendar to view the current calendar.
