# Copy Employee Status Codes

Use this screen to set up employee status codes across
 companies.
In order to use this feature, you must have the appropriate security access in the Payroll
 module for the destination company.
This screen is only available if you have Enterprise Management security access (security category EM). In addition, if you do not have access to manually make changes in the Employees screen in the destination company, then the status code will not be added.
Field
Descriptions

Copy from company
The current company displays in this field.

Copy to company
Enter the company you wish to copy the employee statuses to, or press Enter to accept the default of ALL.
After making your selection, click the Continue button.
