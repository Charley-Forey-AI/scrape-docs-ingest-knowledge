# Form 1095-C Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
FieldsDescriptions
Export WorksheetSelect to open the [Export Employee Worksheet](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/form-1095-c-maintenance/export-employee-worksheet) screen to export a list of all employees with current year earnings from Payroll.
Import WorksheetSelect to open the [Import Employee Worksheet](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/form-1095-c-maintenance/import-employee-worksheet) screen to import a list of employees' offer and coverage codes from the Employee Worksheet for the year specified in the file.
Update Plan Start MonthSelect to open the Update Plan Start Month window. This update populates the employee's plan start month based on the selected criteria:

- Set plan start month to annual coverage renewal month (01-12) for either all employees, or employees hired prior to 2020.

- Set plan start month for employees hired in YEAR to hire/rehire month plus (0-12) months. Employees without a hire date will not be updated.Note: This utility does not overwrite a plan start date already entered, but does populate fields which are blank.

Print FormsSelect to [Print 1095-C Forms](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/form-1095-c-maintenance/print-1095-c-forms) for year-end filing in the United States.
ExportSelect to export 1095-C data for [Processing 1095C with Nelco](/en/spectrum/spectrum/accounting/payroll/period-end-procedures/form-1095-c-maintenance/processing-1095c-with-nelco).
Employee codeEnter the employee code to view, or press Enter to include ALL employees on the maintenance listbox.
EntityIf Entity tracking is enabled for the current company, enter the entity company to view, or leave this field blank for the main company entity.
YearEnter a four-digit year (required).
Employer provided coverage using an HRA?Select this checkbox if the selected employee is receiving coverage through a Health Reimbursement Arrangement (HSA). When selected, enter the Employee age at January 1.
'Offer of coverage' codeEnter the IRS-designated offer of coverage code for each month of the employee's coverage. See the IRS website for current codes and definitions.
Employee share of lowest costEnter the employee share of the lowest-cost monthly premium offered for self-only qualifying coverage. This amount may not be the amount employees paid for coverage if, for example, they chose to enroll in more expensive coverage such as family coverage.
Section 4980H Safe Harbor codeEnter the IRS-designated safe harbor code that employers use to determine which employees are treated as full-time employees. See the IRS website for current codes and definitions.
Zip code (HRAs only)If the Employer provided coverage using an HRA checkbox was selected above, enter the employee's zip code for each month of the employee's coverage.
Plan start monthSpecify the start month for the employee's coverage. Enter 00 for no offer of coverage, otherwise enter the starting month 01-12.Note: This field is required.

Employer provided self-insurance coverage?Select this checkbox if the employee is covered under a policy that is self-insured by the employer.
When selected, the Dependents button is available to add the employee's dependents.
