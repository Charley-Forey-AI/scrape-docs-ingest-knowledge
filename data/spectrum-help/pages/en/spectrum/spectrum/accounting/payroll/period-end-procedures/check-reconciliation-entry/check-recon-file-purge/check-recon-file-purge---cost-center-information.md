# Check Recon File Purge - Cost Center Information

If cost centers are being used in this company, then when
 purging reconciled checks, Spectrum will remove the items only if the operator performing the
 purge has permission to access them in Check Reconciliation Data Entry.
Spectrum compares the cost center assigned in the check with cost centers in your
 operator's assigned cost center scheme, and if the cost center of the check is not
 included, then the reconciled check will not be purged. Likewise, Spectrum also assures
 that the employee's assigned cost center is valid for your operator and if not, then
 checks for that employee will not be purged.
