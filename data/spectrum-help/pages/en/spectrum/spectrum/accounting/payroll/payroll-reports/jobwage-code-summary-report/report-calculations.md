# Report Calculations

Column
Descriptions

Total # of workers
The total number of workers stored in the time card history file for the date range specified.

Total # of hours
The total number of hours stored in the time card history file for the date range specified.

Hourly Pay Rate
The rate stored in the Wage Code File Maintenance screen for the rate level specified in the Job File Maintenance screen. A blank rate level is interpreted as pay level #1.

Hourly Fringe Rate
The fringe rate is computed for the union specified, for the same pay level as above.
