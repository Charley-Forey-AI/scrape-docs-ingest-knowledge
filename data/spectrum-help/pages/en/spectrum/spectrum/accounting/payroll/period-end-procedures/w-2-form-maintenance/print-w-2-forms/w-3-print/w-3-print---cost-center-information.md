# W-3 Print - Cost Center Information

If cost centers are being used in the current company, only employees that the operator has security authorization to access display on the report.
In addition, if entity tracking is being used in the current company, only W-3s for the specified entity code will display, and all employee forms within the authorized entity meeting the other selection criteria will display.
