# Electronic File Formula Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Formula code Description
Enter a unique code and description for the formula.

Formula type
Select a formula type from the available drop-down list.

- If Record is selected, the formula will be calculated after all the totals have been summed.

- If Totals is selected, the formula will be calculated after all the totals have been summed.

- If Date is selected, the formula will be calculated after all the totals have been summed.

Result
Select Answer if this formula is a simple two-part formula, or select T1-T15 if this is a temporary result that will be combined with other temporary results to yield a final answer.

First factor
Enter a variable, number, or result code, or double-click on this field to display the Search File Variables window and select. You may enter a variable that is one of the following: Temporary result (T1-T15), Answer, or Number.

Period
Select the period corresponding to the factor.

Operator
Enter the operator, or double-click on this field to display the Formula Operators window and select from a list of available functions:
 + Add
 - Subtract
 * Multiply
 D Divide
...Take the smaller of the two amounts
...Take the greater of the two amounts

Second factor
Enter a variable, number, or result code. This factor will be used in conjunction with the first factor to yield either an Answer or a Temporary result.

Period
Select the period corresponding to the second factor.

Units
For date type formulas, enter the time period in this column.

Expand
Click this button to display additional information on the variable factors including the file codes and corresponding descriptions.
