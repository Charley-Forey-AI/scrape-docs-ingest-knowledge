# Worker's Compensation Code Maintenance

Use this screen to assign a default worker's compensation
 code to each employee, or to override standard limits on a code-by-code basis (rather than
 strictly on a statewide basis).
You can also manage worker's compensation limits on a non-calendar year
 basis, with the Maintenance screen prompting for the start date (MM/DD) each year, and you
 can assign default worker's compensation codes by job or phase. Base codes must first be
 defined in the [Worker's Comp Base Code Maintenance](/en/spectrum/spectrum/accounting/payroll/maintenance-overview/workers-compensation-code-maintenance/workers-comp-base-code-maintenance) screen. Type a partial entry in the Search field or click the New
 Search button to look up an open item by code, state, or description.
During Pre-Time Card Entry and Time Card Entry, each line will require entry of a code.
Note: For information about current worker's compensation
 rates and codes, contact the insurance company that handles your company's Worker's
 Compensation Insurance, or contact the appropriate state agency.
View the Worker's Compensation code default hierarchy for JOBS

- If the employee has the Always
 use this workers comp code on time card? checkbox selected, then the
 software will use the code from Employees.

- The code first defaults from Phases.

- If no default is defined there, the code defaults from Jobs.

- If no default is defined there, the code defaults from Department Expense Maintenance.

- If no default is defined there, then the code defaults from Wage Code File Maintenance.

- If no default is defined there, then the code defaults finally from Employees.

- If desired, the default assigned here may be overwritten during Pre-Time Card Entry or Time Card Entry.
