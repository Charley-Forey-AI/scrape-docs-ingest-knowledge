# Non-Cash Add-on YTD Log Report

The Non-cash Add-on YTD Log Report is used to print a list
 of the changes made to non-cash add-ons through the Adjust Non-Cash Add-on Balances screen.
This report shows each altered add-on code, its original and revised value, and the date,
 time, and operator making the change. Records from this file will not be purged until the
 associated check is removed from the software using either the Employee Earnings History
 Purge or the Year End Clear update (if the terminated employee is removed from the
 software).
