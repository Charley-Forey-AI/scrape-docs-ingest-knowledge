# Subject-to-Tax Report - Field Descriptions

A reference for completing the fields on this screen.
FieldsDescriptions
EmployeeNarrow the report to just one or multiple employees, or proceed with the default value to include all employees.
Tax codeEnter the tax code (or codes) you want to use to narrow your report selections.
From check dateEnter the first check date to be included on the report.
To check dateThe current Payroll processing date displays, or you can change the date.
Show entire social security number?Select this checkbox to display the full social security number; otherwise leave it clear. This checkbox defaults to clear unless a saved filter is present.
Show 'exempt wage' and 'adjustment' details?Select this checkbox to include the exempt wage detail and deduction/add-on adjustment detail on the report.
Report typeSelect the report format you want to print.
Check cost center
Cost groupThis field only displays if cost centers are enabled in the current company, and the field label may differ if you have a user-defined cost group label specified in Enterprise Installation.
This field works as a filter that allows you to include only a portion of the records your operator has security for, thereby allowing reports to be produced for a segment of employees.
The software does NOT read for authorization of individual cost centers contained within the specified cost group to determine permission for the check cost center.
