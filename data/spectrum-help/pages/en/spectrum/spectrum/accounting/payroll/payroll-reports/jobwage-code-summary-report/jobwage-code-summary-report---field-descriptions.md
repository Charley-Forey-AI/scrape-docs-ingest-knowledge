# Job/Wage Code Summary Report - Field Descriptions

Use the table below for reference when completing the fields
 on this screen.
Fields/Buttons
Descriptions

Job
Enter the job to be included on this report,
 or press Enter accept the system default of ALL.

Period end date
Enter the first and last period end dates to
 be included on this report, or press Enter to
 begin with the earliest processing date and end with the current
 payroll processing date.
