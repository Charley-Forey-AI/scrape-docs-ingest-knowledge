# Reverse Pre-Time Card Transfer - Field Descriptions

A reference for completing the fields on this screen.
FieldsDescriptions
Selections
Batch codeEnter the batch code corresponding to the information that should be moved back into Pre Time Card Entry, or press Enter to undo ALL of the updates which should be reversed.
New batch codeEnter a new batch code to identify the information, or press Enter if the batch code(s) should remain the same.
Check Type
All time cardsLeave selected to return all transferred pre-time cards to Pre-Time card entry.
Layoff and replacement checks onlySelect if you want to limit the transfer to only layoff and/or replacement checks; then select one or both checkboxes.
