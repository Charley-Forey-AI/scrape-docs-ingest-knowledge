# Employee Earnings History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Employee codeEnter or select an employee in order to view their respective earnings history.
EntityIf cost center entities are being used. press F4 and select an entity in order to view entity-specific payment history details. Note: Only the entities set up for the specified employee display in the Search Employee Entities window.

Check buttonSelect to open the [Payroll Check History Inquiry](/en/spectrum/spectrum/accounting/payroll/inquiries-overview/employee-earnings-history/payroll-check-history-inquiry) window, useful for reviewing the selected employee's check information, including hours/earnings, taxes, and totals.
Time Card buttonSelect to open the [Time Card History Inquiry](/en/spectrum/spectrum/accounting/payroll/inquiries-overview/employee-earnings-history/time-card-history-inquiry) window. This window is useful for viewing certified information in the time card detail history lines for this transaction. You can both move and sort the columns in this window to customize the display.
