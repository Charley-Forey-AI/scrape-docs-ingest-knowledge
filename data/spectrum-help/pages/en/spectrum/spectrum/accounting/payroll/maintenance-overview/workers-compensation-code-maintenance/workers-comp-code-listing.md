# Worker's Comp Code Listing

Worker's Comp Code Listing prints the worker's compensation
 information entered in Worker's Comp Code Maintenance and Worker's Comp Base Code Maintenance,
 including multiple rates.
Selections are available to determine the print sequence. Select the option that
 corresponds to the report desired:

- Listing of w/c base codes only - Prints only base codes entered in Worker's Comp Base Code Maintenance, in sequence by code.

- Full listing, sorted by code - Prints all code/state combinations, in sequence first by code, then by state.

- Full listing, sorted by state - Prints all code/state combinations, in sequence first by state, then by code.
