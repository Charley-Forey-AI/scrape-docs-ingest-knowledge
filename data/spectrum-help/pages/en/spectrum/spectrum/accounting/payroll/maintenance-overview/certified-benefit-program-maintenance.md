# Certified Benefit Program Maintenance

Some jurisdictions require Certified Payroll Reporting be done electronically.
Both of the Certified Payroll XML File Formats (eCPR and AASHTOWare) include fringe benefit information. This information needs to be compiled into several pre-defined 'types'. The two formats each use a different list of 'pre-defined types' of fringe benefits.
