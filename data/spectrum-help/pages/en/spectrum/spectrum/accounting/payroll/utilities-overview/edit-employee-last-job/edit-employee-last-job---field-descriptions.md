# Edit Employee Last Job - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Code
The employee code displays in this field. No entry is allowed.

Employee name
The employee's first and last name display in this field. No entry is allowed.

Company
The employee's last recorded company displays in this field. Press Enter to accept the default, or enter another company.

Job
The employee's last recorded job number displays in this field. Press Enter to accept the default, or enter another job. Use the space bar to remove the job if this employee should not have a previous job specified.
The Job name column will also display if the job is valid in the current company.

Permanent
Select the checkbox to suppress the automatic updating of the last job field during Check Calculation.
