# Exemption Amount Exceptions Listing

This feature is only available when Payroll is set for Non-US processing.
This listing displays all 'Active' employees not currently assigned the 'Exemption' amount for the tax code entered on the start screen. The Payroll Managers will want to print this out to identify all employees who will not be updated automatically when the utility is run. They can use this as a worksheet to identify those employees with non-standard exemption amounts that may need to be adjusted manually in Employee Tax Setup.
