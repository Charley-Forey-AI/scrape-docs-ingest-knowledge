# Union File Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Union code
Enter the union code(s) to include on the report, or press Enter to include ALL union codes.

Active deductions and fringes only?
Select this checkbox to only include active deductions and fringes on the report.

Include inactive unions?
Select this checkbox to include inactive unions on the report.

Include benefit tier overrides?
Select this checkbox to include any benefit tier overrides set up for this union on the report.
