# Meter Readings

Use this window to enter meter readings while recording
 equipment usage (EU) transactions and equipment cost (EC) transactions.
This window can also be used when recording stand-by time, although in
 this case, the hours are not computed from the change-in-meter. Based on setup in the Equipment Control > Equipment File Maintenance screen, the window allows entry for up to four meters. If no meter type is
 specified for the equipment, the meter fields are disabled (for example, if only two meter
 types are specified, only two fields will be enabled). After you complete your time card
 entries, the software will automatically create up to four meter transactions in Equipment
 Control for the specified equipment.
Note: Meter readings can only be deleted from the Equipment Control > Meter Reading Transaction Entry screen.
