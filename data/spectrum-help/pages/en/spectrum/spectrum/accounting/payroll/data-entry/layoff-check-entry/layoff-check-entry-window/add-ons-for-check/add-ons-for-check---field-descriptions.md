# Add-ons For Check - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Code
The add-on code displays. Add-ons may not be changed, instead, use Time Card Entry, Layoff Check Entry or Employee Deduction Maintenance to change add-ons.

Descriptions
The description of the add-on code displays.

Amount
The add-on amount for this check displays.

Source
The source of the add-on information displays. Source options include: Employee Deduction Maintenance, Union File Maintenance, or Time Card Entry.
