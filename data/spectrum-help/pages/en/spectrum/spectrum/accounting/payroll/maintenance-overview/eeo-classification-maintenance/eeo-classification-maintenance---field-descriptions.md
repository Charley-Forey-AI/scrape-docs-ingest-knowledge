# EEO Classification Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

EEO code
Enter the code for an Equal Employment Opportunity (EEO) classification.

Descriptions
Enter a brief description of the EEO code.

Job Category
The job category descriptions are hard coded to match the EEO-1 Report data. Use the drop-down menu to select the applicable job category for the selected EEO classification code.
