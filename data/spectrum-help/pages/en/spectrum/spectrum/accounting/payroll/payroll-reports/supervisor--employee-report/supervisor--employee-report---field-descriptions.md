# Supervisor / Employee Report - Field Descriptions

Use the table below for reference when completing the fields
 on this screen.
Fields/Buttons
Descriptions

Employee Supervisor Employee status
 Department
Use these fields to narrow your report
 results. Enter an Employee, Supervisor, Employee status code, and
 Payroll Department, or press Enter to
 include ALL on the report.

Review date
Enter the first and last review dates to be
 included on the report, or press Enter to
 start with the earliest date and end with the current Payroll
 processing date.
