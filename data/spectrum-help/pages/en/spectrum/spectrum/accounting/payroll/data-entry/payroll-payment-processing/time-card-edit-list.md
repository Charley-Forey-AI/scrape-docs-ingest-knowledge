# Time Card Edit List

After all time cards have been entered, print the Time Card
 Edit List and check the report for accuracy.
Make any necessary corrections and reprint the report.
The Crystal-Link format of these reports offers a drill-down to Document Imaging that displays when Preview is used to access the reports. For easy identification, items with an associated Document Imaging display are shown in blue in the Crystal-Link Preview window.
This report will include lines for the main tax codes and any related tax codes. However,
 related tax codes will not appear on summary reports.
Important: This report should be retained as a permanent
 audit record of the company.
