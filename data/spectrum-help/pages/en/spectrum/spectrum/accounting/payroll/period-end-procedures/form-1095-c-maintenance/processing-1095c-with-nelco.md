# Processing 1095C with Nelco

Viewpoint partners with Nelco to provide initial e-filing at no cost.
Any follow-up 1095-C corrections will be handled by Nelco
 for a fee, and printing and mailing of physical forms is also available through Nelco for
 an additional fee.

1. Open the Payroll > Period End > 1095-C Processing screen to add and edit employee-specific reporting information for
 employees who are eligible for health care coverage in the selected calendar
 year.

- Select the Export Worksheet button to export a list of all employees with
 current year earnings from Payroll. This will also include employees'
 dependents when coverage is provided by a self-insured employer.

- Select the Import Worksheet button to import information from the Employee
 Worksheet into this screen. This import will not delete any records; it will
 change existing records based on the import file. Once the update is complete,
 an exceptions window will list lines from the import file (should be a .CSV
 file) that could not be imported. Make corrections to the Employee Worksheet
 and then re-import the .CSV file, as necessary.

- To add employee information, select the New button to open the
 New Form 1095-C
 window and make changes to an individual employee after importing from the
 Employee Worksheet. The 'Offer' fields and the 'Safe Harbor' fields each
 provide convenient combo-box pull-down lists for IRS-designated codes.

- Employee state info will also be included in the Export
 Worksheet for states that require electronic filing of 1095 data (currently New
 Jersey and Washington D.C. only).

1. Once the import has been completed successfully, select the Export button to export the data to a CSV file for electronic processing. The Export to Nelco information window displays. Enter a contact and phone # for your company, and select Continue if you are ready to proceed with the export. A second information window displays letting you know if the export was successful and listing any exceptions (records containing errors) if they exist.Note: Do not open the export file in Excel, as this will remove
 any zeros from the file and leave these fields blank instead. If you need to make
 any changes to the export file, use a text editor, or make changes in Nelco's
 portal instead (see below).

1. Contact Nelco at 1-800-266-4669 to obtain your prepaid
 filing code for 1095-C filing. You will need to provide the total number of customer
 records you are processing, and your Spectrum customer ID. If you do not know your
 customer ID, contact Viewpoint Customer Support by visiting the [Viewpoint
 Customer Portal](https://support.viewpoint.com/s/).

1. Open the [Nelco partner site](https://spectrum.nelcoportal.com/import/gwximport). If you haven't already created an account, you will need to select the Create Account link. Here you'll be
 prompted to enter general info (name, address, email, etc.) about your company.

1. Select the Import Data link and select the
 Affordable Care Act Reporting Form Type and select the
 File to Import.

1. Select the Import button.

1. On the Forms In Progress screen, choose the filing services you wish to use.
 Remember that Federal E-filing is provided at no cost with the prepaid filing code
 from Nelco, and printing and mailing services are available for an additional fee. If
 you only want to file the 1095-C Forms, select the Federal
 E-filing option from the A LA CARTE SERVICES section as shown in this image.


1. Make sure all the employees for your companies are selected on this screen and select the Next button to proceed.

1. If you selected the 'Recipient Mail & Online Retrieval' service, enter recipient email addresses.

1. On the 1094-C Data Entry screen, enter Applicable Large Employer (ALE) Member transmittal information for your company. Refer to the IRS instructions for details on entering this information.

1. Review the recipients in your cart and select the Next button.

1. On the Payment Details screen, enter the Prepaid Filing Code you received from Nelco in Step 1 and select the Apply Code button. If you are having Nelco print and mail the forms for you as well, you will need to enter credit card information on this screen for those services.

1. Select the Pay & Submit button to complete processing.
