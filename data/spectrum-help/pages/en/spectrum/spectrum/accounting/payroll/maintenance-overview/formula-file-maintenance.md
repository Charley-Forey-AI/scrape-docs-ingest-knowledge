# Formula File Maintenance

Use this screen to create deduction formulas for use in
 payroll calculation.
You also have the ability to perform a test of the formula in this screen to ensure the
 formula was set up correctly before Check Calculation occurs.
For complicated union fringe calculations and add-on or deductions, a user-defined formula system is provided. Use basic math functions to calculate the amounts required by various unions and other agencies, and to test the results and set up variables that can be assigned per union or employee (for example, for varying rates or constants).
