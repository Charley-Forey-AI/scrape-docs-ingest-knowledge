# Fee History Window

Use this window to view fee history information for a selected job.
Fields
Descriptions

Description
The description for this default fee displays.

Type
The type of fee displays: Flat fee or Percentage. If a flat fee was entered, the system skips to the Amount column. If a percentage was entered, the system advances to the next field.

Base

- Individual cost types displays if the fee is based on individual cost types.

- Subtotal displays if the fee is based on the subtotal of all cost types

- Total displays if the fee is based on the total (including any add-ons).

If either Subtotal or Total was entered, the system skips to the Amount column. If Individual cost types was entered, the system advances to the next field.

Cost types
The Cost type(s) to be included in this fee display.

Amount
The amount of the flat fee or percentage on which to base the fee displays.

Extension
The extension amount displays. The maximum value is $99,999,999.99

Taxable?
Yes displays if this transaction is taxable, No displays if it is not taxable.
