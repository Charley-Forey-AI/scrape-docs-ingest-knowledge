# Installation Overview

On the Site Map, click System Administration >  Installation >  Time + Material.
Ensure that the checkbox for the Time + Material module has
 been selected in the System Administration >  Installation >  Company >  Modules tab. Click Save and
 return to the Installation list.Note: The first time you select the
 Installation screen, the system beeps and displays the following message: Control record
 HAS NOT BEEN CREATED OK TO PROCEED. After you complete the Installation screen setup,
 this message no longer displays.

1. [Time + Material - Set Default T+M Properties](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---set-default-tm-properties).

1. If applicable to your organization, set up default markups, add-ons, or fees:

- [Time + Material - Create Default Markups](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-markups).

- [Time + Material - Create Default Add-Ons](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-add-ons).

- [Time + Material - Create Default Fees](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-fees).
