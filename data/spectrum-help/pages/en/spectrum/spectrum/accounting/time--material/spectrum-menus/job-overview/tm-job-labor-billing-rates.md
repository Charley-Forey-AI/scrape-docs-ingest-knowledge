# T+M Job Labor Billing Rates

Use this screen to override the default billing rates set up in the standard Labor Billing Rates Maintenance screen. For example, a billing code might be set up so that a lead carpenter is billed at $50 per hour, but for this specific job the customer should be billed only $40 per hour.
The information entered in the [New/Edit Job Labor Billing Rates](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/tm-job-labor-billing-rates/newedit-job-labor-billing-rates) displays in the list box. If you have security authorization, click New or Edit to add or change billing rates.

- If the selected job is a master job, the rates only need to be set up once and they will default into any sub-jobs, provided that the sub-jobs do not have their own billing rates set up. If a sub-job has its own billing rates set up you can add this job, but the billing rates will default from it and not the master job. If this job is a sub-job billed from the master job, billing rates for the sub-job are derived from the master job.

- If this screen is not used, billing rates will default from the standard Labor Billing Rates Maintenance.
