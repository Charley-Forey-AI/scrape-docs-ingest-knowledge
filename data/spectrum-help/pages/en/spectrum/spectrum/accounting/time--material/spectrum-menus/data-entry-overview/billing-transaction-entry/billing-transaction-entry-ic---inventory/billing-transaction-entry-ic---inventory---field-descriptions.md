# Billing Transaction Entry: IC - Inventory - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Item code
Enter or search for the item code in this required field, which may not be changed after saving the information. The item description will display.

Description

Job
Enter or search for the job code in this required field. Based on the number entered, the full job description will display from Jobs.

Phase
Enter or search for the phase code in this required field. Phases containing up to 20 characters can display in this column.
Note: Data entry is prevented if the phase status is set to Complete. A warning
 displays if the status is set to Inactive.

Ct
Enter or search for the Cost type code in this required field.

Quantity
Enter the total quantity issued in this required field.

Sell price
You can press Enter
 to accept the default, or change the selling price for this item. The sell
 price defaults based on the material level number that is set up in the
 T+M Job Billing Setup screen.

Total amount
The total dollar amount (Quantity multiplied by Sell price) displays .

Job name
This information displays after you enter data in the Job and Phase columns.
