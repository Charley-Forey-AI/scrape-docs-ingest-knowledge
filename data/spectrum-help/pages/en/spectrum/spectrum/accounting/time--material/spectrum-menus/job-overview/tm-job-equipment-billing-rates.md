# T+M Job Equipment Billing Rates

Use this screen to override the default equipment billing rates set up in the standard Equipment Billing Rates Maintenance screen. For example, a 14-foot backhoe loader might be set up to bill at $215 per day or $26.875 per hour, but for this specific job the customer should be billed $35 per hour.
The information entered in the [New/Edit Job Equipment Billing Rates](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/tm-job-equipment-billing-rates/newedit-job-equipment-billing-rates) displays in the list box. If you have security authorization, click New or Edit to add or change the job equipment billing rates.

- If the selected job is a master job, the rates only need to be set up once and they will default into any sub-jobs, provided that the sub-jobs do not have their own billing rates set up. If a sub-job has its own billing rates set up you can add this job, but the billing rates will default from it and not the master job. If this job is a sub-job billed from the master job, billing rates for the sub-job are derived from the master job.

- If this screen is not used, billing rates will default from the Equipment Billing Rates Maintenance screen.
