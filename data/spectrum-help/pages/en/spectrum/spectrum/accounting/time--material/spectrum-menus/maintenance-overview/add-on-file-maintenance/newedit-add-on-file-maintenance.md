# New/Edit Add on File Maintenance

Use this window to set up or modify standard add-ons.
Field
Description

Add-on code
Enter or search for the add-on code.

Description
Enter the add-on code description. Examples: Overhead percentages or small tools percentages.

Add-on percent
Enter the add-on percentage of total dollar amount; for example, 10.00 is 10%.
