# Entering a Transaction

If you have expenses for which you have not yet received an
 invoice, the T+ M transaction feature allows you to enter amounts to be updated to the T+ M
 pre-billing file for the purpose of selecting for billing.
It will be necessary
 to note these transactions to, when the actual invoice arrives and is posted, it is not
 billed in duplicate.Note: If you are using Payroll Pre-Time Card Entry, it is
 easy to copy these transactions to T+M so you can bill prior to updating Payroll. Use
 the Time + Materials > Data Entry > Update Pre-time Cards and select the batch you want to bill. When the Payroll module
 updates, these time card entries will not be sent to the Time + Material module to be
 billed because they have been flagged as billed.

1. On the Site Map screen, click
 Time + Material >  Data Entry > Billing Transaction Entry.

1. Enter the date for the transaction, or press
 F4 or double-click on this field to select a date from the
 Date Change window.

1. Select the tab that corresponds to the transaction
 type: JC - Job, AP - Vendor,
 EQ - Equipment, IC - Inventory, or
 PR-Employee.

1. Complete the detail portion of the window. Be sure to
 use the phase number you created as your T+M phase.

1. After you enter the transaction information, click
 Save to return to the Time + Materials
 Transaction Entry screen.

1. Click the Update button.

1. Complete the Job,
 From date (if applicable), and Through
 date fields.

1. In the Transaction type
 section, select the transaction type you want to update.

1. Click Preview. The
 T+M Transaction Update screen automatically displays.

1. Select Continue.Note: This transaction ONLY updates to the T+M pre-billing file
 for the purpose of billing before the live transaction takes place.

1. Click OK.
