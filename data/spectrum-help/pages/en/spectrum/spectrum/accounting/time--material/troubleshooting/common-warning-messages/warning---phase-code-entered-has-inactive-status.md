# Warning - Phase Code Entered Has Inactive Status

If an inactive phase code is entered, this message
 displays. Click OK to continue entering data, if desired.
To change the phase's status, open Job Cost > Maintenance > Phases. Enter the Job
 code and then select Begin
 Editing. Locate the phase in question on the grid and click the wrench icon
 to open the Edit Phases window,
 where you can change the phase status to 'Active'.
