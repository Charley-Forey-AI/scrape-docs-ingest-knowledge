# Process Billings

When it is time for you to create invoices for your
 customers, complete the following procedure.

1. On the Site Map screen, click Time + Material  >  Reports  >  Pre-Billing.

1. Print and review the Pre-Billing Report. This is a listing of all billing transactions that are in the Time + Material module. This report is very useful in determining what is available to bill.

1. Click Save and return to the Site Map screen.

1. Optional Step: Delete any transactions that will not be billed using the Detail Billing Selection screen.

1. Optional Step: Update pre-time cards to the Time + Material module.

1. On the Site Map screen, click Time + Material  >  Data Entry Detail Billing Selection.

1. Select or deselect transactions for billing.Tip: You can run the T+ M Billing Selection option as often as
 necessary. For example, you can run it five times to choose transaction for five
 different jobs.

1. Complete the remaining fields on the screen to further limit your selection or deselection transactions.

1. Optional Step: Modify the detail billings.

1. Optional Step: Change the labor and equipment billing rates.

1. Optional Step: Change markups and add-ons.

1. Select and print the T+M billing in the Cost Type Summary BillingPrint, Phase SummaryBilling Print, or Detail Billing Form screens. Tip: The Crystal-Link formats of this report offer drill-downs to Document Imaging when you click the Preview button. For easy identification, items with associated Document Imaging images are shown in blue in the Crystal-Link preview window.

1. On the Site Map screen, click Time + Material  >  Data Entry  >  Update Billings.

1. Use the Update T+M Billings to Invoice Entry screen to create an invoice in the Accounts Receivable module and mark the transactions as billed, which prevents it from being billed a second time.
