# Change Labor Billing Rates

This screen is used to change labor billing rates on a specific bill only after that bill has been selected. If it is necessary to change the rate on line items that have not been selected, the items must first be selected; make the changes through this screen, then deselect the transactions.
Fields
Descriptions

Job
Enter the Time + Material job number. Based on the number entered, the full job description will display.

Billing #
Enter the billing number to change, or press Enter for the current billing number. If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Selections

Employee
Press Enter for ALL employees, click the arrow to search, or enter the employee code to change the billing rate for a single employee. Based on the code entered, the full employee name will display.

Wage code Union code
Press Enter for ALL wage and union codes, click the arrow to search, or enter the code to change the billing rates for specific wage and union codes.
Note that wage and union codes are linked. When a valid pair has been entered, the codes and descriptions will display.

Old rates

From billing code
Enter the billing code to be changed. Click the arrow to select from a list of available billing codes.

Regular Overtime Double time
Regular, overtime, and double time rates display.

Markup
The Markup displays.

New rates

To billing code
To change rates and billing codes, repeat the code entered in the From Billing Code field, and then enter the new rates. Otherwise, to keep the same rates but change billing codes, enter the new billing code.

Regular Overtime Double time
Regular, overtime, and double time rates display.

Markup
The Markup displays.
