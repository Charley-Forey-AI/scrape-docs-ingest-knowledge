# Billing Transaction Entry: PR - Employee

This screen allows entry of Time + Material labor hours
 which are to be updated to the Time + Material module as "PR" transactions.
When you select the Payroll tab for this transaction, additions are made
 to the [Billing Transaction Entry](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/billing-transaction-entry) screen. They
 do not affect the Payroll or Job Cost modules.
If the employee has a billing code set up in Payroll > Employee File Maintenance, the associated Billing
 rate will be used here; otherwise, the Billing rate will be blank.
Note: This tab is hidden if you do not have security
 access.
