# Detail Billing History Report - VAT processing information

If the 'Utilize value added tax (VAT) tracking?' checkbox
 is selected in General Ledger Installation, all formats of the Detail Billing History Report
 will include the current VAT tax when a VAT tax code is present on the billing.
Note: External billing documents, such as the T+M billing, must display current VAT tax (but
 not retention VAT).
