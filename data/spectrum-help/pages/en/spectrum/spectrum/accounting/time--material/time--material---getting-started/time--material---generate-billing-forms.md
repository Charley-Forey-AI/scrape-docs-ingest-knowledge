# Time + Material - Generate Billing Forms

Select a billing form based on how you want to sort and
 summarize billing information for customers.
Cost Type Billing Form:

- Sorts and summarizes billing information based on a cost type.


- Run the report with or without markup shown, depending on what
 you want customers to see.

- Run a summary report, or run a detailed version by phase or by
 phase with group subtotals.

Phase Billing Form:

- Sorts billing information by phase code and cost type.

- Run with or without markup shown, depending on what you want
 customers to see.

- Run a summary or detailed version (with or without group
 totals).

Detail Billing
 Form:

- Set your own sorting preferences, such as by phase then cost
 type (with or without group totals), by cost type then date, or by phase digits.


- Displays all transaction lines, detailing how the bill was
 calculated.

- Allows for attached images to be appended to the report.
Note: For field descriptions of these options, see [Cost Type Summary Billing Form](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/cost-type-summary-billing-form), [Phase Summary Billing Form](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/sample-reports-overview/phase-summary-billing-form), and [Detail
 Billing Form](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/detail-billing-form).

To generate forms:

1. Select Time + Material > Data Entry > Cost Type Billing Form, or Phase Billing Form, or
 Detail Billing Form.

1. Enter a job.

1. To print detailed job descriptions on the form,
 select the checkbox for Job Notes?.

1. Select Format
 options.

1. Choose a Report type or
 Sort by options.

1. Click Preview.

1. Review the report. You can download reports as PDFs and email them to customers. Attached images display in the order in which they are associated with transactions included in the billing.
