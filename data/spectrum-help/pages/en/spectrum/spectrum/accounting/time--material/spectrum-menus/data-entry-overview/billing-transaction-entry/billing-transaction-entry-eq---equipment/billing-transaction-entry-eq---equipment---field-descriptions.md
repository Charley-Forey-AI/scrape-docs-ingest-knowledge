# Billing Transaction Entry: EQ - Equipment - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Equipment code
Enter or search for the Equipment code and description in this required field. Based on the code entered, the full equipment description will display from Equipment File Maintenance.

Description

Job
Enter or search for the job code in this required field. Based on the number entered, the full job description will display from Jobs.

Phase
Enter or search for the phase code in this required field. Phases containing up to 20 characters can display in this column.
Note: Data entry is prevented if the phase status is set to Complete. A warning
 displays if the status is set to Inactive.

Ct
Enter or search for the Cost type code in this required field.

Pay type
Enter the pay type: Regular, Overtime, or Double time. Entry in this required field will determine the Billing rate for this transaction.

Total hours/qty
Enter the total number of hours for this piece of equipment.

Billing rate
Accept the default rate, or enter a new billing rate for this piece of equipment. The billing rate defaults based on the bill code that is set up in Equipment File Maintenance.

Total amount
The total dollar amount (Billing rate multiplied by Total hours) displays.

Job name
This information displays after you enter data in the Job and Phase columns.

Phase description
