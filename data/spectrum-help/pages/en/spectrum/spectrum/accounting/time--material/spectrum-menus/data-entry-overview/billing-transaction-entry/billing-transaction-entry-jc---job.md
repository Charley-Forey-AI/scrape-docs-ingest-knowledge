# Billing Transaction Entry: JC - Job

This screen allows entry of Time + Material transactions
 which are to be updated to the Time + Material module as "JC" transactions.
When the Job Cost tab is selected for this transaction, additions are
 made to the [Billing Transaction Entry](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/billing-transaction-entry) screen.
 They do not affect the Job Cost module.
