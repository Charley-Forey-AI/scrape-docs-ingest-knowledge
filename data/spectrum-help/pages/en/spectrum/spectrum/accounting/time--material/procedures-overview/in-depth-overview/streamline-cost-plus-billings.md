# Streamline Cost Plus Billings

Submit copies of your receipts and other bills for your Cost Plus projects using the Document Imaging module to create a Crystal Reports version of your report and store it on the server.
You can scan in supporting documentation while making entries.
When its time to pull together the backup for the project, select Distribute from the [Detail Billing Form](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/detail-billing-form) and [Detail Billing History Report](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/reports-overview/detail-billing-history-report) screens.

## ZIP file creation

The system creates a .zip file on the server which includes:

- an RPT file

- a Documents folder that holds the DI Images.

## Uses for the files

You can then:

- Opt to download the zip file from the server and unzip it; this results in the two files being copied.

- Email the report to your client, providing them an audit type information.

## If you share the file with your customer

When the recipient receives and opens the .RPT file:

- If they don't already have a program to view the Crystal Report. they will receive a message prompting them to browse the web to find one.

- Once they have downloaded and installed the viewer, they can view the report file and select any links, which will open the associated image in the Documents folder.Important: Attempting to run the RPT file from the 'unzipped' directory will produce error messages. They must unzip the file and copy the resulting two files to a new directory.
