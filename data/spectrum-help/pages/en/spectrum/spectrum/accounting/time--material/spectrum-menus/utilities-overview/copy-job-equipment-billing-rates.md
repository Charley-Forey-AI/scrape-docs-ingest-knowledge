# Copy Job Equipment Billing Rates

Use this screen to copy special equipment billing rates from one Billing code to another, or from one job to another.
Field
Description

From job
Enter the job code from which you want to copy equipment billing rates.

To job
Enter the job code to which you want to copy equipment billing rates.

Billing code
Enter the billing code to copy the billing rates.
