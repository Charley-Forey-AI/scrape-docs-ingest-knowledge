# Delete Completed Jobs

This screen is used to remove completed job Time + Material
 files from the system, including all job information, current billing, historical billing,
 summary billing transactions, and invoice notes. In order to enter the jobs to be deleted, the
 job must have a Completed status and an actual completion date in the Job File Maintenance
 screen.
Note: The job itself is not deleted from Job File
 Maintenance.Field
Description

Line
Line numbers are automatically assigned by the system.


Job #
Enter the number of the completed job for which the TM
 files will be deleted.

Description
The job description displays.

Complete
The completion date displays. Repeat until all jobs
 which should be deleted have been entered. Click Continue to proceed
 with the deletion.
