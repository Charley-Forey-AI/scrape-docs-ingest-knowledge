# Billing Transaction Entry: IC - Inventory

This screen allows entry of materials from inventory which
 are to be updated to the Time + Material module as "IC" transactions.
When the Inventory Control button is selected for this transaction,
 additions are made to the [Billing Transaction Entry](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/billing-transaction-entry) screen. They do not affect the Inventory Control or Job Cost
 modules.
