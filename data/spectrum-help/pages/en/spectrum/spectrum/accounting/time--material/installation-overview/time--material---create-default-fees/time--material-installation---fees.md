# Time + Material Installation - Fees

Use this tab to select the fee settings for the Time + Material module. These settings can be changed as needed at any time.
Fields
Descriptions

Line
Line numbers are assigned and maintained by the system.

Description
Enter the description for this default fee.

Type
Use the drop-down menu to select the type of fee: F for Flat fee or P for Percent.

- If Flat fee is selected, the cursor skips to the Amount column.

- If Percent is selected, the cursor advances to the Base on column.

Base on
Use the drop-down menu to select what the fee is based on. Select from the following options:

> C
base the fee on individual cost types

S
base the fee on the subtotal of all cost types

T
base the fee on the total (including any add-ons)

 If S or T is selected, the cursor skips to the Amount field. If C is selected, the cursor advances to the Cost types column.

Cost types
Enter the cost type(s) to be included in this fee.

Amount
Enter the amount of the flat fee or percentage on which to base the fee.

Tax
Select this checkbox if this fee is taxable.
