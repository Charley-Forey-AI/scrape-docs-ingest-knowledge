From 'PR' work date
Enter a date here if you want to include payroll transactions only. Job Cost, Accounts Payable, and other transaction types will not be included.
 The work date field is populated when T+M Payroll
 entries are made in Payroll. If you enter Payroll transactions in Time + Material > Transaction Entry, you can use the Modify Billing screen
 to enter a work date.

To 'PR' work date
Enter the last work date to be included, or press Enter
 to end with the current Time + Material processing date.
This field is only used by Payroll Time + Material transactions. The work date
 field is populated when T+M Payroll entries are made in Payroll. If you
 enter Payroll transactions in Time + Material > Transaction Entry, you can use the Modify Billing screen to enter a work date. The Pre-Billing Report is
 correct when you select Payroll transactions and a 'From work date' is
 entered.
