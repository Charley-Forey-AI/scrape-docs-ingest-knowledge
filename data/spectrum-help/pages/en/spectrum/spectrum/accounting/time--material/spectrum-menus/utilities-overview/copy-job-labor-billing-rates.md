# Copy Job Labor Billing Rates

Use this screen to copy special labor billing rates from one Billing code to another, or from one job to another.
Field
Description

From job
Enter the job from which to copy labor billing rates. The job name displays.

To job
Enter the job to which labor billing rates should be copied.

Billing code
Enter the billing code to copy the labor rates.
