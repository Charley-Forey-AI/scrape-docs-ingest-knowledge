# Billing Transaction Entry: JC - Job - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Reference
Enter the reference number and description for job cost adjustments.

Description

Job
Enter or search for the job and phase codes in these required fields. Phases containing up to 20 characters can display in this column.
When adding a new row, if the selected job is a master job the rates only need to be set up once and they will default into any sub-jobs, provided that the sub-jobs do not have their own billing rates set up. If a sub-job has its own billing rates set up you can add this job, but the billing rates will default from it and not the master job. If this job is a sub-job billed from the master job, the system will prompt you to bill the master job and replace the sub-job code with its master job in the grid.
Note: Data entry is prevented if the phase status is set to Complete. A warning
 displays if the status is set to Inactive.

Phase

Ct
Enter or search for the Cost type code.

Total hours/qty
Enter the total number of hours or quantity for this transaction.

Total amount
Enter the total dollar amount for this transaction.

Job name
This information displays after you enter data in the Job and Phase columns.

Phase description
