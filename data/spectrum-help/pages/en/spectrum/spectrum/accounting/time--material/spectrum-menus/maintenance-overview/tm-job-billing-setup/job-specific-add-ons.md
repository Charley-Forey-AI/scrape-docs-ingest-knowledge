# Job-Specific Add-ons

Add-ons may be used to add a percentage to the subtotal of the bill. The percentage is applied against a predefined set of cost types. This is a convenient way to bill for overhead.
You can add, edit, or delete add-ons as necessary. Entries in this window are originally based on defaults established in the Time + Material Installation screen. A window is available which can be used to select any of the standard add-ons set up in Add-On File Maintenance that should apply to this job.
