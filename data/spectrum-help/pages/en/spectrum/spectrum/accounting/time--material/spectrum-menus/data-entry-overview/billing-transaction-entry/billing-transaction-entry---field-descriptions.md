# Billing Transaction Entry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Listing
Click to open the [Time + Materials Edit Listing](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/billing-transaction-entry/time--materials-edit-listing) screen.

Update
Click to open the [Time + Materials Journal](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/billing-transaction-entry/time--materials-journal) screen.

Transaction date
Press Enter to accept the current Time + Materials processing date, or enter the desired date for this transaction. The fiscal year and period display to the right.

Batch code
Enter a batch code to separate your transactions from others for data entry and update purposes.
