# Job Billing/Cost Analysis Report - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen for this company, Spectrum compares the cost center assigned to the job with cost centers the operator has access to. If the operator does not have permission, then access to the job is disallowed. For jobs that the operator has access to, Spectrum compares the cost center assigned to the phase with cost centers the operator has access to. If the operator does not have permission, then access to the phase is disallowed.
