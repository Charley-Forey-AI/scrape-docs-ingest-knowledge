# Billing Transaction Entry: AP - Vendor - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Vendor code
Enter the Vendor code in this required field. Based on the entered code, the vendor name will display from Vendor File Maintenance.

Job
Enter or search for the job and phase codes in these required fields. Phases containing up to 20 characters can display in this column.
Note: Data entry is prevented if the phase status is set to Complete. A warning
 displays if the status is set to Inactive.

Phase

Ct
Enter or search for the Cost type code.

Invoice date
Press Enter to accept the current Time + Materials processing date, or enter the desired Invoice date.

Invoice #
Enter the vendor item code (for example, invoice number or credit memo number).

P.O. #
Enter the purchase order number, check number, and A/P Invoice amount.

Check #

Total amount

Job name
This information displays after you enter data in the Job and Phase columns.

Phase description
