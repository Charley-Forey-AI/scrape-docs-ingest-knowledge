# Job-Specific Fees

Fees may be a flat amount or a percentage based on the total bill, the subtotal, or on individual cost types. Examples include environmental and hazardous materials fees.
You can add, edit, or delete fees as necessary. Entries in this window are originally based on defaults established in the Time + Material Installation screen.
Fields
Descriptions

Line
Line numbers are assigned and maintained by the system.
Enter a description for the fee.

Description

Type
Select the type of fee: Flat or Percent.

- If you select Flat fee, the cursor skips to the Amount column.

- If you select Percentage, the cursor advances to the next field.

Based on

- Select one of the following from the drop-down box:

-  C - Individual cost type

-  S - Subtotal

- T - Total (including add-ons)

Enter C to base the fee on individual Cost types; S to base the fee on the Subtotal of all cost types; or T to base the fee on the on the total (including any add-ons).
If S or T is entered, the cursor skips to the Amount field. If C is entered, the cursor advances to the next field.

Cost type
Enter the cost type(s) to be included in this fee.

Amount
Enter the amount of the flat fee or percentage on which to base the fee. The maximum value is $99,999,999.99.

Taxable?
Select the checkbox if the fee is taxable.
