# Time + Materials Journal - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job
Enter or search for the job number. Based on the number entered, the full job description will display from Jobs; verify that the desired job has been selected.

Dates
Enter the first and last transaction dates to be included, or select a date from the Date Change window. In the Through date field, press Enter to end with the current Time + Materials processing date.

Batch code
Enter a batch code to separate your transactions from others for data entry and update purposes.

Transaction type
Select the transaction type(s) you want to display on the report:

- Accounts Payable

- Equipment

- Inventory

- Job Cost

- Payroll
