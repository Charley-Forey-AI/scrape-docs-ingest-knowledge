# Cost Type Billing History Report

Use this report to obtain a billing history report for a specific job or range of jobs. The format of this report is based on the same sort option in the Cost Type Billing Inquiry. The major sort is by cost type, for example: labor, materials, or subcontractor.
This historical report duplicates information available on the Cost Type Summary Billing Print, before updating the billing.
