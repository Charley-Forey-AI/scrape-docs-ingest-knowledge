# Modify T+M Billing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Job
Enter the Time + Material job number. Based on the number entered, the full job description will display.

Customer
The customer code and name displays.

Billing #
Enter the number of the billing to be modified in this required field. If the billing number exceeds 999, the billing number will automatically be converted to the alpha equivalent (Example: A00 = 1000).

Invoice Notes
Click this button to enter notes for the invoice. These notes will display on the Detail Billing Form and the Detail Billing History Report. When updating, they will transfer to the Accounts Receivable invoice.
