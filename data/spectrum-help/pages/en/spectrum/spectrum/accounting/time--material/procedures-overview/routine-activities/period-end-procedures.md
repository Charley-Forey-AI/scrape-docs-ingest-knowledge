# Period End Procedures

Your T/M Processing Date Change can be set to minimum or
 maximum processing date.
Periodically, use [T/M Processing Date Change](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/data-entry-overview/tm-processing-date-change) to set the minimum and maximum Processing date.
