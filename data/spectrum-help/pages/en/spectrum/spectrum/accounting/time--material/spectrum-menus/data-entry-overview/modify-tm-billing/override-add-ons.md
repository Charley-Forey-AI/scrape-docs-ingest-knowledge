# Override Add-ons

This window is used to view and/or modify add-ons. Add-ons may be added in this window, if desired.
Field
Description

Add-on
Enter the add-on code.

Description
Enter a corresponding add-on code description.

Cost types
Enter the cost type(s) to be included in this add-on.

Add-on %
Enter the add-on percent.

Extension
The system will automatically calculate the extension.

Taxable?
Select to indicate that this add-on line is taxable.
