# Time + Material - Set Up T+M Labor Billing Rates

Optionally, set up T+M labor billing rate codes and
 associated pay rate levels.
Assign these billing codes to employees in the Payroll
 module. These rates override default billing rates set up in the Labor Billing Rates
 Maintenance screen. The system automatically applies T+M labor billing rates when time is
 entered against a T+M job. Employees who work on T+M jobs are paid at their usual rate, and
 T+M jobs are billed at the labor rate that corresponds to the billing code.

1. Select Time + Material > Maintenance > Labor Billing Rates.For field descriptions, see [New/Edit Labor Billing Rates](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/labor-billing-rates-maintenance/newedit-labor-billing-rates).

1. Click New to create a new line
 entry. To modify an existing code, select the code in the grid, and click
 Edit.

1. Enter a Billing code and
 Description.

1. Select a Markup type:
 Percentage, Rate per hour, or
 None.

1. Enter labor billing rate levels 1 through 5 in each of
 the four rate type columns: Regular,
 Overtime, Double time, and
 Markup.

- Use the Markup column to specify a
 value (percentage or flat rate) to add on top of the given regular, overtime,
 or double time rate.

- The Markup column does not apply if
 you selected None as the Markup
 type.
For details on how the system calculates markups for labor
 rates, see [Calculate Markups Based on Pay Type](/en/spectrum/spectrum/accounting/time--material/procedures-overview/in-depth-overview/calculate-markups-based-on-pay-type).

1. When you are finished, click
 OK. The Labor Billing Rates grid
 displays the rates set up for each billing code.

1. [Time + Material - Assign Labor Billing Codes to Employees](/en/spectrum/spectrum/accounting/time--material/time--material---getting-started/time--material---assign-labor-billing-codes-to-employees) who work on Time + Material jobs.
