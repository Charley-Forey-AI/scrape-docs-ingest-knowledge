# Procedures As Required

You can reference the following help topics for procedures
 in Spectrum.

- Add standard labor billing rates as appropriate using [Labor Billing Rates Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/labor-billing-rates-maintenance).

- Add standard equipment billing rates as appropriate using [Equipment Billing Rates Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/equipment-billing-rates-maintenance).

- Add standard markups as appropriate using [Cost Type Markup File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/cost-type-markup-file-maintenance).

- Add standard add-ons as appropriate using [Add-On File Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/add-on-file-maintenance).

- Establish a job master record as needed for each new Time + Material or Cost Plus job using Jobs and [T+M Job Billing Setup](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/job-overview/tm-job-billing-setup).

- If specific jobs have rates other than the customary charges, enter job-specific labor and equipment rates using [Labor Billing Rates Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/labor-billing-rates-maintenance) and [Equipment Billing Rates Maintenance](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/maintenance-overview/equipment-billing-rates-maintenance).

- Copy job-specific labor and equipment rates as desired to save time during setup using [Copy Job Labor Billing Rates](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/utilities-overview/copy-job-labor-billing-rates) and [Copy Job Equipment Billing Rates](/en/spectrum/spectrum/accounting/time--material/spectrum-menus/utilities-overview/copy-job-equipment-billing-rates).
