# Time + Material - Create Default Markups

If your organization uses standard markups for materials or
 labor, set these markups as defaults on the Time + Material Installation - Markups tab.
The system applies these defaults to all T+M jobs; however,
 you can customize these settings per job as needed.

1. Select System Administration >  Installation >  Time + Material.

1. Click the Markups tab.For field descriptions, see [Time + Material Installation - Markups](/en/spectrum/spectrum/accounting/time--material/installation-overview/time--material---create-default-markups/time--material-installation---markups).

1. Click New to create a new line entry.

1. For Ct (Cost type), choose Labor or Materials.

1. Enter a Description for the markup.

1. Select the Type of markup: Percentage or Rate per
 hour.

1. Select the Basis for calculating the markup. The basis
 indicates how you want the system to calculate this markup:

-  Based on All transactions (typically used for
 materials)

- Based on a specific pay type, such as Regular, Overtime, or
 Double time (typically used for labor)
For details on how the system calculates markups for labor
 rates, see [Calculate Markups Based on Pay Type](/en/spectrum/spectrum/accounting/time--material/procedures-overview/in-depth-overview/calculate-markups-based-on-pay-type).

1. In Amount, enter either a percentage or flat rate for the markup.

1. When you are finished, click Save.
