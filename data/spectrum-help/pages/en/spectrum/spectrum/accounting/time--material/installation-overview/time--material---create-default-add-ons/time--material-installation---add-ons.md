# Time + Material Installation - Add-Ons

Use this tab to select the add-on settings for the Time + Material module. These settings can be changed as needed at any time.
Fields
Descriptions

Line
Line numbers are assigned and maintained by the system.

Add-on
Enter the add-on code.

Description
Enter a corresponding add-on code description.

Rate
Enter the add-on amount.

Cost types
Enter or search for the cost type(s) to be included in this add-on.
