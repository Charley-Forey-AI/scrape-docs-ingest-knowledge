# Pre-Billing Report

Use this report to obtain a listing of all billing transactions that are in the Time + Material module. Billing transactions are selected based on billing status. This report is useful in determining what is available to bill. These transactions are generally updated from all of the source modules of Spectrum (Accounts Payable, General Ledger, Job Cost, Equipment Control, Inventory Control, and Payroll).
