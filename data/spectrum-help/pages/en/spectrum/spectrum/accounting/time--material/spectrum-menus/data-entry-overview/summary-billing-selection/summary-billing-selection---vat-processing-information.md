# Summary Billing Selection - VAT processing information

If the 'Utilize value added tax (VAT) tracking?' checkbox
 is selected in General Ledger Installation, the VAT code defaults from the job's tax
 classification code, and when there is a VAT override on the tax classification code will be
 used at the job's VAT code. When a VAT override does not exist, the code default from the one
 defined in G/L Installation.
