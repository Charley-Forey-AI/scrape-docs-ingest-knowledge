# Time + Materials Journal - Update

When the report is finished, the update screen displays.
After the Time + Materials Journal has been run, read the report carefully. Once the report is accurate, save it as a permanent record of the company.
This function updates the Time + Material detail file; after running this update, transaction records will be available for Summary Billing Selection.
