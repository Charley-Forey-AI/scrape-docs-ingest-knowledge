# Set Up New G/L Department

Use this screen to conveniently set up new G/L account codes in companies that use a departmental chart of accounts, based on the current G/L department setup, by copying them to a different G/L account code. Any G/L accounts with a status of Not Used will not be copied.
The update allows users to set up G/L accounts for a new department code based on the account codes already set up for the Copy from department.
