# Fiscal Calendar Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
New YearSelect to [Add New Fiscal Year](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/period-end-overview/fiscal-calendar-maintenance/add-new-fiscal-year).
ListingSelect to begin generating a report for a fiscal year.
ViewSelect to [View Fiscal Calendar](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/period-end-overview/fiscal-calendar-maintenance/view-fiscal-calendar).
History yearSelect to add a [Add Historical Fiscal Year](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/period-end-overview/fiscal-calendar-maintenance/add-historical-fiscal-year).
DeleteOnly the first and last years in the calendar can be deleted.
 Grid
Fiscal yearThe year for which active data is maintained in the software.
# PeriodsThe number of fiscal periods established.
Start dateThe list of fiscal year start dates.Note: The new date cannot be later than the first period end date.

First periodThe first period names (months) display from the General Ledger Installation > Fiscal Calendar tab.
End dateThe list of fiscal year end dates.
Last periodThe list of fiscal year end periods.
