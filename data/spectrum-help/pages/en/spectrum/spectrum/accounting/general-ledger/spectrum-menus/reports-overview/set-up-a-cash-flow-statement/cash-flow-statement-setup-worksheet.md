# Cash Flow Statement Setup Worksheet

Use this tool if needed when setting up or modifying your Cash Flow Statement.
Identify and make note of the beginning and ending G/L account codes for each category. Also note if you need to exclude any codes in the range.
If you print this worksheet, adjust the scale in the print dialog if necessary to ensure all content is showing.
