# Import G/L Account Codes

Use this screen to import G/L account codes into the Change
 G/L Account Codes screens, view file layout and clear any existing unposted G/L code change
 records before importing new records.
Import errors will be listed in the error correction screen, and can be re-entered as
 necessary.
