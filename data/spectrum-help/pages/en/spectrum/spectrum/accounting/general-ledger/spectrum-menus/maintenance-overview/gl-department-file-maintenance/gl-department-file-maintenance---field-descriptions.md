# G/L Department File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Department code
This department code is the first part of the General Ledger account code. The account type and department don't need to match. For example, if department 5 is Washington State Operations, all accounts beginning with 5 will be in the Washington department.
Normally, when departments are used for income/expense tracking, balance sheet accounts (assets, liabilities, and capital) are grouped within department.
Alternately, if the chart of accounts is non-departmentalized, the department is commonly tied to account type 1 for Assets, 2 for Liabilities, and so on.

Description
The description of the G/L department displays. Click Edit to change the description.
