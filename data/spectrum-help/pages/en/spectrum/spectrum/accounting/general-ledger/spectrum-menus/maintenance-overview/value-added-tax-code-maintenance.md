# Value Added Tax Code Maintenance

Use this screen to set up value added tax (VAT) tax codes. This screen will also prompt for the tax percentage, informational text to appear on the invoice forms and G/L and accounts to store VAT payable amounts.
This screen is only available if the Utilize value added tax (VAT) tracking?
 option has been selected in the System Administration > Installation > Country tab.
Click New or Edit to display the Value Added Tax Code window where you can enter or edit tax codes.
