# G/L Budget File Maintenance - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, the [Edit G/L Budget](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/gl-budget-file-maintenance/edit-gl-budget) displays cost centers authorized for the specified account.
When adding, editing, viewing, or deleting a budget entry for an account, the software compares the account's list of shared cost centers with cost centers in the operator's assigned scheme, and if there are no common cost centers, then no cost centers are shown in the New/Edit G/L Budget window.
Budget entries for each accounting period are recorded by cost center for each G/L account.
