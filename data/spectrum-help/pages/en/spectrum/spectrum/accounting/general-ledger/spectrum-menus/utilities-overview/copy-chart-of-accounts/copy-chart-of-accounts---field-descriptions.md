# Copy Chart of Accounts - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

From company
The current company code displays.

To company
Enter the three-digit code of the company to copy the Chart of Accounts to.

Account status
Select at least one G/L status:

- Active

- Inactive

- Not used
