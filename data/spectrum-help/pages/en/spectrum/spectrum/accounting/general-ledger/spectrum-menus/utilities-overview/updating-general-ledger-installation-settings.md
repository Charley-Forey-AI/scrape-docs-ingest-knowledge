# Updating General Ledger Installation Settings

The following procedure describes how to update General
 Ledger installation settings.

1. On the Menu System screen, click Admin > Installation > General Ledger.

1. Press Enter to accept the default account code mask settings.Note: Normally, the account code mask should not be changed
 after the software is installed. However, the account code mask can be changed at
 this time if necessary.

1. Press Enter to accept the department code length settings.Note: It is necessary to change the number in this field ONLY
 if you have changed the account code mask.

1. At the Retained earnings account code field, enter the new retained earnings G/L account code. This is necessary ONLY if you changed the retained earnings G/L account code during the G/L Code Change Update.

1. Change any additional settings on this screen, as needed. Click OK.Depending on your Job Cost Installation screen settings, you may have to [Updating a Job's Division to a New G/L Department](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/utilities-overview/updating-a-jobs-division-to-a-new-gl-department) before processing any Payroll information.
