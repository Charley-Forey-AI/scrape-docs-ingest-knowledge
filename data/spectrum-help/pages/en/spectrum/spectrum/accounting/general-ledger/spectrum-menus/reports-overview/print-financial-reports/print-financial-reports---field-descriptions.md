# Print Financial Reports - Field Descriptions

Use the table below for reference when launching financial reports.
To navigate to this screen, go to General Ledger > Reports > Print Financial Reports.

FieldDescriptions
LineSequence numbers are assigned and maintained by the software.
Report IDEnter the ID code of the report to be printed. If needed, use the search feature.
Period end dateEnter the period end date for this report or press Enter to use the displayed date. The current period and year for the report are determined from the entry in this field.
The date displayed is determined from the fiscal calendar as the most recent period end.

Detail zero rows included?
Header zero rows included?
Total zero rows included?
Select the checkbox if you want information to print even if numeric values are zero.Important: You must select the Header zero rows included? checkbox to cause the headings to print.

Grouping Type

- Summary only - information is printed in consolidated, summary form.

- Individual department - information is printed for individual departments or cost centers only. A separate screen prints for each department or cost center.

- Individual cost center - this selection is available if the report is set up for cost center reporting, and cost centers are set to Yes in the current company.

- Both - the report includes the relevant combination of a summary and individual departments or cost centers.

Summary descriptionIf your previous selections include a summary, you can opt to enter a brief description of the report.
DepartmentsThe departments to be included in the report (for example, ALL departments).
Cost group This field displays only if the cost group feature is enabled in the Enterprise Installation screen.
Enter the cost group you want to include on the report. Before proceeding, Spectrum verifies your permission to access any cost groups you select.

Report nameThe report name reflects what is entered in the Format section of the [Design Financial Reports](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/design-financial-reports) screen.
