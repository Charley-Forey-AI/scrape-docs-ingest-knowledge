# Recurring Journal Entry Report

Use this screen to view a report of recurring journal entries. You can preview the recurring journal entry report by journal entry sequence or by cross-reference to the G/L account codes where the recurring entries are posted. Since the data in the Recurring Journal Entry is not purged at an update, these reports may be previewed at any time.
When applicable, work order details display on this report. With the ability to attach a work order number to a journal entry detail line, it is possible to accrue actual miscellaneous costs to a specific work order.
