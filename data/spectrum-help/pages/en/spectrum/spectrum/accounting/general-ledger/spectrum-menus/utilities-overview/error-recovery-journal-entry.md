# Error Recovery Journal Entry

Use this utility for a one-sided journal entry, as well as a one-sided cost center adjustment.
Warning: This screen requires a level 9 of security category
 ER (Error Recovery) to access, and should only be used under the direction of Viewpoint
 Support.
