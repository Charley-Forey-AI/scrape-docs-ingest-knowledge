# Recurring Journal Entry Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Journal entry
Enter the journal entry number.

Journal entry
Select this option to sort the report by journal entry number. The system will prompt whether or not to update the transaction.

G/L account
Select this option to sort the report according to G/L account code. The system will return to the Journal Entry screen without updating.
