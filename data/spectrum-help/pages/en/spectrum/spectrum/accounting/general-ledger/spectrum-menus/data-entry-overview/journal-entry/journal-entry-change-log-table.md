# Journal Entry Change Log Table

The Journal Entry change log table GL_JOURNAL_ENTRY_LOG_MC (GL.JELOG) may be accessed for review by performing a query. This table logs any additions or changes to the following columns:

- Company – Key

- Year – Key

- Period – Key

- Journal entry number – Key

- Description

- Cost center

- Create date – system date

- Create operator – 3-digit operator code

- Delete date – system date

- Delete operator – 3-digit operator code

- Update date – System date

- Update operator – 3-digit operator code

- Status flag – 'N' = not used, 'E' = entered, 'U' = updated, 'D' = deleted
