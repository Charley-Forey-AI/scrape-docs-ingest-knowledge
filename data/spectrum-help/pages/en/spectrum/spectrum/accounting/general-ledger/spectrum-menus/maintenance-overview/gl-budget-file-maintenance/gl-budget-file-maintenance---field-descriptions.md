# G/L Budget File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Budget year
Spectrum maintains unlimited years of active data. The year associated with the current General Ledger processing date defaults. To change it, either select a different year or enter the two-digit year that corresponds to the fiscal year in which the data is located.

Search bar
Click the arrow to view the Advanced Search Options window.

Status
Select the budget status to display:

- Active

- Inactive

- Not used

If the G/L account code status is set to Inactive or Not used, you can still enter data. This is useful if you want to set up accounts that won't be used until the next year, or if the data is used for statistics or future budgets.
