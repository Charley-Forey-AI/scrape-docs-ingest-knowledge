# More G/L Inquiry Selections

This window displays when you click the Selections button on the G/L Detail Analysis Inquiry screen. If you make changes to the default settings in the fields on this window, the settings will be retained for the remainder of the current session.
Tip: Information buttons adjacent to the Selections options provide more refined search criteria.
Fields
Descriptions

Selections

Reference 1
The [Transaction Types - Reference 1](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/general-ledger-inquiries-overview/more-gl-inquiry-selections/transaction-types---reference-1) for the individual line item displays in the list box.

Reference 2
The [Transaction Types - Reference 2](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/general-ledger-inquiries-overview/more-gl-inquiry-selections/transaction-types---reference-2), if applicable, for the individual line item displays in the list box.

Description
Enter a description for the search criteria (for example, enter Payroll for Payroll G/L transactions), use the SuperSelect options, or press Enter to accept the software default of ALL.

Cost group
If your company is using cost centers, a Cost group field displays. The field name is user-defined.

Sources

Enter the transaction source code you want displayed on the G/L Detail Analysis Inquiry screen, use the SuperSelect options, or press Enter to accept the software default of ALL.

Account status

- Active

- Inactive

- Not used

Select the checkboxes for those account statuses you want to include on the G/L Detail Analysis Inquiry.
