# Budget File Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Year

- Spectrum maintains unlimited years of active data.

- The year associated with the current General Ledger processing date defaults.

- To change the year, enter the two-digit year that corresponds to the fiscal year in which the data is located, or select a different year from the drop-down menu.

G/L account
Enter the G/L account code.

Account types
Select the checkboxes for the types of accounts to display on the report. Press Enter to only display Income and Expense accounts.
