# Unposted Transactions Inquiry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Report
Click to go to the [Unposted Transactions Report](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/reports-overview/unposted-transactions-report).

Selections

Module
Click the drop-down arrow to search or enter the two-character module code to include in the inquiry.

Through transaction date
Enter the latest transaction date to include on the inquiry.

Include only sources with unposted transactions?
Clear this checkbox to include zero-balance transaction lines in the inquiry results; otherwise, leave this checkbox selected.

Buttons

Refresh
Click to refresh the screen with recalculated transaction counts.

Update
To update a selected unposted transaction, highlight the transaction line and click this button. Depending on the transaction source module you have selected to update, an update starting screen will display.

List box

Count
A list of unposted transactions for each Transaction source
 displays.

Transaction source
The transaction source of the unposted transaction displays.
