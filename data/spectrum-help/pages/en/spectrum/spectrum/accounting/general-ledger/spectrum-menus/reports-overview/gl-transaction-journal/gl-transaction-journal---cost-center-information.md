# G/L Transaction Journal - Cost Center Information

If your company is using cost centers, a Cost group field displays. When a cost group or cost center is specified, the report will show only G/L activity assigned to that cost group/cost center; the cost center associated with each transaction line will display on the report. In addition, the report can be sorted by cost center.
If you specify a cost center on this screen, Spectrum verifies you have permission to access that cost center before proceeding. Spectrum will include G/L account activity on the report only if you have permission to access the cost center assigned to the G/L transaction. Spectrum compares the cost center assigned in the G/L transaction detail with cost centers in the your assigned scheme, and if the transaction cost center is not included, then that transaction is not included on the report.
