# Copy Financial Report Parameters

The Copy/Delete Report Parameters screen is used to copy
 existing report parameters to a new report or to delete existing report parameters from the
 software.
When modifications to an existing report are made, it is recommended that you first make a
 copy of the report and then make your modifications to the copied version of the report.
 When all modifications have been made and the report prints correctly, the original report
 can be safely deleted.
