# G/L Summary Analysis Inquiry - Cost Center Information

If cost centers are set to 'Yes' in the current company, the screen header will include an option for Cost group. When a cost group or cost center is specified, the screen shows only G/L details assigned to that cost group/cost center. Spectrum will verify that your operator has permission to access that cost center before proceeding.
