# Unposted Transactions Inquiry

Use this screen to display unposted G/L transactions for a specified account during the selected period.
The system looks for contract records using scheduled dates through the current period that have not been updated (earned) using Earned Revenue Processing. Contracts with an Earned revenue basis of As billed are not included in unposted transactions.
