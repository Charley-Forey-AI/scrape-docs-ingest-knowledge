# Recurring Journal Entry Update

This update makes automatic journal entries. It selects each active recurring journal entry with a frequency match for the fiscal period requested.
After this update is finished, you need to preview the journal entry activity report and perform the associated update.
After Recurring Journal Entries are established using Recurring Journal Entry, the Recurring Journal Entry Update posts these entries to the regular Journal Entry file as required. These journal entries will not be updated to the General Ledger until the Journal Entry Activity Report and Update have been run. Therefore, one-time changes to the recurring journal entry (such as an amount or a G/L code) may be made in the Journal Entry screen prior to the final update.
Note: This update is only available if you have security
 authorization.
When the update is finished, Spectrum will display the from/to journal entry numbers that have been created.
