# New/Edit Journal Entry

The New/Edit Journal Entry window is used to add, change, and look up unposted General Ledger journal transactions. For each journal, up to 999 detail transaction lines can be entered; the software ensures that each journal entry number is unique by automatically assigning a number for each entry based on the last number found in the Journal Entry History log file. The system calculates the balance of these transactions as they are entered.
Any remarks stored in this screen can be used in conjunction with the [ G/L Detail Analysis Inquiry](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/general-ledger-inquiries-overview/gl-detail-analysis-inquiry), [Journal Entry History Report](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/reports-overview/journal-entry-history-report), [Journal Entry Activity Report](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/data-entry-overview/journal-entry-activity-report), [Journal Entry Activity Update](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/data-entry-overview/journal-entry-activity-report/journal-entry-activity-update), and [Recurring Journal Entry](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/data-entry-overview/recurring-journal-entry) screens.
