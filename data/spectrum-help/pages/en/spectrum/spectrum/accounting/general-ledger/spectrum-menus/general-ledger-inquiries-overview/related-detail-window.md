# Related Detail window

This window shows related G/L entries for the currently highlighted transaction and total debit and credit amounts. This is a view-only window.
Fields
Descriptions

Financials

Debit total
This is the sum of the amount displayed in the list box Debit column.

Credit total
This is the sum of the amount displayed in the list box Credit column.

Activity
This is the difference between the Debit total and the Credit total.

Journal remark
Remarks entered in the [Journal Entry](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/data-entry-overview/journal-entry) screen display in this field, and the Source is Journal entry (JE).
The text shown displays from the first entry since the same text is stored for each line of the journal entry, and only a single journal entry appears in this window at a time.
