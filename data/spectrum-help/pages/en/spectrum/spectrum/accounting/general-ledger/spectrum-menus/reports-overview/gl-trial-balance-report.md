# G/L Trial Balance Report

Use the G/L Trial Balance Report screen to print a trial balance. This report may be run either in detail (listing each transaction), or in summary (providing a list of changes in account balance information).
You can use an alignment mask on this report. Additionally, it is capable of displaying numbers up to 13 digits long, allowing extremely large figures to display, rather than a string of asterisks (*). This is helpful when reviewing the total lines for large reports printed across multiple periods.
