# Set up a Cash Flow Statement

Use the Design Financial Reports screen to set up and edit your Cash Flow Statement report.
Full validation of any new Cash Flow Statement requires comparison to a trusted version of one which exists outside the Spectrum application. If you have questions about creating or validating your new report, please consult your CPA.Note: Creating a new report can be made easier if you happen to already have among your reports one which is similar to the report you are creating. If you find one, use it as a starting point as indicated in the steps below.

Creating your own Cash Flow Statement is a three-part process, and each part requires a few steps of their own. The main parts are:

- Create the report heading.

- Define the report output by grouping rows according to your existing Chart of Accounts.Tip: Consider using this [worksheet](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/reports-overview/set-up-a-cash-flow-statement/cash-flow-statement-setup-worksheet), being sure to note if you need to include or exclude certain codes in a specific range.

- Define the report columns.

These steps include all parts. To set up a new Cash Flow Statement:

1. Go to General Ledger > Maintenance > Design Financial Reports.

1. (Optional) If you're using an existing report as a starting point:

1. Click Copy.

1. Enter the two-character ID of the report you wish to copy.

1. Enter a new, unique two-character ID.

1. Name the new report and click OK.

1. In the Report ID field, enter the new report's two-character ID and press Enter or Tab.Skip to Step 4.

1. In the Report ID field, enter a new unique two-character ID and press Enter or Tab.

1. Under Format, click Change Report and choose a report format:

- Financial Report - Cash Flow

- Financial Report - Cash Flow Fiscal

1. As appropriate, select any other checkboxes. If needed, refer to the [Design Financial Reports - Field Descriptions](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/design-financial-reports/design-financial-reports---field-descriptions).The first part - the report heading - is now complete.

1. Click Save.

1. In the Command Bar, select Rows.A new window displays your GL accounts in a grid. The second part starts now.

1. Enter the Report ID you chose when setting up the report heading.

1. Press Enter or Tab.

1. Use the cells in the Start account and End account columns to define the ranges of accounts based on your GL Chart of Accounts.Note: Click on a row to activate it.

1. To insert a new row:

- Identify and select a neighboring row.

- Click the Insert button.

- Choose the appropriate row type.

1. As appropriate, edit other cells. If needed, refer to [Define Financial Report Rows - Field Descriptions](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/design-financial-reports/define-financial-report-rows/define-financial-report-rows---field-descriptions).The second part - defining the report rows - is now complete.

1. Click Save.

1. In the Command Bar, select Columns.

1. Enter the Report ID you chose when setting up the report heading.

1. Press Enter or Tab.A new window displays the report columns. The third part starts now.

1. As appropriate, edit the cells. If needed, refer to [Define Financial Report Columns - Field Descriptions](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/design-financial-reports/define-financial-report-columns/define-financial-report-columns---field-descriptions).The third part - defining the report columns - is now complete.

1. Click Save and close the report designer.

Generate a Statement of Cash Flow report. See [Print Financial Reports](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/reports-overview/print-financial-reports).Important: Validate the accuracy and completeness of your new report by comparing it to a trusted Cash Flow Statement prepared using your Spectrum data but prepared outside Spectrum.
