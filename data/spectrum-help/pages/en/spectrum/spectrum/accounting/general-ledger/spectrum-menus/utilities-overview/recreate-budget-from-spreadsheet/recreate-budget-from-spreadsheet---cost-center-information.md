# Recreate Budget from Spreadsheet - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, cost center information can be included when importing budget information from the spreadsheet. The file layout has been altered to accept a fourteenth column; the informational text on the starting screen for this update references the use of column 14.
