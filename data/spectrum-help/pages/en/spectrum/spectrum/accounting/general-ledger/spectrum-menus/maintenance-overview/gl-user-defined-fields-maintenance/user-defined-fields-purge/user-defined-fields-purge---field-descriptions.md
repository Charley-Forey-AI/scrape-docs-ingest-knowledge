# User-Defined Fields Purge - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Comp
The code for the company that uses this field displays.

Type
The user-defined field type (Alpha, Numeric, or Date) displays.

Prompt
The user-defined field's title (for example, DUE DATE) displays.

Purge
Click this button to purge the selected user-defined field.
