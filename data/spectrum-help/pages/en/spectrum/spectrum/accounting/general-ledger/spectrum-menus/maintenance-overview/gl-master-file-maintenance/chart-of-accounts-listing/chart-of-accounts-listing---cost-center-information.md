# Chart of Accounts Listing - Cost Center Information

If your company is using cost centers, the Chart of Accounts Listing screen includes a cost group field and checkbox in the Selections section.
If a cost center is specified in this field, the listing will display only accounts assigned to that cost group/cost center. The starting screen also includes an option to include cost center security detail on the listing.
If a cost center is specified on this screen, Spectrum verifies that the operator has permission to access that cost center before proceeding.
When running the Chart of Accounts Listing, Spectrum will include the account on the listing only if your operator has permission to access the account. Spectrum compares the account's list of shared cost centers with cost centers in your assigned scheme, and if there are no common cost centers, then that G/L account code does not display on the listing.
