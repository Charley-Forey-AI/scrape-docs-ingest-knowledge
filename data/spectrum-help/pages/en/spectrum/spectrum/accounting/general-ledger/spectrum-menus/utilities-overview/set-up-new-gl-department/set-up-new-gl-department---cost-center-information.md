# Set Up New G/L Department - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, Spectrum will copy only those accounts that you have permission to access in the source department. Spectrum compares the account's list of shared cost centers with cost centers in your assigned scheme, and if there are no common cost centers, then that G/L account is not copied during the update.
