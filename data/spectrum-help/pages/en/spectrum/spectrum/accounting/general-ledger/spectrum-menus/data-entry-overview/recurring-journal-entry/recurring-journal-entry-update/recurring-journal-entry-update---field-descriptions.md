# Recurring Journal Entry Update - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Journal date

Transaction date
The current General Ledger transaction date displays. Press Enter to have recurring Journal Entries posted to this date, or enter a different date. Click the drop-down arrow to select a date from the Search Calendar Dates window.

Fiscal year
The fiscal year associated with the transaction date displays.

Period
The period number in which the specified processing date is contained (number 1-13) displays.

Current period start
The first day of the period associated with the transaction date displays.

Current period end
The last day of the period associated with the transaction date displays.

Selection

Journal entry #
Enter the recurring journal entry number.

Proceed with update

Yes, proceed with update
Click to proceed directly with the update.

No, cancel update
Click to cancel the update and exit the screen.
