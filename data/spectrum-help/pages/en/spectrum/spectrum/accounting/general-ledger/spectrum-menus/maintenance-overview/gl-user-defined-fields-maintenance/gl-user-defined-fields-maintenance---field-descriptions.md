# G/L User-Defined Fields Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Descriptions

Company
If the Show non-company-specific fields (company ALL) checkbox is clear on the G/L Installation screen, then only user-defined fields associated with the current company display.

Type
The field type, Alpha, Amount, or Date, displays.

- Alpha displays if this is a text field.

- Amount displays if this field contains dollar amounts or numbers.

- Date displays if this field contains numerical dates.

Prompt
The name for this field (for example, DUE DATE) displays.

Validated
If the specified user-defined field has validation codes, Yes displays.
If this user-defined field does not have validation codes, No displays.

Mask
The user-defined alignment mask displays.
When a report is initially created, the software produces a default display code that can be overwritten. If you change the field length after creating the report, you must also change the corresponding display code. See the table below for appropriate codes.

Status
The status (Active or Not Used) of the user-defined field displays.
