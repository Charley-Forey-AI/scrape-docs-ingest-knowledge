# G/L Balance Audit Report - Cost Center Information

If your company is using cost centers, a Cost group field displays in the Selections section. When a cost group or cost center is specified, the report will show only G/L activity assigned to that cost group/cost center.
Spectrum includes G/L account activity on the report only if you have permission to access the cost center assigned to the G/L transaction.
