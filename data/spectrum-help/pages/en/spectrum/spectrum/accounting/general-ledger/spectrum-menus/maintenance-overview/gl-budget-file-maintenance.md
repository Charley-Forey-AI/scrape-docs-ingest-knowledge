# G/L Budget File Maintenance

Use the G/L Budget File Maintenance screen to add, change, delete, or view budget amounts for any General Ledger account code in any of the four years it has been maintained in the software.
Note: Any credit balance account entries (for example,
 revenue/income accounts) must be entered as negative numbers.
Click the Import button to open the [Recreate Budget from Spreadsheet](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/utilities-overview/recreate-budget-from-spreadsheet) window.
