# User-Defined Field Numeric Selection

This window displays when an amount user-defined field is selected in the Chart of Account Listing screen. Use this window to define the numeric range for user-defined fields that will display on the Chart of Accounts Listing.
Field
Description

From value
Enter the minimum numeric value to include on the Chart of Accounts Listing.

To value
Enter the maximum numeric value to include on the Chart of Accounts Listing.

Include all values?
Select this checkbox to include all numeric values; otherwise, leave this checkbox clear to select a numeric range.
