# Printing a Spreadsheet to File: Microsoft Excel

Complete the following steps to print a Microsoft Excel®
 spreadsheet to a file.

1. After creating a template in Excel, highlight the
 spreadsheet to select it.

1. Save the spreadsheet file type as Formatted Text
 (space delimited, *.prn).Important: Be sure to type the complete path,
 placing the file in the same directory as the other Spectrum accounting files. The
 file name must end with the .prn extension. Additionally, your spreadsheet must be
 13 columns wide, and the column width should be resized to 11.
If the spreadsheet is comprised of multiple screens, a warning
 will display indicating the file does not support multiple screens. When managing
 workbooks, you must save each active screen as an individual file.
