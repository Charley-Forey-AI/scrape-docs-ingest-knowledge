# Purge G/L Budgets - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Purge through year
Enter the four-digit year for which you want to purge all records from the budget file. Once purged, these records will no longer be available for financial reports or inquiries.

Authorization to Purge
Click Continue to remove the selected records; otherwise, click Cancel to exit without updating.
