# G/L Balance Audit Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Fiscal year
Press Enter to accept the current fiscal year. Otherwise, enter a new year, click the drop-down arrow, or click the Date Calculator to change it.

From fiscal period
Enter the first fiscal period to be included on this report, or press Enter to begin with the current fiscal period based on the current General Ledger processing date.

To fiscal period
Enter the last fiscal period to be included on this report, or press Enter to end with the current fiscal period based on the current General Ledger processing date.

Format
Select to display Exceptions only default or All pertinent records on the report.
