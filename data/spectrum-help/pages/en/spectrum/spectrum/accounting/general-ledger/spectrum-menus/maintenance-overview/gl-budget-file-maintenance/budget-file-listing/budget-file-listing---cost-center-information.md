# Budget File Listing - Cost Center Information

If your company is using cost centers, a cost center field and checkbox display beneath the G/L account field.
When you specify a cost center, the Budget File Report shows only budget entries assigned to that cost center. Spectrum verifies that your operator has permission to access the cost center before proceeding.
When you run the report, Spectrum includes budget entries on the listing only if you have permission to access the cost center.
Spectrum compares the account's list of shared cost centers with cost centers in your operator's assigned scheme. Budgets for that G/L account code will not display on the report if there are no common cost centers.
When an account includes budget detail for both authorized and unauthorized cost centers, this report will show only those values that you are allowed to view.
