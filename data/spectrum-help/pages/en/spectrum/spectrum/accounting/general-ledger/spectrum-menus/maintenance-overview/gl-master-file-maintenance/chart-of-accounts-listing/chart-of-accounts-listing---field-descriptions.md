# Chart of Accounts Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Selections
Enter the G/L account code and Department code. Use the drop-down arrow to view the codes.

Account status
Select to view accounts with Active, Inactive, and/or Not used status.

Sort by
Select one of these options to sort your report by.

- Account code

- Account number

- User-defined field

If you select User-defined field, the fields in the User-defined field selection section become available.

Account types
Select as many account types as you would like to display on the chart of accounts listing.

- Assets

- Liabilities

- Capital

- Income

- Expense

- Statistic

User-defined field
This field is only available if you select the User-defined field option in the Sort by section. Enter the user-defined field you want to display on the chart of accounts Listing.

-  If you select an Amount type user-defined field, the [User-Defined Field Numeric Selection](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/gl-user-defined-fields-maintenance/user-defined-field-numeric-selection) window displays.

- If you select a Date type user-defined field, the [User-Defined Field Date Selection](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/gl-user-defined-fields-maintenance/user-defined-field-date-selection) window displays.

Selection criteria
This field is only available if you select the User-defined field option in the Sort by section.

- The range displays if you enter a numeric range in the User-Defined Field Numeric Selection window or a date range in the User-Defined Field Date Selection window.

- If you enter an alpha user-defined field in the User-defined field, enter the selection criteria.
