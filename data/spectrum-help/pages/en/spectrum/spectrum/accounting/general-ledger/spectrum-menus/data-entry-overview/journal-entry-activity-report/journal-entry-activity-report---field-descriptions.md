# Journal Entry Activity Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Journal entry
 Enter the journal entry number, or press Enter to accept the software default of ALL.

From/To journal entry date
 Use these fields to select a date range for updating journal entries. Enter the earliest and latest date you want to include on the report, or press the Date Calculator button to select from a list of date options.

Sort by
Sort the report either by Journal entry or G/L account.

Workflow
If the Workflow module is present and the Journal Entry workflow is active, select whether to include completed and available workflows on the report, or workflows currently in progress.
