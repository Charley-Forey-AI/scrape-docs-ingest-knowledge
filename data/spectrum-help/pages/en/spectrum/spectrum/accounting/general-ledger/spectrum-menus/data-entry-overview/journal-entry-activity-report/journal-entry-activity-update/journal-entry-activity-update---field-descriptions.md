# Journal Entry Activity Update - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Journal entry #
The journal entry selection displays.

From/To journal entry date
The from/to journal entry date displays.

Cost group
The cost centers associated with the journal entries selected for update display. This field displays only if you are using cost centers in the current company.
