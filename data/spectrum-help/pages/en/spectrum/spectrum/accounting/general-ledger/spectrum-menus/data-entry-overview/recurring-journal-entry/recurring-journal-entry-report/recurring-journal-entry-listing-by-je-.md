# Recurring Journal Entry Listing by JE #

This report will show work order details when there are
 work orders attached to the journal entry detail lines. Attaching work orders to the journal
 entry lines allows actual miscellaneous costs to be accrued to a specific work order. These
 'actual cost' transactions can then be added into Work Order > Other Charges Entry for billing purposes.
