# Define Financial Report Columns

Use the Financial Report screen to define the columns or
 fields that will appear on the financial report.
This screen should be used after the main Design Financial Reports
 screen has been completed and before completing [Define Financial Report Rows](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/maintenance-overview/design-financial-reports/define-financial-report-rows).
