# Change G/L Code Listing

Use the Change G/L Code Listing screen to print a list of
 changes entered in the Change G/L Account
 Codes screen. You should perform this function before accessing any additional
 options.
Warning: You cannot cancel the update after it has been
 performed.
Before updating, review the listing carefully to make sure all account
 codes are the same length and assigned properly.
