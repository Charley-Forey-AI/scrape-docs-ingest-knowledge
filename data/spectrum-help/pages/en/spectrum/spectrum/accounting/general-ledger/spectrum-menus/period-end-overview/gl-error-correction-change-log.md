# G/L Error Correction Change Log

Use the Error Correction Change Log screen to select and print any error listings. Once you make your error correction selections and click Preview or Print, the Log Report will print and the G/L Error Correction Log Purge screen will display with the option to purge the log. Only those items included on the report will be purged. To proceed with the purge, click Yes, proceed with purge and click OK.
