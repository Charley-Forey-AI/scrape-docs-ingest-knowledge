# G/L Summary Analysis Inquiry

Use this screen to display G/L transaction records, including actual and budget amounts for the current period, prior period and year-to-date, plus corresponding prior year amounts for comparison purposes.
The grid can be sorted by re-arranging columns, grouping by specified columns, or filtering based on additional settings by clicking the [Selections, Summary](/en/spectrum/spectrum/accounting/general-ledger/spectrum-menus/general-ledger-inquiries-overview/gl-summary-analysis-inquiry/selections-summary) button.
