# Consolidated Financials Update

The Consolidated Financials Update screen merges information from the General Ledger for those companies defined as consolidated sets, prior to printing consolidated financial reports.
This function should only be executed within companies that have been defined as consolidation companies. If any modules besides General Ledger are installed in the company, the update will not be performed. Additionally, the update will not be performed if there is no company set up in the Consolidated Financials Control screen.
