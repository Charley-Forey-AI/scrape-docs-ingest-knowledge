# Change G/L Account Codes - Cost Center Information

 If the cost center feature is enabled in the Enterprise Installation screen, Spectrum allows you to add a new account code only if you have permission to access the account. Spectrum compares the account's list of shared cost centers with cost centers in your assigned scheme, and if there are no common cost centers, then that G/L account cannot be added. However, G/L codes not in your assigned scheme can still be edited and deleted.
