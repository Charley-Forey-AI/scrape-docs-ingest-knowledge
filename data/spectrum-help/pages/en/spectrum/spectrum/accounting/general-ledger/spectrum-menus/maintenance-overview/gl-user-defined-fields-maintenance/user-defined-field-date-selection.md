# User-Defined Field Date Selection

This window displays when a date user-defined field is selected in the Chart of Accounts Listing screen. Use this window to define the date range for user-defined fields that will display on the Chart of Accounts Listing.
Field
Description

From date
Enter the beginning date that will display on the Chart of Accounts Listing, press F4 to select a date on the Date Change window, or press Enter to start with the first processing date.

To date
Enter the beginning date that will display on the Chart of Accounts Listing, press F4 to select a date on the Date Change window, or enter the current General Ledger processing date.
