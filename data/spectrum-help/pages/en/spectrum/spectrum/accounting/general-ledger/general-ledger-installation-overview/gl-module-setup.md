# GL Module Setup

Steps to begin setting up your company in the General Ledger
 module.

1. On the Menu System screen, select the Admin button.The Installation menu displays.

1. On the Installation menu, double-click Company Installation.

1. Select the Modules tab.

1. Confirm that the General Ledger checkbox is
 selected.

1. Complete the rest of the Company Installation screen and
 select OK to save the
 information and return to the Installation menu.
