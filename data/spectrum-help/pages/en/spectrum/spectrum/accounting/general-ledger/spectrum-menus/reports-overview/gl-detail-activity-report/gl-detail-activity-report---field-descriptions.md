# G/L Detail Activity Report - Field Descriptions

Use the table below for reference when filling out the report launcher.
FieldsDescriptions
G/L accountEnter the G/L account number you want to view. Select the drop-down arrow to open the Search G/L Accounts window.
DepartmentEnter the Department you want to view. Select the drop-down arrow to open the Search G/L Departments window.
Fiscal yearPress Enter to accept this fiscal year or click on the date calculator to select the calculation method.
From fiscal periodEnter the first fiscal period to be included on this report, or press Enter to begin with the current fiscal period based on the current General Ledger processing date.
To fiscal periodEnter the last fiscal period (1-13) of the selected fiscal year to be included, or press Enter to accept the default entry.
Direct costSelect the direct cost types to include on the report.

Report format

- Detail activity - the standard report

- Detail activity CSV - export as a CSV file for deeper analysis. Report headers appear only once, at the top of the CSV file.

SourcesSelect the source of the transactions to be included on the report. Use the Clear All and Select All buttons accordingly.
