# Non-Standard Column Heading

Select No or Both from the Standard columns headings menu in the Design Financial Reports screen to open this window and add the text that displays over each column on the report.
Fields
Descriptions

Line
Line numbers are assigned and maintained by the software.

Start column position
Enter the number of the column in which the heading will begin.

Text
Enter the text that will display in the column heading. A column heading may contain up to three lines.
