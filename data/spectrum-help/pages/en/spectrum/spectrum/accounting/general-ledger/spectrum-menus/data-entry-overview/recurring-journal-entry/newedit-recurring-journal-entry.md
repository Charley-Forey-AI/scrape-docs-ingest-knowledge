# New/Edit Recurring Journal Entry

The New/Edit Recurring Journal Entry window is used to add,
 change, and look up unposted General Ledger recurring journal transactions.
For each journal, up to 999 detail transaction lines can be entered; the software
 automatically assigns a number for each entry. The system calculates the balance of these
 transactions as they are entered.
