# Purchase Order Log Print - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field
Description

P.O. number
Enter the purchase order number,
 double-click in this field to select from a list of available P.O. numbers, use
 the Superselect options, or press Enter to accept the software default of ALL.
Order date
Enter the first and last order dates to
 be included, or press Enter to start with the software default date and end
 with the current P.O. processing date. After the start
 page has been completed, a report will print and the [Purchase Order Log Purge](/en/spectrum/spectrum/accounting/purchase-order/spectrum-menus/period-end-overview/purge-closed-purchase-orders/purchase-order-log-purge) page will
 display.
