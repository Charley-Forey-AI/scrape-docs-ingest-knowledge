# P.O. Status Report - Cost Center Information

If cost centers are set to 'Yes' in the current company, a
 Cost group field will display on this screen.
Enter a valid Cost group code or Cost center code in this field to include purchase
 orders for that cost center on the report. The software will compare the cost center
 assigned to the purchase order with cost centers in the operator's assigned scheme, and
 if the P.O. cost center is not present for the operator, that purchase order will be
 skipped.
