# P.O. User-Defined Fields Purge - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field/Button
Description

Comp
The code for the company that uses the
 purchase order user-defined field displays. Click the
 column header to arrange the data by ascending order; click the column
 header again to arrange the text in descending order. Use your mouse to
 determine the width of this column by sliding the column title's side
 borders.

Type
The purchase order user-defined field's
 type (Amount, Date, or Numeric) displays.
Prompt
The purchase order user-defined field's
 title (for example, DUE DATE) displays.
Purge
Click this button to purge the selected
 purchase order user-defined field.
