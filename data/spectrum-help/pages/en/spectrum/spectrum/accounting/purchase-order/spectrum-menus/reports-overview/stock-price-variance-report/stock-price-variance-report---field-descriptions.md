# Stock Price Variance Report - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field/Button
Description

P.O. number Invoice number Item code Vendor
Enter a purchase order number, invoice
 number, item code, and vendor code, or press Enter to include ALL items on
 the report.
G/L date
Enter the beginning and ending General
 Ledger dates for this report, or press Enter to print beginning with
 the earliest date and through the current Purchase Order Processing
 date.
