# Purchase Order Log Print

This feature prints a log of purchase orders, then allows
 for removal of purchase orders from the software.
The first page which displays is the log start page; after the log has printed a second
 page displays which allows the option of purging some or all of the purchase orders which
 appear on the log.
This feature prints a log of purchase orders within the selected range. Any gaps in the numbering sequence will be noted with a "*". After a log has printed, purging will be allowed.
