# Ship Via File Maintenance - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field/Button
Description

Ship via code
Enter a unique ship via code. Sample
 shipping terms include COD, prepaid, and so forth. Ship
 via id code is the key field for Ship Via File Maintenance. This code is
 used to distinguish one record from all other records in the file. Whenever
 a purchase order is issued, these codes can be used to indicate the method
 by which the order is to be shipped.

Description
Enter a full description of the ship
 via code.
