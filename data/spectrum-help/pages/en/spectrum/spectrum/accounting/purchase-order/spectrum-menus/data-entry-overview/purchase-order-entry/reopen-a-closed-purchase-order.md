# Reopen a Closed Purchase Order

If you have security authorization to reopen a closed
 purchase order, use the following procedure.

1. If the Purchase order status is Closed, click the Reopen button at the top of the screen.

1. Click Continue to reopen the purchase order.

- If you have security authorization to edit the purchase
 order, it will reopen in edit mode.

- If you do not have authorization, the purchase order will
 reopen in view-only mode.
