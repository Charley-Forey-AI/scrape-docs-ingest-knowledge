# P/O Processing Date Change - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field/Button
Description

Processing date
The left column displays the current
 processing information. The information is re-displayed on the calendar on the
 right. Enter the new processing date by typing over the displayed date, or
 click the date on the calendar. To leave the processing date as it is
 displayed, click the Reset button to restore the present setting. Click Cancel to
 exit the screen without saving the changes. If the new
 date is before the Minimum Processing Date or after the Maximum Processing
 Date established during the Installation procedures, the following error
 message displays:
DATE ENTERED IS NOT WITHIN THE
 CURRENT MINIMUM AND MAXIMUM DATE RANGE.
To leave
 the processing date as it is displayed, click Cancel.

Fiscal year
The fiscal year in which the specified
 processing date is contained displays.
Period
The period number in which the
 specified processing date is contained (number 1-13) displays.
Period start/end dates
The day after the previous period's end
 date and the ending date of the period in which the specified processing date
 is contained displays.
Allowable entry range
The earliest date that a processing
 date can be changed and the latest date that a processing date can be changed
 to is displayed in these fields. These dates are set in the Admin >  Installation >  Processing Date Change screen.
