# Packing List Quantity Entry

The Packing List Quantity Entry page is used to record the
 arrival of materials that are ordered using the two-step purchase order receiving method.
The invoice for the order is not recorded until quantity receipts are recorded here. The
 two-step receiving method has the ability to create an inventory receipt transaction for
 each item on the purchase order. It also updates to Inventory and General Ledger.
When you select OK, the Back Order Alert window displays if

- the quantities received are less than the current amount due, AND

- the Warning for partial receipt in P.O. packing list checkbox is selected on the Purchase Order Installation page.
