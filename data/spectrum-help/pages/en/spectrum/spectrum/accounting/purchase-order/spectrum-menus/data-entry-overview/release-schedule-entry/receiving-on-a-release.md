# Receiving on a Release

If you select a specific release date while in Packing List
 Quantity Entry, the quantities scheduled for that date default automatically. Follow the steps
 below to receive quantities on a release.

1. On the Site Map, select Purchase Order  >  Data Entry  >  Packing List Quantity Entry.

1. In the P.O. number field, enter the purchase order number.

1. Complete the Sequence number, Receipt date, Pack list number, and Remarks fields. The Receiving Options window displays.

1. Double-click in the Schedule release field to select the applicable delivery date.

1. The items and quantities scheduled for this date display.

1. Edit the Received on this pack list field if you need to make any changes to the quantities received.

1. Click OK to save your changes and return to the Site Map.
