# Daily Procedures

Follow the daily procedures below for Purchase
 Orders.

- Set the processing date using P/O Processing Date Change.

- Enter purchase orders using Purchase Order Entry.

- Print entered purchase orders using Purchase Order Printing.

- Receive purchase orders, as necessary, using Purchase Order
 Receiving.

- Print the A/P Transaction Register and update.

- MAKE A DAILY
 BACKUP.
