# P.O. Packing List Detail Quantity Edit List

This report is used to view detailed listing of Purchase
 Orders that have current receiving applied to them.
You can specify to view single purchase order, range of purchase orders, or all open
 purchase orders. You can also narrow these down by vendor, job, date, and you can choose
 how to sort the entries.
