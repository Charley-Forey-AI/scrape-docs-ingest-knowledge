# Packing List Log - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field/Button
Description

Purchase order
Enter a valid purchase order number.
 The vendor associated with this purchase order will display to the right of
 this field.
Job / Work Order / Warehouse
The job, work order or warehouse code
 will display in this field, if specified in the purchase order header.
View
Click this button to open the [Packing List Quantity Entry](/en/spectrum/spectrum/accounting/purchase-order/spectrum-menus/data-entry-overview/packing-list-quantity-entry) screen to display details for the selected packing
 list and any images associated with it.

- If the status of the packing list is 'Open' or
 'Printed', this screen will open in edit mode.

- If the status of the packing list is 'Updated' or
 'Invoiced', this screen will open in view-only mode.
