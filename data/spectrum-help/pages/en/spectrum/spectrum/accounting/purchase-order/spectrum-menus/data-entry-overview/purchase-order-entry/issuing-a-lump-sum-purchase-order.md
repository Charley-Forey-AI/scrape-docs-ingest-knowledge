# Issuing a Lump Sum Purchase Order

To issue a lump sum purchase order, complete the following
 step-by-step procedure.

1. On the Site Map, click Purchase Order >  Data Entry  >  Purchase Order Entry.

1. In the Item code field enter an exclamation point <!> to indicate the item is a non-stock item.

1. In the Additional description field enter a detailed description about the items purchased on this lump sum purchase order.

1. In the Unit price field enter the lump sum amount.

1. Click Save to save your changes and print the purchase order.
