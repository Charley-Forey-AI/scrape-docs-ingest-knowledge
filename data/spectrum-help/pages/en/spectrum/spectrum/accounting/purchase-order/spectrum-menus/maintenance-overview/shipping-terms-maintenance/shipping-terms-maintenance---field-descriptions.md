# Shipping Terms Maintenance - Field Descriptions

Use the table below for assistance when completing the fields on this screen.
Field/Button
Description

Shipping terms code
Enter a unique code for each type of
 shipping terms. Sample shipping terms include COD, prepaid, and so forth. Shipping terms code is the key field for Shipping Terms
 Maintenance. This code is used to distinguish one record from all other
 records in the file. Whenever a purchase order is issued, this code can be
 used to indicate the terms for shipment.

Description
Enter a full description of the
 shipping terms.
