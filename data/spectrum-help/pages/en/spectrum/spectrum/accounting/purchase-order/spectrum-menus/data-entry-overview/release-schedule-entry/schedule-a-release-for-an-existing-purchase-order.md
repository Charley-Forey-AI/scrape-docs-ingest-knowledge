# Schedule a Release for an Existing Purchase Order

use the steps below to schedule a release for an existing
 purchase order.

1. In the Purchase order field,
 type the purchase order number.

1. Upon entering a purchase order number, Spectrum will automatically display the Release Schedule window if a schedule has not been set up for the selected purchase order. Enter a schedule of release dates for the purchase order. The schedule can be changed later by clicking the Release dates button.

1. Select the line item that you want to schedule a
 delivery date for and click the Details button to open the
 Release Details window.

1. Specify the quantities you expect to receive by the
 dates listed.Note: If you enter more items than currently listed on the
 purchase order, you will be prompted to adjust the purchase order.

1. Click OK twice to save your
 changes and return to the Site Map.
