# Change Request Status File Maintenance

Use the Change Request Status File Maintenance screen to set up status codes for cost-reimbursable change requests. Status codes set up in this screen are used for cost-reimbursable change requests only. Change requests that are not cost-reimbursable do not affect revenue or billings in any way, so they use only two status codes: Executed (recognized cost revisions only) and Proposed (no recognition of cost revisions). Only the Recognize revenue and cost revisions, billing allowed and Recognize revenue and cost revisions, billing not allowed change requests affect the revised contract amount for contracts, jobs, and phases. You are not permitted to delete a status code that is already assigned to a change request.
