# Change Order Status File Maintenance

Use the Change Order Status File Maintenance screen to set up different status codes with different billing effects for use with change orders. You are not permitted to delete a status code that is already assigned to a change order. Change orders affect the revised contract amount for contracts, jobs, and phases.
