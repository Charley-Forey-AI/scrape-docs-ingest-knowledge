# Warning - Customer Code Entered Has Inactive Status.

If an inactive customer code is entered, this message
 displays. Click OK to continue entering data, if desired.
To change the customer's status, click Accounts Receivable > Maintenance > Customer > Properties and select either Active or Not
 Used in the Status section.
