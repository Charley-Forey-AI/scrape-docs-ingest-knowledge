# Set Up Standard Fixed Billing Items

Use the Set Up Standard Fixed Billing Items screen to enter fixed billing items for fixed-priced contracts and draw requests to be set up for the company. Entries made in this screen will be available for access in Contracts and Pre-Billing Quantity Entry.
