# Delete Contracts - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Line
Line numbers are assigned by the software.

Customer code
Enter the customer code for the contract to be deleted. The Customer name displays to the right of this field.

Job code
Enter the job number for the contract to be deleted. The software will verify the job status is Complete before proceeding. The Job description displays to the right of this field.
Once a job line is entered, press Continue to purge contracts.
