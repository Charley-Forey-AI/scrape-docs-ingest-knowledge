# Item Description/Message Listing

Use the Item Description/Message Listing screen to print a listing of messages and item descriptions entered through Item Description / Message Maintenance.
Select Print messages only to print a listing of messages, select Print item descriptions only to print a listing of item descriptions, or select Print both to print a listing of both messages and items.
