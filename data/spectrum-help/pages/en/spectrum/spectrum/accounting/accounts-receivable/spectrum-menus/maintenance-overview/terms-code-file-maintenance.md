# Terms Code File Maintenance

The Terms Code File Maintenance screen is used to add, change, delete, and look up customer credit terms and related information. Once established, the terms codes are used in the Customers screen to define a default terms code that will appear in Invoice Entry. At that time, a different terms code may be recorded.
