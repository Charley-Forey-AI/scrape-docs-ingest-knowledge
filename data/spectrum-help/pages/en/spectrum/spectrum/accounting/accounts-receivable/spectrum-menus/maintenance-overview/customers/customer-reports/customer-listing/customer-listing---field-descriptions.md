# Customer Listing - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Customer
The current customer code will default in this field. If the report has a saved template, the software will apply those saved settings and then set this field to the current customer code.

Customer type
Enter the customer type you want to include on this report, or press Enter to print ALL customer types.

Salesperson code
Enter the code of the salesperson you want to include on this report, or press Enter to print ALL salesperson codes.

Format
Select the format option you want for this report: Full listing, Full listing without remarks, Name and address - short format or Alternate bill-to address

Sort by
Select the sort option you want for this report: Customer code, Alpha reference or Salesperson

Customer status
Enter the customer status you want to include on this report (Active, Inactive or Not used) or press Enter to print ALL customer statuses.

Cost Center Information
If the cost center feature is enabled in the Enterprise Installation screen, when running the Customer Master File Listing Spectrum includes the customer on the listing only if you have permission to access the customer's information. Spectrum compares the customer's list of shared cost centers with cost centers in your assigned scheme; if there are not common cost centers, then that customer is not included on the listing.
