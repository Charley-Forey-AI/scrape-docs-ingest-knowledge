# Customer Notes

The Customer Notes screen is used to add or edit notes for
 customers.

1. Click the Edit button to add notes for the
 selected topic. Topics that have existing notes will be noted by a green dot. Up to
 999 lines of notes are allowed per customer.

1. Select a different topic to add more notes, if
 available.

1. Click Save.
