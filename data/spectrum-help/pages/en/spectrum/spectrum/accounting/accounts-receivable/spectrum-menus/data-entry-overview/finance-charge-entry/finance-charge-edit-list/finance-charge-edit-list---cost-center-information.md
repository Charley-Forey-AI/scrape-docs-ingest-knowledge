# Finance Charge Edit List - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when printing the Finance Charge Edit List Spectrum will include the customer only if the operator requesting the listing has permission to access the customer's information. Spectrum compares the customer's list of shared cost centers with cost centers in the operator's assigned scheme; if there are no common cost centers, then that customer is not included.
Spectrum also verifies that the operator has permission to access the cost center assigned to the finance charge record. Spectrum compares the cost center assigned to the finance charge with cost centers in the operator's assigned scheme; if the cost center is not included, then that charge is not shown on the listing.
The Current balance column on the report shows the current balance of the associated cost center.
