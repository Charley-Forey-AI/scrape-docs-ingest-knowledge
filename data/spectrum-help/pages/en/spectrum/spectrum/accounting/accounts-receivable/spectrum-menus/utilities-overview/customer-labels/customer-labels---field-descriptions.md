# Customer Labels - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Customer
Enter the customer codes you want to include on the mailing label, or press Enter to include ALL customers.

Customer type
Enter the customer type you want to include on the mailing label, or press Enter to include ALL customer types.

Salesperson
Enter the salesperson codes you want to include on the mailing label, or press Enter to include ALL salesperson codes.

Sort by
Select the sort option you want to include on the mailing label:Alpha,Salesperson or Postal code

Customer status
Select the customer status you want to include on the mailing label: Active, Inactive or Not used

Print options
Select the print option you want: Labels or Mask

Cost Center Information
If the cost center feature is enabled in the Enterprise Installation screen, when printing customer labels Spectrum will include the customer only if you have permission to access the customer's information. Spectrum compares the customer's list of shared cost centers with cost centers in your assigned scheme; if there are no common cost centers, then that customer does not display on the listing.
