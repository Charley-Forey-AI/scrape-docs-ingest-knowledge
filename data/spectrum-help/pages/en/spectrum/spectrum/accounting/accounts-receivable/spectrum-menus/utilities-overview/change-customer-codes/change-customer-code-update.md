# Change Customer Code Update

Use this screen to change customer codes system-wide according to the information entered in the Change Customer Code screen.
You should print the [Customer Code Change Listing](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/utilities-overview/change-customer-codes/customer-code-change-listing) before proceeding with this update.

- This update performs code changes throughout the software. It is a productivity tool to assist you in making code changes on a wide scale. Be advised that any custom files, pre-release and new files, and non-updated data files may not have been included in this automated process. Therefore, confirm that updates have been performed prior to using this feature. Review records carefully, including installation screens, immediately following completion of the program.

- This update should be performed only while there are no users logged in to Spectrum. Confirm that this workstation is the only one currently in use prior to proceeding with the update.
 Note: If applicable, this update substitutes any new alternate bill-to codes in the address file.
Important: Make sure you have an accurate data backup before proceeding. This update will change customer codes system-wide according to information entered in the Change Customer Code screen. You should have a copy of the Customer Code Change Listing before proceeding with this Update.
