# Change Request Entry - Projects Tab

Use this tab to view and add project-related documents to the change request.
Note: If the Project Management module is not installed, this tab will be unavailable.
Field/Button
Description

Add Item
Click this button to add related documents to the change request. The Select Category window will open first to allow you to select a document category, and then the Add Documents window will open with only documents related to that category.

Remove
Click this button to remove the selected document from the change request.

View
Click this link to open the selected document in a new tab. This link will be hidden if the Operator does not have access to the Log Entry screen.
