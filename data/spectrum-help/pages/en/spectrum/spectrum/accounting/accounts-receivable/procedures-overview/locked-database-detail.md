# Locked Database Detail

In certain pages, the application notifies you who is locking
 the record you are trying to access. When this happens, the Locked Database Detail window
 displays.
Because you can see who is causing the lock, you can go directly to
 them, then you can select Retry.
This functionality is located in the following maintenance and data
 entry pages:

## Maintenance Screens

- Employee File Maintenance

- Vendor File Maintenance

- Customer File Maintenance

- Bank Account File Maintenance

- Equipment File Maintenance

- Fixed Assets File Maintenance

- G/L Master File Maintenance

- Inventory Items

- Job File Maintenance

- Phase File Maintenance

## Data Entry Screens

- Applicant Tracking

- Employee Management

- Authorize Recurring Benefits
