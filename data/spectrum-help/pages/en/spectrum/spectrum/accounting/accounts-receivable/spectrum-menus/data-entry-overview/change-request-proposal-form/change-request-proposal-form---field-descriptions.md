# Change Request Proposal Form - Field Descriptions

View field descriptions for Change Request Proposal
 form.
Field/Button
Description

Job Customer Change request
The job number, customer code and change request number will default from the Change Request Log.

Include change request notes?
Select this checkbox to include any change request notes on the report.

Format
Select a report format:

- Change request proposal form

- Change request proposal form with cost detail

- Change request proposal letter

Supporting documents
These options display conditionally if just one record has been selected to
 print. The Document images option only displays if
 the Document Imaging module is present, and the Related
 Documents option only displays if the Project Management
 module is present.
Select whether to include document images associated with the change request and/or related documents.
Click the Edit
 button to manually select which related documents to include with the
 change order. The Include Related Documents window
 only displays records using an HTML form.
Note: When including related documents, selections made in the
 Include Related Documents window (called from
 the Edit button) are not saved. The default window
 displays with all records selected.
