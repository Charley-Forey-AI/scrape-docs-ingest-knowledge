# Contract Summary Report - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when printing the Contract Summary Report Spectrum will include contract information only if the operator requesting the report has permission to access the contract. Spectrum compares the contract's assigned cost center with cost centers in the operator's assigned scheme; if the cost center is not present, then that contract will not be included.
Financial information shown on this report represents total activity for the contract, even if the operator is not permitted to access certain billing items for the contract.
