# New Change Order from Selected Change Requests

Use this window to create a new change order from the change request records selected on the Change Request Log screen.
Field/Button
Description

Change order
Enter a change order number, or press Enter to assign the next available change order number.

Description
Enter a description of this change order. This information can be changed or added at a later time on the Change Order Entry screen.
Note: This description displays on the Change Order Log and Profit Detail Report. It is not the description displayed in the detail of the draw request (which comes from the billing item description).

Date issued
Enter the issuing date for this change order, or press Enter to default to the current processing date. This date must fall within the A/R minimum and maximum processing dates.

Date approved
Enter the approved date for this change order, or press Enter to default to the current processing date. This date must fall within the A/R minimum and maximum processing dates.

Current status
Enter the current status code for the change order you are adding, or double-click on this field to select from a list of available change order status codes. Additional status codes may be added on the Change Order Status File Maintenance screen.

Cost breakdown

Take off total
The sum of all costs of the selected change requests displays in this field. The percentage of the change order total will display to the right of the amount.

Markups
The sum of all markup amounts for the selected change requests displays in this field. The percentage of the change order total will display to the right of the amount.

Subtotal
The sum of the take off total and markup displays in this field. The percentage of the change order total will display to the right of the amount.

Add-ons
The sum of all the add-on costs for the selected change requests displays in this field. The percentage of the change order total will display to the right of the amount.

Job total
The sum of the subtotal and add-ons total displays in this field. The percentage of the change order total will display to the right of the amount. The percentage of the change order total will display to the right of the amount.

Fee
The sum of all fee dollar amounts for the selected change requests displays in this field. The percentage of the change order total will display to the right of the amount.

Billing adjustment
The billing amount minus the job total and fee amounts for the selected change requests displays in this field. The percentage of the change order total will display to the right of the amount.

Change order total
The sum of all revenue billing items of the selected change requests displays in this field. The percentage of the change order total will display to the right of the amount.
