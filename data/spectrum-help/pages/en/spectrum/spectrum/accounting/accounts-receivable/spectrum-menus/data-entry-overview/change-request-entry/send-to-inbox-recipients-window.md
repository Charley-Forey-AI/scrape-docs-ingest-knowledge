# Send to Inbox Recipients window

 Use this window to send information to the architect, subs, owners, and so forth via the Project Management Inbox.
Field
Description

From
The sender's user ID automatically displays.

Regarding
The job number this Project Management inbox item is regarding displays.

Item
The topic this Project Management inbox item is about displays.

Reason
Enter the reason for this Project Management inbox item, or press Enter to accept the software default.

History
Click this button to view prior Project Management inbox item messages regarding this topic. A yellow diamond displays next to this button if this email has any email messages in its history.

To
Enter the recipient's operator or group ID code.

Cc
Enter the operator or group ID codes for anyone who will receive a carbon copy of this email.

Remove All
Click this button to remove all recipients from the To and Cc fields.
