# Change Request Entry - Notes

Use this tab to enter unlimited notes for the change request. Notes entered on this tab will display on the following reports:

- Proposal Form (all versions)

- Change Request Log (log print and price summary versions)

- Change Order Log

Notes entered on this tab will not be committed until the change request is saved. If the status of the job is Executed, this tab will be display-only.
