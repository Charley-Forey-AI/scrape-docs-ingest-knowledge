# Billing Entry by Percent Markup - Billing Detail window

Use this window to look up detailed information for a selected billing item. No entry is allowed.
Field
Description

App #
A list of application numbers displays.

Bill item
A list of billing item numbers associated with the corresponding application numbers displays.

Work completed
A list of work completed amounts associated with the corresponding application numbers displays.

Material stored
A list of material stored amounts associated with the corresponding application numbers displays.

Completed + stored
A summary list of work completed plus material stored amounts associated with the corresponding application numbers displays.
