# Calculate Estimate window

Use this window to enter a new rate per hour and have the software adjust the estimated hours and costs. These changes will then be reflected in the current hours and current cost, or the original hours and original cost fields, respectively (depending on what field you were in when you opened the window).
If any hours or costs have already been entered, they will default to this window, but if you change any two fields in this window the third field will then re-compute automatically.
