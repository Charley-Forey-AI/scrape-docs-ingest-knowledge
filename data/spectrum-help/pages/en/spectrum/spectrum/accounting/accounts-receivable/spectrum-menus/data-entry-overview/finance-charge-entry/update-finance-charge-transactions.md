# Update Finance Charge Transactions

This update transfers finance charges that have been calculated using the Build Finance Charge Transactions window and/or Finance Charge Entry screen. These charges are transferred to the cash receipts transaction file. when the cash receipt journal and update is run these changes are passed to the customer and general ledger files.
