# Change Request Log

The Change Request Log screen displays all proposed or executed change requests for a specified job and customer. Dollar amounts, as affected by change requests, also display. A contract must have previously been entered for this job and customer. Change requests may only be written for existing contracts.
Note: If a blank billing item record was found when creating a change order, an error message will prompt the user to assign a billing item.
