# A/R Customer User-Defined Fields Listing

The Customer User-Defined Fields Listing screen is used to print a listing of customer-related user-defined fields for a specified range of companies. This listing can include customer-related user-defined alpha fields, date fields, numeric fields and/or validation codes for the selected field type(s) and selected companies.
Field
Description

Company
Enter the company code to print, or press Enter to accept the software default of ALL.

Include alpha fields?
Select this checkbox to include the alpha user-defined fields in the Customer User-Defined Fields Listing.

Include date fields?
Select this checkbox to include the date user-defined fields in the Customer User-Defined Fields Listing.

Include numeric fields?
Select this checkbox to include the amount user-defined fields in the Customer User-Defined Fields Listing.

Print the validation codes?
Select this checkbox to include validation codes in the report.
