# Add a Standard Message Code

Use this tab to enter messages frequently printed on invoices. For example, THANK YOU FOR YOUR BUSINESS or PLEASE CALL FOR FURTHER INFORMATION might be included in this file.
Field
Description

Message code
Enter the identification code for this line. Up to ten digits can be entered in this field.

Message text
Enter the desired message. Messages entered here will be available in the Remarks window for use in the Customers and Contracts screens.
