# Unit Of Measure File Maintenance

Use the Unit of Measure File Maintenance screen to bill for multiple job types (for example, some jobs are Time + Material, some are fixed bid, some are unit price, and so forth). This screen will even allow different billing types for different parts of the same job (for example, part of the job can be a fixed price while other parts are Time + Material).
