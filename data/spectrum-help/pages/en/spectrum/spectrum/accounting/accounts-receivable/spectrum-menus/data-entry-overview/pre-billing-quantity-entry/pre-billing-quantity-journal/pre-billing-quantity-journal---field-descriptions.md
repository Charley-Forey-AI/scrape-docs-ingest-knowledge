# Pre-Billing Quantity Journal - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Quantity dates
Enter the earliest and last quantity entry dates to include on the report,or leave these fields blank to begin with the earliest date and end with the current A/R processing date.

Batch code Job number Customer Billing item
Enter the batch code, job number, customer code for the job, and billing item, or press Enter to include ALL on the report.

Entry type
Select the entry type for the report to print: Actual, Engineering or Both.
