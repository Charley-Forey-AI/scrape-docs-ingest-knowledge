# Error - Equipment Code Entered Is Not Available For Use.

If a retired equipment code is entered, this message displays. This error message disallows further data entry.
To change the equipment's status, click Equipment Control > Maintenance > Equipment to open the Equipment File Maintenance screen. At the
 Status field, enter the desired status code.
