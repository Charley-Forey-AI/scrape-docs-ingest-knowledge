# Salesperson Code Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Salesperson code
Enter a unique three-character code for this salesperson.

Name
Enter the full name of this salesperson.

Sales
The month-to-date, year-to-date, and last year sales totals display. These figures are accumulated balances of transactions updated since the last period end clear was executed.

Commission basis
Select to base commissions either on percentage of gross or percentage of margin. The margin is computed by subtracting cost of goods sold from the gross sales total.
If the Order Processing module is not present on your system, this option serves as a memo field only. If Order Processing is present, the commission settings are used during order processing.

Default commission
Enter the default commission percentage. For example, if the salesperson earns 9 ½ percent, the commission would be entered as "9.5".
If the Order Processing module is not present on your system, this option serves as a memo field only. If Order Processing is present, the commission settings are used during order processing.
