# Delete Contracts

Use the Delete Contracts screen to create a list of contracts to delete.
This list should be comprised of contracts that are completely closed and for which history is no longer needed.
The system prevents deletion of contracts that have open jobs or a non-zero balance. It provides a warning if a contract is not closed.
Important: Make a backup before you purge records from the database.
