# Change Order Entry - Add-ons

Use this tab to view all pricing add-ons to the change order. Change orders inherit the add-on codes (and their detail) from the attached change requests.
Add-on details will be displayed as a separate row. When adding a new add-on, search for pricing add-ons created in the Pricing Add-ons Maintenance screen. If a project manager needs to make a change to the add-on, it must be done back at the change request level.
Click the CR Detail button to open the selected change request in the [Change Request Entry](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/change-request-entry) screen to view executed cost changes.
