# Change Order Form - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Selections
Specify the job number, customer code, change order and status to include on the report, or press Enter to include ALL on the report.

Issued dates
Specify the starting and ending change order issue dates to include on the report.

Approved dates
Specify the starting and ending change order approval dates to include on the report.

Job status
Select a job status to print:

- Active

- Inactive

- Complete

Format
Select what report format to print:

- Standard

- Unit price
