# Contract Profit Summary Window

The Contract Profit Summary window is used to look up profit
 summary information for a specified job.
No entry is allowed. Once you finish viewing the necessary information, click OKto return to the previous
 window.
Note: If the Current estimate display only checkbox is not selected in the Job Cost Installation – Properties tab, the Original contract plus Estimate revisions plus Executed changes amount may not equal the Executed contract amount in the Estimated cost column. Changing your current estimate amount directly in the Phases screen changes the Executed contract amount without changing the Estimates revisions or Executed changes amounts. The Estimate revisions and the Executed changes amounts can only be changed through Change Request Entry.
To avoid this scenario, select the Current estimate display only checkbox in the Job Cost Installation screen. Once this checkbox is selected, Current estimates can be changed only by entering a change request.
