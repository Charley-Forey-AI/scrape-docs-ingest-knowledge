# Change Request Log Report

Use the Change Request Log Report to print a list of change requests
Note: The Change Request Log Report displays an asterisk (*) next to the rough order of magnitude amounts in the Proposal or R.O.M. column so that you can easily differentiate between proposal and R.O.M. amounts.
