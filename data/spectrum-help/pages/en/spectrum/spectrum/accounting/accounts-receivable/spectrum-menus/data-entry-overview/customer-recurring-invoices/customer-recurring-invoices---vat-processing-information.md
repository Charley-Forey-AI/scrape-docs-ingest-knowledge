# Customer Recurring Invoices - VAT Processing Information

If the 'Utilize value added tax (VAT) checkbox is selected in General Ledger Installation, select a VAT code (usually either Goods & Services Tax (GST) or Harmonized Sales Tax (HST) in the Financials section to add to the invoice total. If a Retention amount is specified, the Retention VAT will be calculated automatically.
