# Change Order Entry - Notes

Use this tab to enter unlimited notes for the change order. Notes entered here will appear on the following reports:

- Change Order Log

- Change Order Form

Notes entered on this tab will not be committed until the change order is saved. If the status of the job is Executed, this tab will be display-only.
