# A/R Processing Date Change

The A/R Processing Date Change screen is used to change the date under which this module operates.
The left column displays the current processing information. The information is re-displayed on the calendar on the right. Enter a new processing date, or click the date on the calendar. If the new date is before the Minimum Processing Date or after the Maximum Processing Date established during the installation procedures,an error message will display. To leave the processing date as it is displayed, click the Reset button to restore the present setting. Click Cancel to exit the screen without saving the changes.
