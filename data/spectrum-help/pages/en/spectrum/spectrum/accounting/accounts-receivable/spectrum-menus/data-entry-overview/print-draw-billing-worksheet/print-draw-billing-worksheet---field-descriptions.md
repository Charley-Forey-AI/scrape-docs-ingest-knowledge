# Print Draw Billing Worksheet - Field Descriptions

A reference for printing the Draw Request Billing Worksheet.
FieldDescription
Customer codeEnter the customer or press Enter to select all customers.
Job numberEnter the job or press Enter to select all jobs.
Project managerEnter the Project manager's name, or press Enter to select all project managers.Note: This field is applicable only if using job-based security.

Subtotal on billing item?Select to print subtotals for groups of billing items. Otherwise, leave this checkbox clear to list all billing items, with a total at the end. For unit price jobs, a subtotal prints automatically for each bill item.
Active jobs only?Select to include only active jobs on the report.
FormatChoose between Standard and Expanded format.
