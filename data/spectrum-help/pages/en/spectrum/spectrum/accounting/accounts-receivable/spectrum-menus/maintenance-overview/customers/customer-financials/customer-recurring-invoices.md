# Customer Recurring Invoices

Use the Customer Recurring Invoices screen to establish invoices that are sent out on a recurring basis. It serves a different function than the draw request feature. Entry is done only one time; thereafter, the invoices can be routinely generated by the software.
Note: Click the Switch button at any time to
 display a different customer.
