# Add an Item Description

Use this tab to enter item descriptions. These items may
 later be selected to appear on customer's invoices. Based on this selection, appropriate line
 transactions will be permitted.
Field
Description

Item code
Enter the identification code for this line. Up to ten
 digits can be entered in this field.

Description
Enter the full item description.

Unit of measure
Enter the unit of measure for this item, such as EA, HR,
 BF, and more.

Unit price or extension
Enter the default price that should appear on invoices
 which bill for this item. This price will be a default only; it may be
 overridden on individual invoices.
