# Setting Up Transaction Code Accounts

Use this procedure to set up transaction code accounts for
 Accounts Receivable.

1. Open the Transaction Code Maintenance screen.

1. At the Transaction code field,
 enter the alpha-numeric transaction code you want to set up in the software. For
 example, you may type B1 if you are setting up a bad debt account. Press
 Enter.

1. At the Description field, enter
 a description for the new transaction code. For example, if you are setting up a bad
 debt account, type a description such as BAD DEBT DEPT 1. Press
 Enter.

1. At the Type field, enter the
 type of transaction you are setting up: type P for payment or
 type A for adjustment. Press
 Enter.

1. In the General Ledger portion
 of the screen, enter the General Ledger number,
 G/Ldiscount taken #, and A/R G/L account
 # for the transaction code you are adding.

1. Select the Exclude from finance
 charges checkbox if you want balance dollars tagged with this
 transaction code to be excluded from finance charge calculations. Otherwise, leave
 this checkbox clear.

1. Select the Post to Cash
 Management checkbox if you want transactions using the new transaction
 code posted to the cash management module. Otherwise, leave this checkbox
 clear.

1. At the Bank account code and
 Bank account # fields, enter the bank account information
 you want associated with transaction code you are adding.

1. Click OK.

1. Open the Customers
 screen.

1. At the Customer code field,
 select the first customer for whom you are updating finance charges, and then click
 the Properties button.

1. At the Finance charge code
 field, use the drop-down menu to select the finance charge code.

1. At the Finance charge field,
 enter the monthly percentage, and then click OK.

1. Repeat steps 11 - 13 for each additional customer for whom you are updating finance charges.
