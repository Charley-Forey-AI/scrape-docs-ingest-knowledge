# Contract Notes

The Contract Notes screen is used to add or edit notes for
 contracts.
Note:
 Click the Switch button at any time to display a different
 contract.


1. Click the Edit button to add notes for the selected contract. Up to 999 lines of notes are allowed per contract.

1. Select a different topic to add more notes, if available. Topics that have existing notes will be noted by a green dot.

1. Click Save.
