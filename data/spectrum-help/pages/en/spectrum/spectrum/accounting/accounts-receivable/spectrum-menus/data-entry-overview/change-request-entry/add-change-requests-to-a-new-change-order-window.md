# Add Change Requests To A New Change Order window

This window shows all available change requests (including
 newly created change requests) that have not been assigned an owner change order number.
Use this window to assign selected change requests to a change order. If the
 Add checkbox is selected for any of the change requests listed,
 an Add Change Order window will display when the OK button is
 clicked.
Note: Only users who have security to access change orders
 can open this window.
