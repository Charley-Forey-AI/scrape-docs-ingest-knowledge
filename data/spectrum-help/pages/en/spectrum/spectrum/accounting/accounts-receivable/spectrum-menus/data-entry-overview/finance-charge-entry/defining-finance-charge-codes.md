# Defining Finance Charge Codes

Complete the following steps to define finance charge
 codes.

1. On the Site Map, click Admin  >  Installation  >  Accounts Receivable.

1. At the Default Finance charge tran code field, enter the code you want to assign for default finance charge transactions.

1. Click OK to return to the Site
 Map.
