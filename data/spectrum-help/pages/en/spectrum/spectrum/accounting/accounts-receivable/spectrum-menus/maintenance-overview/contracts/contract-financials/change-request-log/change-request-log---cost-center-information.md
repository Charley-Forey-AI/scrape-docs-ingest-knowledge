# Change Request Log - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when inquiring on change requests Spectrum will display change request details only if the operator has permission to access the job. Spectrum compares the job's assigned cost center with cost centers in the operator's assigned scheme; if the cost center is not present, then change requests for that job will not be displayed. Spectrum will also compare the cost center assigned to the contract with cost centers in the operator's assigned scheme; if the contract's cost center is not present, then inquiry on those contracts will be disallowed.
