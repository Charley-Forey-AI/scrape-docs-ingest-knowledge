# Create a Valid List for Alpha Types

User-defined alpha types should be used to create
 consistency in data entry. The alpha-numeric format supports up to 20 characters of entry, as
 well as a validated table to select from.
Follow these steps to create a list of acceptable
 answers.

1. Navigate to the User-Defined Fields
 Maintenance screen where you want to create a list of valid
 answers.

1. Highlight the user-defined field and click Edit, or click New to add a new field.

1. Select the Validated? checkbox. The
 Validation Codes button
 will appear below.

1. Click the Validations Code button.

1. Enter alpha-numeric entries and corresponding
 descriptions.

1. Click OK.
