# Pricing Add-On Maintenance - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields/Buttons
Description

Add-on code
Enter a unique add-on code in this field. When editing an add-on, this is the only field that cannot be changed.

Description
Enter a description for the add-on in this field.

Calculation method
Select which calculation method to use: Percent or Fixed.
If the method is changed later, the Fixed markup and Markup % fields will need to be changed as well.

Automatically default onto new contracts?
Select this checkbox if the add-on code should automatically be applied to new contracts? This option will be noted in the Auto default column of the listbox.

Fixed markup
Enter a fixed markup amount to automatically apply to the contract. Negative numbers and zero are allowed.
This field will only be available if the Fixed calculation method is selected.

Markup %
Enter a fixed markup percentage to automatically apply to the contract. Negative numbers and zero are not allowed.
This field will only be available if the Percentage calculation method is selected.

Markup calculation based on
Select what the markup calculation is based on:

- One cost type If this option is selected, enter a valid cost type for the current company.

- Take off total (all cost types) This option is calculated as the sum of all cost types before any markups, add-ons or fees.

- Subtotal (take off total plus markups) This option is calculated as the sum of all cost types plus their markups, and does not include any add-ons or fees.
Note: If the Fixed markup calculation method is selected, this radio group will be unavailable.
