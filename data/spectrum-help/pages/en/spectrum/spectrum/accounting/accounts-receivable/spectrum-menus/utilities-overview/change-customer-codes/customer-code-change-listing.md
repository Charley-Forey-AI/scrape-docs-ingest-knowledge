# Customer Code Change Listing

The Customer Code Change Listing screen is used to obtain a printed listing of information entered in Customer Code Change Maintenance. This report should be printed before proceeding to the [Change Customer Code Update](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/utilities-overview/change-customer-codes/change-customer-code-update).
