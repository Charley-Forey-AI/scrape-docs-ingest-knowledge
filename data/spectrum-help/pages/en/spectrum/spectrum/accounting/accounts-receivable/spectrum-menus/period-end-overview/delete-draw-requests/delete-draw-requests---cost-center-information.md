# Delete Draw Requests - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when purging draw requests Spectrum will delete only if the operator has permission to access the contract's information. Spectrum compares the cost center assigned to the contract with cost centers in the operator's assigned scheme; if the cost center is not included, then the draw request cannot be added to the deletion list.
