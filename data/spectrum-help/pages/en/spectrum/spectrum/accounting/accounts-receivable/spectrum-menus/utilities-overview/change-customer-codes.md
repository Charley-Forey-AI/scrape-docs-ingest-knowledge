# Change Customer Codes

This screen is used to change customer codes system wide. It will be necessary to run the Customer Code Change Update before codes are actually changed. You should also run the [Customer Code Change Listing](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/utilities-overview/change-customer-codes/customer-code-change-listing) before performing the [Change Customer Code Update](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/utilities-overview/change-customer-codes/change-customer-code-update). Caution should be exercised when performing this function.
