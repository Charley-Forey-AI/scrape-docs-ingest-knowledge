# Change Sales Tax Code - Field Descriptions

Use this table for reference when completing the fields on this screen.
Button
Description

New
Click this button to open the Assign New Sales Tax Code window. Enter and old sales tax code and a unique new sales tax code to proceed.

Listing
This report will show all sales tax code changes awaiting update for the current company. View the [Sales Tax Code Change Listing](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/utilities-overview/change-sales-tax-code/sales-tax-code-change-listing).

Update
The update will change sales tax codes throughout the software according to the information entered on this screen. You should have a copy of the Sales Tax Code Change Listing Report before proceeding.
