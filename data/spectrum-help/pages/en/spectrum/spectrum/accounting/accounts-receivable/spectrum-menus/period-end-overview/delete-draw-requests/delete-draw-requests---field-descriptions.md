# Delete Draw Requests - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Line
Line numbers are assigned by the software.

Job code
Enter the job number for the draw request to be deleted. If this job is not complete, an error message will display. Only completed jobs may be deleted. The Job description displays to the right of this field.

Customer code
Enter the customer code for the draw request to be deleted. The Customer name will display to the right of this field.

Application #
Enter the application number of the draw request to be deleted, or press Enter to delete ALL draw request applications for this job and customer.
Once an application # is entered, press Continue to purge draw requests.
