# Create a Draw Request for a New Job

The steps to create the first draw request application used during later billings.

1. On the Site Map, select Accounts Receivable  >  Maintenance  >  Contract.

1. Complete the Job, Customer code, Original contract, and Contract date fields.

1. Complete the Contracts sub-windows and then select OK until you return to the main menu.

1. On the Site Map, select Accounts Receivable  >  Data Entry > Draw Request.

1. In the Draw Request Billing Entry screen complete the Job, Customer, and Application# fields.

1. Enter an Application date and date through which the job is to be billed, and Architect's project #, if applicable.

1. Enter the draw in the grid. After completing entry, select OK until you return to the Site Map.

The first application has been created and is ready for processing. At the end of processing each application, Spectrum will create the next application, which will be accessed for processing all subsequent billings for each contract.
