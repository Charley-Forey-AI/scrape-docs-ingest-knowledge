# Fixed Price Subcontract Payment Worksheet

The values shown in the A/R Billing section of the report detail represent application-specific percentages, quantities, and amounts (from the specified draw request). The A/P Subcontract section always shows the current information about the subcontract, regardless of the date of the particular draw request application.
