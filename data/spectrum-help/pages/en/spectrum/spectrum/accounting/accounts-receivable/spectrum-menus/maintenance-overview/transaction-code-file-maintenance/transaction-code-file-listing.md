# Transaction Code File Listing

The Transaction Code File Listing screen prints a list of all transaction codes. As a shortcut to making multiple entries, these codes are handy for the user who is entering numerous payments or adjustments. entered in the software and the corresponding General Ledger accounts to which they update.
