# Sales Tax Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Sales tax code
Enter the sales tax code you want to include on this report, or press Enter to print ALL sales tax codes.

Invoice #
Enter the invoice number you want to include on this report, or press Enter to print ALL invoice numbers.

Invoice date
This field will only be available if the AR Installation  >  Sales tax report based on field is set to Invoice date.
Enter the first and last invoice date to print, or press Enter to print the earliest invoice date and end with the current A/R processing date.

G/L date
This field will only be available if the AR Installation  >  Sales tax report based on field is set to Invoice date. Enter the first and last G/L date to print, or press Enter to print the earliest G/L date and end with the current A/R processing date.

Payment date
This field will only be available if the AR Installation  >  Sales tax report based on field is set to Payment date.
Enter the first and last payment date to print, or press Enter to print the earliest payment date and end with the current A/R processing date.

Invoice cost group
Enter the cost group you want to include on the report, press F4 to select from a list of available cost groups, or press Enter to print ALL cost groups on the report. This field is only available if you have cost groups enabled in the current company.

Invoice source
Select the invoice source from the following options:

- Include both types

- Include Accounts Receivable invoices only

- Include Order Processing invoices only

Sort by
Click the sort option that corresponds to how you want the report information sorted:

- Tax code

- Job

Saved Selections
All report starting screens offer a Saved Selections bar and Save Current button that can be used to remember saved user selections the next time this report is accessed. Whenever the Saved Selections bar is available, the Date Calculator feature is also available for date, fiscal year, and fiscal period fields.
Buttons
Save Current
Located in the Saved Selections bar, click this button after you have made your report selections if you think you want to save them for future use. A widow opens that allows you to enter a name for the report "template" that you are saving, and you can set this template as your default. Additional settings allow you to share this template with other operators or to assign it as a system-wide default (thereby ensuring that everyone is seeing the same items).
Preview
After making your report selections, click this button to preview the report in PDF format. From here you can review the information and then print, save or email the file to someone else.
Export
Click this button to open the Export Report window. Here you can choose a report format (including PDF, RPT, RTF, CSV, and XLS), designate a name for the report (or accept the default name) and decide if you want the report to include auto-generated bookmarks.
My Reports
Use the My Reports window to add and maintain any custom Crystal reports. This feature can then be used in conjunction with the report filters feature to set the default report to a custom report.
