# Where does certain information on the draw request come from?

The following information can be found in the header portion of the AIA / Draw Request Format.
Via (Architect)
This information is stored in the Properties
 window of Contracts. (Accounts Receivable > Maintenance > Contracts)
Architect's Project No:
Entered in Draw Request Billing Entry.
Contract Date:
Found on the main screen of Contracts. (Accounts Receivable > Maintenance > Contracts)
Contract For:
Comes from Comment Line #1 found in the Remarks
 window of Contracts. (Accounts Receivable > Maintenance > Contracts)
