# Cash Receipts Check Detail window

Use this window to look up all of the items paid on a selected check. If an image file is attached, an icon will display in the far left column of this window.

- Click the Invoice image button to scan in a new invoice image, or associate an existing image file with the check.

- Click the Check image button to scan in a new check image, or associate an existing image file with the check.
