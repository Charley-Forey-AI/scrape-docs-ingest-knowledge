# Purge Alternate Bill-To Addresses - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when running the Alternate Bill-To Address Purge Spectrum will purge the customer's address only if the operator requesting the purge has permission to access the customer's information. The software compares the customer's list of shared cost centers with cost centers in the operator's assigned scheme; if there are no common cost centers, then addresses for that customer cannot be purged.
