# Summary Billing Entry

When draw requests are printed, billings print on two screens per job. The first screen, called a header sheet, summarizes the billing. The second screen shows detail. This screen, which resembles the AIA form, is used to prepare the header sheet for a draw request. Dollar values display based on Draw Request Billing Entry input. Retainage and tax may be adjusted using this screen.
To assist in A/R billing, a memo field of tax instructions is used from Tax Classification File Maintenance. These instructions will display and print directly below the tax code field on the draw request billing form, and will also appear on the draw request Summary Billing Entry screen. Please refer to the Job Cost manual for more information on tax classification codes.
