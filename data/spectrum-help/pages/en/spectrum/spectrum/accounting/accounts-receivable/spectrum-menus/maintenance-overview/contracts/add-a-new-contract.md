# Add a New Contract

You can add a new contract as needed.

1. Click the New button.

1. Enter a Customer code.

1. Enter a Job code. If the selected job is a Master job, this
 field will be labeled Master job and the grid will include
 sub-jobs for the master.The software will then open the [Contract Main Properties](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/maintenance-overview/contracts/contract-properties/contract-main-properties) screen where the rest of the contract information can be
 entered.
