# Accounts Receivable Data Conversion

Below is a summary of the steps needed to complete
 installation. It is important that these steps be completed in the order listed.

1. Complete all of the tabs on the Accounts
 Receivable Installation screen.

1. Complete the Accounts
 Receivable portion of the Processing Date
 Change screen.

1. Set processing dates using A/R Processing
 Date Change.

1. Enter all applicable sales tax codes using
 Sales Tax Code Maintenance.

1. Print the sales tax codes and check for
 accuracy.

1. Enter all customer terms codes using Terms
 CodeFile Maintenance.

1. Print the customer terms codes and check for
 accuracy.

1. Enter the transaction type codes for payments and
 adjustments using Transaction Code File Maintenance.

1. Print the transaction type codes and check for
 accuracy.

1. Enter salesperson codes, if applicable, using
 Salesperson Code Maintenance.

1. Print the salesperson codes and check for
 accuracy.

1. Enter invoice item descriptions and standard messages
 using Item Description/ Message Maintenance.

1. Print the item and message codes and check for
 accuracy.

1. Enter unit-of-measure pricing information using
 Unit of Measure File Maintenance.

1. Print the unit of measure codes and check for accuracy.

1. Enter standard billing items, if planning to utilize
 Draw Request Billing Entry, using Set Up Standard Fixed Billing
 Items and/or Set Up Standard Unit Billing
 Items.

1. Print a list of standard billing items and review for accuracy.

1. Enter change request status codes, if applicable,
 using Change Request Status File Maintenance.

1. Enter invoice user-defined fields, as needed, using
 Invoice User-Defined Fields Maintenance.

1. Enter customer user-defined fields, as needed, using
 Customer User-Defined Fields Maintenance.

1. Print the Change Request Status File Listing and review for accuracy.

1. Enter change order status codes, if applicable, using
 Change Order Status Code Maintenance.

1. Enter any alternate billing addresses, if applicable, using Alternate Bill-To Address Maintenance.

1. Print the Alternate Bill-To Address Listing and review for accuracy.

1. Print the Change Order Status File Listing and review for accuracy.

1. Enter all customer information using
 Customers.

1. Print the customer file and check for accuracy.Note: If Job Cost is present on your system, complete the Job
 Cost conversion (see the Conversion Checklist in the Job Cost Help file for
 details).

1. Enter all contract information using
 Contracts.

1. Print the contract information and check for accuracy.

1. Enter all current invoice balances for non-job
 accounts using Invoice Entry.

1. Enter all current invoice balances for job contracts
 using Invoice Entry.

1. Enter a lump-sum amount per contract for all prior
 invoices on job contracts using Invoice Entry.

1. On the General Ledger
 Installation screen, clear the Accounts Receivable
 invoices checkbox and clear the Accounts Receivable cash
 receipts checkbox.

1. Print the Invoice Entry Edit Listing and check the
 balance information for accuracy.

1. Print the Sales Journal and update.

1. Enter cash receipts for invoices using Cash
 Receipts/Adjustment Entry.

1. Print the Cash Receipts Journal and check for
 accuracy. Update.

1. Print and retain the Aged Open Items Report.

1. Enter change orders executed prior to the conversion
 date using Change Order Entry.

1. Print change orders and check for accuracy using the
 Change Order Log Print.

1. On the General Ledger
 Installation screen, select the Accounts Receivable invoices and
 Accounts Receivable cash
 receipts checkboxes so that these figures will post to the General
 Ledger.

1. On the Accounts Receivable
 Installation screen, select the Post to Job Cost from A/R invoice
 checkbox. This should be done whether or not the Job Cost module is installed.
