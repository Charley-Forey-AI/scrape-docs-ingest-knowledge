# Customer User-Defined Fields

 Use the Customer User-Defined Fields screen to assign user-defined fields to the customer.
Field/Button
Description

Prompt
The field label displays. No entry is allowed. To assign a value to the selected user-defined prompt, click the Edit button and then complete the Edit Customer User-Defined Field window.

Current setting
Valid user entries for this user-defined field display.

Edit button
Click to display the Edit User-Defined Field window and make changes to a user-defined field.
