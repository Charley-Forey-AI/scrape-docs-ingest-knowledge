# Reclassifying a Retention Receivable Balance to Current Using Draw Requests

Use this procedure if you are reclassifying a retention
 receivable balance to current with a draw request.

- If you are reclassifying a retention receivable balance to current without a
 draw request, see [Reclassifying a Retention Receivable Balance to Current](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/draw-request-entry/reclassifying-a-retention-receivable-balance-to-current).

- If you would like to pay the reclassified amount or offset the retention
 positives and negatives, see [Paying Reclassified Retention Amounts](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/draw-request-entry/paying-reclassified-retention-amounts).

1. Go to Accounts Receivable > Data Entry > Draw Request.

1. On the Draw Request Billing
 Entry screen, click Summary.

1. Enter the job number for the retention balance you
 want to convert, and press Enter.

1. The Customer field will
 default. Press Enter.

1. At the Application # field,
 enter the current draw request application number and press
 Enter.

1. The cursor will move to the
 Retainage field for percentage amount. Enter '0' to
 set the retainage balance percent to zero and press
 Enter.

1. The cursor will move to the
 Retainage field for dollar amount. Press
 Enter to leave this field clear.Note: You can use the
 Retainage field to calculate part of the
 retainage, either by entering a percent or an actual dollar amount (press
 Enter at the percentage field to access the dollar amount field). The draw
 request application can now be updated whether or not additional work has
 been completed; reclassifying this retainage as current can be the only task
 you perform.

1. If there are any materials stored, enter '0' in
 the Materials stored field.

1. Click OK to return to the
 Draw Request Billing Entry screen.

1. Update the draw request and complete processing, as usual.
