# Print Draw Billing Worksheet

Use the Print Draw Billing Worksheet screen to print the draw billing worksheet. The worksheet for each job is typically given to the project manager. The project manager then completes the worksheet and returns it for entry into the Draw Request Billing Entry screen.
This worksheet uses the default unit price and quantity display masks from Contracts. This is useful in designating how unit price and quantity figures display on these reports, on a line-by-line basis, particularly when tracking very large or very small amounts.
