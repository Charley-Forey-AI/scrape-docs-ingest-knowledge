# Build Finance Charge Transactions - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Period-end date
The current Accounts Receivable processing date displays by default. Enter the date that will be used to age the outstanding invoices. This would normally be the last day of the month, but may be any day.

Number of days grace period
Enter the number of days to be allowed as a grace period. This field may be left blank if no grace period is allowed.

Minimum finance charge
Enter the minimum finance charge, if any. If a calculated finance charge is less than this minimum, the customer will be charged the minimum amount. For example, "1.5" will generate a $1.50 minimum charge if any finance charge between 1¢ and $1.49 is computed.
