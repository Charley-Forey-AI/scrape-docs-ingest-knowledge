# Setting Up Draw Request to Match a Contract

To add one or more draw request to a contract, complete the
 following steps.

1. On the Site Map, click
 Job Cost  >  Maintenance  >  Job.

1. Complete the fields in the Jobs
 screen. This enables Spectrum to automatically set up the contract in
 Accounts Receivable  Contracts.


1. Click OK until you return to
 the Site Map.

1. On the Site Map, click
 Accounts Receivable  >  Maintenance >  >  Contract.

1. Complete the Contracts screen.


1. Click the Details button.


1. Enter the billing item information in the
 Contract Detail window and then click
 OK until you return to the Site Map.


1. On the Site Map, click
 Accounts Receivable  >  Data Entry  >  Draw Request.

1. Complete the Job, Customer, and
 Application # fields.

1. Complete the fields in the grid. Once you update the first sequence number, Spectrum automatically creates any subsequent sequence numbers.

1. In the Update Draws to Invoice
 Entry screen, select the Continue button to
 create an invoice in the Accounts receivable module.

1. Select the Next checkbox next
 to the draw request(s) to have Spectrum automatically create subsequent draw request
 numbers.

1. In the Print Report
 confirmation window, click Preview.

1. Click OK. The
 Update window displays.

1. Select Continue.
