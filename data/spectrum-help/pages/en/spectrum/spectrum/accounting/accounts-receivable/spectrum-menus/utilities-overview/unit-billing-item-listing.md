# Unit Billing Item Listing

The Unit Billing Item Listing screen provides a printout of
 all billing items entered through Set Up Standard Unit Billing Items.
This report prints the job number, if any, entered in [Set Up Unit Price Billing Items](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/maintenance-overview/set-up-unit-price-billing-items).
