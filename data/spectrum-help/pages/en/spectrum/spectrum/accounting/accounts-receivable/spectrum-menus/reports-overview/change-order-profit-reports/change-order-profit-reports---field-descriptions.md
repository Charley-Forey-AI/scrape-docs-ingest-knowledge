# Change Order Profit Reports - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Selections
Specify a Job, Customer, Change order and Change order status to include on the report, or press Enter to include ALL.
Select whether to include contractor pricing, subcontractor pricing, change order notes and change request details on the report.

Issued dates
Specify the beginning and ending change order issue dates to include on the report.

Approved dates
Specify the beginning and ending change order approval dates to include on the report.

Format
Select a report format:

- Detail report

- Summary report

Job status
Select the job status(es) to print:

- Active

- Inactive

- Complete
