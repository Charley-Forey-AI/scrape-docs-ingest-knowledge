# Setting up a New Transaction Code

You can set up a new transaction code in the Accounts
 Receivable module.

1. On the Site Map, click Accounts Receivable  >  Maintenance  >  Transaction.

1. Enter a two-character code. (for example, RC = returned check).

1. At the Description field
 enter a description for this code.

1. At the Type field, enter A for
 adjustment.Note: When entering the three G/L account codes, they should be set up the same
 as the cash receipt transaction code that was originally used to enter the
 check.

1. Complete the Bank and
 General Ledger portions of the screen and click
 OK to save your changes.
