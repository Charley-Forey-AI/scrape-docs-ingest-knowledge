# Add Change Requests To Change Order window

Use this window to add one or more change requests to an
 existing change order.

1. In the Add Change Order or Edit Change Order windows,
 click the Add button.

1. In the Add Change Requests To Change Order window,
 select the change requests you want to add to the change order by clicking the
 checkbox in the Add column. Multiple change requests can be
 added to a change order.

1. Click OK.
