# Copy Fixed Billing Items - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

From company
Enter the company code from which the billing item(s) will be copied.

To company
Enter the company code to which the billing item(s) will be copied.

Billing item
Enter the billing item code to copy, or press Enter to copy ALL.
