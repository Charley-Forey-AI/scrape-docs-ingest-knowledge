# Transaction Code File Maintenance

Use the Transaction Code File Maintenance screen to add transaction codes and set up finance charge codes. Transaction codes are used in the Accounts Receivable module to tell the software which General Ledger accounts are to be updated. Examples of transactions codes include: cash payments received, reverse payments (as for NSF checks), write-off bad debt, reverse finance charge, adjustments to customer's account, and so forth.
