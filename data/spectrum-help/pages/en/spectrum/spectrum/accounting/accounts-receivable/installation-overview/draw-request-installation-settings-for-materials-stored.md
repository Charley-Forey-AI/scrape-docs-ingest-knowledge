# Draw Request Installation Settings for Materials Stored

Spectrum offers two different ways to display materials
 stored on a fixed price draw request.

## Option A

The software will add the materials stored column to the previous
 applied amount and will clear the materials stored.

## Option B

The materials stored column will remain the same on the next draw
 request.
Note:
 The following procedure does not affect unit price draw requests.



- On the Site Map, click Admin  >  Installation  >  Accounts Receivable.

- Click the Draw Request tab.

- Option A – Select the Transfer materials stored to previous app
 checkbox. If this checkbox is selected, then the software will add the materials
 stored column to the previous applied amount and clear the materials stored.
 Option B – Leave the Transfer materials stored to previous app checkbox clear.
 If this checkbox is left clear, then the materials stored column will remain the
 same on the new draw request.

- Click OK twice to return to the Site Map.
