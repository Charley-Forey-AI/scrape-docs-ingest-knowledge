# Change Order Entry - Change Requests

This tab will display all the change requests attached to this change order. Changes made on this worksheet will not be committed until the change order is saved.
Field/Button
Description

Add CR
Click this button to open the Add Change Requests to Change Order window. If the Job+Customer+ Change Order exists and the is not already associated with a draw request, select one or more change requests from this window to add to the new change order.
Note: If a blank billing item record was found when creating a change order, an error message will prompt the user to assign a billing item.

Remove
Click this button to remove the selected change request from the change order.

CR Detail
Click this button to open the selected change request in the [Change Request Entry](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/change-request-entry) screen.
