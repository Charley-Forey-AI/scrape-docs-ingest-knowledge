# Add a New Customer

You can add a new customer as needed.

1. Click the New button.

1. Enter a new customer code. The software will verify the customer code is not in
 use.

1. Enter a customer name and press Enter.

1. Select a terms code and sales tax code for the customer and click
 OK. The software will then open the [Customer Main Properties](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/maintenance-overview/customers/customer-properties/customer-main-properties) screen where the rest of the customer
 information can be entered.

Note: If you do not have security authorization to add or edit customers, you
 will only be able to view existing customers.
