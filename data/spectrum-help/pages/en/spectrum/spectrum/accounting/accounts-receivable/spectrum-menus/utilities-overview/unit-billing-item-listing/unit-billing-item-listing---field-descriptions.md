# Unit Billing Item Listing - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Description

Customer code
Enter a customer code, or press Enter to accept the default of ALL.

Cost Center Information
When the Enterprise Installation Use cost centers option is set to Yes or Pending for this company, when running the Unit Billing Item Listing Spectrum will include the customer on the listing only if the operator requesting the report has permission to access the contract. Spectrum compares the contract cost center with cost centers in the operator's assigned scheme; if the cost center is not present, then that customer is not shown on the listing.
Spectrum includes non-job customer billing items on the listing only if the operator requesting the report has permission to access the customer. Spectrum compares the customer's list of shared cost centers with cost centers in the operator's assigned scheme; if there are no common cost centers, then billing items for that customer are omitted.
Inclusion of standard billing items is not affected by cost center setup.
