# Add Tax Rate Revision window

Use the Add Tax Rate Revision window to add new or upcoming tax rates.
Field
Description

Tax code
The tax code defaults from the Sales Tax Code Maintenance screen.

Rate
Add or edit the new tax rate.

Effective as of
Add or edit the date that the new tax rate will take effect. Spectrum determines which tax rates to use based on the Accounts Receivable processing dates.

Comments
Add or edit a brief description of the tax rate or reason for the rate change.
