# Change Request Revenue Detail window - Unit Price

The Change Request Revenue Detail window is used to enter revenue detail information for a selected change request. If the change request is for a fixed price job, the [Change Request Revenue Detail window - Fixed Price](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/change-request-revenue-detail-window---fixed-price) will open instead.
Note: If you are accessing this window from the Edit Change Request window where the change request is associated with an owner change order, entry is not allowed in this window.
Field
Description

Total cost
The total cost of the change request displays.

Profit
Enter the profit percentage for this change request.

Revenue amount
Enter the revenue amount for this change request. Once an amount is entered, double-click on this field to calculate the change request profit.

Seq
The software automatically assigns a sequential line number to the prices listed.

Group
Enter a group number.

Item #
Enter an item number.

Description
A description displays based on the group and item numbers you selected.

Um
The unit of measure code displays based on the group and item numbers you selected.

Unit price
The unit price displays, based on the group and item numbers you selected.

Quantity
Enter the change request quantity for this item.

Amount
Enter the change request revenue amount for this item.

Balance
The balance owed for this billing item displays.
