# Why isn't my change order transferring onto my draw request?

When attempting to transfer a change order / change request, if it does not appear to be transferred, check the following change order/request fields: Status, Approval date, Billing item(s).
It is also possible that the change order / change request has already been
 transferred. To verify if this is the case, print the current or previous draw request
 to see if the change order / change request appears on the draw request. If the change
 request / change order is on the draw request, it has already been transferred. You can
 also open the change order and click the Edit button. If a message displays stating that
 the change order has already been transferred onto a draw request, please contact
 Viewpoint Support by visiting the [Viewpoint Customer Portal](https://support.viewpoint.com/s/).
