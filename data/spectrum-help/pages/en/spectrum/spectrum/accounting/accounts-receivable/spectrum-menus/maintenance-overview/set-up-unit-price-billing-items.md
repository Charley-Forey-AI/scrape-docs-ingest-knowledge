# Set Up Unit Price Billing Items

Use the Set Up Unit Price Billing Items screen to set up unit price billing items for each customer, and build a file of standard billing items for unit price contracts and draw requests. Each billing item represents a portion of the job for which the contractor may bill the owner according to its percentage of completion.
