# Sales Tax Code Listing

The Sales Tax Code Listing screen prints a list of all sales tax codes entered in Sales Tax Code Maintenance. It includes month-to-date, quarter-to-date, and year-to-date sales tax figures accumulated since the last period end clear was performed.
