# Error - Record not in file CR.CNTR.

This error message displays if you are processing job information or performing an inquiry for a job that meets one of the following criteria:

- The Customer code field in the Jobs > Properties window is blank.
--OR--

- The customer code was entered in the Jobs  >  Properties window after the initial job setup was performed.

If either criterion is met, the contract was not automatically set up in Spectrum when the job was set up. To add a contract for the job, go into Account Receivable  >  Maintenance  >  Contractsand complete the applicable fields.
