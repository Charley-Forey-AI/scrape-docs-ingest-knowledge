# Change Order Entry - Costs

Use this tab to view all cost changes made to phases from the selected change request added on the cost worksheet.
Click the CR Detail button to open the selected change request in the [Change Request Entry](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/change-request-entry) screen to view executed cost changes.
