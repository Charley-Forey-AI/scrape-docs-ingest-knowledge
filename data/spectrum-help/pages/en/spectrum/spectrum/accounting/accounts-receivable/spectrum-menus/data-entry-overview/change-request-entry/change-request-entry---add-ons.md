# Change Request Entry - Add-ons

Use this tab to add pricing add-ons to the change request. All values on this tab will default from the Pricing Defaults for Contract Changes screen, but can be overridden if the change request status code is set to 4 or 5. When adding a new add-on, search for pricing add-ons created in the Pricing Add-ons Maintenance screen. Change request status codes are assigned in the Change Request Status File Maintenance screen.
There are two parts to an add-on. The first part is the revenue side. These represent costs to the project manager must include when determining the amount to bill the customer. The second part is that these amounts must also modify the current estimate and projected costs.
