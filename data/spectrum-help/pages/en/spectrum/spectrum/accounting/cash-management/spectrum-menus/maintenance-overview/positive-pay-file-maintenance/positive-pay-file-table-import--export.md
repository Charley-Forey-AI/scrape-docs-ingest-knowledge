# Positive Pay File Table Import / Export

Use this screen to export setup from the Positive Pay File Format screen to another site. When importing, pay close attention to the Overwrite existing tables checkbox to make sure you don't overwrite an electronic table by accident.
To limit what type of checks an operator can export (including Accounts Payable, Payroll, and Direct), click Admin  >  Operator Maintenance  >  Bank Account Security > Export authorization.
