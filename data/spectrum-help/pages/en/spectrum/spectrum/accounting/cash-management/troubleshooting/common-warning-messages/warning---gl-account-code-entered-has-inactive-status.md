# Warning - G/L Account Code Entered Has Inactive Status.

If an inactive General Ledger account code is entered, this
 message displays. Click OK to continue entering data, if desired.
To change the General Ledger account code's status, click General Ledger > Maintenance > Master File to open the G/L Master File Maintenance screen.
 Select either Active or Not Used in the
 Status section.
