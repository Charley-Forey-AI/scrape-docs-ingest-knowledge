# Cash Management Installation

Click the Admin button on the
 toolbar of any menu; the Installation list will appear.
From the Installation list, double-click Company. Note that this screen should
 have been previously completed by the System Administrator.
In the Company Installation >  Modules window, select the Cash Management checkbox, if it is blank.
After the Company Installation screen has been completed, click the
 OK button to save the information and return to the
 Installation list.
