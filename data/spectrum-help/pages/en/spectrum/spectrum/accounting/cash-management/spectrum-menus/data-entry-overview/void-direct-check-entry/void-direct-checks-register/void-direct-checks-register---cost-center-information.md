# Void Direct Checks Register - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, the Void Direct Check Register includes a column displaying the cost
 center assigned to each void transaction. Spectrum will determine whether all distribution
 lines of the invoice are assigned to the same cost center as in the direct check header. If
 there are multiple cost centers, then Spectrum will create balancing debit and credit entries
 for cost center inter-posting; cost center inter-posting account information is defined in the
 Cash Management Installation screen. When cost centers are used and the Allow G/L account
 overrides by cost center checkbox is selected on the Enterprise Installation screen, Spectrum
 will assign inter-posting amounts to multiple General Ledger accounts by cost center based on
 a list of override G/L accounts in the Cash Management Installation > Inter-posting Overrides window.
