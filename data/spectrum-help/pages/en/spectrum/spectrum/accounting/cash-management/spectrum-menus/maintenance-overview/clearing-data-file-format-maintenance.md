# Clearing Data File Format Maintenance

Use the Clearing Data File Format Maintenance screen to maintain file layouts of the ASCII files to be provided by the bank. The Clearing Data File Format Maintenance file contains information about which checks have been received by the bank.
Data file formats entered in the New Data File Format window will be used by the software in all Spectrum companies. Therefore, it is only necessary to set up the formats once, even if multiple companies are set up on the software.
