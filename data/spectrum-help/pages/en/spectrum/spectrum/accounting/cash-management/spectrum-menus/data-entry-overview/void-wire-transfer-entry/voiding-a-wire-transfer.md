# Voiding a Wire Transfer

Learn how to void an existing wire transfer.

1. On the Site Map, click Cash Management  > Data Entry  > Void Wire Transfer.

1. In the Bank accountfield, enter the originating bank account's code or use the drop-down menu to select from a list of available bank accounts.

1. In the Wire # field, enter the number associated with the wire transfer to be voided.

1. Click Save and then click Update. The Void Wire Transfer Register screen displays.

1. In the Bank account field, enter the code of the bank account to be updated or press Enter to accept the default of ALL.

1. In the Wire # field, enter the number associated with the wire transfer to be updated or press Enter to accept the default of ALL.

1. Click Preview.

1. In the Void File Transfers Update screen, select Continue and then click OK.
