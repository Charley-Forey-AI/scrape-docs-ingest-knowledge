# C/M Processing Date Change - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field/Button
Description

Processing date
The left column displays the current processing information. The information is re-displayed on the calendar on the right. Enter the new processing date by typing over the displayed date, or click the date on the calendar. To leave the processing date as it is displayed, click the Reset button to restore the present setting. Click Cancel to exit the screen without saving the changes.

Allowable entry range
The earliest and latest dates that a processing date can be changed to is displayed in these fields. This date is set in the Admin  >  Installation  >  Processing Date Change screen.

Reset
Click this button to restore the present processing date setting.
