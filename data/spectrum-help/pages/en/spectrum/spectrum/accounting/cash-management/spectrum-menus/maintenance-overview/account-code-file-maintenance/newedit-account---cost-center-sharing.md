# New/Edit Account - Cost Center Sharing

If the cost center feature is enabled in the Enterprise
 Installation screen, the New/Edit Account window includes a Cost Center Sharing tab.
This tab provides a checkbox to indicate whether the account is available for all cost
 centers or a specified list of cost centers. When the Available in all checkbox is clear,
 a paging list is provided to enter the applicable cost centers; the operator can also click
 the Build button to open the Shared Cost
 Center Build sub-window to automate the setup.
When a new bank or credit card account is added, the operator may assign any active cost centers, including ones that the operator will not be authorized for future access.
When editing, viewing, or deleting an account, Spectrum compares the account's list of shared cost centers with cost centers in the operator's assigned scheme; if there are no common cost centers, then that account code is not shown. Disallowed accounts are also not displayed in the list box on the main screen.
