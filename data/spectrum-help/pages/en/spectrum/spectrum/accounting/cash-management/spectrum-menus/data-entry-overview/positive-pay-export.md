# Positive Pay Export

Use the Positive Pay Export screen to export an ASCII text file to your banking institution. Upon receipt of your text file, the bank will authorize payments only on the checks that match the information sent to them.
When items are selected for inclusion in the file based on date, void checks will also be included in the export file as of the void check date; the export will write the void date, not the original check date, into the positive pay export file.
Positive pay file formats are set up using [Positive Pay File Maintenance](/en/spectrum/spectrum/accounting/cash-management/spectrum-menus/maintenance-overview/positive-pay-file-maintenance).
