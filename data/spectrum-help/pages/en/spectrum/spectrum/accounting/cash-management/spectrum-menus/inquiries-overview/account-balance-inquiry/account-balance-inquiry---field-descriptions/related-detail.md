# Related Detail

Use the Related Detail window to display additional details for the selected transactions on the Account Balance Inquiry screen.
This window will display transaction entries by transaction date, originating module source, reference description, G/L account code, debit and credit amounts. The debit and credit totals will appear above the listbox.
