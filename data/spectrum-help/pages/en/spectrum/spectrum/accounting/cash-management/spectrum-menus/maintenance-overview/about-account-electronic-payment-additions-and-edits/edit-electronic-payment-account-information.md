# Edit Electronic Payment Account Information

Follow these instructions to edit electronic payment account information.
The Account Code File Maintenance screen stores electronic payment processing information that is required by banks and varies by bank account.

1. In the Account Code File Maintenance screen, select the Go button.A list of bank and credit card electronic payments display.

1. Select an account to edit.The Edit Account window displays with three tabs: Properties, Details, and Electronic Payment.

1. Select the Electronic Payment tab to edit account information.Note: If the account you have selected is a credit card account, the Electronic Payment tab is disabled.

1. Select United States (ACH) or Canada (AFT). Note: See [ACH Electronic Payment Setup (United States) - Field Descriptions](/en/spectrum/spectrum/accounting/cash-management/spectrum-menus/maintenance-overview/about-account-electronic-payment-additions-and-edits/about-ach-electronic-payments-and-nacha/ach-electronic-payment-setup-united-states---field-descriptions) and [AFT Electronic Payment Setup (Canada) - Field Descriptions](/en/spectrum/spectrum/accounting/cash-management/spectrum-menus/maintenance-overview/about-account-electronic-payment-additions-and-edits/about-aft-electronic-payments/aft-electronic-payment-setup-canada---field-descriptions) for field requirements.

1. Make edits, then select OK.

Your next electronic payment will use this updated information.
