# Purge Direct Check History

The Purge Direct Check History screen allows the user to list and remove items from the direct check history file. Items are removed based on the selected GL date. Only checks that have been updated are stored in the direct check history file. A purge listing can be printed before updating.
Important: This report should be retained as a permanent
 audit record of the company.
