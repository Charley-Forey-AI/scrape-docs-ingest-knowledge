# Error - That Data Format Code Is Invalid.

This message appears when attempting to import cleared checks from the Clearing Data File. This message indicates that the format of the data file does not match the format expected in the bank account's Clearing Data File Format.
Review the import file to determine where it is not defined properly in the Clearing Data File Format.
