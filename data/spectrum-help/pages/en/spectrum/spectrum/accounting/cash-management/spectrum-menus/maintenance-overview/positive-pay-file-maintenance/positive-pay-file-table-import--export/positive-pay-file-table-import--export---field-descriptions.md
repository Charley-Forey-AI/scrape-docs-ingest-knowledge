# Positive Pay File Table Import / Export - Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Field
Description

Transfer type
Select the transfer type you wish to
 perform: Export
 format or Import format.
 Additional fields become accessible based on your selection here.


Export
Enter a format code and file name to export, or
 press Enter to accept the default of ALL table codes.
 This field is enabled only if the Export transfer
 type is selected.

File name
If the Export transfer
 type is selected, enter a file name or press Enter to accept
 the default.

Import
If the Import format
 transfer type is selected, select this checkbox if you want to
 overwrite existing tables during the import process. Otherwise,
 leave this checkbox clear to make sure any existing tables are not
 changed.
