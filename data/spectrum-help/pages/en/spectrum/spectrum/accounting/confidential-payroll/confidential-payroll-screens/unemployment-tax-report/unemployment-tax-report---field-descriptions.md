# Unemployment Tax Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Descriptions

Tax table code
Enter the tax table code, or press Enter to include ALL records.

Check date
Enter the earliest/latest check date range to include on this report.

Show entire social security number?
Select this checkbox to display the full social security number. This checkbox defaults to clear unless a saved filter is present.

Report type
Select the Excess option to print the unemployment excess report, or press Enter to accept the software default and print a Standard version. The Excess report recalculates and reports amounts that should have been reported, based on current tax setup information; whereas the Standard format (existing report) provides amounts that were accrued during the actual pay cycles.
The report may serve as an audit document, particularly if tax settings were not correct when payrolls were processed.
