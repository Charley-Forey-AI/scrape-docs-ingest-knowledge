# Build W-2 Forms - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields/ButtonsDescriptions
Selections
EmployeeEnter the employee code, use the SuperSelect options, or double-click on this field to select from a list of available employee codes.
From/to check dateSpecify a check date range for the update.
Social security limitThe screen will default based on the current year's tax table. Press Enter to accept this amount for the W-2s. If your U.S. tax table has already been set to the next year's limits, enter the correct dollar amount.
Retirement planUse the drop-down menu to select the pension plan indicator.

- Y – Yes, check retirement on all W-2s If you select Y, Spectrum places an X in this field of the W-2 form for all employees.

- N – No, leave retirement blank on all If you select N, Spectrum leaves this field on the W-2 form blank for all employees.

- K – Check only for 401k participants If you select K, Spectrum places an X in this field for employees that have a non-zero 401(k) deduction total.Note: During the W-2 Form Update, if you select K – Check only for 401(k)
 participants, Spectrum selects the employee's W-2 Box 13 Retirement
 plan checkbox when either type D or type S amounts are recorded in
 Box 12 of the W-2.

Write 1 state and locality per W-2?Select the checkbox to write one state and locality per W-2 form, or press Enter to leave the checkbox clear and accept the system default which will write two states and localities per W-2 form. An advantage of having one state per form is the ability to print W-2s selectively by state. An advantage of having two states per form is that fewer forms are needed to produce W-2s.

Box 12 - 401k
Alpha codesEnter the alpha code for the Box 12 field. Needs at your company may vary; please refer to IRS Box 12 filing instructions, state filing requirements and your CPA for specific uses relative to your operation.
Codes 1-4:

- Enter the alpha code for the Box 12 field. For example, D for 401(k), C for
 group term life insurance in excess of $50,000, and so forth.

- Box 12a of the W-3 form sums Box 12 figures of the W-2 forms for code letters D, E, F, G, H, S, Y, AA and BB.

- Box 12b of the W-3 form sums Box 12 figures of the W-2 forms for code letter
 CC.Note: Code CC may be entered into
 any of the four box 12 positions on the W-2.

- If a type D or type S amount is recorded in this field and the K – Check only
 for 401(k) participants retirement option is selected in the
 Retirement plan field, Spectrum automatically selects the Box 13
 checkbox on the W-2 Form Maintenance
 screen.

Note for Qualified Equity Grants:

- Employers with employees who have qualified equity grants must report the amount included in gross income under section 83(i) for the calendar year using code GG.

- Employers must also report the aggregate amount of income which employees elect to defer under section 83(i) for the calendar year using code HH.

DeductionsSelecting this button displays the 401(k)
 Deductions window and allows you to enter Box 12 amounts,

 for Box 12 entries, or see IRS "Instructions for Forms W-2 and W-3" for

 to select a code from the Payroll Deductions
 window.
Enter 1, 2, or 3 (and so on) to map this code to the appropriate Box 12 position. Entry in this field should be coordinated to match Box 12 alpha codes entered to the right in the "alpha code:" fields. You may enter an unlimited number of lines in the window to each of the three Box 12 fields. For example, if you have five 401(k) codes, enter a line for each code and set the # to the same field.
Note: Roth 457(b) plans should not be flagged as retirement plans in box
 13.

DescriptionsEnter descriptions for box 14 (if applicable).

Enter a valid deduction code, or enter an asterisk ( * ) followed by a valid

Note for Oregon Statewide Transit Tax:
The state of Oregon requires both the wages and the tax to be reported for
 each employee as part of the Statewide Transit Tax. To do this, enter code
 OR-STT in Box 14 and then link this code to an existing tax table in the
 Other
 Deductions window below by entering a plus sign (+) and the
 Oregon tax table code: +OR.
