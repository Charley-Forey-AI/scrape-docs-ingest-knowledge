# Year-End Procedures

After completing the final payroll cycle for the year, it will be necessary to close payroll records and prepare for the next year. Detailed instructions are provided in the Getting Started Guide and in the User's Guide and Year End help files.

- Make a year-end backup.

- Close the ending year.

- Perform all quarterly procedures.

- Commence W-2 processing by updating W-2s.

- Commence 1099 processing, if applicable, by updating 1099s.

- Clear Year-to-Date using Period End Clear. This must be done before the first payroll cycle of the new year.

- Change processing dates to the new year using Processing Date Change. Set the MINIMUM allowable date to January 1 of the new year to avoid posting further payrolls to the past year.

- Change tax tables as necessary. Each year, during December, Viewpoint provides year-end instructions to all clients on software maintenance, including instructions for updating Federal tax tables. Print the Income Tax Table Report and review it for accuracy. Also update worker's compensation codes as necessary.

- Run the Period End Clear, selected for Year End.

- Before the end of January, complete W-2 processing.

- Print the W-2 Form Listing and review. If necessary, make changes using W-2 Form Maintenance and reprint the listing.

- Confirm that the W-2 format is correct.

- Set the W-2 forms to print on the same printer as payroll checks.

- Print W-2s on plain paper to verify proper format.

- Print W-2s on the W-2 Forms.

- Prepare Electronic W-2s, if necessary, using Export W-2 File

- Print W-3 form.

- Complete 1099 processing, if applicable.

- Print the 1099 Form Listing and review it for accuracy.

- Make additions and changes as necessary using 1099-Misc Form Maintenance; reprint the listing as needed.

- Print 1099s on plain paper to verify proper format.

- Print 1099s on the 1099 forms.

- Prepare Electronic 1099s, if necessary, using Electronic 1099 Reporting.

- Use Pay Cycle Calendar Maintenance to reset for dates in the new year.

- Print the Fiscal Calendar and update as necessary.
