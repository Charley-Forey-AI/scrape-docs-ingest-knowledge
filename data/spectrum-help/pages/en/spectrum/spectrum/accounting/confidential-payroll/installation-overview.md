# Installation Overview

Confidential Payroll is installed through the Payroll Installation screen.
When the Is this a confidential payroll company checkbox is selected in the Payroll Installation screen, the system prompts for the company ID code. Enter the three-digit company code representing your main payroll company.
Select the Admin button on the toolbar of any menu; the Installation list appears. From the Installation list, select Company Installation. This screen should have been previously completed by the System Administrator.
Ensure that the checkbox for this specific module has been selected.
After the Company Installation screen has been completed, select OK to save the information and return to the Installation list.
