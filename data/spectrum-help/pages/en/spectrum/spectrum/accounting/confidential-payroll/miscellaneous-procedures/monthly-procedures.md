# Monthly Procedures

- Print a copy of Payroll Register in the Confidential Payroll Reports menu and save it.

- If needed, print the Union Report in the Confidential Payroll Reports menu and save it.

- If needed, print the Employment Utilization Report in the Confidential Payroll Reports menu.

- Run the Period End Clear, selected for Month-to-date. Be sure that a backup has been made before running this purge.

- Reconcile Checks

- [Checks](/en/spectrum/spectrum/accounting/confidential-payroll/miscellaneous-procedures/monthly-procedures/checks)

- [Deposits & Adjustments](/en/spectrum/spectrum/accounting/confidential-payroll/miscellaneous-procedures/monthly-procedures/deposits--adjustments)

- Print the G/L Trial Balance Report in Detail. It is not necessary to print this for all G/L accounts; print the report only for the cash account.
