# State Tax History Report

Use the State Tax History Report to print wage and tax information for a given date range.
The State Tax History Report provides historical income tax, workers' compensation, SDI/FICA, and unemployment figures.
It details earnings subject to each tax and the tax amounts computed, which is useful for federal, county, and local municipalities, as well as single-state operations.
This report is available both in summary and detail, and it may be sorted both by employee and by tax code.
