# Payroll Installation - Cost Center Information

When the Payroll Update is performed in a company set up
 for confidential payroll processing, the General Ledger of the company specified in the
 installation screen as the 'confidential company' is updated instead of the current (payroll
 entry) company. Inter-company and inter-post account setup in the Payroll Installation of that
 other company is used. In other words, the Confidential Payroll > Payroll Update will be modified to read both the inter-company and inter-post setup in the
 main company.
When setting up a cash management account, it must be set up in the main company with the same account code as the default payroll account in the confidential company.
