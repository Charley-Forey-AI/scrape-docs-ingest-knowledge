# Worker's Comp Rate Per $100 Report - Cost Center Information

If cost centers are being used in the current company, then when running the Confidential Payroll format of the Worker's Compensation Rate Per $100 Report, Spectrum will include the employee on the report only if the operator requesting the report has permission to access the employee's information. Fore employees in the current company, Spectrum compares the cost center assigned to the employee with cost centers in the operator's assigned scheme; if the cost center is not included, then that employee is not shown on the report.
For employees in other companies, cost center validation is performed based on settings in each included company. If the cost center feature is enabled in the Enterprise Installation screen for the non-confidential company, then the cost center assigned to the employee in the non-confidential company must be included in the list of valid cost centers for the operator's scheme in that other company. If the cost center feature is disabled in the Enterprise Installation screen for the non-confidential company, then all employees from that company will display on the report.
