# Worker's Comp Rate Per $100 Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Descriptions

 Worker's compensation code
Enter the worker's compensation code, or press Enter to include ALL records on the report.

 State code
Enter the state code, or press Enter to include ALL records on the report.

 Check date
Enter the first and last check dates to be included on this report, or leave these fields blank to start with the earliest date and end with the current Payroll processing date.

 Sort by
Select to sort the report by State code or Worker's compensation code.
