# State Disability Insurance Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
FieldDescriptions
State codeEnter the code of the state for which the report is to be prepared.
From check dateEnter the first check date to be included on the report.
To check dateThe default date is the current Payroll processing date; press Enter to accept this date or you can change the date.
