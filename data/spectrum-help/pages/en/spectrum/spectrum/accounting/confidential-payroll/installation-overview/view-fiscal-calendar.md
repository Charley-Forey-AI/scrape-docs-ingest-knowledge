# View Fiscal Calendar

Use this tab to select default fiscal calendar settings for the General Ledger module. These settings can be changed as needed at any time.
Fields
Descriptions

Fiscal year
Select the fiscal year you wish to view from the available drop-down list.

#
The list of period numbers for the selected fiscal year display.

Period
The period labels for the selected fiscal year display.

Start date End date
The start and end dates for the periods within the selected year display.
