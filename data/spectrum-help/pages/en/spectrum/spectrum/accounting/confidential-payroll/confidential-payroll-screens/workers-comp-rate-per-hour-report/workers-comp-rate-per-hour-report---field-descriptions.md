# Worker's Comp Rate Per Hour Report - Field Descriptions

Use this table for reference when completing the fields on this screen.
Field
Descriptions

Worker's compensation code
Enter the worker's compensation code, or press Enter to include ALL records.

State code
Enter the state code, or press Enter to include ALL records.

Check date
Enter the first and last check dates to include on the report.

Format
Select Detail to print the detail report, or press Enter to print the summary report.

Sort by
Select to sort the report by State or Worker's compensation code.
