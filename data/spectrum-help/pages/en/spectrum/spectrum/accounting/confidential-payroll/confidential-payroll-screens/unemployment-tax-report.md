# Unemployment Tax Report

The Unemployment Tax Report is a report of the contribution to be paid by the employer for the date range specified. One or all states may be included in the report. This report prints in alphabetical sequence by employee code and includes employees from all companies identified in CP Company Maintenance. The Federal Unemployment Report (FUTA) is also provided for the "US" tax table.
The Excess version of this report re-computes unemployment liability, including resident state exemptions versus the standard report, which presents amounts calculated and accrued during regular payroll cycles. The report includes all companies associated with this confidential payroll company.
The unemployment report uses the check date to determine the number of weeks worked. If two checks had check dates in the same week, then only one week will be counted.
