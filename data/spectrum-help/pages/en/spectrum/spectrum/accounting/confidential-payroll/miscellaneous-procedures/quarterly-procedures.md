# Quarterly Procedures

- Print the Payroll Register from the Payroll Reports menu while in the Confidential PR company. This report does not print combined information - please print the register for each company separately.

- Print the Quarterly Federal Tax Report in the Confidential Payroll Reports menu to obtain the necessary information for your 941 report.

- Print the Unemployment Tax Report in the Confidential Payroll Reports menu to determine the amount of unemployment taxes due.

- Print the State, County, and Local Tax Report in the Confidential Payroll Reports menu.

- Print the State Disability Insurance Report in the Confidential Payroll Reports menu.

- Print the Worker's Comp Report Rate per $100 Report in the Confidential Payroll Reports menu to determine the amount owed by your company.

- Print the Worker's Comp Rate per Hour Report in the Confidential Payroll Reports menu for the quarter.

- If needed, print the Union Report in the Confidential Payroll Reports menu.

- If needed, print the Employment Utilization Report in the Confidential Payroll Reports menu.

- Run the Period End Clear again, selected for Quarter-to-date.
