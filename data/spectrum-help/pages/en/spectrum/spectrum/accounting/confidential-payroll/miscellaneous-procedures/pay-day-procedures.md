# Pay Day Procedures

- If necessary, run Clear Previous Payroll to clear the temporary file of prior payroll cycle information. Set up the payroll cycle with the appropriate dates, payment frequencies, and deduction / add-on information. Follow the instructions for Set Payroll Cycle about selecting the deduction / add-on codes for this payroll.

- Run Pre-Time Card Update to transfer pre-time card entries into the pay cycle, if applicable.

- Enter time cards for all employees including salaried employees using Time Card Entry.

- Print the Time Card Edit List and check that all time card information was entered accurately. If changes are needed, edit the time card information and re-run the edit list.

- When the Time Card Edit List is correct, perform Check Calculation. Note that check calculation must always be run, even for manual or voided checks.

- If voided or manual checks are included in the cycle:

- Enter applicable deductions / add-ons using the Payroll Check Adjustment screen.

- Select to print the State Wage and Tax Listing and the Pre-Check Register and review the reports for accuracy. If corrections are necessary, enter corrections using the Payroll Check Adjustment screen, and print a new State Wage and Tax Listing and the Pre-Check Register.

- Select Print Checks and print the checks.

- After checks have printed correctly, print the Check Register. Save this report.

- Print and save reports:

- Job Distribution Report

- Labor Distribution Report

- Deduction Report

- Add-on Report

- If the content of the above reports is correct, select Update Payroll. This update does not generate a report.

- Print the Payroll G/L Summary Report. Save this report, then run the associated Update.

- If needed, print the Certified Payroll Report for the desired time period.

- Run the Employee Change Log Print and save a copy of the report to keep track of all changes made to employee payroll information.
