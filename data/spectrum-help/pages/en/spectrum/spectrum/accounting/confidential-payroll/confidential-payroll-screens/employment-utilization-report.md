# Employment Utilization Report

The Employment Utilization Report provides a breakdown by ethnic group of the employees working on a particular job. The report prints each job sorted by EEO Classification. The ethnic groups include African American, Hispanic, Asian, and Native American. It also provides a breakdown by gender, but if an employee doesn't have a gender specified on the Other tab of the Employee Maintenance screen, this report will treat the employee as female.
Information from this report is required on some public works jobs.
 Figures included on this report will be totals that have accumulated since the last month
 to date purge. This report includes employees from all companies identified in Confidential Payroll > Company Maintenance.
Employment Utilization Report - Cost Center Information
If cost centers are being used in this company, then when running the Confidential Payroll format of the Employment Utilization Report, Spectrum will include the employee on the report only if the operator requesting the report has permission to access the employee's information. For employees in the current company, Spectrum compares the cost center assigned to the employee with cost centers in the operator's assigned scheme; if the cost center is not included, then that employee's information is not included on the report.
For employees in other companies, cost center validation is performed based on settings in each included company. When the Enterprise Installation option for Cost centers is set to Yes in the non-confidential company, then the cost center assigned to the employee in the non-confidential company must be included in the list of valid cost centers for the operator's scheme in that non-confidential company. When the Enterprise Installation option for Costcenters is set to No or Pending in the non-confidential company, then all employees from that company will be included on the report.
