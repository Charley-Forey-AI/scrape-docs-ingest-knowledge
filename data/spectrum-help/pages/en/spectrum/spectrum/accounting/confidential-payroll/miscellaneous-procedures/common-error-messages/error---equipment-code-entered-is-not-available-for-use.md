# Error - Equipment Code Entered is Not Available for Use.

If a retired equipment code is entered, this message displays and further data entry is not allowed.

- To change the equipment's status, click Equipment Control  >  Maintenance  >  Equipment to open the Equipment File Maintenance screen. In the Status field, enter a different status code, double-click on this field, or press F4 to select from a list of available equipment status codes.
