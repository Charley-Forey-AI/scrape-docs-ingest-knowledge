# Payroll Processing Date Change

P/R Processing Date Change is used to change the date under which this module operates. If the desired processing date is outside the Minimum and Maximum processing dates previously set, refer to the Processing Date Change discussion in the Spectrum Master Help.
Once you exit this screen, the effective rates update runs automatically.
The processing date must be set separately within each application being run on the system. The displayed screen is used to set the processing date in the Payroll and Confidential Payroll modules.
