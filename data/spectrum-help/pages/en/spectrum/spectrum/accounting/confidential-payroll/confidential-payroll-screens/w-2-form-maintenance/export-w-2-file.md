# Export W-2 File

Use this screen to write W-2 information to a disk which may used to file W-2 forms electronically. After the W2REPORT file has been created, use Notepad to confirm the W-2 information has been created. To further review the contents of the electronic file, the Social Security Administration offers AccuWage Software.
