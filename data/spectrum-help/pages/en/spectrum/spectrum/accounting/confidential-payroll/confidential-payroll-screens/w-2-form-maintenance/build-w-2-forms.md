# Build W-2 Forms

The Build W-2 Forms screen clears the W-2 file and calculates W-2 information based on the payroll earnings history file.
This screen allows two localities to be included on each form by leaving the Write 1 state & locality per W-2 checkbox clear.
The previous W-2 for each employee is stored each time this update is performed. This allows you to restore a W-2 if you discover later that a needed W-2 record has been lost due to re-running the update after purging.
The system displays an asterisk ( * ) in boxes 10, 12 and 14 if there is additional detail in the window.
For information about ordering paper forms, visit [Viewpoint Business Forms](https://store.viewpointforms.com/spa#/category/29833).
