# Enter a New Subcontract Note Topic

1. Click the New button

1. In the Topic name field, type a label for the topic. For example, "Safety."


1. At the Shared Topic prompt, select to allow this topic in all companies or
 the current company only.

1. Click OK to save your entry. The topic you entered will now display in
 the Notes screens throughout the software.
