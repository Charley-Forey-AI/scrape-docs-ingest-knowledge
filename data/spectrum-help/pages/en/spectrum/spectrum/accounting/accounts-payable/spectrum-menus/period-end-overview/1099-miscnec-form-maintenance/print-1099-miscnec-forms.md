# Print 1099-MISC/NEC Forms

Use this screen to print 1099-MISC/NEC forms.
If doing year-end processing, see [1099-MISC/NEC Form Maintenance](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/period-end-overview/1099-miscnec-form-maintenance).
Information for the 1099 is generated either through 1099-MISC/NEC Form Update or through 1099-MISC/NEC Form Maintenance.
Your company's payer name, address, and telephone number shown on the 1099-MISC/NEC form are printed based on information stored in the Company Installation screen.

If doing year-end processing, see [Export 1099-MISC/NEC File](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/period-end-overview/1099-miscnec-form-maintenance/export-1099-miscnec-file).
