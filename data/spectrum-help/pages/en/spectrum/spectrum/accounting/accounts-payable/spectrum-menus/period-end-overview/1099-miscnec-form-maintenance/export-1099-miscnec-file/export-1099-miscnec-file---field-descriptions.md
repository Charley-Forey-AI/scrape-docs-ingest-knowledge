# Export 1099-MISC/NEC File - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Entity
This field only displays if cost centers are being used and entity tracking is also enabled in the current company.
After entering or selecting an Entity, the remaining starting screen fields default. Cost center security is maintained to prevent editing vendors that the operator does not have authorization to access.
If the Entity is left blank, the main company entity is used.

Year
Enter the tax year.

Prior year data?
Select this checkbox to include records from the previous year.

Set defaults?
Select this checkbox if information should default from the Company Installation screen. Clear the checkbox to enter the information manually.

Transmitter info

Transmitter's TCC
Enter the five character alpha / numeric transmitter control code. This code is supplied by the Internal Revenue Service (IRS).

Transmitter's TIN
Enter the transmitter's (preparer's) nine-digit tax identification number, as assigned by the Internal Revenue Service (IRS). It may be EIN or SSN.

Foreign?
Select this checkbox if this is a foreign corporation.

Transmitter's name
Enter the transmitter's name. The contact name is the party the IRS will contact should there be a problem. When the name is longer than will fit in this field, the name should be broken between words and continued in the next field, "Name (continued)." See the IRS specifications for further details.

Contact info

Name, address, phone, and more.
Enter the contact name and address here. The contact name is the party the IRS will contact should there be a problem. When the name is longer than will fit in the name field, the name should be broken between words and continued in the next field, "Name (continued)." See the IRS specifications for further details.

Payer info

Payer's TIN
Enter the company's nine-digit taxpayer identification number, as assigned by the IRS.

Foreign?
Select this checkbox if this is a foreign corporation.

Payer's name
Enter the company's name. This will frequently be the same as the transmitter's name. When the name is longer than will fit in this field, the name should be broken between words and continued in the next field, "Name (continued)." See the IRS specifications for further details.

Address, city, state, zip
Enter the payer's complete address here.

Payer name control
Enter the company's name control. This information is assigned by the IRS, and may be found on the mailing label of the 1099 package. The MMR (magnetic media reporting) package label does not contain this information. If the payer's name control is unknown, this field must be left blank.
After you click the OK button, a window appears with the message: file IRSTAX is now ready to copy.

Electronic filing #
If you are using Nelco for electronic processing of the 1099-MISC/NEC forms, enter the Business eFile Number in this field.
