# Process Invoices

Once Automatic Invoicing has populated the header data for the PDFs in the batch, you can review the results, correct any errors, and then send the invoices to Spectrum.
Note:If you start this procedure, and need to exit before finishing, you can save your progress and return to this page at any time. To do so, select Save and Resume Later. The saved batch will be visible on the Invoices page, where you can select Continue to re-open and complete it at another time.

Note: Only vendors with a “Last invoice date” within the last 12 months are recognized and automatically populated by Automatic Invoicing. If the Vendor field is not auto-filled, you can enter a vendor name to search for that option. When the vendor is used in subsequent invoices, Automatic Invoicing will recognize the vendor and automatically populate that field.

Important: For credit memos, make sure the amount fields are entered as negative values in Automatic Invoicing. Otherwise, Spectrum will not process the invoice correctly as a credit memo, and it will need to be changed in Spectrum.

1. Once you have reviewed an invoice, made any corrections, and entered values in the necessary fields, you can send the invoice to Spectrum. Select Send to Spectrum.

1. Alternatively, you can select Send all Invoices to send all invoices in the batch. Note this option is disabled if there are any errors in any invoice.

1. In the Send to Spectrum dialog, select SEND ALL INVOICES to send the batch to Spectrum.

1. Stay on the page and wait until it confirms that the send is complete.

Note: The Send all Invoices options must be enabled in Invoice Settings; see [Edit Invoice Settings](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/automatic-invoicing-for-trimble-construction-one-users/edit-invoice-settings).

The invoice / batch of invoices is sent to the Vendor Invoice Entry > Pending Auto Invoices screen in Spectrum.
Note: To have the Pending Auto Invoices screen appear automatically in Spectrum, from the Vendor Invoice Entry screen, select Preferences > Always open 'Search Pending Automatic Invoices' when adding a new invoice?

When an invoice is successfully sent to Spectrum, a green check mark appears on the thumbnail image.

You can also see a record of the uploads to Spectrum by selecting the Completed tab.

For details on how to continue in Spectrum, see [Enter a Vendor Invoice](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/enter-a-vendor-invoice).You can run a report in Spectrum to view information about invoices entered via Automatic Invoicing. For details, see [Automatic Invoice History Report](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/reports-overview/automatic-invoice-history-report).
