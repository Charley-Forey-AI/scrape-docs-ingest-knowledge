# Check Register History Report

The Check Register History Report provides a record of checks written. There are several parameters that may be specified to print the exact report needed, and it is possible to create a listing of all the checks written for a specific job or to a specific vendor.
The summary format provides one line per check; the detail format provides one line per invoice per check.
