# Vendor Invoice Entry - VAT Processing Information

Enabling VAT processing in General Ledger Installation enables additional 'VAT tax' and 'VAT tax amount' fields in the Vendor Invoice Entry screen.
The default value for the VAT code comes from the Default tax code field in General Ledger Installation, and the Estimated VAT tax is calculated based on the current VAT tax rate.
