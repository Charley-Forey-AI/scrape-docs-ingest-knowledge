# Subcontract User-Defined Fields - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Prompt
 To assign a value to the selected user-defined prompt, click the Edit button and then complete the Edit window.

Current setting
Valid user entries for this user-defined field display.

Field status
Click this hyperlink to determine the records that display on this screen. Unused records are those with a blank setting. Currently set fields are those that contain entries.

Edit button
 Click to display the Edit User-Defined Field window and make changes to a user-defined field.

Switch
Click this button to conveniently enter a different vendor and subcontract.
