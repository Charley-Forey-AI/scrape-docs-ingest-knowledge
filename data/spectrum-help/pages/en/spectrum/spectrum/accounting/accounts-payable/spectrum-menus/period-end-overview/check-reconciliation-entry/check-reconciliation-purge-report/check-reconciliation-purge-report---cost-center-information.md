# Check Reconciliation Purge Report - Cost Center Information

If cost centers are being used, when purging reconciled checks Spectrum removes the items only if you have permission to access them in Check Reconciliation Data Entry. Spectrum compares the cost center assigned in the check with cost centers in your assigned cost center scheme, and if the cost center of the check is not included, then the reconciled check will not be purged. Spectrum also compares the vendor's list of shared cost centers with cost centers in your assigned cost center scheme, and if there are no common cost centers, then checks for that vendor will not be purged.
