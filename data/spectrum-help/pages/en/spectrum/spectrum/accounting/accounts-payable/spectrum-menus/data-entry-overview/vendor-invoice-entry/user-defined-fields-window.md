# User-Defined Fields window

This window displays the user-defined fields for invoice notes, dates, and amounts.
Fields
Descriptions

Prompt
The title for this user-defined field (for example, DUE DATE) displays. No entry is allowed.
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

Current Setting
Valid user entries for this user-defined field display.
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

Hide Unused
Click this button to only show user-defined fields associated with this invoice.
Note: If you give a value to a user-defined field and that value
 is later removed, the user-defined field will display (with a blank value)
 when you click the Hide
 Unused button.

Edit
Click this button to edit the selected user-defined field.
