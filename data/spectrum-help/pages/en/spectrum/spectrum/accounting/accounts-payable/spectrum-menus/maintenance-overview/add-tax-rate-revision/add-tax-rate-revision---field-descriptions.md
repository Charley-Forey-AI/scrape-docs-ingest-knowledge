# Add Tax Rate Revision - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Tax code
The tax code defaults from the [Use Tax Code Maintenance](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/maintenance-overview/use-tax-code-maintenance) screen. No entry is allowed.

Rate
Add or edit the tax rate.

Effective as of
Add or edit the date that the tax rate takes effect or click the window icon to select a date from the Date Change window Spectrum determines which tax rates to use based on the invoice date.

Comments
Add or edit a brief description of the tax rate or the reason for the rate change.
