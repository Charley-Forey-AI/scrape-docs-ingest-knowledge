# Rate History - Field Descriptions

Use this table for reference when completing the fields on
 this screen.
Field
Description

Year Eff date
The year and date the selected tax rate becomes effective
 display. If an effective date falls prior to the first date in the history
 file, Spectrum will use the earliest rate revision. No entry is allowed.
 Spectrum determines which tax rates to use based on the Accounts Payable
 invoice dates.

Tax rate
The current tax rate displays.

Comments
Any comments associated with the selected tax rate
 display.
