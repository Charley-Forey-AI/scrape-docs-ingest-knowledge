# Vendor Recurring Invoices

This screen displays a list of vendor invoices that have been set up for creation a routine basis. No changes are allowed. Use the Status link to define which records display.

- Click the New button to display the [Vendor Recurring Invoice Entry](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/vendor-recurring-invoice-entry) screen.

- Click the Update button to go to the [Recurring Invoice Report/Update](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/vendor-recurring-invoice-entry/recurring-invoice-listing--update) start screen.
 Here you can enter the report parameters and print a listing.
