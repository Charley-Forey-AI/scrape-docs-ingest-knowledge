# Utilities Overview

Selections from the Utilities menu are used to change vendor codes, reset vendor types and vendor terms, copy sales tax codes from A/R to A/P use tax codes, and copy vendor master file information across modules.
Important: Functions contained on the utilities menu are normally reserved for use by supervisory and administrative personnel only.
