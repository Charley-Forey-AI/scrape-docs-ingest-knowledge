# Subcontract File Listing

The Subcontract File Listing prints information from [Subcontracts](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/subcontracts). This provides a listing of all subcontracts for a specific job, or all subcontracts with a specific vendor. Use the available fields to set the report parameters and then click the Preview button.
