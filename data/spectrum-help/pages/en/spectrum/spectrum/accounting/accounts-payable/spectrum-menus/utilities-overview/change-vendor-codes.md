# Change Vendor Codes

This screen displays a list of all codes currently in-progress for change (codes have been entered but not yet updated) and is the first step in making system-wide changes to existing vendor codes. It will be necessary to run the Vendor Code Change Update before codes are actually changed. The user should also run the Vendor Code Change Listing before performing the update.
Important: Caution should be exercised when performing any
 vendor code changes.
