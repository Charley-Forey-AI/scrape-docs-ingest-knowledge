# Deleting an Alternate Payee

Use the following procedure to delete alternate payees
 currently set up for a vendor.

1. On the Site Map, click
 Accounts Payable > Data Entry > Payment Processing.

1. Click the Alternate Payees
 button.

1. In the Vendor field, enter the
 code for the vendor whose alternate payee information you are deleting, or press
 F4 to search and select a vendor code and press
 Enter.

1. In the detail portion of the screen, press Enter through each of the alternate payees to delete the alternate payee information. Repeat this step for each alternate payee line you want to delete.
