# Vendor 1099 Listing - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when running this report Spectrum includes the vendor on the report only if the operator requesting the report has permission to access the vendor. Spectrum compares the vendor's list of shared cost centers with cost centers in the operator's assigned scheme, and if there are no common cost centers, then that vendor is not shown on the report.
