# T5018 Slip Listing

Use this screen to generate a listing report and review T5018 tax information for annual filing.

1. Specify a vendor or press Enter to accept the default of ALL.

1. The payer tax ID number that was assigned during the [Build T5018 Slips](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/period-end-overview/t5018-slip-maintenance/build-t5018-slips) update displays. No changes are allowed to this field.If 'Entity tracking' is enabled in Enterprise Installation, specify an entity code in addition to the vendor code, or leave the Entity field blank to use the main company entity. Only T5018 Slips for the specified entity will be included on the report.
