# Subcontract Phase Details - Unit Price

This screen displays if you have the Unit price checkbox selected on the Subcontract Main Properties screen. If this checkbox is clear, the [Subcontract Phase Details - Fixed Price](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/subcontracts/subcontract-phase-details---fixed-price) displays instead.
