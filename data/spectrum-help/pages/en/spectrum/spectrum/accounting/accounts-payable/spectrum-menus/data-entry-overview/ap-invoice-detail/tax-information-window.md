# Tax Information window

Use this window to modify the tax status and tax code associated with the detail lines.
Fields
Descriptions

Account code
The selected account code displays in this field.

Taxable
Select this checkbox if this detail line is taxable.

Tax code
Enter another tax code (up to 15 characters).

Sales/Use
The corresponding sales or use tax designation displays.

Tax
The tax percentage amount displays. Spectrum calculates the tax percentage based on the invoice date.

Amount
The purchase order total amount displays.

Tax amount
The amount of the tax (tax percent times the tax amount) displays. No entry is allowed, but you may return to the Tax code field and enter another tax code, if desired.
The Tax Amount window displays if an attempt is made to exit the screen before the entire invoice amount has been distributed, allowing you to complete the information entry.

Expand
Click this button to display additional fields and enter Job, Phase, and Cost type information.
