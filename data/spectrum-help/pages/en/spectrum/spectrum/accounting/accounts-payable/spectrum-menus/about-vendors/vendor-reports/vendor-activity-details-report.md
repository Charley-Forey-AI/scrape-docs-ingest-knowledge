# Vendor Activity Details Report

Review all available vendor data, including open items, payment history, notes, and more.

- If you launch this report starting screen from the Vendor Info Bar, the default vendor code is that of the current vendor.

- The dates in the From date and To date fields refer to check dates. Use these fields if you want to limit the report to a certain range of check dates. The default value of the To date field is the current A/P processing date.
