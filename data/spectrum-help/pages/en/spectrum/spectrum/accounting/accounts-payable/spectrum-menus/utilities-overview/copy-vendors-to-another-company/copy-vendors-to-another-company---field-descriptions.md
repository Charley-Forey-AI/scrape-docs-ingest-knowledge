# Copy Vendors to Another Company - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

From company
The company code of the current company displays. This will be the company from which the vendor file information will be copied. No entry is allowed.

To company
Enter the company code you want to copy vendor information to.

Vendor, Vendor type
Enter the vendor code and type you wish to copy, or press Enter to copy ALL vendors.

Status
Select the status of the vendors you want to copy. For example, you may only want to copy Active vendors.
