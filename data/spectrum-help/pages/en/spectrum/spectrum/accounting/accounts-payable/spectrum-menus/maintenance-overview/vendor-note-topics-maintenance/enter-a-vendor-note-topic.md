# Enter a Vendor Note Topic

You can use the Vendor Note Topics Maintenance screen to enter new vendor notes.

1. Click the New button to create a new note topic.

1. In the Topic name field, type a label for the topic. For example, "Safety."

1. At the Shared Topic prompt, select to allow this topic in all companies or the current company only.

1. Click OK to save your entry. The topic you entered will now display in the Notes screens throughout the software.
