# Unapproved Invoice Edit Listing

Use this listing screen to view a report with information about unapproved invoices.
This screen uses the address stored in the Address Master table.
If VAT processing is enabled in General Ledger Installation, VAT tax is included on the report if a VAT tax code is present on the invoice. The report displays the Vendor Current VAT Tax code and Vendor Retention VAT Tax.
