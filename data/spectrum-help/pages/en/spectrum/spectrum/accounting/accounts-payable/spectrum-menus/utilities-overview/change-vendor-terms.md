# Change Vendor Terms

Use this utility to change the payment terms of selected vendors previously entered in Vendors.
If the New payment
 terms selection is set to Do
 not change, AND if the New
 discount terms selection is set to Invoice date, then
 a New Days Due window displays and allows you to enter the discount
 number of days due and new discount percentage.
Important: There is no way to undo this change once it has
 occurred.
