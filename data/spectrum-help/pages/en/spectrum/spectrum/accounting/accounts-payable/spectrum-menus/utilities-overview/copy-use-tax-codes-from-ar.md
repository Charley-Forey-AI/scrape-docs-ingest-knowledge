# Copy Use Tax Codes from A/R

Use this utility to copy the sales tax codes entered in Accounts Receivable > Sales Tax Code Maintenance to the Accounts Payable > [Use Tax Code Maintenance](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/maintenance-overview/use-tax-code-maintenance) screen.
