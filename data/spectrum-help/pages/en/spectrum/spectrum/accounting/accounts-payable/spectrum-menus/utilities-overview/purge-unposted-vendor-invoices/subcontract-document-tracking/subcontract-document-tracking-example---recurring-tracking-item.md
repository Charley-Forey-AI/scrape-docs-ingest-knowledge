# Subcontract Document Tracking Example - Recurring Tracking Item

When a 'Recurring' tracking item is selected a Properties section displays in this window. Here you can specify starting and ending dates and any helpful comments.
Clicking the Recurring
 compliance hyperlink will open the Job Compliance Tracking screen with the
 selected job in context.Note: If you change the Starting
 date, then whenever 'open items' are dynamically ascertained thereafter,
 this date will refigure all iterations from the new start date forward, using the same
 interval settings. Any existing Job Compliance Log History records will remain on file,
 and the new ones will also appear.

You can also see the date-stamped Entered by operator name.
