# Vendor User-Defined Fields - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Prompt
The field label displays. No entry is allowed. To assign a value to the selected user-defined prompt, click the Edit button and then complete the Vendor User-Defined Field Edit window.

Current setting
Valid user entries for this user-defined field display.

Field status button
Click this hyperlink to determine the records that display on this screen. Unused records are those with a blank setting. Currently set fields are those that contain entries.

Edit button
 Click to display the [New/Edit User-Defined Field](/en/spectrum/spectrum/getting-started/understanding-user-defined-fields/newedit-user-defined-field)
 window and make changes to a user-defined field.
