# Export Invoice Data

You can export invoices as a .csv file from multiple places in
 Trimble Construction One. The page you choose determines how
 these invoices are grouped.

- To download all processing invoices, navigate to Accounts Payable > Invoices and select Export Processing
 Invoices.A .csv file containing all processing invoices is exported.

- To download all processing invoices from a given date range, navigate to Accounts Payable > Inbox and select Export
 [Date Range].A .csv file containing all processing invoices from the selected date
 ranges is exported.

- To download all processing invoices from a given date range, navigate to Accounts Payable > Completed and select Export
 [Date Range].A .csv file containing all completed invoices from the selected date
 ranges is exported.
