# Manual Check Register

Use this screen to preview a listing of manual check entries. After the register has printed, review it for accuracy before proceeding with the update.
At the G/L cash account field, enter the account you want this batch of check entries to update. The standard cash account from the installation screen displays as the default cash account. You may have multiple cash accounts for manual check entries.
Note: Spectrum only selects checks assigned to bank account codes to be included in the Manual Check Register. Credit card accounts are not included.
At the Batch code field, enter the batch you want to update or use the drop-down arrow to display the Unposted Manual Check Batches window and make a selection.
Important: This report should be retained as a permanent audit record of the company.
