# Export T5018 File

This screen is used to compile T5018 tax information for annual electronic filing.
If doing year-end processing, see [Print T5018 Slips](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/period-end-overview/t5018-slip-maintenance/print-t5018-slips).
Complete the fields and select Continue.
If 'Entity tracking' is enabled in Enterprise Installation, this export segregates forms by Entity. The Export file will contain the 'entity' name and address, when applicable. When a valid Entity code is present, the application looks first to the Entity to find the tax ID number assigned to Accounts Payable and if a number is not present the 'Payer tax ID number' (defined during the Build T5018 Slips) is used.

If doing year-end processing, proceed as directed at [Year End Processing Screen](/en/spectrum/spectrum/year-end/year-end-for-canada-payrolls/processing/year-end-processing-screen-can).
