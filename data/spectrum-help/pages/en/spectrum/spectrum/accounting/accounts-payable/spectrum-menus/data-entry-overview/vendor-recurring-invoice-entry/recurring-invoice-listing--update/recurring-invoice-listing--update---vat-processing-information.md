# Recurring Invoice Listing / Update - VAT Processing Information

If VAT processing is enabled in the General Ledger Installation screen, an Assign VAT tax code? checkbox and Tax code field will display after the Batch frequency field. These additional fields allows you to specify a VAT code to be assigned to all invoices created during the Recurring Invoice Update. At the Tax code field, a Search Value Added Tax Codes window is available.
