# Manual Check Register - VAT Processing Information

If VAT processing is enabled in the General Ledger Installation screen, the following calculations include VAT amounts:

- Inv /CR Amount includes total VAT

- Previously Applied' includes VAT amounts paid

- This Amount includes total VAT being paid

- Retention Paid includes retention VAT being paid
