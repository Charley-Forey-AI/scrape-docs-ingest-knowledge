# Add a Sub-Tier Vendor

1. To begin, complete the Vendor and Subcontract # fields.

1. Click New to add a sub-tier vendor (or multiple vendors).

1. Use the Relationship field to enter information that will appear in search windows, warning messages, and on [Document Tracking Reports](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/about-vendors/vendor-reports/document-tracking-reports) reports.

1. Click OK to save your changes and return to the main screen. A list of all
 the vendors associated with the subcontract displays. From here you can select a
 vendor and click Edit to
 make changes or you can click the [Vendor Main Properties](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/about-vendors/vendor-main-properties) link to open the Vendor Main
 Properties screen for the selected vendor.Tip: Assign a Not Used
 status to vendors who will only be assigned as sub-tier vendors. This must be done
 in the [Vendor Main Properties](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/about-vendors/vendor-main-properties)
 screen.
