# 1099-MISC/NEC Form Listing

1099-MISC/NEC Form Listing is used to print 1099-MISC/NEC information before the actual forms are generated. Complete the report parameter fields, preview the listing, and check the information carefully before proceeding.
If 'Entity tracking' is enabled in Enterprise Installation, specify an entity code or leave the Entity field blank to use the main company entity. Only 1099s for the specified entity will be included on the report.
