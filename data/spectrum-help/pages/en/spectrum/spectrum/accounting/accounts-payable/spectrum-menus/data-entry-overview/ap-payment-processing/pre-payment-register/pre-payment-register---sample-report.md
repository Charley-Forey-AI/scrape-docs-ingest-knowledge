# Pre-Payment Register - Sample Report

This report sorts primarily by Bank Account so that the printed checks are segregated from the ePayment and Comdata payments when multiple bank accounts are utilized. Your report may look different than the example here (for example, if you are using Alternate Payees or different processing group payment methods).
Note: This report will show separate sequences for each lien-applicable
 subcontract paid in this Payment Group.
