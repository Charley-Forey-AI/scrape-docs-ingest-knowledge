# A/P Processing Date Change - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Current settings

Processing date
 Enter the new processing date by typing over the displayed date, or click the date on the calendar.
 If the new date is before the Minimum Processing Date or after the Maximum Processing Date established during the Installation procedures, an error message displays.

Fiscal year
The fiscal year in which the specified processing date is contained displays.

Period
The period number in which the specified processing date is contained (number 1-13) displays.

Period start date Previous period end
The date range for the selected period displays.

Allowable entry range

Minimum date
The first allowable processing date displays. No entry is allowed. This date is set by the software administrator in the Installation Processing Date Change screen.

Maximum date
The first allowable processing date displays. No entry is allowed. This date is set by the software administrator in the Installation Processing Date Change screen.
