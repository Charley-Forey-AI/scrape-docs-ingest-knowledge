# Change Vendor Codes - Cost Center Information

If cost centers are being used, when adding vendors in Vendor Code Change Maintenance Spectrum allows the vendor code to be added only if you have permission to access the vendor. Spectrum compares the vendor's list of shared cost centers with cost centers in your assigned cost center scheme, and if there are no common cost centers, then that vendor cannot be added. You are able to view, delete or update the vendor records outside your cost center scheme.
