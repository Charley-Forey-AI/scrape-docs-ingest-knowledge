# Automatic Invoice History Report

Use this report to view information about invoices entered via Automatic Invoicing.
With this report you can tell which invoices required user intervention or corrections. Unless you opt otherwise, it also includes relevant data about all invoices meeting your selection criteria.
See [Automatic Invoice History Report - Field Descriptions](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/reports-overview/automatic-invoice-history-report/automatic-invoice-history-report---field-descriptions).
