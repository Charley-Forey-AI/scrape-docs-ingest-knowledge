# Change Vendor Code Listing

This screen is used to print a list of any vendor codes that have been changed using Vendor Code Change entries, and should be run before proceeding to the [Vendor Code Change Update](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/utilities-overview/change-vendor-codes/vendor-code-change-update).
