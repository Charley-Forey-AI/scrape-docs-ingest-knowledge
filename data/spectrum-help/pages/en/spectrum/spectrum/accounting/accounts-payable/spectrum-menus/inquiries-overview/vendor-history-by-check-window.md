# Vendor History by Check window

This window displays all items for the selected check. No entry is allowed.

- If an invoice image(s) is associated with the invoice, a paper clip icon will display in the far left column of this list box for the detail line. Click the Image button to view the invoice image(s).

- Click the Invoice
 button to display the [Accounts Payable Invoice Inquiry - Distribution](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/inquiries-overview/accounts-payable-invoice-inquiry---distribution) window.Note: In this window,
 the Check Number selection (and the display in the Check Number column) are
 the Transaction # when paid from a credit card account.
