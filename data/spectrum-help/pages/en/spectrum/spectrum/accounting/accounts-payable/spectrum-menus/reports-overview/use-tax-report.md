# Use Tax Report

Use this feature to produce a report based on tax type selection: use tax or sales tax.
After selecting a report type and tax code, use the Selections fields to narrow your results.

- When you select report type A/P invoices the cursor skips over the From ship date and To ship date fields.

- When you select report type Job requisitions the cursor skips from the Use tax code selection to From ship date.

- When you select report type Work orders the cursor skips over the Vendor and From ship date/To ship date fields.

The G/L date selection fields do NOT apply to job requisitions.
Important: A copy of this report should be retained as a permanent audit record of the company.
