# Accounts Payable Detail Transaction Register Sorted by Vendor

Crystal reports are designed to conserve paper. Consequently, if several line items have the same Remarks attached to them, only the first item will display the Remarks. Subsequent lines will only display Remarks if they have different messages attached.
Document Imaging Users: A warning will display if only some of the invoices appearing on this report have images attached.
