# Print T5018 Slips - Field Descriptions

A reference for completing the fields on this screen.
FieldsDescriptions
EntityIf 'Entity tracking' is enabled in Enterprise Installation, specify an entity code or leave the Entity field blank to use the main company entity. Only T5018 Slips for the specified entity will be included on the report.
VendorThis field defaults to ALL vendors unless a saved selection is present.
Report ending dateThe date assigned during the Build T5018 Slips update displays.
Print 'Payer's account number'?CRA specifications indicate that the recipient's copy not show the payer's number.
