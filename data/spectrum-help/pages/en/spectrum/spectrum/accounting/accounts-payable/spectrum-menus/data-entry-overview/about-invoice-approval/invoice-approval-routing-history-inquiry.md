# Invoice Approval Routing History Inquiry

Use this window to review the selected invoice and its associated approval notes.
Access this screen by selecting the Routing / Notes button in the Invoice Approval screen.
Fields are view only.
Changes are allowed only in screens that open using the Edit and New buttons.
When adding a new transaction, if you have set a
 preference to access the User-Defined window, it opens after
 this window closes.
