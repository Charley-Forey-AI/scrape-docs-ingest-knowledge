# Tax Percent window

Use this window to change the tax percentage amounts for an invoice or credit memo.
Fields
Descriptions

Tax code
Enter the tax code (up to 15 characters) for this line, or press
 F4 to view a list of valid tax codes.
 When Enter is pressed, the software advances
 to another field where the corresponding tax percentage displays.
 You may enter an alternate tax percentage, if desired.
This field will be reset if the Job, Phase or Cost type (Ct) fields are edited.

Type
Either Use Tax or Sales Tax
 displays (based on the setting in Use Tax Code
 Maintenance). No entry is allowed.

Tax %
The tax percent amount displays from Use Tax Code
 Maintenance. No entry is allowed.

Taxable?
Select the checkbox if taxable, otherwise leave the checkbox clear.
This field will be reset if the Job, Phase or Cost type (Ct) fields are edited.

Tax amount
The system will automatically calculate the tax amount. No entry is allowed.
