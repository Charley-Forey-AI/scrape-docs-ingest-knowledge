# Set Vendor Type

Use this utility to globally change the vendor type previously entered in the main [About Vendors](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/about-vendors) screen.
Important: There is no way to "undo" this change once it has occurred. Caution should be exercised when performing this function.
