# Subcontract Billing Advice Report

This screen provides billing-to-date information, including current and job-to-date quantities and amounts by billing item, for each subcontract progress billing invoice included in the current processing group. Select the combination of vendors and subcontracts you wish to include on the report and click Preview to generate the report.
