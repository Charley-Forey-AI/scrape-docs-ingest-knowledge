# Document Tracking, Check Distribution Action Report

- If sub-tier vendor tracking is being used, this report includes distribution items (triggered by action #5) of pertinent sub-tier vendors. These sub-tier items are listed under the related subcontract.

- For Recurring tracking items, additional lines appear on the report for each recurring iteration of the tracking item that is currently open.

- This report displays both vendor and subcontract tracking items. It also displays all of the open tracking items associated with the vendor/subcontract of that invoice.
