# Lien Details window

Use this window to enter lien detail information for this vendor and subcontract. Lien release printing is a standard feature that is part of the document tracking system. The lien release is a type of "action" item in document tracking. The legal description that prints as part of the lien release is entered at "lien description" in this screen. It will then appear in the paragraph above the signature on the lien release; the job name and address (from Job File Maintenance) and the company name (from Company Installation) also display.
Note: This button is enabled only if a subcontract has been entered.
