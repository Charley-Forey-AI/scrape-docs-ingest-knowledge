# Set Vendor Type - Cost Center Information

If cost centers are being used, during the Set Vendor Type update Spectrum will reset the vendor type for selected vendors only if you have permission to access the vendor. Spectrum compares the vendor's list of shared cost centers with cost centers in your assigned cost center scheme, and if there are no common cost centers, then that vendor is not changed during the update.
