# Running the Transaction Register

1. On the
 Site Map, click
 Accounts Payable  >
 Data Entry  >
 Vendor Invoices.

1. On the
 Vendor Invoice Entry
 screen, click the Update
 button.

1. On the
 A/P Transaction Register
 screen, select the sort option you want. You can
 choose to sort by Vendor,
 Job, G/L
 account, or All
 sorts.Note: If you
 select Vendor, a
 Sort by window displays
 when you print or preview the report. Select the
 Sort by time checkbox if
 you want the invoices to be sorted in the same
 order as you entered them.

1. At the
 Batch code field, enter the
 batch code you want to use for this update, or
 press Enter to use the
 default. Press F4 or
 double-click on this field to select from a list
 of available batch codes.

1. Click
 Preview or
 Print, and review the
 report for accuracy.

1. The
 A/P Transaction Update
 screen displays. Select
 Continue to complete the
 update; otherwise, select
 Cancel to cancel the update
 and return to the Vendor Invoice
 Entry screen.
