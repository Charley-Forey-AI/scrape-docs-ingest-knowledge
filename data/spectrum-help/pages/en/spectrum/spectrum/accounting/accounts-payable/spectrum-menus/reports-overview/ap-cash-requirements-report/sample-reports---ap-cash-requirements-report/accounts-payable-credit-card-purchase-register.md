# Accounts Payable Credit Card Purchase Register

Crystal reports are designed to conserve paper. Consequently, if several line items have the same Remarks attached to them, only the first item will display the Remarks. Subsequent lines will only display Remarks if they have different messages attached.
Document Imaging Users: This report offers links (in blue) to associated images when previewed online. A single link is shown per transaction (even if multiple images are present for that transaction, in which case the earliest stored image file displays). A warning will display if only some of the invoices appearing on this report have images attached.
