# Verifying that a Document Tracking Code is Set Up in Spectrum

1. On the Site Map, click
 Accounts Payable  >  Maintenance  >  Document Tracking.

1. Click the Items button at the
 bottom of the screen.

1. In the Tracking item code
 field, double-click on this field or press F4 to select from a
 list of available document tracking codes.

1. If you cannot locate the code you need, then you will need to set it up.
