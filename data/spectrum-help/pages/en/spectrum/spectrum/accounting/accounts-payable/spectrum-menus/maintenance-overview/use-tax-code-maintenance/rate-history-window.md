# Rate History window

Use this window to review upcoming use tax rates, with the
 newest rate displaying at the top of the list. Whenever the Accounts Payable processing date
 changes, Spectrum examines the rate file to determine if the current processing date equals or
 exceeds the next tax rate's effective date. If so, the next tax rate becomes the current
 rate.
Changes are not allowed directly in this window, but you can select a
 record and click the Edit button
 if you need to make revisions.
