# Change Request Entry

This screen provides a detailed listing of the relevant dates and past due days for each change request.
If the Project Management module is installed, this screen also shows the related documents and notes for each change request. For more information, see [Change Request Entry](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/data-entry-overview/change-request-entry).
