# Invoice Approval Routing Maintenance

Use the Invoice Approval Routing Maintenance screen to set up an unlimited number of reviewers, using operator IDs that have already been set up in Spectrum.
This screen is view only. To make changes, use the action buttons.
Type a partial entry in the Search field or select the New Search button to look up an open item by routing code, and to add records if you have security authorization to do so.

## Routing Codes Maintenance

Use the action buttons according to what you want to do.

- New - add a new Routing Code to the list. Opens the New Routing Code window.

- Listing - view a list of all existing Routing Codes, and the details pertaining to each.

- Edit - choose an existing Routing Code to modify. Opens the Edit Routing Code window.

- Delete - choose an existing Routing Code to delete.

See also [New/Edit Routing Code - Field Descriptions](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/maintenance-overview/invoice-approval-routing-maintenance/newedit-routing-code---field-descriptions) and/or [Edit an Unapproved Invoice Routing Code](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/about-invoice-approval/edit-an-unapproved-invoice-routing-code).

## Fields

- Routing Code - The routing code given when the user created it.

- Description - The description given by the user who created or edited the Routing Code.

- Confirmation operator - The operator that will ultimately finalize each invoice assigned to the Routing Code, after all reviewers have completed the approval process. Confirming an invoice moves it to Invoice / Credit Memo Entry as an unposted item.

- Approval limits - Displays Yes if the Routing Code uses [Approval Limits](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/about-invoice-approval/approval-limits).
