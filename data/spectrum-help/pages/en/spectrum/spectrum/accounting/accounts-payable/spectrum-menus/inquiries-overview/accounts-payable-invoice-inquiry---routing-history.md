# Accounts Payable Invoice Inquiry - Routing History

The Routing History tab displays the invoice approval routing history and any associated approval notes. If there is no routing history for the selected invoice, this tab will appear blank.
Select the Notes button to view any associated approval notes for the selected invoice.
Note: The Routing History tab is unavailable if either applies:

- The invoice did not originate in Invoice Approval processing.

- The Use Invoice Approval Processing checkbox is not selected in the Accounts Payable Installation screen.
