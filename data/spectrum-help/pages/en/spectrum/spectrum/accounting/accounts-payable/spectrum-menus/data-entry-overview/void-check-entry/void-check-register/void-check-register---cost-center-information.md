# Void Check Register - Cost Center Information

If cost centers are being used, Spectrum verifies that your operator has permission to access the vendor before including void items on the Void Check Register. Spectrum compares the vendor's list of shared cost centers with cost centers in your assigned cost center scheme, and if there are no common cost centers, then void items for that vendor are not selected.
The Void Check Register will include transactions only if you have permission to access the cost center associated with the void check. The cost center used for comparison purposes with your operator's allowed codes is the payment transaction cost center associated with the voided item.
