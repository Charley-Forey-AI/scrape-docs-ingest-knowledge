# Release A/P / Payroll Check Print Lock

Spectrum allows you to print both A/P and Payroll checks using the same check stock (and numbering scheme). Protection is provided by Spectrum to prevent Payroll and Accounts Payable users from accessing the respective 'Print Checks' starting screen when another user is currently printing checks in the other module. This utility clears the system-generated lock in the unusual case where the person printing checks was unexpectedly exited from the software (perhaps due to a power failure).
