# Vendor Invoice Inquiry - VAT Processing Information

When VAT processing is enabled in General Ledger Information, VAT information will display in the header of the screen (including the VAT tax code, VAT tax amount, and Retention portion of the VAT tax) when a VAT tax code is present on the invoice.
