# Secure Electronic File Downloads

Controllers or those with a security level of 8 or higher
 can use this screen to download decrypted files for processing.
If the Use secure download page for
 electronic files option was selected on the Installation screen, ACH and
 ePayments files processed by A/P or Payroll clerks will appear on this screen in encrypted
 format. Once the files have been processed by your financial institution, return to this
 screen and delete the files.
