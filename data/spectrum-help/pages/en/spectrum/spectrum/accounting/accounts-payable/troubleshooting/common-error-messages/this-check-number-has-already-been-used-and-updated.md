# This Check Number Has Already Been Used and Updated

This error message displays when you click the
 Preview button in the A/P Check Print screen
 and the check run includes a check number that has been used elsewhere in the
 software.
To clear this error, either re-print the check batch or
 complete the following steps to reprint one check at a time.

1. In the error message window, click
 OK to return to the A/P Check Print
 screen.

1. In the From vendor and
 To vendor fields, select the vendor whose check must be
 assigned with a new number.

1. In the Check number field,
 enter a new check number or press Enter to accept the software
 default of the next check number.

1. Print the new check.
