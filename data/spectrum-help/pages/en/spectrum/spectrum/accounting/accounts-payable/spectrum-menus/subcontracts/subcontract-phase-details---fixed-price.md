# Subcontract Phase Details - Fixed Price

This screen displays if the Unit price checkbox is clear on the [Subcontracts](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/subcontracts) screen. If this checkbox is selected, the [Subcontract Phase Details - Unit Price](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/subcontracts/subcontract-phase-details---unit-price) display instead.
