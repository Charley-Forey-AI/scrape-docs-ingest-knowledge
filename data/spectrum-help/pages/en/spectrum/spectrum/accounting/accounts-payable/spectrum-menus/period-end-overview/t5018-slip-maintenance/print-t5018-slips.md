# Print T5018 Slips

This report allows users to print T5018 Slips for annual filing. The report will print 'plain paper' forms for issuance to vendors.
If doing year-end processing, see [T5018 Slip Maintenance](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/period-end-overview/t5018-slip-maintenance).
This form is not necessarily sent at the calendar year's end, but instead is issued annually, as of the 'report ending date' for the company filing the report.Note: This form does not need pre-authorization.

If doing year-end processing, see [Export T5018 File](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/period-end-overview/t5018-slip-maintenance/export-t5018-file).
