# Subcontract Phase Details - Fixed Price - Field Descriptions

Use this table for reference when completing the fields on this screen.
FieldsDescriptions
Subcontract totals
OriginalThe original contract amount displays.
ChangesAny executed change order(s) amount display.Note: The phase cannot be deleted if the change order amount is non-zero.

RevisedThe revised subcontract amount represents the sum of the original contract amount and any executed change orders. No changes are allowed.
Detail lines (not sortable)
PhaseEnter the phase to which this portion of the subcontract is to be charged. If a default phase was entered on the main screen, that phase will default here.
If this is a sub-job transaction set up on a master job, double-click on this field to search phases for the sub-job. This will allow you to easily select a phase set up on a master job, but not present on the sub-job. Spectrum will add a new phase to the sub job if the current job is a sub-job of a valid master job, the phase lengths of both jobs match, and the Phase + Cost type combination for the current job is valid and not 'Complete'.
Note: The phase cannot be deleted if the amount in the Change Orders field is non-zero. In addition, a phase with a Complete status may not be selected in this field.

CtEnter the cost type to which this portion of the subcontract is to be charged. If a default cost type was entered on the main screen, that cost type defaults here.
After entry of a phase and cost type, the billing description displays but may be overwritten..

Billing itemWhen adding a records, enter the billing item for this portion of the subcontract work to be performed. After the line has been added, the revenue bid item number displays and no changes are allowed. The system allows you to enter multiple lines for each phase, using different billing item numbers.
Ret?Select this checkbox if this line is subject to retainage.
Original contractEnter the original subcontract amount for this portion of the subcontract work to be performed.
Amount billed
Change ordersThe change order amount(s) display if the subcontract amount has been updated in [Subcontract Change Order Log](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/subcontracts/subcontract-financial-summary/subcontract-change-order-log).
Revised contractThe revised subcontract amount represents the sum of the original contract amount and any executed change orders.
Phase and Cost type descriptionThe phase and cost type descriptions display.
