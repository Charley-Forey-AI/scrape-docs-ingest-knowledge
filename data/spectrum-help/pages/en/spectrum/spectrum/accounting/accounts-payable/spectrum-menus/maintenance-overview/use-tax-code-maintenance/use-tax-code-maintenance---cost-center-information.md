# Use Tax Code Maintenance - Cost Center Information

When adding or changing a use tax code, Spectrum allows entry of a G/L account code only if your operator has permission to assign that code. Spectrum compares the G/L account's list of shared cost centers with cost centers in your assigned cost center scheme, and if there are no common cost centers, then that G/L account cannot be assigned.
Once a G/L account code has been assigned to a use tax code (by an authorized operator), this G/L account will be used by all operators for processing involving the use tax code.
