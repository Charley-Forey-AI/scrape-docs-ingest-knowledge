# Electronic Payment Remittance Advice - Field Descriptions

Use this table for reference when completing the fields in the Electronic Payment Remittance Advice screen.
Fields
Descriptions

Subject
Accept or edit the default text; max 215 characters.

Include company name in subject line?
If you want the recipient to sees your company name in the email subject line, select this checkbox.

Message text
Select or edit the default text; max 250 characters.

Blind copy
Enter email addresses to blind carbon copy additional recipients. Note: Separate email addresses with a semicolon.

E-Mail Exceptions
Note: This button displays only when a vendor does not have an email address listed in the Vendor Main Properties screen.

 See [Resolve Email Exceptions](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/data-entry-overview/ap-payment-processing/about-electronic-payment-remittance-advices/resolve-email-exceptions) for steps to resolve exceptions.

Preview distribution report for vendors with no email?
Select this checkbox to review the distribution report when an email address is not present for a vendor.
