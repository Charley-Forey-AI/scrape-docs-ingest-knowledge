# Vendor Labels - Field Descriptions

Use this table for reference when completing the fields on this screen.
FieldsDescriptions
Vendor type and Postal codeEnter the vendor type and zip code to be included on this report, or press Enter to accept the default of ALL.
Include only 1099-applicable vendors?
-OR-
Include only T5018-applicable vendors?
When the Payroll reporting country is set to Canada in Payroll Installation, this screen prompts a selection for Include only T5018-applicable vendors?
Select this checkbox to print a set of labels specifically to mail 1099s. Only vendors with the 1099 flag set selected in the Vendors screen will be selected.
If you select this checkbox, the following three additional prompts display:

- Year This prompt only displays if labels are being printed to include only 1099-applicable vendors. Enter the year for which 1099s are to print.

- Minimum $ This prompt only displays if labels are being printed for 1099s. Enter the minimum amount that must have been paid to the vendor for the vendor to receive a 1099. For example, if $600 is the minimum for 1099s and only $350 was paid to a vendor for the entire year, and that vendor should consequently not receive a 1099, enter "600" in this field.

- Tax ID status This prompt only displays if labels are being printed to include only 1099-applicable vendors. Enter All to print ALL vendors with the 1099 checkbox selected in [About Vendors](/en/spectrum/spectrum/accounting/accounts-payable/spectrum-menus/about-vendors). Otherwise, enter No to print labels for only those vendors with the checkbox selected and who also have no Federal ID or Social Security information entered in the Vendors.
