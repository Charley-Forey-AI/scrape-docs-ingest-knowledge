# Manual Check Entry - VAT Processing Information

If VAT processing is enabled in the General Ledger Installation screen, the following columns include VAT amounts:

- Outstanding includes unpaid VAT amounts

-  On Current rows, only the current VAT is included

- On Retention rows, only the retention VAT is included

- Previously paid includes VAT amounts paid

- On Current rows, only the current VAT is included

- On Retention rows, only the retention VAT is included

- Original amount includes total VAT
