# Reverse Vendor Invoice - VAT Processing Information

If VAT processing is enabled in General Ledger Installation, the Reverse Vendor Invoice update will include VAT tax information (VAT tax code, tax percent, total amount, retention, and override info) when creating and posting a credit memo (reverse invoice) for an invoice that had a VAT tax code assigned to it.
