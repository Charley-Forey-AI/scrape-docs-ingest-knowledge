# Images Vendor

Use this screen to search for and review vendor image files. When you first arrive on this screen you can click the Search button to open the Search Images window and enter search criteria, or click the New button to add an image file (the Cabinet and Folder fields will default when adding a new image).
A list of image files displays on the Vendor Images screen. After you select an image file from the list box, the image displays in the Slider Bar.
Note: This screen is only available if your company is licensed for the Document Imaging module.
