# Find Vendor - Field Descriptions

Use this table for reference when completing the fields on this screen.
Fields
Descriptions

Vendor
Type a vendor code in order to limit the search results on the Vendors screen.

Vendor name
Type a vendor name in order to limit the search results on the Vendors screen.

Alpha reference
Type an alpha reference (the first six letters of the vendor's full name) in order to limit the search results on the Vendors screen.

Cost center
This field only displays if cost centers/cost groups are enabled for the current company in the Enterprise Installation screen.
Specify a cost center. Only vendors that are assigned to this cost center will display on the Vendors screen. Alternatively, you can double-click in this field to select from a list of available codes.

Vendor status
Select the checkbox (or checkboxes) for the vendor statuses that you want to include in the search results on the Vendors screen. For example, you might select all three checkboxes to include ALL vendors, or you might select to include only Active vendors.
