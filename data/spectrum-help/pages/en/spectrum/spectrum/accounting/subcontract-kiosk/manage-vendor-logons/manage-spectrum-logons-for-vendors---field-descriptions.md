# Manage Spectrum Logons for Vendors - Field Descriptions

A reference for completing the fields in the Manage Spectrum Logons for Vendors screen.
Fields/ButtonsDescriptions
NewSelect to create a new record, that is, to grant a vendor access to Spectrum.
Change StatusChoose a record in the grid and select this button to change the vendor's status. Setting the status to Inactive revokes/prevents access by the vendor.
Operator MaintenanceAppears above the grid if you have security access to the Operator Maintenance screen. Choose a contact and select this button to open the screen.
Vendor code
Vendor name
The code and name of a vendor who's been granted access to your organization's instance of Spectrum. This vendor may or may not still have access. Select Change Status to view the status.
Contact nameThe name of the vendor's contact which has access to your organization's instance of Spectrum.
Logon IDThe username previously validated and assigned to the contact.
