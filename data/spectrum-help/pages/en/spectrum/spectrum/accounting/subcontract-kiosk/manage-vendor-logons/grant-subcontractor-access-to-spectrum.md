# Grant Subcontractor Access to Spectrum

Create new Spectrum user records for external users without accessing the standard Operator Security screen.
To add a new record:

- You must have the Subcontract Kiosk module installed, and its checkbox must be selected in the Company Installation screen.

- A vendor record and a contact record must exist in Accounts Payable > Vendors > Contacts.

In order for a vendor to access the Subcontract Kiosk (SK module), they must have a Spectrum logon.
To grant a vendor access to your organization's instance of Spectrum:

1. Go to System Admin > Security > Manage Vendor Logins.The Manage Spectrum Logons for Vendors screen opens.

1. Select the New button.The New Vendor Logon window opens.

1. Complete the fields accordingly. If needed, see [New Vendor Logon - Field Descriptions](/en/spectrum/spectrum/accounting/subcontract-kiosk/manage-vendor-logons/new-vendor-logon---field-descriptions).

1. Select Continue.

1. Select Change Status to confirm the vendor's status is set to Active.

The Spectrum application:

- creates the new operator and grants access to the Subcontract Kiosk

- generates an email message addressed to the vendor, containing the new Logon ID and password, if you selected the pertinent checkbox.
