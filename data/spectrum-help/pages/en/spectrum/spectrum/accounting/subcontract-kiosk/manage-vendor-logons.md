# Manage Vendor Logons

The Manage Spectrum Logons for Vendors screen displays the vendors who are or have been set up to access the Subcontract Kiosk.
Use the Manage Spectrum Logons for Vendors screen for these tasks:

- View the list of vendors which have current or used to have access to Spectrum

- [Grant Subcontractor Access to Spectrum](/en/spectrum/spectrum/accounting/subcontract-kiosk/manage-vendor-logons/grant-subcontractor-access-to-spectrum)

- Create a new password for a vendor or revoke their access. See [Manage Spectrum Logons for Vendors - Field Descriptions](/en/spectrum/spectrum/accounting/subcontract-kiosk/manage-vendor-logons/manage-spectrum-logons-for-vendors---field-descriptions).
