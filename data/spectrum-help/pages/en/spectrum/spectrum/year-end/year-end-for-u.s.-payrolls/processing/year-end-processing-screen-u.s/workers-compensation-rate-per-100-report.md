# Worker's Compensation Rate Per $100 Report

The Worker's Compensation Rate Per $100 Report lists the
 employer's contribution to worker's compensation for one or all worker's compensation codes
 for a specified date range.
This report may be printed in sequence either by state or by worker's compensation code.
 The worker's compensation rate shown on each line represents the last rate used for that
 employee.
When component rates are recorded in the Worker's Compensation Code Maintenance screen, this component breakdown appears on this report.
When applicable, the Covered earnings column shows the Subject to earnings. This means the state specified on the time card line is set up as 'taxable' in the Tax Effects window of the Deduction / Add-on Code Maintenance screen.
