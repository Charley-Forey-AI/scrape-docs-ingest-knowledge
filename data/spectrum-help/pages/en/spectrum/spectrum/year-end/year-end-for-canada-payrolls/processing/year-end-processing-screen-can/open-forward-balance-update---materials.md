# Open Forward Balance Update - Materials

This screen is used to roll the totals for the plant production file from one year to the next.
Enter the two digits from the earliest fiscal year from which to transfer the opening balance (for example, enter 26 for the year 2026).
Select Continue, and a confirmation box indicates that the update has been completed.

If doing year-end processing, proceed as directed from the [Year End Processing Screen (U.S.)](/en/spectrum/spectrum/year-end/year-end-for-u.s.-payrolls/processing/year-end-processing-screen-u.s.) or from the [Year End Processing Screen (CAN)](/en/spectrum/spectrum/year-end/year-end-for-canada-payrolls/processing/year-end-processing-screen-can).
