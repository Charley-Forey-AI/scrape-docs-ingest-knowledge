# Year-End for U.S. Payrolls

Preparing for and completing year-end processing of U.S. payrolls also prepares you for the next year's payroll processing.

## Prior to year-end processing

We recommend that you familiarize yourself with the entire year-end process prior to beginning your year-end processing.

- View the Release Notes. These will be published soon. Once they are published, a link will be provided here.

- Register for or renew your Social Security Administration (SSA) PIN and Password:

- Call toll free at (800) 772-6270.

- Go to the SSA website at [http://www.ssa.gov/bso/bsowelcome.htm](http://www.ssa.gov/bso/bsowelcome.htm).

Tip:Viewpoint provides a data file that automatically updates the Tax Tables for Single and Married filing statuses in Spectrum.

- You may download this file from the [Viewpoint Customer Portal](https://support.viewpoint.com/s/spectrum).

- The file includes all known changes to electronic tax tables for Federal and all 50 states.

- We update this file throughout the year to reflect changes issued by each entity.
