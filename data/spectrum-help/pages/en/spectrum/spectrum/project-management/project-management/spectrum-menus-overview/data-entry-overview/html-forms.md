# HTML Forms

Spectrum Project Management HTML Forms is a way for
 customers to utilize HTML for data entry in lieu of the standard Project Log Entry
 screen.
 Each Project Management[Category Maintenance](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/category-maintenance) can use a
 different HTML entry form, and everything from the layout, styling, and fields can be
 customized. Spectrum includes a set of HTML forms for use with the standard categories,
 though these can also be customized in a number of ways.
The following HTML Forms are currently available in Project Management:

- [Correspondence Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/correspondence-form)

- [Daily Log Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/daily-log-form)

- [Issue Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/issue-form)

- [Meeting Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/meeting-form)

- [Meeting Item Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/meeting-item-form)

- [Memo Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/memo-form)

- [RFI Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/rfi-form)

- [RFP Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/rfp-form)

## Custom Form Templates

Each Project Management category is linked to a pair of template files (.html) on the server: one for Editing and one for Printing. Customers can create their own templates to use with their custom categories. Templates for standard categories are included in Spectrum, though a customer can also modify these templates or replace them with custom versions.

## Assigning and Uploading Custom Forms

Form templates are stored on the Spectrum server similar to Crystal Reports: in a directory called 'Forms' under the Spectrum folder. Templates should be uploaded through Project Management's [Category Maintenance](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/category-maintenance) screen. Open a Category for Edit and select the HTML Form entry method option.
In the [Search HTML Forms](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/search-html-forms) window you can either add a new form by clicking the New button or edit the settings for an existing form by selecting on a form and clicking Edit.
The New/Edit window allows you to upload a form from your workstation. Uploaded forms are stored in the Spectrum/Forms directory on the Spectrum server.
