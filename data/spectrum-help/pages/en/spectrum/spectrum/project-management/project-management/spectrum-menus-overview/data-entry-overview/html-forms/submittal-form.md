# Submittal Form

Submittals are the confirmation of the contractor's intent
 to comply with the design concept.
Submittals are typically shop drawings, material data, samples, and product data. The
 general contractor will use the submittal form to review and approve all submittals prior
 to distributing them to the architect.
Any related documents will be attached to this form, and when previewing the submittal form you can select which of these documents to include.
Depending on the settings in Category Maintenance, this form will also contain buttons to create new submittals, revisions and attach the submittal to a transmittal form:

- Click the Save and
 Create Next button to save the current submittal and create a new one
 with the same data. The new submittal number will increment to the next number.

- Click the Create
 Revision button to close the current submittal and create a new one with
 all the data and related documents from the previous submittal for revision. The revised
 submittal number will increment to the next decimal (that is, 1.1). Use this option if the
 original submittal was rejected and a revised submittal is required.

- Click the Create
 Transmittal button to attach a Transmittal Form to the submittal. All
 related documents linked to the attached form will be included with the current
 Submittal Form.
