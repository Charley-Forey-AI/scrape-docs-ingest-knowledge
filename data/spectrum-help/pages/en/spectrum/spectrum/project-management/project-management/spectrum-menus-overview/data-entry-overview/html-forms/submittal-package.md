# Submittal Package

A submittal package is a collection of submittals that will
 be sent to the general contractor for approval.
Although a package could have multiple submittals, the reviewer approves
 or rejects them individually.
Watch the following video to learn more about entering submittals and
 creating a submittal package to send to reviewers for approval.
