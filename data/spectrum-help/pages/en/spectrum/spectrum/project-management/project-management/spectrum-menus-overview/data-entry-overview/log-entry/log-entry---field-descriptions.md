# Log Entry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job
When first viewing this screen, you
 will be prompted to select a Job code and category. The data of the selected
 job displays in the [Job
 Info Bar](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/info-bar).
Category
The category code also controls whether
 certain elements on the Overview and Details tab display on the screen,
 allowing customization by type of log entry (such as RFI vs. Daily Log).The following category selections will automatically
 generate HTML Forms:

- [Correspondence Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/correspondence-form)

- [Daily Log Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/daily-log-form)

- [Issue Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/issue-form)

- [Meeting Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/meeting-form)

- [Meeting Item Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/meeting-item-form)

- [Memo Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/memo-form)

- [RFI Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/rfi-form)

- [RFP Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/rfp-form)

- [Submittal Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/submittal-form)

- [Submittal Package](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/submittal-package)

- [Transmittal Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/transmittal-form)Note: Only active categories in the
 current company will display in the Search Categories
 window.

Reference number
Click this field to display the
 Search Log Entries for Category window, where you can
 select an existing log entry for a designated category. Leave the reference field blank to automatically add a new reference number
 and record.

Topic
This field is optional but recommended.
 You can enter up to 250 characters.
Ball in court
Once you enter a contact name, Spectrum
 will search for a match in the system. If the match is partial or there is no
 match, the Search Contacts window displays. Click New to add a [new contact](/en/spectrum/spectrum/getting-started/contacts/contacts-administration/newedit-contact).Click the Contact icon to
 associate the contact with this job.

Log status
Enter a status for the project.
Issued date
Enter the date when the item was
 issued. In the New/Edit Category screen, you can set the default to today's
 date.
Respond by
Enter the date a response is needed by.
 In the Dashboard App, My Project Log Status, the contact
 assigned to the Ball in court
 field will see this as the due date.If
 'Defaults days to respond' was specified on the Project Log Defaults screen,
 this field will use the specified amount and calculate the date a response
 is needed by.

Answered
Enter the date when the item was
 answered.
Cost impact?
Select these checkboxes if the item
 has a cost or schedule impact.
Amount
Enter the amount that the project's
 cost will likely be affected.For Change Requests, the
 change request amount displays. This is the same as the Amount column in
 Change Request Entry - 5. Billing.

Schedule impact?
Change request data displays in this
 column. Approved days or Requested days display.
Days
Enter the number of days this project
 will require.
Closed?
Select to close the status.
Activity flag

- Action
 needed items require the project manager or another person
 to perform a task and then respond or follow up.

- Information only items are sent to keep the project
 manager informed, but don't require the project manager to take action.

- Tracking items are correspondence of a repetitive nature,
 for example, plan revisions or daily logs.

Priority flag
Select the item's priority level.

- High

- Medium

- Low

Unlimited Notepad
The note topics have been entered in
 [Project Note Topics Maintenance](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/project-note-topics-maintenance). A green dot indicates that topics have notes.Note: For each new note topic to display in this
 screen, the administrator must select Show? in the Edit Category
 window.
