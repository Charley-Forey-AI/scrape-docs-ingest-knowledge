# Building / Area File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Code Description
Add or edit the code (for example,
 "B5") and a brief description (for example, "Locker facility") for the
 building.
New/Edit/Delete
Click to add new building areas, make
 changes to existing areas, or delete building areas and descriptions.
Area Description
Add or edit the code (for example,
 "001") and a brief description (for example, "Second floor" or "North wing")
 for the building area.
