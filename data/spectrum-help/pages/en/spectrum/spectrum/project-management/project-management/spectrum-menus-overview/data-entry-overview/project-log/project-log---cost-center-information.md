# Project Log - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, when entering Correspondence Log entries for a job Spectrum will allow
 entry for the job only if the operator has permission to access the job.
As you enter the job number, the system compares the cost center assigned to the job
 with cost centers in your assigned scheme, and if the cost center is not included, then
 entries for that job will not be allowed. When an operator has security authorization
 for a job, then all project information pertaining to that job is shown. The cost center
 scheme 'Job' override will be used to determine the list of valid jobs.
