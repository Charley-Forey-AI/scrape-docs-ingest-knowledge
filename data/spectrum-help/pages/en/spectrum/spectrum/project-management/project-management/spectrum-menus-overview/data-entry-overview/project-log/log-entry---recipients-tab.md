# Log Entry - Recipients Tab

Recipients that were added in the Project Log Defaults
 screen will default on this tab.
Use the Recipients tab to maintain an unlimited number of [Contacts](/en/spectrum/spectrum/getting-started/contacts) associated
 with the current log report.
After selecting a contact name, only the Type column can be modified.
