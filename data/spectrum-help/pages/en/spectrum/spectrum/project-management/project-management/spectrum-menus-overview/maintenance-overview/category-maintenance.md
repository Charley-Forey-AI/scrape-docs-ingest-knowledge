# Category Maintenance

Use the Category Maintenance screen
 to add and modify categories that will be used throughout the Project Management
 module.
Settings in this screen affect how the [Log Entry](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/log-entry) screen works across all
 Spectrum companies so that you won't have to enter the same information in each
 company.
Categories are broad in scope and can be anything that helps you
 organize your correspondences, for example, the name of who sent the document or which part
 of the job the document is about.
Pre-defined categories
There are several pre-defined categories. Pre-defined categories
 include: Correspondence, Daily Log, Drawing, Issue, Meeting, Memo, RFI, Sketch, Submittal,
 Submittal Transmittal, and Transmittal., which cannot be edited or deleted. You can,
 however, set the status to Inactive on a company-by-company basis.
Note: User-defined note topics do not appear in pre-defined
 categories.
User-defined categories
You can add an unlimited number of additional categories--settings for these categories are also shared among all Spectrum companies (other than the company-specific status setting).
You can add a new category to replace a pre-defined one (such as an RFI) in order to record customized Show/Hide settings and defaults.
