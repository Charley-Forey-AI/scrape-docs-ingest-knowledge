# Log Entry

Use the Log Entry screen to record and view individual
 entries.
Entries display in the [Project Log](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/project-log). When first viewing this screen, you will be prompted to select
 a Job code and then the Search Categories window displays an you are prompted to select a
 category. Most category options will open one of the [HTML Forms](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms), but a few logs may be
 entered directly on this screen.
