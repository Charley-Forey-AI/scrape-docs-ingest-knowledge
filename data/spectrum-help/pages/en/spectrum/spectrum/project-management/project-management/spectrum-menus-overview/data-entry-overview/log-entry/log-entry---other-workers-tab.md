# Log Entry - Other Workers Tab

Use this tab to record the number of workers for a vendor.
 Enter the vendor code in the Vendor column, and the number of workers associated with that
 vendor in the Worker count column.
The Total workforce field
 includes the number of employees entered in the Employees tab.
