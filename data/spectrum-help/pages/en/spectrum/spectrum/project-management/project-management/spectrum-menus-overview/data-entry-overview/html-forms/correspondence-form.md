# Correspondence Form

Use this form to send project correspondence to project members. Any related documents will be attached to this form, and when previewing the correspondence form you can select which of these documents to include.
Field
Description

Correspondence number
The correspondence number will be
 auto-assigned when setting up a new correspondence, or you can enter a specific
 number to open an existing correspondence.
Date
The system date will automatically be
 entered when setting up a new correspondence, but can be changed if
 desired.
Project
The job entered on Log Entry screen
 will display here.
To
Enter a job contact(s) for the
 correspondence.
From
Enter the job contact the
 correspondence is from.
Topic
Enter a topic for the correspondence.
 If the correspondence originated from an RFI, this field will default from the
 RFI.
Message
Enter a message for the
 correspondence.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
