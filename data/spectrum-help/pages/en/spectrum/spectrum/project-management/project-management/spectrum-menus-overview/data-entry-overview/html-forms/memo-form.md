# Memo Form

Use this form to document statements of fact, decisions
 made, and agreed upon processes from the project.
Any related documents will be attached to this form, and when previewing the memo form you
 can select which of these documents to include.
Depending on the settings in Category Maintenance, this form will also contain a button to create a Transmittal Form to attach to the current Memo Form. All related documents linked to the attached form will be included with the current Memo Form.
Field
Description

Memo number
The memo number will be auto-assigned
 when adding a new memo, or you can enter a specific number to open an existing
 memo.
Date
The system date will automatically be
 entered when adding a new memo, but can be changed if desired.
Project
The job entered on Log Entry screen
 will display here.
To
Select the job contact(s) to send the
 memo to.
From
The selected job company will display
 in this field.
Topic
Enter a topic for this memo in this
 field and enter the memo below.
By
Select the job contact who entered the
 memo.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
