# Transmittal Form

Use this form to create a cover letter that accompanies the
 exchange of both physical and electronic materials with other project team members.
Transmittals contain information that describes the contents of the documents or physical
 items being sent. Any related documents will be attached to this form, and when previewing
 the transmittal form you can select which of these documents to include.
Field
Description

Transmittal number
The transmittal number will be
 auto-assigned when setting up a new transmittal, or you can enter a specific
 number to open an existing transmittal.
Date
The system date will automatically be
 entered when setting up a new transmittal, but can be changed if
 desired.
Project
The job entered on Log Entry screen
 will display here.
To
Enter a job contact(s) for the
 transmittal.
From
Enter the job contact the transmittal
 is from.
We are sending
Enter a description for the transmittal
 form.
For the purpose of
Select what purpose the transmittal
 document(s) are for.
Sent via
Specify the method of delivery for the
 transmittal: email, fax, US mail, and more.
Reference
Select whether the transmittal form
 references an attached document or under separate cover.
Copies
Enter the number of copies, date and
 description of each item attached to this transmittal.
Remarks
Enter any specific remarks for the
 transmittal.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
