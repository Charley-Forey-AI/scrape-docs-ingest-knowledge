# Daily Log Form

Use this form to record and track activities and conditions on the job site.
This form contains sections to track weather conditions, contractor and subcontractor work, schedule, and safety issues. Any related documents can be attached to this form, and when previewing the daily log form you can select which of these documents to include.
Depending on the settings in Category Maintenance, this form may also contain buttons to create new change requests, correspondences, and attach the daily log to a transmittal form.
FieldDescription
Report numberThe report number is auto-assigned when setting up a new log entry, or you can enter a specific number to open an existing report.
DateThe system date populates automatically when you create a new log, but you can change it if you want.
ProjectThe job entered on Log Entry screen displays here.
TopicEnter a topic for the log here. If the correspondence originated from an RFI, this field default from there.
Reported byEnter the job contact who is reporting the information.
Observed WeatherEnter any observed weather conditions for the project site.
Prime Contractor WorkEnter any prime contractor hours for carpenters, electricians, iron workers, laborers, operators, and other. You can enter any notes for contractor work below.
Subcontractor WorkEnter any hours for subcontractors assigned to this project. You can enter any notes for subcontractor work below.
ScheduleEnter any schedule notes in this section.
SafetyEnter any safety notes in this section.
Related DocumentsAny related documents attached to this form, with notations for inbound or outbound messages (Direction) and Document Imaging attachments. Click the Add Item button to add new related documents to this form.
