# Meeting Form

Use this form to log subject matter discussed during
 project meetings.
This form offers entry fields for Topic, Meeting date, Location, Start time and Status. You
 can also build a list of participants (attendees and Cc contacts), and track attendance at
 the meeting by person. This form also provides agenda item tracking where each item has its
 own Topic, Description, due date, Ball in court assignment, and Priority.
Any related documents will be attached to this form, and when previewing the meeting form you can select which of these documents to include.
Note: When adding a new meeting, the meeting number will be
 sequentially assigned the next available number.Field
Description

Meeting number
The meeting number will be
 auto-assigned when setting up a new meeting, or you can enter a specific
 number to open an existing meeting.
Meeting date
Enter a meeting date.
Project
The job entered on Log Entry screen
 will display here.
Topic
Enter a topic for the
 meeting.
Location
Enter a location for the
 meeting.
Organizer
Enter the meeting's
 organizer.
Status
Select a status for the meeting
 request.
Start time
Select a start time for the
 meeting.
Participants
Select which job participants to
 invite to this meeting. Select the checkbox next to the participant name if
 they attend the meeting.
Overview
Enter an overview for this
 meeting.
Agenda Items
Add agenda items for the meeting in
 this section. You can specify a Due date, designated ball in court
 assignment, priority, status and then enter a topic and description for the
 agenda item. Click the Restore
 Open Items button to add any previously deleted open agenda
 items.
Related Documents
Any related documents will be
 attached to this form, with notations for inbound or outbound messages
 (Direction) and Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
