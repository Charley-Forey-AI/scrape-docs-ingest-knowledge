# Print Form

You can create an array of customized Crystal Reports (for
 example: separate reports for RFIs and Daily Logs).
The customizable Form report is based on information entered in [Category Maintenance](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/category-maintenance).
Access the report from [Project Log](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/project-log) or [Log Entry](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/log-entry), and select the default report in the New/Edit Category screen.
