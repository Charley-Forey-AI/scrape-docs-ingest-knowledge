# Submittal Form - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Submittal number
The Submittal number will be
 auto-assigned when setting up a new submittal, or you can enter a specific
 number to open an existing submittal.
Date
The system date will default in this
 field, but can be overwritten if necessary.
Project
The job entered on Log Entry screen
 will display here.
Spec section
Select a spec section that applies to
 the submittal item. These are set up in the Job Spec Section maintenance
 screen, and based on CIS-standard specifications.
Subsection
Enter the paragraph sub-item within the
 spec section to review.
Status
Select a status for this submittal:
 Open or Closed.
Current action
Select the current action for this
 submittal.

- If Not
 yet submitted is selected, the Submitter designated below
 will be given the Ball in court assignment. If there are multiple
 Submitters specified, the Ball in court assignment will be left
 'Unassigned'.

- If Submitted is selected, the operator will be given the
 Ball in court assignment if they are a job contact.

- If In
 review is selected, the Reviewer designated below will be
 given the Ball in court assignment. If there are multiple Reviewers
 specified, the Ball in court assignment will be left 'Unassigned'.

- If Accepted is selected, the operator will be given the Ball
 in court assignment if they are a job contact.

- If Rejected is selected, the operator will be given the Ball
 in court assignment if they are a job contact.

Ball in court
This field will default based on the
 Current action selected above, otherwise select a job contact who is currently
 responsible for the specified submittal.
Topic
Enter a topic for the
 submittal.
Submitter Reviewer
Enter Submitter(s), Reviewer(s), and
 any additional parties for this submittal.
Date submitted
Enter the date of the submittal and a
 due date.
Released for review
Enter the date the submittal was
 released for review and a due date.
Date returned
Enter the date the submittal was
 returned.
Date closed
Enter the date the submittal was
 closed.
Required on site date
Enter the required project on site
 date.
Notes
Enter any notes for the
 submittal.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
