# Selections - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Search

Key words
Project Log Entries Search for key
 words such as topic text, reference numbers, log classification, spec section
 codes, phase codes and descriptions, reference, responsibility, weather,
 external IDs, and notes.  Change Requests Search for key
 words such as change request numbers, descriptions, and notes.

Name
Project Log Entries Search for
 Ball-in-court contact name, From and To recipient (including names, email
 addresses, and organizations).  Change Requests Search
 for customer codes and names.

Dates

From/To created
Use the Search Calendar
 window and [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)
 icon to select dates. The To fields default to one year after the current job cost
 processing date.The selection criteria first restricts
 the records to display in the grid and then when presented as 'grouped' will
 display the new oldest date on that header column.

 Help link from Accounts Receivable.

 of Change Request Entry equaling non-zero.Select
 No to display
 records with no cost impact.

Schedule impact

 with a schedule impact and change request records with Requested days or Approved days on the Change
 Request Entry - Dates tab equaling non-zero.Select
 No to display
 records without an impact on the schedule.
