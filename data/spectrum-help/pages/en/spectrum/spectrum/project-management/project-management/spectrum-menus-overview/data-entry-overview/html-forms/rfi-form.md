# RFI Form

Use this form to request information or clarification on
 the project from contractors.
Responses from contractors can be added directly to this form. Any related documents will
 be attached to this form, and when previewing the RFI form you can select which of these
 documents to include.
Depending on the settings in Category Maintenance, this form will also contain buttons to create an RFP, Change Request, and Transmittal Form to attach to the current RFI Form. All related documents linked to the attached forms will be included with the current RFI Form.
Watch the following video to learn more about setting up RFIs.
Field
Description

RFI number
The RFI number will be auto-assigned
 when setting up a new RFI, or you can enter a specific number to open an
 existing RFI.
Issued date
The system date will automatically be
 entered when setting up a new RFI, but can be changed if desired.
Project
The job entered on Log Entry screen
 will display here.
To
Enter a job contact and anyone else you
 wish to copy on the request.
Ball in court
Select a job contact who is currently
 responsible for the specified RFI.
Current action
Select the current action for this
 RFI.
Return requested date
Enter a requested date for a response,
 or leave this field blank.
Return required date
Enter the required date for a
 response.
Topic
Enter a Topic for the RFI.
Spec section
Select a spec section for the RFI.
 These are set up in the Job Spec Section maintenance screen, and based on
 CIS-standard specifications.
Paragraph
Enter the paragraph sub-item within the
 spec section to review.
Cost impact?
If this change will impact the cost of
 the project, select this checkbox and enter the cost impact amount in the
 field below.
Schedule impact
If this change will impact the project
 schedule, select this checkbox and note the number of days the change will add
 to the project in the field below.
External number
If there is a secondary RFI number from
 the subcontractor, enter it here.
Drawing impact
If this change will impact any project
 drawings, select this checkbox and add a drawing reference number and drawing
 detail number in the fields below.
Discipline
Enter the specialization group number
 for this RFI in this field.
Priority
Select a priority for the RFI.
Status
Select whether the RFI is Open or
 Closed.
Request
Enter the detailed request for
 information in this section, and select the project team member who requested
 this information.
Response
Responses from contractors are added in
 this section. Enter the name of the respondent and response date below.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
