# Issue Form

Use this form to enter any project issues and note any cost
 and schedule impacts.
Any Related Documents will be attached to this form, and when previewing the issue form you
 can select which of these documents to include.
Depending on the settings in Category Maintenance, this form will also contain buttons to create an RFI, Change Request, and Transmittal Form to attach to the current Issue Form.
Field
Description

Issue number
The issue number will be auto-assigned
 when setting up a new issue, or you can enter a specific number to open an
 existing issue.
Date
The system date will automatically be
 entered when setting up a new issue, but can be changed if desired.
Project
The job entered on Log Entry screen
 will display here.
Topic
Enter a topic for the issue. If the
 issue originated from an RFI, this field will default from the RFI.
Responsibility
Specify the person, or position within
 the project team, responsible for this issue.
Cost impact
If this issue will create a cost impact
 on the project, select this checkbox and enter the cost impact amount
 below.
Schedule impact
If this issue will create a schedule
 impact on the project, select this checkbox and enter the number of days this
 will add to the schedule below.
Status
Select whether this issue is Open or
 Closed.
Priority
Select a priority for this
 issue.
Discipline
Enter the specialization group number
 for this issue in this field.
Reference
Enter a drawing reference number in
 this field, if applicable.
Issue
Describe the issue in this
 section.
Comments
Add any additional comments in this
 section.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
