# Spectrum Menus Overview

The Spectrum Menu online Help is organized so that it
 mirrors Spectrum menu options as they appear on the main Menu System screen - beginning with
 Maintenance and ending with Utilities.
Brief menu descriptions are provided here, followed by links to the menu
 selections.
The Maintenance menu can be used to enter and maintain various
 project-related codes, such as submittal status codes, change request status codes, and
 building codes.
The Data Entry menu can be used to enter a variety of information on
 day-to-day project happenings, as well as issue RFIs or submittals and record daily log
 events. Custom issues and change requests are also issued using this menu.
