# Log Entry - Related Tab

Use this tab to record a list of other Project Log records and change requests related to the current entry.
When a related entry is added to this list, a two-directional link between the two entries and the current entry is automatically added to the related list of the other entry. For example, if an RFI entry is edited to include a particular change request in its list of related entries, then this RFI will also display in the Change Request Entry - Related tab.
