# Project Log Defaults

Use the Project Log Defaults screen to set up a default number of days to respond to RFI's and Submittals, and build a list of default recipients by category.
Field
Description

Job code
The job code entered on the starting
 screen displays in this field.
Category
The selected category and description
 displays in this field.
Default 'days to respond'
Enter the number of days to respond to
 RFI's and Submittals for the selected category.
Type
Select a type option for the
 response:

- To

- From

- Courtesy copy

- Blind copy

Name
Enter a valid contact name for the
 recipient. The selected contact's organization name and e-mail address will
 display to the right of this field.
Role
Enter a role for the recipient. This
 field can be left blank if desired.
