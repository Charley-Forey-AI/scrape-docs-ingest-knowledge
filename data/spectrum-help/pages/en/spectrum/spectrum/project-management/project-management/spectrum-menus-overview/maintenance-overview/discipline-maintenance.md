# Discipline Maintenance

Use this screen to enter (up to 50 characters) or remove
 log classification codes.
This information is set up separately in each company.
Log classifications vary widely. Examples include the status of the item
 (high, medium, low) or the contact assigned to a task.
Typically, a log classification defines the type of information the record is related to. It can be a delay notification, a defective work notice, or who is responsible for completing a task. Basically, the log classification is the reason that item was sent or received.
