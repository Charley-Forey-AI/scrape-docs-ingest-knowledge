# Send the Project Form by Email

You can send email related to the selected
 project.

1. In the [Project Log](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/project-log) screen, after selecting a job, select a row for which you
 want to send email.

1. Click Form. The Print
 Form box displays.

1. Click Preview to compile the report for the project log entry. If you a
 selected a project management entry row (such as a correspondence or an RFI), the [Project Log Form](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/project-log/project-log-form) opens in a
 new tab. If you select a change request, the [Change Request Log Report](/en/spectrum/spectrum/accounting/accounts-receivable/spectrum-menus/reports-overview/change-request-log-report)
 opens in a new tab.Note: The Change Request Log Form can also be
 accessed from AR >  Inquiries >  Change Request Log, by selecting a row and clicking Proposal Form.

1. Click E-Mail. The Send
 Email starting screen displays.Note: If you receive
 an error message when trying to send e-mail, see [Common Error Messages](/en/spectrum/spectrum/project-management/project-management/troubleshoot/common-error-messages).

1. Email addresses default into the fields. A PDF version of the form
 is attached. You can edit the recipients, subject line, list of attachments, and enter
 any message text.

You can store a record of each email sent from the Project Management module, including recipients and message text. Manage and save this optional feature in the [Project Management Installation](/en/spectrum/spectrum/project-management/project-management/project-management-installation) screen.
