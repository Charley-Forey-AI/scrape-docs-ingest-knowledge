# Submittal Package - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Submittal package number
The Submittal package number will be
 auto-assigned when setting up a new package, but can be changed if
 necessary.
Date
The system date will default in this
 field, but can be overwritten if necessary.
Project
The job entered on Log Entry screen
 will display here.
To
Enter the reviewer(s) to send the
 submittal package to.
Spec section
Select a spec section for the submittal
 item. These are set up in the Job Spec Section maintenance screen, and based on
 CIS-standard specifications.
Ball in court
Select a job contact who is currently
 responsible for the specified submittal package.
Status
Select a status for this package: Open
 or Closed.
Topic
Enter a topic for the submittal
 package.
Notes
Enter any notes for the reviewer(s) to
 accompany the submittal package.
Submittal Documents
Click the Add Item button to add
 submittals to this package.
