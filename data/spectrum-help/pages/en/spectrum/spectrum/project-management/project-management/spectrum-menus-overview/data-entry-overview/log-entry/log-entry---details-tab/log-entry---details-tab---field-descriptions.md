# Log Entry - Details Tab - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Discipline
Enter or search for the applicable
 discipline code. Log classifications vary widely;
 possible examples include the status of the correspondence (high, medium,
 low) or the person in the organization that the task is assigned to. A log
 classification typically defines the type of information the record is
 related to. It can be a delay notification, a defective work notice, or the
 person responsible for completing a task. Basically, the log classification
 is the reason that item was sent or received.
You
 can add new log classification codes in [Discipline Maintenance](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/discipline-maintenance).

Spec section
Enter the specification section code
 associated with this job. If a valid spec section is available, a description
 displays.
Subsection
The value displays if all rows contain
 the same value in a group.
Phase
Select the Phase that applies to this
 job.
Building Area
Select the building and area associated
 with this job. Building/area information is maintained in [Building / Area File Maintenance](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/maintenance-overview/building--area-file-maintenance).
Responsibility Reference
Enter the name of the person
 responsible for taking action based on the correspondence. For example, this
 could be the person responsible for completing a task detailed in the
 correspondence. In the Reference field, enter a specific record identifier. Up to 250
 characters are allowed for both fields.
Sent via
Enter the method the item is sent, such
 as Email, hand delivered, or USPS.
Direction
Select Inbound if you are the
 recipient, or Outbound
 if you created the item.
E-mail date
Select the date the email message was
 created.
Weather AM temperature PM temperature Precipitation Precipitation amount
 Wind
Enter a description (up to 250
 characters) and additional details about the weather.
External ID Owner number
Enter the ID of the external software
 you are using.
Log created Logged by
The date, time, operator code, and name
 of the user who created this log displays.
