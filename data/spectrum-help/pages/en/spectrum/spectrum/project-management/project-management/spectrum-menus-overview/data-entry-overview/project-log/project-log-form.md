# Project Log Form

Use this report to communicate ongoing issues regarding the
 project.
You can email RFIs, transmittals, and other standard forms, as well as maintain a record of
 these emailed reports.
[Email Reports](/en/spectrum/spectrum/getting-started/reports--printing-overview/email-reports)
Fields
Descriptions

Email
Click to email the form to selected
 recipients. The Send Email screen will populate with pre-defaulted email
 address and Job # in the Subject line. You can edit the
 recipients, subject line, list of attachments, and enter message text. A PDF
 version of the form will be attached.
Note: If you receive an error message when trying to send
 e-mail, see the topic [Common Error Messages](/en/spectrum/spectrum/project-management/project-management/troubleshoot/common-error-messages).

Download
Click to download a PDF version of the report.

Archive
If you have the Document Imaging module, you can save this report to a
 specified cabinet. Click to display the [Archive Reports Maintenance](/en/spectrum/spectrum/tools/document-imaging/document-imaging-features/archive-reports-maintenance) window.
