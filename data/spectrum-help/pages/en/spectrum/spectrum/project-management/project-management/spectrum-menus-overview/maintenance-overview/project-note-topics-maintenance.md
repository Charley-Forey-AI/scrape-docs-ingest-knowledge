# Project Note Topics Maintenance

Use this screen to add or modify note topics for the
 project.
You can select whether the topic is shared in all companies, or specific
 to the current company. View the [Notes](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/notes) topic for more
 information.
Project Note Topics include the pre-defined topics listed below. These are available in all companies, but can be removed.

- Correspondence

- Daily delays

- Daily subcontracts

- Daily work completed

- Message

- Reply

- Transmittal
