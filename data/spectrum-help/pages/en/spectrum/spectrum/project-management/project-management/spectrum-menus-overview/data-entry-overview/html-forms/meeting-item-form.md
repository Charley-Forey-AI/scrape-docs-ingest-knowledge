# Meeting Item Form

Any agenda items added to the Meeting Form will
 automatically become Meeting Item forms for the selected project.
Use this form to track open issues on the project and manage related project documents. Any
 related documents will be attached to this form, and when previewing the meeting item form
 you can select which of these documents to include.
Field
Description

Item number
The item number will be auto-assigned
 when setting up a new meeting item, or you can enter a specific number to open
 an existing meeting item.
Date
The system date will automatically be
 entered when setting up a new meeting item, but can be changed if
 desired.
Project
The job entered on Log Entry screen
 will display here.
Topic
Enter a topic for the meeting
 item.
Description
Enter a description for the meeting
 item.
Ball in court
Select a job contact who is currently
 responsible for the specified meeting item.
Due date
Enter a due date for this meeting
 item.
Priority
Select a priority for this meeting
 item.
Status
Select whether this meeting item is
 Open or Closed.
Related Documents
Any related documents will be attached
 to this form, with notations for inbound or outbound messages (Direction) and
 Document Imaging attachments. Click the Add Item button to add new
 related documents to this form.
