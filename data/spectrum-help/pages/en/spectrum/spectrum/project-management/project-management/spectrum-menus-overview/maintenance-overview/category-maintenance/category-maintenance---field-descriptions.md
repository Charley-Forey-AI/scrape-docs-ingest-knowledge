# Category Maintenance - Field Descriptions

Use this table as a reference when completing the fields in the New/Edit Category window. The fields that display will vary slightly depending on which Entry method is selected.
Fields
Descriptions

Category
Enter the name of the new category, or
 select one of the pre-defined categories. The category determines the type of
 form produced after completing a [Log Entry](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/log-entry).
Inactive in current company (disallow new entries)?
Select this checkbox to inactivate
 unwanted categories in the current company. This is a company-specific override
 to the system-wide Active setting for companies.
Auto-record email response
Select this checkbox to record
 response notes from correspondence into the RFI or other categories flagged for
 auto response. The response notes are editable in the RFI, just as if the
 operator had copied or typed them into the screen.Note: This option only applies to operators using the e-mail
 subscription feature. When an e-mail is imported using this subscription
 feature, the parent document (that is, the original RFI that was emailed) will
 have the response auto-populated when this option is
 selected.

Meeting header? Meeting item category
Select the Meeting header checkbox to
 add a custom HTML form for the parent meeting form. When this checkbox is
 selected, enter a new Meeting
 item category code. This must not be the current category code,
 or associated with another category code. This option will only be available
 when the Entry type = HTML form.
Entry Method
Select an entry method for new
 categories:

- Log Entry screen: Entry in the grid below.

- Use HTML form: Select a HTML form for entry, or
 create a custom form. [HTML Forms](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms)

HTML form code + Edit button
This field and button combination only display when the HTML form entry method
 is selected.
Use the Edit button
 to open the [Search HTML Forms](/en/spectrum/spectrum/project-management/project-management/spectrum-menus-overview/data-entry-overview/html-forms/search-html-forms)
 window select an HTML file (NOT printable Crystal form).
In the Search HTML Forms window, when using the Download Edit or
 Download Print
 buttons, you must have an [HTML editor](http://en.wikipedia.org/wiki/Comparison_of_HTML_editors)
 in order to open the downloaded files.
