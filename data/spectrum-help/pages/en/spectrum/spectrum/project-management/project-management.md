# Project Management

The Project Management module is used to track the status of current projects. This module provides the tools to track field questions as well as log project conditions, such as weather, work accomplished and subcontractor progress. This module can be used to record contract specifications, phone lists, employee start and release dates, and information for the subcontractors, vendors, and designers being used on any specific project.
Project Management supports job-based security. You can choose to restrict operator access to disallowed jobs in entry screens throughout the Project Management module by selecting the Utilize job-based security checkbox in Job Cost Installation - Security.
This module is integrated with the Job Cost, Payroll, Accounts Payable and Accounts Receivable modules. In addition, Project Management is integrated with Document Imaging to scan and retrieve project documents, change requests, custom issues, daily logs, RFIs, submittals, and transmittals.
