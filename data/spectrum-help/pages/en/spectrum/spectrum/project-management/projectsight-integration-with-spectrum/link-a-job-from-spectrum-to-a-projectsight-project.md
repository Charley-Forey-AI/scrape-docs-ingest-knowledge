# Link a Job from Spectrum to a ProjectSight Project

Sync jobs in Spectrum Job Cost with new or existing projects in ProjectSight.

1. Create a new job or edit an existing job in
 Spectrum.For more information about setting up
 jobs in Spectrum, see [Job Main Properties](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-properties/job-main-properties).

1. Assign the job status to Active. Important: Setting the
 Active status
 is required to sync the record to ProjectSight.

1. In ProjectSight, link the Spectrum job to either a new or existing project:

- Create a new ProjectSight project and link it to
 Spectrum. This pulls
 data from the Spectrum
 job record into the ProjectSight project.ProjectSight Help: For more information about creating new
 projects in ProjectSight,
 see the ProjectSight Help:
 [Create a
 Project](https://prod.projectsightapp.trimble.com/Web/Web%20Help/Content/Online_Help/Projects/Projects.htm#create).

- Link an existing ProjectSight project to your
 Spectrum job. This
 syncs the Spectrum job
 with the ProjectSight project
 so the data in both systems match. Job data flows from Spectrum into ProjectSight.

Note: You can link a job in
 Spectrum to only
 one ProjectSight project at
 a time.

ProjectSight Help: For more information about linking in ProjectSight, see the ProjectSight Help on [Project Linking](https://prod.projectsightapp.trimble.com/Web/Web%20Help/Content/ERP/Web-ERP-Project-Linking.htm).
Note: Avoid
 deleting linked records. If you delete a record in Spectrum that is linked to
 ProjectSight, the ProjectSight record will not
 update.
