# Cost Type File Maintenance

The Cost Type File Maintenance screen is used to create and maintain a file for various cost types such as Labor and Materials. These cost types are then linked to phases set up for each job in [Job Phases](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/job-phases) (for example, Form work - Labor, Form work - Materials, and so on).
Click the Summary button to create a summary name to define a set of cost types.
