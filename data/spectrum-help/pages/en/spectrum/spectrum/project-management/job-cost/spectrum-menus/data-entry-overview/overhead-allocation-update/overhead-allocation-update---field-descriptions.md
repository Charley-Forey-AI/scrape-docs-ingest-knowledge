# Overhead Allocation Update - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields/ButtonsDescriptions
Buttons
ReverseClick to [Overhead Allocation Reverse](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/overhead-allocation-update/overhead-allocation-reverse).
Transaction dateThe current Job Cost processing date displays. To change it, enter the desired transaction date.The fiscal year and period of the entered transaction date display.
Selections
Batch Job DivisionEnter or search for the batch codes, job numbers, and job divisions for which overhead allocations should be performed.
From dateThe first date in the fiscal period specified at the top of the screen
 displays. Press Enter to accept the default of the current Job Cost
 processing date, press F4 or double-click on this field to select a date from the
 Search Calendar Dates window, or enter the date with
 which the overhead allocation should begin.
Through dateThe date entered above as the transaction date displays. Press Enter to accept the
 current default Job Cost processing date, press F4 or double-click on this
 field to select a date from the Search Calendar Dates
 window, or enter the date through which the overhead allocations should
 occur.
Job statusSelect the job status to determine which records will display. Records with Active and Inactive statuses will default, but you can use any combination of statuses to customize your display.
