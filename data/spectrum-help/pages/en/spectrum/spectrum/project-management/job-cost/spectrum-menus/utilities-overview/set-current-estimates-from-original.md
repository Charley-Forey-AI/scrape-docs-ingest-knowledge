# Set Current Estimates from Original

Use this screen to set the current Phase estimate equal to the original estimate previously entered in Job Phases. When the Current estimates display only checkbox is selected in the Job Cost Installation screen, the phase estimates are reset based on original estimates plus or minus change request cost revisions (actions #1 and #2). When the Job Cost Installation checkbox is clear, then all phases with zero current estimates are reset to the original amount and phases with non-zero amounts are not changed.
