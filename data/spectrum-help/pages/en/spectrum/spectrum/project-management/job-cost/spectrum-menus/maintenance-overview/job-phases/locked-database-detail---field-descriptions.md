# Locked Database Detail - Field Descriptions

This window displays when you attempt to access a file while it is in use by another user.
All fields are view only. No entry is allowed.
All information pertains to the operator currently using the record.

FieldsDescriptions
User IDThe operator's code.
NameThe operator's name.
EmailThe operator's email address.
Time onThe time that the operator began the current session.
CompThe code of the company in use by the operator.
Comp nameThe name of the company in use by the operator.
TermThe terminal number in use by the operator.
