# Overhead Allocation Maintenance - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when adding or changing an overhead allocation line Spectrum will allow entry of a G/L account code only if you have permission to assign that code. Spectrum compares the G/L account's list of shared cost centers with cost centers in your assigned scheme; if there are no common cost centers, then that G/L account cannot be assigned.
Once a G/L account code has been assigned to an overhead allocation line (by an authorized operator), then that G/L account will be used by all operators during processing involving that overhead allocation code.
