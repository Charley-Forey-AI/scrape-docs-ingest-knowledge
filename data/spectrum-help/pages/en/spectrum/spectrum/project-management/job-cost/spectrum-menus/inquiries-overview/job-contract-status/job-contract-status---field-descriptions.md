# Job Contract Status - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Create date
This field displays if a create date was specified in Jobs. Normally, the create date on a job (specified by the user in Jobs) will pre-date all job activity. If for some reason the create date has been set to a period later than the first costs, the calculations in this screen will treat that earlier activity as if it all happened in the same period as the create date.

Complete date
This field displays if the selected job is marked as complete in Jobs.
In the unusual instance where the complete date has been set to a date in a period earlier than the latest costs, the calculations on this screen will display costs and billings in periods after the complete date, up through the period end of the current Job Cost processing date, in order to provide a more comprehensive view of the job as a whole.
