# Adding New Phase Categories

Complete the following steps to add a new phase
 category.

1. On the Site Map, click Job Cost  >  Maintenance  >  Phase Categories to open the Phase Category Maintenance screen.

1. Click New to open the [New/Edit Phase Category](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/phase-category-maintenance/newedit-phase-category) window.

1. Enter a new Code and Category name. In the Properties section, you can create a new Phase mask as well as major and minor groups.

1. Click OK to save your changes and return to the main screen.
