# Procedures as Required

Find a list of procedures, which are only done as
 required.

- Enter non-routine transactions, as needed using Job Cost Transaction Entry. Print reports and update.

- As jobs are completed, print reports as directed by management:
Job Cost Analysis Report
J/C Summary Analysis Report
Job Cost History Report
Job Profitability Report
Job Billing History Report
Job Productivity Report
Job Summary Cost Report
Labor Analysis Report
Payroll Hours Analysis Report
Unit Cost Report
Job Overview Report
Billing Item Analysis Report

- As jobs are completed, enter the actual completion date for the job, and change the job status to 'complete' using Jobs or Job Status Maintenance.
