# Quantity History Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Job
Enter the job code, or press Enter to include ALL on the report.
If cost centers are enabled for the selected company, the operator must have cost center security to the job.

Phase
Enter the phase code, or press Enter to include ALL on the report.

Cost type
Enter the cost type, or press Enter to include ALL on the report.
If cost centers are enabled for the selected company, the operator must have cost center security to the phase and cost type.

From/To Date
Enter the dates to include on the report. The to date will default to the current processing date.
