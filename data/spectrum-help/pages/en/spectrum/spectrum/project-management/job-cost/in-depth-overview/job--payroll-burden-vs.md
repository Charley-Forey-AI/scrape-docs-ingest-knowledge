# Job % Payroll Burden vs. Actual Payroll Burden in General Ledger

Payroll overhead can be used to charge each job with an estimated administrative cost. Payroll burden assesses taxes, worker's compensation, union fringes, and so forth.
If the Actual payroll burdens checkbox is selected on the Job Cost Installation – Payroll tab, Spectrum posts the actual payroll burden when the payroll cycle is updated. If the percentage burden method is preferred, leave the Actual payroll burdens checkbox clear. The setting of the Actual payroll burdens checkbox does not affect existing jobs; it only affects jobs that will be entered in the future. Any changes made to existing jobs must be done in Jobs.
