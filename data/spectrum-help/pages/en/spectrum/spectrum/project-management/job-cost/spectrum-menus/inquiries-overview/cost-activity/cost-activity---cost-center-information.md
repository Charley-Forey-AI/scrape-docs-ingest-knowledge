# Cost Activity - Cost Center Information

If cost centers are being used in the current company, the system will validate the job code entered in the heading of the page to assure the operator has permission for the selected job.
When master jobs are being used, all values for the current job display in the grid, without regard for the cost centers assigned to individual transactions for the job. All values for sub-jobs of an authorized master job, regardless of cost centers assigned to the particular job, phase or transaction detail, will also be included.
