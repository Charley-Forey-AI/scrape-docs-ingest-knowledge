# Payroll Hours Analysis Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Selections

Job Phase Employee Division
Enter the job number, Phase number, employee code, and job Division to include on this report.

Superintendent Estimator Project manager
Enter the superintendent, estimator, and project manager of jobs to include on this report.

From transaction date To transaction date
Do one of the following:

- Enter the first and last dates to include on this report.

- Press Enter to leave the fields blank and begin with the earliest transaction date on file, or end with the current Job Cost processing date.

- Press F4 to display the Date Calculator window and select a calculation method.

Include A/P contract labor hours with Payroll hours?
Select to include Accounts Payable contract labor hours with Payroll hours on this report.
This checkbox displays only if the Accounts Payable Installation screen is set up to use contract hours.
