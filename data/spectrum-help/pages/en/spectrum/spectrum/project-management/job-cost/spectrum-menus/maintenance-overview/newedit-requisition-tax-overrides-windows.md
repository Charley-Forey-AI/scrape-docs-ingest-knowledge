# New/Edit Requisition Tax Overrides windows

Find details on adding or editing requisition tax
 overrides.

## Requisition Tax Overrides by Cost Type

Use this window to add or edit requisition tax overrides by cost type.
Fields
Descriptions

Tax classification
The tax classification of the selected cost type displays.

Override cost type
Enter the cost type you want to override.

Taxable?
Select this checkbox to make the requisition tax override by cost type taxable.

## Requisition Phase-Level Tax Overrides

Use this window to enter a second series of overrides by phase.
Field
Description

Phase
Enter the phase code to be overridden by the cost type.

Description
Enter the phase code description.

Taxable?
Select this checkbox to make the phase-level override taxable.
