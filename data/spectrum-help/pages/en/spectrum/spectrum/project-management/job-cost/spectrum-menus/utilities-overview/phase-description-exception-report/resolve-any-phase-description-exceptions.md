# Resolve any Phase Description Exceptions

You can resolve any phase description exceptions in the job
 phases section.
1. Go to [Job Phases](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-properties/job-phases).
2. Enter the job and phase referenced on the exception report.
3. Enter through the description and all cost types.
4. Click OK
 until you reach the Site Map
 screen.
