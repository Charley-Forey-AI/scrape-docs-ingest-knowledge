# Job User-Defined Fields Maintenance - Cost Center Information

 If the cost center feature is enabled in the Enterprise Installation screen, when purging a job user-defined field Spectrum will remove the records only if you have permission to access all affected jobs. Spectrum compares the cost center assigned to each affected job with cost centers in you assigned scheme; if any cost center is not included, then the user-defined field will not be purged and no job changes will be recorded.
