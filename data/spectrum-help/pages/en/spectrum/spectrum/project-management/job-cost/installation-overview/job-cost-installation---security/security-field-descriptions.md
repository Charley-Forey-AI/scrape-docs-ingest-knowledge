# Security Field Descriptions

Use the Security field descriptions when completing actions
 on this screen.
Fields/buttons
Descriptions

Utilize job-based security?
Select this checkbox to use job-based operator
 security in this company. This security restricts access to Job Cost
 reports, inquiry screens, quantity entry, and projected cost entry
 based on the operator's security settings. Job-based operator
 security also restricts access to Project Management data entry,
 inquiries, and reports.

New operator defaults (These fields are
 available only if the Utilize job-based security checkbox is
 selected.)

Project manager Superintendent Estimator Master
 job Division
Entries made in these fields will serve as the
 defaults for any future operators who are assigned to this (current)
 company.
For example, you may choose to complete only the
 Superintendent field so that only those jobs assigned to the
 specified Superintendent will display on job inquiries.

Grid columns: Operator-specific settings

Code Name
The list of individual operators who currently
 have security access to Job Cost or Project Management modules in
 this company displays. No entry is allowed.

Project Manager
Superintendent
Estimator
Master Job
Division
The selected operator's settings display. Use
 these columns to further specify which jobs and projects operators
 will be able to access.

1. To change any of
 these settings:

- Highlight an operator code/name.


- Make any changes, for example,
 designate a new Project Manager to give future
 operators access to jobs assigned to the specified
 Project Manager. Leave the column blank to give the
 operator access to jobs assigned to all project
 managers.

Regarding the Division
 column, if the other columns on this screen are blank, but
 this one is set to a specific value, the operator will be
 locked out of all jobs.
If all the other fields on this screen are
 set to "ALL" and the Division
 column is set to a specific value, the operator will have
 access to jobs in that specific division.
