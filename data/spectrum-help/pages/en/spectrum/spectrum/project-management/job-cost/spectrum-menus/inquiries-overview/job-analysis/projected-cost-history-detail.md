# Projected Cost History Detail

Use this screen to review the contract's projected cost history in detail. Changes shown in this screen reflect entries from projected cost processing (in addition to several installation selections). Clicking any of the column heading buttons allows you to sort the data accordingly.
Field
Description

Phase
The phase for this contract displays.

CT
The cost type for each phase displays.

Description
The phase description displays.

Date
The date the change was entered displays.

Time
The time the change was entered displays.

OPR
The operator responsible for the contract change displays.

Projected change
The total projected change in revenue for each phase displays.

Note
Any text that was entered in the memo field of the [Projected Cost Entry (by Phase summary)](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-entry-by-phase-summary) screen displays.
