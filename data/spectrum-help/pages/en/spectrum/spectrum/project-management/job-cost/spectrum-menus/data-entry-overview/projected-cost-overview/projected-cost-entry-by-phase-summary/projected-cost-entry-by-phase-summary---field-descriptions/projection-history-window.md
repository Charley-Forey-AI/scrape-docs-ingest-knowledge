# Projection History window

Use the Projection History window to look up the projected cost history of a selected phase line. Each projected cost change made for the phase displays in the list box, including information about the date, time, operator, dollar amount (for the new projected amount), and any related notes.
Field
Description

Phase
The phase code and description for the selected phase line displays.

Cost type
The cost type and description for the selected phase line displays.

Date
A list of dates, times, operators, amounts, and memos for any projected cost changes made to the phase display in the list box.

Time
A list of times for any projected cost changes made to the phase displays in the list box.

Operator
A list of operators who made any projected cost changes to the phase displays in the list box.

Amount
A list of amounts for any projected cost changes made to the phase displays in the list box.

Memo
A list of memos made for any projected cost changes displays in the list box.
