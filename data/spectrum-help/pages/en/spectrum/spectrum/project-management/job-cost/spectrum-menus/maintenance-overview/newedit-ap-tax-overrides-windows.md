# New/Edit A/P Tax Overrides windows

## A/P Tax Overrides by Cost Type

Use this window to add or edit A/P tax overrides by cost type.
Fields
Descriptions

Tax classification
The tax classification of the selected cost type displays.

Override cost type
Enter the cost type you want to override.

Taxable?
Select this checkbox to make the A/P cost type override taxable.

Click Edit or New to modify or add A/P tax overrides by phase.

## A/P Phase-Level Tax Overrides

Use this window to enter a second series of overrides by phase.
Field
Description

Phase
Enter the phase code to be overridden by the cost type.

Description
Enter the phase code description.

Taxable?
Select this checkbox to make the A/P phase override taxable.
