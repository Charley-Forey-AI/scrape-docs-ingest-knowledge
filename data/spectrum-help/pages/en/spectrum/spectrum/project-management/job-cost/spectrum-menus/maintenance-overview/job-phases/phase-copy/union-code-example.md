# Union code example

Given the following wage/union codes, if "276" and "756" were entered at the "union code" prompts, L1, L3, and C1 would be set up as wage codes in the job master file.
L1 = General Laborer
276 - Bellingham
242 - Seattle
440 - Seattle
252 - Tacoma
L2 = Jackhammer Operators
242 - Seattle
440 - Seattle
252 - Tacoma
L3 = Drill Operator
276 - Bellingham
242 - Seattle
440 - Seattle
252 - Tacoma
 L4 = Drill Helper
242 - Seattle
440 - Seattle
252 - Tacoma
 C1 = Carpenter
470 - Tacoma
756 - Bellingham
1144 - Seattle
 I1 = Ironworkers
86 - Seattle
14 - Spokane
114 – Tacoma
505 - Bellingham

For each union code specified on the starting screen for Phase Copy, if
 that union is recorded in the Job Phases > Properties > Wage Codes window, the wage/union code will be copied to the new job.
