# New/Edit Proposed Job

Use this window to create a new proposed job or to edit an existing job. Information from the New Proposed Job and Edit Proposed Job windows displays in the main Proposed Job Maintenance screen immediately upon closing the window.
Fields
Descriptions

Proposed job
Enter a proposed job code (if creating a new job). If you are editing an existing proposed job, the code will default here and no changes may be made to this field.
Cost Center Information
If the cost center feature is enabled in the Enterprise Installation screen, a must-enter Cost center field will appear beneath the Proposed job field. The software will compare the cost center with cost centers (or cost groups) in the Operator's assigned cost center scheme, and if the proposed cost center is not present entry is not allowed.
When adding a new proposed job, the cost center entry must be a valid and active code authorized for the Operator. When editing a proposed job with an existing cost center, the Operator will not be permitted to enter a disallowed cost center code and the existing cost center code will remain in this field.

Description
Enter or edit a job description for this proposed project.

Entered date
Enter the date that this proposed job was entered into Spectrum (today's date defaults), or click the drop-down arrow use the Date Change calendar to select a date.

Remarks
Enter any additional information you want to include for this proposed job.

Estimates

Start date Complete date
Enter the estimated start and complete dates for this proposed job or click on the drop-down arrow to select a date on the Search Calendar Dates window.

Contract amount
Enter the estimated contract amount for this proposed job. The amount entered here will be used in the Totals calculations that display on the main Proposed Job Maintenance screen.

Cost
Enter the estimated cost of this proposed job. The amount entered here will be used in the Totals calculations that display on the main Proposed Job Maintenance screen.

Win probability
This field is used to calculate the expected value of contract for proposals. For example, a $10,000,000 contract with win probability of 40% = $4,000,000. If the job Type is set to Awarded or Signed, this percentage automatically displays as 100%.

Status
Select one of these options to display jobs in the Proposed, Awarded, or Signed status.
