# Projected Cost Calculations window

Use this window to customize how calculations work for your operator. The options you select determine which fields should be calculated based on data entry, and which fields should be locked (if you enter a value, certain values will remain locked while others are recalculated).
The Projected quantity field is never recalculated.
For example, if the dollars are entered, you can choose to recalculate either the hours or the dollars per hour. If the hours are recalculated, the units per hour and dollars per unit would also be recalculated.
