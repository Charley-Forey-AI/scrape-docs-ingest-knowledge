# Projected Cost Entry (by Group)

Use the Projected Cost Entry screen to enter information
 from the project manager's Projected Cost Worksheet for both fixed
 price and unit price jobs.
The line transaction portion of this screen corresponds to the
 Projected Cost Worksheet from which entry will typically occur. Each line transaction shows
 the estimate, percent complete, and the projected cost per hour.

- When the Group option is being used, the Update button does not appear. Use the Phase format to verify information in detail, and to archive the detail report.

- When performing the [Build Projected Cost Transaction File](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/build-projected-cost-transaction-file), if Major group only or Through minor group options are selected in the Group summary  section, then the Projected Cost Entry screen displays for groups.

- Select the Calculations button to open the [Projected Cost Calculations window](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-calculations-window) window and specify which fields are to be recalculated.

- If a job is selected that is not already built, the software opens the Build Projected Cost Transactions for Job window to quickly build the specified job using the saved selections from the Build Projected Cost Transaction File Starting screen.
