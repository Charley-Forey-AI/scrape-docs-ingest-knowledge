# Building Projected Costs

Building the projected costs creates entries in the Project Cost Transactions file.
The Build function builds a transaction file for use in entering projected cost transactions. The transaction file is built from estimate and projected cost information in Job Phases. Actual costs-to-date through the specified year and period will be included in the transaction file.
The ability to select by group summary has been incorporated into this file. If major or minor grouping is selected at the time of the build, the projected entry screen will show only the major or minor group totals for entry. When entering a percent complete, cost to complete, or final projection for the group, the system will automatically allocate this percentage complete to all the detail phases of this group.
