# Cost Type Summary Report

The Cost Type Summary Report provides useful information when balancing job cost totals with General Ledger account balances. The report gives summary totals by cost type, and is especially useful when General Ledger work-in-progress accounts are linked to the cost types in job cost. The Summary report summarizes cost type amounts across jobs, per period. The Detail report details cost type amounts for each job, per period.
