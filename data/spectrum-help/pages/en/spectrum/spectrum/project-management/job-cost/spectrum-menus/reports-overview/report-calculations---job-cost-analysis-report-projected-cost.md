# Report Calculations - Job Cost Analysis Report (Projected Cost)

The following table describes how each column is calculated on this report.
Field
Description

Committed Cost
The Committed cost figure displays from Job Phases.

Actual Cost
The actual job cost amount for the selected period, as well as the job-to-date amount, displays for each phase and Cost type. These figures are accumulated balances of transactions updated through the year and period requested.

Projected Cost
The total Projected cost at completion for each phase/cost type displays from Job Phases.

Estimate
The current Estimate amount displays from Job Phases.

Variance
Variance is calculated as the difference of the two preceding columns; that is, the projected cost less the estimated cost.

% Var
Percent variance is calculated as the quotient of the two preceding columns; that is, the variance amount divided by the estimated cost, multiplied by 100.

% Comp
The percent complete is calculated as the actual job-to-date figure divided by the projected cost, multiplied by 100.
