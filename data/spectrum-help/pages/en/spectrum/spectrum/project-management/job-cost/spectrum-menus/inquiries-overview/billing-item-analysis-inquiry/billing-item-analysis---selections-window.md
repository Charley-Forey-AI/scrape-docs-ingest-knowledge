# Billing Item Analysis - Selections window

To accept the system defaults, press Enter. To limit the data displayed, select only the checkboxes of the desired information.
Field
Description

Use projected cost
Select the checkbox to use projected cost, or leave the checkbox unmarked to use actual costs. Entry in this field is based on a default from the Job Cost Installation screen.

Include pre-time card entries
Select the checkbox to include pre-time card entries; otherwise leave the checkbox unselected to leave them out. The default in this field is based on the setting for the Include pre-time cards checkbox in Job Cost Installation.
If you use this feature and if you post Payroll burdens to separate phases, verify that the burden phases are added to the phase detail file before any pre-time cards are entered using these phases.

Batch
When the Include pre-time card entries checkbox is selected, enter the pre-time card batch code, press Enter for ALL batch codes, or use the SuperSelect options.
This field displays only when the Include pre-time card entries checkbox is selected.

Include unapproved A/P invoices
Select this checkbox to include unapproved invoices; otherwise, leave the checkbox clear.

Include A/R unposted billings and pre-billings
Select the checkbox to include Accounts Receivable unposted draw billings. In order to exclude Accounts Receivable unposted draw billings, do not select the checkbox.

Batch
When the Include A/R unposted billing and pre-billings checkbox is selected, enter the Accounts Receivable code, or press Enter for ALL batch codes.
This field displays only when the Include A/R unposted billing and pre-billings checkbox is selected.
