# Set Current Estimates from Original - Cost Center Information

If the cost center feature is enabled in the Enterprise
 Installation screen, when performing the Set Current Phase Estimates update Spectrum will
 reset the phase only if you have permission to access the phase. Spectrum compares the cost
 center assigned to the job with cost centers in your assigned scheme. If the cost center is
 not included, then the estimates are not reset on that job. In addition, Spectrum compares the
 cost center assigned to each phase, if applicable with cost centers in your assigned scheme.
 If the cost center is not included, then the estimate in that phase is not reset. If a job
 does not have a cost center, this utility will only automatically change records that the
 operator is allowed to manually access. In these instances, the operator must open Jobs and
 edit the job accordingly.
