# Job Contract Status

Use the Job Contract Status screen to show period-by-period
 contract status, back to the beginning of the job.
The grid shows one row for each fiscal period (newest to oldest). This
 screen shows historical information only, beginning with the last month and working
 backwards. The current contract status displays on the [Job Analysis](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-analysis) screen. If the selected job is a Master job, the Job code field will be labeled Master job and the grid will include
 amounts from sub-jobs for the master.
The calculations on this tab are computed based on actual historical information, built from cost history, billing history, projected cost history, and actual approval and executed date of change orders and change requests. For calculation details, see [Contract Status Calculations](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/inquiries-overview/job-contract-status/contract-status-calculations).
