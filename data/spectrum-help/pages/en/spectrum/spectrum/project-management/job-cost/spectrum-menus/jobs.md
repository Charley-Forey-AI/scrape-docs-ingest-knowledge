# Jobs

Use this screen to manage your jobs. You can set up new
 jobs, change existing jobs, or look up job information.
Security restrictions apply if Utilize Job-based security is selected in [Job Cost Installation - Security](/en/spectrum/spectrum/project-management/job-cost/installation-overview/job-cost-installation---security). If you don't have security authorization, you cannot make changes.
Double-clicking a row in the list box takes you to view-only details.
Note: It is possible to set up change orders in this screen.
 However, if the Accounts Receivable module is installed, we recommend that you process
 change orders through Accounts Receivable > Data Entry > Change Order Log/Entry.
If you have security rights to add jobs, click New to open the [New Job](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-properties/new-job) window and get started. Otherwise, in the
 Search bar, enter the job you want to look up and then click
 Edit to modify a job in the
 [Job Analysis](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-analysis) screen.
