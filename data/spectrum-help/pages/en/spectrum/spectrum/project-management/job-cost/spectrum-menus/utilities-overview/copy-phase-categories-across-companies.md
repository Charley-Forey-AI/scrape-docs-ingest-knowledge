# Copy Phase Categories Across Companies

Use the Copy Phase Categories Across Companies screen to copy phase file detail information from one company to another. Caution should be exercised when performing this function.
Note: This function does not copy major/minor group summary
 information.
Operators with security access can use this screen to copy phase, group, and phase category records to multiple companies during the same update, and to overwrite existing information in the destination companies. If the corresponding phase category codes were not set up previously, they will be added during the update.
