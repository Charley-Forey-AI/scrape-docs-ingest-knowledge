# Job Comparison Report

Use this report to obtain a comparison of the costs for up to five selected jobs. Actual figures for the phases/cost types are displayed so that the user can see any correlation between similar jobs. There are no special calculations for this report. All figures are printed directly from Job Phases.
So that similar phases can be compared even when the phase mask is different across jobs, this report examines the "alternate phase" defined in Job Phases.
