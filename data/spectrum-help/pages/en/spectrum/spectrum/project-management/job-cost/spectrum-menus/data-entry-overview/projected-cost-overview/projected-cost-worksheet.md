# Projected Cost Worksheet

The Projected Cost Worksheet screen is used to print a worksheet for the project manager to complete. The job project manager should fill in the appropriate blanks on the printed worksheet. This completed worksheet is then used as the source document for Projected Cost Entry.
Note: The Projected Cost Worksheet calculates the projected cost based
 on the date specified in the Through period
 end date field and records stored in the projected cost history file.
