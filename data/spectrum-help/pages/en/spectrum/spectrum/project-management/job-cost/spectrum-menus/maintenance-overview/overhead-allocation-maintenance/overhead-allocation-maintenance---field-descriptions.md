# Overhead Allocation Maintenance - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields/Buttons
Descriptions

Batch code
The batch is compared with the batch entered in the Overhead window in the Properties window of the [Jobs](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs) screen.
Enter the batch code of the group of jobs for which overhead allocation will be specified.

Latest transaction
The transaction date displays from the last time the [Overhead Allocation Update](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/overhead-allocation-update) was performed.

From/to history date
The from and through dates from the last time the Overhead Allocation Update was performed display.

Overhead total
The overhead total displays.
