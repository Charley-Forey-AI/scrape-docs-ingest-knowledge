# Reclassify Job Cost History: Transaction Type IC - Inventory - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, for IC transaction type entries Spectrum will allow you to edit a transaction for an item only in you have permission to access the warehouse's information. Spectrum compares the warehouse cost center with cost centers in your assigned scheme; if the cost center is not present, then transaction entry for that warehouse will be disallowed. Validation will also take place if you attempt to assign a disallowed warehouse to a transaction line.
Spectrum also verifies that you have authorization for the item code. Spectrum compares the item's list of shared cost centers with cost centers in your assigned scheme; if there are no common cost centers, then transaction entry fort hat item will be disallowed.
The credit cost center displays based on the cost center of the warehouse code. No changes are permitted.
