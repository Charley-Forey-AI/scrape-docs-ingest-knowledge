# Tax Classification File Maintenance - VAT Processing info

 If the 'Utilize value added tax (VAT) tracking?' checkbox is selected in General
 Ledger Installation, the Tax Classification File Maintenance screen offers an 'Override VAT'
 column that displays the tax code that defaults into customer invoices for jobs assigned to a
 particular tax classification code. In addition, the tax code plus percentage
 displays.
