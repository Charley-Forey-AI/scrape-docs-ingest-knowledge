# Build Job List

Use this window to define job information before displaying the [Set Job Status](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/utilities-overview/set-job-status) screen.
Fields
Descriptions

Selections

Job
Customer
Division
Enter or search for the job number, customer number, and division.

Personnel

Superintendent
Estimator
Project
 manager
Enter the superintendent, estimator, and project manager.

Dates

From last cost
 date
To last
 cost date
Enter the earliest and latest last cost dates to include, or press Enter to accept the default dates. To quickly enter dates, view the Date Shortcuts.

From last bill
 date
To last
 bill date
Enter the earliest and latest last bill dates to include, or press Enter to accept the default dates. To quickly enter dates, view the Date Shortcuts.

Job status

Active?
Inactive?
Complete?
Select the Job status to determine which records will display. Records with an Active and Inactive status will default, but you can use any combination of statuses to customize your display.

Defaults

Complete date
Enter the default complete date for jobs set to Complete during this session.

Allow override of complete date?
Select this checkbox to allow the job completion date to be entered for jobs
 set to Complete
 during this session. If this checkbox is not selected, the complete date
 specified above will be used for jobs set to complete.Note: If this checkbox is selected,
 an additional window will prompt you to approve the "Complete date"
 before exiting the Set Job Status screen.

New status
Select a new status default.
If the Job Cost Installation screen Set projected costs, quantity, and
 hours to actual when job is complete checkbox is selected and
 a job's status is set to Complete, then the job's projected quantity is automatically
 set to equal the actual quantity amount.

Default all 'Complete' jobs to disallow revenue
 entry?
If the Disallow revenue
 entry checkbox is selected in Job Main
 Properties, select this checkbox to default this setting for
 all 'Complete' jobs.

User-defined fields
Enter text in the entry fields associated with the user-defined fields. Press
 F4 to select
 from a list of available user-defined fields.Note: Only alpha user-defined fields display; numeric and date user-defined
 fields are not permitted.

The field names are defined in Job Cost > Job User-Defined Fields Maintenance.
