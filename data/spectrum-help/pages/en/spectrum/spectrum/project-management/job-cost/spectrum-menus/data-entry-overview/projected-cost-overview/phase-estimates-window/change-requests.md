# Change Requests

Use this screen to review the change request information for the selected company. Click any of the column heading buttons to sort the data accordingly.
Fields
Descriptions

Total Executed, Approved, or Proposed Changes
Depending on the type of change you are viewing, the total for the executed, approved, or propose change requests displays as a percentage.

Change Request, Change Order
Click the links to review change request details such as the current status, important dates, permissions, and possible time extensions.

CR Number
The Change Request number displays based on your selections in the Job, Customer, andStatus fields. To edit, add, or delete a selected change request, click the Edit, Add, or Delete buttons.

Owner CO
The owner-assigned number for the change order displays.

Description Submitted Approved Status Amount Customer
The change request description, date submitted, date approved, status, amount, and customer display.
