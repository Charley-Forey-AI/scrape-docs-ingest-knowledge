# Purge Job Cost Quantity History

Use this function to preview, and then Purge data from the quantity history file. It is possible to purge only a portion of the data previewed.
This feature should only be used when severe limitations exist on the user's operating system, such as a shortage of disk space. Viewpoint does not usually recommend use of the purge feature.
CAUTION: Once quantities have been purged, they cannot be
 accessed for inquiry or reports, so extreme care should be taken when using this
 feature.
This function will first preview data from the quantity history file. Once you have
 carefully previewed the report, the Quantity History Purge screen displays. Click
 Continue to proceed with the
 purge, or Cancel.
