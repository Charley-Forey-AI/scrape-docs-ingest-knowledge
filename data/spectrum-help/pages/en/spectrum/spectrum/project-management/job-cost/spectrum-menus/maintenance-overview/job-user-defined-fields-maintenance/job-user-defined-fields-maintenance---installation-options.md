# Job User-Defined Fields Maintenance - Installation Options

Use the table below for user-defined installation options and their result on the Job User-Defined Fields screen.
Installation tab checkbox
Option
Result on Job User-Defined Fields screen

Allow setup of company-specific fields?
Select
Company-specific user-defined fields can be set up for the current company.

Allow setup of non-company-specific fields (company ALL)?
Select
 Users are permitted to set up user-defined fields for all companies.

Clear
Users are not permitted to set up user-defined fields that are not company-specific.

Show non-company-specific fields (company ALL)?
Select
User-defined fields for all companies can be edited.

Clear
Only user-defined fields associated with the current company can be edited.
