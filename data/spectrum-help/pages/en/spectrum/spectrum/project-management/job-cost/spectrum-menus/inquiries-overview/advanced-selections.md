# Advanced Selections

Use this window to adjust calculation parameters for the Job Analysis Inquiry screen. Entries in this window will be retained for as long as the operator remains in the Job Analysis Inquiry screen.
Fields
Descriptions

Job-to-date calculations

Include open commitments in job-to-date cost?
Select this checkbox to include open commitments in job-to-date cost totals. This field will default based on the 'Include open commitments' checkbox in Job Cost Installation (unless a saved filter is present).

Include unapproved A/P invoices in job-to-date cost?
Select this checkbox to include unapproved A/P invoices in job-to-date cost totals. This field will default based on the 'Include unapproved Accounts Payable invoices' checkbox in Job Cost Installation (unless a saved filter is present).

Include A/P contract labor hours with Payroll hours?
Select this checkbox to include A/P contract labor hours with Payroll hours. This field will default to checked if one or more cost types is specified in 'Contract labor cost types' in Accounts Payable Installation, otherwise this field will default to cleared (unless a saved filter is present).

Include Payroll pre-time card cost and hours?
Select this checkbox to include Payroll pre-time card cost and hours. This field will default based on the 'Include pre-time cards' checkbox in Job Cost Installation (unless a saved filter is present).

Week-to-date calculations

From date
Enter a start date for week-to-date calculations. This field will default to seven days before the current Job Cost processing date (unless a saved filter is present).

To date
Enter an end date for week-to-date calculations. This field will default to the current Job Processing date (unless a saved filter is present).

Quantities

Multiple quantity cost types?
Select this checkbox to include multiple quantity cost types in the report calculations. This field will default based on the 'Use multiple quantity cost types"' selection in Job Cost Installation (unless a saved filter is present).

Quantity cost type
Select a quantity cost type to include in the calculation parameters:

- Burden

- Equipment

- Hauling

- Labor

- Materials

- Overhead

- Subcontracts

- Trucking

This field will be hidden if the Multiple quantity cost types? checkbox is selected.

Filter
Click this button to create a filter for searching based upon the settings in Advanced Selections.
