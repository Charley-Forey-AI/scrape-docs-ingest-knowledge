# Weekly Procedures - As Needed

Find information on typical weekly procedures.

- Quantity Tracking
Print the Quantity
 Complete Worksheet, and/or the  Projected Cost Worksheet,
 depending on the company's needs. These should be given to the appropriate job
 superintendents for completion.
From the worksheets returned by job superintendents, perform
 Quantity Complete Entry
 and/or Projected Cost
 Entry.

- Print the associated reports. Check the reports for accuracy. If accurate, Update.

- Print the Job Cost Analysis Report for week-to-date amounts, if desired.

- Print the Summary Analysis Report for week-to-date amounts, if desired.

- Print the Job Productivity Report for week-to-date amounts, if desired.

- Print the Job Summary Cost Report for week-to-date amounts, if desired

- Print Job Unit Cost Report for week-to-date amounts, if desired.
