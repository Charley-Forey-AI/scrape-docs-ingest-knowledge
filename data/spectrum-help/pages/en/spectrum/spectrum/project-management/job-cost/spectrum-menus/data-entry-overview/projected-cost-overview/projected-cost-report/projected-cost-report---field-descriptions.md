# Projected Cost Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Selections

Batch
The operator's login initials default as the batch code, but you can enter a new one.

Job
Enter the job number to include on this report. You must specify a job number before proceeding.
Note: The default job number that displays is the last job number
 entered in this field during the current session. If you are accessing this
 screen for the first time during a session, this field defaults to blank.

Include rates detail?
Select to include the rate detail for each type (Dollars, Hours, and Quantity) selected for the report.

Include

Dollars?
Hours?
Quantity?
Select at least one checkbox that corresponds to the data you want to display on the report. For each checkbox selected, a separate line will be included for the selection on the report.

Report format

Phase/Cost type
Phase summary
Group
Select the Phase/Cost
 type option to include all jobs currently built from group or
 phase.
Select the Phase summary option to include all jobs built from phase, but exclude all jobs for group summary.
Select the Group
 option to include all jobs built from group, but exclude jobs built by
 Phase/cost type and Phase summary.
