# Cost Type Summary Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Description

Selections

Job
Enter a job number, press F4 select from a list of valid job numbers, use the SuperSelect options, or press Enter to accept the system default of ALL.

Phase
Enter a Phase number, use the SuperSelect options, or press Enter to accept the system default of ALL.

Cost type
Enter a cost type number, use the SuperSelect options, press F4 to select from a list of valid cost types, or press Enter to accept the system default of ALL.

Division
Enter the job Division number, use the SuperSelect options, or press Enter to accept the system default of ALL.

From period start date
To period end date
Enter the period start and end dates, or press Enter for the first day of the current fiscal year, based on the current Job Cost processing date. The corresponding fiscal year and period automatically display (no entry is allowed in these two fields).
This field is not applicable when the Totals field is set to Job-to-date.
The fiscal year and period display based on the period end dates.

Report type

Summary
Detail by job
Select to print the report in Detail by job report or using the Summary report format.

Totals

Year-to-date
Job-to-date
When Job-to-date is
 selected, totals are shown for all costs through the to period-end date.
 When Year-to-date
 is selected, totals are shown for all costs for the from/to date range
 specified.

Job status

Active?
Inactive?
Complete?
Enter the Job status to display on the report.
