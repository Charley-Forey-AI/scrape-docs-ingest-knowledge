# Job Cost Installation - Properties

Use this tab to set the default properties settings for the Job Cost module. These settings can be changed as needed at any time.
Note: When the Job Cost Installation screen is accessed for
 the first time, the system will beep and a message stating "MISSING CONTROL RECORD FOR JOB
 COST - OK TO PROCEED" will display. Once this screen has been completed and saved, the
 message will not display again.
