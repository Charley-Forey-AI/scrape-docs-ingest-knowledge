# Delete Unused Phases

Use this screen to remove all phases for the selected job that have no actual hours, quantities, amounts or open commitments regardless of any existing estimates or projects. Phases that contain estimated or projected figures, but no actual costs will be deleted. When phases are deleted, any associated wage code/union detail is also removed. Phase notes will be removed when the last cost type for a given phase is deleted.
