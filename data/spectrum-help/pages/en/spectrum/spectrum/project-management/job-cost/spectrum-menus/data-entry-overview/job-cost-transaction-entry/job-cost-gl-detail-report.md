# Job Cost G/L Detail Report

The Job Cost G/L Detail Report screen is used to print general ledger information pertaining to the cost transaction report. This report should be run after transactions are updated using Cost Transaction Update and before information is updated to the general ledger. Once the report has printed, this report starting screen will be replaced with the update screen.
Important: This report should be retained as a permanent audit record of the company.
