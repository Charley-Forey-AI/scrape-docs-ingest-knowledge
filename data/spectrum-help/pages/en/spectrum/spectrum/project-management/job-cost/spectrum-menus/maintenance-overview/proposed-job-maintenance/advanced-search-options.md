# Advanced Search Options

Use the Advanced Search Options window to narrow down the list of
 records that will display in the proposed job list on the Proposed Job Maintenance screen.
This window is especially useful if you have a large number of jobs and partial information
 about the specific code or codes you are trying to look up or edit. For example, you can
 search on all the proposed jobs that have already been awarded.
FieldDescription
Proposed jobUse this field to search on the job code or double-click to select from a list of available job codes. To accept the default of ALL, press Enter.Tip: If you only know the first character or characters of the code you want to search on, enter these characters and then enter an * (asterisk).

DescriptionUse this field to search on the job description. To accept the default of ALL, press Enter.Tip: If you only know the first letter or letters of the alpha code you want to search on, enter these characters and then enter an * (asterisk).
