# New/Edit Job Group - Properties tab

Use this table for reference when completing the fields in this window.
Fields
Descriptions

Template code
In the New Job Group window, enter the job group code. In
 the Edit Job Group window, this field is view-only.

New job

Job type
Enter the description for this job group.

Job mask
Enter the job mask number, for example, SJXXX if jobs in this group should be
 assigned SJ001, SJ002, SJ003, and so forth.
Otherwise, leave this field blank to allow the operator to specify the job number as part of the procedure for adding each new job in this group.

Next job number/code
Job code is only available if you are using the Job mask above. Either accept the default or enter the next available job sequence number.

Copy from

Job
Enter the job number from which to copy. If job-based security is enabled, Spectrum verifies that you have permission to access this job.

Phase mask
The phase mask for the copied job displays. You can view up to 20 characters.
