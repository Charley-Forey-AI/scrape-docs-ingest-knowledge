# Set Job Status - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, Spectrum displays the selected job only if you have permission to access the job. Spectrum compares the cost center assigned to the job with cost centers in your assigned scheme, and if the cost center is not included, then that job is not shown on the Job Status Maintenance screen.
