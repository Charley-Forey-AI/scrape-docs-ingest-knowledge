# Month-End/Quarter-End Procedures

Find information on typical month-end/quarter-end
 procedures.

- Print the Job Cost reports for the current month as directed by management.

- Make a month-end backup.

- Perform Earned revenue processing using Prior Period Earned Revenue Entry.

- Print the Contract Status Report.
