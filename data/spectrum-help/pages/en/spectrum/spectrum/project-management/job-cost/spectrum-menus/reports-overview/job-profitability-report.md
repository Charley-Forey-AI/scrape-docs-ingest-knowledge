# Job Profitability Report

Use the Job Profitability Report to identify job-to-date profit margins. You can preview the report with or without projected costs. The report displays in sequence by job number or project manager.
Note: This report only displays jobs with activity (billing
 or cost) as of the report date, regardless of the job start date.
