# J/C Processing Date Change - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Current Settings

Date
The left column displays the current processing information. The information
 is re-displayed on the calendar on the right. Enter the new processing date
 by typing over the displayed date, or click the date on the calendar. To
 leave the processing date as it is displayed, click the Reset button to restore
 the present setting. Click Cancel to exit the screen without saving the changes.

Fiscal year
The fiscal year in which the specified processing date is contained displays.

Period
The period number in which the specified processing date is contained (number 1-13) displays.

Period end
The ending date of the period in which the specified processing date is contained displays.

Previous period end
The ending date of the previous period displays.

Min/Max Settings

Minimum date
The earliest date that a processing date can be changed to is displayed in
 this field. This date is set in the System Administration > Installation > Processing Date Change screen.

Maximum date
The latest date that a processing date can be changed to is displayed in this
 field. This date is set in the System Administration > Installation > Processing Date Change screen.
