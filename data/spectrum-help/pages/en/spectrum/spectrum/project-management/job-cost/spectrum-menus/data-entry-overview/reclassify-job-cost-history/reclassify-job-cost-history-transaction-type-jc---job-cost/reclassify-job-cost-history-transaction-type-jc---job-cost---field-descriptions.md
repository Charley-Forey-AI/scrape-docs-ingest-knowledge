# Reclassify Job Cost History: Transaction Type JC - Job Cost - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

A check mark displays in this column for each line that has been reclassified.

Reference Description Job
Enter the reference number, description, and job number.

Phase
Enter or search for a phase code.
Note: Entry is prevented if the phase's status is set to
 Complete.

Ct Hours
Enter the cost type and number of hours associated with this phase.

Amount
Press Enter to accept the default dollar amount of the transaction, or enter a new dollar amount.

G/L debit
The G/L account code and description for the debit displays. No entry is allowed if a G/L account code is specified for the cost type in Cost Type File Maintenance.
Entry is prevented if the account's status is set to Not used
Note: This must be a direct job cost code.

G/L credit
 Enter the G/L account to be credited.
Entry is prevented if the account's status is set to Not used.
Note: This cannot be a direct job cost code.

Job name Phase description
The name of the job and phase description display.
