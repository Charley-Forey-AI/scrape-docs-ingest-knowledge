# Labor Analysis Report

The Labor Analysis Report is the traditional budget vs actual look at the job. On this report we see estimated, projected and actual hours and dollars. This report does not differentiate between regular, overtime and double time.
The Labor Analysis Report can be printed either with or without projected costs. The expanded format of this report is designed for use by companies with unit price jobs, when projected hours and quantity are tracked. The information for each job prints in sequence by Phase.
