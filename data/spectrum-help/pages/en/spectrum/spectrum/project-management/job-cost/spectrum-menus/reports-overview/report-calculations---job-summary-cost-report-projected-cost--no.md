# Report Calculations - Job Summary Cost Report (Projected Cost = No)

The following table describes how each column is calculated on this report.
Field
Description

Period
Actual period-to-date costs for the designated year and period display.

Job To Date
Actual job-to-date costs through the year and period display.

Estimate
The current estimated amount displays from Job Phases.

Standard Format
Figures in the labor, materials, and subcontract columns are for the corresponding cost types for each phase (the labor and subcontract columns contains cost types designated on the start screen; all other cost types are contained in the materials column). The totals columns provide the sums of figures in the three cost-type columns.

Expanded Format
Figures in the labor, material, subcontract, cost type #4 and cost type #5 columns are for the corresponding cost types for each phase. The labor, subcontract, cost type #4 and cost type #5 columns contain the cost type(s) specified on the starting screen for this report. All other cost types are contained in the materials column. The totals columns provide the sums of figures in the five cost type columns.
