# Time Card History Detail window

The Time Card History Detail window is used view all time cards included on this employee's check pertaining to the current job. The columns in this window are moveable and sortable, to allow you to customize the display.
Note: If a job has been reclassified from the original transaction, an alert message will appear beneath the list box to reflect this.
Fields
Descriptions

Empl Images
Use this button to access any time card images associated with the specified employee.
Note: The Payroll Update stores the job time card image links, if present. If there are no job time card image files, then the Payroll Update will store the employee time card image.
Tip: You can press ALT + M on your keyboard to access images for this check stored in the Employee Images folder.

Employee
The employee associated with the selected detail line displays here.

Check#
The check # associated with the selected detail line displays in this field.

Job Image
Use this button to access any job time card images associated with the job referenced on the list box. If multiple jobs are displayed, select the one you want to view from the available list box.
Links to job time card images are also available on the [Job Cost History Report](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-reports/job-cost-history-report).
Note: The Payroll Update stores the job time card image links, if present. If there are no job time card image files, then the Payroll Update will store the employee time card image, if found.
Tip: You can press ALT +J on your keyboard to access images for this check stored in the Job Images folder.
