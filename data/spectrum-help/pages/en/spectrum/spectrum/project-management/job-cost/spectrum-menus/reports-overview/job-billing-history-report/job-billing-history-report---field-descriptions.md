# Job Billing History Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Job Invoice Division
Enter the job number, invoice number, and Division of the jobs to include on this report.

From G/L date
To G/L date From invoice date To invoice date
Enter the first and last General Ledger date and Invoice date to include on this report.
Click the date calculator icon to optionally add a date calculation to the report filter parameters.

Combine sub-job activity into master job?
Select this checkbox to include transactions for sub-jobs if the job code specified above is a master job.
