# Copy Phase Categories Across Companies - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Copy from

From company
The current company code will be used to copy the phase information.

Phase category
Enter a phase category or click the arrow to open the Search Phase Categories
 window.

Copy to

To company
Enter the three-digit code of the company to which to copy phase information, or click the arrow to open the Search Companies window.

Overwrite existing setup?
Select to overwrite the existing information in the standard phase file in the destination company. If selected, standard phases and categories will only be set up in the companies where the current operator has security access to add codes manually into the [Phase Category Maintenance](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/phase-category-maintenance) screen.

Continue/Cancel
Select Continue to complete the update process or Cancel to exit without saving your changes.
