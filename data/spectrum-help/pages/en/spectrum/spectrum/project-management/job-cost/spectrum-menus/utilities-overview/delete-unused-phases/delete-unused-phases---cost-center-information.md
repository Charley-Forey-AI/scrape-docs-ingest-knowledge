# Delete Unused Phases - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when performing the Delete Phases update Spectrum will remove the phase only if you have permission to access the phase's information. Spectrum compares the cost center assigned to the job with cost centers in your assigned scheme; if the cost center is not included, then phases are not deleted on that job. In addition, Spectrum compares the cost center assigned to each phases, if applicable, with cost centers in your assigned scheme. If the cost center is not included, then the phase is not deleted.
If a job does not have a cost center, this utility will only automatically change records that the operator is allowed to manually access. In these instances, the operator must open Jobs and edit the job accordingly.
