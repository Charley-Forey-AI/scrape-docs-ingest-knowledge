# Cost Type Summary Report - Cost Center Information

If the cost center feature is enabled in the
 Enterprise Installation screen, the starting screen includes a
 selection for a cost group. When a cost group or cost center is specified, then the report
 will show only jobs assigned to cost centers in that group (based on the cost center recorded
 in the Jobs > Edit window).When you specify a cost center on the starting screen, Spectrum
 verifies that you have permission to access that cost center before proceeding. When printing
 the Job Cost Type Summary Report, Spectrum will include a job only if you have permission to
 access the job. Spectrum compares the cost center assigned to the job with cost centers in
 your assigned scheme; if the cost center is not included, then the job is not shown. In
 addition, Spectrum compares the cost center assigned to each phase, if applicable, with cost
 centers in your assigned scheme. If the cost center is not included, then that phase's
 activity is not shown.
