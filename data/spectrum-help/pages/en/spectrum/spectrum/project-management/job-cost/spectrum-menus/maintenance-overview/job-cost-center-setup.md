# Job Cost Center Setup

Use this screen to apply new cost center codes to jobs that currently use the cost centers feature.
Note: This screen displays only if the cost center feature
 is enabled in the Enterprise Installation screen
Fields
Descriptions

Job number
Enter the number of the job to be included on this screen, double-click on
 this field or press F4 to select from a list of available job numbers, use the
 SuperSelectoptions, or press Enter to accept the system default of ALL.

Current cost center
Enter the number of the cost center currently in use to be included on this
 screen, double-click on this field or press F4 to select from a list
 of available cost center numbers, use the SuperSelect options, or press
 Enter to accept
 the system default of ALL.

New cost center
Enter the number of the new cost center to be included on this screen, use the
 drop-down menu to select from a list of available cost centers, or select
 <no
 change> to view jobs whose cost centers have not changed.

Job
The job number displays. No entry is allowed.
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

Description
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

Division
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

State
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

Old cost center
Click the column header to arrange the data in ascending order; click the column header again to arrange the text in descending order. Use your mouse to determine the width of this column by sliding the column title's side borders.

New cost center
Enter the number of the new cost center to be included on this screen, use the
 drop-down menu to select from a list of available cost centers, or select
 <no
 change> to view jobs whose cost centers have not changed.
