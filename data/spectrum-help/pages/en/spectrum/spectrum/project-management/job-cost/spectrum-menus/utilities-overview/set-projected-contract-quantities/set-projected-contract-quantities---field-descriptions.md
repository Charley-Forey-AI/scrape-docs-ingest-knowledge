# Set Projected Contract Quantities - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Selections

Job
Enter the job number of the unit price job that you want to include in this update, or double-click in this field to select from a list of available jobs.

Phase
Enter the phase number that will automatically be set up to use projected contract quantities when the Contract Status Report is run.

Cost type
Enter the cost type associated with the phase that will automatically be set up to use projected contract quantities when the Contract Status Report is run. If a cost type is setup in the Job Cost Installation screen, this cost type will display here as the default.
