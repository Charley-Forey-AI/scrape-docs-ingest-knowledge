# Job Cost Analysis Report

You can preview the Job Cost Analysis Report with or without projected costs. This data comes from the Job Analysis Inquiry screen. The information for each job displays in sequence by phase code.
If the Include open commitments and Include A/P invoices in progress checkboxes are selected in the [Job Cost Installation - Report Defaults](/en/spectrum/spectrum/project-management/job-cost/installation-overview/job-cost-installation---report-defaults) screen, subcontract invoice amounts will be omitted.
If subcontract billings are routed through the approval process, then
 this report should be run with the Include
 open commitments checkbox clear. This will show the unapproved subcontract
 billings in the job-to-date costs, and the open commitments will display in a separate
 column on the report.
