# Error - Equipment Code Entered Is Not Available For Use

If a retired equipment code is entered, this message displays. This error message disallows further data entry.
To change the equipment's status, clickEquipment Control >  > Maintenance >  >  > Equipmentto open the Equipment File Maintenance screen. In the Status field, enter a different status code, or press F4 or double-click on this field to select from a list of available equipment status codes.
