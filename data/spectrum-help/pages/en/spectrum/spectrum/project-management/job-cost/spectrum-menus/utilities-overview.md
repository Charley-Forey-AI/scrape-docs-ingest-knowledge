# Utilities Overview

On the Site map, click on the Job Cost folder, and then click on the Utilities folder to access the Utilities menu selections discussed below.
Selections from this menu are used to copy Standard phase file information across companies; set job group range to match the Job Cost Installation screen; set the earning calculation method; reset the OverheadBatch code in the job file; reset the current Phase estimate to the original estimate; rebuild the major and minor groups; delete any unused phases; set the price method in the phase file; and generate a phase description exception report.
