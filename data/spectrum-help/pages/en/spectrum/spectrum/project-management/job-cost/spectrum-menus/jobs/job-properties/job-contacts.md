# Job Contacts

Use this screen to view the existing key personnel and an unlimited list of contacts associated with the job.
Fields/Buttons
Descriptions

Organizations
Click to open the Other Organizations File Maintenance screen.
Note: This button will not display if you do not have security authorization to access that screen in the current company.

Job code
Enter the number of the job you want to view.

Key personnel

Edit Key Personnel Superintendent Estimator Project manager
The Superintendent, Estimator, and Project manager display. The Employee code will also display if the Utilize employee code for job personnel checkbox is selected in the Job Cost Installation screen.
Click Edit Key Personnel to select a different name. This button is only available if you have security authorization.

Job contacts

Add Contact
Click to open the Search Contacts window and select a contact. You can select multiple contacts by highlighting the contact with the mouse and pressing Ctrl or Shift.

Edit
Click to open the Edit Contact window for the selected contact.

Remove
Click to remove the contact associated with this job.

E-Mail
Click to launch your email application and send a message to the selected contact.

Status
Click to determine which contacts to display: Active, Inactive, or Both.
