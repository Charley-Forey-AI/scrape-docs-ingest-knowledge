# Billing Item Analysis Report - Cost Center Info

If the cost center feature is enabled in the Enterprise Installation screen, when printing the Billing Item Analysis Report Spectrum will include activity only if you have permission to access the job. Spectrum compares the cost center assigned to the job with cost centers in your assigned scheme; if the cost center is not included, then the job will not display. Activity for all billing items and phases of authorized contracts is included on the report.
For example:
Job 503 = cost center 102
contracts 250, 503 = cost center 101
->Operator with cost center 102 only sees the job on the report.
