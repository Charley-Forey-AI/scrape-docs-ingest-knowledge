# Projected Cost Entry (by Phase summary)

Use the Projected Cost Entry screen to enter information from the project manager's Projected Cost Worksheet for both fixed price and unit price jobs. All the necessary phase information displays on one screen, where you can view and perform projections.

- Select the Calculations button to open the [Projected Cost Calculations window](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-calculations-window) window and specify which fields are to be recalculated.

- If a job is selected that is not already built, the software opens the Build Projected Cost Transactions for Job window to quickly build the specified job using the saved selections from the Build Projected Cost Transaction File Starting screen.

As each row is modified, the software stores projected costs entered in Phase summary by phase + cost type in the Projected Cost Transaction Table.
