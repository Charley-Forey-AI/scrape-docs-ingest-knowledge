# Cost Transaction Report Update

This update records cost transaction data in Job Cost History and Time + Material for billing. Do not continue until you have carefully previewed the Cost Transaction Report. Once it is accurate, save a copy and click Continue to update the report.
Balanced entry protection is incorporated into this function to ensure the General Ledger will stay in balance. If debits do not equal credits (such as due to a power failure or data error), Spectrum will display a screen message and disallow update until the problem is corrected.
Consult the support desk at Viewpoint for instructions if it is necessary to correct an out-of-balance error.
Once you successfully update the report, the [Job Cost G/L Detail Report](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/job-cost-transaction-entry/job-cost-gl-detail-report) displays.
