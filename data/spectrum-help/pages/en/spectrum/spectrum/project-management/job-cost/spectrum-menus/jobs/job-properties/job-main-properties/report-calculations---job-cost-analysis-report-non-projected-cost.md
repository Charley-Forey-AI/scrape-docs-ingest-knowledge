# Report Calculations - Job Cost Analysis Report (Non-Projected Cost)

The following table describes how each column is calculated on this report.
Field
Description

Committed Cost
The Committed cost figure displays from the Job Phases screen.

Actual Cost
The actual job cost amount for the selected period, as well as the job-to-date amount, displays for each phase and Cost type. These figures are accumulated balances of transactions updated through the year and period requested.

Estimate
The current Estimate amount displays from Job Phases.

Variance
Variance is calculated as the difference of the two preceding columns - that is, the actual job-to-date cost less the estimated cost.

% Var
Percent variance is calculated as the quotient of the two preceding columns -- that is, the variance amount divided by the estimated cost, multiplied by 100.

% Comp
The percent complete is calculated as the actual job-to-date figure divided by the estimated cost, multiplied by 100.

Cost Per Unit
The cost per unit is calculated as the actual job-to-date figure divided by the number of job units from Jobs.
