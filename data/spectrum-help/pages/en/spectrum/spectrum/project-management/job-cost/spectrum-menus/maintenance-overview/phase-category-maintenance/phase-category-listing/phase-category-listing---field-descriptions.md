# Phase Category Listing - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Phase category
Enter a phase category, or click the arrow to open the Search Phase Categories window. The search window does not show company-wide phase categories.

Include company-wide settings?
Select to include phases that are not assigned to specific jobs. The company-wide categories will display in the report first.
