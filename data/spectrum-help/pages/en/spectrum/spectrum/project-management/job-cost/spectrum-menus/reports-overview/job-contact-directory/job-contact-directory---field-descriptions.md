# Job Contact Directory - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Job
Enter the job code in this field.
If job-based security is enabled in Job Cost Installation or cost centers have been set to 'Yes' for the current company, you must have security authorization for the requested job.

Sort by
Select whether to sort by the contact's Last name or Organization name.
