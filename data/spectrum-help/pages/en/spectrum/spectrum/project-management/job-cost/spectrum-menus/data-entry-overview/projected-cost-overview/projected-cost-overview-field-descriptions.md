# Projected Cost Overview Field Descriptions

Use the table below for reference when completing fields on
 this screen.
Fields/Buttons
Descriptions

Build
Click this button to open the[Build Projected Cost Transaction File](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/build-projected-cost-transaction-file) .
Note: this screen is only available if
 you have security authorization.

Calculations
Click to open the [Projected Cost Calculations window](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-calculations-window).

Worksheet
Click to open the [Projected Cost Worksheet](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-worksheet).
Note: This screen is only available if
 you have security authorization.

Update
Click to open the [Projected Cost Report](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-report). This
 option is only available from Projected Cost Entry by Phase (instead
 of by Group).
Note: this screen is only available if
 you have security authorization.

Job
Enter or search for the Job number you want to
 view.
In the Build Projected Cost Transaction File
 screen, If Phase was selected, the Projected Cost Entry screen
 will display by [Projected Cost Entry (by Phase summary)](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-entry-by-phase-summary).
If Major or Minor group was
 selected, the screen will display b y[Projected Cost Entry (by Group)](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-entry-by-group)
 .
