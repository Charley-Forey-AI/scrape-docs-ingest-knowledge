# Expand Phase Code for Job - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Job
Enter the job code for the phase you want to change. Jobs with a completed status will not be allowed.

Current phase setup

Phase mask
The existing phase mask format for this job displays.

Major group start position Major group end position Minor group start position Minor group end
The major group start, major group end, minor group start, and minor group end positions for this job display.

New phase setup

If the job specified in the heading is a sub job, the fields in this section will be hidden. An exception to this rule is if a particular field in the sub-job is different from the master job, that field will be available to edit and make the same as the master job.

Phase mask
Enter the new phase mask. The system prevents entry of a new phase length that is the same or shorter than the current setup, or longer than 20 characters.
If the current job is a sub-job and the master job's phase mask is less than or equal to that of the sub-job, changes will be disallowed. If the master job's phase mask is greater than the sub-job, the sub-job phase mask can be expanded to match the master.

Major group start position Major group end position Minor group start position Minor group end position
Enter the new major group start, major group end, minor group start, and minor group end positions for this job.
