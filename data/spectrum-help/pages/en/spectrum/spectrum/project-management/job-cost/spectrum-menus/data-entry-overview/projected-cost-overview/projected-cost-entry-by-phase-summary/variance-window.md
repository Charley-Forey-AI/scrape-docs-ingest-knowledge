# Variance window

This window displays the variance for the selected phase line based on the variance of the current projection (at completion) and the current estimate and last projection.
When this window is accessed in context to a job built for 'Phase summary,' only Dollar variance summed across all selected cost types for the current phase and Quantity variance for the quantity cost type will be displayed.
