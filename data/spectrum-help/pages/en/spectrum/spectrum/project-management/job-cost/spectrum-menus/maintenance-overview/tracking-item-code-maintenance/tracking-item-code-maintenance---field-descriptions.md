# Tracking Item Code Maintenance - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Item code Description
Enter an item code and description for the tracking item.

Trigger basis
Select a trigger for this tracking item:

- Date of expiration

- Yes/No completion

- Recurring Yes/No completion

If recurring is selected, select an interval type (weeks or months) and specify the number of weeks or months per interval.
