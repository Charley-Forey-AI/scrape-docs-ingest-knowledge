# Phase User-Defined Fields Maintenance - Installation Options and Results

Use the table below for reference on configuring the Installation screen for use with this screen.
Installation tab checkbox
Option
Result on Phase User-Defined Fields screen

Show non-company-specific fields (company ALL)
Select

- User-defined fields for all companies can be added and edited in this screen.

Clear
Only user-defined fields associated with the current company can be added, edited, and viewed.

Allow setup of company-specific fields
Select
Company-specific user-defined fields can be set up for the current company.

Allow setup of non-company-specific fields (company ALL)
Select
You can set up user-defined fields for all companies.

Clear

- You cannot set up user-defined fields that are not company-specific.
