# Installation Overview

To access the Company Installation screen, select the System Administration  option the toolbar of any menu; the Installation menu will appear.
On the Installation menu, select Company Installation.
If this screen has not yet been completed, contact your System administrator.
Ensure that the Job Cost checkbox has been selected.
After ensuring that the Job Cost module is selected for this company, select OK to save the information and return to the Installation menu.
