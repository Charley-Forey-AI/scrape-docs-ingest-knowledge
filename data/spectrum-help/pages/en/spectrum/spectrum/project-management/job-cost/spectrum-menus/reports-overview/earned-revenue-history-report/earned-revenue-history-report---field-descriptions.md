# Earned Revenue History Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
FieldDescription
JobEnter the job number to include on the report, or press Enter to include ALL jobs on the report.Note: If cost centers are enabled for the current company, the user must have access to the selected job to include it on the report.

Through period end dateEnter the period end date for the report. The default is the current processing date for Job Cost. The fiscal year and period from the selected date will display to the right of this field.
Job statusEach checkbox is selected by default. Use these to filter your report output.
