# Set Earned Revenue Calculation Method

This screen is used to determine and set the preferred
 method for earnings calculation in Jobs based on entries on the starting screen of this
 update. During the update, the calculation method stored in the Job Contract
 Status window will be reset.
CAUTION: Extreme caution should be exercised when performing
 this update. It may affect how calculations are subsequently performed on the Contract
 Status Report.
