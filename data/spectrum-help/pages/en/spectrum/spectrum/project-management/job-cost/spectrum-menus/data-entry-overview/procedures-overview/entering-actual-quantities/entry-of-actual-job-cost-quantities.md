# Entry of Actual Job Cost Quantities

Job Cost quantities can be entered into Spectrum through various methods:

- Payroll Pre-Time Card Entry

- Payroll Pre-Time Card Quantity Import

- Payroll Time Card Entry

- Job Cost Quantity Entry
Regardless of the method used to entry the actual quantities, the end result is that they are updated to the Job Cost Quantity Transaction History file (JC.QHIST), job phase file (JC.PHSE), and Job Cost Balance file (JC.CBAL).

On the Job Cost Installation screen, there is a selection option to Enter additional quantity. This is asking whether you want to enter a new job-to-date complete quantity, or enter the incremental quantities completed. This setting is global for each company, and cannot be set on a job-by-job basis.
