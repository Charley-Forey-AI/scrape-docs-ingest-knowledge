# Projected Cost Overview

Use the Projected Cost Entry screen to track cost-to-complete in the various phases of a project. The project manager, who supplies figures for completion of each phase, can print a worksheet. This information is then used to track job costing during the project, and to compare estimated cost with projected cost at completion.
When the Group option is being used, the Update button does not display. Use the Phase format to verify information in detail, and to archive the detail report.
The Projected Cost Entry screen version that displays depends on the Group summary option you select during the Build Projected Cost Transaction File build.
If you enter a job code that has not been built yet , the Build Projected Cost Transactions for Job window displays, allowing you to quickly build a job. This window will display if you have security for the [Build Projected Cost Transaction File](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/build-projected-cost-transaction-file).
