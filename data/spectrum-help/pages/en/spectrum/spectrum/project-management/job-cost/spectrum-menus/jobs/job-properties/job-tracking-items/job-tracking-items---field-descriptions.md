# Job Tracking Items - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Job
The selected job code will default in this field.

Tracking Item
Enter a job tracking item to add to the selected job.

Trigger basis
The trigger basis for the selected tracking item will display in this field. No entry is allowed.
If 'Recurring' was selected, specify a start and end date for this trigger.
