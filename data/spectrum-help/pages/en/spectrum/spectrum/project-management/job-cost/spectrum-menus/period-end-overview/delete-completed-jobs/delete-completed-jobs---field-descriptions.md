# Delete Completed Jobs - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Line
Line numbers are assigned and maintained by the system.

Job Description
Enter a valid job number. The job must have both a status of Complete and an actual completion date entered in Jobs. The selected job description displays to the right of this field.

Complete date
The job's actual completion date displays.
