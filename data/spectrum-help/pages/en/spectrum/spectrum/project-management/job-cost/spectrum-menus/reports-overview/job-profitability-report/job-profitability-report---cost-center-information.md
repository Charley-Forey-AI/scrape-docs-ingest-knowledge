# Job Profitability Report - Cost Center Information

If your company is using cost centers, a Cost group field displays beneath Through period end date.
When a cost group or cost center is specified, the report will show only jobs
 assigned to cost centers in that group (based on the cost center recorded in the Jobs > Edit window). If a default cost group or cost center is specified in Cost
 Center Scheme Maintenance, that code will default in this starting screen.
When you specify a cost center on the starting screen, Spectrum verifies that you have permission to access that cost center's information before proceeding.
When previewing the Job Profitability Report, Spectrum includes a job only if you have permission to access the job. Spectrum compares the cost center assigned to the job with cost centers in your assigned scheme; if the cost center is not included, then the job does not display.
Activity for all phases of authorized jobs is included on the report.
If cost center entities are being used, this report displays the Entity name in the heading of the report whenever all cost centers included in the selected 'Cost group' for the report are assigned to the same entity.
