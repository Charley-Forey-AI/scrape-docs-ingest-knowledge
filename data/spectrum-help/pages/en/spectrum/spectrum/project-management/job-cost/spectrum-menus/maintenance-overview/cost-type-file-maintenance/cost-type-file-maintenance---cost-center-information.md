# Cost Type File Maintenance - Cost Center Information

When adding or changing a cost type code, you can enter a G/L account code only if you have permission to assign that code. Spectrum compares the G/L account's list of shared cost centers with cost centers in your assigned scheme. If there are no common cost centers, then that G/L account cannot be assigned.
Once a G/L account code has been assigned to a cost type by an authorized user, then all users can access that G/L account for processing with the cost type code.
