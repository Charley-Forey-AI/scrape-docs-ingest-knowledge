# Overhead Allocation Update

The Overhead Allocation Update screen is used to allocate additional overhead to specific jobs. In Jobs, a batch code is assigned to every job. The batch code is then used to group jobs; this update can charge overhead to one or more groups of jobs, based on the specified batch code(s). Up to 99,999 overhead transactions can be applied per job per update.
This update will enter your overhead transactions into Job Cost Transaction Entry; it will be necessary to run the Job Cost Transaction Report in order to complete the process. After running the Job Cost Transaction Report, run the Job Cost Update to complete the process.
