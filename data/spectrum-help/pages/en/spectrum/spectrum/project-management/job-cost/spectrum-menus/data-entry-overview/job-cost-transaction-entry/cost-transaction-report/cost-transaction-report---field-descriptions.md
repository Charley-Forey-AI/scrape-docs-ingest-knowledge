# Cost Transaction Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

From date To date
Enter the first and last transaction dates to include on the report. You can use date shortcuts.

Batch
Press Enter to
 accept the default, or enter or search for a new batch code.

Sort by

- Select Transaction type/Reference to sort the report in sequence by reference.

- Select Transaction type/Job to print the report in sequence (within the transaction type) by job number.

Transaction type
Select one or more transaction types to include in this update.

- Accounts Payable

- Equipment

- Inventory

- Job Cost

- Payroll

If you select Inventory, the software will update the quantity to the phase cost balance file.

Cost group
When you specify a cost group, the report will show only jobs assigned to cost centers in the specified group.
Note: This field displays only when the Enterprise Installation
 Use cost
 centers radio group is set to Yes for this
 company.
