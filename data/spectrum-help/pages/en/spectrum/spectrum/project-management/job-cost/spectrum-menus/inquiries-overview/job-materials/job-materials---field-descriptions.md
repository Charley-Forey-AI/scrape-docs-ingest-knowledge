# Job Materials - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Job code
 Enter a job code to display materials for the selected job in the grid. If the selected job is a Master job, this field will be labeled Master job and the grid will include sub-jobs for the master.

New P.O.
 Click to display the Purchase
 OrdersPurchase Order Entry screen.

New Req
Click to display the [Job Requisition Entry ](/en/spectrum/spectrum/materials/inventory-control/spectrum-menus-overview/data-entry-overview/job-requisition-entry) screen. This option will be hidden if the
 Inventory Control module is not installed, or the operator does not have
 security to Job Requisition Entry.

Switch
Click to select a different job code.

Selections
Click to display the [Job Materials - Selections window](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/inquiries-overview/job-materials/job-materials---selections-window) window.

Job code
The selected job number and description display in this view-only field.

Details
This link will open one of two inquiry windows, depending on the 'Type' of material item selected:

- If the selected row is a purchase order, the Purchase Order
 Detail window will open.

- If the selected row is a valid job requisition, the Job Requisition
 Inquiry window will open.

- If the selected row is for requisition history and the requisition header is
 no longer present, no window will open.
