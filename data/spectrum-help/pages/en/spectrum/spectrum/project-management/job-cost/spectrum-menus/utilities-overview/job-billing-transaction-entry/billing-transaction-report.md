# Billing Transaction Report

Use the Billing Transaction Report to update the job and billing history files with the billing activity. Before the transactions are updated to the Job Cost Detail file, a report displays those transactions for verification.
You should preview the report before billing information is updated to the job files. Once the report runs, the Billing Transaction Update window displays. Select Continue to update the information from the Billing Transactions Report.
Important: This report should be retained as a permanent
 audit record of the company.
