# What if we have different markups negotiated with a single
 customer?

If we have different markups negotiated with the same
 customer for different groups of jobs, how can we avoid having to set these up for each
 job?
Answer: You can set up markups by master job. Set up master jobs for the customer with the
 correct markups. Then, when individual jobs are set up, leave the markups blank but enter
 the correct master job. This will cause the individual job to use the master job's
 markups.
