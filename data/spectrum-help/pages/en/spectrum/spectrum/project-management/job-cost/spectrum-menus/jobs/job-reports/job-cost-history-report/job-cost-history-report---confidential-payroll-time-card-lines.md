# Job Cost History Report - Confidential Payroll Time Card Lines

Learn about Confidential Payroll Time Card
 Lines

- Confidential Payroll time card lines may post as multiple lines (one for each time card line), showing up as 'PR SUMMARY' on the field company's Job Cost History Report, even if the confidential company's payroll posting is set to post by 'period end' or 'check date'. This is due to the Post to jc history in detail setting in [Cost Type File Maintenance](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/cost-type-file-maintenance) in the field company. If this option is selected, it will post in detail for each work date. When this option is NOT selected, the software aggregates employees and work dates.

- When 'Include pre-time card entries' is selected on the starting screen, this
 report can hide 'pre-time cards' for confidential employees based on the selections
 to 'Post in summary to job and equipment history' or 'Post in summary to job and
 equipment history and suppress hours' (found in [Employee Pay Rates](/en/spectrum/spectrum/accounting/payroll/employees/employee-properties/employee-pay-rates)). When these check
 boxes are selected and payroll is updated, the employee name and other details will
 not display on the report.

- Confidential Payroll time cards with Document Imaging images charged to a field company can be viewed by a user in the field company, but only if that user also has access to the Confidential Payroll company. Employees that should not be able to view Confidential Payroll time card line images should NOT have access to the confidential company.

- If in a Confidential Payroll company, time card entry is done 'by job' charging to the field company, then employees in the field company WILL be able to view the time card image because the image is stored by the job, which is in the field company.
