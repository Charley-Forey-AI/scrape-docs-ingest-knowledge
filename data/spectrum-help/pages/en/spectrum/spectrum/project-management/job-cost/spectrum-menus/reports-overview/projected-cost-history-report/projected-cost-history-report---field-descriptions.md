# Projected Cost History Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Field
Description

Job
Enter a job(s) for the contract to include on the report, or leave the field blank to include ALL jobs on the report.

Phase
Enter a phase(s) for the contract to include on the report, or leave the field blank to include ALL phases on the report.

Cost type
Enter a cost type(s) for the contract to include on the report, or leave the field blank to include ALL cost types on the report.

Date
Enter the transaction start and end dates to include on the report, or leave these fields blank to include all dates on the report.
