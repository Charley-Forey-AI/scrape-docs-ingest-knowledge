# Contract Status Report

The Contract Status Report, also
 called the WIP Report, Bonding Report, or Bank Report, can be previewed with or without
 projected costs. The report displays in sequence first by the job status of Active, then Inactive, then Complete, and then by the job number within
 each status.

- Projected formats: The projected formats of the
 Contract Status Report provide date-sensitive financial
 information for both projected costs and change order amounts. This is achieved by
 storing (by period) the change in the projected cost of a job during the Projected
 Cost Update and other screens affecting project cost including the A/P Transaction
 Update and the Quantity Complete Update. On the revenue side, any changes to the
 revenue are stored by period during both Accounts Receivable Change Order
 Log/Entry and Change Order Status File
 Maintenance. By storing these figures on a period-by-period basis, it is
 possible to backdate the projected formats of the Contract Status
 Report to present statements of contract status at various points in
 time.

- Non-projected formats: The non-projected formats of the
 Contract Status Report display using the current revised
 estimates.
