# Report Calculations - Summary Job Cost Analysis, Non-Projected

The following table describes how each column is calculated on this report.
Field
Description

Committed Cost
The committed cost figure displays from Job Phases.

Actual Cost
The actual job cost amount for the selected period.

Job-to-date
The job-to-date amount displays for each phase. These figures are accumulated balances of transactions updated through the year and period requested.

Cost Per Unit
The cost per unit is calculated as the actual job-to-date figure divided by the number of job units from Jobs. This is calculated on a line-by-line basis, including subtotals and totals.

Estimate
The current estimate amount displays from Job Phases.

Variance
Variance is calculated as the difference of the actual job-to-date cost less the estimated cost.

% Variance
Percent variance is calculated as the variance amount divided by the estimated cost, multiplied by 100.

% Complete
The percent complete is calculated as the actual job-to-date figure divided by the estimated cost, multiplied by 100.
