# Daily Procedures - As Needed

Enter cost transactions. As indicated in the following steps, Job Cost transactions will typically be performed as part of the routine procedures in other Spectrum modules.

- Enter Job Cost transactions.

- Enter equipment E/C transactions for jobs. Note that these are generally done in the source software (Equipment Control).

- Enter all vendor invoices received for jobs. Note that these are generally entered in the source software (Accounts Payable).

- Enter inventory transactions for jobs. Note that these are generally done in the source software (Inventory Control).

- Enter payroll information for all jobs. Note that these are generally done in the source software (Payroll).

- If transactions are entered in the Job Cost module, run the CostTransaction Edit List. Check the report for accuracy. If accurate, update.
OR

- If transactions are not done in Job Cost, but are entered in the respective source modules and should be updated to Job Cost, run the respective updates in the various modules.

- Enter billing transactions. Note that these are generally done in the source software (Accounts Receivable).

- If billing transactions are entered in the Job Cost module, run the Billing Transaction Report. Check the report for accuracy. If accurate, update.
OR

- If billing transactions are not done in Job Cost, but are entered in Accounts Receivable, run the Sales Journal Update.

- Make a Daily Backup
