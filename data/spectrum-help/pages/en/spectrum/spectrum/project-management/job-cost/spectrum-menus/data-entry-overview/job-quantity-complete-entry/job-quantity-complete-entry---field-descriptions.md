# Job Quantity Complete Entry - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields/ButtonsDescriptions
Header
JobEnter the job number for which quantities will be entered. The job description will display to the right of this field.
Batch codeEnter the batch or job number corresponding to the pre-time card information to be printed. Press Enter to print ALL batches and jobs entered in Pre-Time Card Entry.
Work dateEnter the work date (for example, for hours that the employee worked on the 14th, enter "14"). This field will default to the current processing date. The software will validate that the date is within the Job Cost minimum/maximum date range.
Phases to includeEnter any phase codes to include in the grid, or press Enter to include ALL.Note: This screen only includes phases with an
 Active status. If you wish to enter quantities for
 an inactive phase, use the Quantity Entry screen in
 Payroll or the Projected Cost Entry screen in Job Cost.
 Or you can change the phase status to Active before
 accessing this screen.

Cost types to includeThe cost types to include in the grid will default from the Job Cost
 Installation screen, but can be overridden.
Grid
Phase Ct DescriptionThe phase code, cost type code, and phase description displays in these fields.
UmThe unit of measure displays from the job cost phase file. For example, Labor quantities might be "hours", Material quantities might be "each", and so forth.
Est quantityThe revised estimated quantity for the phase/cost type displays in this field.
JTD quantityThe job-to-date quantity through the specified work date above displays in this field.
Additional quantityEnter any additional quantities that were completed for this phase and cost type.
New JTD quantityThe sum of the JTD quantity and Additional quantity displays in this field. If a new value is entered in this field, the Additional quantity will be recalculated.
