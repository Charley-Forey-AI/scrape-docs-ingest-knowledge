# Job Comparison Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Selections

Job 1 - 5
Up to five jobs may be compared. At each of the five job number prompts, enter the number of a job to display on the report. These jobs must have already been set up in the Jobs screen. If less than five are needed, simply press Enter to leave the appropriate job number spaces blank.

Phase
Enter the phase code(s) to include on the report.

Include group subtotals?
Select to include group subtotals on this report. The default is controlled by the Show group subtotals field on [Job Cost Installation - Report Defaults](/en/spectrum/spectrum/project-management/job-cost/installation-overview/job-cost-installation---report-defaults).

Report format

- Detail

- Phase summary

 Select Phase summary to preview each phase of the jobs being compared . Select Detail to print the report (including the Cost type details) for each phase.
