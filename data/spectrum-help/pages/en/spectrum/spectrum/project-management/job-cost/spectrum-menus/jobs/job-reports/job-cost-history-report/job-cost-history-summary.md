# Job Cost History (Summary)

Document Imaging Users: This report offers links (in blue) to associated images when previewed online. A single link is shown per transaction (even if multiple images are present for that transaction, in which case the earliest stored image file displays). If more than one image exists for a transaction,when the image link is clicked a window displays a list of the available image files.
