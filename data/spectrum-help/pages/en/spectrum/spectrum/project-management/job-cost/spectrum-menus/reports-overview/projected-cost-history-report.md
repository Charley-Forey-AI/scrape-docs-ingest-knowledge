# Projected Cost History Report

Use this report to track net changes to the contract, date/time stamps, who made changes and any messages. The report will be sorted by job, then by phase and cost type. Subtotal amounts will be provided in phase + cost type, then by phase and by job.
Note: If the Year/Period column on the report has an asterik (*) next to it, there is a date mismatch in the tables. Contact Viewpoint Support for assistance with this.
