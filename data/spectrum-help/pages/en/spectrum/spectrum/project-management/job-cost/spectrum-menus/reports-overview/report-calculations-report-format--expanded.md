# Report Calculations (Report Format = Expanded)

The following table describes how each column is calculated on this report.

## Hours

Field
Description

Reg
Regular payroll hours for this job display. This includes all hours entered as R and ER pay types during Payroll Time Card Entry.

O/T
Overtime payroll hours for this job display. This includes all hours entered as O and EO pay types during Payroll Time Card Entry.

D/T
Double-time payroll hours for this job display. This includes all hours entered as D and ED pay types during Payroll Time Card Entry.

Other
Other payroll hours for this job display. This includes all hours entered during Payroll Time Card Entry with pay types other than those displayed in the three previous columns.

Total
Total hours display as the sum of the previous hours columns.

## Earnings

Field
Description

Reg
The dollars earned for all regular hours display. If labor posts including the Burden, this figure includes burden dollars. If burden posts to a different Cost type than labor, this figure does not include burden dollars.

O/T
The dollars earned for all overtime hours display. If labor posts including the burden, this figure includes burden dollars. If burden posts to a different cost type than labor, this figure does not include burden dollars.

D/T
The dollars earned for all double-time hours display. If labor posts including the burden, this figure includes burden dollars. If burden posts to a different cost type than labor, this figure does not include burden dollars.

Other
The dollars earned for all other hours display. If labor posts including the burden, this figure includes burden dollars. If burden posts to a different cost type than labor, this figure does not include burden dollars.

Total
Total dollars display as the sum of the previous earnings columns.

Burden Total
The portion of total earnings which is the burden amount displays.
