# New Job Checklist

Required steps:
____Set up the customer in the customer file in the Customers screen (Accounts Receivable  >  Maintenance  >  Customer).
____Set up the job in the Jobs screen (Job Cost  >  Maintenance  >  Job). Remember to enter the customer code on the Other tab. This will automatically set up
 the contract for the job in Contracts. If this is a Time + Material or Cost plus project, the job will
 also be set up in the Time + Material module.
Tip: If the customer code is not set up, there is a window available to add it on-the-fly while adding the job.
____Set up the job's phases and original estimates in Job Phases (Job Cost > Data Entry > Phase).
Tip: This can also be done by importing information from the Project Setup module or by copying the phases from a similar job.
____Set up the job's projected costs to be the same as the current estimates in Projected Cost Entry. (Job Cost  >  Data Entry  >  Projected Cost)
____Review the retention, sales tax, and revenue G/L account code information in the Contracts  >  Properties window (Accounts Receivable  >  Maintenance  >  > Contract  >  Properties).
____Set up billing items for the job in the Contracts  >  Detail window.
Optional steps, depending on the project's billing method:
____If this job is going to be billed with a draw request, set up the
 first draw request in the Accounts Receivable  >  Draw Request Billing Entry screen.
____If this is a Time + Material or cost plus job, review the JobBilling Record and verify that the markups,
 add-ons, and billing rates are correct.
____If the Equipment Control module is present and standard equipment billing rates do not apply to this job, set up special billing rates for the job in Job/Equipment Rate File Maintenance.
____If the Materials Management module is present and standard hauler rates or other charges do not apply to this job, set up special contract rates for this job in Job Contracts.
