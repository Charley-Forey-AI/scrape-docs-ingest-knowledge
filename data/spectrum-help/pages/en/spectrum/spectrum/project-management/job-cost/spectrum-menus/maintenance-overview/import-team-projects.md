# Import Team Projects

When a project is created in the Viewpoint Team application, project data from Team will display in this screen and can be linked to a job in Spectrum for further use.

- Click the Create Job button to create a new job in Spectrum with the selected project created in Team. The project code from Team will default as the job code in Spectrum, but can be changed as necessary.

- Click the Link Existing Job button to find a job in Spectrum, and then add the selected project to the existing job.
