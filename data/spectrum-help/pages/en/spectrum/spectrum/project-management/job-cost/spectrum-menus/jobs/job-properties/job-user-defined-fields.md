# Job User-Defined Fields

Use this screen to assign user-defined categories to a job. User-defined categories are set up in [Job User-Defined Fields Maintenance](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/job-user-defined-fields-maintenance).
Fields
Descriptions

Field Status
Click to change the status display.

- Currently set: Select to display valid current entries, which show data in the Current setting column.

- Unused: Select to display fields that are not being used, which are blank in the Current setting column. These won't display if you do not have security authorization.

Edit
Click to modify a selected user-defined field entry.

Prompt
The title of the user-defined field displays.

Current setting
Valid user entries display.
