# Job Overview Report - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

Selections

Job Division
Enter the job number and Division to include on the report.

Superintendent Estimator Project manager
Enter the superintendent, estimator, and project manager of jobs to include on this report.

Customer
Enter a customer code to include on this report.

Through period end date
Enter the through date, press Enter to accept the current Job Cost processing date, or
 view the Date Calculator window and select a
 calculation method.

Fiscal year Period
These fields display based on the period-end date.

Use projected cost?
Select this checkbox to print the Projected cost. This checkbox defaults
 based on the Show projected
 cost field in Job Cost
 Installation.
Note: This checkbox is irrelevant for the Detail Job-To-Date
 Variance by Cost and Detail Job-To-Date Variance by Hours reports.

Include job notes? Include contract remarks?
Select these checkboxes to include [Job Notes](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-documentation/job-notes) and [Contract Remarks window](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/reports-overview/job-overview-report/job-overview-report---field-descriptions/contract-remarks-window). If multiple contracts exist, only the contract remarks for the primary contract display on the report.

Include open commitments?
Select to include open commitments in Job-to-Date costs. This checkbox defaults based on the Include open commitments field in Job Cost. The open commitment calculation will include use tax recorded on purchase orders and job requisitions.

Include Payroll in progress?
Select this checkbox to include pre-time card and time card entries.
If you use this feature and post Payroll burdens to separate phases, verify that the burden phases are added to the phase detail file before any pre-time cards are entered using these phases.
When this checkbox is selected, Spectrum computes the total payroll cost
 (including burdens) that has been entered into Payroll > Pre-Time Card Entry but has not yet been calculated or posted. This cost is added
 to the actual cost of each job.
Note
This is only an estimate and does not reflect the actual computed costs. If
 the job uses actual burdens, the burden cost is estimated using the [Job Payroll Setup](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-properties/job-payroll-setup) > Payroll burden section. In addition, if the payroll information is changed
 before the payroll is calculated and updated, the actual costs that post to
 the job will be different from what is reflected on the previously printed
 report. Estimated payroll costs are also included in the current week/month
 on this report.

Include A/P invoices in progress?
Select to include unapproved and unposted A/P invoices. Spectrum computes the total invoice amount.This amount is added to the actual cost of each job. If the invoice information is changed before the invoice is posted or updated, the actual costs that post to the job will be different from what is reflected on the report when it was previously previewed. The estimated invoice costs are also included in the current week/month on this report.

Include A/P contract labor hours with Payroll hours?
Select to include Accounts Payable contract labor hours with Payroll hours on this report.
This checkbox displays only if the Accounts Payable
 Installation screen is set up to use contract hours.
