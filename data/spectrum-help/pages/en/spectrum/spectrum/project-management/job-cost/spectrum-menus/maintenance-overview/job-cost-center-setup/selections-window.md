# Selections window

Use this window to select the type of cost center
 information you want to include on the Job Cost Center Setup screen.
Fields
Descriptions

Selections

Active only?
Clear this checkbox to display active and non-active jobs on the Job Cost Center Setup screen; otherwise, leave this checkbox selected to display only active jobs.

Job number
Enter a job number in order to limit the data results on the Job Cost Center Setup screen, press F4 or double-click on this field to select from a list of available job codes, use the SuperSelect options, or press Enter to accept the system default of ALL.
