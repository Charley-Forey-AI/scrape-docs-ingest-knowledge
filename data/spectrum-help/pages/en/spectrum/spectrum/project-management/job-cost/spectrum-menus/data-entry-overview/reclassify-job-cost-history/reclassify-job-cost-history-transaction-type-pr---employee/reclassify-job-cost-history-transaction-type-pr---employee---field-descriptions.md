# Reclassify Job Cost History: Transaction Type PR - Employee - Field Descriptions

Use the table below for reference when completing fields on this screen.
Fields
Descriptions

A check mark displays in this column for each line that has been reclassified.

Employee Name
Enter the employee code. The full employee name will display once a code has been entered.

Job Phase
Enter the job number and phase code.
Note: Entry is prevented if the phase's status is set to
 Complete.

Ct
Enter the cost type, press Enter to accept the default, or press F4 or double-click on this field to select from a list of available cost types.

Total hours
The total number of hours for this transaction displays.

Rate
Enter the rate per hour (including burden) for this employee.

Total gross
The number that displays is the total dollar amount of the transaction.

Check #
Enter the check number, or press Enter to accept the default.

Billing rate
Enter the billing rate per hour for this employee. This field only displays if entry is for a T + M job and the employee has a billing code set up in Employee Maintenance-Rates. When this entry is posted, it will be updated to T + M transaction files.

Message
Enter a brief message describing the transaction.

G/L debit
 The G/L account code and description for the debit displays. No entry is allowed if a G/L account code is specified for the cost type in Cost Type File Maintenance.
Entry is prevented if the account's status is set to Not used
Note: This must be a direct job cost code.

G/L credit
 Enter the G/L account to be credited.
Entry is prevented if the account's status is set to Not used.
Note: This cannot be a direct job cost code.

Job name Phase description
The name of the job and phase description display.
