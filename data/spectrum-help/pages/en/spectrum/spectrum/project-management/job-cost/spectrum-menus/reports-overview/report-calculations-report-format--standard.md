# Report Calculations (Report Format = Standard)

The following tables describes how each column is calculated on this report.

## Hours

Field
Description

Reg
Regular payroll hours for this job display. This includes all hours entered as R and ER pay types during Payroll Time Card Entry.

O/T
Overtime payroll hours for this job display. This includes all hours entered as O and EO pay types during Payroll Time Card Entry.

Total
Total hours display as the sum of the two previous columns; that is, regular hours plus overtime hours.

## Earnings

Field
Description

Reg
The dollars earned for all regular hours display. If labor posts including the Burden, this figure includes burden dollars. If burden posts to a different Cost type than labor, this figure does not include burden dollars.

O/T
The dollars earned for all overtime hours display. If labor posts including the burden, this figure includes burden dollars. If burden posts to a different cost type than labor, this figure does not include burden dollars.

Total
Total dollars display as the sum of the two previous columns; that is, regular earnings plus overtime earnings.

Burden Total
The burden amount displays.
