# WIP Schedule with Backlog

When running your contract status report, if you want contract values strictly based on projection contract revenue, select the Use projected contract for unit price jobs checkbox. This causes the report to provide the projected contract amount for both the Original and Revised Contract amounts.If instead you want to see the contract values based on the revised contract amount, clear the checkbox.
