# Phase Defaults window

Use this window to set the defaults for this phase code. The information entered in these fields is linked to several other screens including the Contracts, Unit of Measure File Maintenance, and Worker's Compensation Code Maintenance screens. After completing this window, a window displays to give you to opportunity to update only modified fields or to update all cost types to the new defaults.
