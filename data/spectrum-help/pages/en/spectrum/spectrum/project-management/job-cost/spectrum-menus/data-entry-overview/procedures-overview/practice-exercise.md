# Practice Exercise

Create a new job in Spectrum with several phases, and after
 entering actual costs, change the projected costs to both increase and decrease from the
 original estimate.
After each update of projected costs, check the job files to see the changes, and run the
 Contract Status Report for both estimates and projections.
