# Phase Estimates window

This window displays the original estimate, the changes to the estimate, and the current estimate amounts for the selected phase on the [Projected Cost Entry (by Phase summary)](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/projected-cost-entry-by-phase-summary) screen.
Fields
Descriptions

Original estimate
The original estimate displays from [Job Phases](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs/job-properties/job-phases).

Executed changes
The executed change amount displays based on all change requests (or change orders associated with a change request) for the phase line.
Change Requests
Click the adjacent arrow to view detailed information about [Change Requests](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/phase-estimates-window/change-requests) for the selected phase.

Approved changes
The approved change amount displays based on all change requests (or change orders associated with a change request) for the phase line.
Change Requests
Click the adjacent arrow to view detailed information about [Change Requests](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/phase-estimates-window/change-requests) for the selected phase.

Other changes
This field displays change amounts if there is a discrepancy between the revised estimate and the original estimate, plus any approved and executed changes (for example, amounts entered directly into Job Phases will display here). Any subcontractor pricing amounts on Approved and Executed change requests and orders will also be included in this amount.

Revised estimate
The revised estimate amount displays based on the current estimate from the phase file. If this amount does not match the original estimate, plus any executed changes, plus any approved changes, this other amount will display in the Other changes field.

Proposed changes
The proposed change amount displays, which includes all contractor pricing for the current phase for change requests with a proposed status.
Change Requests
Click the adjacent arrow to view detailed information about [Change Requests](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/data-entry-overview/projected-cost-overview/phase-estimates-window/change-requests) for the selected phase.
