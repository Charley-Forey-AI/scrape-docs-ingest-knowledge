# Set Job Group Range to Company Default - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, when performing the Set Job Group Range update Spectrum will reset the job only if you have permission to access the job. Spectrum compares the cost center assigned to the job with cost centers in your assigned scheme; if the cost center is not included, then the group range is not reset on that job.If a job does not have a cost center, this utility will only automatically change records that the operator is allowed to manually access. In these instances, the operator must open Jobs and edit the job accordingly.
