# Quick Job Setup Maintenance

The Quick Job Setup screen is used to set up groupings of standard jobs with phases (for example, large or small jobs) for quick access.
You can define which fields to copy from the job file (such as contract type, price, division) and phase file while defining the job code mask with the next sequential number. This provides a numbering scheme based on a job group. When a new job is being created based on a 'Quick Job Setup' template, the system always copies the 'auto-overtime' setting of the 'Copy from job'.
To create the Quick Job Setup template, enter the Template code, and then complete the fields in the [New/Edit Job Group - Properties tab](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/quick-job-setup-maintenance/newedit-job-group---properties-tab), select the [New/Edit Job Group - Job Fields Tab](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/quick-job-setup-maintenance/newedit-job-group---job-fields-tab) you want to copy from the source job to the new job, and select the [New/Edit Job Group - Phases Tab](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/quick-job-setup-maintenance/newedit-job-group---phases-tab) you want to copy from the source job to the new job.
