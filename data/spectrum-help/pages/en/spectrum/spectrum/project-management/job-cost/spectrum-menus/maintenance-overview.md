# Maintenance Overview

On the Site Map screen, click the Job Cost option and then click the Maintenance tab to access the Maintenance menu options discussed below.
The Maintenance menu is used during conversion to define standard phase and cost types, and to enter all information pertinent to existing jobs. After initial installation, maintenance menu options can be used to update this information as necessary.
The following screens can be accessed from the Maintenance menu:

- [Cost Type File Maintenance](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/cost-type-file-maintenance)

- [Import Team Projects](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/import-team-projects)

- [Jobs](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/jobs)

- Job Note Topics Maintenance

- [Job Phases](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/job-phases)

- [Phase User-Defined Fields Maintenance](/en/spectrum/spectrum/project-management/job-cost/spectrum-menus/maintenance-overview/phase-user-defined-fields-maintenance)
