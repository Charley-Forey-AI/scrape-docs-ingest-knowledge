# Import a Change Request

In addition to importing the base contract estimate, you
 can import an unlimited number of change request estimates for jobs. Estimates for change
 requests will update to Accounts Receivable > Change Request Log/Entry.
Note: You can only import change requests that have an
 action code 4. In other words, you can only import change requests when there is no
 recognition of revenue, cost, or billing.
The change request import procedure is divided into multiple sections to clarify the process.

## PART A: Importing the Estimate

1. On the Site
 Map screen, click Project Setup  >  Job Setup.

1. On the Job
 Setup screen, click the Import button.

1. On the Standard
 Job Import screen, complete the Job field and then press Enter. Note: To add a job on-the-fly, double-click the Job field. In the Search Jobs window, click the
 Maintenance button.
 In the Jobs window,
 complete the Job,
 Description, and any
 other relevant fields, and then click OK.

1. In the Estimate
 type field, select the Change request option from the drop-down menu.

1. In the Reference
 field, enter the reference number for the change request estimate or
 press F4 to select from a
 list of valid change request reference numbers. Note: To add a
 change request on-the-fly, double-click the Reference field. In the
 Change Requests for Selected
 Contract window, click the Maintenance button. Complete the
 Add Change Request
 window and then click OK
 until you return to the Standard Job
 Import screen.

1. In the Transfer
 file name field, enter the full path of the file to be imported or
 press F4 to select a valid
 path and file.

1. Select the Import
 job with bid group, Clear phase setup, Clear billing detail setup, and Price method checkboxes, if
 appropriate.

1. In the Price
 method field, select the correct price method for the job to be
 imported.

1. In the Proceed
 with import section, select Continue and then click OK.

## PART B: Review and Modify the Estimate

Project Setup has the ability to import phases with
 their estimate information.

1. On the Job
 Setup screen, press Enter through the Job, Estimate
 type, and
 Description fields to
 accept the import defaults.

1. Review the Phase, Ct,
 Description, Um, Unit cost, Quantity, Amount, Hours, Projected quantity, Billing item, Alt phase, Worker's compensation, and
 Vendor fields for each
 phase. Note: If the phase numbers do not import, open the
 Job Cost  >  Job Cost Maintenance  >  Properties  >  Phase Setup window and edit the Phase number mask to match the phase mask set up for the imported
 job and then re-import the job in Project Setup.

1. Click OK.

## PART C: Review and Modify the Change Request

Use Project Setup to import and set up your contract
 and billing detail.

1. On the Job Setup screen, press
 Enter through the Job and
 Estimate type fields to accept the import defaults.

1. Click the Billing button.

1. On the Billing Detail Setup screen, press
 Enter through the Job,
 Estimate type, and Customer code
 fields.

1. Review the billing detail information and make and necessary
 changes.

1. Click OK twice to save your changes and
 return to the Job Setup screen.

## PART D – Review and Modify the Subcontract

Use Project Setup to create A/P subcontracts complete
 with billing information.

1. On the Job
 Setup screen (Project Setup  >  Job Setup), click the Subcontract button.

1. On the Subcontract
 Setup screen, complete the Job and Estimate type fields.

1. In the Reference field, enter the reference number for the subcontract.

1. Complete the Cost
 type and Subcontract fields.

1. Review the subcontract detail information, making any necessary
 changes. Note: Enter a vendor code and subcontract number to
 create the subcontract in the Accounts Payable module. Use different vendor code /
 subcontract number combinations to set up multiple subcontracts. Not every phase
 has to be assigned to a subcontractor.

1. Click OK twice to save your changes and return to the Job
 Setup screen. Note: You are not required to
 assign a vendor to every phase. In this case, a warning window displays asking if
 you want to clear the subcontract number. Select Clear subcontract number from records with
 no vendor when you are not assigning a vendor code to every phase
 code. Select Do not clear
 subcontract numbers if you want to save your changes and plan on
 returning to finish this screen later.

## PART E – Update

1. When the phases, subcontracts, and contract have been reviewed
 and verified, click the Update button.

1. In the Select
 Estimate for Update window, complete the Job, Estimate type, and Reference fields and then click
 OK.

1. On the Change
 Request Setup window, complete the Customer and Change request fields.

1. Select the Transfer contractor pricing checkbox to transfer the contracting
 pricing information to the Accounts Receivable >  Data Entry >  Change Request
 Log/Entry >  Edit Change Request >  Change Request Contractor
 Pricing window.

1. Select the Transfer subcontractor pricing checkbox to transfer the contracting
 pricing information to the Accounts Payable >  Accounts Receivable  >  Data Entry >  Change Request
 Log/Entry >  Edit Change Request  >  Change Request Subcontractor
 Pricing window.

1. Select the Transfer revenue checkbox to transfer the contracting pricing
 information to the Accounts Payable >  Accounts Receivable >  Data Entry >  Change Request
 Log/Entry >  Edit Change Request >  Change Request Revenue
 Detail window.

1. In the Proceed
 with Update section, select Continue and then click OK until you return to the
 Site Map screen.
