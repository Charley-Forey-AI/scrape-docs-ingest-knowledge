# Cost Types window

Enter the Spectrum cost types that should be assigned to Hard Dollar costs when importing.
For each cost type, press Enter to accept the default defined in the Job Cost Installation screen. After a cost type code has been entered, the cost type description will display from the Cost Type File Maintenance screen.
