# Create an Export File Within Timberline (PE Method)

This Project Setup module can be used to link a
 non-Spectrum estimating program to the Spectrum Construction Management software.

- At main menu, select PE for Produce Estimate.

- The system will prompt Create new estimate file? Enter No.

- The system will prompt Name of this estimate? Enter estimate
 name.

- Next, select IF for Interface Options.

- Then, select GJ
 for Generate Estimate to Job Cost.

- The Generate to Job Cost Options screen will appear:

- Job #: ___________ (enter the Spectrum job number)

- Transaction Date: _____________(enter the date to be associated
 with the transactions generated)

- Reference Number: ____________

- (enter C
 for Job "C"ost category to use amounts for Labor, Materials, Sub, Equipment and Other
 (columns) from the spreadsheet)

- Description: _____

- (enter C
 to use the Job Cost (C)ategory Descriptions)

- Is Estimate a Change Order?: ____ (enter No)

- Round Quantities?: ____ (enter No)

- Allocate Add Ons?: ____ (enter No)

- Generate Burden Entries: ____ (enter No)

- Optional User Field 1: _____ (enter B for Blank)

- Optional User Field 2: _____ (enter B for Blank)

- The system will prompt: "Select where to print the report."Enter
 F for (F)ile. If desired,
 you may also print the export file to a printer to review information. If printing the
 export file to a printer, respond No at the Change Print
 Style prompt. The phase descriptions may be blank on the printout but
 will be contained in the export file.

- The system will prompt: "Print file name?" Enter estimate name.

- The file is now ready for import by Spectrum Project Setup.
