# Project Setup Installation for Hard Dollar - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Estimating method
Select the Hard Dollar option to
 indicate that the estimating package being used is Hard Dollar's software. A
 drop-down list is available for viewing all source-software options. After Hard
 Dollar's estimating software is specified, additional entry fields will appear.
Properties

Default tax cost type
Enter the Spectrum cost type code
 corresponding to the Hard Dollar tax information. After you press Enter, the full cost type
 description displays from the Cost Type File Maintenance screen. Entry in this
 field is optional; it simply assigns a default.
Default rental cost type
Enter the Spectrum cost type code
 corresponding to the Hard Dollar rental information. After you press Enter, the full cost type
 description displays from the Cost Type File Maintenance screen. Entry in this
 field is optional; it simply assigns a default.
Default quantity cost type
Enter the Spectrum cost type code
 corresponding to the Hard Dollar quantity information. After you press
 Enter, the full
 cost type description displays from the Cost Type File Maintenance screen.
 Entry in this field is optional; it simply assigns a default.
Default labor burden cost type
The default labor burden cost type
 displays, but may be overwritten.
Default supplies cost type
The default supplies cost type
 displays, but may be overwritten.
Default other cost type
The default other cost type displays,
 but may be overwritten.
Billing item mask
Enter the default mask number for the
 billing item.
