# HCSS Heavy Bid Create Historical Costs - Field Descriptions

This update screen is used to create historical costs in ASCII file format that can then be imported into Heavy Construction Systems Specialists, Inc. (HCSS) system.
Fields/Buttons
Descriptions

Cost Types button
This window displays the cost type
 codes. These cost types are based on settings in the Project Setup Installation
 and Job Cost Installation screens.
Job
Enter the number of the Spectrum job to
 be transferred back into the estimating software, or press Enter to accept ALL jobs.
Priority quantity cost type
Enter the priority quantity cost type,
 or press Enter to
 accept the system default from the Project Setup Installation screen.
Labor cost type
Enter up to five labor cost types, or
 press Enter to
 accept the system default. This field defaults based on the Labor cost type field in Job
 Cost Installation.
Set activity field to

- Select Phase to set the phase code in the historical file to the
 Spectrum phase.

- Select Alternate phase if you want to set the phase code field
 to the Spectrum alternative phase.

- Select Import to set the phase code based on the phase code
 imported. To determine the imported phase, the Project Setup Standard
 Phase File is used to match the estimating phase to the Spectrum phase
 code.
