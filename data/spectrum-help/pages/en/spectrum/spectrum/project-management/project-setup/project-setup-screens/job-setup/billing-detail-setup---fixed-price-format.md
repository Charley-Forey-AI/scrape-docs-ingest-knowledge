# Billing Detail Setup - Fixed Price Format

All jobs are initially imported into the Billing Detail
 Setup screen if applicable data exists.
This screen may be used to verify imported data before proceeding with the Update Project
 Setup and shows details associated with the specified estimate only. If billing detail
 information was not included in the import, it may be entered using this screen prior to
 executing the Update Project Setup.
Note: You can change the format in the Price method section in the [Import Standard Job](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/import-standard-job) screen.
