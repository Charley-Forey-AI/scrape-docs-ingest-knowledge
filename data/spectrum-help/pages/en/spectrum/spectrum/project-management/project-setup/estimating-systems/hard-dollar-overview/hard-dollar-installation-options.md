# Hard Dollar Installation Options

The following information should be noted prior to completing the Hard Dollar installation.

- The estimating method should say Hard Dollar.

- The default transfer file should specify the full path and file
 name for the Hard Dollar File that will be used to create the job setup worksheet. For
 example: A:\BUDGET.TXT

- Default tax cost
 type: Enter the cost type. This may be different than the material cost
 type if the user wishes to see this as a separate line item in the phase file. If the
 user does not want to separate the tax from other phase line items, just enter the same
 cost type for tax as you specified for material on the Job Cost Installation
 screen.

- Default rental cost
 type: Enter the cost type. This information will be used during the
 import to create phase line items for "Rental".

- Default quantity cost
 type: Enter the cost type. This information will be used during the
 import to record the "Forecast quantity" for each activity code. We normally recommend
 the labor cost type.

- Default labor burden
 cost type: Enter the cost type. This may be different than the labor cost
 type if the user wishes to see this as a separate line item in the phase file. If the
 user does not want to separate the labor burden from the direct labor, just enter the
 same cost type for the burden as specified for labor in the Job Cost Installation
 screen.

- Default supplies cost
 type: This may be different than the material cost type if the user
 wishes to see this as a separate line item in the phase file. If the user does not want
 to separate supplies from other phase line items, just enter the same cost type for
 supplies as specified for material on the Job Cost Installation screen.

- Default other cost
 type: Enter the cost type. This information will be used during the
 import to create phase line items for Other.
