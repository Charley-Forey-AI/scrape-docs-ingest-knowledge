# Project Setup Timberline Overview

This Project Setup module can be used to link the
 Timberline estimating program to your Spectrum modules.
Click to select Project
 Setup from the Installation list. The Project Setup Installation screen displays. Select
 the source estimating software in use by your company. The lower portion of the screen will
 then be added.
