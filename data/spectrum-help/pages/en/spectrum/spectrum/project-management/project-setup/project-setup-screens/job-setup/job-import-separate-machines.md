# Job Import (Separate Machines)

This screen can be used to move information from McCormick to Spectrum. The system will read the user-created conversion document and import the data into Spectrum. Review the instructions in creating the conversion document before proceeding with this import.
Important: If McCormick and Spectrum Project Setup are on
 the same machine, refer to the Help file topic about "Same Machine." If McCormick and
 Spectrum Project Setup are on different computers, refer to the Help file topics about
 "Separate Machines."

- Enter the Spectrum job number to which the McCormick job data should be imported.

- Select the import options and price method that you want to use.

- Click Continue.
