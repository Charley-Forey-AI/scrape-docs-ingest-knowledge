# Subcontract Setup - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Job code
Enter an imported job number. To be
 valid, the Spectrum job must have been imported using Job Import or added in
 Job Setup. Entry in this field is required.Click on the
 job code to clear the current job and estimate and re-enter.

Estimate type
Use the drop-down menu to select the
 estimate type. If Change
 request is selected, the Reference field displays. The
 job number, estimate type, and reference number default based on the last
 estimate accessed by the user, but may be overridden.
Cost type
Enter the cost type for subcontract
 activity. The cost type defaults from the Job Cost Installation field for
 subcontract.
Default subcontract
Enter the default subcontract number to
 assign for this job. The job number defaults. Press Enter to accept the default
 or enter another subcontract number. The subcontract
 detail displays information that has been imported. Only phases for the cost
 type specified above displays. If changes are desired, press Enter to place the cursor
 on the number of the line to be modified. Subcontracts will only be created
 during the Update Project Setup for phases with both a subcontract number
 and a valid vendor code specified. It is not necessary to assign all
 subcontract phases in order to perform the Update Project Setup.


Subcontract total

Switch button
Click this button to access subcontract
 details for another job estimate.
Grid columns

Phase

Description
The phase description displays. No
 entry is allowed. To edit the description, use Job Setup to access this field.
Vendor
Enter the vendor code or use the
 drop-down list to select the vendor to be assigned the subcontract for this
 phase. When a valid vendor code is recorded in this field, the vendor name
 displays on the next line. Note: Entry is
 prevented if the vendor status is Not Used, and a warning is provided if the
 status is Inactive

Subcontract #
The subcontract number recorded in the
 heading portion of the screen displays. Press Enter to accept the default or
 enter the desired subcontract number for this phase.
Amount
The estimated amount for this phase
 displays. No entry is allowed. To edit the amount, use Job Setup to access this
 field.
Retention
Click to select this field (an X will
 appear when it is selected) if this subcontract will be subject to retainage.
 Otherwise, leave this field clear to set the retention percentage to 0.00%.
Subcontract name
Enter the subcontract name or any notes
 about this phase.
Vendor name

Buttons

Job Setup
Click this button to display the [Job Setup](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup) screen.
Billing Detail
Click this button to display the [Billing Detail Setup - Unit Price Format](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/billing-detail-setup---unit-price-format) screen.
Listing
Click this button to display the [Job Setup Report](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/job-setup-report) starting
 screen.
