# Job Setup

All jobs are initially imported into the Job Setup screen. This screen may be used to verify imported phase data before proceeding to the Project Setup Update screen.
If the import file only has a unit of measure on some of the lines, then when reviewing the job in this screen, only those phase/cost type lines show the unit-of-measure. However, when the update to job occurs, the unit-of-measure drags to subsequent cost type records for the same phase.
Note: Job Cost users may access this screen to enter jobs
 when doing manual estimating or during initial Job Cost conversion to set up phases and
 estimates. .
