# Change Request Setup Update - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job
The job number entered in the
 Select Estimate for Update window. No entry is
 allowed.
Description
The job's description displays. This
 description defaults from the Select Estimate for Update
 window. No entry is allowed.
Reference
The change request reference number
 displays. This reference number defaults from the Select Estimate
 for Update window. No entry is allowed.
Selections

Customer

Change request
Enter the change request number, or use
 the drop-down arrow to select from a list of valid change requests.
Transfer contractor pricing?
If you select this checkbox and then
 update the change request information, the contractor pricing information
 associated with this change request will populate the Accounts Receivable > Data Entry > Change Request > Edit Change Request > Contractor Pricing window and any existing contractor costs will be overwritten.
Transfer subcontractor pricing?
If you select this checkbox and then
 update the change request information, the subcontractor pricing information
 associated with this change request will populate the Accounts Receivable > Data Entry > Change Request > Edit Change Request > Subcontractor Pricing window and any existing contractor costs will be overwritten.
Transfer revenue?
If you select this checkbox and then
 update the change request information, the revenue information associated with
 this change request will populate the Accounts Receivable > Data Entry > Change Request > Edit Change Request > Change Request Revenue Detail window.
