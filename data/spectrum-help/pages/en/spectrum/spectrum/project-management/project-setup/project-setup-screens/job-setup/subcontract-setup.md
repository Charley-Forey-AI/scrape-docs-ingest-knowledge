# Subcontract Setup

All jobs are initially imported into the Subcontract Setup
 screen if subcontract data exists.
This screen may be used to verify imported data before proceeding with the Update Project
 Setup. If subcontract information was not included in the import, it may be entered using
 this screen prior to executing the Update Project Setup. The only fields that need to be
 set up in this screen are the vendor code and subcontract number for phases that need to be
 set up in Accounts Payable.
Note: Job Cost users may access this screen to enter subcontracts when
 doing manual estimating or during initial Job Cost conversion to set up subcontracts. Also,
 a phase needs to be assigned to the subcontract cost type for you to be able to enter
 information in this screen.
If any phases have not been assigned to a valid Spectrum vendor, a warning window displays once you click OK. Select the Clear subcontract number from records with no vendor option if you want to clear the subcontract numbers; otherwise, select the Do not clearsubcontract numbers option.
Important: The Update Project Setup cannot be performed if
 any phases have a vendor without a subcontract number or a subcontract number without a
 valid vendor code.
