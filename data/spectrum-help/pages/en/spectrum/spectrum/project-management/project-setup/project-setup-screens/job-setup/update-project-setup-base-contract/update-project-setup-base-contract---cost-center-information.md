# Update Project Setup (Base Contract) - Cost Center Information

You may assign phase level cost centers during the Project
 Setup import update and in the Job Setup entry screen.
Any operator who was authorized to perform this update before the cost center
 functionality was added will continue to be allowed to do so, regardless of operator
 cost center security restrictions. When cost centers are being used, the 'Job cost
 center' must be assigned in Jobs when the job is first added (before import) and is not
 part of the import file or Project Setup entry.
