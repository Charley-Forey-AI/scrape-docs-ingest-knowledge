# Standard Phase File Maintenance

Use this screen to establish relationships between phase
 codes used in your estimating package and those used by Spectrum.
If the phase codes used in your estimating software do not correspond to the phase codes set up in Spectrum, use this screen to assign a correlation between the two different numbering systems.
Use the [New, Edit, View, and Delete buttons](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/standard-phase-file-maintenance/new-edit-view-and-delete-buttons) to add or edit import phases (field descriptions for the
 New/Edit Standard Phase window are provided in the table below).
Important: This function lets you assign a phase code to any
 import phase. Any description entered here will override the import file description. Any
 job phases or cost types without estimates will only be imported if the "Include phases
 with no estimated cost, hours or quantity?" checkbox is selected in the [Update Project Setup (Base Contract)](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/update-project-setup-base-contract) screen.

## Example:

If a company with an Import phase code of 123456 wants to assign this phase code to Spectrum phase code 789 when importing the phase to Project Setup using a text file, enter 789 in the Phase code field on this screen. As a result, the phase will be 789 regardless of the job phase setup or the Job Cost Installation settings for leading and trailing zeros.
