# Project Setup Installation for IntelliBid Software Systems - Field
 Descriptions

This Project Setup module can be used to link IntelliBid,
 version 7 to the Spectrum Construction Management software.
Use the table below for reference when completing the fields on this
 screen.

1. The export uses the Average Hourly Rate calculated by
 the system. Click the Dollars radio button.

1. From the Tool bar in Summary, click the arrow in the
 Print Reports icon and select Spectrum (formerly Forefront).

1. The Report Preview window displays the export
 file.

1. Click the Export button and select "Text". The file is
 copied to Notepad. Save the file to a location on your computer for import into
 Spectrum.

Fields
Descriptions

Estimating method
Select the estimating package
 being used. A drop-down list is available for viewing all source-software
 options. After the estimating software is specified, additional entry
 fields display.
Default quantity cost type
Enter the Spectrum cost type code
 corresponding to the estimating software quantity information. After you
 press Enter,
 the full cost type description displays from the Cost Type File
 Maintenance screen. Entry in this field is optional; it simply assigns a
 default.
User-defined cost types
You can specify up to four
 user-defined cost types and descriptions for use in creating the
 historical cost file. These cost types are in addition to Labor
 (specified on the starting screen for Create Historical Costs), Material,
 Subcontract, and Cost Types #4 and #5 (specified in the Job Cost
 Installation screen). An asterisk (*) displays if entries have been
 recorded in the window. Click the drop-down arrow
 to display the User-Defined Cost Types window.

You can now use the text file you just saved to import the data into
 Spectrum.
