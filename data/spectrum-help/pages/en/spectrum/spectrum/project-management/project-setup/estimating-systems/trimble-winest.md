# Trimble WinEst

This Project Setup module can be used to link a
 non-Spectrum estimating program to the Spectrum Construction Management software.
Note: The current supported version of Trimble WinEst is 14.0.239. These
 instructions support back to version 12.
The Project Setup Installation screen is provided in two parts. First,
 specify the software that will be used for estimating. Based on this estimating method, the
 system will complete the remainder of the installation screen and allow further entry.
Select Project
 Setup from the Installation list. The Project Setup Installation screen displays.
