# Job Setup Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Job
Enter or select a valid job number.
Estimate type
Use the drop-down menu to print the
 job's Base contract
 or Change request
 data.
Reference
This field only displays if Change request was selected.
 Enter a change request reference number or select one from the available
 drop-down list.
Report type
Select the report type you want:

- Phase - prints information from Job Setup

- Subcontract - prints information from Subcontract
 Setup. If Subcontract is selected, then the following field will
 display.

- Subcontract cost type: This field only appears if
 the Subcontract report option has been selected. Press Enter to
 accept the default subcontract cost type to be printed on the
 report.

- Billing Detail - prints information from Billing Detail
 Setup.
