# Job Import (Same Machine)

This feature can be used to move information from McCormick to Project Setup. The system will automatically create the conversion document, then will read the conversion document and import the data into Spectrum.
Important: If McCormick and Spectrum Project Setup are on
 the same machine, refer to the Help file topics about "Same Machine." If McCormick and
 Spectrum Project Setup are on different computers, refer to the Help file topics about
 "Separate Machines."

- Enter the Spectrum job number to which the McCormick job data should be imported.

- Select the import options and price method that you want to use.

- Click Continue.
