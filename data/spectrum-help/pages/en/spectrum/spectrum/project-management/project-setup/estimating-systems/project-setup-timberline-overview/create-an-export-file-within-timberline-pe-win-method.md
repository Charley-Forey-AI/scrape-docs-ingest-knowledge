# Create an Export File Within Timberline (PE Win Method)

If the Timberline spreadsheet contains columns for Group
 and Phase, the phase description is transferred to Spectrum.
If the spreadsheet contains columns for Group, Phase and Item, the phase description may
 not successfully transfer when estimate amounts are associated with items rather than
 phases.

- With PE
 Win on the screen, and the estimate spreadsheet loaded and displayed,
 click to select the Interface
 pull down menu.

- From the Interface pull down menu, select Job Cost to access the transfer
 wizard.

- The Job Cost Transfer Wizard dialog box will appear, listing the
 four steps involved in creating the transfer file. Click on Step 1 to begin.

- In Step
 1, record the transfer file name, for example, ESTIMATE.JCI, the transaction
 date, the job number and the file directory. When the information displayed on the
 screen is complete, click Step
 2 to go on.

- In Step
 2, indicate whether the estimate is an original estimate or a change order.
 Also, specify whether to round quantities and whether to allocate add-ons among
 appropriate items. Then click Step
 3 to continue.

- In Step
 3, specify the desired job cost reference and the job cost description, then
 click Step 4 to go on.

- In Step
 4, click the radio button to indicate the report destination is "file", then
 click the Generate
 Transactions button to begin the compilation.
