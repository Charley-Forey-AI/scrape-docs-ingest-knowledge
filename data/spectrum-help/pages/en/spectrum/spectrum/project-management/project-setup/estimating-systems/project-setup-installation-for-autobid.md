# Project Setup Installation for AutoBid

This Project Setup module can be used to link a
 non-Spectrum estimating program to the Spectrum Construction Management software.
Note: The current Quick-Pen AutoBid version supported is version 2013.1.
 This product has been supported by Spectrum since 2004.
The Project Setup Installation screen is provided in two parts. First, specify the software that will be used for estimating. Based on this estimating method, the system will complete the remainder of the installation screen and allow further entry.
Click to select Project Setup from the Installation list. The Project Setup Installation screen displays.
To export from the Trimble AutoBid Estimating System, use the AutoBid Code Manager, available on the "Launch Excel" screen. The AutoBid Video Library has a tutorial video that explains the process.
