# Change Request Setup Update

This update is only relevant for existing Proposed (status
 type 4) change requests.
After the change request number has been recorded, this screen will
 assure that another operator is not already editing the particular contract before it will
 allow you to proceed. An error message will appear if the change request file is locked and
 the update screen will close.
