# Project Setup Installation for Hard Dollar

Click to select Project Setup from the Installation list. The Project Setup Installation screen displays.
This Project Setup module can be used to link the Hard Dollar estimating program to your Spectrum modules. The Project Setup Installation screen is provided in two parts. First, specify the software that will be used for estimating. Based on this estimating method, the system will complete the remainder of the Project Setup Installation screen and allow further entry.
