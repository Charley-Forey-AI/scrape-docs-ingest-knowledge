# Project Setup Data Conversion

Once you have installed the Project Setup module, there are a few steps you must take to complete the setup process. It is important that these steps be completed in the order listed.
Conversion Summary

- Install the Project Setup module using Project Setup Installation.

- Enter standard phase file maintenance information using Project
 Setup Standard Phase File Maintenance. It is possible to copy standard phases from Job
 Cost using Copy Standard Phases.
