# Import a Base Contract

In addition to importing the base contract estimate, you
 can import an unlimited number of change request estimates for jobs. Estimates for change
 requests will update to Accounts Receivable > Change Request Log/Entry.
Once a base contract bid has been completed in the estimating software,
 use the Standard Job Import screen to move information into the
 Project Setup module.
Important: Any job phases or cost types without estimates
 will NOT be imported.
Base contract estimates are recorded in:

- Job Cost > Jobs and Phases

- Account Payable > Subcontracts

- Accounts Receivable > ContractsThe following procedure is divided into multiple
 sections to clarify the process.

## Part A: Importing the Estimate

1. On the Site
 Map screen, click Project Setup >  Job Setup.

1. On the Job
 Setup screen, click the Import button.

1. On the Standard
 Job Import screen, complete the Spectrum job number field and then
 press Enter. Note: To add a job on-the-fly, double-click the Job field. In the Search Jobs window, click the
 Maintenance button.
 In the Jobs window,
 complete the Job,
 Description, and any
 other relevant fields, and then click OK.

1. In the Estimate
 type field, select the Base contract option from the drop-down menu.

1. In the Transfer
 file name field, enter the full path of the file to be imported or
 press F4 to select a valid
 path and file.

1. Select the Import
 job with bid group, Clear phase setup, Clear billing detail setup, and Price method checkboxes, if
 applicable.

1. In the Price
 method field, select the correct price method for the job to be
 imported.

1. Click Continue.

## Part B: Review and Modify the Estimate

Project Setup has the ability to import phases with their estimate information.

1. On the Site
 Map screen, click Project Setup  >  Job Setup.

1. Press Enter
 through the Job, Estimate
 type, and Description
 fields to accept the import defaults.

1. Review the Phase, Ct,
 Description, Um, Unit cost, Quantity, Amount, Hours, Projected quantity,  Billing item, Alt phase, Worker's compensation, and
 Vendor fields for each
 phase and make any required edits. Note: If the phase numbers
 do not import, open the Job Cost >  Jobs >  Properties >  Phase Setup window and edit the Phase number mask to match the phase mask set up for the imported
 job and then re-import the job in Project Setup.

1. Click Save.

## PART C: Review and Modify the Contract

Use Project Setup to import and set up your contract and billing detail.

1. On the Job
 Setup screen, press Enter through the Job and Estimate
 type fields to accept the import defaults.

1. Click the Billing button.

1. On the Billing
 Detail Setup screen, press Enter through the Job, Estimate type, and Customer code fields.

1. Review the billing detail information and make and necessary
 changes.

1. Click Saveand return to the Job Setup screen.

## PART D: Review and Modify the Subcontract

Use Project Setup to create A/P subcontracts complete with billing information.

1. On the Job
 Setup screen (Project Setup >  Job Setup), press Enter through the Job, Estimate
 type, and Description fields to accept the import defaults.

1. Click the Subcontract button.

1. On the Subcontract
 Setup screen, press Enter through the Job, Estimate
 type, Cost
 type, and Subcontract
 number fields.

1. Review the subcontract detail information, making any necessary
 changes.Note: Enter a vendor code and subcontract number to
 create the subcontract in the Accounts payable module. Use different vendor code /
 subcontract number combinations to set up multiple subcontracts. Not every phase
 has to be assigned to a subcontractor.

1. Click Save and return to the Job Setup screen.Note: You are not required to
 assign a vendor to every phase. In this case, a warning window displays asking if
 you want to clear the subcontract number. Select Clear subcontract number from records with
 no vendor option when you are not assigning a vendor code to every
 phase code. Select Do not clear
 subcontract numbers if you want to save your changes and plan on
 returning to finish this screen later.

## PART E: Update

1. When the phases, subcontracts, and the contract have been
 reviewed and verified, click the Update button.

1. In the Select
 Estimate for Update window, complete the Job and Estimate type fields and then click
 OK.

1. On the Update
 Project Setup screen, select the Set up subcontracts in A/P check
 box to set up the contract's subcontracts in Accounts Payable >  Subcontracts. Otherwise, leave this checkbox clear.

1. Select the Set up
 contract in A/R checkbox to set up the contract in Accounts Receivable >  Contracts. Otherwise, leave this checkbox clear.

1. Select the Set up
 billing detail in A/R checkbox to set up the contract's billing detail
 in Accounts Receivable >  Contracts. Otherwise, leave this checkbox clear.

1. In the Customer
 code field, enter the contract's customer code.

1. In the Tax
 code, enter the contract's tax code or press Enter to accept the system
 default.

1. In the Taxable field, use the drop-down menu to select whether this contract
 is taxable or press Enter
 to accept the system default.

1. In the Quantity
 cost type to create field, enter the cost type for quantities or leave
 this field blank to not create a quantity cost type for this contract.

1. Click Save until you return to the Site Map screen.
