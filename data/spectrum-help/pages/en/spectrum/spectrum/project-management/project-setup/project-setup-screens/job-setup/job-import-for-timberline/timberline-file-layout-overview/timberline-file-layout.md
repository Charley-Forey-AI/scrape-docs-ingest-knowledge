# Timberline File Layout

The Timberline transfer file is a comma-delimited ASCII
 text file (.TXT extension).
It contains three types of records: job cost header record, phase record and category
 record. The file layout for each type of record, including the maximum length, is available
 by clicking one of any of the links that follow.
