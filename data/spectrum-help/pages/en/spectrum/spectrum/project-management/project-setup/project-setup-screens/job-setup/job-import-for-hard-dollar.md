# Job Import (for Hard Dollar)

This screen can be used to move information from the Hard
 Dollar estimating software into the Spectrum Job Cost module.
The system will read the user-created budget text file and import the data into Spectrum,
 and includes total dollar amounts. The Hard Dollar lines will be summarized by phase and
 cost type. The phase description, billing item, unit of measure, and quantity will always
 come from the Hard Dollar account code. Labor and Equipment hours are imported into
 Spectrum if entered in Hard Dollar. The billing item description is split between the
 description and additional description fields.
During the import, the estimating (import) phase is cross-referenced with phases recorded in the Project Setup Standard Phase File. If the estimating phase is found, the corresponding Spectrum phase, unit of measure, and worker's compensation code will be used. If any of these fields are blank in the Project Setup Standard Phase File, data from the import file will then be used instead.
