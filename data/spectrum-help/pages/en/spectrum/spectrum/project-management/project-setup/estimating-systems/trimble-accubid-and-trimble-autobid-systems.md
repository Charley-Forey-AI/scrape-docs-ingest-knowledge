# Trimble Accubid and Trimble Autobid Systems

The Project Setup module can be used to link Trimble
 Accubid and Trimble
Autobid estimating packages to the Spectrum Construction Management
 software.
Current Accubid versions supported include v11 for Accubid Pro, and version 6 for Accubid Enterprise.
These estimating packages use the Spectrum standard import as follows:
FieldsDescriptions
Estimating methodSelect Trimble
 Accubid or Trimble
Autobid.
Default quantity cost typeEnter the Spectrum cost type code
 corresponding to the estimate quantity information. After you press Enter, the
 full cost type description displays from the Cost Type File Maintenance screen.
 Entry in this field is optional; it simply assigns a default.
Default subcontract cost typeEnter the default subcontract cost type
 code, if available. Entry in this field is optional; it simply assigns a
 default that can be overridden.
User-defined cost typesYou can specify up to four user-defined
 cost types and descriptions for use in creating the historical cost file. These
 cost types are in addition to Labor (specified on the starting screen for
 Create Historical Costs), Material, Subcontract, and Cost Types #4 and #5
 (specified in the Job Cost Installation screen). An asterisk (*) displays if
 entries have been recorded in the window. Select the drop-down arrow to display
 the User-Defined Cost Types window.
