# Example Record Layouts

The following are examples of different record layouts used in Project Setup.
Important: When entering record layouts, you must type the
 record lines directly into the text editor. If you attempt to cut and paste a line, any
 blank spaces preceding a field in the cut text will appear as blank data in the import.

## Job Record:

*, WELD CITY, 41 3RD ST, GREELY, CO, 80632

## Phase Record:

P, 201, FOUNDATION, M, 2.00, CY, 200.00, .01, 1, 20, 3.00, 0301, AMELUM, Seattle Sand & Gravel

## Revenue Record:

R, 460, Class I Conc. Pave, Class I Bit. Conc. Pave. O/G, Y, TN, 23.50, 8000.00, 188000.00, 8500.00, 199750.00, Sub this work
