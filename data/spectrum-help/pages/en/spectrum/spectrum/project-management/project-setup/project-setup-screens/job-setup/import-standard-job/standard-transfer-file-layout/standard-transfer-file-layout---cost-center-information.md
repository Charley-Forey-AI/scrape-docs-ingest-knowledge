# Standard Transfer File Layout - Cost Center Information

You may assign phase level cost centers during the Project
 Setup import update and in the Job Setup entry screen.
You may also import cost centers for billing detail entries and edit
 cost centers in the Billing Detail Setup screen and then these
 cost centers will be updated to Job CostPhase and
 Accounts ReceivableContract Detail. The
 Update Project Setup also allows you to set the contract cost
 center equal to the job's cost center.
