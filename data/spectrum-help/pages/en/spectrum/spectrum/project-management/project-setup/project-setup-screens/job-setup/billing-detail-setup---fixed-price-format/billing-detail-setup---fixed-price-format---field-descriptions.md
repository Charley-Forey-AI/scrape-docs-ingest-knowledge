# Billing Detail Setup - Fixed Price Format - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons Descriptions
Job codeEnter or select an imported job number. To be valid, the job must have been imported using Job Import or added in [Job Setup](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup). Entry in this field is required.
Estimate typeUse the drop-down menu to select the estimate type.

- If you select Change request, the Reference field displays. In the Reference field, enter an eight-digit alphanumeric reference for the change request estimate or press F4 to select from a list of valid change requests. An unlimited number of change requests can be recorded per job.

- For Base contracts, the customer specified in the Job Setup  >  Properties window will default but may be overwritten. For change request estimates, the customer code will be display-only if a change request has been recorded for the estimate.

CustomerEntry in this field is not required, but a valid customer code will be needed during the Project Setup Update in order to create a contract and record the billing detail in Accounts Receivable. The customer code defaults from the Jobs if it is already set up there.
Total contractThe total contract amount imported for this job displays.
Projected revenueThe sum of the contract amounts and projected revenue amounts of the billing items display.
Job SetupSelect to return to the [Job Setup](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup) screen.
SubcontractsSelect to go to the [Subcontract Setup](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/subcontract-setup) screen.
Listing buttonSelect to go to the [Job Setup Report](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/job-setup-report) start screen.
New/Edit/DeleteUse these [New, Edit, View, and Delete buttons](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/standard-phase-file-maintenance/new-edit-view-and-delete-buttons) to add or change records.
 Grid fields
Billing groupEnter the billing group and billing item. Press F4 to search for standard billing items for unit price jobs and billing groups.
Billing item
DescriptionThe billing item description displays. Press Enter to accept the default or enter the desired text. A second field is available to enter an additional description for this billing item.
Add'l description
Taxable?Select if this specific billing item is taxable.
UmEnter the unit of measure for this specific item. Press F4 to search for standard units of measure.
Unit priceEnter the unit price for this specific billing item. A search window is provided to set the unit price display mask. Not applicable if the unit of measure is LUMP SUM (LS).
Contract qtyEnter the contract quantity for this billing item. A search window is provided to set the quantity display mask. Not applicable if the unit of measure is LS.
Contract amountEnter the contract amount for this billing item.
Projected qtyEnter the projected quantity for this billing item, if desired. Not applicable if the unit of measure is LS.
Projected amountEnter the projected revenue amount for this billing item, if desired.
MemoEnter any memo information about this billing item, for example, the subcontractor.
