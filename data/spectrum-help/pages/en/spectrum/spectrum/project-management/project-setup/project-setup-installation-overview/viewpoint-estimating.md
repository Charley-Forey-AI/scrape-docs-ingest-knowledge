# Viewpoint Estimating

The Project Setup module can be used to link Viewpoint Estimating and Viewpoint Estimating MEP to the Spectrum Construction Management software. These estimating packages use the Spectrum standard import as follows:
FieldsDescriptions
Estimating methodSelect Viewpoint Estimating or Viewpoint Estimating MEP.
Default quantity cost typeEnter the Spectrum cost type code corresponding to the estimate quantity information. After you press Enter, the full cost type description displays from the Cost Type File Maintenance screen. Entry in this field is optional; it simply assigns a default that can be overridden.
Default subcontract cost typeEnter the default subcontract cost type code, if available. Entry in this field is optional; it simply assigns a default that can be overridden.
User-defined cost typesEnter the cost type to be used during the Create Historical Costs update.
Estimating systemEnter the name of the estimating system used.
