# Project Setup Installation for AutoBid - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Estimating method
Select the source estimating software
 in use by your company.A drop-down list is available to select the estimating
 package type being used. After estimating software is specified, additional
 entry fields will appear.
Default quantity cost type
Enter the Spectrum cost type code
 corresponding to the estimate quantity information. After you press Enter, the
 full cost type description displays from the Cost Type File Maintenance screen.
 Entry in this field is optional; it simply assigns a default.
User-defined cost types
You can specify up to four user-defined
 cost types and descriptions for use in creating the historical cost file. These
 cost types are in addition to Labor (specified on the starting screen for
 Create Historical Costs), Material, Subcontract, and Cost Types #4 and #5
 (specified in the Job Cost Installation screen). An asterisk (*) displays if
 entries have been recorded in the window. Click the drop-down arrow to display
 the User-Defined Cost Types window.
