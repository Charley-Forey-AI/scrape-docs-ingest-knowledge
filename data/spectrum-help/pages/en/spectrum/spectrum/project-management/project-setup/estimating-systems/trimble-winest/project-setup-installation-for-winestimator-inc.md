# Project Setup Installation for WinEstimator, Inc. - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Estimating method
Select the estimating package being
 used. A drop-down list is available for viewing all source-software options.
 After the estimating software is specified, additional entry fields display.
Default quantity cost type
Enter the Spectrum cost type code
 corresponding to the estimating software quantity information. After you press
 Enter, the full
 cost type description displays from the Cost Type File Maintenance screen.
 Entry in this field is optional; it simply assigns a default.
User-defined cost types
You can specify up to four user-defined
 cost types and descriptions for use in creating the historical cost file. These
 cost types are in addition to Labor (specified on the starting screen for
 Create Historical Costs), Material, Subcontract, and Cost Types #4 and #5
 (specified in the Job Cost Installation screen). An asterisk (*) displays if
 entries have been recorded in the window. Click the
 drop-down arrow to display the User-Defined Cost Types window.
