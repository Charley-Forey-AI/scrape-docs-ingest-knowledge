# Billing Detail Setup - Unit Price Format

All jobs are initially imported into the Billing Detail
 Setup screen if data exists.
This screen may be used to verify imported data before proceeding with the Project Setup
 Update and shows details associated with the specified estimate only. If billing detail
 information was not included in the import, it may be entered using this screen prior to
 executing the Project Setup Update.
If changes are desired, press Enter to place the cursor on the number of the line to be modified, then Tab. If billing items were not imported or additional billing items are needed, add the billing detail.
Note: You can change the format in the Price method section in the [Import Standard Job](/en/spectrum/spectrum/project-management/project-setup/project-setup-screens/job-setup/import-standard-job) screen. Unit price is the default option. Job
 Cost users may access this screen to enter billing detail when doing manual estimating or
 during initial Job Cost conversion to set up contract billing detail.
