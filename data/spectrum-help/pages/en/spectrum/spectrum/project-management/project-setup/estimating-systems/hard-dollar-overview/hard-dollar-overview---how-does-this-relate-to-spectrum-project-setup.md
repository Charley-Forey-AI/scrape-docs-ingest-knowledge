# Hard Dollar Overview - How does this relate to Spectrum Project Setup?

The following items ARE transferred during the Project Setup Import:

- General Overhead Items

- Work Plan Percent Add-ons

- Direct Bid Cost by Bid Item (all 5 levels of detail)

- Job Management & Expense

- Unassigned Work Plan Activities

- Indirects

- Bid Items The following items ARE NOT
 presently transferred during the Project Setup Import because there is no account
 number associated with them:

- Total unadjusted Direct Bid Cost

- Total Escalation for Direct Bid Cost

- Total Markup for Direct Bid Cost

- Total Unadjusted Overhead Bid Cost

- Total Escalation for Direct Bid Cost

- Total Escalation for Overhead Bid Cost

- Total markup for Overhead Bid Cost

- Tax Add-ons

- Directed Add Cuts

- Add Cuts - All Items

- Finance

- Bond Account

- Markup Account

- Escalation Account

All these items do not have the ability to enter an account number, so they are not transferred into Spectrum Project Setup. These items are generally overall adjustments; the Hard Dollar user will create one or more new direct cost items in the estimate to account for these adjustments. Then these will be transferred into Project Setup just like any other direct cost item.
