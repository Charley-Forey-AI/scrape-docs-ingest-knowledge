# Import Standard Job

Once a bid has been completed in the estimating software,
 this screen can be used to move information from that estimating software into the Spectrum
 Project Setup module.
The system will read the user-created transfer file and import the data
 into Spectrum.

Important: Any job phases or cost types without estimates
 will NOT be imported. During the import, the estimating (import) phase is cross-referenced
 with phases recorded in the Project Setup Standard Phase File. If the estimating phase is
 found, the corresponding Spectrum phase, unit of measure, and worker's compensation code
 will be used. If any of these fields are blank in the Project Setup Standard Phase File,
 data from the import file will then be used instead.
