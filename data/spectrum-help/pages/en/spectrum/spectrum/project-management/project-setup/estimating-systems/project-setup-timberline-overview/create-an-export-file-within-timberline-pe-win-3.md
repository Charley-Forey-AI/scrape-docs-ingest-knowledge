# Create an Export File Within Timberline (PE Win 3.0+Higher Method)

If you are going to export information to a JC system during the installation of Timberline, select (O)ther from the list of available program options.
To use the Job Cost Transfer Wizard, go to INTERFACE and select 'Job Cost.'

- "Step 1" tab: enter the file name, date, job number, and path.

- "Step 2" tab: enter the original or c.o. amount, round qty's?
 (yes/no), and allocate add-ons to the line items (this allows you to allocate the profit
 and overhead to the line items).

- "Step 3" tab: WBS (work breakdown structure) codes and ways to
 incorporate a location code into the WBS.

- "Step 4" tab: Report generation allows you to run a report of the
 information, preview, or file. It is recommended that you select the preview option to
 review the report prior to creating the file.
