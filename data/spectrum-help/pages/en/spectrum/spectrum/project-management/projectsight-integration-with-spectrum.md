# ProjectSight Integration
 with Spectrum

With this integration enabled and configured, you can sync
 project financial data between ProjectSight and
 Spectrum, keeping your records up to
 date across both applications and eliminating dual entry.

## Integration Requirements

In order to use this integration, you must have:

- Access to both Spectrum and ProjectSight.

- A Trimble Construction One
 subscription to Spectrum.
 (This integration is not available to on-premises customers.)

## Integrated Workflows

The integration currently supports the following workflows:
Project Setup

- Link Spectrum jobs
 to ProjectSight
 projects. For details, see [Link a Job from Spectrum to a ProjectSight Project](/en/spectrum/spectrum/project-management/projectsight-integration-with-spectrum/link-a-job-from-spectrum-to-a-projectsight-project).

- Import or link Spectrum
 customers and vendors to ProjectSight companies. For details, see
 [Link a Customer / Vendor from Spectrum to a ProjectSight Company](/en/spectrum/spectrum/project-management/projectsight-integration-with-spectrum/link-a-customer--vendor-from-spectrum-to-a-projectsight-company).

Budget Management

- Automatically create ProjectSight budget
 codes for each Spectrum phase code and cost type, and
 automatically create Spectrum phase codes and cost types
 for each ProjectSight budget code. Automatically
 sync modifications to budget estimates between the systems.
 For details, see [Sync Job Phase and Cost Type Records Between Spectrum and ProjectSight](/en/spectrum/spectrum/project-management/projectsight-integration-with-spectrum/sync-job-phase-and-cost-type-records-between-spectrum-and-projectsight).

Commitments

- Manually send ProjectSight
 subcontracts to Spectrum (authorized users only). For
 details, see [Send Subcontracts from ProjectSight Into Spectrum](/en/spectrum/spectrum/project-management/projectsight-integration-with-spectrum/send-subcontracts-from-projectsight-into-spectrum).

- Manually send ProjectSight
 purchase orders to Spectrum (authorized users only). For
 details, see [Send Purchase Orders from ProjectSight Into Spectrum](/en/spectrum/spectrum/project-management/projectsight-integration-with-spectrum/send-purchase-orders-from-projectsight-into-spectrum).
