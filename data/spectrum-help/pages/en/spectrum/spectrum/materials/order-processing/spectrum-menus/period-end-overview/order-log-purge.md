# Order Log Purge

The Order Log Purge screen first prints a report from the file containing the status of all sales order numbers entered in Spectrum. Information is maintained regarding the customer for whom each sales order is issued, its status, what invoice number is created, if any, and whether it has been deleted. Spectrum uses this log to prevent the operator from issuing more than one sales order with the same number.
Enter an order number or press Enter to include all order numbers, then click Preview.
Important: This report should be retained as a permanent
 record of the company.
