# Alternate Ship-To Address Listing

The Alternate Ship-To Address Listing screen prints a listing of additional customer ship-to addresses in customer code sequence. You can print addresses for some or all customers based on the parameters you specify in the screen's three fields.
Note: The area code of the inventory items shipped also
 appears on the report.
