# Invoice Profitability Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Invoice number
Enter an invoice number with which to begin the report, use the SuperSelect options, or press Enter to include ALL invoices

From/to invoice date
Enter the invoice date range that you want to include on this report.

Print status
Press Enter to accept the system default and print ALL invoices, or select to print unprinted or printed invoices only.
