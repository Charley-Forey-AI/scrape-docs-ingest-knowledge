# Invoice Edit List

This screen provides information about invoices within a specified invoice number and date range. This report can be requested for unprinted invoices in order to review accuracy before printing the invoices themselves, but printing of this report is not required prior to printing invoices. The Invoice Detail Edit List may instead be printed for only printed invoices in order to exclude those items not yet issued. Since detailed invoice information remains on file even after the invoice has been updated, this report can be printed at any time. The information on this report is available until the invoice purge is done.
There are two formats for the Invoice Edit List: the Summary format presents sales totals for each invoice; the Detail format also includes all line items for each invoice.
Export the file - Several Spectrum report starting screens off the
 ability to export files. Click the Export button, select the file format, and enter the file NAME in the
 'Export file' field. The file path is not necessary because the file save location depends
