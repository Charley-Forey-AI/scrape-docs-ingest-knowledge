# Invoice G/L Summary Update

Do not continue until the Invoice G/L Summary Report has printed correctly. Once it has printed and is accurate, continue with the update.
This update will post all information appearing on the Invoice G/L Summary Report to the General Ledger, and also updates the Inventory file.
"Balanced Entry" protection is incorporated into this function to assure the General Ledger will stay in balance. If debits do not equal credits (due to a power failure or data error, for example), Spectrum will provide a special screen message and disallow update until the problem is corrected. Consult the support desk at Viewpoint for instructions if it is necessary to correct an out-of-balance error.
