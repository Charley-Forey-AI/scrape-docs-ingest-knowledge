# Item Price Inquiry

To use this screen, the customer for whom a quote is being prepared is entered, followed by the desired item code. The screen then displays the price levels, any promotional discounts, the customer's special pricing, if any, and quantity discounts authorized for that customer. On-hand, committed, on order and back ordered quantities also display. Additionally, the quantity available and the customer's lowest price displays across the top of the screen.
Note: It is not necessary to enter a specific customer code
 to use this screen; the Customer code
 field can be left blank so that customer special pricing doesn't
 display.
