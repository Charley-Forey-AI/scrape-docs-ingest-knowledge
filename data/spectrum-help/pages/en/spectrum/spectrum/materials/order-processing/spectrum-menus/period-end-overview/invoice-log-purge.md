# Invoice Log Purge

The Invoice Log is a report from the file containing the status of all invoice numbers entered in Spectrum. Information is maintained regarding the customer to whom each invoice is issued, its status, what order number it was created from, if any, and whether it has been deleted. The system uses this log to prevent more than one invoice to be issued with the same number.
Important: This report should be retained as a permanent
 record of the company.
