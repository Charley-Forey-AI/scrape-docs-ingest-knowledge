# Order Confirmation - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Date
The date of the invoice to be created from this order defaults to the current date for the Order Processing system. Press Enter to accept the displayed date, or enter a different invoice date.

Freight
Enter the freight amount to be assessed on the invoice that will be created during this order confirmation.
