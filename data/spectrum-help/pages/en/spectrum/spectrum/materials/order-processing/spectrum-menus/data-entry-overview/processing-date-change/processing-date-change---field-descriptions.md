# Processing Date Change - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Current settings

Processing date
The current processing date displays. Enter the new
 processing date by typing over the displayed date. To leave the processing
 date without making any changes, click Cancel.
If the new date is before the Minimum Processing Date or
 after the Maximum Processing Date established during the installation
 procedures, the following error message displays:
DATE ENTERED IS NOT WITHIN THE CURRENT MINIMUM AND
 MAXIMUM DATE RANGE.

Fiscal year
The fiscal year in which the requested date is located
 displays.

Period
The period number in which the requested date is located
 (number 1-13) displays.

Period start date
The starting date of the period in which the requested
 date is located displays.

Period end date
The ending date of the period in which the requested date
 is located displays.

Allowable entry range

Minimum date
The first and last allowable processing dates display.
These dates are set by the system administrator in the
 Installation Processing Date Change screen.
