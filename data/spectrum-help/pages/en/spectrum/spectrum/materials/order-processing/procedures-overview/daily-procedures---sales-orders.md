# Daily Procedures - Sales Orders

You can enter sales orders on a daily basis by going to Site Map > Order Processing > Data Entry > Order Entry.

1. On the Site Map screen, click Order Processing > Data Entry >  Order Entry. Use the Order
 Entry screen to enter or change sales orders. The program has two
 parts. When you finish entering information on the main screen, the Order Properties window displays.
 Use this window to add or modify order, shipping, and salespeople information.

1. On the Order Entry screen, click Sales Print. Use the Print Sales Order Form screen to
 print sales orders.Note: Sales orders that have already been printed before can be
 reprinted you can print all orders entered since the last time sales orders were
 printed.

1. On the Order Entry screen, click Picking. Use the Picking Sheet Entry screen to print
 picking sheets that show what should be pulled from inventory in order to prepare
 orders for shipment.

1. On the Picking Order Print screen, click Pack List. Use the Packing List Print screen print
 packing lists to be enclosed with orders for customer's information, as needed.

1. On the Picking Order Print screen, click Ship Labels. Use the Ship Labels screen to print labels
 with order, address and carrier information on them. Shipping labels that have been
 printed before may be reprinted, as needed.

1. On the Order Entry screen, click the Confirmation button to view the
 order and indicate whether or not the order was shipped in full. If the order was
 shipped in full, an invoice is automatically created. If the warehouse was only able
 to make a partial shipment, a new order is created for the backordered amounts and an
 invoice is created for the partial shipment. Some heading information, the date of
 the invoice and freight charges may be altered during shipping confirmation.
