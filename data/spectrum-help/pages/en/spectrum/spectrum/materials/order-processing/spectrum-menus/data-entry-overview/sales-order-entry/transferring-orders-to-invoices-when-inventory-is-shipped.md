# Transferring Orders To Invoices When Inventory Is Shipped

The following procedure describes how to transfer and order
 to an invoice.

1. On theSite Mapscreen, click Order Processing >  Data Entry >  Sales Orders.

1. Complete the Sales Order Entryscreen, including
 the Order Properties and
 Order Detailswindows for
 the order you are adding.

1. Optional: Click the Print Pick List button to print a
 list of items included on the order. Once the report has printed, you will have to
 enter the order number
 again before transferring the order to an invoice.

1. Click the Confirm button.

1. In the Order Confirmation window, complete the Invoice and Default quantities confirmed
 sections as they apply to the order and click OK.

1. The Confirmation Details window displays; review each line item that will
 be included on the order for accuracy and click OK.Note: The only fields where entry is allowed are Qty ord/ship and Message. Any changes that are
 required in other fields must be recorded in the Sales Order
 Entry screen.

1. In the Order Confirmation window, select Confirm order and click OK.

1. In the Order Confirmation screen, review the data in the Confirmation section, click
 Print Invoice, and then
 click OK.

1. You will be returned to the Sales Order Entry screen after the
 invoice has printed. Click OK or click Cancel to return to the Site Map screen.The order is now an invoice and can be changed as needed in the
 Order Processing Invoice Entry screen.
