# Open Order Reports

Use the Open Order screen to print information about current sales orders. Information can be printed in various formats. The Open Order format provides sales order information on an order-by-order basis, detailing items included on each order. The Item on order format provides sales order information on an item-by-item basis, detailing each order that includes the item. The Fill by order and the Fill by customer formats provide information that can be used to make decisions about which orders should be processed/shipped when limited quantity is available on hand.
