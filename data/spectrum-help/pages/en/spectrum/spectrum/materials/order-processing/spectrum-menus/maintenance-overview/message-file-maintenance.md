# Message File Maintenance

Use the Message File Maintenance screen to establish various quick entry codes for use during Quote Entry, Order Entry and Invoice Entry. The message code can be one or two characters, either alpha or numeric. For example, TM can be created for "Our truck will deliver in your area on Monday." The message code may then be utilized during entry in the message field of a line item; by typing *TM, the system would show the full message on the quote, order or invoice. The message code is also recognized in the reference and remarks fields of quotes, orders and invoices, and during invoice printing.
