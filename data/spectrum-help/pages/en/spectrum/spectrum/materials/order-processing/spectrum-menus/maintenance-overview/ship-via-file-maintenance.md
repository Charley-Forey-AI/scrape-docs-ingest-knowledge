# Ship Via File Maintenance

Use the Ship Via File Maintenance to establish various quick entry codes for use during Quote Entry, Order Entry and Invoice Entry. The ship via code can be a single letter or number. For example, B can be created for "UPS Blue Label", or T for "Company Truck," or W for "Will Call." These codes may then be utilized during entry at the Ship Via Field by typing *W; the system will show "Will Call" on the order or invoice.
