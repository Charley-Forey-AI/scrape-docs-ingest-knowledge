# Invoice G/L Summary Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

From invoice date
Enter the date with which this report is to begin, or press Enter to include the earliest date to be posted or press F4 to display the Date Change window

Through invoice date
Enter the date with which this report is to end or press F4 to display the Date Change window .

Print button
Click the Print button to print the Invoice G/L Summary Report and display the Invoice G/L Summary Update screen
