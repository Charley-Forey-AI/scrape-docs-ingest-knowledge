# Alternate Ship-To Address Listing- Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Customer code
Enter the code of the customer who is to print on the Alternate Ship-To Address Listing, use the SuperSelect options, or press F4 to select from a list of available customer codes. Press Enter to print ALL customers, beginning with the first customer record.

Salesperson code
Enter the code of the salesperson's customers to print on the report, use the SuperSelect option, or press F4 to select from a list of available salesperson codes. If you want ALL salespersons to appear on the report press Enter.

Customer type
Enter the type of customer who is to appear on the report or use the SuperSelect options. To print ALL customer types on the list, press Enter.
