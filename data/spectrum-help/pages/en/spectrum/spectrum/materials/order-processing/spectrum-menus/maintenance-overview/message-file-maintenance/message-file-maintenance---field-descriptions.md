# Message File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
The fields in the table below display after you have selected New to create a new record, or chosen an existing record and selected the Edit button.
Fields/ButtonsDescriptions
SeqSequence numbers are assigned and maintained by the system. No entry is allowed.
NoEnter the message code.
MessageEnter the message to be used during quote, order and invoice processing.
New/Edit/Delete buttonsUse the [New, Edit, View, and Delete buttons](/en/spectrum/spectrum/materials/order-processing/spectrum-menus/maintenance-overview/customer-ship-to-addresses/customer-ship-to-addresses---field-descriptions/new-edit-view-and-delete-buttons) to add new records, make changes to existing records, or delete records.
