# Customer Ship-to Addresses

The Customer Ship-to Addresses screen is used to maintain additional customer addresses for use during Quote Entry, Order Entry, and Invoice Entry.
The address contained in the Customers screen is used for billing and mailing purposes. Additional customer addresses can be recorded here, such as additional facilities and branch offices to which merchandise can be shipped. There is virtually no limit to the number of ship-to addresses that can be entered for a customer.
