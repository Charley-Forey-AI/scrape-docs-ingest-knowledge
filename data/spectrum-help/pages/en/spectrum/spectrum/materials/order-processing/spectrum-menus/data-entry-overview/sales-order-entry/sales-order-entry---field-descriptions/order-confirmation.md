# Order Confirmation

Use the Order Confirmation screen to confirm the order entry.
If only certain users should be permitted to confirm orders, define that access at a group or user level. If needed, see [User Security Maintenance](/en/spectrum/spectrum/tools/info-link/info-link-screens-overview/user-security-maintenance).
