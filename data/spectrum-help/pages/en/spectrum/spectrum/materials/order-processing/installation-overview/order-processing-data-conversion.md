# Order Processing Data Conversion

You can learn how to complete the Order Processing module
 Installation.
It is important that these steps be performed in the order
 listed below.

1. On the Company Installation screen, select the Accounts Receivable module if
 it is not already selected. Complete the Accounts Receivable Installation
 screen.

1. Complete the Order Processing Installation
 screen.

1. Enter customer information, if desired, using Accounts Receivable > Customers.

1. Print the Customer File Listing and check for
 accuracy.

1. Enter ship via information, if applicable, using Order Processing > Ship Via File
 Maintenance.

1. Print the Ship Via File Listing and check for
 accuracy.

1. Enter shipping terms information using Shipping Terms Maintenance.

1. Print Shipping Terms Listing and check for accuracy.

1. Enter the alternate customer ship to information using
 Customer Ship-to
 Addresses.

1. Print the Alternate Ship-to Address Listing
 and check for accuracy.

1. Enter standard messages using Message File Maintenance.

1. Enter quantity discount information, if applicable,
 using Quantity Discount
 Maintenance.

1. Set the processing dates to the conversion date using
 Processing Date
 Change.

1. Convert Accounts Receivable balances. Please refer to
 the Accounts Receivable Help file for instructions.

1. Enter the General Ledger departments to be used in
 posting various types of sales using G/L Department File Maintenance.

1. Enter the outstanding sales orders as of the
 conversion date using Order
 Entry.

1. Set the processing date to the new period for Order
 Processing using Processing Date
 Change.
