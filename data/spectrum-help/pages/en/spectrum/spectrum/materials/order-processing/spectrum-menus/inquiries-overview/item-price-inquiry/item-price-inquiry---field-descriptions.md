# Item Price Inquiry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Header section

Customer code
If you don't need to inquire about a specific customer's special pricing, you can leave this field blank. Otherwise, enter the customer code for the customer for whom the quotation is being prepared .

Item code
Enter an item code to inquire about a specific inventory item or select from a list of available inventory item codes.

Warehouse code
Enter a warehouse code to inquire about a specific warehouse.

Category
The item category displays. No entry is allowed.

Item summary

Quantity available Lowest price Method
The current available quantity of this item at the specified warehouse displays, along with the customer's "best" price. This is also the price per item that would be offered by the system during Quote Entry, Order Entry or Invoice Entry. This is the lowest price among price tables, special pricing and promotional prices.
The method defines how the system arrived at the lowest price, for example,
 LIST, PROMOTIONAL PRICE, and so forth.

Current quantity

On hand Committed On order

- The current total quantity on hand of the item at the specified warehouse displays.

- The current total quantity of items committed for sale from the specified
 warehouse displays. This number is updated from entry in Inventory Control > Job Requisition Entry, Work Order Material Entry,
 Order Entry, Invoice
 Entry, and Inventory Control > Mix Assembly Transaction Entry. Quote-type orders are not included.

- The current total quantity of items on order from supplier (shown in purchase units) to the specified warehouse displays.

Price table

Price tables 1-5 Price level
The established price level for this customer displays.

Quantity breaks

Quantity Amount

- The quantity at which each price break occurs is displayed.

- The amount of each quantity discount is displayed.
