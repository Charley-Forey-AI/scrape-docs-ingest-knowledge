# Order Delete

The Order Delete screen is used to delete erroneous or outdated orders from Spectrum.

## To use this screen

1. In the Order number field, enter the order number to be deleted and press Enter.

1. Click the Delete button.

1. Repeat steps 1 and 2 for each additional order that you are deleting.

1. Once you have finished deleting orders, click OK.
