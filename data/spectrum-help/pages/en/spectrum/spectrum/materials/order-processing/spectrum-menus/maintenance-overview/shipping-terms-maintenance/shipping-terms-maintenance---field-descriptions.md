# Shipping Terms Maintenance - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
The fields in the table below display after you have clicked the
 New button to create a new
 record, or selected an existing record and clicked the Edit button.
Fields/Buttons
Descriptions

Shipping terms code
Enter a unique shipping terms code, or click the
 drop-down arrow to select from a list of available shipping terms codes.
 These codes are used for quick entry while issuing quotes, taking orders
 and entering invoices.

Description
Enter the shipping terms description.

Listing button
Clicking this button in Spectrum displays the
 Shipping Terms Listing screen. This screen prints a report of all
 shipping terms on record, sorted in shipping terms code sequence.
