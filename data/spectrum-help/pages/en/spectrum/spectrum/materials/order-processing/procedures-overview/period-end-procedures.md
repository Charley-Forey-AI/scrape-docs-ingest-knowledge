# Period End Procedures

The following tasks should be completed at period
 end.

1. Print each of the following reports. These reports can
 be accessed and printed from the Order Processing > Reports menu.

- Sales
 Commission Report — Review the commission split amongst all
 salespeople for all invoices found in the invoice file.

- Open Orders
 Report — Review the report for any errors found in all open
 orders.
Note: If errors are noted in the Open Order
 Report, use Order Entry to make corrections. The
 Order Entry screen can be accessed from the Order Processing > Data Entry menu.

1. Print a copy of the Order Log Print to see the order
 numbers used since the last purge, together with the date and status of each order.
 The Order Log Print screen
 can be accessed from the Order Processing Period End menu.

1. Print a copy of the Invoice Log Print to see the invoice numbers
 used since the last purge, together with the date and status of each invoice. The
 Invoice Log Print screen
 can be accessed from the Order Processing Period End menu.

1. Use the Processing Date Change to set minimum and maximum processing dates as
 needed. This screen is accessed from the Order Processing > Data Entry menu.
