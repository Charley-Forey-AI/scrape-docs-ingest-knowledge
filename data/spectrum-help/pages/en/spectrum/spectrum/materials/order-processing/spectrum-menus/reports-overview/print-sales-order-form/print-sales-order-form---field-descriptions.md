# Print Sales Order Form - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Order
Enter the number of the sales order to print,
 double-click to select from a list of available sales orders, or press
 Enter to print
 ALL orders on file.

Print status
Select to reprint previously printed orders or to print
 only orders which have not previously been printed.

Status
Select to include orders and/or active and inactive
 quotes on the form.
