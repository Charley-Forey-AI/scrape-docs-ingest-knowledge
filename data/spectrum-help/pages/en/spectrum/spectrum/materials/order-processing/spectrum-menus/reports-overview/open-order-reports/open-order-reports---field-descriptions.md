# Open Order Reports - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Selections
Use the selection fields to specify which information you
 want to include on the report. All report starting screens offer a Saved
 Selections bar and Save Current button that can be used to remember saved
 user selections the next time this report is accessed.

Drop ship option
Select the drop ship orders you want to appear on the
 report.

Vendor
Enter a vendor code, or double-click to select from a
 list of available vendor codes.

Report format
Select the preferred report format from the options
 provided.

Order shipping status
Select which orders you want to appear on the report
 according to their order status.

Status
Select to include or exclude quote records.

Print orders with quantities
When either the Fill by customer or the
 Fill by order
 options are selected, the Print orders with
 quantities section is enabled to specify which quantity
 orders to include on the report:

- Complete – report includes orders which contain at
 least one item that can be shipped in full.

- Partial – report includes orders which contain at
 least one item that can be shipped in full, though there are more
 orders than stock on hand to fill all orders.

- Unavailable –report includes orders which contain at
 least one item that cannot be shipped in full.

- All
 – report includes all orders regardless of quantity
 availability.
