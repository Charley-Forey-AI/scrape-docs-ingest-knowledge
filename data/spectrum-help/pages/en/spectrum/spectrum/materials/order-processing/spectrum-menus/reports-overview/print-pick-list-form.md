# Print Pick List Form

The Print Pick List Form screen prints picking sheets for the specified range of orders and requested dates of shipment. It is possible when selecting picking sheets to specify printing only new orders or only back orders. If a picking sheet has already been printed once, the operator must specify that the picking sheet is to be reprinted. One company might periodically print all picking sheets that have not yet been printed while other companies may opt to print picking sheets as needed at the time the order is placed. It is not necessary to print picking sheets before confirmation of shipment or invoicing.
