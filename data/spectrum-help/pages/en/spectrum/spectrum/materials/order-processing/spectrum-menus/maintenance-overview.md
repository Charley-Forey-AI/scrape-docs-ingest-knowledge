# Maintenance Overview

The Maintenance menu is used during conversion to establish records in specified order processing files. Following conversion, the data in these files can be revised as necessary.
Important: Some maintenance data may have already been
 entered in the Accounts Receivable module; in this case, duplicate entry is not
 necessary.
