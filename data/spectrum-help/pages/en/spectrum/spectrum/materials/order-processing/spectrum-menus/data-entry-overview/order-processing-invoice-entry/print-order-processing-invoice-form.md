# Print Order Processing Invoice Form

This screen is used to print invoices. The standard screen for Order Processing invoices accommodates contractors using custom forms.
Export the file
Several Spectrum report starting screens off the ability to export files. Click the Export button, select the file format, and enter the file NAME in the 'Export file' field. The file path is not necessary because the file save location depends on the browser settings.
