# Print Order Processing Invoice Form - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Format
Select a format for the invoice form:

- The Invoice form format will optionally include the client company bitmap in the heading portion of each invoice. To print the company logo, the applicable bitmap must be placed on the PC where the invoices are printed, and named C:\Spectrum\COMPANY.BMP. It is recommended that a company bitmap be created if Crystal-Link forms are used.

- The Invoice form with zero lines suppressed format suppresses the printing of freight and other non-stock lines with zero extensions.

Selections

From/thru invoice date
Enter an invoice date range to print a series of invoices.

Invoice
Press Enter to begin printing ALL invoice numbers on file. Otherwise, enter the invoice number to be included, or select from a list of available invoice numbers.
If this screen was launched from the Customer Open Items or Customer Payment History screens, the invoice number will default in this field.

Customer
Press Enter to begin printing ALL customer codes on file, enter the customer code to be included, or select from a list of available customer codes.
If this screen was launched from the Customer Open Items or Customer Payment History screens, the customer code will default in this field.

Message
Enter a message to be printed on this batch of invoices, if desired. Otherwise enter * and a valid message number to access a pre-established message.

Print status

- First print only - select this option button to print invoices which have not been previously printed.

- Reprint only - select this option button to print previously printed invoices.

- All
