# Warning: Vendor Code Entered Has Inactive Status.

If an inactive General Ledger account code is entered, this
 message displays. Click OK to If an
 inactive vendor code is entered, this message displays. Click OK to continue entering data, if desired.
To change the vendor's status, click Accounts Payable > Maintenance > Vendor > Properties and select either Active or Not
 Used in the Status
 section.
