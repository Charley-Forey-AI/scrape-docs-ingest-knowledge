# How Quantity Discounts Work in Order Processing

You can set up quantity discounts in Order Processing by
 going to Order Processing > Maintenance > Quantity Discount Maintenance.

1. Set up a quantity discount in Order Processing  >  Maintenance  >  Quantity Discount Maintenance.

1. Create an Order Processing invoice.

1. Verify that the Terms discount overrides quantity discount checkbox (located in the Order Processing  >  Maintenance  >  Terms Code File Maintenance screen) is clear for the term code being used on the invoice.

1. Update the Order Processing invoice.

The quantity discount will take effect when you process an
 Accounts Receivable cash receipt for the invoice.
