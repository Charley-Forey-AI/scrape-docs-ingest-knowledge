# Looking Up Information in Order Processing

Use the following procedures to look up information in Order Processing.
To review inventory information

- Click Order Processing  >  Inquiries  >  Item. Item Price
 Inquiry displays information for a specific customer or for all
 customers.

To review information
 about a specific customer

- Click Order Processing  >  Inquiries  >  Customer Order /  > Invoice. Customer Order/Invoice
 Inquiry displays information about a designated customer.

To review information about a designated
 item

- ClickOrder Processing  >  Inquiries  >  Item / Order. Item / Order Inquiry displays information about a designated item.


To review total orders

- Click Order Processing  >  Inquiries  >  Order Totals. Display Order Totals
 displays order totals for one or more orders.

To review total sales

- Click Order Processing  >  Inquiries  >  Sales Totals. Display Sales Totals
 displays sales totals for one or more invoices.
