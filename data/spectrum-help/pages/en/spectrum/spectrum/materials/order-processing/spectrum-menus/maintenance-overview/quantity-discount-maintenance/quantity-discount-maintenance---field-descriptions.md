# Quantity Discount Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
The fields in the table below display after you have clicked the New button to create a new record, or selected an existing record and clicked the Edit button.
Fields/Buttons
Descriptions

Type
Select the discount record type from the drop-down list. Selections include Item or Category.

Code
Enter the item or category code, or press F4 to select from a list of available category codes.

Discount rate
Enter the rate for this discount. The quantity discount is calculated by
 multiplying this rate by the invoice quantity.

Listing button
Clicking this button in Spectrum displays the Quantity Discount Listing screen. This screen prints a report of all quantity maintenance files on record.
