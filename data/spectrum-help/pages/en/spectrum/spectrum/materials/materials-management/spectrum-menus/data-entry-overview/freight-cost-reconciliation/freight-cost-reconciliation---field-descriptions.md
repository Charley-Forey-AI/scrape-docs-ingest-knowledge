# Freight Cost Reconciliation - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Vendor
Enter the vendor code, or press F4 to search a list of
 valid freight vendor codes.
Note: If the vendor's status is Not used then entry is not allowed.

Invoice #
Enter the invoice number. If you specified a vendor at the Vendor
 code field, you can click the Preview
 button from this field to preview a listing of all unassigned tickets
 associated with the vendor you selected.
The system validates the invoice numbers entered here to make sure that the numbers do not already exist in Accounts Payable (including any rejected invoices). If any of the invoice numbers have already been used, an error message will display. Use this screen to replace the duplicated number with a unique number.

Invoice date
Enter the date of the invoice or select a date from the Date
 Change window. This information is later transferred to the
 A/P Invoice/Credit Memo Entry.
This date must be within the Materials Management minimum/maximum processing
 date range.
Note: If the Stop
 entry if invoice date after maximum date checkbox is
 selected in the Accounts Payable Installation > Properties tab, the date entered in this field must fall before the
 maximum Accounts Payable processing date.

Tickets included
The number of tickets that are selected in the grid displays.

Reconciled hours Reconciled quantity Reconciled cost
The total hours, amount, and cost reconciled displays.

Adjustments
The total dollar amount for all ticket adjustments on this vendor's invoice displays. Click the button to display the [Truck Summary Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/freight-cost-reconciliation/freight-cost-reconciliation---field-descriptions/truck-summary-inquiry) window.

Other tickets
The total dollar amount for all other tickets on this vendor's invoice displays.

Invoice total
The invoice total displays.

Include All
Click to select all items in the list box below.

Clear All
Click to clear all selected items.

Search
Click to display the [Freight Cost Reconciliation - Search Transactions](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/freight-cost-reconciliation/freight-cost-reconciliation---field-descriptions/freight-cost-reconciliation---search-transactions) window.

Ticket
Click to open the [Scale Ticket Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/scale-ticket-inquiry) window for the selected item.

Grid

Include?
Select this checkbox to reconcile the freight cost for this line item.

Plant / Pit
The plant code displays.

Ticket
The ticket number displays. With the ticket number highlighted, it is possible
 to click the Ticket button to view the entire ticket
 history.

Date
The ticket date displays.

Rate type
Use the drop-down menu to select whether this ticket's rate is based on the Hour or the Ton.

Hours
Enter the actual hourly quantity for this ticket.

Quantity
The total quantity displays. This information can be changed during the reconciliation process, if necessary.

U/m
The total unit of measure (UM) displays. This information can be changed during the reconciliation process, if necessary.

Rate Cost
The rate and cost display.

Hauler
The hauler code and description display.

Item
The item code displays.

Ticket type
The types of tickets are: Job,
 Customer, Transfer, or
 Receipt.

Name
The following displays for each name:

- Job trips: Code and description

- Customer trips: Customer code and description

- Receipts: Warehouse code and description

- Warehouse trips: Warehouse code and description

Item description
The item code description displays.
