# Scale Ticket Inquiry

Use the Scale Ticket Inquiry window to look up ticket history for any ticket listed on the main inquiry screen. Entry is not allowed in this window.
Fields
Descriptions

Images
Click the link to view images for this ticket.

Batch code
The Batch code displays.

Plant / pit
The plant/pit code displays based on the plant/pit you selected in the [Scale Ticket History Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/inquiries-overview/scale-ticket-history-inquiry) window.

Ticket #
The ticket number displays based on the ticket you have selected on the main screen. The warehouse code and description of the assigned plant/put display to the right of this field.

Ticket type
The type of ticket displays: Customer, Job Receipt, or Transfer.

Customer
This field displays when the Ticket type is Customer.

Job
This field displays when the Ticket type is Job.

To warehouse
This field displays when the Ticket type is Transfer.

Date / time
The time and date display based on the ticket type.

Sales order
Any sales order processing number assigned to this ticket displays.

Customer P.O.
The customer's purchase order associated with this ticket displays.

Message
A message for this ticket displays.

Financials

Total extension Freight charge Misc. charge
The total extension, freight charge, and miscellaneous charge for the ticket displays.

Misc. charge taxable?
This checkbox is selected if the miscellaneous charges associated with this ticket are taxable; otherwise, if the miscellaneous charges are not taxable, it will appear clear.

Total $ (before tax)
This is calculated as the Total extension plus Freight charges plus Misc. charges.

Sales tax
The sales tax amount, if applicable, displays in this field.

VAT code
If the Utilize value added tax (VAT) tracking? checkbox is selected in Company Installation, select the appropriate VAT code for this ticket. Once selected, the VAT amount will automatically display.

Ticket total
The total amount for this ticket displays in this field.

Posted to...

Order Processing
Displays if the ticket has been updated to Order Processing Invoice Entry.

Accounts Receivable
Displays if the customer ticket has been updated to A/R Customer Invoice Entry.

Inventory Control
Displays if the Ticket type is Job, Receipt, or Transfer.

Click to view the [Scale Ticket Inquiry - More Scale Ticket Information](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/scale-ticket-inquiry/scale-ticket-inquiry---more-scale-ticket-information) window.

User-Defined
Click to view the User-Defined fields window.
