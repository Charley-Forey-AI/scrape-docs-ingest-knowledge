# Ticket Error User-Defined Fields

Use this window to view and edit user-defined fields for
 the ticket selected in Ticket Error Correction Entry.
Fields/Buttons
Descriptions

Edit
Click to edit the selected user-defined field.

Field status
Click to view fields that display data in the Current setting column, that are not used, or both.

Prompt
The title for this user-defined field (for example, DUE DATE) displays.

Current Setting
Valid user entries for this user-defined field display.
