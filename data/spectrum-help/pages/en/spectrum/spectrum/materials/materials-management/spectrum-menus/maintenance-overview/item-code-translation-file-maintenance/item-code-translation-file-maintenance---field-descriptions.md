# Item Code Translation File Maintenance - Field Descriptions

Use the table below for reference when completing the fields
 on this screen.
Field
Description

Scale item
Enter or search for the code of the item as it is
 entered in the scale software.
Important: This window does not look at the Spectrum item
 code, nor does it read codes from the scale software package; it
 provides a listing of scale codes that were previously entered at
 this prompt using this translation screen.

Plant/pit
Enter or search for the plant/pit code, or press
 Enter to leave blank if this is the same item code for all
 plants.

Spectrum item
Enter or search for the corresponding Spectrum
 item code. This is the only field that can be modified in the Edit
 Scale Item window.
