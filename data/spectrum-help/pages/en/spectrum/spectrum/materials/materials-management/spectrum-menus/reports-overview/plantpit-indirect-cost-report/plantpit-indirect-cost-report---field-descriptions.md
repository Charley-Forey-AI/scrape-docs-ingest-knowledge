# Plant/Pit Indirect Cost Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Plant / Pit
Enter a plant/pit location to include on the report, or press Enter to include ALL on the report.

Phase
Enter phase codes for the plant/pit location, or press Enter to include ALL on the report.

Cost type
Specify cost types for the plant/pit location, or press Enter to include ALL on the report.
Note: This will be helpful for companies that use a separate cost
 type for processing Job Cost adjustments.

From date To date
Enter the first and last dates to be included, or press Enter to begin with the earliest date and end with the current Purchase Order processing date.
