# Ticket Import User-Defined Fields Listing - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

Company
Enter the company code to print, or press
 Enter
 to accept the system default of ALL.

Include alpha fields
Select this checkbox to include the alpha
 user-defined fields in the Ticket Import User-Defined Fields
 Listing.

Include date fields
Select this checkbox to include the date
 user-defined fields in the Ticket Import User-Defined Fields
 Listing.

Include numeric fields
Select this checkbox to include the numeric
 user-defined fields in the Ticket Import User-Defined Fields
 Listing.

Print the validation codes
Select this checkbox to include validation codes
 in the listing.
