# Freight Adjustment File Maintenance

Use Frieght Adjustment File
 Maintenance screen to view freight adjustment codes. Codes established in the
 New Adjustment Code window can be applied to freight during data
 entry, and the software will automatically calculate the adjustment amount. Adjustments can
 increase or decrease freight amounts used in the Freight Cost Reconciliation > Truck Summary Inquiry.
