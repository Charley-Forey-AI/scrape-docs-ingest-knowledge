# Setting Up Hauler Rates for Job Contracts

Set up hauler rates for job contracts to apply special
 hauler rates to a designated job contract. If you do not set up special hauler rates for a job
 contract, Spectrum will use the standard default hauler rates set up in Hauler File
 Maintenance.
Note: This procedure must be completed when adding new hauler rates or updating existing
 hauler rates for a specified job contract in Spectrum.

1. On the Site Map screen, click
 Materials Management  >  Maintenance  >  Job Contract.

1. Complete the Job and
 Area code fields for the job contract hauler rates you are
 designating. Press F4 to select at either field.

1. Click the Hauler Rates
 button.

1. Complete the Hauler code field
 or press F4 to select a valid hauler code.

1. In the Cost rate type option
 box, select the default cost rate type for the selected hauler:
 Ton or Hour. The rate type specified
 here will default during Scale Ticket Date Entry.

1. Complete the Properties section
 of the window with the appropriate Cost per tone rate,
 Cost per hour rate, and Minimum cost
 for the selected job.Note: If any of the fields in the
 Properties section are left blank, rate information will
 alternatively default from Hauler File Maintenance during
 Scale Ticket Date Entry.

1. Click OK.

1. Repeat steps 4 – 7 for each additional special hauler rate you want to apply to the selected job contract.

1. Once you have added all of the special hauler pricing
 and completed the Job Material Contract Maintenance screen,
 click OK to return to the main menu.
