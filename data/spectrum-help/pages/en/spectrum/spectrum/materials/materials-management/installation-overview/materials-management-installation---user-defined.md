# Materials Management Installation - User-Defined

Use this tab to install properties information for the
 Materials Management module.
Fields
Descriptions

Allow setup of company-specific fields?
Select this checkbox to allow the setup of
 vendor-related company-specific fields for companies other than the current
 company.
If you select this checkbox, users are permitted to set
 up and maintain user-defined fields associated with the current company.


Allow setup of non-company-specific fields (company
 ALL)?
Select this checkbox to allow the setup of
 vendor-related non-company-specific fields for all companies.
If you select this checkbox, users are permitted to set
 up and maintain user-defined fields associated with all companies.

Show non-company-specific fields ticket data entry
 (company ALL)?
Select this checkbox to display ticket-related
 non-company-specific fields for all companies in the Scale Ticket
 Transactions screen.
If you do not select this checkbox, only the
 user-defined fields associated with the current company will be available in
 the Scale Ticket Transactions screen.
