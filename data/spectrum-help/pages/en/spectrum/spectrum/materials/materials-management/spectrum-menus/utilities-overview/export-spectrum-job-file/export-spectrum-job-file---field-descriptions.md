# Export Spectrum Job File - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Job
Enter a job number to be exported, or press Enter to accept the software default of ALL.

Division
Enter the division of jobs to be exported, or press Enter to accept the software default of ALL.

Customer
Enter the customer code for the jobs to be exported, or press Enter to select all customers.

Export file
The export path and file name will default to FFJOB.TXT.
Click the File Layout button to look up optional informational describing the file layout. This window shows the variables, type (alpha or numeric) and maximum length, and is presented in the order in which the information will appear in the resulting comma-delimited ASCII export file.

Continue
Click this button to proceed directly to the update.
