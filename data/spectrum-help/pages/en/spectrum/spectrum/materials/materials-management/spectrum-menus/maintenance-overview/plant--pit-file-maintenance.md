# Plant / Pit File Maintenance

The Plant / Pit file maintenance window provides
 information about individual plants and pits in the Material Purchase Reconcilliation screen.
Spectrum will route the invoice created based on that code after the
 update is complete.
 If the hauler does not have a routing code, Spectrum will route the
 invoice based on the routing code specified for the plant/pit referenced on the Material
 Purchase Reconciliation screen. If the plant/pit does not have a routing code, Spectrum
 will route the invoice based on the routing code specified for the job referenced on the
 Material Purchase Reconciliation screen. If the job does not have a routing code, Spectrum
 will route the invoice based on the routing code specified on the vendor referenced on the
 Material Purchase Reconciliation screen. If the vendor does not have a routing code,
 Spectrum will route the Invoice based on the routing code specified in the Accounts Payable
 Installation screen. Finally, if a routing code is not specified in the Accounts Payable
 Installation screen, Spectrum will prompt for you to enter one. If a routing code is
 defined for the hauler referenced in Freight Cost Reconciliation, then Spectrum will route
 the invoice created based on that code after the update is complete. More info If the
 hauler does not have a routing code, Spectrum will route the invoice based on the routing
 code specified for the plant/pit referenced on the Freight Cost Reconciliation.
 If the plant/pit does not have a routing code, Spectrum will route the
 invoice based on the routing code specified for the job referenced on the Freight Cost
 Reconciliation. If the job doesn't have a routing code, Spectrum will route the invoice
 based on the routing code specified on the vendor referenced on the Freight Cost
 Reconciliation. If the vendor does not have a routing code, Spectrum will route the invoice
 based on the routing code specified in the Accounts Payable Installation screen. Finally,
 if a routing code is not specified in the Accounts Payable Installation screen, Spectrum
 will prompt for you to enter a routing code.
