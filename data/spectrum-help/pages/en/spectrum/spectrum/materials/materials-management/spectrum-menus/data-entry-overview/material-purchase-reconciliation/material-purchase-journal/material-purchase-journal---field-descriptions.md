# Material Purchase Journal - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Vendor code
Press Enter to update and create invoices for all vendors, or use SuperSelect to enter the desired selection criteria.

Invoice no.
Press Enter to update all invoices, or use SuperSelect to enter the desired selection criteria.
The software validates the invoice numbers entered here to make sure that the numbers do not already exist in Accounts Payable. If any of the invoice numbers have already been used, a Material Cost Error Report will be created. Use the Material Purchase Reconciliation screen to replace the duplicated number with a unique number.

Through invoice date
Press Enter to include invoices through the current Materials Management processing date; otherwise, press F4 to display the Date Change window and select.

G/L date
Enter the General Ledger date to use for this update. This date must be within the Materials Management minimum/maximum processing date range. The software defaults to the current Materials Management processing date.

Assign VAT tax code?
If the Utilize value added tax (VAT) tracking? checkbox is selected in Company Installation, select this checkbox to add VAT to the vendor invoice. When selected, specify the Tax code below.
