# Customer Material Contract Maintenance

Use material contract maintenance screen to set up any
 special overrides for item pricing (level#), freight pricing, miscellaneous charges, tax
 source, or dollar limit. This screen is only used for special changes; otherwise, the software
 looks at the Materials Management Installation set up for these items.
