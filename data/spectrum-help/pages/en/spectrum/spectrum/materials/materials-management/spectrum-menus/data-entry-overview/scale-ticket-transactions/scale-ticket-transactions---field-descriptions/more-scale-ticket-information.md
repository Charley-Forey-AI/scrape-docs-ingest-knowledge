# More Scale Ticket Information

Use this window to set up hauler information pertaining to this ticket.
In the Ticket type field of the New/Edit Scale Ticket window, if Customer is selected , you can establish customer, salesperson, and sales tax information. If Job is selected, you can establish area code, phase, and cost type information.
