# Mix Processing Adjustments - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Batch code
Enter the Batch code, or search for batches in progress.

Components
Click to display further unit cost information in the [Mix Components window](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/mix-processing-adjustments/mix-components-window).

Item code
The item codes with the Auto-create mix checkbox selected in Inventory Control Inventory Item File Maintenance - Properties will be shown on this screen for potential modifications.

Description Quantity
The item description of the mix and net mix quantity display.

U/m Unit cost
The mix's unit of measure (UM) and unit cost display.
