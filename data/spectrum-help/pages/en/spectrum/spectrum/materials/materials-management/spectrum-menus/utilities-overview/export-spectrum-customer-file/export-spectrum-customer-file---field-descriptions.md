# Export Spectrum Customer File - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Customer code
Enter the customer code to be exported. Use the SuperSelect options to define the search, or press Enter to select all customers.

Customer type
Enter the customer type to be exported. Use the SuperSelect options to define the search, or press Enter to select all customer types.

Export file
The export path and file name will default to "C:\FFCUST.TXT:" from the Company Installation screen. Click the File Layout button to look up optional informational describing the file layout. This window shows the variables, type (alpha or numeric) and maximum length, and is presented in the order in which the information will appear in the resulting comma-delimited ASCII export file.

Continue
Click this button to proceed directly to the update.
