# Material Purchase Journal

Use this screen to view a journal of all the reconciled material tickets and to create invoices in A/P as unprocessed invoices either in Unapproved Invoice Entry or in Invoice/Credit Memo Entry. You can then route the unapproved invoices or run the A/P Transaction Register / Update to process these material purchase invoices throughout the software. This update will also post the actual material purchase cost back to the ticket history file for later reporting. If a use tax code was applied to the item in job requisitions, this same code will also show up on the invoice line for job variance lines. This material purchases file will be cleared upon finish of the update.
The A/P general ledger account to debit will be the same non-stock accrual account from the item category file credited with the posting of the customer invoices or job requisitions.
