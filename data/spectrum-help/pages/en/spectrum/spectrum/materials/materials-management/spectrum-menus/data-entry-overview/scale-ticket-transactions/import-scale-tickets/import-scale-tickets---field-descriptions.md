# Import Scale Tickets - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Plant/pit
Enter the plant/pit code from which ticket information will be imported. A search window is available for viewing valid codes.

Data format
The data format specified in Plant/Pit File Maintenance defaults. Press Enter to accept the default, or override the default if desired. A search window is available for viewing valid data format codes.

Batch
 Your user ID will default. Otherwise, enter the batch code to which these tickets are to be assigned if no batch code is specified in the import file, or search for a list of valid batch codes.

Ticket date
If the Data Format file specifies "DATE" at the date format prompt, enter the ticket date for this file. If a format (that is, MMDDYY) is specified at the date format prompt, the date will be imported along with other ticket information and entry here is not necessary. The date specified in this field will automatically be assigned to tickets if an invalid date format is set up in Scale Data Format Maintenance.
