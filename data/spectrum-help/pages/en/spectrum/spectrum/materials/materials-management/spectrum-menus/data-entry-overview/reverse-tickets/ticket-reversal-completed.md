# Ticket Reversal Completed

Once the reversal is complete, this screen will display. Each ticket that has been reversed displays, along with a corresponding message describing whether the transaction was deleted, a credit memo was created, or a requisition was created.
Field
Description

Listing
Click to open the Print Report window. You can [MM Ticket Reversal Report](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/reverse-tickets/ticket-reversal-completed/mm-ticket-reversal-report), Export, Cancel the report; or click the My Reports button to customize it.

Transaction type
The transaction type displays: Customer, Job, Receipt, or Transfer.

Reference #
A reversal reference number for each transaction displays.

Result
A message detailing the result of the reversal process for each reference number displays.
