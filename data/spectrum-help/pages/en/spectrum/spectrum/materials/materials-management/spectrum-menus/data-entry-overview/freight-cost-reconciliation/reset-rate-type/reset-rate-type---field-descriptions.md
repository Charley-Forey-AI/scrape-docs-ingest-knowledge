# Reset Rate Type - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Customer
Press Enter to select ALL customers, jobs, plant/pits, trucks, and haulers to include in the global change, or search and select from a list of valid codes.

Job

Plant/pit

Truck

Hauler

Phase
Press Enter to select ALL Phases to include in the global change, or enter a phase code.

Cost type
Press Enter to select ALL Cost types to include in the global change, or search and select from a list of valid cost type codes.

Tickets
Press Enter to select ALL tickets to include in the global change, or search
 for a valid scale ticket using the search Scale Tickets window.

From date
Enter or search for the start date you want to include in the global change.
Example: If you enter 04/01/16 in this field, the change will affect only those tickets occurring on or later than this date.

To date
Enter or search for the end date you want to include in the global change.
Example: If you enter 04/01/16 in this field, the change will affect only those tickets occurring on or before this date.

New rate type

- Hour

- Ton

The rate type you select here will display in the Rate type column on the [Freight Cost Reconciliation](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/freight-cost-reconciliation) screen for all tickets included on the update.
