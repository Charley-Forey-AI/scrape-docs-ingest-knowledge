# Making Global Changes Using Revise All Import Errors

In the event that errors exist in imported tickets, you can
 make global changes to the erroneous tickets so that they can successfully complete the import
 process.

1. On the Site Map, click Materials Management  >  Data Entry  >  Scale Tickets.

1. Click the Import button. The Import Scale Tickets screen displays.

1. Click the Errors button. The Ticket Error Correction Entry screen displays.

1. Click the Revise All button. The Revise All Import Errors screen displays.

1. Use any of the fields in the Selections section to make global changes to the erroneous tickets. For example, to change the customer who appears on a group of tickets, enter the customer code in the Customer field and then enter the correct customer code in the corresponding Replace with field.
