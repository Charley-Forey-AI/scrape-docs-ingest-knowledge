# Truck Summary Inquiry

Use this window to look up any adjustments that have been
 made to trucks associated with the selected scale ticket on the main screen. This window can
 be used to make additional adjustments to a selected truck, for example, adjustments for the
 cost of fuel.
Fields
Descriptions

Vendor
The Vendor code, description, and Invoice number display.

Invoice

Adjustment Entry
Enter any adjustments for the designated truck. The Freight Cost
 Adjustments window is used to further define and calculate
 assigned adjustments using pre-defined freight adjustment codes.
Making an adjustment

1. Click the Adjustments button .

1. At the Adj. Code field, enter the freight adjustment
 code or search and select an adjustment code.

1. At the Quantity field, enter the positive or negative
 quantity amount, and then press Enter.
Note: To enter a negative quantity amount, type a hyphen before
 the quantity amount. For example, if you are entering a quantity of
 10, you would first type a "-", and then type the digits "1" and "0".
 When you press Enter to proceed to the next field, the quantity amount
 will appear as 10.00–.

1. Press Enter to proceed through the
 Price and Total
 fields. the software will automatically calculate the price and the
 total amount.

Truck code
The truck code and corresponding description associated with the scale ticket
 selected on the Freight Cost Reconciliation screen
 display.

Description

# of tickets
The number of tickets selected for this truck on the main screen displays.

Charges
All charges associated with the selected truck display.

Adjustments
All adjustments that have been made for the selected truck display. No entry
 is allowed. Click the Adjustments button to enter any
 further adjustments for the selected truck.

Total cost
The total for the truck after including any assigned adjustments displays.
