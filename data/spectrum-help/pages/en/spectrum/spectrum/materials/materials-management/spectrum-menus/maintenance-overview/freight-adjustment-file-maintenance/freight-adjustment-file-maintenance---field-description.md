# Freight Adjustment File Maintenance - Field Description

Use the table below for reference when completing the fields
 on this screen.
Fields
Descriptions

Adjustment code
In the New Adjustment Code window, enter or search for the adjustment
 code you want to use for the selected freight adjustment.

Description
Enter or edit a description of the adjustment
 code.

Unit of measure
Enter, search for, or edit the unit of measure
 (UM).

Default rate
Enter or edit the default freight rate.

Adjustment type
Select the adjustment type you want associated
 with the adjustment code you are adding or editing.

- If you select Add-on, the amount added to
 freight will be positive.

- If you select Deduction, the amount added
 to freight will be negative.
