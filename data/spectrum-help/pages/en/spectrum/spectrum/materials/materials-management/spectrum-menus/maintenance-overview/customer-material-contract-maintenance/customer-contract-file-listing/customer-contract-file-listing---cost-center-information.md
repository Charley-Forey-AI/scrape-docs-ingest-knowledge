# Customer Contract File Listing - Cost Center Information

Use the table below for reference when completing the fields on this screen.
If the cost center feature is enabled in the Enterprise Installation screen, the Operator will only be able to view records they have cost center security for. The operator will be unable to access a record if he or she does not have cost center security for the cost center assigned to the record.
