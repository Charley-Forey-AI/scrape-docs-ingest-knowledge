# Hauler Special Rates window

The Hauler Special Rates window allows you to set up multiple haulers with their cost and billing rates and respective minimums. If there are no special rates for the hauler, then the software will use the standard rate from the hauler file. If the freight is charged directly to the customer for certain haulers, then you can set up these haulers with no freight rate. A blank hauler code may be entered, if all haulers for this customer are paid a standard rate. You will then be able to set up standard haul rates for a particular customer or particular customer/customer job combination.
