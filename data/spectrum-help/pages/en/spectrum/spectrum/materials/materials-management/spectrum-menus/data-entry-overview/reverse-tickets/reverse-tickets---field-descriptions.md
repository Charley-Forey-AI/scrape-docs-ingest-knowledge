# Reverse Tickets - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields/Buttons
Descriptions

Transactions to be reversed

Plant
Enter or search for the plant you want to include on the reversal.

Pit
Enter the pit you want to include on the reversal.

Job Customer
Enter or search for the job number and customer you want to include on the reversal.

Invoice Job requisition
Enter the Invoice number and job requisition number you want to include on the reversal.

From/To ticket date
Enter the from and through ticket dates to update. Or press
 Enter to leave blank and update from the first
 date or through the current Materials Management processing date. Click the
 arrow to select a date from the calendar or click the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator) icon select a calculation method to determine the
 default date.

Include transaction types

Include transaction types
Select any of the checkboxes to indicate the types of transactions you want to include in the reversal.

- Customer

- Job

- Receipt

- Transfer

New transactions

Batch code
The Batch code defaults to the current operator. Click the arrow to select a different batch code.

Reversal date
This date defaults to the current Materials Management processing date. The
 date must be within the minimum/maximum date range, which is set in the Systems Administration > Installation > Processing Dates screen.
A single date will be assigned to all transactions reversed in this update.
When tickets are updated directly to Accounts Receivable.
If the reversal date is changed for new items, this information will be saved by the system except with A/R credit memos; A/R credit memos use the G/L date for the reversal date and the original invoice date for the invoice date of the credit memo.
