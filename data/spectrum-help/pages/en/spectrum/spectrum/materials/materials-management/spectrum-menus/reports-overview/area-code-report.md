# Area Code Report

The Area Code Report provides a detailed report of special pricing areas. This report shows the total quantity (tons) delivered or sold by the area code regions. It shows the quantity by item within the region, where the company is selling the materials, what type of material is being sold, and by which salesperson. The Summary format sorts by plant/pit, then area code and item code. The Detail format sorts by plant/pit, then area code, item code, date and ticket number.
Many contractors maintain special rates for various regions surrounding each plant/pit. This report serves as a management tool showing total quantity delivered or sold by region. Since the contractor may have multiple salespeople servicing these regions, the report also includes information regarding salesperson performance.
