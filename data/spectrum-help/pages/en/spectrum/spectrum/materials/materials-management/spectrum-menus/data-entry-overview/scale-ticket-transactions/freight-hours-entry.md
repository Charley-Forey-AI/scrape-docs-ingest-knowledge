# Freight Hours Entry

Use the Freight Hours Entry screen to reconcile the freight hours for trucks before they are posted to an invoice. This process of reconciliation is an optional step; otherwise, the freight hours are reconciled in the Freight Hours Reconciliation screen after the ticket is posted. Using this screen to reconcile freight hours prior to posting them to tickets is useful if the billed freight is billed hourly.
The freight rate type information defaults from the Hauler File Maintenance screen. Freight hours and freight quantity are stored separately on each ticket.
