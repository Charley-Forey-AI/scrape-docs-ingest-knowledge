# Scale Data Format Maintenance

When ticket data has already been entered into a different database (such as scale software), it is not necessary to re-enter that information into Spectrum. You can use the other database to create a file that will transfer data into Spectrum. Files with up to 2,000 characters per line can be imported.
Data formats will be used by the software in all Spectrum companies. Therefore, it is only necessary to set up the formats once, even if multiple companies are set up on the software. In the Data format field, enter a unique code to identify the data being formatted. For example, set up one code for each scale. To view existing data format codes, click the drop-down arrow.
