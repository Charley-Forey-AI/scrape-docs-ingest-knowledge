# Installation Overview

Click the Admin button on the toolbar of any menu; the Installation list will appear. From the Installation list, select Company Installation. Note that the software Administrator should have previously completed this screen. Ensure that the flag for this specific module has been checked and then click Continue to save the information and return to the Installation list.
Select Materials Management from the Installation list. When the installation screen is selected for the first time, Spectrum will beep and a dialog box stating "S/I MODULE HAS NOT BEEN INSTALLED OK TO PROCEED" will display on the screen. Once this Installation screen has been completed and the information saved, the message will not appear again.
Spectrum provides an Help for almost all the fields within this screen. Press F4 at the desired prompt to obtain detailed instructions for completing the fields and screens.
