# Freight Cost Reconciliation - Search Transactions - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Selections

Plant/pit
Press Enter to select all plants/pits, or search for valid plant/pit information.

Ticket
Press Enter to select all tickets, or search for valid scale ticket information.

Batch code
Enter a Batch code, or search for valid freight reconciliation batches.

Truck
Enter a truck code in this field, or search for valid truck codes.

From date
Enter the date from which to select/deselect, or press Enter to select ALL.

To date
This field defaults to the current Materials Management processing date. Press Enter to accept this date, or enter a new ending date.

Ticket display

- Show all tickets

- Show open tickets

Click the ticket display option you want to view on the Freight Cost Reconciliation screen.
