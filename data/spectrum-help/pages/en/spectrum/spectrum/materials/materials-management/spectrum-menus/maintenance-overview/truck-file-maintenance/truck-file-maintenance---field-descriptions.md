# Truck File Maintenance - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Fields
Descriptions

Truck CodeIn the New Truck window, enter or
 search for a unique code to identify the truck. If truck and trailer are
 used together as a unit, enter them as a single unit, not as two
 separate entries.
Truck NameEnter a full name for the truck.
 For instance, the code might be "D001," and the full description "Dump
 Truck #1."
Tare weightEnter the truck tare weight in
 pounds, and the last tare date for this truck.
Last tare date
Convert net weight?Select this checkbox to convert the scale net
 weight from pounds to tons in scale entry. It is useful to calculate
 the freight quantity in tons when importing scale tickets because it
 makes sense to use the same weights for all tickets instead of
 calculating the weight by truck.

Hauler codeEnter or search for the hauler code to default
 for this truck. The code entered here must have been previously set up
 in Hauler File Maintenance. When this optional field is blank, it
 will be necessary to import both truck and hauler from the scale
 software. If both truck and hauler are entered in the scale software
 from which data is imported to Spectrum, hauler is easily imported;
 this field is provided for the convenience of those whose scale
 software prompts for a truck code but does not prompt for a hauler.
