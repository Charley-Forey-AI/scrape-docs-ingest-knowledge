# Freight Hours Reconciliation - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Batch code
Enter a batch code, or search for existing freight quantity batches in progress. The batches in progress display. Press Tab to select the desired batch, or press Cancel to return to the screen.

Truck
Enter or search for valid truck codes.

From date
Enter a date with which to begin reconciliation, or press Enter to leave blank and begin with the first date.

To date
Enter a date with which to end reconciliation, or press Enter to accept the current Materials Management processing date.

Tickets included
The number of tickets that are selected in the grid displays.

Reconciled hours
The total hours, amount, and cost reconciled displays.

Reconciled quantity

Reconciled cost

Include All
Click to display the [Freight Cost Reconciliation - Search Transactions](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/freight-cost-reconciliation/freight-cost-reconciliation---field-descriptions/freight-cost-reconciliation---search-transactions) window.

Clear All
Click to open the [Scale Ticket Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/scale-ticket-inquiry) window for the selected item.

Ticket
Click to open the [Scale Ticket Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/scale-ticket-inquiry) window for the selected item.

Status
Click to select open tickets only or all tickets.

Grid

Include?
Select this checkbox to reconcile the freight quantity for this line item. The software will move the cursor to the next line item. Enter additional freight reconciliations, if desired.

Plant / pit
The corresponding plant code for this line item displays.

Ticket
The ticket number displays. Click the Ticket button to view detailed ticket history for this line item.

Date
The date for this line item displays.

Start time
If a start time was previously entered for the ticket, it will display.

End time
If an end time was previously entered for the ticket, it will display.

Cycle time
If a total cycle time was previously entered for the ticket, it will display.

Freight hours
Change the freight hours for the selected ticket as needed. Press Enter once you have made your selection to proceed to the next field.

Rate type
Use the drop-down menu to select a rate type: Hour or Ton. Press Enter once you have made your selection to proceed to the next field.

Rate
Enter the freight rate to use for this ticket.

Freight qty
Enter the freight quantity (for rate per ton calculations).

U/m
The unit of measure (um) for this line item displays.

Freight cost
Enter the total freight cost for this item.

Item code
The item code and description for this line item display. No entry is allowed. If multiple items were entered for this ticket, this field will display the item code as *MULTIPLE*. Click the Items button to view each item code for this multiple entry.
