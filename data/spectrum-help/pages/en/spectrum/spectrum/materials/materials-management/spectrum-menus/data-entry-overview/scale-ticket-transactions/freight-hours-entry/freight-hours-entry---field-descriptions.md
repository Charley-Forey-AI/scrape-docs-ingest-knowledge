# Freight Hours Entry - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Calculate Cycles
Click to view the [ Calculate Cycle Times](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/calculate-cycle-times) window.
Note: This button is only available if you have security authorization.

Quantity delivered
The total quantity delivered for all trucks stored on open tickets displays.

Number of trips
The total number of trips made by all trucks stored on open tickets display.

Average cycle time
The average cycle time for all trucks stored on open tickets displays.

Truck
Click to view the [Truck Cycle Details window](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/freight-hours-entry/truck-cycle-details-window) window.

List box

Truck
A list of truck codes displays for the open tickets.

Description
The descriptions associated with the corresponding truck codes display. This information defaults from the Hauler file.

Quantity delivered
The total quantity delivered for all trucks stored on open tickets displays.

# of trips
The total number of trips made by all trucks stored on open tickets display.

Cycle time
A list of cycle times associated with the corresponding truck code display.
 This information defaults from the Calculate Cycle
 Times screen.
