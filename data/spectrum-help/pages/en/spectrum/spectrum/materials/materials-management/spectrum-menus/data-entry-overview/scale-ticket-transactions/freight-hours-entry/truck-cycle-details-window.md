# Truck Cycle Details window

Use this window to edit freight hours information before tickets are posted to the software. If you are reconciling hourly billed freight information prior to posting it to an invoice, all edits that need to be made to the this information are done using this window.
Fields
Descriptions

Truck
The truck code that is selected on the main screen displays.

Tare weight
The tare weight of the selected truck displays

Ticket
Click to display the New/Edit Scale Ticket window.

Grid

Plant / pit
A list of plant codes displays.

Cycle time
A list of cycle times associated with the selected ticket displays.

Rate type
When you open this window, the cursor will display in this field. Use the drop-down menu to select the rate type. Options include Hour or Ton. Once you have made your selection, press Enter to proceed to the next field.

Freight rate
Enter the freight haul rate for this ticket, either in tons or hours, depending on the rate type you have selected.

Hours
Enter the number of freight hours for this ticket.

Freight charge
the software calculates the freight charge based on the freight rate you entered. This calculation can be overwritten.

Ticket total
the software calculates the ticket total.

Ticket type
The type of ticket--job, customer, transfer, or receipt--displays.

Name
The following will display for each different ticket type:

> Ticket type
Column

Job trips
Job code (description)

Customer trips
Customer code (description)

Receipts
Warehouse code (description for Plant / pit)

Transfers
Warehouse code (description)

Quantity
Enter any quantity information for the truck ticket in this field.
