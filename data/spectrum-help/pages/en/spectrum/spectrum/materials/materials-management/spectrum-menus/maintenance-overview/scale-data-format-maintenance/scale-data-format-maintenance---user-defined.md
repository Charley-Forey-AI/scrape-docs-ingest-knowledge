# Scale Data Format Maintenance - User-Defined

This tab displays the user-defined fields for invoice notes, dates, and amounts.
Field
Description

Edit
Click this button to display the Scale Import User-Defined Field Edit window.
Start-Stop positions display for the fixed width field, while Field # displays for the Comma-delimited field. Use this window to change these fields within the import file.
Once the field number or start/stop position is defined, the updated information will display in the Current Setting column in the User-Defined Fields window.

Field status
Click this Status filter button to select to view Currently set or Unused fields associated with this invoice.

Prompt
The title for this user-defined field (for example, DUE DATE) displays.

Current Setting
Valid user entries for this user-defined field display.
