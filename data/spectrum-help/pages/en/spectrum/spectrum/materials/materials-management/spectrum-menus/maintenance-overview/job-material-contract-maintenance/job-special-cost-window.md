# Job Special Cost window

Use this window to set up item codes with special costs
 that override the standard cost used on job tickets. Special costing can be used when the
 Transfer job costs with actual costs checkbox is clear on the Inventory Control
 InstallationProperties tab. When actual costs are used,
 then actual LIFO, FIFO, or average cost values from the Inventory Warehouse detail file are
 applied to jobs instead of the special cost.
Field
Description

Job
The job number and area code display based on the job and area code you
 selected in Job Material Contract Maintenance.

Area code

New/Edit/Delete
Click to add, modify, or remove items from the grid.

Item code
Enter the item code or search inventory items for the special cost you are setting up.

Cost
Enter the special standard cost to use for this contract.

Description
The description of the item you entered displays.
