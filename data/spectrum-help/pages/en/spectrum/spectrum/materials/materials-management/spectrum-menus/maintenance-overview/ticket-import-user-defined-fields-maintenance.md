# Ticket Import User-Defined Fields Maintenance

Use the Ticket Import User-Defined Fields
 Maintenance screen to set up virtually unlimited job-specific user-defined
 fields. This screen is used to view, add, edit, and delete scale ticket user-defined fields.
Note: User-defined fields are divided into two types:
 standard and unlimited. Standard user-defined field screens limit the number of
 user-defined fields to ten alphanumeric fields, five date fields, and five numeric
 fields.
The fields in this screen default based on the settings on the Materials Management Installation > User-Defined Fields tab.

- If the Show non-company-specific
 fields (company ALL) checkbox is selected on the Materials Management Installation > User-Defined tab, user-defined fields for all companies can be added and edited in
 this screen.

- If the Allow setup of
 company-specific fields checkbox is selected on the Materials Management Installation > User-Defined tab, company-specific user-defined fields can be set up for the
 current company.

- If the Allow setup of
 non-company-specific fields (company ALL) checkbox is selected on the Materials Management Installation > User-Defined tab, users are permitted to set up user-defined fields for all
 companies
