# Importing Scale Tickets From Other Software Packages

Ticket information can easily be imported into Spectrum
 from other scale software packages so that you do not need to re-enter scale information. If
 any errors occur during the import, this information is saved to a file that you can view, as
 needed.

1. On the Site Map screen, click Materials Management  >  Data Entry  >  ScaleTicket Data Entry.

1. Click the Import button.

1. At the Plant/pit field, enter a plant or pit code for the import, or press F4 to search valid plant/pit codes in Spectrum.

1. At the Data format field, the data format code will default based on the plant/pit you selected. This information can be overridden. Press F4 to select from a list of scale data formats.

1. At the Batch code field, the current batch code defaults. This information can be overridden. Press F4 to select from a list of batch codes.

1. At the Ticket date field, enter the ticket date for the import, or press F4 to view Spectrum's calendar.

1. At the Transfer file name field, the file transfer path will default from Scale Data Format Maintenance. Press Enter to accept the default, or override with a different path, if necessary. Click the folder icon to search your system's directories for the path to the file you want to import.

1. Click Continue.
