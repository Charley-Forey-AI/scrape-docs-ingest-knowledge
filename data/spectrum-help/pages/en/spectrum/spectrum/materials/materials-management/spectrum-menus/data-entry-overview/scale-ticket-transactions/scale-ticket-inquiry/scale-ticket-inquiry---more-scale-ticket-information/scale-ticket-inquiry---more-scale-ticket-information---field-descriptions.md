# Scale Ticket Inquiry - More Scale Ticket Information - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Description

Freight

Truck
The truck code for hauling and description display.

Truck gross weight
The truck's gross weight and tare weight display.

Truck tare weight

Hauler
The hauler code and description display.

Posted freight cost
Any posted freight and reconciled costs associated with this ticket display.

Reconciled cost

Cycle time
Cycle time is calculated only when a truck is assigned to two or more tickets. the software sorts the tickets by date and time for each truck. The time difference between tickets is the calculated cycle time. For example, if Truck A is assigned ticket 100 at 8:30 a.m. and ticket 200 at 10:00 a.m., then the cycle time is 1.5 hours for ticket 100.

Freight time
The freight hours for the selected hauler display.

Freight charge
The sum of the freight charges displays.

Freight taxable?
This checkbox is selected if the freight is taxable. The tax displays on the ticket total.

Customer ticket info

Customer's job
The customer's job number, salesperson, sales tax code and associated descriptions display.

Salesperson

Sales tax code

C.O.D.?
This checkbox is selected when this ticket is updated to the invoice and C.O.D. terms are used.

Job ticket info

Area code
The area code, Phase, and Cost type display.

Phase

Cost type

Reconciled cost details

Rate type
The rate type, freight hours, freight quantity, and freight charge display.

Freight hours

Freight quantity

Freight charge

Adjustments
Any adjustments made to the actual freight cost displays.

Vendor
The vendor's code and name display.

Invoice
The invoice number associated with the freight displays.

Freight charge details

Rate type
The rate type for the hauler used on this ticket displays.

Freight hours
The number of freight hours used for this hauler displays.

Freight quantity
The freight quantity associated with this ticket displays.

Freight rate
The freight rate associated with this ticket displays.

Freight charge
Any freight charges issued for this ticket display.
