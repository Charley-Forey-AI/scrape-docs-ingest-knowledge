# Customer Special Pricing window

This window displays the special pricing setup from the inventory special price setup for this customer. If the Inventory Control module is installed, this information is also accessible through Customer Special Price Maintenance in Inventory Control. For convenience, entry of special pricing may be performed in either screen.
Fields
Descriptions

Customer
The customer's code and job number display.

Customer's job

New/Edit/Delete
Click to add, modify, or remove items from the grid.

Grid

Code type
Select whether the code type is a category or an item.

Category / item
Enter the item or category code and press Enter to display the description.

Description

Price method
Select the type of pricing: Price, Discount, or Price table code. The field that displays next varies depending on the option you select here.

Amount
This field displays if you selected Price in the Pricing method field. Enter the special price amount for this customer and item code.

Multiplier
This field displays when the Price method is Discount. Enter the multiplier for this customer and item code.

Table code
This field displays when the Price method is Price table code. Enter the special price level for this customer and item code.

Effective from
Enter the from and to dates from which this special pricing is effective.

Effective to

Message
Enter any messages (for example, "Volume Discount/Valued Customer").
