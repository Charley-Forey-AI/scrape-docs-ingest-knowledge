# Job Material Contract Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job
In the New Material Contract window, enter or search for the job number.

Area code
Enter or search for the area code for which rates are to be assigned, or leave this field blank to assign rates to the entire job. The area entered here must have been previously set up in the Inventory Control module.
The area code is used to set up special rates for different areas within a job. If the job is large and has many different locations that have different haul rates or miscellaneous charges, then these can be set up using the area code.
The area code would have to be entered in the scale package along with the job number in order to use these rates.
If no area code is entered, then the software uses only the job number for the special rates. If the job is not set up here, then the Import Scale Tickets update looks at the hauler file for the rates and the Materials Management Installation screen for the miscellaneous charges, if any.

Properties

Misc. charge type
Enter the miscellaneous charge type: Fixed, Rate, Percentage, or None.

Misc. cost amount
Depending on your selection in the Misc. charge type field above, enter the miscellaneous cost amount (fixed), the rate, or the percentage amount, or press Enter to accept the software default.

Misc. charge category
Enter or search for the miscellaneous charge item category code, or press Enter to accept the software default.

Nonstock misc. item Nonstock misc. desc Nonstock misc. um
Enter the miscellaneous charge non-stock item code, description, and unit of measure to use, or press Enter to accept the software default.

Special Cost
Click to view the [Job Special Cost window](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/maintenance-overview/job-material-contract-maintenance/job-special-cost-window).

Hauler Rates
Click to view the [Hauler Special Rates window](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/maintenance-overview/job-material-contract-maintenance/hauler-special-rates-window).
