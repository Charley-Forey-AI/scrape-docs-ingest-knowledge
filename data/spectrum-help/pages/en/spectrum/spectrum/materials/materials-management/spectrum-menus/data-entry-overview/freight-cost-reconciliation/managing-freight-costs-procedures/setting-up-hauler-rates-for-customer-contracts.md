# Setting Up Hauler Rates for Customer Contracts

You can apply special hauler rates to a designated customer
 contract by going to Materials Management > Customer Contract.
If you do not set up
 special hauler rates for a customer contract, Spectrum will use the standard default hauler
 rates set up in Hauler File Maintenance. Note: This procedure must be completed when adding new hauler rates or updating existing
 hauler rates for a specified customer contract in Spectrum.

1. On the Site Map screen, click
 Materials Management > Maintenance > Customer Contract.

1. Complete the Customer and
 Customer's job fields for the customer contract hauler
 rates you are designating. Press F4 to select at either
 field.

1. Click the Hauler Rates
 button.

1. Complete the Hauler code field,
 press F4 to select a valid hauler code, or press
 Enter to accept the default hauler code.

1. In the Default freight rate
 type option box, select the default cost rate type for the selected
 hauler: Ton or Hour. The rate type
 specified here will default during Scale Ticket Date Entry.

1. Complete the Properties section
 of the window.Note: If any of the fields in the
 Properties section are left blank, rate information will
 alternatively default from Hauler File Maintenance during
 Scale Ticket Date Entry.

1. Click OK.

1. Repeat steps 4 – 7 for each additional special hauler rate you want to apply to the selected job contract.

Once you have added all
 of the special hauler pricing and completed the Customer Material Contract
 Maintenance screen, click OK to return to the main
 menu.
