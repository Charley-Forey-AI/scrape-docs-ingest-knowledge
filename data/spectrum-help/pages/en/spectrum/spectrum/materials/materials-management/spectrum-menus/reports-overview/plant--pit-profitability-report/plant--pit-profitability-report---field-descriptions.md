# Plant / Pit Profitability Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Plant / pit
Press Enter to select tickets from all plants for this report, or click the arrow to search for valid plant/pit codes.

Show unit prices?
Select to show unit prices on the report.
If you select this checkbox, there will be three lines per plant with the total quantity produced on the second line and the unit prices on the third line, as follows:
total $: xxxxx xxxxxx xxxxx xxx xxxxx xxxxx xxxxx xxx xxxxx xxxxx xxxxx xxx
qty produced: xxxxxxxx.xx xxxxxxx.xx xxxxxx.xx
unit price: xxx.xx xxx.xx xxx.xx xxx.xx xxx.xx xxx.xx xxx.xx xxx.xx xxx.xx

Period end date
Enter the period end date to apply to the report. Click the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)
 icon to select a calculation method.

Cost calculation
If you select Use standard cost, the total standard cost that is calculated and stored in the Materials Management balance file will be used.
If you select Use actual cost, that information will default from the plant/pit file in which a new field for actual cost job appears.
This is an optional entry; if there is an actual job for that plant and actual costs is selected, the report will calculate this total cost from the Job Cost cost balance file for this year and period. If no job is entered, then no actual cost will show on this report.
