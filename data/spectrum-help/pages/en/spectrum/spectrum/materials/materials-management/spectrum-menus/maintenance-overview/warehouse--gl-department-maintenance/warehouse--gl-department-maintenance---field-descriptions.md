# Warehouse / G/L Department Maintenance - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

Warehouse
In the Add Warehouse window, enter or search for a
 warehouse code (up to 10 characters) for this plant/pit. Entry in
 this required field allows the software to accurately track
 materials.

Stock sales G/L department
G/L department refers to Inventory Control G/L
 Department File Maintenance. The software will look first at the
 Inventory Control Category File Maintenance for defaults; if blank,
 it will then look at this file for defaults.
Enter or search for the inventory General Ledger
 department code for stock sales.

Freight G/L department
Enter or search for the inventory General Ledger
 department code for freight charges.
Entry here will be superseded by any G/L
 Department entry in the Hauler File Maintenance screen.

Misc. charges G/L department
Enter or search for the inventory General Ledger
 department code for miscellaneous charges.
