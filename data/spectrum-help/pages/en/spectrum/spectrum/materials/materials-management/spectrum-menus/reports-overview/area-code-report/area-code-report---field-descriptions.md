# Area Code Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Job Customer Plant/pit Salesperson Hauler Truck Item code Area/cust job Ticket # Phase Cost type P.O. #
Press Enter to preview ticket information from all job codes, Customer codes, plant/pits, salespeople, haulers, trucks, item codes, areas (for job tickets), customer jobs (for customer tickets), ticket numbers, Phase codes, Cost types, or purchase order numbers; or click the arrow to search for valid codes.

From date To date
Enter the from and through ticket dates to update. Or press Enter to leave
 blank and update from the first date or through the current Materials
 Management processing date. Click the arrow to select a date from the
 calendar or click the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)
 icon select a calculation method to determine the default date.

Ticket types
Select one or more ticket types to display on the report:

- Customer

- Job

- Receipt

- Transfer

Report type
Select the report format to display: Summary or Detail
