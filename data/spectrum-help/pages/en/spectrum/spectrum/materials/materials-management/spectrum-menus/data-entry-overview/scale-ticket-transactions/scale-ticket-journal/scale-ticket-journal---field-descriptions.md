# Scale Ticket Journal - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Batch code Plant/pit Ticket # Customer Job
Search for available batch codes, plant/pit codes, ticket numbers, customer codes, and job numbers, or press Enter to include ALL on the update.

From/To ticket date
Enter the from and through ticket dates to update. Or press Enter to leave
 blank and update from the first date or through the current processing date.
 Click the arrow to select a date from the calendar or click the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)
 icon select a calculation method.

G/L transaction date
Enter the General Ledger transaction date to use. The date must be within the Materials Management minimum/maximum processing date range.

Transaction type
Select the option that corresponds to the particular tickets that are to be updated.

- Job requisitions

- Invoices

- Transfers

- Receipts
