# Customer Contract File Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Customer
Press Enter to display information for all customers, or click the arrow to look up valid customer codes.

Press Enter to display information for all of the customers' jobs.

Show special pricing?
Select this checkbox to include special pricing on the report.

Select this checkbox to include special hauler rates on the report.
