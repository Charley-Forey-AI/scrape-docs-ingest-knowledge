# Mix Components window

This window displays all the detailed information relating to the sequence line you have selected on the Mix Processing Adjustments screen. Use this window to view the mix of components that are set up to produce the unit cost shown for this item.
You may also add/change component items and their respective quantities for this batch. When the scale tickets are updated to Order Processing/Accounts Receivable or Inventory, the Mix Processing Update will be performed. If the mix has been adjusted here for the batch being updated, then that will be used; otherwise, the software will process the standard batch that is set up in Inventory Control.
