# Scale Ticket History Inquiry

This inquiry screen is used to view information on scale tickets. A cross-reference file has been included to increase speed on this screen. Selections on job or customer, plant/pit, and date range will increase the speed significantly; if no selections are made on these fields, the inquiry will be much slower.
