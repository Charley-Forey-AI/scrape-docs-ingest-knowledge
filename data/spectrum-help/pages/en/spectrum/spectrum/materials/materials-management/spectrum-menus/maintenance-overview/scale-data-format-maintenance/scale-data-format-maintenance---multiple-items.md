# Scale Data Format Maintenance - Multiple Items

This tab allows multiple item code setup. Additionally, quantity and price can be specified for import for each item.
Field
Description

Data format
The data format displays.

Line
the software assigns each line transaction a sequence number as items are entered one by one.

Variable
The three variables per additional import item display (three variables per row). A description for each variable displays.

Type
The type of variable, Alpha or Numeric displays.

Max length
The maximum length for this variable displays.

Required?
Yes or No displays to indicate whether or not this variable is required.

Validated?
Yes or Nodisplays to indicate whether or not this variable requires validation.

Field#
Enter the field number for this variable. If the Data type field on the Properties tab is set to Comma-delimited, then this required field displays.

Start Stop
Enter the start and stop position for this variable. If the item is set to a fixed length, this field will read Start  >  Stop.
