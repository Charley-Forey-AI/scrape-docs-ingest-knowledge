# Job Material Contract Maintenance

Use this screen to set up certain defaults for tickets assigned to particular jobs. Specifically, the miscellaneous charges can be set up to default by job and area code onto the scale ticket. It is also possible to establish default freight charges based on the hauler code for the particular job.
You can also set up hauler default charges with a blank hauler code, which permits the setup of rates for a job without regard to area.
