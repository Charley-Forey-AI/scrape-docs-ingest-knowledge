# Reverse Tickets

Use the Reverse Tickets screen to reverse any range of
 tickets according to the data you enter in the fields on this screen
Once you have made your selections below for the pending reversal, click
 Next to proceed to the second [Reverse Tickets - Display](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/reverse-tickets/reverse-tickets---display) screen, which displays all of the tickets slated for reversal
 based on your selections on the current screen. You can remove any tickets that should not
 be reversed.
Once the reversal is complete, those tickets that have been reversed will be removed from the history file and added back to the current ticket file.
