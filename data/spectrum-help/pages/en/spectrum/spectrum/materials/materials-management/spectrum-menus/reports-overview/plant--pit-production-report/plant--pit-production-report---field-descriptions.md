# Plant / Pit Production Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Plant / pit
Press Enter to select tickets from all plants for this report, or click the arrow to search for valid plant/pit codes.

Item category
Press Enter to select items from all categories this report, or click the arrow to search for valid category codes.

Item code
Press Enter to select all items for this report, or click the arrow to search for valid item codes.

Period end date
Enter the period end date or use the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)
 to apply to the report.

Subtotal by item category?
Select this checkbox to display subtotals by item category; otherwise, leave this checkbox clear.

Display
Select to display the report by sale price or standard cost.

- Sale price

- Standard cost

Period type
Select which period type to display the report.

- Month-to-date

- Week-to-date

Week-to-date begin
Enter the period beginning date to include on the report. This field appears only if the Week-to-date option is selected.

Week-to-date end
Enter the period ending date to include on the report. This field appears only if the Week-to-date option is selected.
