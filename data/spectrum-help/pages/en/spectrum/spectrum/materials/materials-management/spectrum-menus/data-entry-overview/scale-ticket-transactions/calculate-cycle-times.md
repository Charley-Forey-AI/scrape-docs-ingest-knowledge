#  Calculate Cycle Times

Use the Calculate Cycle Times screen to calculate the cycle time, freight hours, and hourly freight amounts for selected tickets. Once you have made your selections, these freight amounts can then be viewed and corrected prior to updating this information in the software using the [Freight Hours Entry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/freight-hours-entry) screen.
It is not necessary to update using this screen, because the software will calculate the cycle time and hourly freight amount as a part of the ticket update. This screen simply allows you to view this information and make any necessary updates prior to the update.
