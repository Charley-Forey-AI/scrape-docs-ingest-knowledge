# Sales Journal Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Ticket types
If you the select either the Job or Customer ticket type, then those fields become available in the Selections section below. If you select All, then ALL jobs and customers will be automatically selected.

Report type
Select the report format to display: Detail or Summary

Job
If you selected the Job ticket above, enter or click the arrow to search for the job number to use for this report.
If you selected All tickets above, then ALL job tickets are automatically selected.
If you selected Customer tickets above, then this field becomes unavailable.

Customer
If you selected the Customer ticket above, enter or click the arrow to search for the customer code to use for this report.
If you selected All tickets above, then ALL customer tickets are automatically selected.
If you selected Job tickets above, then this field becomes unavailable.

Plant/pit
Press Enter to select tickets from all plants for this report, or click the arrow to search for valid plant/pit codes.

Salesperson
Press Enter to select tickets from all salespeople, or click the arrow to search for valid salesperson codes.

Hauler
Press Enter to select tickets from all haulers, or click the arrow to search for valid hauler codes.

Truck
Press Enter to select tickets from all trucks, or click the arrow to search for valid truck codes.

Item code
Press Enter to select all items for this report, or click the arrow to search for valid item codes.

Area/cust job
Press Enter to select tickets from all areas (for job tickets) and all customer jobs (for customer tickets), or click the arrow to search for valid area codes.

Ticket #
Press Enter to select all tickets for this report, or enter the ticket number you want to display on the report.

Phase
Press Enter  to select tickets from all phases for this report, or enter the phase number you want to display on the report.

Cost type
Press Enter to select tickets from all cost types for this report, or enter the specific cost type you want to display on the report.

P.O. #
Press Enter to select all purchase orders for this report, or enter the specific purchase order number you want to display.

From date
Enter the date from which to preview, or press Enter to leave blank and preview from the first date.

To date
Enter the from and through dates to display on the report. Or press Enter to leave blank and display the report from the first date or through the current Materials Management processing date.
Click the arrow to select a date from the calendar or click the [Date Calculator](/en/spectrum/spectrum/getting-started/spectrum-interface-faqs/user-interface-elements/date-calculator)
 icon select a calculation method.
