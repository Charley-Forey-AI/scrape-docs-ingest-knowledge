# Freight Cost Journal/Update - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

Vendor code
Press Enter to update and create invoices for all
 vendors, or use SuperSelect to enter the desired selection criteria. Click
 on the Info icon for assistance with SuperSelect. A search window is
 available for viewing valid vendor codes.

Invoice no.
Press Enter to update and create invoices for all
 vendors, or use SuperSelect to enter the desired selection criteria. Click
 on the Info icon for assistance with SuperSelect. A search window is
 available for viewing valid vendor codes.
The system validates the invoice numbers entered here to
 make sure that the numbers do not already exist in Accounts Payable. If any
 of the invoice numbers have already been used, a Freight Cost Error Report
 will be created. Use the Freight Cost Reconciliation screen to replace the
 duplicated number with a unique number.

Through invoice date
Include invoices through this date. The system defaults
 to the current Materials Management processing date.

G/L date
Enter the General Ledger date to use for this update.
 This date must be within the Materials Management minimum/maximum processing
 date range. The system defaults to the current Materials Management
 processing date.

Assign VAT code?
If the Utilize value added tax (VAT) tracking? checkbox is selected
 in Company Installation, select this checkbox to add VAT to the vendor
 invoice. When selected, specify the Tax code below.
