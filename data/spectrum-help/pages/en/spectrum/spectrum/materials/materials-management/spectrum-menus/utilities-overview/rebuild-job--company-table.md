# Rebuild Job / Company Table

Use this screen to rebuild the Materials Management job / company table by adding all jobs from all designated companies, and automatically creating records in the Job / Company File Maintenance screen.
Click Continue, and a confirmation box will indicate whether the update has been completed.
Field
Description

Company code
Enter ALL or specific company codes for jobs to include in the build.

Job status
Select at least one Job status.

- Include active jobs

- Include inactive jobs

- Include complete jobs
