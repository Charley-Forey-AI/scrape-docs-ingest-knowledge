# Scale Data Format Maintenance - File Layout

Use this tab to determine the appearance of imported information and to recalculate ticket pricing for cash-only sales or charge only sales. More Info Cash-only sales are determined using the COD? checkbox in the Import Scale Tickets screen. If a ticket contains both a job and a customer, the job code will be used as long as it is valid. The customer code will be used only if the job code is blank or invalid. This allows the scale program to record the customer's job number in a memo capacity. If both the job and the customer fields are in the file for a ticket, but the job is not set up in Spectrum, the job information is not used for pricing purposes.
