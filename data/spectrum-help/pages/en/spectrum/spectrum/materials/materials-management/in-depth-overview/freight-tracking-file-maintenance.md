# Freight Tracking File Maintenance

The Hauler File Maintenance, Customer Material Contract Maintenance and Job Material Contract Maintenance screens allow you to designate rates per ton or rates per hour for freight costs. This is done in each screen by designating the rate type as Ton or Hour. The appropriate file listings for these maintenance screens also show the cost rate method Ton or Hour.
