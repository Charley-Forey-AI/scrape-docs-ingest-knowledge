# Multiple Items on Ticket

Use the Multiple Items on Ticket window to view items by code for the ticket highlighted in the Error Display line transactions, along with the items' corresponding detailed information, including unit of measure, quantity, unit price, and so forth. These items can be changed as needed.
You can make corrections in the Item code, Quantity, Unit Price, and Sell price columns if errors exist.
