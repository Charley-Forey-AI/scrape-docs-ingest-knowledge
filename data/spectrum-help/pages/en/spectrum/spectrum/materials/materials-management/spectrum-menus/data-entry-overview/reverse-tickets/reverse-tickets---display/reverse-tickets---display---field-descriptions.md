# Reverse Tickets - Display - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Fields
Descriptions

Buttons

Remove from List
Click to remove the highlighted row from the list of tickets to be reversed.

Ticket
Click to open the [Scale Ticket Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/scale-ticket-inquiry) window.

Invoice
Click this button to display the Order Processing Invoice Inquiry or A/R Invoice window. You can look up invoices associated with the selected reference number. This window is available only for invoice transaction types.
This button is inactive if the highlighted row is a Job or Inventory transaction.

Alert
Click to open the Tickets on Transaction window, which displays all of the tickets associated with the selected reference number as well as any alerts that may have been entered for any of the tickets displayed.

List box

Transaction type
The transaction type displays: Customer, Job, Receipt, or Transfer.

Reference #
A reversal reference number for each transaction displays. A reference number can contain any number of tickets; to view the tickets attached to a reference number, highlight the reference number and click the Tickets button. Clicking this button will also display any warnings associated with any of the referenced tickets.

Date
The date of each referenced transaction displays.

Plant / pit
The plant code associated with the corresponding reference number displays.

Ticket
The ticket number associated with the reference number displays. If more than one ticket is associated with the corresponding reference number, the number of additional tickets will display in parenthesis. For example, if there are four tickets associated with a reference number, then the first ticket number will display followed by a note in parenthesis that reads (3 more).

Ticket type
The type of ticket displays: Job, Customer, Transfer, or Receipt.

Name
The following displays for each name:

- Job trips: Code and description

- Customer trips: Customer code and description

- Receipts: Warehouse code and description

- Warehouse trips: To warehouse code and description

Alert
If any warnings have been selected for a referenced ticket, a Yes will be indicated in this column. Click the Tickets button to display the Tickets on transaction window and view the specific warning message(s).
 Keep in mind that unless deleted from this list, even tickets marked with warnings will be reversed.
If any warning flags display in the Warn column, highlight the column with the warning flag and click the Tickets button to review the warning message.
