# Truck File Maintenance

Use this screen to identify the trucks used for deliveries. If the Convert net weight checkbox is selected, then the New/Edit Scale Ticket screen will convert net weight from pounds to tons when calculating from the tare weight (tare weight is entered in pounds; quantity is tracked in tons).
