# Material Purchase Reconciliation - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Vendor
Enter the vendor code, or press F4 to view a list of
 outside material vendors.
Note: If the vendor's status is Not used, then entry is not allowed.

Invoice #
Enter the invoice number. If you specified a vendor at the Vendor
 code field, you can click the Preview button from this
 field to preview a listing of all unassigned tickets associated with the
 vendor you selected.
The system validates the invoice numbers entered here to make sure that the numbers do not already exist in Accounts Payable (including any rejected invoices). If any of the invoice numbers have already been used, an error message will display. Use this screen to replace the duplicated number with a unique number.

Invoice date
The invoice date defaults from the selected invoice, but can be changed. This information is later transferred to the A/P Invoice/Credit Memo Entry.
Note: If the Stop entry if invoice date after maximum date check
 box is selected in the Accounts Payable Installation > Properties tab, the date entered in this field must fall before the
 maximum Accounts Payable processing date.

Tickets included
The number of tickets that are selected in the grid displays.

Reconciled quantity
The total hours, amount, and cost reconciled displays.

Reconciled cost

Buttons

Include All
Click to select all items in the list box below.

Clear All
Click to clear all selected items.

Search
Click to display the [Freight Cost Reconciliation - Search Transactions](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/freight-cost-reconciliation/freight-cost-reconciliation---field-descriptions/freight-cost-reconciliation---search-transactions) window.

Ticket
Click to open the [Scale Ticket Inquiry](/en/spectrum/spectrum/materials/materials-management/spectrum-menus/data-entry-overview/scale-ticket-transactions/scale-ticket-inquiry) window for the selected item.

Grid

Include
Select this checkbox to reconcile the outside material purchase.

Plant/pit
The plant code displays. No entry is allowed.

Ticket
The ticket number displays. No entry is allowed. If multiple items were entered for this ticket, this screen lists the ticket number once for each item, permitting each item to be reconciled separately.
Click the Ticket button to view the entire history of
 this ticket.

Date
The ticket date displays. No entry is allowed.

Item
The item code displays. No entry is allowed.

Quantity
Press Tab to access this field. The outside material
 purchase quantity display. Press Enter to accept, or
 enter an alternate quantity.

U/m
The unit of measure (UM) displays. No entry is allowed.

Unit cpst
Press the Tab key to
 access this field. The outside material unit displays. Press
 Enter to accept, or enter an alternate unit.

Total cost
Press the Tab key to access this field. The actual
 outside material total cost displays. Press Enter to
 accept, or enter an alternate total cost.
Note: If job contract special costing is present, then cost will be based on contract special cost instead of inventory special cost.

Hauler
The hauler code and description display.

Ticket type
The types of tickets are: Job,
 Customer, Transfer, or
 Receipt.

Name
The following displays for each name:

- Job trips: Code and description

- Customer trips: Customer code and description

- Receipts: Warehouse code and description

- Transfers: Warehouse code and description

Item description
The item code description displays.
