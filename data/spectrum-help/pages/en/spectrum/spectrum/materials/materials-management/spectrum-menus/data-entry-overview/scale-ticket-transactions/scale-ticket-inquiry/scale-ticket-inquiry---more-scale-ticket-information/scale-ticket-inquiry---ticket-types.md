# Scale Ticket Inquiry - Ticket Types

If the ticket type is Customer, the Customer ticket info section displays. You can view the customer's job, salesperson, and sales tax information.
If the ticket type is Job, the Job ticket info section displays. You can view the area code, phase, and cost type information.
