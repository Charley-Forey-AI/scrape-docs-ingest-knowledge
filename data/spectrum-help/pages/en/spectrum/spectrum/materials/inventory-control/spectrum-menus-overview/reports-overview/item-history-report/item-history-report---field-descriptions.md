# Item History Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Warehouse Item code Category ABC class Job Equipment Transaction reference
Enter a warehouse code, item code, category code, ABC class code, job code, equipment code, and transaction reference code, or press Enter to preview ALL codes.

Transaction date
Press Enter if the date range of the transactions on this report is to begin with the first transaction date on file and end with the current Inventory Control processing date. If this report is to begin and end with specific transaction dates, enter those dates in these fields.

Transaction types to include
Select the transaction types you want to include on the report:

- Adjustments

- Cost adjustments

- On-order

- Receipts

- Transfers

- Sales

- Returns

- Job requisitions

- Mixes / usage

- Equipment

Report format
Select how to sort this report:

- Warehouse/item

- Warehouse/transaction

- Job/item

- Warehouse/item/job
