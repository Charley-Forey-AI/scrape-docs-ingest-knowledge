# Change Item Codes - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field
Description

Old item code
Enter the existing item code you wish to change, or
 double-click in this field to select from a list of valid codes. Entry in
 this field is required.

New item code
Enter the item code to which you wish to change the
 existing code. Entry in this field is required.
A note about use of numeric coding:
Because many users prefer alphabetic or combination alpha
 and numeric coding, this codes is not a numeric-only field. Because of this,
 the code left-justifies when entered. Users preferring a strictly numeric
 coding scheme should be advised to use leading zeros when adding codes in
 order to produce desired sort results on screens and reports. Without
 leading zeros, '01', '10', '100' will appear before '2', '20', '200'.
 Instead, codes should be set up '001', '002', and more. If more than 1000 codes
 are anticipated, instead enter '0001', '0002', and more.
