# Warehouse File Maintenance- Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, the Warehouse File Maintenance screen includes a Cost center field. This field associates the warehouse with a particular cost center.
When adding a new warehouse, the Cost center field must be completed, and the entry must be a valid cost center for the operator who is adding the warehouse.
When editing, viewing, or deleting a warehouse, Spectrum compares the warehouse's assigned cost center with cost centers in the operator's assigned scheme; if the warehouse's cost center is not present, then that warehouse code is not accessible.
Throughout Spectrum, the Search Warehouse window displays only warehouses associated with cost centers that the operator is authorized to access.
