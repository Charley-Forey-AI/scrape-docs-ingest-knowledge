# Item Warehouse Details - Field Descriptions - Cost Center
 Information

If the cost center feature is enabled in the Enterprise
 Installation screen, Spectrum will allow an item to be entered only if the operator has
 permission to access the item. Spectrum compares the item's list of shared cost centers with
 cost centers in the operator's assigned scheme; if there are no common cost centers, then that
 item cannot be entered.
Spectrum will verify that the warehouse has a valid and active cost
 center set up and will not allow the warehouse to be entered if one is not set up. The
 warehouse must also be in the list of valid cost centers for the item. After an item has
 been set up for a warehouse, all operators that have access to the item will have access
 to warehouse details for that item in this screen. The same validation takes place on
 the Details tab of the New/Edit Inventory Item window when adding a warehouse for an
 item.
