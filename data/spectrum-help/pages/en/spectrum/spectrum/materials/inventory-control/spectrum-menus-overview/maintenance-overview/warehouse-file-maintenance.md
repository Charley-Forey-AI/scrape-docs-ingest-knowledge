# Warehouse File Maintenance

Use the Warehouse File Maintenance screen to set up the
 different warehouses that you want to track inventory quantities.
 You must set up at least one warehouse
 in order to sell and receive items. The warehouse code is prompted for in all sales
 transactions (for example, Order Processing, Equipment Tracking, Materials Management, Work
 Order), receiving transactions (Purchase Order, Inventory Control), and job requisitions
 (Inventory Control).
If you want to track inventory for different control purposes (for example, for each service truck, or for different locations within the same warehouse), additional warehouse codes may be set up for this purpose. In addition, if you are using the Materials Management module for tracking pits or plants, each pit or plant would be set up as a different warehouse.
You can also choose to default the General Ledger department code by warehouse. This is necessary for companies that want to track inventory adjustments, usage, and so forth, by each G/L department.
