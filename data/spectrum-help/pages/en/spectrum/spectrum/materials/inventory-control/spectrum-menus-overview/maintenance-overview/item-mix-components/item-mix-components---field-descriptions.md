# Item Mix Components - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Item code
Enter the desired item code for the new "parent" item; this must be a valid item code as defined in Inventory Items.

Category
The category code of the specified item displays.

Standard cost
The standard cost of the specified item displays.

Detail grid

Component item
Enter the item code for the component going into this assortment. This must be
 a valid item code as defined in Inventory Item File Maintenance.

Item description
The description of the component item code displays.

Standard cost
The standard cost of the specified item displays.

Quantity per mix
Enter the number of units of the component item code to be used in preparing one assortment.
Repeat these steps until all component items have been entered. As each line is completed, the standard cost field above the table reflects the cost of the additional components.

Category
The category code and description display in this field.
