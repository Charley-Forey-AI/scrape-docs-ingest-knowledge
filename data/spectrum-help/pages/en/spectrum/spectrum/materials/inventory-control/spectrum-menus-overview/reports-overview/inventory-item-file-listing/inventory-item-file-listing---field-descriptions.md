# Inventory Item File Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Item Category Discount code ABC class
Enter the item code, product category code, discount code, and ABC class code to be included on the report, or press Enter to include ALL codes on the report.

Include item remarks?
Select this checkbox to include any item remarks.

Taxable status
If both taxable and nontaxable inventory items are to be included in the report, select All. Otherwise, select to include either sales taxable items or nontaxable items in the report.

Commission status
If both commissionable and non-commissionable inventory items are to be included in the report, select All. Otherwise, select to include either commissionable or non-commissionable inventory items in the report.

Status
Select whether to print Active,
 Discontinued, and/or Not used items.
Note: Items are designated as "discontinued" in the Inventory Item Maintenance
 screen. The Discontinued checkbox only affects
 Order Processing orders and invoices, and if an operator attempts to use
 a discontinued item, a warning message will display in the Order Processing > Invoice Entry, Order Entry, and Quote
 Entry screens (Work Order, Purchase Order, and Inventory
 Control job requisitions are not impacted).
