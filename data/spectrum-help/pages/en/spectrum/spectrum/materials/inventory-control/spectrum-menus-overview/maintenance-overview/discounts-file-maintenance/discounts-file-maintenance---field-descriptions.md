# Discounts File Maintenance - Field Descriptions

Use the table below for reference when completing the
 fields on this screen.
Field/Button
Description

Discount code
Enter the discount code.

Description
Enter the discount code description.

Discount percent
Enter the discount percentage associated with this
 discount code.

Effective from
Enter the date the discount percentage takes effect. The
 discount will be applied if the transaction date is on or after the date
 entered.

Effective to
Enter the discount percentage end date. The discount will
 be applied if the transaction date is on or before the date entered.
