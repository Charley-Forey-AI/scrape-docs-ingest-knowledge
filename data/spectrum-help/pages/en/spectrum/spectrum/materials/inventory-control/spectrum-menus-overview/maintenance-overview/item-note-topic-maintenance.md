# Item Note Topic Maintenance

Use the item note topic maintenance screen to add or edit
 notes for items.

1. Click the Edit button to add notes for the selected topic. Topics that have existing notes will be noted by a green dot. Up to 999 lines of notes are allowed per item.

1. Select a different item to add more notes, if available.

1. Click Save.
