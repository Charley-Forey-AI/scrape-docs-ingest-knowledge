# Purge Item
 History

The Purge Item History screen is used to remove older
 transactions from the item history file.
 This purge will not produce a report
 during the purge procedure; therefore, print an Item History Report with the same selection
 parameters before performing this purge.
Important: The Item History
 Report should be run prior to this purge and retained as a permanent audit record of the
 company.
