# Inventory G/L Report/Update - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Batch
Enter the unposted inventory batch code in this field. If this screen was accessed from the Inventory Transaction Report/Update screen, the selected batch will default in this screen, otherwise it will default to ALL batches.
