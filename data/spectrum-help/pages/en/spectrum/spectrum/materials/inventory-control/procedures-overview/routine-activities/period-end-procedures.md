# Period End Procedures

Learn about the period end procedures.

- At the end of each fiscal period, or on demand, print the Inventory Activity Report, which summarizes sales figures and profitability for each inventory item. File the report for reference.

- Items-On-Hand Valuation Report is available at any time to show the total dollar value of items in inventory. This report is often used at period end to determine the asset value of inventory for accounting purposes.
