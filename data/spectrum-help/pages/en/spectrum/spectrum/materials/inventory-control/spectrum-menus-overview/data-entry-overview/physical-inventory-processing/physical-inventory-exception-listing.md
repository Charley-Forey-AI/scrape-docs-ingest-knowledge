# Physical Inventory Exception Listing

This screen prints a report of items not entered during Physical Count Data Entry, which is used to identify items inadvertently omitted during a complete Physical Inventory. The report should be reviewed for item codes that should have been entered in Physical Count Data Entry.
This report should not be printed if the Physical Inventory covers only a portion of the total inventory.
This selection is only available for use when a physical inventory is in progress, previously initiated using Physical Inventory Status Update.
