# Item/Warehouse Detail Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Warehouse
Enter the warehouse code to be included on the report, or press Enter to accept the software default and print ALL warehouse codes. Warehouse codes may be up to 10 characters long.

Category
Enter the category code to be included on the report, or press Enter to accept the software default and print ALL category codes.

Item code
Enter the item code to be included on the report, or press Enter to accept the software default and print ALL item codes.

Discount code
Enter the discount code to be included on the report, or press Enter to accept the software default and print ALL discount codes.

Print quantity detail?
If quantity and cost detail is to be printed on the report, select this checkbox. If only summary data is to be printed, leave this checkbox clear. If this checkbox is selected in this field, each of the quantities and corresponding costs of each of the 20 buckets is included on the report.

Sort by
Select Item / warehouse if the Item/Warehouse Detail Listing is to print in order first by inventory items and then by warehouses. Select Warehouse / item if the list is to print in order first by warehouse and then by items in that warehouse.

Taxable
If both taxable and nontaxable inventory items are to be included in the report, select Both. Otherwise, select either Taxable only or Nontaxable only, depending on what you want to appear on the report.

Commissionable
Select the type of inventory items you want to appear on the report:

- Both

-  Commissionable only

-  Noncommissionable only

Status
Select whether to print Active,
 Discontinued, and/or Not used items.
Note: Items are designated as "discontinued" in the Inventory Item
 Maintenance screen. The
 Discontinued checkbox only affects Order
 Processing orders and invoices, and if an operator attempts to use a
 discontinued item, a warning message will display in the Order Processing > Invoice Entry, Order Entry, and Quote
 Entry screens (Work Order, Purchase Order, and Inventory
 Control job requisitions are not impacted).
