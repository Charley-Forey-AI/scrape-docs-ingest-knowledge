# Mix Pick
 Sheet

The Mix Pick Sheet report shows the components for a mix
 assembly with the mix item in the header of the report--even if that item has not been added
 to the warehouse yet.
 The mix assembly transaction cannot be
 processed until the mix item has been added to the warehouse.
