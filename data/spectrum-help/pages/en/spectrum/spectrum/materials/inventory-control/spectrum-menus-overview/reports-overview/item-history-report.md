# Item History Report

Every inventory transaction is recorded in the inventory
 history file. Transactions include receipts and credit returns, adjustments, and transfers
 among warehouses and sales.
The Item History Report details these
 transactions as a tool for isolating discrepancies between system records and the actual
 physical count.
