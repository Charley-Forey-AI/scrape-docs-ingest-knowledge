# Prerequisites

Accounts Receivable must be installed on the software for the Customer Special Price Maintenance to operate fully; however, if it is not this function can be used in a limited way. Special pricing information may be maintained for customers; but information about the customers themselves, such as names, addresses, and telephone numbers, cannot be maintained in Inventory Control.
This screen is integrated with the Work Order and Order Processing modules. If Materials Management is installed, this information is also accessible through Customer Contract File Maintenance. For convenience, entry may be performed in either screen.
