# Reorder Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Warehouse
If inventory from only one warehouse is to appear on the report, enter that warehouse code. If all warehouses are to be included in the report, press Enter.

Vendor code
If items purchased from a specific vendor are to be included on the report, enter that vendor code. If items purchased from all vendors are to be included in the report, press Enter.

Category
Enter a category code, or press Enter to include ALL category codes.

Item code
Enter an item code, or press Enter to include ALL item codes.

ABC class
Enter an ABC class code, or press Enter to print ALL codes.

Reorder point
This field determines what items will be printed by comparing the quantity available to the reorder point (safety stock). The lower the number entered, the closer the quantity available must be to the allowed safety margin. Entering 1% will list items that need to be ordered immediately; leaving the field blank will list all warehouse items, regardless of the reorder point.

As of date
Enter the date to be used for purposes of quantity history on the report, or press Enter to accept the current Inventory Control processing date.
This report contains information appearing under a column titled "turnover." This information is calculated as follows:

Total of the Month-End available for 12 months*
%
12

Select an item status to filter report results.
