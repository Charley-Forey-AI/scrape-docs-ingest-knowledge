# Item
 Location File Maintenance

The Item Location File Maintenance screen is used to add
 and maintain an item's location within a specified warehouse.
This information is then used in Item
 Detail File Maintenance. Location tracking may be required or optional if the Validate item
 location checkbox in Inventory Control Installation is selected.
