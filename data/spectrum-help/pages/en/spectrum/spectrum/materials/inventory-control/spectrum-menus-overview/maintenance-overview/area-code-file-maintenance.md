# Area Code File Maintenance

The Area Code File Maintenance screen is used for pricing
 options by area of work and for categorizing on inquiry screens and reports.
Once set up, the user is allowed to set
 up special pricing by job and area code for categories of items or specific items using Job
 Contract File Maintenance in Materials Management.
Field
Description

Area code
Enter the area code in this field.

Description
Enter the corresponding area code description in this field.
