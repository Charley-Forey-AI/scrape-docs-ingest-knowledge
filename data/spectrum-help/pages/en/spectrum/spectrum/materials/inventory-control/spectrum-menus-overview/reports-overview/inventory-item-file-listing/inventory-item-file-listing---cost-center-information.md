# Inventory Item File Listing - Cost Center Information

If the cost center feature is enabled in the Enterprise Installation screen, the this screen includes a selection for a Cost group. When a cost group is specified, then the report will show only items assigned to that cost group. Spectrum compares the item's list of shared cost centers with cost centers in the operator's assigned scheme; if there are no common cost centers, then that item is not shown.
