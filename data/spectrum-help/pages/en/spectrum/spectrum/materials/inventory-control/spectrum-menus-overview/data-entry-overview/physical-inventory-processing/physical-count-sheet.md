# Physical Count Sheet

The Physical Inventory Count Sheet is a worksheet that is
 used to record inventory during the actual counting of stock. It prints by warehouse, in item
 location sequence, to facilitate speed of counting.
 The item location should already have
 been entered in the Item Detail File Maintenance or the bottom of Inventory Item File
 Maintenance. If no locations are assigned, the report will print in item code order within
 the warehouse.
Tip: Some companies without pre-assigned locations on file take the opportunity to note warehouse location when using this; then, at a convenient time before the next physical inventory, item location information can be updated using Item Detail File Maintenance.
