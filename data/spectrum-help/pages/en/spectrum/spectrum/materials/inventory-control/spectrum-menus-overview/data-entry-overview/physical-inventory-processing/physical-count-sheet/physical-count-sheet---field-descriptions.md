# Physical Count Sheet - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Warehouse
The system defaults to the warehouse code selected in the Physical Inventory Status Update. To print the default sheets, press Enter. To print sheets for another warehouse, enter the warehouse code. Warehouse codes may be up to 10 characters long.

Item location
Enter a valid item location code, or press Enter to include all the locations on file.

ABC class
Enter the first ABC class that is to appear on this report, or press Enter to include all of the ABC classes on file.

Include discontinued items?
Select this checkbox to include discontinued inventory items on the count sheet.
