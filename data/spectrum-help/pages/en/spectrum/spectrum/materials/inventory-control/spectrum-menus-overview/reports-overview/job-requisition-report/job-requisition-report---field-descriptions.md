# Job Requisition Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Unposted requisition
Enter the unposted requisition to include on the report, or press Enter to accept the software default of ALL.

Requisition status
Select to print Shipped requisitions only, Open requisitions only, or All requisitions.

Sort by
Select to sort the report by Requisition number or by Job.
