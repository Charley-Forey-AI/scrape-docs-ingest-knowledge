# Copy Items to New Warehouse - Field Description

Use the table below for reference when completing the fields on this screen.
Field
Description

Item
Enter the item code with which to begin this transfer, or press Enter to transfer ALL.

Category
Enter the category code with which to begin this transfer, or press Enter to transfer ALL.

Copy from warehouse
Enter the warehouse from which to copy information. Warehouse codes may be up to 10 characters long. Entry in this field is required.

Copy to warehouse
Enter the new warehouse to which you wish to copy information. Warehouse codes may be up to 10 characters long. Entry in this field is required.

Copy reorder point?
Select this checkbox to copy the reorder report; otherwise, leave this checkbox clear.
