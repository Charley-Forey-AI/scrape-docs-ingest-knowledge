# Warning - Item Does Not Exist In Warehouse

If an item does not exist in the warehouse selected on the Mix Assembly Transaction Entry screen, this message displays.
To change the warehouse, click OK until you return to the Whs field in the details portion of the Mix AssemblyTransactionEntry screen. Manually enter a different warehouse code or press F4 to select from a list of warehouse codes.
