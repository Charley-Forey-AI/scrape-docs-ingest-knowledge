# Set Taxable Flags for Items - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Item
Enter the item code you wish to set the sales tax and use tax subject flags for, or press Enter to set taxable flags for ALL item codes.

Category
Enter the category code you wish to set the sales tax and use tax subject flags for, or press Enter to set taxable flags for ALL category codes.

Sales tax subject to
Specify whether the item and category code(s) are subject to sales tax:

- Yes, item is taxable

- No, item is not taxable

- Don't change

Use tax subject to
Specify whether the item and category code(s) are subject to use tax:

- Yes, item is taxable

- No, item is not taxable

- Don't change
