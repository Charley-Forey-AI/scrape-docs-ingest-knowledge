# Items-on-Hand Valuation Report - Cost Center Information

If the cost center feature is enabled in the
 Enterprise Installation screen, the Cost group field will appear in the
 Selections section to specify which cost centers to include on the
 report. This report will only include items in warehouses that the operator has authorization
 to access, based on the cost center assigned in Warehouse File
 Maintenance. Spectrum compares the warehouse's assigned cost center with cost
 centers in the operator's assigned scheme; if the cost center is not present, then items in
 that warehouse will not be included.
