# Clear Physical Inventory Status

The Clear Physical Inventory Status screen releases the
 Inventory Control module to return to normal operations.
 This selection is the final step in
 Physical Inventory Processing and has the effect of canceling limitations placed on
 operations through Set Physical Inventory Status. This would normally be the final step of
 Physical Inventory Processing.
To assure that all physical count transactions are fully updated, the clearance will be disallowed if there are Adjustment entries remaining in the Inventory Transaction file. This condition would occur if the Update Physical Inventory is performed, but not the Inventory Transaction Update.
Note: Items and quantities
 entered in Physical Inventory Count Entry will not be cleared. These entries should be
 cleared manually, if desired.
