# Adjust Price Levels - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Markup based on
Select what you would like the markup to be based on:

- Standard cost

-  List price

Item
Enter the item to be included on this update, or press Enter to accept the software default of ALL.

Category
Enter the category to be included on this update, or press Enter to accept the software default of ALL.

Primary vendor
Enter the vendor to be included on this update, or press Enter to accept the software default of ALL.

Minimum cost amount
If Standard Cost is selected, the software displays the minimum cost amount field. If List Price is selected, the software displays the minimum price amount field. Enter the minimum cost/price to be updated.

Maximum cost amount
If Standard Cost is selected, the software displays the maximum cost amount field. If List Price is selected, the software displays the maximum price amount field. Enter the maximum cost/price to be updated.

Set price levels
Enter the percentage by which to increase or decrease the standard cost. This first price level field is only available if the markup is based on the standard cost (instead of the list price). Repeat for levels 2-5 for standard prices and list prices.
