# I/C Processing Date Change - Field Definitions

Use the table below for reference when completing the
 fields on this screen.
Field/Button
Description

Processing date
The left column displays the current processing
 information. The information is re-displayed on the calendar on the right.
 Enter the new processing date by typing over the displayed date, or click
 the date on the calendar. To leave the processing date as it is displayed,
 click the Reset
 button to restore the present setting. Click Cancel to exit the screen
 without saving the changes.
 If the new date is before the Minimum Processing Date or
 after the Maximum Processing
 Date established during the Installation procedures, an error
 message will display.

Fiscal year
The fiscal year in which the specified processing date is
 contained displays.

Period
The period number in which the specified processing date
 is contained (number 1-13) displays. The period start and end date for the
 specified processing date displays.

Allowable entry range
The earliest and latest dates that a processing date can
 be changed to is displayed in this field. These dates are set in the Admin > Installation > Processing Date Change screen.

Reset
Click this button to return the field selections on this
 screen to their default settings.
