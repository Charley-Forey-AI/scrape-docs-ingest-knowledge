# Item/Warehouse Detail Listing

The Item/Warehouse Detail Listing is a report of all
 warehouses on record.
Information found in the Item Detail File
 Maintenance screen is printed in warehouse code sequence. If printed for quantity detail,
 the contents of each of the warehouses (up to 20 aged lines) will print; otherwise, there
 will be a summarization of the quantity and cost on hand. Only those items currently
 on-hand will be included on the listing.
