# Inventory Transaction Report / Update - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Batch
Enter the unposted inventory batch code in this field. The current operator code will default in this field, but can be changed. The selected batch code will exclude transactions for other batches and will be saved in Inventory History when the transaction is updated.

Transaction types to include
Select which transaction types to include on the report:

- Job requisitions

- Equipment parts

- Transactions

- Mixes
