# Correcting an Inventory Item set up with the Wrong UM

Learn how to correct an inventory item that has been
 assigned the wrong unit of measure (UM). The new UM will only be applied to new
 transactions.

1. On the Site Map screen, click Inventory Control  >  Maintenance  >  Inventory Item.

1. Enter the item code for the inventory item you want to correct or press F4 to select a valid item code.

1. If the unit of measure is incorrect for the item when it was purchased: In the Purchase section of the screen, in the Unit of measure field, enter the updated unit of measure abbreviation, for example, PR for pairs, DZ for dozen, SL for sling, or EA for each. If the unit of measure is incorrect for the item when it was sold: In the Sell section of the screen, in the Unit of measure field, enter the updated unit of measure abbreviation, for example, PR for pairs, DZ for dozen, SL for sling, or EA for each.

1. Click OK until you return to the main menu.
