# Change Item Codes Update

The change item codes update is used to update the
 information entered in Inventory Item Number Change Maintenance by
 changing the inventory item numbers system-wide.
 The user should first have an accurate
 data backup and obtain a printout using Inventory Item Number Change Listing before
 proceeding with this update.
Important: This update will
 perform code changes throughout the software. It is a productivity tool to assist you in
 making code changes on a wide scale. Be advised that any custom files, pre-release and new
 files, and non-updated data files may not have been included in this automated process.
 Therefore, confirm that updates have been performed prior to using this feature. Review
 records carefully, including installation screens, immediately following completion of the
 program. THIS UPDATE SHOULD NEVER BE PERFORMED WHILE OTHERS ARE LOGGED INTO Spectrum.
 Confirm that this workstation is the only one currently in use prior to proceeding with the
 update.
