# Adjust List Price/Standard Cost - Field Description

Use the table below for reference when completing the fields on this screen.
Field
Description

Change type
Select Price to update the price of inventory items, or select Cost to update the cost of the inventory item(s).

Category
Enter the category code that you want to update. A list of codes is available by running the Category File Listing. Entry in this field is required.

Item
Enter the item code that you want to update, or press Enter to accept the FIRST item code.

Primary vendor
Enter the primary vendor code that you want to update, or press Enter to accept the software default of All.

Minimum price amount
Enter the minimum amount to be changed, or press Enter to leave this field blank.

Maximum price amount
Enter the maximum amount to be changed, or press Enter to accept the default of 99,999.9999.

Change options
Select the change option you want for this update:

- Dollar amount

- Percentage

After you have selected a change option, enter the change amount (either by dollar or percentage) in the Change amount field.
