# Set Up Items in New Warehouse - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Item
Enter the item code with which to start this transfer, or press Enter to accept the software default of ALL.

Category
Enter the category code with which to start this transfer, or press Enter to accept the software default of ALL.

Warehouse
Enter the warehouse for which you are creating new inventory items or double-click in this field to select a valid warehouse code.
Note: The warehouse code must have already been set up in
 Warehouse File Maintenance. Warehouse codes may be up to 10 characters
 long.
