# Inventory Count Listing Screen

The purpose of this report is to explain how many errors were found in the entire import file. Only records with errors will appear on the listing. The 'Additional count for the item?' label will state 'Yes Warning' when multiple files have been imported for the same item and warehouse. This is a visual indicator that the counts will be added together during the update.
The final page of the report will state 'Errors found in XX of YY records'. Only records without errors may be updated.
