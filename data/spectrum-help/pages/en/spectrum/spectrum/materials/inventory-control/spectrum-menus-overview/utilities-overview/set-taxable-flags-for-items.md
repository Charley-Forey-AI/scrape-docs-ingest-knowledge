# Set Taxable Flags for Items

Use the Set taxable Flags for Items utility to set the
 sales tax and use tax subject to flags for the selected items.
 This utility is helpful when setting up inventory during
 implementation, or if tax laws change for a particular item in your state. Changes made
 here will be reflected on the [Item Main Properties](/en/spectrum/spectrum/materials/inventory-control/spectrum-menus-overview/maintenance-overview/item-main-properties) screen.
Important: Caution should be exercised when performing item
 code changes.
