# Item Mix Components

The Item Mix Components screen provides a simple method of
 building "mixes." It allows two or more inventory stock items to be combined to create a
 unique new stock item.
 An assortment of "component" items is
 assembled to create a "mix" item. The full cost basis of the component items is transferred
 to the newly assembled assortment item. In addition, during processing, quantities on hand
 of component items are reduced as quantities on hand of assembled items increase. No change
 in total inventory value occurs as a result of mix processing; but following assembly,
 "cost" is associated with the assortment item.
Before entry is made on this screen, both the assortment and the component item codes must be created in Inventory Items. The standard cost of the item to be assembled should be set to zero prior to use of this screen. The standard cost of the assembled item will automatically become the sum of the component standard costs as the assortment is defined on this screen.
Think of "mix" items as the separate ingredients in a cake: eggs, flour, vanilla, and so forth. Each individual "mix" component is entered in this screen and used whenever a mix is produced with Mix Processing.
