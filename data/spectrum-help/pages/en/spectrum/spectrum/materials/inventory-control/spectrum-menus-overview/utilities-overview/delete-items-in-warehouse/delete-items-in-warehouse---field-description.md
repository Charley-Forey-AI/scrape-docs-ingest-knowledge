# Delete Items in Warehouse - Field Description

Use the table below for reference when completing the fields on this screen.
Field
Description

Item
Enter the item code to be purged, or press Enter to select this system default of ALL.

Category
Enter the category code to be purged, or press Enter to accept the software default of ALL.

Warehouse
Enter the warehouse code from which you wish to purge items; warehouse codes may be up to 10 characters long. Entry in this field is required.

Include items with non-zero quantity on hand?
Select this checkbox to delete all items meeting the criteria specified above. If this checkbox is NOT selected, any items with positive quantity on hand in the specified warehouse will be skipped.
