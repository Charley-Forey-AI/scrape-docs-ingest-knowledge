# Report Calculations - Excess Inventory Report

The following table describes how each column is calculated on this report.
Column
Description

Current O/H
This column displays the current quantity on hand.

Total cost
The total cost is the total extended cost/value of the inventory item (this should also be the figure seen in the Items-On-Hand Valuation Report for this item).

Avg Monthly Use
Based on the "through date" and "months to use" fields on the starting screen, the Avg Monthly Use column is the sum of the historical quantities, divided by the number of months.

Months in Stock
This figure equals the current quantity-on-hand divided by the average monthly use.

Excess
The excess amount equals the total cost– (number of months of excess (from the starting screen)X average monthly usage X unit cost).
