# Purge Item History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Warehouse
To purge the history of a single warehouse enter that warehouse code , or press Enter to accept the software default of ALL.

Item code
Enter the item code, or press Enter to accept the software default of ALL.

Purge through
Enter the date through which items should be purged. Items with transaction dates on or before this date will be purged.
