# Physical Count Adjustment window

Use this window enter an additional count for this item or to create a total new quantity.
Field
Description

Physical count
The current actual physical count displays. No entry is allowed.

Additional count
Enter the number of additional items.

Total
The new total displays, based on the data in the Physical count and Additional count; otherwise, enter the new total.
