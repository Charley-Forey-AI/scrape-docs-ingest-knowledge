# Quantity Breaks File Listing

The Quantity Breaks File Listing screen is used to preview
 all quantity breaks and related information entered during Quantity Breaks File Maintenance.
The report first prints the category
 quantity breaks, then the item quantity breaks.
Note: This screen is relevant
 only if the Order Processing or Materials Management modules are installed. If the
 Materials Management module is installed, this information is also accessible through
 Customer Contract File Maintenance. For convenience, entry of special pricing may be
 performed in either screen.
