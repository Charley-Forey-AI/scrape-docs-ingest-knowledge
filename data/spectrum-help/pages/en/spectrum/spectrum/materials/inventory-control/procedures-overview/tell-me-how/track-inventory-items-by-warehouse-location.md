# Track Inventory Items by Warehouse Location

Learn how to Track Inventory Items by Warehouse
 Location.
Before beginning this procedure, you should have established
 the warehouse and item location code(s) you will need to track inventory.

1. On the Site Map screen, click Admin >  Installation >  Inventory Control.

1. Make sure the Validate item location checkbox is
 selected. Once it has been selected, click OK twice.

1. On the Site Map screen, click Inventory Control >  Maintenance >  > Inventory Item.

1. Enter the first item code you want to add to the
 software.Note: If the item code you want to track by warehouse location
 already exists and you want to update its tracking information, press
 F4 to select the item code you are updating. Then,
 proceed to step 7.

1. In the Description field, enter the description of the item.

1. Complete the Classification, Purchase, and Sell portions of the screen.

1. Click the Details button.

1. In the Whse column, use the drop-down menu to select the warehouse in which
 the inventory item is stored and press Enter.

1. In the Location column, use the drop-down menu to select the location of the
 item within the warehouse and press Enter.

1. Complete the Reord and Sugg qty fields.

1. Click OK repeatedly until you return to the main menu.
