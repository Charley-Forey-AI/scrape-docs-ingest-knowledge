# Inventory Item File Listing

Use the Inventory Item File Listing screen to print a
 report that provides information (other than pricing) entered in the Inventory Items
 screen.
 The report prints
 category-by-category, and then in item code order with categories. A variety of selections
 are provided, such as category, item code and ABC class. This screen can be used to obtain
 a hard copy of all items that have been entered.
