# Transfer Freight Misc. Charges - Field Definitions

Use the table below for reference when completing the fields on this screen.
Field
Description

Freight Charge
Enter the freight charge amount you want applied to this transaction.

Department
Enter the department code for the freight charge you entered, or double-click in this field to select from a list of valid department codes.

Misc Charge
Enter the miscellaneous charge amount you want applied to this transaction.

Department
Enter the department code for the miscellaneous charge you entered, or double-click in this field to select from a list of valid department codes.
