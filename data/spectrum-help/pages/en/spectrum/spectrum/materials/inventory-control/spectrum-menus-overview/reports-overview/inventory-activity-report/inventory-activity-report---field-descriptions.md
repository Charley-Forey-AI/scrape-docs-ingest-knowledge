# Inventory Activity Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Category
Enter the category to be included on this report, or press Enter to include all inventory categories in this report.

ABC class
Enter the class to be included on this report, or press Enter to include all ABC classes in this report.

Item code
Enter the item to be included on this report, or press Enter to include all inventory items in this report.

Year-to-date sales
Enter the lowest and highest year-to-date dollar amounts to be included on this report, or press Enter to begin with the lowest dollar amount and end with the highest dollar amount (up to $99,999,999.00).

Sort by
Select to sort the report by the Year-to-date sales dollar amount or by item Category.
