# Item P.O. Receiving Report - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Warehouse Item Vendor
Enter the warehouse code, inventory item, and vendor code you wish to include on the report, or press Enter to accept the default of ALL.

From date To date
Enter the first and last dates to be included, or press Enter to begin with the earliest date and end with the current Purchase Order processing date.
