# Physical Inventory Reports - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Count transaction listing?
Select this checkbox to print a copy of the Physical Count Transaction Listing. This report prints a complete listing of the physical inventory count.
Click the associated Selections button to choose a sort option for this report: Warehouse, Item code, or Location

Exception listing?
Select this checkbox to print a copy of the Physical Count Exception Listing. This report lists all items for which there is no physical count entered.
Click the associated Selections button to choose whether to include discontinued items

Variance report?
Select this checkbox to print the Physical Inventory Variance Report. This report compares the actual inventory quantities counted with the quantities entered in Spectrum.
Click the associated Selections button to choose a sort option for this report: Warehouse or Item code
