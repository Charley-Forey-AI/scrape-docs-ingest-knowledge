# Inventory Price Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Category
Enter the product category code to be included on the report, or press Enter to accept the software default and print ALL category codes.

Item code
Enter the item code to include on the report, or press Enter to accept the software default and print ALL item codes.

Discount code
Enter the discount code to be included on the report, or press Enter to accept the software default and print ALL discount codes.

ABC class
Enter the ABC code to be included on the report, or press Enter to accept the software default and print ALL ABC codes.

Report format
Select Job to include phase and cost type information on the items listed on the report. Select Price level and standard cost to include the standard cost and price level table for the range of items on the report. Select Price table only to include the price level table, but suppress standard cost information on the report.
