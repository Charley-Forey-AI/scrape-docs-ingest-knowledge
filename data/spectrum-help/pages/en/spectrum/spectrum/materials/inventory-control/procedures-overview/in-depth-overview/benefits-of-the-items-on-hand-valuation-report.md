# Benefits of the Items-on-Hand Valuation Report

The Items-on-Hand Valuation Report is a convenient way to accurately and properly report asset valuation. To get the most out of this report, run it every time a physical inventory is done (at least annually or semi-annually).
This report serves many purposes, including the following.

- Balances to G/L (one or more asset accounts named "inventory")

- Internally used for CFO or controller for source document

- Used for insurance replacement costs

- Helps track and account for "shrinkage" and "spoilage."

- Based on a periodic inventory schedule, which allows you to run the report for past periods
