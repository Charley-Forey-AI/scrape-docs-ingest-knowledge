# Inventory File Labels - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Item code Category Vendor Warehouse
Enter the item code, category code, vendor code, and warehouse code for which labels are to print, or press Enter to print ALL items.

Print options
Select the print option you want: Label print or Alignment mask.
 To print an alignment mask:

1. Load the labels into the printer.

1. Select the Alignment mask option. This prints XXX's on the labels where information will be printed.

1.  Check whether the XXXs are aligned correctly. If the alignment is incorrect, adjust the printer. Repeat steps 1 and 2 to print the alignment mask again. Repeat until alignment is correct.

1. When alignment of the XXXs is correct, select Labels to print the actual labels.
