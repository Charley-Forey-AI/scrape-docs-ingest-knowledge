# Inventory File Labels

The Inventory File Labels screen is used to print labels
 for warehouse shelves. Labels are sorted by warehouse and printed in sequence by location
 within the warehouse.
Items must be assigned to warehouses to
 be included on this report. You can use the [Set Up Items in New Warehouse](/en/spectrum/spectrum/materials/inventory-control/spectrum-menus-overview/utilities-overview/set-up-items-in-new-warehouse) utility to add items
 to a warehouse so that they display on this report.
Each inventory label is 30 columns wide and contains the following information:
ITEM: ITEM CODE CAT: 1234
ITEM DESCRIPTION
NAME OF PRIMARY VENDOR
PART: VENDOR'S PART # UM: CS*BX
WH: A LOC: A015 PT: 0
ITEM: The item code displays.
CAT: The category code displays.
ITEM DESCRIPTION: The item description displays.
NAME OF PRIMARY VENDOR: The name of the primary vendor of this item displays, as entered in Inventory Item File Maintenance.
PART: The vendor's part number for this item displays, as entered in Inventory Item File Maintenance.
WH: The warehouse code displays. Items stored in multiple warehouses will have a label for each warehouse.
LOC: The location of the item within the warehouse displays, as entered in Inventory Item File Maintenance.
PT: The reorder point for the item displays, as entered in Inventory Item File Maintenance or Item Detail File Maintenance.
