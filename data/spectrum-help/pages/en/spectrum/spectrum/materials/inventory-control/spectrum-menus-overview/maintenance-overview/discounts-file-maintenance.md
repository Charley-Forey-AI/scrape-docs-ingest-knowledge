# Discounts File Maintenance

The Discounts File Maintenance screen is used to establish
 and maintain discount codes to be used for Order Processing and Work Order promotional
 discount pricing.
An example of discount code usage is for
 "Monthly Specials;" perhaps a group of items will be on sale during the month of June.
The applicable discount code would be entered on the Inventory Item File Maintenance screen for all items to be discounted. During Order Processing Order Entry, Invoice Entry(OP) and Work Order processing, the software will offer the promotional price only during the date range identified here. It is not necessary to remove the discount code from Inventory Item File Maintenance at the end of the promotional discount period; the software will disregard the discount code after the effective date range has passed. In fact, the promotional discount may be reactivated at any later time simply by changing the effective date range on this screen to the current/future period.
Note: If the Order Processing
 module is not installed, entry in this screen is irrelevant.
