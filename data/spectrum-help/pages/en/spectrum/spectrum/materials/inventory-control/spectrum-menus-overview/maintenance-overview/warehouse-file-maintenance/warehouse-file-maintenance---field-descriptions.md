# Warehouse File Maintenance - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Warehouse code
Enter the warehouse code. Warehouse codes may be up to 10 characters long.

Description
Enter the name of the warehouse, or press Enter to accept the corresponding default.

Address
Enter the address and phone number of the warehouse in this section.

Sales tax code
Enter the sales tax code for this warehouse (for Materials Management transactions).

Use tax code
Enter the use tax code for this warehouse. Up to 15 characters are allowed.

Department
Enter the General Ledger inter-departmental code for this warehouse. The default hierarchy for department codes is as follows:
The system first looks to the Category File Maintenance for the G/L department code. If the field is blank there, then the software will look to the Warehouse File Maintenance. If this is also blank, you will be prompted to enter a valid code.
