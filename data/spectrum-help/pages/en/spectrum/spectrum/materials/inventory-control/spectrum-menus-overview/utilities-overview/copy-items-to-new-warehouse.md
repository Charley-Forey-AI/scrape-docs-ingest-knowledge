# Copy Items to New Warehouse

The Copy Items to New Warehouse screen is used to copy item
 information from one warehouse to another.
If the copy from warehouse is not set up
 for a particular item, no new warehouse will be created for that item. If the copy to
 warehouse already exists, no existing data will be overwritten.
