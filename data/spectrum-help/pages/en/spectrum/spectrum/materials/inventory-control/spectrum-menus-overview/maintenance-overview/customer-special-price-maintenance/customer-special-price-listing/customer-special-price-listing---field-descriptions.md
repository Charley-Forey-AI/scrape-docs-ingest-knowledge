# Customer Special Price Listing - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Customer type
Enter a specific customer type, or press Enter to accept the software default and print ALL customer types.

Customer
Enter the customer code of the first customer to print, or press Enter to accept the software default and print ALL customers on file.

Item/category code
Enter the first item code to print, or press Enter to accept the software default and print ALL item/category codes.
