# Inventory Items

The Inventory Items screen may be used to record a variety
 of information about each inventory item stocked by the company. This screen contains
 information on cost, vendors, and item category, as well as optional pricing and discount
 figures for construction companies.
After defining an item for inventory, it is necessary to add an item
 detail record to specify the warehouse(s) in which the item will be stored. An item detail
 record must be added for every warehouse where this item will be stocked.
