# Change Item Codes

The Change Item Codes screen is used to change existing
 item codes system-wide.
It will be necessary to run the [Change Item Codes Update ](/en/spectrum/spectrum/materials/inventory-control/spectrum-menus-overview/utilities-overview/change-item-codes/change-item-codes-update) before codes are actually
 changed. The user should also run the [Change Item Codes Listing](/en/spectrum/spectrum/materials/inventory-control/spectrum-menus-overview/utilities-overview/change-item-codes/change-item-codes-listing) before performing the update.
Important: Caution should be
 exercised when performing item code changes.
