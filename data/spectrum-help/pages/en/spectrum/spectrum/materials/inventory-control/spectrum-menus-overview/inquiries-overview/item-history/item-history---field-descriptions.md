# Item History - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field
Description

Item code
Enter the desired item code for inquiry.

Warehouse
Enter the warehouse code for inquiry, or double-click in this field to select from a list of valid warehouse codes. Warehouse codes may be up to 10 characters long.

Details

Date
The date of the transaction displays.

Type
The transaction types display. Valid transaction types include:
 Adjustments, Credit
 Returns, On Order,
 Receipts, Transfers,
 Sales, Customer Returns,
 Job Requisition Entries, Miscellaneous
 Cost Adjustment, Equipment, or
 Usage.

Reference
The reference number of the transaction displays.

U/m
The unit of measure (um) of the transaction displays.

Quantity
The quantity of the transaction displays.

Unit cost
The unit cost of the transaction displays.

Extension
The extension displays as quantity multiplied by the unit cost.

Use tax $
The use tax amount displays.

Use tax code
The use tax code, description, and percentage displays.

P.O. #
The purchase order number displays.

Job
The job code and description displays. If multi-company is in use, the company code will also display.

Phase
The phase code and description displays.

Cost type
The cost type code and description displays.

Equipment
The equipment code and description displays.

Cost category
The cost category code and description displays.

Eq. work order
The equipment work order code displays.

Cost adjustment source
The cost adjustment source code and description displays.

Transfer to warehouse
The warehouse code and description where the item will be transferred displays.

Message
Any text from the Adjustment reason column displays in this field.

Inventory batch
The inventory batch code displays in this field.
