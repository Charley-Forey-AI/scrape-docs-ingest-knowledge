# Mix Pick Sheet - Field Descriptions

Use the table below for reference when completing the fields on this screen.
Field/Button
Description

Mix item code
Enter a specific item code, or press Enter to accept the default if the report is to include all assortment item codes.

Warehouse
Enter the appropriate warehouse code for this report, or press Enter to accept the default warehouse code.

Quantity to be made
Enter the desired quantity of assortment items to be made.
