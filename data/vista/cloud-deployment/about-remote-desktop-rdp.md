# About Remote Desktop (RDP)

Use of RDP in Vista is an alternative connection method for users whose use of the VRL Cloud connection method results in poor performance.
It's common for some number of VRL Cloud users in each organization to experience what appears to be system slowness. This is usually correlated with heavy usage, such as payroll or payment processing and/or generating certain reports.
Using Vista from a remote desktop can resolve this slowness.
This section provides information about RDP, with specifics for both admins and end users.
Note:This information refers to features associated with Vista's modern cloud infrastructure. If you do not see some of these options, you may be running on a legacy version of cloud. Contact your account representative for more information.
