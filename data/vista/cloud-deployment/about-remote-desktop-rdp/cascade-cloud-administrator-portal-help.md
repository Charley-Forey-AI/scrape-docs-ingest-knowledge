# Cascade Cloud Administrator Portal Help

Cascade is a user management portal, leveraged by cloud admins across multiple Trimble products.
If your product has access to Cascade, you can use the portal to perform administrative functions to manage your users' Azure Virtual Desktop and product application accounts, including maintaining profiles and passwords, creating groups, and setting folder permissions.

## Log Into the Cascade Portal

Go to [web.cascade.trimble.com](https://web.cascade.trimble.com) and log in to Cascade
 using the credentials provided by Trimble.

## Access Cascade Help for Your Product

Choose your product below to go to the Cascade Portal Help within your product's help system.
[Vista Cloud Admin Portal Help](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:vista_60061)
[ProContractor Cloud Admin Portal Help](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:PC_000006)
[Viewpoint Estimating Cloud Admin Portal Help](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:VE_000002)
If you don't see your product listed, reference the [Vista Cloud Admin Portal Help](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:vista_60061). The information applies to all products using the Cascade portal.
