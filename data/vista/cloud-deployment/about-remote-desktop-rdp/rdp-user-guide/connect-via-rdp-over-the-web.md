# Connect via RDP over the Web

Microsoft's AVD remote desktop web client offers an alternative method for accessing your Vista application.
If you are an RDP user, the preferred method of logging
 in to Vista is to download
 [Windows App
 from the Microsoft Store](https://apps.microsoft.com/detail/9N1F85V9T8BN). If needed, use the method shown on
 this page only as an alternative.
If you're an RDP user, and if using the remote desktop client is problematic, try connecting to your Vista application over the Internet.

1. Go to [Connect to Azure Virtual Desktop with
 the Remote Desktop Web client](https://learn.microsoft.com/en-us/previous-versions/remote-desktop-client/client-features-web-cloud).

1. Follow the instructions to access your Azure Virtual Desktop resources directly from a web browser without needing to install a separate client.

Your should now be able to log in to Vista without the desktop client.
A one-time configuration may be necessary for logging in to Vista. See [Access Vista - Remote Desktop (RDP)](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/access-vista---remote-desktop-rdp).
