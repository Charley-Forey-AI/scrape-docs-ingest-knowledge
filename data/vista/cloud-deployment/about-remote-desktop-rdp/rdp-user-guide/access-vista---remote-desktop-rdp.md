# Access Vista - Remote Desktop (RDP)

As an RDP user, you must configure your local Vista client to access your internet-hosted server in order to log in to the application.
Important: This information is for new RDP users.
Before you can access Vista, make sure the following
 is complete:

- A system administrator must add you to the [cloud portal](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:vista_60060) and share with you your
 Azure Virtual Desktop credentials.

- A system administrator must create a Vista account for you and
 share with you your User Name and
 Password.

- You must download [Windows App from the Microsoft
 Store](https://apps.microsoft.com/detail/9N1F85V9T8BN) so that you can connect to the Azure Virtual Desktop and access
 Vista.

To access Vista from your Azure Virtual Desktop:

1. Log in to Azure Virtual Desktop using the credentials provided by your
 administrator.

1. Open the Vista application.

1. Choose your login method.For details about login methods, see [Log in to Vista](/en/vista/vista/on-premises-deployments/installations/major-release-installation/vista-client-application/log-in-to-vista).

1. Enter your credentials and complete the login process.
