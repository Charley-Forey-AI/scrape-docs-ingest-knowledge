# Enable Workstation Printing

If you are a Vista RDP user needing to print checks, reports, and documents from the Vista application, you must install TSPrint on your computer.
You must have already configured your Vista client.
To download and install the TSPrint client:

1. Go to [https://www.terminalworks.com/remote-desktop-printing/downloads](https://www.terminalworks.com/remote-desktop-printing/downloads).

1. Choose and download the proper version client for your workstation.

1. Navigate to the downloaded file and open it.

1. Follow the steps in the wizard.

1. When installation is complete, close the TSPrint Client setup wizard.

Your workstation can now print from Vista and TSPrint options should appear in print dialogues.
