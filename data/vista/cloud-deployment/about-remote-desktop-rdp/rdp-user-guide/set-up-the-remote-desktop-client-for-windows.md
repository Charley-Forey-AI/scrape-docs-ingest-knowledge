# Set up the Remote Desktop Client for Windows

If you are an RDP user, download and install the Windows App
 desktop client so you can access Vista.
Note: Starting March 27, 2026,
 the Remote Desktop client for Windows (MSI) will no longer be
 supported.
Users should begin migrating to Windows App to
 ensure continued access to their Azure Virtual Desktop and Windows 365
 resources after the date above. If you've determined that an RDP connection
 is required for the type of work you do in Vista, complete the following steps:

1. Download [Windows App from the Microsoft Store](https://apps.microsoft.com/detail/9N1F85V9T8BN).

1. Follow the instructions to install the desktop
 client.For a video demonstration:

Your workstation now has the necessary client installed for you to log in to Vista.If for some reason the desktop client method isn't working for you, see [Connect via RDP over the Web](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/connect-via-rdp-over-the-web).

A one-time configuration is necessary for logging in to Vista. See [Access Vista - Remote Desktop (RDP)](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/access-vista---remote-desktop-rdp).
