# RDP User Guide

If you access Vista using Remote Desktop Protocol (RDP), take these steps to get started.
To set up your connection and get started using Vista:

1. Enable your connection to Vista. Download [Windows App from the Microsoft Store](https://apps.microsoft.com/detail/9N1F85V9T8BN).Note: Starting March 27, 2026, the Remote Desktop client
 for Windows (MSI) will no longer be supported. See this [FAQ
 page](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/transition-to-windows-app-faq) for more information.

1. [Access Vista - Remote Desktop (RDP)](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/access-vista---remote-desktop-rdp#ID-00051130--en__ID-00051130).

1. If you need to print from Vista using your local printers, [Enable Workstation Printing](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/enable-workstation-printing#ID-0005118e--en__ID-0005118e).

1. If you need to scan from Vista using your local scanner, [Enable Workstation Scanning](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/enable-workstation-scanning#ID-000511ff--en__ID-000511ff).

1. Orient yourself with the cloud File Explorer. Saving to or from Vista requires use of your cloud File Explorer. See [About File Management with RDP](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/about-file-management-with-rdp#ID-0005115d--en__ID-0005115d).

1. Review details about drag-and-drop functionality. See [Copy Local Files to the Cloud](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/copy-local-files-to-the-cloud#ID-0005103a--en__ID-0005103a).

1. Review how Microsoft application plug-ins work in Vista. See [About Microsoft Office with RDP](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/rdp-user-guide/about-microsoft-office-with-rdp#ID-0005116f--en__ID-0005116f).
