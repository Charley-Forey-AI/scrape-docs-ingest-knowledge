# Cascade Cloud Administrator Portal - Vista

Use the Cascade cloud portal to perform administrative functions for
 your organization, including managing users, creating groups, and setting folder
 permissions.
You must be an
 administrator to access the Cascade portal.
If your organization has a cloud deployment hosted in Azure, each RDP
 user needs a Cascade user record before they are able to log in to Vista from the remote
 desktop.
