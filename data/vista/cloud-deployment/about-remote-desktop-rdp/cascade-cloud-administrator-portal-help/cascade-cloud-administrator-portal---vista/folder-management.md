# Folder Management

In the Folder Management section of the Cascade cloud admin
 portal, admins can set folder permissions by user or group.
In the virtual desktop, the shared folders serve as a virtual network drive,
 helping your users share and access the directories they need.
You must be an
 administrator to access the Cascade portal.
