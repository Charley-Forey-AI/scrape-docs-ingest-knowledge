# Group Management

In the Group Management section of the Cascade cloud admin
 portal, you can add new groups, add members to groups, remove members from groups, and
 delete groups.
These groups allow you to control permissions and access to specific
 directories and files within the virtual desktop.
Note:You must be an
 administrator to access the Cascade portal.
