# User Management

In the User Management section of the Cascade cloud admin
 portal, you can add users, set passwords, unlock accounts, reset MFA, and delete users.
Note:You must be an
 administrator to access the Cascade portal.

In addition to setting up users in the portal, you must set them up in your
 product application as well.
