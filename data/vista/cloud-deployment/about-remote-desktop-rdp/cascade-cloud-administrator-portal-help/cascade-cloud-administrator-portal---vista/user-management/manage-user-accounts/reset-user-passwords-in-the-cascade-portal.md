# Reset User Passwords in the Cascade Portal

Admins can create or reset user passwords in the cloud admin
 portal from the User Management page.
Tip: See at a glance which of
 your users has an expiring password. The Password Expiration column
 indicates the date that the password expires and the color represents expiration
 status.

- Red: password is expired

- Yellow: password is expiring in 7 days or less

- Green: password is active

1. Select the
 user icon  to open the User Management page.

1. For the user needing a new password, in the User
 Actions column, select Reset
 Password.You can also see the password expiration details and password reset option
 when you select a User Name to view the user's details.

1. On the Edit User Password page, enter a password
 in the New Password
 field. The password must comply with the guidelines listed. Start
 typing to see the guidelines.

1. Type the same password again in the Confirm Password
 field.

1. Select Save Changes.

1. Share the new password with the user.
