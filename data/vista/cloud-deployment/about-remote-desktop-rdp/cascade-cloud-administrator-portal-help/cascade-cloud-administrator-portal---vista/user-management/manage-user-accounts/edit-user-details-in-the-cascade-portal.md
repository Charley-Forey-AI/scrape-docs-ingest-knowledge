# Edit User Details in the Cascade Portal

You can edit details about a user such as their display name
 in the portal, first or last name, and phone number.
To edit which group a user belongs to, see [Add Members to a Group in the Cascade Portal](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp/cascade-cloud-administrator-portal-help/cascade-cloud-administrator-portal---vista/group-management/manage-members/add-members-to-a-group-in-the-cascade-portal).

1. Select the
 user icon  to open the User Management page.

1. For the user you want to edit, select their User
 Name, then select Edit.

1. On the Edit User Details page, change the user
 details as needed. The only field you cannot change is the User Name.

1. Select Save Changes.
