# Unlock User Accounts from the Cascade Portal

Admins can unlock a user's Azure Virtual Desktop account from
 the cloud admin portal.
If a user has five invalid
 login attempts, they will be locked out of their account for a period of time. As an
 admin, you can manually unlock their account.

1. Select the
 user icon  to open the User Management page.

1. For the user you want to unlock, in the User Actions column, select
 Unlock Account.

1. In the Unlock User Account window, verify that you want to unlock the user's
 account by typing their user name in the field.

1. Select Unlock.You will see a message saying *User has been successfully unlocked.*

That user will be able to log into their Azure Virtual
 Desktop account again.
