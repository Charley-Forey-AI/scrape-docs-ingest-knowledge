# Reset User Multifactor Authentication in the Cascade Portal

Admins can reset a user's MFA to prompt them to re-register
 their authentication method the next time they log into their Azure Virtual Desktop
 account.
You will need to reset MFA
 if one of your users loses their phone or needs to set up MFA on a new device.

1. Select the
 user icon  to open the User Management page.

1. For the user you want to reset, in the User Actions column, select
 Reset MFA.

1. In the Reset User Multifactor
 Authentication window, verify that you want to reset MFA for
 that user by typing their user name in the field.

1. Select Reset.You will see a message saying *The user's MFA has been successfully reset.*

The next time that user attempts to log into their
 Azure Virtual Desktop account, they will be prompted to reset their MFA.
