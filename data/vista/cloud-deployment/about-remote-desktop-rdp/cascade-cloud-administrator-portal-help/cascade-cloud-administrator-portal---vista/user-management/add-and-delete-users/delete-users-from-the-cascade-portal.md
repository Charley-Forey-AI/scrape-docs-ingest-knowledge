# Delete Users from the Cascade Portal

You can remove users from the cloud admin portal in the User
 Management section.
Deactivate the user's
 profile in the product application *before* you remove
 them from the Cascade portal.

1. Select the
 user icon  to open the User Management page.

1. For the user you want to delete, select the delete
 icon .

1. In the Delete User confirmation window, verify
 that you want to delete that user by typing their user name in the field.

1. Select Delete.
