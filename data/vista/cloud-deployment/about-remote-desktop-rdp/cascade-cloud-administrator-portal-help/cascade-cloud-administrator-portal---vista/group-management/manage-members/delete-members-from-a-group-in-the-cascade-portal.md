# Delete Members from a Group in the Cascade Portal

Remove members from a group in the Cascade cloud admin portal.

1. Select the group icon  to open
 the Group Management page.

1. For the group you want to remove members from,
 select Edit GroupYou can also select the Group Name, then Edit.

1. For each user you want to remove from the group,
 select the delete icon  on the right.

1. To confirm your updates, select Save Changes.

Note: The only group you cannot remove
 members from is the All
 Users group, which automatically adds all users in your
 organization.
