# Add Members to a Group in the Cascade Portal

Add members to an existing group in the cloud admin portal.
You must be an
 administrator to access the Cascade portal.

1. Select the group icon  to open
 the Group Management page.

1. For the group you want to add members to, select
 Edit GroupYou can also select the Group Name, then Edit.

1. On the Edit Group Members page, select Add Members.

1. Select the checkbox next to each user you want to add to the group.You can also search for users to add to the group.

1. Select Add.

1. When you are finished, select Save Changes.
