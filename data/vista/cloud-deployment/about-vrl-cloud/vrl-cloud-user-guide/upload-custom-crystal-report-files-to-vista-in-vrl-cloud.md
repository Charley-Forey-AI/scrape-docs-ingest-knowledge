# Upload Custom Crystal Report files to Vista in VRL Cloud

You can add existing custom report files to Vista by dragging them onto the Info tab of the RP Report Locations form.
If your deployment method is VRL Cloud, you don't need to use VPN (Virtual Private Network) or RDP (Remote Desktop Protocol), nor do you need Windows file Share access in order to add new reports into Vista.To add one or more custom report files to Vista:

1. Open the RP Report Locations form.

1. Select the Grid tab.

1. In the Location column, locate and select the row named Custom.The entry in the Path column for the Custom row should be formatted like this:\\your_companyD1\Viewpoint Repository\Reports\Custom\

1. Select the Info tab.

1. Drag and drop your custom RPT file(s) onto the RP Report Locations form.

The Name, Size, and Date Modified columns display each report's information.
You must now create a record for this custom report in the RP Report Titles form and link that record to this one. For instructions, see [Set Up New Custom Vista Reports in VRL Cloud](/en/vista/vista/cloud-deployment/about-vrl-cloud/vrl-cloud-user-guide/set-up-new-custom-vista-reports-in-vrl-cloud).
