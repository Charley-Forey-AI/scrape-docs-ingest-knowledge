# What's New in 2025.6

The Vista 2025.6 release includes several enhancements and new features, as well as defect fixes that apply to all geographies.
Some of the new features in this release include improved handling of Federal tax on supplemental earnings, support for multiple 401k contribution limits (US), and improvements to SM work orders, as well as several other features to help improve your workflow.
Important:Introducing the new Vista release cadence in 2025.6:Beginning with this June release, we are introducing the new
 release cadence for Vista. The
 version number now reflects the year and the month of release (6 = June). The
 2025.6 version will be followed by successive releases, approximately every two
 months throughout the year. For more information, review the [Changes to Vista
 Releases in 2025 FAQ](/en/vista/vista/whats-new/whats-new-in-vista/whats-new-in-2025.6/changes-to-vista-releases-in-2025-faq).
For Vista Remote Link (VRL) users,
 this release requires that you update the Vista Client and Crystal Reports to ensure
 compatibility with the latest features. After you install 2025.6, please install
 the following items (found in the [Downloads](https://support.viewpoint.com/s/vista) section of the Viewpoint Customer Portal) on any system that will
 be running the Vista Client:

- Vista
 Client (VistaClient.25.6.13.9.msi)

- ODBC Driver 17 (The prerequisite file
 (VC_redist.x64.exe) and driver (msodbcsql.msi))

- Crystal Reports SP25
 (CRRuntime_64bit_13_0_25.msi)
