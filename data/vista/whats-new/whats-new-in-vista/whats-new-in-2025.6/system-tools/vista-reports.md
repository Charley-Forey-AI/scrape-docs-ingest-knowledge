# Vista Reports

The following discusses updates made to standard reports for the 2025.6 release.

## Viewpoint ePayments Renamed to ePayments

The Viewpoint ePayments feature in Vista is now called ePayments. This change affects the following reports. For form and field changes, see [Accounts Payable Features](/en/vista/vista/whats-new/whats-new-in-vista/whats-new-in-2025.6/accounting/accounts-payable-features):
ReportRpt IDChange
AP EFT/Viewpoint ePayments Remittance26Changes include:

- The report's title changed to AP EFT/ePayments Remittance

- In the Payment Method column, Viewpoint ePayments transactions now show as "ePayments".

- Updated the report description to show "ePayments" where applicable

AP EFT/Viewpoint ePayments Remittance Vendor1032Changes include:

- The report's title changed to AP EFT/ePayments Remittance Vendor

- In the Payment Method column, Viewpoint ePayments transactions now show as "ePayments".

- Updated the report description to show "ePayments" where applicable

AP Payment Preview with Compliance54Changes include:

- In the report parameters and the report heading, the Print Viewpoint ePayments field is now titled "Print ePayments".

- The Viewpoint ePayments section of report is now titled "ePayments"

AP Vendor Payment History72Changes include:

- In the report parameters, the Print Viewpoint ePayments and Show Viewpoint ePayments Fulfillment Info fields are now titled "Print ePayments" and "Show ePayments Fulfillment Info" (respectively).

- On the report cover page and in the Pay Method column of report, all references to Viewpoint ePayments are now "ePayments".

AP Vendor Payment History Drilldown73
AP Vendor Spend Analysis - Viewpoint ePayments1316The report's title is now AP Vendor Spend Analysis - ePayments
AP Payment Audit List53In the Payment Method column, Viewpoint ePayments transactions now show as "ePayments".

## Updated Leave Accrual / Leave Balance Reports

In conjunction with the improvements made to the decimal precision of leave accruals and available balances (see [Payroll release notes](/en/vista/vista/whats-new/whats-new-in-vista/whats-new-in-2025.6/hr-and-payroll/payroll-features#concept-21c0f0fc-c2ea-45a1-b407-0618ce08a2cf--en__ImprovedLeave)), the following reports now show available balance, accrual, usage balances, and adjustments with five decimal places.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
