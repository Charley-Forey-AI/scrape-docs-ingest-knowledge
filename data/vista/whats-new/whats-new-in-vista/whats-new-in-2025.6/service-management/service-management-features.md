# Service Management Features

Vista 2025.6 delivers on user-requested Service Management enhancements, fixes, and other improvements.

## New Work Order Scope Cost Summary

You can view a summary of costs for scopes on a work order.
A new Summary tab was added to the scope section of SM Work Orders. This display-only tab provides a summary of costs for a work order scope by cost type category and cost type, based on the work completed lines posted to the scope. Totals shown will be the sum of all work completed lines posted to the same cost type category/cost type.
For more information, see [Work Order Scope Cost Summary](/en/vista/vista/service-management/work-orders/work-order-scope-cost-summary).

## Improvements to SM Work Orders

Several minor changes were made to improve the SM Work Orders form. These changes include the following:

- For Flat Price work order scopes, the Margin field is now to the right of the Price field, and the Split button is now to the right of the Margin field. Since entry of a Margin is required before you can add/update entries in the SM Flat Price Revenue screen (accessed via the Split button), this change improves the entry workflow.

- For work order scopes, the Price Method field was expanded to show the full description of all Price Method options.

- On the Work Order Scope Grid tab, several columns were repositioned to improve entry workflow and to be consistent with the field order on the Info tab. These fields are as follows:

- Moved the Rate Template field after the Price Method field.

- Moved the Tax Source field before the Tax Type field.

- Moved the Derived field after the Tax Rate field.

- Moved the Ins Code field after the Bill To field.

- Moved the Not to Exceed field after the Rate Template field.

- Moved the Use Agreement Rates field after the Revision field.

- Moved the Cost Total, Cost Total Derived, Labor Hours, Labor Hours Derived, Craft, and Class fields after the Derived field.
