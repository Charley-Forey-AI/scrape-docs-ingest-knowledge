# General Features

Vista 2025.6
 delivers on general user-requested enhancements, fixes, and other improvements.

## Updates to Software Requirements

For the latest software requirements, see [Vista Software Requirements](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements/vista-software-requirements).
Crystal Reports Runtime 2016 SP25If you are running Crystal Reports and the 64-bit
 version of the Vista
 client, you must upgrade to Crystal Reports Runtime 2016
 SP25 for Vista version 2025.6 and later. The Crystal
 Reports runtime is available to download with the Vista 2025.6 installers from
 the Customer Portal.
ODBC Driver 17 for SQL ServerFor Vista 2025.6, you must upgrade the ODBC
 driver for Crystal Reports to ODBC Driver
 17.
If you already have SQL Server Management Studio
 on your system, check to make sure ODBC Driver 17 is installed. ODBC
 Driver 17 must be installed on the local PC running the Vista client.
If you are using VRL, ODBC Driver 17 must be
 installed on the Vista
 application server *and* the local
 PC running the Vista
 client.

## For Customers on the Cloud: Read-Only Reporting Database
 Now Available

If you are a cloud-hosted Vista customer, you may be eligible to purchase a
 Vista Enterprise Database, a secondary read-only replica
 of your existing Vista database. This
 secondary database gives you the ability to use the primary transactional Vista database for other functions while
 executing certain reports. For more information, see [Vista Databases](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements/vista-databases).
Contact your account representative to learn more about this offering.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
