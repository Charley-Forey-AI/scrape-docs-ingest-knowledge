# Viewpoint Administration Features

The Vista 2025.6
 release delivers on user-requested Viewpoint Administration
 enhancements, fixes, and other improvements.

## Updated Single Sign-On Settings in VA Site
 Settings

To simplify login settings and eliminate potential configuration
 conflicts, the Allow Social
 Logins option on the VA Site Settings form is no longer
 available.
With this update, social logins are allowed by default.
To disallow social logins and require users to sign in with a
 federated company email address, select the User Email Enforcement checkbox
 and set the MFA Required
 field to Federated.

For more information, see [Configure Multifactor Authentication and Additional Login Settings](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/single-sign-on-with-trimble-id-admin-information/configure-multifactor-authentication-and-additional-login-settings).

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
