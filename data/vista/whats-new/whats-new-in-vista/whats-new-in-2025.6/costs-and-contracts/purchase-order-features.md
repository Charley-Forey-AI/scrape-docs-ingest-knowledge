# Purchase Order Features

Vista 2025.6 delivers on user-requested Purchase Order enhancements, fixes, and other improvements.

## Cost Tax Defaults for PO Add Item and PO Item Distribution

In support of the [Tax Handling](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2024-releases/whats-new-in-vista-2024-r1/service-management/service-management-features#concept-k1quaafr1002--en__SMTaxHandlingRN) feature included in Vista 2024 R1, the default behavior was modified in PO Add Items and PO Item Distribution for tax-related fields on purchase order items/lines posted to an SM work order (line type 6-SM Work Order).
The Tax Type and Tax Code fields for SM-related transactions will now default based on the PO item/line's SM Call Type and the Material Tax Override and Tax Option Override settings for the specified the work order scope. For detailed default behavior information, please see the F1 help for the Tax Type and Tax Code fields in PO Add Items and PO Item Distribution.
