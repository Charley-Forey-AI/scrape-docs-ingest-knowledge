# Remote Site Deployment

Remote clients require thin client technology as provided with Win2008/2012
 Terminal Services. Citrix is another viable solution. The ability to scan
 remotely can be facilitated by third-party software, such as RemoteScan.
 This allows a remote user to scan a document locally and route the scanned
 image through a remote desktop connection (for example, Terminal Services,
 Citrix, etc.).
You should review all remote access requirements with your network analyst /
 integrator for further details.
Vista Remote Link (VRL) is also
 available for remote clients to connect over the internet via
 HTTPS. No Citrix or remote desktop services are required when
 using VRL.
