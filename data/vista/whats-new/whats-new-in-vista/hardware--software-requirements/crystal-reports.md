# Crystal Reports

The two types of Crystal Reports licenses are Runtime and Developer.
RuntimeRuntime is used by all Vista users. You don't need to purchase a license because use of Vista includes a runtime license for all users to run Vista reports.For each workstation that needs to use the Vista application, but before installing the Vista application, install Crystal Reports runtime. [Download the correct version](https://support.viewpoint.com/s/product-more-resources?product=Vista&type=Downloads&links=true&title=Downloads%20for%20Vista) from the Viewpoint Customer Portal, located alongside the other download files for currently supported Vista versions.DeveloperTo customize reports, you must purchase at least one copy of Crystal Reports Professional directly from SAP.The minimum supported developer version is listed in the Client Workstation section of the [Vista Software Requirements](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements/vista-software-requirements). For more information on Viewpoint-provided Crystal Reports training, visit the [Trimble Construction Academy](https://academy.viewpoint.com/learn).
