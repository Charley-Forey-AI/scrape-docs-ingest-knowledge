# Server / Network Management and Maintenance

You should staff one or more individuals qualified to assume responsibility for
 management, administration, and support of your organization's hardware, Windows Server
 operating system, SQL server, and related network products.
Database monitoring and tuning by your Network / SQL Administrator may be
 necessary in order for your users to obtain optimal performance in your environment.
