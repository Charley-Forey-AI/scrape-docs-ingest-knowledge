# Software Installation

Your organization’s Network / SQL Administrator is responsible for
 installation and configuration of the items listed in these
 sections on [hardware](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements/vista-hardware-requirements) and [software](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements/vista-software-requirements) requirements.
New customers are assigned a Viewpoint Deployment Engineer to
 assist with reviewing system requirements, installation
 procedures, installing SQL Server, and installation of one
 Vista
 client application. This assistance is provided remotely via
 telephone and Internet. Initial installation assistance and
 services are provided with the new customer contract.
Once the Vista server application
 and one client are installed, Viewpoint Support is available to
 provide guidance when installing Vista on additional clients
 without charge and as part of Software Assurance. There is a
 charge for assistance moving the Vista application to a new
 server or reinstalling the application on the existing server
 (for example, upgrading the O/S or in a Disaster Recovery
 situation).
The Vista live databases may only
 be installed on a single instance of SQL Server. Support and
 assistance is only provided for your production databases.
