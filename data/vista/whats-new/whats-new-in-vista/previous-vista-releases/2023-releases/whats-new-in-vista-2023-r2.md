# What's New in Vista 2023 R2

The Vista 2023 R2 release includes several enhancements and new features, defect fixes that apply to all geographies.
Some of the features contained in this release include the ability to create separate work order scopes for service items, updates to the Job Billing invoice delivery feature, and the ability to access Vista Web's Pending PO / Subcontract Review page from Vista, as well as several other features to help improve your workflow.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
