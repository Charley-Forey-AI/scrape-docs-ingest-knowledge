# Equipment Management Features

Vista 2023 R1 delivers on user-requested Equipment Management enhancements, fixes, and other improvements.

## Improve Tool Tracking and Job Cost Allocations Using Hilti ON!Track with Vista

This powerful new integration enhances tool tracking and automatically allocates tool use and costs to jobs.
With the integration enabled and Equipment Management module configured, Hilti ON!Track records and tracks tool movements between locations, and these changes automatically update in Vista. ON!Track calculates the time tools and equipment spend at job sites, and you can use this data to better allocate costs to jobs in Vista.
For more information about setting up the Hilti ON!Track integration with Vista, contact your consultant.
Learn more about the [Hilti ON!Track Integration with Vista](/en/vista/vista/job-resources/equipment-management/equipment-transfers/hilti-ontrack-integration-with-vista).
