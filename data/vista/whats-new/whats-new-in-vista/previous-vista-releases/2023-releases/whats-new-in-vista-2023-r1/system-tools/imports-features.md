# Imports Features

Vista 2023 R1 delivers on user-requested Imports enhancements, fixes, and other improvements.

## Command Line Importing no longer supported

The use of Command Line is no longer supported and we have removed references to this functionality from the online Help.
If you used the Command Line Import function prior to this version, you'll need to perform your imports differently. See [About Importing Files Automatically](/en/vista/vista/administration/imports/setup-and-maintenance/about-importing-files-automatically).

## Update to the AP Vendors Import

The AP Vendor (APVendMaster) import form now includes the following fields:

- Email1099YN (Identifier #603)

- Tax ID Type (Identifier #252)

- Tax First Name (Identifier #256)

- Tax Middle Initial (Identifier #257)

- Tax Last Name (Identifier #258)

These import fields correlate to the 1099 Info fields on the AP Vendors form, Add'l Info tab.

##  Added Email 1095-C flag to ACA 1095-C Import (U.S. only)

The PR ACA 1095-C Employee (PR1095Header) import form now includes a new Email1095CYN identifier (#210). The system will automatically update all existing templates using this import form with the new identifier.

Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
