# Work Center Features

Vista 2023 R1 delivers on user-requested Work Center enhancements, fixes, and other improvements.

## Added Support for Subcontract Workflow Process

In support of the Subcontract review/approval workflow process added in this release, you now have the ability to review and approve subcontracts using the PM Work Center.
When a subcontract requiring approval is submitted for review/approval in PM Subcontracts, the system notifies assigned reviewers that subcontracts are ready for their review. Those subcontracts will then display in the My Documents to Review query in the Document Review section of the work center.
The reviewer can then review subcontracts/items, edit them (if needed), add comments and attachments, review/add related items, and approve or reject the subcontracts.
Note: If a subcontract is rejected, it is pulled out of the workflow process and must be resubmitted for review/approval once edits are completed.

For more information about reviewing/approving subcontracts, see [Review Subcontracts Submitted for Approval](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow/review-subcontracts-submitted-for-approval).
 For more information about the Subcontract Review/Approval feature added in this release, see the [Workflow](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2023-releases/whats-new-in-vista-2023-r1/system-tools/workflow-features) release notes.
Note: If you are using using the PO/SL review feature in Vista Web, you can also review and approve subcontracts via the Approval menu.
 For more information, see [Pending PO /
 Subcontract Review](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:FM_000034:FM:FM).
