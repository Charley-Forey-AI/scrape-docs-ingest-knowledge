# General Features

Vista 2023 R1 delivers on General user-requested enhancements, fixes, and other improvements.

## Access Trimble Work Center from Vista

You can now access the Trimble Work Center from the Vista main menu by clicking the Trimble Work Center icon () in the toolbar.
If you have Trimble Construction One
 enabled on your system, clicking this icon takes you to the Trimble Work Center. If
 you do not have Trimble Construction One enabled on your
 system, clicking the icon presents you with the [Trimble Construction
 One](https://construction.trimble.com/en/trimble-construction-one) website.
If you log into Vista using SSO, the Trimble Work Center launches automatically on a separate tab. If you close the Work Center, you can quickly reopen from Vista using the icon.
For more information about the Trimble Work Center, see [Trimble Work Center](/en/vista/vista/system-tools/work-centers/trimble-work-center).

## Updates to the Login Screen and Desktop Icon

For both on-premises and cloud-deployed organizations, we've updated the login screen. Your login experience remains largely unchanged, and the number of changes you see depend in part on how your Vista instance is deployed.

- On-premises - the changes are in appearance only, that is, branding and styling; the refreshed screen's fields are named the same and behave the same as before.

- Cloud and Trimble Construction One - while the same branding and styling updates apply, the other main changes are to buttons on the login screen, and are apparent only if you have single sign-on (SSO) enabled:

- The Log In With Viewpoint ID button has been renamed to Sign In With Trimble.

- If you log in using your username and password (and not SSO), you can select the new Continue With Classic ID button.

In conjunction with changes to the login screen, the Vista desktop icon (shown below) was updated for branding consistency.

## Aatrix History Filtering by Report Type

Aatrix history is now filtered by report type. This means that when you select the History button in any of the Aatrix access forms (AP 1099 Aatrix Print and eFile, PR Aatrix - W2 Print and eFile, PR Aatrix - T4 Print and eFile, etc.), the history screen will now show the history for the selected report type (previously, it would show the history for all report types).
The only exception to the filtering functionality involves ACA reports. Report type filtering has not yet been implemented for ACAs; therefore, when you access ACA history (via PR Aatrix - ACA Print and eFile), the history screen will continue to show the history for all report types. Additionally, ACA report history will also show on other history screens (W-2s, 1099s, and other general payroll history lists). Report type filtering for ACAs will be added in a future release.
In a related feature, you now have the ability to specify an Aatrix history path by user. For more information, see the [Viewpoint Administration](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2023-releases/whats-new-in-vista-2023-r1/administration/viewpoint-administration-features#concept-8569--en__AatrixHistoryPath) release notes.
