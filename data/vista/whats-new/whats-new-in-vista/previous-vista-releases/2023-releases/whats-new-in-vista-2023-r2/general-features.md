# General Features

Vista 2023 R2 delivers on General
 user-requested enhancements, fixes, and other improvements.

## Viewpoint Log Viewer Displays the Current Server Date and
 Time

You can now see the current Server Date/Time on the Viewpoint Log Viewer.
If your Vista client is in a different time zone from the server,
 you may find it useful to see the server time when reviewing logs. The server is on
 Pacific time.
Note: This update does not change functionality—the Log Viewer
 still lists timestamps in server time. Search criteria should be entered in server time.
