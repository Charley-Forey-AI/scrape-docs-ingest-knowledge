# Project Management Features

Vista 2023 R2 delivers on user-requested Project Management enhancements, fixes, and other improvements.

## Access Vista Web's Pending PO / Subcontract Review from PM

You can now access the Pending PO / Subcontract Review approval page in Viewpoint Field Management™ from the Project Management module. The Programs menu in PM now includes a PM Pending PO / SL Review link that when selected, opens the Pending PO / Subcontract Review page in a web browser.

## Updated Informational Display for Subcontract Estimates

The informational display of Original and Available Estimate values in PM Subcontracts were updated for better visual clarity.
 The Original Estimate values, previously displayed above the Project field, are now displayed with the Available Estimate values below the Subcontract field. In addition, the Remaining Estimate field now displays the entire label and estimate value (previously, the field was cut off and did not display the estimate value).

## Add Interface Month to PO and Subcontract Change Orders

When viewing the item detail for subcontract and purchase order change orders, you can now see the original interface month for each transaction. This is especially useful if using the Correct Item feature (TasksCorrect Item), since corrections must be interfaced in the same month as the original interface month.
The following forms were updated to include the interface month as indicated:

- PM Subcontract Detail - Added an Interface Month field to the Interfaced tab.

- PM Material Detail - Added an Intfc Month field to the Interfaced tab.

- PM Subcontract Change Orders - Added an Interface Month field to the Grid tab and an Interface Mth field to the Info tab.

- PM PO Change Orders - Added an Interface Month field to the Grid tab and an Interface Mth field to the Info tab.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
