# Cloud Deployment Features

Vista 2023 R2 delivers on user-requested Cloud Deployment enhancements, fixes, and other improvements.

## Removed RDP Attachment Search Functionality

If you are an RDP user, you can no longer search attachments from the main Vista menu. This was an RDP-only feature, and it has been removed due to limited adoption.
Removal of this attachment search functionality does not impact other methods of searching. You can still use the DM Attachment Index Search form in the Document Management module. You can also still search for forms and reports from the main Vista menu.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
