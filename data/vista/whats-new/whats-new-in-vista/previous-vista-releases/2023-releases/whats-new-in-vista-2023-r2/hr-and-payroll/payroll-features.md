# Payroll Features

Vista 2023 R2 delivers on user-requested Payroll enhancements, fixes, and other improvements.

## Box 14 Reporting for W-2s in Aatrix

Box 14 values entered on the State Box 14 Information tab in PR W-2 Process are now correctly passed to Aatrix. Aatrix reports these values in the appropriate area of the W-2 based on the Aatrix tax type associated with the deduction/liability code in PR Deductions/Liabilities.
