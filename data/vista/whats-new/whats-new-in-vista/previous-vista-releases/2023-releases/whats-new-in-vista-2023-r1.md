# What's New in Vista 2023 R1

The Vista 2023 R1 release includes several enhancements and new features, as well as defect fixes that apply to all geographies.
Some of the features contained in this release include a review/approval workflow process for subcontracts, invoice delivery functionality for Job Billing invoices, the ability to assign project structure types to jobs/projects, and several other features to help improve your workflow.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
