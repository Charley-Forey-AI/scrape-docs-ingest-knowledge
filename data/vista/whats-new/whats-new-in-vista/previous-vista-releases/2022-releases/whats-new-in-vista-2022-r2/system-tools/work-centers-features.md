# Work Centers Features

Vista 2022 R2 delivers on user-requested Work Center enhancements, fixes, and other improvements.

## Make User Searches Easier

We now provide instructions for applying the necessary treatment to your Work Center inquiries so that users can use terms and tactics which they are already familiar with, making it easier for them to more easily find what they're looking for.

## Interrupt Long-running Work Center Inquiries

If the Inquiry you have just run is taking longer than you want to wait, you can cancel it using the new button visible in the grid just above the column names.
