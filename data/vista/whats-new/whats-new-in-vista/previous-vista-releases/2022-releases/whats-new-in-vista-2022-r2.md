# What's New in Vista 2022 R2

The Vista 2022 R2 release includes several enhancements and new features, as well as defect fixes that apply to all geographies.
Some of the features contained in this release include electronic delivery consent for W-2s, 1099s, and 1095s, the ability to flag employees as exempt from Federal tax, and an easier workflow for selecting and paying released retainage on AP invoices, as well as several other features to help improve your workflow.

Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.

## Improved Field-Level Help Experience

This release improves your experience when accessing Help for fields on a form.
 Field-level information for each form is now presented in a single Help article. When you press F1 from a field, you are taken to the appropriate Help location, but it now displays with all other fields on that form in a single article. This allows you to scroll through the Help for other fields on the form without having to press F1 again from the same Vista form.
This new format also eliminates lengthy lists of field definitions in the Help Table of Contents. You will now only see a "field descriptions" article for each form.
Tip: If you are on a Field Definitions article that contains a large number of fields and you know the name of the field you are looking for, you can press Ctrl + F and type in the field name, and the cursor will move to that section in the article automatically.
