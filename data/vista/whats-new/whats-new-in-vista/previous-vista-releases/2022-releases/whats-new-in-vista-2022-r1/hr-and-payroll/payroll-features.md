# Payroll Features

Vista 2022 R1 delivers on user-requested Payroll enhancements, fixes, and other improvements.

## Change to Export File Type for PR EFT Payments

The Export File Type field on the Download tab in PR EFT Payments is now enabled for U.S. companies only (that is, those companies with the Default Country field set to US in HQ Company Setup).
For all other countries, the Export File Type field is set to Active and disabled.

## Use YTD Accumulations to Correct Rounding Errors (Canada only)

The Use YTD accumulations to correct rounding errors checkbox in PR Deductions and Liabilities now defaults as enabled when you select Federal as the Calculation Category and 1-Withholding or 4-Employment Insurance as the Federal Type. The checkbox still defaults as selected, but may be unselected as needed.
 This is especially useful when an employee is switched to a Reduced EI rate part-way through the year. Deselecting the checkbox prevents the system from applying the new rate to all amounts for the year.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
