# Vista 2022 R1.03

This release contains defect fixes only; there are no enhancements included with this release.

## CM Seq # Enabled for Manual Checks

When entering manual checks in AP Payment Posting, you can now access the CM Seq # field for manual entry. The CM Seq # field was previously disabled for all payment methods; however, it is now enabled only if the Pay Method is C-Check and the Check Type is M-Manual.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
