# What's New in Vista 2022 R1

The Vista2022 R1 release includes several enhancements and new features, as well as defect fixes that apply to all geographies.
Some of the features contained in this release include auto adding new customers and service sites from work order quotes, an improved void payments process, and integration with field tickets for PM purchase orders, as well as other features to help improve your workflow.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.

## Reports Catalog Now in Viewpoint Help

 You can now access the Vista Reports Catalog in Viewpoint Help. The Reports Catalog provides information about each of the reports available in Vista, including the report location, a description of the report and report parameters, and a screenshot of the report itself.
 To access the Reports Catalog, click the Reports Catalog icon on the Vista 2022 R1 home page.

 Reports are grouped by module and then by country (United States, Canada, and Australia). Reports that apply to all regions are located in the General Reports folder for each module.
