# Project Management Features

Vista 2021 R2 delivers on user-requested Project Management enhancements, fixes, and other improvements.

## Allow Syncing Projects to ProjectSight

If you are using the ProjectSight web application, you can now sync projects to ProjectSight.
To enable this functionality, the PM Projects form now includes a new Sync to ProjectSight checkbox (on the Addl Info) tab. When you select this checkbox, the system sends the selected project to the ProjectSight web application, setting it up as a Project.

## Ability to Assume Ownership of a PCO Approval

When a PCO is in the approval process, it is rendered into a 'locked' status. If Vista then closes unexpectedly (such as a power outage), it remains in a locked status. If that occurs, you can now assume ownership of the PCO even if you were not the user originally working on the PCO approval.
 When you select the PCO in PM Approve PCOs (accessed via PM Pending Change Orders by selecting Tasks > Approve PCOs), the message displayed now includes an Override? option. If you select Yes, a second message displays stating "Please ensure other user is not actively editing this PCO. Continue?" If you select Yes, you can then access the PCO and approve it. If you select No to either message, you are returned to the PM Pending Change Orders form.
Note: Before you take ownership of a PCO approval from another user, it is highly recommended that you converse with the other user. Once you take ownership, the changes made by the other user are deleted and you must start over with the PCO approval process.
