# Viewpoint Administration Features

Vista 2021 R2 delivers on user-requested Viewpoint Administration enhancements, fixes, and other improvements.

## Reorganized VA Site Settings

The VA Site Settings form is now comprised of two tabs: Settings and Add'l Settings. The Settings tab contains general information (such as license total, max number of row settings, and security settings), while connection settings (such as Comdata FTP and Viewpoint ePayments) are now on the Add'l Settings tab. This reduces the clutter on the form and allows room for additional fields.

## Ability to Designate URL for Vista web Forms

In conjunction with the Web Integration feature, which allows direct access to Vista web forms from within the Vista client, a new Vista Web URL field was added to the VA Site Settings form for designating the base URL for Vista web forms.
This new field is automatically populated when you install the Vista web portal and displays the base URL used for Vista web forms. If you are using a Direct Company Login (PRCo Specific URLs), this field may not auto-populate with the company-specific URL, in which case, you will need to enter the base portal URL.

## Access Vista web Forms from the Vista App

You can now access selected forms in the Viewpoint HR Management for Vista, Viewpoint Financial Controls, Viewpoint Field Management, and Vista Field Service applications from the Vista Main Menu. Links to these forms are located in the Programs folder of the related Vista module, and when selected, open the form in a web browser. For a full list of the available forms, see [Vista Web Forms](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/vista-web-forms).

## Ability to Create Custom Links to External Web Page in Vista

You can now create custom links to eternal web pages using the new VA Web Forms form. Custom links are added to the Programs menu of the User Database module.
For more information about the VA Web Forms, see [VA Web Forms](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-web-forms).

## License Type Fields

In preparation for a future release, to enable determining the correct license level for users, a new License Type field was added to the VA User Profile form in Cloud-deployed Vista instances. There are two license types displayed:

-  0-Office - This license type is intended for those needing full use of Vista - both accounting and operational capabilities.

-  1-PM - This license type is intended for those accessing just the Vista forms and reports aligned with PM functions.

## Improved Queries and Inquiries

Limiting the number of rows returned by Work Center queries can improve system performance by ignoring records beyond a certain number. This release brings more consistent application of the the number you specify in the Maximum Rows field in the Inquiry Settings form.
If you prefer your Work Center query results to include more data, or if data that you expect to see is not appearing in the results, please double-check the Maximum Rows field and, if appropriate, raise the value.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
