# Human Resources Features

Vista 2021 R2 delivers on user-requested Human Resources enhancements, fixes, and other improvements.

## Non-Binary Gender Designations

You can now assign non-binary genders to resources and dependents in the Human Resources module.
In some states, employers are required to base gender designations on the gender with which each employee identifies. Adding new gender information fields to the HR Resources form and the HR Resource Dependents form allows employers to comply with gender identification requirements.
To implement this functionality, the following changes were made to the HR Resources and HR Resource Dependents forms:

- Gender Identity - Use this new field to specify the gender an employee identifies with. Options are: Male, Female, Non-Binary, and Other. Selecting Other enables the gender Description field (unlabeled).

- Description (unlabeled) - This new field is used to enter a gender description for employees with an "Other" gender identity designation (for example, Agender, Androgyne, Bigender, etc.).

- Gender - Changed the label of this field to "Reporting Gender", as this field is currently used for gender designation on EEO reports.

The new Gender Identity and gender Description fields are informational only; they are not currently used for EEO reporting.
Note: Changes to an resource's gender information are updated to PR Employees automatically if the Name update option is selected in HR Company Parameters and the Active PR Employee and Exists in PR checkboxes are selected for the employee in HR Resources.

For information about the gender information changes in PR Employees, see the [Payroll](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r2/hr-and-payroll/payroll-features) release notes.
