# Accounts Receivable Features

Vista 2021 R2 delivers on user-requested Accounts Receivable enhancements, fixes, and other improvements.

## Allow Syncing Customers to ProjectSight

If you are using the ProjectSight web application, you can now sync customers to ProjectSight.
To enable this functionality, the AR Customers form now includes a new Sync to ProjectSight checkbox (on the Addl Info) tab. When you select this checkbox, the system sends the selected customer to the ProjectSight web application, setting it up as a Company.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
