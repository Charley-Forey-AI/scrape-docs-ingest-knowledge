# General Features

Vista 2021 R2 delivers on General user-requested enhancements, fixes, and other improvements.

## Vista Version Info Added to Main Menu

You can now see the current version of Vista without having to go to Help > About Vista. The current database, release version, and tax update version are now displayed in the second pane of the main menu Status bar.

## Improved Vista Help Menu

To improve clarity, functionality, and promote product standardization, the Vista Help menu has been reorganized and updated at the Main Menu and Form levels.

## Update to Licensing Limit Enforcement

With this release, enforcement of the Licensing limit has been removed. What this means for you is:

- You can now log in to Vista, even if the number of licenses has been exceeded. The system no longer presents you with the "all licenses are in use" message.

- The Logged in Users form (Help > Support > Logged on Users) now shows only the number of licenses being used of each license type (Office or PM); it no longer shows how many licenses are available.

## Removed Cosential Link from Main Menu

The Cosential item in the Links menu (on Vista's Main Menu) was removed, as the web page is no longer in use.

## New Four-Digit Year Option in Field Properties

You now have the ability to set certain date fields to use a four-digit year format using the Field Properties form.
A new 4 digit years checkbox was added to the System Overrides tab in the Field Properties form (accessed by pressing F3 in a field). This new field is enabled only for date fields with a bDate datatype (for example, the Hire Date field in PR Employees).
When you select this checkbox for a valid date field, dates entered in that field display in a four-digit year format. If you enter a two-digit year, the system automatically converts it to a four-digit year format. For example, 070121 is converted to 7/1/2021.

## Removal of Unsupported Applications

With this release, Resource Management, Financial Manager, and Viewpoint Connects are no longer included in the Vista installers or updates, as support for these applications ended some time ago. While the components and most references have been removed, underlying data is preserved.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
