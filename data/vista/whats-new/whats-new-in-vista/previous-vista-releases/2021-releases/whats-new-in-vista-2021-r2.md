# What's New in Vista 2021 R2

The Vista 2021 R2 release includes several enhancements and new features, as well as defect fixes that apply to all geographies.
Some of the features contained in this release include merging duplicate vendors for a single vendor, invoice delivery options for service agreements, proportional salary distribution for auto earnings, updates for field tickets, and several other features to help improve your workflow.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
