# Human Resources Features

Vista 2021 R1 delivers on user-requested Human Resources enhancements, fixes, and other improvements.

## Updated HR-VETS Reporting

In compliance with current Department of Labor (DOL) standards, several changes were made to the HR-VETS reporting feature.
These changes are as follows:

- HR Resource Master - To align the veteran classifications in HR Resource Master with current DOL standards, added a new Active Duty Wartime or Campaign Badge checkbox to the Other Info tab (Veteran Status section). Selecting this checkbox indicates the employee is a veteran who served on active duty during a war or in a campaign for which a badge was authorized.
Also removed the Vietnam and Other Veteran checkboxes, as they are no longer applicable.
Note: When you install this release, the system automatically selects the new Active Duty Wartime or Campaign Badge checkbox for any employee who previously had either the Vietnam or Other Veteran checkboxes selected.

- HR Hiring Location Setup - Added new DUNS Number and NAICS Code fields to allow overriding the company DUNS and NAICS values by hiring location. If you leave these fields blank for a hiring location, the system uses the company values for HR VETS-4212 reporting.

- Headquarters - Added a new NAICS field to enable inclusion on the HR VETS-4212 reports. For more information, see the [Headquarters](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/administration/headquarters-features) release notes.

- IM Imports - Added a new ActiveDutyWartimeVetYN identifier (#466) to the HRResourceMaster import form. For more information, see the [Imports](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/administration/imports-features) release notes.

- Reports - Two new reports were created that meet current DOL requirements and veteran classifications: HR Federal Contractor VETS-4212 and HR Federal Contractor VETS Resources. In addition, several existing reports were modified to align with the current veteran classifications..
For more information about these reports, see the [Vista Reports](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/system-tools/vista-reports) release notes.

- VA Inquiries - Modifications were made to three HR-related inquiries to align with the current veteran classifications. For more information, see the [Viewpoint Administration](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/administration/viewpoint-administration-features) release notes.

-  Workflow Notifier Queries - Modifications were made to several HR-related notifier queries to align with the current veteran classifications. For more information, see the [Workflow](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/system-tools/workflow-features) release notes.
