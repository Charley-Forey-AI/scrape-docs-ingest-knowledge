# New Trimble Analytics Features for Vista: Dashboards and Vista Reports

Trimble Analytics 2021 R11 introduces two new features to help meet your needs for data insights and data mobility: Dashboards provide in-depth visualizations of your Vista data, and Vista Reports enable you to access your Vista Crystal Reports from the web.

## Dashboards

Dashboards in Trimble Analytics provide visualizations of key project data to enable you to make data-driven decisions for your enterprise.
From each Dashboard, you can filter the data to see details for a particular project, modify how the data is displayed, and export the data as either an .xlsx or .csv file.
Dashboards retain any security settings configured for a user in Vista.
For more information, see [Dashboards](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:Analytics_000004:Analytics:Analytics) in the Trimble Analytics Help.

## Vista Reports

With Vista Reports, you can easily access your Vista Crystal Reports on the web from Trimble Analytics without opening Vista.
Any custom reports and default parameters that you have set up in Crystal Reports are available in Vista Reports.
Vista Reports retain any security settings configured for a user in Vista.
For more information, see [Vista Reports](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:Analytics_000005:Analytics:Analytics) in the Trimble Analytics Help.

Note: For both Dashboards and Vista Reports, your Vista enterprise must be using single sign-on (SSO).
