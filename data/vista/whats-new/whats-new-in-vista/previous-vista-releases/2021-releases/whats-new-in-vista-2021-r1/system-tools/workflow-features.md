# Workflow Features

Vista 2021 R1 delivers on user-requested Workflow enhancements, fixes, and other improvements.

## Support Veteran Classification Changes in HR Notifier Queries

Modified the following queries to align with the current veteran classifications:

- HREBEffectDate (Resource Benefit Effective Date)

- HREBEligDate (HREB Elig Date)

- HREBEndDate (HREB End Date)

- HREBReinstateDate (HREB Reinstate Date)

- HREDFollowUpDate (HRED Follow-Up Date)

- HREGFinalAppealDate (Final Appeal Date)

- HREGHearingDate (HREG Hearing Date)

- HRERNextReviewDate (HRER Next Review Date)

- HRRMI9ReviewDate (HRRM I9 Review Date)

- HRRMPhysExpireDate (HRRM Physical Expire Date)

- HRSHNextDate (HRSH Next Date)

Changes include removing references to the Vietnam and Other Veteran statuses, which are no longer being used, and including the new Active Duty Wartime or Campaign Badge status.
For more information about the veteran classification changes, see the [Human Resources](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/hr-and-payroll/human-resources-features) release notes.
