# Document Management Features

Vista 2021 R1 delivers on user-requested Document Management enhancements, fixes, and other improvements.

## Added SM tab to Attachments Indexes

You can now view indexes for attachments added to Service Management data records. An SM tab was added to the DM Attachment Index form so that you review the existing indexes for an attachment or add new ones.

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
