# Vista 2021 R1.03

This release includes updates to PR Department of Labor EEO-1 Report (U.S.), as well as defect fixes that apply to all geographies.

## Updated PR Department of Labor EEO-1 Report (U.S. only)

The PR Department of Labor EEO-1 report (ID# 795) has been updated to facilitate reporting for the 2019 and 2020 tax years as follows:

- Removed the Annual Salary column

- Removed the Job Category pay range bands

- Modified report to improve appearance and performance

Additionally, removed the PR EEO Component 2 E-file report (ID# 1328) from the Reports menu, as it is obsolete (as of reporting tax year 2019).

## Check Issue Status

To see a list of defects fixed in this version, go to the [Track Cases/Issues page](https://support.viewpoint.com/s/browse-cases-and-issues) of the Viewpoint Customer Portal. Use the filter options to select your product and module. In the Fixed in Version field, select this version, then select Apply Filters.
