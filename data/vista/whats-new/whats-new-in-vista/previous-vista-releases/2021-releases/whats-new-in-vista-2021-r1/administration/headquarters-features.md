# Headquarters Features

Vista 2021 R1 delivers on user-requested Headquarters enhancements, fixes, and other improvements.

## Support for NAICS and DUNS for HR VETS-4212 Reporting

In compliance with current Department of Labor (DOL) standards, several changes were made to the HR VETS-4212 reporting feature, which includes adding support for NAICS and DUNS identifiers.
In the Headquarters module, a new NAICS Code field was added to the HQ Company Setup form (Info tab). You will use this field to record the NAICS code (up to 6 digits) for each company in which you do business.
When you run the HR Federal Contractor VETS-4212 report (used for exporting your data for e-filing) and the HR Federal Contractor VETS Resources report (used for viewing/printing VETS-4212 detail), the system now automatically populates the reports with the NAICS and DUNS identifiers; you no longer have to enter these identifiers in the report parameters
Note: You can override the NAICS Code by hiring location in HR Hiring Location Setup.

For more information about the changes made for HR VETS-4212 Reporting, see the the [Human Resources](/en/vista/vista/whats-new/whats-new-in-vista/previous-vista-releases/2021-releases/whats-new-in-vista-2021-r1/hr-and-payroll/human-resources-features) release notes.
