# What's New in 2026.3

The Vista 2026.3
 release includes several enhancements and new features, as well as defect fixes that
 apply to all geographies.

## General Features

This release delivers on general user-requested enhancements, fixes, and other improvements.
Trimble Assistant Main Menu IconIf you use Trimble ID to log in to Vista, you now have access to the Trimble Assistant. The Assistant in Vista is now accessed from an icon  on the main menu. Selecting the Trimble Assistant icon will open the Assistant in your default web browser.Note: The Trimble
 Assistant is a chat agent you can use to ask specific questions about
 the tasks you can perform in Vista. The assistant gets its information
 from the Vista Help.

## Human Resources Features

This release delivers user-requested HR enhancements, fixes,
 and other improvements.
New field in HR Resources: Work Email
The Work
 Email field found on the Info tab in the
 HR
 Resources form, allows users to separate professional
 email communication, if desired. For more details about this field, see
 [Work Email](/en/vista/vista/hr-and-payroll/human-resources/applicantsresources/resource-setupmaintenance/about-the-hr-resources-form/field-definitions-hr-resources-form#concept-1t0tritn--en).

## Payroll Features

This release delivers user-requested Payroll enhancements, fixes, and other improvements.
New field in PR Employees: Work Email
The Work Email field, found on the Info tab in the PR Employees form, allows users to separate professional email communication, if desired. For more details about this field, see [Work Email](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form/field-definitions-pr-employees-form#concept-214b77a1-a165-4182-9ac4-fa7e9366cbc9--en).
Smart Payroll: Automated Overtime validation
The PR Timecard Validation Rules form now includes an Overtime Validation rule. This feature automatically identifies the most beneficial overtime rule for the employee and flags timecard transactions for overpayments or underpayments before they hit an employee's paycheck.
For more information on this validation, see [Overtime Validation](/en/vista/vista/hr-and-payroll/payroll/time-entry/trimble-smart-payroll/about-validation-rules#concept-7ec33fd6-8ed1-497d-b592-b5e53072aeb8--en__overtime-validation).
Smart Payroll: Meal Break Compliance validation
The PR Timecard Validation Rules form also now includes a validation, Meal Break Compliance Rule, that checks hourly, non-exempt employee timecards against state-mandated meal break requirements. This helps payroll stay compliant with local labor laws by flagging missing or insufficient meal breaks before the payroll is processed.For more information on this validation, see [Meal Break Compliance](/en/vista/vista/hr-and-payroll/payroll/time-entry/trimble-smart-payroll/about-validation-rules#concept-7ec33fd6-8ed1-497d-b592-b5e53072aeb8--en__meal-break-compliance).

## Viewpoint Administration Features

API Published checkbox visible in VA InquiriesThe Published checkbox field is now visible in the VA Inquiries form. Most users will not interact with the field, as it indicates if the inquiry has been made available to an API, such as AppXChange. Be very careful about editing any inquiries that have this checkbox selected. They have already been made available to the API, and any changes could break existing API integrations.
For more information on the field, see [API: Published](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/va-inquiries-form/field-definitions-va-inquiries-form#ID-00048ebc--en)
Increased form and report access for PM License Type usersSeveral additional Vista forms and reports are now included under the PM License Type for users. Note: Designating a license type does not grant or restrict a user's access to programs/forms and reports. Instead it serves to limit—by license type—which users can run certain forms and reports.
Form names:

- JC Std Item Codes (JCSI)

- PO Compliance (POComplyHeader)

- SLComplyHeader

Reports:

- JC Progress Posting (#535)

- JC Projections (#544)

- JC Revenue Projections (#548)

- PO Change Order (#718)

- PO Entry (#727)

- SL Change Order (#883)

- SL Entry (#891)

For more information, see [License Types](/en/vista/vista/getting-started/getting-started-with-vista/vista-licensing/license-types).
