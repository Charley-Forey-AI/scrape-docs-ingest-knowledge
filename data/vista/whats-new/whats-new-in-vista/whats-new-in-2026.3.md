# What's New in 2026.3.1

An emergency fix has been deployed to customers on Vista version 2026.3. The 2026.3.1 release includes
 two defect fixes.
This hotfix resolves the following software issues:

- 120173: Sales tax being
 incorrectly duplicated in Job Cost totals and Actual Cost fields.

- 120158: Validation error
 preventing users from deleting Job Type transactions within AP Entry
 batches.
