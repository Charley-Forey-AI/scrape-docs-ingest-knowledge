# Hardware / Software Requirements

These are the minimum hardware and software requirements for running Vista™ by Viewpoint as of the latest Vista release.
This information changes frequently, and we recommend you contact Viewpoint prior to purchasing your hardware and software if you have any concerns about suitability.
Note:Attention VRL Cloud Customers:Trimble Viewpoint
 manages architecture, performance, and applications on behalf of customers whose
 Vista instance is hosted in the
 cloud. However, as a VRL cloud customer, you are responsible for installing the
 Vista client on your local machine.

The customer agrees to provide a computer system(s) that complies with the specifications of the current minimum hardware and software requirements as listed, and to have the system configured by a qualified person in accordance with those requirements.
These requirements and guidelines consider only a single dedicated server scenario for Vista. Discuss additional installation considerations with your Viewpoint project manager or deployment engineer.
