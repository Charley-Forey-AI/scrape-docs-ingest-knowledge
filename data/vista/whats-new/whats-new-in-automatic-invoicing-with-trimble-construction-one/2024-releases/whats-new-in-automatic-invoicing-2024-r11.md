# What's New in Automatic Invoicing 2024 R11

With this release, when reviewing invoice details, you can now
 select description text in the PDF to update the Description field.

## Description Field Includes Select Tool

When reviewing invoice details on the Process page, you can now
 use the select tool in the Description field to select text from the PDF. The
 Description field will be updated with the selected text.
