# What's New in Automatic Invoicing 2025 R1

With this release, when processing an invoice, Automatic Invoicing now populates the Description
 field automatically.

## Description Field Automatically Populated

With this release, when you upload an invoice, if the invoice
 includes a description, Automatic Invoicing automatically enters the description text in the
 appropriate field. If your invoice does not contain a description, Automatic Invoicing generates one based on
 other information within the invoice. You can edit the field as needed.
