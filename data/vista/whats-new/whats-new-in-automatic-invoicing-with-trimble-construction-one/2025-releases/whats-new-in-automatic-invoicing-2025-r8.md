# What's New in Automatic Invoicing 2025 R8

This release of Automatic Invoicing includes a new layout for processing invoices.

## New Layout for Processing Invoices

An improved layout on the Process page makes it easier to review
 and edit invoice details. You can enable the new layout by selecting the New UI option in the upper
 right-hand corner of the page.
