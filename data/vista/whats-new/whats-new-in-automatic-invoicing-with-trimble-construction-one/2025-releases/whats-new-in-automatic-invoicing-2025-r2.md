# What's New in Automatic Invoicing 2025 R2

With this release, you can now easily toggle the automatic
 description field feature from the Invoice Settings tab.

## New Automatic Invoice Description Setting

With this release, you can now toggle the feature to automatically
 capture invoice descriptions from the Invoice Settings tab. For more information,
 see [Edit Invoice Settings](/en/vista/vista/accounting/accounts-payable/invoices/unapproved-invoices/automatic-invoicing-for-trimble-construction-one-users/edit-invoice-settings).
