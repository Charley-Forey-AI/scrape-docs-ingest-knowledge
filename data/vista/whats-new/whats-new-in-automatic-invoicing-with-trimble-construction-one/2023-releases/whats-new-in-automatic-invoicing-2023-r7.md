# What's New in Automatic Invoicing 2023 R7

This release introduces Invoice Settings that apply to all new invoices you add with Automatic Invoicing. The Accounts Payable Admin can modify the settings to remove special characters from the Invoice Number field.

## Invoice Settings: Remove Special Characters from the Invoice Number Field

With this release, you can now modify settings that apply to all new invoices added to Automatic Invoicing. Invoice settings currently include the option to automatically remove any special characters from the Invoice Number field when new invoices are processed.
