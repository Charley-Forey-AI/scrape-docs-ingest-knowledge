# What's New in Automatic Invoicing 2023 R11

This release adds an optional Description field to the invoice details
 when processing an invoice.

## New Description Field in Invoice Details

With this release, on the Process page, when reviewing the Invoice
 Details, you can now add a description for the invoice in the Description field. The
 Description field is optional. Descriptions must be 30 characters or less.
