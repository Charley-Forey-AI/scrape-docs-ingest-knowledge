# About the IM Template Form

A template must be set up before you can import data from third-party programs into Vista™ data files.
Templates are set up with certain specifications enabling the system to recognize, use, and convert the data file created by the third-party program. Template specifications identify the file format of the import file, the organization structure of the import (record types), the fields/data being imported (including location in the data file, as well as any default values), and cross-references that assist in data conversion.
You can use a standard template, which Vista provides for importing data. See [Standard Import Templates](/en/vista/vista/administration/imports/setup-and-maintenance/standard-import-templates) for a list of available templates. If you are using a standard import template, you do not need to use the IM Template form.
You can also create a custom template, modify a copy
 of a template, or export/import a template.
