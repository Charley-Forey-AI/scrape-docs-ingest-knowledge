# Field Definitions: IM Template Copy Form

The following is a list of field descriptions for the IM
 Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Source Template

 This field automatically defaults the currently active template on IM Template.
 Accept the default or enter another template to copy. Press F4 for a list of valid templates.

##  Destination Template

 Enter a name or number to uniquely identify this new template, up to 10 characters.

##  Description

 Enter a description for this template, up to 30 characters.
