# About the IM Template Copy Form

Use this form to copy an existing import template to a new
 template. You can access this form by selecting File > Copy
 Template in the IM Template form.When you first access this form, the Template field defaults the current template
 from IM Template, but you can select other source templates, as necessary. Once you
 copy a template, the system copies all of the settings (including information on the
 Info, Record Types, and Template Detail tabs) from the source template to a new
 template. You can then modify data on the new template as needed.
For more information on copying a
 template, see [Copying Import
 Templates](/en/vista/vista/administration/imports/setup-and-maintenance/copy-import-templates).
For more
 information on creating or modifying a template, see [Creating an Import Template](/en/vista/vista/administration/imports/setup-and-maintenance/create-an-import-template).
