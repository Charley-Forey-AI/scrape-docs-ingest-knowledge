# Direct Imports

The system uses direct imports to update non-batch processing forms/tables. While processing direct imports, the system uploads data directly into the associated database tables.
If you import the wrong data while running a direct import, you must restore your system to a previously backed-up version or manually remove imported data. Make sure to review all data carefully in [IM Work Edit](/en/vista/vista/administration/imports/processing/about-the-im-work-edit-form) before running the [IM Upload](/en/vista/vista/administration/imports/processing/about-the-im-upload-form) form.
