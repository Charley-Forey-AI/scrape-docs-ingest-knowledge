# Import Forms

Each import template you create must specify an Import Form.
 Import Forms identify the database tables into which data is imported.
The following import forms are currently available in the system. If additional information is available for an import form, an Add'l Info link displays in the Sample / Add'l Info column. If a sample file is available for an import, you can access that file by selecting the sample link. Sample files show which fields require entry, which fields are importable, and which fields have Viewpoint Defaults available, along with specific comments for each field.
