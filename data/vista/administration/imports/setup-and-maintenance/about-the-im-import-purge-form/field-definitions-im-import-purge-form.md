# Field Definitions: IM Import Purge Form

The following is a list of field descriptions for the IM
 Import Purge form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Import ID

 For Import ID Purge only.
 Specify the import ID to purge. Press F4 for a list of valid import IDs for the specified template. Then click the Purge button. The purge process will purge the import ID for the specified template from IM Work Edit. The template will not be purged.

##  Import Template

 If purging an import template, specify the import template to purge and click the Purge button. The purge process will delete the template header, template detail, record types, cross-references, and any existing pending import IDs for the template.
 If purging an import ID, specify the template to which the import ID is associated.

##  Select Type of Purge

 Select one of the purge options:

- Import Template Purge – to purge an import template and associated detail.

- Import ID Purge – to purge an import ID for a specific template.

- Orphaned Message – to purge messages related to imports that no longer exist.
