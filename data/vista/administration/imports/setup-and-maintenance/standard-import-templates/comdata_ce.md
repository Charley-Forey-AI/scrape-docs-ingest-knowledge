# COMDATA_CE

This is a batch import template for importing Comdata CEMS data into AP Transaction Entry.
Imports using this template will insert records into an AP Transaction Entry batch for processing.
Note: You may need to modify the template to provide a default Comdata Vendor (APHB Identifier #35) and a default GL Account (APLB Identifier #155). Since Viewpoint does not prevent edits to the standard template, it is highly recommended that you copy this template and make your modifications to the copy. This will ensure the standard template maintains its integrity.
