# JC Change Order

 Additional information about the JC Change Order template.
This import supports new change order records as well as updates to existing change order records.
The imported Change Order data imports budget changes as approved change orders. Budget changes will include quantities, costs, and hours by job, phase, and cost type.
This import has three record types. These record types are HDR (JC Change Order Header), ITEM (JC Change Order Items), and ITEMDTL (JC Change Order Item Detail).

## ProjectSight

Importing Change Order data from ProjectSight into Vista creates a JC Change Order batch that you can then validate and post in Vista.
