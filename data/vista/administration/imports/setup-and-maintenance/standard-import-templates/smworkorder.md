# SMWorkOrder

This is an "insert" only import. You cannot use this import to update existing work orders.
Additionally, if you import a work order that is missing a work order number, and you do not assign a work order number in IM Work Edit, the upload process will auto-assign a work order number using the highest existing work order number plus 1.
