# SMAgreementDeferral

This is a direct import (i.e., data is uploaded directly to the data table) that allows for inserting new records and updating existing ones.
Note: You can only import agreement revenue deferral schedules for agreement revisions that are in Quote status. If an agreement revision has already been activated, you will need to either deactivate the agreement or create an amendment.
