# HR Resources

The HRResMast template contains only fields for the actual
 HRRM table.
It does not include fields for related tables such as Applicants, Contacts, Dependents, Benefits, etc. Also, in this template, you cannot change the value of a last name, social security number, sort name, or employee number for an HR Resource.
