# SMCustomer

Customers being imported for upload to SM Customers must first be set up in AR Customers.
You can set them up manually or import them using the AR Customer import. Although you can run the SM Customer import at any time, customers must be set up in AR before you upload the customers to SM Customers (in IM Upload).
