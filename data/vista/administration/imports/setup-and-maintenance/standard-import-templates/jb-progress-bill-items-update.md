# JB Progress Bill Items Update

Use this template to import either “to-date” or “current” data.
If both “to-date” and “current” data should be imported (e.g. some of your customers have “to-date” and some have “current”), it is recommended that multiple templates are set up to accommodate this need.
Note: The Direct Import Type for this template is set to “Update Only.” Although the “Insert & Update” and “Insert Only” options are also available, this import is currently designed to update values for existing items only, not insert new items. Therefore, it is recommended that this option is set to “Update Only”.
