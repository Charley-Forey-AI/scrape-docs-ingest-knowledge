# SMWorkCompleted

This is a direct import (that is, data is uploaded directly to the data table) that allows for inserting new records and updating existing ones.

Imported work completed Equipment, Miscellaneous, or Inventory lines are set with a status of Provisional in SM Work Orders. This means that unlike manually entered work completed lines, posting does not occur automatically; you must batch and process these lines using the SM Work Order Cost Posting.
