# PO Entry - ProjectSight

Additional Information about the PO Entry - ProjectSight template.
You can import data for Vista Purchase Orders from ProjectSight.
The imported data inserts new purchase orders/items, as well as new items for existing open purchase orders, into a PO Entry batch. Note: The imported data will not update existing PO Items. Manual edits must be made for these changes.
