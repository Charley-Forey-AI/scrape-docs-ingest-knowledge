# About the IM Cross Reference Form

Use this form to create cross references.
Cross references have three uses:

- Convert data from one value to another

- Link incompatible third-party data formats with a valid Vista format

- Link multiple data file records with a single Vista field

Once you create cross-references, you need to associate them with import template fields in [IM Template ](/en/vista/vista/administration/imports/setup-and-maintenance/about-the-im-template-form).
