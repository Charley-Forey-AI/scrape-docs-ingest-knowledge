# Copy Import Templates

You can copy existing import templates and modify them to meet your needs using the IM Template Copy form.

You may find it necessary to create a new import template based on an existing one. In fact, it is recommended that you do not modify standard import templates, but rather copy them and modify the copy. For a list of standard templates, see [](/en/vista/vista/administration/imports/setup-and-maintenance/standard-import-templates)[Standard Import Templates](/en/vista/vista/administration/imports/setup-and-maintenance/standard-import-templates).
 The following instructions detail how to copy import templates.

1. Select File > Copy Template from the IM Template form.  The system displays the IM Template Copy form.

1. Enter the name of the template you are copying from in the Source Template field. Press F4 for list of available templates.Note: The Source Template field defaults to the current template on the IM Template form.

1. Enter the name of the template you are creating in the Destination Template field.

1. Enter a description of the template in the Description field.

1. Select Copy. The system displays a confirmation dialog box.

1. Select OK.

1. Close IM Template Copy.

Locate the new template in IM Template and make modifications as necessary. For more information, see [Creating an Import Template](/en/vista/vista/administration/imports/setup-and-maintenance/create-an-import-template).
