# Field Definitions: IM Upload Form

The following is a list of field descriptions for the IM
 Upload form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Import ID

 Enter the Import ID of the import work file to upload into the system’s data files. This field defaults the last active import ID. Press F4 for a list of valid Import IDs.
