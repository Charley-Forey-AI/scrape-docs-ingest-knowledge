# About the IM Error Messages Form

This form is used to view any errors that exist in the work
 file.
To access this form, select Options > View
 Errors on the IM Work Edit form. If the work file does not have any errors,
 this option is unavailable. When errors exist, a message displays at the top of the
 IM Work Edit form to alert users.
When an error exists, this form displays the Record
 Sequence, Identifier, Column Name, and Column Description for the error. The Message
 field displays a message that identifies the error. Errors should be corrected
 before processing the upload with IM Upload.
All fields on this form are display only.
