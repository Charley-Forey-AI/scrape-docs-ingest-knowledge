# Clearing the Work File

Click Clear Work File to remove all
 records from the work file for the specified Import ID.
Once the work file is cleared, the records can only
 be retrieved by re-importing the data file. Additionally, the work file can be cleared
 by selecting Options > Clear Work
 File.
