# About the IM User Input Detail Form

This form displays when the import process encounters a record
 that has the Prompt
 on Input box checked in IM Template (Template Detail tab).
Use this form to enter a value for the specified
 column. This option is typically used when user defaults are entered, but the
 default changes with each import (e.g. Company, CM Account, PR Group, Vendor Group,
 etc.). Having the import prompt for a value eliminates having to set up multiple
 templates with differing values, changing the value manually in the template detail
 before importing the text file, or editing the value in the work file.
