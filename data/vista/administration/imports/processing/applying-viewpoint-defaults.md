# Applying Viewpoint Defaults

Select Options > Apply Viewpoint
 Defaults to reapply Viewpoint defaults to the values for a selected
 record.
Typically, this feature is only used if changes
 were made to fields that specified a Viewpoint value and you want the system to revert
 back to the original default.
