# About the IM Import Form

Use this form to create the temporary work file needed to transfer data from third-party software packages into a limited set of Vista™ software data files.
Imported data typically includes, but is not limited to, payroll timecard detail, job cost adjustment detail, equipment usage and cost adjustment detail, fuel usage, and bank reconciliation data. Use IM Work Edit to edit the work file as necessary.
