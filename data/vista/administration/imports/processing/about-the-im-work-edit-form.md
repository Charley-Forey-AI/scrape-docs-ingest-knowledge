# About the IM Work Edit Form

Use this form to edit the records in the work file (created when importing a data file with IM Import).
This form displays imported data records
 sorted by import ID and record type. For each existing record, you can add, delete, or
 modify data as necessary. However, only the Upload Value field can be edited for a
 record; all other fields are display only.
The data displayed in the grid differs
 depending on the import template and the record type. When importing multiple files,
 edits can be made to records associated with each record type. Once all edits are
 complete, work file data must be uploaded into the appropriate system data files using
 IM Upload. For more information, refer to IM Upload in Related Topics below.
