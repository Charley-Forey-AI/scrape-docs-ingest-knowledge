# About Updating All Record Sequences

Use the “Update all Record Sequences” option, available from
 the Records menu, to specify whether to automatically update all record sequences each time
 an upload value is changed for a specific record sequence.
If selected, all record sequences for an Import ID/Record Type/Import
 Template/Identifier will be updated with the same value each time an upload value is
 changed for any sequence in the work file. If not selected, the upload values for each
 record sequence must be changed individually.
