# Edit an Existing Custom Table/Form

Once you have created and saved a custom form, you can edit it as necessary.
Before making changes to a form, carefully consider the data stored in the table before making any significant changes.
Note: When setting up custom UD tables and forms, do not use special characters in column or table names.The underscore ( _ ) is the only special character allowed. Use of any other special character will result in a processing failure for any dependencies.

To make changes in a form:

1. In the UD User Table and Form Setup form, make changes in the upper and lower sections.

1. If you make changes you want to undo, click Undo Changes, and the table will revert to its previous state.

1. When all changes are made to your satisfaction, click Update Table. Important: Once the Update Table button is clicked, the table is saved and the Undo Change button becomes unavailable. If changes are needed, columns, or the entire table, will need to be deleted and recreated with the proper settings.
