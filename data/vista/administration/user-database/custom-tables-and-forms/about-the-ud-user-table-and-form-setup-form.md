# About the UD User Table and Form Setup Form

You can use the UD User Table and Form Setup form to create
 custom tables and forms.
From here, you will define the table name, as well as the columns within the table. Once you save the table and columns, the new form appears in the UD Programs folder. You can then move the form to any module folder(s) as appropriate.
Note: When setting up custom UD tables and forms, do not use special characters in column or table names.The underscore ( _ ) is the only special character allowed. Use of any other special character will result in a processing failure for any dependencies.
