# About the UD User Lookups Copy Form

You can use the UD User Lookups Copy form to copy any standard lookup.
Standard lookups cannot be modified; therefore, if you want to make changes to a standard lookup, copy it first and then modify the copy. This form is accessed by selecting File > Copy Lookup from the UD User Defined Lookups form.
