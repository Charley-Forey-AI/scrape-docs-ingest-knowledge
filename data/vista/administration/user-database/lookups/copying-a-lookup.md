# Copying a Lookup

Use the User Lookups Copy form to copy any standard lookup.
Standard lookups cannot be modified, so if you want to make changes to a standard lookup, copy it first and then modify the copy.

To copy a lookup:

1. In the UD User Defined Lookups form, press F4 in the Lookup field to select from a list of lookups.

1. Click File  Copy Lookup. The User Lookups Copy form opens.

1. In the Source Lookup field, specify the lookup to copy by pressing F4 and selecting from a list of valid lookups. Note:User defined lookups have a “ud” prefix and display at the bottom of the list.

1.  In the Destination Lookup field, enter the name of the new lookup. Make sure to begin the name with the letters ud.

1.  In the Destination Lookup Title field, enter a description of the new lookup .

1.  Click Copy. A confirmation dialog box displays.

1. Click Close. Focus returns to the UD User Defined Lookups form with the new lookup displayed in the Lookup field.
