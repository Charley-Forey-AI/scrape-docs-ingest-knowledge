# Field Definitions: UD User Lookups Copy Form

The following is a list of field descriptions for the UD User
 Lookups Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Source Lookup

 Specify the standard lookup to copy. Press
 F4
 for a list of available lookups.

## Destination Lookup

Enter the name of the new lookup, including a “ud” prefix. Up to 30 characters are allowed, no spaces (for example “udCompanyVehicles”). This name is used throughout Vista™ whenever assigning this lookup to a field.

##  Lookup Description

 Enter a description (title) for the new lookup, up to 30 characters, including spaces.This will be the description/title shown for this lookup whenever F4 is pressed at Lookup.
