# Field Definitions: VA Scheduled Changes Form

The following is a list of field descriptions for the VA
 Scheduled Changes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Effective Date

If you are adding a scheduled data change for a field, enter the date that the system should process the scheduled data change that you are creating. Additionally, you can click the Calendar button to select a date from the calendar window that displays.
After you save the scheduled change record, you cannot update this field. If you used an incorrect date, you will need to create a new scheduled change record.
If you are trying to find an existing record,
 you can press F4 from this field to find additional dates that share the existing form
 and field specified in the Form and Field fields.
 Once you select a date from the lookup window, the system places focus on the Key ID to
 Update field.

## New Value

If you are entering a new scheduled data change, enter the value that should display in the field after the effective date. Once you save the record or tab off of the field, the system will validate that the format of the entry is correct for the field. For example, if you are entering an address in a field that stores 60 characters, and the data you enter is 70 characters, the system will truncate your entry to 60 characters.
If you are reviewing existing scheduled change records, this field displays the data that will be applied on the effective date. You can enter a new value if you need to update the scheduled change data.

## Field

This field is enabled when you access VA Scheduled Changes from the VA Programs folder on the Main Menu.
You cannot enter new values in this field; use
 this field to search for scheduled data changes in the system. Press F4 to see a
 list of fields, on the form that you specified in the Form field, that have associated
 scheduled data changes. Once you select a field from the lookup window, the system place
 focus in the Effective Date field.

## Form

This field is enabled when you access VA Scheduled Changes from the VA Programs folder on the Main Menu.
You cannot enter new values in this field; use
 this field to search for scheduled data changes in the system. Press F4 to see a
 list of forms that have associated scheduled data changes. Once you select a form from the
 lookup window, the system places focus on the Field field.

## Key ID to Update

This field is only available when you access VA Scheduled Changes from the VA Programs folder on the Main Menu.
You cannot enter new values in this field; use
 this field to search for scheduled changes in the system. Press F4 to see a
 list of key IDs (that are associated with the form, field, and effective date that you
 specified) that have associated scheduled changes.
Note: This field helps to identify individual data change
 records that have the same form, field, and effective date for this form only. If you need
 to identify the exact record in the system, review the information displayed in the
 Key
 Fields field. The Key Fields field lists the record's key
 field values from the form displayed in the Form field.
