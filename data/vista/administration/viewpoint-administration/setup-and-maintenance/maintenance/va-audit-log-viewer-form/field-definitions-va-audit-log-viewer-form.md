# Field Definitions: VA Audit Log Viewer Form

The following is a list of field descriptions for the VA
 Audit Log Viewer form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Company

Company field on the VA Audit Log Viewer form
Enter the company number for filtering audit
 records. Press F4 for a list of valid companies.
If you leave this field blank, the system will display records for all companies.

## Field

Field field on the VA Audit Log Viewer form
Enter the field for filtering audit records.
 Press F4 for a list of valid fields.
Note: To use the F4, you must first specify a table in the Table field.
If you leave this field blank, the system will display records for all fields in the system.

## Table

Table field on the VA Audit Log Viewer form
Enter the table for filtering audit records.
 Press F4 for a list of valid tables.
If you leave this field blank, the system will display records for all tables in the system.

## User

User field on the VA Audit Log Viewer form
Enter the user for filtering audit records. Press F4 for a list of system users.
If you leave this field blank, the system will display records for all users in the system.
