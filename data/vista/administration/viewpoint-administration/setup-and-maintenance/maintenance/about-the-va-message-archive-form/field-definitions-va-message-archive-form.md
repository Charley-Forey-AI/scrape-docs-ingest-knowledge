# Field Definitions: VA Message Archive Form

The following is a list of field descriptions for the VA
 Message Archive form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Attempts

This field displays the number of attempts the system made to deliver the message.

## Body

 This field displays the body text of the message.

## CC

 Enter the courtesy copy address for filtering the message archive.

## CC

 This field displays the address that a courtesy copy of the message was sent to.

## Date Sent

 This field displays the date the message was sent.

##  Failure Date

 This field displays the date that the system failed to send the message, if applicable.

## Failure Reason

 This field displays the reason the message was not delivered, if applicable.

## From

 Enter the From address for filtering the message archive.

## From

 This field displays the address that sent the message.

##  Sent

 Select this checkbox to filter by sent messages.

## Source

Enter the source for filtering the message archive.

## Source

This field displays the source of the message (e.g., Notifier, Workflow, PTO requests).

## Subject

Enter the subject line for filtering the message archive.

## Subject

This field displays the subject line of the message.

##  To

 Enter the To address for filtering the message archive.

## To

 This field displays the address that the message was sent to.
