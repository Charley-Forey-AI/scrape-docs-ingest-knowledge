# VA Change Server Name Form

Use this form to change the Vista™ server name for report and PM Word document locations.
This form has two fields: Current Viewpoint Server and New Viewpoint Server. The Current Viewpoint Server field displays the existing Viewpoint server name. To change the name, enter the new server name’s in the New Viewpoint Server field and click OK to update the report and PM Word document locations.
Note: If the system stores attachments by file location, changing the server name might make these attachments unavailable. Make sure to check these paths in DM Attachment Options and change the server name if necessary. You should also check to make sure your subdirectory structure is applicable to the new server. If it is not, modify the path or restructure the subdirectory as applicable.
Once you change the server name, reconfigure all clients towards the new server.
