# Field Definitions: VA Change Server Name Form

The following is a list of field descriptions for the VA
 Change Server Name form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## New Viewpoint Server

New Viewpoint Server field on the VA Change Server Name form
 Enter the name of the new Viewpoint server. Enter only the portion that comes before the Viewpoint share on the new server. For example, if your new server path is \\NewServer\Viewpoint\Reports, enter “NewServer” in this field.
