# Setting a Datatype Label

Use this form to set labels (field description) for fields associated with certain datatypes.
This is useful when standard Vista™ software terminology does not quite match your organization’s terminology. For example, many customers may adjust the ‘Phase Code’ label to read ‘Cost Code.’ This is also known as “soft” or “dynamic” labeling.
 To set a new label for the datatype:

1. In the VA Datatype label
 form, in the Datatype field, press
 F4 to select from a list of datatypes.

1. Enter the new
 description in the Label field and save the
 record. All fields using that datatype now display the new label.
