# About the VA Scheduled Changes Form

The function of this form differs depending on how you access it.
If you accessed this form by clicking Scheduled Changes from the shortcut menu of a field, you can use this form to specify a data change for a field and enter the effective date for the change. If you are a system administrator, you can access this form from the VA Programs folder on the Main Menu to monitor all scheduled changes in the system and make adjustments as necessary.
Regardless of how you access this form, it displays information regarding scheduled changes such as the value of the change, who created the change record, the time and date that the change record was created, the date that the change was applied, the update status (Pending, Applied, and Error), and any update status messages.
