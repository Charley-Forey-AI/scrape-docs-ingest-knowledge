# Field Definitions: VA Process User Temp Copy Form

The following is a list of field descriptions for the VA User
 Temp Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Copy Options

Select the copy option that you want to use when creating a new subfolder for a user's My Tasks folder (Menu tab, Main Menu).

- Copy and Append - Select this option to add the subfolder to the user's My Tasks folder, but not remove any existing subfolders.

- Copy and Replace - Select this option to add the subfolder to the user's My Tasks folder and remove all existing subfolders.
