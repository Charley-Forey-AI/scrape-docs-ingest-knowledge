# VA Business Intelligence Mgmt Form

This form applies only to customers that purchased the Business Intelligence module.
Use this form to:

- Select which companies should be included in the OLAP cubes.

- Identify your SQL Server Analysis Services (SSAS) server. The system uses the SSAS server to process information for the Business Intelligence (BI) dashboards — this does not apply to the SSRS report parts released beginning in version 6.4.0.

- Process the Viewpoint OLAP (Online Analytical Processing) database.

BI dashboard reports rely on the Viewpoint OLAP database. OLAP information is static; the system processes the Viewpoint OLAP database in specified intervals. Processing can be resource-intensive, so it typically occurs during off-hours.
