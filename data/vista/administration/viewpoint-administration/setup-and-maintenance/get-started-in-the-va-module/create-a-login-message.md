# Create a Login Message

You can use VA Site Settings form to create a message that displays when users log in to the system.
Typical uses of this feature might be to alert users to system maintenance or post a reminder.
To create a log on message:

1. In VA Site Settings,
 enter the message in the Login Message text box.


1. To have the message
 display when users log in, select the Login Message Will Be
 Displayed checkbox.

1. Click OK
 to close the form. The message now displays when users access the system. To stop displaying the message,
 deselect the Login Message Will Be
 Displayed checkbox.
