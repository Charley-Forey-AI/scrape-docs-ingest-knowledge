# Field Definitions: VA Menu Templates Form

The following is a list of field descriptions for the VA Menu
 Templates form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Active

Active checkbox on the VA Menu Templates form
Select this checkbox to activate this template. When this checkbox is selected, this template is available for users to use when adding a subfolder to a root folder on the Menu tab of the Main Menu.

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)

## Folder Template #

Folder Template # field on the VA Menu Templates form
Enter the folder template number. If you are
 adding a new template number, enter N, New, or + to create a new template.
Note: The system begins numbering custom templates at 10,000. This helps to distinguish them from the standard templates that arrive with the system.

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)

## Folder Title

Folder Title field on the VA Menu Templates form
Enter the name of the template, up to 30 characters.

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)

## Item

Item field on the VA Menu Templates form, Menu Items tab
Enter the item information in this field. If you selected F-Form from the Item Type drop-down, enter the correct form name. If you selected R-Report from the Item Type drop-down, enter the correct report ID. Press F4 for a list of valid forms or reports.

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)

## Item Type

Item Type field on the VA Menu Templates form, Menu Items tab
Select the correct type for this folder item:

- F-Form

- R-Report

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)

## MenuSeq

MenuSeq field on the VA Menu Templates form, Menu Items tab
Enter the display sequence for the item.
Tip: To make it easier to add additional items to your folder template, enter the display sequence in increments. For example, you might enter the first item with a sequence of 10, the next one with a sequence of 20, and so on. This allows you to easily add items to the middle of your sequence without having to renumber each item.

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)

## Module

Module field on the VA Menu Templates form
Enter the module that this template relates to. When users add a subfolder to the specified module's root folder on the Menu tab of the Main Menu, this template will be available for selection.

Related information

- [VA Menu Templates Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-menu-templates-form)
