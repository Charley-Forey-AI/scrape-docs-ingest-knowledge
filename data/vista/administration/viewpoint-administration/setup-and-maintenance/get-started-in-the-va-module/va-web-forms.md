# VA Web Forms

The primary use for this form is for Vista site admins to link web page addresses (URLs) to commonly accessed sites directly into the Vista Main Menu.
You can easily navigate to custom sites or other web sites from a centralized and convenient location under User Database >  Programs. You can use these links to access virtually any site, such as internal company sites, collaborative project sites, or commonly used third-party sites and applications.
Form security allows site security admins to control access to these links as they would with any standard form. For more information, see [VA Form Security Form](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-form-security-form).
