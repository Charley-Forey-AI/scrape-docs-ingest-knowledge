# Identify the SSAS Server

The system uses the SSAS server to process information for the Business Intelligence (BI) dashboards, including which companies are included in the OLAP cubes. In some cases, you may need to identify your SSAS server.
To identify your SSAS server:

1. Open the VA Business Intelligence Mgmt form.

1. Enter the name of the
 server in the SSAS Server Name field.


1. Enter the name of the
 Viewpoint OLAP (Online Analytical Processing) database in the SSAS
 Database field.

1. In the SSAS
 Job field, enter the name of the job that will process the OLAP
 database.

1. In the Companies to
 Process field, select all companies that you want to include in
 processing the OLAP database.

1. Click Save.

1. Process the OLAP database. For more information, see [About Processing the Viewpoint OLAP Database](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/process-the-viewpoint-olap-database).

1. To identify the server
 hosting the Viewpoint OLAP database, enter the name of the server in the
 SSAS
 Server Name field and click Update.
