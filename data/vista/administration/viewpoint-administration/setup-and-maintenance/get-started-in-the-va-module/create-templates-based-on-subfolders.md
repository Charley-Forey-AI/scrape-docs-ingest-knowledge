# Create Templates Based on Subfolders

You can create templates based on a user's existing My Tasks
 subfolders using VA Menu Templates.
When you create a template based on an existing subfolder, you will specify the template name, an associated module, and the folder items (forms and reports) that you want associated with the template.
To create templates based on existing subfolders.

1. From VA Menu Templates,
 select File > Copy
 Template. The system displays the VA User Template Copy form.

1. In the Source section of the form, select the User radio button. To the right of the radio button, specify the user. Press F4 to see a list of valid user logins.

1. In the Destination section of the form, select the New Template radio button.

1. Enter a name for the
 template in the Name field.

1. Enter the associated
 module in the Module field. Note: When you associate a module with this
 template, users can create a subfolder based on this template in the root
 folder for the module on the Folders pane of the Main Menu. For more
 information on users adding subfolders to the Folders pane, see [Menu Tab](/en/vista/vista/system-tools/user-interface-guide) or [Main Menu Options](/en/vista/vista/system-tools/user-interface-guide/about-the-main-menu/main-menu-options). In order for this function
 to work, however, you must first activate the template. See [Creating and Managing Folder Templates](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/create-and-manage-folder-templates) for more
 information.

1. Click Initialize. The system displays the VA Process User Temp Copy
 form.

1. Use the arrow buttons between the Source and Copy selection boxes to specify which items should be copied to the destination template. See [VA Process User Temp Copy](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-process-user-temp-copy-form) for more information.

1. Click Apply. The system displays a success message.
