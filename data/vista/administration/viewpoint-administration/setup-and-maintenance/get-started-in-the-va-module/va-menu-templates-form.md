# VA Menu Templates Form

Use this form to create and manage folder templates, which users can then apply to their application windows.
For example create a folder for the PM module that contains all of the forms used to process changes orders, including the order in which the forms should be used. A user that processes change orders can then apply this template to their application window, and all of the forms that they need will be grouped and ordered.
A number of standard templates are available for your use, or you can create your own. To distinguish between standard and custom templates, the system begins numbering custom templates at 10,000.
