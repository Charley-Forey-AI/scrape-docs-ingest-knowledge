# VA Process User Temp Copy Form

Use this form in conjunction with VA User Template Copy to create subfolders in the My Tasks folder for users.
These subfolders are based on folder templates created in VA Menu Templates, and contain the items (forms and reports) specified for the template. Additionally, you can use the two forms to copy existing subfolders and add them to a user's My Tasks folder, or create a folder template based on that existing subfolder and its items.
You will use this form to specify the exact folder items that you want to copy from the source (user or template) to the destination (user, security group, or template).
This form has two boxes: the Source list box and the Copy To list box. The Source box displays the settings associated from the source (a template or a user's My Tasks folder). The Copy To box displays the root folder (either a template name or the user's My Tasks folder) , as well as any items that you add to the folder from the source location.
Both the Source box and the Copy To box display a hierarchical view of the template/folder. At the top level of the hierarchy is the template/folder. All associated subfolders display below the root folder. To expand a folder, click the plus sign to the left of the folder. To collapse a folder, click the minus sign.
To create folders or items in the destination location, you will need to move them from the Source list box to the Copy To list box. To move items between the two list boxes, you can drag and drop items. You can also drag and drop items within a list box to determine the hierarchy in which the items and folders display.
Note: When you are creating a template based on multiple subfolders from a user, the system takes all user items and places them in the template's root folder. Templates cannot have nested subfolders. You can use this form to copy subfolders with nested subfolders to a user's My Tasks folder, however.
You can also use the buttons on this form to move items or change the view of the list box. The following table provides a description of each of the buttons on this form.
Button
Description

Copy all folders and items from the Source list box to the Copy To list box.

Copy currently selected folder or item from the Source list box to the Copy To list box.

Remove the currently selected folder or item from the Copy To list box.

Remove all folders and items from the Copy To list box.

Collapse all open folders in the list box.

Expand all folders in the list box.

Moves an item up into an adjacent subfolder. If no subfolder exists, moves the item into the root folder. This button has no effect for items at the root folder level.

Moves an item down into an adjacent subfolder. If there is no additional subfolder under the current subfolder, the item remains in place.
