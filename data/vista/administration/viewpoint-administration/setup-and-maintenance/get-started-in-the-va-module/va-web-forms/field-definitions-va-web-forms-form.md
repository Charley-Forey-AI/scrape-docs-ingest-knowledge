# Field Definitions: VA Web Forms Form

The following is a list of field descriptions for the VA Web
 Forms form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Form Name

The Form Name field on the VA Web Forms form.
Enter the form name. Entry must start with "ud". For example, udviewpoint.

## Menu Title

The Menu Title field on the VA Web Forms form.
Enter the title you would like to display on the User Database Programs menu in Vista.

## URL

The URL field on the VA Web Forms form.
Enter the URL for your entry. The URL must include "https://" at the beginning. For example, https://help.viewpoint.com.

## Notes

The Notes field on the VA Web Forms form.
Enter any notes for your entry. The notes only display in the VA Web Forms form.
