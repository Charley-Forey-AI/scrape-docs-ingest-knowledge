# Field Definitions: VA User Template Copy Form

The following is a list of field descriptions for the VA User
 Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Destination

Template section on the VA User Template Copy form
Select the destination where you want to create a new subfolder or template:

- User - Select this option to create a new subfolder for a specific user.

- Group - Select this option to create a new subfolder for all users associated with a specific security group.

- New Template - Select this option to create a new template based on an existing template, or an existing user subfolder.

Related information

- [VA User Template Copy Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-user-template-copy-form)

## Module

Name field on the VA User Template Copy form
Enabled if you selected the New Template
 option in the Destination section.
Enter the module that this template relates to. When users add a subfolder to the specified module's root folder on the Menu tab of the Main Menu, this template will be available for selection.

Related information

- [VA User Template Copy Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-user-template-copy-form)

## Name

Name field on the VA User Template Copy form
Enabled if you selected the New Template
 option in the Destination section.
Enter the name of the new template that you are creating based on an existing template or user subfolder.

Related information

- [VA User Template Copy Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-user-template-copy-form)

## Source

Source section on the VA User Template Copy form
Select the source type that the system will use to create a subfolder or template:

- User

- Template

Related information

- [VA User Template Copy Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-user-template-copy-form)

## Template

Template field on the VA User Template Copy form
Enabled if you selected the Template option
 in the Source section.
Enter the template number that you are using to base a new subfolder or template on. Press F4 for a list of valid folder templates.

Related information

- [VA User Template Copy Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-user-template-copy-form)

## User

User field on the VA User Template Copy form
Enabled if you selected the User option in
 the Source section.
Enter the user login that you are using to base a new subfolder or template on. Press F4 for a list of valid user logins.

Related information

- [VA User Template Copy Form](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/get-started-in-the-va-module/va-user-template-copy-form)
