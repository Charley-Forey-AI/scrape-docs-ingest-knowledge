# About Assigning Roles to User Accounts

Use the Roles tab of VA User Profile to assign roles to a user account.
For example, if the user is a job superintendent, you can associate them with the job superintendent role.
Roles are created and maintained using the [HQ Roles](/en/vista/vista/administration/headquarters/role-setup/hq-roles-form) form. You can also associate a user with a role using the Users tab on this form.
Note: You will not be able to delete a user account if it is associated with any roles on this tab. You also won't be able to delete a role from this tab if there are spending limits set up for the user on that role using the [HQ User Spending Overrides](/en/vista/vista/administration/headquarters/role-setup/hq-user-spending-overrides-form) form.
