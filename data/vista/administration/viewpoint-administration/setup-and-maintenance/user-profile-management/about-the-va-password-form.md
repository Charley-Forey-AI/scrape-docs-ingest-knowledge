# About the VA Password Form

Use this form to create a SQL Server login for a new
 Vista user
 account.
Unless a SQL Server login already exists for the
 user, this form opens automatically when you save a new Vista user account using the [VA
 User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) form.

1. Enter and then re-enter a password.
 Note:If the user already exists as a
 SQL user, you can either:

-  Enter the existing SQL
 password to use as the Vista password, or

-  Enter a new Vista
 password.

1. Click OK.
