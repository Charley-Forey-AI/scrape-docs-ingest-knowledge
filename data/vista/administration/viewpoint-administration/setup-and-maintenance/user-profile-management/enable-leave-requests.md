# Enable Leave Requests

If you are using the leave request functionality of the HR module, you must enable leave requests for users in VA User Profile.
To enable leave requests:

1. Open the VA User Profile form.

1. In the HR Co and HR
 Resource fields, assign an HR company and
 resource number, respectively.The system associates the user name with a resource record created in the HR
 Resources form. See [Leave Requests](/en/vista/vista/hr-and-payroll/human-resources/leave/about-leave-requests) for more information.
