# About the VA Copy Security Form

Use this form to copy the security setup from one user to another.
Access this form by selecting the Copy Security button in VA User Profile.
When copying security using this form, the copy process includes module, form, tab, report, attachment, and data security settings. Additionally, the system assigns the security groups from the source account to the destination account.
 When you copy security permissions, the system copies only those permissions from companies to which your user account also has access. Your access is determined by the security permissions given to your user account in VA Form Security. If desired, you can run the VA Copy Security Report to preview which permissions will be added.
Note: The copy process does not affect global/user level access. The copy function focuses on a specific company and the global company is treated as an individual company for this purpose.

For more information about using this form, see [Copy Security Using the VA User Profile Form](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/copy-security-using-the-va-user-profile-form).
