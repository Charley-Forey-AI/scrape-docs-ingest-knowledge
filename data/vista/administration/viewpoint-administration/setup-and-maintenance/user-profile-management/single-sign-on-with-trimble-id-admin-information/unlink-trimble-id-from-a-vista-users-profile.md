# Unlink Trimble ID from a Vista User's Profile

Administrators can reset a user's Vista profile, unlinking it from Trimble ID and updating information.
This is useful for
 troubleshooting Trimble ID account linking
 issues, such as updating a user's email address or user name. The next time the user
 attempts to log in to Vista, they will be prompted to migrate their account to Trimble ID again.

1. From the Vista main menu, go to Viewpoint Administration > Programs > VA User Profile.

1. Open the record for the user you want to unlink
 from Trimble ID in order to edit
 their profile.

1. On the Info tab, deselect the Is SSO Account checkbox.For more information about this field, see [Is SSO Account](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form/field-definitions-va-user-profile-form#ID-iw7154de--en).

1. A popup window asks you to confirm unlinking Trimble ID. Select Yes to continue.

1. Select Save to unlink the account.The user's Vista profile is no longer an SSO account and is no longer linked
 to Trimble ID.

1. If you need to edit any of the user's information, such as their
 Email Address, enter that information in the form.
 Save again.The user's profile is reset to a non-SSO account.

1. Instruct your user to log in again. They must choose Sign in with Trimble ID and
 migrate their account to Trimble ID
 to connect their profiles again.
