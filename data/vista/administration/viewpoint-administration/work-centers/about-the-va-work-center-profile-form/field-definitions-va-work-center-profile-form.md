# Field Definitions: VA Work Center Profile Form

The following is a list of field descriptions for the VA Work
 Center Profile form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Name

Use this field to name the Work Center
 profile. This is the text that shows up in the User Profile drop down menu on the [Work Centers](/en/vista/vista/system-tools/user-interface-guide/about-the-main-menu/main-menu-options/about-the-work-centers-form) form when users are selecting which profile to
 apply.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Profile ID

Enter '+' to create a new Work Center profile. This field will automatically populate with the next available Profile ID number if you click the New Record icon () and then press TAB to move through the field.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Security Group

Use this field to assign a security group to the work center profile. Press F4 to select a security group from a list or enter a security group number.
Security groups are created and maintained using [VA Security Groups](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-groups-form). Click [here](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-groups-form) for more information on security groups.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Navigation

If this profile is for a Dashboard work center (Work Center field is set to
 1), leave this field blank.
If this profile is for a PM, SM, or Accounting work center (Work Center field is set to
 2, 3, or 4,
 respectively), enter the navigation style to use or press F4 to select
 from a list of defined navigation styles. The navigation style selected here determines how
 the navigation menu appears on the selected work center.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

- [About the VA Work Center Tab Navigation Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-tab-navigation-form)

## Sequence

This field identifies each unique item on this tab. By default it will populate with the next available sequence number when creating a new item.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Tab Name

Use this field to name the tab. This is the name that will display on the tab at the top of the main application window.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Tab Number

Use this field to define the order in which the items will appear on the main application window. The item with a tab number of "1" will display farthest to the left next to the Menu tab, followed by the other items in sequential order.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Template Name

If you are creating a Dashboard Work Center
 tab (you entered "1" in the Work Center field), use this field to
 select the format of the Dashboard Work Center. Press F4 to select the format from a
 list.
 If you are creating a PM Work Center tab (you
 entered "2" in the Work Center field), enter PM in this field.
 If you are creating an SM Work Center tab
 (you entered "3" in the Work Center field), enter SM in this
 field.
 If you
 are creating an Accounting Work Center tab (you entered "4" in the Work Center
 field), enter Accounting in this field.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Work Center

Press F4 in this field to select the type of
 Work Center that you want to add to the profile.

- Dashboard - Click [here](/en/vista/vista/system-tools/work-centers/about-the-dashboard-work-center) for an overview on Dashboard Work Centers.

- Project Management - Click [here](/en/vista/vista/system-tools/work-centers/about-the-project-management-work-center) for an overview on PM Work Centers.

- Service Management - Click [here](/en/vista/vista/system-tools/work-centers/about-the-service-management-work-center) for an overview on SM Work Centers.
