# Field Definitions: VA Work Center Tab Navigation Form

The following is a list of field descriptions for the VA Work
 Center Tab Navigation form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Description

Use this field to enter a description of the custom tree menu.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Navigation ID

Enter + or N to create a new tree menu item. The
 system will automatically assign the next available number.
If you clicked the New Record icon (), this field automatically populates with the next available number.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)

## Template

Enter the type of Work Center that you are
 creating a template for, for example enter PM to create a template for a PM Work Center.
 You can also press F4 to select a template from a list.

Related information

- [About the VA Work Center Profile Form](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)
