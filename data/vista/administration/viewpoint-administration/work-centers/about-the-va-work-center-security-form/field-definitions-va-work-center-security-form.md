# Field Definitions: VA Work Center Security Form

The following is a list of field descriptions for the VA Work
 Center Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Group/User

Group/User section on the VA Work Center Security form
 Select an option in this section for setting work center template security.

- Security Group – Select this option to filter by group. Then, in the [ Security Group ](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-security-form/field-definitions-va-work-center-security-form#ID-0004a3dd--en) text box, enter a specific group or leave blank for all groups.

- User – Select this option to filter by user. Then, in the [ User ](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-security-form/field-definitions-va-work-center-security-form#ID-0004a3ee--en) text box, enter a specific user or leave blank for all users.

## Security Group

Security Group field on the VA Work Center Security form
Enter the security group that you want to set
 up security on, or press F4 to select it from a list.
You can also leave this field blank if you want to filter on all security groups.
Note: Only Security Groups that have a Group Type of 2-Reports (set in [VA Security Groups](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-groups-form)) can be assigned Work Center Security.

## Standard

Standard checkbox on the VA Work Center Security form
 Use this checkbox in the grid to filter the templates in the grid. There are three available settings for this checkbox:

- Checked – Displays all standard templates

- Unchecked – Displays all custom templates.

- Mixed – Displays all templates.

User field on the VA Work Center Security form
 This text box is enabled if the User option was selected in the [Group/User](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-security-form/field-definitions-va-work-center-security-form#ID-0004a3ca--en) section.

## Company

Company section on the VA Work Center Security form
 Select an option in this section for filtering by a single or multiple companies.

- Select Company – Select to specify a single company to set up security for. Then enter the specific company in the [Select Company](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-security-form/field-definitions-va-work-center-security-form#ID-0004a3bb--en) text box to the right.

- Across All Companies/Across Companies I have permissions for – Select to set up work center security for all companies for which you have form security permissions.

- If you have permissions for all companies, this
 option displays as Across All companies.

- If you have permissions for only some companies,
 this option displays as Across Companies I have permissions
 for.

## Select Company

Select Company field on the VA Work Center Security form
This text box is enabled if the Select Company
 option was selected in the [Company](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-security-form/field-definitions-va-work-center-security-form#ID-0004a3a3--en) section.
Enter the company for filtering, or press
 F4
 to select from a list of valid companies. If you do not see some companies in the list, you
 may not have been granted form security permissions to administer work center security for
 that company.
