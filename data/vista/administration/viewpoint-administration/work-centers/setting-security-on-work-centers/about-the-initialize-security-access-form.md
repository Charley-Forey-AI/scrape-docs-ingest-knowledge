# About the Initialize Security Access Form

Use this form to set access levels to reports for all users/groups listed in the grid on VA Work Center Security.
Access this form by clicking the Initialize Access Level button on VA Work Center Security.
 Template access levels determine whether users or groups can access a work center template.
 For more information about setting access levels, see Access Level.
