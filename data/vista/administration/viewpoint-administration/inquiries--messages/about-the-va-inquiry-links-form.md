# About the VA Inquiry Links Form

Use this form to link queries together and create an inquiry.
To open this form and create an inquiry, double-click on a row in the Links tab of the [VA Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/va-inquiries-form) form and then enter detailed information about the inquiry relationship.
Each parameter of the query selected in the Related Query field will display as an item on the Parameters tab. Setting up how the parameters of the child query relate to the parent query define how the queries relate.
See [Creating Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/creating-inquiries) for an example that creates two queries and then links them together to create an inquiry.
