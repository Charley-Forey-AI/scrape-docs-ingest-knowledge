# Copying Queries

You cannot edit standard queries, but you can copy one and
 then modify the copy.

1. From the VA Inquiries
 form, open the query that you want to copy.

1. Select File >
 Copy Query. The Copy Query form displays.

1. Enter the name for the
 new query in the New Query Name field.

1. Click Copy. The query is added to the Grid tab and you can modify it
 as necessary. Note: The Copy Query feature will not copy
 any links that have been set up using the Links tab on [VA Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/va-inquiries-form).
