# Field Definitions: VA Messages Form

The following is a list of field descriptions for the VA
 Message form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Body

 This field displays the body text of the message.

##  CC

 This field displays the addresses that received a courtesy copy of this message.

## Read

 Select this checkbox to indicate that you have read this message.

##  Selected

Check this checkbox to indicate that the message is ready for deletion.
For example, open the Grid tab, check this box next to each message that you would like to delete, and then click the Save icon to save your selections. Now select File > Delete Selected Messages and the selected messages are deleted.

##  Subject

 This field displays the subject line of the message.

##  To

 This field displays the address the message was sent to.
