# Viewing Message Attachments in VA Messages

If a message in VA Messages has an attachment, the system enables the Attachments button in the form so you can view those attachments.
To view the attachment:

1. Click the Attachments button. The Message Attachments window displays.
 The window shows the file name of the attachment, description, file location,
 and attachment ID number.

1. Select the attachment
 from the grid and click the Launch button to open the
 application associated with the attachment (e.g., if the attachment is a Word
 document, MS Word opens).
