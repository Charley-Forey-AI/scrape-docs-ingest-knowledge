# Deleting Messages from VA Messages

You can delete messages stored in VA Messages.
These are messages sent by Vista™ - for example, workflow notifications.
To delete a message:

1. Open the VA Messages form from the VA Programs folder or by clicking the pop-up icon from the system tray of your desktop

1.  Select
 the message and click the Delete button or select
 Records > Delete. To delete
 multiple messages:

1. Open the VA Messages form from the VA Programs folder or by clicking the pop-up icon from the system tray of your desktop

1. Select
 the Selected checkbox for each message,
 then select File > Delete Selected
 Messages. The system deletes all selected messages.
