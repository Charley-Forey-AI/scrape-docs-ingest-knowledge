# Field Definitions: VA Inquiry Security Form

The following is a list of field descriptions for the VA
 Inquiry Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Select Company

Select Company field on the VA Inquiry Security form
This text box is enabled if the Select Company
 option was selected in the [Company](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form/field-definitions-va-inquiry-security-form#ID-00049029--en) section.
Enter the company for filtering, or press
 F4
 to select from a list of valid companies. If you do not see some companies in the list, you
 may not have been granted form security permissions to administer inquiry security for that
 company.

Related information

- [Set Security for Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/set-security-for-inquiries)

- [About the VA Inquiry Security Form](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form)

## Company

Company section on the VA Inquiry form.
 Select an option in this section for filtering by a single or multiple companies.

- Select Company – Choose to specify a single company to set up inquiry security for. Then enter the specific company in the [Select Company](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form/field-definitions-va-inquiry-security-form#ID-00049041--en) text box to the right.

- Across All Companies/Across Companies I have permissions for – Select to set up inquiry security for all companies for which you have form security permissions.

- If you have permissions for all companies, this
 option displays as Across All companies.

- If you have permissions for only some companies,
 this option displays as Across Companies I have permissions
 for.

Related information

- [Set Security for Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/set-security-for-inquiries)

## Group/User

Group/User section on the VA Inquiry form
 Select an option in this section for setting attachment security.

- Security Group – Select this option to filter by group. Then, in the [Security Group](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form/field-definitions-va-inquiry-security-form#ID-00049063--en) text box, enter a specific group or leave blank for all groups.

- User – Select this option to filter by user. Then, in the [User](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form/field-definitions-va-inquiry-security-form#ID-00049070--en) text box, enter a specific user or leave blank for all users.

Related information

- [About the VA Inquiry Security Form](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form)

## User

User field on the VA Inquiry form
 This text box is enabled if the User option was selected in the [Group/User](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/about-the-va-inquiry-security-form/field-definitions-va-inquiry-security-form#ID-00049050--en) section.
Enter the user to filter by, or press
 F4
 to select from a list of users, or leave blank to filter for all users.

Related information

- [Set Security for Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/set-security-for-inquiries)

## Security Group

Security Group field on the VA Inquiry form
Enter the security group that you want to set
 up security on, or press F4 to select it from a list.
You can also leave this field blank if you want to set up security on all security groups.

Related information

- [Set Security for Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/set-security-for-inquiries)

## Standard

Standard checkbox on the VA Inquiry form
 Use this checkbox in the grid to filter the queries in the grid. There are three available settings for this checkbox:

- Checked – Displays all standard queries.

- Unchecked – Displays all custom queries.

- Mixed – Displays all queries.
