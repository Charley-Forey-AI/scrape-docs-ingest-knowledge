#  About the VA Messages Form

Use this form to view messages sent by Vista™ - for example, workflow notifications.
Messages will display in this form only if your user profile is set to display Vista messages instead of emails. This is set up using the Notification Preferences tab on the [VA User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) form.
You can open the VA Messages form from the VA Programs folder or by clicking the pop-up icon from the system tray of your desktop (see image below).
