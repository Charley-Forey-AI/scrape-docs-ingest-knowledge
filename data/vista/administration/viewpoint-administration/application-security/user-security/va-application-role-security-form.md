# VA Application Role Security Form

Use this form to protect access to your Vista™ data and security tables.
This form enables you to:

- Prevent unauthorized users from modifying Vista™ data tables via third-party applications such as MS Query or MS Access.

- Prevent unauthorized users from giving themselves security permissions by using third-party applications to modify Vista security tables.

- Change the application role security password.

Note: To access this form, you must have permissions set up via [VA Form Security](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-form-security-form) and have system administrator permissions in SQL Server. Tasks
