# Field Definitions: VA Copy Company Security Form

The following is a list of field descriptions for the VA Copy
 Company Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Sort Group By ID

Sort Group By ID field on the VA Copy Company Security form
 Select this checkbox to sort groups in the list box by their IDs instead of by their names. This setting does not apply when the list box displays Users.

## Copy Security

Copy Security section on the VA Copy Company Security form

- Security Group – Select this option
 to copy form security for one or more program security groups.

- Report Group – Select this option
 to copy report, inquiry, and work center security for one or more report security
 groups.

- Attachment Group – Select this
 option to copy attachment type security for one or more attachment type security
 groups.

- User(s) – Select this option to
 copy security from a single user, multiple users, or all users.

 Once you select an option, specify by selecting the appropriate checkboxes in the list below.

## Delete Existing Security in Destination Company Before Copying

Delete Existing Security in Destination Company Before Copying checkbox on the VA Copy Company Security form
 Select this checkbox to replace the security setup in the destination company. Permissions already defined for the selected users will be deleted and replaced with the new permissions.
 Do not select this checkbox if you do not want the existing security setup to be replaced. Instead, only those permissions not already defined for the selected users in the destination company will be copied. All existing permissions will be left intact.

## Destination Company

Destination Company section on the VA Copy Company Security form

- Company – Select this option to copy security
 settings to a single company. Then specify the company in the text box to the right
 by pressing F4 to select from a list. If you do not see some companies in the
 list, you may not have been granted form security permissions to administer security
 for that company.

- All Companies – Select this option to copy security settings to all companies.

## From Company

From Company field on the VA Copy Company Security form
 Specify the company from which to copy the specified group’s (or user’s) security setup.
Press F4 to select from a list. If you do not
 see some companies in the list, you may not have been granted form security permissions to
 administer security for that company.

Company field on the VA Copy Company Security form
Enabled when the Company option
 is selected in the Destination Company section.
Specify the company by pressing F4 to select
 from a list. If you do not see some companies in the list, you may not have been granted
 form security permissions to administer security for that company.
