# Field Definitions: VA Add Users to Group Form

The following is a list of field descriptions for the VA Add
 Users to Group form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Security Group

Security Group field on the VA Add Users To Groups form
 Enter the security group for adding users to. This field defaults to the active security group from VA Security Groups.
