# VA Add Admin Form

Use this form to create Master Security Administrators.
Access this form by clicking Options > Create Master Administrators in [VA Security Administration](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-administration-form).
A Master Security Administrator can access VA Security Administration to restrict another user's security form access to administering only specific module/company combinations in security forms.
