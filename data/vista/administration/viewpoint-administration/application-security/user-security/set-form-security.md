# Set Form Security

Administrators can use these steps to assign form security
 permissions to users and groups.
Follow the steps below to set up security on forms. For
 more information, follow the links in each step.

1. Filter the grid to
 select the groups or users to which you will apply access levels and
 permissions. See [Filtering the VA Form Security Grid](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings/filtering-the-va-form-security-grid).

1. Apply form security
 settings to the selected users or groups. See [Applying Form Security Settings](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings).  Note: For header/detail forms, you can set security levels
 separately for each section by selecting the Allow
 security for this form to be maintained separately checkbox
 in the form's Form Properties. For more information, see [Form Properties](/en/vista/vista/system-tools/user-interface-guide/system-forms/about-the-form-properties-form).

1. If desired, apply tab
 security settings to the selected users and groups. See [Applying Tab Security Settings](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-tab-security-settings).
