# Record and Attachment Permissions

 In VA Form Security, you can assign permissions for users or groups to work with records and attachments. Here is how group-level permissions interact with user-level record and attachment permissions:

- Where there are record permissions (the Add, Update, and Delete checkboxes in VA Form Security), the user setting always takes precedence over the group setting.

- Where there are attachment permissions (the View Only; Add; Add, Edit; and Add, Edit, Delete options in VA Form Security), the user setting always takes precedence over the group setting. In forms where there are attachments, and both user and group access levels are set to None , the user cannot add, edit, or delete attachments.
