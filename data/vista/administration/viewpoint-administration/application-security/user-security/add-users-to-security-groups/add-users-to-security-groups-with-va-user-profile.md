# Add Users to Security Groups with VA User
 Profile

Once you have created security groups, you can add users to them.
Using [VA User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) to add users to security groups is
 typically done if you are adding users to the system after group security has been
 implemented. Creating users in this form will update that group in VA Security
 Groups.Important: In order to add or delete users from security groups, you must have permissions for all relevant areas relating to security, including:

- Form security access to VA Security Groups form

- Form security access to *all* of the companies where that security group is used

- Access to all other types of security where that security group is used (such as Forms, Reports, Inquiries, Work Center, or Attachment Types).

To add users to
 Security Groups with VA User Profile:

1. In VA User Profile, in
 the User
 Name field, enter the user's user name, or press F4
 for a list of valid users.

1. Select the Security Groups tab. The tab displays all security groups assigned to the user.

1. In the Security
 Group field, enter the group number, or press F4
 to select from a list of available security groups.  You can also delete
 security groups from this user from this tab by selecting a security group
 record and clicking the Delete button on the toolbar.
