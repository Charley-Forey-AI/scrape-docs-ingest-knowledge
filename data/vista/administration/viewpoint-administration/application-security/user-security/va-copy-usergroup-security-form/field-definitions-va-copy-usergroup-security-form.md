# Field Definitions: VA Copy UserGroup Security Form

The following is a list of field descriptions for the VA
 CopyUserGroup Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## From

 Specify where security is being copied from.

## Name/Group Number

When copying security from a user, enter the
 user name. Press F4 to see a list of valid user names.
 When copying security from a group, enter the
 group number. Press F4 to see a list of valid group numbers.

## Names

 Select the user(s) or group(s) to copy security settings to.

## Security Type

 Specify the type of security to copy.

## To

 Specify where security is being copied to.
