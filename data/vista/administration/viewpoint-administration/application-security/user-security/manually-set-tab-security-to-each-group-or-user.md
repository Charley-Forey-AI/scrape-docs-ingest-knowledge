# Manually Set Tab Security to Each Group or User

When setting security for individuals groups or users, use
 the grid in VA Tab Security to assign access levels.
The following instructions detail how to manually
 assign security. For more information on specific fields, see the F1 help. To set security for individual tabs:


1. In VA Form Security,
 filter for the appropriate records. For more information, see [Filtering the VA Form Security Grid](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings/filtering-the-va-form-security-grid).

1. Select a record in the
 grid and select 1-By
 Tab from the Access column drop-down. Note: You can also initialize this access
 level to multiple records in the VA Form Security grid. For more information
 see [Applying Form Security Settings](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings).

1. Click the Apply button. Select the record again and click the Tab
 Security button. The VA Tab Security form displays with the
 company, form, and the specific security group (or user) set from VA Form
 Security.

1. Select the appropriate
 tab record in the grid and select an access level from the Access column drop-down.

1. Repeat step 4 for all
 applicable tabs.

1. Click the Apply button. The system displays a confirmation dialog box.


1. Click Close.

1. Close the form. The
 system returns focus to VA Form Security. Note: If you re-initialize settings for all
 tabs, the system changes any modifications made to individual tabs. For more
 information, see [Initialize Tab Security to All Groups or Users](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/initialize-tab-security-to-all-groups-or-users).
