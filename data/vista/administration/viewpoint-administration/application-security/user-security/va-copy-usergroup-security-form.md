# VA Copy User/Group Security Form

The primary use for this form is to convert current Viewpoint version 5 user security settings to Vista™’s group security settings.
Once you successfully copy security, use of this form is unlikely. However, it can be used to transfer security settings from an individual user or group to another user or group.
