# Field Definitions: VA Security Groups Form

The following is a list of field descriptions for the VA
 Security Groups form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Description

Description field on the VA Security Groups form
 Enter a description of the security group. The description can be up to 256 characters long.

## Group Type

Group Type field on the VA Security Groups form
 Select the group type for the security group. This type limits the group to a specific type of security. If users need a larger range of access, add them to multiple groups.
 Available group types include:

- 0-Data

- 1-Program

- 2-Reports

- 3-Attachment Type

Note: Group types may be changed, as long as no security entries exist for the group in VA Form Security (including tab security), VA Data Security Access, and VA Report Security.

## Name

Name field on the VA Security Groups form
Enter a name for the security group. The name can be up to 30 characters long and is used to identify the report group.

## Security Group

Security Group field on the VA Security Groups form
 Enter a number (up to 7 characters), to
 identify this security group. You can also press F4 to select from a list of existing
 security groups.
Note: The system is installed with Group 0 (Default Security), which cannot be deleted. This group is automatically granted permission to access any new datatype that is set up as secured on the system. All other users and groups must be assigned access.

User Name field on the VA Security Groups form, Users tab
Enter a user to add to the security group or
 press F4 to select a user from a list.
