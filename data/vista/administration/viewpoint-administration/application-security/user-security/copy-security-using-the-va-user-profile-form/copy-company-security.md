# Copy Company Security

You can copy the security setup from one company to another.
This can be very useful if you have multiple companies and you have groups or users that require the same security setup in all companies.
 The copy function copies permissions only for module, form, tab, report security, inquiry, work center, and attachment type security. The process does not copy data level security.
Note: It is recommended that you set security for multiple companies during initial implementation using VA Form Security. Using this form, you can set access for multiple, or all, companies at once.
To copy company security:

1. In VA Copy Company Security, specify the source company in the From Company field. Press F4 to select from a list. If you do not see a company in this list, you may not have been granted form security permissions to administer security for that company.

1. In the Destination
 Company section, select whether to copy security to a single company or all
 companies. When selecting one company, specify the company in the text field to
 the right of the Company radio button. Press
 F4 to select from a list. If you do not see a company in this
 list, you may not have been granted form security permissions to administer
 security for that company.

1. Select whether to delete security settings in the destination company:

- To delete all existing security settings in the destination company, select the Delete existing security in destination company before copying? checkbox.

- To copy only the permissions not already defined for the selected groups or users in the destination company, leave the Delete existing security in destination company before copying? checkbox unselected. This option leaves existing permissions intact.

1. In the Copy Security section, select the type of security to copy.

1. If you have selected one of the group types in the Copy Security section, you can select the Sort Group by ID checkbox to sort the groups in the list.

1. Select one or more items in the list. You can use the All or None buttons to help make your selections.

1. Click the Copy Security button to begin the copy process. A confirmation message displays.

1. Click OK to proceed with the copy. A message displays stating that the process is complete.

1. Repeat the process as necessary for each applicable company.

1. Click Close to close the form.
