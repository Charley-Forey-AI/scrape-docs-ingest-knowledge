# Field Definitions: VA Application Role Security Form

The following is a list of field descriptions for the VA
 Application Role Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Lockdown Security Tables

Checking this checkbox prevents users from
 giving themselves security permissions by using third-party applications to modify Vista™ security
 tables. This checkbox is checked automatically when the Use Application
 Security checkbox is checked.
Uncheck this checkbox to allow users to modify
 Vista security
 tables via third-party applications. To uncheck this checkbox, you must also uncheck the
 Use Application
 Security checkbox.
Click the Update Security button for the selection
 to take effect.
Important: This checkbox is not checked by default when Vista is installed or updated. To enable this functionality, you must check this checkbox manually.
For information on how this checkbox interacts
 with the Use
 Application Security checkbox, see [Use Application
 Security](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-application-role-security-form/field-definitions-va-application-role-security-form#ID-00047f4d--en) and [VA
 Application Role Security](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-application-role-security-form).

## New Password

 Enter a new password for the application
 role. Once you re-enter the new password in the Re-enter Password field and click the
 Update
 Password button, the new password overwrites the previously set password.

## Re-enter Password

 Re-enter the password you entered into the
 New
 Password field. When you click Update Password, this new password replaces the old password.

##  Use Application Security

 Select this checkbox to prevent
 modifications to Vista™ data tables via third-party applications. Checking this
 checkbox automatically checks the Lockdown Security Tables checkbox.
Clear this checkbox to allow Vista to be modified via third-party applications.
Click the Update Security button for the selection
 to take effect.
For information on how this checkbox interacts
 with the Lockdown
 Security Tables checkbox, see [VA
 Application Role Security](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-application-role-security-form) and [Lockdown Security
 Tables](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-application-role-security-form/field-definitions-va-application-role-security-form#ID-00047f3b--en).
