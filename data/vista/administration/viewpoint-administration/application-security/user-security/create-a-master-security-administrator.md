# Create a Master Security Administrator

The VA Add Admin form allows the user to add Master Security Administrators and the Security Administration form allows users to be assigned as Security Administrators and thus, restrict another user's security form access to administering only specific company/module combinations.
A Master Security Administrator is a user who has access to the [VA Security Administration](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-administration-form) and [VA Add Admin](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-add-admin-form) forms.
There are two methods of creating a Master Security Administrator:
Becoming the first Master Security Administrator
To become the first Master Security Administrator:

1. Confirm that you already have been given Global access to all companies in VA Form Security. (If you do not have access, the form's icon will be grayed out in the Viewpoint Administration Programs menu.)

1. In VA Site Settings, select the Enable Security Administration checkbox. Selecting the checkbox makes the first user to do so the first Master Security Administrator and enables the VA Security Administration form.

1. Click OK.

Creating a subsequent Master Security Administrator
To create a Master Security Administrator:

1. Confirm that the user who you want to create as a Master Security Administrator has already been given Global access to all companies in VA Form Security. (For users who do not have access, the form's icon is grayed out in the Viewpoint Administration Programs menu.)

1. Open the VA Security Administration form.

1. In the form menu, select Options and then Add Master Administrators.

1. In the VA Add Admin form, press F4 in the User Name field and select from a list of users.

1. Save the form.
