# Field Definitions: VA Data Security Access Form

The following is a list of field descriptions for the VA Data
 Security Access form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Allow Access

Allow Access checkbox on the VA Data Security Access form
 Select this checkbox to allow an individual group access to a datatype.
Note: If you reinitialize access for all groups, the system overrides this selection.

Related information

- [Assign Security to Secured Datatypes](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/assign-security-to-secured-datatypes)

- [VA Data Security Access Form](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-data-security-access-form)

## Company

Company field on the VA Data Security Access form
 Enter a company for filtering or leave blank
 to filter for all. Press F4 to select from a list of valid
 companies.
If you do not see some company options in this list, you may not have been granted form security permissions to administer security for that company.

Related information

- [Assign Security to Secured Datatypes](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/assign-security-to-secured-datatypes)

- [VA Data Security Access Form](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-data-security-access-form)

## Datatype

Datatype field on the VA Data Security Access form
Enter the datatype for assigning security. A description of the datatype displays to the right of this field.

Related information

- [Assign Security to Secured Datatypes](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/assign-security-to-secured-datatypes)

- [VA Data Security Access Form](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-data-security-access-form)

## Group

Group field on the VA Data Security Access form
 Enter a security group for filtering or leave
 blank to filter for all. Press F4 for a list of valid security groups.

Related information

- [Assign Security to Secured Datatypes](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/assign-security-to-secured-datatypes)

- [VA Data Security Access Form](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-data-security-access-form)

## Initialize Access

Initialize Access section on the VA Data Security Access form
 Select the correct level of access to apply to the security groups in the grid.

- Allowed – Use this option to allow groups access to this datatype.

- Not Allowed – Use this option to block access to this datatype.
