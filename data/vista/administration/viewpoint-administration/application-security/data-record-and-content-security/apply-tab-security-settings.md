# Apply Tab Security Settings

When setting up form security, you have the option to assign security by tab.
 You can assign tab security to groups or users. There are two ways that you can assign tab security:

- Initialize the same security settings to all group or users in the grid, or

- Manually assign tab security to each group or user

Click on the links below for information about setting tab security.
