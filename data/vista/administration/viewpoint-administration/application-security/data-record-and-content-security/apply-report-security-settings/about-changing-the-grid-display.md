# About Changing the Grid Display

To change the display in the grid, check the Grouping
 checkbox.
The grid then displays in a standard table
 format, which allows for filtering by specific values in each column.
If desired, you can then regroup the grid into a
 custom hierarchy by dragging column headers to the gray area along the top of the grid.
 Additionally, you can use a wildcard (*, %) to filter information in the grid. For
 example, if you want to filter for all AP forms, enter *AP in the filter bar under the
 Mod column.
