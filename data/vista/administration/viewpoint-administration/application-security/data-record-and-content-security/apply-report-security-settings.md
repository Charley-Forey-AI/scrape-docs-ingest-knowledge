# Apply Report Security Settings

About applying report security settings.
Once you have [filtered the grid](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-report-security-settings/filtering-the-va-report-security-grid) in VA Report Security, you can assign security to groups or users in two ways:

-  initialize the same security settings to all groups or users in the grid, or

- manually assign security to each group or user

If VA Security Administration is turned on, you may be restricted to applying report security in only some company/module combinations. For more information, see [VA Security Administration Form](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-administration-form).
Important: Report security will be applied for SSRS reports only if the Sync Security checkbox is selected in [RP RS Server](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/rp-rs-server-form).
