# Field Definitions: VA Attachment Type Security Form

The following is a list of field descriptions for the VA
 Attachment Type Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Select Company

Select Company field on the VA Attachment Type Security form
This text box is enabled if the Select Company
 option was selected in the [Company](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form/field-definitions-va-attachment-type-security-form#ID-00047feb--en) section.
Enter the company for filtering, or press
 F4
 to select from a list of valid companies. If you do not see some companies in the list, you
 may not have been granted form security permissions to administer security for that
 company.

## Attachment Type

Attachment Type field on the VA Attachment Type Security form
Enter the attachment type to set security for,
 or press F4 to select it from a list.
Leave this field blank to set up security on all attachment types that have been set up as secured.
Note: You can set up security only on attachment types that
 have been set up to be secured. To see if an attachment type is set up as secured, press
 F4,
 find the attachment type in the lookup, and then see if the attachment type is marked as Y
 in the Secured
 column.
Note: To set up an attachment type as secured, check the
 Secured
 box on the [DM Attachment Types](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/dm-attachment-types-form) form.

## Company

Company section on the VA Attachment Type Security form
 Select an option in this section for filtering by a single or multiple companies.

- Select Company – Use this option to set attachment type security for a specific company. Then enter the specific company in the [Select Company](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form/field-definitions-va-attachment-type-security-form#ID-00048003--en) text box to the right.

- Across All Companies/Across Companies I have permissions for – Select to set up attachment type security for all companies for which you have form security permissions.

- If you have permissions for all companies, this
 option displays as Across All companies.

- If you have permissions for only some companies,
 this option displays as Across Companies I have permissions
 for.

## Group/User

Group/User section on the VA Attachment Type Security form
 Select an option in this section for setting attachment security.

- Security Group – Select this option to filter by group. Then, in the [Security Group](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form/field-definitions-va-attachment-type-security-form#ID-00048024--en) text box, enter a specific group or leave blank for all groups.

- User – Select this option to filter by user. Then, in the [User](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form/field-definitions-va-attachment-type-security-form#ID-00048034--en) text box, enter a specific user or leave blank for all users.

## Security Group

Security Group field on the VA Attachment Type Security form
Enter the security group that you want to set
 up security on, or press F4 to select it from a list.
You can also leave this field blank if you want to set up security on all security groups.
Note: You can enter only security groups that are set up as
 Attachment Type groups. This is defined using the Group Type drop down on the [VA Security Groups ](/en/vista/vista/administration/viewpoint-administration/application-security/user-security/va-security-groups-form) form.

## User

User field on the VA Attachment Type Security form
 This text box is enabled if the User option was selected in the [Group/User](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form/field-definitions-va-attachment-type-security-form#ID-00048011--en) section.
Enter the user to filter by, or press
 F4
 to select from a list of users, or leave blank to filter for all users.
