# Manually Assign Security to Each Group or User

When setting security for individuals groups or users, use
 the grid to assign access levels and record permissions.
The following instructions detail how to manually
 assign security. For more information on specific fields, see the F1 help.

1. In the VA Form Security
 form, filter for the appropriate records. For more information, see [Filtering the VA Form Security Grid](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings/filtering-the-va-form-security-grid).

1. In the grid, expand the
 hierarchical view so that you can see all of the items in the grid.

1. For each record in the grid
 for which you want to assign security, select an access level from under the
 Access column drop-down list.Note: The system allows you to select the access
 level by Tab page only for forms with multiple tabs. For more information on
 setting Tab security, see [Apply Tab Security Settings](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-tab-security-settings).

1. Select the appropriate
 record permission checkboxes in the Add, Update,
 and Delete columns, as necessary. When you select a box, the user is able
 to perform that action on the specified form.

1. Select an attachment
 permission level from the Attachments column drop-down, as
 necessary.

1. Click the Apply
 button. The system displays a message that the security was successful. The system
 updates the group (or user) security.

1. If desired, click the
 Refresh
 Grid button again to clear the grid.Note: If you later initialize settings for all
 users, any modifications made to individual users will be changed to match that of
 all users. For more information, see [Initialize Security Settings to All Groups or Users in the Grid](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings/initialize-security-settings-to-all-groups-or-users-in-the-grid).
Note: If you deny access to a form, its icon is
 disabled in the associated module’s Programs folder. To hide or view inaccessible
 forms, users can select View > Display
 Accessible Items Only from the Main Menu toolbar.
