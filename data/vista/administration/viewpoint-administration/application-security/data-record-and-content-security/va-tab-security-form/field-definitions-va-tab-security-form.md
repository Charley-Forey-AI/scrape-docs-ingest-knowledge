# Field Definitions: VA Tab Security Form

The following is a list of field descriptions for the VA Tab
 Security form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Company

The Company field on the VA Tab Security form.
Do not change the company selected in this field.
The company for which you are setting up security is pulled from the VA Form
 Security form. Changing the company selected in this field will only create an
 error when you try to apply security using the Apply button.

##  Form

The Form field on the VA Tab Security form.
 Do not change the form selected in this field.
The form for which you are setting up security is pulled from the VA Form
 Security form. Changing the form selected in this field will only create an
 error when you try to apply security using the Applybutton.

## User Name

The User Name field on the VA Tab Security form.
Do not change the user name selected in this field.
The user name for which you are setting up security is pulled from the VA Form
 Security form. Changing the user name selected in this field will only create an
 error when you try to apply security using the Apply button.

##  Select Security for all Records in the Grid

The Select Security for all Records in the Grid checkbox on the VA Tab Security form.
If you have NOT designated security By Tab for this module/form/user in VA Form Security, you must do so before you can continue in this form.

- Full – User has full access to this tab on selected form

- Read Only – User can view details on tab, but cannot add, edit, or delete records.

- None – User does not have access to this tab on the selected form.

Note: Click the Initialize button to have the access level updated for all items in the grid.
