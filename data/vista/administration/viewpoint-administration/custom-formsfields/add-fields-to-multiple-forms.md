# Add Fields to Multiple Forms

When adding a custom field to a form that has a corresponding form (e.g., JC Contracts/PM Contracts) or related form (e.g., PM Subcontract Detail/SL Entry, SL Entry Items) elsewhere in the system, you will have the option to add that field to the additional forms.
Once you define all other parameters for the custom field, the wizard displays a checklist of all corresponding/related forms. Select the checkbox for each form to add the field. From the drop-down list, specify the tab location for the field. All checked forms, as well as the original form, now display the custom field.
