# Format a Notes Field

You can use the VA Custom Fields Wizard to add a notes field
 that includes text formatting (font face, font size, etc), list formatting, and paragraph
 formatting to a standard form in the application.
For example, you can use this feature to add formatted help text to the forms in the application.
 The toolbar below will display above the notes field.

Follow the steps below to add a formatted notes field to a form in the application.

1. Open the VA Custom Fields Wizard form.

1. Select Add Custom
 Field and then click the Nextbutton.

1. Complete the first few fields in the form.

1. Select Text in the
 What
 is the general type of input? field.

1. Check the Use a
 predefined data type box.

1. Press F4
 in the enabled field and select bFormattedNotes from the list
 that displays.

1. Click the Next button.

1. Make sure that
 FormattedNotes is still selected in the Control to
 use field.

1. Complete the remaining fields in the wizard.

1. Open the form that you added the formatted notes field to.

1. Position and size the formatted notes field. For more details, see [Change the Position of a Field on the Form](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/change-the-position-of-a-field-on-the-form).
