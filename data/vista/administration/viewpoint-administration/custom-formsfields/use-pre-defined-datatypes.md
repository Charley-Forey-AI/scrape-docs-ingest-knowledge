# Use Pre-Defined Datatypes

Depending on the type of custom field, you may have the
 option to use a pre-defined datatype.
When using a pre-defined datatype, the
 system formats the custom field the same as the datatype. For example, if you create a
 field called ‘Units’ and assign the ‘bUnits’ datatype, the custom field uses the
 following formatting:
Input Type:
Numeric

Input Mask:
###,###,##0.000

Input Length:
15

Precision:
3 (Numeric

Note: If you plan to implement data level security for the
 custom field, associate it with a securable datatype. See “Setting Data Level Security
 for a Custom Field” in Related Topics below.
