# Field Definitions: VA Custom Form Tabs

The following is a list of field descriptions for the VA
 Custom Form Tabs form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Control Type

Control Type field on the VA Custom Form Tabs form
 Select the type of information that you want to add to the tab.

- 0-Default - Select this option if
 you want to add a user-defined table to the tab. This enables the Grid Form
 field.

- 3- Query - Select this option if
 you want to add a query to the tab. This enables the Query Name field so that you can
 select a query.

## Form

Form field on the VA Custom Form Tabs form
 Enter the parent form to associate the
 user-defined table with. Press F4 to select from a list of valid forms. You can use the
 F4
 lookup for this field to view all forms or just forms with associated custom tabs.

## Grid Form

Grid Form field on the VA Custom Form Tabs form
 Enter the name of the user-defined table or
 press F4 to select from a list of valid tables.
This field is enabled when 0-Default is
 selected in the Control Type drop down.

GridKeySeq field on the VA Custom Form Tabs form, Grid Links tab

## Query Name

Query Name field on the VA Custom Form Tabs form
This field is enabled only when 3-Query is selected in the Control Type field.
Use this field to associate the tab with a query. Press F4 to select a query from a list. Only queries that are linked to the form query of the parent form display in the list. You can do this using the Links tab on the [VA Inquiries ](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/va-inquiries-form) form. Click [here](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/adding-a-query-to-a-tab) for information on how to add a query to a tab on a form.

## Tab#

Tab# field on the VA Custom Form Tabs form
 Enter the custom related tab’s number. Press
 F4
 to select from a list of valid custom related tabs for the parent form.
Note: You must create the custom related tab prior to linking it with the user-defined table. If this field is blank, then a custom related tab has not been created and you cannot link the user-defined table using this form. For more information, refer to Linking Related Tabs to UD Tables in Related Topics below.

## Title

Title field on the VA Custom Form Tabs form
Use this field to change the name of the tab on the form. The form name can be up to 30 characters long.
