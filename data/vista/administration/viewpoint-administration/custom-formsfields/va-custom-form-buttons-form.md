# VA Custom Form Buttons Form

Use this form to create custom form buttons that can open other forms, reports, and external applications, or initiate stored procedures.
Additionally, you can specify parameters to pass on to the item you are accessing with the button. You can place custom buttons on all standard forms, as well as custom forms that you create using the UD module.
For more information about using this form, see [Create Custom Form Buttons](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/create-custom-form-buttons).
