# VA Custom Field Combo Boxes Form

Use this form to create combo box (drop-down list) types for custom fields.
Using a combo box is recommended when you want users to select from a predefined list of available inputs. When using the Custom Fields Wizard to create a field with either a ‘Text’ or ‘Numeric’ type of input, the combo box control is available. When using this control, all custom defined combo box types are available for selection.
 Use the Items tab to specify the number of selections that are available for the combo box you are creating. From here, specify the display sequence of the items, as well as the Database and Display value for each item.
