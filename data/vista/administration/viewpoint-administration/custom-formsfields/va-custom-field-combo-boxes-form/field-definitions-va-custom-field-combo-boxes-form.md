# Field Definitions: VA Custom Field Combo Boxes Form

The following is a list of field descriptions for the VA
 Custom Field Combo Boxes form.

## Combo Box Type

Combo Box Type field on the VA Custom Field Combo Boxes form
Enter a unique code to identify the combo box type. This is an alphanumeric field with a maximum of 20 characters.
The type entered here, as well as the description, is available for selection when using the Custom Fields Wizard and “3-Combo box” has been selected as the control to use.

Related information

- [VA Custom Field Combo Boxes Form](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/va-custom-field-combo-boxes-form)

## DatabaseValue

DatabaseValue field on the VA Custom Field Combo Boxes form, Items tab
 Enter the value that displays in the database for this combo box item.
 This value is also used by custom reports to display, sort, summarize, or restrict data.

Related information

- [VA Custom Field Combo Boxes Form](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/va-custom-field-combo-boxes-form)

## Description

Description field on the VA Custom Field Combo Boxes form
 Enter a description for the combo box type. This is an alphanumeric field with a maximum of 30 characters.
 The description entered here, as well as the type, is available for selection when using the Custom Fields Wizard and “3-Combo box” has been selected as the control to use.

Related information

- [VA Custom Field Combo Boxes Form](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/va-custom-field-combo-boxes-form)

## DisplayValue

DisplayValue field on the VA Custom Field Combo Boxes form, Items tab
 Enter the value that displays in the combo box field. This is the description of the combo box item.

Related information

- [VA Custom Field Combo Boxes Form](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/va-custom-field-combo-boxes-form)

## Seq

Seq field on the VA Custom Field Combo Boxes form, Items tab
 Enter the sequence number for this item; this number determines the order in which items display in the combo box.

Related information

- [VA Custom Field Combo Boxes Form](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/va-custom-field-combo-boxes-form)
