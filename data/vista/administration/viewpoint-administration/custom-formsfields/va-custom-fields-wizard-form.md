# VA Custom Fields Wizard Form

This wizard guides you through adding, editing, and removing custom (user-defined) fields from forms in the application.
Generally, custom fields are used for recording additional information related to the form. Custom fields are usually set up after you have been using the system for a while because it takes some understanding of the module and a familiarity with the forms before you can determine the need for additional fields.
