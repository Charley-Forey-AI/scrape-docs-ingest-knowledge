#  All Invoice

The All Invoice checkbox on the HQ Compliance Groups form, Compliance Group Codes tab.
 Display only, indicating whether this compliance code will be verified for all non-PO/SL invoices (checked) or only purchase order and subcontract invoices (unchecked).
