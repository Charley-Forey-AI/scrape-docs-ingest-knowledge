# About the HQ Standard Note Form

Use this form to set up standard and/or frequently used information, such as billing procedures, shipping procedures, special requirements, or company policies.
Once set up, these standard notes can be pulled into header or item notes in
 various forms throughout the system (e.g. subcontracts, purchase orders, change
 orders, and work orders).
