# About the HQ Insurance Codes Form

Use this form to set up the insurance codes you will use when processing payroll.
(Master File)
 The codes will be used in conjunction with a state code (determined when payroll is processed) and based on the job state, SM work order state, or resident state.
Once set up here, you can then use PR State Insurance Codes to define rates for workman’s comp and/or general liability insurance by state and insurance code. You can also set them up on templates in JC Insurance Template to define worker’s comp insurance by phase.
