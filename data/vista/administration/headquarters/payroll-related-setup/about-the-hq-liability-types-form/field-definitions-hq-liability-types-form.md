# Field Definitions: HQ Liability Types Form

The following is a list of field descriptions for the HQ
 Liability Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Liability Type

Enter a number (0-9999) that identifies this liability type. Liability types are used to group liabilities together for interfacing from Payroll to General Ledger and Job Cost.
Example:
Type
Description

1
Taxes

2
Benefits

3
Unempl Insurance

4
Worker's Comp

##  Description

 Enter a description of this liability type, up to 30 characters.
