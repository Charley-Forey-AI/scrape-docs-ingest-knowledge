# Set up superannuation fund clearing houses

You can use the HQ Super Clearing House form to set up clearing houses that will hold the superannuation funds for your employees.
The system displays the HQ Super Clearing House form
 only if your default country is set to Australia (
 Default
 Country
 field, HQ Company Setup).

1.  Enter an identification code in the Clearing House field, up to 20 characters.

1. In the Name field, enter the name of the clearing house.

1. In the Clearing House Format field, select the format that is supported by this clearing house for generated Super Contribution .csv files.

1. In the Contact and Phone fields, enter the name and the telephone phone number of the contact person at the clearing house.

1. In the Address section, enter the address of the clearing house.

1. Confirm that there are no fund records associated with the clearing house in [PR Super Clearing House Funds](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/superannuation/about-the-pr-super-clearing-house-funds-form). Note: To delete clearing houses, select the clearing house in the Clearing House field and click the Delete icon.

1. Select the clearing house in the Clearing House field, and click the Delete icon.

Once you finish, you can then use the PR Super Clearing House Fund form to associate the clearing houses with superannuation funds that have been set up in HQ Super Funds.
