# Field Definitions: HQ Super Cleaning House Form

The following is a list of field descriptions for the HQ
 Super Cleaning House form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Clearing House

Required.
Enter a code, up to 20 characters, that uniquely
 identifies this clearing house, or press F4 for a list of existing clearing
 house.

## Name

Enter a the actual name or a description of the clearing house.

## Clearing House Format

Required.
Select the format that is supported by this clearing house for generated Super Contribution .csv files.

## Contact / Phone

In the Contact and Phone fields,
 enter the name and the telephone phone number of the contact person at the clearing
 house.

## Address

Enter the address of the clearing house.

- Address

- City

- State

- Postal Code

- Country

- Add'l Address
