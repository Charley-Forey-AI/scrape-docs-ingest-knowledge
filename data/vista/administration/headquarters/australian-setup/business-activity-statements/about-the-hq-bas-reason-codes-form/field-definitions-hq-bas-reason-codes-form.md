# Field Definitions: HQ BAS Reason Codes Form

The following is a list of field descriptions for the HQ BAS
 Reason Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Reason Code

Enter the identifying number for this reason
 code. Refer to [www.ato.gov.au](http://www.ato.gov.au/) to determine
 what codes you will need to set up in this form.

## Reason

Enter a description for this reason code, up to 60 characters.
