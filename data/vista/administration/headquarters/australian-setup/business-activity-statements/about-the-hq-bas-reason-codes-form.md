# About the HQ BAS Reason Codes Form

Use this form to set up reason codes for Business Activity Statement (BAS) reporting.
You will use the reason codes set up in this form when specifying reason codes for varying PAYG or FBT instalment amounts in [HQ BAS Process](/en/vista/vista/administration/headquarters/australian-setup/business-activity-statements/about-the-hq-bas-process-form).
Note:This form only displays for Australian companies (as set up in HQ Company Setup).

To set up a BAS reason code, enter the number in the
 Reason
 Code field. In the Reason field, enter the description
 of the code. Refer to [www.ato.gov.au](http://www.ato.gov.au/) to
 determine what codes you will need to set up in this form.
