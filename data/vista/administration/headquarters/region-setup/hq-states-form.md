# HQ States Form

Use the HQ States form to set up and maintain state or region codes (for example, provinces, territories, etc.).
(Master File)
The Vista software comes with a pre-defined master file of the standard U.S. Postal Service abbreviations for all 50 U.S. states. You will need to set up any other regions you plan to use.

## Reciprocal States

Use this tab to set up reciprocal states; that is, states that have reciprocal agreements for inter-state employment. Reciprocal agreements allow employees who live in one state and work in another, to pay state withholding tax only for the state in which they reside.
The system uses reciprocal states information to determine the tax state for withholding when posting in Payroll's Time Card Entry or Automatic Earnings forms. If the selected employee resides in a state other than the job state, the system checks the Headquarters State Reciprocal (HQSR) file to see if the job state has a reciprocal agreement with the resident state. If so, it designates the resident state as the tax state for withholding.

## Compliance

Use this tab to set up the state compliance requirements used in conjunction with defined timecard validation rules to detect discrepancies and errors when entering timecards. You can also double-click in the Compliance grid to access the HQ State Compliance form. For more information about state compliance requirements, see [HQ State Compliance Form](/en/vista/vista/administration/headquarters/company-setup/hq-state-compliance-form).
