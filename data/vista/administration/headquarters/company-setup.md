# Company Setup

Prior to setting up company information in other modules, you must set up the company first in the HQ module
There are a number of items you can set up in HQ, but some of the primary functions include setting up your first company, setting up additional companies, setting company logos, creating/assigning groups, setting up contacts, creating reason codes, and setting up units of measure.
