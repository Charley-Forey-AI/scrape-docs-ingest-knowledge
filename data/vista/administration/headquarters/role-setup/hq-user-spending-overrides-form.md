# HQ User Spending Overrides Form

Use the HQ User Spending Overrides form to set spending limits and thresholds on specific
 users who are assigned to a role.
Access this form by clicking the User Spending
 Overrides button on the Users tab of the HQ Roles form.
 This form only applies to the WF module's Process Workflow feature.
The spending limits and thresholds set up here override the limits and thresholds defined for the role on the Spending Limits tab of the [HQ Roles](/en/vista/vista/administration/headquarters/role-setup/hq-roles-form) form.
For example, if you have several project managers but some of them should have different spending limits for purchase orders and/or subcontracts, you can set up a generic spending limit for the project manager role and then use this form to set up specific spending limits for applicable users.
