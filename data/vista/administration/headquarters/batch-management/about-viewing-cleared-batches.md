# About Viewing Cleared Batches

All batches throughout the system are tracked in HQ Batch Control.
The information on this form is set up when you
 create a batch in any of the posting programs (e.g. AP Transaction Entry, GL Journal
 Transaction Entry, PR Timecards, etc.).
Batches cleared in any of the batch processing forms
 (via Clear Batch, File menu) will show with a status of 6-Cancelled
 in this form. If you need to need to see who cleared the batch, use the VA Other Event
 Statistics report.
