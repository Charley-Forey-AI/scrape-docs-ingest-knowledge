# About the HQ Batch Control Form

All batches throughout the system are tracked in HQ Batch Control.
The information on this form is set up when you create a batch in any of the posting programs (e.g., AP Transaction Entry, GL Journal Transaction Entry, PR Timecards, etc.). You can edit the Restricted Access, Status, and In Use/Posted By fields under certain conditions. However, the remaining information cannot be changed and will be grayed out.
Note:You may want to consider restricting access to this form so that only the System Administrator has access to it.
