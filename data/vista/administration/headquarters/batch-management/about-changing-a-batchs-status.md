# About Changing a Batch's Status

All batches throughout the system are tracked in HQ Batch Control.
The information on this form is set up when you
 create a batch in any of the posting programs (e.g., AP Transaction Entry, GL Journal
 Transaction Entry, PR Timecards, etc.). However, you can edit the Status under certain
 circumstances.
As a batch goes through its various levels of
 processing, the system will assign a value in the Status field of the HQ Batch Control
 form. This value identifies where the batch is in the processing cycle. The different
 statuses assigned to a batch are described in [Status](/en/vista/vista/administration/headquarters/batch-management/about-the-hq-batch-control-form/field-definitions-hq-batch-control-form#ID-0000ea8d--en).
You can manually change a batch’s status only if the status is less than 4, in which case you can change it only to 0 (open).
