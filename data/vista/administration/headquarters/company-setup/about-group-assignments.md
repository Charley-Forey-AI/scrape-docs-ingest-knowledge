# About Group Assignments

Group assignments are defined on the Add'l Info tab in HQ
 Company Setup and are used to specify which code files the company will use.

Groups are set up in HQ Groups and are used for code files that are not master files; that is, they are either specific to a single company or shared by multiple companies (unlike master code files,
 which are used by all companies). These code files include AR Customers, AP Vendors, PM
 Firms, HQ Materials, HQ Tax Codes, JC Phases, JC Cost Types, EM Cost Codes, EM Revenue
 Codes, EM Shops, and HQ Contacts.
If you want your code files to be specific to a company, you will assign different groups to each company. For example, Co #1 would use Group #1, Co #2 would use Group #2, and so on. When you add, delete, or change codes in these files, you must be in the associated company. If you want multiple companies to share code files, you will assign the same groups to each company that will share the file. For shared code files, you can add, change, or delete codes from within any of the companies assigned the same group.
