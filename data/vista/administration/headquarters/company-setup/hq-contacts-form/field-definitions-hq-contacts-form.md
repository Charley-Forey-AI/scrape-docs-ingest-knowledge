# Field Definitions: HQ Contacts Form

The following is a list of field descriptions for the HQ
 Contacts form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Contact Seq

Enter the contact sequence number for this
 contact, or enter N , New , or + to
 have the system automatically assign the next sequence number.

## Courtesy Title

Enter a courtesy title (e.g. Mr., Ms, Jr., etc.) for this contact, if applicable. Up to 10 characters allowed.

## First Name

Enter the contact's first name, up to 30 characters.

## Middle Initial

Enter the contact's middle initial, if applicable.

## Last Name

Enter the contact's last name, up to 30 characters.

## Title

Enter the contact's title, if applicable (e.g. President, Controller, Project Manager, etc.).

## Organization

Enter the company to which this contact is associated. Up to 30 characters allowed.
Note:The organization specified here can be any company already set up in the system (as a vendor, customer, or firm) or any company not set up in the system.

## Phone

Enter the contact's phone number. Up to 20 characters allowed.

## Ext

Enter the extension number for this contact. Up to 5 characters allowed.

## Cell

Enter the contact's cell phone number, if applicable. Up to 20 characters allowed.

## Fax

Enter the contact's fax number, if applicable. Up to 20 characters allowed.

## Address

Enter the contact's address, up to 60 characters.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## Add'l Address

Use this field to enter additional address information for this contact (up to 60 characters). For example, if the contact receives mail at a P.O. Box for this address, then you might enter the P.O. Box in the Address field above, and use this field to enter the street address.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## City

Enter the city, up to 60 characters.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## State

Enter a valid state (as defined in HQ States).
 The system validates the state based on the Default Country specified in HQ Company
 Setup for the active company. If not valid, an error displays; however, the entry will be
 allowed. You must then enter a valid country for this state in the Country field below.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## Zip Code

Enter the zip code for this address, up to 12 characters.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## Country

Enter the 2-character country code. Entry in this field is only required when the address exists outside the Default Country specified in HQ Company Setup for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## Email Address

The Email Address field on the HQ Contacts form.
Enter the email address for this contact. You can
 enter multiple email addresses for this contact; however, you must separate them with a
 semi-colon. For example, jane.doe@email.com; johnd@email.com;
 joe.smith@email.com.
Note: Clicking the Send button () automatically opens Microsoft Outlook and sets up an outgoing message for you using the email address specified in this field.
