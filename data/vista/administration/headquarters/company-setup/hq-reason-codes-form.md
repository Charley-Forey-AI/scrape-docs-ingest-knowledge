# HQ Reason Codes Form

Use this form to create and maintain reason codes.
Reason codes allow you to identify why a transaction was entered, changed, or deleted. For example, you can create a reason code that represents an explanation for issuing credit on an invoice. Other codes can be used to identify corrections or additions to a transaction.
