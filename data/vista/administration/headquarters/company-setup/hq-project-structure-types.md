# HQ Project Structure Types

Use the HQ Project Structure Types form to create project structure types that you can use to classify project data for Analytics reporting.

If you use Trimble Analytics™, the project structure types defined here allow you to identify and group project data (for example, estimates, actuals, change orders, etc.) in a more accurate and meaningful manner for Analytics reporting.
Vista™ provides a list of standard project structure types for your use. However, if you prefer, you can set up user-defined project types and link them to the standard project types.
Once defined here, you can assign project structure types to applicable jobs in the [JC Jobs Form](/en/vista/vista/costs-and-contracts/job-cost/jobs-and-contracts-setup/jc-jobs-form).
Note: If you do not use Trimble Analytics, you can still assign project types to jobs for informational purposes.

For more information about Trimble Analytics, see [About Viewpoint
 Analytics.](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:Analytics_000001:Analytics:Analytics)
