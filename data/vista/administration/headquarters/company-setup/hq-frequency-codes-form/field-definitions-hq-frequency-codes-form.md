# Field Definitions: HQ Frequency Codes Form

The following is a list of field descriptions for the HQ
 Freqeuency Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Frequency Code

 Enter a frequency code, up to 10 characters. This code will be used to group certain recurring types of entries, such as GL Auto Journal Entries, AP Recurring Invoices, and PR Auto Earnings.

##  Description

 Enter a description of this frequency code, up to 30 characters, that indicates when this code is activated (e.g. Always, Monthly, 1st Week, 2nd Week, etc.).
