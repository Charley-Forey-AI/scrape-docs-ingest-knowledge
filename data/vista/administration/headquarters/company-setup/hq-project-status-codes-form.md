# HQ Project Status Codes Form

Use this form to set up the status codes that your company uses to track the various 'states' of potential projects (set up in PC Potential Project).
The following is a list of suggested project
 status codes:

- Active

- Awarded

- Active Construction

- Closed

- Completed

- In Negotiation

- Not Awarded
