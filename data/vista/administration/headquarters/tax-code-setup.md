# Tax Code Setup

You can set up and maintain single and multi-level tax codes for the system using the HQ Tax Codes form.
Single-level tax codes are useful when only one tax rate is applicable. If you require multiple tax rates, you can use a multi-level code and associate it with all applicable single-level codes.
