# About the HQ Material Categories Form

Use this form to define material categories.
Categories typically represent types of materials that you want sorted/grouped together on reports (e.g. concrete, plumbing, general, etc.).
If a category will contain materials not set up in Inventory, you will need to assign a Non-Stocked GL Account. This account will be debited when non-stocked materials in this category are included on an expense invoice in AP, and credited if non-stocked materials are posted to jobs or equipment in JC and/or EM.
Note:When you create a new material code, it is associated with the Material Codes group assigned to the current active company. All information set up here only applies to the current active company and any companies that share that Material Codes group.
