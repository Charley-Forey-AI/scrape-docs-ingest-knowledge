# WF Notifier Queries Report

You can use the WF Notifier Queries Report report to display the details from the
 Notifier Query Maintenance form by selecting Workflow > Reports > WF Notifier Queries Report.
This report will display the details from the Notifier Query
 Maintenance Form. Query parameters will be displayed if they exist. The Email Fields
 that are displayed in the form but not accessible will also display in the report.
