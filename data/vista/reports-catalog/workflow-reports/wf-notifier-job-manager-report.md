# WF Notifier Job Manager Report

You can use the WF Notifier Job Manager Report by selecting Workflow > Reports > WF Notifier Job Manager Report.
This report will print out the detail set up in Notifier Job Manager program.
Report Parameters
Description
Beginning Job
Click the Field Lookup
 button or press F4 to select the beginning
 workflow notifier job or leave blank for all.

Ending Job
Click the Field Lookup
 button or press F4 to select the ending
 workflow notifier job or leave blank for all.

Print Notes?
checkbox to print notes.
