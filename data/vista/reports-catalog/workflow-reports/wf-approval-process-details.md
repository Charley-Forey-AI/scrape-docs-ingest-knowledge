# WF Approval Process Details

You can use the WF Approval Process Details report to see
 the setup of details of a process and where that process is being used within Viewpoint by
 selecting Workflow > Reports > WF Approval Process Details.
This report allows the user to see the setup of details of a process and where that process is being used within Viewpoint. It will show a list of members of the process and whether they are attached to the process directly or via a role. Below the member list the report will show the company, module, and or data type (Job, EM Department, or IN Location) the process is currently used in.
Display\Parameter Restrictions:
Please note that if the report is filtered on a Role the section that shows members associated with a process directly (not through a role) will not be displayed. Also, since this is a WF report and WF is not bound by company, the Company report parameter is blank by default.
