# SL Textura Subcontract Export

You can use the SL Textura Subcontract Export report to
 obtain unformatted data for the Textura Application by selecting Subcontract Ledger > Reports > SL Textura Subcontract Export.
This report is used to obtain unformatted data for the Textura Application. Data comes from Subcontractor Header and Item data, as well as related Change Order data. Each subcontract change order is unique by the Sub CO#. Contract No (Viewpoint's 30 character Subcontract number) has been truncated to twenty characters. Also, the change order description, which uses the Notes field from SL Change Orders, is truncated to 145 characters. Report should be exported using the Text (TXT) options. Characters per inch = 20, and number of lines per page = 0.
