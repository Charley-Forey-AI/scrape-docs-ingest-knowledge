# SL Payment to AP Update Preview

You can use the SL Payment to AP Update Preview report by
 selecting Subcontract Ledger > Reports > SL Payment to AP Update
 Preview.
The SL Payment to AP Update Preview is NOT an audit list, but is helpful in checking the accuracy of the SL Worksheet data that will be transferred to Accounts Payable as invoices. The report may be run from either the SL Worksheet program under the File menu or the Viewpoint Menu. There is an option to have a new page for each subcontract.
