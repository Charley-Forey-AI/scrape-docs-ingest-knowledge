# SL Textura Subcontract Compliance Export

You can use the SL Textura Subcontract Compliance Export
 report by selecting Subcontract Ledger > Reports > SL Textura Subcontract Compliance
 Export.
This report is used to export subcontract compliance CSV formatted data for Textura. Data comes from SL Subcontract Entry and SL Compliance Detail. The report accepts Company, Beginning Job and Ending Job parameters. Report should be exported using the Text (TXT) options. Characters per inch = 20, and number of lines per page = 0. Contract No (Viewpoint's 30 character Subcontract number) has been truncated to twenty characters.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.
