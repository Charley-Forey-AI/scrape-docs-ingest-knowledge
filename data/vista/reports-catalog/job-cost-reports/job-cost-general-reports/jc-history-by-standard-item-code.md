# JC History by Standard Item Code

You can use the JC History by Standard Item Code report to
 list purged Contract information sorted by Contract, Std Item Code or and Unit of Measure in
 the JCHC table by selecting Job Cost > Reports > JC History by Standard Item
 Code.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract
Select the Field Lookup button or press F4 to select the beginning contract.

Ending Contract
Select the Field Lookup button or press F4 to select the ending contract.

Sort by Std (I)tem Code or (R)egion
