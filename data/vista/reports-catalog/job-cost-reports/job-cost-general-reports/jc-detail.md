# JC Detail

You can use the JC Detail report for a list of details for
 Jobs, Phases, and Cost Types from the JCCD table by selecting Job Cost > Reports > JC Detail.
This report lists details for Jobs, Phases, and Cost Types from the JCCD table. Different information is printed depending upon the Transaction Type and Source. The report includes only interfaced and posted details.
