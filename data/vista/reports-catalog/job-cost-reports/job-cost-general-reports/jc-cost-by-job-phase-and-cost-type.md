# JC Cost by Job Phase and Cost Type

You can use the JC Cost by Job Phase and Cost Type report
 by selecting Job Cost > Reports > JC Cost by Job Phase and Cost
 Type.
The JC Cost By Job, Phase and Cost Type provides actual hours, units, and
 dollars in a cross tab format by listing cost types (7 cost types per page. as
 columns and job/phases in rows. The Job and Phase descriptions will repeat on
 subsequent pages when eight or more cost types print. Input ranges (Beginning and
 Ending) are available for job, phase, cost type, and month.
