# JC Job Phases

You can use the JC Job Phases report for a list of Phases
 on Jobs by selecting Job Cost > Reports > JC Job Phases.
This report provides a list of Phases on Jobs. Inputs for range of Jobs and
 Phases.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Beginning Phase
Select the Field Lookup button or press F4 to select the beginning phase.

Ending Phase
Select the Field Lookup button or press F4 to select the ending phase.

Print Phase Notes?
checkbox to print phase notes.

Include Cost Type Details?
checkbox to include cost type details.

Print Cost Type Detail Notes?
checkbox to print cost type detail notes.

New Page Per Job?
Select checkbox to have page break per job.

Include Inactive Phases?
checkbox to include inactive phases.
