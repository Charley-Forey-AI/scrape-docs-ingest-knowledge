# JC Projections Detail - Cost

You can use the JC Projections Detail - Cost report to
 print one line per Resource Transaction showing the Budget Code, Date Range, and Resource
 Detail. Resource Detail will print either PR or EM info from the JCPR (JC Projection
 Resource Detail) by selecting Job Cost > Reports > JC Projections Detail - Cost.
This report prints one line per Resource Transaction showing the Budget Code,
 Date Range, and Resource Detail. Resource Detail will print either PR or EM info
 from the JCPR (JC Projection Resource Detail). Report will print projection data
 even when resource detail does not exist. Resource detail column headings will be
 suppressed when that level of detail does not exist.
