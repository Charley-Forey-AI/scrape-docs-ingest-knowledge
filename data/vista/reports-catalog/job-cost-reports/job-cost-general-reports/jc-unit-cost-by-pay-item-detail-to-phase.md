# JC Unit Cost By Pay Item Detail To Phase

You can use the JC Unit Cost By Pay Item Detail To Phase report
 by selecting Job Cost > Reports > JC Unit Cost By Pay Item Detail To
 Phase.
This report, grouped and sorted by Contract, Item, and Phase provides comparison of units, unit cost (costs/unit), and dollars between Estimated, This Period, and Job To Date time frames, as well as Variance To Date Unit Cost (Current Estimated Cost/Current Estimated Units - Actual Units/Actual Units) and Variance To Date Cost (Actual Units * Estimated Unit Cost - Actual Cost). The units and cost for the columns in "This Period" summarize job cost detail entered between the Beginning and Ending Date inputs, which is useful for showing weekly job performance. The report further offers the choice of restricting job cost detail between either actual or posted date. The phase subtotal level accumulates units only for phase/cost types with the phase unit flag checked in the JC Job Phases program. Likewise, the total item units only add units for phase/cost types with the item unit flag checked. The Include Jobs with No Activity ? input provides the ability to exclude jobs with only estimate data, or in other words, jobs without any actual costs through the End Date parameter.
Access this report by clicking Job Cost > Reports > JC Unit Cost By Pay Item to Phase.
