# JC Change Order Detail w/Est Profit

You can use the JC Change Order Detail w/Est Profit report
 for approved Change Order report that gives Revenue, Cost, and Estimated profit information
 about each Change Order Item by selecting Job Cost > Reports > JC Change Order Detail w/Est
 Profit.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract
Select the Field Lookup button or press F4 to select the beginning contract.

Ending Contract
Select the Field Lookup button or press F4 to select the ending contract.

Beginning Change Order
Select the Field Lookup button or press F4 to select the beginning change order.

Ending Change Order
Select the Field Lookup button or press F4 to select the ending change order.

checkbox to include approved change order item subtotals.
