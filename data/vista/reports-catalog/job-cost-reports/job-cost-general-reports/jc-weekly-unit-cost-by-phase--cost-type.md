# JC Weekly Unit Cost By Phase & Cost Type

You can use the JC Weekly Unit Cost By Phase & Cost
 Type report by selecting Job Cost > Reports > JC Weekly Unit Cost By Phase & Cost
 Type.
This report, identical to the "JC Unit Cost Report By Pay Item to CT" except for the Projected Final Cost and Variance columns, provides comparison of units, unit cost (costs/unit), and dollars between Estimated, This Period, and Job To Date time frames down to the Cost Type level. The Projected Final Cost is normally the Job To Date UC multiplied by the Current Estimated Units. For lump sum phases/cost types, Projected Final Cost equals Actual Cost when Actual Cost exceeds Current Estimated; otherwise it prints Current Estimated Cost. The report further offers the choice of restricting job cost detail between either actual or posted date. The phase subtotal level accumulates units only for phase/cost types with the phase unit flag checked in the JC Job Phases program. Likewise, the total item units only add units for phase/cost types with the item unit flag checked. The Include Jobs with No Activity ? input provides the ability to exclude jobs with only estimate data, or in other words, jobs without any actual costs through the End Date parameter.
