# JC History By Job/Phase/CT

You can use the JC History By Job/Phase/CT report to list
 purged Job information sorted by Phase, Cost Type and Unit of Measure in the JCHJ table. by
 selecting Job Cost > Reports > JC History By Job/Phase/CT.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Beginning Phase
Select the Field Lookup button or press F4 to select the beginning phase.

Ending Phase
Select the Field Lookup button or press F4 to select the ending phase.

Beginning Cost Type
Enter or select the applicable beginning cost type.

Ending Cost Type
Enter or select the applicable ending cost type.

Unit of Measure
Select the Field Lookup button or press F4 to select the unit of measure.
