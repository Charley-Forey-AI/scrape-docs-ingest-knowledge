# JC Unit Cost Drilldown

You can use the JC Unit Cost Drilldown report to print and
 group by the same columns as the "JC Unit Cost Report By Pay Item Detail to CT" report (but
 presents it in a drill down format by) selecting Job Cost > Reports > JC Unit Cost Drilldown.
This report prints and groups by the same columns as the "JC Unit Cost Report By Pay Item Detail to CT" report, but presents it in a drill down format. Furthermore, job cost detail transaction data is available at the lowest drill down levels.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract
Select the Field Lookup button or press F4 to select the beginning contract.

Ending Contract
Select the Field Lookup button or press F4 to select the ending contract.

Select by (P)osted Date or (A)ctual Date
Enter P or A.

Beginning Date
If By Date or Month is D, enter beginning date (MM/DD/YY).

Ending Date
If By Date or Month is D, enter ending date (MM/DD/YY).

Through Month
Enter through month (MM/YY).
