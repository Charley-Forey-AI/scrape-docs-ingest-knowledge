# JC Cost Type Phases Report

You can use the JC Cost Type Phases Report report for cost
 type(s) and print the phases that contain only the designated cost type(s) by selecting
 Job Cost > Reports > JC Cost Type Phases Report.
This report allows you to input cost type(s) and print the phases that contain
 only the designated cost type(s). Example. this report would be used to list a jobs
 phases that use the labor cost type. Tables used are JCCH, JCCT, JCJP, and JCJM.
 Parameters are Company and selected cost types.
