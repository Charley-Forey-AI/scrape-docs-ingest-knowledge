# JC Costs/Estimates by Project Manager

You can use the JC Costs/Estimates by Project Manager
 report to print Estimated Hours and Dollars, Actual Hours and Dollars and Balance Hours and
 Dollars to the Cost Type level of detail by selecting Job Cost > Reports > JC Costs/Estimates by Project
 Manager.
This report prints Estimated Hours and Dollars, Actual Hours and Dollars and
 Balance Hours and Dollars to the Cost Type level of detail. The report sorts by
 Company, Project Manager, Job, Phase and Cost Type.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project Manager
Select the Field Lookup button or press F4 to select the beginning project manager.

Ending Project Manager
Select the Field Lookup button or press F4 to select the ending project manager.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Through Month
Enter through month (MM/YY).
