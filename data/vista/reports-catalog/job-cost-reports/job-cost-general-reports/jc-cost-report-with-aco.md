# JC Cost Report With ACO

You can use the JC Cost Report With ACO report for JC
 Costs with Approved Change Order Detail by selecting Job Cost > Reports > JC Cost Report With ACO.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Through Month
Enter through month (MM/YY).

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Beginning Phase
Select the Field Lookup button or press F4 to select the beginning phase.

Ending Phase
Select the Field Lookup button or press F4 to select the ending phase.

Beginning Project Manager
Select the Field Lookup button or press F4 to select the beginning project manager.

Ending Project Manager
Select the Field Lookup button or press F4 to select the ending project manager.

Page by Job?
checkbox to have page break per job.
