# JC Phase Codes

You can use the JC Phase Codes report for a list of Phase
 codes by Phase Group by selecting Job Cost > Reports > JC Phase Codes.
This report provides a list of Phase codes by Phase Group. The phase group
 determines which set of Phase codes will be used for each company. The phase group
 is set up in the HQ Company Parameters program.
