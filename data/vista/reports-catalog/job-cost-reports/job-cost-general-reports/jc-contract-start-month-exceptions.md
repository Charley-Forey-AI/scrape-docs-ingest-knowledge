# JC Contract Start Month Exceptions

You can use the JC Contract Master Start Month Exception report by selecting Job Cost > Reports > JC Contract Start Month Exceptions.
The month used to store Original Contract and Estimate values has changed from JCCM Start Month to JCCI Start Month. Many JC Reports rely on JCCM Start Month when aggregating and reporting cost and/or revenue values. "JC Contract Start Month Exceptions Report" will help identify contract items affected by the change, by listing Contract Items where JCCM Start Month is different than JCCI Start Month.
