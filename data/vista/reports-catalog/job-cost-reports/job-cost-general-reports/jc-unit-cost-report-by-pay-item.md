# JC Unit Cost Report By Pay Item

You can use the JC Unit Cost Report By Pay Item report by
 selecting Job Cost > Reports > JC Unit Cost Report By Pay
 Item.
This report, identical to the "JC Unit Cost Report By Pay Item to CT" except for the last two columns, provides comparison of units, unit cost (costs/unit), and dollars between Estimated, This Period, and Job To Date time frames down to the Cost Type level. The last two columns on this report show variances between current estimated and actual for Unit Cost (Cost per Unit) and dollars. The report further offers the choice of restricting job cost detail between either actual or posted date. The phase subtotal level accumulates units only for phase/cost types with the phase unit flag checked in the JC Job Phases program. Likewise, the total item units only add units for phase/cost types with the item unit flag checked. The Include Jobs with No Activity ? input provides the ability to exclude jobs with only estimate data, or in other words, jobs without any actual costs through the End Date parameter.
