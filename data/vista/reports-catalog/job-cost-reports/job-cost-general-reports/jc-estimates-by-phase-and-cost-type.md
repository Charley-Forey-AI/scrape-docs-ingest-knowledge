# JC Estimates By Phase and Cost Type

You can use the JC Estimates By Phase and Cost Type report
 for a list of current estimated Units, Hours, and cost by Job, Phase, and Cost Type by
 selecting Job Cost > Reports > JC Estimates By Phase and Cost
 Type.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.
