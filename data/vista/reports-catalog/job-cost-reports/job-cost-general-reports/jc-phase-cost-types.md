# JC Phase Cost Types

You can use the JC Phase Cost Types report for a list of
 the Phase and Cost Type combinations set up for each Phase Group by selecting Job Cost > Reports > JC Phase Cost Types.
This report provides a list of the Phase and Cost Type combinations set up for
 each Phase Group. Phase Codes are set up in the JC Phase Master form.
