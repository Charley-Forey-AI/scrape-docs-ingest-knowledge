# JC Standard Item Codes

You can use the JC Standard Item Codes report for a list
 of Standard State Item Codes by SI Region by selecting Job Cost > Reports > JC Standard Item Codes.
This report provides a list of Standard State Item Codes by SI Region. These
 codes were set up in the JC Std Item Codes maintenance form.
