# JC Textura Change Order Export

You can use the JC Textura Change Order Export report to
 obtain unformatted data for the Textura Application by selecting Job Cost > Reports > JC Textura Change Order Export.
This report is used to obtain unformatted data for the Textura Application.
 Data comes from Change Order header and Item detail. The report accepts Company,
 Beginning Job and Ending Job. This needs to be exported using the Text (TXT)
 options. Characters per inch = 24, and number of lines per page = 0.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.
