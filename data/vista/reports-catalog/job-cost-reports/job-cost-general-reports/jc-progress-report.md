# JC Progress Report

You can use the JC Progress Report to show the percent of
 units complete at the phase/cost type level for each job by selecting Job Cost > Reports > JC Progress Report.
Report showing the percent of units complete at the phase/cost type level for each job. The percentage is based on the projected units, if entered, or the current estimated units when projected units equal 0 (Actual Units / Projected Units) or (Actual Units / Current Estimated Units).
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Beginning Phase
Select the Field Lookup button or press F4 to select the beginning phase.

Ending Phase
Select the Field Lookup button or press F4 to select the ending phase.

Through Date
Enter through date (MM/YY).
