# JC Insurance Template by Job Phase

You can use the JC Insurance Template by Job Phase report
 to compare all phases set up in the JC Job Phases form and JC Insurance Template form for
 open jobs by selecting Job Cost > Reports > JC Insurance Template by Job
 Phase.
This report is designed to compare all phases set up in the JC Job Phases form and JC Insurance Template form for open jobs. The report can be run by JC Company for a range of Jobs with options to print missing phases only and/or include inactive phases. If printing with option 'Print Missing Phases Only?'=Yes, the report will list the significant part of the JC Job Phases that are missing from the JC Insurance Template per Job. If printing with option 'Print Missing Phases Only?'=No, the report will list all phases set up in the JC Job Phases. The insurance code will print next to the phase description if the phase is set up on the Insurance Template and has an insurance code assigned. If the job phase is not set up, then the report will compare the valid part of the Template Phase with the valid part of Job Phase. If the if they are the same, the insurance code assigned to the valid part of the phase will print for that job phase. (Valid Part of Phase is set up in JC Company Parameters).
