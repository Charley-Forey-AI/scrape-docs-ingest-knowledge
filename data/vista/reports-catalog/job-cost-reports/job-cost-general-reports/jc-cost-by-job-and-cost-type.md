# JC Cost by Job and Cost Type

You can use the JC Cost by Job and Cost Type report to
 print actual cost in a cross tab format by listing cost types (7 cost types per page) as
 columns and jobs in rows by selecting Job Cost > Reports > JC Cost by Job and Cost Type.
The JC Cost by Job and Cost Type prints actual cost in a cross tab format by listing cost types (7 cost types per page) as columns and jobs in rows. The job descriptions will repeat on subsequent pages when eight or more cost types print. Input ranges (Beginning and Ending) are available for job, cost type, and month. The Status parameter for Open, Soft, Closed jobs can help with reconciling to WIP accounts according to the Cost Type/GL Account assignments in JC Departments. Furthermore, the Beginning /Ending Closed Month parameters apply when running the report for closed jobs only.
Access this report by clicking Job Cost > Reports > JC Cost by Job and Cost Type.
