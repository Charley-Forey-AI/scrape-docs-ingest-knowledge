# JC Cost Header Detail

You can use the JC Cost Header Detail report for a list
 all JCCH cost header records on file by selecting Job Cost > Reports > JC Cost Header Detail.
The JC Cost Header Detail report lists all JCCH cost header records on file.
 Data comes from JCCH, JCJM, and JCJP. The report sorts by Company, Job, and Phase
 and prints at the Cost Type level. Columns include Cost Type, Unit of Measure, Bill
 Flag, Item Unit Flag, Phase Unit flag, Buy Out, Last Projected Date, Plugged Flag,
 Active Flag, Original Hours, Original Units, and Original Cost.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.
