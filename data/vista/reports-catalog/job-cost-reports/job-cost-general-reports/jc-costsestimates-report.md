# JC Costs/Estimates Report

You can use the JC Costs/Estimates Report report for Hours
 and Dollars for Estimated, Actual, and Balance Cost information for each Job Phase and Cost
 Type by selecting Job Cost > Reports > JC Costs/Estimates Report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Through Month
Enter through month (MM/YY).

Exclude Zero Estimate/Cost Amounts?
checkbox to exclude zero estimate/cost amounts.
