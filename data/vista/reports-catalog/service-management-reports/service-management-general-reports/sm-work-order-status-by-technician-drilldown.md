# SM Work Order Status By Technician Drilldown

You can use the SM Work Order Status By Technician
 Drilldown report by selecting Service Management > Reports > SM Agreement Analysis
 Drilldown.
This report is used for scheduling and displays all Technicians and Work Orders they have been assigned as Lead Technicians, attended Trips or have Work Completed. It also lists whether or not a Work Order needs attention: if a Billing is ready or if a Work Scope is due. Furthermore the report's layout is grouped by Work Order Status and can be drilled down to the Work Order itself for more information.
