# SM Work Order Activity

You can use the SM Work Order Activity report to see the
 state of costs and billings for work order within the date range specified by the parameters
 by selecting Service Management > Reports > SM Agreement Analysis
 Drilldown.
The Work Order Activity report shows the state of costs and billings for work order within the date range specified by the parameters.
The report groups Work Order data by Work Order and Pricing Method. This means that if you have a work order that has Scopes with mixed pricing methods (T&M and Flat Price, for example), you would expect to see the work order represented twice on the main report: once for the T&M Scopes, and once for the Flat Price Scopes.
The report excludes Work Order Scopes with Price Method = Non-Billable
