# SM Serviceable Items List

You can use the SM Serviceable Items List report by
 selecting Service Management > Reports > SM Serviceable Items List.
The SM Serviceable Item List shows the Name, Description, Class\Type, Manufactured Year, Manufacturer (Make), Model, Warranty Dates, Last Work Order that addressed the Item, and the Location of SM Service Items. The report can be filtered on Company, Customer, Service Site, Class, Type, Make, Model, and Service Site Status. The report also has two levels of sorting. It can first be sorted by either Customer Name or Service Site Name and then by Item Class or Item Type. In regards to Class and Type, if the report is sorting by Class it will show the Class name as a group header and the Class\Type column will only show the Type value. Vice versa if the report is being sorted by Type.
