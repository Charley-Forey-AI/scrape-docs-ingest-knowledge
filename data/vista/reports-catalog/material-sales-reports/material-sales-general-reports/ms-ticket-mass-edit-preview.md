# MS Ticket Mass Edit Preview

You can use the MS Ticket Mass Edit Preview report to see
 processed edits of your MS Ticket Entry batch by selecting Material Sales > Reports > MS Ticket Mass Edit Preview.
This report shows processed edits of your MS Ticket Entry batch. The changes
 shown on this report will change the data on your MS Ticket Entry batch. If a batch
 has been previously posted and then pulled back in for mass editing purposes the
 report will show the transaction number related to the entry. If unposted the report
 reflects the sequence number in the unposted batch. The report can be run directly
 from the Mass Edit program by right clicking directly above the Batch Detail tab or
 it can be run from the MS Reports Menu.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Month
Enter month (MM/YY).

Batch ID
Press F4 to select the batch ID number.
