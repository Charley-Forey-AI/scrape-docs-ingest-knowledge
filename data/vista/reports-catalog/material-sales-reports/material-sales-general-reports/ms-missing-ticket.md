# MS Missing Ticket

You can use the MS Missing Ticket report to list
 any ticket numbers that are not stored in the database for the ticket number range used in
 the report input fields by selecting Material Sales > Reports > MS Missing Ticket.
This report will list any ticket numbers that are not stored in the database for the ticket number range used in the report input fields. Only tickets with numeric ticket numbers will be included in the report. Ticket numbers that have alphabetical characters, punctuation, or special characters will not be evaluated in the report.
