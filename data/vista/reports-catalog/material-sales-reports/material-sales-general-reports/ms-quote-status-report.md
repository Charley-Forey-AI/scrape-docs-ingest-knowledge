# MS Quote Status Report

You can use the MS Quote Status Report report to display
 the sum of Units sold for MS Quotes by selecting Material Sales > Reports > MS Quote Status Report.
This report will display the sum of Units sold for MS Quotes. It uses a custom report view to gather the Quote Details and the Price Override information with the MS Ticket Details for the sold units.
