# MS Truck Types List

You can use the MS Truck Types List report for a list of
 the details from the MS Truck Types maintenance program by selecting Material Sales > Reports > MS Truck Types List.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Truck Type
Click the Field Lookup
 button or press F4 to select beginning truck type, or leave
 blank for all.

Ending Truck Type
Click the Field Lookup
 button or press F4 to select ending truck type, or leave blank
 for all.

checkbox to print notes.
