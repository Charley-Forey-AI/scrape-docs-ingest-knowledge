# IN Location Company Overrides

You can use the IN Location Company Override report to see GL Account overrides to locations by company. To access this report, select Inventory > Reports > IN Location Company Overrides.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.
