# IN Inventory Price List

You can use the IN Inventory Price List report to see a
 listing by location (then material, standard price, customer price, job rate, inventory
 rate, equipment rate, standard cost) by selecting Inventory > Reports > IN Inventory Price List.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.

Beginning Material
Select the Field Lookup button or press F4 to select the beginning material.

Ending Material
Select the Field Lookup button or press F4 to select the ending material.
