# IN Value of Inventory at Month End

You can use the IN Value of Inventory at Month End report
 to see a detailed list of materials by location and their value per month. To access this report, select Inventory > Reports > IN Value of Inventory at Month
 End.
This report will detail the values of inventory at month end for the Cost and Valuation methods chosen. It is necessary to run the IN Monthly Reconciliation process before the data is available for this report.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.
