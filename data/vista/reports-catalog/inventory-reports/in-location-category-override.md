# IN Location Category Override

You can use the IN Location Category Override report to see GL Account overrides to locations by category. To access this report, select Inventory > Reports > IN Location Category Override.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.
