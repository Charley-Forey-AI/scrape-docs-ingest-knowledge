# IN Negative Price/Cost Report

You can use the IN Negative Price/Cost Report by selecting
 Inventory > Reports > IN Negative Price/Cost Report.
You should run this report prior to running
 the IN Monthly Reconciliation process. The report identifies inventory materials that have
 negative amounts in one or more of the following fields: Standard Price, Customer
 Rate, Job Rate, Inventory Rate, Equipment Rate, Average Unit Cost, Standard Unit
 Cost, or Last Unit Cost. The rate fields represent the cost plus markup depending on
 the Pricing Options in the IN Company Parameters form.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.

Beginning Material
Select the Field Lookup button or press F4 to select the beginning material.

Ending Material
Select the Field Lookup button or press F4 to select the ending material.
