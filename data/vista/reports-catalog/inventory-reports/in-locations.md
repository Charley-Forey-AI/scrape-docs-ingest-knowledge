# IN Locations

You can use the IN Locations report to see setup data from IN Locations. To access this report, select Inventory > Reports > IN Locations.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.

Print Active Only?
checkbox to print only active locations.

Print Notes?
checkbox to print notes.
