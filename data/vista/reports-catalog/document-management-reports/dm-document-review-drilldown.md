# DM Document Review Drilldown

You can use the DM Document Review Drilldown report to
 sort Documents for review by Reviewer or by the Added by person by selecting Document Management > Reports > DM Document Review Drilldown.
This report's drilldown will sort Documents for review by Reviewer or by the
 Added by person. The Sort will be either the Reviewer or Added by person based on
 the Sort parameter then it will sort by Attachment ID and will have a Beginning and
 Ending Date Added Range and will show a link to the Attachment(s).
Report Parameters
Description

(A)dded by or (R)eviewer
Enter A or R.

User Name
Click the Field Lookup
 button or press F4 to select beginning attachment ID or leave
 blank to select User Name.

Beginning Date Added
Enter beginning date or leave blank for all.

Ending Date Added
Enter ending date or leave blank for all.

Include Completed?
checkbox to include document reviews that have been completed.
