# DM Document Routing Status Codes

You can use the DM Document Routing Status Codes report to
 list the HQ Document Review Status Code by selecting Document Management > Reports > DM Document Routing Status
 Codes.
Report Parameters
Description

Beginning Status Code
Click the Field Lookup
 button or press F4 to select beginning status code or leave
 blank for all.

Ending Status Code
Click the Field Lookup
 button or press F4 to select ending status code or leave blank
 for all.
