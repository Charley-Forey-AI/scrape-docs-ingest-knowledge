# DM Attachment Audit Log

You can use the DM Attachment Audit Log report to report
 all DM attachment activity by attachments or a time range by selecting Document Management > Reports > DM Attachment Audit Log.
The DM Attachment Audit Log has an option to exclude "View" events. The
 attachment will be identified by the current name but if it has been renamed the
 original name will also be displayed.
This report may be used by all personnel to track attachment activity; however, its primary audience is IT staff and security personnel.
