# IM Cross Reference Detail

You can use the IM Cross Reference Detail report to list import values and the corresponding Vista value for a range of import templates. To access this report, select ImportsReportsIM Cross Reference Detail.
Report Parameters
Description

Beginning Template
Click the Field Lookup
 button or press F4 to select the beginning template.

Ending Template
Click the Field Lookup
 button or press F4 to select the ending template.
