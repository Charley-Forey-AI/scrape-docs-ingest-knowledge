# IM Upload Error Report

You can use the IM Upload Error Report to see a list of errors that were generated during the upload process. To access this report, select
 Imports > Reports > IM Upload Error Report.
You can change the records in the IM Work Edit form, or the import file can
 be corrected and uploaded again.
Parameters

Import ID (Required)
Click the Field Lookup
 button or press F4 to select the import ID.
