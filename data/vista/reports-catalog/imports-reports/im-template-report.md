# IM Template Report

You can use the IM Template Report to see a list of templates, grouped by header and detail information. To access this report, select Imports > Reports > IM Template Report.
Report Parameters
Description

Beginning Import Template
Click the Field Lookup
 button or press F4 to select the beginning import template.

Ending Import Template
Click the Field Lookup
 button or press F4 to select the ending import template.
