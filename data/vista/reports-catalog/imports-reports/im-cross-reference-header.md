# IM Cross Reference Header

You can use the IM Cross Reference Header report to see a list of fields for a range of templates from the IM Cross Reference Header form. To access this report, select
 Imports > Reports > IM Cross Reference Header.
Report Parameters
Description

Beginning Template
Click the Field Lookup
 button or press F4 to select the beginning template.

Ending Template
Click the Field Lookup
 button or press F4 to select the ending template.
