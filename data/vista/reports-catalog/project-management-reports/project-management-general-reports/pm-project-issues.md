# PM Project Issues

You can use the PM Project Issues report by selecting
 Project Management > Reports > PM Project Issues.
The PM Project Issues report, sorted by Project and Issue, prints the issue date, action, login, and action date for all issue history entries. The report includes all types of documents entered with issue numbers in the PM module.
