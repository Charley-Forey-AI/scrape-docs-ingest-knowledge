# PM Phase Submittals – 6.5

You can use the PM Phase Submittals – 6.5 report by
 selecting Project Management > Reports > PM Phase Submittals – 6.5.
This report provides phase submittals set up in PM by phase group. Table is
 PMPS.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Phase
Enter or select the applicable beginning phase.

Ending Phase
Enter or select the applicable ending phase.

Submittal Type (Blank for All)
Click the Field Lookup
 button or press F4 to select the submittal type or leave blank
 for all.
