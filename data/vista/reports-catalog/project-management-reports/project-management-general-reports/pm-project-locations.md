# PM Project Locations

You can use the PM Project Locations report for a list of
 Project locations entered in the Project Locations form by selecting Project Management > Reports > PM Project Locations.
This report provides a list of Project locations entered in the Project
 Locations form. The report includes project, location, location description, and
 notes (if requested).
