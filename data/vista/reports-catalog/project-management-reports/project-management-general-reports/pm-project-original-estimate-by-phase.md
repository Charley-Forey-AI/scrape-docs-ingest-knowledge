# PM Project Original Estimate by Phase

You can use the PM Project Original Estimate by Phase
 report by selecting Project Management > Reports > PM Project Original Estimate by
 Phase.
The PM Project Detail by Phase report lists each Job, Phase, and Cost Type
 original hours, units, and original cost from the JCCH table. The division is a sort
 based on the first two characters of the Phase Code.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.
