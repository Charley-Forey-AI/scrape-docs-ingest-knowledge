# PM Subcontract Change Orders - All

You can use the PM Subcontract Change Orders - All report
 by selecting Project Management > Reports > PM Subcontract Change Orders -
 All.
This report prints Subcontract Change Orders for an individual Project,
 including the status of each change order. The report will sort by PMCO, Project,
 SLCo, SL, and SLItem.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Project (Required)
Select the Field Lookup button or press F4 to select the project.

Beginning Subcontract
Click the Field Lookup
 button or press F4 to select the beginning subcontract.

Ending Subcontract
Click the Field Lookup
 button or press F4 to select the ending subcontract.
