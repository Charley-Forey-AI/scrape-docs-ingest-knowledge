# PM Project Drilldown

You can use the PM Project Drilldown report by selecting
 Project Management > Reports > PM Project Drilldown.
This Drilldown report lists Project cost totals by company for a range of projects. By drilling down, the report breaks down the cost totals at the Phase level, then at the Cost Type level. The Cost Type level shows Approved and Pending Change Orders, with additional drilldown buttons for Committed Costs and Actual Costs. The Committed Costs level lists JC Transaction Type, Source, Record Type (i.e. PO#, MO#, AP Ref), Remaining Units and Costs. The Actual Costs level lists month, transaction, posted and actual dates, source, description and actual hours, Units and Costs. At this level there is an option to drilldown further at the transaction level to see additional details.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Project Manager (Blank for All)
Select the Field Lookup button or press F4 to select the project manager or leave blank for all.
