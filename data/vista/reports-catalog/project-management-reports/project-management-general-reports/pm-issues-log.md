# PM Issues Log

You can use the PM Issues Log report to list the issues
 related to a project by date, initiator, master issue, status and resolved date by selecting
 Project Management > Reports > PM Issues Log.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Beginning Issue
Select the Field Lookup button or press F4 to select the beginning issue.

Ending Issue
Select the Field Lookup button or press F4 to select the ending issue.

Status: (O)pen, (C)losed, or blank for All
Enter O, C, or leave blank for all.
