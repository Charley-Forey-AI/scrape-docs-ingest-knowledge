# PM Change Order Request – Phase

You can use the PM Change Order Request – Phase report by
 selecting Project Management > Reports > PM Change Order Request –
 Phase.
The PM Change Order Request - Phase report prints pending change order forms addressed to the owner by Pending Change Order Type (PCOType), Number (PCO), and Item (PCOItem) with phase detail. In addition to the standard pending change order data, the form includes the item notes from the pending change order item. The form also prints markups (if 'Y' chosen for Show Markups? input) entered via the Totals button in PM Change Orders if change order cost detail is entered for the item. Add-ons also print for each item as long as fixed amounts are not entered.
