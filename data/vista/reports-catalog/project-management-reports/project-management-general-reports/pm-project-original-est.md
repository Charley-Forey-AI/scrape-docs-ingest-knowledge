# PM Project Original Est. by Cost Type

You can use the PM Project Original Est. by Cost Type
 report by selecting Project Management > Reports > PM Project Original Est. by Cost
 Type.
This report visualizes PM Project Original Estimates by Cost Type in a cross tab. The rows define the various phases and the columns represents the various cost types.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Project (Required)
Select the Field Lookup button or press F4 to select the project.
