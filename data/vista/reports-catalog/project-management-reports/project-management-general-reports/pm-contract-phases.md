# PM Contract Phases

You can use the PM Contract Phases report to list phases
 by contract items on selected projects by selecting Project Management > Reports > PM Contract Phases.
This report lists phases by contract items on selected projects. It includes the original contract amount and the original estimated costs and calculates a variance or profit/loss for each contract item. Original contract amounts are pulled from JCCI and original estimated costs from JCCH. The brptJCUnitCost stored procedure is used on this report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.
