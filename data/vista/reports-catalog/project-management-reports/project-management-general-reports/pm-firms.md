# PM Firms

You can use the PM Firms report by selecting
 Project Management > Reports > PM Firms.
This report provides a list of Firms and their static information. Firm
 information includes Firm Name and number; Firm Type; Firm Sort Name; Vendor #;
 Mailing, Shipping, and e-mail addresses; Contact; Fax #; and Internet Address.
 Parameters include Company, Beginning and Ending Firm Sort Name and Print Notes?
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Firm Sort Name
Click the Field Lookup
 button or press F4 to select the beginning firm sort name.

Ending Firm Sort Name
Click the Field Lookup
 button or press F4 to select the ending firm sort name.

Print Notes?
checkbox to print notes.
