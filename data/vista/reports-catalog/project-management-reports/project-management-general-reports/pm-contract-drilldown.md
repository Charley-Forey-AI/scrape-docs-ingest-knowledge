# PM Contract Drilldown

You can use the PM Contract Drilldown report by selecting
 Project Management > Reports > PM Contract Drilldown.
This Drilldown will start at Contract, then go to Contract Item, then Phase, then Cost Type and finally to Subcontract/Material.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract
Select the Field Lookup button or press F4 to select the beginning contract item.

Ending Contract
Select the Field Lookup button or press F4 to select the ending contract item.

Project Manager (Blank for All)
Select the Field Lookup button or press F4 to select the project manager or leave blank for all.
