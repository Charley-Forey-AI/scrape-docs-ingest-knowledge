# PM Change Order Status Report

You can use the PM Change Order Status Report report to
 list Approved Change Orders and Pending Change Orders per Project with the option of only
 printing by Project Manager by selecting Project Management > Reports > PM Change Order Status Report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Project Manager (Blank for All)
Select the Field Lookup button or press F4 to select the project manager or leave blank for all.
