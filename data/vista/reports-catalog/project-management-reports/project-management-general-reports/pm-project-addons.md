# PM Project Addons

You can use the PM Project Addons report for a list of
 each project and the project's Add-ons by selecting Project Management > Reports > PM Project Addons.
This report provides a list of each project and the project's Add-ons. The
 report includes Add-on, Add-on Description, Basis, Percent, Amount, Phase, Cost
 Type, Contract, and Contract Item. Parameters include beginning and ending project
 and whether to print notes.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Print Notes?
checkbox to print notes.
