# PM Pending Contracts List

You can use the PM Pending Contracts List report by
 selecting Project Management > Reports > PM Pending Contracts List.
This report provides a list of Pending Contracts. This report prints contracts
 with a status flag = 0.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract
Select the Field Lookup button or press F4 to select the beginning contract.

Ending Contract
Select the Field Lookup button or press F4 to select the ending contract.
