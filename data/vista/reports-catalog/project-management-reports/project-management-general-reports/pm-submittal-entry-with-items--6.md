# PM Submittal Entry with Items – 6.5

You can use the PM Submittal Entry with Items – 6.5 report
 to display the Submittal Items by selecting Project Management > Reports > PM Submittal Entry with Items –
 6.5.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Enter S, T, or N.

checkbox to print header notes.
