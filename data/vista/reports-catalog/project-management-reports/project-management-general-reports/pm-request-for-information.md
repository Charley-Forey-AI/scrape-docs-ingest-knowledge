# PM Request For Information

You can use the PM Request For Information report to print
 the details for RFI type documents maintained in the PM Request for Information program by
 selecting Project Management > Reports > PM Request For Information.
The PM Request For Information report prints the details for RFI type documents maintained in the PM Request for Information program. The report sorts by Project, RFI Type, and RFI code.
