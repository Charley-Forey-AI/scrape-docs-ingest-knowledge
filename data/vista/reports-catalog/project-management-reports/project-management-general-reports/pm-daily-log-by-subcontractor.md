# PM Daily Log by Subcontractor

You can use the PM Daily Log by Subcontractor report to
 sort daily log information by Company, Project, and subcontract by selecting Project Management > Reports > PM Daily Log by Subcontractor.
The PM Daily Log by Subcontractor report sorts daily log information by
 Company, Project, and subcontract. It prints only subcontract-related items. Columns
 on this report include Log Date, Start Time, End Time, Supervisor, Foreman,
 Journeyman, Apprentices, and Location.
