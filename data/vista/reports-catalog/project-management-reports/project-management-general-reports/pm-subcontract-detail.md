# PM Subcontract Detail

You can use the PM Subcontract Detail report by selecting
 Project Management > Reports > PM Subcontract Detail.
PM Subcontract Details mimics the Subcontract Detail form in Project Management. The report sorts by Project; Record Type (Original Estimate, Pending Change Order, or Approved Change Order), Unassigned (no SL#)/Assigned (with an SL#)/Interfaced (has an interfaced date), Subcontract Number, and then finally Sequence. The report includes parameters for beginning and ending project and a selection to print Original Estimates Only, Approved Change Orders Only, Pending Change Orders Only, or All.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

(O)riginal, (A)CO’s, (P)CO’s (Blank for All)
