# PM Pending Change Order Summary

You can use the PM Pending Change Order Summary report by
 selecting Project Management > Reports > PM Pending Change Order
 Summary.
The PM Pending Change Order Summary is a concise listing of pending change orders by Project, PCO Type and Pending Change Order Number, similar to what is seen on the first level of the Pending tab in the PM Change Orders program.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Project Manager (Blank for All)
Select the Field Lookup button or press F4 to select the project manager or leave blank for all.
