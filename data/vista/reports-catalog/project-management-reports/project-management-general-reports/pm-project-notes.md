# PM Project Notes

You can use the PM Project Notes report to print PM
 Project Notes sorted by PM Company, Project, Date Added and Added-by by selecting
 Project Management > Reports > PM Project Notes.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Beginning Date Added
Enter beginning date for added notes.

Ending Date Added
Enter ending date for added notes.

Added by
Click the Field Lookup
 button or press F4 to select added by.
