# PM Daily Project Log With Options

You can use the PM Daily Project Log With Options report
 by selecting Project Management > Reports > PM Daily Project Log With
 Options.
The PM Daily Project Log With Options report, like the PM Daily Project Log
 report, lists the activities related to a job by day, but includes options for
 printing the daily log distribution for the different log types (employees, crews,
 subcontracts, equipment, activities, conversations, deliveries, accidents, and
 visitors).
