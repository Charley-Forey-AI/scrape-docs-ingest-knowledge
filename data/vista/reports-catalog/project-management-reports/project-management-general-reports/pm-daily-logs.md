# PM Daily Logs

You can use the PM Daily Logs report by selecting
 Project Management > Reports > PM Daily Logs.
The PM Daily Project Log report is a succinct report that prints one line per daily log by project.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Beginning Log Date
Enter beginning log date (MM/DD/YY).

Ending Log Date
Enter ending log date (MM/DD/YY).
