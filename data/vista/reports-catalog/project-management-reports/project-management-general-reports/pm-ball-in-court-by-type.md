# PM Ball In Court by Type

You can use the PM Ball In Court by Type report to show
 all open items for Transmittals, Requests for Information, Submittals, Other Documents and
 Requests for Quotes further sorted by Document Type by selecting Project Management > Reports > PM Ball In Court by Type.
This report shows all open items for Transmittals, Requests for Information, Submittals, Other Documents and Requests for Quotes further sorted by Document Type. The Project parameter is required and the Date parameter is used to calculate days overdue. Items marked as 'Final' will not be included at all. The report also has an option to print Pending Change Order Items. When answering "Y" for the Include Future Due Dates input, the report will include items with due dates later than the inputted Date parameter.
