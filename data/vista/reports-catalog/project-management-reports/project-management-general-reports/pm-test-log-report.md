# PM Test Log Report

You can use the PM Test Log Report report to list the test
 activities related to a project by selecting Project Management > Reports > PM Test Log Report.
The PM Test Log report lists the test activities related to a project.
 Additional information such as type, date, code, location, firm/contact, tester,
 status and issue are also included.
