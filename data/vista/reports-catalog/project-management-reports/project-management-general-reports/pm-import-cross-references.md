# PM Import Cross References

You can use the PM Import Cross References report to list
 cross references set up by template for Phases, cost types, unit of measures, materials, and
 vendors by selecting Project Management > Reports > PM Import Cross References.
This report lists cross references set up by template for Phases, cost types,
 unit of measures, materials, and vendors. (PMUX Table)
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Template
Select the Field Lookup button or press F4 to select beginning template. Leave blank to include all templates.

Ending Template
Select the Field Lookup button or press F4 to select ending template. Leave blank to include all templates.

Page By Template?
checkbox for page break for each template.
