# PM Request For Information Log

You can use the PM Request For Information Log report by
 selecting Project Management > Reports > PM Request For Information Log.
The PM Request for Information log is a concise version of the PM Request for Information report with less RFI header information. When run in summary, the report prints one line per RFI Type and RFI showing the RFI date, Firm and Contact Name, and Status. When run in detail, the same fields shown under the summary option print in a separate group heading, and the report includes the Sent To Firm and Contact, Information Requested, Date Requested, Response, and Date Received.
