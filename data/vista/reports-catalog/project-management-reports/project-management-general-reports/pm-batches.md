# PM Batches

You can use the PM Batches report to display all PM
 Projects related batches by selecting Project Management > Reports > PM Batches.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.
