# PM Import Cost Types

You can use the PM Approved Change Order report for a list
 of the Import Create Cost Types table (PMUC) by selecting Project Management > Reports > PM Approved Change Order.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Template
Select the Field Lookup button or press F4 to select beginning template. Leave blank to include all templates.

Ending Template
Select the Field Lookup button or press F4 to select ending template. Leave blank to include all templates.

Beginning Phase Group
Click the Field Lookup
 button or press F4 to select the beginning phase group.

Ending Phase Group
Click the Field Lookup
 button or press F4 to select the ending phase group.
