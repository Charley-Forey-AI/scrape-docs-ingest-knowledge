# PM Drawing Log Report

You can use the PM Drawing Log Report report to list the
 activities related to a project by selecting Project Management > Reports > PM Drawing Log Report.
The PM Drawing Log report lists the activities related to a project.
 The report is sorted by Company, Project, Drawing Type, Drawing, Issue Date, Revision
 and Revision Date.
