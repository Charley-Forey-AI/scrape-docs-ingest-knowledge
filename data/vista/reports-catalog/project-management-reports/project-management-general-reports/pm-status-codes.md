# PM Status Codes

You can use the PM Status Codes report for a list of
 Status Codes by selecting Project Management > Reports > PM Status Codes.
This report provides a list of Status Codes. These codes are entered in the
 Status Codes form. The report includes Status, Status Description, and Code Type
 (Beginning, intermediate, or final).
