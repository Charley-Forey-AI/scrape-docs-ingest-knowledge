# PM Interfaced Buy Out By ACO Item Report

You can use the PM Interfaced Buy Out By ACO Item Report
 report by selecting Project Management > Reports > PM Interfaced Buy Out By ACO Item
 Reportr.
The PM Interfaced Buy Out by ACO Item report, first sorted by project, phase, and cost type, prints the savings between current estimated and committed cost for original interfaced estimates and then by Approved change order and item. The Include Cost Types parameter gives the ability to specify a selected list of Cost Types by inputting them as follows: 2,3,30, etc.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Cost Types to Include (1,2,3) (Blank for all)
Enter cost types to include separated by comma (e.g. 1,2,3) or leave blank for all.
