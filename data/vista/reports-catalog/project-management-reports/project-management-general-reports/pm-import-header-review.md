# PM Import Header Review

You can use the PM Import Header Review report by
 selecting Project Management > Reports > PM Import Header Review.
A list for reviewing purposes of Header information loaded into the (PMWH) table to be uploaded into PM. The items may also be reviewed in the program which shows them the errors on line.
Report Parameters
Description
Import (Blank for All)
Select the Field Lookup button or press F4 to select the Import ID or leave blank for all.
