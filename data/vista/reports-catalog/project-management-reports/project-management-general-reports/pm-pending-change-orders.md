# PM Pending Change Orders

You can use the PM Pending Change Orders report to print
 change order costs for each phase and cost type pertaining to a pending change order by
 selecting Project Management > Reports > PM Pending Change Orders.
The Pending Change Orders report prints change order costs for each phase and cost type pertaining to a pending change order. The report also lists the add-ons related to each PCO Item. Only add-ons assigned to specific phases are included since these become additional costs after the change order is approved.
