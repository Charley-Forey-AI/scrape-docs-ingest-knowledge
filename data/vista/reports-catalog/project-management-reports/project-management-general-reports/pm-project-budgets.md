# PM Project Budgets

You can use the PM Project Budgets report to list the
 budget of a PM Project including units, hours, UM, unit cost and amount by selecting
 Project Management > Reports > PM Project Budgets.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Project (Required)
Select the Field Lookup button or press F4 to select the project.

Budget Number
Click the Field Lookup
 button or press F4 to select the beginning budget number.

Ending Budget Number
Click the Field Lookup
 button or press F4 to select the ending budget number.

Print Notes?
Enter Y or N
