# PM Batch Control Error List

You can use the PM Batch Control Error List report by
 selecting Project Management > Reports > PM Batch Control Error List.
PM Batch Control Error is run from the PM Interface program only when errors occur during the Validation process. The report uses the HQBE table, which PM Interface updates with any error information. The report also lists validation errors for quotes in a subreport that runs off of the PMBE table.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Month
Enter Month (MM/YY).

BatchId’s
Enter batch IDs.
