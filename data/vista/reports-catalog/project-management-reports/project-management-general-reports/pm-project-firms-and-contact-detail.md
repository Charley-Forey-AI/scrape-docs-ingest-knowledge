# PM Project Firms and Contact Detail

You can use the PM Project Firms and Contact Detail report
 by selecting Project Management > Reports > PM Project Firms and Contact
 Detail.
This report provides a list of each of the firms with which each project
 works. The report includes project, firm, contact person from the firm, and a
 description. Parameters include beginning and ending project and whether to print
 notes. This report also includes the contact phone, fax, email and preferred method
 of contact.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Print Firm Notes?
checkbox to print firm notes.

Print Contact Notes?
checkbox to print contact notes.

Print Project Firm Notes?
checkbox to print project firm notes.
