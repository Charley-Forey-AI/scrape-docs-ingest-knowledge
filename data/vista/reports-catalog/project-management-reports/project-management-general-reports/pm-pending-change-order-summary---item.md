# PM Pending Change Order Summary - Item

You can use the PM Pending Change Order Summary - Item
 report to summarize the pending change order by PCO item and PCO type by selecting
 Project Management > Reports > PM Pending Change Order Summary -
 Item.
The PM Pending Change Order Summary - Item report summarizes the pending change order by PCO item and PCO type. Information such as issue, pending amount, approved amount, status, ACO and ACO item are also included.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Project Manager (Blank for All)
Select the Field Lookup button or press F4 to select the project manager or leave blank for all.
