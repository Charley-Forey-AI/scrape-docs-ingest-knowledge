# PM Project Cost Type Markups

You can use the PM Project Cost Type Markups report for a
 list of Project Cost Types and their markups by selecting Project Management > Reports > PM Project Cost Type Markups.
This report provides a list of Project Cost Types and their markups.
 Parameters include beginning and ending project.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.
