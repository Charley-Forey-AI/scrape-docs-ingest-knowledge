# PM Import Templates

You can use the PM Import Templates report by selecting
 Project Management > Reports > PM Import Templates.
Printout of the Import templates set up in the Import Template Master (PMUT) which shows tables for Info, Import parameters, Edit parameters, and upload parameters.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Template
Select the Field Lookup button or press F4 to select the beginning template or leave blank for all.

Ending Template
Select the Field Lookup button or press F4 to select the ending template or leave blank for all.

Page By Template?
checkbox for page break for each template.

Print Notes?
checkbox to print notes.
