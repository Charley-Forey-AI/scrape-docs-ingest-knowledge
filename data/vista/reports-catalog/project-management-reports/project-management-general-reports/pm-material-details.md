# PM Material Details

You can use the PM Material Details report by selecting
 Project Management > Reports > PM Material Details.
The PM Material Details report mimics the Material Detail form in Project Management. The report sorts by Project, Record Type (Original Estimate, Pending Change Order, Approved Change Order or Purchase Order Change Order), Unassigned (no PO or MO #)/Assigned (with a PO or MO #)/Interfaced (has an interfaced date), Material Code, and then finally Sequence. The report includes parameters for beginning and ending Project and a selection to print Original Estimates Only, Approved Change Orders Only, Pending Change Orders Only, or All. This report prints landscape.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

(O)riginal, (P)ending, (A)pproved (Blank for All)
Enter O, P, A, or leave blank for all.
