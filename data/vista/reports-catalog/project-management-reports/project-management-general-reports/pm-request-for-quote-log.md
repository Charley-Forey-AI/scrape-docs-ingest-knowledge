# PM Request For Quote Log

You can use the PM Request For Quote Log report by
 selecting Project Management > Reports > PM Request For Quote Log.
This report displays all of the information necessary for tracking RFQs: RFQ number, RFQ Description, the RFQ and Due Dates, the Firm and Contact information, RFQ Status, Item Scopes, and Notes. You can select data by specifying an RFQ, Job, and Status, and there is an option to include the item notes.
This report only displays RFQs that were created using the new PM Request For Quote form, or legacy RFQs that were imported into the new form using the PM Migrate Data form. The report will also only display RFQs with RFQ items.
