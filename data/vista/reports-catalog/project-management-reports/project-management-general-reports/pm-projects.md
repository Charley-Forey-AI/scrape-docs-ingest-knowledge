# PM Projects

You can use the PM Projects report by selecting Project Management > Reports > PM Projects.
This report lists the projects with description, mail address, phone, fax, customer, project manager, project status, and SMSA. There is an option to run the report for full details which will list all the fields on each tab in the PM Projects form. The inputs are for PM Company, Projects, Open/Closed/all Jobs, Detail or Summary, and an input to print notes.
