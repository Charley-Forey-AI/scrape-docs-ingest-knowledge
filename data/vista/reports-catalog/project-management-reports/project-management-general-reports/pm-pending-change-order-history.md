# PM Pending Change Order History

You can use the PM Pending Change Order History report to
 present issue history information of pending change orders by project, pending change order
 type, and pending change order number by selecting Project Management > Reports > PM Pending Change Order
 History.
The PM Pending Change Order History report presents issue history information of pending change orders by project, pending change order type, and pending change order number. The report continues sorting by pending change order item and issue number and prints the action taken and issue date for each document revision. Only pending change orders assigned an issue number will print on this report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.
