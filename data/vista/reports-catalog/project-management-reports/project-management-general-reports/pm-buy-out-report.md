# PM Buy-Out Report

You can use the PM Buy-Out Report report to compare
 estimated and committed costs by Project, Phase, and Cost Type by selecting Project Management > Reports > PM Buy-Out Report.
The PM buy-out report compares estimated and committed costs by Project, Phase, and Cost Type. The report draws from PM tables, so information will be displayed whether interfaced to JC or not. The Include Cost Types parameter gives the ability to specify a selected list of Cost Types by inputting them as follows: 2,3,30, etc.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Include Cost Types (ie: 2,3)
Enter cost types to include, separated by commas, i.e. 2,3,30 or leave blank for all.
