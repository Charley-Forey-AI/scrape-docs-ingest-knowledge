# PM Change Order Request – Item

You can use the PM Change Order Request – Item report by
 selecting Project Management > Reports > PM Change Order Request – Item.
The PM Change Order Request - Item report prints pending change order forms addressed to the owner by Pending Change Order Type (PCOType), Number (PCO), and Item (PCOItem). In addition to the standard pending change order data, the form includes change order item units, unit price, and notes.
