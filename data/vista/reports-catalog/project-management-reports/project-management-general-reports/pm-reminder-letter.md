# PM Reminder Letter

You can use the PM Reminder Letter report by selecting
 Project Management > Reports > PM Reminder Letter.
The PM Reminder Letters report serves as a form used for presenting submittal items, not yet returned or acknowledged, to architectural engineering firms. The report only includes those items where the due date (Due back from Architect) is blank or less than the current date the report is generated.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Project
Select the Field Lookup button or press F4 to select the beginning project.

Ending Project
Select the Field Lookup button or press F4 to select the ending project.

Return By Date (Top of Form) (Required)
Enter return by date.
