# PM Request For Quote Log 6.6

You can use the PM Request For Quote Log 6.6 report by
 selecting Project Management > Reports > PM Request For Quote Log 6.6.
This report provides the user with information necessary to track RFQs. the RFQ number, the PCO, the PCO Type, RFQ Description, the RFQ and Due Dates, the Firm and Contact information, RFQ Status, and Notes. The user can select data by specifying RFQ, Job, PCO and/or PCO Type, and also has the option of displaying data contained in the Notes field.
