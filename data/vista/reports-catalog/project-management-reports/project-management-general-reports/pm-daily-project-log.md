# PM Daily Project Log

You can use the PM Daily Project Log report to list the
 activities related to a job by day by selecting Project Management > Reports > PM Daily Project Log.
The PM Daily Project Log report lists the activities related to a job by day.
 The report is sorted by Company, Project, Log Date, Log Number, and Log Type
 (employees, crews, subcontracts, equipment, activities, conversations, deliveries,
 accidents, and visitors). The report further shows daily log distribution
 information for each log type.
