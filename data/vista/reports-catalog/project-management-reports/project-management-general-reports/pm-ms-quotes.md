# PM MS Quotes

You can use the PM MS Quotes report by selecting
 Project Management > Reports > PM MS Quotes.
The PM MS Quotes report prints in a similar layout as the MS Quote form, but instead shows material sales quotes from the PM material detail table PMMF. If printed in detail, the report will also show header information from the info tab in MS Quote Header. With the summary option selected, the report prints in a more concise format showing only the project, quote #, description, quote date, and expiration date from MS Quote Header.
