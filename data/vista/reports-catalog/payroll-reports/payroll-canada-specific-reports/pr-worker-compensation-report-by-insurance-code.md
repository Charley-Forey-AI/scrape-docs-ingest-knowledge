# PR Worker Compensation Report by Insurance Code

You can use the PR Worker Compensation Report for the
 information required to complete Worker's Compensation reports for each Province by
 Insurance Code report by selecting Payroll > Reports > PR Certified Export Template -
 eMars.
The PR Insurance Report provides the information required to complete Worker's Compensation reports for each Province. The report sorts by Province, Insurance Code, and Deduction/Liability Code (DL Code). The report will also print a separate line for each rate and each DL Code. The Gross Amount column will exclude earnings that are flagged as Subject Only on the workers compensation deduction/liability code set up. The Subject Amount column includes all Earning Codes where the deduction/liability code for workers compensation is assigned. The Eligible column represents the amount used to calculate the premiums and is multiplied by the DL Code rate. The Payroll Processing programs calculate and store the amounts in the columns described above in the PRIA (PR Insurance Accumulations) table used by this report.
When the report is run by payroll ending date the amounts reflected on the report are for all payrolls that have been processed. When the report is run for paid month the open payrolls must be paid in order for the amounts to show up on the report.
