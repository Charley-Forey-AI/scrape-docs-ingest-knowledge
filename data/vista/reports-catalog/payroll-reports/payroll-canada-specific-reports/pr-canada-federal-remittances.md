# PR Canada Federal Remittances

You can use the PR Canada Federal Remittances report to review remittance amounts by employee for a date range of your choice by selecting Payroll > Reports > PR Canada Federal Remittances.

This report displays, by employee, the deduction and liability amounts for specified federal deduction and liability codes from employee pay sequence detail. Report also includes amounts for provincial/territorial tax deductions configured for the specified payroll company.
Inputs for the PR Company and federal deduction and liability codes are required. Each CPP, EI, or Federal Tax input field is intended for a single deduction or liability code representing the appropriate deduction or liability.
 The Payroll Program Account Number input field is optional. If you supply a valid value, the report displays payroll data for that account only. If you leave the input field blank, the report displays data for all accounts.
The Paid Date fields are also optional. If you supply date values, the report restricts data to amounts from paid employee sequences whose paid dates fall within the range of your specified dates, inclusively. If you leave these fields blank, the report displays data from all employee pay sequences regardless of paid date, including any unpaid sequences.
You can use this report to validate regular remittance amounts as frequently as each pay period, or as infrequently as quarterly, for instance.
