# PR Canada PD7A Information

You can use the PR Canada PD7A Information report to assist users in completing form PD7A as required by the Canada Revenue Agency by selecting Payroll > Reports > PR Canada PD7A Information.

This report displays payroll data for a specified PR company, Payroll Program Account, and Beginning Paid Date and Ending Paid Date range, inclusively. The time period between the beginning/ending paid dates is known as the "remitting period".
Each CPP, CPP2, and EI input field is intended for a single EDL Code representing the appropriate Deduction or Liability.
The Tax DLCodes to Include field is intended for one or more comma-separated EDL Codes representing Deductions (or Liabilities) whose amounts are to be summed up in the "Tax deductions" line on the report. This is typically used for income tax deductions from employee pay. If more than one EDL Code value is specified in this input field, values must be separated by commas with no intervening space characters (as in: "1,2,3").
The "No. of employees" line on the report displays a count of distinct employees who were paid for true earnings for the last pay period related to the remitting period.
