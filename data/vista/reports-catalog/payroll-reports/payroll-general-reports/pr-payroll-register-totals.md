# PR Payroll Register Totals

You can use the PR Payroll Register Totals report to print
 the grand totals on the PR Register by selecting Payroll > Reports > PR Payroll Register Totals.
This report prints only the grand totals on the PR Register. It can be run for a range of pay periods and a range of payroll groups.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning PR Group
Click the Field Lookup
 button or press F4 to select the beginning PR group.

Ending PR Group
Click the Field Lookup
 button or press F4 to select the ending PR group.

Beginning PR Ending Date
Select the Field Lookup button or press F4 to select the beginning pay period end date.

Ending PR Ending Date
Select the Field Lookup button or press F4 to select the ending pay period end date.
