# PR Hours Exception Report

You can use the PR Hours Exception Report to list
 employees where their total posted hours for the selected pay period are not equal to the
 standard hours entered in PR Pay Period Control by selecting Payroll > Reports > PR Hours Exception Report.
The PR Hours Exception Report only lists employees where their total posted hours for the selected pay period are not equal to the standard hours entered in PR Pay Period Control.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

PR Group (Blank for All)

PR End Date (Required)
Click the Field
