# PR Employee Leave History Detail

You can use the PR Employee Leave History Detail report to print details for leave accrual and usage posted from PR Leave Entry batches by selecting Payroll > Reports > PR Employee Leave History Detail.
This report prints details for leave accrual and usage posted from PR Leave Entry batches. You may sort by Employee Name or Number, and then by either Month/Leave Code or Leave Code/Month. The Current Carry Over column shows the potential carry over (which is the available balance up to the carry over limit). This column is not impacted by the Date Range report parameters.
 The Date Range Available Balance also shows the difference between accrued, balance adjustment (from resets) and taken within the selected date range. If this Date Range Available Balance exceeds the maximum available balance allowed (from the leave code setup or the employee leave override), which may occur if the selected time frame overlaps a year, the Date Range Available Balance displays the maximum available balance from the set up tables. In order for the report to calculate correctly, the PR Leave Code Limit and/or PR Employee Leave Limit Override should be set.
