# PR Craft Template Jobs

You can use the PR Craft Template Jobs report to list by
 PR Craft Template the jobs in the JC Job Master that are assigned to the craft template by
 selecting Payroll > Reports > PR Craft Template Jobs.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

PR Craft Template (Blank for All)
Click the Field Lookup
 button or press F4 to select the craft template or leave blank for
 all.

Open (O), Closed (C) or All (A) Jobs
Enter O, C, or A.
