# PR Routine List

You can use the PR Routine list report to List of all the
 tax routines for payroll taxes by selecting Payroll > Reports > PR Routine List.
This report provides a list of all the tax routines for payroll taxes.
 This includes any special program required to calculate State and Local Taxes.

Company
