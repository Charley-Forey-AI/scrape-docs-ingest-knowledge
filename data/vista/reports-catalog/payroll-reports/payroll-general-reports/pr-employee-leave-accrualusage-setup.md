# PR Employee Leave Accrual/Usage Setup

You can use the PR Employee Leave Accrual/Usage Setup
 report to list each employee and the leave codes set up to accumulate leave information by
 selecting Payroll > Reports > PR Employee Leave Accrual/Usage
 Setup.
PR Employee Leave Accrual/Usage Setup lists each employee and the leave codes set up to accumulate leave information. Leave information reported will be from the Leave Code unless there is an employee level override. Included information is Employee Number, Employee Name, Leave Code, Earning Code, Accrual Basis and Rate, and Usage Basis and Rate.
