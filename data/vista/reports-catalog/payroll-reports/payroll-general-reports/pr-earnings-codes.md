# PR Earnings Codes

You can use the PR Earnings Codes report for a list of PR
 Earnings Codes by selecting Payroll > Reports > PR Earnings Codes.
This report provides a list of PR Earnings Codes. This report contains
 a subreport that will print Deduction and Liability Codes that are subject to this
 Earnings Code, when requested at runtime.
