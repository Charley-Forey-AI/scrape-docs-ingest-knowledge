# PR State Information

You can use the PR State Information report for a list of
 State Codes in payroll and the Income Tax and Unemployment information for each by selecting
 Payroll > Reports > PR State Information.
This report provides a list of State Codes in payroll and the Income
 Tax and Unemployment information for each. Notes may be printed on request.
