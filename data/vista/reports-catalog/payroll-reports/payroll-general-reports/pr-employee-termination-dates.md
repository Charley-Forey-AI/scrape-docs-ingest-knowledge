# PR Employee Termination Dates

You can use the PR Employee Termination Dates report by
 selecting Payroll > Reports > PR Employee Termination Dates.
The PR Employee Termination Dates report lists terminated employees for a range of Termination Dates and provides the option of sorting by Employee Number or Sort Name. This report can be run for a single PR Group, or for all PR Groups by leaving the input blank.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

PR Group (Blank for All)
Select the Field Lookup button or press F4 to select the PR group, or leave blank for all.

Beginning Employee Number
Select the Field Lookup button or press F4 to select the beginning employee number.

Ending Employee Number
Select the Field Lookup button or press F4 to select the ending employee number.

Sort By Employee (S)ort Name or (N)umber
Enter S or N.

Beginning Termination Date
Click the Field
 Lookup button or press F4 to select
 the beginning termination date.

Ending Termination Date
Click the Field
 Lookup button or press F4 to select
 the ending termination date.
