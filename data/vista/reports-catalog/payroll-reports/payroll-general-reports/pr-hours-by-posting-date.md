# PR Hours by Posting Date

You can use the PR Hours by Posting Date report to print
 one line per employee and earn code showing hours in seven separate columns for each day of
 the week by selecting Payroll > Reports > PR Hours by Posting Date.
The PR Hours by Posting Date report prints one line per employee and earn code showing hours in seven separate columns for each day of the week. The date entered at runtime is the date for the first column.
