# PR Departments

You can use the PR Departments report for a list of
 payroll deparments by selecting Payroll > Reports > PR Departments.
This report provides a list of Payroll Departments. This report
 contains subreports for Liability Types and Earnings Types.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Department
Select the Field Lookup button or press F4 to select the beginning department.

Ending Department
Select the Field Lookup button or press F4 to select the ending department.

Print Liability Types?
checkbox to print liability types.

Print Notes?
checkbox to print notes.
