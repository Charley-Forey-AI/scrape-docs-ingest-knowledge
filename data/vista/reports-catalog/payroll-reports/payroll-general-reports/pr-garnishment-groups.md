# PR Garnishment Groups

You can use the PR Garnishment Groups report for a list of
 Payroll Garnishment Groups by selecting Payroll > Reports > PR Garnishment Groups.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Garnishment Group
Click the Field Lookup
 button or press F4 to select the beginning garnishment group.

Ending Garnishment Group
Click the Field Lookup
 button or press F4 to select the ending garnishment group.
