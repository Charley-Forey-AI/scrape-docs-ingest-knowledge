# PR State Insurance Codes

You can use the PR State Insurance Codes report for a list
 of Insurance Codes that are set up for each State (including Deduction/Liability Codes to be
 calculated for the Insurance Code. Notes may be printed on request) by selecting
 Payroll > Reports > PR State Insurance Codes.
List of Insurance Codes that are set up for each State, including Deduction/Liability Codes to be calculated for the Insurance Code. Notes may be printed on request.
