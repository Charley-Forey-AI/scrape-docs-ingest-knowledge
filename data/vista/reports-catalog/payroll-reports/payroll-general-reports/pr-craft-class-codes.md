# PR Craft Class Codes

You can use the PR Craft Class Codes report for a list of
 the Standard Craft Class Codes by selecting Payroll > Reports > PR Craft Class Codes.
This report provides a listing the Standard Craft Class Codes. There are several sections of the
 form when Craft Classes are entered, and each of these sections are in a
 "subreport." There are selections for beginning and ending Craft and Class as well
 as choices for whether to include Pay Rates, Variable Earnings, Add-on Earnings, and
 Deduction and Liability Codes.
