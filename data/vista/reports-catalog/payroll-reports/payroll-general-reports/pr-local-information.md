# PR Local Information

You can use the PR Local Information report for PR Local
 set-up Information, including Deduction/Liability Codes to be calculated for the Local Code
 by selecting Payroll > Reports > PR Local Information.
This report provides PR Local set-up Information, including
 Deduction/Liability Codes to be calculated for the Local Code. Notes may be printed on
 request.
