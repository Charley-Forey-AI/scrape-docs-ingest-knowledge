# PR Employee Leave Codes

You can use the PR Employee Leave Codes report to display the set-up of employee leave codes including their earning codes by selecting Payroll > Reports > PR Certified Export Template - eMars.
This report displays the set-up of employee leave codes including their earning codes. It provides detail information on the employee's initial code set-up and subsequent overrides of codes. Leave codes not set up with Accrual and Usage information will not be displayed.
