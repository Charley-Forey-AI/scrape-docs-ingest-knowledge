# PR Employee Deduction/Liability Override

You can use the PR Employee Deduction/Liability Override
 report to display settings saved in program PR Employee Dedns/Liabs by selecting
 Payroll > Reports > PR Employee Deduction/Liability
 Override.
This report displays settings saved in program PR Employee Dedns/Liabs. (These settings may also be viewed and saved in program PR Employees, on the Dedns/Liabs tab.)
The records are ordered first by Employee Sort name, then by Dedn/Liab Code.
Optional inputs allow the user to restrict the results to active employee’s only, "Employee-Based" deductions/liabilities only, or "Eligible for Arrears" deductions only. Any number of optional restrictions may be applied simultaneously. Additionally, there is an option to print Notes associated with the record, if any.
Report Parameters
Description

Select the Field Lookup button or press F4 to select the ending employee sort name.

checkbox to include inactive employees.

checkbox to only print “employee-based” deductions/liabilities.
