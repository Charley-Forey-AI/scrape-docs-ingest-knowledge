# PR Insurance Report by Job/WO

You can use the PR Insurance Report by Job/WO by selecting
 Payroll > Reports > PR Insurance Report by Job/WO.
The PR Insurance Report by Job/WO report prints insurance liability amounts by Job or SM Work Order, Insurance State, Insurance Code, Liability Code with one line per employee and phase. The Gross Amount includes all timecard amounts posted to the Insurance Code plus add-on earnings. The STE column is the straight time equivalent portion of timecard wages plus add-on earnings.
Time coded to SM Work Orders follow time coded to JC Jobs. Time coded to both appear under the JC Job, and can be broken out into Work Order groups underneath the job, or summarized in the job, based on the parameter "Break out Job Work Order time in separate groups".
