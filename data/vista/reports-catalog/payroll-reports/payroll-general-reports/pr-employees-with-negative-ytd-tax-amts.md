# PR Employees with Negative YTD Tax Amts

You can use the PR Employees with Negative YTD Tax Amts
 report by selecting Payroll > Reports > PR Employees with Negative YTD Tax
 Amts.
This report will return Employees who have negative tax deduction amounts which will cause an error during W-2 preparation. Only deduction codes with the Calculation Category of ‘F’, 'S' or 'L' will be included. The user may input a code to exclude, like EIC, so it would not display on the report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Payroll Year Ending Month (Required)
Enter payroll ending month.

Deduction Code to exclude (i.e. EIC)
Click the Field Lookup
 button or press F4 to select the deduction code to exclude.
