# PR Report of Earnings

You can use the PR Report of Earnings report for a list of
 hours and wages for a selected company and earning code by selecting Payroll > Reports > PR Report of Earnings.
This report provides a list of hours and wages for a selected company
 and earning code. This would allow them to print the amounts for SEC125 or 401K earnings
 by range of months.
