# PR Pay Period Control

You can use the PR Pay Period Control report for a list by
 Company, Group and Pay period ending date of the set up in Pay period control by selecting
 Payroll > Reports > PR Pay Period Control.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

PR Group (Blank for All)
Select the Field Lookup button or press F4 to select the PR group, or leave blank for all.

Beginning Payroll Ending Date
Click the Field Lookup
 button or press F4 to select the beginning pay period end date.

Ending Payroll Ending Date
Click the Field Lookup
 button or press F4 to select the ending pay period end date.

Print (O)pen, (C)losed, or (B)oth
Enter O, C, or B.
