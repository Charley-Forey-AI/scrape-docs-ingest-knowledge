# PR Craft Report – Full Detail

You can use the PR Craft Report – Full Detail report to
 summarize Craft earnings, deductions, and liabilities by code for a range of pay period end
 dates by selecting Payroll > Reports > PR Craft Report – Full Detail.
This report summarizes Craft earnings, deductions, and liabilities by
 code for a range of pay period end dates, however, the report design accommodates
 monthly reporting best. The report, sorted by Craft, Employee (name or number based on
 input), and Class, includes the following columns. Code Description, Hours, Basis,
 Eligible (for deductions and liabilities), and Calculated Amount. The source of the data
 comes from the PRCA table (PR Craft Accumulations), which the software updates after
 running the Payroll Processing program and contains any overrides that were done.
