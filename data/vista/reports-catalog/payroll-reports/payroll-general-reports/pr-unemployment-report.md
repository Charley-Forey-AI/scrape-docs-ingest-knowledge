# PR Unemployment Report

You can use the PR Unemployment Report report for
 information specific to each state's needs as set up in the State Information form by
 selecting Payroll > Reports > PR Unemployment Report.
The Unemployment report provides information specific to each state's needs as set up in the State Information form. This allows the user to include only the information required by the individual State, such as hours or weeks worked, year-to-date or period gross pay, amounts under limits, and liability amounts. The report may be used as the backup sheet detailing each employee's earnings and attached to the summary form provided by the State. The report may be run for a specific State or, if the State field is left blank, then all states will be printed.
The reporting period option "Current Month" defines the month data will be reported through. For example, if the month of October is entered in this option and the "Quarter" reporting option is selected, only data through October will be reported on, even though data may exist for November and December.
A reporting option for monthly, quarterly or annually is available. If the monthly or quarterly option is selected, the total year to date employee count will be displayed along with employee count for the specific reporting period (Monthly or Quarterly).
