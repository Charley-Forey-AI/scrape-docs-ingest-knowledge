# PR Company Parameters

You can use the PR Company Parameters report for a list of
 company setups by selecting Payroll > Reports > PR Company Parameters.
This report provides a list of Company setups. If 0 is entered for the company, then all
 companies will print.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.
