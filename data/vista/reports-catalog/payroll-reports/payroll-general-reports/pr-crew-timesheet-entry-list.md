# PR Crew Timesheet Entry List

You can use the PR Crew Timesheet Entry List report to
 list entries made in the PR Crew Time Sheet batch and can be printed prior to interfacing
 with payroll by selecting Payroll > Reports > PR Crew Timesheet Entry List.
The report will list entries made in the PR Crew Time Sheet batch and can be printed prior to interfacing with payroll. The report also provides the option for including/excluding the data entered on the Equipment Usage, Other Job Earnings, and Non-Job Earnings tabs in PR Crew Timesheet Entry.
