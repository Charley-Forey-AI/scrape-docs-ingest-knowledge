# PR to JC Update Report

You can use the PR to JC Update Report report to print the postings from Payroll (by
 Job, Phase and Cost Type) that have been updated to Job Cost by
 selecting Payroll > Reports > PR to JC Update Report.
This report prints by Job, Phase and Cost Type the postings from Payroll that have been updated to Job Cost. The report may be printed at any time after the PR Ledger process has been run for a PR End Date.
Parameters

Company
Accept the default, or press F4 to select a company.

Beginning JCCo
Select the Field Lookup button or press F4 to select the beginning job cost company.

Ending JCCo
Select the Field Lookup button or press F4 to select ending job cost company or leave blank to select all.

Beginning Job
Select the Field Lookup button or press F4 to select the beginning job.

Ending Job
Select the Field Lookup button or press F4 to select the ending job.

Beginning PR Ending Date
Select the Field Lookup button or press F4 to select the beginning pay period end date.

Ending PR Ending Date
Select the Field Lookup button or press F4 to select the ending pay period end date.

Print Cost Type Recap Only?
checkbox to only print cost type recap.
