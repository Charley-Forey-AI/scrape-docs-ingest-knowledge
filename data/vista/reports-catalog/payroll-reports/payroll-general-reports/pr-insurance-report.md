# PR Insurance Report

You can use the PR Insurance Report report, which provides
 the information required to complete Worker's Compensation reports for each State, by
 selecting Payroll > Reports > PR Certified Export Template -
 eMars.
The PR Insurance Report provides the information required to complete Worker's Compensation reports for each State. The report sorts by State, Insurance Code, and Deduction/Liability Code (DL Code). The report will also print a separate line per rate for each DL Code. The Gross Amount column excludes earnings that are flagged as Subject Only against the insurance DL Code whereas the Subject column includes all earnings based on the DL Code (see grid in PR Earnings Codes). The Basis column is the basis of earnings, such as straight time equivalent, hours, days, etc. used for calculating the DL Code amount. The Eligible column represents the Basis up to the DL Code limit, which the Payroll Processing program multiplies by the DL Code rate. The Payroll Processing programs calculate and stores the amounts in the columns described above in the PRIA (PR Insurance Accumulations) table used by this report.
