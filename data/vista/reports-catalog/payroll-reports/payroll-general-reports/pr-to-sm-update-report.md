# PR to SM Update Report

You can use the PR to SM Update Report report to print the
 postings from Payroll that have been updated to Service Management by Work Order, Scope, and
 SM Cost Type by selecting Payroll > Reports > PR to SM Update Report.
This report prints the postings from Payroll that have been updated to Service Management by Work Order, Scope, and SM Cost Type. The report may be printed at any time after the PR End Date has been closed and PR Ledger Update process was run.
