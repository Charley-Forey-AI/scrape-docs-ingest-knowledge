# PR Standard Craft Codes

You can use the PR Standard Craft Codes report for a
 listing of the standard Craft Codes by selecting Payroll > Reports > PR Standard Craft Codes.
This report provides a listing of the standard Craft Codes. With this
 report comes the option to print Earnings, Deductions and Liabilities, Capped Codes, and
 Capped Basis.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Craft Code
Select the Field Lookup button or press F4 to select the beginning craft.

Ending Craft Code
Select the Field Lookup button or press F4 to select the ending craft.

Include Earnings, Deductions, Liabilities?
checkbox to include earnings, deductions, and liabilities.

Include Capped Codes?
checkbox to include capped codes.

Included Capped Basis?
checkbox to include capped basis.
