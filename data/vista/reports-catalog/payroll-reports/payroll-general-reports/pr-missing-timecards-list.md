# PR Missing Timecards List

You can use the PR Missing Timecards List report by
 selecting Payroll > Reports > PR Missing Timecards List.
This report compares PRTH with PREH and lists any employees that are
 active that do not have any timecard entries in a selected PR End Date. Shows PR Dept
 assigned to, craft, and last job to help identify where to run down missing
 timecards.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

PR Group (Blank for All)
Select the Field Lookup button or press F4 to select the PR group, or leave blank for all.

PR End Date (Required)
Select the Field Lookup button or press F4 to select the ending pay period end date.

Sort by Employee (S)ort Name or (N)umber
Enter S or N.
