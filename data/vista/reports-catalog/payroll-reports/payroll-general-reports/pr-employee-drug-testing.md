# PR Employee Drug Testing

You can use the PR Employee Drug Testing report to list
 employees to be tested by selecting Payroll > Reports > PR Employee Drug Testing.
This report lists employees to be tested. The report uses a special random number function that assigns each employee a number. When this report is run, a number is entered. One out of every X (the # entered) Employees will be tested. If you need to test 25% of the Employees, enter 4 (1 in every 4 employees). Only those employees to be tested appear on the report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

# of Employees to print on report
Enter number of employees to be tested. If you need to test 25% of the Employees, enter 4 (1 in every 4 employees).

Include Inactive?
checkbox to include inactive employees.
