# PR Crew Timesheet Flash Report

You can use the PR Crew Timesheet Flash Report report to
 summarize Progress, hours and usage posted in PR Timesheet entry and compares that to the
 estimates posted in Job Cost by selecting Payroll > Reports > PR Crew Timesheet Flash Report.
The report summarizes Progress, hours and usage posted in PR Timesheet entry and compares that to the estimates posted in Job Cost. Estimated units only include the units on cost types where the phase unit flag is checked in JC Job Phases.
The report compares expected amounts against the actual amounts posted in PR Timesheets. Expected amounts are calculated based on an estimated cost/unit multiplied by the progress units posted in PR Crew Timesheet entry. Actual amounts are the total hours posted in PR Crew Timesheets multiplied by the pay rates. There is also an input parameter to apply a burden rate against the actual costs. If a rate is entered it will increase the timesheet amounts by the burden rate.
The report does not include timecards entered directly through PR Timecard entry.
