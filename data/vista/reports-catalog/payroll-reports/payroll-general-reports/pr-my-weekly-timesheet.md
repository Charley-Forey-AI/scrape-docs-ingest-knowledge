# PR My Weekly Timesheet

You can use the PR My Weekly Timesheet report by selecting
 Payroll > Reports > PR My Weekly Timesheet.
The PR My Weekly Timesheet reports provides a user the ability to
 print timesheets they have entered for validation and/or record keeping processes. The
 user may select a particular timesheet or all the timesheets they entered for a specific
 start date. When selected, the "Print Notes" option will allow notes associated with the
 timesheet to be printed. The user's logon is used to identify timesheets they have
 entered.
When the "Print Blank Timesheet" option is selected a blank timesheet will be generated base on the Start Date entered regardless of other parameters selected.
