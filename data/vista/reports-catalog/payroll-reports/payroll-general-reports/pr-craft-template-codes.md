# PR Craft Template Codes

You can use the PR Craft Template Codes report by
 selecting Payroll > Reports > PR Craft Template Codes.
PR Craft Template Codes. Options exist to include or exclude Earnings, Deduction, and Liability Codes; Job Craft Items; and Craft/template-level notes.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Craft
Select the Field Lookup button or press F4 to select the beginning craft.

Ending Craft
Select the Field Lookup button or press F4 to select the ending craft.

Beginning Template
Select the Field Lookup button or press F4 to select the beginning template or leave blank for all.

Ending Template
Select the Field Lookup button or press F4 to select the ending template or leave blank for all.

Include Earning, Deduction Liability Details?
checkbox to include earnings, deductions, and liability details.

Include Job Craft Items?
checkbox to include job craft items.

Print Craft Template Notes?
checkbox to print craft template notes.
