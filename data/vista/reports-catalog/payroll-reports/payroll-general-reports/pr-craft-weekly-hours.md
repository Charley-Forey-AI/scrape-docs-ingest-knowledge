# PR Craft Weekly Hours

You can use the PR Craft Weekly Hours report by selecting
 Payroll > Reports > PR Craft Weekly Hours.
The PR Craft Weekly Hours report provides a way to compare hours for up to five different weeks shown in separate columns. The report further breaks down craft hours for straight time, overtime, and double-time. The report uses the PR Craft Accumulation table as the source of information, which is updated by the Payroll Processing program.
