# PR Craft Class Template Hierarchy

You can use the PR Craft Class Template Hierarchy report
 to display how specific crafts and classes are paid with and without associated templates by
 selecting Payroll > Reports > PR Craft Class Template
 Hierarchy.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Craft
Select the Field Lookup button or press F4 to select the beginning craft.

Ending Craft
Select the Field Lookup button or press F4 to select the ending craft.

Beginning Class
Click the Field
 Lookup button or press F4 to select
 the beginning class.

Ending Class
Click the Field
 Lookup button or press F4 to select
 the ending class.

Beginning Template
Select the Field Lookup button or press F4 to select beginning template. Leave blank to include all templates.

Ending Template
Select the Field Lookup button or press F4 to select ending template. Leave blank to include all templates.

Template Sort
checkbox to sort by template.
