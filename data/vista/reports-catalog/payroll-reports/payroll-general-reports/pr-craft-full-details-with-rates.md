# PR Craft Full Details With Rates

You can use the PR Craft Full Details With Rates report to
 summarize Craft earnings, deductions, and liabilities by code and rate for a range of pay
 period end dates by selecting Payroll > Reports > PR Craft Full Details With
 Rates.
This report summarizes Craft earnings, deductions, and liabilities by code and rate for a range
 of pay period end dates. The report, sorted by Craft, Employee (name or number based
 on input), and Class, includes the following columns. Code Description, Hours, Rate,
 Eligible (for deductions and liabilities), and Calculated Amount. The source of the
 data is the PRCX table (PR Craft Accum Rate Detail), which is updated using the
 current rate during the Payroll Processing Program and does not contain
 overrides.
