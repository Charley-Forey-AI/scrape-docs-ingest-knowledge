# PR Job/WO Cost Report

You can use the PR Job/WO Cost Report report to print earning
 and liability amounts from Payroll sorted by JC Job and SM Work Order for a range of Payroll
 Ending Dates by selecting Payroll > Reports > PR Job/WO Cost Report.
What's displayed are actual amounts as calculated in Payroll. Therefore, if using JC Liability Templates that are based on rates rather than exact, and/or using Fixed Rates, amounts charged to Job Cost will differ from this report.
All information is available for this report after payroll is processed and is not based on the interface to Job Cost or Service Management. This report may be used to report actuals which might then be used to compare to burden rates charged to Jobs and Work Orders.
