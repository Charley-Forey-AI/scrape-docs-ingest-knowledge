# PR Employees List

You can use the PR Employees List to list detail payroll
 information for an employee by selecting Payroll > Reports > PR Employees List.
This report lists detail payroll information for an employee. If the employee's filing status, deductions, liabilities and auto earnings are not set up they will not be displayed. Employee leave information will not be displayed on this report.
