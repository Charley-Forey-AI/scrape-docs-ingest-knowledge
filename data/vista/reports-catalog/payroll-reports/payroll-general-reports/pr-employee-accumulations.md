# PR Employee Accumulations

You can use the PR Employee Accumulations report by
 selecting Payroll > Reports > PR Employee Accumulations.
The Employee Accumulations are used to see Employee Month to Date, Quarter to Date and Year to Date information for each Earnings, Deduction, Liability, and Unemployment Code updated from payroll. There is also an option to run the report for Earnings, Deductions or Liabilities only. This option will also show employee detail or Grand Totals Only. A selection for monthly totals is included, which will print the code amounts for each month updated for the runtime requested periods. The report may also be printed for grand totals only which will not print employee information, but rather grand totals for each Earnings, Deduction, Liability, and Unemployment Code. The report may be printed in alphabetical order by Employee sort name or numeric order by Employee number. A range of Employee number and/or Employee Sort names may be input.
