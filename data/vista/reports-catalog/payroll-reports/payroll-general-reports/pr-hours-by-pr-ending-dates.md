# PR Hours by PR Ending Dates

You can use the PR Hours by Period Ending Dates report to
 print total hours worked for a range of PR Ending Dates by selecting Payroll > Reports > PR Certified Export Template -
 eMars.
This report can be sorted by:

- Employee

- Job

- Job Co

- SM Work Order

- SM Co

And secondarily sorted by:

- Employee

- Job

- SM Work Order

- None (Leave the field blank. The report will simply summarize the first group.)

At the end of the report, a Grand Total and Employee Count is displayed for the PR Company.
