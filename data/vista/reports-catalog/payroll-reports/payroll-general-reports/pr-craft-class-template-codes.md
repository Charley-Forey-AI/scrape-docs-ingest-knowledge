# PR Craft Class Template Codes

You can use the PR Craft Class Template Codes report by
 selecting Payroll > Reports > PR Craft Class Template Codes.
PR Craft Class Template Codes includes static information about the Craft Class Templates as well as Pay Rate, Variable Earnings, Add-on Earnings, and Deduction and Liability details. An option is given to include Pay Rates, Variable Earnings, Add-on Earnings, and Deduction and Liability details. Subreports are used on this report.
