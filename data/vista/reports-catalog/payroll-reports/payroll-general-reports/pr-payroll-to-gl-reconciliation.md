# PR Payroll to GL Reconciliation

You can use the PR Payroll to GL Reconciliation report by
 selecting Payroll > Reports > PR Payroll to GL
 Reconciliation.
The PR Positive Pay Template report is designed to create a positive
 pay electronic file for transmission to a bank. After the report has been run, export it
 to an electronic file for transmission to a bank.
This report is only a template and a custom version of it will need to be created with the specific bank’s data format.
