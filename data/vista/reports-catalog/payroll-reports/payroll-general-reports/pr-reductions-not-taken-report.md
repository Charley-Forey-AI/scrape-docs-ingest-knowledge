# PR Reductions Not Taken Report

You can use the PR Reductions Not Taken Report report to
 determine which amount based deductions and salary reductions (negative earnings) were
 missed from active employees for a range of payroll periods by selecting Payroll > Reports > PR Reductions Not Taken Report.
This report may be run to determine which amount based deductions and salary reductions (negative earnings) were missed from active employees for a range of payroll periods. It may be used to determine what arrearages should be taken in future payrolls. Earnings will be reported if set up as automatic earnings. Deductions must be set up in PR Employee Deduction Overrides with 'Calculations' checked for either Override standard Rate/Amt or Override with a fixed Amount. They will be included if no payroll was processed for the employee for the pay periods specified and the frequency codes used for the earnings/deductions are set up in the Pay Period Control.
