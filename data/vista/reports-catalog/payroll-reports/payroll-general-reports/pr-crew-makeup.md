# PR Crew Makeup

You can use the PR Crew Makeup report for a list of Crew
 Codes and their members by selecting Payroll > Reports > PR Crew Makeup.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Crew Code
Click the Field
 Lookup button or press F4 to select
 the beginning crew code.

Ending Crew Code
Click the Field
 Lookup button or press F4 to select
 the ending crew code.
