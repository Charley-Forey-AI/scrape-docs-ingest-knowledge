# PR Craft Accumulations

You can use the PR Craft Accumulations report to print
 each Craft and Class earning, deduction, and liability amount, including basis and eligible,
 updated after running Payroll Processing by selecting Payroll > Reports > PR Craft Accumulations.
The PR Craft Accumulations report prints each Craft and Class earning, deduction, and liability amount, including basis and eligible, updated after running Payroll Processing. The report sorts by Employee, Pay Period End Date, Type of Code (Earning, Deduction, Liability), and Craft. The basis column includes the appropriate subject amount, such as hours or dollars, depending on the method of calculation for the earning, deduction, or liability code (EDL Code). The eligible column reports the portion of the basis column up to the limit for the EDL Code.
