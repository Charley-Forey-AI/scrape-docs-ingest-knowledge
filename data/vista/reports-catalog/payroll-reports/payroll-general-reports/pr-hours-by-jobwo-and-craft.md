# PR Hours by Job/WO and Craft

You can use the PR Hours by Job/WO and Craft report to
 summarizes hours by Job, SM Work Order, Employee, and Craft by selecting Payroll > Reports > PR Hours by Job/WO and Craft.
This list summarizes hours by Job, SM Work Order, Employee, and Craft. The hours are displayed by Regular, Overtime, and Double Time as determined by the Earn Code used in time card entry. The report may be run for a range of PR End Dates, Jobs, SM Work Orders, or Crafts.
The "Show time for (J)obs, (S)ervice, or (B)oth" parameter tells the report whether to show only Job-related time (time applied only to a JC Job), Service-related time (time applied only to an SM Work Order), or both.
For time that is entered to both a Job and SM Work Order, the "Break out Job Work Order time in separate groups" parameter controls whether that time is in separate Work Order groups underneath the Job, or if that time is grouped within the one Job group.
