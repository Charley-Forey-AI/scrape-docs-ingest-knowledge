# PR Labels

You can use the PR Labels report PR Labels to print labels
 for each selected employee by selecting Payroll > Reports > PR Certified Export Template -
 eMars.
Prints labels for each selected employee. Report input parameters include beginning and ending group an option for whether only active employees should be printed and an additional option for sorting alpha or numeric. The label is approximately 1" x 2 5/8". There are 3 labels wide and 10 labels down on each page.
Report Parameters
Description
