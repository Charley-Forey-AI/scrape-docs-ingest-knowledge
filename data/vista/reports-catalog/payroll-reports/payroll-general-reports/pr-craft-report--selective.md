# PR Craft Report – Selective

You can use the PR Craft Report – Selective report by
 selecting Payroll > Reports > PR Craft Report – Selective.
The PR Craft Report - Selective data comes from the PRCX table (tracks RATE detail for Craft
 Accumulations), which is updated using the current rate during the Payroll
 Processing Program and DOES NOT contain overrides. The report shows an asterisk
 beside amounts that have been overridden in the PRSQ table (tracks Pay status and
 employee processing detail including overrides). The report prints in the same
 format as the other PR Craft Detail reports, but provides additional flexibility on
 how the information is printed. The input parameters allow summarization by
 earnings, deductions and liabilities by code or Craft (Totals Only). Additionally,
 the report can summarize at the Class level by answering "Y" to the Include Class
 Info parameter.
