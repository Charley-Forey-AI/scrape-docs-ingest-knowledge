# PR Leave Codes

You can use the PR Leave Codes report for a list of Leave
 Codes that are set up by selecting Payroll > Reports > PR Leave Codes.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Leave Code
Click the Field
 Lookup button or press F4 to select
 the beginning leave code.

Ending Leave Code
Click the Field
 Lookup button or press F4 to select
 the ending leave code.
