# PR Overtime Schedules

You can use the PR Overtime Schedules report for a list of
 each overtime schedule set up in PR showing regular hours and OT earnings code by selecting
 Payroll > Reports > PR Overtime Schedules.
This report lists each overtime schedule set up in PR showing regular
 hours and OT earnings code. Table is PROT.
