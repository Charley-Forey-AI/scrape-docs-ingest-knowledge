# PR Monthly Wage Report for Illinois (IDES)

You can use the PR Monthly Wage Report for Illinois (IDES)
 report by selecting Payroll > Reports > PR Monthly Wage Report for Illinois
 (IDES).
Employers that are required to submit their quarterly contribution and wage reports electronically to the Illinois Department of Employment Security (IDES) must also submit eight additional monthly wage reports annually beginning January 2013, per Public Act 097-0689, the Medicaid Reform "SMART" Act.
This Vista report enables you to comply with that regulatory requirement.
The Company and Month parameters on the report launcher are required. A third parameter provides an option to display column headings in the report.
You may wish to display column headings when you use the report to preview the contents of your e-file submission for IDES.
When you use the report to create the e-file itself, you should opt not to print column headings. Use the Export button on the report launcher or report viewer, specifying "Excel Format Data Only (.xls)" for the Export Type. Open the exported file in Excel, then save it as a "CSV (Comma delimited)" file. As a comma-separated (.csv) plain ASCII text format file, your file now conforms to the IDES TaxNet system's e-file requirements.
As a final step before submitting your e-file, you may wish to rename the file. TaxNet recommends that you include the month, year, and a sequence number in the file name to assist in tracking submissions. Specifically, TaxNet recommends that you use the following naming convention:
ILUIMonthlyWageMMYYYY_NN.csv -- where MM is the two-digit month, YYYY is the four-digit year, and NN is the two-digit file sequence.
Here is a sample file name per this convention. ILUIMonthlyWage012013_01.csv
Note that the TaxNet system treats the file name as case insensitive.
