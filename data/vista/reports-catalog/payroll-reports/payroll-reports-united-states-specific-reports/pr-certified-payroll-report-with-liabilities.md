# PR Certified Payroll Report with Liabilities

You can use the PR Certified Payroll Report with
 Liabilities to print earnings and deductions in the same format as the PR Certified Report
 w/ Class information, and also includes liability information, by selecting Payroll > Reports > PR Certified Payroll Report with
 Liabilities.
The PR Certified Report with Liabilities prints earnings and deductions in the same format as the PR Certified Report w/ Class information, but also includes liability information. See the PR Certified Report for further explanation of fields and parameters used on the report.
