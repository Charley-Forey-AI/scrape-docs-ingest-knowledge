# PR New Hire Report by Check Date

You can use the PR New Hire Report by Check Date report by
 selecting Payroll > Reports > PR Certified Export Template -
 eMars.
This report is similar to the PR New Hire Report by Hire Date except it only displays employees that have been paid after the cutoff date.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Check Dates For The Period
Enter beginning check date for the period.

Ending Check Dates For The Period
Enter ending check date for the period.
