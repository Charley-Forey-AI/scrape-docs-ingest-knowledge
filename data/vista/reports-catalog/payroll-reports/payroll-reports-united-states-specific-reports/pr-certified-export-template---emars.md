# PR Certified Export Template - eMars

You can use the PR Certified Export Template - eMars
 report to export payroll certified information suitable for uploading to eMars by selecting
 Payroll > Reports > PR Certified Export Template -
 eMars.
PR Certified Export Template - eMars is a Crystal report to export payroll certified information suitable for uploading to eMars.
To export to Excel use the Export Option - Excel Format Data Only (.xlsx)
