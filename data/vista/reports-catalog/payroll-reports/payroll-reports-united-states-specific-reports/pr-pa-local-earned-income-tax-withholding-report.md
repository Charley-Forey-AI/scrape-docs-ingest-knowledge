# PR PA Local Earned Income Tax Withholding Report

You can use the PR PA Local Earned Income Tax Withholding
 Report by selecting Payroll > Reports > PR PA Local Earned Income Tax Withholding
 Report.
The Employer Quarterly Return - PA Local Earned Income Tax Withholding report has required input fields for PR Company, tax year, tax quarter, and local code. The remaining fields are required entries to meet the specifications for quarterly reporting in Pennsylvania. Please note. PO Boxes are not advised for employer address fields.
The Credit or Adjustment Amount and Total Payments Made This Quarter (dollar amount fields) and Penalty or Interest Percent values should be entered prior to running the report to meet the information requirements for local tax reporting.
Employees will only be included if there is a local tax code for the employee in PR Employees.
If there is no data to display for a given local code, the report will return an empty form.
