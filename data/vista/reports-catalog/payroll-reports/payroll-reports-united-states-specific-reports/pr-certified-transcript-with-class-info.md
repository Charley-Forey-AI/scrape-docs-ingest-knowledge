# PR Certified Transcript With Class Info

You can use the PR Certified Transcript With Class Info
 report by selecting Payroll > Reports > PR Certified Transcript With Class
 Info.
The PR Certified Transcript With Class Info shows the same format and information as the PR Certified Report, but prints additional earning details for each craft/class in which the employee worked. See the PR Certified Report for further explanation of fields and parameters used on the report.
