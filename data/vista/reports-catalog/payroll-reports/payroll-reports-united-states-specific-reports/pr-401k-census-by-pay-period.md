# PR 401(k) Census by Pay Period

You can use the PR 401(k) Census by Pay Period Report By
 Material report to print requisitions by material, with the option to sort the data by
 Material Category or by Route (purchase/stock/quote) by selecting Payroll > Reports > PR 401(k) Census by Pay Period.
This report prints the same columns and format as the PR 401(k) Census report, but instead prints hours, wages, employee deferral, and employer match by employee for a range of payroll ending dates. Hours and wages include all earnings codes posted with the subject to auto earnings "Y"(checked) in PR Earnings Codes set up. The employee deferral column totals the amounts from the earnings codes and pre-tax deductions entered at runtime. The employer match column calculates in the same manner for each of the liability codes. The PR Group input restricts the data to one payroll group (this is the PR Group assigned in PREH (PR Employees). If an employee's group assignment is changed within the time frame used as an input parameter for this report, data will only be returned for the current group assigned in PR Employees. PRDT (PR Detail tables) are used for this report. This report now allows comma-delimited input of 5 codes each for Earnings, Pre-Tax and Liability codes.
