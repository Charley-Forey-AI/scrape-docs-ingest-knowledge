# PR W2 Employee Copies Report

You can use the PR W2 Employee Copies report for printing or exporting employee copies of W2 forms by selecting Payroll > Reports > PR W2 Employee Copies Report.
This report provides a print out of a specified employee's W2 for the current tax year.Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Tax Year
Click the Field Lookup button or press F4 to select the earning code.

Copy 2: (S)tate only or (L)ocal and State
 Enter S or L.
Beginning Employee Sort Name
Select the Field Lookup button or press F4 to select the beginning employee sort name.

Ending Employee Sort Name
Select the Field Lookup button or press F4 to select the ending employee sort name.

Include Employees Notified by Email


 Select the check box to force printing of W-2s for employees that have consented to receive W-2s by email. When this box is unchecked, the report will print W-2s only for employees that have declined the option to receive W-2s by email.
Truncate Employee SSNSelect the check box to obscure all but the last four digits of the employee's Social Security Number.
