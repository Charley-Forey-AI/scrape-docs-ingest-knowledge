# PR 940/941 Information

You can use the PR 940/941 Information report to print the
 detail necessary to fill in the Federal Form 940 and 941 by selecting Payroll > Reports > PR 940/941 Information.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Reporting Month
Enter reporting month.

(M)onth, (Q)uarter or (Y)ear Information
Enter M, Q, or Y.

Employee Sort Order (A=Sort Name, N=Numeric)
Enter A or N.

Federal Withholding Code
Select the Field Lookup button or press F4 to select the federal withholding code.

FICA Medicare Withholding Code
Click the Field Lookup
 button or press F4 to select the FICA Medicare withholding
 code.

FICA Medicare Deduction Rate – in decimal format
Enter current Medicare deduction rate in decimal format -- 1.45% = .0145

FICA Social Security Withholding Code
Click the Field Lookup
 button or press F4 to select the FICA social security
 withholding code.

FICA SS Deduction Rate – in decimal format
Enter current social security deduction rate in decimal format -- 6.2% = .062

FICA Medicare Liability Code
Click the Field Lookup
 button or press F4 to select the FICA Medicare Liability
 code.

FICA Medicare Liability Rate – in decimal format
Enter current Medicare liability rate in decimal format -- 1.45% = .0145

FICA Social Security Liability Code
Click the Field Lookup
 button or press F4 to select the FICA social security liability
 code.

FICA SS Liability Rate – in decimal format
Enter current social security liability rate in decimal format -- 6.2% = .062

EIC Deduction Code
Click the Field Lookup
 button or press F4 to select the EIC deduction code.

FUTA Liability Code
Click the Field Lookup
 button or press F4 to select the FUTA Liability code.

FUTA – Code Rate – in decimal format
Enter current FUTA liability rate in decimal format -- .8% = .008

FICA Additional Medicare Code
Click the Field Lookup
 button or press F4 to select the FICA Additional Medicare
