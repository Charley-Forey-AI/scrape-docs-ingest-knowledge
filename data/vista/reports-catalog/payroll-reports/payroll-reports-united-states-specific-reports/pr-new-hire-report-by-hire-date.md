# PR New Hire Report by Hire Date

You can use the PR New Hire Report by Hire Date report to conform with government
 requirements for new hire reporting by selecting Payroll > Reports > PR New Hire Report by Hire Date.
This report was designed to conform with government requirements for new hire reporting. The report lists the last name, first name, Address, social security number, and hire date for employees hired between two specified dates. The report will not list employees who were hired during the specified time but also terminated between those two dates.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning New Hire Date
Enter beginning new hire date.
