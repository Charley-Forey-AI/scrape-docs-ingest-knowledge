# PR W2 Item Codes

You can use the PR W2 Item Codes report to print a list of
 items from PRWC and PRWI which include all the items necessary for printing W2's and the
 Magnetic media text file. (i.e. Federal Wages, Federal Tax etc.) by selecting Payroll > Reports > PR W2 Item Codes.
