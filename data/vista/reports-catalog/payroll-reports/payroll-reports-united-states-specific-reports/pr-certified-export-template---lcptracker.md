# PR Certified Export Template - LCPtracker

You can use the PR Certified Export Template - LCPtracker
 by selecting Payroll > Reports > PR Certified Export Template -
 LCPtracker.
PR Certified Export Template - LCPtracker is a Crystal report to export payroll certified information suitable for uploading to LCPtracker.
To export to Excel use the Export Option - Excel Format Data Only (.xlsx)
