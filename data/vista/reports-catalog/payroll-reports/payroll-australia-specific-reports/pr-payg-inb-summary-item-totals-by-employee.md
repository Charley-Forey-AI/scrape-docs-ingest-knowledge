# PR PAYG INB Summary Item Totals by Employee

You can use the PR PAYG INB Summary Item Totals by Employee
 report by selecting Payroll > Reports > PR PAYG INB Summary Item Totals by
 Employee.
PR PAYG INB Summary Item Totals by Employee report summarizes each employee's ATO Item amounts and Misc. Code amounts by Tax Year to assist with reconciliation with PR Employee Accumulations amounts.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Tax Year
Click the Field Lookup
 button or press F4 to select the tax year.

Employee
Click the Field Lookup
 button or press F4 to select the employee.
