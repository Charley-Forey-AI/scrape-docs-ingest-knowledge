# PR Labels - NZ

You can use the PR Labels - NZ report to print labels for
 each selected employee by selecting Payroll > Reports > PR Labels - NZ.
Prints labels for each selected employee. Report input parameters include beginning and ending PR group, an option for printing only active employees, and an additional option for sorting alpha or numeric. The labels are designed to print on Avery L7159 63.5 x 33.9 mm laser labels. There are 3 labels wide and 10 labels down on each page.
