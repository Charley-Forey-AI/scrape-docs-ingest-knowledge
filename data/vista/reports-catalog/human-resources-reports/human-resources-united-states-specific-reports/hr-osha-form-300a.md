# HR OSHA Form 300A

You can use the HR OSHA Form 300A report for information
 flagged as OSHA type for the calendar year and displays it in the format of the OSHA 300A
 form by selecting Human Resources > Reports > HR OSHA Form 300A.
This report pulls information flagged as OSHA type for the calendar year and displays it in the format of the OSHA 300A form. The report can be printed for all OSHA locations or for one specific location and with an option to override the Establishment Name, Street, City, State and Zip. If no override information is entered, the report will pull the information from HQCO.
Injury/Illness types have been changed on the OSHA 300A report. Therefore, injury/illness type accidents that occurred prior to 1/1/02 will no longer show on this report.
