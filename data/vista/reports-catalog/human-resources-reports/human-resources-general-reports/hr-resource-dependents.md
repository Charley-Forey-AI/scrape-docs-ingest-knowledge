# HR Resource Dependents

You can use the HR Resource Benefits report for a list of
 all the dependents set up for a resource with their static information by selecting
 Human Resources > Reports > HR Resource Benefits.
This report provides a list of all the dependents set up for a resource with
 their static information. This can be ran for a range of HR resources. HRDP.
