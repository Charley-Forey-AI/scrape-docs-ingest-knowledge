# HR Benefit Enrollment By Plan

You can use the HR Benefit Enrollment By Plan report for a
 list of employees and dependents covered by PLAN with the effective date, end date, cost,
 for reconciling to vendors invoice by selecting Human Resources > Reports > HR Benefit Enrollment By Plan.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Benefit Code
Select the Field Lookup button or press F4 to select the beginning benefit code or leave blank for all.

Ending Benefit CodeSelect the Field Lookup button or press F4 to select the ending benefit code or leave blank for all.

Include Inactive?
checkbox to include inactive resources.
