# HR Resource Contacts

You can use the HR Resource Contacts report for a list of
 contacts for each HR resource by selecting Human Resources > Reports > HR Resource Contacts.
This report provides a list of contacts for each HR resource. HREC table.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Resource
Select the Field Lookup button or press F4 to select the beginning resource or leave blank for all.

Ending Resource Select the Field Lookup button or press F4 to select the ending resource or leave blank for all.

Print Notes?checkbox to print notes.

Include Inactive?
checkbox to include inactive resources.
