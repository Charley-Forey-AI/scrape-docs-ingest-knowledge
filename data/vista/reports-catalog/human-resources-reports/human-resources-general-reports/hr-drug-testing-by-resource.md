# HR Drug Testing By Resource

You can use the HR Drug Testing By Resource report for a
 listing of drug testing history by HR resource which can be run by JC Co., Job, and/or
 Resource range by selecting Human Resources > Reports > HR Drug Testing By Resource.
This report provides a listing of drug testing history by HR resource which
 can be run by JC Co., Job, and/or Resource range. It shows test results, action
 taken, and notes from the HRDT table.
