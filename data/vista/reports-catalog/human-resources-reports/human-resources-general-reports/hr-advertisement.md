# HR Advertisement

You can use the HR Advertisement report for the position
 description, salary range, job qualifications, & training required for a position by
 selecting Human Resources > Reports > HR Advertisement.
A sample of a report that can be ran off of the HRPC table for an advertisement. Shows the position description, salary range, job qualifications, & training required. Send inquiries to. Company info from HQCO.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Position Code
Press F4 to select the position code or leave blank for all.

Print Notes?
checkbox to print notes.
