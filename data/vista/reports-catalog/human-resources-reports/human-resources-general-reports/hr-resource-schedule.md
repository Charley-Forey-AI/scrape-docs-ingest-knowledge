# HR Resource Schedule

You can use the HR Resource Schedule report report that
 shows each resource by date and schedule code with description of the event by selecting
 Human Resources > Reports > HR Resource Schedule.
This schedule report shows each resource by date and schedule code with
 description of the event. It can be ran by selected company or all companies, range
 of resources and selected schedule codes. HRES.
