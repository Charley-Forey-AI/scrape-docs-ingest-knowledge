# HR Benefits by Expiration

You can use the HR Benefits by Expiration report to group
 by Benefit and list HR References by Expiration date in a Range of Benefit codes and/or an
 Expiration Date Range by selecting Human Resources > Reports > HR Benefits by Expiration.
This report groups by Benefit and lists HR References by Expiration date in a
 Range of Benefit codes and/or an Expiration Date Range. Information mostly comes
 from HREB but the report also uses HRBC, HRDP and HRRM.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Benefit Code
Select the Field Lookup button or press F4 to select the beginning benefit code or leave blank for all.

Ending Benefit Code Select the Field Lookup button or press F4 to select the ending benefit code or leave blank for all.
