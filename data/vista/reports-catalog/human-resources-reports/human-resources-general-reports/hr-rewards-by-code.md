# HR Rewards by Code

You can use the HR Rewards by Code report to show by
 reward code the resource, date, description, reason and reward received by selecting
 Human Resources > Reports > HR Rewards by Code.
This report shows by reward code the resource, date, description, reason and
 reward received. The report can be run for all or specific resources and rewards.
 Table used on this report is HRRD.
