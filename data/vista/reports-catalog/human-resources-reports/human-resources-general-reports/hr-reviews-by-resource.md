# HR Reviews by Resource

You can use the HR Reviews by Resource report for a list
 of each review for a resource and ratings by ratings code by selecting Human Resources > Reports > HR Reviews by Resource.
This report lists each review for a resource and ratings by ratings code. The tables used are HRRM, HRER, HRPC, HRRP, and HRCM. It can be ran by range of HR resources and range of review dates.
