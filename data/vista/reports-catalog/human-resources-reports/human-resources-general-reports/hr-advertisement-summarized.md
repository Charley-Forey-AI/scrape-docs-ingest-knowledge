# HR Advertisement Summarized

You can use the HR Advertisement Summarized report for a
 recruiting advertisement listing all open positions from the HRPC table by selecting
 Human Resources > Reports > HR Advertisement Summarized.
This provides a sample report for a recruiting advertisement listing all open
 positions from the HRPC table.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Position Code
Press F4 to select the position code or leave blank for all.

Print Notes?
checkbox to print notes.
