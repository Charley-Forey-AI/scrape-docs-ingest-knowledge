# HR Recruiting Activity

You can use the HR Recruiting Activity report for a list
 of each position code showing requirements for the position, advertisement information,
 qualifications, performance ratings, training required and any employees that have been
 assigned the position code by selecting Human Resources > Reports > HR Recruiting Activity.
This report provides a list of each position code showing requirements for the
 position, advertisement information, qualifications, performance ratings, training
 required and any employees that have been assigned the position code. HRPC. The
 report can be restricted to only Active Employees.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Position Code
Select the Field Lookup button or press F4 to select the beginning position code or leave blank for all.

Ending Position Code
Select the Field Lookup button or press F4 to select the ending position code or leave blank for all.
