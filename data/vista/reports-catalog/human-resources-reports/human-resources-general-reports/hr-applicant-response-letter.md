# HR Applicant Response Letter

You can use the HR Applicant Response Letter report as an
 example of an applicant response letter by selecting Human Resources > Reports > HR Applicant Response Letter.
This is an example of an applicant response letter. This report prints by HR Company with options to pull specific or all values for Position Code and Reference. The position code pulls in the description of the position as well as the contact writing the letter.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Position Code (Blank for All)
Press F4 to select the position code or leave blank for all.

Resource (Blank for All)
Select the Field Lookup button or press F4 to select the resource or leave blank for all.
