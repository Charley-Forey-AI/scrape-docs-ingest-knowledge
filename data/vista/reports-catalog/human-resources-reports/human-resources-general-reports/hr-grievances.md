# HR Grievances

You can use the HR Grievances report for a list of
 grievances by HR resource from the HREG table by selecting Human Resources > Reports > HR Grievances.
This report provides a list of grievances by HR resource from the HREG table.
 The date of the grievance, the outcome, hearing date, final appeal, and notes. This
 can be ran by selected HR resource and selected HR company or (0) selects all.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Resource (Blank for All)
Select the Field Lookup button or press F4 to select the resource or leave blank for all.

Print Notes?
checkbox to print notes.
