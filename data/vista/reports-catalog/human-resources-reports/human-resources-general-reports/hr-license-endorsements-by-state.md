# HR License Endorsements by State

You can use the HR License Endorsements by State report to
 list, by State, the License Codes and information about the License Code for that state by
 selecting Human Resources > Reports > HR License Endorsements by
 State.
This report lists by State the License Codes and information about the License Code for that state. The Description field is 250 characters.
Report Parameters
Description
State (Blank for All)
Accept the default, or press F4 to select a company.

(E)ndorsement, (R)estriction, (C)ertificate, (A)ll
Enter E, R, C, or A.
