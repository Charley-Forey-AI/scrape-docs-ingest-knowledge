# HR to PR Initialize Check

You can use the HR to PR Initialize Check report for a
 list of HR Resources and detail that may need correction prior to updating payroll by
 selecting Human Resources > Reports > HR to PR Initialize Check.
Report Parameters
Description
Beginning Resource
Select the Field Lookup button or press F4 to select the beginning resource or leave blank for all.

Ending Resource
Select the Field Lookup button or press F4 to select the ending resource or leave blank for all.

PR CompanyClick the Field
 Lookup button or press F4 to select the
 payroll company.
HR Company
Click the Field Lookup
 button or press F4 to select the human resources company.
