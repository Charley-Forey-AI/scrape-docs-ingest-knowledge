# UD Table and Form Setup

You can use the UD Table and Form Setup report to print
 out the fields used to create User Database Tables and Forms by selecting User Database > Reports > UD Table and Form Setup.
This report will print out the fields used to create User Database Tables and Forms. There is an input to choose whether to print the Validation if it has been set up for any fields.
