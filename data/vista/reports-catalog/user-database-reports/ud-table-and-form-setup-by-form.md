# UD Table and Form Setup by Form

You can use the UD Table and Form Setup by Form report to detail how the UD
 Table/Form is set up by selecting User Database > Reports > UD Table and Form Setup by Form.
This report will detail how the UD Table/Form is set up. This report would be useful to recreate any user defined tables. Recommended to print this report for all user defined tables to save as a reference.
