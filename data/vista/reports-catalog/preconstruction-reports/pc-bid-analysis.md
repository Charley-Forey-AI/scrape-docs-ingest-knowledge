# PC Bid Analysis

The PC Bid Analysis report allows an estimator to analyze the bidding performance of projects and/or companies that you are inviting to bid. To access this report, select
 Pre-Construction > Reports > PC Bid Analysis.
This report provides the optional ability to slice the data in three groups making analysis easier. Audience for this report include estimator, department heads, and senior management.
