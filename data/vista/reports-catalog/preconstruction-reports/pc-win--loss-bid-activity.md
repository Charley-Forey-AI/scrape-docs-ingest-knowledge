# PC Win / Loss Bid Activity

You can use the PC Win / Loss Bid Activity report to get a
 picture of the activity within the estimating department and track performance at several
 levels (listing data from the PC Potential Projects form). To access this report, select Pre-Construction > Reports > PC Win / Loss Bid Activity.
The report is grouped by project type and department; each group includes bid price, bid gross profit, and the markup percent, as well as other information. Each group also contains a sub report showing the same data, but grouped by bid result. The data for all projects and departments are summarized at the end of the report.
