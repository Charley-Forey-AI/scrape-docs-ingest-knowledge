# AP Worker Comp Liabilities for Subcontractors - Canada Report

You can use the AP Worker Comp Liabilities for Subcontractors - Canada Report to
 track vendors (subcontractors) who do not have a valid clearance certificate for worker
 compensation. You can access this report by selecting Accounts Payable >  Reports  > AP Worker Comp Liabilities for Subcontractors - Canada Report.
This report displays subcontracts at the item level so labor charges can be uniquely
 identified. A recommendation is that you set up labor charges for subcontracts as a
 unique cost type.
