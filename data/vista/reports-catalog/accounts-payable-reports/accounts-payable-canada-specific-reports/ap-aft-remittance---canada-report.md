# AP AFT Remittance - Canada Report

You can use the AP AFT Remittance - Canada report to act as a stub for the vendor.
 The report prints a new page for each cheque with information about the funds that have been
 electronically transferred to their account. You can access this report by selecting Accounts Payable >  Reports  > AP EFT Remittance - Canada.
Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Month
Enter or select the month.

BatchId
Press F4 to select the batch ID number.

Effective Date
Enter or select the effective date.
