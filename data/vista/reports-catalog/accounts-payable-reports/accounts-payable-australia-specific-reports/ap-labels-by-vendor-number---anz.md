# AP Labels by Vendor Number - A/NZ

You can use the AP Labels by Vendor Number - A/NZ report to print vendor address labels on Avery L7159 63.5 x 33.9 mm laser labels. You can access this report by selecting Reports >  Reports  > AP Labels by Vendor Number - A/NZ.
This report prints labels across and then down. Input options to print for a range of Vendors, which vendor address to use and whether to include inactive vendors.Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Beginning Vendor Number
Press F4 to select the beginning vendor number for sorting.

Ending Vendor Number
Press F4 to select the ending vendor number for sorting.

Print (V)endor Address or (P)O Address
Enter V to to print the vendor address or P to print the PO address.

Include Inactive:
Select the checkbox to include inactive vendors.
