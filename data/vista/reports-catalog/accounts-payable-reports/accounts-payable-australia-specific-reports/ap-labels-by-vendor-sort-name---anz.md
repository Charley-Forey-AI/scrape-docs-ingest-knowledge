# AP Labels by Vendor Sort Name - A/NZ

You can use the AP Labels by Vendor Sort Name - A/NZ report to print vendor address labels on Avery L7159 63.5 x 33.9 mm laser labels. You can access this report by selecting Reports >  Reports  > AP Labels by Vendor Sort Name - A/NZ.
The report prints labels across and then down in Vendor Sort Name order. There is an input for a range of Vendors to print and an option to print either vendor address or the PO address. Additionally there is an option to print for only the active vendors.Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Beginning Vendor Name
Press F4 to select the beginning vendor name for sorting.

Ending Vendor Name
Press F4 to select the ending vendor name for sorting.

Print (V)endor Address or (P)O Address
Enter V to to print the vendor address or P to print the PO address.

Include Inactive:
Select the checkbox to include inactive vendors.
