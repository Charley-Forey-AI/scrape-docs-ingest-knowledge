# AP Labels by Vendor Sort Name Report

You can use the AP Labels by Vendor Sort Name report to print vendor address labels
 on Avery 5160 1" x 2 5/8" laser labels. You can access this report by selecting Accounts Payable >  Reports  > AP Labels by Vendor Sort Name.
The report prints labels across and then down in Vendor Sort Name order. There
 is an input for a range of Vendors to print and an option to print either vendor
 address or the PO address. Additionally there is an option to print for only the
 active vendors.Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Beginning Vendor Name
Press F4 to select the beginning vendor name for sorting.

Ending Vendor Name
Press F4 to select the ending vendor name for sorting.

Print (V)endor Address or (P)O Address
Enter V to to print the vendor address or P to print the PO address.

Include Inactive:
Select the checkbox to include inactive vendors.
