# AP Vendor Compliance Report

You can use the AP Vendor Compliance report to display a list of compliance codes for
 a range of vendors. You can access this report by selecting Accounts Payable >  Reports  > AP Vendor Compliance.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Vendor
Press F4 to select the beginning vendor number for sorting.

Ending Vendor
Press F4 to select the ending vendor number for sorting.

Print Vendor Compliance Memos?
Select the checkbox to include vendor compliance
 memos in the report.

Include Compliance Codes not flagged All Invoices?
Select the checkbox to include compliance codes
 that do not have the All Invoices box
 checked in the HQ Compliance Codes form.

Show only Codes that are not in compliance?Select this checkbox to show only compliance codes with which the vendor is not currently in compliance.
