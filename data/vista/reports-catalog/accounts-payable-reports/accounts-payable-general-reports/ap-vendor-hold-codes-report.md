# AP Vendor Hold Codes Report

You can use the AP Vendor Hold Codes report to view all hold codes that apply to a
 vendor. You can access this report by selecting Accounts Payable >  Reports  > AP Vendor Hold Codes.
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Beginning Vendor
Select the Field Lookup button or press F4 to select the beginning vendor.

Ending Vendor
Select the Field Lookup button or press F4 to select the ending vendor.

Beginning Hold Code
Click the Field Lookup
 button or press F4 to select the Beginning Hold Code.

Ending Hold Code
Click the Field Lookup
 button or press F4 to select the Ending Hold Code.

checkbox to Print Memo.
