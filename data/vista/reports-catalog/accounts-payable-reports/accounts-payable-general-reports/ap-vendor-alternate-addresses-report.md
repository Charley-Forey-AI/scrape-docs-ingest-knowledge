# AP Vendor Alternate Addresses Report

You can use the AP Vendor Alternate Addresses report to display a list of the
 alternate addresses for vendors. You can access this report by selecting Accounts Payable >  Reports  > AP Vendor Alternate Addresses.
Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Beginning Vendor
Press F4 to select the beginning vendor number for sorting.

Ending Vendor
Press F4 to select the ending vendor number for sorting.
