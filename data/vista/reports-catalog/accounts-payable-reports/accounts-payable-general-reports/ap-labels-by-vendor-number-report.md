# AP Labels by Vendor Number Report

You can use the AP Labels by Vendor Number report to print vendor address labels on
 Avery 5160 1" x 2 5/8" laser labels. You can access this report by selecting Accounts Payable >  Reports  > AP Labels by Vendor Number.
This report prints labels across and then down. Input options to print for a
 range of Vendors, which vendor address to use and whether to include inactive
 vendors.Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Beginning Vendor Number
Press F4 to select the beginning vendor number for sorting.

Ending Vendor Number
Press F4 to select the ending vendor number for sorting.

Print (V)endor Address or (P)O Address
Enter V to to print the vendor address or P to print the PO address.

Include Inactive:
Select the checkbox to include inactive vendors.
