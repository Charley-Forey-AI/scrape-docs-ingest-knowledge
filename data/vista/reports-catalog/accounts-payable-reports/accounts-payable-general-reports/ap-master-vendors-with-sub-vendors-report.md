# AP Master Vendors with Sub Vendors Report

You can use the AP Master Vendors with Sub Vendors report to view a list of vendors
 that you designated as master vendors and any related sub-vendors. You can access this
 report by selecting Accounts Payable >  Reports  > AP Master Vendors with Sub Vendors.
Report ParametersDescription
Company
Accept the default, or press F4 to select a company.

Beginning Vendor
Press F4 to select the beginning vendor number for sorting.

Ending Vendor
Press F4 to select the ending vendor number for sorting.
