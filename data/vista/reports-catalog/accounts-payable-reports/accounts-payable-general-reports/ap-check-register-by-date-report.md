# AP Check Register by Date Report

Use the AP Check Register by Date report to print payment information for all checks,
 EFTs, and credit service payments processed in AP within a specified date range. You can
 access this report by selecting Accounts Payable >  Reports  > AP Check Register by Date.
The report allows you to print payment information sorted by Vendor number or
 by CM Reference. If you select the Pay Type Recap option, the report will include a
 summary of GL accounts and amounts by Pay Type.Report ParametersDescription
