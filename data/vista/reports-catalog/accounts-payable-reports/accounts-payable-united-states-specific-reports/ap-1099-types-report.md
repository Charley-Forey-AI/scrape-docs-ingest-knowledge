# AP 1099 Types Report

You can use the AP 1099 Types report to provide a list of 1099 types. You can access
 this report by selecting Accounts Payable >  Reports  > AP 1099 Types.
1099 types are user defined; however, the 1099 form print programs use this type to print the
 correct style 1099 form. Make sure the hard-coded types used in standard reports are
 set up as types. The Federal Form 1099 requires the use of 'MISC', 'DIV', and 'INT'
 currently for the types of 1099 forms.
