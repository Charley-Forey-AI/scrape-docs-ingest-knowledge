# EM Kilometres Driven - CAN

You can use the EM Kilometres Driven - CAN report for a
 list by equipment summarizing the number of hours and miles driven between a range of
 selected dates by selecting Equipment Management > Reports > EM Kilometres Driven - CAN.
This report provides a list by equipment summarizing the number of hours and
 miles driven between a range of selected dates. Parameters are Company, range of
 equipment, range of dates from the EMMR table.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Reading Date
Enter beginning reading date (MM/DD/YY) for transactions to be included or leave blank for all.

Ending Reading Date
Enter ending reading date (MM/DD/YY) for transactions to be included or leave blank for all.
