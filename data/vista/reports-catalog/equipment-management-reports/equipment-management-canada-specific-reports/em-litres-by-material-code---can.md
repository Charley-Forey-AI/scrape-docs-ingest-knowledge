# EM Litres by Material Code - CAN

You can use the EM Litres by Material Code - CAN report to
 summarize total litres and tax amount by material, tax code, and equipment by selecting
 Equipment Management > Reports > EM Litres by Material Code -
 CAN.
This report summarizes total litres and tax amount by material, tax code, and equipment. In order to print only fuel data, specify either (or all) the fuel cost code, cost type, or material in the input parameters.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Month
Select the date-range arrows to select the Beginning Month

Ending Month
Select the date-range arrows to select the Ending Month

Beginning Material Code
Select the Field Lookup button or press F4 to select ending material code or leave blank to select all.

Ending Material Code
Select the Field Lookup button or press F4 to select ending material code or leave blank to select all.
