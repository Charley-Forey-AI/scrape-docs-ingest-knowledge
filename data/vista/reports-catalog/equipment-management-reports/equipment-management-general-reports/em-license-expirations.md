# EM License Expirations

You can use the EM License Expirations report for an
 equipment list showing vehicle registration expiration dates for a selected company (by
 range of category, equipment, and expiration dates) by selecting Equipment Management > Reports > EM License Expirations.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Category
Select the Field Lookup button or press F4 to select beginning category, leave blank to include all.

Ending Category
Select the Field Lookup button or press F4 to select ending category or leave blank to select all.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending EquipmentSelect the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Expiration DateEnter beginning expiration date (MM/DD/YY) for equipment to be
 included or leave blank for all.
Ending Expiration DateEnter ending expiration date (MM/DD/YY) for equipment to be
 included or leave blank for all.
Status. (A)ctive, (I)nactive, (D)own (Blank for All)
Enter A, I, D or leave blank.
