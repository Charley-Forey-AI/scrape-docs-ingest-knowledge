# EM Components

You can use the EM Components report for a list of the
 setup information entered in EM Equipment for Component Type equipment by selecting
 Equipment Management > Reports > EM Components.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Component
Select the Field Lookup button or press F4 to select beginning component. Leave blank to include all components.

Ending Component
Select the Field Lookup button or press F4 to select ending component. Leave blank to include all components.

Print Information Tab?
checkbox to include data on the Information Tab, leave unchecked to exclude data from the Information Tab.

Page at Component?
checkbox to include a Page Break at each Component.
