# EM Automatic Usage Template

You can use the EM Automatic Usage Template report for
 setup information for category as well as equipment by selecting Equipment Management > Reports > EM Automatic Usage Template.
Beginning and ending template and option to print notes may be
 entered at runtime.Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Template
Select the Field Lookup button or press F4 to select beginning template. Leave blank to include all templates.

Ending Template
Select the Field Lookup button or press F4 to select ending template. Leave blank to include all templates.

Print Notes
checkbox to print notes.
