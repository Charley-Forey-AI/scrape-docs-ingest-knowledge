# EM Parts

You can use the EM Parts report for a list of material
 (equipment parts) by equipment by selecting Equipment Management > Reports > EM Parts.
This report provides you with a list of material (equipment parts) by
 equipment. Report columns include part number and description, headquarters material
 code and description, quantity and unit of measure. Report input parameters include
 company, beginning, and ending equipment number and an option to print part
 notes.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Print Notes?
checkbox to print notes.
