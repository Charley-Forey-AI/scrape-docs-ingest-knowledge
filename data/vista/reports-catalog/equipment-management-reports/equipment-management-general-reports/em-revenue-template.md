# EM Revenue Template

You can use the EM Revenue Template report for a list of
 Revenue Templates with details by category code and equipment code by selecting Equipment Management > Reports > EM Revenue Template.
This report provides a list of Revenue Templates with details by category code
 and equipment code. A sub-report prints the details by revenue code category
 including revenue code, rate, hours/time UM, % of rate and allow override flag. A
 second sub-report prints revenue code equipment information. Input parameters
 include beginning and ending revenue template, option to print category rates and
 option to print equipment rates.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Template
Select the Field Lookup button or press F4 to select beginning template. Leave blank to include all templates.

Ending Template
Select the Field Lookup button or press F4 to select ending template. Leave blank to include all templates.
