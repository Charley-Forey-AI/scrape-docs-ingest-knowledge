# EM Meter Readings Per Month

You can use the EM Meter Readings Per Month report to see
 EM meter readings per month by selecting Equipment Management > Reports > EM Meter Readings Per Month.
This report provides EM meter readings per month sorted by Equipment
 then month from the EMMR file. Shows the first and last meter reading date for each
 month and the meter change between the two for meter hours and odometer. Parameters are
 range of equipment and range of months.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Month
Select the date-range arrows to select the Beginning Month

Ending Month
Select the date-range arrows to select the Ending Month
