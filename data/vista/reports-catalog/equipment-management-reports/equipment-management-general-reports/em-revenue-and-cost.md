# EM Revenue and Cost

You can use the EM Revenue and Cost report for a summary
 of cost detail and revenue detail by equip category / equipment by selecting Equipment Management > Reports > EM Revenue and Cost.
This report provides a summary of cost detail and revenue detail by equip
 category / equipment. Parameters are company, range of categories, months, and
 equipment. Tables used in stored procedure are EMAR and EMMC.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Category
Select the Field Lookup button or press F4 to select beginning category, leave blank to include all.

Ending Category
Select the Field Lookup button or press F4 to select ending category or leave blank to select all.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending EquipmentSelect the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Month Select the date-range arrows to select the Beginning Month

Ending Month
Select the date-range arrows to select the Ending Month
