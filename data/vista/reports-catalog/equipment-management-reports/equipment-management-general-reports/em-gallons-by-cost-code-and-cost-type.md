# EM Gallons by Cost Code and Cost Type

You can use the EM Gallons by Cost Code and Cost Type
 report to show totals of gallons purchased by cost code, cost type, equipment, and material
 code by selecting Equipment Management > Reports > EM Gallons by Cost Code and Cost
 Type.
This report shows totals of gallons purchased by cost code, cost type, equipment, and material code. In order to print only fuel data, specify either (or all) the fuel cost code, cost type, or material in the input parameters.
