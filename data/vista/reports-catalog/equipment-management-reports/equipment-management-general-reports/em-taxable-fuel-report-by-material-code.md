# EM Taxable Fuel Report by Material Code

You can use the EM Taxable Fuel Report by Material Code report
to show by equipment, the total taxable gallons for a material code. Access this report by selecting Equipment Management > Reports > EM Taxable Fuel Report by Material Code.
This report shows, by equipment, the on road/off road miles, total miles, total gallons and taxable (on road) gallons for a selected material code.
Report ParametersDescription
CompanyAccept the default, or press F4 to select a company.

Material CodeEnter the fuel material code or press F4 to select a material code.
Beginning EquipmentEnter or select the applicable beginning equipment.

Ending EquipmentEnter or select the applicable ending equipment.

Begin MonthEnter or select the applicable beginning month.

Ending MonthEnter or select the applicable ending month.
