# EM Equipment Components

You can use the EM Equipment Components report for a list
 of components, printed by equipment that it is attached to. EMEM table by selecting
 Equipment Management > Reports > EM Equipment Components.
This report provides a list of components, printed by equipment that it is
 attached to. EMEM table. Parameters are Co., beg - end equip and beg - end component
 type.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Component Type
Click the Field Lookup
 button or press F4 to select beginning component type or leave
 blank to select all types.

Ending Component Type
Click the Field Lookup
 button or press F4 to select ending component type or leave
 blank to select all types.
