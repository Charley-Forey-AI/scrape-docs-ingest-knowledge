# EM Usage Rules

You can use the EM Usage Rules report to see the usage
 rules set up in the EM Usage rules table with revenue code to use, based on minimum and
 maximum hours and hours basis, by selecting Equipment Management > Reports > EM Usage Rules.
This report shows the usage rules set up in the EM Usage rules table with
 revenue code to use; based on minimum and maximum hours and hours basis. Tables EMUR
 & EMUD.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Rules Table
