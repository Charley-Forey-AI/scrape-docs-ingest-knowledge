# EM Equipment Cost Detail Report

You can use the EM Equipment Cost Detail Report for a list
 of the setup information entered in EM Equipment for Component Type equipment by selecting
 Equipment Management > Reports > EM Equipment Cost Detail
 Report.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Month
Select the date-range arrows to select the Beginning Month

Ending Month
Select the date-range arrows to select the Ending Month

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending EquipmentSelect the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning EM Cost TypeSelect the Field Lookup button or press F4 to select beginning cost type, leave blank to include all.

Ending EM Cost TypeSelect the Field Lookup button or press F4 to select ending cost type, leave blank to include all.
