# EM Work Order History By Equipment / Cost Code

You can use the EM Work Order History By Equipment / Cost
 Code report for summary detail and status of work order information in order by equipment
 number then cost code by selecting Equipment Management > Reports > EM Work Order History By Equipment / Cost
 Code.
This report provides summary detail and status of work order information in
 order by equipment number then cost code. Tables used EMEM, EMWH, and EMWI.
