# EM Revenue Breakdown Cost Report

You can use the EM Revenue Breakdown Cost Report report
 for EM Cost detail by Equip, Revenue code, Revenue breakdown code by selecting Equipment Management > Reports > EM Revenue Breakdown Cost
 Report.
This report provides EM Cost detail by Equip, Revenue code, Revenue breakdown
 code. Parameters are company, range of equipment and range of months. Tables used in
 stored procedure are EMRB, EMRD, and EMCD.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Month
Select the date-range arrows to select the Beginning Month
