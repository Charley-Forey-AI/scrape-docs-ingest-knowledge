# EM Cost Types

You can use the EM Cost Types report for a list of EM Cost
 Types from the set up program by selecting Equipment Management > Reports > EM Cost Types.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning EM Cost Type
Select the Field Lookup button or press F4 to select beginning cost type, leave blank to include all.

Ending EM Cost Type
Select the Field Lookup button or press F4 to select ending cost type, leave blank to include all.

Print Notes?
checkbox to print notes.
