# EM Equipment Location

You can use the EM Equipment Location report to list
 equipment and its current Location by selecting Equipment Management > Reports > EM Equipment Location.
Note: If the input for View Transaction History is Yes, then below each equipment is
 a list of transfer transactions for the time chosen in the input
 parameters.

Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Month
Enter beginning month or leave blank for all.

Ending Month
Enter ending month or leave blank for all.

View Transaction History?
checkbox to include transaction history.

Print Transfer Memos?
checkbox to include equipment transfer memos.

Print Notes?
checkbox to print notes.
