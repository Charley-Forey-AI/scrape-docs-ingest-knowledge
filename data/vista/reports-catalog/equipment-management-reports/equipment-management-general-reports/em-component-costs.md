# EM Component Costs

You can use the EM Component Costs report for a list of
 the equipment and summarized costs by Cost Code and Cost Type with totals for Month to Date,
 Fiscal Year to Date, and Total to Date by selecting Equipment Management > Reports > EM Component Costs.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Component
Select the Field Lookup button or press F4 to select beginning component. Leave blank to include all components.

Ending Component
Select the Field Lookup button or press F4 to select ending component. Leave blank to include all components.

Beginning Cost Code
Enter or select the applicable beginning cost code.

Month
Enter or select the month that you want to report through.
