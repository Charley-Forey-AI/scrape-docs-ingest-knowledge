# EM Equipment Work Order Log

You can use the EM Equipment Work Order Log report to list
 the Work Orders in WO number with the Work Order Description, and the Equipment that is
 assigned to the work order by selecting Equipment Management > Reports > EM Equipment Work Order Log.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Work Order
Enter or select the applicable beginning work order.

Ending Work Order


 Enter or select the applicable ending work order.
Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.
