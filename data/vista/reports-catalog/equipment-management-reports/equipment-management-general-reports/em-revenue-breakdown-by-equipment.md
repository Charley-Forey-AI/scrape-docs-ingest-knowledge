# EM Revenue Breakdown By Equipment

You can use the EM Revenue Breakdown By Equipment report
 for a list of Equipment revenue breakdown codes by equipment code by selecting Equipment Management > Reports > EM Revenue Breakdown By
 Equipment.
This report provides a list of Equipment revenue breakdown codes by equipment
 code. Report columns include Revenue Code, revenue breakdown code, and rate. Input
 parameters include group, beginning, and ending equipment code.
Reprot Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Equipment
Select the Field Lookup button or press F4 to select beginning equipment code or leave blank to select all equipment.

Ending Equipment
Select the Field Lookup button or press F4 to select ending equipment code or leave blank to select all equipment.

Beginning Category
Select the Field Lookup button or press F4 to select beginning category, leave blank to include all.

Ending Category
Select the Field Lookup button or press F4 to select ending category or leave blank to select all.
