# EM Litres by Cost Code and Cost Type – A/NZ

You can use the EM Litres by Cost Code and Cost Type –
 A/NZ report for totals of gallons purchased by cost code, cost type, equipment, and material
 code by selecting Equipment Management > Reports > EM Litres by Cost Code and Cost Type –
 A/NZ.
This report shows totals of gallons purchased by cost code, cost type, equipment, and material code. In order to print only fuel data, specify either (or all) the fuel cost code, cost type, or material in the input parameters.
