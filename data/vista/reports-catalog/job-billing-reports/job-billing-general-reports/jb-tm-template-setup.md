# JB T&M Template Setup

You can use the JB T&M Template Setup report to
 display all the template info It use by selecting Job Billing > Reports > JB T&M Template Setup.
The report displays all the template info It use. JBTM (template header), JBTS (template seq), JBTC(template cost types), JBTA(template addons) wit. an option to include the JB Equip Rates, JB Labor Rates and Labor Rate Overrides.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Template
Select the Field Lookup button or press F4 to select the beginning template or leave blank for all.

Ending Template
Select the Field Lookup button or press F4 to select the ending template or leave blank for all.

Print Notes?
checkbox to print notes.

Print Seq Notes?
checkbox to print sequence notes.

Print Rates And Overrides Details
checkbox to print rates and overrides details.
