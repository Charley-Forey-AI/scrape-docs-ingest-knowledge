# JB Change Order Billing

You can use the JB Change Order Billing report to compares
 Change Order amounts from Job Cost against amounts billed amounts in Job Billing by
 selecting Job Billing > Reports > JB Change Order Billing.
This report compares Change Order amounts from Job Cost against amounts billed
 amounts in Job Billing. Leaving the Checkbox checked for "Show Variances Only?" will
 result in seeing only Change Orders with a variance, if unchecked all change orders
 will show.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract:
Select the Field Lookup button or press F4 to select the beginning contract or leave blank for all.

Ending Contract:
Select the Field Lookup button or press F4 to select the ending contract or leave blank for all.

Show Variances Only?
checkbox to only show variances.
