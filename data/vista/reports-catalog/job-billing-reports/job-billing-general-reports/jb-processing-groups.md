# JB Processing Groups

You can use the JB Processing Groups report for a list of
 Job billing processing groups by selecting Job Billing > Reports > JB Processing Groups.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Processing Group:
Click the Field Lookup
 button or press F4 to select the beginning processing group or
 leave blank for all.

Ending Processing Group:
Click the Field Lookup
 button or press F4 to select the ending processing group or
 leave blank for all.

checkbox to print notes.
