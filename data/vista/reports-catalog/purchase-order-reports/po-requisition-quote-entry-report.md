# PO Requisition Quote Entry Report

You can use the PO Requisition Quote Entry Report to print
 the information that is displayed in PO Quote Entry by selecting Purchase Order > Reports > PO Requisition Quote Entry
 Report.
This report will print the information that is displayed in PO Quote Entry. The user has the option of printing notes and/or requisition detail if they exist. There is the option to print a range of Quotes or to print quotes created by a single user.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Quote ID
Click the Field Lookup
 button or press F4 to select the beginning quote ID.

Ending Quote ID
Click the Field Lookup
 button or press F4 to select the ending quote ID.

Created By (Blank for All)
Click the Field Lookup
 button or press F4 to select created by or leave blank for
 all.

Print RQ Detail?
checkbox to print quote detail.

Print Notes?
checkbox to print notes.
