# PO Shipping Locations

You can use the PO Shipping Locations report for a list of
 possible Shipping Locations for Purchase Orders by selecting Purchase Order > Reports > PO Shipping Locations.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Location
Select the Field Lookup button or press F4 to select the beginning location.

Ending Location
Select the Field Lookup button or press F4 to select the ending location.

Print Notes?
checkbox to print notes.
