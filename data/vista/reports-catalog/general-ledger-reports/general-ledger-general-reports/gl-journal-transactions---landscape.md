# GL Journal Transactions - Landscape

You can use the GL Journal Transactions - Landscape report
 for GL Journal transaction detail listed by Journal, Month, and GL Reference by selecting
 General Ledger > Reports > GL Journal Transactions -
 Landscape.
This report provides GL Journal transaction detail listed by Journal, Month,
 and GL Reference. This report uses table GLTD and can be printed at any time.
