# GL Monthly Budgets

You can use the GL Monthly Budgets report for a list of
 Accounts and budget amounts by month by selecting General Ledger > Reports > GL Monthly Budgets.
This report provides a list of Accounts and budget amounts by month. Inputs
 for up to four budget codes.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Account
Enter or press F4 to select a beginning GL account.

Ending Account
Enter or press F4 to select an ending GL account.

Beginning Month
Enter beginning month (MM/YY) or leave blank for all.

Ending Month
Enter ending month (MM/YY) or leave blank for all.

First Budget Code
Click the Field Lookup
 button or press F4 to select first budget code.

Second Budget Code
Click the Field Lookup
 button or press F4 to select second budget code.

Third Budget Code
Click the Field Lookup
 button or press F4 to select third budget code.

Fourth Budget Code
Click the Field Lookup
 button or press F4 to select fourth budget code.
