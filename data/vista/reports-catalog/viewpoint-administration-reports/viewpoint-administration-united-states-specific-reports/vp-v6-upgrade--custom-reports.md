# VP V6 Upgrade – Custom Reports

You can use the VP V6 Upgrade – Custom Reports report by selecting Viewpoint Administration > Reports > VP V6 Upgrade – Custom Reports.
This report lists custom reports (Report ID 10000 and higher) that will require modifications in order to run properly in the Viewpoint application. The RP Report Layout program must be run before printing this report. See the Viewpoint V6 Software Printing Guidelines document in the V6 Upgrade section of the Viewpoint Customer Area website for specific instructions on making corrections.
Parameters

Print Report Formulas?
checkbox to print report formulas.
