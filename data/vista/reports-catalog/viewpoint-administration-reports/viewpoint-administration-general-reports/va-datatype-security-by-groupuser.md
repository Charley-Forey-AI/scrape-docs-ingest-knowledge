# VA Datatype Security By Group/User

You can use the VA Datatype Security By Group/User to list groups or user that have
 access to datatype instances (ie Job, CMAcct) by selecting Viewpoint Administration > Reports > VA Datatype Security By Group/User.
This report lists groups or user that have access to datatype instances (ie Job, CMAcct). The report groups all the company datatypes first and then prints all the "non-company" datatypes next. Users have the option of sorting the report for each security group or for each individual user.
