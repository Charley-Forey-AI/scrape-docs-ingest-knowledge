# VA Web Usage By Viewpoint Browser

You can use the VA Web Usage By Viewpoint Browser report to display the URL of the
 web sites accessed and associated times by users by selecting Viewpoint Administration > Reports > VA Web Usage By Viewpoint Browser.
This report will display the URL of the web sites accessed and associated times by users.
Report Parameters
Description
Beginning Date
Enter beginning date.

Ending Date
Enter ending date.

User Name
Select the Field Lookup button or press F4 to select the user.
