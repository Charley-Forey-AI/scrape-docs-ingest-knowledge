# Viewpoint View Columns

You can use the Viewpoint View Columns report by selecting Viewpoint Administration > Reports > Viewpoint View Columns.
The Viewpoint View Columns report was designed to assist with writing Crystal report. The report lists each view in the Viewpoint system along with the columns for each view. Other information that is helpful such as type of field, length, precision, and number of decimals.
