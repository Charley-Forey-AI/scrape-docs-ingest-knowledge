# VA Inquiry Security

You can use the VA Inquiry Security report to verify inquiry access for an individual user or user group by selecting Viewpoint Administration > Reports > VA Inquiry Security.
This report shows security access levels by company and user or user group for a selected inquiry or all inquiries.
.
Report Parameters
Description
Beginning Company
Select the Field Lookup button or press F4 to select the beginning company.

Ending Company
Select the Field Lookup button or press F4 to select the ending company.

User or Group (required)

 Enter a user name or security group or press F4 to select from a list of valid Viewpoint Users or Report Security Groups.
Enter InquiryName

 Enter the inquiry name or press F4 to select from a list of valid VA Queries. Leave blank to show the security settings for all inquiries.
