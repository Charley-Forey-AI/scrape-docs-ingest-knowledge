# VA Work Center Security

You can use the VA Work Center Security report to verify work center access for an individual user or user group by selecting Viewpoint Administration > Reports > VA Work Center Security.
This report shows security access levels by company and user or user group for a selected work center template or all work center templates.
.
Report Parameters
Description
Beginning Company
Select the Field Lookup button or press F4 to select the beginning company.

Ending Company
Select the Field Lookup button or press F4 to select the ending company.

User or Group (required)
 Enter a user name or security group or press F4 to select from a list of valid Viewpoint Users or Report Security Groups.
Enter Template Name
 Enter the work center template name or press F4 to select from a list of valid work center templates. Leave blank to show the security settings for all work center templates.
