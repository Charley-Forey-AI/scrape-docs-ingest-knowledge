# Viewpoint Lookup Usage

You can use the Viewpoint Lookup Usage report for a list that will display the Form
 or Report where a Lookup (F4) is being used by the system by selecting Viewpoint Administration > Reports > Viewpoint Lookup Usage.
This report provides a list that will display the Form or Report where
 a Lookup (F4) is being used by the system. If the Form or Report is a custom then
 "Custom" is displayed in the Custom column.
Parameters

Beginning Lookup
Click the Field Lookup
 button or press F4 to select the beginning
 lookup or leave blank for all.

Ending Lookup
Click the Field Lookup
 button or press F4 to select the ending
 lookup or leave blank for all.
