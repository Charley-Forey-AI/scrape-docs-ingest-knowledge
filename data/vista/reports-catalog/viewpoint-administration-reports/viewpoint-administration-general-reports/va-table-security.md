# VA Table Security

You can use the VA Table Security report by selecting Viewpoint Administration > Reports > VA Table Security.
This report is a listing of the tables. If the InUse column is set to 0, this means security for this table is not set. Where the InUser column = 1, security is set. A listing of only those tables where security is turned on may be acquired by setting the flag for Table security, InUse Only, to Y.
