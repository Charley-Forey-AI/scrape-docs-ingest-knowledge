# VA Application Log By Object

You can use the VA Application Log By Object report to list the Application Log by
 Event for "FormOpened" and "ReportOpened" and associated Object in reverse chronological
 order by selecting Viewpoint Administration > Reports > VA Application Log By Object.
Report Parameters
Description
