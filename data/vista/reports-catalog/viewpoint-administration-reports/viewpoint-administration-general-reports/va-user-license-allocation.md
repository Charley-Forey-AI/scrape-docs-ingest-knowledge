# VA User License Allocation

You can use the VA User License Allocation Report by selecting Viewpoint Administration > Reports > VA User License Allocation.
Use the VA User License Allocation Report to see your organization's current Vista license information and a comprehensive breakdown of user license distribution. You can use this information when reconciling on-demand usage.
This multi-page report includes the following information: Vista ID, License Model, License Enforcement Status, Total Licenses Contracted, Total Licenses Allocated, usage per license type (Office or PM), licenses available, and user login details.
Available Licenses = Total Contracted – Total Allocated
Note: If you have available licenses, this number will show as green. If you have exceeded your number of available licenses, the number shows as negative in red.

Report Parameters

There are no parameters for this report.
