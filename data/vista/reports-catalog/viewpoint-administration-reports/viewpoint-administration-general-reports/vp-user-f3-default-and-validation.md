# VP User F3 Default and Validation

You can use the VP User F3 Default and Validation report by selecting Viewpoint Administration > Reports > VP User F3 Default and Validation.
This report lists by Viewpoint User, the Form and Sequence that has
 been modified with user F3 values. If the sequence also has a user validation procedure,
 then it will be listed with the standard Viewpoint validation procedure.
Parameters

User (Blank for All)
Select the Field Lookup button or press F4 to select the user.

Beginning Form
Click the Field Lookup
 button or press F4 to select the beginning
 form.

Ending Form
Click the Field Lookup
 button or press F4 to select the ending
 form.
