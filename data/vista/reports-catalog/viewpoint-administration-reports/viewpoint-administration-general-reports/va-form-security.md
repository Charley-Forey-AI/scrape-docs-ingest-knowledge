# VA Form Security

You can use the VA Form Security report to verify form access for an individual user
 by selecting Viewpoint Administration > Reports > VA Form Security.
This report is intended to verify form access for an individual user.
 It lists form security access for a specific user. The report will list the form
 security access level along with record permissions(Add, Update, Delete) and associated
 attachments. If tab level security is defined on a specific form it will also be
 listed.
When selected, the parameter 'Include "All Companies"' displays applied security for all active companies. This report uses a special View called VASecurityByForm.
