# VA Security Group by User

You can use the VA Security Group by User report to list users and their associated
 security groups along with the type of security by selecting Viewpoint Administration > Reports > VA Security Group by User.
Report Parameters
Description
Beginning Security Group
Click the Field Lookup
 button or press F4 to select the beginning
 security group.

Ending Security Group
Click the Field Lookup
 button or press F4 to select the ending
 security group.

User (Blank for All)
Select the Field Lookup button or press F4 to select the user.

Security Group Type Data (0), Program (1), Reports (2) Blank (All)
Enter 0, 1, 2, or leave blank for all.
