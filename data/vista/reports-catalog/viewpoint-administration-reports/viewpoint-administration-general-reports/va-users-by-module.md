# VA Users by Module

You can use the VA Users by Module report to display a list of active users and the modules they have accessed for a given month. Access the VA Users by Module report by selecting Viewpoint Administration > Reports > VA Users by Module.
This report displays, in grid format, a list of active Users and all available modules in Vista. In each module column, the report shows the number of times the given user has opened a form in that module. If a cell is blank, it means that user does not have access to that module. If a zero displays, the user has access to the module, but has not opened any forms for that module in the specified month.

MonthEnter or select the month.
