# VA Application Log By User

You can use the VA Application Log By User report to list system activity for users
 in reverse chronological order by selecting Viewpoint Administration > Reports > VA Application Log By User.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Date
Select the Field Lookup button or press F4 to select the beginning date.

Ending Date
Select the Field Lookup button or press F4 to select the ending date.

User
Select the Field Lookup button or press F4 to select the user.
