# VA License Usage by Day Report

Use the VA License Usage by Day Report to assess and understand your enterprise-wide login frequency and license utilization.
This drilldown report shows maximum concurrent usage by license type for each day, along with user login details. To learn more, visit these Help pages:

- [License Type](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form/field-definitions-va-user-profile-form#concept-8148--en)

- [Assess License Usage](/en/vista/vista/getting-started/getting-started-with-vista/vista-licensing/assess-license-usage)
