# VA Security Groups

You can use the VA Security Groups report for a list of Security Groups with
 descriptions and associated users by selecting Viewpoint Administration > Reports > VA Security Groups.
Report Parameters
Description
Beginning Security Group
Click the Field Lookup
 button or press F4 to select the beginning
 security group.

Ending Security Group
Click the Field Lookup
 button or press F4 to select the ending
 security group.
