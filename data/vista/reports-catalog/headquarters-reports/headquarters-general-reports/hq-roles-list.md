# HQ Roles List

You can use the HQ Roles List report to see the full setup
 details of a given role (or all roles) from either a role or member
 perspective. To access this report, select Headquarters > Reports > HQ Roles List.
 From the role perspective the report displays:

- The role description and its active status

- Default values for limits and thresholds for each type\sub-type combination defined on the role

- What modules the role is currently enabled on

- What members have been associated with the role and their active status

- If members have any overrides on the role's limits

From the Member perspective the report displays:

- Each role the member is associated with

- The role's description and active status

- What modules the role is currently enabled on

- What the final limit\threshold values are for each type\sub-type on the role and whether or not those values are default or override values
