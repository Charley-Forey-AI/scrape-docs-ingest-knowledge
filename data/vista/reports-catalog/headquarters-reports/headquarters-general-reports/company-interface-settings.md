# Company Interface Settings

You can use the Company Interface Settings report to see company parameters information for each system module. To access this report, select Headquarters > Reports > Company Interface Settings.
The main data source for this report is the company parameter form for each appropriate module. It uses the selected HQ company to pull the modular company information. The detail included in this report consists of the interfacing company for each related module, along with the available interface levels. If you have not purchased a module, it will display on the report, but the information bill be blank.
Report Parameters
Description

Beginning Company
Select the Field Lookup button or press F4 to select beginning company.

Ending Company
Select the Field Lookup button or press F4 to select ending company.
