# HQ Batch Control Attachment Drilldown

You can use the HQ Batch Control Attachment Drilldown
 report to review batches that you either created or posted. To access this report, select Headquarters > Reports > HQ Batch Control Attachment
 Drilldown.
This drilldown displays only batch IDs that were either created by or posted by the user running the report. The data on the drilldown will also show batch IDs that have had their batch reports automatically added to HQ Batch Control (you selected the Attach Batch Reports to HQ Batch Control checkbox in the company parameters form for the corresponding module).
Report Parameters
Description

Company
Accept the default, or press F4 to select a company.

Month
Enter batch month (MM/YY).

BatchID (Blank for All)
Click the Field Lookup
 button or press F4 to select batch ID or leave blank for
 all.
