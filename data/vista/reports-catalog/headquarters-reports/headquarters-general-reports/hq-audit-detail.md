# HQ Audit Detail

You can use the HQ Audit Detail report to review audit details. To access this report, select Headquarters > Reports > HQ Audit Detail.
The HQ Audit Detail report lists records from the HQMA database table. This information is useful to audit or track changes made to tables including what the old value, new value, date of change and user who made the change to a column in a table. Some tables like the AP Vendor Master and the AR Customer tables will need the Company/Group input to be the group from HQ Groups as there is no company stored in those tables.
You can also view audit records of changes made to field property
 (F3) overrides in the Audit Log. See [View Changes to F3 Field Property System Overrides](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/maintenance/view-the-audit-log/view-changes-to-f3-field-property-system-overrides) for more information.
