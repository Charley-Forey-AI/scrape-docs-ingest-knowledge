# HQ Implementation Record Counts

You can use the HQ Implementation Record Counts report to see a list of record counts related to the tasks in the Workflow (WF) Template Tasks form. To access this report, select HeadquartersReportsHQ Implementation Record
 Counts.
This report uses the selected HQ Company to pull the WF Template Tasks and appropriate record counts. The detail included in this report consists of the company for each database table, along with the record counts for the company/table. Tables with zero record counts display in red.
Report Parameters
Description

Enter Company (0 for All)
Accept the default, or press F4 to select a company.
