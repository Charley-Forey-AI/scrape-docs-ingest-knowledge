# HQ Material U/M Conversions

You can use the HQ Material U/M Conversions report to see materials (by group) and their UM conversion data. To access this report, select Headquarters > Reports > HQ Material U/M Conversions.
Report Parameters
Description

Material Group
Select the Field Lookup button or press F4 to select material group.

Beginning Material
Select the Field Lookup button or press F4 to select beginning material code or leave blank for all.

Ending Material
Select the Field Lookup button or press F4 to select ending material code or leave blank for all.
