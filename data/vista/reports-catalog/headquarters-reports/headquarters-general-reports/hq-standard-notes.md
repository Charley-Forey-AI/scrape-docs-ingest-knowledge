# HQ Standard Notes

You can use the HQ Standard Notes report for a list of standard notes set up and available for use throughout the system. To access this report, select HeadquartersReportsHQ Standard Notes.

Beginning Standard Note

Click the Field Lookup
 button or press F4 to select ending standard note or leave
