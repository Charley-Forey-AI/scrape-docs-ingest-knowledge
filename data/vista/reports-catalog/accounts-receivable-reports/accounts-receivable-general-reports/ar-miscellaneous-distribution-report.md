# AR Miscellaneous Distribution Report

You can use the AR Miscellaneous Distribution Report for a
 list of Miscellaneous Distribution codes and the monthly totals for each transaction posted
 to each code. You can access this report by selecting Accounts Receivable > Reports > AR Miscellaneous Distribution
 Report.
This report also uses the Miscellaneous Code table in combination with the
 Miscellaneous details to derive an extended amount. For example, the basis amount
 entered for the Miscellaneous Distribution code is printed along with the rate setup
 for the Dist Code with a calculated extended amount.
