# AR Transactions by Receivable Type

You can use the AR Transaction by Receivable Type report
 to print AR transactions sorted by receivable type for a range of months. You can access
 this report by selecting Accounts Receivable > Reports > AR Transactions by Receivable
 Type.
This report prints AR transactions sorted by receivable type for a range of
 months. Transactions can be restricted to include only those from any combination of
 AR, JB, and MS. If transactions from AR are included, additional inputs are
 available to include any combination of invoices, receipts, and finance charges
 (adjustments, credits, write-offs and released retainage transactions are included
 when invoices are included.)
