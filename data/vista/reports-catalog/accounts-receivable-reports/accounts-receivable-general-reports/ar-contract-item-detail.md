# AR Contract Item Detail

You can use the AR Contract Item Detail report to list out
 the Detail from table ARTL. You can access this report by selecting Accounts Receivable > Reports > AR Contract Item Detail.
This report is limited to a single company, with ranges available for Contract
 and Contract Item. The Through Month input will restrict transactions beyond the
 input date. This report could be used to display detail for other Job to Date
 reports.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Contract
Press F4 to select the beginning contract for sorting.

Ending Contract
Press F4 to select the ending contract for sorting.

Beginning Item
Press F4 to select the
 Beginning Item.

Ending Item
Press F4 to select the Ending
 Item.

Through Month
Select the date-range arrows to select the month for reporting through.
