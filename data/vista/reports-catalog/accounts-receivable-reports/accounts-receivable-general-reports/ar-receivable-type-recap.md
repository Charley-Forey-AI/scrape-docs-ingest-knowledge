# AR Receivable Type Recap

You can use the AR Receivable Type Recap report to print
 accounts receivable and retainage totals for each receivable type for all invoices that are
 within the month range entered. You can access this report by selecting Accounts Receivable > Reports > AR Receivable Type Recap.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Month
Select the date-range arrows to select the Beginning Month

Ending Month
Select the date-range arrows to select the Ending Month
