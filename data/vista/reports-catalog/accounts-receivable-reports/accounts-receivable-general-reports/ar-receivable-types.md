# AR Receivable Types

You can use the AR Receivable Types report for a list of
 AR Receivable types. You can access this report by selecting Accounts Receivable > Reports > AR Receivable Types.
Report Parameters
Description
Company
Accept the default, or press F4 to select a company.

Beginning Receivable Type
Click the Field Lookup
 button or press F4 to select the
 Beginning Receivable Type

Ending Receivable Type
Click the Field Lookup
 button or press F4 to select the Ending
 Receivable Type
