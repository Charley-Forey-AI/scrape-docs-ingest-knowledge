# Date Formats

The format of the dates and periods in the application (e.g.
 fiscal periods, pay periods, batch months, effective dates, etc.) is determined by the
 region setting of your workstation.
To view or change the region setting of your
 workstation, go to the Start menu on your Windows Task Bar, select Control
 Panel > Region and
 Language Options, and then open the Format tab on the Region and Language window. The
 Short
 Date field defines how dates/periods display in the system.
The date/period format used for reports is
 determined by the Report Date Format field in HQ
 Company Setup. This option allows you to define the date form in each company.

- 1 – (mm/dd/yy)

- 2 – (dd/mm/yy)

- 3 – (yy/mm/dd)

Note: It is suggested that you select the format that
 matches the date format defined for your PC.
