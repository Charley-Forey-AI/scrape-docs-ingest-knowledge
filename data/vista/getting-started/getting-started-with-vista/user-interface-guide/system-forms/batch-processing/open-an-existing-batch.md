# Open an Existing Batch

Posting programs in the system use batches to record data.
 You can use the Batch Selection form to open an existing batch.

1. Select the Use an existing
 batch checkbox.

1. Specify a month in the
 Batch Month field.

1. To display only batches
 that you created, select the Show my batches only check
 box.

1. Select the Include
 unposted batches checkbox to show unposted batches of any
 status.

1. In the Unposted Batches
 grid, select the desired batch.

1. Click OK
 to open the entry form.
