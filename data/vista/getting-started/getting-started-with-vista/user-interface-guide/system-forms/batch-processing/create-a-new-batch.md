# Create a New Batch

Posting programs in the system use batches to record data. You
 can use the Batch Selection form to create a new batch.

1. Select Create a new
 batch.

1. Specify a month in the
 Batch Month field.

1.  If you are the only
 user who should have access to the new batch, select the Restricted
 Access checkbox.

1. Click OK
 to open the entry form.
