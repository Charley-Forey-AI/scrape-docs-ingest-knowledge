# Date Field Shortcuts

You can use shortcuts to enter information into date
 fields.
ShortcutDescription
T or tSet the
 date to the current date.
MMDDFour digit month and
 day
Enter a
 four digit month and date (MMDD) and the system will automatically add
 the current year.
+The
 system will automatically set the date to tomorrow.
+5The
 system will automatically set the date to 5 days in the future. You can actually enter
 any value after the +, for example you can enter +7 to set the
 date to next week.

-The
 system will automatically set the date to the previous day.
-5The
 system will automatically set the date to 5 days in the past.Just like with +, you
 can enter any value after the -, for example you can enter
 -7 to set the
 date to the previous week.
