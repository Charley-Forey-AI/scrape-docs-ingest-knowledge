# Print Grid Data

Most forms in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can print and print
 preview the records that display on the Grid tab.

1. Right-click in the grid
 and select Print Option.Note: The Print Options menu item is not
 available on the Dashboard, PM, or SM Work Center grids.

1. In the context menu,
 select either Print or Print
 Preview. The Print/Export Options form
 opens. In the TrueDBGrid Print/Export Options form, the
 Action drop-down defaults to the option you
 selected.

1. Accept the defaults in
 the rest of the form, or if desired, use the remaining options to control how
 the data will be presented when printed.Note: The File Name field and
 Options button are disabled when using Print or Print
 Preview.

1. Click OK.
 If you select Print Preview, the PrintPreview window
 opens, where you can save or print the file. If you select
 Print, the file is sent immediately to your default
 printer.
