# Delete Grid Records

Most forms in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can delete records from
 the Grid tab.

1. Delete a record by clicking the
 Delete button, pressing the Delete key, or selecting Records > Delete.

1. To delete multiple records, select the desired records (using the Shift key for
 consecutive selection or the Ctrl key for non-consecutive selection) and click
 the Delete button or select Records > Delete. You cannot use the Delete key for multiple record deletions). A
 message displays indicating that you are about to delete multiple rows.

1. Click Yes to continue or No to abort.
