# Hide Grid Descriptions

Most form in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can temporarily hide the
 display of descriptions to allow more records to show in the grid.

1. Select Tools > Field Properties.

1.  In the Description in Grid field of the Field Properties (F3) form
 select Show Above Grid Only.Once you set a description to show above the grid, it automatically sets this
 option to on. To turn it off, go to the View menu, right-click to open the
 context menu menu, or open the drop-down menu of the Grid button, and then
 deselct the Descriptions Above Grid
 option. It will remain off until you manually turn it back on.
