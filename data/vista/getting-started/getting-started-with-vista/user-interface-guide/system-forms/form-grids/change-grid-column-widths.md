# Change Grid Column Widths

Most forms in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can change the width of
 the columns on the grid, as well as the grid splitter that separates the key columns from
 the non-key columns to allow more records to show in the grid.
The display of column headings differs based
 on field type justification. Numeric, String to Numeric (Employee, Vendor, etc.), and
 Date/Month fields are right justified, while text fields are left justified.Note: If the column for a drop-down field is not wide
 enough to display both the largest value and the drop-down arrow,
 press F4 to enter the character in a lookup, or click the drop-down arrow
 to select the correct option.

1. Point your cursor to the
 right border of the column heading until the cursor changes to the column resize
 pointer ().

1. Drag the border to the
 size you want—right to increase the size, left to decrease it. If you want to
 widen a key column, you will generally need to change the position of the
 splitter to make room. Just move the cursor to any position on the splitter bar
 until it changes to the splitter reposition pointer (). Then drag the splitter
 until it is in the desired position. If there is extra room left between the
 splitter and the last key column, you can manually adjust the splitter as
 necessary; however, it will automatically reset to accommodate the size of the
 key columns once you close the form. Once you have made changes to
 the column widths and the splitter position, they will remain in effect
 until you change them or reset them.
To restore the column widths to their original size:

- In the View menu, point to Columns, and then select Reset
 Column Widths. All columns are reset to their original
 widths and the splitter bar is reset to its original position.
