# Field Definitions: VPMenu Form Search Form

The following is a list of field descriptions for the VPMenu
 Form Search form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Limit Records on Form

Select an option for the number of records to be returned on the originating form.
Defaults to the System option. This is the number set company-wide in VA Site Settings or by user in VA User Profile or User Options.
If you select another option, it will override the site-wide setting in VA Site Settings and the by-user setting in VA User Profile or User Options.
The limit you specify here affects only the originating form. For example, if you opened the [Search panel](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/system-forms/search-form-records/about-the-search-panel) from PR Employees, then the limit you set there would apply only to PR Employees.
