# Save a Form Search

If you have one or more searches that you use regularly to locate data records in a form, you can save your searches to eliminate having to re-enter the search criteria each time you need to filter those records.
To save a search:

1. Open the Vista form for which you want to create and save a search.

1. Access the Search panel in one of the following ways:

- Select the  icon in the toolbar

- Select Records > Form Search from the form menu

1. Enter your search criteria in the upper section of the search panel.Note: The search criteria available depends on the form.

1. Indicate the order in which to sort search results as follows:

- Latest Records - Sort search results in order by the latest records (descending order)

- Earliest Records - Sort search results in order by the earliest records (ascending order)

1. In the Name of Search text box, enter a name to easily identify the search.Note: If you need to delete a saved search, choose the search you want to delete in the Name of Search dropdown and select Delete This Search.

1. If shared searches are enabled for your login (in VA User Profile) and you want to share this search, select the Shared radio button.If you do not want to share this search or if shared searches are not enabled for your login, select the My Saved radio button.

1. Select Save This Search.The system saves the search and adds it to the Name of Search drop-down list.
