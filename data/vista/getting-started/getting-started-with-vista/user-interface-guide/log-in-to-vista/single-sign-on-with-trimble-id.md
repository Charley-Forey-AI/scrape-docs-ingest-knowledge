# Single Sign-On with Trimble ID

SSO with Trimble ID allows you to use the same login credentials for Vista as for other Trimble-licensed products.
Note: Once implemented in early 2025, Trimble ID will be available to Trimble Construction One cloud-hosted Vista customers only.If you are a legacy cloud or on-premises customer, contact your account representative to learn about how you can transition to the modern cloud and gain access to all the latest features.

The information included here is for Vista users. If you are an admin, see the following article for admin-specific setup tasks related to Trimble ID: [Single Sign-On with Trimble ID: Admin Information](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/single-sign-on-with-trimble-id-admin-information).

## Set Up Your SSO Login with Trimble ID

Once Trimble ID is enabled for your company, you must set up the following in order to use Trimble ID to log into Vista:
Create a Trimble IDAs a Vista user, you must create your own Trimble ID profile: [Upgrade Your Vista Login to Trimble ID](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/log-in-to-vista/single-sign-on-with-trimble-id/upgrade-your-vista-login-to-trimble-id).
To see a walkthrough of this process, watch the following video: [Upgrade Your Vista Login to Trimble ID - Video](/en/vista/vista/videos/administration/upgrade-your-vista-login-to-trimble-id---video).
Log in Using Trimble IDAfter you upgrade your login, going forward you are required to [log in using Trimble ID](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/log-in-to-vista/single-sign-on-with-trimble-id/log-in-to-vista-using-trimble-id).
Manage Trimble ID Login CredentialsManage your profile and login credentials at [myprofile.trimble.com](https://myprofile.trimble.com/home). For more
 information, see [Manage My
 Trimble ID Profile](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:accountservices_003:accountservices:accountservices).
