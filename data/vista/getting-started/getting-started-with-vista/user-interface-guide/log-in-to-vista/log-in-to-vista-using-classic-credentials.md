# Log in to Vista Using
 Classic Credentials

You can log in to Vista using either your Windows login
 (the credentials you use to log in to your workstation) or using your Vista system SQL
 login.
Note: If you are a Trimble Construction One cloud-hosted Vista customer, this option to log in using classic credentials (SQL or Windows logins) will no longer be supported in the fall of 2025. At that point, you will have to log in using [Trimble ID](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:vista_60004).

Before you can log in for the first
 time, an admin must set up a user profile on your behalf in the VA User Profile
 form, and your user profile must have a default Company assigned. If applicable, you
 must also have configured the connection to your server via internet or LAN.
Tip: If you already configured a Trimble IDlogin, you must use it when
 signing in. If you try to use a classic login method (SQL
 login or Windows domain login) after setting up your Trimble ID, you will not be able to use
 this account and see an error saying *Your account
 has been deactivated*. Instead, [Log in to Vista Using Trimble ID](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/log-in-to-vista/single-sign-on-with-trimble-id/log-in-to-vista-using-trimble-id).

To log in to Vista with classic credentials:

1. Open the Vista application.

1. On the Vista log in screen, select the plus sign next to the Vista Server field and enter the URL provided by your administrator, starting with *https://*.Any entries that appear in this list are servers that you
 successfully signed into previously.

1. Select the check mark to validate your entry.

1. Then select Sign in with Classic ID.

1. Do one of the following:

- Select the Use Windows Login
 checkbox. The User Name field populates
 automatically with your domain and user name.

- Enter your SQL user name in the User
 Name field. If this field is disabled, clear the Use Windows
 Login checkbox.

1. Enter your
 Password.

1. Optional: Select the Save
 Password checkbox to auto-fill your password for future login
 instances.Tip: If the Save
 Password checkbox does not display, your administrator has
 disabled it in the VA Site Settings form.
