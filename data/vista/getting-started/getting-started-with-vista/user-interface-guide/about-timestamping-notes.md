# About Timestamping Notes

You can enter timestamped notes or comments on several forms throughout Vista.

For some Notes fields, timestamping is provided as a standard. For other Notes fields, you can [enable timestamping](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/about-timestamping-notes/enable-timestamping-for-notes) using the Display as Timestamp Notes option on the System Overrides tab in the Field Properties form (F3). However, to enable this feature, you must be designated as a Form Administrator in VA User Profile.
When entering timestamped notes, you must first double-click in the field to which you are adding the notes. This opens the Time Stamp Notes window, which you can then use to enter any miscellaneous notes about the item.
The Time Stamp Notes window is divided into two panes. The upper pane is display only, and shows all previously entered notes. Each note is time/date-stamped and includes the login name of the person who entered the notes. Existing notes cannot be edited or deleted.
The lower pane is used to add new notes. Once you have entered your note text, click the Add button. This will add the note, along with the date, time, and login name, to the upper pane of the window.
Tip: To use the Tab feature (such as to indent the first line of a paragraph or create columns), press Ctrl + Tab for each tab increment.

Add a Standard NoteStandard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into a Notes field with Timestamp Notes enabled, place focus in the bottom pane of the Time Stamp Notes window and select the Timestamp Notes icon () in the toolbar or right click the mouse and select Standard Notes from the shortcut menu. This opens the Standard Note Copy window. Then enter the standard note to copy (or select from F4 lookup) and click OK. The system inserts the selected note into the field.
Spelling CheckYou can use the Spelling Check feature in either the upper or
lower pane of the Time Stamp Notes window. Just place focus in the desired pane
and either select the Spelling icon () on the toolbar or right click the mouse and select Spelling from the
shortcut menu.
