# User Interface Features and Overrides

The following table provides a guide of user interface features available at the system, form, and field levels of the Vista™. Additional information is available for certain features and can be accessed by clicking the associated link.
X = Feature is/can be controlled at the indicated level.
* = Limited overrides apply based on Vista standard settings. Cannot change property if set at a higher level (e.g. standard or system wide).
** = When setting system or user overrides for these features, the checkboxes default as null (box shaded green). This indicates no override has been set and that the standard default defined for the input will be used. You can view the standard default for the related field using the Property Values tab in the Field Properties form (F3).
