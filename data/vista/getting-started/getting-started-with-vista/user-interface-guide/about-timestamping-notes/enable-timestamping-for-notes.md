# Enable Timestamping for Notes

You can enable timestamping for certain Notes fields to track entry times and dates, as well as the user who entered the data.

The timestamp feature is enabled using the System Overrides tab in the Field Properties form (F3). To set overrides on this tab, you must be designated as a Form Administrator in VA User Profile. In addition, if your company has set up security for this tab, you must also have access permissions.
Timestamping is only available for Notes fields using a bNotes datatype, which you can check using the Property Values tab in the Field Properties form.The following details how to enable the timestamping feature.

1. Open the form containing the Notes field you want to timestamp.

1. Place your cursor in the desired field and press F3.The Field Properties form displays.

1. Click on the System Overrides tab.

1. Select the Display as Timestamp Note checkbox. Note: This checkbox is only enabled for fields with a bNotes datatype.

1. Click Apply or OK.A message displays indicating that you must close and reload the form in order for your changes to take effect.

1. Select OK to save the change and reload the form.

Once the form reloads, the Timestamp Notes feature is enabled. For more information, see [About Timestamping Notes](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/about-timestamping-notes).
