# About the User Options Form

You can change your general user options from the Main Menu using the User Options
 form. You can access this from by selecting Options >  User Options from the Main Menu.
The User Options form and the VA User Profile form update each
 other, so you need to set only one to affect both. However, the Map Site Option and the F8
 Hot Key fields are available only on this form.
