# Active Project

The Active Project field on the Change Active Information form.
Enter the project number to make active. Press F4 for a list of valid projects.
The project specified here defaults each time you access
 a form with a Project key field (such as PM Projects, PM Change Orders, PM Copy Project, PM Material Detail,
and PM Meeting Minutes). Changing the project in any of those forms automatically updates the active project specified here.
