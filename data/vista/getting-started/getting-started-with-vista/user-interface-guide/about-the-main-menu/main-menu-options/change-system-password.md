# Change System Password

You can change your system password at any time from the Main Menu.
If your company is using the Security feature, you
 will need to be given access to this form in VA Form Security.

If you logged in using a Domain User login, the Change
 Password option is disabled.Logging in as a Domain User uses your Windows login. If you
 need to change the password for your Windows login, refer to the Microsoft Windows Help for
 instructions.

1. From the Main Menu, select Options > Change Password.

1. In the Old
 Password field, enter your current password.

1. In the New
 Password field, enter a new password.

1. Confirm the new password in
 the Retype
 New Password field.

1. Click OK.

The
 new password is saved and activated.
Log out and log in using the new password.
