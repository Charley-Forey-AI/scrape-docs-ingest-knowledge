# About the System Information Form

This form provides information about your system, such as hardware resources,
 components, and software environment.
You can access this form from the Main Menu or any form menu by selecting Help > System > System Information.
This is a standard Windows utility, so for information about this
 feature and/or how to navigate through the information, refer to the help that Windows
 provides in the utility (Help > Contents).
