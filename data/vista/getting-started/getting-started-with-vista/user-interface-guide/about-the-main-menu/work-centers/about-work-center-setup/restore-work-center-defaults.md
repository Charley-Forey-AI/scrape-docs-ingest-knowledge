# Restore Work Center Defaults

You can remove your changes to Work Centers and restore the default
 settings.
Proceed if you want to reset the Work Centers on
 your application window to the settings for the profile selected in the
 User Profile drop down.
 For example, if you added queries to a PM Work Center menu but want to
 remove them, you can restore the standard menu on the PM Work Center, which
 removes the queries that you added to the menu.

1. From the main
 application window, select Options > Work
 Centers. The Work Centers form opens.

1. Click Reset.

1. Close and reopen
 the application.
