# Configure the Report Section

In the Report section of the Dashboard Work Center, you can
 select an SSRS report part, enter report parameters, and view and print the
 report.
The Report section displays SSRS (SQL Server
 Reporting Services) report parts, not Crystal reports. If you have the
 Business Intelligence module, there are some standard SSRS report parts
 included in the Business Intelligence library. For more information, see
 SSRS - Adding Standard SSRS Reports to Work Centers.

1. In the Report section, click
 Edit. The system displays the
 Report Configuration window.

1. Select an SSRS report from the
 Report drop-down field. The Memo and
 Description fields display information about the report.
 This information is contained in the RP Report Titles
 form.

1. Use the Maximum
 Rows field to set the maximum number of
 records that you want to appear.

1. On the Parameters tab, use the
 Value field to configure each report
 parameter, as applicable.

1. Click OK to
 close the Report Configuration window. The system displays
 the report in the Report section.
