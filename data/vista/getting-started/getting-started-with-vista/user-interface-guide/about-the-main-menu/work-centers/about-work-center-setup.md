# About Work Center Setup

Get started on setting up your organization's Work Centers.
Setting up the Work Centers feature has two basic steps:

1. Admin set up - a user with access to the VA Viewpoint Administration module must set up Work Centers. For more information, see [Setting Up a Work Center - Admins](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/about-the-main-menu/work-centers/about-work-center-setup/set-up-a-work-center---admins).

1. User set up - once the admin set up is complete, you can add Work Centers to your Vista application windows and set them up for their own use. Changes you make only affect you. For more information, see:

- [Adding a Work Center - Users](/en/vista/vista/getting-started/getting-started-with-vista/user-interface-guide/about-the-main-menu/work-centers/about-work-center-setup/add-a-work-center---users)
