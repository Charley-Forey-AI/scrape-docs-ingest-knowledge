# Audit and Update Licenses in Vista

If you use on-demand licensing to have access to named
 Vista licenses when you need them, you
 should periodically review your named user count to make sure your company does not exceed
 the number of named user licenses purchased in your contract. Your company will be charged
 for any additional named user licenses.
Note: This information applies only to
 administrators of Vista organizations
 with a Named User License (NUL) model using on-demand licensing.
Users with a named user license have a User Type of Vista and a License Type of Office:

1. From the Vista main menu, go
 to Viewpoint Administration > Reports > VA User License
 Allocation and run the report.After running the report, you will see a list of all named
 users that count toward your billing.
For information about this report, see [VA User License Allocation](/en/vista/vista/reports-catalog/viewpoint-administration-reports/viewpoint-administration-general-reports/va-user-license-allocation).

1. Review each user listed on the report and
 determine which users require named user licenses. Standard users of the
 Vista application will need named user licenses.You only need to review named users belonging to your
 company and third-party consultants.
Trimble Viewpoint does not charge for
 named users that are used by Trimble Viewpoint personnel,
 such as KDSLookup or VCSSupport accounts. These Trimble named users are automatically filtered out of
 on-demand billing quantities.
Tip: If you are
 unsure if a user will be filtered out, you should probably remove that
 user.

1. Deactivate any named users that you do not need
 and those who do not require full access to the Vista
 application.

1. Go to Viewpoint Administration > Programs > VA User
 Profile.

1. Enter the User Name of the
 named user you want to deactivate.

1. On the Info tab, select the Deactivated
 checkbox.

1. Save the record.

1. Alternatively, you can reassign User Type and License Type for users to
 change their license type instead of deactivating their profile.

1. Go to Viewpoint Administration > Programs > VA User
 Profile.

1. Enter the User Name of the
 named user you want to reassign.

1. If needed, reassign the User Type.Note: Only reassign the User Type to
 User Application for data collection
 users, not standard users of the Vista
 application.

1. If needed, on the Info tab, reassign the
 License
 Type.For more information, see [License Types](/en/vista/vista/getting-started/getting-started-with-vista/vista-licensing/license-types).

1. Save the record.
