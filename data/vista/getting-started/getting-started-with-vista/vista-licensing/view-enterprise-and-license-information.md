# View Enterprise and License Information

In Trimble Construction One,
 administrators can view the enterprise name and license information.

Select your name in the upper right corner and select Admin Center.Note: If you don't see Admin Center, you have not been set up as an administrator. Contact your System Administrator for assistance.

If you have more than one enterprise, select it from the Admin Center list.

The selected Enterprise Info tab appears.

The Enterprise Info tab shows the following:

- The General section displays the Enterprise name.

- The License (Assigned /
 Contracted) section displays the product licenses in use (on
 the left side of the forward slash) and the total number of available licenses
 (on the right side of the slash). You can export this information to a .CSV
 file using the Export Data button.

- (For Team only) The Enterprise
 Logo section allows you to upload a logo for your Enterprise.
 Once uploaded, the logo appears in your Team email notifications.
