# About Trimble Assistant in Vista

Trimble Assistant is a chat agent you can use to ask specific questions about the tasks you can perform in your product.
To start a chat, from the Vista main menu, go to Links > Vista Help Assistant. The Vista Help Assistant is only available to Vista users who have
 Trimble ID credentials. In order to access the assistant at [vistaagent.trimblecms.com](https://vistaagent.trimblecms.com/), you must sign in with your Trimble ID.
Important:Trimble Assistant is trained only on
 product-related content on the Trimble Help website. This help
 assistant does not have access to any of your company's data or any data within
 the application database.

## Tips for Chatting With Trimble Assistant

- Trimble Assistant is still learning. To improve results, ask specific questions. If you do not get a helpful answer the first time, try rephrasing your question and asking it again.

- Trimble Assistant is looking for feedback. Select the Rate button in the lower right of the chat window to rank and comment on the answer to your question. Your feedback will be used to improve the assistant and documentation as needed.

## Trimble Assistant FAQs

Can Trimble Assistant perform tasks within the product?No. Trimble Assistant cannot complete tasks in your product. However, it can provide detailed instructions on how you can complete a given task.
Can I ask Trimble Assistant context-specific questions? For example, what is my vendor balance?No. Trimble Assistant can answer only product-related questions that are covered in the Help content.
What should I do if I don't see the Trimble Assistant link in my
 Vista Links?If you do not see the assistant added to your Vista Links menu, and
 you have a Trimble ID login, you can add the link yourself.

1. From the Vista main menu, go to Links > Organize Links.

1. In the Organize Links window, select Add
 Link.

1. In the Add/Edit Link window, enter the following values:

- Name: Vista Help
 Assistant

- Link Address:
 https://vistaagent.trimblecms.com/

1. Select OK to add the link.

1. Select OK again to close out of the
 window.
