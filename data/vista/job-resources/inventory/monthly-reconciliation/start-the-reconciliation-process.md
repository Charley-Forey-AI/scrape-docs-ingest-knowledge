# Start the Reconciliation Process

Begin the month-end reconciliation process in the IN Monthly Reconciliation form.
Before you start your month-end reconciliation

- print the "IN Inventory Totals by Matl and Location" report, which provides a summary of information used in the reconciliation process.

- See [How the Reconciliation Process Works](/en/vista/vista/job-resources/inventory/monthly-reconciliation/how-the-reconciliation-process-works) to review and understand the process.

- Close or clear any open batches.

1. In the Month field, enter the month you want to reconcile.

1. Select Initialize.  The form populates beginning values based on the prior month’s ending values, and a record is created for each location and material.

1. Select Print to print and review the Reconciliation Report.

1. Select Update to create a batch of Adjustments for correcting entries.  The IN Batch Process form displays.

1. Process the batch normally. For more information, see [ About the IN Batch Process Form](/en/vista/vista/job-resources/inventory/production/about-the-in-batch-process-form).

You can edit Adjustment batches in the IN Adjustment Entry form, but it is not recommended unless you plan to reprocess the month.
