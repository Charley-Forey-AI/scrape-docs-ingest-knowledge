# IN Physical Count Worksheet Form

Use this form to perform physical count updates. Count information is stored until added to an Adjustment batch.
Worksheets, and the resulting adjustment batches, are stored by user. When a worksheet is created, only the user who created the worksheet will have access to it. Materials existing on a user's worksheet (for a location) cannot be added to another user's worksheet for that same location. When the adjustment batch is created, only materials marked as ready for the current user's worksheet(s) are added to the batch. Materials marked as 'ready' on worksheets created by other users will be bypassed.
Once you create a worksheet, you can either manually add materials to the worksheet or you can add them via initialization. Initialization allows you to add materials to the worksheet by category range, material range, or a combination of category and material, for one or more locations, and to include the system count. For more information, see [Initialize the Physical Count Worksheet](/en/vista/vista/job-resources/inventory/monthly-reconciliation/initialize-the-physical-count-worksheet).

## Print a Worksheet

Once you have manually added or initialized materials to this form, you can then print the worksheet so that you can record actual quantities as you perform a physical count of the inventory. To print the worksheet, select File > Physical Count Worksheet Report, save to a PDF, then print the PDF.
