# IN Physical Count Initialize Form

This form can only be accessed by selecting File >  Initialize from the IN Physical Count Worksheet form, and is used to initialize materials to the worksheet.

## Initialize a Worksheet

Although you can manually enter materials on the physical count worksheet, it is much easier to use the initialize feature to add materials. Initialization allows you add materials to the worksheet by category range, material range, or a combination of category and material, for one or more locations, and to include the system count. For more information about initializing worksheets, see [Initialize the Physical Count Worksheet](/en/vista/vista/job-resources/inventory/monthly-reconciliation/initialize-the-physical-count-worksheet).

## Clear a Worksheet

Although this form is typically used to initialize worksheets, you can also use it to clear (or partially clear) worksheets. As with initialization, you can specify a range of categories and/or a range of materials, and select the locations to clear. You also have the option to clear only those materials that are not flagged as Ready or clear all materials, regardless of whether they are flagged as Ready. For more information, see [Clear a Physical Count Worksheet](/en/vista/vista/job-resources/inventory/monthly-reconciliation/clear-a-physical-count-worksheet).
