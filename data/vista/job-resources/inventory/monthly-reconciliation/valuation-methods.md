# Valuation Methods

The valuation method is used for monthly reconciliation. It allows you to capture beginning, ending, and summarized transaction activity by month and/or to make adjusting entries to correct inventory values when "ins" and "outs" are not posted in sequence.
There are two valuation methods available: FIFO and LIFO. Select the links below for more information about these valuation methods.
[Valuation Method: FIFO (First In, First Out)](/en/vista/vista/job-resources/inventory/monthly-reconciliation/valuation-methods/valuation-method-fifo-first-in-first-out)
[Valuation Method: LIFO (Last In, First Out)](/en/vista/vista/job-resources/inventory/monthly-reconciliation/valuation-methods/valuation-method-lifo-last-in-first-out)
