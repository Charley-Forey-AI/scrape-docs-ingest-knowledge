# About Committed Costs

Material Orders represent a job's committed costs, which are
 stored in the JCCP (Costs By Period) database table.
When a material order is set up, the total committed costs for the job
 increases. These costs only change when material orders are added, changed, or deleted.
 The Remaining Committed Costs column in the JCCP database table accumulates all of the
 order information and is then decreased each time a material order is confirmed (in the
 Material Order Confirmation form).
