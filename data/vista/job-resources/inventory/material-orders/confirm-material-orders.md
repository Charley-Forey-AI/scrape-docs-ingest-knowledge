# Confirm Material Orders

Use the IN Material Order Confirm Init form to quickly confirm material orders.

1. Enter the confirmation date and material order numberall items on the material order automatically display in the grid.

1. (Optional) Confirm all units for all items by selecting
 the Confirm the
 'Remaining Units' for all Items checkbox.

1. Edit confirmed units if needed.

1.  If you want the batch to include items with zero
 confirmed units, select the Include Items where 'Confirm This Time' equals
 zero checkbox.

1. click the Update button to create the confirmation batch.All confirmed items will be cleared from the grid

1. Repeat the process for each material order you need to confirm.The system adds each material order to the existing batch.

1. Post the batch by selecting File > Batch Process.If you want to

 review/edit transactions before posting, go to the IN Material Order Confirmation form.
