#  About the IN Add Transaction to Batch Form

Use this form to add material orders to the currently selected
 batch.
 Access this form by selecting File > Add Material Order in the IN Material Order Entry form.
Enter the material order number in the IN Material
 Order field and click Add to Batch. The material order is now
 included in the batch.
