# Field Definitions: IN Location Groups Form

The following is a list of field descriptions for the IN
 Location Groups form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Location Group

 Enter a number (0-255) to identify this location group.

##  Description

 Enter a description of this location group, up to 30 characters.

##  Location

 This field is display only and shows location numbers assigned to the
 Location Group (in IN Locations).

##  Description

 This field is display only and shows the description of the location
 assigned to the Location Group (in IN Locations).
