# Field Definitions: EM WO Item Init Form

The following is a list of field descriptions for the EM WO
 Item Init form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Component

 This field is only accessible if you checked the Assigned Component option.
 Specify the component for which to initialize items for this work order. Must be a valid component for the primary equipment specified in the work order header. At least one standard maintenance group must be set up for the component in EM Standard Maintenance Groups.

##  Standard Maintenance Group

 Specify the standard maintenance group for the primary equipment or component (if Assigned Component option checked) from which to initialize items for this work order.
Note: If items in the standard maintenance group exist on other open work orders, a warning displays, but entry is allowed; however, initialization will only add those items that are not already on an open work order.
