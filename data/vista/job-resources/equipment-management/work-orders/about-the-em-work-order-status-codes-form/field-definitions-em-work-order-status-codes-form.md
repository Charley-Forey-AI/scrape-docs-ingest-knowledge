# Field Definitions: EM Work Order Status Codes Form

The following is a list of field descriptions for the EM Work
 Order Status Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Status Code

 Enter a code, up to 10 characters, that identifies a work order status (e.g. Pending, Complete, Cancelled, etc.).

##  Description

 Enter a description of this status code, up to 30 characters.

##  Status Type

- Normal – Select this option to indicate a normal status type (i.e., requires no special handling). Use this status type for ‘beginning’ or ‘in progress’ status codes.

- Final – Select this option if the status code represents completion or final disposition of a work order item. Work order items with this type of status are eligible for deletion.

- Parts – Select this option when you want the work order status to show on the Parts reports. For example, you may not want any work order items with a ‘final’ status to show on the Parts report because they have been completed, but you would want to see those items that are ‘in progress’.
