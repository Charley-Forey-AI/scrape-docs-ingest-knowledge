# Field Definitions: EM Work Order Parts Status Form

##  Parts Status Code

 Enter a code, up to 10 characters, that identifies this WO parts status (i.e. Ordered, Received, Invoiced, etc.).

##  Description

 Enter a description of the status code, up to 30 characters.
