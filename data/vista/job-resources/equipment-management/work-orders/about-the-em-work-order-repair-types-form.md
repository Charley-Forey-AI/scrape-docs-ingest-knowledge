# About the EM Work Order Repair Types Form

Use this form to define the types of repairs you may want to track.
Some examples of repair types are:

- Scheduled Maintenance

- Accident

- Abuse

- Warranty

- Unscheduled Repairs

Once set up here, you can assign repair types to work order items. Repair types are used primarily to sort maintenance items for reporting purposes.
