# Field Definitions: EM Work Order Parts Init Form

The following is a list of field descriptions for the EM Work
 Order Parts Init form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Work Order

 Specify the work order (set up in EM Work Order Edit) for which to initialize parts.

##  Select Parts w/Status Code

 Enter the parts status code (from EM Work Order Parts Status) by which to restrict initialization of parts. Initialization will only include parts with this parts status.

##  Change to Parts Status Code

 Required.
 Specify the parts status code (from EM Work Order Parts Status) to assign to all parts initialized for this work order.
Initialize
 Click this button to implement the initialization process. Once initialization is complete, the message box at the bottom of this screen indicates the number of parts initialized. Additionally, a message displays asking if you want to initialize parts from another Work Order. Click Yes to continue or No to close the form and return to EM Work Order Parts Posting form.
