# About the EM Work Order Parts Init Form

Use this form (accessed by selecting File > Initialize Parts from Work Order in EM Work Order Parts Posting) to initialize parts to a work order.
You specify the work order from which to initialize parts, enter the status code by which to restrict the parts pulled (if applicable), and specify the new parts status to assign to the parts. Then click the Initialize button to implement the initialization process. Once initialization is complete, the message box at the bottom of the screen indicates the number of parts initialized. Additionally, a message displays asking if you want to initialize parts from another Work Order. Click Yes to continue or No to close the form and return to the EM Work Order Parts Posting form. All parts initialized are set up as separate sequences, which you can edit as necessary.
