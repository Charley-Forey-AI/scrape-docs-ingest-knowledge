# About the EM Work Order Initialize Form

Use this form to create work orders for equipment maintenance.
By entering specific criteria, you can create a list of equipment due for maintenance, and then create work orders for each item on the list. This is useful if you want to create work orders automatically for all maintenance items due for equipment at a specific job or location, for all equipment within a specified category, or for a range of maintenance groups.
