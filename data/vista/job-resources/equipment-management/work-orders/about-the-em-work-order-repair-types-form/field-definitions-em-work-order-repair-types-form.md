# Field Definitions: EM Work Order Repair Types Form

The following is a list of field descriptions for the EM Work
 Order Repair Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Description

 Enter a description of this repair type, up to 30 characters.

##  Repair Type Code

 Enter a code, up to 10 characters, that identifies this repair type (e.g. Maint, Warranty, Accident, etc.).
