# About the EM Work Order Parts Status Form

Use this form to set up and define status codes used when posting parts on work orders.
Some examples are:

- Issued

- Ordered

- Received

- Invoiced

- Completed

Once set up here, you can assign these status codes when posting parts to work orders in EM Work Order Parts Posting and EM Work Order Edit. You should designate one of these status codes as the default ‘Beginning Parts Status’ in EM Company Parameters. The system will automatically assign this status to all newly added parts on a work order.
