# Field Definitions: EM Shops Form

The following is a list of field descriptions for the EM
 Shops form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Shop

 Enter a code, up to 20 characters, that uniquely identifies this shop.

##  Description

 Enter a description of this shop, up to 30 characters.

##  Address

 Enter the address of this shop, up 30 characters.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Add'l Address

Enter the additional address for this shop, up to 30 characters, if applicable. For example, if this shop receives its mail at a P.O. Box, you might enter the P.O. Box in the address field above, and use this field to enter the street address.

##  City

 Enter the city in which this shop is located, up to 20 characters.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

## State

Enter a valid state (as defined in HQ States). The system validates the state based on the Default Country specified in HQ Company Setup for the active company. If not valid, an error displays. You must then enter a valid State for this Country in the State field or a valid Country for this State in the Country field.
Note: The name of this field differs depending on the
 Default
 Country specified in HQ Company Setup. For the United States (US) and
 Australia (AU), the label displays as "State". For Canada (CA), the label displays as
 "Province".
If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Zip

 Enter the zip code, up to 12 characters.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Country

 Enter the 2-character country code. Entry in this field is required when the address exists outside the Default Country specified in HQ Company Parameters for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Phone

 Enter the shop’s phone number, up to 20 characters.

##  Fax

 Enter the shop’s fax number, up to 20 characters.

##  Manager

 Specify the manager or supervisor for this shop. Up to 30 characters allowed.

##  INCo

 Enter a valid IN company (from IN Company Parameters) for this shop, if applicable. Defaults the IN Co# specified in EM Company Parameters.

##  IN Location

 Enter a valid inventory location (from IN Locations) for this shop, if
 applicable.

##  Last WO #

This field is only accessible if you selected
 the Auto Sequence
 Work Order # checkbox and the Shop work order option in EM Company
 Parameters (Work Orders tab).
 Enter the last work order number for this shop, up to 10 digits. For example, to begin auto-numbering work orders at ‘100’, enter 99 here. After the initial entry, the system updates this field automatically.
