# Field Definitions: EM Equipment Code Change Form

The following is a list of field descriptions for the EM
 Equipment Code Change form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Old Equipment

 Enter the equipment code to change.
 Once you Tab or Enter off this field, the system runs a check to see if the code you are trying to change is already in the process of being changed by another user. If it is, a message displays telling you it is being changed and you cannot go any further.
 If the code is not being changed and the equipment code exists in any open batches, you will receive a warning. You will need to post all batches listed on the Open Batch List before you can change the equipment’s code.
 If the equipment does not exist in any open batches, the Table List tab shows all tables in which the code exists, as well as the number of records in each table.

##  New Equipment

 Enter the new equipment code and click the Change button to implement the change. The system will cycle through all tables listed in the grid and update the new equipment code to all records referencing the old equipment code.
