# Field Definitions: EM Component Types Form

The following is a list of field descriptions for the EM
 Component Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Component Type

 Enter a code, up to 10 characters, that identifies this component type (e.g. ENGINE, FIN DRIVE, TRANS, etc.).

##  Description

 Enter a description of this component type, up to 30 characters.

##  Cost Code

 Enter the cost code to be used as the default when posting costs to this component. Default may be overridden at time of posting if the ‘Allow Cost Code Change’ option is checked in EM Company Parameters (Work Orders tab).
