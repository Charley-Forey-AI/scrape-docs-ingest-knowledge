# About the EM Component Types Form

Use this form to set up the different types of components associated with equipment, such as engines, transmissions, drivelines, and so forth.
By using component types, mechanics and data entry operators can easily associate a component with a piece of equipment without having to enter the component code. For example, if work is being done on the engine of a specific piece of equipment, the equipment number and component type (engine) can be entered, and the system will pull up the engine assigned to the specified equipment. If multiple components of the same type exist for a specific piece of equipment, the F4 lookup can be used to determine which component to use.
When you set up each component type, you have the option to specify a cost code. If a cost code is entered here, it will be used as the default when posting costs to the component.
