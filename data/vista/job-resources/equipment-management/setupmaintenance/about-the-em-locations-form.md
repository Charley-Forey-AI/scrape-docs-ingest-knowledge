# About the EM Locations Form

Use this form to define location codes for the location-tracking feature.
Location tracking is done with both a job code and a location code, either of which may be blank. So a piece of equipment may be assigned simply to a job (job code with no location), a non-job location such as a shop, or to a combination job code/location code (designating a specific location on that job).
 Locations are active by default, but you can designate a location as inactive. An inactive location can no longer be used for new records, but its history remains and is available on reports.
