# About Meter Security

Users will not automatically have access to the EM Meter
 Reading Adjustment form.
You will need to provide access to all
 applicable employees using VA Form Security.
Important: Because this form can potentially affect
 numerous meter reading transactions, it is suggested that it be used with extreme
 caution and that its use be restricted to only those users who should have access to
 it.
