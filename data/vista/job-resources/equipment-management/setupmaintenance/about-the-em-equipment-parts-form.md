# About the EM Equipment Parts Form

Use this form to set up all of the parts associated with a piece of equipment.
Parts set up here will be included in the lookups available for some part code inputs.
You can set up both valid and non-valid parts (those not associated with HQ materials) in this form; however, the use of non-valid parts when posting transactions depends on the ‘Validate Parts’ option in EM Company Parameters (Parts tab). If checked, posting programs (e.g. EM Cost Adjustments, EM Work Order Edit, etc.) will only allow entering parts that are associated with valid HQ materials. If unchecked, posting programs will allow entry of any part code, regardless of whether it is associated with a valid HQ material.
Note: You can also use this form to set up fluids (such as oil, transmission fluid, brake fluid, etc.) and their quantities.
