# About the EM Equipment Code Change Form

Use this form to change equipment codes and have the new codes updated to all Viewpoint tables.
You can change only one equipment code at a time. However, if the equipment code
 exists in multiple companies, it will be changed in all companies.
It is recommended that you backup your
 database and verify backup before initiating a change. It is also recommended that
 you limit access to this form.
