# About the EM Standard Maintenance Groups Form

Use this form to define groups of maintenance tasks for a piece of equipment.
Each group can have several individual tasks (including a list of parts needed for those
 tasks), but all tasks in one group must be for maintenance that will be done at the same
 interval.
Setting up maintenance groups allows you to:

- Track the date and time equipment is next due for
 maintenance by comparing the Last Done meter readings to the equipment’s current
 meter reading (EM Meter Readings).

- Easily create maintenance work orders by identifying
 the maintenance group for a piece of equipment so that the work order is
 automatically set up with pre-defined tasks and parts.

- Track component maintenance on an individual basis
 without referencing the primary piece of equipment.
