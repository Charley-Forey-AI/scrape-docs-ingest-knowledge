# About the EM Revenue Template Copy Form

Use this form (accessed via File > Copy Revenue Template in EM Revenue Templates) to copy revenue templates.
The copy ‘from’ template automatically defaults the template on which you currently have focus; however, you can copy from any existing template. The copy ‘to’ template must be a new template—you cannot copy to an existing template.
You have the option to copy category and/or equipment rates from the existing template. All revenue codes/rates and revenue breakdowns defined by category and/or equipment for the existing template are copied to the new template.
Once you click the Copy button, the copy process creates the new template. You can modify the category/equipment rates for the new template as needed.
Note: Template Type (Percent of Standard Rate or Rate Override) defaults from the existing template and cannot be changed.
