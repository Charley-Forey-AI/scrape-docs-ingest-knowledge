# Field Definitions: EM Revenue Breakdown Codes Form

The following is a list of field descriptions for the EM
 Revenue Breakdown Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Revenue Breakdown Code

 Enter a code, up to 10 characters, that identifies this revenue breakdown. For example, if you have a revenue code of HOURLY, you might set up the following breakdown codes:
Breakdown Code
Description

1
Ownership Costs

2
Depreciation Costs

3
Fuel Costs

4
Maintenance Costs

##  Description

 Enter the description of the revenue breakdown code. The description can be up to 30 characters long.
