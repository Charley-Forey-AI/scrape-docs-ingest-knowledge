# About the EM Usage Rules Table Form

Use this form to define rules tables for automatic usage posting to jobs.
The rules table specifies which revenue code will be used to determine usage rates for a piece of equipment based on the number of hours the equipment is on the job. For example, you may use an hourly rate up to 6 hours and switch to a daily rate when you exceed 6 hours.
You also have the option to specify whether to look at the job-to-date hours or billing period hours. If using job-to-date hours, the billing rate will be determined by the total number of hours the equipment has been on the job since it was first placed on the job. If you are using billing period hours, the billing rate will be determined by the total number of hours the equipment has been on the job during the billing period.

## Usage Rules

The Usage Rules tab is used to define the usage rules for the table. For each revenue code you set up, you specify the minimum and maximum number of hours the equipment must be used on the job in order for the revenue code (and its defined rates) to apply.
Rules tables, once set up here, are assigned to auto usage templates in the EM Auto Usage Posting Template form.
