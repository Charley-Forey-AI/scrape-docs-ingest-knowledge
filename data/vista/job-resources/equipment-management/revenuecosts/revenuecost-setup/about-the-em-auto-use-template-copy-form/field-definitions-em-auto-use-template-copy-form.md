# Field Definitions: EM Auto Use Template Copy Form

The following is a list of field descriptions for the EM Auto
 Use Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Copy From: Template

 Specify the auto usage template (from EM Auto Usage Posting Template) to copy. Source template must exist in the active company; no cross-company copying allowed.

##  Copy to: Description

 Enter a description for the new auto usage template, up to 30 characters.

##  Copy to: Template

 Specify the auto usage template to which you are copying the existing template. Destination template must be new; cannot copy to an existing template. Up to 10 characters allowed.

##  Include Equipment

 Check this box to copy the equipment setup from the source template to the new template.
 Leave this box unchecked if not copying the equipment setup from the source template. System will only copy the category setup.
