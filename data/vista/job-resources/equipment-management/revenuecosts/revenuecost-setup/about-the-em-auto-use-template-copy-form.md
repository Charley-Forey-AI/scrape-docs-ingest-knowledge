# About the EM Auto Use Template Copy Form

Use this form to copy auto usage posting templates.
You can copy only the category setup to the new template or specify to include the equipment setup as well.
Once you specify the template to copy, enter the new template and description, and indicate whether to include the equipment setup.
Note: For equipment-only templates (those with no category setup),
 you must check the Include Equipment box to ensure the
 equipment information is copied.
 When you have completed entering the copy criteria, click the Copy button. The system will copy the category and/or equipment setup (as applicable) from the existing template to the new template, along with any notes defined for the source template. Once the copy process is complete, you can use the EM Auto Usage Template form to edit the new template as necessary.
