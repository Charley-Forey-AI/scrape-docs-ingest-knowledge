# Field Definitions: EM Revenue Template Copy Form

The following is a list of field descriptions for the EM
 Revenue Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  From Revenue Template

 Enter the revenue template to copy. Defaults the template on which you currently have focus (in EM Revenue Template)

##  To Revenue Template

 Enter the revenue template to which you are copying information from the source template. Template must be new; cannot copy to an existing template.

##  To Revenue Template: Description

 Enter a description of the new template, up to 30 characters. If not selecting to copy category and/or equipment rates, click OK to implement the copy process.

##  Category Rates

 Check this box to copy the category rates/revenue breakdown from the existing revenue template to the new template.
 Leave this box unchecked if not copying category rates/revenue breakdown to new template.
 Click OK to implement the copy process.

##  Equipment Rates

 Check this box to copy the equipment rates/revenue breakdown from the existing revenue template to the new template.
 Leave this box unchecked if not copying equipment rates/revenue breakdown to new template.
 Click OK to implement the copy process.
