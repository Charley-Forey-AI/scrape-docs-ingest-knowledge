# About Work Units

The Work Units with Usage option determines whether the
 productivity of all pieces of equipment assigned to a specific category will be tracked
 during usage posting.
For example, you may want to track how many tons of rocks are hauled by all
 dump trucks in a specific category.
When you select this option, you will be
 able to post work units along with usage in the EM Usage Posting form. You will also
 need to specify the default unit of measure in which you want the equipment’s
 productivity stored in the EM Annual Revenues (EMAR) and EM Revenue Detail (EMRD) files.
 If the default is overridden during posting, only the EMAR file will be updated.
Work units are also updated to Job Cost when
 this option is in use. As with other JC updates, the units update the JC Cost Detail
 table (JCCD) in all cases, but only update the JC Costs by Period table (JCCP) if the
 unit of measure is the standard one for the specified phase/cost type.
