# About Show All Type Columns

This option is found in the Options menu and is only
 available when focus is on the Grid tab.
It allows you always show all columns for
 all line types (checked) or to only show the columns associated with the current line’s
 type (unchecked). For example, if unchecked and line type is ‘Equip’, only
 equipment-related columns display (e.g. Equipment, Comp Type, Component, etc.). If you
 then check the option, you will also see work order-related columns (e.g. Work Order, WO
 Item, etc.), regardless of line type. Keep in mind that displaying all line type columns
 will increase the size of the grid, which can make data entry more cumbersome.
Note: The intended use of this option is for viewing
 purposes only. Once you close the form, the grid automatically resets to the default
 (unchecked).
