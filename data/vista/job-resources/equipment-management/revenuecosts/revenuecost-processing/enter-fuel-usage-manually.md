# Enter Fuel Usage Manually

You can manually enter fuel usage transactions if you prefer to post usage for individual pieces of equipment rather than a group of equipment.
Manual entry provides more flexibility because you can enter transactions only for the equipment you want. In addition, you have the ability to override the default equipment information if necessary.

1. From the main menu, select Equipment Management  >  Programs  >  EM Fuel Posting .The Batch Selection form displays.

1. Create a new batch:

- Select Create a new batch.

- In the Batch Month field, enter the batch month.

- Click OK.

The EM Fuel Posting form displays.

1. In the Seq # field, enter N or + to add a new transaction.The system automatically assigns a sequence number for you.

1. In the Equipment field, enter the equipment or press F4 to select from a list of valid equipment.

1. In the EM Cost Code field, accept the defaulted equipment cost code (from EM Equipment) or enter a new cost code. Press F4 to select from a list of valid equipment cost codes.

1. In the EM Cost Type field, accept the defaulted equipment cost type (from EM Equipment) or enter a new cost type. Press F4 to select from a list of valid equipment cost types.

1. In the Usage Date field, enter or select the posting date for all equipment in this batch.

1. (Optional) In the IN Co field, enter the inventory company that provided the fuel or press F4 to select from a list of valid IN companies. Leave this field blank to include all inventory companies that provided fuel.

1. (Optional) In the IN Loc field, enter the inventory location that provided the fuel or press F4 to select from a list of valid inventory locations. Leave this field blank to include all inventory locations that provided fuel.

1. In the Fuel Matl Code field, accept the defaulted fuel material code (from EM Equipment) or enter a new material code. Press F4 to select from a list of valid fuel material codes.

1. (Optional) In the UM field, accept the defaulted unit of measure or press F4 to select from a list of valid UMs for the IN or HQ material.

1. In the Units field, enter the fuel usage units.

1. In the Unit Price field, accept the defaulted unit price or enter a new unit price.For information about how this field defaults, see the [Unit Price](/en/vista/vista/job-resources/equipment-management/revenuecosts/revenuecost-processing/about-the-em-fuel-posting-form/field-definitions-em-fuel-posting-form#ID-0000c2cf--en) F1 help.

1. (Optional) In the Tax Code field, enter the tax code or press F4 to select from a list of valid tax codes.

1. In the Trans Acct field, accept the defaulted GL account or press F4 to select from a list of valid GL accounts.For information about how this field defaults, see the [Transaction Account](/en/vista/vista/job-resources/equipment-management/revenuecosts/revenuecost-processing/about-the-em-fuel-posting-form/field-definitions-em-fuel-posting-form#ID-0000c2e1--en) F1 help.

1. (Optional) In the Offset Acct field, accept the defaulted GL account or press F4 to select from a list of valid GL accounts.For information about how this field defaults, see the [Offset Account ](/en/vista/vista/job-resources/equipment-management/revenuecosts/revenuecost-processing/about-the-em-fuel-posting-form/field-definitions-em-fuel-posting-form#ID-0000c2ed--en) F1 help.

1. In the New Odo field, enter the odometer reading at the time of refueling.

1. In the New Hours field, enter the hour meter reading at the time of refueling.

1. Save the record.

1. Enter additional transactions as needed. Once you have completed entering transactions, you can post the batch. For more information, see [EM Batch Process ](/en/vista/vista/job-resources/equipment-management/revenuecosts/revenuecost-processing/about-the-em-batch-process-form).
