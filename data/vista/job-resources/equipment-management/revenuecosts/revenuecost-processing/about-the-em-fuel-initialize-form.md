# About the EM Fuel Initialize Form

Use this form to post fuel usage by batch.
Access this form from EM Fuel Posting by selecting File > Fuel Posting Initialize.
 Posting fuel usage in this form has two advantages:

- You can select equipment based on search criteria - for example all of the equipment on a job.

- Quick data entry - The system provides standard defaults (i.e. equipment, cost code, cost type, usage date, location, and fuel material code) based on information set up for the equipment in EM Equipment.

Once you initialize transactions, you are returned to the EM Fuel Posting form, where you can edit transactions as needed and post the batch.

## Post Fuel to Equipment Assigned To

This section allows you to use a set of
 criteria to filter the equipment pulled into the batch. You can filter by JC Co,
 location, category, department, and/or shop. The more criteria you enter, the more
 restrictive the selection will be. For example, you can post fuel usage for all
 equipment on a specific job, or you can restrict entries based on job, location, and
 category.
After you enter the desired criteria,
 click the Refresh button to pull all equipment into the batch meeting the criteria.
 You can repeat this process as needed until you have added all desired equipment.
 Then use the grid to enter the units, miles, and hours.
