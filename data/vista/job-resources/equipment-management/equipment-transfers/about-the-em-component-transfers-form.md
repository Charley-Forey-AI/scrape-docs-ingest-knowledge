# About the EM Component Transfers Form

Use this form to transfer a component from one piece of equipment to another.
When you enter the component code, the
 equipment to which the component is currently attached displays in the Prior Master
 Equipment field (Info tab). You must then specify the equipment to which
 you are transferring the component and if applicable, the reason for the transfer.

## Equipment On

Once you specify the equipment to which you are transferring the component, this
 section defaults the current hours, miles/kilometres, and fuel readings at the
 ‘transfer in’ time for both the equipment and the component. You should not need to
 change this information; however, if you do make changes, be aware that the system
 will not update the changes to the Equipment Master (EMEM). You will need to specify
 the effective date of the transfer; that is, the date you transferred the component
 to the equipment.

## Equipment Off

The ‘transfer off’ date and hours,
 miles/kilometres, and fuel readings in this section are disabled and null for each
 new transaction you enter; however, the system will update these fields once you
 transfer the component to another piece of equipment. You can edit this information
 if necessary, but the system will not update the changes to the Equipment Master.
Note: If you set up a new component in EM Equipment, a
 new transfer record is automatically set up here. The Prior Master
 Equipment field defaults to ‘No Info’, since there is no previous
 transfer record available. The ‘transfer to’ equipment will be the equipment to
 which you assigned the component in EM Equipment. Hours, miles/kilometres, and fuel
 for both the equipment and component will default from EM Equipment.
