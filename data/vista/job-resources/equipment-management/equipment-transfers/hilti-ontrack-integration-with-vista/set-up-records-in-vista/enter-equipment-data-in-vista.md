# Enter Equipment Data in Vista

Enter or update equipment data in the Vista EM Equipment record. This data will go into Hilti ON!Track Assets.
Important: If you add a new piece of equipment in ON!Track, the integration creates a new equipment record in Vista and links these two records together. However, you must *make all changes to this record in Vista*, as the data syncs from Vista into ON!Track.

1. From the main menu, select Equipment Management > Programs > EM Equipment.

1. On the Info tab, enter values in the required fields:
 Category and Department.

1. Enter information in the following fields:

- Description

- VIN/Serial #

- Location

- JC Co (Job Cost Company)

- Rev Code (Revenue Code)

1. Go to the Revenue Rates tab and enter a
 Rate for each Rev Code in the grid.
