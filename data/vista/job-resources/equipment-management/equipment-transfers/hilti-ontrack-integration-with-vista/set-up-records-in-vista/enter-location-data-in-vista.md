# Enter Location Data in Vista

Enter project and equipment location data in Vista. This data will go into Hilti ON!Track Locations.

- Project location is the geographical location of the job site. When a piece of equipment is being used on a particular project, it should be assigned to the project location.

- Equipment location is any location that is *not* at the project location. When a piece of equipment is not being used on a project, it should be assigned according to where it is located, such as in the shop. Equipment location designations can include the following types of sites:

- Yard

- Trailer

- Shop

1. Enter project location data in Vista. This is data about the geographical location of your project.

1. From the main menu, select Project Management > Programs > PM Projects.

1. Enter a Description to name the project.

1. Enter Mailing and Shipping address information for the project location.

Alternatively, you could also update these fields on the JC Jobs form.

1. Enter equipment location data in Vista.This is data about the location of equipment not currently in use for a project, such as the equipment yard, trailer, or shop.

1. From the main menu, select Equipment Management > Programs > EM Locations.

1. On the Info tab, enter the
 equipment's Location, or press F4 to select a
 location from the lookup list.

1. Use the auto-populated Description, or enter your own text to describe the equipment location.

1. Select the Active checkbox to activate the location.

1. Select the Notes tab to enter additional information about the equipment location.

1. Select the Grid tab to view or edit the location fields.

1. Save the record.
