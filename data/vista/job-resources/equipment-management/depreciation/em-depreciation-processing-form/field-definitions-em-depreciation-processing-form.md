# Field Definitions: EM Depreciation Processing Form

The following is a list of field descriptions for the EM
 Depreciation Processing form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Date

 Specify the date to assign as the posting date for all transactions in this batch.

##  Post Depreciation for Select Assets

 Check this box to post depreciation only for the equipment and/or asset specified below. Use this option to bring an asset that has been added or changed up to date with the rest of your assets. You can only use this option after you have posted depreciation for all assets.
 Leave this box unchecked if posting depreciation for all assets.
Note: The system will only update the ‘Last Month Calculated’ in EM Company Parameters when posting depreciation for all assets (i.e. when this option is unchecked).

##  Equipment

 This field is only accessible if
 you checked the ‘Post Depreciation for Select Assets’ option above.
 Specify the equipment for which to post depreciation. If a specific asset is not indicated in the Asset field below, depreciation will be posted for all assets set up for this equipment.
Note: If you specify a piece of equipment flagged as
 'Inactive’' (in EM Equipment), you will receive a warning that the equipment's status is
 inactive, but you will be allowed to proceed.

##  Asset

 Specify the asset (for the equipment specified above) for which to post depreciation. If left blank, depreciation will be posted for all assets of the equipment specified above.
