# About Depreciation

The Depreciation feature in EM provides you with the ability to set up equipment asset records so that the system will create and post depreciation entries to GL when you run the EM Depreciation Processing program.
You can depreciate all assets or selected assets, and generate these depreciation adjustments monthly, quarterly, or yearly. Additionally, you can set the Calculations Required option (in EM Company Parameters) so that your closing in General Ledger checks to make sure you have created these entries before allowing you to close the GL month or year.
For more information about the basic steps for setting up depreciation, see [Set Up Equipment Depreciation](/en/vista/vista/job-resources/equipment-management/depreciation/set-up-equipment-depreciation).
