#  About the MS Price Template Copy Form

Use this form to copy an existing pricing template.
 Access this form by selecting File > Copy Template from MS Price Templates.
 This form defaults the currently active MS company and price template, but allows you to copy from other existing templates. The destination template can be a new or existing template. If you are copying to an existing template, a warning displays stating that the system will only add new records to the template.
 Select the Copy Source Price Template Notes checkbox to copy notes from the source template to the new template. If you are copying to an existing template, the system disables this field and you cannot copy notes, although you can use a manual cut-and-paste procedure.
