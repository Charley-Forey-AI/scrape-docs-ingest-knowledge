# Field Definitions: MS Price Template Copy Form

The following is a list of field descriptions for the MS
 Price Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Source Price Template

 This field automatically defaults the currently active price template. Accept the default or enter the price template to copy. Press F4 for a list of valid templates.

##  Destination Price Template

 Enter the destination price template, up to four digits. If you enter an existing price template, a warning displays. If you continue, the system only adds records that do not already exist on the template.

##  Description

 Enter a description for the new price template, up to 30 characters.
Note: If copying to an existing template, this field is inaccessible.

## Copy Source Price Template Notes

Note: This field is inaccessible if copying to an existing template.
Check this box to copy all notes entered for the source template (in MS Price Templates) to the new template.
Do not check this box if you do not want notes for the source template copied to the new template.
