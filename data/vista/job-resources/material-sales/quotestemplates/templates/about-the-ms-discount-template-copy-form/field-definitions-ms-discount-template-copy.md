# Field Definitions: MS Discount Template Copy

The following is a list of field descriptions for the MS
 Discount Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Source Discount Template

 This field automatically defaults the currently active discount template. Accept the default or enter the price template to copy. Press F4 for a list of valid templates.

##  Destination Discount Template

 Enter the destination discount template, up to four digits. If you enter an existing discount template, a warning displays. If you continue, the system only adds records that do not already exist on the template.

##  Description

 Enter a description for the new discount template, up to 30 characters.
Note: If copying to an existing template, this field is inaccessible.

##  Copy Source Discount Template Notes

Note: This field is inaccessible if copying to an existing template.
 Check this box to copy all notes entered for the source template (in MS Payment Discount Templates) to the new template.
 Do not check this box if you do not want notes for the source template copied to the new template.
