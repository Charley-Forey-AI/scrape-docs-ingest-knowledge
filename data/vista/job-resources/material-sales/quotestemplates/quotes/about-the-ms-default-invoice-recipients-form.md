# About the MS Default Invoice Recipients Form

Use this form to override the default invoice recipients defined on the AR Default MS Recipients form in AR Customers. Access this form by clicking the Default Recipients button on the Invoices tab of the MS Quotes form.
 You will only need to use this form if the default invoice recipients you defined for a customer differ for invoices associated with quotes for the selected customer.
