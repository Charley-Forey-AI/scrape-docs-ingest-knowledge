# Field Definitions: MS Quote Copy Form

The following is a list of field descriptions for the MS
 Quote Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Quote Type

 Specify the type of quote to copy. This will also be the quote type for the destination quote.

##  Source: Select By

 Select how to identify the source quote. This field determines what set of ‘purchaser’ fields display.

##  Source: Quote

 This field is accessible when you select “1-Quote #” in the Select By drop-down field. Use for Customer, Job, and Inventory quotes.
 Specify the quote number to copy. Press F4 for a list of valid quote numbers.

##  Source: Customer

 This field is accessible when you select “2-Purchaser” in the Select By drop-down field. Use for Customer quotes.
 Specify the customer (from AR Customers) from which to copying quote information.

##  Source: Customer Job

 This field is accessible when you select “2-Purchaser” in the Select By drop-down field. Use for Customer quotes.
 If applicable, specify the customer job to copy quote information from. Press F4 for a list of jobs for the specified customer.

##  Source: Customer PO

 This field is accessible when you select “2-Purchaser” in the Select By drop-down field. Use for Customer quotes.
 If applicable, specify the customer PO# to copy quote information from. Press F4 for a list of job POs for the specified customer.

##  Source: JC Co#

 This field is accessible when you select “2-Purchaser” in the Select By drop-down field. Use for Job quotes.
 Specify the Job Cost company to copying quote information from.

##  Source: Job

 This
 field is accessible when you select “2-Purchaser” in the Select By drop-down field. Use for
 Job quotes.
 Specify the job (from JC Jobs) to copy quote
 information from.

##  Source: IN Co#

 This field is accessible when you select “2-Purchaser” in the Select By drop-down field. Use for Inventory quotes.
 Specify the Inventory company to copy quote information from.

##  Source: To Location

 This field is accessible when you select “2-Purchaser” in the Select By
 drop-down field. Use for Inventory quotes.
 Specify the ‘sell to’ location (from IN
 Locations) to copy quote information from.

##  Destination: Customer

 Use for Customer quotes.
 Specify the customer (from AR Customers) for the copied quote information.

##  Destination: Customer Job

 Use for Customer quotes.
 Specify the customer job for the copied quote information, up to 20 characters.

##  Destination: Customer PO

 Use for Customer quotes.
 Specify the customer purchase order for the copied quote information, up to 20 characters.

##  Destination: JC Co#

 Use for Job quotes.
 Specify the job for the copied quote information, up to 20 characters.

##  Destination: Job

 Use
 for Job quotes.
 Specify the job (from JC Jobs) for the copied
 quote information, up to 20 characters.

##  Destination: IN Co#

 Use for Inventory quotes.
 Specify the customer purchase order for the copied quote information, up to 20 characters.

##  Destination: To Location

 Use for Inventory quotes.
 Specify the ‘sell to’ inventory location
 (from IN Locations) for the copied quote information, up to 20 characters.

##  Destination: Quote

 For Customer, Job, or Inventory quotes.
 If you are selected the ‘Auto Generate Quote #s’ checkbox (MS Company Parameters), enter “N” or “+”, and the system automatically generates a quote number based on the ‘Last Used Quote #’ field (MS Company Parameters).
 If you are assigning quote numbers manually, enter a number, up to 10 characters. The number must not already exist in the system.

##  Copy: Quote Detail

 Check this box to copy quote detail from the source quote. The system only copies materials and their pricing;-all units initialize as 0.00.

##  Copy: Price Overrides

 Check this box to copy pricing overrides from the source quote.

##  Copy: Discount Overrides

 Check this box to copy payment discount overrides from the source quote.

##  Copy: Haul Codes Defaults

 Check this box to copy haul code defaults from the source quote.

##  Copy: Haul Codes Overrides

 Check this box to copy haul rate and minimum amount overrides from the source quote.

##  Copy: Haul Codes

 Check this box to copy haul zone defaults from the source quote.

##  Copy: Pay Codes

 Check this box to copy pay code defaults and pay rate override information from the source quote.

##  Copy: Quote Header Notes

 Check this box to copy quote header notes from the source quote.

## Quote Header Custom Fields

 Check this box if you want to copy any custom fields on the Info tab of the MS Quote form to the destination quote.
By default this box is not checked.
