# Pay Rate Hierarchy - Table 1

The following table identifies the levels and search order used by the system to find
 a default pay rate when the pay basis is “1-Per Unit”, “4-Units Per Mile”, 5-Units Per Hour”,
 or "7-Percent of Surcharge Total".
Note: Pay basis 7-Percent of Surcharge Total is used for
 surcharges only, and will determine the amount of the assessed surcharge that will be
 passed to the haul vendor.
Each column in the table represents a
 column on the Rates tab in MS Pay codes. An X denotes a supplied value. The first row
 represents the minimum set of information required for a pay code with one of the three
 specified pay basis rates. Each successive row provides additional information and
 overrides all preceding levels. For example, rates defined by location for a unit-based
 pay code override rates defined for the location group, rates defined by material
 override rates defined for the category, and so forth.
