# Pay Rate Hierarchy - Table 2

The following tables identify the levels and search order used by the system to find a
 default pay rate when the pay basis is “2-Per Hour”, “3-Per Load”, or “6-Percent of Haul”.
Each column in the table represents a column on
 the Rates tab in MS Pay codes. An X denotes a supplied value. The first row represents the
 minimum set of information required for a pay code with one of the three specified pay
 basis rates. Each successive row provides additional information and overrides all
 preceding levels. For example, rates defined by material override rates defined for the
 category, and so forth. Pay Basis:  2-Per
 Hour  3-Per Load  6-Percent of Haul
