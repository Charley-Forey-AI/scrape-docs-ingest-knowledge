# Field Definitions: MS Truck Types Form

The following is a list of field descriptions for the MS
 Truck Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Truck Type

 Enter a code, up to 10 characters, that identifies this truck type.

##  Description

 Enter a description of this truck type, up to 60 characters.
