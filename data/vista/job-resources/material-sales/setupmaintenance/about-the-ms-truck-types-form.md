# About the MS Truck Types Form

Use this form to set up the types of trucks that are used for hauling/delivering materials - for example 3 Axle End Dump or 6 Axle Belly Dump.
You can use truck types in the following ways:

- Assign them to trucks using the [ MS Vendor Trucks ](/en/vista/vista/job-resources/material-sales/setupmaintenance/about-the-ms-vendor-trucks-form) form

- Assign them to rates when setting up haul rates in [MS Haul Codes ](/en/vista/vista/job-resources/material-sales/setupmaintenance/about-the-ms-haul-codes-form) - for example if a haul rate applies to a specific type of truck

- Assign them to rates when setting up vendor pay rates in [MS Pay Codes ](/en/vista/vista/job-resources/material-sales/setupmaintenance/about-the-ms-pay-codes-form) - for example if a pay rate applies to a specific type of truck

Note: At least one truck type must be set up if you will be posting haul charges when entering tickets.
