# Field Definitions: MS Surcharge Groups Form

The following is a list of field descriptions for the MS
 Surcharge Groups form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Surcharge Code

Enter a valid surcharge code (from MS Surcharge Codes) to add to this surcharge group.
 You must assign a minimum of one surcharge charge code to this group in order for surcharges to be assessed during ticket entry (MS Ticket Entry); however, you can assign as many surcharges to the group as necessary.

## Description

Enter a description of this surcharge group, up to 30 characters.

## Surcharge Group

Enter a number (0-255) to represent this surcharge group.
