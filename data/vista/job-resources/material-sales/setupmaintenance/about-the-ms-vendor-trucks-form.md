# About the MS Vendor Trucks Form

Use this form to set up all of the trucks operated by outside haul vendors.
Assign each truck a truck number and truck type unique to the vendor. The system uses gross, tare, and capacity weights entered for each vehicle as defaults when entering tickets. If you do not use an average tare weight when weighing trucks, it is suggested that this field be left at zero. In other words, if you obtain a new tare weight each time a truck hauls a load, then it is not necessary to set up a tare weight default here because you will enter the tare weight when entering the ticket.
 You can enter other information, such as the licensing information and permit/insurance dates, for each truck, but these fields are informational only and are not used anywhere in the system.
