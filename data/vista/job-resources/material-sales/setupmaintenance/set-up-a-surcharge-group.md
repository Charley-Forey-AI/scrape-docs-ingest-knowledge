# Set Up a Surcharge Group

Set up a surcharge group.

1. In the Surcharge
 Group field, enter a number (0-255) to represent the surcharge
 group.

1. In the Description field, enter the surcharge group description.

1. Click the Codes
 tab.

1. Using the Surcharge
 Code field, enter the surcharges to include in this surcharge
 group. Press F4 to select from a list of
 valid surcharge codes.
