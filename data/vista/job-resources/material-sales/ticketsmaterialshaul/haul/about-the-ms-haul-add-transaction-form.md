# About the MS Haul Add Transaction Form

Use this form to pull existing hauler time sheets into the
 current batch.
You can access this form by selecting File > Add Time
 Sheets from MS Hauler Time Sheet Entry. You can pull a time sheet into the
 batch using either the freight bill number or the haul transaction number.
Only hauler time sheets posted within the same batch
 month that are not currently in another batch can be added to the current batch.
 Time sheets can then be edited or marked for deletion, and the batch posted as
 normal.
