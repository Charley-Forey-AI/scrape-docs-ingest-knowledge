# Field Definitions: MS Haul Add Transaction Form

The following is a list of field descriptions for the MS Haul
 Add Transaction form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Freight Bill

 Specify the freight bill number of the haul transaction you want to add to the batch. Press F4 for a list of haul transactions by freight bill number. Click the Add to Batch button to add the transaction to the time sheet batch.

##  Haul Transaction

 Specify the haul transaction to add to the batch. Press F4 for a list of valid haul transactions by transaction number. Click the Add to Batch button to add the transaction to the time sheet batch.
