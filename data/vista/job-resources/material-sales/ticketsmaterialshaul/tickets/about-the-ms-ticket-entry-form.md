# About the MS Ticket Entry Form

Use this form to enter tickets for material sales to customers, jobs, and/or inventory.
You can enter tickets in two ways:

- Upload from a PC Scale package in IM Import

- Manually enter the ticket in MS Ticket Entry

If you will be uploading tickets from a Scale package, use the Imports (IM) module to import, edit, and upload the work file into Material Sales.
Note: Certain setup options in MS Company Parameters, as well as the Ticket Entry options in the Options menu, may affect data entry in this form. See MS Company Setup Options and Ticket Entry Options in Related Topics below for information.
