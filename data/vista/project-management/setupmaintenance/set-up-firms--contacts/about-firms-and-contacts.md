# About Firms and Contacts

The Project Management module includes firms and contacts so
 that you can associate documents and records with project contacts - for example vendors,
 subcontractors, architects, and suppliers.
 Firms and contacts are available to all users of the software, depending on security permissions. For example, some companies will limit the number of persons who can set up Firms.

## AR Customers, AP Vendors, and PM Firms

The firms and contacts in the PM module are unique to PM, but you
 can import customers from the AR Customers form, or vendors from the AP Vendors
 form.

You can import a range of AR customers into PM Firms using the PM Customer Range form
 (PM
 Firms > File > Initialize Customer Firms). After a customer is imported into PM Firms, any change that you
 make using PM Firms will not update the customer
 record in AR Customers.
You can also import a range of AP
 vendors into PM Firms using the PM Vendor Range form (PM
 Firms > File > Initialize Vendor Firms). After a vendor is imported into PM Firms, any change that you make
 using PM Firms will update the vendor record in
 AP Vendors based on the selections in the Allow Add from PM and Allow Update from
 PM checkboxes (AP Company Parameters from, Audit Options tab).
