# About the PM Add Firm Contact Form

Use this form to create a new PM firm contact when using the
 Create and Send feature to generate a project document or send a communication.
You can only access this form from PM Send Documents by clicking in the To, Cc, or Bcc rows of the Message tab. The PM Send Documents form is accessed from the Create and Send menu by clicking on any of the following options:

- Send Document

- Send Message

- Send with Transmittal

You will typically use this feature when a contact you wish to include in an
 email is not already set as a contact in PM Firm Contacts. Once you add the contact in this
 form, the contact is added to PM Firm Contacts and you can edit the new contact's information
 as needed.
