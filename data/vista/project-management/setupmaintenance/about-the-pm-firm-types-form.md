# About the PM Firm Types Form

Use this form to create and maintain the different types of firms with which you do business.
Firm types allow you to identify different categories of firms, such as
 vendors, subcontractors, suppliers, engineers, and architects, to name a few. Once you
 set up firm types, you can then assign them to firms in PM Firms. You will typically use
 firms type for reporting purposes.
