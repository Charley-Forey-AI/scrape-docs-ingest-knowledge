# About the PM Employee Range Form

This form is accessed by selecting the Initialize
 Contacts option from the File menu of PM Firm Contacts, and is used to
 initialize contacts from employees set up in PR Employees.
Note: The Initialize Contacts option is only enabled when
 the firm selected in the Firm Number field on PM Firm Contacts
 is the same firm defined in the Our Firm field on the Info tab in
 [PM Company Parameters ](/en/vista/vista/project-management/setupmaintenance/pm-company-parameters-form).
You select the range of employees to
 initialize and the initialization process will set up each employee as a contact with a
 number identical to the employee number. If an employee’s number already exists in PM,
 that employee is skipped during initialization and must be added manually.
Other information, such as the employee’s
 name, sort name, phone numbers (phone, fax, and mobile), and email address will default
 into the corresponding fields in PM Firm Contacts. The employee’s designated Craft will
 default as the contact’s Title. Information can be modified in PM Firm Contacts as
 necessary; however, changes made to the contact’s information will not be updated in PR
 Employees.
