# Create a Firm Type

Create a firm type in the PM Firm Types form.

1. Open the PM Firm Types
 form.

1. Click the New Record
 icon ().

1. Enter the firm type that
 you would like to create in the Firm Type field - for example
 vendor, supplier, architect, engineer, etc.. The text entered in this field is
 used to select the firm type in the PM Firms form.

1. Enter a description of
 the firm type in the Description field. This field
 will also display when selecting a firm type, and it can be up to 30 characters
 long.

1. Use the Notes tab to
 enter any notes on the firm type.

1. Click the Save icon
 () when complete.

1. You can now add the
 field type to a firm using the Firm Type field on the PM
 Firms form.
