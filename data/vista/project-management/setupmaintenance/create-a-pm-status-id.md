# Create a PM Status ID

Create status ID codes to use in tracking change orders, submittals, RFIs, and other documents.

1. From the main menu,
 select Project Management > Programs > PM Status IDs.

1. Click the New Record
 icon to create a new record.

1. In the Status
 Code field, enter a name for the new status id. This will be
 used to identify and select the status ID.

1. From the
 Project Options drop-down, select how to handle
 pending change orders assigned this status ID in projects. For more information,
 see the [F1](/en/vista/vista/project-management/setupmaintenance/pm-status-ids-form/field-definitions-pm-status-ids-form#ID-0002d22c--en) help.

1. In the Type
 of Code section, select the type that best identifies this code. The code types (beginning, intermediate, and final) are used in reporting and
 other places in the application. See the [F1](/en/vista/vista/project-management/setupmaintenance/pm-status-ids-form/field-definitions-pm-status-ids-form#ID-0002d24b--en) help more information.

1. If the status code
 applies to all document categories, select the Active for All
 Forms checkbox. Otherwise, leave unselected.

1. If you did not select
 the Active for All Forms checkbox, use the
 Document Category field to indicate the category to
 which this status code applies. Click [here](/en/vista/vista/project-management/setupmaintenance/pm-status-ids-form/field-definitions-pm-status-ids-form#ID-0002d2f7--en) for more information.

1. Click
 Save.
