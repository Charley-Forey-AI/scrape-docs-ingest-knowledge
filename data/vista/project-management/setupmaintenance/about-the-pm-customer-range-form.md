# About the PM Customer Range Form

Use this form to create PM firms using the customers set up in AR Customers.
Access this form from PM Firms by selecting File > Initialize Customer Firms.
When creating firms from customers, each customer in the selected range is set up as a firm in PM Firms with a firm number that is identical to the customer number. If the firm number or sort name already exists in PM Firms, the customer is skipped and you must manually set them up in PM Firms.
Once the initialization process is complete, you can edit the firms as needed. Changes made to a firm's information are not updated to the related customer in AR Customers.
