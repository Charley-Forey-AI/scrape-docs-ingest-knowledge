# About the PM Phase Submittals - 6.5 Form

Use this form to set up a standard set of submittals by phase. For each applicable phase, you define each of the submittal documents that are commonly needed when the phase is included on a project. Then when initializing submittals for projects using [PM Submittal Initialization - 6.5](/en/vista/vista/project-management/issuessubmittalsother-documents/submittals/about-the-pm-submittal-init---6.5-form), if the phase exists on the project and is assigned the submittal document type requested, it will be initialized and set up on the project automatically.
