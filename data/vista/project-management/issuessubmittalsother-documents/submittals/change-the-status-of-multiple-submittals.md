# Change the Status of Multiple Submittals

You can change the status of multiple submittals at one time using the Update Submittal Register form.
You can individually change the status of a submittal using the Status field in the Submittal Register grid. However, if you want to change the status of multiple submittals at once, do the following:

1. From the main menu, select Project Management > Programs > PM Submittal Register.

1. In the grid, select the submittals to change, using the SHIFT key for consecutive selection or the CTRL key for random selection.

1. From the toolbar, select Tasks > Update Submittal Register.The PM Submittal Register Update form displays.

1. In the Status field, enter the status or press F4 to select from a list of valid statuses.

1. Click OK.

You are returned to the PM Submittal Register form and the system updates each of the selected submittals with the new status.
