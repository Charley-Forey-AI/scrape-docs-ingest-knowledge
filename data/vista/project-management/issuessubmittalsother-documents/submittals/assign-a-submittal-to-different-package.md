# Assign a Submittal to Different Package

You can change the package assignment for a submittal via PM Submittal Package.
To change the submittal package assignment for a submittal, you must first remove the submittal from the existing submittal package, and then add it to the new submittal package.The following steps describe this process.

1. From the main menu, select Project Management > Programs > PM Submittal Package.The PM Submittal Package form displays.

1. In the Project field, enter the project for the submittal or press F4 to select from a list of projects.

1. In the Package field, enter the submittal package to which the submittal is assigned. Press F4 to select from a list of packages for the project.

1. In the Revision field, enter the package revision to which the submittal is assigned or press F4 to select from a list of revisions for the submittal package.The submittals grid (bottom of the form) displays the submittals assigned to the selected package/revision.

1. In the grid, select the submittal you want to reassign.

1. Click the Delete () button in the toolbar.The submittal is removed from the submittal package.

1. Return to the Package field and enter the new package to which you are assigning the submittal.

1. In the Revision field, enter the package revision or press F4 to select from a list of revisions for the submittal package.

1. In the Seq field of the grid, enter the sequence number of the submittal you are reassigning or press F4 to select the submittal from a list of submittals.

The selected submittal is added to the new package.
