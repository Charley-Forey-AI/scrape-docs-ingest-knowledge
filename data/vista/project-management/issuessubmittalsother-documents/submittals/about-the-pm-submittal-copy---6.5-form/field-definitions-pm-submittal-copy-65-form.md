# Field Definitions: PM Submittal Copy-65 Form

The following is a list of field descriptions for the PM
 Submittal Copy-65 form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project

 Required.
 Specify the project from which to copy submittals.

##  Submittal Type

 Specify the submittal type by which to restrict submittals displayed in the grid. Must be a valid submittal type (i.e. set up in PM Document Types with a category of 'Submittal'). Leave blank to include all submittal types.

##  Copy Items

 Check this box to copy submittal items from the source project to the destination project. All submittals initialized to the grid will default with this option checked. May be overridden.
 Leave this box unchecked if you do not submittal items copies. All submittals initialized to the grid will default with this option unchecked. May be overridden.

##  Copy Revisions

 Check this box to copy submittal revisions from the source project to the destination project.
 Leave this box checked to copy only original submittals (those with Revision of ‘0’) from the source project.

## Responsible Person

Required.
Specify the contact to designate as the responsible person for all submittals copied to the destination project. Must be a valid contact for the 'Project Our Firm' in PM Projects or, if no ‘our firm’ specified for the project, the ‘Our Firm’ specified in PM Company Parameters. May be overridden in PM Submittals - 6.5.
Refresh - Click this button to populate the grid with all submittals meeting the selection criteria.
Copy - Select the submittals to copy by highlighting them in the grid, using the Shift key for consecutive selection or the Ctrl key for random selection. Then click this button to initiate the copy process.

##  Date Due

 Specify the due date to be assigned to all submittals copied to the destination project. If no date is specified here, Date Due will default as null for all copied submittals.
Refresh - Click this button to populate the grid with all submittals meeting the selection criteria.
Copy - Select the submittals to copy by highlighting them in the grid, using the Shift key for consecutive selection or the Ctrl key for random selection. Then click this button to initiate the copy process.

## Generate Next Submittal Numbers

Check this box to generate new submittal numbers based on the numbering scheme specified for the project in PM Projects.
Leave this box unchecked if you do not want submittal numbers generated. Copied submittals will be assigned the same submittal number as the source submittal.
Refresh - Click this button to populate the grid with all submittals meeting the selection criteria.
Copy - Select the submittals to copy by highlighting them in the grid, using the Shift key for consecutive selection or the Ctrl key for random selection. Then click this button to initiate the copy process.
