# Field Definitions: PM Phase Submittals-65 Form

The following is a list of field descriptions for the PM
 Phase Submittals-65 form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Phase

 Enter the phase you want set up on a submittal. If this phase’s assigned Document Type is specified when initializing submittals in PM Submittal Entry, this phase will be added to the document.

##  Document Type

 Enter a valid document type for this phase. Document type must be assigned the ‘SUBMIT’ document category.

##  Sequence

 Create a new phase submittal
When creating a new phase submittal, you are creating it for the selected phase code and document type.
There are several ways to create a new phase submittal:

- Enter a "+" and then press TAB to exit the field. The system will
 automatically create a new phase submittal and assign it the next available number.

- Click the New Record icon () and then
 complete the Phase Code and Document Type fields. The Sequence
 field will automatically populate with the next available number. Press TAB to accept
 the default and create the new phase submittal.

- Enter a phase submittal number that does not exist for the phase
 code and document type. You can also press F4 to see a list of phase submittals that
 have already been created.Select an
 existing phase submittal

Open the Grid tab and select a phase submittal.

## Description

Enter a description for the submittal document. The description can be up to 30 characters long.
