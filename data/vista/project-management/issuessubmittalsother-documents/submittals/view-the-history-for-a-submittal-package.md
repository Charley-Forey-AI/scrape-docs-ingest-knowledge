# View the History for a Submittal Package

Use the PM Document History form to view all of the changes that have been made to a submittal register or submittal package / package revision.
To view the history for a submittal package:

1. From the main menu, select Project Management > Programs > PM Submittal Package.

1. From the toolbar, select Tasks > Document History.The PM Document History Detail form displays.

1.
The Grouping Bar displays (in dark gray) at the top of the grid.

1. Drag-and-drop the Submittal Pkg and Submittal Pkg Rev column headings into the grouping bar.The submittals are now grouped by submittal package and revision.

1. To view the changes made to a specific revision, expand the submittal package and then the specific package revision.
