# Field Definitions: PM Transmittal Add Form

The following is a list of field descriptions for the PM
 Transmittal Add form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Transmittal

 Enter a new transmittal number, or enter a
 ‘+’ if you want the system to automatically assign it the next available number.
Existing Transmittal
Press F4 and select an existing transmittal
 from the list that displays.
Selecting an existing transmittal will add the
 originating document to the Documents tab of the [PM Transmittals](/en/vista/vista/project-management/issuessubmittalsother-documents/transmittals/about-the-pm-transmittals-form) form,
 and many of the fields on the PM Transmittal Add form will be disabled since this
 information has already been set up using the PM Transmittal form.

## Subject

Enter a subject line for this transmittal, up to 255 characters.
This field defaults based on the originating form.
Note:This field is disabled if you selected to add this document to an existing transmittal.

##  Transmittal Date

Enter the date of this transmittal. Initially defaults the current date.
 This field is disabled if you selected to add this document to an existing transmittal.
 Date Shortcuts
T
 or tSet the date
 to the current date.
MMDDFour digit month and
 day
Enter a four
 digit month and date (MMDD) and the system will automatically add the current
 year.
+The system
 will automatically set the date to tomorrow.
+5The system
 will automatically set the date to 5 days in the future. You can actually enter any
 value after the +, for example you can enter +7 to set the date to next
 week.

-The system
 will automatically set the date to the previous day.
-5The system
 will automatically set the date to 5 days in the past.Just like with +, you can enter
 any value after the -, for example you can enter -7 to set the date to the previous week.

## Responsible Person

Enter the responsible person for this transmittal. Must be a valid contact defined for this project’s ‘Our Firm’ in PM Firm Contacts. If the originating form has a responsible person already assigned, the designated responsible person defaults here and this field is disabled.
Note:This field is disabled if you selected to add this document to an existing transmittal.

## Copy Document Attachments to Transmittal

Check this box to copy the attachments from the current record in the originating form (e.g. PM Pending Change Orders, PM Other Documents, etc.) to this transmittal.
Leave this box unchecked if you do not want to copy attachments from the current record to this transmittal.
