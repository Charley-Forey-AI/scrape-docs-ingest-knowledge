# Copy a Request for Quote

If you have RFQs that are similar, you can save time when creating new RFQs by copying an existing RFQ using the PM Request for Quote Copy form.

1. Open the RFQ that you want to copy in the PM Request For Quote form.

1. Select Tasks > Copy RFQ.The PM Request for Quote Copy form opens, defaulting the RFQ you selected in PM Request For Quote.

1. In the Project field, select the project for which you are creating the new RFQ. This can be the same project or a different project.

1. The RFQ field automatically defaults the next available RFQ number for the selected project. Accept the default or enter a new RFQ number.

1. The Description field defaults the description of the RFQ you copied. Accept the default or enter a new description.

1. Select OK.

The new quote is created and opened in the PM Request for Quote form. You can modify the RFQ as needed.
