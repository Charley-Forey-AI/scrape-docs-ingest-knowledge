# PM Request For Quote Copy Form

Use the PM Request For Quote Copy form to copy an existing RFQ. This can save you time when creating a new RFQ that is similar to an existing one. Using this feature you can copy an RFQ to a different project, or to the same project.
When you copy an RFQ, the system copies everything on the RFQ except the following:

- Contacts on the Distribution tab

- Changes orders associated with the RFQ using the Related Items feature (these are the change orders displayed on the Change Orders tab on the PM Request For Quotes form).

Once the copy process is complete, you can modify the new RFQ as needed.
