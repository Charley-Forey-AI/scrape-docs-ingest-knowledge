# About the PM Copy RFI Form

Use this form to create a copy of an existing RFI.
For example, if you would like to create a new RFI that is a
 revision or clarification of an existing RFI, you can create a copy of the existing RFI and
 then make changes to the copy. This feature also allows you to close and copy an RFI, and
 then work from the copied version.
This form will copy all information on the selected RFI,
 including all responses to the original question, but it will not copy attachments or
 documents generated using the [Create and
 Send](/en/vista/vista/project-management/create-and-send/about-the-create-and-send-feature) feature.
