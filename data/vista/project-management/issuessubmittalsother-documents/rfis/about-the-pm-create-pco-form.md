# About the PM Create PCO Form

Use this form to create a pending change order from either a project issue or RFI.
You can open this form from either the PM Request
 For Information or PM Issues form. Click Create > Pending
 Change Order. Any default values are based on the ‘Schedule Impact and Cost’
 information entered on the RFI or project issue:

- The Change in Days field populates with the value in the Days
 field on PM Request For Information or PM Issues form.

- If the Fixed Amount Flag checkbox is
 selected and the PCO item's unit of measure is lump sum, the Fixed
 Amt field populates with the value entered in the ROM
 Price field in the PM Request For Information or PM Issues
 form.
