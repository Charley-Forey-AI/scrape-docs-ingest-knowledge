# Field Definitions: PM Request For Information Copy Form

The following is a list of field descriptions for the PM
 Request For Information Copy form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Project

Enter the project that should be associated
 with the new RFI or press F4 to select it from a list.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/rfis/about-the-pm-copy-rfi-form)PM Request For Information Copy

## RFI

Enter the RFI number you would like to create or press '+' to populate the field with the next available number.
You cannot select an existing RFI. The value in this field must be a unique number for the project and RFI type combination.
Press F4 to see the RFI numbers that have
 already been created.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/rfis/about-the-pm-copy-rfi-form)PM Request For Information Copy

## RFI to Copy

Enter the RFI to copy or press F4 to select it
 from a list.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/rfis/about-the-pm-copy-rfi-form)PM Request For Information Copy

## RFI Type

Enter an RFI type to assign it to the new RFI or press F4 to select an RFI type from a list.
RFI types are document types with a document
 category of 'Request for Information.' Document types are created and maintained in [PM Document Types](/en/vista/vista/project-management/setupmaintenance/pm-document-types-form) and document categories are assigned to a
 document type using the Document Category field in PM Document
 Types.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/rfis/about-the-pm-copy-rfi-form)PM Request For Information Copy

## Subject

Enter the subject of the new RFI. The text entered in this field will overwrite the subject of the RFI being copied.
The subject can be up to 60 characters long.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/rfis/about-the-pm-copy-rfi-form)PM Request For Information Copy
