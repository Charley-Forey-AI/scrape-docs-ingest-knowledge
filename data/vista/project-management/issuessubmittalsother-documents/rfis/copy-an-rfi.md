# Copy an RFI

You can create a copy of an existing RFI using the PM Copy RFI form.
For example, if you would like to create a new RFI that is a revision or clarification of an existing RFI, you can create a copy of the existing RFI and then make changes to the copy. This feature also allows you to close and copy an RFI, and then work from the copied version.

1. Open the PM Request for Information form.

1. From the toolbar, select Tasks  >  Copy Request for Information.The PM Copy RFI form displays.

1. In the RFI to Copy field, enter the RFI to copy or press F4 to select from a list of RFIs.

1. In the New RFI section, use the Project, RFI
 Type, and RFI fields to specify the new RFI to which you are copying the source RFI.Note: Document types are created and maintained in [PM Document Types ](/en/vista/vista/project-management/setupmaintenance/pm-document-types-form). Only document types with a document category of 'Request for Information' will be available in the RFI Type field.

1. In the Subject field, enter the subject of the new RFI. The value in this field will overwrite the subject of the RFI being copied.

1. Click OK.

 A message displays asking if you would like to open the new RFI. Click Yes to open the new RFI in PM Request For Information.
