# Field Definitions: PM Other Sub Initialization Form

The following is a list of field descriptions for the PM
 Other Sub Initialization form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project

 Required.
 Initially defaults to currently selected project. May be overridden.
 Enter the project for which to initialize subcontract documents.

##  Document Type

 Required.
 Enter the document type.Must be a valid, active document type with a category of 'Other' (as defined in PM Document Types).Press F4 for a list of document types.

##  Responsible Person

 Required.
 Specify the person from your firm who is responsible for this document.
Select SL’s to Initialize
 Using the grid below, select the subcontracts to initialize. Use the ‘Select All SL’s’ button to select all subcontracts or use the grid to manually select subcontracts. You can use the Shift or Ctrl keys to consecutively or randomly select SL’s (respectively).
 Once you have selected the desired SL’s, click the Initialize button to begin the initialization process.

##  Copy Attachments

 Check this box to copy attachments defined for a subcontract to the generated document.
 Leave this box unchecked if you do not want SL attachments copied.
Select POs to Initialize
 Using the grid below, select the subcontracts to initialize. Use the ‘Select All SL’s’ button to select all subcontracts or use the grid to manually select subcontracts. You can use the Shift or Ctrl keys to consecutively or randomly select SL’s (respectively).
 Once you have selected the desired SL’s, click the Initialize button to begin the initialization process.
