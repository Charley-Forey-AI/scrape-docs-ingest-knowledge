# Field Definitions: PM Other Documents Form

The following is a list of field descriptions for the PM
 Other Documents form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project

 Enter the project associated with the
 document or press F4 to select one from a list.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

## Document Type

Enter the document type. Must be a valid, active document type with a category of 'Other' (as defined in PM Document Types). Press F4 for a list of document types.
Note: To display a description field for this field on the
 Grid tab, access Field Properties (F3) and select an option from the Description in
 Grid drop-down.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

## Document

Use this field to identify a specific document
 of the type selected in the Document Type field.
Create a new document
There are several ways to create a new document using this field:

- Enter a '+' and then press TAB to exit the field. The system will automatically assign the new document the next available number.

- Click the New Record icon () and then complete the
 Document
 Type field. The system will automatically populate the Document
 field with the next available number. Press TAB to move through the Document
 field and create the new document.

- Enter a document number that has not been used and
 press TAB to exit the field. The system will automatically create a new document. You
 can also press F4 to see a list of documents that have already been created.

Note: The system will ignore alphanumeric numbers when
 determining the next number to assign. Select an existing document

 Enter an existing document number or press F4 to select one from a list.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

##  Description

 Enter a description of this document, up to 60 characters.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

##  Related Firm

 Enter a firm number or press F4 to select a
 firm from a list.
Tip:The sort name can be used instead of the firm number
 when selecting firms in PM module forms. Generally the sort name is the first several
 letters of the firm name.  For example when creating a subcontract in the [PM Subcontracts](/en/vista/vista/project-management/subcontracts/pm-subcontracts-form) form for 'Bryan Electrical', you can
 enter the sort name 'bryan' in the Vendor field instead of the firm
 number '10042'. The sort name of a firm is set up using the Sort Name
 field on the [PM Firms](/en/vista/vista/project-management/setupmaintenance/about-the-pm-firms-form) form.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

## Status

Enter the status (from PM Status IDs) of this document.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

## Responsible Party

Enter the person from your firm who is responsible for this document or press F4 to select them from a list.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents

##  Location

 Specify where the document is currently located. This may be a file path or a physical location.

##  Date Due

Generally this field is used to enter the date the document is due to the related firm.
Date field shortcuts
T or tSet the date to the current date.
MMDDFour digit month and day
Enter a four digit month and date (MMDD) and the system will automatically add the current year.
+The system will automatically set the date to tomorrow.
+5The system will automatically
 set the date to 5 days in the future. You can actually enter any value after the +, for
 example you can enter +7 to set the date to next week.

-The system will automatically set the date to the previous day.
-5The system will automatically set the date to 5 days in the past.Just like with +, you can enter any value after the -, for example you can enter -7 to set the date to the previous week.

[](/en/vista/vista/project-management/issuessubmittalsother-documents/other-documents/pm-other-documents-form)PM Other Documents
