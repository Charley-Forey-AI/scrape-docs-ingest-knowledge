# Update Cost Projections

You can update the actual and committed values included in a projections batch, and calculate the projected values based on the updated amounts.

1. From the main menu,
 select Project Management  >
 Programs > PM
 Cost Projections.

1. In the Projection
 Date field, enter the date of the projection batch or press
 F4 to select from a list of existing projection batches.

1. In the Project field, enter the project to work with.Note: If you used the F4 Lookup to select a
 projection date in Step 1, this field automatically defaults the
 project.

1. Click the Update Projections button below the grid.The PM Update Projection Options screen
 displays.

1. In the Write Over
 Plugged Values section, select how you want to update projection values. For
 information about each of the available options, see the F1 help.

1. From the Initialized Worksheet Detail Option drop-down, select how to initialize detail projections.

1. Click Update.

The system updates the cost information in the projections grid based on the selected update options.
