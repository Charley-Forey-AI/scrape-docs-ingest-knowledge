# Field Definitions: PM Update Projection Options Form

The following is a list of field descriptions for the PM
 Update Projections Options form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Write Over Plugged Values

The Write Over Plugged Values radio options on the PM Update Projection Options form.
The selection in this field does not affect the plugged values entered in the current batch.

- Never — Never write over existing plugged values.

-  Always — Always write over plugged values.

-  Only When Actual + Committed Costs Exceed Plugged Costs — Initialization will only write over plugged values when the actual plus remaining committed costs exceed the plugged costs.

-  Only When Actual + Committed Costs Exceed Plugged Costs, Keep as Plugged — Plugged values are only overwritten when the actual plus remaining committed costs exceed the plugged costs; however, values will remain flagged as ‘plugged’.This is the option used by most customers

## Initialized Worksheet Detail Option

The Initialized Worksheet Detail Option drop-down on the PM Update Projection Options form.
Select how to initialize detail to the projections worksheet:

- None

- Initialize Detail Projections with values

- Initialize Detail Projections without values
