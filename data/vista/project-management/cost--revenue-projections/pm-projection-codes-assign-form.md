# PM Projection Codes Assign Form

 Use the PM Projection Code Assign form to quickly remove or add phases and cost types to the selected projection code.
Once you create a projection code, you have the option to manually assign phases and cost types to the code using the Phase/Cost Type tab. This is typically only useful when you have a select few phases/cost types that you want assigned to the projection code. However, you can quickly assign multiple phases and cost types to a projection code by using this form to select a group of phases/cost types already associated with the project, and then assigning them to the projection code.
For more information, see [Assign Phases/Cost Types to a Projection Code](/en/vista/vista/project-management/cost--revenue-projections/assign-phasescost-types-to-a-projection-code).
