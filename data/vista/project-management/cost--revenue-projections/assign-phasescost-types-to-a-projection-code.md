# Assign Phases/Cost Types to a Projection Code

You can quickly assign phases and cost types to a projection code using the PM Projection Codes Assign form.
Phases/cost types assigned to a projection code must be assigned to the project in PM Project Phases.

1. Open the PM Projection Codes form.

1. In the Project field, enter the project to work with or press F4 to select from a list of valid projects.

1. In the Projection Code field, enter the projection code to which you are assigning phases/cost types or press F4 to select from a list of projection codes for the project.

1. In the form toolbar, select Assign Phases ().The PM Projection Codes Assign form displays, showing both the Available Phases and the Assigned Phases.

1. From the Available Phases selection box, select the phases/cost types to add. Use the SHIFT key for consecutive selection or the CTRL key for random selection.

1. Select Apply.

The system adds all selected phases to the Phase/CT grid (in PM Project Codes) for the projection code.
