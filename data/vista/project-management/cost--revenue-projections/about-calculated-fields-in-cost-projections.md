# About Calculated Fields in Cost Projections

The table below describes most of the calculated fields on the PM Cost Projections form (Grid and Phase/Cost Type tab), and PM Cost Projection Editor form.
These fields are disabled and may not display on the Grid tab (PM Cost Projections) of the based on your selected layout. For more information, see [PM Projection Layouts](/en/vista/vista/project-management/cost--revenue-projections/about-the-pm-projection-layouts-form).

 Field
Description

Actual + Committed Hours
Actual costs plus remaining
 committed hours for the phases and cost types associated with the
 projection code

Actual + Committed Cost
Actual costs plus the remaining committed costs for a
 phase and cost type

Actual Cost
Actual costs for the phases and cost types associated
 with the projection code

Actual Hours
Actual hours posted to JCCD/JCCP

Actual Production
The calculation of this
 field will vary based on the production type that is selected in the
 Production Type field
 or the  icon on the PM Cost Projection Editor
 form.

Actual Units
Actual units posted to JCCD/JCCP

Current Estimated Cost
Current estimated cost for the phases and cost types
 associated with the projection code

Current Estimated Hours
Current estimated hours of the phases and cost types
 associated with the projection code

Current Estimated Production
The calculation of this field will vary based on the
 production type that is selectedin the Production Type field or the  icon on the PM Cost Projection Editor
 form).

Current Estimated Unit Cost
Current estimated units to actual units

Current Estimated Units
Current estimated units for phases and cost types
 associated with the projection code

Current Estimate + Included Cost
Current estimated costs including pending change order
 / approved change order cost, and add-on cost

Current Estimated + Included Units
Current estimated units including pending change order
 / approved change order estimated units

Current Estimated + Included Hours
Current estimated hours including pending change order
 / approved change order estimated hours

Displayed CO Cost
Pending and approved change
 order estimates that have not been interfaced and are set up to be
 included in projections
PCO is assigned a document
 type set up to be included in projections (PM Document Types > Include
 in JC Projections field), and assigned a status that is set up to be
 included in projections (PM Status IDs > Projections Option field)

Forecast Final Cost
Projected cost at completion

Forecast Final Hours
Projected hours at completion

Forecast Final Unit Cost
Forecast final cost divided by forecast final
 units

Forecast Final Units
Displayed Projected Units at
 completion
The Projected Final Units
 calculation subtracts Previous Projected Units, but the Forecast Final
 Units calculation adds the Previous Projected Units resulting in the
 original Projected Units value.

Forecast Production
The calculation of this field will vary based on the
 production type that is selected in the Production Type field or the  icon on the PM Cost Projection Editor
 form).

Future Actual Cost
Actual costs posted to JCCD / JCCP after the
 projection date / month

Future CO Cost
Estimated pending change order and approved change
 order cost that has not been interfaced with a status ID of Display in
 Cost Projections

Future CO Units
Estimated pending change order and approved change
 order units that have not been interfaced

Included CO Cost
Estimated costs on pending
 and approved change orders that have not been interfaced with a status ID
 of Display and Calculate in Cost Projections

Included CO Hours
Estimated hours on pending and approved change orders
 that have not been interfaced

Included CO Units
Estimated units on pending and approved change orders
 that have not been interfaced

Original Estimated Hours
Original estimated value for the job, phase, cost
 type

Original Estimated Units
Original estimated value for the job, phase, cost
 type

Original Estimated Cost
Original estimated value for the job, phase, cost
 type

Over / Under Production
The calculation of this field will vary based on the
 production type that is selected in the Production Type field or the  icon on the PM Cost Projection Editor
 form.

Percent Complete Production
The calculation of this field will vary based on the
 production type that is selected in the Production Type field or the  icon on the PM Cost Projection Editor
 form.

Previous Production
The calculation of this field will vary based on the
 production type that is selectedin the Production Type field or the  icon on the PM Cost Projection Editor
 form.

Previously Projected Cost
The previously projected
 cost that was posted.
The value in this field also
 depends on the selection in the Previous projected through prior
 month field (PM Cost Projection Options).

Previously Projected Unit Cost
Previously Projected Cost divided by Previously
 Projected Units

Previously Projected Units
Previous Projected Units
 posted to JCCD / JCCP
The value in this field also
 depends on the selection in the Previous projected through prior
 month field (PM Cost Projection Options).

Projected Final Cost Variance
Difference between the
 Project Projected Final Cost and the Previously Project Cost.
The value in this field
 includes any approved change orders (entered in JC Change Orders only).
 The variance amount for ACOs is calculated as the Current Estimated Cost
 less the Original Estimated Cost.

Projected Final Production
The calculation of this field will vary based on the
 production type that is selected (PM Cost Projections form> Grid
 tab> Production Type field
 or the  icon on the PM Cost Projection Editor
 form).

Remaining Committed Cost
Remaining committed costs posted to JCCD/JCCP

Remaining Committed Hours
Remaining committed costs posted to JCCD / JCCP

Remaining Committed Units
Remaining committed units posted to JCCD / JCCP

Remaining (Projected Unit Cost)
Remaining projected cost / remaining projected
 units

Remaining (Projected Units)
Difference between projected final units and actual
 units (invoiced through AP)

Remaining Production
The calculation of this field will vary based on the
 production type that is selected in the Production Type field or the  icon on the PM Cost Projection Editor
 form.

Total Committed Cost
Total committed costs posted to JCCD/JCCP

Total Committed Units
Total committed units posted to JCCD/ JCCP

UM
This is the unit of measure
 associated with the projection code.
For information about using a common
 unit of measure, see [About Setting Common UMs for Projection Codes](/en/vista/vista/project-management/cost--revenue-projections/about-setting-common-ums-for-projection-codes).

Uncommitted Cost
(Estimated budgeted costs) - (total committed costs +
 PCO uncommitted cost)
