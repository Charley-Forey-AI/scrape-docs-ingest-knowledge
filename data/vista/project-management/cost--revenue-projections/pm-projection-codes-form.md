# PM Projection Codes Form

Use this form to create and maintain the projection codes used when entering cost projections in PM Cost Projections.
Projection codes allow you to group phases and cost types, and determine what level of detail you can use when entering cost projections in the PM Cost Projections form.
The following describes three ways to create projection codes.
Generate Projection CodesYou can generate project codes automatically based on the phases, cost types, phases and cost types, or contract items on the job. This can save you time and reduce data entry when creating cost projections using the enhanced cost projections process. For more information, see [Create Projection Codes Automatically](/en/vista/vista/project-management/cost--revenue-projections/create-projection-codes-automatically).Create Projection Codes ManuallyYou can manually create projection codes and assign specific phases and cost types to each code using the Phase/Cost Type tab or use the Assign Phases feature to quickly assign phases and cost types to each projection code. For more information, see [Assign Phases/Cost Types to a Projection Code](/en/vista/vista/project-management/cost--revenue-projections/assign-phasescost-types-to-a-projection-code).Copy Existing Projection CodesYou can copy projection codes from one project to another. There are two ways to accomplish this:

- Use the Copy Projection Codes option (Tasks menu) to copy the projection codes from one project to another. For more information, see [Copy Projection Codes](/en/vista/vista/project-management/cost--revenue-projections/copy-projection-codes).

- When copying projects in PM Copy Project, select the Projection Codes checkbox in the Additional Information to Copy section. Once you initiate the copy process, the system copies the projection code and other selected information from the source project to the destination project. For more information, see [PM Copy Project Form](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form).
