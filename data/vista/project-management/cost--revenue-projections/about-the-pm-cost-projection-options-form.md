# About the PM Cost Projection Options Form

 Use this form to set up the cost projection options. The
 selections on this form impact only your user account
This form automatically displays when the PM Cost Projections form is opened
 if the Show this window on form
 open box is checked. You can also access this form from PM Cost
 Projections by selecting File > Cost Projection Options.
Press F1 when any field in the form is selected for detailed information about
 that field.
