# About the PM Projection Layouts Form

Use projection layouts to control how the Grid tab on the PM Cost Projections form displays.
Customer projection layouts allow you to change what columns appear in the projections and the order in which they appear, as well as change column
 widths, add highlight colors, and change the column headings. The layouts that you create
 are saved in a library, and you can share these layouts with other users.
Once a projection layout has been applied, you can also
 manually adjust the column order and column width using the PM Cost Projections form.
 Any manual change made using the PM Cost Projections form will not affect the projection layout that was
 applied.
