# Copy Projection Codes

You can copy projection codes from one project to another using the PM Projection Code Copy form.

1. Open the PM Projection Codes form.

1. In the Project field, enter the source project or press F4 to select from a list of valid projects.

1. In the Projection Code field, enter the projection code to copy or press F4 to select from a list of projection codes for the project.

1. From the toolbar, select Tasks > Copy Projection Codes. The PM Projection Code Copy form displays, and the selected project defaults as the Source Project.

1. In the Destination Project field, enter the project to copy the projection to or press F4 to select from a list of valid projects.

1. Click Copy. Note: If project codes already exist for the destination project, a message displays warning that projection codes exist and will be cleared before the copy process. If you want to clear the old projection codes, select Yes. If you do not want to clear the existing projection codes, select No.

Projection codes are copied from the source project to the destination project.
