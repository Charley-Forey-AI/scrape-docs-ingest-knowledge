# Shared Tables

Several forms in the Project Management module share tables
 with forms in other modules.
PM Form
Module/Form
Shared Tables

Project Master
JC Jobs
JCJM

Project Phases/Cost Types
JC Job Phases/JC Original Estimates
JCJP, JCCH

Contract Master/Items
JC Contracts/Items
JCCM, JCCI

PO Header, PO Items
PO Purchase Order Entry
POHD, POIT

SL Header, SL Items
SL Subcontract Entry
SLHD, SLIT

Note: Any notes associated with these tables are also shared.
Because these tables are shared, entries made in one form will update the other. If you initially set up the information in Project Management, it is automatically set up in the related accounting form; however, the data is considering ‘pending’ until the project is interfaced to accounting. You will not be able to access the pending data in the target module until the data is interfaced.
