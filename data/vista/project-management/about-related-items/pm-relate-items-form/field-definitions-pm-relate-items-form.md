# Field Definitions: PM Relate Items Form

The following is a list of field descriptions for the PM
 Relate Items form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Project

Enter the project associated with the record that you are trying to locate or press F4 to select it from a list. By default this field will populate with the project number of the currently selected record, but you can select a different project number and relate records from different projects.

## Category

This field is used to filter the records that display in the Relate Records section by record category, for example RFI, PCO, RFQ, etc. Click the drop down and select a category or leave the field blank to include records from all categories.

## Search For

This field is used in conjunction with the
 Keyword field.
Select General if you want to search records by the description associated with the record.
Select Vendor/Firm if you want to search
 records by the vendor/firm associated with the record. For example select Vendor/Firm and
 then enter a vendor name or vendor number into the Keyword field to locate all records
 associated with a vendor.

## Keyword

Use this field to filter the records by a keyword.
The field is used in conjunction with the
 Search For
 field.

- Vendor/Firm - When Vendor/Firm is selected in the Search For field,
 enter a vendor or firm name in the Keyword field to locate all records
 associated with that vendor or firm.

- General - When General is selected in the Search For field,
 enter a keyword in the Keyword field to locate records with the
 associated keyword. For example, you can use this field to search for a keyword in the
 original question on an RFI, or in the Details  field on a PCO.

Note: The system will search the main PM tables, but it will
 not include all related records. For example, the search will not include RFI responses since
 they are not stored in the main RFI table.
