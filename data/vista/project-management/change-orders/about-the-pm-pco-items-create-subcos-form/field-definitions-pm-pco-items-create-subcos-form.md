# Field Definitions: PM PCO Items Create SubCOs Form

The following is a list of field descriptions for the PM PCO
 Items Create SubCOs form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Add to SCO: Add to Subcontract Change Order

You can use this field to add an item to an existing SubCO, or create a new SubCO.
Note:This field is disabled if you select items that are
 associated with multiple subcontracts. All of the items on a subcontract change order
 must be associated with the same subcontract, so those detail items cannot be grouped
 together onto a single SubCO. You can see the subcontract an item is associated with
 using the SL column.
Add an item to an existing SubCO

Enter an existing subcontract change order number or press F4 to select it from a list.
Create a new SubCO
Leave this field blank. The system will
 automatically create a new subcontract change order and assign it the next available number
 when you click the Create SubCOs button.

## Keyword

Use this field to filter the items that display by a keyword. The system will search all of the columns in the grid for the keyword as you type.
