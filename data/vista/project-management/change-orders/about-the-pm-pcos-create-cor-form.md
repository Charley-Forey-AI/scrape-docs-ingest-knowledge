# About the PM PCOS Create COR Form

Use this form to:

- Create a new change order request (COR) using an unapproved PCO
 selected in PM Pending Change Orders

- Add an unapproved PCO selected in PM Pending Change Orders to an
 existing COR

You can open this form from [PM Pending Change Orders ](/en/vista/vista/project-management/change-orders/pm-pending-change-orders-form) by selecting Create > Change Order
 Request.
Click on a link below for more information.
Create a new COR
Follow the steps below to create a new change order
 request using the PCO selected in PM Potential Change Orders.

1. Open PM Pending Change Orders and select the pending change order that should be added to a COR. There must be items on the selected PCO that have not been approved. Note: Potential/pending change order items are approved using
 [PM
 Approve PCOs](/en/vista/vista/project-management/change-orders/about-the-pm-approve-pcos-form).

1. Select Create > Change
 Order Request. This will open the PM PCOs Create COR form.

1. Verify that the contract
 that displays in the Contractfield is the contract that
 should be changed.

1. Check the Create New
 COR box.

1. Enter the change order
 request that you would like to create in the COR field. By default, this field
 will populate with the next available COR number.

1. Enter a description of the
 new COR in the Descriptionfield.

1. Click the OK button
 to create the COR. A message will appear. Click Yes if you would like to open the
 new COR using [PM Change Order Requests](/en/vista/vista/project-management/change-orders/about-the-pm-change-order-requests-form).

Add the PCO to an existing COR
Follow the steps below to add the PCO to an existing
 change order request.

1. Open PM Pending Change Orders and select the pending change order that should be added to a COR. There must be items on the selected PCO that have not been approved.

1. Potential/pending change order items are approved using [PM Approve PCOs](/en/vista/vista/project-management/change-orders/about-the-pm-approve-pcos-form).

1. Select Create >  Change
 Order Request. This will open the PM PCOs Create COR form.

1. Verify that the contract
 that displays in the Contract field is the contract that
 should be changed.

1. Check the Add to Existing
 COR box.

1. Press F4 in the COR field
 to select the change order request that you would like to add the PCO to.

1. Click the OK button
 when complete. A message will appear. Click Yes if you would like to open the
 selected COR in [PM Change Order Requests](/en/vista/vista/project-management/change-orders/about-the-pm-change-order-requests-form).

Related Items
The pending change orders added to a change order request will automatically be related using the Related Items feature. Click [here](/en/vista/vista/project-management/about-related-items) for more information Related Items.
