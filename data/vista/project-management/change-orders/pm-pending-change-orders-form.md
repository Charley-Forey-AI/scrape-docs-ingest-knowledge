# PM Pending Change Orders Form

Use the PM Pending Change Orders form to create and maintain pending/potential change orders (PCOs).
Pending change orders are used to record information about a potential change to an area of work assigned to a project, including detailed pricing information as it relates to current estimates, commitments, and contract item amounts.
There are several tasks you can perform using this form, some of which include:

- Adding new items to the pending change order

- Generating a change order document

- Creating a change order request

- Copying a pending change order

- Creating a transmittal from the pending change order

Once you've entered a pending change order, you can approve the change order using PM Approve PCOs, and then interface it to applicable accounting modules (Job Cost, Subcontract Ledger, Purchase Orders, Inventory, and/or Material Sales) using PM Interface.

## Available Estimate / Non-Interfaced / Remaining Estimate

These display-only fields are only visible when you select the Estimate/Purchase Details tab (PCO Item section), and show estimate information for the selected phase/cost type. These values are calculated as follows:
FieldCalculation
Available EstimateCurrent Estimated Cost - Actual Cost - Remaining Committed Cost + Non-Interfaced Estimated Cost
Non-InterfacedThis is a more complex calculation that takes a number of factors into consideration. However, in general, this calculation is the sum of the amount and taxes from PM Material Detail and based on certain factors, the subtraction of some PO Item and PO Change Order detail costs (which can include taxes), plus Non-Interfaced Estimated.
Remaining EstimateAvailable Estimate - Non-Interfaced
