# Field Definitions: PM PCOS Create COR Form

The following is a list of field descriptions for the PM PCOS
 Create COR form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## COR: Change Order Request

Creating a new COR
 Use this field to enter the COR that will be
 created. By default, this field will populate with the next available COR number when the
 Create New
 COR box is checked.
Adding to an existing COR
If you are adding the pending change order to
 an existing COR, use this field to select the COR that the PCO should be added to. Either
 enter the COR number or press F4 to select the COR from a list.

## Create New COR / Add to Existing COR

Check the Create New COR box if you would like to
 create a new COR using the PCO selected on PM Pending Change Orders.
Check the Add to Existing COR box if you would like
 to add the PCO selected on PM Pending Change Orders to an existing change order
 request.

## Description

Enter a description of the new change order
 request. This field is only enabled when the Create New COR box is checked.
The value in this field can be up to 60 characters long.
