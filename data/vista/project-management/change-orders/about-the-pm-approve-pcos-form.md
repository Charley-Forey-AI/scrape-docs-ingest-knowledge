# About the PM Approve PCOs Form

Use this form to approve pending change orders and assign them to ACOs.
Click [here](/en/vista/vista/project-management/change-orders/change-management---overview) for general information on approving PCOs and
 grouping them onto ACOs.
You can also use this form to add those
 ACOs to a new or existing contract change order (CCO). Click [here](/en/vista/vista/project-management/change-orders/pm-contract-change-orders-form) for information on contract change orders.
You can open this form by:

- Selecting
 Tasks > Approve
 PCOs from [PM Pending Change Orders](/en/vista/vista/project-management/change-orders/pm-pending-change-orders-form)

- Selecting Tasks > Approve
 PCOs from [PM Change Order Requests](/en/vista/vista/project-management/change-orders/about-the-pm-change-order-requests-form)

- Clicking the Approve button on the Pending tab
 of [PM Change Orders ](/en/vista/vista/project-management/change-orders/about-the-pm-change-orders-form)

[About Creating SubCOs/SCOs and POCOs from the PCOs](/en/vista/vista/project-management/change-orders/about-creating-subcosscos-and-pocos-from-the-pcos)
[Approve PCOs](/en/vista/vista/project-management/change-orders/approve-pcos#task-4461--en__task-4461)

- Interface with Accounting - After the change orders have
 been approved, you can interface them with the accounting modules using PM
 Interface. You can open this form by selecting Tasks > Open
 PM Interface.

- Approve additional PCOs - Select Tasks > Approve PCOs if you want to approve several PCOs at the same time. This will
 open the [PM Add Records ](/en/vista/vista/project-management/change-orders/about-the-pm-add-records-form) form, which allows you to
 select the PCOs that you want to approve. The selected PCOs will be added to the
 Grid tab on the PM Approve PCOs form. Select a PCO on the Grid tab and then use
 the Info tab to select the ACO and the PCO Items tab to select the PCO items
 that will be approved.

- Updates to Custom Fields - If you are using custom fields created using the
 [VA Custom Fields Wizard ](/en/vista/vista/administration/viewpoint-administration/custom-formsfields/va-custom-fields-wizard-form), any data entered
 in a custom field on a pending change order will update the ACO as long as
 you have set up identical custom fields in both PM Pending Change Orders and
 PM Approved Change Orders. If you also have the custom field set up in JC
 Change Orders, it will be updated when the approved change order is
 interfaced to Job Cost.
