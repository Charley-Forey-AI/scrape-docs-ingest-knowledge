# Field Definitions: PM CO Items Load Costs Form

The following is a list of field descriptions for the PM CO
 Items Load Costs form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Beginning/Ending Actual Date

 Specify the beginning and ending actual date by which to restrict the JC detail transactions to be used when accumulating phase cost type totals. Only JC detail transactions within this date range will be used to load actual units, hours, and costs into phase/cost types for the change order item.

##  Beginning/Ending Month

 Specify the beginning and ending month by which to restrict the JC detail transactions to be used when accumulating phase cost type totals. Only JC detail transactions within this month range will be used to load actual units, hours, and costs into phase/cost types for the change order item.

##  Replace Existing Values?

 Check this box to replace the units, hours, or costs for phase cost types on the selected change order item. Existing units, hours, and costs will be updated with accumulated values from JCCD (Cost Detail).
 Leave this box unchecked if you do not want units, hours, or costs replaced for phase cost types. Only phase cost types with zero amounts will be updated; phase cost types with existing amounts will be skipped.
