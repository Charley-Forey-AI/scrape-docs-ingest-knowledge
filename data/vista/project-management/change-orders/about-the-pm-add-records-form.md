# About the PM Add Records Form

This is a generic form launched from several places in the PM module, and it is used to select multiple records.
Use the Keyword Search field to filter the
 records that display - the system will search through all of the columns in the form and
 display only records that include the search text.
There are several ways to select multiple records:

- Range of records- Hold down SHIFT and click on the first and last record in the range, and then check or uncheck the selection box.

- Specific records - Hold down CTRL and click on each record that should be selected, and then check or uncheck the selection box.

The function of this form varies depending on how it is launched.

- PM Approve PCOs > Tasks > Add PCOs - Select the PCOs
 that you want to approve and then click the OK button. The selected PCOs
 will be added to the Grid tab on the [PM
 Approve PCOs](/en/vista/vista/project-management/change-orders/about-the-pm-approve-pcos-form) form. This allows you to approve multiple PCOs at the
 same time, even if the PCOs are from different projects.

- PM Change Order Requests > Tasks > Add PCOs - Select
 the PCOs that you would like to add to a change order request and then click the
 OK button. You cannot select potential/pending change orders
 that do not have any unapproved PCO items.

- PM Contract Change Orders > Tasks > Add ACOs - Select
 the ACOs that you would like to add to a contract change order and then click
 the OK button. The status of the selected ACOs will change from
 "Approved" to "Pending", which means that they cannot be interfaced with the
 accounting modules until the CCO has been approved.

- PM Distribution Groups > Tasks > Add Contacts - Select the PM
 firm contacts that you would like to add to the distribution group and then
 click the OK button. Click [here](/en/vista/vista/project-management/create-and-send/pm-distribution-groups-form) for more information about distribution groups.

- PM Change Orders > Tasks > Add Items from PCO Detail - Select changes to
 a PO that have already been entered on a PCO using the Estimate / Purchase
 Detail tab on the [PM Pending Change Orders](/en/vista/vista/project-management/change-orders/pm-pending-change-orders-form) form. Click [here](/en/vista/vista/project-management/change-orders/pm-po-change-orders-form) for more information.

- PM Subcontract Change Orders > Tasks > Add Items from PCO Detail - Select
 changes to a subcontract that have already been entered on a PCO using the
 Estimate / Purchase Detail tab on the [PM Pending Change Orders](/en/vista/vista/project-management/change-orders/pm-pending-change-orders-form) form. Click [here](/en/vista/vista/project-management/change-orders/pm-subcontract-change-orders-form) for more information.
