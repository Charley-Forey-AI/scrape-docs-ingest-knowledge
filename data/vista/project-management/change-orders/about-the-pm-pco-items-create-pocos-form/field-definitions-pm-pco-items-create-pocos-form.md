# Field Definitions: PM PCO Items Create POCOs Form

The following is a list of field descriptions for the PM PCO
 Items Create POCOs form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Add to POCO

Use this field to associate the selected items with an existing PO change order (POCO). Enter a PO change order or press F4 to select one from a list.
If you leave this field blank, the system will create a new POCO.

## Keyword

Use this field to filter the items that display by a keyword. The system will search all of the columns in the grid for the keyword as you type.
