# About Options for Copying

The Notes, Response,
 Information
 Requested, and Attachments checkboxes at both the PCO
 header and item levels control what is copied to the pending change order.
If you check these boxes, the corresponding
 tabs in the RFI header will be copied to the Notes tab for the pending change order
 header and/or items as specified.
Note: If you are copying Response and/or Information
 Requested information, be aware that only the RFI header information will be copied.
 Responses and requested information entered for a firm/contact in the Distribution grid
 are not copied to the pending change order.
If you are copying attachments, checking the
 attachments option at the header level will copy any RFI attachments to the pending
 change order header. If checked for the PCO item, any RFI attachments will be copied to
 the pending change order item
