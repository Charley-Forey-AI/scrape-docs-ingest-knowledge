# Field Definitions: PM Copy Project Budget Form

The following is a list of field descriptions for the PM Copy
 Project Budget form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Source Project Budget: Project

Enter the project from which to copy the project budget. Must be a valid project set up PM Projects.

##  Source Project Budget: Budget

 Enter the budget to copy. Must be a valid budget set up for the source project (in PM Project Budgets).

##  Destination Project Budget: Project

 Display only, the project to which you are copying the budget. Will always be the active project.

##  Destination Project Budget: Budget

 Enter the project budget to which you are copying project budget detail or enter ‘+’ to default the next sequential budget number for the project. Must be a new project budget; cannot copy to an existing project budget.

##  Destination Project Budget: Description

 Enter a description of the destination budget, up to 60 characters. If left blank, the budget description will default as null in PM Project Budgets.
