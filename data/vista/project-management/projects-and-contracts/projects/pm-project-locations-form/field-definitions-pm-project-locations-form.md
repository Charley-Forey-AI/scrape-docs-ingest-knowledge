# Field Definitions: PM Project Locations Form

The following is a list of field descriptions for the PM
 Project Locations form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Project

Enter the project the location is associated with or press F4 to select a project from a list. the project to which this location is to be assigned.

## Location

Location field on the PM Project Locations form
Enter a unique code to identify the location. This field can be up to 10 characters long.
Some examples of project locations could include:
Location Code
Description

B
Building

S
Site Work

B1
Building 1

B1-1
Building 1, 1st Floor

B1-120
Building 1, Room 120

B1RF
Building 1, Roof System

SNE
Site, Northeast Section

Related information

- [PM Project Locations Form](/en/vista/vista/project-management/projects-and-contracts/projects/pm-project-locations-form)

## Description

Enter a description of the location. The description can be up to 30 characters long.
