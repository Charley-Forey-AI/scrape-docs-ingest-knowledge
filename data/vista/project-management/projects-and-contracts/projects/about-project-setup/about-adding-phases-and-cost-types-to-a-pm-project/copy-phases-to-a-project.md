# Copy Phases to a Project

You can add multiple phases to a project at one time using the project phase copy feature.
The phase grid displays all phases created using JC Phases.

1. From the Vista main menu, select Project Management > Programs > PM Project Phases .

1. In the Project field, enter the project to which you are adding phases or press F4 to select from a list of projects.

1. Select File > Project Phases Copy.The PM Project Copy Phases form opens and displays all phases created using JC Phases.

1. In the grid, select the phases to add to the project. Use SHIFT to select consecutive records or CTRL to select specific ones.

1. Click Copy.

 The system add the phases to the sprecified project.
