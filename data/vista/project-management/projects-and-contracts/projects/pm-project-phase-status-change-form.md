# PM Project Phase Status Change Form

Use this form to globally change the status for phases and/or cost types on a project.
You can only access this form by selecting
 the Phase Status Change option from the File menu of PM Project Phases.

## Phases

Use this tab to change the status (in JC Job Phases) for phases on a project. You can
 change the status for all phases on the project or only those within a specified
 phase range. Once you specify whether to change the status for the specified phases
 to ‘Active’ or ‘Inactive’ and click Update, the system cycles through all specified
 phases and changes the status for all phases not already set to the specified
 status. For example, if you select to change the phase status to 'Inactive', the
 system will change all phases flagged as 'Active' to 'Inactive', and skip all phases
 already flagged as 'Inactive'.

## Cost Type

Use this tab to change the update status
 (in JC Cost Header) for cost types on a project. You can change the status for all
 phases and/or cost types on the project or for only phases and/or cost types within
 a specified range. You must specify whether to set the selected cost types as ‘Ready
 to Interface’ or ‘Not Ready to Interface’. Once you click Update, the system cycles
 through all specified phases and cost types and changes the status for all cost
 types not already set to the specified status. For example, if you select to change
 the cost type status to ‘Ready to Interface’, the system changes all cost types
 flagged as ‘N-Not Ready to Update’ to ‘Y-Ready to Update’, and skips all cost types
 already flagged as ‘Y-Ready to Update’. In addition, the system skips cost types
 flagged as ‘I-Interfaced’ or ‘J-Job Cost’.
Note: You can also change the status for phases in JC
 Job Phases; however, the cost type update is available in PM only.
