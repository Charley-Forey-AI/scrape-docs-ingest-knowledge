# PM Project Firms Form

Use this form to set up specific firms and contacts on a project.
The firms and contacts set up for a project here are used for lookups, as well as to create distribution lists for submittals, change orders, daily logs, or any other information or documents on a project.
Once you enter the project, specify each firm and firm contact you want added to the project. As you enter the firm contact, the screen displays information about the contact (title, email, phone, etc.) as defined in PM Firm Contacts.
The Inactive checkbox indicates the contact's active status. The system automatically sets this flag based on how the Exclude from Lookups checkbox is set for the contact in PM Firm Contacts (for example, if the checkbox is selected, the Inactive checkbox here is selected). For more information about the Exclude from Lookups checkbox, see the F1 help.
Note: You can quickly set up firms and contacts for a project using the Project Firms Copy option in the File menu of this form. For more information, see[Copy Project Firms/Contacts to Another Project](/en/vista/vista/project-management/projects-and-contracts/projects/copy-project-firmscontacts-to-another-project).

## Distribution Defaults/Groups

You can assign distribution defaults to define which project contacts should receive documents when using the Create and Send feature. This feature applies to many forms in the PM module.
For example, if a contact at an architecture firm should receive a copy of all drawing logs of document type 'ARCH', use this feature to set up that contact as a default distribution for document type 'ARCH'. When drawing logs of that type are created in PM Drawing Logs, the contact will by default be included in the correspondence. For more information about this feature, see [PM Assign Distribution Defaults Form](/en/vista/vista/project-management/create-and-send/pm-assign-distribution-defaults-form).
Note: Assigning a new distribution default will not change the distribution of the current document. The new distribution default will only be applied when you create a new document (RFI, change order, etc.)

You can also set up distribution groups, which allow you to group contacts together that you regularly send correspondence to--for example, a group of contact that receive submittals. You can then add these groups to communications created using the Create and Send feature. For moe information, see [PM Distribution Groups Form](/en/vista/vista/project-management/create-and-send/pm-distribution-groups-form).
