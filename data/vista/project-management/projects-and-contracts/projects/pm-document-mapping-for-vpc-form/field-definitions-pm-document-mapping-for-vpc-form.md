# Field Definitions: PM Document Mapping For VPC Form

The following is a list of field descriptions for the PM
 Document Mapping For VPC form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Document Category

Select the document type (e.g., Request for Information, Change Orders, etc.) that you wish to map to VFP from the Document Category pull-down list.

## Seq

Enter the sequence number of an existing document, or enter N, New, or + to add a new sequence. The system will automatically assign the next available sequential number.

## Document Type

This is an optional field. The system allows entry in this field for user-defined sequences only.
Enter a document type that will associate this index sequence with the selected document. Press F4 for a list of valid document types set up in PM Document Types.

Related information

- [PM Document Types Form](/en/vista/vista/project-management/setupmaintenance/pm-document-types-form)

## Container

To populate the
 Container
 field, click the
 Select Container
 button to select the VFP container to map to. From the Select Container To Associate form, select the appropriate container, and click
 OK
 . The selected container will then populate the
 Container
 field.

## Report

Click this box to publish a PM document as a report instead of a template, in an instance where you want to use the report as a collaborating document, or if a Word template does not exist for a particular PM document. Select F4 for a list of valid reports.

## Publish as PDF

Select in order to publish your documents to VFP in PDF format. Selecting this option also provides you with the option to collate your published VFP documents into a single PDF document.

Related information

- [Collate to a Single Document](/en/vista/vista/project-management/projects-and-contracts/projects/pm-document-mapping-for-vpc-form/field-definitions-pm-document-mapping-for-vpc-form#ID-0002be15--en)

## Include Attachments

Click to include attachments associated with the source record that you want included with the document published to VFP.

## Collate to a Single Document

Click to collate all of your attachments into
 a single PDF for publishing to VFP. The Include Attachments option must be
 selected in order for this option to appear.
