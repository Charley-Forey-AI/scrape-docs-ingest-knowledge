# Field Definitions: PM Notes Review Management Form

The following is a list of field descriptions for the PM
 Notes Review Management form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project

 Enter the project by which to filter project notes, or leave blank if not filtering by project.

##  Review Status

 Select the review status by which to filter project notes, or leave blank if not filtering by review status.

- New – Select this option to display only those project notes with a review status of ‘New’.

- Reviewed – Select this option to display only those project notes with a review status of ‘Reviewed’.

## Search by Date

Select the date by which to filter project notes.

- None – Select this option if not filtering project notes by date.

- Date Added – Select this option to filter project notes by the ‘added on’ date (i.e. the date a project note was added in PM Notes).

- Status Date – Select this option to filter project notes by the ‘review status’ date (i.e. the date the review status was last updated for a project note in PM Notes, PM Notes Review, or this form).

##  Beginning/Ending Date

 These fields are enabled only when the Search By Date option is set to ‘Date Added’ or ‘Status Date’.
 Enter the beginning and ending date in a range of ‘added on’ or ‘status review’ dates by which to filter project notes.

##  Reviewers

 Select the reviewers by which to filter project notes, using the Shift key for consecutive selection, or the Ctrl key for random selection. Select ‘All’ if not filtering by reviewer.

## Review Status

This field indicates the review status of this project note. You may change the status if applicable; status will be updated to PM Notes and PM Notes Review.

- New – This project note entry has not been reviewed and/or signed off by this reviewer.

- Reviewed – This project note entry has been reviewed and signed off on by this reviewer.

Note:If status is changed from Review to New, the system automatically sends an email to the assigned reviewer informing them that there is a project note for them to review.
