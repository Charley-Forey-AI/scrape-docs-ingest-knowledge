# PM Copy Project Phases Form

Use this form to add multiple phases to a project at one time.
You can access this form from PM Project Phases by selecting
 File >  Project Phases Copy.
During the copy process, the system assigns phases
 to contract item ‘1’ and defaults the projection minimum percent specified for the phase in
 JC Phases. If no minimum is specified, the Projection Minimum for the project/phase is set
 to 0.00.
Subcontract detail records with zero values are generated in PM
 Subcontract Detail for all phases with subcontractor cost types matching the cost types
 designated for subcontracts in PM Company Parameters (SL Parameters tab). In addition,
 material detail records are generated in PM Material Detail for all phases with material
 cost types matching the cost types designated for materials in PM Company Parameters
 (Material Parameters tab).
