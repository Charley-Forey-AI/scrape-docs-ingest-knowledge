# Field Definitions: PM Copy Project Form

The following is a list of field descriptions for the PM Copy
 Project form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Source Project

Enter the project to copy or press F4 to select a project from a list.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  PM Company

 Enter the destination PM company.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Destination Project

Enter a new project number. This is the project that will be created by the copy. This cannot be an existing project number. Press F4 to see a list of existing projects.
Note: Custom fields on the source project in PM Projects, PM Project Phases, and/or PM Cost Types Detail will be copied to the new project.

## Description

Enter a description of the new project, up to 60 characters. Initially defaults the description of the source project, which may be overridden.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Liability Template

Enter the liability template (from JC Liability Template) for the new project. Initially defaults the liability template for the source project, which may be overridden.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## State

Enter the state code (from HQ States) for the new project that identifies where the project is located. Initially defaults the state code for the source project, which may be overridden.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Destination Contract

 Enter the contract for the new project, up to 9 characters (including dashes). Initially defaults the code entered for the destination project, which may be overridden.
 You may specify an existing contract, if desired, as long as the contract’s status is ‘Open’ (1) or ‘Pending’ (0). If an existing contract is specified, remaining contract fields are disabled and will default the information specified for the existing contract. Additionally, only contract items that exist for the source contract but not for the destination contract will be copied.
Note: If custom fields exist for the source contract and/or contract items, they will be copied to the new contract/contract items.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Description

Enter a description of the contract, up to 60 characters. Initially defaults from the destination job description, but may be overridden.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Department

 Enter the department for the new contract. Must be a valid department (from JC Departments) for the destination company.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Customer

 Enter the customer for this contract, if applicable. Must be a valid customer (from AR Customers) for the destination company.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Retainage %

 Enter the retainage percent for the contract.

- This will be applied to all of the items that are created on the
 new contract.

- This will also be used as the default retainage percentage
 (PM Contracts > Info tab > Default
 Retainage % field). [More](/en/vista/vista/project-management/projects-and-contracts/contracts/pm-contracts-form/field-definitions-pm-contracts-form#ID-000242db--en)

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Start Month

 Enter the new contract’s start month. Initially defaults the current month and year.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Copy Amounts

- None – Do not copy original or current estimate amounts to the new project.

- All Amounts – Copy both original and current estimate amounts to the new project.

- Original Estimates Only – Copy only the original estimate amounts to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Subcontract Detail

 This field is disabled if the phase group differs between the source and destination companies.
 Check this box to copy subcontract detail from the source project to the new project.
 Leave this box unchecked if not copying subcontract detail to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Material Detail

This field is disabled if the phase group differs between the source and destination companies.
Check this box to copy material detail from the source project to the new project.
Leave this box unchecked if not copying material detail to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Project Firms

 This field is disabled if the phase group differs between the source and destination companies.
 Check this box to copy project firms from the source project to the new project.
 Leave this box unchecked if not copying project firms to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Drawing Logs

 Check this box to copy all drawing logs/revisions set up for the source project to the new project.
 Leave this box unchecked if not copying drawing logs/revisions to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Change Order Header

Check this box to copy change order headers from the source project to the new project.
Leave this box unchecked if not copying change
 order headers to the new project. (Note: If unchecked, the Change Order
 Items and Change Order Detail checkboxes are
 disabled.)
[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Change Order Items

This field is only accessible if the
 Change Order
 Header box is checked.
Check this box to copy change order items from the source project to the new project.
Leave this box unchecked if not copying change
 order items to the new project. (Note: If unchecked, the Change Order Detail
 checkbox is disabled.)
[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Change Order Detail

This field is only accessible if the
 Change Order
 Items box is checked.
Check this box to copy change order detail from the source project to the new project.
Uncheck this box if not copying change order detail to the new project.
[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Project Budgets

 Check this box to copy project budgets from the source project to the new project.
 Leave this box unchecked if not copying project budgets to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Other Documents

Check this box to copy ‘Other’ documents (those with a document category of ‘Other Documents’) from the source project to the new project.
Leave this box unchecked if not copying ‘Other’ documents to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Submittals - 6.5

 Check this box to copy the submittals created and maintained using the [PM Submittals - 6.5](/en/vista/vista/project-management/issuessubmittalsother-documents/submittals/about-the-pm-submittals---6.5-form) form. The copy does not include submittal header/item dates or copies sent/received.
This option will not copy any submittals created using the enhanced submittals process. [More](/en/vista/vista/project-management/issuessubmittalsother-documents/submittals/about-submittals)

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Submittal Items 6.5

 This field is only accessible if the 'Submittals' option is checked.
 Check this box to copy submittal items from the source project to the new project.
 Leave this box unchecked if not copying submittal items to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Item Retainage

Check this box to copy item retainage from the source contract items to the destination contract items.
Leave this box unchecked if not copying item retainage from the source contract items to the destination contract items. The destination contract items will default the retainage percent specified for the destination contract (above).

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

##  Item Departments

 Check this box to copy the departments from the source contract items to the new contract items. When checked, a warning displays (in red) to the right indicating that you are copying source departments.
 Leave this box unchecked if you are not copying item departments from the source contract. All items added for the new contract will automatically be assigned the department specified for the destination contract above.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## JB Bill Groups

Check this box to copy bill groups defined for the source contract (in JB Bill Groups) to the new contract.
Leave this box unchecked if you are not copying bill groups from the source contract. No record will be set up in JB Bill Groups for the new contract.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Project Reviewers

Check this box to copy the reviewer setup from the source project to the destination project.
Leave this box unchecked if you do not want to
 copy the reviewer setup from the source project to the destination project. Reviewers will
 need to be set up manually for the new project in PM Projects (Reviewers tab).

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Job Roles

Check this box to copy role assignments set up using the Roles tab on the [PM Projects](/en/vista/vista/project-management/projects-and-contracts/projects/pm-projects-form) form.
You might want to check this box if you use the WF module Process Workflow feature. [More](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow)
Note:Checking this box enables the Job Phase
 Roles checkbox, allowing you to copy roles at the job phase level.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Job Phase Roles

This checkbox is enabled only when the
 Job
 Roles box is checked.
Check this box to copy job phase role assignments from the source project to the destination project.
Uncheck this box if not copying job phase role assignments to the new project.

[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-copy-project-form)PM Copy Project

## Projection Codes

Check this box if the projection codes set up on the source project should be created for the new project.
Projection codes are created and maintained using the [PM Projection Codes ](/en/vista/vista/project-management/cost--revenue-projections/pm-projection-codes-form) form, and they only apply to projections that are created using the PM Cost Projections form.
Copy the projection codes to an existing project
You can also copy the projection codes from one project to another using the [Copy Projection Codes ](/en/vista/vista/project-management/cost--revenue-projections/pm-projection-code-copy-form) form.
