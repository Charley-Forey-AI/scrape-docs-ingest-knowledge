# PM Document Mapping for VPC
 Form

Use this form to ensure that your document categories are mapped
 to your containers for successful integration with Viewpoint For Projects™ (VFP).
You can select whether you want to use a Word Doc Template,
 Report, Publish as PDF, Include Attachments, and if you want to Collate to a Single
 Document (PDF only).
