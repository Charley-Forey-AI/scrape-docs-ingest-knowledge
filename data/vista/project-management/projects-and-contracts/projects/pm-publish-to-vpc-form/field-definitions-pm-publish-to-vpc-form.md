# Field Definitions: PM Publish to VPC Form

The following is a list of field descriptions for the PM
 Publish to VPC form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Document Category

Select the Document Category associated with the document type that you want to publish to VFP.

## Include Previously Published Documents

Check this box to include previously published documents prior to publishing documents to VFP.

## Publish

Click the checkbox next to the Doc Type that
 corresponds with the selected Document Category. Click All to automatically select all Doc Types
 to publish. Click Publish once you select documents to publish to VFP.
