# PM Copy Template Form

Use this form to add the phases on a template to a project.
Templates are created and maintained using PM
 Template Phases. The destination project must already be set up in PM Projects before
 the phases can be copied into it.

## Phase Costs

Estimated units, hours, dollars, and contract amounts are not defined when templates
 are set up; therefore, you will need to set up this information manually in PM
 Project Phases and PM Contract Items.

## Copying Multiple Templates

You can copy more than one template into a project. The initial
 copy sets up the data in the PM Projects and PM Contracts. Additional roll-ins will
 add any new phases (bypassing any duplicate phases) and new contract items.
