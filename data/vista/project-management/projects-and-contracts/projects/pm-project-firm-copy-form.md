# PM Project Firm Copy Form

Use this form to copy the firms and contacts from one project to another.
Access this form by selecting File >  Project Firms Copy
  in PM Project Firms. This can save time if you have
 projects that use some of the same subcontractors, vendors, architects,
 and engineers.
The copy process allows you to select only a single source project at one time; however,
 you can repeat the process as many times as necessary to get all the firms and contacts you
 want set up for your destination project.
Once you select a source project, the system populates the grid with the firms and contacts
 defined for the project. However, if a firm/contact is already set up for the destination
 project, it is not included in the grid.
