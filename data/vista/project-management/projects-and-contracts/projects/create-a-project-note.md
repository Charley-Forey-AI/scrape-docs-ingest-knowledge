# Create a Project Note

You can create a project note by going to Project Managment > Programs > PM Notes.

1. From the PM Notes form, click the New Record () icon. By default, the system
 assigns the note the next available sequence number. This is the number that
 displays in the Sequence field.

1. In the
 Summary field, enter a summary for the project note.


1. In the Status field, enter the status of the note or
 accept the defaulted status. Press F4 to select from a
 list of valid statuses.

1. In the Firm field, enter the firm to which this project
 note applies or press F4 to select from a list of valid
 firms.

1. In the Firm Contact field, enter the contact to which
 this project note applies or press F4 to select from a
 list of valid firm contacts.

1. On the Notes tab (or Notes field
 if in Grid view), enter any relevant notes.

1. Double-click on the Notes tabThe Time Stamp Notes form displays.

1. In the lower portion of the form, enter any applicable notes.

1. Click Add.The notes are saved and added to the upper portion of the
 form.

1. Close the form.

1. If applicable, assign
 one or more reviewers to the project note for review.Open the Reviewers tab. Use
 this tab to assign one or more reviewers to a project note entry for review.
 When a reviewer is assigned to a project note, an email is sent notifying the
 reviewer that they have a note to review.

1. Click on the Reviewers tab.

1. In the Reviewer field, enter a reviewer or press
 F4 to select from a list of valid
 reviewers.

1. Leave the Status field set to New. The reviewer
 will change this field to Reviewed once they have
 completed their review.

1. Save the project note
 record.
