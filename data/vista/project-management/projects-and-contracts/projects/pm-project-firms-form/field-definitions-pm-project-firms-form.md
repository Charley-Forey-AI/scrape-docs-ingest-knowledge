# Field Definitions: PM Project Firms Form

The following is a list of field descriptions for the PM
 Project Firms form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project

 Specify the project for which to set up firms and contacts.

[](/en/vista/vista/project-management/setupmaintenance/set-up-firms--contacts/about-firms-and-contacts)Overview: Firms and Contacts
[](/en/vista/vista/project-management/projects-and-contracts/projects/pm-project-firms-form)PM Project Firms

## Firm

Enter a firm number or press F4 to select
 one from a list. Firms are created and maintained using the [PM Firms](/en/vista/vista/project-management/setupmaintenance/about-the-pm-firms-form) form.
Tip: The sort name can be used instead of the firm number
 when selecting firms in PM module forms. Generally the sort name is the first several
 letters of the firm name.  For example when creating a subcontract in the [PM Subcontracts](/en/vista/vista/project-management/subcontracts/pm-subcontracts-form) form for 'Bryan Electrical', you can enter the sort name 'bryan'
 in the Vendor
 field instead of the firm number '10042'. The sort name of a firm is set up
 using the Sort
 Name field on the [PM Firms](/en/vista/vista/project-management/setupmaintenance/about-the-pm-firms-form) form.

##  Contact

 Enter the contact (from PM Firm Contacts) to assign to this project. If you do not know the contact number/code, you can enter the first few characters of the sort name (typically the last name) and press Enter, and the system will locate and display the first closest match.

 Enter a description for this firm/contact, up to 30 characters.

## Email Option

Specify the default email option for this
 contact. The setting defined here determines how the CC checkbox is set for the contact when
 added to a distribution grid in any form with Create and Send functionality.

- None – Select this option if this contact is a primary contact and
 is not typically courtesy copied (Cc) or blind courtesy copied (Bcc) when sending
 documents via email or fax.

- Cc – Select this option if you typically courtesy copy this contact
 when sending documents via email or fax. Contact’s name will be visible to other email
 or fax recipients.

- Bcc – Select this option if you typically blind courtesy copy this
 contact when sending documents via email or fax. Contact’s name will not be visible to
 other email or fax recipients.
