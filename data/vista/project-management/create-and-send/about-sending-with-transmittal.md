# About Sending With Transmittal

When you click on the Create and Send () icon in the toolbar at the top of a form the following
 options display: Send Message, Send Document, Send with
 Transmittal, and Create Document From Template.
Use the Send With Transmittal option to
 create a transmittal and bundle it with the project documents, reports, and attachments
 included in the communication. Any project documents generated for the email will
 automatically be related using the [Related
 Items](/en/vista/vista/project-management/about-related-items) feature.
Click on a step in the diagram for more detailed information about that step.
