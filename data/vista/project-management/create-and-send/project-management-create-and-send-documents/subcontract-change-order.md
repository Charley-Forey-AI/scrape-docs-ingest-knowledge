# Subcontract Change Order

Use this form to record a change in scope to an existing subcontract - for example
 additions or subtractions to existing items, or the creation of new scope items. Use this
 form to create and maintain subcontract change orders (SCOs/SubCOs), generate and send an
 SCO document to a list of contacts using the Create and Send feature, and then approve the
 SCO.
Access this report by clicking:

1. Project Management > Programs > PM Subcontract Change Orders

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
