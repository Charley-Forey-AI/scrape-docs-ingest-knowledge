# Change Order Request

Use this form to create and maintain change order requests (CORs). A change order request is a formal request to an owner/customer for a change in scope, and it is an agreement acknowledging that the customer/owner will pay for the changes. Once the change order request is signed by the owner/customer, work can proceed and billings can occur.
Access this report by clicking:

1. Project Management > Programs > PM Change Order Requests

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
