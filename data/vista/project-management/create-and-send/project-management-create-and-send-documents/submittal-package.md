# Submittal Package

Use this form to create and maintain submittal packages. Once submittals have been
 entered in the register using the PM Submittal Register form, they can be bundled in a
 submittal package using this form.
Access this report by clicking:

1. Project Management > Programs > PM Submittal Package

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
