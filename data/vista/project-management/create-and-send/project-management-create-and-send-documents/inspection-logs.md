# Inspection Logs

Use the PM Inspection Logs form to create and maintain inspection logs, and then use the Create and Send feature to generate an inspection log document.
Inspection logs are categorized by inspection type and a unique code. The inspection types (fire, electrical, rebar, etc.) are set up by an admin using the PM Document Types form.
Access this report by clicking:

1. Project Management > Programs > PM Inspection Logs

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
