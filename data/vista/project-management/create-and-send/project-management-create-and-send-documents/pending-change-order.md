# Pending Change Order

Use this form to create and maintain pending/potential change orders (PCOs). Pending change orders are used to record information about a potential change to an area of work assigned to a project, including detailed pricing information as it relates to current estimates, commitments, and contract item amounts.
Access this report by clicking:

1. Project Management > Programs > PM Pending Change Orders

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
