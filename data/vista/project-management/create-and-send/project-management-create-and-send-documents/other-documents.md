# Other Documents

Use this form to create and maintain subcontracts.
Access this report by clicking:

1. Project Management > Programs > PM Other Documents

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list (Note there are several default documents available)

1. Click View Document

Fax Cover Sheet

Formal Letter

Letter of Intent

Memo
