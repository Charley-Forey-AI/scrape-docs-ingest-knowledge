# Request For Information

Use the PM Request For Information form to create and maintain RFIs, generate and send RFI documents, and receive and track the responses.
An RFI is generally used when you need specific information about a project or have questions for a project contact.
Access this report by clicking:

1. Project Management > Programs > PM Request For Information

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
