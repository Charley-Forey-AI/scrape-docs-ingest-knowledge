# Submittal Register

Use this form to create and maintain the submittal register of each project. Once
 submittals have been entered in the register, they can be bundled into a submittal package
 using the PM Submittal Package form.
Access this report by clicking:

1. Project Management > Programs > PM Submittal Register

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
