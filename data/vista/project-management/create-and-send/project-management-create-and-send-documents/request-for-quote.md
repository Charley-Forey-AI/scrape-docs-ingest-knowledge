# Request For Quote

Use this form to create and maintain RFQs. RFQs are used to get an itemized breakdown
 of what it will take to complete the items on a change order, for example the labor,
 materials, equipment, and overhead costs.

Access this report by clicking:

1. Project Management > Programs > PM Request For Quote

1. Click the Create and Send icon

1. Document Source = Template

1. Document > Select from drop-down list

1. Click View Document
