# PM Create & Send Locations Form

Use the PM Create & Send Locations form to create and maintain document template locations. Document template locations define where document templates are stored.
 Locations are not company-specific. Once you create a location, it is available
 in all PM companies.
The standard document templates provided by Viewpoint (as a
 starting point for creating your own templates) use a location called PMStandard, and
 are stored in a standard repository (\Viewpoint Repository\Document Templates\Standard).
However, user-created templates cannot be saved in the PMStandard location, so you must to set
 up at least one location before creating your own templates. For instructions, see [Create a Document Location](/en/vista/vista/project-management/create-and-send/create-a-document-location#task-5869--en__task-5869).
