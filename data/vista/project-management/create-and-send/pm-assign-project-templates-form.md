# PM Assign Project Templates Form

Use the PM Assign Project Templates form to set up default templates on projects and define which templates users can select when generating project documents.

## Assign Default Document Templates

You can set up a default document template for each applicable project. Document templates allow you to define which formats should be used when generating project documents using the [Create and Send](/en/vista/vista/project-management/create-and-send/about-the-create-and-send-feature) feature. For example, if all RFIs for a project should use a specific RFI document format, you should set up that format as a document template using the PM Create & Send Templates form, and then configure that template as the default for the project. When an RFI is generated for the project using the Create and Send feature, by default the RFI will be in the desired format.Note: You can also set up default document templates at the company level using the Document Templates tab in PM Company Parameters. However, default templates defined at the project level will always override those defined at the company level. For more information about defining default templates for a company, see [Default Template](/en/vista/vista/project-management/setupmaintenance/pm-company-parameters-form/field-definitions-pm-company-parameters-form#ID-00023e16--en).

## Define Templates Selectable by Users

You can also define which document templates users can select when when generating project documents (RFI, RFQ, PCO, etc.). When using the [Create and Send](/en/vista/vista/project-management/create-and-send/about-the-create-and-send-feature) feature, all of the document templates of the selected document category display in a drop-down menu. This can be problematic if you have hundreds of document templates. You can use the PM Assign Project Templates form to configure which document templates can be selected when generating project documents, making it easier for users to select the correct template. For example, if you have a lot of document templates but only a few will be used on a specific project, you can set up the project so that only a select number of the document templates are available to users.

To set up the default document templates on a project, and/or configure which document templates can be used to generate project documents, click the Distribution Templates () icon.
