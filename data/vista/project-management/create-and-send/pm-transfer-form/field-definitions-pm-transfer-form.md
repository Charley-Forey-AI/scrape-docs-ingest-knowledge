# Field Definitions: PM Transfer Form

The following is a list of field descriptions for the PM
 Transfer form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Source

Enter where you would like to remove the document templates from or press F4 to select it from a list.
Document templates are created and maintained using [PM Create & Send Locations ](/en/vista/vista/project-management/create-and-send/pm-create--send-locations-form).

## Destination

Enter the document location where the template will be moved to or press F4 to select one from a list.
Document locations are created and maintained using [PM Create & Send Locations ](/en/vista/vista/project-management/create-and-send/pm-create--send-locations-form).

## Word Templates (.dot)

Check the box next to each template you would
 like to transfer. You can use SHIFT to select a consecutive list of templates or you can
 click Select
 All to select all of the templates in the list.
NOTE: The list will not include:

- templates that are saved in both the source and destination document locations

- templates that have not be set up using [PM Create & Send Templates ](/en/vista/vista/project-management/create-and-send/pm-create--send-templates-form)

## Select All

Check this box to move all templates in the list.
NOTE: The transfer process will skip all templates that were not set up using [PM Create & Send Templates ](/en/vista/vista/project-management/create-and-send/pm-create--send-templates-form).
