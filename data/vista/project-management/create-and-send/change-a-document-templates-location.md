# Change a Document Template's Location

Transfer a document template from one document location to
 another.
This process only moves
 document templates that have been set up using [PM Create & Send Templates ](/en/vista/vista/project-management/create-and-send/pm-create--send-templates-form). If you created document templates outside
 of the application, you cannot move them using this process until they are set up in PM
 Create & Send Templates.

1. Open [PM Create & Send Templates ](/en/vista/vista/project-management/create-and-send/pm-create--send-templates-form) and select
 File > Document Template Transfer.

1. Select where the
 document template will be moved from in the Source field. Press F4 to
 select a document location from a list.

1. Select where the
 document template will be moved to in the Destination field. Press F4
 to select a document location from a list.

1. Select the template you
 would like to move in the list box.Note: The list will not include document
 templates that are saved in both the source and destination document
 locations. This list will also only include document templates that have
 been set up in PM Create & Send Templates.

1. Click Copy when complete to move the selected document templates.
