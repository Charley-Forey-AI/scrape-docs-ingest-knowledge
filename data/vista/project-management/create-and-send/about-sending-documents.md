# About Sending Documents

Use the Create and Send () Send Document option to generate
 a project document and email it to a list of contacts.
This diagram outlines the process. Click on a step in the diagram for more detailed information.
If you need to edit the generated document before it
 is sent, use the Create
 Document From Template option. [More](/en/vista/vista/project-management/create-and-send/about-generating--editing-project-documents)

Tip: If you are using Microsoft Word documents with IF statements, do the following to ensure your Conditional Statements are properly merged into PDF:

1. Select Options from the File pull-down menu.

1. Click Display.

1. Under Printing Options,
 select the following: Update fields before printing
 and Update linked data before printing.

1. Click OK.
