# PM Response Values Form

This form only applies to setting up automated response forms. For more information on setting up automated response forms, see [About Automated Response Forms](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/about-automated-response-forms)..
Use this form to customize the response values that display on the Invitation to Bid automated bid response form. For example, you can use this form to change the "No, I am not interested in bidding" field caption on the Invitation to Bid automated response form to "No, I will not bid."
Do not delete the ITBResponse record because it contains the standard return values for the Invitation to Bid automated response form. If you delete this record, the Invitation to Bid automated response forms will not work.
Follow the steps below to change the response values that display on the automated response form. Currently, this is the only change that should be made using this form. Modifying other elements my break the automated response forms.

1. Select the ITBResponse
 record using the Grid tab.

1. Open the Items tab. This tab
 displays the response values that will display on the ITB automated response
 form.

1. Change the values in the
 Display
 Value column to change how the response value will display on the
 automated response form. You can enter up to 255 characters into these fields, but
 the drop down field on the ITB automated form will only display up to approximately
 35 characters.

1. Click the Save icon
 to save your changes. If the Save icon is disabled, the changes have already been
 saved.
