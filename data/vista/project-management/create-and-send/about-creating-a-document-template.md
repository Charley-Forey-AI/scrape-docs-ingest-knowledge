# About Creating a Document Template

You can create custom templates to use for generating project documents, such as
 RFI's and RFQ's, in the PM module.
The diagram below outlines the four basic steps.
Note: Custom document templates are stored in document locations.
 Before following these steps, open the PM Create & Send form and make sure that
 there is at least one document location.
