# Field Definitions: PM Response Values Form

The following is a list of field descriptions for the PM
 Response Values form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Name

Use this field to enter the name of the automated response record. The value in this field can be up to 20 characters long and it must be unique.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values

## Description

Use this field to enter a description of the automated response record. The value in this field can be up to 60 characters long.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values

## Sequence

Use this field to identify a unique line item on the Response Fields tab. When creating a new line item, this field will automatically populate with the next available number.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values

## Database Value

Do not change the value in this field.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values

## Display Value

Use this field to define what will display on the automated response form. The value in this field can be up to 255 characters long, but the drop down field on the ITB automated response form will only display the first 35 characters.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values

## Notes

Use this field to enter notes. Double click in this field to open the Grid Notes form.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values

## Notes

Use this field to enter notes. Double click in this field to open the Grid Notes form.

[](/en/vista/vista/project-management/create-and-send/pm-response-values-form)PM Response Values
