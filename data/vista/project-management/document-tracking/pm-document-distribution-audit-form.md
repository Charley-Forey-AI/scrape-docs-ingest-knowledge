# PM Document Distribution Audit Form

Information about the PM Document Distribution Audit form.
You can use this form to view:

- All of the documents generated using the [Create and Send](/en/vista/vista/project-management/create-and-send/about-the-create-and-send-feature) feature - for example change orders, RFIs, drawing logs, project issues, etc.

- Any communications sent using the Create and Send feature - for example if you resend a document to a project contact.

The Create and Send feature allows you to generate multiple documents at the same time using the Documents tab on the PM Send Documents form - for example you can create a transmittal and submittal document at the same time. The additional documents generated using this tab do not create distribution audit records on the records used to generate those documents.
