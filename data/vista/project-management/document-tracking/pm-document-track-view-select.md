# PM Document Track View Select

 Use this form to change the active document tracking view used by PM Document
 Tracking.
You can only access this form by selecting the Select Document
 Tracking View option from the File menu of PM Document Tracking.
The selection box lists all valid document tracking views that
 have been set up in PM Document Tracking Views. Once you have selected the view you wish to
 use, click the Select button to save your selection and exit the form. The PM Document
 Tracking form is refreshed (which may take a few seconds) to redisplay the data using the
 new view. The new view will remain in effect until you change it again.
Note: Override views are not company specific; therefore, changing the document
 tracking view in one company will affect all PM companies that you can access.
