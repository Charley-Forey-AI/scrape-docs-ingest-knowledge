# PM Document Edit Manager Form

Use this form to edit generated project documents before you send them to applicable contacts.
For example, you can generate an RFI document and then edit the RFI before emailing it to a project contact. If there are multiple contacts on the email, you can edit the document generated for each contact.
There are two ways to edit the generated project document. Use the one that best fits your workflow.

- Create & Send () >  icon > Create document from template - Generate and edit the project document, and then use the PM Send Documents form to include the edited document in the email. For more information, see [About Generating & Editing Project
 Documents](/en/vista/vista/project-management/create-and-send/about-generating--editing-project-documents).

- PM Send Documents >  Edit Document(s) button - Edit the generate document(s) directly from the PM Send Documents form. For more information, see [Edit a Generated Project Document Using PM Document Edit Manager](/en/vista/vista/project-management/document-tracking/edit-a-generated-project-document-using-pm-document-edit-manager).
