# PM Document History Form

Use this form to view the changes that have been made to PM module documents. For example use this form to view the changes made to the PCOs associated with a project.
 Each time you add, change, or delete a PM document (RFI, PCO, project issue, submittal, submittal package), an entry is added to this form. Each entry shows the change that was made, the old and new value, and the user that performed the change. Note: Not all changes display in the form. For example, adding, changing, or deleting an entry on the Distribution tab of a document does not create an item on this grid.
Once you ener a project and select a document category, you can use the filter bar and group bar to locate specific information.

- Filter Bar - Use the filter bar to filter the information that displays in the form - for example if you want to view all of the changes to a specific submittal package. To use the filter bar, right click in the grid and select Filter Bar from the menu that displays, and then enter a value in any of the columns in the row that displays at the top of the grid.

- Grouping Bar - Use the grouping bar to group the information by the entry in a specific row - for example group the PCOs that display in the form by PCO number or PCO type. To use the grouping bar, right click in the grid and select Grouping Bar from the menu that displays, and then drag and drop a column heading into the section at the top of the grid.
