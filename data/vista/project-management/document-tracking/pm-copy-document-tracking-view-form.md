# PM Copy Document Tracking View Form

Use this form to copy an existing view.
Access this form from PM Document Tracking Views by
 selecting File > Copy View.
Copying views is most useful for creating
 similar views that require minor modifications to meet other needs.
To copy a view, enter the source and destination views (the view you are copying and the new view), and click the Copy button. The system copies the grid setup from the source view to the new view.
You can modify new views as necessary using PM Document Tracking Grids, and then assign them t to companies in PM Company Parameters and/or select them as override views in PM Document Tracking.
