# Field Definitions: PM Document Tracking Form

The following is a list of field descriptions for the PM
 Document Tracking form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project

 Specify the project whose documents you wish to view. To view a specific type of document, click on the desired tab. To access a specific document in its originating form, double-click the grid row or select the document and click the Setup Form button.
Note:Upon initial access to this form, the default view will be the Document Tracking View specified in PM Company Parameters. If no default view is specified in PM Company Parameters, the standard ‘Viewpoint’ view is used. However, you can override the default document tracking view at any time via the Select Document Tracking View option (File menu).
