##  Tab Page Title

 Enter the title for this tab, up to 30 characters. Initially defaults the standard title. The title specified here will become the title displayed for the corresponding tab in PM Document Tracking.
IMPORTANT! It is suggested that you keep your titles as minimal as possible. The longer your titles are, the wider you may need to resize the PM Document Tracking form in order to see all tabs in the row.

##  Table View

 This column initially defaults the tables containing the columns applicable to this tab. You cannot change the table for a record; however, you can add or delete tables as necessary. Press F4 for a list of available tables.

##  Column Name

 This column initially defaults the columns applicable to this tab. You cannot change the column name; however, you can add or delete columns as necessary. Press F4 for a list of available columns for the selected table.

##  Column Title

 This column initially defaults the standard title for each column, which may be overridden. Up to 30 characters allowed.
Note: Changing the column name here will not affect the column's standard column name.

##  Visible

 Check this box if you want this column displayed in the grid when the related tab is selected for this view in PM Document Tracking.
 Leave this box unchecked if you want this column hidden when the related tab is selected for this view in PM Document Tracking.
