# Field Definitions: PM Document Tracking View Copy Form

The following is a list of field descriptions for the PM
 Document Tracking View Copy form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Source View

 Specify the view to copy. Defaults the view specified in PM Document Tracking Views.

##  Destination View

 Enter a name or code for the new document tracking view, up to 10 characters.

##  Description

 Enter a description of the destination view, up to 30 characters.
