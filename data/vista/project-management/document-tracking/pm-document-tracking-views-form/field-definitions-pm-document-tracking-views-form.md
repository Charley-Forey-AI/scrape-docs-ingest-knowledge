# Field Definitions: PM Document Tracking Views Form

The following is a list of field descriptions for the PM
 Document Tracking Views form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Document Tracking View

 Enter a name or code for this document tracking view, up to 10 characters. Once set up here, you can assign this view to one or more companies in PM Company Parameters.

##  Description

 Enter a description of this document tracking view, up to 30 characters.
