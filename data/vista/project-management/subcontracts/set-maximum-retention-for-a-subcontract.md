# Set Maximum Retention for a Subcontract

You can

 set a maximum retention amount for a subcontract to prevent further withholding of retention once the maximum retention amount is reached for the subcontract. 

 When you enter a subcontract, you have the option to set a
 maximum retention amount. You can base this amount on a percentage of the subcontract or a
 flat amount.Once the system reaches the maximum retention amount, it will no longer
 continue withholding retainage for the specified subcontract. See [Applying Maximum Retention Amounts to Subcontracts](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-subcontract-maximum-retention-amounts) for more information.
The
 following instructions detail how to set the maximum retention amount for a
 subcontract.

1. In the Vista main menu, go to Project Management > Programs > PM Subcontracts.

1. In the Project field, enter the project to work
 on or press F4 to select from a
 list of valid projects.

1. In the
 Subcontract field, select the subcontract to work on or press
 F4 to select from a list of valid subcontracts for the
 project.

1. In the Set WC Maximum Retention
 group box, choose one of the followingOptionDescription
Percent of Subcontract

1. In the % of Subcontract field, enter the maximum percent
 of the subcontract amount that is subject to retention.The system defaults the
 maximum retention amount in the Max Amt by % field based on the
 original subcontract amount or the current subcontract amount, depending on
 whether you are including change orders in maximum retention percent (see next
 step). If you have not set up any subcontract items, this field deaults as
 0.00.

1. If you want to include change order amounts in percentage-based maximum
 retainage calculations, select the Include Chg Orders in Max Retention
 % checkbox.

1. From the Adjust Maximum Invoice drop-down, select the correct method for
 retention distribution when the system reaches the maximum amount. For more
 information, see the [Adjust Maximum Invoice](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-subcontract-entry-form/field-definitions-sl-subcontract-entry-form#ID-0003f369--en) F1 help.

Maximum Amount

1. In the Retention Amount field, enter the maximum amount
 of retainage that can be withheld for this subcontract.

1. From the Adjust Maximum Invoice drop-down, select the correct method for
 retention distribution when the system reaches the maximum amount. FOr more
 information, see the [Adjust Maximum Invoice](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-subcontract-entry-form/field-definitions-sl-subcontract-entry-form#ID-0003f369--en) F1 help.

1. Save the record.
