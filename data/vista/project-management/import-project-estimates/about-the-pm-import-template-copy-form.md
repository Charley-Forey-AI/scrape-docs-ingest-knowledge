# About the PM Import Template Copy Form

This form can only be accessed by selecting the Template Copy option from the File menu of PM Import Estimates Template, and is used to copy an existing import template to a new import template.
Once you specify the source and destination templates, you must specify which cross-reference records you want copied to the new template. Cross-reference files available for copying are as follows:

- Phases

- Cost Types

- Unit of Measures

- Materials

- Vendors

- Create Cost Types

Once you have made your selections, click the Copy button. The copy process will copy the template header, template detail, and all selected cross-references to the new import template. You can edit the template as necessary in PM Import Estimates.
