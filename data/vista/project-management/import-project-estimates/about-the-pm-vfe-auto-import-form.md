# About the PM VFE Auto Import Form

Use this form to set up and initiate an
 automatic VFE data import and upload or import only.
You can access this form from the PM Projects form by selecting Tasks > Import from Viewpoint for Estimating - MEP or Tasks > Import from Viewpoint for Estimating - AccuBid .
 Some fields have default values that
 can be overridden. Press F1 on each of those fields for more
 information.
After you click Import, the PM
 VFE Auto Import Results form will appear. If there are data errors detected in your import,
 you will have the ability to launch the Work Edit form.
