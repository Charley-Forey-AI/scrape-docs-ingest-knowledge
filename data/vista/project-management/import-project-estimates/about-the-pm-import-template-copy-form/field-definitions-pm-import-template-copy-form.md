# Field Definitions: PM Import Template Copy Form

The following is a list of field descriptions for the PM
 Import Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Source Template

Specify the import template to copy. Defaults
 to the active template (i.e. template specified in PM Import Templates).

## Dest Template

Enter the import template, up to 10 characters, to which the existing template will be copied. Must be a new template; cannot copy to an existing template.

##  Description

 Enter a description of the new template, up to 30 characters.

## Copy Template Cross Reference Files

Check the box for each of the cross-reference files you want copied to the new template.

- Phases

- Cost Types

- Unit of Measures

- Materials

- Vendors

- Create Cost Types
