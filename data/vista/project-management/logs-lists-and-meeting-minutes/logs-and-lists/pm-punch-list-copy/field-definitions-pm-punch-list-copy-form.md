# Field Definitions: PM Punch List Copy Form

The following is a list of field descriptions for the PM
 Punch List Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## From Punch List

 Enter the punch list to copy from. Must be a valid punch list for the current project. All items existing for the source punch list will display in the Punch List Items grid below.

## To Punch List

Enter the punch list to copy to, up to 10 characters.

## Select/Unselect All Items

Once the source punch list has been entered,
 all of the items defined for that punch list are displayed in the Punch List Items grid
 below.

- Select All Items – Use this button to
 select all of the items in the grid for copying.

- Unselect All Items – Use this button to undo all selections.

To select items individually, highlight the
 desired item (by clicking in the row from the far right column). To select multiple items
 consecutively, press the Shift key and select the first and last record in a block of
 records to copy. To select multiple records randomly, press the Ctrl key and select each
 desired record.

## Unfinished Items Only

Check this box to copy only those punch list items that are unfinished (i.e. do not have a ‘finished’ date). If you use the ‘Select All Items’ feature and then check this box, finished items will be cleared from the grid.
Leave this box unchecked to copy all items, regardless of whether they are ‘finished’ or ‘unfinished’.

## Renumber Items

Check this box to renumber copied items on the new punch list. When checked, the Start Renumbering and Increment By inputs are enabled.
Leave this box unchecked if items are to be copied to the new punch list using the existing item numbers.

## Start Renumbering At

Enter the number with which to begin renumbering items. Items will be renumbered based on the Increment By option below.

## Increment By

Enter the amount by which to increment punch list items. For example, if renumbering begins with 1 and the ‘increment by’ amount is 2, punch list items will be numbered as follows:  1, 3, 5, 7, 9, and so forth.
