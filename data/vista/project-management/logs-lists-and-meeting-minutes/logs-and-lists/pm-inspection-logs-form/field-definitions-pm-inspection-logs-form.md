# Field Definitions: PM Inspection Logs Form

The following is a list of field descriptions for the PM
 Inspection Logs form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Project

Specify the project (from PM Projects) for which to set up inspection logs.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

## Inspection Type

Specify the inspection type for this inspection log. Must be a valid, active document type with a category of 'Inspection Logs' (as defined in PM Document Types).
Note:To display a description field for this field on the
 Grid tab, access Field Properties (F3) and select an option from the Description in
 Grid drop-down.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

##  Inspection Code

Create a new inspection
When creating a new inspection code, you are
 creating a new inspection for the type selected in the Inspection Type field.
There are three ways to create a new inspection using this field:

- Enter a '+' and then press TAB to exit the field. The system will
 automatically create a new inspection log of the selected type and assign it the next
 available inspection code number.

- Click the New Record icon () at the top
 of the form and then select a type in the Inspection Type field. The system will
 automatically populate the Inspection Code field with the next
 available inspection number. Press TAB to exit the field and the system will create the
 new inspection log.

- Enter an inspection code number that does not exist for the type
 selected in the Inspection Type field and then press TAB to exit the field. You can also
 press F4 to see a list of inspection numbers that have already been created. The
 inspection code can be up to 10 characters long.

Note: The system will ignore alphanumeric numbers when
 determining the next number to assign. Select an existing inspection

Enter an existing inspection code or press F4
 to select it from a list. Only inspections of the type selected in the Inspection Type
 field will display in the list.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

##  Description

 Enter a description of this inspection, up to 30 characters.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

##  Location

 Enter the location of the inspection. Must be a valid location set up for the project in PM Project Locations.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

##  Date

 Enter the date of the inspection.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

##  Firm

 Enter the firm performing the inspection or
 press F4 to select a firm from a list. Firms are created and maintained using the
 [PM Firms](/en/vista/vista/project-management/setupmaintenance/about-the-pm-firms-form) form.
Tip: The sort name can be used instead of the firm number
 when selecting firms in PM module forms. Generally the sort name is the first several
 letters of the firm name.  For example when creating a subcontract in the [PM Subcontracts](/en/vista/vista/project-management/subcontracts/pm-subcontracts-form) form for 'Bryan Electrical', you can enter the sort name 'bryan'
 in the Vendorfield instead of the firm number '10042'. The sort name of a firm is
 set up using the Sort Name field on the [PM Firms](/en/vista/vista/project-management/setupmaintenance/about-the-pm-firms-form) form.
[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

## Contact

Enter the contact from the specified firm or
 press F4 to select one from a list.
Must be a valid contact set up for the firm in
 PM Firm Contacts. This can be either the person performing the inspection or the person to
 contact regarding the inspection.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

##  Inspector

 Enter the name or initials of the person who performed the inspection. Up to 30 characters allowed.If the contact specified in the previous field is the person who performed the inspection, you can either enter their name/initials here or leave the field blank.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs

## Status

Enter the status of the inspection. Must be a valid status code set up in PM Status IDs.

[](/en/vista/vista/project-management/logs-lists-and-meeting-minutes/logs-and-lists/pm-inspection-logs-form)PM Inspection Logs
