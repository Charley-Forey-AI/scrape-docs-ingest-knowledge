# PM Drawing Logs Form

Use the PM Drawing Logs form to create and maintain drawing logs.
Use the upper section of this form to
 set up information about the drawing log (i.e., drawing type/number, description,
 status, and if applicable, any issue associated with the log), and then use the
 lower portion of the form to enter revisions on the drawing log.
You can add a revision to a drawing log by creating a new item in the lower portion
 of the PM Drawing Logs form.
If you need to add a revision to several drawing logs at once, select File >  Initialize Revision.
