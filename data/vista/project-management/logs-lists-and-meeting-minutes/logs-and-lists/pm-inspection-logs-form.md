# PM Inspection Logs Form

Use the PM Inspection Logs form to create and maintain
 inspection logs, and then use the Create and Send feature to generate an inspection log
 document.
Inspection logs are categorized by inspection type and a unique
 code. The inspection types (fire, electrical, rebar, etc.) are set up by an admin using
 the PM Document Types form.
Tip: The PM Inspection Logs form and PM Test Logs form are almost
 identical.
The History tab displays all of the
 additions, changes, and deletions made to the inspection log. Each history record shows
 what change that was made, who made the change, and when it was changed.
