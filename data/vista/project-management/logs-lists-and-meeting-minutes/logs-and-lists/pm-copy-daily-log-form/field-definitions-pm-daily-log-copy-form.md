# Field Definitions: PM Daily Log Copy Form

The following is a list of field descriptions for the PM
 Daily Log Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Daily Log Date

 Enter the date of the new log.

##  Daily Log

 Enter a number for the new log (0-32,767). Duplicate numbers can only be used if logs have different dates.

## Header Notes

Check this box to copy header notes to the new daily log. When checked, the copy will include all notes entered for the source daily log on the Notes tab.
Leave this box unchecked if you do not want to copy header notes to the new daily log.

## Header User Memos

Check this box to copy header user memos (custom fields) to the new daily log. When checked, the copy will include all user memos entered for the source daily log (on the Info tab or any custom tabs).
Leave this box unchecked if you do not want to copy header user memos to the new daily log.

## Copy Options

Check the box for each option corresponding to the information you want copied to the new daily log.

- Employees

- Crews

- Subcontracts

- Distribution

- Equipment

- Activities

- Header Notes

- Weather Info

- Header User Memos

- Detail Notes

- Detail User Memos

Leave the box unchecked for options corresponding to information you do not want copied to the new daily log. (Note: Conversations, Deliveries, Accidents, and Visitors cannot be copied.)
Copy
Click this button to implement the copy process. Once completed, you are returned to PM Daily Logs, with focus on the new daily log.
