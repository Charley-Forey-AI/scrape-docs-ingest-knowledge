# PM Punch List Copy

Use this form to copy an existing punch list to a new punch list.
This form is accessed by selecting the Punch List Copy option from the File menu of PM Punch List.
To initiate the copy, specify the punch list to copy (must be a valid punch list for the current project). All of the items defined for the source punch list are displayed in the Punch List Items grid below.
Next, enter the new punch list and use the grid to select the punch list items to copy. You can select items individually, using the Shift key to select items consecutively or the Ctrl key to select items randomly. If you want only ‘unfinished’ items (those not assigned a Finished Date) to be copied, check the Unfinished Items Only option. All ‘finished’ items are cleared from the grid.
The Renumber Items option allows you to renumber copied items on the new punch list. When checked, the ‘start renumbering’ and ‘increment by’ options are enabled, allowing you to define how the items will be renumbered on the new punch list. For example, if you want to begin renumbering at 100, but you want to leave room between the existing items to enter new items, you might set the ‘increment by’ value to ‘5’ to allow for adequate spacing. When the existing items are renumbered, the numbers will be assigned as follows: 100, 105, 110, 115, etc. Later, if you want to manually add additional items, you can use the unused numbers: 101, 102, 103, 104, and so on.
Once you have entered the item parameters, click the Copy button. The copy process will copy the items, item detail, and notes (header and item) to the new punch list. You can modify the new punch list as necessary in PM Punch List.
