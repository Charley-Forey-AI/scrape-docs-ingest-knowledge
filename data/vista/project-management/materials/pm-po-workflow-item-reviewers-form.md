# PM PO Workflow Item Reviewers Form

Use the PM PO Workflow Item Reviewers form to see a list of reviewers assigned to the purchase order, along with the approval status.

This form is part of the [Process Workflow](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow) feature.
Access this form by selecting the Workflow button on the
 PM Purchase Orders form. The Workflow button is only enabled if the PO is included in a workflow (for example, when the amount on the PO item is greater than your spending limit). If a workflow doesn't apply, Approval Not
 Required displays in the Workflow Status field and the Workflow button is disabled.
Note:The Workflow button only
 displays if you have the Workflow module.

This form changes depending on the Workflow Status. If the purchase order requires approval, but has not yet been submitted for approval, this form displays as Pending Reviewers and shows the workflow that applies to each reviewer for each PO item, along with their approval limit and whether they are a required or optional reviewer.
Once the PO is submitted for approval, the title changes to PM PO Workflow Item Reviewers and the form displays the progress of each reviewer; that is, the approval status and any comments or notes entered for that item. The PO Item field at the top of the form allows you to select the PO item to view.
Note: Users cannot approve their own purchase orders. Therefore, if the user who created the PO is also a reviewer, they are automatically removed from the review/approval workflow.
