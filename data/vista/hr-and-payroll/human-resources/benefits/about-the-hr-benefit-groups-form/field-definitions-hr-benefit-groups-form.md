# Field Definitions: HR Benefit Groups Form

The following is a list of field descriptions for the HR
 Benefit Groups form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Benefit Group

 Enter a code, up to 10 characters, that uniquely identifies this benefit group.

##  Description

 Enter a description of this benefit group, up to 30 characters.

##  Benefit Code

 Enter a valid benefit code (set up in HR Benefit Codes) for this benefit group.

##  ElectiveYN

 Check this box to indicate that this is an elective benefit. If checked, this benefit code is initialized as inactive, and will need to be manually activated in HR Resource Benefits.
 Leave this box unchecked if this is not an elective benefit. Benefit will be initialized as ‘Active’.
