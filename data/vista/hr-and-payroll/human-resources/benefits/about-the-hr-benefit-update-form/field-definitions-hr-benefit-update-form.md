# Field Definitions: HR Benefit Update Form

The following is a list of field descriptions for the HR
 Benefit Update form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  HR Benefit Update

##  Benefit Code

 Enter the benefit code to update. Press F4 to see a list of valid benefit codes.

##  Include Inactive Resources

 Select this checkbox to have the system update inactive resources.

##  Only Update Matching Rates

 Select this checkbox to updated resources whose benefit rate (earnings or deduction/liability) matches the Old Rate for the benefit code.
