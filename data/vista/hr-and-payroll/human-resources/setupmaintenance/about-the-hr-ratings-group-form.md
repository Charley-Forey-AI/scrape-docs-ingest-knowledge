# About the HR Ratings Group Form

Use this form to set up ratings groups.
Ratings groups allow you to group together rating codes (set up in HR Codes) that will be initialized to position codes or a resource’s review.
 For example, if you have a group of employees for which you rate performance, attendance, working relationships, and general attitude, you would set up each item as a rating code, then add them to a Ratings Group. Then, whenever you perform a review or evaluation, you can initialize the ratings group to the resource and each of the items (rating codes) will automatically be set up for the resource.
 You can also initialize a ratings group to a position code so that each time the position is filled, the resource is automatically set up with the rating codes for future performance ratings.
