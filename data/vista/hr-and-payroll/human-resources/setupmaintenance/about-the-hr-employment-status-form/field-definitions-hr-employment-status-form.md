# Field Definitions: HR Employment Status Form

The following is a list of field descriptions for the HR
 Employment Status form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Employment Status Code

 Enter a code, up to 10 characters, that uniquely identifies an employment status.
 Examples:
 APPL = Applicant
 EMPL = Employee
 MED = Medical Leave
 MAT = Maternity Leave
 RET = Retired

##  Description

 Enter a description of this employment status code, up to 30 characters.

##  Update Employment History

 Check this box if use of this status code in
 HR Resources will be logged as an event in the HR Employment History (HREH) table. When
 checked, each time this status code is added to or removed from a resource, an event will
 be logged in HREH.
 Leave this box unchecked if use of this status code will not be logged in HR Employment History.

##  History Code

 Enabled only if this status code is flagged to update the HR Employment History table.
 Specify the history code (HR Codes, Type H)
 to which use of this employment status code will be logged in the HR Employment History
 table.
