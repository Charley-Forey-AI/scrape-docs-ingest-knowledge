#  About the HR Initialize Rating Groups Form

This form is used to initialize performance rating codes to a
 position code or a resource/review.
It can be accessed as follows:

- In HR Position Codes – Select the Initialize Performance Ratings option from the File menu.

- In HR Resource Review – Select the Initialize Performance Ratings from Group option in the File menu.

 Rating codes (set up in HR Codes) identify items
 that are rated during a performance review or
 evaluation. These codes can then be assigned to
 ratings groups in HR Ratings Group. Initializing
 from a ratings group here allows you to quickly
 assign all applicable performance rating codes to a
 position code (HR Position Codes) or a
 resource/review date (HR Resource Review).
Note: You can initialize rating codes from more than one ratings
 group. Initialization will add only those ratings codes that do not already exist for
 the position or the resource/review.
