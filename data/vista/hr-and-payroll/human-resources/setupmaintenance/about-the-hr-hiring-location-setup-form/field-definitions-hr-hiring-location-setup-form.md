# Field Definitions: HR Hiring Location Setup Form

The following is a list of field descriptions for the HR
 Hiring Location Setup form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Location ID

The Location ID field on the HR Hiring Location Setup form.
Enter a location ID number to uniquely identify this location. This entry does not appear in the HR Federal Contractor VETS-4212 report.

## Name

The Name field on the HR Hiring Location Setup form.
 The name entered in this field appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## Address 1

The Address 1 field on the HR Hiring Location Setup form.
 Enter the street address for the hiring location. This entry appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## Address 2

The Address 2 field on the HR Hiring Location Setup form.
 Enter additional address information for the hiring location. This entry appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## City

The City field on the HR Hiring Location Setup form.
 The city entered in this field appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## State

The State field on the HR Hiring Location Setup form.
Enter a valid state (as defined in HQ States). The system validates the state based on the Default Country specified in HQ Company Setup for the active company. If not a valid state/country combination, an error displays. You must then enter a valid country for this state in the Country field. Press F4 to view and select from a list.
 The state entered in this field appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## Zip

The Zip field on the HR Hiring Location Setup form.
 The zip code entered in this field appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## Country

The Country field on the HR Hiring Location Setup form.
Enter the 2-character country code. Entry in this field is required when the state is outside the Default Country specified in HQ Company Parameters for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.
This entry does not appear in the HR Federal Contractor VETS-4212 report.

## County

The County field on the HR Hiring Location Setup form.
 Enter the county for this location. Entry in this field appears in the HR Federal Contractor VETS-4212 report if there is at least one employee assigned to this location and the employee has timecard records during the applicable reporting period.

## DUNS Number

The DUNS Number field on the HR Hiring Locations form.
Enter the 9-digit DUNS number for this hiring location. The number entered here overrides the DUNS number specified for the company in HQ Company Setup.
If you leave this field blank, the system will use the DUNS number specified in HQ Company Setup.
The system includes this number on the HR Federal Contractor VETS-4212 report (used for exporting data for e-filing) and the HR Federal Contractor VETS Resources report (used for viewing or printing).

## NAICS Code

The NAICS Code field on the HR Hiring Location Setup form.
Enter the North American Industry Classification System (NAICS) code (up to 6 digits) for this hiring location. The code specified here overrides the NAICS code specified for the company in HQ Company Setup.
If you leave this field blank, the system will use the NAICS code specified in HQ Company Setup.
The system includes this code on the HR Federal Contractor VETS-4212 report (used for exporting data for e-filing) and the HR Federal Contractor VETS Resources report (used for viewing or printing).
