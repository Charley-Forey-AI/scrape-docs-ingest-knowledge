# Field Definitions: HR Ratings Group Form

The following is a list of field descriptions for the HR
 Ratings Group form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Rating Group

 Enter a code, up to 10 characters, that uniquely identifies this performance ratings group.

##  Description

 Enter a description of this ratings group, up to 30 characters.

##  Rating Code

 Enter the rating code (HR Codes, Type R) to
 add to this performance ratings group.
 Rating codes identify items that are to be rated during a performance evaluation. If initializing performance ratings to a position code or resource/review (via HR Initialize Rating Groups), all rating codes set up here are automatically set up for the position or resource/review date.
