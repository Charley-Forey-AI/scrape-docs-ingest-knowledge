# About the HR Lic Restrict/End Copy Form

 Use this form to copy license endorsements/restrictions from one state to another.
You can access this form by selecting
 File > Copy State License from HR License
 Endorsements.
Once you have specified both the source and
 destination states, click Copy to implement the copy process.
 All endorsements, restrictions, and certificates defined for the source state will be
 copied to the destination state. License codes already existing in the destination state
 will be skipped.
Note: The system will not enable the Copy button until you
 enter both a source and destination state.
