# Field Definitions: HR License Endorsements Form

The following is a list of field descriptions for the HR
 License Endorsements form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Country

 Enter the 2-character country code. Entry in this field is required when the address exists outside the Default Country specified in HQ Company Parameters for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.

## State

Enter a valid state (as defined in HQ States) for these license endorsements. The system validates the state based on the Default Country specified in HQ Company Setup for the active company. If not valid, an error displays, but entry is allowed. You must then enter a valid country for this state in the Country field.

##  Lic Code Type

 Specify the license code type.
 R = Restriction
 E = Endorsement
 C = Certificate

##  Lic Code

 Enter the license code (up to 10 characters) that identifies this restriction, endorsement, or certificate, as defined by the state specified above.

##  Description

 Enter a description of the license restriction, endorsement, or certificate, up to 250 characters.
