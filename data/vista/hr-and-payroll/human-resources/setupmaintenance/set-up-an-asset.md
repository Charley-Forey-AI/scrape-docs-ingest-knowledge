# Set Up an Asset

Set up an HR Asset.

1. Enter information in the
 Category field. Note: The asset category is a non-validated
 field. However, it is used on reports to group assets by category, so you
 may want to develop a naming scheme and usage rules suited to your company's
 needs.

1. Enter information as
 applicable into the Description through
 Warranty Exp Date fields. For more information on each field,
 press F1 for field Help topics.

1. In the Status field, select an option. The status of an asset
 indicates whether it is available for use. See [Status](/en/vista/vista/hr-and-payroll/human-resources/setupmaintenance/about-the-hr-company-assets-form/field-definitions-hr-company-assets-form#ID-000107bb--en)
 for specific information.

1. Enter information in the
 Status Memo field.
