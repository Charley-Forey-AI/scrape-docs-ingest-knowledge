# Field Definitions: HR Mine Form

The following is a list of field descriptions for the HR Mine
 form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  MSHA ID

 Enter the MSHA (Mine Safety & Health Administration) mine identification number, up to 10 characters.

##  Mine Name

 Enter the name of the mine, up to 30 characters.

##  Operator

 Specify who the operator of the mine is, up to 30 characters.

##  Begin Date

 Enter the date the specified operator began operation of the mine.

##  Status

 Enter the status of the mine (e.g. Active, Abandoned, etc.), up to 10 characters.

##  Status Date

 Enter the date the mine status (indicated above) went into effect (i.e. the date the status last changed).

##  Mine Material

 Enter the type of material being mined (e.g. crushed sandstone, lime, etc.), up to 30 characters.

##  Mine Type

 Specify the type of mine (e.g. Surface, Facility, etc.), up to 15 characters.

##  Location

 Enter the location of the mine, up to 30 characters.

##  Address

 Enter the address of the mine, up to 60 characters.
Note: If you have Internet access, you can click the Map
 button for direct access to the default map site for your login (as defined in User
 Options, Main Menu). Map will default the approximate location of the specified country and
 address. If a country is not specified, attempts to locate address based on Default Country
 specified in HQ Company Setup.

##  City

 Enter the city in which the mine is located, up to 30 characters.
Note: If you have Internet access, you can click the Map
 button for direct access to the default map site for your login (as defined in User
 Options, Main Menu). Map will default the approximate location of the specified country and
 address. If a country is not specified, attempts to locate address based on Default Country
 specified in HQ Company Setup.

## State

Enter a valid state (as defined in HQ States). The system validates the state based on the Default Country specified in HQ Company Setup for the active company. If not valid, an error displays, but entry is allowed. You must then enter a valid country for this state in the Country field.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

## Zip Code

Enter the zip code, up to 12 characters.
If you have Internet access, you can click the
 Map button () to view the address using your
 default map site (Options > User Options). If you did not specify a country, the system uses the Default Country
 defined in HQ Company Setup.

##  Country

 Enter the 2-character country code. Entry in this field is required when the address exists outside the Default Country specified in HQ Company Parameters for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.
Note: If you have Internet access, you can click the Map
 button for direct access to the default map site for your login (as defined in User
 Options, Main Menu). Map will default the approximate location of the specified country and
 address. If a country is not specified, attempts to locate address based on Default Country
 specified in HQ Company Setup.

##  Phone

 Enter the phone number, up to 20 characters.
