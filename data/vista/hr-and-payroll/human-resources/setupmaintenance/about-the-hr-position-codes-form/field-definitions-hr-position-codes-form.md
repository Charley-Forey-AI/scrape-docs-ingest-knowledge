# Field Definitions: HR Position Codes Form

The following is a list of field descriptions for the HR
 Position Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Position Code

 Enter a code, up to 10 characters, that uniquely identifies this position.

##  Job Title

 Enter this position’s title, up to 30 characters.

## Description

Use this box to describe the position (what the job entails). You may also consider using this section to store an official description of this job as used in newspaper ads or internal postings. The space allowance is virtually unlimited.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.

##  Annual Salary Range: Beginning

 Specify the starting salary for this position. Informational only.

##  Annual Salary Range: Ending

 Specify the ending salary for this position. Informational only.

##  Part-Time?

 Check this box if this position is a part-time position.
 Leave this box unchecked if this is a full-time position.

##  Part-Time Weekly Hours

 Enabled only if the Part-time option is checked.
 Enter the number of hours per week allowed for this part-time position.

##  Bonus Eligible?

 Check this box if this position is eligible for bonuses.
 Leave this box unchecked if this position is not eligible for bonuses.

##  Bonus %

 Enabled only if the Bonus Eligible option is checked.
 Specify the percent of this position’s salary that is allowable for bonuses.

##  Type

 Specify the position type.

- Exempt – Select this option if this position is classified as exempt.

- Non-Exempt – Select this option if this position is classified as non-exempt.

- Union – Select this option if this is a union-type position.

##  Number of Open Positions

 Specify the number of openings available for this position.

##  Closing Date

 Specify the closing date for this position; that is, the date on which postings for this position will expire.

##  Position Contact

 Specify the person who should be contacted in regards to this position.

##  Contact Phone

 Enter the contact’s phone number, up to 20 characters.

##  Fax

 Enter the contact’s fax number, up to 20 characters.

##  Advertised?

 Check this box if this position was advertised.
 Leave this box unchecked if this position was not advertised.

##  Mode of Advertisement

 Specify the medium used to advertise this position (e.g. newspaper, bulletin board, Internet, etc.)

##  Reporting Level

 This field is informational only, and is intended for use in the construction of organizational charts.
 Specify what level (numeric) this position reports to, up to 4 digits. For example, the President’s position would be Level 1, the Vice President’s would be Level 2, and so forth.

##  Report-to Position

 Specify what position this position reports to (e.g. President, Project Manager, etc.)

##  Resource #

 Display only, each of the resources currently assigned to this position.

##  ActiveYN

 Display only, indicating if the specified
 resource is currently an ‘active’ payroll employee (i.e. the Active PR Employee flag in HR
 Resources is checked).

##  Qualifications: Skill Code

 Using skill codes defined in HR Codes (Type
 S), identify what skills are preferred for this position. A description of the skill code
 displays to the right of this field.

##  Qualifications: RequiredYN

 For each skill code specified, designate whether or not the skill is required to attain this position.

##  Training: Train Code

 Using training codes defined in HR Codes
 (Type T), indicate the training necessary for this position. A description of the training
 code displays to the right of this field.

##  Training: RequiredYN

 For each training code specified, designate whether or not the training is required to attain or maintain this position.

##  Sequence

 Display only, the system-assigned sequence number for this performance code. Number is assigned once the performance code is entered.

##  Rating Code

 Using rating codes defined in HR Codes (Type
 R), specify what items will be rated for this position. A description of the rating code
 displays to the right of this field.
Note: If you have set up ratings groups (HR Ratings Group), you can initialize a group of ratings codes to a position using the HR Initialize Performance Rating Groups form.
