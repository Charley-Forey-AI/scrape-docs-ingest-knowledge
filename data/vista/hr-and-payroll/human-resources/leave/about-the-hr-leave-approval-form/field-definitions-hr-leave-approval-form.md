# Field Definitions: HR Leave Approval Form

The following is a list of field descriptions for the HR
 Leave Approval form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Approval Group

 Enter the approval group for filtering, if you belong to more than one approval group.

##  Status

 Select a status for filtering the grid. Leave blank to filter for all statuses. The following statuses are available:

- New

- Approved

- Denied

- Cancelled

##  From Date

 Enter the date that you want to start filtering from.

##  Resource

 Enter the number for the resource (employee) for filtering the grid.

##  Select All

 Select this checkbox to have the system select every leave request checkbox in the grid.

##  Action on Selected

 Use this drop-down to select the status for all requests selected in the grid. The following statuses are available from this field:

- New

- Approved

- Denied

- Cancelled

##  Action

 Select a status for this leave request. The following statuses are available:

- New

- Approved

- Denied

- Cancelled

##  Approver Comments

 Enter any comments for this leave request. Comments entered here display on the HR Leave Request form.
