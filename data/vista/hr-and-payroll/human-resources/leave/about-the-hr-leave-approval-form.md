# About the HR Leave Approval Form

Use the HR Leave Approval form to review and approve leave
 requests.
Once an employee associated with your
 approval group enters a leave request in HR Leave Request, the request displays on this
 form.

## Approving Leave Requests

When you first access this form, the
 Approver display field automatically defaults to your resource
 number. The grid displays all new leave requests for resources associated with your
 approval group. Depending on the resource name selected on the grid, the system
 displays available information based on the leave code in the Leave Totals by
 Resource and Code section of the form. This section displays the available leave
 total for the code, pending leave (scheduled leave that has not been taken yet),
 balance leave (available leave minus pending leave), and the last date that leave
 information was posted from the Payroll (PR) module.
Note: Leave totals may not be accurate if information
 has not been posted recently from PR. If the date in the Last PR Post
 Date field is more than a few weeks old, you may want to
 double-check leave totals with your organization’s payroll department. The PR Auto
 Leave Accrual/Usage form is used to post PR leave information to HR.
To approve a leave request, select the
 checkbox for the request record in the grid and select a new status for the request
 from the Action drop-down. Available statuses include: New, Approved,
 Denied, and Cancelled. Additionally, you can enter any applicable comments in the
 Approver
 Comments field. Repeat for all requests that require review and
 approval and then click Update. The system sends an email
 notification to the requesting resource via their notification preference. For more
 information on setting notification preferences, see [Setting User Notification Preferences](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/set-user-notification-preferences).
Tip: To approve all requests in the grid, check the
 Select
 All box. The system selects the checkbox for each request in the
 grid. Select a new status for all requests using the Action
 drop-down.

## Viewing Leave Requests

If there are a number of requests in the
 grid, you can use the fields in the Show Requests By section of the form to search
 for requests by approval group, status, the earliest date to view, and/or resource.
 Enter the filtering criteria in the fields and click Refresh
 button. The grid displays all requests meeting the criteria.
