# Field Definitions: HR Leave Request Form

The following is a list of field descriptions for the HR
 Leave Request form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Start Date

 Enter the starting date in a range for filtering leave requests.

##  End Date

 Enter the ending date in a range for filtering leave requests.

##  Leave Code

 Enter the leave code for filtering leave requests.

##  Date

 Enter the date of the leave request.

##  Leave Code

 Enter the type of leave code for this request. Press F4 to see a list of available leave codes.

##  Hours

 Enter the number of hours for the leave request.

##  Requester Comments

 Enter any comments you have about the leave request.
