# Creating a Leave Approval Group

You can use the HR Approval Groups form to create a leave
 approval group.
 To create an approval group:

1. Enter a number for the group in the Approval Group field. Press F4 to see a list of previously created groups.

1. Enter a group description in the Approval Group Description field.

1. Enter a resource number in the Primary Approver field. Press F4 to see a list of resources.

1. If necessary, enter a resource number in the Secondary Approver field. Press F4 for a list of resources.

1. Select the Email Notification checkbox for the approver(s) if the system should send a notification when a resource enters a leave request. Note: If you specify a primary and secondary approver, and both have the Email Notification checkbox selected, both approvers receive the email automatically. The system does not enforce an approver hierarchy.

1. Save the record as normal.

1. Assign the group number
 to each applicable resource in HR Resources (Leave Approval Group field, Other
 Info tab). Note: Once you assign the group number to
 employees, their resource number and name displays on the Resources tab of
 this form.
