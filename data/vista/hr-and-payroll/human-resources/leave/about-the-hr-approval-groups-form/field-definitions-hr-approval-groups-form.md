# Field Definitions: HR Approval Groups Form

The following is a list of field descriptions for the HR
 Approval Groups form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Approval Group

Enter a number for the approval group, up to
 three numeric characters. Press F4 to see a list of previously created
 approval groups.

## Approval Group Description

Enter a description for the approval group, up to 30 characters.

##  Primary Approver

 Enter a resource number for the group’s primary approver. Press F4 to see a list of resources. This is a required field.

## Email Notification

Select this checkbox to have the system send the primary approver a notification when a resource associated with the group enters a leave request (HR Leave Request). Email notifications are sent via the approver’s chosen delivery method in VA User Profile. For more information on selecting a notification delivery method, see VA User Profile in Related Topics below.

[](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form)VA User Profile

##  Secondary Approver

 Enter a resource number for the group’s secondary approver, if necessary. Press F4 to see a list of resources.

## Email Notification

Select this checkbox to have the system send the secondary approver a notification when a resource associated with the group enters a leave request (HR Leave Request). Email notifications are sent via the approver’s chosen delivery method in VA User Profile. For more information on selecting a notification delivery method, see VA User Profile in Related Topics below.
Note: If you specify a primary and secondary approver, and both have the Email Notification checkbox selected, both approvers receive the email automatically. The system does not enforce an approver hierarchy.
