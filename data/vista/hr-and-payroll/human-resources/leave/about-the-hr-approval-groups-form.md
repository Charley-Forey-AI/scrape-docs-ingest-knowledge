#  About the HR Approval Groups Form

Use this form to create leave approval groups.
Leave approval groups have at least one associated approver and any number of employee resources. When employees request time off using the HR Leave Request form, the approver(s) are responsible for approving or denying the leave request using the HR Leave Approval form. For more information on requesting and approving leave, refer to HR Leave Request and HR Leave Approval in Related Topics below.
