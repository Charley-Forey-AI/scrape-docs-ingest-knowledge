# About the HR Resource Skills Form

Use this form to define and track specific skills for
 resources.
 Access this form by clicking the Skills button at
 the bottom of the Skills tab in HR Resources, selecting the Skills option from the File
 menu of HR Resource Training, or from the Viewpoint Main Menu.
 Skill codes (set up in HR Codes, Type S) are used
 to identify all of the skills an applicant/employee has. For those skills where
 certification applies, a certification date can be entered. If the certification must be
 renewed periodically, an expiration date can be entered to indicate when
 re-certification is due.
 You can quickly view all of the skills defined for
 an employee/applicant in HR Resources, Skills tab. Additionally, you can track changes
 to skill information by checking the Skills option in the Employment History Event
 Option section of HR Company Parameters.
