# Field Definitions: HR Resource Review Form

The following is a list of field descriptions for the HR
 Resource Review form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Disabled if this form was accessed
 from HR Resources.
 Indicate the resource (from HR Resources) for
 whom you are setting up review information.

##  Review Date

 Enter the review date. Press F4 for a list of review dates for this employee.

##  Reviewer

 Enter the name of the person conducting this review, up to 30 characters.

##  Position Code

 This field initially defaults to
 the position code specified for this resource in HR Resources. If the position has changed,
 enter the code (from HR Position Codes) that identifies the position for which this
 employee is being reviewed.

##  New Review Date

 Enter the date of this employee’s next review. This can be used in reports to indicate which employees have reviews coming up.

## Review Notes

Use this section to enter any notes pertaining to this review.  The space allowance in this box is virtually unlimited.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.

##  Seq #

 Display only, the system-assigned sequence number for this performance rating. Add items in the order you wish them to appear.
 If you used the ‘Initialize Performance Ratings from Group’ or ‘Initialize Performance Ratings from Position’ features, all of the rating codes assigned to the specified ratings group or position code (respectively) are automatically set up in this grid with sequence numbers automatically assigned by the system.

##  Rating Code

 Enter the code (from HR Codes, Type R) that
 identifies an item being evaluated for this review.
 If you used the ‘Initialize Performance Ratings from Group’ or ‘Initialize Performance Ratings from Position’ features, all of the rating codes assigned to the specified ratings group or position code (respectively) are automatically set up here.

##  Rating

 Enter a number (0-255) that represents how this employee performed this specific item. Most companies have a predefined rating scheme. For example:
 1=Poor
 2=Fair
 3=Average
 4=Above Average
 5=Exceeds Expectations
