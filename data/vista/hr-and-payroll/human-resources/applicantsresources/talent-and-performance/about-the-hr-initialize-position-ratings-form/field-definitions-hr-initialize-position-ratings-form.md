# Field Definitions: HR Initialize Position Ratings Form

The following is a list of field descriptions for the HR
 Initialize Position Ratings form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Position Code

 Specify the position code (defined in HR Position Codes) to use for initializing performance rating codes. Then click the Initialize Ratings button to begin initialization. All rating codes set up for this position code will be assigned to the specified resource/review date (in HR Resource Review).
