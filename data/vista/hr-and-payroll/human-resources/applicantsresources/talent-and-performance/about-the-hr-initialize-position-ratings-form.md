#  About the HR Initialize Position Ratings Form

This form is accessed by selecting the Initialize Performance
 Ratings from Position option in the File menu of HR Resource Review, and is used to
 initialize performance rating codes for a resource/review.
 Rating codes (set up in HR Codes) identify items
 that are rated during a performance review or evaluation. These codes can then be
 assigned to position codes in HR Position Codes. Initializing from a position code
 here allows you to quickly assign all applicable performance rating codes to the
 specified resource/review date.
Note: You can initialize rating codes from more than one position code. Initialization will add only those ratings codes that do not already exist for the resource/review date.
