# Field Definitions: HR Resource Skills Form

The following is a list of field descriptions for the HR
 Resource Skills form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Enter the resource (defined in HR
 Resources) for whom you are defining skills.

##  Skill Code

 Enter the skill code (defined in HR Codes,
 Type S) that identifies this resource’s skill.

##  Certification Date

 Enter the date this resource was certified for this skill.

##  Expire Date

 If the certification for this skill is one that needs to be renewed periodically, enter the date the current certification expires.

##  Skills Tester

 Indicate the person responsible for testing this resource to verify that he/she has this skill.
