# Field Definitions: HR Resource Employment History Form

The following is a list of field descriptions for the HR
 Resource Employment History form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Specify the resource (from HR Resources) for
 which you are entering/reviewing employment history.
Note: If this form was accessed via the HR Resources, this
 field defaults the resource specified in HR Resources and is disabled.

##  Seq #

 Enter N, New, or + to add a new employment history event. The system will automatically assign the next available sequential number once you move off the field.

##  Date

 Specify the date this employment history event occurred.

## Type

 Specify the type of code for this employment history record.
 The type specified here will help differentiate records having the same code, but different code types.

## Employment History Code

 Enter the code that identifies this
 employment history event, or press F4 for a list of codes.
Depending on the selection you made in the
 [Type](/en/vista/vista/hr-and-payroll/human-resources/applicantsresources/talent-and-performance/about-the-hr-resource-employment-history-form/field-definitions-hr-resource-employment-history-form#ID-00011076--en)
 field, you can specify codes from [ HR
 Codes](/en/vista/vista/hr-and-payroll/human-resources/setupmaintenance/about-the-hr-codes-form) (Types H or N) or [ HR Position Codes](/en/vista/vista/hr-and-payroll/human-resources/setupmaintenance/about-the-hr-position-codes-form).

##  Supervisor

 Specify who the supervisor was for this employee at the time this event occurred, up to 30 characters.
