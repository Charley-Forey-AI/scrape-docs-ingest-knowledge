# Field Definitions: HR Resource Rewards Form

The following is a list of field descriptions for the HR
 Resource Rewards form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Disabled if
 this form was accessed from HR Resources.
 Enter the resource (from HR Resources) for
 which you are entering reward information.

##  Seq #

 Enter a sequence number (1-9999) for this reward entry or enter N, New, or + to have the system auto-assign the next available sequential number.

##  Date

 Enter the date on which the employee received this reward.

##  Reward Code

 Enter the code (defined in HR Codes, Type W)
 that identifies the type of reward received by this employee.

##  Description

 Enter a description of this reward, up to 30 characters.

##  Reward Amount

 Enter the amount of this reward. Cannot be left blank, so if no dollar value is applicable, enter 0.00.

##  Reason

 Enter the reason this employee received this reward, up to 30 characters.
