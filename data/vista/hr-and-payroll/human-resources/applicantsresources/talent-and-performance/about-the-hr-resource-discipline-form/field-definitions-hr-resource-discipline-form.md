# Field Definitions: HR Resource Discipline Form

The following is a list of field descriptions for the HR
 Resource Discipline form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Disabled if this form was accessed
 from HR Resources.
 Specify the resource (HR Resources) for which
 you are entering this discipline incident.

##  Seq #

 Enter the sequence number (1-9999) for this disciplinary incident entry or enter N, New, or + to have the system auto-assign the next available sequential number.

##  Date

 Enter the date on which the incident occurred.

##  Discipline Code

 Enter the code (defined in HR Codes, Type N)
 that identifies the reason for this disciplinary action.

## Incident Description

Use this section to describe the incident. The space allowance in this box is virtually unlimited.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.

## Disciplinary Action

Use this section to describe any disciplinary action taken as a result of this incident.  The space allowance in this box is virtually unlimited.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.

##  Days Suspended

 If the employee has been suspended from work, indicate how many days (0-255) of suspension were ordered.

##  Warning Notice?

 Check this box if a warning notice was issued to the employee.
 Leave this box unchecked if no warning notice was issued to the employee.

##  Follow-up Date

 Enter the date on which a follow-up review of this incident will take place.
