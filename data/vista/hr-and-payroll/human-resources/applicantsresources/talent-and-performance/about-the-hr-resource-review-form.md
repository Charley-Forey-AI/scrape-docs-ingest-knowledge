#  About the HR Resource Review Form

Use the HR Resource Review form to enter and maintain
 information about employee reviews.
 This form may be accessed from the main menu or
 from the Reviews tab in HR Resources (by double-clicking the desired record or clicking
 anywhere in the line and selecting the Open Related Record in Form option from the
 Records menu).
 The Info tab is used to enter information about the employee’s review; that is, the date of the review, who performed the review, the position held by the employee, and when the next review is due. Space is also provided for entering notes about the review.
 The Performance Ratings tab is used to rate all of
 the items that are being evaluated during the review (e.g. attendance, attitude, job
 performance, quality/quantity of work, etc.). Rating codes (defined in HR Codes) are
 used to identify the items being evaluated. Performance rating Items can be entered
 manually or by initialization.

## Initialize Performance Ratings

 If you elect to initialize performance
 rating items, there are two options available (from the File menu) for your use:

- Initialize Performance Ratings
 from Group – This option allows you to quickly set up all of the items that
 will be rated during this employee’s performance evaluations based on a
 Rating Group (set up in HR Ratings Group). All performance rating items
 defined for the rating group will be added for the selected resource/review
 date. See Related Topics below for information about HR Initialize Rating
 Groups.

- Initialize Performance Ratings
 from Position – This option allows you to quickly set up all of the items
 that will be rated during this employee’s performance evaluations based on a
 position code (set up in HR Position Codes). All performance rating items
 defined for the position code will be added for the selected resource/review
 date. See Related Topics below for information about HR Initialize Position
 Ratings.

## Salary History

 This option (available from the File
 menu) opens the HR Resource Salary History form, allowing you to review the
 resource’s salary history. See Related Topics below for more information about HR
 Resource Salary History.
Note: You can track changes to review information by
 checking the Reviews option in the Employment History Event Options section of HR
 Company Parameters.
