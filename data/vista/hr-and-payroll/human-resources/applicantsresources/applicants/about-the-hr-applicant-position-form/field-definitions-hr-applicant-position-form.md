# Field Definitions: HR Applicant Position Form

The following is a list of field descriptions for the HR
 Applicant Position form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Disabled if this form was accessed from HR
 Resources.
 Enter the resource (as defined in HR
 Resources) applying for this position.

##  Position Code

 Enter the code (as defined in HR Position Codes) that identifies the position for which this applicant is applying. Use F4 to view all positions that the applicant has applied for.

##  Full/Part Time?

 Indicate whether this applicant is applying for a full-time or part-time position.
 F = Full-time
 P = Part=time

##  Part Time Hours

 If this applicant is applying for a part-time position, specify the number of hours per week this applicant is willing to work.
