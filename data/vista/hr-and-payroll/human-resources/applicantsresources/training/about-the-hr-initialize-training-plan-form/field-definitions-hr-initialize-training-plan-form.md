# Field Definitions: HR Initialize Training Plan Form

##  Resource

 Specify the resource (HR Resources)
 for which you are initializing a training plan.

##  Position Code

 Enter the position code from which to initialize training. Click the Initialize Plan button to initialize all training codes set up for the position code (in HR Position Codes) to the specified resource.
