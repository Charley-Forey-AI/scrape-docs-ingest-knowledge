# Field Definitions: HR Training Class Registration Form

The following is a list of field descriptions for the HR
 Trainig Class Registration form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  PR Company

 Required if filtering by PR Group or Crew.
 Specify the PR company by which to filter resources for registration.
 Click the Get Resources button to populate the Unassigned selection box with all resources meeting the filter criteria. Then use the Add or Add All buttons to add resources to the Assigned selection box. (Note: If necessary, you can use the Remove or Remove All buttons to move resources back to the Unassigned selection box.)
 Once you have finished adding resources to the Assigned selection box, click the Create Registration button to initialize the resources to the selected class.
 Click the Clear button to clear all resources from both selection boxes.
