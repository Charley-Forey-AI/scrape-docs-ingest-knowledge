# Field Definitions: HR Resource Training Form

The following is a list of field descriptions for the HR
 Resource Training form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Enter the
 resource (as defined in HR Resources) for whom you are setting up this training/education
 information.

##  Seq #

 Enter the sequence number (1-9999) for this training entry or enter N, New, or + to have the system auto-assign the next available sequential number.

##  Training Code

 Enter a valid training code (from HR Codes,
 Type T) to identify the training that this resource is being signed up for, is in the
 process of completing, or has completed.

##  Class Seq

 Enabled for new resource/sequence # records only.
 Enter the sequence of the class (defined in HR Training Classes) for which you are setting up this resource. For example, if there are multiple sequences representing the different time slots in which the class is being held, enter the sequence for the appropriate time slot. Press F4 for a list of valid class sequences.
 Once you enter a valid sequence, the form is populated with information set up in HR Training Classes.

## Description

Enter a description of the training, up to 30
 characters. Initially defaults the description from the training code (Description
 field, HR Codes) or will default the training code description (Class Desc
 field, HR Training Classes) if you enter a value in the Class Seq
 field. You can override the default.

##  Institution

 Specify the institution at which this class is being (or was) held. Up to 30 characters allowed. If you specified a class sequence above, defaults the institution specified for this class in HR Training Classes.

##  Class

 Specify the name of the training class, up to 30 characters. If you specified a class sequence number above, defaults the description assigned to the class in HR Training Classes.

##  Date

 Specify the date that this class begins. If you specified a class sequence above, defaults the start date specified for this class in HR Training Classes.

##  Status

 Identifies the status of this training for this individual. If you specified a class sequence above, defaults the status specified for this class in HR Training Classes.
 U – Unscheduled
 S – Scheduled
 I – In Progress
 C – Completed
 A – Absent
 X – Cancelled

##  Date Complete

 Enabled only when Status is 'C-Completed'.
 Specify the date on which this resource completed this class. This field will be populated automatically when updating training (via HR Training Class Status) where the status is set to Complete.

##  Grade

 Indicate the grade this resource received for this class, up to 10 characters.

##  CEU Credits

 Indicate the number of CEU (Certified Education Units) credits this resource received (or will receive) for completing this class. Up to 5 digits allowed (including decimals). If you specified a class sequence number above, defaults the CEU credits specified for this class in HR Training Classes.

##  Hours

 Indicate the number of credit hours this resource received (or will receive) for this course. May also be used to indicate the number of hours the class takes to complete. Up to 10 digits allowed. If you specified a class sequence number above, defaults the hours specified for the class in HR Training Classes.

##  Degree Program

 Check this box if this course/class is part of a degree program.
 Leave this box unchecked if this class is not part of a degree program.

##  Degree Description

 If this class is part of a degree program, enter a description of the degree, up to 30 characters.

##  Training Cost

The Training Cost field on the HR Training Classes form.
 Enter the cost of this training class, if applicable. Otherwise, enter 0.00.

##  Employee Reimbursed

 Check this box if this employee will be reimbursed for the cost of this class.
 Leave this box unchecked if this employee will not be reimbursed for the cost of this class.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.

##  1099

 Check this box if the cost of this training class is subject to 1099 reporting (i.e. the class instructor will be issued a 1099). If checked, specify the associated vendor to the right.
 Leave this box unchecked if the cost of this training class is not subject to 1099 reporting.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.

##  Vendor

 Enabled only if you checked the 1099 option to the left.
 Specify the vendor to which a 1099 will be issued.
 If you specified a class sequence number above, defaults the vendor designated for this class in HR Training Classes (if applicable).

##  OSHA

 Check this box if this training is OSHA (Occupational Safety and Heath Administration) related.
 Leave this box unchecked if this training is not OSHA related.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.

##  MSHA

 Check this box if this training is MSHA (Mine Safety and Heath Administration) related.
 Leave this box unchecked if this training is not MSHA related.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.

##  First Aid

 Check this box if this training is first aid related.
 Leave this box unchecked if this training is not first aid related.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.

##  CPR

 Check this box if this training is CPR related.
 Leave this box unchecked if this training is not CPR related.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.

## Safety Related

The Safety Related checkbox on the HR Resource Training form.
Select this checkbox if this training is safety related. Otherwise, leave unselected.

##  Work Related

 Check this box if this training is work related.
 Leave this box unchecked if this training is not work related.
 If you specified a class sequence number above, defaults as set for the class in HR Training Classes.
