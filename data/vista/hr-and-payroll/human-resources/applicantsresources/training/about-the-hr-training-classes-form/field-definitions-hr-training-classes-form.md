# Field Definitions: HR Training Classes Form

The following is a list of field descriptions for the HR
 Training Classes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Training Code

 Enter the training code that identifies this
 training class. Must be a valid training code (type T) set up in HR Codes.

##  Class Seq

 Enter the sequence (1-9999) of this class. Typically, each sequence of a class will represent a different time slot or date range. For example, if the class will be held at two different time slots (e.g. 8-12 a.m. and 5-9 p.m.), you might set up the 8-12 a.m. time slot as 'Seq 1' and the 5-9 p.m. time slot as 'Seq 2'.

##  Description

 Enter a description of the class, up to 30 characters.

##  Instructor

 Enter the name of the instructor, up to 50 characters.

##  Start Date

 Enter the date on which this class begins.

##  End Date

 Specify the ending date by which to filter the classes shown in the grid. Classes with a end date equal to or prior to this date will be displayed.

##  Class Time

 Enter the start time for this class, in 24-hour format.

##  Time Desc

 Enter the time description for this class. For example:Mon–Fri, Wed & Fri, etc. You can also include the time range if desired (e.g. Mon-Fri, 8-10 a.m.). Up to 30 characters allowed.

##  Status

 Specify the current status of this training class. The status will be updated automatically when updating from HR Training Class Status.
 U – Unscheduled
 S – Scheduled
 I – In Progress
 C – Completed
 A – Absent
 X – Cancelled

##  Max Attend

 Specify the maximum number of attendees allowed for this class/sequence.

##  Hours

 Indicate the number of credit hours employees will receive upon completion of this course. You can also use this field to indicate the total number of hours this class takes to complete. Up to 10 digits allowed.

##  CEU Credits

 Indicate how many CEU units (Certified Education Units) employees will receive for completing this class. Up to 5 digits allowed (including decimal).

##  1099

 Check this box if the cost of this class is subject to 1099 reporting. If checked, you will specify the associated vendor (instructor) to the right.
 Leave this box unchecked if the cost of this class is not subject to 1099 reporting.

##  Vendor

 If the cost of this class is subject to 1099 reporting, specify the vendor (AP Vendors) to
 which this applies.

##  OSHA

 Check this box if this class is OSHA (Occupational Safety and Heath Administration) related.
 Leave this box unchecked if this class is not OSHA related.

##  MSHA

 Check this box if this class is MSHA (Mine Safety and Heath Administration) related.
 Leave this box unchecked if this class is not MSHA related.

##  First Aid

 Check this box if this class is first aid related.
 Leave this box unchecked if this class is not first aid related.

##  CPR

 Check this box if this class is CPR related.
 Leave this box unchecked if this class is not CPR related.

##  Work Related

 Check this box if this class is work related.
 Leave this box unchecked if this class is not work related.

##  Training Costs

 Indicate what the cost is for this training class.

##  Employee Reimbursed

 Check this box if employees will be reimbursed for the cost of this class.
 Leave this box unchecked if employees will not be reimbursed for the cost of this class.

##  Institution

 Enter the institution where this class/course is to be held. Up to 30 characters allowed.

##  Address

 Enter the address of the institution, up to 60 characters.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup..

##  City

 Enter the city in which the institution is located. Up to 30 characters allowed.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

## State

Enter a valid state (as defined in HQ States). The system validates the state based on the Default Country specified in HQ Company Setup for the active company. If not valid, an error displays, but entry is allowed. You must then enter a valid country for this state in the Country field.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Zip

 Enter the zip code, up to 12 characters.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Country

 Enter the 2-character country code. Entry in this field is required when the address exists outside the Default Country specified in HQ Company Parameters for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Phone

 Enter the 10-digit phone number of the institution, including dashes.

##  Room

 Enter the number of the room (e.g. room, suite, etc.) in which the class is to be held.

##  Contact

 Enter the name of the person to contact in regards to this class. Up to 50 characters allowed.

##  Skill Code

 Enter any skills associated with this class
 (i.e., skills that an employee will obtain by taking the class). For example, if taking a
 Red Cross class, the employee might obtain CPR and/or First Aid skills. Must be a valid
 skill code (type ‘S’) set up in HR Codes.
Note: Once the class has been completed and updated (via HR Training Class Status), all skills defined here will be updated for each resource in HR Resource Skills.

##  Show

 Specify how to restrict the display of classes in the grid.

-  Open Classes – Select this option to restrict display to only those classes that are open (i.e. not flagged as Complete).

-  All Classes – Select this option to show all classes, regardless of their status.

##  Begin Date

 Specify the beginning date by which to filter the classes shown in the grid. Classes with a start date equal to or greater than this date will be displayed.

##  End Date

 Enter the date on which this class ends.

##  Resource

 Specify the resource to assign to this class. Defaults resources registered via HR Training Class Registration or HR Resource Training.

##  Status

 Specify the status of this training for this resource. If resource was added manually or by initialization (in HR Training Class Registration), status will default as ‘S-Scheduled’.
 U–Unscheduled
 S–Scheduled
 I–In Progress
 C–Completed
 A–Absent
 X–Cancelled

##  Date Complete

 Enabled only when status is ‘C-Completed’.
 Specify the date this resource completed this class. If you updated the status to ‘Complete’ using HR Training Class Status, this date will default the date specified during the status update.
