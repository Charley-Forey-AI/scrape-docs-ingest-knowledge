# Field Definitions: HR Training Class Status Form

The following is a list of field descriptions for the HR
 Training Class Status form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Tester

 Enabled only when status is ‘C-Completed’.
 Defaults the instructor specified for the class in HR Training Classes. May be overridden. If changed, will update the Skills Tester designated in HR Resource Skills for each skill code associated with the training code/sequence. No update to the instructor specified in HR Training Classes will occur.

##  Date Complete

 Enabled only when status is ‘C-Completed’.
 Defaults the End Date specified for the class in HR Training Classes. May be overridden. Changes will be updated for the resource in HR Training Classes and HR Resource Training.

##  Status

 Specify the current status of the training class. This status will be updated for the class in HR Training Classes and for each resource registered for the class in HR Training Classes and HR Resource Training.
 U – Unscheduled
 S – Scheduled
 I – In Progress
 C – Completed
 A – Absent
 X – Cancelled
