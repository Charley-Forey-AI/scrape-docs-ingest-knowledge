# About the HR Initialize PR Employees Form

Use this form to initialize employees set up in PR Employees
 into HR Resources, or resources set up in HR Resources into PR Employees.
You will only need to use this program during
 initial setup of employees/resources, when adding new employees or resources, or when
 adding the HR module after Payroll has already been implemented. You should not use this
 program to update changes to an employee's static information or benefits/salary
 information. Updates to an employee's static information should be handled through HR
 Resources, and updates to an employee's benefit or salary information should be handled
 using the HR Update Benefit/Salary to PR program.
