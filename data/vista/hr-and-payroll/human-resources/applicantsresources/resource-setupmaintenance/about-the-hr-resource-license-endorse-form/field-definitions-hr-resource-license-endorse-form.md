# Field Definitions: HR Resource License Endorse Form

The following is a list of field descriptions for the HR
 Resource License Endorse form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Lic Code Type

 Specify the license code type for this employee.
 R = Restriction
 E = Endorsement
 C = Certificate

##  Lic Code

 Specify the license code (from HR License Endorsements) that applies to this employee/state. A description of the license restriction, endorsement, or certificate displays to the right.
