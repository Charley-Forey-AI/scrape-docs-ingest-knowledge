# Field Definitions: HR Resource Schedule Form

The following is a list of field descriptions for the HR
 Resource Schedule form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource

 Enter the resource for this schedule or leave request. Press F4 for a list of resources.

##  Date

 Enter the date that applies to this schedule item/event.

##  Seq

 Enter the sequence number you want to view.
 If entering a new schedule item/event, enter “N”, “New”, or “+”.

##  Schedule Code

 Enter the code that identifies the schedule item/event for this resource. Press F4 for a list of valid schedule codes.

##  Description

 Enter a description of this schedule item/event, up to 30 characters. For example, you might use this field to enter the hours of a shift or the location of an onsite.

##  Schedule Hours

 Enter the number of hours that applies to this schedule item/event.

##  Requester Comments

 For schedule codes assigned to leave types, this field displays the leave requester comments from HR Leave Request. You can make changes to this information as necessary.

##  Approver Comments

 For schedule codes assigned to leave types, this field displays any applicable leave approver comments from HR Leave Approval. You can make changes to this information or add additional comments.

##  Request Status

 For schedule codes associated with a leave type, select a status for the leave request The following statuses are available:

- New

- Approved

- Denied

- Cancelled
