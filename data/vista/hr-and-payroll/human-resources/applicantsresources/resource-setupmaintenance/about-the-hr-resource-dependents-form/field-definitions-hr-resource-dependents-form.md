# Field Definitions: HR Resource Dependents Form

The following is a list of field descriptions for the HR
 Resource Dependents form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Disabled if this form was accessed
 from HR Resources.
 Enter the resource (defined in HR Resources)
 for whom you are setting up dependent information.

##  Dependent Seq #

 Enter the sequence number (1-9999) for this dependent or enter N, New, or + to have the system auto-assign the next available sequential number.

## Last

The Last name field in the HR Resource Dependents form.
 Enter this dependent’s last name, up to 30 characters.

## First

The First name field in the HR Resource Dependents form.
Enter this dependent’s first name, up to 30 characters.

## Middle

The Middle name field in the HR Resource Dependents form.
Enter this dependent’s middle name or inital, up to 15 characters.

## Suffix

The Name Suffix field in the HR Resource Dependents form.
Enter this dependent’s suffix,(e.g. Sr., Jr., III, etc.), up to 4 characters.

##  Relationship

 Indicate what this dependent’s relationship is to the resource (e.g. spouse, son, daughter, parent, etc.), up to 30 characters.

##  Birth Date

 Enter the dependent’s date of birth.

## Social Security #

 Enter the dependent’s Social Security Number. (Note: If dashes are not entered, they will be added automatically.)

## Gender Identity

The Gender Identity drop-down on the HR Resource Dependents form, Info tab.

Select this dependent's gender identity.

- M-Male

- F-Female

- X-Non-Binary

- O-Other (enables the text box to the right for entering the gender description)

## Description

The Description field (unlabeled) on the HR Resource Dependents form, Info tab.
This field is enabled only when the Gender Identity drop-down is set to O-Other.
Enter a description of the dependent's gender (for example, Agender, Androgyne, Bigender, etc.).

##  Reporting Gender

The Reporting Gender field on the HR Resource Dependents form, Info tab.
 Indicate whether this dependent is male or female.

##  Address

 Enter the resident address of this dependent, up to 60 characters.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  City

 Enter the city of residence for this dependent, up to 30 characters.
Note: If you have Internet access, you can click the Map
 button for direct access to the default map site for your login (as defined in User
 Options, Main Menu). Map will default the approximate location of the specified country and
 address. If a country is not specified, attempts to locate address based on Default Country
 specified in HQ Company Setup.

## State

 Enter this dependent’s resident state (as defined in HQ States). The system validates the state based on the Default Country specified in HQ Company Parameters for the active company. If not valid, an error displays, but entry is allowed. You must then enter a valid country for this state in the Country field.
Note: If you have Internet access, you can click the Map
 button for direct access to the default map site for your login (as defined in User
 Options, Main Menu). Map will default the approximate location of the specified country and
 address. If a country is not specified, attempts to locate address based on Default Country
 specified in HQ Company Setup.

## Zip Code

 Enter the dependent’s zip code, up to 12 digits.
Note: If you have Internet access, you can click the Map button for direct access to the default map site for your login (as defined in User Options, Main Menu). Map will default the approximate location of the specified country and address. If a country is not specified, attempts to locate address based on Default Country specified in HQ Company Setup.

##  Country

 Enter the 2-character country code. Entry in this field is required when the address exists outside the Default Country specified in HQ Company Parameters for the active company. Country must be valid for the specified state (e.g. state, province, territory, etc.) as defined in HQ States.
Note: If you have Internet access, you can click the Map
 button for direct access to the default map site for your login (as defined in User
 Options, Main Menu). Map will default the approximate location of the specified country and
 address. If a country is not specified, attempts to locate address based on Default Country
 specified in HQ Company Setup.

##  Phone

 Enter the home phone number of this dependent, up to 20 characters.

##  Work Phone

 Enter the work phone number for this dependent, if applicable, up to 20 characters.
