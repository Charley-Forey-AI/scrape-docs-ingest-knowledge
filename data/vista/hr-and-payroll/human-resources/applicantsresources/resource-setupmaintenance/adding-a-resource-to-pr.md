# Adding a Resource to PR

After the initial setup of resources in HR Resources (whether
 manually or via initialization from PR), any new resources that you add can be initialized
 to PR on the fly, without having to use the HR Initialize PR Employees program.

To use this feature:

1. In HR Resources, specify
 a PR employee number and check the Active PR Employee option.

1. Assign a PR Group, Department, Ins Code and State, UnempState, and Earn Code.

1. Save the record.

1. Click Add to
 PR button. Once the employee is added to
 PR, the system will check the Exists in PR display-only
 box. The system also checks this box when you add the employee to HR from
 PR.
