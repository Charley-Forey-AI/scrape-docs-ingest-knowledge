# Field Definitions: HR Resource Drug Testing Form

The following is a list of field descriptions for the HR
 Resource Drug Testing form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Resource #

 Specify the
 resource (from HR Resources) for which you are entering drug testing information.

##  Date

 Enter the date on which the drug test was performed. (Note: Pressing F4 will display a list of dates on which the specified employee has undergone drug testing.)

##  JCCo

 Enter the Job Cost company the employee was working for at the time of the testing.

##  Job

 Enter
 the job (from JC Jobs) on which the employee was working at the time of the testing.

##  Test Type

 Enter the code that identifies the type of
 drug test or the reason for performing the drug test. Must be a valid drug test code set up
 in HR Codes (Type D).

##  Location

 Enter the location at which the drug test was performed, up to 30 characters.

##  Tester

 Enter the name of the person who performed the drug test, up to 30 characters.

##  Status

 Enter the status code for this drug test.
 Press F4 to see a list of valid status codes (set up in HR Codes). This field defaults with
 the code you specified in the Default Status Code field in HR Company Parameters; if that
 field is blank, this field defaults blank.

## Results

Use this section to enter the results of the drug testing.The space allowance in this box is virtually unlimited.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.
