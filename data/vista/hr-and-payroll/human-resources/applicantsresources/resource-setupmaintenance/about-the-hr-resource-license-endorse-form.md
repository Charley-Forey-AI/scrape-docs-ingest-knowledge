#  About the HR Resource License Endorse Form

This form can only be accessed by clicking the Endorsements
 button in the Driving Info section of HR Resources (Other Info tab), and is used to set up
 and maintain license endorsements by employee.

You will typically only need to use this form for employees that require/have a CDL (Commercial Driver's License).
 Before you can set up endorsements/restrictions for an employee, you must have already set up license endorsements and restrictions by state in HR License Endorsements and you must have designated a license state for the employee (Driving Info section). Employee’s license state will display above the gridand will determine the endorsements/restrictions available for assignment.
