#  HR Hospital Information Form

Use this form to set up hospital information (i.e. hospital name, address, phone number, and email address).
The information set up in this form is used when entering accident information in HR Resource Accident.
If you have Internet access, you can use the Map button to quickly access the default map site for your login (as defined in User Options, Main Menu) and locate the specified country and address.  The Email button will automatically set up an outgoing email message (in Microsoft Outlook) to the email address specified for the hospital.
