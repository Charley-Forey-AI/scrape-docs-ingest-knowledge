# About the PR Arrears/Payback History Form

Use the PR Arrears/Payback History form to view arrears and
 payback history for employees who are active for arrears calculations. The system updates the information on this form
 during payroll processing.
Access this form from the main menu or by
 double-clicking a record on the Arrears/Payback History tab on the PR Employee
 Dedns/Liabs form.
As needed you can manually enter new records in this
 form, or alter or delete manually entered records.
If you wish to remove records that were added through payroll processing and posting, you must delete them using payroll processing.
