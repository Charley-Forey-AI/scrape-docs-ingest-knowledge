# Set Automatic Overtime to Use the Timecard Rate

Set automatic overtime to use the timecard rate.

1. In PR Company
 Parameters, click on the Info tab.

1. Select the Using Automatic
 Overtime checkbox.

1. In the Overtime
 Earnings Code field, enter the earnings code to use for
 overtime.

1. Save the record.

The system calculates overtime rates based on the
 timecard's posted rate multiplied by the factor for the specified overtime earnings
 code.
