# About the PR Salary Distribution Form

Use this form to distribute an employee’s salaried amount to multiple jobs, equipment, or accounts by calculating an hourly wage based on the hours entered.
The system only uses hours keyed into PR Timecard Entry to calculate this hourly wage; automatic earnings do not apply.
When using this form you can restrict distribution by employee, payment sequence, or both. All employee records for the pay period that match the criteria are processed. For more information on using this form, see [Distributing a Salary](/en/vista/vista/hr-and-payroll/payroll/payroll-processing/payroll/distribute-a-salary).
