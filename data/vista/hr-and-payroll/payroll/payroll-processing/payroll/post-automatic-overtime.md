# Post Automatic Overtime

You can use PR Auto Overtime to calculate and post overtime
 for timecards that you entered in PR Timecard Entry.
The following instructions detail how to post automatic overtime.

1. Double-click the PR
 Auto Overtime icon in the PR Programs folder on
 the Main Menu. The Batch Selection form
 displays.

1. Select the correct
 unposted batch from the grid, and make sure the
 values in the Batch
 Month, PR
 Group, PR End
 Date, and Pay
 Seq fields match what you indicated
 when you entered timecards in PR Timecard
 Entry.

1. Click OK.  The PR Auto Overtime form
 displays.

1. If you want to
 restrict automatic overtime posting by a specific
 employee, check the Restrict by
 Employee box and enter the employee
 number in the Employee field. Press F4
 from the Employee field to see a list of valid
 employees.

1. If you want to
 restrict automatic overtime posting to a single
 payment sequence, check the Restrict
 Posting to a Single Payment Sequence #
 box and enter the sequence in the Payment Seq
 # field.Note: If you do
 not choose to restrict automatic overtime posting
 by specifying an employee or a payment sequence,
 the system will bring in all employees and
 sequences for the pay period.

1. Click Process.  The system displays a
 confirmation message.

1. Click Close. The employees matching the criteria display in the grid.

1. Select File > Process
 Batch.  The PR Batch Process form
 displays.

1. [Process the batch](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/maintenance/pr-batch-process-form). The system then applies the appropriate rates based
 on the overtime schedule that you assigned to the employee (OT
 Schedule field, PR Employees), job (PR Overtime Schedule
 field in JC Jobs), craft (OT Schedule field, PR
 Crafts), or craft template (OT Schedule field in PR
 Craft Templates). For more information on overtime schedules, see [PR Overtime Schedule](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-overtime-schedule-form).
Tip: To see how the system posted automatic overtime, view the
 Batch List report from the PR Batch Process form after you validate the
 batch.
You can now [distribute salaries](/en/vista/vista/hr-and-payroll/payroll/payroll-processing/payroll/distribute-a-salary) or [post automatic earnings](/en/vista/vista/hr-and-payroll/payroll/payroll-processing/payroll/about-posting-automatic-earnings/post-automatic-earnings).
