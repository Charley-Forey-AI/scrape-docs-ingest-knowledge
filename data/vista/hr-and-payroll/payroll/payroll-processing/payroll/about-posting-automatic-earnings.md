# About Posting Automatic Earnings

When PR Post Auto Earnings is run, a transaction line is created for each appropriate entry based on the frequency code specified.
You may edit and/or delete any of these entries in PR Timecard Entry before printing checks.
When you are finished entering the automatic earnings, print a list of them using the PR Employee Auto Earnings report (Report menu).
