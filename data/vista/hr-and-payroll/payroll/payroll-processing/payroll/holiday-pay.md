# Holiday Pay

Automatic overtime uses the holidays defined in PR Pay Period
 Control (Holidays tab) to determine holiday pay, based on the overtime schedules assigned to
 employees and jobs.
However, if you have holidays that are specific to a
 craft, you must process auto overtime by craft. That is, timecards must be posted to
 a craft, the employee must be assigned an overtime type of "Craft" (in PR Employees,
 Info tab), and the craft must be assigned an overtime schedule (in PR Crafts).
If an employee is posted to multiple crafts within the week, each day's craft entry uses its own overtime schedule to determine holiday pay. If an employee is posted to multiple crafts in a single day, and they use the same overtime schedule, the first craft posted determines the holiday for that day. If you want each craft to use its own rules, you must set up separate OT schedules.
Craft holidays are in addition to the standard holidays specified in Pay Period Control.
