# Local Codes

Similar to state tax and unemployment, the local codes (set up
 in PR Local Codes) may be based on the employees resident local or the local area in which
 they are working.
The default flags are located in the PR Employees
 and PR Company Parameters forms.
In PR Employees
Always Use Employee’s
 Resident Local Code — If this flag is checked, then the employee’s
 standard local code is always used. If unchecked, then the Use Job, SM Work
 Order or Office Local for Local Tax flag in the PR Company
 Parameters form controls the default.
In PR Company Parameters
Use Job, SM Work Order or
 Office Local for Local Tax — If this flag is checked, and the
 Always
 Use Employee’s Local Code option (PR Employees) is unchecked, then
 the Local Code will default based on the job. If no job is entered, then the PR
 Company’s office local will be used. If no office local has been set up, then the
 employee’s standard local code will be used. If unchecked, then the default will
 always be the Employee’s standard Local Code.Note: You can opt to override the local code in PR Timecard Entry.
