# Field Definitions: PR Auto Overtime Form

The following is a list of field descriptions for the PR Auto
 Overtime form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Restrict by Employee

 Checked – Calculate automatic overtime for a selected employee (designated below).
 Unchecked – Calculate automatic overtime for all employees

##  Employee

 Enabled only when Restrict by Employee option is checked (above).
 Specify the employee for which to automatically calculate and post overtime.

## Restrict Posting to a Single Payment Sequence #

Check this box to restrict the calculation of automatic overtime to a specified payment sequence (designated below). Only timecards posted to the specified payment sequence will be included in overtime calculations.
Uncheck this box to calculate automatic overtime on all payment sequences.

## Payment Seq #

Enabled only when Restrict Posting to a Single Payment Seq# is checked (above).
Specify the payment sequence number by which to restrict automatic overtime calculations. Only timecards posted to this payment sequence will be included in automatic overtime calculations.
