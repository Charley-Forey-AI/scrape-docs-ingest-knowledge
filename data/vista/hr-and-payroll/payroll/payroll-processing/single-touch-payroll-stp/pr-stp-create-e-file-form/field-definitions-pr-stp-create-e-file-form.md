# Field Definitions: PR STP Create E-File Form

The following is a list of field descriptions for the PR STP
 Create E-File form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Validation

Validation text display field on the PR STP Create E-File form
The PR STP Download form appears when you click Generate from the PR STP Generate Submission Data form.
If Vista's Single Touch Payroll (STP) validation process does not encounter any errors in your STP e-file, the Save As dialog box appears.
If the validation process encounters one or more errors, this field displays information about the error(s).Note: Even if errors are encountered, you are permitted to save your e-file.
