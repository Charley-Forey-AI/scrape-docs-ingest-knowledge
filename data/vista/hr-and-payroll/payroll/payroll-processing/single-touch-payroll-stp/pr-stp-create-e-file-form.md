# PR STP Create E-File Form

The PR STP Create E-File form (Australia only) appears during the normal course of processing Single Touch Payroll (STP).

When the system finishes creating the file and passing it through a validation process, it prompts you to save the file. After saving, lodge the submission with the ATO on or before the pay date.
