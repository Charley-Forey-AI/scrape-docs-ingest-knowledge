# Equipment Class Override

This option is accessed from the Options menu of PR Timecard Entry and is available when the Post Equipment Usage option has been selected.
The Equipment Class Override option determines where the EM Co and Equip columns display in the grid for Job and customer SM Work Order timecards. If selected, the EM Co and Equip columns display before the craft classification (Class column). If not selected, these fields display after the Class columm.
This option should only be selected if the type of equipment an employee operates determines the rate of pay. The equipment entered is tied to its category. If a classification is set up in EM Categories, the form uses this class and the employee’s craft. The rate of pay is then pulled from the craft/class as always.
Note: This option does not apply to crew timesheets. The
 craft/class set in PR Crew Timesheets is automatically used.
