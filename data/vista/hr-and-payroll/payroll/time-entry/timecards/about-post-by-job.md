# About Post by Job

This option is available from the Options menu of PR Timecard Entry.
If you select this option, the Job field
 on each Job timecard line defaults to the previous line. Typically, this is used when
 timecards come in as a daily timesheet for each job, with all employees listed.
If this option is not selected, the job defaults to the job currently stored in the employee’s master record (PR Employees). This job is updated with each timecard entry and defaults to whatever job was last entered for the employee. This option is typically used when the timecards come in by employee, listing all the days and jobs the employee worked.
