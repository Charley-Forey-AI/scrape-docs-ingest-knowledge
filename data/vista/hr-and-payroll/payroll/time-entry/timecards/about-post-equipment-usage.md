# About Post Equipment Usage

This option is only available from the Options menu of PR
 Timecard Entry if you have checked the Enter Equipment Usage with Timecards box
 in PR Company Parameters (Equipment tab).
It determines whether the active user can enter equipment usage with Job and customer SM Work Order timecards.
If you select this option, equipment usage fields display in the grid, allowing you to enter equipment usage with each job or customer SM Work Order timecard line.
