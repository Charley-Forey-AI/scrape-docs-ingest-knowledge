# Review and Accept/Correct Timecard Discrepancies

If you are using the Smart Payroll feature and the timecard validation process encounters discrepancies, you must either accept or correct the discrepancies before you can post the timecard batch.
Once you select to process a timecard batch, the system automatically runs the timecard validation process. If discrepancies are found, the PR TimeCard Compliance form displays and lists all violated rules that are flagged as Not Allowed or Allow With Warning. You must then accept or correct discrepancies before you can post your batch.

1. For violated rules highlighted in yellow (those with a severity level of Allowed With Warning):

1. To accept a discrepancy without correcting it, select the Accept checkbox.

1. To correct a discrepancy, adjust the values as appropriate for all fields highlighted in yellow.

1. To delete a discrepancy, choose the record and select Delete.

1. For violated rules highlighted in red (those with a severity level of Not Allowed):

1. To correct the discrepancy, adjust the values as appropriate for all fields highlighted in red.

1. To delete a discrepancy, choose the record and select Delete.

1. Select Submit.The system updates the timecard batch accordingly. If additional discrepancies are found, repeat the process. If no other discrepancies are found, the PR Batch Process form displays.

1. Validate and process the batch.

Once you process the batch, violations that you accepted without correcting are updated to the Compliance tab in PR Pay Period Control. Records that you deleted are removed from the timecard batch the same as if you had deleted the record in PR Timecard Entry. If you still need to post those hours, you'll need to create a new timecard batch and re-enter the line.
