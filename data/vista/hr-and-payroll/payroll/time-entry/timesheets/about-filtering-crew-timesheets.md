# About Filtering Crew Timesheets

You can set PR Crew Timesheet Entry to filter for specific timesheets according to their status.
There are a number of filtering options available from the Records menu. You can use each option separately, or use a combination of options for more restrictive filtering.
The following table details each filtering option.
Filter Option
Description

Show My Timesheets
Selecting this option filters for timesheets created
 by you (indicated by the Created By
 display field).

Show Initial
Selecting this option filters for timesheets without a status (unlocked timesheets).

Show Awaiting Approval
Selecting this option filters for timesheets that are locked and are ready for approval.

Show Ready to Send
Selecting this option filters for timesheets that have been approved.

Show Send in Progress
Selecting this option filters for timesheets that were not successfully sent to a PR batch.

Show Send Complete
Selecting this option filters for timesheets that were successfully sent to a PR batch.
