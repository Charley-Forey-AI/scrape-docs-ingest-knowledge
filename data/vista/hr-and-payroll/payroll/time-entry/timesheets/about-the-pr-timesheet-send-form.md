# About the PR Timesheet Send Form

This form has two primary purposes.
First, you can send crew timesheets to PR, JC, and EM batches. Second, you can send employee created timesheets (in PR My Timesheet), that have been approved by their managers (in PR My Timesheet Approval), to a PR processing batch.
To send crew timesheets, you will use the Crew Timesheets tab. To send timesheets created by employees, you will use the My Timesheets tab. Both tabs function separately, so if you have both crew and employee-created timesheets you will have to process them separately.
Because of the sensitive nature of this form, it is recommended that you limit security access to payroll personnel. See [Applying Form Security Settings](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-form-security-settings) for more information.
