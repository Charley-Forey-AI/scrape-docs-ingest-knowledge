# About the PR My Timesheet Copy Form

Use this form to copy timesheet detail information from one timesheet to another.
You can access this form by selecting File  > Initialize
 Timesheet from PR My Timesheet.
When copying timesheet details, you will specify a source timesheet and the system will copy all details to the new timesheet. Copied details include employee (if you have permissions to enter timesheets for other employees), JC company, job, phase, craft, class, shift, earnings codes, and, optionally, hours worked per day.
For more information on using this form, see [Copying Timesheets](/en/vista/vista/hr-and-payroll/payroll/time-entry/timesheets/copy-timesheets).
