# Field Definitions: PR My Timesheet Copy Form

The following is a list of field descriptions for the PR My
 Timesheet Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Start Date

Enter the start date of the timesheet you are copying data from. Click the calendar button ()to select the date from the calendar.

## Sheet

Enter the sheet number of the timesheet you are copying data from.

## Include Hours

Check this box if you want to include the hours for each employee record in the copy process. In other words, the system will copy the hours from the source timesheet to the new timesheet.
