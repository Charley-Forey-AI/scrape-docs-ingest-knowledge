# Tracking Employee Start and Stop Times

The system allows you to track the start, stop, and break time for your employees.
You can track this time using two different methods:

- You can use the PR Employee Daily Time form to track the daily start and stop time, as well as the break hours, for employees. Use this form if you need to report these details to a local jurisdiction requiring this information. For example, California requires this information from an employer to track qualification for Dual Wage Classifications.

- If you need start/stop/break time to be associated with timecard records, you can customize PR Timecard Entry to display fields for entering this information.

These two methods of tracking start and stop times are independent and do not relate to one another, other than by concept. This topic discusses how to use PR Employee Daily Time to track start/stop/break times. For information on associating start/stop/break times with timecard records, see [Entering Start and Stop Time on Timecards](/en/vista/vista/hr-and-payroll/payroll/time-entry/timecards/enter-start-and-stop-time-on-timecards).
Note: Prior to entering start, stop, and break time you must
 verify that the setting of the Timecard Stop Time must be greater than Start Time
 checkbox in PR Company Parameters is correct. For more information, see [Timecard Stop Time must be greater than Start
 Time](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-company-parameters-form/field-definitions-pr-company-parameters-form#ID-000358ed--en).

1. You can access the PR Employee
 Daily Time from the PR Programs folder in the main menu or by selecting Tasks > Open PR Employee
 Daily Time from the PR My Timesheet, PR Crew Timesheet Entry, or PR Timecard Entry
 forms .

1. Enter the employee's number in the Employee field. Press F4 for a list of employees.

1. Enter the work date in the Date field.

1. Enter N, New, or
 + in the Seq field to
 create a new time record.

1. Enter the starting time (24-hour format) in the Start Time field.

1. Enter the stop time (24-hour
 format) in the Stop Time field. If you checked the Timecard Stop Time must be greater than Start
 Time box in PR Company Parameters, the time you enter here cannot be less
 than the Start
 Time value. If the Timecard Stop Time must be greater than Start
 Time box is not checked, all time after 24 hours will be considered the next
 day.

1. Enter the number of hours that
 the employee took for a break in the Break Hours field.

1. Save the record.
