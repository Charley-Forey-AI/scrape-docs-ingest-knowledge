# Set Display Options for Crew Timesheets

You can modify the display settings in PR Crew Timesheet Entry to show more information on each tab.
 All display settings are accessible from the Options menu item. The following table details each of the display options.
Display Option
Description

Default Phases on Sheet #1 Only
Select this option so that only the first sheet defaults the standard phases as set up in PR Crews. Each subsequent sheet defaults with blank phase fields. This option is recommended when you need to add more than 8 phases for the crew on a particular work day.

Display Totals
Select this option to display the totals worked for the job/phase for all employees at the bottom of the Employee Hours and Equipment Usage tabs.

Craft
Select this option to display a Craft field for each employee on the Employee Hours, Other Job Earnings, and Non-Job Earnings tabs.

Overtime Hours
Select this option to display a OT Hours field for each job/phase on the Employee Hours tab. You can use this field to specify any overtime hours for the specified employee.

Doubletime Hours
Select this option to display a DBL Hours field for each job/phase on the Employee Hours tab. You can use this field to specify any doubletime hours for the specified employee.

Cost Type
Select this option to display a Cost Type field for each job/phase on the Equipment Usage tab. You can use these fields to specify a cost type for the equipment and phase.

Rev Code
Select this option to display a Rev Code field for each job/phase on the Equipment Usage tab. You can use these fields to specify a revenue code for the equipment and phase.
