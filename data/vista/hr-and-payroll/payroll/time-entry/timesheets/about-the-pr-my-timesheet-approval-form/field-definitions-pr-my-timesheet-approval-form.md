# Field Definitions: PR My Timesheet Approval Form

The following is a list of field descriptions for the PR My
 Timesheet Approval form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Reviewer

Enter your reviewer ID. Press F4 to see a list of reviewer IDs that are associated with your system login.

## Start Date

This field defaults the most recent date
 associated with a timesheet that is related to your reviewer group (via settings in JC
 Jobs, PM Projects, or PR Employees) after you enter your reviewer ID in the Reviewer.
To view another start date associated with your reviewer ID, use the navigation arrows in the toolbar, or enter the date in this field. You must enter a date that is associated with a timesheet which is in turn associated with your reviewer group.
Note: If you have been given Enter/Change/Approve for Self
 and Others in VA User Profile (My Timesheet Permissions field), this
 field automatically defaults to the most recent date associated with a timesheet without an
 associated reviewer group.
See [Creating Reviewer Groups for Timesheet Entry](/en/vista/vista/administration/headquarters/reviewer-setup/create-reviewer-groups-for-timesheet-entry) for more information.

## Approved

Check this box if you want to approve this timesheet record. Once you check this box, the record is ready for sending to a PR timecard batch (typically done by payroll personnel using PR Timesheet Send).

## JC Co

This field defaults the JC company entered for
 the timesheet in PR My Timesheet. Accept the default or change the company as necessary.
 Press F4 for a list of JC companies.

## Job

This field defaults the job entered for the timesheet in PR My Timesheet. Accept the default or change the job as necessary. Press F4 for a list of valid jobs.

## Phase

This field defaults the phase entered for the timesheet in PR My Timesheet. Accept the default or change the phase as necessary. Press F4 for a list of phases.

## SM Co

Displays for technician timesheets only (i.e.
 Type
 is S-SM Work Order).
This field defaults the SM company entered for
 the timesheet in PR My Timesheet. Accept the default or change the company as necessary.
 Press F4 for a list of SM companies.

## Work Order

Displays for technician timesheets only (i.e.
 Type
 is S-SM Work Order).
This field defaults the SM work order entered
 for the timesheet in PR My Timesheet. Accept the default or change the work order as
 necessary. Press F4 for a list of SM work orders.

## Scope

Displays for technician timesheets only (i.e.
 Type
 is S-SM Work Order).
This field defaults the work order scope
 entered for the timesheet in PR My Timesheet. Accept the default or change the scope as
 necessary. Press F4 for a list of work order scopes.

- If the scope specified on the timesheet is closed, you will be able to approve the timesheet; however, if you change the scope, you must enter an open scope. Selecting a closed scope will display an error and you will be unable the record.

- If the work order is job-related and the scope is not assigned a phase, a warning displays and you be unable to save the record. You can press F5 from this field to access SM Work Orders and assign a phase to the scope. Once you assign a phase to the scope, you can save the record.

## Pay Type

Displays for technician timesheets only (i.e.
 Type
 is S-SM Work Order).
This field defaults the SM pay type entered
 for the timesheet in PR My Timesheet. Accept the default or change the pay type as
 necessary. Press F4 for a list of SM pay types.
Note: Once the work order/scope specified for this entry is closed, you will be unable to change the pay type.

## SM Cost Type

Displays for technician timesheets only (i.e.
 Type
 is S-SM Work Order).
This field defaults the SM cost type entered
 for the timesheet in PR My Timesheet. Accept the default or change the cost type as
 necessary. Press F4 for a list of SM cost types.
Note: Once the work order/scope specified for this entry is closed, you will be unable to change the SM cost type.

## JC Cost Type

Displays for technician timesheets only (i.e.
 Type
 is S-SM Work Order).
This field defaults the JC cost type entered
 for the timesheet in PR My Timesheet. Accept the default or change the cost type as
 necessary. Press F4 for a list of JC cost types.

## Earnings Code

This field defaults the earnings code entered for the timesheet in PR My Timesheet. Accept the default or change the earnings code as necessary. Press F4 for a list of earnings codes.

## Day

This field defaults the hours entered for the timesheet for this date in PR My Timesheet. Accept the default or change the hours as necessary.

## Craft

This field defaults the craft entered for the
 timesheet in PR My Timesheet. Accept the default or change the craft as necessary. Press
 F4
 for a list of crafts.

## Class

This field defaults the class entered for the
 timesheet in PR My Timesheet. Accept the default or change the class as necessary. Press
 F4
 for a list of classes.

## Shift

This field defaults the shift entered for the timesheet in PR My Timesheet. Accept the default or change the shift as necessary.
