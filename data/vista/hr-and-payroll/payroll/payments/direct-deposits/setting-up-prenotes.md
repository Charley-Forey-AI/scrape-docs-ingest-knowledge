# Setting Up Prenotes

Every employee who participates in direct deposits will have
 to go through a prenote process.
This allows your bank to identify the validity of each
 employee’s account.

1. In the PR Employees
 form, select the
 Prenote
 radio button on the Direct Deposit tab. Note: The system produces a live check for the employee when
 you select the Active radio
 button.

1. Download the file using
 PR EFT Payments. Select the Prenote radio button in the
 Export File Type section (Download tab).

1. If your bank requires a
 test file, set up the fields just as for prenote, except select the Test radio button in the PR EFT Payments form (Download tab).
 When running the test, it is not necessary to set up all employees, as only the
 transfer process and the file format will be tested.
