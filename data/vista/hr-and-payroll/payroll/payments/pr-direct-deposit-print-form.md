# PR Direct Deposit Print Form

Use the PR Direct Deposit Print form to print direct deposit advice stubs for Employee and Payment Sequences that have already been assigned a CM Reference number in the PR EFT Download form.
You may reprint a deposit stub as often as necessary, even if the Pay Period is closed.
This form, like the PR Check Print form, relies on a Crystal report layout to perform the actual printing. Because of the complexity of a deposit stub, a certain amount of preprocessing is anticipated. Information from Timecard Headers, Add-ons, Sequence Totals, Employee Accumulations, and Sequence Control are combined and sorted into a PR Stub Print table. This table temporarily holds the information needed to print both check and deposit stubs.
