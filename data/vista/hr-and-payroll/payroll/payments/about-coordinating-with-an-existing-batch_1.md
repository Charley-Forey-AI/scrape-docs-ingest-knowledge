# About Coordinating with an Existing Batch

If timecard detail has not been entered for any employees in
 the sequence, simply change the desired Employee’s PR Employees form Direct Deposit field to
 Not Used and input payroll as usual.
The Employee will receive a physical check
 for the entire payment amount.
If there is timecard detail already entered
 in the system, the Employee’s payment method must be changed in the PR Employee Pay
 Sequence Control form and individually reprocessed in this form or PR Payroll Process.
 Note that selecting the “Employees with Changed Earnings or Overrides” option in PR
 Payroll Process will not catch and process the change to the payment method.
