# About the PR Employee ROE Generate Form

(Canada only) Use the PR Employee ROE Generate form to generate the ROE XML E-file and/or PR Canada Record of Employment report.
Access this form from the PR Record of Employment form by selecting Generate ROE.
You can use this form to generate an original or amended e-file or ROE report for a single employee or for all employees.

- Original ROE - you must specify the ROE Date for the employee(s). The system generates the report or e-file for only those records with the specified ROE Date.

- Amended ROE - the system uses the value you enter in the Amend Date field to determine the records to include in the report or e-file.
