# Field Definitions: PR Employee ROE Generate Form

The following is a list of field descriptions for the PR
 Employee ROE Generate form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Employee

Employee field on the PR Employee ROE Generate form.
Enter the employee for which to generate an
 ROE report or press F4 to select from a list of employees. Leave this field blank to generate
 an ROE report for all employees meeting the date criterion.

## Report/Filing Option

Report/Filing Option field on the PR Employee ROE Generate form.
Select the applicable print option:

- Original - Select this option to
 generate or print an original ROE report for the selected employee(s) / ROE
 Original Date.

- Amended - Select this option to generate or print an amended ROE report for the selected employee(s) / Amended Date.

## ROE Original Date

ROE Original Date field on the PR Employee ROE Generate form.
This field is enabled when you select the Original report/filing option.
Enter the original ROE date for this filing.
 This is the date that you specified in the ROE Date field on the PR Record of
 Employment form. When you generate the ROE report, the system will only add records to the
 report that have this matching date.

## Amended Date

Amended Date field on the PR Employee ROE Generate form.
This field is enabled when you select the Amended report/filing option.
Enter the amended date for this ROE filing.
 This is the date that you specified in the Amend Date field on the PR Record of
 Employment form. When you generate the ROE report, the system will only add records to the
 report that have this matching date.

## Print Option

Print Option field on the PR Employee ROE Generate form.
Select the applicable print option:

- Preview - Select this option to
 preview the ROE report before printing or generating the E-file.

- Print - Select this option to print the ROE report.

Neither of these options generates a report
 that you can submit to Service Canada directly. Using the data from the report you will
 need to either fill out an official Service Canada ROE form or manually enter the data
 using the ROE web application on the [Service Canada](http://www.servicecanada.gc.ca/eng/home.shtml) website.
