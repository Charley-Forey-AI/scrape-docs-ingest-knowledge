# About the PR Employee ROE Workfile Form

(Canada only) Use the PR Employee ROE Workfile form to create Record of Employment (ROE) records for employees for which you will be submitting an ROE report.
Use this form to [Create Employee ROE Records](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/create-employee-roe-records):

- select employee records

- add specific details to each record

- prompt the system create an ROE record for each employee.

The system use these records to generate the ROE reports (in the PR Employee ROE Generate form).
