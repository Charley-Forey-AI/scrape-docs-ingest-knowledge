# Generate an ROE Paper Report

(Canada only) Generate an ROE Paper report to make it easier to complete an official paper or web form.
Before you can generate your Record of Employment (ROE) reports, you must first

- [Create Employee ROE Records](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/create-employee-roe-records) for each employee for which you will be submitting the report, and

- review the records for accuracy. See [About Reviewing Employee ROE Records](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/about-reviewing-employee-roe-records).

For information on submitting an amended report, see [Generate Amended ROE Reports](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/generate-amended-roe-reports).

Generating an ROE paper report does not generate a report that you can submit to Service Canada directly.
Instead, you can use the data from the report to either fill out an official Service Canada ROE form or manually enter the data using the ROE web application on the Service Canada website.
To generate an ROE paper report:

1. From the PR Record of Employment form, select Generate ROE.The system displays the PR Employee ROE Generate form.

1. In the Employee field, enter the employee for which to generate an ROE paper report or press F4 to select from a list of employees. You can also leave this field blank to generate an ROE report for all employees that meet your date criterion.

1. In the Report/Filing Option section, select the Original radio button.

1. In the ROE Original Date field, enter the ROE date.This is the date that you specified in the ROE Date field on the PR Record of Employment form. When you generate the ROE report, the system will only add records to the report that have this matching date.

1. In the Print Option section, select either the Preview or Print radio button.

1. Select Report.
