# About Reviewing Employee ROE Records

Use the PR Record of Employment for to review employee ROE records for accuracy before you generate ROE reports.
During the process of creating employee records for ROE reporting (see [Creating Employee ROE Records](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/create-employee-roe-records)), the system adds records to the PR Record of Employment form. In doing so, the system retrieves data from the PR Employee ROE Workfile form, as well as other employee and payroll data from the system. Once the employee records are added to the PR Record of Employment form, you should review them for accuracy before you generate ROE reports.
For more information on a particular field, see the F1 help.

## Grid Tab

The majority of the field values on the Grid tab come from employee records in the PR Employees form, and the PR Employee ROE Workfile form, if you used it to create employee ROE records.

## Insurable Earnings Tab

The system defaults the employee's insurable earnings per pay period on this tab during the process for creating employee ROE records (see [Creating Employee ROE Records](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/create-employee-roe-records)). The information on this tab comes from payroll data and is based on earnings codes and liability codes that have been identified as being insurable. For more information, see [Determining ROE Settings for Earnings Codes](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-earnings-codes-form/field-definitions-pr-earnings-codes-form#ID-00037064--en) and [ROE Insurable](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-deductionsliabilities-form/field-definitions-pr-deductions-liabilities-form#ID-0005bf5e--en).
The system displays all pay periods from the last 12 months, plus one additional pay period (to meet Service Canada's guidelines). If you entered the employee record manually on the Grid tab, no information will default on this tab. In this case, enter all pay period records and amounts from the last 12 months plus one additional pay period.
Note: All earnings representing insurable separation payments are allocated exclusively to the employee's final pay period, regardless of the pay period in which the payments were actually made. This conforms with Service Canada requirements for ROE reporting.

## Separation/Special Payments Tab

The information on this tab comes from payroll data and is based on earnings codes that have been set up as separation or special (paid sick leave, maternity, WLI) payments. For more information, see [Determining ROE Settings for Earnings Codes](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-earnings-codes-form/field-definitions-pr-earnings-codes-form#ID-00037064--en).
The system displays payments that occurred within the last 12 months plus one additional pay period. You can also add additional records to this tab, if necessary. For more information on adding record, or editing existing ones, see [Manually Entering ROE Separation/Special Payment Records](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/about-the-pr-record-of-employment-form/field-definitions-pr-record-of-employment-form#ID-0003a7ef--en).
