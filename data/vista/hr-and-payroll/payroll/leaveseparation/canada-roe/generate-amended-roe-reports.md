# Generate Amended ROE Reports

(Canada only) Generate amended Record of Employment (ROE) reports.
When amending a (ROE) report, you can either:

-  Update the information directly in the PR Record of Employment form (which does not update data elsewhere in the system), or

- Delete the record from the form, reprocess the employee's data in the Payroll module or make other needed corrections, and follow the steps to create an ROE report (see [About Record of Employment Reporting](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/canada-roe/about-record-of-employment-reporting)).

In most cases, the second option is preferred because it ensures that your system data matches up with the data that is displayed on the PR Record of Employment form. In either case, there are some specific things that you must do to generate an amended ROE report.
The following discusses how to generate an amended ROE report:

1. In the PR Record of Employment form, locate the amended employee ROE record.

1.  In the ROE Serial Number field, enter the serial number you received from Service Canada for the original ROE. You must enter a value in this field to generate an amended file.

1. In the Amend Date field, enter the amended date. You must enter a date in this field to generate an amended file.

1. Select Generate ROE.The system displays the PR Employee ROE Generate form.

1.  In the Employee field, enter the employee for which you are generating an amended ROE report. Press F4 to see a list of employees. You can also leave this field blank to generate an amended ROE for each employee that meets your date criterion.

1. In the Report/Filing Option section, select the Amended radio button.

1. In the Amended Date field, enter the amended date for which to generate the report. This is the same date that you entered in the Amend Date field in step 3.

1. Select Generate E-file.The system displays the Save Export File As window.Note: If you want to create a paper report, in the Print Option section, select Preview or Print and then select Report.

1. Save the record to a location on your network.

Submit the file using the ROE Web application on the Service Canada website.
