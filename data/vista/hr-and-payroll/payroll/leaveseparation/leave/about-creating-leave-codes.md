# About Creating Leave Codes

You can associate two accrual types with a leave code.
These include:

- Fixed Amounts - Use fixed amount leave codes to accrue leave units based on a specific time-period, or frequency.

- Rate-Based - Use rate-based leave codes to have leave accrue at a rate based on applicable earnings codes.

There are three steps to creating leave codes: you must create the code based on an accrual type, set accrual and usage information, and associate employees with the leave code. Select the following links for detailed information on each step.
