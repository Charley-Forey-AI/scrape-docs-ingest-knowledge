# PR Leave Codes Form

Use this form to create and maintain leave codes.
Leave codes track paid employee absences (such as vacation, holiday, and sick time). When you set up a leave code, you'll associate an accrual type, determine limits, enter usage tracking, and associate employees.
When you add a new leave code, the system automatically selects the Active checkbox. While you can deselect it initially, you must select this checkbox before assigning the leave code to employees in PR Employee Leave (either manually or via initialization).
If you deactivate a leave code after you have already assigned it to employees, the system prompts you to deactivate the leave code for the applicable employees. If you reactivate a leave code, you must manually reactivate it for all applicable employees.
If you deactivate an already assigned leave code, the system prompts you to deactivate it for those employees. Reactivating a leave code requires you to manually reactivate it for all applicable employees.
Note: Settings on this form are standard, but you can modify settings for associated employees using PR Employee Leave. Tasks
