# PR Employee Leave Initialization Form

Use this from to initialize an active leave code for all, or active, employees within a PR group.
Access this form by selecting the Initialize Employees button on the PR Leave Codes form.
When you initialize employees with this form, the system adds all leave code settings to PR Employee Leave. If the employee/leave code combination already exists in PR Employee Leave, the system does not make any changes. All new entries initialize without overrides and with zero balances; in other words, the standard leave code settings. Use PR Employee Leave to override any settings for specific employees.
For more information on using this form, see [Initializing Leave Codes for Multiple Employees](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/leave/about-creating-leave-codes/initialize-a-leave-code-for-multiple-employees).
