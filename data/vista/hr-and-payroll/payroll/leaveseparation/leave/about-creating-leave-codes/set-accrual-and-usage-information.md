# Set Accrual and Usage Information

The following describes how to set accrual and usage information for a leave code on the PR Leave Codes form.

1. Select Payroll > Programs > PR Leave Codes.

1. Select an existing leave code or create a new [fixed amount](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/leave/about-creating-leave-codes/create-a-fixed-amount-leave-code) or [rate-based](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/leave/about-creating-leave-codes/create-a-rate-based-leave-code) leave code.

1. Select the Accrual/Usage Info tab.

1. In the Earn Code field, enter the applicable earnings code.

1. From the Type dropdown, select A-Accrual or U-Usage. Note: You do not need to enter an accrual type for fixed amount leave codes.

1. From the Basis dropdown, select A-Amount or H-Hours. Note: When creating leave codes, you will typically select H-Hours as the basis. You might select A-Amount if you are using this form to track accrual and usage for something other than a leave code (e.g., medical spending, dependent care, etc.)

1. In the Rate field, enter the accrual or usage rate.

1. Save the record.

1. Repeat as needed for additional earnings codes.

When running the leave process, the system multiplies the basis by the rate to determine the accrual or usage amount for the specified earnings code.
