# PR Add Leave Trans to Batch Form

Use this form to change or delete a previously posted leave transaction.
Access this form by selecting File > Add Transaction in the PR Leave Entry form.
Note: The system does not allow adding transactions to a batch where the leave code is no longer active (Active checkbox is deselected for the employee leave code in PR Employee Leave or for the company in PR Leave Codes).
