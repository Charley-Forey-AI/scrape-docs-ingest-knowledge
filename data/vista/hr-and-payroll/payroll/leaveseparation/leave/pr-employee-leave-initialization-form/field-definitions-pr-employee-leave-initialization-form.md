# Field Definitions: PR Employee Leave Initialization Form

The following is a list of field descriptions for the PR
 Employee Leave Initialization form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Initialize All or Active Employees

 Select whether All Employees or only Active Employees are to be initialized for this leave code. All new entries will be initialized with no overrides and 0.00 balances. Employee leave codes that already exist will not be changed.

##  PR Group

 Enter a valid PR group to which you want to initialize leave.

##  Eligible Date

 Enter an eligibility date for this leave initialization.
