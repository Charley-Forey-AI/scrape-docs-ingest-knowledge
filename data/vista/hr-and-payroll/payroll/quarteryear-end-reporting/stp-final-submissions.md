# STP Final Submissions

(AUS only) At the end of each fiscal year, use the STP Process form to finalize your STP submissions.
You should only generate the finalisation file after you have paid the year's final pay sequences. Select the link below for more information about creating and submitting the STP Finalisation File.
Note: For more information about STP processing, see [Single Touch Payroll (STP)](/en/vista/vista/hr-and-payroll/payroll/payroll-processing/single-touch-payroll-stp).
