# Generate and Save an Electronic Report

Generate and save an electronic report.

1. In the State field, enter a state code, or press F4
 to select from a list of states.

1. In the Quarter
 Ending field, enter a date. The ending quarter must be dated one
 of the following: 03/XX, 06/XX, 09/XX, or 12/XX.

1. Click Export. The Save Export File As window displays.

1. Save the electronic
 file. The file name and extension defaults based on the state for which you are
 generating the report and should be accepted as is (unless your state has
 specific file name or extension requirements that differ from the default).
