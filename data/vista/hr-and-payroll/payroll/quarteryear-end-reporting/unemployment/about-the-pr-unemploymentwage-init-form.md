# About the PR Unemployment/Wage Init Form

Use this form to initialize quarterly unemployment and wage reporting data for a specific quarter.
Access this form by selecting File  > Initialize or clicking the Initialize button on the [PR Unemployment Process](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/about-the-pr-unemployment-process-form) form.
You can initialize data for all states or for a specific state.
Unemployment and wage data is pulled from the Employee Accumulations and State Information tables. Data is then stored in the PRUH (PR Unemployment Report Header) and PRUE (Unemployment Report Employees) tables.
Unemployment and wage data can be re-initialized for
 any state if necessary. If you want the existing information to be cleared before the
 new information is initialized, select the Replace Existing Data - All Current Information
 Will Be Cleared checkbox. Otherwise, only new information that was not
 previously initialized is added.
