# About the PR Generate Electronic Filing Form

Use this form to generate a Quarterly Unemployment and Wage Report based on information stored in the PRUH (PR Unemployment Report Header) and PRUE (Unemployment Report Employees) tables. Generated reports on a state-by-state basis for each specific quarter.
Access this form by selecting File > Electronic
 File from the [PR
 Unemployment Process](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/about-the-pr-unemployment-process-form) form.
