# Field Definitions: PR Generate Electronic Filing Form

The following is a list of field descriptions for the PR
 Generate Electronic Filing form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## State

Enter the state (from HQ States) for which to
 generate electronic filing.
Important: Vista is transitioning to the advanced
 e-filing capabilities provided by Aatrix. As states change their reporting requirements
 and e-filing specifications, the standard Vista e-file format may no longer comply. In
 this case, you will need to use Aatrix for SUI/Wage reporting.For a list of states
 that must now use Aatrix for SUI/Wage reporting, see [States Requiring Aatrix for Unemployment Reporting (United States)](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/states-requiring-aatrix-for-unemployment-reporting-united-states).
For information about e-filing unemployment reports using Aatrix, see [Create / Submit Quarterly Unemployment Report Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/create--submit-quarterly-unemployment-report-using-aatrix).

##  Quarter Ending

 Enter the last month of the quarter for which you are printing this Unemployment/Wage Report.
