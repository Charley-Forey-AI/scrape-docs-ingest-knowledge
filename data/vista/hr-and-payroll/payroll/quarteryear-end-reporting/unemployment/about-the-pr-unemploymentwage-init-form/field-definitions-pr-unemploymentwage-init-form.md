# Field Definitions: PR UnemploymentWage Init Form

The following is a list of field descriptions for the PR
 UnemploymentWage Init form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Restrict By State

 Check this box to initialization unemployment/wage report individually by state. Only the state indicated below will be initialized.
 Do not check this box if you want all states initialized at one time.

##  State

 If initializing by state, enter the state for which you are initializing the unemployment/wage report.
Important: Vista is transitioning to the advanced
 e-filing capabilities provided by Aatrix. As states change their reporting
 requirements and e-filing specifications, the standard Vista e-file format may no
 longer comply. In this case, you will need to use Aatrix for SUI/Wage
 reporting.For a list of states that must now use Aatrix for SUI/Wage
 reporting, see [States Requiring Aatrix for Unemployment Reporting (United States)](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/states-requiring-aatrix-for-unemployment-reporting-united-states).
For information about e-filing unemployment reports using Aatrix, see
 [Create / Submit Quarterly Unemployment Report Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/create--submit-quarterly-unemployment-report-using-aatrix).

##  Quarter Ending

 Enter the last month for the calendar quarter (03/YY, 06/YY, 09/YY, or 12/YY) to which this report applies.

##  Replace Existing Data - All Current Information Will Be Cleared

 Check this box if you want all data previously initialized to be cleared and replaced.
 Do not check this box if you want existing values to remain, and only values not previously entered to be initialized.
