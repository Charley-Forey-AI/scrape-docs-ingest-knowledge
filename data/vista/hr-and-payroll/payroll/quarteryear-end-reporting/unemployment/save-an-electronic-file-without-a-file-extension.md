# Save an Electronic File without a File Extension

When you first save the electronic file, Windows
 automatically attaches a file extension to the file name. However, some states require a
 specific file name without the extension.
Although you cannot save the file without the
 extension being added, you can remove the extension from the name by renaming the file
 after it has been created.

1. Use the Windows Explorer
 to locate the text file.

1. Right-click the file
 name.

1. In the context menu,
 select Rename, remove the extension from the file name, and press
 Enter.Note: This may trigger a message indicating
 the filename extension may make the file unusable. You have the option to
 make the proceed or cancel the action.
