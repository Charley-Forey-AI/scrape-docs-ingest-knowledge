# Field Definitions: PR Unemployment Process Form

The following is a list of field descriptions for the PR
 Unemployment Process form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Quarter Ending

 Enter the last month for the calendar quarter (03/YY, 06/YY, 09/YY, or 12/YY) to which this report applies.

##  State

 Enter the state for which to set up unemployed report information.
Important: Vista is transitioning to the advanced e-filing
 capabilities provided by Aatrix. As states change their reporting requirements and
 e-filing specifications, the standard Vista e-file format may no longer comply. In
 this case, you will need to use Aatrix for SUI/Wage reporting.For a list of
 states that must now use Aatrix for SUI/Wage reporting, see [States Requiring Aatrix for Unemployment Reporting (United States)](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/states-requiring-aatrix-for-unemployment-reporting-united-states).
For information about e-filing unemployment
 reports using Aatrix, see [Create / Submit Quarterly Unemployment Report Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/unemployment/create--submit-quarterly-unemployment-report-using-aatrix).

##  Company Name

 Enter the name of the company to which this report applies; maximum of 50 characters. If the initialize feature was used previously (Initialize option, File menu), the company name defaults from HQ Company Setup.

##  Mailing Address

 Enter the company’s mailing address; maximum of 40 characters. If the initialize feature was used previously (Initialize option, File menu), the address defaults from HQ Company Setup.

##  Location Address

 Enter the location address as defined by state; maximum of 22 characters.

##  City

 Enter the city in which the company is located; maximum of 25 characters. If the initialize feature was used previously (Initialize option, File menu), the city defaults from HQ Company Setup.

## State

Enter a valid state (as defined in HQ States). The system validates the state based on the Default Country specified in HQ Company Setup for the active company. If not valid, an error displays, but entry is allowed. If you used the initialize feature (Initialize option, File menu), the state defaults from HQ Company Setup.

##  Zip

 Enter the 5-character zip code. If you used the initialize feature (Initialize option, File menu), the zip code defaults from HQ Company Setup.

##  Zip Ext.

 Enter the 4-digit zip code extension.

##  Federal Tax ID#

 Enter the Federal Tax ID# (aka Employer Identification Number), omitting all non-numeric characters, such as hyphens. If you used the initialize feature (Initialize option, File menu), this number defaults from HQ Company Setup.
Note: Florida requires this field when generating electronic unemployment files.

##  Other EIN #

 If an IRS form 941 (Employer’s Quarterly Federal Tax Return) or 943 (Employers Annual Tax Return) was submitted for the same tax year as this report, and the form used an EIN number other than the one indicated above (Federal Tax ID#), enter that number here. Otherwise, leave blank.
Maine Only: Use this field to enter your Revenue Service ID number.

##  Contact

 Enter the name of the person (up to 30 characters) to contact concerning problems with this report. If the initialize feature was used previously (Initialize option, File menu), the contact name defaults from PR State Information (Unemp/Wage Reporting tab).
Note: Florida requires this field when generating electronic unemployment files.

##  Phone

 Enter the contact’s telephone number; up to 15 characters. If the initialize feature was used previously (Initialize option, File menu), the contact’s phone number defaults from PR State Information (Unemp/Wage Reporting tab).
Note: Florida requires this field when generating electronic unemployment files.

##  Contact Email

 Enter the contact’s email address. If the initialize feature was used previously (Initialize option, File menu), the contact’s phone number defaults from PR State Information (Unemp/Wage Reporting tab).
Note: Florida requires this field when generating electronic unemployment files.

##  Ext

 Enter the contact’s telephone extension number; up to of five characters. If you used the initialize feature (Initialize option, File menu), the contact’s telephone extension defaults from PR State Information (Unemp/Wage Reporting tab).

##  Unemployment ID #

 If a separate ID# is required for reporting unemployment information, enter the ID#; up to 20 characters. If you used the initialize feature (Initialize option, File menu), the unemployment ID # defaults from PR State Information (Unemp/Wage Reporting tab).
Note: Florida requires this field when generating electronic unemployment files.

##  Transmitter ID #

 Enter the media transmitter or authorization number, up to six characters, assigned to you by the State. If you used the initialize feature (Initialize option, File menu), the transmitter ID # defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Establishment ID #

 Enter the establishment identification number, up to four characters. This ID# may be used to designate various store or factory locations or types of payroll when a file contains multiple records with the same EIN#. If you used the initialize feature (Initialize option, File menu), the establishment ID # defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  C-3 Data

 Enter C-3 data (1 character), as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the C-3 data defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Plant #

 If applicable, specify the plant code as defined by state, up to two characters.
 If you used the initialize feature (Initialize option, File menu), the Plant code will default from PR State Information, if defined for this state.

##  Tax Type

 Enter the 1-character tax type, as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the tax type defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Branch

 If applicable, specify the branch code (up to three characters) as defined by state.
 If you used the initialize feature (Initialize option, File menu), the Branch code will default from PR State Information, if defined for this state.

##  Electronic Filing Type

 Select the electronic filing type for the State specified above.
· Original
· Amended

##  Suffix Code

 Enter the suffix code (up to five characters) as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the suffix code defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Taxing Entity

 Enter the taxing entity (up to 10 characters) as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the taxing entity defaults from PR State Information (Unemp/Wage Reporting tab)
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Document Control ID

 Enter the document control identification number (up to 10 characters) as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the document control ID# defaults from PR State Information (Unemp/Wage Reporting tab).
· Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  State ID#

 Enter the 2-digit Federal postal code assigned to the State specified above. This is NOT the 2-character postal abbreviation. If you used the initialize feature (Initialize option, File menu), the state ID# defaults from PR State Information (Unemp/Wage Reporting tab).

##  State Control #

 Enter the state control number (up to seven characters) as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the state control number defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Unit ID#

 Enter the unit identification number (up to five characters) as defined by the state requiring this information.
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  County

 Enter the county code (up to three characters) as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the county code defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Out of County

 Enter the out of county employee data (up to seven characters) as defined by the state requiring this information. If you used the initialize feature (Initialize option, File menu), the out of county data defaults from PR State Information (Unemp/Wage Reporting tab).
 Used by the ICESA (Interstate Conference of Employment Security Agencies) format.

##  Create Date

 Enter the media creation date.

##  Computer

 Enter the manufacturer's name, up to eight characters.

##  Unemployment Insurance Tax Rate

 Defaults the unemployment insurance tax rate based on the SUTA liability code specified for the state in PR State Information and the rate assigned that code in PR Deductions/Liabilities; may be overridden.

##  Employer Assessment Rate

 Enter the employer assessment rate, up to nine digits (including decimal). This is an optional field used by some states for electronic filing.

##  Employer Assessment Amount

 Enter the employer assessment amount. This is an optional field used by some states for electronic filing.

##  Employee Assessment Rate

 Enter the employee assessment rate, up to nine digits (including decimal). This is an optional field used by some states for electronic filing.

##  Employee Assessment Amount

 Enter the employee assessment amount. This is an optional field used by some states for electronic filing.

##  Previous Underpayment

 Enter the previous underpayment amount. This is an optional field used by some states for electronic filing.

##  Interest

 Enter the interest amount. This is an optional field used by some states for electronic filing.

##  Penalty

 Enter the penalty amount. This is an optional field used by some states for electronic filing.

##  Additional Penalty

 Used by the state of New Mexico only.
 Enter the additional penalty amount. This is the ‘late report penalty’ as defined by the state of New Mexico.

##  Overpayment

 Enter the overpayment amount. This is an optional field used by some states for electronic filing.

##  Allocation Amt

 Enter the allocation amount. This is an optional field used by some states for electronic filing.

##  Total Due

 Enter the total due amount. This is an optional field used by some states for electronic filing. Check your state’s requirements to determine if this information is needed.
For the state of Florida, this is a required field and is used to populate the 'total due' in the electronic media file.
For the state of New Mexico, this is the payment amount and includes tax due, interest, late report penalty, and late payment penalty.

##  Total Remittance

 Enter the total remittance amount. This is an optional field used by some states for electronic filing.

## Other

The Other tab contains inputs that are used
 only for Louisiana quarterly reporting. Fields on this tab are visible only when LA is entered
 in the State field on the Info tab.

##  Multiple County Industry (LA only)

Multiple County Indisutry checkbox in the PR State Unemployment Form, Other tab.
Note: This field appears only when the State field on the Info tab is set to LA.

Select this checkbox if this firm has employees in more than one county/industry included in this report.

##  Multiple Worksite Location (LA only)

Multiple Worksite Location checkbox in the PR State Unemployment Form, Other tab.

Note: This field appears only when the State field on the Info tab is set to LA.

Select this checkbox only if this firm has employees at more than one location within the same county included in this report.

##  Multiple Worksite Indicator (LA only)

Multiple Worksite Indicator checkbox in the PR State Unemployment Form, Other tab.
Note: This field appears only when the State field on the Info tab is set to LA.

Select this checkbox only if this firm is including multiple worksite data on wage reporting diskette in lieu of Form BLS-3200.
 If this firm will report multiple worksite data using Form BLS-3200, do not check this box.

##  Electronic Funds Transfer Indicator (LA only)

Electronic Funds Transfer Indicator checkbox in the PR State Unemployment Form, Other tab.
Note: This field appears only when the State field on the Info tab is set to LA.

Select this checkbox only if this firm participates in electronic funds transfer (EFT) of quarterly UI payroll taxes.

##  Multi-State Employers Indicator (LA only)

Multi-State Employers Indicator checkbox in the PR State Unemployment Form, Other tab.
Note: This field appears only when the State field on the Info tab is set to LA.

Select this checkbox only if this firm has employees included in this report who worked in multiple states.

##  Export File Type (VT only)

Export File Type options in the PR State Unemployment Form, Other tab.
Note: This field appears only when the State field on the Info tab is set to VT.

If this firm has 250 or fewer employees, select Small Employer.
If this firm has 251 or more employees, select Large Employer. The system enables the remaining fields.

- FTE Employee Count - refer to the state's definition of FTE.

- Health Care Amt Due - refer to the state's requirement for this field's value.

##  FTE Employee Count (VT only)

FTE Employee Count field in the PR State Unemployment Form, Other tab.
Note: This field appears only when the State field on the Info tab is set to VT.

The system enables this field when you select Large Employer.
Refer to the state's definition of full-time-equivalent employee.

##  Health Care Amt Due (VT only)

Health Care Amt Due field in the PR State Unemployment Form, Other tab.

Note: This field appears only when the State field on the Info tab is set to VT.

The system enables this field when you select Large Employer.
Refer to the state's requirement for this field's value.
