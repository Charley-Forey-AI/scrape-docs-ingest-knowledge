# About Regulatory Reporting Using Aatrix

Aatrix Software specializes in tax form creation and electronic filing.
Integrated into Vista's existing regulatory reporting options, Aatrix uses your Vista data to create your company's reports and electronic files. You can also enroll with Aatrix to submit your files for you.
Reports and electronic files that Aatrix can output include W2s, 1099s, 941s, T4s, and T5018s, along with many others. You can use Aatrix to process multiple regulatory reports and filings required throughout the year.

## Prerequisites to using the Aatrix Application

- Download and install the application to your local workstation. For instructions, see [Enabling Aatrix on a workstation](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/enable-aatrix-on-a-workstation).

- [Enroll with Aatrix](https://partner.aatrix.com/vista). This is required in order to submit your electronic files.

- Enter your [DUNS #](/en/vista/vista/administration/headquarters/company-setup/hq-company-setup-form/field-definitions-hq-company-setup-form#ID-0000ebcc--en) in the HQ Company Setup form.

- For state and/or local payroll regulatory files, you must enter an [Aatrix Tax Type](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-deductionsliabilities-form/field-definitions-pr-deductions-liabilities-form#ID-00036b63--en) in each applicable deduction or liability code in the PR Deductions/Liabilities form (Addl Info tab).

## Using the Aatrix Application

Access to the Aatrix application depends on the reports or electronic files that you are processing. Most regulatory reporting requires access directly through the PR Aatrix Report Selection form. However, some processes require access through other forms within Vista.
FormHow to Access Aatrix
W-2

1. Open the PR W-2 Process form and enter the Tax Year.

1. Select Tasks > Launch Aatrix Print and eFile.

1. In PR Aatrix - W2 Print and eFile, select the applicable W2 form.

1. Select Continue.

For more information about processing W-2s, see [Process W-2s](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/w-2s/process-w-2s).

1099

1. Open the AP 1099 Processing form and enter the Year Ending.

1. Select Tasks > Download.

1. In AP 1099 Download, enter the Calendar Year Ending Month and remaining information.

1. In AP 1099 Aatrix Print and eFile, select the appropriate 1099 form.

1. Select Launch Aatrix.

For more information about processing 1099s, see [Prepare and Process 1099s](/en/vista/vista/accounting/accounts-payable/year-end-reporting/1099s/prepare-and-process-1099s).

1094/1095-C1094/1095-B

1. Open the PR ACA Process form and enter the Tax Year.

1. Select Tasks > Launch Aatrix Print and eFile.

1. In Select the applicable form.

1. Select Continue.

For more information, see [Create, Print, and E-File 1094s and 1095s Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/create-print-and-e-file-1094s-and-1095s-using-aatrix).

T5018

1. Open the AP T5018 Payments form and enter the Reporting Period End Date.

1. Select Tasks > Launch Aatrix.

1. In AP T5018 Aatrix Print & eFile, select the T5018 form.

1. Select Continue.

For more information, see [Generate T5018 Reports Using Aatrix](/en/vista/vista/accounting/accounts-payable/year-end-reporting/t5018s/about-generating-t5018-reports/generate-t5018-reports-using-aatrix).

T4

1. Open the PR Canada T4 form and enter the Tax Year.

1. Select Tasks > Launch Aatrix Print and eFile.

1. In PR Aatrix - T4 Print and eFile, select the T4 form.

1. Select Continue.

For more information, see [Create and Submit T4s Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/create-and-submit-t4s-using-aatrix).

Note: The Aatrix application does not send data back to Vista. If you discover errors and need to make corrections after you have already submitted a filing, you must do so directly in Aatrix. For more information, see [Make Corrections to Regulatory Filings via Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/make-corrections-to-regulatory-filings-via-aatrix).

## Quarterly Unemployment Reporting

To create and submit unemployment reports, use the PR Aatrix Report Selection form. For more information, see [Create / Submit Quarterly Unemployment Report Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/create--submit-quarterly-unemployment-report-using-aatrix).
Although using Aatrix is currently optional for many states, there are some states that must use Aatrix for unemployment reporting. For a list of these states, see [States Requiring Aatrix for Unemployment Reporting (United States)](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/create--submit-quarterly-unemployment-report-using-aatrix/states-requiring-aatrix-for-unemployment-reporting-united-states).

## Federal 941 Reporting

If you electronically file your 941 Employer's Quarterly Federal Tax Return, you must use Aatrix. You can access the 941 report from the PR Aatrix Report Selection form, which auto loads with PR Pay Period Detail values. You can edit values as needed directly within the report and then either print (at no cost) or e-file the result. For more information, see [Create / Submit Regulatory Filings using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/create--submit-regulatory-filings-using-aatrix).

## Other Regulatory Filings

To create or submit any other available regulatory filing, use the PR Aatrix Report Selection form. For more information, see [Create / Submit Regulatory Filings using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/create--submit-regulatory-filings-using-aatrix).
