# PR Aatrix Report Selection Form

Use the PR Aatrix Report Selection Form to create and submit regulatory reports and electronic files.
You will only need to use this form for regulatory reporting not handled elsewhere in the system. For example, you can use this form for Federal 941, I9, and state regulatory reporting, but you cannot use this form for W-2s, 1099s, 1095s, T4s, and T5018s.
 For more information about using this form, see [Create / Submit Regulatory Filings using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/create--submit-regulatory-filings-using-aatrix).
