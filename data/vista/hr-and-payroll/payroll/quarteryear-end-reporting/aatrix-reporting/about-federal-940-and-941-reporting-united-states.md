# About Federal 940 and 941 Reporting (United States)

Vista provides options for filing your Federal 940 and 941 tax
 returns.

## Federal 940 (Employer's Annual
 Federal Unemployment (FUTA) Tax Return)

Vista supports the PR 940/941 Information report (Report
 ID#764) to assist with your Employer’s Annual Federal
 Unemployment (FUTA) Tax Return. This report displays annual FUTA values
 that you must manually transfer to a Federal 940
 form.Vista also supports annual FUTA reporting
 through its Aatrix integration. A current version of the 940 is
 available from the PR Aatrix Report Selection form. The report loads
 with PR Pay Period detail values for the specified year and allows
 you to view, edit and print an official 940 return
Note: Vista's standard PR 940/941 Information
 report pulls data from PR Employee Accumulations (PREA) and
 includes adjustments and corrections made directly to those
 monthly accumulation balances. This report can produce inconsistent
 results if you make adjustments to PR Employee Accumulations
 that impact an employee’s eligible FUTA earnings.Because the Aatrix 940 Schedule A report relies on PR Pay
 Period Detail, it will agree and be fully
 reconcilable with your actual payroll
 data.

## Federal 941 (Employer’s Quarterly Federal Tax Return)

This form is used to report income taxes, social security tax, and/or Medicare tax withheld from employee's paychecks, as well as to pay the employer's portion of social security and/or Medicare tax. Form 941 must be mailed or e-filed to the IRS.
You must report and e-file your 941 Employer’s Quarterly Federal Tax Return using Vista’s standard Aatrix integration. This report is accessible from the PR Aatrix Report Selection program and auto loads with PR Pay Period Detail values. You can edit values as needed directly within the report and then either print (at no cost) or e-file the result. For more information, see Create/Submit Regulatory Filings using Aatrix.
