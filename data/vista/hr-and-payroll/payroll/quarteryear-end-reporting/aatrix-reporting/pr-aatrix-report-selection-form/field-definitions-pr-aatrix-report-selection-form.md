# Field Definitions: PR Aatrix Report Selection Form

The following is a list of field descriptions for the PR
 Aatrix Report Selection form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Tax Year

The Tax Year field in the PR Aatrix Report Selection form.
The default is the current tax year.
Override if needed.

Related information

- [About Regulatory Reporting Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/about-regulatory-reporting-using-aatrix)

## Federal/State

The Federal/State drop-down on the PR Report
From the drop-down, select Federal or a state for which to create/submit regulatory reports.

## Form Type

The Form Type text box on the PR Aatrix Report Selection form.
From the list, select the form type for regulatory reporting. The forms available in this list will differ based on your selection in the Federal/State drop-down.

## Description

The Description text box on the PR Aatrix Report Selection form.
Display only, the description of the selected form type.

## Period

The Period drop-down on the PR Aatrix Report Selection form.
This field only displays for regulatory reports that are submitted annually or quarterly.
From the drop-down, select Annual or the applicable reporting quarter.
Note: The Annual option is only available for reports created/submitted annually. The four quarter options are only available for reports created/submitted quarterly.

## Start Date / End Date

The Start Date and End Date fields on the PR Aatrix Report Selection form.
These fields only display for regulatory reports that are not submitted annually or quarterly.
Enter the starting and ending date for regulatory reporting.
