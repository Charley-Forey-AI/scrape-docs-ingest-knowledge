# Make Corrections to Regulatory Filings via Aatrix

Use Aatrix to correct and submit regulatory filings ( W-2s, 1099s, ACA, T4s, and T5018s).
Once you generate regulatory filings and submit them to the appropriate taxing authority (via Aatrix), any corrections to those filings must be done directly in Aatrix. Since Aatrix does not send data back to Vista, you may want to manually update your data in Vista to match the corrections you make in Aatrix; however, you cannot submit corrections to Aatrix via Vista.
Note: If you have not completed the filing process in Aatrix (that is, you have not printed or submitted your filings to the taxing authority via Aatrix), you can cancel the filing in Aatrix, correct your data in Vista, and then re-transmit the data to Aatrix. Even though you have corrected your data, the records are still considered Original filings and are flagged accordingly.

The following discusses the initial steps for making corrections to regulatory filings in Aatrix; however, once you complete the initial steps, the process differs depending on the form you are correcting.

1. Access the Aatrix report selection form for the filing you are correcting.OptionDescription
1099In AP 1099 Processing, select Tasks > Download. Then in AP Download, select Launch Aatrix.
ACAIn PR ACA Process, select Tasks > Launch Aatrix Print and eFile.
W-2In PR W-2 Process, select Tasks > Launch Aatrix Print and eFile.
T5018In AP T5018 Payments, select Tasks > Launch Aatrix.
T4In PR Canada T4, select Tasks > Launch Aatrix Print and eFile.
Other (U.S.)Open the PR Aatrix Report Selection form.

The corresponding Aatrix launch form displays (for example, AP 1099 Aatrix Print and eFile).

1. Select the History button.The Aatrix Open Report screen displays.

1. From the grid, select the checkbox next to the form type and period to correct.

1. Select the Edit button.The History File Options screen displays, which will differ depending on the form you are correcting (for example, W-2 History File Options).
