# Field Definitions: PR Aatrix Print and eFile Form

The following is a list of field descriptions for the PR Aatrix Print and eFile form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Tax Year

The Tax Year field in the PR Aatrix - T4 Print and eFile form.
Enter the reporting tax year. Initially defaults the tax year selected in the PR Canada T4 form. You may override as needed.

## Form Type

The Form Type selection box on the PR Aatrix - T4 Print and eFile form.

## Payroll Program Account

The Payroll Program Account field in the PR Aatrix - T4 Print and eFile form.
Enter a valid payroll program account number or press F4 to select from a list of valid account numbers. This number must begin with RP and be followed by 4 digits (RP####).Note: Valid program account numbers are determined by the employee slips generated for the tax year. You can only enter or select a Payroll Program Account number that exists in the employee T4 slips table for the selected tax year. If you enter a program account number that does not exist, you will receive an error that the number is not valid for the tax year, even if you enter the Program Account Number specified for the company in PR Company Parameters.
If you use the F4 lookup to select an account and the list is blank, it means you have not generated employee slips. You will need to generate employee slips before you can continue.
