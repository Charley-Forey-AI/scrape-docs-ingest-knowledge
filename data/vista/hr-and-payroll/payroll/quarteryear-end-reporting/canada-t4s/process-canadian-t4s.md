# Process Canadian T4s

(Canada only) An overview of the T4 process of generating T4 slips and summaries for tax reporting.
Select the links for more detailed information.

1. Set up T4 information for the current tax year (company information, T4 box items, other codes, and employees). For more information, see [Set T4 Information for the Current Tax Year](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/set-t4-information-for-the-current-tax-year).

1. Create, print, and e-file T4s using Aatrix. For more information, see [Create and Submit T4s Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/create-and-submit-t4s-using-aatrix).
