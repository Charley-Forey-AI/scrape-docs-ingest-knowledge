# PR Aatrix - T4 Print and eFile Form

(Canada only) Use the PR Aatrix - T4 Print and eFile form to create your T4 slips and submit them to the Canada Revenue Agency (CRA).
Access this form by selecting Tasks > Launch Aatrix Print and eFile from the toolbar in the PR Canada T4 form.
Note: Before you can use the Aatrix application, you must have already installed it on your system. For details, see [Enable Aatrix on a Workstation](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/enable-aatrix-on-a-workstation).
Once you set up T4 information in the PR Canada T4 form, you will then use the Aatrix wizard to complete the reporting process; that is, printing and electronically filing your T4s. For more information, see [Create and Submit T4s Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/create-and-submit-t4s-using-aatrix).
Slips included in this process are based on the Tax Year and the Payroll Program Account specified for each slip in the PR Employee T4 Slips form.
If you need to make corrections after you have already submitted your T4s, you must do so directly in Aatrix. For more information, see [Make Corrections to Regulatory Filings via Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/make-corrections-to-regulatory-filings-via-aatrix).
