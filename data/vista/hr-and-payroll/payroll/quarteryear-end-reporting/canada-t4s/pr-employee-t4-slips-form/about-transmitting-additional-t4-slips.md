# About Transmitting Additional T4 Slips

(Canada only) Use the T4 Flag "targeted-selection" if you need to generate a second set of original T4 slips after having already submitted a first set of original or amended slips to the CRA for the same tax year.

Even though both sets of slips will be set to Return Type O-Original on the PR Employee T4 Slips form, you can select the T4 Flag checkboxes for only the slips that make up your second set. Then, if you select the Include slip only if its T4 flag is set checkbox on the PR Canada T4 Slip form and proceed to generate original slips, only the slips in your second set will be included.

## Clear All T4 Flags

The 'Clear All T4 Flags' action clears all flags for employees' slips that are set on the PR Employee T4 Slips form. You can clear the T4 Flag for every employee T4 slip in the selected Tax Year by going to Tasks > Clear All T4 Flags from the PR Canada T4 form once you submit the batch to the CRA.
