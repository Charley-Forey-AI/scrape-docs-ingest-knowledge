# PR Canada T4 Form

(Canada only) Use the PR Canada T4 form to set up T4 reporting information for the current tax year.
Each year you must:

- initialize the T4 box items for the year and assign specific earnings or deduction/liability codes to each box

- initialize T4 codes and assign specific earnings or deduction/liability codes to each T4 code

- initialize employee T4 slips for the current year.

Once you are done setting information for the current tax year, you can create and submit T4s using Aatrix (Tasks > Launch Aatrix Print and eFile).
