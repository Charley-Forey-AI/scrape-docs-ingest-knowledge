# Create and Submit T4s Using Aatrix

Use Aatrix to create your T4 slips and submit them to the Canada Revenue Agency (CRA).

- You must have Aatrix installed on your local workstation. For more information, see [Enable Aatrix on a Workstation](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/enable-aatrix-on-a-workstation).

- You must have set up T4 reporting information for the reporting tax year in PR Canada T4. For more information, see [Set T4 Information for the Current Tax Year](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/set-t4-information-for-the-current-tax-year).

To create and submit T4s using Aatrix:

1. From the main menu, select Payroll > Programs > PR Canada T4.The PR Canada T4 form displays.

1. In the Tax Year field, enter the reporting tax year.

1. From the toolbar, select Tasks > Launch Aatrix Print and eFile.The PR Aatrix - T4 Print and eFile form displays.

1. The Tax Year field default value is the tax year specified in PR Canada T4. Accept the default or change the tax year as necessary.

1. In the Form Type selection box, select T4 Official Form.

1. In the Payroll Program Account field, enter the payroll program account to report or press F4 to select from a list of valid account numbers. This number must begin with RP and be followed by 4 digits (RP####).

1. Click Continue.The Aatrix T4 Setup Wizard opens.

1. Follow the prompts in the Aatrix wizard to complete the process.Note: As you run through the wizard, you must resolve any areas highlighted in red before you can print and submit your T4s. Aatrix does not send data back to Vista, so changes made in the Aatrix wizard do not update the PR Canada T4 form.

If you need to make corrections after you have already submitted your T4s, you must use Aatrix. For more information, see [Make Corrections to Regulatory Filings via Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/canada-t4s/process-canadian-t4s/make-corrections-to-regulatory-filings-via-aatrix).
