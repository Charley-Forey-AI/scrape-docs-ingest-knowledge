# PR Aatrix - W2 Print and eFile Form

Use this form to process W-2s using Aatrix.
Access this form via the PR W-2 Process form by selecting Tasks > Launch Aatrix Print and eFile.
Once you enter the tax year and select the W-2 form type, clicking Continue launches the Aatrix wizard. You must then follow the prompts to complete the process. For more information, see [File W-2s Using Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/w-2s/file-w-2s-using-aatrix).
If you need to make corrections after you have already submitted your W-2s, you must do so directly in Aatrix. For more information, see [Make Corrections to Regulatory Filings via Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/w-2s/make-corrections-to-regulatory-filings-via-aatrix).
