# Field Definitions: PR Aatrix - W2 Print and eFile Form

The following is a list of field descriptions for the PR
 Aatrix - W2 Print and eFile form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Tax Year

The Tax Year field in the PR Aatrix - W2 Print and eFile form.
This field defaults the tax year selected in the PR W-2 Process form. You may override if needed.

## Form Type

The Form Type selection box on the PR Aatrix - W2 Print and eFile form.

Select the W-2 form type to process for your location (for example, for U.S. customers, select W-2/W-3).
