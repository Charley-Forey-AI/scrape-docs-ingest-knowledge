# Field Definitions: PR W-2 Create Attachments Form

The following is a list of field descriptions for the PR W-2
 Create Attachments form.

## Tax Year

The Tax
 Year field on the PR W-2 Create Attachments form.
Enter the tax year you want to pull the W-2 files from.

## Beginning Employee Number

The Beginning Employee Number field on
 the PR W-2 Create Attachments form.
Enter the employee number at the beginning of the range of employees you want
 to retrieve.

## Ending Employee Number

The Ending Employee Number field on the
 PR W-2 Create Attachments form.
Enter the employee number at the end of the range of employees you want to
 retrieve.

## List of Employee Numbers

 The List of
 Employee Numbers: Comma, Space, or Tab separated field on the PR W-2 Create
 Attachments form.
Fill out the employees you wish to retrieve using their employee number,
 separating each by a comma, space, or tab.

## Truncate SSN

The Truncate
 SSN field on the PR W-2 Create Attachments form.
Select the Truncate SSN box to display only the last four digits of the social
 security number, instead of the entire SSN.
