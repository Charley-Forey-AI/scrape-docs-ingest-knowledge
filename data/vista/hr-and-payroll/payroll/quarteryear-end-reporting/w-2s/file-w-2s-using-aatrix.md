# File W-2s Using Aatrix

You will use Aatrix to generate the electronic files for federal, state, and local W-2s.
Beginning December 2021, you must print and eFile W-2s using Aatrix. If you have not yet installed Aatrix on your local workstation, you must do so prior to processing W-2s. For more information, see [Enable Aatrix on a Workstation](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/enable-aatrix-on-a-workstation) and [Prerequisites to using the Aatrix application](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aatrix-reporting/about-regulatory-reporting-using-aatrix).
Note: Before you print and eFile your W-2s via Aatrix, you should print the PR W-2 Preview Report to make sure that all of the necessary information has been entered and is correct. If information has not been set up correctly, you can recreate the W-2 files using PR W-2 Process. For more information, see [Preview Federal and State W-2 Reporting Information](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/w-2s/preview-federal-and-state-w-2-reporting-information) .

1. From the main menu, select Payroll > Programs >  PR W-2 Process.The PR W-2 Process form displays.

1. In the Tax Year field, enter the tax year for W-2s.

1. Make sure that you entered the required information for federal filing. This information includes the following fields: User ID#, Contact Name, Phone, and Delivery Address.

1. Select Tasks > Launch Aatrix Print and eFile.The PR Aatrix - W2 Print and eFile form displays.

1. The Tax Year field defaults the year you specified in the PR W-2 Process form. Accept the default or enter the correct reporting year.

1. In the Form Type selection box, select the appropriate W-2/W-3 report for your location.

1. Select Continue.The Aatrix applications displays.

1. Follow the prompts in the Aatrix application to complete the process.Note: The Aatrix application does not send data back to Vista. If you need to make corrections once you have submitted your W-2s, you must do so directly in Aatrix. For more information, see [Make Corrections to Regulatory Filings via Aatrix](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/w-2s/make-corrections-to-regulatory-filings-via-aatrix).

1. For New Jersey users only:If you have employees with NJ Private Family Leave
 Insurance (NJPFLI) or NJ Private Disability Insurance (NJPDI), you will need to
 manually set up that information in Aatrix (on the Verify Employee Information
 grid); Aatrix does not pull this information from Vista. For instructions on how to do this, see the
 [Aatrix NJ Wage and Tax
 Formats](https://www.aatrix.com/support/knowledgebase/windows/us/standard/state-local/nj-formats/) webpage.Note: When you print W-2s, the
 Private Plan account numbers will appear in Box 14; however, the actual
 amounts will appear in Boxes 18, 19, and 20.
