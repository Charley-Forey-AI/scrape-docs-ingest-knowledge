# PR W-2 Create Attachments Form

Use the PR W-2 Create Attachments form to generate and store
 PDF attachments of each employee's W-2 form, providing an exact copy of the W-2 form
 as it was issued.
Access this form from the Payroll module, or from the PR W-2
 Process form by going to Tasks >  Launch W-2 Create
 Attachments.

A generated PDF for the employee is attached to
 their specific W-2 record in the PR W-2 Process form,
 and given a unique attachment ID which differentiates the new attachment
 from other versions.
Additionally, when users run this attachment
 process, all W-2 documents generated from it are automatically assigned a
 new standard DM Attachment type, called PR W2 Forms.
