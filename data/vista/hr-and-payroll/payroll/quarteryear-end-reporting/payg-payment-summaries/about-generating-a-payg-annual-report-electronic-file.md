# About Generating a PAYG Annual Report Electronic File

To generate a PAYG Annual Report electronic file, select Generate PAYG Annual Report E-File from the Tasks menu on the PR ETP Summary Process toolbar.
Note: Before you generate the PAYG electronic file, you should print the PR PAYG Payment Summary - Individual Non-Business and the PR PAYG Payment Summary - Employee Termination Payment to make sure that all of the necessary information has been entered and is correct.
For more information about generating electronic files, see [Generate Electronic Files for PAYG Summaries](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/generate-electronic-files-for-payg-summaries).
