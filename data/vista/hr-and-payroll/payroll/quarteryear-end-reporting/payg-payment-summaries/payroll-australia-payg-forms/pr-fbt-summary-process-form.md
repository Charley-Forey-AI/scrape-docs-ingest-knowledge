# PR FBT Summary Process Form

Use the PR FBT Summary Process form to prepare Fringe Benefit Tax (FBT) returns for the current tax year.
The PR Fringe Benefit Tax Return report retrieves values from this form.
When you first access this form, the system retrieves contact information from the HQ ATO Tax Years form. You can modify the default values and enter additional details required for the report.
Once you are finished filling in information, you can generate the reporting amounts that will display in the Return calculation details section of the PR Fringe Benefit Tax Return report.
Note: If you pay your FBT using instalments, you must use the HQ BAS Process form to prepare a Business Activity Statement (BAS) for each quarterly instalment prior to preparing your annual FBT return. For more information, see [Business Activity Statements for FBT Instalments](/en/vista/vista/administration/headquarters/australian-setup/business-activity-statements/business-activity-statements-for-fbt-instalments).
