# PR Limits and Rates Form (Australia)

(Australia only) Only change the information in the PR Limits and Rates form if the ATO changes rates mid-year and you have not yet installed a new release of Vista since the ATO rate change.
Use this form to define the limits and rates used by the system to calculate termination payments and associated tax deductions.
Any changes to limits and/or rates are updated automatically when you install a new release of Vista.
Important: You should never need to add or edit information in this form, and it is strongly recommended that you do not. You should only change this information if the ATO changes rates mid-year and you have not yet installed a new release of Vista since the ATO rate change.
Note: Also, if you make a change to a field that results in an empty value, that field's value will automatically default to zero 0.
