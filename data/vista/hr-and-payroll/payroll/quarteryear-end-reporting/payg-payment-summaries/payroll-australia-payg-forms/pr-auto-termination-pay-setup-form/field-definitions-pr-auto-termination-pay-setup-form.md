# Field Definitions: PR Auto Termination Pay Setup Form

The following is a list of field descriptions for the PR Auto
 Termination Pay Setup form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## ETP/Leave Earn Code

Enter the earnings code that calculates this ETP type. This field applies, and is enabled for, the following ETP types:

- Annual Leave (and Leave Loading) - For more information, see [Setting Up Automatic Annual Leave ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments).

- Long Service Leave - For more information, see [Setting Up Automatic Long Service Leave ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-long-service-leave-payments).

- Standard ETPs (including "in lieu of notice", unused personal leave, and unused RDO) - For more information, see [Setting Up Automatic Standard ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-standard-employment-termination-payments-australia).

## Redund ETP/Leave Earn Code

Enter the earnings code for calculating
 redundancy payments for this ETP type record. Press F4 for a list of earnings codes.
This field allows you to enter earnings codes setup with an ATO Category of LSAR (for annual or Long Service Leave payment types) or ETPR (for ETP payments) subject to special tax deduction (Tax Free Component) considerations due to an employee’s job having been made redundant.
Additionally, this field allows for entry of earnings codes with an ATO Category of either ERP or ETPR. Upon the release of 6.8, it was unclear if these types of payments are taxed as part of Redundancy calculations (where a Tax Free component is considered) or if these payments are always taxed as a normal ETP (similar to RDO where they are not a part of Tax Free component consideration). By removing restrictions on this field for these two payment types, you are free to enter the appropriate earnings code that would be a member of the Subject Basis of the appropriate tax deduction code. Freedom to enter either type of earnings code will put the user in control of how these two types of payments are taxed.

## Termination Payment Type

Select the ETP type for this record. You can choose from the following options:

- AL - Annual Leave - For more information
 on setting up this type of record, see [Setting Up Automatic Annual Leave ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments).

- ALL - Annual Leave Load - This must be
 set up separately from annual leave on this form, but the two types are tied together. For
 more information, see [Setting Up Automatic Annual Leave ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments).

- LN - In Lieu of Notice - For more
 information, see [Setting Up Automatic Standard ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-standard-employment-termination-payments-australia).

- PL - Personal Leave - For more
 information, see [Setting Up Standard ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-standard-employment-termination-payments-australia).

- RDO - Rostered Days Off - For more
 information, see [Setting Up Automatic Standard ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-standard-employment-termination-payments-australia).

- R - Redundancy/Early Retire - This is for
 redundancy payments, not for an ETP payment made due to redundancy. For more information,
 see [Setting Up Automatic Redundancy/Early Retirement ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-setting-up-redundancy-etps).
