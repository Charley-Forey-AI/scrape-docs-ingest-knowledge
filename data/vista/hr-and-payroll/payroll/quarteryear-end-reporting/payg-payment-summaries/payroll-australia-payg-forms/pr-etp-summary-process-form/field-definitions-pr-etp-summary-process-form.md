# Field Definitions: PR ETP Summary Process Form

The following is a list of field descriptions for the PR ETP
 Summary Process form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Signature of Authorised Person

This field initially defaults from the
 Signature of
 Employer field on the [HQ ATO Tax Years](/en/vista/vista/administration/headquarters/australian-setup/hq-ato-tax-years) form. Accept the default, or enter the
 signature of the person authorised to submit this report to the ATO. If you change the
 information here, the system will not update HQ ATO Tax Years.

## Report Date

Defaults from the Report Date
 field on [HQ ATO Tax Years](/en/vista/vista/administration/headquarters/australian-setup/hq-ato-tax-years).
Accept the default, or enter the reporting date. If you change the information here, the system will not update HQ ATO Tax Years.

## Close/Open Tax Year

To lock the current ETP tax year, and prevent
 any unintentional updates to the Employee ETP By Date tab, select ETP Tax Year
 Closed.
To open the ETP tax year and change data on
 the Employee ETP By Date tab, select ETP Tax Year Open.

## Employee

Required.
Populated when employee amounts are generated with [PR ETP Generate Amounts](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/payroll-australia-payg-forms/pr-etp-generate-amounts-form).
You can also manually enter a valid employee
 number, or press F4 for a list of employee numbers.

## Date of Payment

Required.
Populated when employee amounts are generated with [PR ETP Generate Amounts](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/payroll-australia-payg-forms/pr-etp-generate-amounts-form).
You can also manually enter a date of payment made to this employee as a result of being terminated during the current year. More than one date may be entered per employee, but there can be no more than one date of payment per pay period.

## Given Name

This field defaults from the First field in
 the Name section of [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## Middle Name

This field defaults from the Middle field in
 the Name section of [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## Surname

This field defaults from the Last field in
 the Name section of [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## Address

This field defaults from the Address field
 on [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## City

This field defaults from the City field on
 [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## State

This field defaults from the State field for
 this employee in [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default in this field by
 entering a valid state or pressing F4 for a list. However, edits in this
 field will not update PR Employees.

## Post Code

This field defaults from the Postal Code
 field on [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## Date of Birth

This field defaults from the Birth Date
 field on [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default text in this field, but it will not update PR Employees.

## Tax File #

Required.
This field defaults from the TFN field on
 [PR
 Employees](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employees-form).
You can change the default number in this field, but it will not update PR Employees.

## Tax Year

This field defaults to the latest tax year for existing records in the [PR ETP Summary Process](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/payroll-australia-payg-forms/pr-etp-summary-process-form) form.
To view ETP details for a previously set up
 tax year, enter the year in this field or press F4 for a list of tax years.
To create a new tax year, you must first set it up in [HQ ATO Tax Years](/en/vista/vista/administration/headquarters/australian-setup/hq-ato-tax-years).
