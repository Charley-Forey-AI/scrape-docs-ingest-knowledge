# PR ETP/PAYG E-File Unset/Set Form

Use the PR ETP/PAYG E-File Unset/Set form to set or unset the Include in E-File checkboxes for all employees listed in PR ETP Employee Amounts and/or PR Employee PAYG Summaries for the applicable tax year.
You can access this form from either PR ETP Summary Process or PR PAYG Summary Process by selectingTasks > Unset/Set 'Include in E-File' checkboxes.
The Include in E-File checkbox in the PR ETP Employee Amounts form (for terminated employees) and the PR Employee PAYG Summaries form controls whether an employee's payment records will be included in the ATO (Australian Taxation Office) e-file. When you generate summaries/amounts, the system automatically sets the Include in E-File checkbox to selected and the E-File Amendment Indicator to O-Original for all employees/amounts.
If you are preparing to submit original payment summaries, you should not need to change these settings. However, if you have made manual changes to individual summary records and want to reset all employees/summaries, you can use the Set All "Include in E-File" option in this form to automatically select this checkbox for all employees. In the case of PAYG employee/summaries, it will select the checkbox for each summary. In the case of ETP employee/amounts, it will select the checkbox for each DL code entry and also reset the E-File Amendment Indicator for all entries to the default value of O-Original.
If you are preparing to lodge an amended report, it is strongly recommended that you do not use the "set all" option, since the system does not distinguish between Original and Amended summaries. Instead, use the Unset All "Include in E-File" option to deselect the Include in E-File checkbox for all items, and then manually select the checkbox for each item and set the E-File Amendment Indicator to A-Amended.
Note: If you have original summaries that you have not sent to the ATO, you can include them with amended summaries. However, you will need to manually check the Include in E-File checkbox for those summaries as well.
Important: This setting impacts the records of only those employees to which you have been given access through Employee Data Type security and PR Group security.
