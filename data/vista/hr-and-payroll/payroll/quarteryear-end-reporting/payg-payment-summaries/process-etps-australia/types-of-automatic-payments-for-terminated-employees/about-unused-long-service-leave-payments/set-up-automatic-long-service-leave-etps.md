# Set Up Automatic Long Service Leave ETPs

You can use the PR Auto Termination Pay Setup form to create automatic long service leave ETP records.
When you set up a record in this form, the system automatically creates timecard records (during payroll processing) for employees who qualify for a long service leave ETP.To set up automatic long service leave ETPs:

1. In the PR Auto Termination Pay Setup form, enter N, New, or + in the Seq field to generate the next available number.

1. In the ETP/Leave Earn Code field, enter the earnings code for long service leave (accrual after 17 Aug 1993).

1. In the Pre18Aug93 Leave Earn Code field, enter the earnings code for long service leave (accrual prior to 18 Aug 1993).

1. In the Redund ETP/Leave Earn Code field, enter the earnings code for long service leave due to redundancy.

1. In the Leave Code field, enter the leave code associated with this type of long service leave.

1. Save the record.Note: If you enter multiple long service leave ETP type records, and the ETP/Leave Earn Code and Pre18Aug93 Leave Earn Code fields match for all records, the system combines the long service leave balances. If you want each type to be processed separately, you must create a separate earnings code for each long service leave type.

During payroll processing, the system will now generate a timecard record for all applicable earnings codes for each employee eligible for an ETP.
In order for payroll processing to include all applicable employees, you must set each applicable employee as eligible for an ETP. See [Process ETPs: Australia](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia).
