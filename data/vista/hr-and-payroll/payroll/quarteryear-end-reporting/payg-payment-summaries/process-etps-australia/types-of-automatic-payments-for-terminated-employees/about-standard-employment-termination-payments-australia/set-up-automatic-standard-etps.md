# Set Up Automatic Standard ETPs

Use the PR Auto Termination Pay Setup form to create standard ETP type records.
Standard ETP payment types include in lieu of notice, unused personal leave, and unused rostered days off (RDO).When you set up a record in this form, the system automatically creates timecard records (during payroll processing) for employees who qualify for a standard ETP.
To set up automatic standard ETPs:

1. In the PR Auto Termination Pay Setup form, enter N, New, or + in the Seq field to generate the next available number.

1. Select the standard ETP payment type from the Termination Payment Type drop-down:

- Select LN - In Lieu of Notice if this is an in lieu of notice ETP.

- Select PL - Personal Leave if this is an ETP for unused personal leave.

- Select RDO - Rostered Day Off if this is an ETP for unused RDO.

1. In the ETP/Leave Earn Code field, enter the earnings code for the ETP type.

1. In the Redund ETP/Leave Earn Code field, enter the earnings code for a standard redundancy ETP.

1. For Personal Leave ETP types, enter a leave code in the Leave Code field.

1. Save the record.

During payroll processing, the system will now generate a timecard record for all applicable earnings codes for each employee eligible for an ETP.

In order for payroll processing to include all applicable employees, you must set each applicable employee as eligible for an ETP. See [Process ETPs: Australia](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia).
