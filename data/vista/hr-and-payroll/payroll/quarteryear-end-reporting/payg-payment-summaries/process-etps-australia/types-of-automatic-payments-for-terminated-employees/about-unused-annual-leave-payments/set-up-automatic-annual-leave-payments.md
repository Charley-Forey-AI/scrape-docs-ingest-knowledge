# Set Up Automatic Annual Leave Payments

Use the PR Auto Termination Pay Setup form to create ETP type records.
When you set up a record in this form, the system automatically creates timecard records for employees who qualify for an annual leave ETP.To set up automatic annual leave ETP payments:

1. In the PR Auto Termination Pay Setup form, enter N, New, or + in the Seq field to generate the next available number.

1. From the Termination Payment Type drop-down, select AL - Annual Leave.

1. In the ETP/Leave Earn Code field, enter the earnings code for normal termination. For more information, see [Creating Earnings and Deduction Codes for Annual Leave - Normal Termination](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments/create-earnings--deduction-codes-for-annual-leave-normal-termination).

1. In the Redund ETP/Leave Earn Code field, enter the earnings code for termination due to redundancy, invalidity, or early retirement. For more information, see [Creating Earnings and Deduction Codes for Annual Leave - Termination Due to Redundancy, Invalidity, or Early Retirement](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments/create-earnings--deduction-codes-for-annual-leave-termination-due-to-redundancy-invalidity-or-early-retirement).

1. In the Leave Code field, enter the leave code associated with this type of annual leave. For more information, see [Creating Leave Codes](/en/vista/vista/hr-and-payroll/payroll/leaveseparation/leave/about-creating-leave-codes).

1. Save the record.

1. Create a new record by entering a new number in the Seq field.

1. From the Termination Payment Type drop-down, select ALL - Annual Leave Load.

1. In the ETP/Leave Earn Code field, enter the leave loading earnings code for normal termination. For more information, see [Creating Earnings and Deduction Codes for Annual Leave - Normal Termination](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments/create-earnings--deduction-codes-for-annual-leave-normal-termination).

1. In the Redund ETP/Leave Earn Code field, enter the leave loading earnings code for termination due to redundancy, invalidity, or early retirement. For more information, see [Creating Earnings and Deduction Codes for Annual Leave - Termination Due to Redundancy, Invalidity, or Early Retirement](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/types-of-automatic-payments-for-terminated-employees/about-unused-annual-leave-payments/create-earnings--deduction-codes-for-annual-leave-termination-due-to-redundancy-invalidity-or-early-retirement).

1. Save the record.The system will now generate a timecard record for all applicable earnings codes for each employee that is set as eligible for an unused annual leave payment. For more information, see [Generating Employee ETPs](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/process-etps-australia/generate-payments-for-terminated-employees).
