# Types of Automatic Payments for Terminated Employees

You can set up and process a number of payment types for terminated
 employees.
Vista allows you to set up and process a number of payment types for terminated employees. These types can be broken into two categories:

- payments that can be generated automatically and

- payments that you must enter manually.

The following figure displays the payment types and their associated category.
Note: The table items in gray are referred to as standard ETPs. The standard designation is used only to group ETP types together that are processed and taxed in a similar fashion.
