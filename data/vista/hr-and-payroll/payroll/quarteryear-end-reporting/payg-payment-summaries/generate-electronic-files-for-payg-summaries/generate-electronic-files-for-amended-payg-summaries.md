# Generate Electronic Files for Amended PAYG Summaries

Use the PR PAYG Summary Process and PR ETP Summary Process forms to generate electronic files (e-files) for submission of amended PAYG summaries.
Note: Prior to generating electronic files, make sure you have already reviewed and/or edited PAYG items and amounts as necessary. If you have not, you should do so before proceeding here. For more information, see [Review/Update PAYG Information for Employees](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/prepare-the-payg-payment-summary---individual-non-business/reviewupdate-payg-information-for-employees) and [PR ETP Employee Amounts](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/payroll-australia-payg-forms/pr-etp-employee-amounts-form). You can also print the PR PAYG Payment Summary - Individual Non-Business and the PR PAYG Payment Summary - Employee Termination Payment to make sure that all of the necessary information has been entered and is correct.

 The file you create will include both amended PAYG Individual Non-Business (INB) summaries and PAYG Employment Termination Payment (ETP) summaries as applicable.To generate an electronic file for amended INB and ETP summaries:

1. Open the PR PAYG Summary Process or PR ETP Summary Process form.

1. In the Tax Year field, enter the tax year for generating the file or press F4 to select from a list of valid tax years.

1. From the toolbar, select Tasks > Unset/Set 'Include in E-File' checkboxes.The PR ETP/PAYG E-File Unset/Set form displays.

1. In both the ETP Summary and PAYG Individual Non-Business Summary sections, select Unset All "Include in E-File".

1. Click Update E-File.The system clears the Include in E-File checkbox for all employees/summaries (employed and terminated).

1. Click Cancel to close the PR ETP/PAYG E-File Unset/Set form.

1. For amended INB summaries, update the E-File settings for each applicable employee.

1. In PR PAYG Summary Process, select on the Employees tab.

1. Select the employee whose summaries you are amending and double-click in the grid to open PR Employee PAYG Summaries.

1. On the PAYG INB Summaries tab, for each applicable summary, select the Include in E-File checkbox and set the E-File Amendment Indicator to A-Amended.

1.  Repeat Steps b and c for each applicable employee.

1. For amended ETP summaries, update the E-File settings for each applicable employee:

1. In PR ETP Summary Process, select on the Employee ETP by Date tab.

1. Select the employee whose summaries you are amending and double-click in the grid to open PR ETP Employee Amounts.

1. In the Date of Payment field, select the applicable payment date.

1. In the Employee ETP Amounts by DL Code grid, select the Include in E-File checkbox and set the E-File Amendment Indicator to A-Amended for each applicable payment code/DL code.

1. Repeat Steps b-d for each applicable employee.

1. Return to the PR PAYG Summary Process or PR ETP Summary Process form.

1. From the toolbar, select Tasks > Generate PAYG Annual Report E-File.The system displays the Save Export File As window.

1. Save the text file to a location on your network.The system saves the file and a message displays informing you of the number of records that were processed.

1. Click Close. The system returns to the PR PAYG Summary Process or PR ETP Summary Process form.

1. You can now lodge the electronic file with the ATO.

For more information about how to lodge your PAYG withholding annual reports, see the Australian Taxation Office website ([www.ato.gov.au](http://www.ato.gov.au/)).
