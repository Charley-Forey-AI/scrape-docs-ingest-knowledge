# Field Definitions: PR Aatrix ACA Print eFile Form

The following is a list of field descriptions for the PR
 Aatrix ACA print eFile form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Tax Year

The Tax Year field in the PR Aatrix - ACA Print and eFile form.
Enter the tax year. Defaults the tax year selected in the PR ACA Process form.

## Form Type

The Form Type selection box on the PR Aatrix - ACA Print and eFile form.
Select the form type to process.

- ACA 1095/1094-B - Select to process 1095/1094-B Health Coverage Information Returns

- ACA 1095/1094-C - Select to process 1095/1094-C Employer Provided Health Insurance Offer and Coverage Information Returns

- Electronic 1095 Consent - Select to print an electronic consent form for each of your employees for the 1095 B or C.
