# Test ACA Transmittal Data for Accuracy in Aatrix

You can run the 1094/1095 process in test mode in Aatrix to ensure your transmittal data is accurate and formatted correctly before you begin filing for the current year.
When you are in test mode, no payroll information is saved or transmitted to the Aatrix eFile center. You can only use test mode to see how the process works and to make sure your data is accurate before you print or eFile your 1094s and 1095s.

1. In PR ACA Process, select Tasks > Launch Aatrix Print and eFile.Important: If you haven't yet installed Aatrix on your local workstation, a dialog box appears. Select the provided link to begin the process. Once installation is complete, select Tasks > Launch Aatrix Print and eFile again.

The PR Aatrix - ACA Print and eFile form opens.

1. In the Tax Year field, enter the tax year.

1. From the Form Type selection box, select the form type to test.

- 1095/1094-B

- 1095/1094-C

1. Select Continue.

1. In the ACA Setup Wizard, select one of the following:

- I want to test drive the 1095-C/1094-C Wizard with only the first 25 of my employees

- I want to test drive the 1095-C/1094-C Wizard with all of my employees

1. Select Next and follow the prompts in the Aatrix application.
