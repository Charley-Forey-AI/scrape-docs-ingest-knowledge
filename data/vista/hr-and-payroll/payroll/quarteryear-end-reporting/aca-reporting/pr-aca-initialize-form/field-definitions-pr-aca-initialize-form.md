# Field Definitions: PR ACA Initialize Form

The following is a list of field descriptions for the PR ACA
 Initialize form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Tax Year

Tax Year field on the PR ACA Initialize form
The 4-digit tax year is determined by what is
 selected in the Tax
 Year field of the [PR ACA Process](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-process-form) form.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Overwrite Existing Date

Overwrite Existing Data checkbox on the PR ACA Initialize form
Select this checkbox to delete and overwrite existing data.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Initialize From

Initialize From section on the PR ACA Initialize form
Select to initialize from Payroll or Human Resources.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## HRCo

HRCo field on the PR ACA Initialize form
Enter an HR company, or press F4 to select
 from a list. Press F5 to open the [HR Company Parameters](/en/vista/vista/hr-and-payroll/human-resources/setupmaintenance/about-the-hr-company-parameters-form) form.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Include Dependents

Include Dependents checkbox on the PR ACA Initialize form
Select this checkbox if:

- You selected the Human
 Resources option in the Initialize From section of the form, and

- The offered coverage is self-insured and the
 Self-Insured checkbox is selected on in the ACA section of the Info
 tab in HR Resource Benefits form.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Employees to Initialize

Employees to Initialize section on the PR ACA Initialize form
Select to initialize by full-time or all employees. Full-time employees are defined as those working 30 hours per week or 130 per month as recorded in posted timecards with earnings codes set up to track ACA hours.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## MEC Offer

MEC Offer checkbox on the PR ACA Initialize form
Select this checkbox if you want the system to check the MEC Offer Indicator for all 12 months on the 1094-C.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Aggregated Group

Aggregated Group checkbox on the PR ACA Initialize form.
Select this checkbox if you want the system to check the Aggregated Group Indicator for all 12 months.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Offer Code

Offer Code field on the PR ACA
 Initialize form.
If you are initializing from Payroll, select
 an IRS 1095-C series one offer code from the Offer Code list to designate the coverage
 offer.
If you are initializing from Human Resources,
 ignore this field. The data will be pulled automatically from the ACA History tab in HR
 Resources.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## Self Cost Only

Self Cost Only field on the PR ACA Initialize form
If you offer self-only coverage, enter the employee share of the cost for that coverage. Leave the field blank if you do not offer self-only coverage.

Related information

- [PR ACA Initialize Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/aca-reporting/pr-aca-initialize-form)

## 4980H Safe Harbor Code

4980H Safe Harbor Code field on the PR ACA Initialize form
Select an IRS 4980H Safe Harbor code.

## Plan Start Month

Plan Start Month on the PR ACA Initialize form.
From the drop-down menu, select the calendar month (01- January thru 12 - December) in which the plan year begins. If more than one plan year could apply (e.g. the ALE Member changes the plan year during the year), select the earliest applicable month.
If no health plan coverage was offered to the
 employee, select 00
 - No Health Plan Offered.
Note: The start month specified here will default for all initialized employees. You may override as needed for each employee in PR ACA 1095-C Employee.
