# Assign Add-on Earnings to a Craft

You can assign add-on earnings to a craft using the Add-on
 Earnings tab in PR Crafts.
Add-on earnings represent earnings that are added to
 an employee's total whenever he works in this craft. Typical add-on earnings are
 added to the classification's pay rate, but need to be reported separately (for
 example, union vacation or dues). This may also be used for fringes paid directly to
 non-union employees for prevailing wage jobs. Add-ons from PR Crafts are always
 applied, but rates can be overridden at the class, craft template, and/or craft
 class template levels.
The following instructions detail how to assign add-on earnings to a craft.

1. In PR Crafts, select the
 Add-on Earnings tab.

1. In the Earn
 Code field, enter an earnings code or press F4
 to select from a list of earnings codes. The system defaults values in the
 Description and Method fields.

1. If the calculation
 method is V (variable), in the Factor field, enter the earnings factor that
 applies to this craft. For all other calculation methods, the factor is set to
 0.000000 and cannot be changed. In the Old Rate field, enter the old
 rate for the earnings code. The system will use this rate for timecards entered
 prior to the date specified in the Effective Date field (Info
 tab). In the New Rate field, enter the new
 rate for the earnings code. The system will use this rate for timecards entered
 on or after the date specified in the Effective Date field (Info
 tab).

1. Save the record.
