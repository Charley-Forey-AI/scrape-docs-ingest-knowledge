# About the PR Craft Classes Form

Class codes identify the classifications of work types within
 a craft.
Use this form to define the pay rates and variable
 earnings, as well as overrides to add-on earnings and deduction (including pre-tax) and
 liability rates for each class. This form can be accessed from the Main Menu and also by
 double-clicking a record on the Classes tab in PR Crafts.
