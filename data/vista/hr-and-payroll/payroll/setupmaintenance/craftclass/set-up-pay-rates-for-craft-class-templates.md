# Set Up Pay Rates for Craft Class Templates

You can use the Pay Rates tab in PR Craft Class Templates to
 override pay rates for the craft class.
You can set up pay rates for different shifts, as well as determine old and new pay rates.
Important: Pay rates defined at the craft template, craft/class, or craft levels (as applicable) do not need to be set up here unless you are overriding the existing rate(s). If you do not override an existing rate here, the rate defined at the next hierarchical level will be used (i.e. craft template rates override craft class rates). If you want a specific rate excluded from calculations, set it up here with a 0.00 amount. (Note: Requiring only override rates to be defined here eliminates having to update multiple levels each time a rate change occurs.)
The following instructions detail how to set up pay rates for craft class templates.

1. Select the Pay Rates tab in PR Craft Class Templates.

1. Enter a shift number in
 the Shift field.

1. Enter rates in the
 Old
 Rate and New Rate fields. The system
 uses the effective date specified in PR Crafts or PR Craft Templates to
 determine which limit is used. The old limit is used for timecards entered prior
 to the effective date and the new limit for timecards entered on or after the
 effective date.

1. Save the record as normal.
