# Copying an Existing Template

The following instruction details how to copy an existing craft template.

1. In PR Template Copy, select the
 Template
 radio button in the Template Copy Source section of the form. The system enables the
 Template
 field.

1. Enter the template you are copying from in the
 Template
 field.

1. You can copy all crafts/classes defined for the template in PR Craft Templates and PR Craft Class Templates or you can select a specific craft to copy craft/class information from. Select the appropriate option:
 All Crafts
 or
 Selected Craft
 .

1. If you are copying from a selected craft, enter the craft in the
 Craft
 field. Press
 F4
 for a list of crafts.

1. In the Include section of the form, check the boxes for the type of class/craft information that you want copied to the new template. Options include:
 Pay Rates
 ,
 Addon Earnings
 ,
 Job Craft Items
 ,
 Variable Earnings
 ,
 Dends/Liabs
 , and
 Notes
 .

1. Enter the template you are copying to in the
 Template
 field.

1. Enter a description for the template in the
 Description
 field.Note: You can select an existing template to copy new information to. In that case, the system will not overwrite any information on this template, but will add any data that did not exist previously. Additionally, the
 Description
 field will default automatically and you will be unable to make changes.

1. Click
 Copy
 . The system copies information to the new template.
