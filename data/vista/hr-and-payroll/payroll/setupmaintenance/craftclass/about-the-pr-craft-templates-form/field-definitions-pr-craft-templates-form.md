# Field Definitions: PR Craft Templates Form

The following is a list of field descriptions for the PR Craft Templates form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Craft

 Enter a craft that has already been set up in
 the PR Crafts.

##  Template

Enter a template that has already been set up
 in the PR Templates.

##  Override Effective Date

 Check this box if you want to override the
 standard Craft’s date. If overridden, all timecards posted to this Craft and job using this
 template will use the Effective Date entered on this form to determine rates.
 Do not check this box if using the Effective
 Date set up for this craft in PR Crafts.

##  Effective Date

 Enter the effective date of rate changes if you checked the Override Effective Date box.

##  Override Overtime

 Check this box if a different set of daily overtime “rules” apply to jobs covered by this template.
 Do not check this box if using the standard overtime rules for all jobs covered by this template.

##  OT Schedule

 Enter a valid overtime schedule if you checked the Override Overtime box.

## Type

Type drop-down on the PR Craft Templates form,
 Capped Basis tab.
From the drop-down list, select the capped basis code type.

-  E-Earnings

- L-Liability

## Earn/Liab Code

Earn/Liab Code field on the PR Craft Templates
 form, Capped Basis tab.
If you selected E-Earnings from the Type drop-down,
 enter a valid earnings code or press F4 to select from a list of valid
 earnings codes. Earnings code must have a method of H-Rate per hour (in PR Earnings
 Codes).
If you selected L-Liability from the
 Type
 drop-down, enter a valid liability code or press F4 to select from a list of valid
 liability codes. Liability code must have a method of H-Rate per hour or A-Amount AND a
 calculation category of C-Craft, E-Employee, or A-Any (in PR Deductions/Liabilities).
