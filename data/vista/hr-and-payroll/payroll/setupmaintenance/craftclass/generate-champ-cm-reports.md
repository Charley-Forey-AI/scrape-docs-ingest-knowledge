# Generate Champ-CM Reports

To generate the export files in the PR Generate Champ-CM Exports form, click Employee Export or Work Hour Export; the “Save Export As” window displays. Save the file as either a *.CSV (Comma separated) or *.txt (Tab separated) file type and click the Save button. Once completed, a message displays notifying you of how many records were processed.
