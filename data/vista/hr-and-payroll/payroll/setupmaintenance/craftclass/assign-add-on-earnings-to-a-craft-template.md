# Assign Add-on Earnings to a Craft Template

You can assign add-on earnings to a craft template using the
 Add-on Earnings tab in PR Craft Templates.
 You can override add-on earnings specified in PR
 Crafts or PR Craft Classes by entering them here and adjusting information as
 necessary. You can also specify additional add-on earnings that are required by this
 template. The system will use add-ons set up here unless overridden in PR Craft
 Class Template.
Important: Add-ons defined at the craft or class level do not need to be set up in PR Craft Templates unless you are overriding the existing rate(s). If you do not override an existing rate here, the rate defined at the class or craft level will be used. If you want a specific rate excluded from calculations, set it up here with a 0.00 amount. Requiring only override rates to be defined here eliminates having to update multiple levels each time a rate change occurs.
The following instructions detail how to assign add-on earnings to a craft template.

1. In PR Craft Templates,
 select the Add-on Earnings tab.

1. Enter an earnings code
 in the Earn Code field. Press F4 to select from a list of
 earnings codes. The system defaults values in the Description and Method fields.

1. If the calculation
 method (Method field) is variable (V), enter the earnings factor that
 applies to this craft template in the Factor field. For all other
 calculation methods, the factor is set to 0.000000 and cannot be changed.

1. Enter the old rate for
 this earnings code in the Old Rate field. The system
 uses this rate for timecards entered prior to the specified effective date in PR
 Crafts or PR Craft Template. You can override this rate at the craft class
 template level.

1. Enter the new rate for
 this earnings code in the New Rate field. The system
 uses this rate for timecards entered on or after the date specified in PR Crafts
 or PR Craft Template. You can override this rate at the craft class template
 level.

1. Save the record as normal.
