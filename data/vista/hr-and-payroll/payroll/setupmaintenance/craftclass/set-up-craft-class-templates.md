# Set Up Craft Class Templates

You can use the PR Craft Class Templates form to set up the
 most detailed information for determining pay rates, add-on earnings, deductions (including
 pre-tax) and liabilities for crafts/classes.
The following instructions detail how to set up craft class templates.

1. Open the PR Craft Class Templates form from one of the following locations:

- From the Main Menu

- From PR Craft Classes, by double clicking in the
 Craft Class Templates grid

- From PR Craft Templates, by double-clicking in the
 Craft Class Templates grid

1. Enter the Craft, Class, and/or Template values as applicable:

If you opened this form via the Main Menu:

1. In the Craft field, enter a
 valid craft or press F4 to select from a
 list of crafts. Craft must already be associated with a craft template
 in PR Craft Templates.

1. In the Class field, enter a
 class or press F4 to select from a
 list of valid classes for the craft.

1. In the Template field, enter
 the template or press F4 to select from a
 list of valid templates. This must be a template that has already been
 set up for the specified craft in PR Craft Templates.

If you opened this form via PR Craft Classes:

- The Craft and Class fields display the craft and class selected in PR
 Craft Classes and are disabled.

- In the Template
 field, enter the template or press F4 to select from a list of valid templates.
 Template must be set up in PR Templates and for the craft in PR
 Craft Template.

If you opened this form via PR Craft Templates:

- The Craft and
 Template fields display the craft and template selected
 in PR Craft Templates and are disabled.

- In the Class field,
 enter a class or press F4 for a list
 of valid classes for the craft.

1. To override the
 Prevailing Wage values for the craft/class, select the Override Class
 Prevailing Wage checkbox. This will enable the related
 prevailing wage fields. See [Step 5](#ID-0003203d--en__ID-00032073)
 below. If Prevailing Wage values for
 the specified craft/class do not differ by job, leave this checkbox
 unselected. The system will use the Prevailing Wage values defined for the
 craft class in PR Craft Classes.
Note: This checkbox is disabled if Capped
 Codes are not set up for the Craft (in PR Crafts).

1. For Australia users: If
 you pay a weekly minimum for superannuation for this class, in the Superannuation
 Weekly Minimum field, enter the weekly amount. For more
 information, see [Setting Weekly Minimums for Superannuation](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/superannuation/set-up-superannuation/set-weekly-minimums-for-superannuation).


1. If you
 selected the Override Class Prevailing
 Wage checkbox in Step 3, enter the prevailing wage
 information.

1. From the
 Calculation Method
 drop-down, select the calculation method for the craft/class/template.
 Options include: Standard, Cash Fringe Factored, Generous Fringe Benefit
 Pkg #1, Generous Fringe Benefit Pkg #2, and Constant Fringe Benefit Rate. For more information
 about each of these options, see [Calculation Method](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/craftclass/about-the-pr-craft-classes-form/field-definitions-pr-craft-classes-form#ID-00035fdb--en).

1. Enter hourly,
 benefit, or total wage rates as applicable:

If you selected the Cash Fringe Factored calculation method:

1. In the Old Total Wage Rate
 and New Total Wage Rate
 fields, enter the old and new total prevailing wage rate for this
 craft/class/template. The system will determine which rate to use based
 on the Effective Date specified for the craft in PR Crafts.

1. Save the record.

For all other calculation methods:

1. In the Old Basic Hourly Rate
 and New Basic Hourly Rate
 fields, enter the old and new basic hourly rate for the
 craft/class/template. The system will determine which rate to use based
 on the Effective Date specified for the craft in PR Crafts.

1. In the Old Fringe Benefit
 Rate and New Fringe Benefit
 Rate fields, enter the old and new fringe benefit rate
 for the craft/class/template. The system will determine which rate to
 use based on the Effective Date specified for the craft in PR
 Crafts.

1. Save the record.

1. [Set up pay rates for the class](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/craftclass/set-up-pay-rates-for-craft-class-templates).

1. [Set up variable earnings for the class](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/craftclass/set-up-variable-earnings-for-craft-class-templates).

1. [Set up add-on earnings for the class](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/craftclass/assign-add-on-earnings-to-craft-class-templates).

1. [Set up deductions/liabilities for the class](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/craftclass/assign-deductions-and-liabilities-to-craft-class-templates).

1. [Set up allowance earnings](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/craftclass/assign-allowances-to-craft-class-templates).
