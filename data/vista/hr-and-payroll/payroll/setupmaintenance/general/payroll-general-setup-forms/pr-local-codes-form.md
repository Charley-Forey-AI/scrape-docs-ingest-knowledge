# PR Local Codes Form

Use this form to define income tax and other deduction and liability information associated with a local code.
Note: Do not use this form to set up local codes for Ohio School District tax. See the [Tax and Worker's Comp Routines](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/about-payroll-routines/deduction-and-liability-based-routines-united-states/north-carolina-philadelphia) table for information on how to set up those codes.
