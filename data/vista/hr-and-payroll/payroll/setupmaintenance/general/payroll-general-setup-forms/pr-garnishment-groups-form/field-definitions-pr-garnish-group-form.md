# Field Definitions: PR Garnish Group Form

The following is a list of field descriptions for the PR
 Garnish Group form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Garnishment Group

 Enter a number (0-255) that will be used to identify this garnishment group of earnings and deductions, which are used as a basis for calculating garnishment deductions.

##  Description

 Enter a description for this garnishment group, up to 30 characters.

##  Type

 Select either “E” for earnings or “D” for deduction.

## Earnings/Deduction Code

Enter a valid earnings or deduction code. The description displays in the ED Code Desc field.
