# JC/EM Fixed Rate Accounts

There are GL Accounts that you need to specify for posting
 payroll to the General Ledger if you will be using the "Fixed Rate" options (in PR
 Employees) for posting time to jobs or equipment.

- JC Fixed Rate GL Account — This
 account is credited when a fixed rate (as set up by Employee in PR Employees) is
 used to calculate an Employee’s labor and burden for JC. Offsets expenses
 debited to the JC Department’s Labor Accounts assigned by Earnings Type.

- EM Fixed Rate GL Account — This
 account is credited when a fixed rate (as set up by Employee in PR Employees) is
 used to calculate an Employee’s labor and burden for EM. Offsets expenses
 debited to the EM Department’s Labor Account.

- SM Fixed Rate GL Account -
 This account is credited when a fixed rate (as set up by Employee in PR
 Employees) is used to calculate an Employee’s labor and burden for SM work
 orders. Offsets expenses debited to the SM Department’s standard labor account
 or override labor account (if overrides are defined by call type and/or cost
 type).
