# Earnings Types

This tab allows you specify the Earnings Expense account to
 debit when earnings are posted to this Department.
Specific earnings are grouped by type in PR
 Earnings Codes and PR Deductions/Liabilities; therefore, you must specify the GL account
 to use for each of the earnings types (defined in HQ) that will be used when earnings
 are posted.
In addition, for each earnings type, you
 must specify the “JC Applied Earnings”, “EM Applied Earnings”, "SM Applied Earnings",
 and “Interco Applied Earnings” accounts. These accounts will be used to offset (credit)
 job, equipment, work order, and intercompany earnings (respectively).
