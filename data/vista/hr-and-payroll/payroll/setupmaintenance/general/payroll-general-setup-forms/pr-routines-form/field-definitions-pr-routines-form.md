# Field Definitions: PR Routines Form

The following is a list of field descriptions for the PR
 Routines form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Routine

The Description field on the PR Routines form, Info tab.
For user defined routines, enter the routine name up to 10 characters.
For more information on existing routines, see [Routines](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/about-payroll-routines).

##  Description

The Description field on the PR Routines form, Info tab.
This field defaults the description provided during
 initialization; you can override the default as necessary.
When creating a new routine, enter a description of the routine, up to 30 characters.

##  Procedure Name

The Procedure Name field in PR Routines, Info tab.
For standard Vista routines, the procedure name defaults during routine initialization and identifies the SQL stored procedure that performs the calculation for the selected routine. Do not change this default.
 The procedure updates to the most current stored procedure after you load ta release that contains routine updates (for example, tax updates).
For user-defined tax routines (for example, Exempt
 Rate of Gross for applicable locals) or earnings routines, press F4 to select a
 procedure to use for the routine.
For more information on existing routines, see [Routines](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/about-payroll-routines).

##  Last Updated

The Last Updated field on the PR Routines form, Info tab.
This field displays the date this routine was last initialized.
Note: Be aware that when you initialize existing routines or add new ones, this date is updated for all routines, whether the routine has changed or not. For more information, see [Initializing Routines](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/about-payroll-routines/initialize-routines).
When creating a new earnings routine for Australia, this field is optional; however, consider entering a creation date for informational purposes.

## Misc Amt #1, #2, #3, #4

The Misc Amt #1, #2, #3, and #4 fields on the PR Routines form.
The amounts that you enter here are dependent upon the setup information for the routine. Use the routine tables to locate the setup information for the routines you use.
