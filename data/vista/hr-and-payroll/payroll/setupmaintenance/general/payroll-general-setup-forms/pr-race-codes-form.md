# PR Race Codes Form

Use this form to set up Race Codes that are used for reporting purposes.
The system validates these codes against values in the PR Employees and HR Resources forms when you enter the employee's race.
Important: If you are using the PR Certified Export - eMars, ReportID 1315 (U.S. only), set up one-character values only to insure that your export file is compatible with the eMars system.
Although you can choose your own number and/or letter and enter your own descriptions for each race code, you must make a selection from the EEO Category drop down list; these categories are required for the PR Department of Labor EEO Report and the PR Department of Transportation Report.
See the F1 field help for the EEO Categories field for a list of valid entries.
