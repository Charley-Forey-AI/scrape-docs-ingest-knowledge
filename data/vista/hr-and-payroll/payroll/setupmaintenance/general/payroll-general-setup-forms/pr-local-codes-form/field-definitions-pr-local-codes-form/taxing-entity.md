# Taxing Entity

Enter the taxing entity code as defined by the local
 agency, up to 10 characters. The code that you enter here will default in the Tax Entity
 field in PR W-2 Employee Edit (Local Info tab) when you initialize W-2 information for
 the specified tax year in PR W-2 Process.
For the State of Pennsylvania: Enter “MATCB” for employees who are a resident of the Middletown jurisdiction.
For the state of Ohio: Enter the 5 digit taxing entity code. If you enter more than 5 characters, the system will truncate the code to 5 digits when you generate the local W-2 electronic file.
