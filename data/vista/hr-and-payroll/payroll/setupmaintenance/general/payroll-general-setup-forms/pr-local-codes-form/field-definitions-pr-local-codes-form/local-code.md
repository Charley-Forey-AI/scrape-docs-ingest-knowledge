# Local Code

Enter a code, up to 10 characters, that will be used to identify a city, county, or other local taxing district.
Note: Do not use this form to set up local codes for Ohio School District tax. See the Tax and Worker's Comp Routines table for information on how to set up those codes (click [here](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/about-payroll-routines/deduction-and-liability-based-routines-united-states/north-carolina-philadelphia)).
