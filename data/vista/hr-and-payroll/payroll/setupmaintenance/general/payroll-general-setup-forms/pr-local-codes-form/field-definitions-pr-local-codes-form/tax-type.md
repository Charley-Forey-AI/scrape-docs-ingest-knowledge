# Tax Type

 Select the tax type for this local code. When W-2's are initialized, the system uses this tax type as the default tax type on the Local Info tab (PR W-2 Employee Information). The following options are available:

- C - City Income Tax

- D - County Income Tax

- E - School District Tax

- F - Other
