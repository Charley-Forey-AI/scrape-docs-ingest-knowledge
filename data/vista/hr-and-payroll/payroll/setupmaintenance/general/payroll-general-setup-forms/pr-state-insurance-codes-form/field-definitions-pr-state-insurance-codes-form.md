# Field Definitions: PR State Insurance Codes Form

The following is a list of field descriptions for the PR
 Satate Insurance Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## State

State field on the PR State Insurance Codes
 form.
 Enter the state to which Workers Compensation
 deductions and liabilities apply. Press F4 for a list of valid states.

##  Subject to Weekly Earnings Limit

Subject to Weekly Earnings Limit checkbox on
 the PR State Insurance Codes form, Info tab.
Select this checkbox if deductions and liabilities calculated for this insurance code are subject to a weekly wage limit.
Note: The limit is applied to each insurance code. Employees working on multiple insurance codes may have total deductions or liabilities for workmen’s compensation based on wages that exceed this limit.
 Do not select this checkbox if deductions and liabilities associated with this insurance code are not subject to a weekly earnings limit.

## Use Override Code When Pay Rate Exceeds Threshold Rate

Use Override Code When Pay Rate Exceeds
 Threshold Rate checkbox on the PR State Insurance Codes form, Info tab.
Select
 this checkbox to use an override insurance code when the employee's pay rate and
 applicable add-on earnings for a timecard are equal to or greater than the specified
 threshold rate. When selected, entry in the Threshold Rate and Override Insurance
 Code fields is required.
Do not select this checkbox if you do not want the insurance code to be overridden when entering timecards.

## Threshold Rate

Threshold Rate field on the PR State Insurance
 Codes form, Info tab.
Only used when the Use Override Code When Pay
 Rate Exceeds Threshold Rate checkbox is selected.
Enter the threshold rate for this insurance code. This rate is used to determine which insurance code to use when entering timecards or automatic earnings. If the employee's combined pay rate and applicable add-on earnings are equal to or greater than this threshold rate, the override insurance code (specified below) is used. If the employee's combined pay rate and applicable add-on earnings are less than the threshold rate, the insurance code specified for the employee (in PR Employees) is used.

## Override Insurance Code

Override Insurance Code field on the PR State
 Insurance Codes form, Info tab.
Enabled only when the Override Code When Pay Rate
 Exceeds Threshold Rate checkbox is selected.
 Enter the insurance code that will override
 the normally defaulted code or press F4 to select from a list of valid
 insurance codes.
Note: The system will only use this code if the employee's pay rate and applicable add-on earnings are equal to or greater than the Threshold Rate indicated for the specified state/insurance code. Applicable add-on earnings are those where the earnings code is included in the liability's basis codes, and where the calculation method is F-Factored Rate per Hour, H-Rate per Hour, or V-Variable Factored Rate.
