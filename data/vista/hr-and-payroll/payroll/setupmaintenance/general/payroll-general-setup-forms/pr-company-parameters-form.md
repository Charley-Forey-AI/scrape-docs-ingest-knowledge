# PR Company Parameters Form

Use this form to set up Payroll processing for each applicable company.
This form allows you to set up the default settings for timecard posting, interfacing, and auditing functions, as well as what reports to use for check print, EFTs, and pay stubs.
 If you have enabled timecard compliance (checkbox on the Info tab), you can use the Timecard Rules tab to enable validation rules and define their severity levels. You can also use it to access the PR Timecard Validation Rules form for further setup. For more information, see [PR Timecard Validation Rules Form](/en/vista/vista/hr-and-payroll/payroll/time-entry/trimble-smart-payroll/smart-payroll-forms/pr-timecard-validation-rules-form).
Note: If you leave the Enable Timecard Compliance checkbox unselected, you will be unable to access the Timecard Rules tab or the PR Timecard Validation Rules form (main menu).
