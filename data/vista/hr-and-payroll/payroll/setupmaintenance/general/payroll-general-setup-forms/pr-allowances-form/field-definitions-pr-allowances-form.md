# Field Definitions: PR Allowances Form

The following is a list of field descriptions for the PR
 Allowances form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Allowance

Allowance field on the PR Allowances form.
Enter a name for this payroll allowance type. Up to 16 characters allowed.

## Description

Description field on the PR Allowances
 form.
Enter a description for this payroll allowance. Up to 30 characters allowed.

## Rule Set Location

Rule Set Location drop-down on the PR
 Allowances form.
From the drop-down list, select the rule set location for this allowance.

- Craft Master

- Craft Template

- Craft Class

- Craft Class Template

The rule set location determines what
 craft/class/template level the system will use to locate the processing rules. Therefore,
 you can only add allowances to the specified craft/class/template level. For example, if
 you create an allowance with a Craft Class rule set location, you can only add that
 allowance to a Craft/Class in PR Craft Classes.
