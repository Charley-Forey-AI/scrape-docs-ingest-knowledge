# Field Definitions: PR Occupational Categories Form

The following is a list of field descriptions for the PR
 Occupational Categories form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Occup Category

The Occup Category field on the PR Occupational Categories form.
Enter Standard Occupational Classification (SOC) codes for the 10 major job categories,
 which you also must assign a [ Report Seq#](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-occupational-categories-form/field-definitions-pr-occupational-categories-form#ID-00039883--en).
In states with unemployment reporting:For states that require unemployment reporting (AK, LA, SC, WA, WV), you must
 also enter unique SOC codes. You can find a list of valid SOC codes on the
 [U.S. Bureau of Statics](https://www.bls.gov/) website. These values are
 used in the following Aatrix unemployment reports:

The Report Seq# and associated major job categories are used for EEO
 reporting on the PR Department of Labor EEO-1 Report.
In states with *no* unemployment reporting:The Occupational Categories and Report Seq# values are the same, used for EEO
 reporting on the PR Department of Labor EEO-1 Report.

##  Description

The Description field on the PR Occupational Categories form.
Enter a description for this occupational category, up to 30 characters.

##  Report Seq#

The Report Seq# field on the PR Occupational Categories form.
Enter the report sequence number (1-10) for this
 occupational category to indicate the job category. This number is used to control the
 order in which the data is printed, as well as the description of the job category on
 the PR Department of Labor EEO-1 report, as defined by the [Equal Employment Opportunity Commission (EEOC)](https://www.eeoc.gov/).
