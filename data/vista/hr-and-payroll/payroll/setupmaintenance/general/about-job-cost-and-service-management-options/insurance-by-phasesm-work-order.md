# Insurance By Phase/SM Work Order

The Insurance Based on Phase or SM Work Order
 Scope option controls the insurance code default when entering
 timecards.
If you check this option, timecards posted to a
 job will default the insurance code based on the job’s insurance template and phase.
 Timecards posted to a service work order will default the insurance code from the work
 order scope. If this option is left unchecked, then the employee’s standard insurance
 code is used.
