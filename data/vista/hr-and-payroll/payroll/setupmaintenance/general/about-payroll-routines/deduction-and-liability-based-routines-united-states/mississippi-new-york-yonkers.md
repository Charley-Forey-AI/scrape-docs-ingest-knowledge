Mississippi
MS Tax
bspPRMSTXX
PR Employee (Filing Status tab)
Use filing status M, S, or B
M = Married or head of family, only one spouse working
S = Single
B = Married, both working
Enter the exemption amount in Misc Amount #1.

Montana
MT Tax
bspPRMTTXX
PR Employees (Filing Status tab)
Use filing status of S, M, or H only.
S = Single, Married Filing Separately, or Married Filing Joint with Both Spouses Working
 M = Married Filing Jointly or Qualifying Widower
 H = Head of Household
Note: Use the information provided on the employee's Form MW4 (in Box 1 or Box 2) to determine the filing status.

NE Tax

New Jersey
NJ Tax
bspPRNJTXX
PR Employees (Filing Status tab)
Use number of regular exemptions only. Enter number of desired table in the Misc Factor field.
Table A = 1
Table B = 2
Table C = 3
Table D = 4
Table E = 5
If no table is specified, Table A is automatically assigned for single status, and Table B for married status.

New Mexico
NM Tax
bspPRNMTXX
PR Employees (Filing Status tab)
Use filing status of S, M, or H only.
S = Single or Married filing separately
M = Married filing jointly (or Qualifying widow(er))
H = Head of Household
Use number of regular exemptions only.

New York Disability
NY Disability
bspPRNYDXX
PR Deductions/ Liabilities
Set up a State deduction, using Rate of Gross as Method. Limit by Calculated Amount, enter the limit amount, and apply by Pay Period.
Set up a Liability, using Routine as Method and NYDis Tax as the Routine. Limit by Subject Amount, enter limit amount, and apply by Pay Period.
