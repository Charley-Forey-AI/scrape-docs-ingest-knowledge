# Set Automatic Overtime to Use the Highest Available Rate

Set automatic overtime to use the highest available
 rate.

1. In PR Company
 Parameters, click on the Info tab.

1. Select the Using Automatic
 Overtime checkbox.

1. In the Overtime
 Earnings Code field, enter the earnings code to use for
 overtime.

1. Select the Use Factored
 Employee Rate if Higher checkbox.

1. If you use variable
 earnings, select the Use Variable Earnings Rates
 checkbox.

1. Save the record. The system will compare the timecard rate, the
 craft/class rate (if you checked the Use Variable Earnings Rates
 box), and the employee rate. The one with the highest rate will be used.
