# Set up Payroll Crews

You will use the PR Crews form to set up payroll crews.
Crews identify groups of employees and/or equipment that commonly work together.
The following instructions detail how to set up a payroll crew.

1. In the Crew
 field, enter a unique code for the crew you are creating, up to 10 characters. Press
 F4 to see a list of crews.

1. Enter a description for the
 crew in the Description field.

1. In the JC Co#
 field, enter the default JC company for this crew. This is the company where the
 crew's time will generally be posted to. Press F4 for a list of valid companies.


1. In the Job
 field, enter the job where the crew's time will generally be posted to. Press
 F4 for a list of valid jobs.

1. In the Timesheet
 Phases fields, enter the phases where time for the crew will generally
 be posted. You can enter up to eight phases for the crew. Note: If you require more than eight phases, you
 can manually add them to additional timesheets for the crew in PR Crew Timesheet
 Entry.

1. Enter the override earnings
 code for regular, overtime, and doubletime hours for this crew in the Regular,
 Overtime, and Doubletime fields, respectively.
 The system uses these earnings codes when sending timesheets to a payroll timecard
 batch.

1. In the PR Group
 field, enter the payroll group for this crew. Press F4 to see
 a list of valid groups. Note: The group that you enter here must be the
 payroll group assigned to each of the crew members in PR Employees (PR
 Group field).

1. In the Shift
 field, enter the shift for this crew. Note: This shift defaults when you [initialize timesheets and/or timecards for this
 crew](/en/vista/vista/hr-and-payroll/payroll/time-entry/timesheets/about-the-pr-initialize-crew-timecards-form).

1. Click the Timesheets Require
 Approval box if you want crew timesheets to be approved before sending
 them to a payroll, job cost, or equipment usage batch. Note: You can approve crew timesheets using the PR
 Crew Timesheet Approve form. [Timesheets are sent to batches](/en/vista/vista/hr-and-payroll/payroll/time-entry/timesheets/send-crew-timesheets) using PR Timesheet
 Send.

1. [Add employees to the crew](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/set-up-payroll-crews/add-employees-to-payroll-crews) using the Crew Members tab.

1. Save the crew record as normal. The crew is now set up and you can begin [entering timesheets for the crew](/en/vista/vista/hr-and-payroll/payroll/time-entry/timesheets/enter-crew-time) in PR Crew Timesheet Entry.
