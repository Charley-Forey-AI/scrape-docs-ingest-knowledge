# Set up Separation Payments

(Canada only) Before you can enter ROE Separation Payments you must set up the payment information.
Note: Prior to setting up separation payment information, you must first [set up the associated earnings code](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/set-up-earnings-codes-canada).
For information on how to determine when you may need to select any of the separation payment options, refer to the "How to Complete the Record of Employment Form" document on the Service Canada website.

Separation payments are earnings paid to an employee due to separation (for example, vacation, statutory holiday, or other monies). These payments are reported in Block 17 of the Record of Employment (ROE) report.
To ensure payments are included in Block 17 in
 ROE reporting, you must:

- create dedicated earnings codes that will be used
 exclusively for each type of separation payment

- for each earnings code,
 set up the applicable separation payment information in the Record of Employment
 section of the PR Earnings Codes form.

To set up special payment information for an earnings code:

1. In the PR Earnings Codes form,
 select the earnings code for which you are setting up separation payment
 information.

1. Click on the Addl Info
 tab.

1. If this is an earnings for which EI premiums
 are paid, or if CRA rules specify this earnings to be insurable, select the Insurable
 Earnings checkbox.If you do not
 select this checkbox, earnings associated with this earnings code will not be
 included within Insurable Earnings in ROE reporting.

1. If the hours worked for this earnings are
 insurable per CRA rules, select the Insurable
 Hours checkbox.If you do not select this checkbox, hours associated
 with this earnings code will not be included within Insurable Hours in ROE
 reporting.

1. Select the Separation checkbox.

1. From the Category drop-down, choose between V-Vacation, SH-Statutory
 Holiday, or OM-Other Monies to indicate
 the separation payment category.

1. Depending on the category you selected, select the appropriate Pay Type.

1. Save the record.

For each earnings code,
 set up the applicable separation payment information in the Record of Employment
 section of the PR Earnings Codes form.
