# About Deduction and Liability Rates

There are two rate/amount fields for each record in the PR
 Deductions/Liabilities form.
When creating deduction and liability codes, use the
 Rate/Amount #1 field to indicate the primary rate or amount that
 the system will use when calculating a rate-based deduction or liability. The system
 also uses this rate as the resident rate for some state and local taxes. The
 Rate/Amount #2 field is the secondary, or "non-resident"
 rate/amount used for some state and city tax routines. Typically, you will only use
 the Rate/Amount #1 field.
Note: The system determines residency by checking whether the
 posted to state or city matches the information in the employee's static
 file.
