# Assigning Deductions and Liabilities

Once you have created deduction and liability codes in PR Deductions/Liabilities, you will need to assign them to one of the following levels: Federal, State/Provincial, Local, Craft/Class, Insurance, or Employee.
The following table displays where you can assign deduction/liability codes to each level.
Form
Description

PR Federal Information
Use this form to identify all of your Federal deductions and liabilities. Canadian users also use this form to set up QPP and QPIP deductions and liabilities.
Additional deductions/liabilities that apply to all employees without regard to the State, Local, Craft/Class, or Insurance code that their earnings are posted to may be set up here on the Add'l Federal Based Dedns/Liabs tab. You may find this to be more convenient than setting up the deduction/liability codes individually for each employee in PR Employees.

PR State Information / PR Province Information
Use this form to identify tax, unemployment, and other deduction and liability information associated with a state/province.

PR Local Codes
Use this form to identify income tax and other deduction and liability information associated with a city, county, or other local taxing district.

PR Crafts
Use this form to identify deductions (including pre-tax) and liabilities associated with different types of crafts (unions).

PR Craft Classes
Deductions (including pre-tax) and liabilities in this form override the deductions and liabilities set up in PR Crafts.

PR Craft Templates
Use this form to override standard Craft information set up in PR Crafts.

PR Craft Class Templates
This form is the most detailed level for determining deductions (including pre-tax) and liabilities. Deductions and liabilities set up here override those at the Craft, Class, or Craft Template levels.

PR Employee Dedns/Liabs
Use this form to set up all employee-specific information related to deductions (including pre-tax) and liabilities. This includes Federal and State tax filing status, tax add-ons, calculation overrides, and also employee deductions (i.e. garnishments, insurance, and liabilities). Access this form from the Payroll Programs folder on the Main Menu or by double-clicking the Dedns / Liabs tab in PR Employees.

If you have [created a pre-tax deduction](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/about-creating-deductionliability-codes/create-deduction-and-liability-codes) and want to assign it at the craft/class/template levels, you must first add the pre-tax deduction to the Dedns/Liabs tab in PR Crafts. Once you add the pre-tax deduction to PR Crafts, you can then specify overrides in PR Craft Templates, PR Craft Classes, or PR Craft Class Templates. If you attempt to add a pre-tax deduction to the Dedns/Liabs tab for one of the these forms without adding it to PR Crafts, the system will display a warning and prevents you from entering the deduction.
