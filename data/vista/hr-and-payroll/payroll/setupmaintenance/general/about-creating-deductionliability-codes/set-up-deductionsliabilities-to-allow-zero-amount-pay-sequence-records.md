# Set up Deductions/Liabilities to Allow Zero-Amount Pay Sequence Records

You can set up deductions and liabilities to allow zero-amount pay sequences to display in PR Employee Pay Seq Control for processed payrolls.
By default, once you've processed a payroll, the PR
 Employee Pay Seq Control form shows non-zero amount deduction/liability records for an
 employee/pay sequence.If you prefer to see all payment sequence
 records, including those with zero amounts, you can set up applicable deductions and
 liabilities to allow showing those records.

1. Open PR Deductions/Liabilities.

1. In the Dedn / Liab Code field, enter the deduction or liability code. Press F4 to select from a list of valid deduction/liability codes.If this is a new code, enter the new code number and then set up the remaining information as needed.

1. Select the Preserve zero-amount records in pay sequence checkbox.

1. Select the Basis Codes tab and make sure all applicable earnings codes are assigned to the deduction/liability.

1. Save the record.

If you have not already done so, assign the deduction/liability to all applicable employees in PR Employee Dedns/Liabs, making sure to select the Employee-Based checkbox.
