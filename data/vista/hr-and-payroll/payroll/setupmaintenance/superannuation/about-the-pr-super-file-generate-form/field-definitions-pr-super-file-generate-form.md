# Field Definitions: PR Super File Generate Form

The following is a list of field descriptions for the PR
 Super File Generate form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## PR Group

To generate a file for every employee in the PR company, leave this field blank.
To generate a file for a specific PR group
 within the PR company, enter the PR group number, or press F4 for a list.

## Begin Date / End Date

Required.
Enter the range of dates by which to filter records for the Super Contributions file. Pay period ending dates (set up in [PR Pay Period Control](/en/vista/vista/hr-and-payroll/payroll/payroll-processing/payroll/pr-pay-period-control-form)) that fall within this range determine the pay periods that are used for contribution calculations. (Includes closed pay periods only.)
The Begin Date must be on or before the
 End
 Date.

## Employee

To generate a file for every employee in the PR Group, leave this field blank
To generate a file for a specific employee,
 enter an employee number, or press F4 to select from a list.

## Clearing House

Required.
Enter the clearing house for which you want to
 generate a Super Contributions file, or press F4 to select from a list.

## File Payment Date

Required.
Defaults to today's date.
Enter the date for which you want to generate the Super Contributions file.

## File Reference

Enter a reference code or description, up to 18 characters. This reference is used by some clearing houses to validate the file as unique.
