# About the PR Super Clearing House Funds Form

(Australia only) Use the PR Super Clearing House Funds form to associate superannuation funds with clearing houses within a PR company.
This form only displays if your default country is set to Australia (Default Country field, HQ Company Setup form).
You can do these tasks with this form:
