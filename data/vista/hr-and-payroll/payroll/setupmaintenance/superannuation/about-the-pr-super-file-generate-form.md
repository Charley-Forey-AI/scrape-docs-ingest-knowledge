# About the PR Super File Generate Form

(Australia only) Use the PR Super File Generate form to generate Super Contributions files.
Once generated, you can submit the file to the selected
 clearing house so that it can manage the distribution of data and cash related to its
 associated superannuation funds.
Note: The system displays this form only if your default
 country is set to Australia (Default Country field, HQ Company
 Setup form).
The file is a summary of all employer and employee
 superannuation contributions by fund. It includes data from closed pay periods only for
 employees of a specific PR company. The file is generated in .csv file format, and is
 formatted to meet the requirements of the selected clearing house.
Use this form to [Generate a Superannuation Contributions File in .csv Format](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/superannuation/generate-a-superannuation-contributions-file-in-.csv-format#task-1433--en__task-1433).
