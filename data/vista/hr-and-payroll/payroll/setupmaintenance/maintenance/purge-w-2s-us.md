# Purge W-2s (US)

(Unites States) You can use the W-2s tab in PR Purge to purge federal, state, and local W-2 records for a given tax year.
This tab deletes the associated records from the W-2 employee amounts (PRWA), W-2 state (PRWS), and W-2 local (PRWL) tables for the specified tax year.
Note: The W-2 tab displays if the default country for the company
 you are processing in has been set to United States (Default
 Country field, HQ Company Setup). If you are in an Australian company, this tab displays as the ATO tab. For more information, see [Purging ATO Data](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/maintenance/purge-ato-data-au).
If you are in a Canadian company, this tab displays as the T4s tab. For more information, see [Purging T4s](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/maintenance/purge-t4s-ca).

The following instructions detail how to purge W-2s.

1. In PR Purge, select the W-2s tab.

1. Check the Purge W-2
 Data box. The system enables the Tax
 Year field.

1. In the Tax
 Year field, enter the year for purging federal, state, and local
 W-2 data.

1. Click Purge. The system displays a confirmation message.

1. Click Yes
 to purge W-2s.

The system purges all W-2 records meeting the specified criteria.
