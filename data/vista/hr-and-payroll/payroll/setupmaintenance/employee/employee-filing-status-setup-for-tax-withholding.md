# Employee Filing Status Setup for Tax Withholding

You can use the Filing Status tab in PR Employees to set up employee filing statuses for tax withholding.
You can also enter the same information in PR Employee Dedns/Liabs; on that form you can also override the calculation or indicate an add-on amount. However, this tab is more streamlined if you just need to enter the filing status and indicate the number of exemptions.
The fields on this tab function differently depending on your country. Select the correct link below for setting up employee filing status.
