# About Synchronizing Employee Data with HR

You can use the Synchronize HR option in both the PR
 Employees and the PR Employee Dedns / Liabs forms (File menu) to update the information to
 HR so that both the PR and HR information are synchronized.
Synchronization can be done at any time, regardless
 of whether changes have been made to the employee's information in the current session.
 This is useful in situations where you elected to not update HR when a change was
 originally made. When you synchronize HR, the system updates all applicable fields in HR
 Resources based on PR Update Options in the HR Company Parameters form.
Note: This feature is only available in View mode; that is, you
 cannot use this feature to update HR when in Add or Change mode. Additionally, feature
 can only be used for employees that already exists in and are active in HR.
