# Amount

For United States and Australian companies

- For deductions — If the deduction code entered for this line has a method of Rate of Gross, the amount is calculated based on the Rate specified in [PR Deductions/Liabilities](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-deductionsliabilities-form). Otherwise, enter or edit the month-to-date amount for this deduction code.

- For earnings — Enter or edit the month-to-date amount for this earnings code.

- For liabilities — If the liability code entered for this line has a method of Rate of Gross, an amount is calculated based on the Rate specified in [PR Deductions/Liabilities](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-deductionsliabilities-form). Otherwise, enter or edit the month-to-date amount for this liability code.

For Canadian companies
This field is available only for adding records. For adding a new record, enter the following:

- For deductions — If the deduction code entered for this line has a method of Rate of Gross, the amount is calculated based on the Rate specified in [PR Deductions/Liabilities](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-deductionsliabilities-form). Otherwise, enter or edit the month-to-date amount for this deduction code.

- For earnings — Enter or edit the month-to-date amount for this earnings code.

- For liabilities — If the liability code entered for this line has a method of Rate of Gross, an amount is calculated based on the Rate specified in [PR Deductions/Liabilities](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/general/payroll-general-setup-forms/pr-deductionsliabilities-form). Otherwise, enter or edit the month-to-date amount for this liability code.

For editing an existing record, double-click the record to edit using the [PR Employee Accumulations Detail](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/pr-employee-accums-detail-form-ca) form.
