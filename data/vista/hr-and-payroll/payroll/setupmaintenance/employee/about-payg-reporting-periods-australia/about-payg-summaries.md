# About PAYG Summaries

The system uses each employee's reporting periods to determine the summaries to generate.
 A single summary will be created for each reporting period in the specified tax year with a Period End Date that falls on or before the "process through" date you specify when generating summaries.
If you don't provide a Period End Date for a reporting period, the system uses the specified "through date". For more information, see [PR PAYG Summary Process Form](/en/vista/vista/hr-and-payroll/payroll/quarteryear-end-reporting/payg-payment-summaries/payroll-australia-payg-forms/pr-payg-summary-process-form).
