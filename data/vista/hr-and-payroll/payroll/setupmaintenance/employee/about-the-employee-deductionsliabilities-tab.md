# About the Employee Deductions/Liabilities tab

The Dedns/Liabs tab in PR Employees shows the deductions and liabilities set up for the employee in PR Employee Dedns/Liabs, along with the rate/amount and limit for each.
You can add deductions and liabilities for an employee directly in the Dedns/Liabs grid; however, you must access the PR Employee Dedns/Liabs form (by double-clicking in the grid) to enter additional information not included in the grid. For example, Additional columns in the grid represent other information contained in the PR Employee Dedns/Liabs form (i.e., Employee-Based, Override Misc Amount #1, etc.). A checkmark shown in any of these columns denotes that there is existing information defined for the employee in that section of the deduction/liability overrides form.
