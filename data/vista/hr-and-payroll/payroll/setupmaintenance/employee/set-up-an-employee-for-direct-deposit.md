# Set up an Employee for Direct Deposit

For employees receiving pay checks via direct deposit, you will use the Direct Deposit tab in PR Employees to set up the appropriate banking information.
Typically, employers will implement the prenote process prior to activating direct deposit for the employee to ensure validation of routing and account numbers by the bank. During the prenote process, the employee receives a check instead of a deposit. This ensures the employee is paid should the bank reject the prenote transaction due to invalid information. However, you are not required to implement the prenote process to activate direct deposit for an employee.
Click on the links below for more information.
To set up an employee for the pre-note process:

1. In PR Employees, select the employee for which to set up direct deposit and click the Direct Deposit tab.

1. In the Direct Deposit
 section, select Pre-Note. The employee will
 continue to receive paper checks until the pre-note process is complete.

For US and Canada customers:

- In the Routing Transit #
 field, enter the routing transit number of the bank accepting the direct
 deposit for this employee.

For Australian customers:

- In the BSB # field, enter
 the bank state branch number of the bank accepting the direct deposit
 for this employee.

1. In the Bank Account
 # field, enter the employee's bank account number.

1. In the Account Type
 section, select the type of account receiving direct deposits for the employee.
 Options are: Checking and Savings.

1. From the Acct
 Type drop-down, select the type of account receiving direct
 deposits for the employee. Options are: Checking and Savings.

1. In the Bank Account
 # field, enter the employee's bank account number.

1. In the Account Type
 section, select the type of account receiving direct deposits for the employee.
 Options are: Checking and Savings.

1. In the Account Type
 section, select the type of account receiving direct deposits for the employee.
 Options are: Checking or Savings.

1. Save the record.

Once you have completed the pre-note process (the bank has verified the account information is valid), you can set up the employee to begin receiving direct deposits.
To activate direct deposit for an employee:

1. In PR Employees, select the employee for which to set up direct deposit and click the Direct Deposit tab.

1. In the Direct Deposit
 section, select Active.

1. If you did not previously implement the prenote process for this employee, do the following:

For US and Canada customers:

- In the Routing Transit #
 field, enter the routing transit number of the bank accepting the direct
 deposit for this employee.

For Australian customers:

- In the BSB # field, enter
 the bank state branch number of the bank accepting the direct deposit
 for this employee.

For US and Canada customers:

- In the Routing Transit #
 field, enter the routing transit number of the bank accepting the direct
 deposit for this employee.

For Australian customers:

- In the BSB # field, enter
 the bank state branch number of the bank accepting the direct deposit
 for this employee.

1. In the Bank Account # field,
 enter the employee's bank account number.

1. In the Account Type section, select the type of
 account receiving direct deposits for the employee. Options are:
 Checking and
 Savings.

1. In the Account Type
 section, select the type of account receiving direct deposits for the employee.
 Options are: Checking or Savings.

1. Select the Direct Deposit
 Applies to Only One Pay Sequence checkbox if direct deposit
 applies to only one pay sequence (specified in the Pay Seq field) for this
 employee. Direct deposit payment will default for the specified pay sequence for
 all pay periods; however, you may override the default in PR Employee Pay Seq
 Control. Do not select the Direct
 Deposit Applies to Only One Pay Sequence checkbox if direct
 deposit payment applies to all pay sequences for the employee.

1. If you selected the
 Direct Deposit Applies to Only One Pay Sequence checkbox, in
 the Pay
 Seq field, enter the pay sequence to pay by direct deposit. This
 pay sequence will default a direct deposit payment method (E-EFT) for all pay
 periods for this employee. You may override the default in PR Employee Pay Seq
 Control.

1. Save the record. The amount method (percentage or flat amount) and multiple direct deposits are set up on the Add’l Direct Deposits tab. For more information, see [Setting up Additional Direct Deposits for an Employee](/en/vista/vista/hr-and-payroll/payroll/setupmaintenance/employee/set-up-additional-direct-deposits-for-an-employee).
