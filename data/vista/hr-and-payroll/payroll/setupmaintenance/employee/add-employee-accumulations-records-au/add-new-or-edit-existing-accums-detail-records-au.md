# Add New or Edit Existing Accums Detail Records (AU)

(Australia) Add new or edit existing detail records in the PR Employee Accums Detail form.
Once PR Employee Accumulations is set up, earnings,
 deductions, and liabilities accumulations are automatically created in this form.
 However, you can add new detail records for an employee or edit detail records to insert
 values not directly related to an existing PR Timecard or the processing of a timecard.
 See the F1 help for more information.Note: You cannot add new accumulations records using the PR
 Employee Accums Detail form. You can only add or edit detail for an existing
 accumulations record. If you need to add an accumulations record, you must do so
 in PR Employee Accumulations before you can add detail records for the new
 accumulation record.

1. Launch the PR Employee Accumulations form.

1. In the Employee field, enter the employee to work with or press F4 to select from a list of valid employees.

1. Double-click any record in the grid.The PR Employee Accums Detail form opens, displaying the accums detail records related to the record you selected in PR Employee Accumulations.

1. In the Employee field, accept the default or enter a different employee to work with. Press F4 to select from a list of employees who have records in PR Employee Accumulations.

1. In the Month field, select a month for which there are existing records for the selected employee in PR Employee Accumulations.

1. In the Type field, press F4 to select an EDL type for which there are existing records for the selected employee and month combination in PR Employee Accumulations.

1. In the EDL Code field, press F4 to select an EDL code for which there are existing records for the selected employee, month, and EDL type combination in PR Employee Accumulations.

1. If editing existing detail, select the record to edit. Then update the Hours, Subject Amount, Eligible Amount, and/or Amount fields as applicable. These fields are available depending on which option is selected in the Type field.

1. If adding a new detail record:

1. Click in the Record ID field to automatically generate a new sequence number.

1. Enter the Income Stream Type, and if the income stream type is WHM (Working holiday maker), also enter the Income Stream Country.

1. Enter values in the Hours, Subject Amount, Eligible Amount, and/or Amount fields as applicable. These fields are available depending on which option is selected in the Type field. See the F1 help for more information.
