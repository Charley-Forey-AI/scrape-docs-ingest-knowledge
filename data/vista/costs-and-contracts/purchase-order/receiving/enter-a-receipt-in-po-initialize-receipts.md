# Enter a Receipt in PO Initialize Receipts

You can enter receipts for purchase orders using the PO Initialize Receipts form.

When you open the PO Initialize Receipts form, the Receipt Date and Received By fields populate with default information.

1. In the PO # field, enter the PO to receive or press F4 to select from a list of valid purchase orders.The items grid displays all items for the selected purchase order that are set up to be received.Note: If you use the PO Item Distribution feature, the grid displays the distributed items for each PO item. For more information about distributing PO items, see [PO Item Distribution Overview](/en/vista/vista/costs-and-contracts/purchase-order/purchase-order-entry/po-item-distribution-overview).

1. In the Receiver # field, enter the receiver number for the purchase order (for example, the packing list number).

1. Receive units using one of the following methods:

- Auto-receive all units for all items in the grid by selecting the Receive All checkbox.

- Manually enter received units in the Recvd This Time field for each applicable item in the grid.

1. Once you have updating received units, select Update.The receipt is saved and added to the batch, and the cursor moves back to the PO # field so you can enter additional receipts, if needed.

1. Once you have completed entering receipts, post the batch (File > Process Batch).
