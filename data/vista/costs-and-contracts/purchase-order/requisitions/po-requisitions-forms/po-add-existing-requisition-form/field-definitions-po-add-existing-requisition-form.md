# Field Definitions: PO Add Existing Requisition Form

The following is a list of field descriptions for the PO Add
 Existing Requisition form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## RQ ID

Enter the requisition from which to select a requisition line to add to this quote line. Press F4 for a list of valid requisitions.

## RQ Line

Enter the requisition line to add to this quote line. Press F4 for a list of valid requisition lines.
Once you enter the requisition ID and requisition Line, press OK. If the requisition line is eligible (i.e. has the same material code and is not already on a quote or purchase order), a message displays indicating that the requisition line was successfully added to the quote line.
If requisition line is not eligible (i.e. has a different material code and/or is already on a quote or purchase order), a message displays indicating that the requisition line was not added to the quote line.
