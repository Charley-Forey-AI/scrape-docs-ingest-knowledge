# PO Add Existing Requisition Form

The PO Add Existing Requisition form is used to add requisition lines to a quote line.
Access this form via the RQ Quote Edit form by selecting Add Existing Req (Req Lines tab).
Specify the requisition number and line and press OK. If the requisition line is eligible (i.e. has the same material code and is not already on a quote or purchase order), a message will display indicating that the requisition line was successfully added to the quote line.
If the requisition line is not eligible (i.e. has a different material code and/or is already on a quote or purchase order), a message displays indicating that the requisition line was not added to the quote line.
Note: Adding requisition lines using the Add Existing Req button disregards the Approval Required on RQ for Quote checkbox in PO Company Parameters (Requisition Info tab).
