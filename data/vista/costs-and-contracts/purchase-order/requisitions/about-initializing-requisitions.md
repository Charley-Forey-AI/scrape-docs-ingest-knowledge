# About Initializing Requisitions

You can use information from a variety of modules to initialize requisitions to PO Requisition Entry.
To initialize requisitions, you must be using the Equipment Management (EM), Inventory (IN), and/or Project Management (PM) modules.
Note: No matter which module you are initializing from, all lines default with a route type of Quote. If you do not change the Route Type field, the requisition lines must go through the quote process. Alternately, you can change the route type to either Purchase or Stock. For more information on routing options, refer to Creating Requisitions in Related Topics below.
