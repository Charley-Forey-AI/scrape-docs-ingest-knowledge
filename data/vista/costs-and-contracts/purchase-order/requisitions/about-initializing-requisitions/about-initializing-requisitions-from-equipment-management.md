# About Initializing Requisitions from Equipment Management

Use of the EM module allows you to initialize requisitions
 from open work orders.
The initialization process cycles through
 all open work orders and creates requisition lines for the parts needed to complete the
 work orders. To enable a successful initialization, you must have set up the parts on
 the work order (EM Work Order Edit, Parts tab). This includes selecting ‘P-Purchase’
 from the P/S Flag drop-down, and checking the Required box. Use PO Requisition
 Initialization to initialize requisitions from EM. Once you initialize the requisition,
 the system assigns it a Work Order line type. For more information, see PO Requisition
 Initialization in Related Topics below.
