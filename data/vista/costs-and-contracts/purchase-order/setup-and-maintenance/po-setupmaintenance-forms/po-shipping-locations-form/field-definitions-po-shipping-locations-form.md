# Field Definitions: PO Shipping Locations Form

The following is a list of field descriptions for the PO
 Shipping Locations form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Shipping Location

 Use this field to create a unique code that will identify the shipping location. The shipping location code can be up to 10 characters long.
This is the shipping location code that is entered on a purchase order, and it provides the default “Ship To” address for the purchase order.

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations

##  Description

 Enter a description of the shipping location. The description can be up to 30 characters long.

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations

##  Tax Code

This is an optional field.
Press F4 to add a tax code to the shipping
 location. The selected tax code will be used as the default when entering “non-job” lines
 in the [PO Purchase Order Entry ](/en/vista/vista/costs-and-contracts/purchase-order/purchase-order-entry/po-purchase-order-entry-forms/po-purchase-order-entry-form) form.
Tax codes are created and maintained using the [HQ Tax Codes ](/en/vista/vista/administration/headquarters/tax-code-setup/about-the-hq-tax-codes-form) form.

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations

##  Address, City, Zip Code

Use the Address, City, and
 Zip
 Code fields to enter the “ship to” address of the shipping location. This is
 the address that will display in the Ship To section of the purchase order.

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations

## State

Enter a valid state or press F4 to select a
 state from a list. States are created and maintained using the [HQ States ](/en/vista/vista/administration/headquarters/region-setup/hq-states-form) form.
If a country is not selected in the Country field,
 the system validates that the state is correct based on the default country set up on the
 active company (HQ Company
 Setup > Info
 tab > Default
 County field).

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations

##  Country

Press F4 to select a country from a list.
 Countries are created and maintained using the [ HQ Countries ](/en/vista/vista/administration/headquarters/region-setup/create-country-codes) form.
You can skip this field if the shipping
 location is in the default country of the currently selected company. The default country
 is set up using the [HQ company Setup](/en/vista/vista/administration/headquarters/company-setup/hq-company-setup-form) form (Info tab> Default Country
 field).

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations

##  Additional Address

 Use this field to enter additional address information for this shipping location. The value in this field can be up to 60 characters long. For example, if the shipping address is in an office building, you might use this field to enter the building number and/or suite/room number: Bldg 1, Suite 250.
 The address information entered here will display in the Ship To section of the purchase order (e.g. PO Purchase Order Form, PO Purchase Order Form w/Chg Ord, etc.).

[](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/po-setupmaintenance-forms/po-shipping-locations-form)PO Shipping Locations
