# Add a New PO Item

You can add a new purchase order item to a purchase order using the PO Add Item form.

The following details now to add a new PO item to an existing purchase order in the PO Change Order Entry form.

1. Open the PO Change Order Entry form.

1. In the Seq # field, enter + or N.The system automatically assigns the next available sequence number.

1. In the PO # field, enter the purchase order or press F4 to select from a list of valid purchase orders.

1. In the Item# field, enter a new item number. You can press F4 to see a list of existing items for the selected purchase order.Once you move off the Item # field, the system displays a message indicating the PO Item does not exist and asking if you want to add the new item.

1. Select Yes.The PO Add Item form displays.

1. Enter the PO item information and select Add.The new PO item is created and you are returned to the PO Change Order Entry form.

1. In PO Purchase Order Entry, iff you created a unit-based PO item:

1. In the Units field, enter the number of units by which to change the purchase order's current units.

1. In the Unit Cost field, enter the change to the unit cost or leave at 0.00 if there is no change to the unit cost.For example, if the original unit cost was $100 and for the new item, it is $110, enter 10.00 in the Unit Cost field.

1.  If you created a lump sum PO item:

1. In the Total Cost field, enter the total cost of the new item.

1. Save the record.The system updates the purchase order cost information (shown in bottom of form) accordingly. For more information, see [Purchase Order Cost Information](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/purchase-order-cost-information#ID-0002f085--en__ID-0002f085).

1. Process the batch.
