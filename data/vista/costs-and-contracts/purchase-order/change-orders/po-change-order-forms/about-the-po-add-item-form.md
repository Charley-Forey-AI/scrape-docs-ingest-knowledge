# About the PO Add Item Form

Use the PO Add Item form to add a new item to a purchase order.
You can only access this form via PO Change Order Entry by entering a new number in the Item# field and then selecting Yes to the "Do you wish to add this item" prompt that displays.
Most of the fields on this form match the fields on the PO Purchase Order Entry form; however, this form does not include a Units or Total Cost field. To enter this information, complete the fields on this form and then select the Add  button. This will return you to the PO Change Order Entry form, where you can enter the unit cost and total cost information (in the Change to Current section).
