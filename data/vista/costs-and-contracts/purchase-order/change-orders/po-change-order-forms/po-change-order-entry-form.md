# PO Change Order Entry Form

Use this form to create new purchase order change orders (POCOs), or edit existing PO change orders.

A PO change order is a change to an existing purchase order. Before you create a change order, you should understand how the system tracks purchase order cost information. For more information, see [Purchase Order Cost Information](/en/vista/vista/costs-and-contracts/purchase-order/setup-and-maintenance/purchase-order-cost-information).
When creating new PO change orders you can:

- Change an existing PO item

- Add a new PO Item to an existing PO

When editing existing PO change orders, you can:

- Change a PO change order by either adding a new PO item or changing an existing one

- Delete an existing PO change order
