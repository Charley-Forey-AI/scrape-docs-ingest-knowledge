# PO Quote Edit Form

Use this form to review and/or modify existing quotes.

Quotes cannot be added manually in this form. If you need to create a new quote, you must use PO Quote Initialization.
The quote header shows the quote's creation date, who created the quote, and the quote's description. However, only the description can be edited. You also have the option to specify whether the quote is locked. When the Locked option is checked, quote lines cannot be edited or deleted and will be grayed out. In addition, the Apply button will be disabled to prevent any updates via the Apply to All Quote Lines feature.
