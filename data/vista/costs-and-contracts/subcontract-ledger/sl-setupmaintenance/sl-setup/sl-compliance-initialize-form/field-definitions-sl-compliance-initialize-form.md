# Field Definitions: SL Compliance Initialize Form

The following is a list of field descriptions for the SL
 Compliance Initialize form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Compliance Code

 Enter the compliance code to apply to multiple subcontracts. Press F4 for a list of valid compliance codes.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Apply Compliance Code to Pending Subcontracts

 Check this box to apply this compliance code to pending subcontracts in addition to open subcontracts.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Beginning JC Co#

 Optional field.
 Enter the beginning number in a range of JC companies for applying the compliance code to subcontracts.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Job

 Optional field.
 Enter the beginning number on a range of jobs for applying the compliance code to subcontracts.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Ending JC Co#

 Optional field.
 Enter the ending number in a range of JC companies for applying the compliance code to subcontracts. This field defaults the company that you enter in the Beginning JC Co# field.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Job

 Optional field.
 Enter the ending number on a range of jobs for applying the compliance code to subcontracts. This field defaults the company that you enter in the Beginning JC Co# field.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Include all Subcontracts for the Selected Job(s)

 Check this box to add the compliance code to all open subcontracts for the specified range of jobs. If you checked the Apply Compliance Code to Pending Subcontracts box, the system will also add the code to all pending subcontracts for the specified range of jobs.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

## Beginning Subcontract

Optional field.
Enter the beginning number in a range of subcontracts for
 applying the compliance code.
[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL
 Compliance Initialize

##  Ending Subcontract

 Optional field.
 Enter the ending number in a range of subcontracts for applying the compliance code.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize

##  Vendor

 Optional field.
 Enter a vendor to filter subcontracts by for adding the compliance code.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-setupmaintenance/sl-setup/sl-compliance-initialize-form)SL Compliance Initialize
