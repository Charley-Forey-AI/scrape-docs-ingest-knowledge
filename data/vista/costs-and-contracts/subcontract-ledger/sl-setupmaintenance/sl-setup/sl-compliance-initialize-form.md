# SL Compliance Initialize Form

Use this form to initialize a single compliance code to multiple subcontracts.
You can access this form by selecting File > Initialize Compliance Code from SL Compliance.
When you initialize a compliance code, you can filter for specific subcontracts using a range of JC companies, jobs, and subcontract numbers, or filter by a specific vendor. Once you set the filtering options and select Update, the system initializes the code to all open subcontracts that meet the filtering criteria. If you selected the Apply Compliance Code to Pending Subcontracts checkbox, the system also applies the compliance code to pending subcontracts.
Note: This form only initializes one compliance code at a time. If you have multiple codes to initialize, you must run this form for each code.
