# About the SL Worksheet Initialization Form

Use this form to initialize multiple subcontracts to a
 worksheet.
You can access this form by selecting
 File > Initialize
 Worksheet from SL Worksheet.
The pay control code is used to group invoices together for payment, and it allows you to
 easily pay the subcontractors when you get paid by the owner.
When using [AP Payment Posting ](/en/vista/vista/accounting/accounts-payable/payments/ap-payment-forms/ap-payment-posting-form) or [AP Payment Workfile ](/en/vista/vista/accounting/accounts-payable/payments/ap-payment-forms/ap-payment-workfile-form), you can select the invoices that
 should be paid by payment control code, and all invoices that have the same payment
 control code are added to the payment work file.
