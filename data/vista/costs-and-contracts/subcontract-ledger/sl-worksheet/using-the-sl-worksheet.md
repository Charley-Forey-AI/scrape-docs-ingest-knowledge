# Using the SL Worksheet

Use the SL Worksheet to create AP invoices for subcontracts.
There are several tasks associated with using the
 subcontract worksheet, from adding subcontracts to the worksheet, to entering invoice
 information and sending the invoices to Accounts Payable.

1. Add subcontracts to the worksheet using one of the following methods:

- [Initialize the worksheet](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/initialize-subcontracts-to-worksheets#ID-0003e244--en__ID-0003e244)

- [Manually add a
 subcontract to the worksheet](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/add-a-subcontract-to-a-worksheet#ID-0003e20d--en__ID-0003e20d)

1. [Enter the
 invoice information and amounts](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/enter-subcontract-invoice-information#task-9629--en__task-9629).

1. For each subcontract that is ready to invoice, select the
 Ready checkbox. For more information about this checkbox,
 see the [F1](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/sl-worksheet-form/field-defintions-sl-worksheet-form#ID-0003f57c--en) help.

1. Run the SL to AP Update Preview report. by selecting
 File > SL to AP
 Update Preview.

1. Select File > SL
 to AP Update Preview

1. Send the invoices to Accounts Payable as follows:

- If you are updating to AP Transaction Entry,
 select File > Update
 to Accounts Payable > Update
 to AP Transaction Entry. For more information, see [Send Subcontract Invoices to AP Transaction
 Entry](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/send-subcontract-invoices-to-ap-transaction-entry).

- If you are updating to AP Unapproved Invoice
 Entry, select File > Update
 to Accounts Payable > Update
 to AP Unapproved Invoice Entry. For more information, see [About the SL Update to AP Unapprove Form](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/about-the-sl-update-to-ap-unapprove-form).

Once you invoice a subcontract, it is removed from the
 worksheet.

1. Optional. [Clear the
 worksheet](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-worksheet/clear-the-sl-worksheet#ID-0003e2fe--en__ID-0003e2fe) to remove any remaining subcontracts.
