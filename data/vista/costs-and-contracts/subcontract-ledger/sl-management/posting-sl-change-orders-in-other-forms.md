# Posting SL Change Orders in Other Forms

The SL Change Order Entry form is one of three forms
 available for posting change orders.
The other two forms are JC Change Orders and
 PM Change Orders. The following table displays what form you should use, depending on
 which totals to impact. For example, the table shows that a change order that affects
 both committed and estimated job costs should be entered in both SL Change Order Entry
 and JC Change Orders. In this case, if using the Project Management (PM) module, change
 orders would only need to be entered in PM Change Orders.

Note: This form can be used to make any necessary changes
 to a subcontract change order that was interfaced from PM. When the subcontract change
 order batch is posted, PM Subcontract Detail (PMSL database table) is updated for the
 change order record if the SL batch month and transaction are found in PMSL.
