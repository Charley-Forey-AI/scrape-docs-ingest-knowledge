# About Entering Change Order Items

 Change Order Items can be entered in SL Subcontract Entry,
 but are typically entered using SL Change Order Entry, which can add the item and update the
 current amount in one step.
 When setting up a change order item on a
 subcontract in SL Subcontract Entry, identify the job, phase, cost type, and
 description. No units, unit costs, or total costs may be entered—that must be done in SL
 Change Order Entry.
 If a subcontract change order has already
 been set up in SL Change Order Entry and the subcontract is accessed in SL Subcontract
 Entry, the form displays the change order item on the subcontract, but only the
 description, WC Ret%, and SM Ret% fields can be changed.
