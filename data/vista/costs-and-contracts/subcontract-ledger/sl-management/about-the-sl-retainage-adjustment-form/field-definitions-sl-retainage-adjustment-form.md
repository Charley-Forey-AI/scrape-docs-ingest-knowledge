# Field Definitions: SL Retainage Adjustment Form

The following is a list of field descriptions for the SL
 Retainage Adjustment form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  JC Co#

 Specify the Job Cost company in which retainage adjustments will be made.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

##  Job

 Specify the job for which retainage adjustments will be made.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

## Process All Open Subcontracts on this Job

Check this box if you want to adjust retainage on all open subcontracts for this job.
Do not check this box if you want to select the subcontract(s) for which to adjust retainage.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

##  Beginning Subcontract

 Enter the subcontract to begin the retainage adjustment process.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

##  Ending Subcontract

 Enter the subcontract to end the retainage adjustment process.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

## Change Retainage Types

- Work Completed – Select this option to adjust the retainage percentage for Work Completed only.

- Stored Materials – Select this option to adjust the retainage percentage for Stored Materials only.

- Both – Select this option to adjust the retainage percentage for Work Completed and Stored Materials.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

##  Update Retainage % on all Items

 Check this box to adjust the retainage percentage for all items on the specified subcontract(s).
 Do not check this box if you want to define the criteria for selecting subcontracts eligible for retainage adjustment.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

##  Update Only if Current Retainage % Equals

 Specify the retainage percentage to use when selecting which subcontract items to process.
 All items with a current retainage percentage equal to the percentage specified here will be updated.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment

##  New Retainage %

 Enter the new percentage that will replace the current percentage on all selected items.

[](/en/vista/vista/costs-and-contracts/subcontract-ledger/sl-management/about-the-sl-retainage-adjustment-form)SL Retainage Adjustment
