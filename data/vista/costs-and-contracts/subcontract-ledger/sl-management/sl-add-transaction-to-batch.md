# SL Add Transaction to Batch

You can use this form to add subcontracts to the currently selected batch.
Access this form by selecting File > Add Transaction from SL Subcontract Entry.
Enter the subcontract number in the Subcontract field and click the Add to Batch button. The subcontract is now included in the batch.
