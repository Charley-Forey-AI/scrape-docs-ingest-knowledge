# Reopen a closed subcontract

You can use the SL Subcontract Entry process to reopen a subcontract that has been closed using the SL Close form - for example if you accidentally closed a subcontract.
Note: When reopening subcontracts, committed costs are not reestablished. It is recommended that new items are added to create the committed cost.

1. Create a new SL Subcontract Entry batch or open an existing one.

1. When the SL Subcontract
 Entry Detail batch displays, select File > Add
 Subcontract and then add the closed subcontract to the batch.

1. Change the Status field on the Info tab in the upper portion of the form
 to Open.

1. Select File > Process Batchand then process the batch. Once the batch has been posted, the
 subcontract will be reopened.
