# Job Billing

Welcome to the Vista Help for Job Billing. It is intended as a guide and reference tool that will help you take advantage of all the features available in Job Billing.
The Implementation Checklists and Processing Logs for this module are available in the Work Flow module. To view the checklists and logs, use the WF Checklist Template form. For information on processing the checklists and logs, see [Work Flow](/en/vista/vista/system-tools/work-flow).
If you do not have the Work Flow module, use the WF Templates report (VA Reports menu) to view or print the checklists and logs.
