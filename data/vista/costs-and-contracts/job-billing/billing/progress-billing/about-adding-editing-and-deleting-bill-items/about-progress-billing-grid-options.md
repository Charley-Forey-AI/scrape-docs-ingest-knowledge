# About Progress Billing Grid Options

When working with Items on a billing in JB Progress Billing, you can set the grid to display additional fields for entry using the Options menu.

Once you have entered billing information (in [JB Progress Billing](/en/vista/vista/costs-and-contracts/job-billing/billing/progress-billing/jb-progress-billing-form), Info tab), item information for the bill displays on the Items tab. The Items tab displays standard information that can be edited for each item on the invoice. However, you can enter additional information for each items using the entry options available in the Options menu. The following is a list of each option, along with a description of its use.

- Item Unit Entry - Displays columns for Unit Price, Contract Units, Previous Units, This Bill Units, and To Date Units columns. If the unit of measure is LS (lump sum), these fields are not available for input/edit.Note: Changes to Contract Units, Contract Amount, Previous Units, and Previous Amount can only be made if the Allow Changes to Previous and Contract Amounts checkbox in JB Company Parameters is selected.

- Item Retainage Entry – Displays columns for Retainage % and Retainage.

- Item Notes Entry – Displays a column for Notes.

- Show Standard Item Code – Displays a column for Std Item on the Items and Item SM/Tax tabs. This display-only column shows the standard item code assigned to each contract item.

- Show Bill Group – Displays a column for Bill Group on the Items and Item SM/Tax tabs. This display-only column shows the bill group assigned to each contract item.

Note: You can set the show/hide function for both the Std Item and Bill Group columns independently on the Items and/or Items SM/Tax tabs using the F3 Properties form. However, be aware that selecting/deselecting the corresponding options from the Options menu (Show Standard Item Code and Show Bill Group) automatically updates the F3 setup; therefore, if you wish to maintain the individual settings, you should not use show/hide options in the Options menu. Sorting or filtering by the Std Item or Bill Group column on one tab will automatically sort/filter by the selected column on the other tab.
