# Field Definitions: JB T M Template Copy Form

The following is a list of field descriptions for the JB T M
 Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Source Co

 Enter the company to copy the billing template from.

##  Source Template

 Enter the T&M template (JB T&M Template Setup) to copy.

##  Destination Company

 Specify the JB company the T&M template will be copied to.

##  Destination Template

 Enter the name of the destination T&M template. Up to 10 characters allowed.

##  Description

 Enter a description of the new template, up to 30 characters.

##  Copy Cost Types

 Check this box to copy all cost types defined for the source template to the destination template.
 If this box is unchecked, cost types are not copied to the new template.

##  Copy Add-ons

 Check this box to copy all add-on sequences defined for the source template to the destination template.
 If this box is unchecked, add-on sequences are not copied to the new template.

##  Copy Labor Rates

 Check this box to copy all labor rates defined for the source template (in JB T&M Template Labor Rates or JB T&M Template Labor Rate Overrides) to the destination template.
 If this box is unchecked, labor rates are not copied to the new template.

##  Copy Equipment Rates

 Check this box to copy all equipment rates defined for the source template (in JB Template Equipment Rates) to the destination template.
 If this box is unchecked, equipment rates are not copied to the new template.

##  Copy Material Rates

 Check this box to copy all material rates defined for the source template (in JB Material Rate Overrides) to the destination template.
 If this box is unchecked, material rates are not copied to the new template.
