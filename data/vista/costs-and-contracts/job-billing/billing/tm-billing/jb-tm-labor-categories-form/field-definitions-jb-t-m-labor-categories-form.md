# Field Definitions: JB T M Labor Categories Form

The following is a list of field descriptions for the JB T M
 Labor Categories form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Labor Category

 Enter a labor category, up to 10 characters.

##  Description

 Enter a description of this labor category, up to 30 characters.

##  Restrict by Craft

 Check this box to restrict which crafts will be included in billing when the selected labor category is defined on a template sequence.
 Leave this box unchecked if you want all crafts to be included in billings when the selected category is defined on a template. If left unchecked, cursor skips the Class column and defaults to null (blank).

##  Craft

 If payroll detail is being restricted by craft, specify the craft code (from PR Crafts) to
 assign to this labor category. Only payroll detail posted to this craft will be included in
 billings when this labor category is defined on a template.

##  Restrict by Class

 Check this box to restrict which classes (for the specified crafts) will be included in a billing when the selected labor category is defined on a template.
 Leave this box unchecked if you want all classes to be included in billings when the selected category is defined on a template. If left unchecked, cursor skips the Class column and defaults to null (blank).

##  Class

 If payroll detail is being restricted by class, specify the class (from PR Craft Classes) to assign to this labor category. Payroll detail for this class (for the specified crafts) will be included in billings when the labor category is defined on a template.
