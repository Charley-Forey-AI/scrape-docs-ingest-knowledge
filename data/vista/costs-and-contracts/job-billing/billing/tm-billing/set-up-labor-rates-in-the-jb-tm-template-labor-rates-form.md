# Set Up Labor Rates in the JB T&M Template Labor Rates
 Form

Specify how labor is billed for each labor category on the JB
 T&M Template Labor Rates Form.
Access this form from
 either the JB Programs folder or by double-clicking a record on the Labor Rates tab
 in JB T&M Template Setup.
Labor categories must be set up in JB Labor
 Categories before setting up labor rates. Labor categories group specified classes and
 crafts together on a billing. Once categories are set up, this form displays all
 templates and labor categories.

1. Restrict the labor rate
 by earn type, factor, and/or shift by checking the Restrict by
 Earn Type, Restrict by Factor, and/or
 Restrict by Shift boxes. Leave the checkbox blank to indicate
 “all.” Depending on the selected checkboxes, the Earn
 Type, Factor, and/or Shift fields are enabled.

1. Indicate the earn type,
 factor, or shift in the Earn Type,
 Factor, and/or Shift fields, as
 necessary.For example, you can restrict a
 labor category by a specific earnings type, but include all earning factors
 and shifts. Any combination of the three restricting options can be
 used.

1. Select a rate option
 from the Rate Option drop-down: C-Actual Costs, F-Earnings Factor, or R-Labor
 Rate.Note: If you select C-Actual Costs, labor is
 billed using the actual costs posted to JC. If you select R-Labor Rate,
 labor is billed using the rate specified for the labor category. If you
 select F-Earning Factor, labor is billed using the earnings factor posted to
 JC multiplied by the labor category’s rate.

1. Enter the new rate in
 the New
 Rate field.

Note: You can update old rates for all entries in the grid by
 selectingFile > Move New
 Rates to Old.When this option is selected, the Old
 Rate field is automatically updated with the values in the
 New
 Rate fields. The New Rate fields do not change and
 must be manually updated.
The system will apply the new rate based
 on the date indicated in the
 New
 Labor Rate Effective Date
 field. To update this date, use the JB TM Template Setup form.
