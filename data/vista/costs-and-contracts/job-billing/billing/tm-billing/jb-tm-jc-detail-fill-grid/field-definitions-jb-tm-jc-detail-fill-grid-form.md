# Field Definitions: JB T&M JC Detail Fill Grid Form

The following is a list of field descriptions for the JB T&M JC Detail Fill Grid form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Cost Detail Beginning Date

The Cost Detail Beginning Date field on the JB T&M JC Detail Fill Grid form.
Enter the beginning date in a range of dates for which to pull Job Cost detail for this bill line sequence.
Note: The system will only pull cost detail that does not already exist on the billing for the specified bill line sequence.

## Cost Detail Ending Date

The Cost Detail Ending Date field on the JB T&M JC Detail Fill Grid form.
Enter the ending date in a range of dates for which to pull cost detail for this bill line sequence.
Note: The system will only pull cost detail that does not already exist on the billing for the specified bill line sequence.

## Cost Detail Cutoff Month

The Cost Detail Cutoff Month field on the JB T&M JC Detail Fill Grid form.
Enter the month through which to pull cost detail from Job Cost for the specified bill line sequence. If left blank, the system will pull all available costs not already on the billing.

## Use Different Labor Cutoff Date

The Use Different Labor Cutoff Date checkbox on the JB T&M JC Detail Fill Grid form.
Select this checkbox to use a different cutoff date (than the Cost Detail Cutoff Date) for pulling in labor cost detail. Use the Labor Cutoff Date field to specify the date through which to pull labor cost detail.
Clear this checkbox to pull in labor cost detail based on the Cost Detail Cutoff Date.

## Labor Cutoff Date

The Labor Cutoff Date field on the JB T&M JC Detail Fill Grid form.
This field is enabled once you select the Use Different Labor Cutoff Date checkbox.
Enter the date through which to pull labor costs from Job Cost. If left blank, the system will populate the grid with all available labor-type costs.
Note: The system will only pull labor costs that do not already exist on the billing for the specified bill line sequence.
