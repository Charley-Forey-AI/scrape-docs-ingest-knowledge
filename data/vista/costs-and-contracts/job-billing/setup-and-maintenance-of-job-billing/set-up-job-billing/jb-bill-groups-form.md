# JB Bill Groups Form

Use this form to set up billing groups.
Billing groups are set up by contract and are used to process certain contract items together on billing invoices. This form is not necessary to create bill groups and is used only to provide a description for a group of items.
 Each bill group is assigned a description. This
 description prints on the JB Progress Invoice by Bill Group and can be included when
 creating custom invoice formats (in Crystal Reports). This allows you to identify each
 grouping of contract items, with the contract item description or contract item “bill
 description” (assigned in JC Contract Items) used to further identify the breakdown of
 items within the group. For example:
