# Field Definitions: JC Projections Job Close-Out Form

The following is a list of field descriptions for the JC
 Projections Job Close-Out form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Job

 Enter the job (from JC Jobs) to close out.
Note: If you enter a soft- or hard-closed job and you do not allow posting to closed jobs (flags in JC Company Parameters), a message displays indicating the job’s status and entry is not allowed.
 Click the Generate button to implement the closeout process. All projected costs for active phases/cost types will be set equal to actual costs, and remaining values set to zero.

##  Set Projections As Plugged

 Check this box to set projections for this job as plugged. During the closeout process, projected values generated for active phases/cost types will be flagged as 'plugged' (i.e. an asterisk will display in the P column of the display grid).
 Leave this box unchecked if you do not want projections for this job set as plugged. Projected values will be displayed without the ‘plugged’ indicator.
