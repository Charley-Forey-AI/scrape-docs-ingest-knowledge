# Field Definitions: JC Overrides Copy Cost Revenue Form

The following is a list of field descriptions for the JC
 Overrides Copy Cost Revenue form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Include Notes from Prior Month

 Check this box to copy the projection override notes (for jobs if copying cost overrides or contracts if copying revenue overrides) from the previous month to the current projection month.
 Leave this box unchecked if you do not want to copy projection override notes from the previous month.
 Click the Yes button to implement the copy process. The New Overrides column (on the Cost tab if copying cost overrides or the Revenue tab if copying revenue overrides) will be populated using the Previous Override amount for each job or contract.
