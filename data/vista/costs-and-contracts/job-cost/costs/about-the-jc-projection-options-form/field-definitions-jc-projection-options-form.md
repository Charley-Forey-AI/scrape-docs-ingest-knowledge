# Field Definitions: JC Projection Options Form

The following is a list of field descriptions for the JC
 Projection Options form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Show Only Changed Since Last Projection

Check this box to display only those phases
 where the current calculated forecast is not equal to the previous calculated forecast. The
 forecast is the computer-generated calculation of projections as of the time you brought
 the job into the batch. It is compared to the previous calculated forecast. The previous
 projection may be either the last time projections were posted in the current month or the
 last time in the previous month, depending on whether the Previous Projected Through
 Prior Month box is checked.
Leave this box unchecked to display all phases on the job.

## Show Only Item Unit Cost Types

Check this box to display only cost types with
 the Item
 Unitsbox checked on either JC Phases or JC Job Phases (Cost Types tab).
Leave this box unchecked to display all cost types for a phase, regardless of how the Item Unit flag is set.

## Show Only Phase Unit Cost Types

Check this box to display only calculations for
 cost types with the Phase Units box checked in JC Job Phases (Cost Types tab) or JC Phases.
Leave this box unchecked to display all cost types for a phase, regardless of how the Phase Unit flag is set.

## Show Linked Cost Types

Check this box to show both primary and linked cost types.
You can link cost types using the Link Progress Cost Type field on the [JC Cost Types ](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-cost-types-form) form.
Leave this box unchecked to show only the primary cost types for each phase.

## Show This Window on Form Open

Check this box if the JC Projection Options form should open automatically each time the JC Cost Projections 6.7 form is opened.
Leave this box unchecked if you do not want this form opened automatically when opening JC Cost Projections. Form must be opened manually via the Options window.

## Cost Type Options

Specify the cost types to display on the Cost Projections grid.
All Cost Types – Select this option to display all cost types for all phases.
Selected Cost Types Only – Select this option to display only those cost types indicated in the Selection box below for each phase.

##  Previous Projected Through Prior Month

 Check this box to have the Previous projections column display amounts through the month prior to the current batch month (i.e. if batch month is 07/07, shows amounts through 06/07).
 Leave this box unchecked to have the Previous projections column display amounts through the current batch month (i.e. if batch month is 07/07, shows amounts through 07/07).
Note: Amounts displayed will not be date sensitive. Therefore, a projection done on 07/15/07 will be included in the Previous amount for a batch created on 07/10/07.

## Order By

Select the sort order of the projections grid.

- Contract Item – Select this option to sort grid records by contract item. Grid will display all phases/cost types assigned to the contract item specified in the Item field in lower section of the form.

- Phase– Select this option to sort grid records by phase. Grid will display all phases/cost types in numeric order based on the phase range selected in the Phase Options below.

## Cycle Through Items or Phases One at a Time

Check this box to cycle through contract items
 or phases (depending on the setting of the Order By drop-down) one at a time. The
 Order
 By drop-down field determines how you will cycle through records:

- If you are sorting by Contract Item, the grid will
 show all phases/cost types assigned to the contract item specified in the lower
 section of form. Use the Page Down key to move through each phase/cost type for the
 contract item. To move to the next contract item, use the left/right arrow buttons
 (/ ) to the left of the Item
 field.

- If you are sorting by Phase, the grid will show one
 phase at a time, beginning with the first phase in the selected range. Use the Page
 Down key to move through each of the cost types for the selected phase. To move to
 the next phase, use the left/right arrow buttons (/ ) to the left of the Phase
 field.

Leave this box unchecked if not cycling through items or phases individually. All items or phases will be included in the grid in numeric order.

## Projection Method

Indicate the override projection method that
 will be used to calculate cost projections for this batch. Method specified here overrides
 the method specified in JC Company Parameters. This field initially defaults to the method
 that you specified in JC Company Parameters (Projection Method field, Projections
 tab).

- Actual - Cost projections will be calculated using only actual costs.

- Actual
 + Committed Costs - Cost projections will be calculated using both
 actual costs and committed costs.

## Include Inactive Phases & Cost Types

This checkbox allows you to specify whether cost projections will include inactive phases and cost types. Check this box if phases and cost types flagged as 'inactive' in [JC Job Phases ](/en/vista/vista/costs-and-contracts/job-cost/jobs-and-contracts-setup/jc-job-phases-form) should be included in cost projections. The inactive phases and cost types will be highlighted in gray on the JC Cost Projections form.
Leave this box unchecked to exclude include inactive phase and cost types in projections.
Note: This box is only be enabled when the Allow Inactive Phases and
 Cost Types box on the Projections tab of JC Company Parameters is
 checked.
