# About the JC Projection Future CO Form

This display-only form is accessed by clicking the Future CO’s button at the bottom of the JC Cost Projections screen, and is used to view pending and approved change order items that have been set up for the job/phase/cost type in Project Management (phase must be active), but have not yet been interfaced to Job Cost.
Note: Pending change orders will only display in this form if the 'Include in Projections Flag' is checked for the document type assigned to the pending change order.

The Projection Option field above the grid displays the projection option defined for the status code assigned to the selected pending change order in PM Status IDs. If a status code's projection option is set to None, pending change orders assigned that status code will be excluded from display in this form, regardless of whether the Include in Projections option is checked for the PCO's document type. However, if the status code assigned to the pending change order has a projection option of Display in Projections or Display & Calculate in Projections, the Include in Projections option must also be checked for the PCO's document type in order for the change order to display here.
Note: If the projection option is set to Display & Calculate in Projections, the pending change order item amount will be included in the projection calculation, as well as the Future CO Cost and Include in CO Amount columns on the JC Cost Projections grid.
