# About Auto-Setting Remaining Unit Cost

When updating projections in JC Cost Projections, you can have the system automatically set the remaining unit
 cost for a selected phase/cost type to the current estimated unit cost or actual unit cost.

 To have the system set remaining unit costs for phases/cost types, click the Set
 Remain UC button (below grid). The system sets the remaining unit cost based on
 the Select Which
 Unit Cost to Use for Remaining drop-down option selected in JC Company
 Parameters (Projections tab). If you selected ‘E-Current Estimate’, the system sets the
 Remaining
 U/C column equal to the Current Est’d U/C column. If you
 selected ‘A-Actual’, the system sets the Remaining U/C column equal to the
 Actual
 U/C column.
Note: In order for this feature to work, the current estimated
 unit cost or actual unit cost must not be zero, depending on which JC Company option
 applies. If there are no existing values, then clicking the Set Remain UC button will
 not change the remaining unit cost from its current value.
