# Column Display

You can change the sort order of the records in the
JC Cost Projections grid by clicking on any column heading.
Clicking on a column heading again will
 toggle between descending and ascending sort order.
You can also control which columns display
 in the projections grid, as well as the order in which they display, using the Display
 Columns tab in JC Projection Options. For more information, see Projection Options in
 Related Topics below.
Note: If you have adjusted the width for a column and then
 move that column, the width adjustment will not move with the column. You will need to
 readjust the widths for columns after you have completed moving them to the desired
 positions. Width adjustments will be saved once you close the form.
