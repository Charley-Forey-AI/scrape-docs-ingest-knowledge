# Field Definitions: JC Job Phase Cost Detail Form

The following is a list of field descriptions for the JC Job
 Phase Cost Detail form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Phase

 Specify the job phase for which to view cost detail. Initially defaults the phase selected in JC Cost Projections.

[](/en/vista/vista/costs-and-contracts/job-cost/costs/jc-job-phase-cost-detail-form)JC Job Phase Cost Detail

##  Cost Type

 Specify the phase cost type for which to view cost detail, or leave blank to include all cost types for the phase. Initially defaults the cost type selected in JC Cost Projections.

[](/en/vista/vista/costs-and-contracts/job-cost/costs/jc-job-phase-cost-detail-form)JC Job Phase Cost Detail

##  Begin/End Month

 Specify the beginning and ending month for which to show cost detail, or leave blank to include all months.

[](/en/vista/vista/costs-and-contracts/job-cost/costs/jc-job-phase-cost-detail-form)JC Job Phase Cost Detail

##  Show Actual Units Only

 Check this box to show only those detail records for the phase/cost type with actual units (i.e. Actual Units are not 0.00).
 Leave this box unchecked to show all detail records for the phase/cost type, regardless of whether there are actual units.

[](/en/vista/vista/costs-and-contracts/job-cost/costs/jc-job-phase-cost-detail-form)JC Job Phase Cost Detail

## Costs

Specify the type of costs to display in the
 detail grid below. Then click Refresh Grid to reload the grid with the
 selected values.

- Actual – Select this option to show all actual costs for the selected job/phase/cost type.

- Committed – Select this option to show all committed costs for the selected job/phase/cost type.

- Estimate – Select this option to show all estimated costs for the selected job/phase/cost type.

- Projected – Select this option to show all projected costs for the selected job/phase/cost type.

[](/en/vista/vista/costs-and-contracts/job-cost/costs/jc-job-phase-cost-detail-form)JC Job Phase Cost Detail
