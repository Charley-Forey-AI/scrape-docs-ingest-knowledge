# About the JC Soft Close Job Form

Use this form to close a single job tied to a multi-job contract. This allows you to ensure that a particular job does not keep accruing costs when its contract is still open.
When you select to soft close a job, the system runs a check to determine if the job is eligible for soft-closing. If errors are encountered (such as the job is in one or more open batches), the screen displays a list of the errors.. You must correct all the items in the list before you can soft-close the job.
