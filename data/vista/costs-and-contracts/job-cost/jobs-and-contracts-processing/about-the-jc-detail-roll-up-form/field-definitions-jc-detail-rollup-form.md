# Field Definitions: JC Detail Rollup Form

The following is a list of field descriptions for the JC
 Detail Rollup form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Roll-Up Code

 Enter the roll-up code to process. Once entered, the screen displays a summary of the setup options you chose in JC Detail Roll-Up Codes.

[](/en/vista/vista/costs-and-contracts/job-cost/jobs-and-contracts-processing/about-the-jc-detail-roll-up-form)JC Detail Rollup

##  Beginning/Ending Job

 Enabled for Cost Detail roll-ups only where roll-up code applies to ‘Selected’ jobs.
 Specify the beginning and ending in a range of jobs to roll up. If the roll-up code applies to all jobs, defaults as ‘All’ and cannot be changed.

[](/en/vista/vista/costs-and-contracts/job-cost/jobs-and-contracts-processing/about-the-jc-detail-roll-up-form)JC Detail Rollup

##  Beginning/Ending Contract

 Enabled for Revenue Detail roll-ups only where roll-up code applies to ‘Selected’ contracts.
 Specify the beginning and ending in a range of contracts to roll up. If the roll-up code applies to all contracts, defaults as ‘All’ and cannot be changed.

[](/en/vista/vista/costs-and-contracts/job-cost/jobs-and-contracts-processing/about-the-jc-detail-roll-up-form)JC Detail Rollup

##  Batch Options: Roll-Up Only?

 Specify the status of contracts or jobs to include in the detail roll-up.

- Open – Select this option to roll up detail for open jobs (cost roll-up) or contracts (revenue roll-up) only.

- Closed – Select this option to roll up detail for closed jobs or contracts only.

- Both – Select this option to roll up detail for open and closed jobs or contracts.
