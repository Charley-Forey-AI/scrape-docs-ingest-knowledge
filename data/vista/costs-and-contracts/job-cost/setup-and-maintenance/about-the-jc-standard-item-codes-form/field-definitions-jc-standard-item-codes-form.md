# Field Definitions: JC Standard Item Codes Form

The following is a list of field descriptions for the JC
 Standard Item Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Std Item Region

 Enter the location or region, up to 6 characters that applies to this standard item code. For example: WA, OR, PDX, Fed.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-standard-item-codes-form)JC Standard Item Codes

##  Std Item Code

 Enter a code, up to 16 characters, that identifies this standard item. A standard item may be a state bid item.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-standard-item-codes-form)JC Standard Item Codes

##  Description

 Enter a description of this standard item, up to 60 characters.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-standard-item-codes-form)JC Standard Item Codes

##  Unit of Measure

 Specify the unit of measure (from HQ Units of Measure) that applies to this standard item. Contract items using standard item codes should use either the metric unit of measure or the unit of measure specified for the standard item.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-standard-item-codes-form)JC Standard Item Codes

##  Metric Unit of Measure

 Specify the unit of measure (from HQ Units of Measure) to use as the metric unit of measure for this standard item. Contract items using standard item codes should use either the metric unit of measure or the unit of measure specified for the standard item.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-standard-item-codes-form)JC Standard Item Codes

##  Std Unit Price

 Specify the standard unit price for this code if one exists. (Optional.)

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-standard-item-codes-form)JC Standard Item Codes
