# Field Definitions: JC Liability Template Form

The following is a list of field descriptions for the JC
 Liability Template form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Liability Template

 Enter a number (1-9999) to identify this liability template.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Description

 Enter the description, up to 30 characters, for this template.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Phase

 If you want the premium portion of overtime earnings redirected to a specific phase, enter that phase here. Otherwise, leave blank to have the premium portion of overtime earnings go to the phase specified when entering timecards (PR Timecard Entry).

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Premium Cost Type

 If you want the premium portion of overtime earnings redirected to a specific cost type, enter that cost type here. Otherwise, leave blank to have the premium portion of overtime earnings go to the cost type specified for the earnings codes in PR Earnings Codes.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Liability Type

 Enter a valid liability type (defined in HQ Liability Types) for this template. A liability type includes one or more Payroll liabilities.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Phase

 If you want the burdened costs associated with the liability type to be directed to a specific phase code, enter that phase code here. Otherwise, leave blank to have the burdened costs go to the same phase as the earnings.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Cost Type

 If you want the liability costs to be redirected to a specific cost type, enter that cost type here. Otherwise, leave blank to have the liability costs go to the same cost type as the earnings. You can use any valid cost type (from JC Cost Types), even if the cost type is not used on any of the phases set up for the associated job and the phases are not locked in PM Projects and JC Jobs.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

## Calculate Method

Indicate the method by which liability amounts will be calculated.

- R-Rate - Use the rate specified in
 the Liability Rate field (to the right).

Note: You will need to set up at least one basis earnings code for this liability type on the Basis Earnings Codes tab. If you do not, a warning displays (in red) indicating the basis earnings codes are missing. Message will clear once you enter an earnings code for the liability type.

- E-Exactly as Payroll - Charge
 burden to Job Cost based on exact amounts as calculated in Payroll.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Liability Type

 Enter a valid liability type (from HQ Liability Types) that has already been set up on the liability template.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

## Earn Codes

Enter the earnings code (set up in PR Earnings Codes) that identifies the earnings you want included in the basis of the calculation for this liability.
Note: If you set up an earnings code here and that earnings code is later deleted in PR Earnings Codes, the earnings code will not be deleted from this grid. You will need to delete it from the grid manually.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

##  Liability Rate

 Enabled only if Calculate Method is ‘Rate’.
 Enter the rate at which this liability will be calculated.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-liability-template-form)JC Liability Template

## Notes

Use this tab to enter any miscellaneous information about this item.
The space allowance is virtually unlimited.
Spelling Check
 Click the Spelling icon on the toolbar or select
 Tools > Spelling
  to spell check the text in this field.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field, right
 click the mouse while focus is in the field and select Standard
 Notes from the shortcut menu, which opens the Standard Note Copy window.
 Then enter the standard note to copy (or select from F4 lookup) and click OK. The
 system inserts the selected note into the field.
