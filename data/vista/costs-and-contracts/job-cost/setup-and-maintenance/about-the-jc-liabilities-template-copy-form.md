# About the JC Liabilities Template Copy Form

Use this form to copy one liability template to another.
When copying you can specify whether you want to
 copy the template from one company to another. You can access this form by selecting
 File > Copy
 Template from JC Liability Template.
The copy ‘from’ template automatically defaults the template on which you currently have focus; however, you can copy from any existing template. The copy ‘to’ template must be a new template—you cannot copy to an existing template. Once you click the Copy button, the copy process creates a new template with a description of “Copy of ##” (where ## represents the source template number), which may be overridden. The premium phase/cost type, as well as all of the liability types and earnings codes set up on the source template are copied to the new template. You can modify the new template as needed.
