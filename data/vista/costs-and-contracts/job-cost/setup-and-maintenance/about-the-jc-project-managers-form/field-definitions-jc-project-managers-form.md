# Field Definitions: JC Project Managers Form

The following is a list of field descriptions for the JC
 Project Managers form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Project Manager

 Enter a number, up to 10 digits, that identifies this project manager.
Note: Project managers are not validated against the employee file in Payroll. They are used only for assignment to jobs in the Job Master.  
[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers

##  Name

 Enter the name of the project manager, up to 30 characters.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers

##  Phone

 Enter the phone number for this project manager. Up to 20 characters allowed.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers

##  Fax

 Enter the Fax number for this project manager. Up to 20 characters allowed.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers

##  Mobile Phone

 Enter the mobile phone number for this project manager. Up to 20 characters allowed.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers

##  Pager

 Enter the pager number for this project manager. Up to 20 characters allowed.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers

##  Internet

 Enter the Internet address for this project manager.
Note: Clicking the Go button to the right of this field will take you directly to the web site indicated in this field.  
[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-project-managers-form)JC Project Managers
