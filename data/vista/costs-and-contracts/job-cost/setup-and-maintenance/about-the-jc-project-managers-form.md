# About the JC Project Managers Form

Use this program to create and maintain a list of project managers in the JC module.
In addition to the project manager’s name and phone number, space is provided to enter a mobile phone, pager, and email address.
The primary purpose of identifying the project managers is to facilitate writing reports either sorted by the name of a project manager or restricted to only those jobs that are the responsibility of a selected project manager.
Project managers are not validated against the employee file in Payroll. They are used only for assignment to jobs in the Job Master.
