#  About the JC Ins Template Initialize Form

This form can only be accessed by selecting the Initialize Phases option from the File menu of JC Insurance Template, and is used to initialize a range of phases to the template.
Initialization provides a quick method for adding phases to a template; however, you can still add phases manually if desired.
To initialize phases, specify the insurance code to assign to all
 initialized phases, specify the beginning and ending phase, then click OK to begin
 the initialization process. All phases within the specified range will be added to
 the Phases tab in JC Insurance Template, along with the assigned insurance code. You
 are then returned to JC Insurance Template, where you can modify insurance codes as
 necessary, and/or delete any unwanted phases.
