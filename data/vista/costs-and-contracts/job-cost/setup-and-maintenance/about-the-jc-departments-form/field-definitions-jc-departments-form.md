# Field Definitions: JC Departments Form

The following is a list of field descriptions for the JC
 Departments form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Department

 Enter a code for this department, up
 to 10 characters. Once added, a department code may be deleted only if it is not currently
 assigned to any contracts.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

## Description

Enter a description of the department. The value in this field can
 be up to 30 characters long.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

## Open Revenue Acct

Enter the liability or income account where revenue on open
 contracts should be posted.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Closed Revenue Acct

 Required.
 Specify the income account to which revenue will be posted after the contract is closed if posting to a closed job is allowed. JC Contract Close may also use this entry to move amounts from open to closed accounts.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Cost Types: Cost Type

 Enter the cost type for which to designate open/closed GL accounts
 for work-in-progress.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Cost Types: Open WIP Account

 Specify the asset or expense account
 to which costs for open jobs will be posted.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Cost Types: Closed WIP Account

 Specify the expense account into
 which all costs will be posted after the job is closed if posting to a closed job is
 allowed. JC Contract Close may also use this entry to move amounts from open to closed
 accounts.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Phase Overrides: Phase

 Enter a phase or press F4 to select one from a list.
The phase in this field does not have to be
 set up using the [JC Phases ](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-phases-form) form.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Phase Overrides: Open WIP Account

 Specify the GL account for open jobs that will be debited when
 posting costs to this phase.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Phase Overrides: Closed WIP Account

 Specify the GL account to debit when posting costs to this phase
 after the job is closed if posting to a closed job is allowed. JC Contract Close may also
 use this entry to move amounts from open to closed accounts.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Phase Overrides: Exclude Payroll

 Check this box if this phase
 override will not be used for transaction's with a Payroll source. In which case, the GL
 account specified for the earnings type, liability type, or cost type will be used.
 Leave this box unchecked if this phase override will always be used, regardless of the transaction's source.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Liability Types: Liability Type

 Specify a valid liability type for which separate accounting is
 required. This is only required if burden expense is to be posted to a separate GL account
 other than the cost type account.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Liability Types: Open Burden Acct

 Enter the account to debit for open jobs if burden expense is to
 go to a different account than the one assigned to the cost type.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Liability Types: Closed Burden Acct

 Enter the account to debit for closed jobs if burden expense is to
 go to a different account than the one assigned to the cost type. This will be used if you
 allow posting to closed jobs, and may also be used by JC Contract Close to move amounts
 from open to closed accounts.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Earnings Types: Earnings Type

 Only used if you assigned any special earnings types to earnings
 codes in Payroll so those earnings are posted to a different debit account than regular
 earnings.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Earning Types: Open Labor Acct

 Specify the expense account for open jobs to debit for any of the
 special earnings categories you are using.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments

##  Earnings Types: Closed Labor Acct

 Specify the expense account to debit for closed jobs for any of
 the special earnings categories you are using. This will be used if you allow posting to
 closed jobs, and may also be used by JC Contract Close to move amounts from open to closed
 accounts.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-departments-form)JC Departments
