# Field Definitions: JC Insurance Template Form

The following is a list of field descriptions for the JC
 Insurance Template form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Insurance Template

 Enter a number (1-9999) to identify this insurance template.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-insurance-template-form)JC Insurance Template

##  Description

 Enter a description for this insurance template, up to 30 characters.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-insurance-template-form)JC Insurance Template

##  Phase

 Enter the phase code for which to designate an insurance code. Timecards posted to this phase on a job using this template will automatically default the insurance code specified (to the right) for this phase. Timecards posted to phases not defined in the template will use the employee’s standard insurance code.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-insurance-template-form)JC Insurance Template

##  Insurance Code

 Specify the insurance code (from HQ Insurance Codes) that applies to this phase.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-insurance-template-form)JC Insurance Template

## Notes

Use this tab to enter any miscellaneous information about this item.
The space allowance is virtually unlimited.
Spelling Check
 Click the Spelling icon on the toolbar or select
 Tools > Spelling
  to spell check the text in this field.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field, right
 click the mouse while focus is in the field and select Standard
 Notes from the shortcut menu, which opens the Standard Note Copy window.
 Then enter the standard note to copy (or select from F4 lookup) and click OK. The
 system inserts the selected note into the field.
