# About the JC Insurance Template Copy Form

Use this form to copy one insurance template to another.

When copying you can specify whether you want to
 copy the template from one company to another, provided that the phase group is the
 same between both companies. You can access this form by selecting File > Copy
 Template in JC Insurance Template.
When the copy process is implemented, the new template is given a description of “Copy of ##” (which can be overridden), and all phases and insurance codes are copied to the new insurance template.
Note: The system will copy templates with phases that have been
 deleted from JC Phases as long as the valid part of the phase still exists. If a
 valid part of the phase does not exist, the system will copy the template, but will
 not include the phase.
