# Field Definitions: JC Allocations Initialize Form

The following is a list of field descriptions for the JC
 Allocations Initialize form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Include Soft Closed Jobs

 This field is available only when initializing Jobs to an allocation code (i.e. you selected File > Initialize Jobs from JC Allocation Codes).
 Check this box to include soft-closed jobs in the Available Jobs list.
 Leave this box unchecked to exclude soft-closed jobs from the Available Jobs list.
Note: Unchecking this box will automatically remove any soft-closed jobs that you added to “Jobs to be included in allocation” list box.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-allocations-initialize-form)JC Allocation Initialize

## Include Hard Closed Jobs

 This field is available only when initializing Jobs to an allocation code (i.e. you selected File > Initialize Jobs from JC Allocation Codes).
 Check this box to include hard-closed jobs in the Available Jobs list.
 Leave this box unchecked to exclude hard-closed jobs from the Available Jobs list.
Note: Unchecking this box will automatically remove any hard-closed jobs that you added to “Jobs to be included in allocation” list box.

## Notes

Use this tab to enter any miscellaneous information about the allocation code. The space allowance is virtually unlimited.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
