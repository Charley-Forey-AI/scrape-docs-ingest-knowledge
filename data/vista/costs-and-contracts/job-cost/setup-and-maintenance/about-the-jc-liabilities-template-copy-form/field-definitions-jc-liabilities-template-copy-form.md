# Field Definitions: JC Liabilities Template Copy Form

The following is a list of field descriptions for the JC
 Liabilities Template Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## From JCCo

Enter the JC company number that you are copying the template from.
This field initially defaults to the current JC company.

## From Template

Enter the template number that you are
 copying. Press F4 to see a list of valid templates. This field initially defaults the
 insurance template specified in JC Liability Template.

## To JCCo

Enter the JC company that you are copying the template to.
This field initially defaults to the current JC company.

## To Template

Enter a template number to copy settings to. The number must be a new template number; you cannot copy to an existing template.
