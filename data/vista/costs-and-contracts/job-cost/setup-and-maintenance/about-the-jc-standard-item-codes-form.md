# About the JC Standard Item Codes Form

Use this form to create and maintain standard item codes by region.
This has two uses:

-  Allows contract items to have
 both a sequential contract item number and the state bid item number

- Utility work where jobs are
 based on a standard price list of contracted items

For each standard item code, you assign
 a description, unit of measure, and metric unit of measure. A standard unit price
 may also be assigned. These will default on contract items when they are created.

- Add a standard item code to a contract item - See [Add a Standard Item Code to a Contract Item](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/add-a-standard-item-code-to-a-contract-item#task-9915--en__task-9915).

- Add standard item code to database if not on file box - The Add standard
 item code to database if not on file box on the JC Company Parameters form
 that changes the functionality of this feature. [More](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/jc-company-parameters-form/field-definitions-jc-company-parameters-form#ID-00016ed5--en)
