# Field Definitions: JC Allocations Initialize Form

The following is a list of field descriptions for the JC
 Allocations Initialize form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Include Soft Closed Jobs

 This field is available only when initializing Jobs to an allocation code (i.e. you selected File > Initialize Jobs from JC Allocation Codes).
 Check this box to include soft-closed jobs in the Available Jobs list.
 Leave this box unchecked to exclude soft-closed jobs from the Available Jobs list.
Note: Unchecking this box will automatically remove any soft-closed jobs that you added to “Jobs to be included in allocation” list box.
