# Field Definitions: JC Ins Template Initialize Form

The following is a list of field descriptions for the JC Ins
 Template Initialize form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Template

 Specify the template (from JC Insurance Template) for which you are initializing phases.

##  Insurance Code

 Specify the insurance code (from HQ Insurance Codes) to assign to all phases being initialized to this template.

##  Beginning/Ending Phase

 Specify the beginning and ending in a range of valid phases you want initialized to this insurance template. Then click OK to begin initialization.
