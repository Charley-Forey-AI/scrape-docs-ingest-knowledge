# Field Definitions: JC Fixed Rate Template Form

The following is a list of field descriptions for the JC
 Fixed Rate Template form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Rate Template

 Enter a number (1-9999) to identify this fixed rate template.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Description

 Enter a description of this fixed rate template, up to 30 characters.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

## Effective Date

Enter the effective date for the new rates defined on this template. Job timecards with a date that falls on or after this date will use the ‘new’ rates; timecards with a date prior to this date will use the ‘old’ rate.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Sequence

 Enter N, New, or + to add a new sequence. The system will automatically assign the next available sequential number.

[](/en/vista/vista/costs-and-contracts/job-cost/revenue/about-the-jc-revenue-adjustments-form)JC Revenue Adjustments

##  PR Co

 Required.
 Enter a valid Payroll company for this rate detail line.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Craft

 If restricting by craft, enter the craft (from PR Crafts) to which the fixed rate applies.
 Only timecards posted to this craft and meeting all other criteria will use the ‘old’ or
 ‘new’ rate specified to the right.
Note: Entry in this field is required if you are restricting by Class.  
[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Class

 If restricting by class, enter the class to which the fixed rate applies. Only timecards posted to this class and meeting all other criteria will use the ‘old’ or ‘new’ rate specified to the right.
Note: If restricting by class, you must also specify a craft (previous field).

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Shift

 If restricting by shift, enter the shift to which the fixed rate applies. Only timecards posted to this shift and meeting all other criteria will use the ‘old’ or ‘new’ rate specified to the right.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Earn Factor

 If restricting by earnings factor, enter the earnings factor to which the fixed rate applies. Only timecards posted to this earnings factor and meeting all other criteria will use the ‘old’ or ‘new’ rate specified to the right.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Employee

 If restricting by employee, enter the employee to which the fixed rate applies. Only timecards posted to this employee and meeting all other criteria will use the ‘old’ or ‘new’ rate specified to the right.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  Old Rate

 Enter the old rate for this template detail sequence. Timecards posted with a date prior to the Effective Date specified for the template (on the Info tab) will use this rate.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

##  New Rate

 Enter the new rate for this template detail sequence. Timecards posted with a date on or after the Effective Date specified for the template (on the Info tab) will use this rate.

[](/en/vista/vista/costs-and-contracts/job-cost/setup-and-maintenance/about-the-jc-fixed-rate-template-form)JC Fixed Rate Template

## Notes

Use this tab to enter any miscellaneous information about this item.
The space allowance is virtually unlimited.
Spelling Check
 Click the Spelling icon on the toolbar or select
 Tools > Spelling
  to spell check the text in this field.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field, right
 click the mouse while focus is in the field and select Standard
 Notes from the shortcut menu, which opens the Standard Note Copy window.
 Then enter the standard note to copy (or select from F4 lookup) and click OK. The
 system inserts the selected note into the field.
