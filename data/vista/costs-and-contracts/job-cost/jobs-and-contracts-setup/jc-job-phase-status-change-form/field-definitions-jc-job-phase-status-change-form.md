# Field Definitions: JC Job Phase Status Change Form

The following is a list of field descriptions for the JC Job
 Phase Status Change form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Job

The Job field on the JC Job Phase Status Change
 form.
Specify the job (from [JC Jobs](/en/vista/vista/costs-and-contracts/job-cost/jobs-and-contracts-setup/jc-jobs-form)) for which to globally change phase status.

## Change By

The Change By radio buttons on the JC Job Phase Status Change
 form.

Select the change by option for this phase status change:

- All Phases - Select this option to change the status for all phases on the
 selected job.

- Phase Range - Select this option to specify a range of phases to include in the
 status change.

## Phase Range

The Beginning/Ending Phase fields on the JC Job Phase Status
 Change form.

The Beginning Phase and Ending Phase fields are enabled only when the Phase Range radio
 button is selected in the Change By section.
Enter the beginning and ending phase in a range of phases for which to change the active
 status. The system will update the status for all phases within this phase range.

## Status Change

The Status Change radio buttons on the JC Job Phase Status
 Change form.

Select the status to apply to the selected phases:

- Active - Select this option to set all selected phases to an "active" status.
 Once active, you can post costs, estimates, and commitments to the selected
 phases.

- Inactive - Select this option to set all selected phases to an "inactive"
 status. Once inactive, you can no longer posts costs, estimates, or commitments
 to the selected phases.
