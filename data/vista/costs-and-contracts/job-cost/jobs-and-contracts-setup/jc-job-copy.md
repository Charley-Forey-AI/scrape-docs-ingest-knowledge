# JC Job Copy

Use the JC Job Copy form to copy an existing job to a new job.
You might typically use this feature when you have multiple jobs that are similar or to create sub-jobs of master jobs.
If you are using data level security for jobs and/or
 contracts, the destination job and contract will inherit the security
 groups assigned to the source job and contract. You may override these
 assignments in the related programs. For more information, see [Job Level Security](/en/vista/vista/costs-and-contracts/job-cost/job-level-security).
