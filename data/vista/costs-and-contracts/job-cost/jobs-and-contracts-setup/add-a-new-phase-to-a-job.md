# Add a New Phase to a Job

Follow the steps below to add a new phase to the job.

1. Enter or select a job in
 the Job field on the JC Original Estimates form.

1. Select a phase that
 isn't already associated with the job using the Phase field. The JC Add New Phase form displays.

1. Enter the contract item
 that is associated with the new phase in the Item field. You cannot add a new contract item using this
 form.

1. Click OK.The JC Add New Phase form closes.

1. In the Cost Type field, enter the
 cost type for the new phase or click Initialize to add all cost
 types assigned to the phase (in PM Phases).

The system adds the new phase and cost type(s) to the job.
