# Field Definitions: PC Create and Send Form

The following is a list of field descriptions for the PC
 Create and Send form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Vendor

Enter the vendor by which to filter the potential bidders in the Bid List grid. Only contacts for this vendor meeting all other filter criteria will be displayed. Leave blank if not filtering by vendor.
Note: The vendors/contacts displayed in the Bid List grid
 will depend on how you set the Related to Bid Package checkbox. If
 checked, the grid will display all vendors/contacts defined as potential bidders for the
 specified bid package. If unchecked, the grid will display all vendors/contacts defined as
 potential bidders from all bid packages for the project.

## Contact

Enter the contact sequence by which to filter the potential bidders in the Bid List grid. Only contacts with this sequence number meeting all other filter criteria will be displayed. Leave blank if not filtering by contact.
Note: The vendors/contacts displayed in the Bid List grid
 will depend on how you set the Related to Bid Package checkbox. If
 checked, the grid will display all vendors/contacts defined as potential bidders for the
 specified bid package. If unchecked, the grid will display all vendors/contacts defined as
 potential bidders from all bid packages for the project.

## Scope

Enter a valid scope (from PC Scope Codes) by which to filter potential bidders in the Bid List grid. Only vendors/contacts assigned this scope (in PC Pre-Qualification Info, Scope/Phases tab) and meeting all other filter criteria will be displayed. Leave blank if not filtering by scope.

## Phase

Enter a valid phase (from JC Phases) by which
 to filter potential bidders in the Bid List grid. Only vendors/contacts assigned this phase
 (in PC Pre-Qualification Info, Scope/Phases tab) and meeting all other filter criteria will
 be displayed. Leave blank if not filtering by phase.

## Message Status

From the drop-down menu, select the message status (assigned by the system) by which to filter potential bidders in the Bid List grid. Available options are:

- N - Invite Not Sent

- S - Invite Sent

- F - Invite Failed

 Only vendors/contacts assigned the selected status and meeting all other filter criteria will be displayed. Leave blank if not filtering by message status.

## Bid Response

From the drop-down menu, select the bid response (assigned to vendors/contacts in PC Bid Package, Bid List tab) by which to filter potential bidders in the Bid List grid. Available options are:

- W - Will Bid

- D -Declined

- U - Undecided

- N - No Response

 Only vendors/contacts assigned this bid response and meeting all other filter criteria will be displayed. Leave blank if not filtering by bid response.

## Reply To Email:

This field only applies to emails sent using SMTP. This field will not work on emails sent using MS Outlook.
Enter the email address that is used to process the response emails. Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/about-pc-module-email-integration) for an outline of using SMTP with the email integration feature.

## Subject

Enter a subject line for this email message. Up to 998 characters allowed. Initially defaults a description of "Notification for XXX", where XXX represents the project name.
Note: There is a token that will be appended to the subject line. This token will be used by the Outlook Add-In to associate the reply email with the parent record. This is part of the PC module email integration feature. Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/about-pc-module-email-integration) for more information.

## Message

Use this text box to enter the message that will appear in the body of the email or fax message. Space is virtually unlimited.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
