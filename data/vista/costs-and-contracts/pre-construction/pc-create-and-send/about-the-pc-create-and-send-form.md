# About the PC Create and Send Form

Use this form to create and send documents such as bid invitations, addenda, or general notifications to vendors that are associated with a potential project.
This form generates documents using the standard or customized document templates in the PM module, and then it prints, faxes, or emails those documents to a list of vendors. This feature is very similar to the Create and Send feature in the PM module.Note: You can access this form from the PC Bid Package form by clicking the Message icon
 and selecting Select a Template or Create without a Template.
