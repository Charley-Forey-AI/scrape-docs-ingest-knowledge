# Document Templates

Viewpoint provides the standard templates for sending bid invitations, addendums, and notifications to contacts on your bid lists.
These include:

- Invitation to Bid

- Addendum

- Notification

These templates cannot be deleted or modified, but you can copy one using [PM Copy Document Template ](/en/vista/vista/project-management/create-and-send/pm-copy-document-template-form) and then modify the copy using [PM Create & Send Templates ](/en/vista/vista/project-management/create-and-send/pm-create--send-templates-form).
Click [here](/en/vista/vista/project-management/create-and-send/about-creating-a-document-template) for an overview on creating a customized document template using a standard template.
