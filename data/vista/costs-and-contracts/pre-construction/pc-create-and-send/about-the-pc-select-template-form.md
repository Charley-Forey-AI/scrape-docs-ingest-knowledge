# About the PC Select Template Form

Use this form to select which document template should be used to generate the document that you want to send.
For example if you want to send an Invitation to Bid automated response form, select that document template from the this.
Note: You can access this form by clicking the Create and Send icon on [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form) or [PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form), and then choosing Select a Template.
The Template list box displays all of the templates that you can select, including standard templates (Invitation to Bid, Addendum, or Notification), and any custom templates that you have created. Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/document-templates) for information on creating custom templates.
Highlight a template and click OK.
 This will open [PC Create and Send](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/about-the-pc-create-and-send-form) and attaches the selected
 document. Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/about-the-pc-create-and-send-form) for more information on using the Create and
 Send feature in the PC module.
