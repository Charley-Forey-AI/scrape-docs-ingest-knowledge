# Field Definitions: PC Region IDs Form

The following is a list of field descriptions for the PC
 Region IDs form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Region Code

Enter a code, up to 10 characters, to identify a geographical region. Codes defined here will be used in the PC Pre-Qualification Info form (Regions tab) to identify the regions in which a subcontractor has performed/or is scheduled to perform work.
Examples:

State

- CA

- OR

- WA

County

- OrangeCty

- MarionCty

- LinnCty

Directional

- NE

- NW

- SE

- SW

##  Description

 Enter a description of this geographic region, up to 60 characters.

## Active Lookup

Check this box if this geographic region is active and will be included in geographical region lookups.
Leave this box unchecked if this geographic region is inactive and will be excluded from geographical region lookups.
Note: Setting a geographic region to inactive does not prevent its use when setting up regions in PC Pre-Qualification Info (Regions tab).
