# Field Definitions: PC Contact Types Form

The following is a list of field descriptions for the PC
 Contact Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Contact Type Code

Enter a code, up to 10 characters, to identify a contact type. The types defined here will be used when setting up contacts for vendors in the PC Pre-Qualification Info form (Contacts tab).
Examples:

- Primary

- Secondary

- Estimating

- Scheduling

- Other

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-contact-types-form)PC Contact Types

##  Description

 Enter a description for this contact type, up to 60 characters.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-contact-types-form)PC Contact Types

## Active Lookup

Check this box if this contact is active and will be included in Contact lookups.
Leave this box unchecked if this contact is inactive and will be excluded from Contact lookups.
Note: Setting a contact type to inactive does not prevent its use when setting up contacts in PC Pre-Qualification Info (Contacts tab).
