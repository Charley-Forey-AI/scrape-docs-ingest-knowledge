# Field Definitions: PC Reference Types Form

The following is a list of field descriptions for the PC
 Reference Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Reference Type Code

Enter a code, up to 10 characters, that identifies a type of work/business reference. The types defined here will be used when setting up references for subcontractors for vendors in the PC Pre-Qualification Info form (References tab).
Examples:

- Supplier

- Contractor

- Sub

##  Description

 Enter a description of this reference type, up to 60 characters.

## Active Lookup

Check this box if this reference type is active and will be included in reference type lookups.
Leave this box unchecked if this reference type is inactive and will be excluded from reference type lookups.
Note: Setting a reference type to inactive does not prevent its use when setting up references in PC Pre-Qualification Info (References tab).
