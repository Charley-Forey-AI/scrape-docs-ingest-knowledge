# About the PC Reference Types Form

Use this form to set up reference type codes. Reference types identify the different types of work/business references set up for a subcontractor.
Note: Reference types are associated with a vendor group (defined
 in HQ Company Setup, AP Vendors and PM Firm Codes field);
 therefore, PC companies pointing to the same vendor group will share reference
 types.
Examples:

- Supplier

- Contractor

- Subcontractor

Once you have defined reference types here, you can assign them to work/business references set up for the subcontractor in the PC Pre-Qualification Info form (References tab).
