# About the PC Region IDs Form

Use this form to set up geographical region codes.
Region codes are used to identify the different areas in which subcontractors perform work. These areas can represent states, counties, directions, countries, etc.
Note: Region codes are associated with a vendor group (defined in
 HQ Company Setup, AP Vendors and PM Firm Codes field);
 therefore, PC companies pointing to the same vendor group will share region
 codes.
Examples:
State

- CA

- OR

- WA

County

- OrangeCty

- MarionCty

- LinnCty

Directional

- NE

- NW

- SE

- SW

Once you set up region codes here, use them in the PC Pre-Qualification Info form to identify the regions in which a subcontractor has worked and/or is scheduled to work, as well as what percentage of previous work was done and/or is to be done in each region.
