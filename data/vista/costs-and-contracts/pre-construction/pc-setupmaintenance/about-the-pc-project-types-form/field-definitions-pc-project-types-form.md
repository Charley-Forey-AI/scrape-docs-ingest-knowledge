# Field Definitions: PC Project Types Form

The following is a list of field descriptions for the PC
 Project Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Project Type Code

Enter a code, up to 10 characters, to identify a type of project. The types defined here will be used in PC Pre-Qualification Info (Proj Types tab) to identify the types of work a subcontractor has done or is scheduled to do.
Examples:

- Casino

- Resort

- Industrial

- Commercial

- Correctnl

- Retail

##  Description

 Enter a description of the project type, up to 60 characters.

## Active Lookup

Check this box if this project type is active and will be included in project type lookups.
Leave this box unchecked if this project type is inactive and will be excluded from project type lookups.
Note: Setting a project type to inactive does not prevent its use when setting up project types in PC Pre-Qualification Info (Proj Types tab).
