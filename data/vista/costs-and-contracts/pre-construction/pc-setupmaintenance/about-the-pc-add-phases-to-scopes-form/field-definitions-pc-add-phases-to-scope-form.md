# Field Definitions: PC Add Phases to Scope Form

The following is a list of field descriptions for the PC Add
 Phases to Scope form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Scope

Enter the scope code for which to initialize selected phases. Initially defaults the scope code selected in PC Scope Codes.
