# About the PC Project Types Form

Use this form to set up project type codes. Project types are
 used to identify the different types of jobs your company works on.
Note:Project types are associated with a vendor group
 (defined in HQ Company Setup, AP Vendors and PM Firm Codes
 field); therefore, PC companies pointing to the same vendor group will share project
 types.

Examples:

- Casinos

- Clubhouse/Resorts

- Commercial

- Correctional Facilities

- Healthcare

- Industrial Building

- Retail

- Sports/Entertainment

Once you set up project types, use [PC Pre-Qualification Info](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form) to identify the types of projects a subcontractor has done and/or is scheduled to do, as well as what percentage of their previous and/or scheduled work consists of each project type (e.g. 1% of their jobs are casinos, 3% of jobs are hospitals, etc.).
