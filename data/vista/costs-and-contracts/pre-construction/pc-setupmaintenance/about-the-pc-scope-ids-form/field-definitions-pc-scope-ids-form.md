# Field Definitions: PC Scope IDs Form

The following is a list of field descriptions for the PC
 Scope IDs form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Scope Code

Enter a code, up to 10 characters, to identify a scope of work. Codes set up here will be used in the PC Pre-Qualification form (Scope tab) to identify the scopes of work a subcontractor has performed or is scheduled to perform.
Examples:

- Site

- Concrete

- Roofing

- Electrical

- Flagging

##  Description

 Enter a description of this scope of work, up to 60 characters.

## Active Lookup

Check this box if the scope is active and should be included in scope code lookups.
Leave this box unchecked if the scope is inactive and should be excluded from scope code lookups.
Note: Setting a scope code to inactive does not prevent its use when defining scopes of work in PC Pre-Qualification Info (Scope tab).

## Phase

Enter a phase (from JC Phases) that identifies
 a division of work associated with this scope.
