# Create PM Subcontract Detail

You can create PM module subcontract detail using the awarded vendor bids on a PC potential project after that potential project has been used to create a PM project.
Note: Before you can follow the steps in this process, you must create a PM module project using the potential project. For information, [click here.](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/potential-project-conversion)
Follow the steps below to create PM module subcontract detail using the awarded bids on a potential project.

1. Open a potential project in [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).

1. Open the [PC Bid Coverage](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-coverage-form) tab to view the vendor bids on the potential project.

1. Make sure that there are awarded vendor bids on the potential project. These bids will be used to create the subcontract detail. Note: Check the Bid Awarded box to award a bid to a vendor. Amounts entered into the Bid Amount  field will be used to create the subcontract detail. If there is no amount entered in the  Bid Amount field, the subcontract detail will be created with no amount, but you will be able to modify the subcontract detail after it has been created in [PM Subcontract Detail ](/en/vista/vista/project-management/subcontracts/pm-subcontract-detail-form). Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-coverage-form) for more information on awarding and maintaining bids on the Bid Coverage tab.

1. Click the Tasks icon and select Create PM Subcontract Detail from the menu that appears. This will open [PC Create PM Project](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/potential-project-conversion). Note: You can only access PC Create PM Project if you have form security set up to add records to PM Projects. Form security is set up using [VA Form Security ](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-form-security-form).

1. Enter the PM project number in the Project field or press F4 to select it from a list. This is the PM project associated with the subcontract detail that will be created.

1. Check the Create Subcontract Detail box.

1. Enter the cost type that should be assigned to the subcontract detail in the Cost Type field. You cannot change the cost type assigned to the subcontract detail once it has been created. Note: By default, the Cost Type field will populate with the subcontract cost type set up in the SL Cost Type 1 field on the Subcontract Parameters tab of [PM Company Parameters ](/en/vista/vista/project-management/setupmaintenance/pm-company-parameters-form). If the SL Cost Type 1 field is blank, it will display the value in the SL Cost Type 2 field.

1. Click OK when complete to create the subcontract detail.

1. An information window will display the number of subcontract detail items created by the form. Click Yes to view the subcontract detail in [PM Subcontract Detail ](/en/vista/vista/project-management/subcontracts/pm-subcontract-detail-form). Click [here](/en/vista/vista/project-management/subcontracts/about-subcontract-buyout) for an overview of the Subcontract Buyout process in the PM module. Note: By default, the subcontract detail will be assigned a lump sum unit of measure but you can change the UM using PM Subcontract Detail if it does not apply.
Note: If there are inclusions and exclusions on the scopes/phases on the bid package associated with the awarded bid used to create the subcontract detail, those inclusions and exclusions will be copied to the PM subcontract and will display on the new Inclusions/Exclusions tab of PM Subcontracts.
