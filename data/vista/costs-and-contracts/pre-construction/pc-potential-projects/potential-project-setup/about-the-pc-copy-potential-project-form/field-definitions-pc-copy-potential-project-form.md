# Field Definitions: PC Copy Potential Project Form

The following is a list of field descriptions for the PC Copy
 Potential Project form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Potential Project to Copy

Enter the potential project to copy or press F4 to select it from a list. This is the potential project that contains the information that you would like to copy into another potential project.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Copy to new Potential Project / Copy to existing Potential Project

Check the
 Copy to new Potential Project
 box to create a new potential project and copy the selected information into it. This will enable the
 ID
 and
 Name
 field to enter the new potential project number and name.
Check the Copy to existing Potential Project box to
 copy the selected information into an existing potential project. This will enable the
 ID field
 to select the existing potential project.
Note: When using the Copy to existing Potential Project box, be
 aware that the information you copy to the existing potential project will overwrite existing
 information. For more information on how the information will be overwritten, refer to[PC Copy Potential Project](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form).
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## ID

Enter the id of the potential project that you would like to create.
Note: You cannot select a project id that has already been used because you are creating a new potential project. Press F4 to view the potential projects that have already been created.
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Name

Enter the name of the potential project that
 you are going to create. This field will populate with the project name of the potential
 project selected in the ID field, but it can be modified.
The name of a potential project is entered
 into the Project
 Name field on the Info tab in [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## ID

Enter the id of the potential project that the information should be copied into or press F4 to select it from a list.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Info

Check this box if the information on the Info
 tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form) should be copied from the potential
 project selected in the Potential Project to Copy field and
 pasted into the selected potential project.
Information on the Info tab of PC Potential
 Projects includes: project name and description, site location, project details like
 project status, type and size, and bid result information.
Note: If you are copying information to an existing
 potential project, checking this box means that any information already entered on the Info
 tab in PC Potential Project will be erased and then overwritten with the copied
 information.
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Bid Info

Check this box if the information on the Bid
 Info tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form) should be copied from the potential
 project selected in the Potential Project to Copy field and
 pasted into the selected potential project.
Information on the Bid Info tab of PC
 Potential Projects includes bid information like the bid day, and pre-bid meeting
 instructions, as well as the cost breakdown.
Note: If you are copying information to an existing project,
 checking this box means that any information already entered on the Bid Info tab in PC
 Potential Project will be erased and then overwritten with the copied information.
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## M/WBE Goals

Check this box if the information on the M/WBE
 tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form) should be copied from the potential
 project selected in the Potential Project to Copy field and
 pasted into the selected potential project.
Note: If you are copying information to an existing potential project, checking this box means that any certificates already entered on the M/WBE tab in PC Potential Project will be erased and then overwritten with the copied information.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Project Team

Check this box if the project team members on
 the potential project selected in the Potential Project to Copy field should be
 copied and pasted into the selected potential project.
Note: If you are copying team members to an existing
 project, checking this box means that any team members already entered on potential project
 will be erased and then overwritten with the copied information.
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Bid Packages

Check this box if the bid packages on the
 potential project selected in the Potential Project to Copy field should be
 copied and pasted into the selected potential project.
Bid packages are created and maintained on a potential project using the Bid Packages tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).
Note: If you are copying bid packages to an existing
 project, checking this box means that any bid packages already entered on the existing
 project will be erased and then overwritten with the copied information.
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project

## Forecast Info

Check this box if the information on the
 Forecast Info tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form) should be copied from the potential
 project selected in the Potential Project to Copy field and
 pasted into the selected potential project.
Information on the Forecast Info tab of PC
 Potential Projects includes: start and completion date, forecasted revenue, cost and
 profit, and the projected chase amount.
Note: If you are copying information to an existing project, checking this box means that any forecast information already entered on the Forecast Info tab in PC Potential Project will be erased and then overwritten with the copied information.
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-copy-potential-project-form)PC Copy Potential Project
