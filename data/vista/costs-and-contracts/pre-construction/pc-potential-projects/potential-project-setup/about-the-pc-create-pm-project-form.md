# About the PC Create PM Project Form

This form can be launched by click the Task icon, and selecting Create PM Project or
 Create PM subcontract Detail.
Use this form to:

-  Create a new PM project from an existing PC potential project. [More](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/potential-project-conversion)

- Create PM subcontract detail after a PM project has been created from a PC potential project. [More](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/potential-project-conversion/create-pm-subcontract-detail)

Note: If a user does not have form security set up to add
 records to PM Projects, they will not be able to access this form. Form security is set up
 using [VA Form Security ](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-form-security-form).
