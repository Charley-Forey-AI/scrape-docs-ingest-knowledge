# About the PC Copy Potential Project Form

This form is used to copy information from a potential project
 to either a new potential project or an existing potential project, and it is intended to
 save time and reduce data entry.
For example, if
 you need to input two similar potential projects, you can input the information for
 one potential project and then copy the general project information, bid
 information, project team members, forecast information, and M/WBE Goals to the
 similar potential project.
This
 form can be opened from the main PC module Program folder or by selecting
 File > Copy
 Potential Project in [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).
