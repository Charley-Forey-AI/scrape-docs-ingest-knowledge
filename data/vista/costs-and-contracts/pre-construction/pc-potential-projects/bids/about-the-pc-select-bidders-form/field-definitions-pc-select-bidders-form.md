# Field Definitions: PC Select Bidders Form

The following is a list of field descriptions for the PC
 Select Bidders form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Bid Package

Enter the bid package that you want to
 generate the bid list for or press F4 to select it from a list.
If you accessed PC Select Bidders by clicking
 the Select
 Bidders button on [PC
 Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form), this field will be enabled and you must enter a bid package.
If you accessed PC Select Bidders by clicking
 the Select
 Bidders button on [PC Bid
 Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form), this field will be disabled and it will display the bid package selected
 in PC Bid Package.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Scope

Enter a scope or press F4 to select one from a list. Only vendors associated with this scope on the Scope/Phase tab of [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form) will be included in the search results.
By default, this field will populate with the
 scopes on the bid package selected in the Bid Package field. If "Multiple"
 displays, there is more than one scope on the bid package, and they will all be used to
 filter the vendors that display in the search results.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Phase

Enter a phase or press F4 to select one from a list. By default this field will populate with the phases set up on the bid package using the Scope/Phase tab on[PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form). If "Multiple" displays, there is more than one phase on the bid package and they will all be used to filter the vendors that display in the search results.
Only vendors associated with this phase will be included in the search results. Vendors are associated with a phase using the Scope/Phase tab on [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## M/WBE

Enter an M/WBE certificate type or press F4 to select one from a list. By default this field will display the M/WBE certificate associated with the potential project using the M/WBE Goals tab on [PC Potential Projects.](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form) If "Multiple" displays, there is more than one M/WBE certificate on the potential project and they will all be used to filter the vendors that display in the search results.
Only vendors associated with the selected certificate type will be included in the search results. Vendors are associated with a certificate type using the Certificates tab on[PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).
Certificates are created and maintained using [PC M/WBE Certificate Types](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-mwbe-certificate-types-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Vendor

Enter a vendor or press F4 to select one from a list. Only contacts for this vendor will display in the search results.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Contact

Enter a contact or press F4 to select one from
 a list. This field allows you to filter the search results by a single vendor contact. Be
 sure to select a vendor in the Vendor field before using this field.
Contacts are added to a vendor using the
 Contacts tab on [PC Pre-Qualification Info. ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form)

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Contact Type

Enter a contact type or press F4 to select one from a list. Only vendor contacts assigned the selected contact type will be included in the search results.
Vendor contacts are assigned a contact type using the Contacts tab in [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Country

Enter the country or press F4 to select one from a list. By default, this field will populate with the site location country, which is set up on the Info tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).
Only vendors associated with this country will be included in the search results. Vendors are associated with countries and states using the States tab in [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).
Countries are created and maintained using [HQ Countries .](/en/vista/vista/administration/headquarters/region-setup/create-country-codes)

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## State/Province

Enter a state or province or press F4 to select
 one from a list. By default, this field will populate with the site location state or
 province, which is set up in the State field on the Info tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).
Only vendors associated with the selected
 state or province will be included in the search results. Vendors are associated with a
 state or province using the States tab in [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).
States are created and maintained in [HQ States ](/en/vista/vista/administration/headquarters/region-setup/hq-states-form).
Note: If you entered a value in the Country field,
 the selected state/province must be associated with the correct country. States/provinces
 are associated with countries using [HQ States ](/en/vista/vista/administration/headquarters/region-setup/hq-states-form).
[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Region

Enter a region or press F4 to select one from a list. By default this field will populate with the site location region, which is set up on the Info tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).
Only vendors associated with the selected region will be included in the search results. Vendors are associated with regions using the Regions tab in [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).
Regions are created and maintained using [PC Region IDs ](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-region-ids-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Project Type

Enter a project type or press F4 to select one from a list. By default this field will populate with the project type set up on the potential project using the Info tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form)
Only vendors associated with the selected project type will display in the search results. Vendors are associated with a project type using the Project Types tab of [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).
Project type are created and maintained using [PC Project Types ](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-project-types-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Largest Project: Ever

Enter a minimum and/or maximum contract value
 to filter vendors by the largest project they have ever performed.
Largest project information is added to a
 vendor using the $
 Value of Work field in the Largest Project Ever section on the Project tab
 of [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form)

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Largest Project: Last Year

Enter a minimum and/or maximum contract value to filter vendors by the largest project they performed during the last year.
Largest project information is added to a
 vendor using the $
 Value of Work field in the Largest Project Last Year section on the Project
 tab of [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form)

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Bid Contacts Only

Check this box to include only vendor bid
 contacts in the search results. Vendor contacts are set up as bid contacts by checking the
 Bid
 Contact box on the Contacts tab of [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).
If this box is not checked, all vendor contacts set up on the Contacts tab of PC Pre-Qualification Info will be included in the search results.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Qualified Vendors Only

Check this box to include only qualified vendors in the search results.
Vendor are set up as qualified vendors by
 checking the Qualified box on the Additional Information tab of [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form).

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders

## Preferred Vendors Only

Check this box if only preferred vendors should display in the search results.
A vendor is set up as preferred using the
 Preferred
 box on the Additional Info tab on the [PC Pre-Qualification Info ](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/subcontractor-prequalification/about-the-pc-pre-qualification-info-form) form.

[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form)PC Select Bidders
