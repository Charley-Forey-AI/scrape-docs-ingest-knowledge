# Field Definitions: PC Bid Package Copy Form

The following is a list of field descriptions for the PC Bid
 Package Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Copy Bid List

Check this box to copy the bid list of the source bid package to the new bid package.

## Copy Inclusions \ Exclusions

Check this box to copy the inclusions and exclusions on the source bid package to the new bid package.
Inclusions and exclusions are added to a bid package using the Inclusions/Exclusions tab on PC Bid Package.

## Copy Scope \ Phase

Check this box to copy the scopes and phases on the source bid package to the new bid package.

## Destination Bid Package

Enter a bid package number for the copied bid package. You cannot use the copy feature to overwrite or add information to an existing bid package, so you cannot select an existing bid package in this field.
You can press F4 to see a list of bid packages that have already been created.

## Destination Potential Project

Enter the potential project that the bid package should be copied to. This field will default to the current potential project.
Press F4 to select the potential project from a list.

## Source Bid Package

Enter the bid package that you would like to copy or press F4 to select it from a list.

## Source Potential Project

Enter the potential project that contains the bid package you would like to copy or press F4 to select it from a list.
This field will default to the current potential project, but you can change it if it does not apply.
