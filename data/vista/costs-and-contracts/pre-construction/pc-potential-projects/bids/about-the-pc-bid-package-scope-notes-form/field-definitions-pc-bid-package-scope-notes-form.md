# Field Definitions: PC Bid Package Scope Notes Form

The following is a list of field descriptions for the PC Bid
 Package Scope Notes form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Sequence

This field displays the inclusion/exclusion sequence number, which is a sequential number assigned when a new inclusion or exclusion is created.
Enter '+' or 'N' in this field to create a new inclusion or exclusion. This will create a new record and assign it the next available number.

## Type

Select if this record is an inclusion or exclusion.

## Scope

Enter a scope code or press F4 to select one from a list.
You can only select a scope that has already been assigned to the bid package.
You can add a scope or phase to a bid package using the Scope/Phase tab on [PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form).

## Phase

Enter a phase code or press F4 to select one from a list.
You can only select phases that have already been added to a bid package.
You can add a scope or phase to a bid package using the Scope/Phase tab on [PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form).

## Detail

Enter detail information about the inclusion/exclusion.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.

## Notes

Enter any notes on the bid package inclusion/exclusion record.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.
