# About the PC Add Scopes & Phases Form

Use this form to add scopes and phases to a bid package.
Scopes and phases represent the areas of work that are included in a bid package.
You can access this form by:

- Clicking the
 Add
 Scopes & Phases
 button on [PC Bid
 Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form)

- Clicking the
 Add
 Scopes & Phases
 button on the Bid Packages tab of [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form)

Before using this form, scope codes and their
 associated phases must be set up using [PC
 Scope Codes.](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-scope-ids-form)
Adding scopes and phases to a bid package
Use the Scope field to filter the grid and
 only display the phases associated with the selected scope code, and then highlight the
 scopes and phases that should be added to the bid package and click the Add to Bid
 Package button.
Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/adding-scopes-and-phases-to-a-bid-package) for more detailed information on adding scopes and phases to a bid package.
