# Field Definitions: PC Bid Coverage Form

The following is a list of field descriptions for the PC Bid
 Coverage form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Bid Response

Use the drop down menu to select the bid response. The selection in this field will affect the color of the selected row in the grid.
This field is also available on the Bid List tab of [PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form) when you add vendors to the bid list on a bid package.

## Bid Received

Check this box if the bid from the selected
 vendor was received.
Note: If Bid Response is set to No Response, you
 will not be able to check this box and save the change.

## Bid Awarded

Check this box if the bid has been awarded to the selected vendor.
When this box is checked, the line item will create subcontract detail items when use PC Create PM Project to create subcontract detail.

## Bid Amount

Enter the bid amount of the selected vendor.
If this bid is awarded and you use it to create a subcontract detail item, the bid amount will become the subcontract detail item amount.
Note: This field will not be enabled if 'No Response' is
 selected in the Bid
 Response field.
