# About the PC Copy Bid Package Form

Use this form to copy a bid package on a potential project.
This can reduce data entry and improve efficiency when setting
 up new bid packages. For example, if you are creating a bid package and a similar bid
 package has already been set up, you can copy the existing bid package so that you don't
 have to create the new bid package from scratch.Select File > Copy Bid
 Package from [PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form) to launch this form.
Note: PC Bid Package is launched from the Bid Packages
 tab of[
 PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form).
