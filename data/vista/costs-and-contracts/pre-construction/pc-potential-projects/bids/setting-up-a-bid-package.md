# Setting up a Bid Package

You can use the PC Bid Package form to create and maintain the bid packages on a potential project.
You can have more than one bid package on a
 potential project, so you can create a bid package for each area of work (for example, site
 work, structural, heating and cooling, etc.), or you can create a single bid package for
 all work that needs to be bid on.To reduce
 data entry and improve efficiency, you can also copy an existing bid package rather than
 creating a new one from scratch. Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-copy-bid-package-form) for information on copying an existing bid package.
Follow the steps below to create a bid
 package.

1. From the Pre-Construction Programs menu, open [PC Potential Projects](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/about-the-pc-potential-projects-form). 

1.  Enter the project you are setting up the bid package for in the Potential Project ID field or press F4 to select it from a list.

1.  Click Set Up Bid Package. This will open [PC Bid Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form). 

1. In the Bid Package ID field, enter a number or code to identify the bid package.

1.  Enter the name of the bid package in the Name field.

1. Enter details about the bid package in the Description field. For example the areas of work included in the bid package.

1. If bidding is sealed for this project, check the Sealed Bid box.

1. In the Bids Due section, enter the date and time that all bids are due. Note: You can enter the date manually or by clicking the calendar button and selecting the correct date from the calendar. Time must be entered in 24-hour format (e.g. enter 8:00 a.m. as 08:00, 8:00 p.m. as 20:00).

1. In the Walk-Thru section,
 enter the date and time that the project walk-thru will occur. (See the NOTE in the
 previous step for information about entering date and time.)

1. Enter any notes about the project walk-thru in the Notes field.

1. In the Primary Contact section, enter the name, phone number, and email address of the primary contact for this bid package (typically the person bidders should contact for information about the bid package).

1. In the Secondary Contact section, enter the name phone number, and email address of the secondary contact for the bid package (typically the person bidders should contact for information about the bid package when the primary contact is unavailable).

1. You are now ready to add
 scopes and phases to your bid package. You can add scopes and phases manually (using
 the Scope/Phase tab) or initialize them using the PC Add Scopes & Phases form.
 The following steps will guide you through the initialization process.

1. [Add scopes and phases to the bid package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/adding-scopes-and-phases-to-a-bid-package).

1. Create a bid list by
 selecting potential bidders for the bid package. Click the Select
 Bidders button to create a list of vendors that should be included on
 the bid list. This will open [PC Select Bidders](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form). Click [here](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-select-bidders-form) for more information on creating a bid list.


1. Add any inclusions and exclusions to the scopes/phases on the bid package using the Inclusions/Exclusions tab.

Now that you have finished setting up the bid package, you can
 [send a bid invitation](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/create-and-send-a-custom-invitation-to-bid), [addendum](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/create-and-send-an-addendum-using-a-template), or [notification](/en/vista/vista/costs-and-contracts/pre-construction/pc-create-and-send/create-and-send-a-notification-using-a-template) to contacts on your bid list.[](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/potential-project-setup/overview-pc-potential-project-to-pm-project)Overview: PC potential
 project to PM project
