# Field Definitions: PC Add Scopes-Phases Form

The following is a list of field descriptions for the PC Add
 Scopes-Phases form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Scope

Enter a scope code or press F4 to select one from a list. This will filter the grid so that only phases associated with the selected scope code will display.
Phases are associated with scope codes using [PC Scope Codes ](/en/vista/vista/costs-and-contracts/pre-construction/pc-setupmaintenance/about-the-pc-scope-ids-form).
