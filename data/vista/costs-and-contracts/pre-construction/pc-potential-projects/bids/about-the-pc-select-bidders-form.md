# About the PC Select Bidders Form

Use this form to generate a list of potential bidders for a bid package.

You must set up at least one bid package on
 the potential project before you can create the list of potential bidders. Click [ here](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/setting-up-a-bid-package) for more information on creating bid packages.
You can also manually add potential bidders
 using the Bid List tab on [PC Bid
 Package](/en/vista/vista/costs-and-contracts/pre-construction/pc-potential-projects/bids/about-the-pc-bid-package-form)
