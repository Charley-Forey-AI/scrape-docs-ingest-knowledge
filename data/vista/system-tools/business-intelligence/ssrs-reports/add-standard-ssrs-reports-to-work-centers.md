# Add Standard SSRS Reports to Work Centers

You can add the standard SSRS reports included in the Business
 Intelligence library to Dashboard Work Centers.
Before following these steps:

- SQL Server Reporting Services and SQL Server version 2008 R2
 or later must be installed, configured, and running.

- The Business Intelligence module installer and Viewpoint
 SSRS Reports installer have both been run. For more information, see the
 Installation Manual.

- After running the Viewpoint SSRS Reports installer, you have
 completed the steps in [SSRS - Setting
 up SSRS reports after install](/en/vista/vista/system-tools/business-intelligence/ssrs-reports/set-up-ssrs-reports-after-install).

- Select the Activate Work Centers checkbox in VA Site Settings. This
 allows users access to the Work Centers form ( Options > Work
 Centers).

This topic can be used in
 conjunction with the Customer Training content on SSRS reports, which you can open
 from the main application window by selecting Help  > Training.

1. Set up security for SSRS reports using the VA Report Security form. For more information, see [Applying Report Security Settings](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/apply-report-security-settings).

1. Give users access to the "4 Reports" and/or "1 Report" templates using the VA Work Center Security form. These are the Dashboard templates designed for viewing the SSRS reports, and granting access to these template allows users to create a Dashboard Work Center with the "4 Reports" and/or "1 Report" configuration.

1. Create a work center profile. For more information, see [Create a Work Center Profile](/en/vista/vista/administration/viewpoint-administration/work-centers/create-a-work-center-profile).

1. (Optional) Set up the
 application so that users can customize standard or create new SSRS reports and
 then add them to Dashboard Work Centers. See [SSRS
 - Setting Up Customized SSRS Reports](/en/vista/vista/system-tools/reports/ssrs-overview-and-security-considerations/setting-up-customized-ssrs-reports) for more information.

1. Have individual users apply the work center profile to their accounts. See [Assign a Work Center Profile to your user
 account](/en/vista/vista/system-tools/work-centers/about-work-center-setup/assign-a-work-center-profile-to-your-user-account).

1. Have users add the appropriate SSRS reports to Dashboard Work Centers. See [Add an SSRS report to a Dashboard Work Center](/en/vista/vista/system-tools/business-intelligence/ssrs-reports/add-an-ssrs-report-to-a-dashboard-work-center).
