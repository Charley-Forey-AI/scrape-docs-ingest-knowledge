# Add an SSRS report to a Dashboard Work Center

You can add SSRS reports to Dashboard Work Centers as long as
 you are using either the 4 Reports or 1 Report templates.
Prior to performing this task, apply a Work Center
 profile to your login. See [Assign a Work Center Profile to your user
 account](/en/vista/vista/system-tools/work-centers/about-work-center-setup/assign-a-work-center-profile-to-your-user-account).

1. Open a Dashboard Work Center that uses the "4
 Reports" or "1 Report" template. This is a Work Center with 1 report section or
 a grid of four report sections.Note: If the SSRS reports that you would like
 to add to the Work Center does not display in the drop down, you have not
 been given access to it by an administrator using VA Report Security.

1. Click the Configuration icon in the top right hand
 corner of a report section on the Work Center. This will open the Report
 Configuration form.

1. Select an SSRS report from the Report
  drop down menu.Note: If the SSRS reports that you would like
 to add to the Work Center does not display in the drop down, you have not
 been given access to it by an administrator using VA Report Security.

1. Click OK.The SSRS report displays in
 the Work Center. Repeat the steps above to add more report parts to the Work
 Center. If you would like to use more SSRS reports, you can add more Dashboard
 Work Centers to your application window and then add SSRS reports to those
 additional Work Centers. Note: If "The permissions granted to user
 "XXXXX" are insufficient for performing this operation.
 (rsAccessDenied)" displays in the Work Center, you have not been granted
 access to that BI report part.
