# About the BI Operational Target Copy Form

Use this form to copy an operational target that has been created using the BI Operational Target form.
This can save time and reduce data entry if you want to create a new operational target that is similar to a target that you have already created. This will copy every field on the source target except the notes entered on the Notes tab in the both the upper and lower portion of the BI Operational Targets form.
