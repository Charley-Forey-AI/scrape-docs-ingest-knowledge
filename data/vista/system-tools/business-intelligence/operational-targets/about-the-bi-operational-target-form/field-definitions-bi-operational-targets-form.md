# Field Definitions: BI Operational Targets Form

The following is a list of field descriptions for the BI Operational Targets form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Target Name

Enter the name of the operational target.
The value in this field can be up to 50 characters long.

## Description

Use this field to enter a description of the operational target - for example "To meet revenue numbers, the company must bid on 10 jobs per month."

## Target Type

Select the view that will be used to group and
 filter the report. Press F4 in this field to select it from a
 list.
This field is used in conjunction with the
 Grouping
 Level, Grouping Value, Filter Field,
 and Filter
 Values fields.
You can skip this field if you do not want to group and/or filter the report.

## Actuals

Use this field to select the query that
 contains the actual amounts that you would like to compare to the target. For example, if
 you are creating a target for equipment usage, the query needs to include equipment usage.
 Press F4 to select a query from a list. You can only select a SQL or View
 query.
Queries are created and maintained using the [VA Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/va-inquiries-form) form.

## Begin Date / End Date

Use these fields to enter the length of time
 that the target should apply. For example enter 1/1/19 and 12/31/19 if the
 target applies to 2019.
Once you enter the period length, use the
 Period field to select how the period will be divided - for example by
 month, by week, by day, etc.
The selection in the Begin Date,
 End
 Date, and Period field determines which fields will
 display on the Budget tab in the lower portion of the form. That is where the target values
 are entered.

## Notes

Enter notes on the target. You can also enter notes on specific target revisions using the Notes tab on the lower portion of the form.
Spelling Check
 Click the Spelling icon on the toolbar or
 select Tools > Spelling
  to spell check the text in this field.
Add a Standard Note
Standard notes allow you to insert frequently used text into some fields in the application. This text is created and maintained using the [HQ Standard Note](/en/vista/vista/administration/headquarters/standard-notes/about-the-hq-standard-note-form) form.
To insert a standard note into the field,
 right click the mouse while focus is in the field and select Standard Notes
 from the shortcut menu, which opens the Standard Note Copy window. Then enter the standard
 note to copy (or select from F4 lookup) and click OK. The system inserts the selected note
 into the field.

## Revision

This field allows you to create multiple versions of the targets for a single period - for example it allows you to create a gold, silver, and bronze level target.
Create a new revision
Enter a + in this field and the system will
 create a new revision and automatically assign it the next available revision number.

## Target Level

Enter a target level on an operational target,
 or press F4 to select one from a list.
Target levels are created and maintained using the [BI Operational Target Level](/en/vista/vista/system-tools/business-intelligence/operational-targets/about-the-bi-operational-target-level-form) form.
This is an optional field.

## Goal

Enter the goal value for a specific period.
The selection in the Begin Date,
 End
 Date, and Period fields on the Info tab in the upper portion of the form determine what
 periods will display on the Budget tab. For example if you enter 1/1/19 in the
 Begin
 Date field, 12/31/19 in the End Date field, and
 select By
 Month in the Period field, you will be able to enter a
 goal value for each month in 2019.
