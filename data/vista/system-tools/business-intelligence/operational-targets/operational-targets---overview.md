# Operational Targets - Overview

To use Operation Targets feature, you need to understand the Vista data structure, and you need to be able to create queries and custom reports.
What is the operational target feature?
The operational target feature allows you create targets on key indicators and then create reports that track your progress towards those targets - for example, utilization of equipment, number of labor hours per month, on-time completion of projects, etc.
This feature has three basic steps:

- Step 1: Enter the targets using the BI Operational Target form - For
 example, you would like equipment usage to be 75% in 2019 and 80% in 2020, and you
 would like to measure equipment usage every month. You can also set up several
 different target levels that apply, for example a gold, silver, and bronze standard.
 For more information, see [Create an operational target](/en/vista/vista/system-tools/business-intelligence/operational-targets/create-an-operational-target).

- Step 2: Select or create a query that contains the actual values that you
 would like to compare to the targets - Use the [VA Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/va-inquiries-form) form to create a query that contains
 the actual values that you would like to compare, and then associate it with the
 operational target using the Actuals field on the Info tab in
 the upper portion of the BI Operational Target form.

- Step 3: Create a customer report - Once the targets have been entered and
 they are associated with a query, create a custom report that compares the targets
 and the actual amounts. What are
 target levels?

Target levels are created and maintained using the BI Operational Target Level form, and they are associated with the targets created using the BI Operational Target form. This allows you to maintain a standard list of target levels that can be associated with many different targets - for example AGC of America Gold Award, AGC of America Silver Award, etc. [More](/en/vista/vista/system-tools/business-intelligence/operational-targets/about-the-bi-operational-target-level-form)
