# Field Definitions: BI Operational Target Copy Form

The following is a list of field descriptions for the BI Operational Target Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Destination Target

Enter the target that you would like to create in the Destination Target field. The destination target can be up to 50 characters long.

Related information

- [About the BI Operational Target Copy Form](/en/vista/vista/system-tools/business-intelligence/operational-targets/about-the-bi-operational-target-copy-form)

## Source Target

Enter the target that you would like to copy in the Source Target field or press F4 to select it from a list.
