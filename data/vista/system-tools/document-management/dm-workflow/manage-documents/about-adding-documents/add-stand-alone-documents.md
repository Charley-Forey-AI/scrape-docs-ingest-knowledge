# Add Stand Alone Documents

Documents added to the system are typically attached to a data record. But you can also add "stand alone" documents to the system which are not associated with any data record.
You may want to store documents electronically that do not necessarily have a direct tie to any records in the system.
To add stand alone documents to the system:

1. Select DM Stand Alone Documents from the DM Programs folder on the Main Menu.The Stand Alone Documents form displays.

1. Click Add.The system displays the Attachment Detail form.

1. Click Browse to locate the document that you want to attach.The system displays the Open dialog.

1. Locate the document and click Open.The network path of the document displays in the File Name field.

1. Enter a description for the document in the Description field.

1. Enter an attachment type in the Attachment Type field, if necessary.Press F4 for a list of attachment types. Attachment types are used to group and organize attachments, as well as for applying security. For more information, see [DM Attachment Types Form](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/dm-attachment-types-form).

1. Click OK.

The document appears in the grid on the Stand Alone Documents form.
