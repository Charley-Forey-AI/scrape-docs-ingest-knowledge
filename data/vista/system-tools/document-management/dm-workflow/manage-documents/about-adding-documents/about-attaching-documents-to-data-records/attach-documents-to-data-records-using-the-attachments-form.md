# Attach Documents to Data Records using the Attachments Form

You can use the Attachments form in any form (that allows attachments) to attach documents to data records.
The following describes how to attach documents to data records using the Attachments form.

1. Open the form that contains the data record that you want to attach the document to and locate the record. Note: Some header/detail forms allow attachments at the header and line levels, so make sure focus is in the key field of the appropriate section (header or detail). This ensures the document is attached to the correct record.

1. Click the Attachments icon on the toolbar. The system displays the Attachment form.

1. Click Add.The system displays the Attachment Detail form.Note: You can also launch the Attachment Detail form by clicking on the drop-down next to the Attachments icon on the toolbar and selecting Add Attachment.

1. Click Browse to locate the document that you want to attach. The system displays the Open dialog.

1. Locate the document and click Open.The network path of the document will display in the File Name field.

1. Enter a description for the attachment in the Description field.

1. Enter an attachment type in the Attachment Type field, if necessary. Press F4 for a list of attachment types. Attachment types are used to group and organize attachments, as well as for applying security. For more information, see [DM Attachment Types Form](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/dm-attachment-types-form).Note: The Attachment Type field defaults an attachment type if there is one defined by the user or for the system on the Form Properties form. For more information, see [About the Form Properties form](/en/vista/vista/system-tools/user-interface-guide/system-forms/about-the-form-properties-form).

1. Click OK.

The document appears in the grid on the Attachment form and is attached to the data record.
