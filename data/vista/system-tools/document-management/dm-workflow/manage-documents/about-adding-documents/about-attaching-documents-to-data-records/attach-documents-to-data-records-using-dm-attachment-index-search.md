# Attach Documents to Data Records using DM Attachment Index Search

You can attach documents to a record/transaction that are associated with other
 records/transactions using the DM Attachment Index Search form.
 The following describes how to add attachments from another record/transaction to
 the current record or transaction.

1. Open the form containing the record/transaction to which you are adding the
 attachment(s).

1. On the Attachments button in the toolbar, click the drop-down arrow and select
 Index Search.The DM Attachment Index Search form displays.

1. Enter search criteria to locate the attachment(s) you want and click
 Search.The Results grid displays all attachment IDs meeting the selection
 criteria.

1. Select the desired attachments, using SHIFT for consecutive selection or CTRL
 for random selection.Note: If you are adding a stand-alone document, you can delete the original
 stand-alone document when adding it to the record by selecting the
 Delete Stand Alone when
 Attached checkbox. This will delete the attachment prior to
 adding the attachment to the current data record.

1. Click Attach Selected to Current Record.

The system copies the document to the currently selected record on the original
 form. The document remains in its original location.Note: If you added an attachment on
 the DM Transaction Attachments form, the system adds the attachment to all selected
 attachments/transactions.
