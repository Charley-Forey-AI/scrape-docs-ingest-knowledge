# Field Definitions: DM Attachments DM Stand Alone Docs Form

The following is a list of field descriptions for the DM
 Attachment DM Stand Alone Docs form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## ID

This field defaults to the ID for the first
 attached document in the grid. If there are no documents in the grid, this field defaults
 blank. This field is inaccessible from the Grid tab. On all other tabs, enter the document
 ID to view. Press F4 for a list of available documents. If you enter a value that is not
 associated with a document, the system automatically displays the Attachment Detail form.
