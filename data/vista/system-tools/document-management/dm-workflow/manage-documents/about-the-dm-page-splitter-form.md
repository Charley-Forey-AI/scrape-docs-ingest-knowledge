# About the DM Page Splitter Form

Use the Page Splitter form to break apart PDF files that include attachments pertaining to more than one Vista record. You can then either attach the broken out section to to Vista records or save them anywhere the file can be dragged.
Access this form by right-clicking in any relevant form and selecting Attachments  > Page Splitter.Note: You can only use this form to split PDF files.
For instructions on breaking PDF files into multiple PDF files, see [Split a PDF File into Separate Files](/en/vista/vista/system-tools/document-management/dm-workflow/manage-documents/split-a-pdf-file-into-separate-files).
