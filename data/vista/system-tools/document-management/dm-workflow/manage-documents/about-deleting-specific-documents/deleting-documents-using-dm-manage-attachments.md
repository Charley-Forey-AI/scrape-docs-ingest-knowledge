# Deleting Documents using DM Manage Attachments

1. Search for the documents using DM Attachment Index Search. For more information, see [Searching for Documents](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm).

1. From the Results grid, select the documents you want to delete. You can use Shift-click to select multiple consecutive documents in the grid or use Control-click to select multiple non-consecutive documents.

1. Click
 Manage Selected Attachments
 . The Manage Selected Attachments form displays.

1. Select the Delete tab.

1. Click Delete.
  A confirmation dialog box displays.

1. Click
 Yes
 to delete the selected documents. The system deletes the documents and any related indexes and displays a confirmation dialog box.

1. Click
 Close
 .Note: If you checked the
 Retain deleted attachments
 box in DM Attachment Options, the system saves the document, but removes all links and indexes. You can search for these deleted documents using DM Attachment Index Search. Deleted documents display in the Results grid with Deleted displaying in the
 Current State
 field. If you are deleting a document with a current state of deleted, the system will display a warning stating it will purge the document. Click
 Yes
 to completely purge the document from the system.
