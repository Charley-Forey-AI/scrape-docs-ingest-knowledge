# Deleting Documents using Attachment/Stand Alone Documents Form

To delete a document using the Attachment/Stand Alone Documents form, select the document in the grid and click
 Delete
 . The system displays a warning. Click
 Yes
 to delete the document. The system removes the document from the grid; data records associated with attached documents are unaffected.
Tip: You can use Shift-click of Control-click to select several documents for deletion.
Note: If you checked the
 Retain Deleted Attachments
 box in DM Attachment Options, the system saves the document and sets its status to Deleted. To locate this document, use DM Attachment Index Search.
