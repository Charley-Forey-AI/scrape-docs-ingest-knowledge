# Copying Specific Documents to a New Location

You can choose to copy documents from their current storage location to a new location.
The documents will stay in their original location once you copy them and the system will continue to reference these documents from that location.You might want to copy documents to a new location if you need to gather related documents together to provide to someone who is not a user of the Vista software. For example, you might use this feature to copy all of the documents associated with a particular job to a specific network folder.
The following instructions detail how to copy specific documents to a new location.
Note: In order to move documents to new locations, the system ignores any datatype or attachment type security settings.

1. Search for the documents using DM Attachment Index Search. For more information, see [Searching for Documents](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm).

1. Select the documents you want to copy from the Results grid. You can use Shift-click to select multiple consecutive documents in the grid or use Control-click to select multiple non-consecutive documents.

1. Click Manage Selected Attachments. The Manage Selected Attachments form displays.

1. Select the Copy tab.

1. Enter the network location in the Copy attachments to the specified location field, or use the Browse button to locate the appropriate directory. This is the location where the system will copy the documents to.

1. Click Copy. The system copies the selected documents to the new location.
