# About the DM Attachment Detail Form

You can use this form to browse for and add a document to the system or edit details about the document.
Access this form by clicking the Add or Edit buttons from the [DM Attachments/DM Stand Alone Attachment Form](/en/vista/vista/system-tools/document-management/dm-workflow/manage-documents/attachment-formdm-stand-alone-documents).
