# About Deleting Specific Documents

You can delete specific documents from the system using the DM Manage Attachments form or the DM Attachment/Stand Alone Documents form.
Note: For information on purging documents from the system based on attachment type and retention settings, see [Purging Expired Documents](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/dm-maintenance/purging-expired-documents).
