# Scan a Stand-Alone Document

While most documents you scan into the system are associated to a data record, you can scan "stand-alone" documents into the system that are not associated with any specific data record.
 You may want to add stand-alone documents to your system to store documents electronically that do not have a direct tie to any records in the system.

To scan a single stand-alone document into the system:

1. Go to the Main Menu and select DM ProgramsDM Stand Alone Documents.

1. From the toolbar, select Import > Scan Single Image.

1. If prompted, define your scan settings.Scan settings differ from scanner to scanner, but may include resolution (DPI), paper size, bit-depth (e.g., 8-bit Gray, 24-bit Color, etc.), scaling, document feeder (e.g., flatbed or ADF load), and more.

1. Click Scan.Once the document is scanned, the Single Scan Viewer displays the scanned document.

1. Review the scanned document to verify accuracy. For more information on viewing options, see [Single Scan Viewer Toolbar Options](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-a-stand-alone-document/single-scan-viewer-toolbar-options).

1. (Optional) Re-scan or add pages. Choose an option and follow the same steps as adding the original scanned item.

- Re-scan - to replace the previously scanned document, click the Scan button again.

- Add a Page - to add one or more pages to the previously scanned document, click the Add a Page button ().

1. (Optional) In the Attachment Type field, enter an attachment type.For more information, see [DM Attachment Types Form](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/dm-attachment-types-form).

1. In the Description field, enter a description for the document.

1. Click OK.The system closes the Single Scan Viewer. The document displays in the grid on the Stand Alone Documents form.
