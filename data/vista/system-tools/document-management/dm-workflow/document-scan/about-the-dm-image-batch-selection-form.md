# About the DM Image Batch Selection Form

Use this form to view existing batches, add documents to a batch, and/or clear a batch.
Use this form to view existing batches, add documents to a batch,
 and/or clear a batch. This form can be accessed in three ways:

- selecting the Open Image Batch option from the Scanning
 button () on most forms

- selecting Scan > Open
 Image Batch from the Tools menu of most forms

- selecting DM Scan Batch Viewer on the Document Management
 menu

When you open this form, the
 Available Image Batches grid displays each batch, when it was created, who created
 the batch, who is currently using the batch, whether the batch is restricted, the
 number of documents per batch, and how many documents within the batch are
 unattached to data records.
By
 default, the available batches are sorted first by Created Date (newest to oldest)
 and then by the Description. You can click the column headers to sort the
 batches.
