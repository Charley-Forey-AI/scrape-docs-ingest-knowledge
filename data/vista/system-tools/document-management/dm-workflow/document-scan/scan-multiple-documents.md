# Scan Multiple Documents

You can scan multiple documents into the system at the same time using a batch process.
When adding documents in in a batch, you can also attach the documents to data records automatically or manually.
These steps are to scan multiple documents into the system and attach them to a data record. Consider these pages for similar tasks:

- [Scan and Attach Single Documents to Data Records](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-and-attach-single-documents-to-data-records)

- [Scan Multiple Stand Alone Documents](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/scan-multiple-stand-alone-documents)

1. Create a scan batch with the Image Batch Selection form. For more information, see [Create a Scanned Document Batch](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/create-a-scanned-document-batch).

1. Add documents to the system using the Batch Scan Viewer. Using this form, you can add documents via a scanner or by uploading documents from a network directory. For more information, see [Add Documents to a Scan Batch](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/add-documents-to-a-scan-batch).

1. Attach the scanned documents to data records. For more information, see [Attach Documents in a Scan Batch to Data Records](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/attach-documents-in-a-scan-batch-to-data-records).
