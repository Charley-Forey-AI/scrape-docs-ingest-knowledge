# Single Scan Viewer Toolbar Options

The following table discusses the toolbar options available in the Single Scan Viewer form.
Display/Attachment OptionDescription
Use the View button to rearrange the layout of the viewer by clicking the arrow and selecting the desired option. Options include:

- Thumbnail and Image - displays both the Thumbnail and the Image panes. This is the default option.

- Thumbnail only - the viewer only displays the Thumbnail pane.

- Image only - the viewer only displays the Image pane.

Re-scan the document. The system replaces the current scanned document with the new document. See [Scan and Attach Single Documents to Data Records](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-and-attach-single-documents-to-data-records) for more information.
Add additional pages to the scanned document by scanning an additional document. Once the scan is complete, the new scanned document image will display in the Single Scan Viewer. See [Scan and Attach Single Documents to Data Records](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-and-attach-single-documents-to-data-records) for more information.
Zoom in.
Zoom out.
Activate the selection cursor. Use the selection cursor to draw a box around the area of the image that you wish to zoom in to.
Fit the image to the width of the Image pane.
Fit the image to the height of the Image pane.
Rotate the image to the left in the Image pane.
Rotate the image to the right in the Image pane.
