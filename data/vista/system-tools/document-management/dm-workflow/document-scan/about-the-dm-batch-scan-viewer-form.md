# About the DM Batch Scan Viewer Form

Use this form to view documents in a scan batch, add documents to a batch, and attach documents to data records.
All documents scanned into a batch are stored in the Vista software database.
Access this form by selecting a batch from the Image Batch Selection form. From the main menu, go to Document Management > Programs > DM Scan Batch Viewer. From the Image Batch Selection form, select a batch to open the Batch Scan Viewer form.
The Batch Scan Viewer's layout defaults in two panes. The thumbnail pane (left-hand side) displays all batch documents in thumbnail format. The image pane (right-hand side) shows a larger preview of the document. The header area of the form displays the batch description, total number of documents in the batch, the number of unattached documents, and the default attachment type for the batch.
