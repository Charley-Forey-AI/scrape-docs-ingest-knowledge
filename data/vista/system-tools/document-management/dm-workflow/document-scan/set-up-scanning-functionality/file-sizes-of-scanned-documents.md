# File Sizes of Scanned Documents

The image file size of scanned documents depends on the contents of the image, color depth, number of pages and file type (e.g. TIFF, JPG).
JPEG compression is used only if you selected the Use
 .JPG Files for Single Page Color Documents checkbox in the DM Attachment Options
form and you are scanning a single-page color document.
Multiple-page color documents
 (including 8-bit Grayscale) are scanned in TIFF format (uncompressed).
CCIT Compression
 works only with black and white documents, but can be used for single and multiple
 pages.
 For estimating purposes, this table provides file sizes for a single page based on DPI (dots per inch).
