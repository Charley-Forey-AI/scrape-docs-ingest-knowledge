# Field Definitions: DM Image Batch Selection Form

##  Batch Name

 Specify a name for the new batch; maximum of 30 characters.

## Restricted

Select this box to limit access to the batch. When selected, no other users may view the batch.
Note: This option cannot be changed later. To allow access, clear the batch and create a new, unrestricted one.

##  My Batches Only

 Select this box to display your batches only in the Available Image Batches grid.
