# DM Scan Batch Viewer

Select the DM Scan Batch Viewer to open the Image Batch Selection form.
DM Scan Batch Viewer enables you to:

- add large numbers of scanned documents to batches

- remove documents from scan batches

- clear scan batches

- attach documents to data records in Vista

See [Image Batch Selection](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/about-the-dm-image-batch-selection-form) for more information.
