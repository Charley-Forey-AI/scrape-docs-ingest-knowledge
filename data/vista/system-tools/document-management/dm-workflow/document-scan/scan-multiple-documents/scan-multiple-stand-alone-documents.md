# Scan Multiple Stand Alone Documents

You can scan multiple stand alone documents into the system at the same time using a batch process.
For more information on stand alone documents, see [Add Stand Alone Documents](/en/vista/vista/system-tools/document-management/dm-workflow/manage-documents/about-adding-documents/add-stand-alone-documents).
The following steps provide an outline for adding multiple stand alone documents to the system using a scan batch process. For more information on scanning single stand alone documents into the system, see [Scan a Stand-Alone Document](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-a-stand-alone-document).

1. Create a scan batch with the Image Batch Selection form. For more information, see [Create a Scanned Document Batch](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/create-a-scanned-document-batch).

1. Add documents to the system using the Batch Scan Viewer. Using this form, you can add documents via a scanner or by uploading documents from a network directory. For more information, see [Add Documents to a Scan Batch](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/add-documents-to-a-scan-batch).

1. Select DM Stand Alone Documents from the DM Programs folder on the Main Menu.The Stand Alone Documents form displays.

1. From the toolbar, select
 Import > Open
 Scan Batch.The Image Batch Selection form displays.

1. Select the appropriate
 batch from the Available Image Batches grid and click OK.The Batch Scan Viewer displays.

1. If needed, from the
 Thumbnail pane on the left side, select a document to add as a stand alone
 document and click Add Stand Alone Attachment.
 Repeat as needed.

1. Close the Batch Scan
 Viewer.The documents appear in the grid on the Stand Alone Documents
 form, each with a file name of VCSScanning.

1. Identify each document in the grid and add a description to help identify its
 contents in the future. For additional information, see [Add Stand Alone Documents](/en/vista/vista/system-tools/document-management/dm-workflow/manage-documents/about-adding-documents/add-stand-alone-documents).
