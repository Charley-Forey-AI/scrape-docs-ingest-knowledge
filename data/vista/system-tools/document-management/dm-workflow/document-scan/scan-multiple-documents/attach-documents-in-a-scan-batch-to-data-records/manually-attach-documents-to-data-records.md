# Manually Attach Documents to Data Records

The following instructions detail how to manually attach
 documents to data records.

1. Open the form and find
 the record that you want to attach the document to.

1. Click the arrow to the
 right of the Scanning button () and select Open
 Image Batch.
The Image Batch Selection form displays. Tip: Select the My Batches Only checkbox on the Image Batch Selection form to display only your
 batches in the Available Image Batches grid.

1. Select the batch to open
 from the grid and click OK.
The Batch Scan Viewer form displays. Note: The In Use By column in the Available Image Batches form indicates whether a
 batch is currently in use and by whom. If a batch is "in use," it does not
 necessarily mean that a user is currently in the batch entering data. In
 some cases, the system flags a batch as “in use” because of an interrupted
 validation or program error. If the batch was created by your login, you may unlock it by selecting the batch
 in the grid, right-clicking your mouse, and selecting Unlock Batch. Batches
 that are “in use” by another login must be unlocked by that user.

1. Before attaching
 documents, review them to verify accuracy. For more
 information on viewing options, see [Batch Scan Viewer Toolbar Options](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/about-the-dm-batch-scan-viewer-form/batch-scan-viewer-options).

1. Select the thumbnail of
 the document that you want to add from the left-hand pane of the form.
The image
 displays in the right-hand pane. Tip: To select multiple documents to add,
 shift-click consecutive documents or control-click to select non-consecutive
 documents.

1. Enter the attachment
 type for the document in the Attachment Type field or accept the default. Press F4 for a list of attachment types.Note: If you add an attachment type, the
 system will default it for each document in the batch when you attach
 documents to records using this form. If you change the attachment type
 later, each subsequent document will default with the new attachment
 type.

1. Click Attach.
The system attaches the scanned document to the selected data
 record. Note: If you choose the Hide Attached option, attached documents will no longer display in the
 Batch Scan Viewer.

1. Repeat steps 5-7 to add
 additional documents to the selected data record.
