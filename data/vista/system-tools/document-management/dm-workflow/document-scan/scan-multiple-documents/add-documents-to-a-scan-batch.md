# Add Documents to a Scan Batch

There are two ways to add documents to a scan batch using the Batch Scan Viewer.
Documents can be added:

- From a file directory on your network

- Via a scanner attached to your computer

Note:Functionality of the DM Scan Batch form is contextual:In order to add documents to a batch, you must access the DM Scan Batch form starting in the DM menu.
If you access the DM Scan Batch form using the Scan button () on an individual form, the documents can be added only to the pertinent record.
