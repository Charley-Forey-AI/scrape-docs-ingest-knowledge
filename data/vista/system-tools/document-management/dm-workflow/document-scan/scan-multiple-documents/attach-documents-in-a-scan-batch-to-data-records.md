# Attach Documents in a Scan Batch to Data Records

There are two ways that you can attach scanned documents to data records.
A scan batch must have documents in it before taking the steps on this page. If needed, see
 [Add Documents to a Scan Batch](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/add-documents-to-a-scan-batch).
You can attach documents in a scan batch to specific data records. Doing so moves them from the database to the document storage location that you set up for the system.
There are two methods for attaching scanned documents to data records:

- you can [add documents manually](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/attach-documents-in-a-scan-batch-to-data-records/manually-attach-documents-to-data-records#task-4651--en__task-4651), or

- you can have the system [automatically attach documents](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/attach-documents-in-a-scan-batch-to-data-records/automatically-attach-documents-to-data-records#task-3774--en__task-3774) as you create new records in the system.

Once scanned documents are attached to data records, you can treat them like any other document:

- access them via the Attachments button () on the toolbar of any form

- send them through the document routing process

- manage them using the DM Manage Attachments form
