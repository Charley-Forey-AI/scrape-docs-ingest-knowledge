# Add Documents from a File Directory

This option allows you to copy all existing documents from a specified network folder to
 a table in the database.
Typically, this option is used when a large
 number of documents have been scanned on to your network and you need to bring the
 batch into Vista™. Supported file types include: .pdf,
 .jpg, .jpeg, .tif, and .tiff.
Note: When displaying .pdf files, the Batch Scan Viewer uses a
 snapshot of the PDF, which may make the resolution less than ideal. To adjust the
 resolution, change the resolution setting using the PDF View
 Resolution field in the DM Attachment Options form. For more information, see
 [Set Document Viewing Options](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/set-document-viewing-options).
