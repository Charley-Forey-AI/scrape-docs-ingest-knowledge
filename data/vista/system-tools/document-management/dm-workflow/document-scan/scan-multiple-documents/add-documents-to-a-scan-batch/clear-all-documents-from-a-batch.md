# Clear All Documents from a Batch

You can delete the contents of a scan batch.
After clearing a batch, any associated documents must be re-scanned if you want to add them to a new
 batch.
To clear a batch of all its documents:

1. Select File > Clear Batch from either the Image Batch Selection form or the Batch Scan
 Viewer.The system displays a warning.

1. Click Yes.

All documents are removed from the batch.
