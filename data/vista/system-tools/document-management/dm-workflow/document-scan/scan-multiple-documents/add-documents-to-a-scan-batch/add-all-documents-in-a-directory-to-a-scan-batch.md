# Add All Documents in a Directory to a Scan Batch

Add all documents in a directory to a scan batch.

1. Select DM Scan Batch Viewer from the DM Programs folder in the Main Menu. The system displays the Image Batch Selection form.

1. Select the batch you want to add documents to from the Available Image Batches grid and click OK . You can also double-click the batch in the grid. The Batch Scan Viewer displays.Note: You can also create a scan batch at this step if an appropriate one does not exist. For more information, see [Create a Scanned Document Batch](/en/vista/vista/system-tools/document-management/dm-workflow/document-scan/scan-multiple-documents/create-a-scanned-document-batch).

1. Click the Get Images button () and select From Directory. A Browse for Folder window displays.

1. Select the directory that contains the scanned documents and click OK. A dialog box displays asking if you want the documents deleted from the original location.

1. Click Yes to delete the files from their original location or click No to leave the documents in their original location. All documents from the specified directory display in the Batch Scan Viewer in descending order by creation date.

1. Enter an attachment type in the Attachment Type field. Press F4 for a list of attachment types.Note: Adding an attachment type to a batch makes it the default type for each document in the batch.
