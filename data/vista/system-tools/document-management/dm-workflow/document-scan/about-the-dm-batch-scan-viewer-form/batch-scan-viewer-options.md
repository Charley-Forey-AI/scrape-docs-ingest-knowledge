# Batch Scan Viewer Options

The following table discusses the options available in the Single Scan Viewer form.
Display/Attachment Option
Description

Use the View button to rearrange the layout of the viewer by selecting the arrow and selecting the desired option. Options include:

- Thumbnail and Image - displays both the Thumbnail and the Image panes. This is the default option.

- Thumbnail only - the viewer only displays the Thumbnail pane.

- Image only - the viewer only displays the Image pane.

When you set this option to checked (the green checkmark displays) the system will automatically add selected documents to the next record that you create on a form.This button does not display if you access the form from the Stand Alone Documents form (by selecting Import > Open Scan Batch).

When you set this option to checked, the system will not display documents in the batch that have been previously attached.
Right-Select the thumbnail of a document and select this option to manually set the attachment status to "attached." When marked as attached, the document does not display when you select the Hide Attached option.Make sure that the document has been selected (visible in the right-hand preview pane) before right-Selecting; otherwise, the selected attachment status will not take affect.

Right-Select the thumbnail of a document and select this option to manually set the attachment status to "unattached." When marked as unattached, the document will display when you select the Hide Attached option.Make sure that the document has been selected (visible in the right-hand preview pane) before right-Selecting; otherwise, the selected attachment status will not take affect.

Select this button to manually add a document to a data record. This button does not display if you access the form from the Stand Alone Documents form (by selecting Import > Open Scan Batch).

This button only displays if you access the form from the Stand Alone Documents form (by selecting Import > Open Scan Batch).Select this button to add the currently selected document to the system as a stand alone document.

The Description Picker option allows you to designate how the description for each image is generated. Select the Description Picker icon and drag it to the desired field on your entry form. For example, if you are entering AP invoices and you want the image description to be the AP Reference number, drag the icon to the field (making sure that a black box surrounds the field). Each time you enter and save a transaction header, the Auto Attach feature pulls the transaction’s AP Reference number and uses it as the image description.
The Previous and Next arrows allow you to move to the next or previous page in a multi-page documents. The number displays the current page number that you are viewing.

This button allows you to zoom out.
Select this button to activate the selection cursor. Use the selection cursor to draw a box around the area of the image that you wish to zoom in to.
Select this button to fit the image to the width of the Image pane.
Select this button to fit the image to the height of the Image pane.
Select this button to rotate the image to the left in the Image pane.
Select this button to rotate the image to the right in the Image pane.
