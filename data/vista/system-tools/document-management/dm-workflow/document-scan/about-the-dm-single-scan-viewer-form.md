# About the DM Single Scan Viewer Form

You can use this form to scan single documents and attach them directly to the current data record that you are viewing.
You can access this from by by clicking the Scanning button () from the toolbar, and it is only available if you have purchased scanning functionality for the Document Management (DM) module.
The layout for this form defaults in two panes. The left pane displays all scanned documents in thumbnail format, while the right pane shows a larger preview of the document. There are a variety of viewing options on this form that allow you to change the display settings for the document.
