# About the DM Document Review Form

This form displays the documents assigned to the current user for review.
When you open the form, it automatically displays documents assigned to you for the past month. Documents are assigned to reviewers using DM Document Routing. For more information, see [Document Review Process](/en/vista/vista/system-tools/document-management/dm-workflow/document-review-process).
When reviewing a document you can add your comments, set the review status, view the document, and see if there are other reviewers assigned to the document.
