# Field Definitions: DM Document Review Manage Form

The following is a list of field descriptions for the DM
 Document Review Manage form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Status Code

Enter the status code for filtering documents.
 Press F4 to see all available status codes.
Leave this field blank to filter for all
 status codes.

##  Reviewers

 Select the reviewer for filtering documents. To filter by multiple reviewers, shift-click or control-click each applicable reviewer. Select All to filter by all reviewers.

##  Search by Date

 Select the appropriate radio button for filtering by date.

- None – If you select this option, the system will not include dates in the filter.

- Date Added – If you select this option, the system filters for documents based on the date they were added to the system. Additionally, when you select this option, the system enables the Beginning Date and Ending Date fields.

- Status Date – If you select this option, the system filters for documents based on the date you applied the last status to the document. Additionally, when you select this option, the system enables the Beginning Date and Ending Date fields.

##  Beginning Date

 Enter the beginning date in a range for filtering documents.

##  Ending Date

 Enter the ending date in a range for filtering documents.

##  Send Attachment

 Check this box to attach the document to emails that you send reviewers (by clicking the Email Reviewer button).

##  Instructions

 This field defaults from the Instructions field from DM Document Routing. Accept the default or make changes as necessary.

##  Reviewer Comments

 This field defaults from the Reviewer Comments field from DM Document Review. Accept the default or make changes as necessary.

##  Status

 This field displays the current document review status. Accept the default or update the status as necessary.
