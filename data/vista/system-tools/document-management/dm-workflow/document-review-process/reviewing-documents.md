# Reviewing Documents

If you have set up a document review/approval process, your reviewers can review documents assigned to them using the DM Document Review form.
When users log into this form, all of the documents currently assigned to them will display.
The following information details how to review documents.

1. From the DM Programs folder, select DM Document Review. The DM Document Review form displays, along with all documents assigned to you for review.

1. Select a record in the grid and click View Attachment to view the document. The system displays the document in it's standard viewer or in the DM Document Viewer (if set up in DM Attachment Options).Tip: If you have a number of documents in the grid, you can filter out documents that you have already set to a completed status by checking the Hide Completed Items box. You can also limit the number of records that display in the grid by date. Enter a date in the Added Since field and the grid will only display items that were assigned to you after the date you specify.

1. Enter any comments you
 might have in the Reviewer Comments
 field.

1. From the Status drop-down, select a review status for the document.Note: The status you select only applies to you and does not affect other reviewers assigned to the document.
Tip: To see who else has been assigned as a reviewer to this document, click View Routing. The system will display the DM Document Routing form. The grid on this form will display all reviewers assigned to the document. You can also use the DM Document Routing form to assign additional reviewers. For more information, see [Assigning Reviewers to Documents](/en/vista/vista/system-tools/document-management/dm-workflow/document-review-process/assigning-reviewers-to-documents).

1. Save the record as normal.
