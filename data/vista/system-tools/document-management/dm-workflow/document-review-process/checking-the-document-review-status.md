# Checking the Document Review Status

You can use the DM Document Review Manage form to check the status of document reviews.
When checking the review status, you will first perform a filter to locate documents under review. From there you can update the status, view routing information, and view the attached document. The following instructions detail how to check the review status for documents within the system.

1. Enter a review status code for filtering in the Status Code field. Press F4 for a list of available status codes. You can also leave this field blank to filter for all status codes

1. Select a reviewer for filtering from the Reviewers field. To filter for multiple reviewers, shift-click or control-click users for filtering. To filter for all reviewers, select All.

1. In the Search by Date section, select a date filtering option: None, Date Added, or Status Date. If you select either Date Added or Status Date, the system enables the Beginning Date and Ending Date fields. Note: When you select Date Added, the system searches for documents based on when they were added to the system. When you select Status Date, the system searches for documents based on the date when the last status was applied.

1. If you selected either the Date Added or Status Date radio button, enter a date range in the Beginning Date and Ending Date fields.

1. Click Search. The grid displays all reviewers, and their associated documents, that meet the filtering criteria.

1. Once you have filtered the grid, you can update the review instructions or review status as necessary. To update the review instructions, click in the Instructions field and enter new text. To change the review status, select a new status from the Status drop-down.

1. To view the document, click View Document. The system displays the document in its default viewer or in the DM Document Viewer (if you activated it in DM Attachment Options).

1. To see routing information for the document, click View Routing. The DM Document Routing form appears and displays all reviewers that are currently associated with the document. You can also assign new reviewers to the document using the DM Document Routing form. For more information, see [Assigning Reviewers to Documents](/en/vista/vista/system-tools/document-management/dm-workflow/document-review-process/assigning-reviewers-to-documents).

1. To email the reviewer, click Email Reviewer. An email message appears; the body text of the email includes the document description, instructions, and notes. If you wish to attach the document to the email, check the Send Attachment box prior to clicking Email Reviewer.
