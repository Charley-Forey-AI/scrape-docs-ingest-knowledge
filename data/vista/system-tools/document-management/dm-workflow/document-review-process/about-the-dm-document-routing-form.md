# About the DM Document Routing Form

The primary use for this form is to assign reviewers and instructions to documents, but you can also use it to track progress for other reviewers assigned to the document.
You can access this form in several ways:

- Double-clicking a record on the Routing tab in [DM Attachments/DM Stand Alone Documents ](/en/vista/vista/system-tools/document-management/dm-workflow/manage-documents/attachment-formdm-stand-alone-documents)

- Clicking the Document Routing button in [DM Attachment Index Search ](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm/about-the-dm-attachment-index-search-form)

- Clicking the View Routing button in [DM Document Review ](/en/vista/vista/system-tools/document-management/dm-workflow/document-review-process/about-the-dm-document-review-form)

- Clicking the View Routing button in [DM Document Review Manage ](/en/vista/vista/system-tools/document-management/dm-workflow/document-review-process/about-the-dm-document-review-manage-form)

Although you might typically assign reviewers at the
 time you add the document, reviewers can be assigned to an attachment at any time, no
 matter which form is used to access this form. For more information, see [Assigning Reviewers to Documents](/en/vista/vista/system-tools/document-management/dm-workflow/document-review-process/assigning-reviewers-to-documents).
