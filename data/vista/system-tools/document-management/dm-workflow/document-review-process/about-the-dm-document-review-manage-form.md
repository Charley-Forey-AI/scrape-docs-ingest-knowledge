# About the DM Document Review Manage Form

Use this form to manage the document review process.
When using this form you can filter for specific reviewers and associated documents; update reviewer instructions; set the review status for documents; email reviewers; view routing information; view the attached document,; and/or email the reviewer. This form is intended for document review managers, and security should be set appropriately.
