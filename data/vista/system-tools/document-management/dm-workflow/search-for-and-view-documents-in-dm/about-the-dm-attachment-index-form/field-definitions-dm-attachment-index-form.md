# Field Definitions: DM Attachment Index Form

The following is a list of field descriptions for the DM Attachment Index form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Index Seq

Initially defaults the first index sequence created (by the system) for the associated document. System generated sequences are not editable.
If adding a user-defined index sequence, enter a sequence number (-2,147,483,648 - 2,147,483,647) or 'N' for the next sequential number.
