# Field Definitions: DM Attachment Index Search Form

The following is a list of field descriptions for the DM
 Attachment Index Search form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## AP Company

Enter an AP company for restricting the index search. Only document indexes containing this AP company, and meeting all other search criteria, will display in the grid.

## Vendor Group

Enter a vendor group for restricting the index search. Defaults the vendor group for the AP company specified above. Only documents with indexes containing this vendor group, and meeting all other search criteria, will display in the grid.

## Created Date From

Enter a beginning date in a range for filtering documents. The ‘created date’ refers to the date that you added the document to the system.

## Created Date To

Enter an ending date in a range for filtering documents. The ‘created date’ refers to the date that you added the document to the system.

## Created By

Enter a user name for filtering attachments. The search process looks for documents added by the specified user.

## Attached From

Enter the form associated with the documents
 you are searching for. Press F4 for a list of forms.
Tip: To search for all forms within a module, use a wild card search. For example, if you want to search for all AP forms, enter AP*.

## Attachment Type

Enter an attachment type for limiting the
 search. Press F4 for a list of available attachment types.

## Description

Enter a document description for limiting the search.
Tip: To search for all descriptions with a similar name, use a wild card search. For example, if you want to search for a document associated with an AP record, you might enter AP*.

## Keyword

Enter a keyword to search for within your documents. When entering a keyword in this field, the system will look for all documents containing the keyword that match the other criteria that you specified.

## Include Attached

Check this box to search for documents associated with data records.
This box is selected by default and remains checked if you click the Clear button. You must manually uncheck this box if you wish to search for attached documents.

## Include Stand Alone

Select this checkbox to search for stand-alone documents. See [Document States](/en/vista/vista/system-tools/document-management/about-document-states) for more information.

## Include Deleted

Select this checkbox to search for deleted documents. See [Document States](/en/vista/vista/system-tools/document-management/about-document-states)for more information.

## Delete Stand Alone when Attached

Check this box to have the system delete the original stand-alone document once you attach a copy to a data record.
Note: The system displays this checkbox when you access the DM Attachment Index Search form from another form (selecting
 Index Search
 from the
 Attachments
 button drop-down on the toolbar).
