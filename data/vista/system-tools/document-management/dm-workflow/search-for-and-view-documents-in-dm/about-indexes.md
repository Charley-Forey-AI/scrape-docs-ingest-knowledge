# About Indexes

When adding documents to the system and attaching them to data records, the system automatically applies an index to the attached document.
Indexes provide details about the document based on the data associated with the record it is attached to.
For example, if the document was attached to an Accounts Payable (AP) invoice the system will add index entries for the AP company, vendor group, vendor, AP reference number, and the AP check number. The index may also contain additional information depending on the line type of the invoice. So, if it is a Job line type, the system will also add index entries for JC company, job, phase group, phase, and cost type. If there are multiple lines on the invoice, a separate index sequence is generated for each line on the invoice.
Indexes enable you to search for documents within the system. When searching for a document (in DM Attachment Index Search), you can enter criteria that will help locate the document based on different index values. In some cases, you may find that the system-generated indexes are not adequate for your needs. While you cannot edit system-generated indexes, you can create custom indexes for documents to help tailor your searches. In addition, you can create indexes in the same way for stand alone documents in the system.
