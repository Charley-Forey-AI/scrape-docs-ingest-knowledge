# Refreshing Indexes for All Documents

The following instructions detail how to refresh indexes for all documents.
Note: For posted transactions, the indexes automatically refresh with the batch process.

1. From the DM Programs folder, select the DM Attachment Index Refresh form.

1. In the
 Modules with Attachments
 section, select the module(s) for updating documents.Tip: You can click
 All
 to check all of the module boxes. You can also click
 None
 to uncheck all of the boxes.

1. Click
 Refresh Indexes
 .  The status bar indicates the refresh progress. Note: Depending on the number of attachments associated with the modules that you have selected, this process can take several minutes. If you need to stop the process for any reason, click
 Stop
 .

1. When the process is complete click
 Close
 to close the form.
