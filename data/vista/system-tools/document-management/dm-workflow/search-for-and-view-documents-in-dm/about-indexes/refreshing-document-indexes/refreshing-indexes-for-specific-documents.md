# Refreshing Indexes for Specific Documents

To refresh indexes for specific documents, use the DM Manage Attachments form.

1. Search for the documents using DM Attachment Index Search. For more information, see [Searching for Documents](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm).

1. Select the documents you want to copy from the Results grid. You can use Shift-click to select multiple consecutive documents in the grid or use Control-click to select multiple non-consecutive documents.

1. Click
 Manage Selected Attachments
 . The Manage Selected Attachments form displays.

1. On the Indexes tab, click
 Refresh
 . The system refreshes indexes and displays a confirmation dialog box.

1. Click
 Close
 .
