# Creating Document Index Cross-References

The following instructions detail how to create a document index cross-reference.
For more information on the types of cross-references, see [Document Index Cross-References](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm/about-indexes/document-index-cross-references).

1. Select DM Index Cross-Reference from the DM Programs folder in the Main Menu.

1. Enter the name of the master column name in the Master Column Name field. Press F4 for a list of column names.

1. If you are creating a module cross reference, enter the module in the Module field. Press F4 for a list of modules.Note: If you are creating a form cross-reference, you can specify a module but it is not required.

1. If you are creating a form cross-reference, enter the form in the Form field. Press F4 for a list of forms.Note: Do not enter a form if you are creating a module cross-reference.

1. Enter the name of the column you are cross referencing in the Column Name field.Tip: You cannot use the F4 key from the Column Name field. To determine what to enter in this field, locate the field that you want to use as an index option and press F3 to open the Field Properties form. The name of the field will display in the Column Name display field at the top of the Field Properties form. Enter that name in the Column Name field on the DM Attachment Cross Reference form.

1. Enter a cross-reference description in the Description field.

1. If you are creating an exception and do not want to include the column in the cross-reference, check the Skip Index box. Otherwise, make sure that it is unchecked.

1. Save the record as normal. The system will check the Custom display box.
