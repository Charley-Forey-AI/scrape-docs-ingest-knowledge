# About the DM Attachment Index Form

Use this form to view an index for a document.
The system automatically creates an index when assigning an attachment to a data record. These indexes allow users to easily search for a specific document using search criteria. Additionally, you can use this form to create an additional index sequence to help locate the document. For more information on searching for documents, see [Searching for Documents](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm).
