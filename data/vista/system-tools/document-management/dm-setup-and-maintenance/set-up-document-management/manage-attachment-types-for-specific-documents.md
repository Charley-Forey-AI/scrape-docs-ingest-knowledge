# Manage Attachment Types for Specific Documents

You can manage attachment types for specific documents using the DM Manage Attachments form. Managing attachment types includes adding attachment types to specific documents or removing them.

1. Search for the documents using DM Attachment Index Search. For more information, see [Searching for Documents](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm).

1. Select the documents you want to manage from the Results grid. You can use Shift-click to select multiple consecutive documents in the grid or use Control-click to select multiple non-consecutive documents.

1. Click Manage Selected
 Attachments. The Manage Selected Attachments form displays.

1. Select the Attachment Types tab.

1. To add attachment types to
 the documents, select the Add Attachment Type option and
 enter the attachment type in the text box. Press
 F4
 for a list of attachment types.To remove attachment types from the
 documents, select the
 Remove
 Attachment Type
 option.

1. Click Apply to
 All. The system adds or removes the attachment types and displays a
 confirmation dialog box.

1. Click Close.
 The system returns to the Manage Selected Attachments form. Note: If an attachment is already associated with
 an attachment type, you must have access to the original to update it. If you do
 not have access, the system displays a warning and the attachment type does not
 update. Click Yes to see a list of all
 non-updated attachments.
