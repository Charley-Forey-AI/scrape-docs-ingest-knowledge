# Index and Send Emails from Outlook to Vista

You must install and configure MS Outlook. If needed, start here: [About the Vista Add-in for Microsoft Outlook](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/email-integration-with-microsoft-outlook/about-the-vista-add-in-for-microsoft-outlook).
Once you have Installed and configured MS Outlook to integrate with Vista, you can then create indexes for emails from within Outlook and then send them directly to Vista. When sent, the email documents are added to the system as stand-alone documents.
To index and send emails to Vista:

1.  Access the Indexes form inside of Outlook. You can access this form in two ways:

- Highlight an email in the Outlook Inbox, and click the Send To Viewpoint button located in the Outlook ribbon.

- Right-click an email and select Send To Viewpoint.

1. Associate indexes with a selection of Project Management, Accounts Payable, or PM Document criteria by using the lookup or manually entering index values.[1](#fntarg_1)

Outlook sends the email document to Vista, including the specified indexes.
You can view stand-alone documents using the [Attachment Form/DM Stand Alone Documents](/en/vista/vista/system-tools/document-management/dm-workflow/manage-documents/attachment-formdm-stand-alone-documents) form or by using the [DM Attachment Index Search](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm/about-the-dm-attachment-index-search-form) form to locate the document based on the index created.Note: When searching for stand-alone documents using the DM Attachment Index Search form, select the Include Stand Alone checkbox on the Advanced tab.

[1](#fnsrc_1) The Indexes form:

- Provides lookup functionality. However, some lookups are dependent upon data being present in another field. For example, the lookup for the Job/Project field doesn't work if you have not first specified a value in the Company field.

- Does not provide validation on new entries. If you make any data entry errors in this form, correct them using the DM Attachment Index form.
