# Field Definitions: DM Attachment Types Form

The following is a list of field descriptions for the DM
 Attachment Types form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Attachment Type ID

The field displays the ID for the current attachment type.
Enter New , N , or
 + to create a new attachment
 type. This field is display only when on the Grid tab.

##  Name

 Enter the attachment type name. The system allows up to 50 characters.

##  Description

 Enter a description for the attachment type. The system allows up to 255 characters.

## Retention Time

Enter the time in months for the system to retain documents with this type. Once this time has passed, you can delete documents with this type using [DM Expired Documents Purge](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/dm-maintenance/about-the-dm-expired-documents-purge-form).

## Secured

Check this box if you want to set up security on the attachment type. When this box is checked, you can set up security on the attachment type using the [VA Attachment Type Security ](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form) form.
This box defaults to checked when you create a new attachment type record.
