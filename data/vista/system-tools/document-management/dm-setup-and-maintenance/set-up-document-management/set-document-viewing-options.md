# Set Document Viewing Options

If you have purchased scanning functionality for the Document Management module, or if you want to use the DM Document Viewer to view documents in the system, you will need to set two viewing options on the DM Attachment Options form.
You can choose the resolution and your preferred program for viewing documents.

1. Open the DM Attachment Options form.

1. In the PDF View Resolution field, set the resolution for PDF documents.This field's default value is 120 dpi, but you can choose a setting between 72 and 300 for viewing documents in the DM Document Viewer, Batch Scan Viewer, and Single Scan Viewer forms.

1. Note: The system uses the default program installed on your computer when displaying all document file types.
If when viewing .TIF, .PDF, and .JPG files you want them to display in the DM Document Viewer, select the Use Viewpoint Viewer checkbox.
