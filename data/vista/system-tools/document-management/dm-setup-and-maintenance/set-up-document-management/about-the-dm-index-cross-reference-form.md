# About the DM Index Cross Reference Form

You can use this form to create document index cross-references.
Cross-references are used when form fields have the same purpose as fields in the [DM Attachment Index ](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm/about-the-dm-attachment-index-form) and [DM Attachment Index Search ](/en/vista/vista/system-tools/document-management/dm-workflow/search-for-and-view-documents-in-dm/about-the-dm-attachment-index-search-form) forms, but have differing database column names. Cross-references can help to tie these fields together and provide increased search capabilities. Additionally, you can use this form to create custom index options.
