# Updating Documents with a New File System Directory Path

When you are storing documents in a file system, you can update document paths without having to move the files.
This option is useful if you have changed servers, causing the existing path to be invalid.
The following instructions guide you through the process of updating the directory path for documents.

1. Select the DM Manage Attachments form from the DM Programs folder on the Main Menu. The DM Manage Attachments form displays.

1. Select the Update tab.

1. Enter the original path name in the Old Path field. Optionally, click the Browse button to locate the appropriate directory.

1. Enter the new path name in the New Path field. Optionally, click the Browse button to locate the appropriate directory.

1. (Optional) Check the Include Subfolders box to include all documents in the specified directory’s subfolders, if any exist.

1. (Optional) Check the Keep Folder Structure box to duplicate the folder structure in the new path.

1. Click Update.  The system updates the path without moving the files.
