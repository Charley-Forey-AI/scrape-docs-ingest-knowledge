# Copying Documents to a New File System Directory

When you are storing documents in a file system, you may need to copy all of the documents to a new directory. To copy all documents, you will use the DM Manage Attachments form.
The following instructions detail how to copy documents to a new directory.

1. Select the DM Manage Attachments form from the DM Programs folder on the Main Menu. The DM Manage Attachments form displays.

1. Select the Copy tab.

1. Enter the location to copy the document from in the Copy attachments from field. Optionally, click the Browse button to locate the appropriate directory.

1. Enter the location to copy attachment to in the Copy attachment to the specified location field. Optionally, click the Browse button to locate the appropriate directory.

1. (Optional) Check the Include Subfolders box to copy all documents in the specified directory’s subfolders (if any exist). Note: For example, if there is a directory structure of \\Viewpoint\Attachments\Month, and you specify \\Viewpoint as the source location, the system copies all documents in the Viewpoint directory, as well as any that exist in the Attachments and Month directories.

1. (Optional) Check the Keep Folder Structure box to duplicate the folder structure from the source location in the destination location. Note: The directory structure must be the same in both locations.

1. Click Copy.  The system copies the documents from the source location to the destination location.
