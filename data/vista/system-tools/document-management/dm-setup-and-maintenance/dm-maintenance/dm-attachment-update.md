# DM Attachment Update

If you decide to change your document storage location to either a file system or location or the Vista™ database, you will use this form to move existing documents to the new location.
You can also use this form to move documents stored in a file system to a new subdirectory structure.
You can access this form by clicking the Update Attachments button in DM Attachment Options or from the DM Programs folder on the Main Menu.
