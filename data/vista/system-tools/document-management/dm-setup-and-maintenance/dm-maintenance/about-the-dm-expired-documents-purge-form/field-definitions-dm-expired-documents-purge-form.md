# Field Definitions: DM Expired Documents Purge Form

The following is a list of field descriptions for the DM
 Expired Documents Purge form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Filter

Select an option for backing up or deleting documents:

- Single Attachment Type - Select this option to back
 up or delete documents associated with a specific attachment type. If you select this
 option, enter an attachment type in the Attachment Type field to the right.

- All Attachment Types - Select this option to back up or delete documents associated with all attachment types.

## Attachment Type

Enter the attachment type associated with the
 documents that you want to back up or purge. Press F4 for a list of valid attachment
 types.
