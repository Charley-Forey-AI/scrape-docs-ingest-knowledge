# Deleting Documents from a File System Directory

The following instructions detail how to delete documents from a file system directory.

1. Select the DM Manage Attachments form from the DM Programs folder on the Main Menu. The DM Manage Attachments form displays.

1. Select the Delete tab.

1. Enter the directory to delete documents from in the Delete attachments with the following path field. Optionally, click the Browse button to locate the appropriate directory.

1. (Optional) Check the Include Subfolders box to delete all documents in the specified directory’s subfolders, if any exist.

1. Click Delete.  The system deletes the documents. Note: If you checked the Retain deleted
 attachments box in DM Attachment Options, the system saves
 the document, but removes all links and indexes. You can search for these
 deleted documents using DM Attachment Index Search. Deleted documents
 display in the grid with a current state of "Deleted." If you are deleting a
 document in a "deleted" state, the system displays a warning stating it will
 purge the "deleted" document. Click Yes to completely purge
 the document from the system.
