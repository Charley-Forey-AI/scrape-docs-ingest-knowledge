# Backing Up Expired Documents

You can back-up documents to a network directory based on attachment type and retention time using the DM Expired Documents Purge form.
You can back-up all documents associated with a single attachment type or for all attachment types. Typically, you may want to back-up your documents prior to purging them from the system. For more information on setting retention limits for documents, see [Setting Up Attachment Types](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/set-up-attachment-types).
The following instructions detail how to backup expired documents in the system.

1. Select the appropriate radio button from the Filter section: Single Attachment Type or All Attachment Types.

1. If you selected Single Attachment Type, enter the attachment type in the field to the right of the radio button. Press F4 to see a list of valid attachment types.

1. Click Backup. The system displays a reminder that you should have security access for the attachment type.

1. Click Yes to proceed. The system opens a Browse For Folder window.

1. Locate and select the directory for backing up the documents.

1. Click OK to back up the documents. A confirmation message displays.

1. Click Close to return to the DM Expired Documents Purge form. You can now purge the documents from the system. For more information, see [Purging Expired Documents](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/dm-maintenance/purging-expired-documents).
