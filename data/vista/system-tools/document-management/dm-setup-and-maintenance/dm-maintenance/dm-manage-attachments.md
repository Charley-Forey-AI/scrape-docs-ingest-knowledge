# DM Manage Attachments

Use this form to manage documents.
If you are storing documents in a network file system, you can access this form from the DM Programs folder in the Main Menu and use it to copy all documents to a new network location, delete all documents from a network directory, or update all documents with a new network directory path. You cannot access this form in this way if you are storing documents in a database.
Regardless of your document storage location, you
 can access this form by clicking the Manage Selected Attachments button in
 DM Attachment Index Search. When accessing the form in this way, you can copy
 specific documents to a new location, delete specific documents, manage attachment types
 for specific documents, and refresh document indexes for specific documents.
For more information on document storage locations, see [Document Management Storage Options](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/set-up-document-management/document-management-storage-options).
