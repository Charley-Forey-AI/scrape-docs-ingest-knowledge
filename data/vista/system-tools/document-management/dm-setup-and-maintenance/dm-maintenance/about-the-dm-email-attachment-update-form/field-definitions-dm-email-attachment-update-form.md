# Field Definitions: DM Email Attachment Update Form

The following is a list of field descriptions for the DM
 Email Attachment Update form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Update

Check this box for each attachment in the grid that you want to update.
Uncheck this box for those attachments that you do not want to update.
Note: You can click
 Select All
 to check the
 Update
 box for all attachments in the grid or
 Clear All
 to uncheck the
 Update
 box for all attachments.
