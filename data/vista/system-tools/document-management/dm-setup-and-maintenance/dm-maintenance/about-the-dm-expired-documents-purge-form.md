# About the DM Expired Documents Purge Form

Use this form to purge documents with a specific attachment type from the system.
The system only deletes documents that are older than the time you specified in DM Attachment Types. Optionally, you can have the system back-up documents to a specific network directory before you purge them from Vista™. When using this form you can choose to purge/back-up all documents associated with a single attachment type, or for all attachment types.
When you purge documents with this form, the attachments are removed from the system, regardless of the settings in DM Attachment Options. For more information on these settings, see [Configuring Document Retention and Auditing Settings](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/dm-maintenance/configuring-document-retention-and-auditing-settings).
Note: Before backing-up or purging documents, you must have
 security access to the attachment type. See [VA Attachment Type Security](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form) for more information on
 setting security access.
