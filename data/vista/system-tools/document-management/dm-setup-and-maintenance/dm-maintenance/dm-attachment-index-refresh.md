# DM Attachment Index Refresh

Because information for a data record is editable, information in a document index can become inaccurate, preventing you from accurately searching for a document.
Use this form to refresh document indexes by module, allowing you to keep document indexes up-to-date. In addition, you can use this form to create custom index options that are not available using the standard index fields.
