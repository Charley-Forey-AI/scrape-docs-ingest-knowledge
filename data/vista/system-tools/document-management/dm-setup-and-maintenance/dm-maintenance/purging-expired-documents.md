# Purging Expired Documents

You can purge documents from the system based on attachment type and retention time using the DM Expired Documents Purge form.
You can purge all documents associated with a single attachment type or for all attachment types.
Note: In order to purge documents from the system, you must have security access to the associated attachment type. See[VA Attachment Type Security](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-attachment-type-security-form) for more information.
The following instructions detail how to purge documents from the system.
Note: Prior to purging documents, you may want to back them up to a network directory. For more information, see [Backing Up Expired Documents](/en/vista/vista/system-tools/document-management/dm-setup-and-maintenance/dm-maintenance/backing-up-expired-documents).

1. From DM Expired Documents Purge, select the appropriate radio button from the Filter section: Single Attachment Type or All Attachment Types.

1. If you selected Single Attachment Type, enter the attachment type in the field to the right of the radio button. Press F4 to see a list of valid attachment types.

1. Click Purge. The system displays a reminder that you should have security access for the attachment type.

1. Click Yes to proceed. The system displays a purging confirmation message .

1. Click Yes to purge the documents. A confirmation message displays.

1. Click Close to return to the DM Expired Documents Purge form.
