# Change the Active Company

You can work in multiple companies in the system. You can
 change the company you are working in from the Main Menu.
You can change the active company by selecting the
 company from the Company drop-down list; the
 system will automatically display the selected company in the logo panel (if you
 selected View > Display Logo Panel). In addition, all processing will be posted to the active
 company as defined in the company files for each module.Note: This option is also available from the
 standard Dashboard Work Center, as well as any additional Work Centers you
 have added to the Main Menu.
Important: It is recommended that you do not
 change the active company using the Company drop-down list
 while you are still working in a form. Instead, save your work and close all
 open forms before switching to another company.
