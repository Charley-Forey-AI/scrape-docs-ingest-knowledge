# Active Job

The Active Job field on the Change Active Information form.

Enter
 the job number to make active. Press F4 for a list of valid jobs.
The job specified here defaults when you access the following forms: JC Jobs, JC Job Phases, JC Original Estimates, JC Change Orders, JC Job Copy, and SL Subcontract Claims. Changing the job in any of those forms automatically updates the active job specified here.
