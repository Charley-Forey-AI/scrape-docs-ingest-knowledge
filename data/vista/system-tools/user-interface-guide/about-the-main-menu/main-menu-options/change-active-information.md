# Change Active Information

You can change the active information for jobs, projects, and pay periods from the Main Menu. The active information affects how forms default this information.
The changes made in this form do not affect Dashboards, Accounting Work Centers, SM Work Centers, or PM Work Centers.

1. From the Main Menu, select View > Change Active Info.

1. To set a new active project, enter the project number in the Active Project field. This project will default each time you access a form with a Project key field (e.g., PM Projects, PM Change Orders, PM Material Detail, PM Meeting Minutes, etc.).

1. To set a new active job, enter the job number in the Active Job field. This job defaults when you access the following forms: JC Jobs, JC Job Phases, JC Change Orders, JC Cost Projections, and JC Revenue Projections.

1. To set a new active payroll period, enter the payroll group in the Active Group field and the payroll ending date in the Active End Date field. This information defaults when you open PR Pay Period Control.

1. Click OK to save the active information.
