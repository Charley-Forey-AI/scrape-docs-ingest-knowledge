# About the Logged On Users form

You can use the Logged On Users from to view a list of users currently logged on to the system.
You can access the Logged On Users form by selecting Help > System > Logged On Users from the main menu and from most forms.
In addition to displaying a list of the users currently logged into Vista, you can also see how many licenses are in use, select to show your logins or all logins, and email a specific user or all users.
Note: The form displays only if you have form security permissions to view it or the Logged On Users use Form Security checkbox is not selected in VA Site Settings.
