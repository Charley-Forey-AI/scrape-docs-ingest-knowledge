# Search the Main Menu

You can easily find forms and reports by using the Main Menu
 Search functionality.
 You can search all folders in the Vista Main Menu,
 individual modules, and the My Tasks folder. The Search functionality will look for
 forms and reports whose names contain the text that you enter into the search
 box.

1. To search all modules, select the
 Search folder in the folders pane. In the
 Search field, located above the Items pane, enter
 some or all of the form or report name. Forms and reports whose name contain the
 text you enter will display in the Items pane.

1. To search with a module, open the module you want in the Folders pane and click
 either the Programs or Reports
 folder. In the Search field, enter some or all of the
 form or report name. Forms and reports whose name contain the text you enter
 will display in the Items pane.

1. To search within the My Tasks folder, select the My Tasks folder and in the
 Search field, enter some or all of the form or report
 name. Forms and reports whose name contain the text you enter will display in
 the Items pane.

Once the form or report you are looking for displays
 in the Items pane, you can launch it from there. You can also copy it into a
 user-defined subfolder or the My Tasks folder.
