# About the Work Centers form

You can use the Work Centers form to add, remove, or customize Work Centers that display on your application window. You can access this form by selecting Options > Work Centers from the main application menu.
Use this form to:

- Add, remove, or customize the Work Centers that display on your application window.

- Apply a Work Center that another user has created and made public.

- Apply a Work Center profile (created using [VA Work Center Profile](/en/vista/vista/administration/viewpoint-administration/work-centers/about-the-va-work-center-profile-form)) to your user account.Tasks
