# Password

Locking a SessionThis prompt only displays if you did not use a password when you logged on to Vista™.
Enter a temporary password. This password will be used to unlock the software session and will remain in effect until you exit out of the software.
Unlocking a SessionEnter the password you used to log on to the software or enter the temporary password you created when you locked your session and select the Unlock Session button.
