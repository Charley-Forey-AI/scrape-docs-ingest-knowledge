# Add Links to External Websites

You can add links to external websites directly in the system.

1. From the Main Menu, select
 Links > Organize
 Links.

1. In the Organize Links form,
 click Add
 Link. If you are editing an existing link, click Edit
 Link.

1. In the Add/Edit Link form,
 enter or edit the name for the link in the Name field.

1. Enter the website's URL in
 the Link
 Address field. Make sure to enter the entire address. For example, a
 link to Google would be entered as: www.google.com.

1. To reorder the list of links click Move Up or Move Down
 for each link you want to move.

1. Click OK. Note: To delete a link, select Links > Organize Links, select the link, and click Delete Link
