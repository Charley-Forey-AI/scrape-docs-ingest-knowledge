# Trimble Work Center

The Trimble Work Center provides access to multiple Trimble applications from one easy-to-use dashboard. It also includes optional dashboards that you can select to suit your needs.

If you have Trimble Construction One enabled on your system, you can quickly access the Trimble Work Center from Vista by clicking the Trimble Work Center icon () on the main menu toolbar.
Note: Access to the Trimble Work Center requires that your company's subscription is part of our Trimble Construction One contract, and that you sign in with a Trimble ID.

Logging into Vista version 2024 R1 or higher using Trimble ID automatically launches the Trimble Work Center on a separate browser tab.
 For first time users of Trimble ID, logging into Vista presents the following prompt:

 Select Open ViewpointURILauncher to continue with the login process.
If you do not have Trimble Construction One enabled on your system, clicking the Work Center icon presents you with the [Trimble Construction One](https://construction.trimble.com/en/trimble-construction-one) website.
For information about the Trimble Work Center, see [Introducing the Trimble Work Center](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:WC_000002:Workcenter:Workcenter).
