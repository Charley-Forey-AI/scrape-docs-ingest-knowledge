# Configure the Grid Section

In the Grid section of the Dashboard Work Center, you can
 customize the grid in a query to hide or display certain columns.
If the query that you add to the grid is related to other queries, you can drill down to view more detailed information.

1. Click the Inquiry Settings () icon at the top of the Grid section.

1. Select the query in the Base Inquiry field. Queries
 are created and maintained using the VA Inquiries form.Note: A query will display in this
 drop-down menu only if:

- The Assignable in Work
 Center checkbox is selected for this query
 in the VA Inquiries form, and

- It is associated with the PM Work Center
 template on the Associations tab on the VA Inquiries form,
 and

- You have security set to access it in the
 VA Inquiry Security form.

1. In the Name field, enter the name of
 the query.Note: This field allows you to have multiple
 instances of the same query in a single Work Center - for example if you
 have several versions of the Hot List on the PM Work Center but changed the
 parameters on each so they display different results.

1. Enter the maximum number of rows that you want the grid to display in the Maximum Rows field.

1. Use the options in the Toolbar Visibility section
 to select which buttons are included in the toolbar at the top of the Reports
 panel. If you clear all of these options, the toolbar will not display.

1. To hide a column in the grid, clear the box next
 to the column name in the Visible Columns selection box.

1. On the Parameters tab, use the Parameter Value field to
 configure each grid parameter, as applicable. For more information on available
 parameters, see [About Parameter Values](/en/vista/vista/system-tools/user-interface-guide/about-the-main-menu/work-centers/about-the-dashboard-work-center/about-parameter-values).

1. Click OK to close the Inquiry
 Settings form.The Grid section displays the
 query information that you selected.

1. To reorder grid columns in the display, click and
 drag the column headers to the correct location. For filtering, right click in
 the grid and select Show Filter Bar. The filter bar displays at the top of the
 grid.
