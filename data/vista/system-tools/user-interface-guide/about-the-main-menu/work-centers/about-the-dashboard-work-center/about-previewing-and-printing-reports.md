# About Previewing and Printing Reports

When working in the Dashboard Work Center, you can use the toolbar across the top of the Report section to change report view settings, print the report, and more.

- Document Map – display the document map to the left of the
 report, used to locate each new section of the report.

- Navigation Arrows – scroll through the report. The arrow
 options include: First Page, Previous Page, Next Page, and Last Page.

-  Current Page – displays the current page of the report. To
 go to a new page, enter the Page Number and press Enter.

-  Stop Rendering – stop the report from generating.

-  Refresh – update the report with current data.

- Print – the print dialog box appears.

- Print Layout – see how the system formats the report for
 printing.

- Page Setup – change the page setup including, the paper
 size, source, orientation, and margins. You can also print directly from the
 Page Setup window.

- Export – export to either an Excel spreadsheet or to PDF.

- Zoom – select a zoom setting from this drop-down list.
