# Keyboard Shortcuts for Moving Through Form Fields

When entering data into the system, there are several ways to
 move from one field to another.
MethodDescription
TABMoves
 you through the fields on a form in a pre-defined order.
SHIFT +
 TABSame as
 TAB, but in the opposite direction.
ENTERWorks
 the same as TAB if the Enter as Tab option
 is checked on the [VA User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) or [User Options](/en/vista/vista/system-tools/user-interface-guide/about-the-main-menu/main-menu-options/about-the-user-options-form) forms.
Accelerator KeysAccelerator keys are identified by an underscore (‘_’) on an field's
 label. When you first access a form, the accelerator keys may not be
 immediately identifiable (depending on Windows settings). If this
 occurs, press the ALT key to display the accelerator keys available in
 the menu and on the form (if applicable), then press the desired letter
 to take you to the menu or input.
Arrow
 KeysIf you
 are working in a grid, the arrow keys will move you up/down and
 right/left through the grid.When on the other tabs on a form, they will only
 move you through the data in a field or, if a radio button control,
 through the options on the control.

HOME /
 ENDIf you
 are working in a grid, the Home key will move you to the first column in
 the grid and the End key will move you to the last column in the grid.If working on a
 standard form entry tab, the Home and End keys will only move you to
 the beginning or ending positions in the current field
 (respectively), with the exception of checkboxes and radio buttons,
 in which case the Home and End keys have no function. In a drop-down
 field, the Home keys will move you to the first option in the list
 and the End key will move you to the last option in the
 list.

PAGE UP/ PAGE
 DOWNWhen on
 the Grid tab of a form, use these keys to move through the records in
 the grid.When on a
 non-grid tab in a form, use these keys to move between the header
 and footer of the form. This will also save any information that has
 been changed.
