# Set the Grid as the Default Tab

Most forms in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can set this tab as the
 default view when opening forms.
In most forms, when you open a form, the Info
 tab is open by default. However, you can set the form to open to the grid by default.
 Each user can make the Grid tab be the default by editing the form's
 properties.

1. In the Items pane of the
 Main Menu, select the form, and then in the Items menu, select Form
 Properties. The Form Properties form opens.

1. In the User's Default
 Tab drop-down list, select 0-Grid Tab.

1. Click Close.

The next time you open the form, it will open
 to the Grid tab.
