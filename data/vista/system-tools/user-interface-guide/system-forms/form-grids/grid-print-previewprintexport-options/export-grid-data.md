# Export Grid Data

Most forms in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can export the records that display on the Grid tab.

1. Right-click in the grid, and in the context menu, select Export Grid.

1. In the TrueDBGrid Print/Export Options form, select an export format option in the Action field.

1. Click Browse to navigate to and select a download location.

1. Enter a file name for the export file in the File name field.

1. If desired, click the Options button to open a Save As window, where you can enter additional setup options based on the selected export format. For example, if exporting to PDF, you can specify whether to use compression, embed True Type fonts, define document security, allow editing, etc.

1. Accept the defaults in the rest of the form, or if desired, use the remaining options to control how the data will be presented when exported.Note: The Options button is disabled when exporting to C1 or C1 Open XML documents.

1. Click OK. The file is exported to the directory you specified.
