# Edit Grid Records

Most form in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. To edit records within a
 grid, you must activate the edit mode for the cell you wish to change.
Tabbing to a cell or clicking in a cell with the
 mouse only selects the cell; it does not automatically place you in edit mode.

1. Activate edit mode using one of the following
 options:

- When using the mouse, double-click in the desired cell.

- When using the keyboard, move to the desired cell using the Tab or Enter key, and press the
 F2 function key.

1. Edit the cell and press the Tab key to exit edit mode and move to the next
 cell.
