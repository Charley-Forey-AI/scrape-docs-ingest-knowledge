# Hide Grid Field Descriptions

Most forms in the system have a Grid tab. It is used for
 viewing, entering, or editing records in a spreadsheet format. You can hide field
 descriptions in the grid to eliminate the need to decrease the size of the grid.
Many fields have an associated decription field. You can hide the description without
 removing the field itself, or you can make the description display above the grid.

1. In the field whose
 description you want to hide, press F3.

1. In the Field Properties
 form, select a tab to determine the level at which you want to hide the
 description of this field:

- System Overrides (accessible if you are
 set as a Form Administrator in VA User Profile) - affects all users,
 or

- User Overrides - affects only the
 current user.

1. Select an option in the
 Description in Grid list.
