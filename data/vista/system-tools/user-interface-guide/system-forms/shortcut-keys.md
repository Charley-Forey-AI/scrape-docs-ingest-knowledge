# Shortcut Keys

You can use shortcut keys for quicker data entry which can reduce the need for the mouse.
Shortcut KeyDescription
Ctrl +
 CCopy currently selected data from a field or grid cell.You can also select Edit > Copy from the menu at the top of the form.

Ctrl +
 VPaste copied or cut data from the previously selected field or grid cell.You can also select Edit > Paste from the menu at the top of the form.
Note: May also be pasted into an
 existing record; however, be aware that it will replace the existing
 data.

Ctrl + Shift +
 CCopy data from the entire selected row.
Ctrl + Shift +
 VPaste data from the previously selected row.
Ctrl +
 HMove to first record.You can also use the
 First Record button () on the
 toolbar or select Records > First from the menu at the top of the form.

Ctrl +
 JMove to previous record.You can also use the
 Previous Record button () on the
 toolbar or select Records > Previous from the menu at the top of the form.

Ctrl +
 KMove to the next record.You can also use the
 Next Record button () on the
 toolbar or select Records > Next from the menu at the top of the form.

Ctrl +
 LMove to last record.
Ctrl +
 SSave record.You can also use the
 Save button () on the toolbar or select Records > Save from the menu at the top of the form.

Ctrl +
 XCutting data removes currently selected data from a field or grid cell.
