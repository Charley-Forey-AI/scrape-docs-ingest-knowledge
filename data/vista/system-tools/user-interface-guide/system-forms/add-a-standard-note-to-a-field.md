# Add a Standard Note to a Field

You can add standard notes to fields in the system using the Standard Note Copy
 form.
You must create a standard note in the HQ Standard
 Note form.
Access this form by right-clicking in a field that
 has the standard notes feature and selecting Standard Notes from the context
 menu.

1. In the field you want to add the standard note to, right click and select
 Standard
 Notes from the context menu.

1. In the Standard Note to
 Copy field, press F4 to select a note from a
 list.Note: To create and use a new standard note, press
 F5 in the Standard Note To Copy field to
 open the HQ Standard Note form.

1. Click OK. This
 will close the form and populate the standard note in the selected field.
