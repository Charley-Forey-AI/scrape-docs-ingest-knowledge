# Configure the Vista Client for LAN Connection

You must configure the Vista client at each workstation to access your server.
The following steps are for new users or when you have
 just re-installed the client on a workstation that's already been set up. These
 instructions are specific to users whose organization's deployment method is
 on-premises.Before you can access Vista, make sure the
 following is complete:

- [Install the Vista Client Application on a
 Workstation](/en/vista/vista/system-tools/user-interface-guide/access-vista---local-area-network-lan/install-the-vista-client-application-on-a-workstation).

- Obtain your server name, database name, and host from
 your system administrator, or by viewing the client configuration on a
 workstation that is already connected to the server.

To connect your workstation client to the
 server:

1. Open the Vista application.A dialog box appears.

1. Select Yes to review connection settings.The Vista Configuration screen appears.

1.  Select the LAN option.

1. Enter the information in the Server Name, Host, and Name fields. Select OK.The Log In screen appears.

You can now log in to the Vista application using your username and password or your
 Windows login. For details, see [Log in to Vista Using
 Classic Credentials](/en/vista/vista/system-tools/user-interface-guide/log-in-to-vista/log-in-to-vista-using-classic-credentials).
