# Log in to Vista

Your login options vary depending on your organization's
 Vista deployment method.
When Vista is installed or re-installed, you must enter the name of the
 Vista application server you are connecting to
 in order to log in.

1. Open the Vista application.What displays next varies by your organization's current
 deployment method.

1. Take the appropriate actions to authenticate and
 log in to Vista:

- If you have a Trimble ID:
Select Sign In With
 Trimble ID and complete
 the login process. For details, see [Log in to Vista Using Trimble ID](/en/vista/vista/system-tools/user-interface-guide/log-in-to-vista/single-sign-on-with-trimble-id/log-in-to-vista-using-trimble-id).

- If you do *not* have a
 Trimble ID:
Select Sign In With
 Classic ID.

- To use a SQL login, enter your VistaUser Name and Password.

- To log in with Windows credentials, select the Use Windows Login checkbox, and
 enter your Windows domain password.

For details, see [Log in to Vista Using
 Classic Credentials](/en/vista/vista/system-tools/user-interface-guide/log-in-to-vista/log-in-to-vista-using-classic-credentials).
Note: If you are a Trimble Construction One cloud-hosted Vista customer, this option to log in using classic credentials (SQL or Windows logins) will no longer be supported in the fall of 2025. At that point, you will have to log in using [Trimble ID](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:vista_60004).
