# Access Vista - Local Area Network
 (LAN)

Use the log in screen to configure Vista to securely
 connect to your company's on-premises Vista database via a
 Local Area Network (LAN).
The steps on this page are for new users or when you
 have just re-installed the Vista client on a workstation that may have
 already been set up. These instructions are specific to users whose
 organizations have on-premises deployment.Before
 you can access Vista, make sure the following is
 complete:

- [Install the Vista Client Application on a
 Workstation](/en/vista/vista/system-tools/user-interface-guide/access-vista---local-area-network-lan/install-the-vista-client-application-on-a-workstation).

- [Configure the Vista Client for LAN Connection](/en/vista/vista/system-tools/user-interface-guide/access-vista---local-area-network-lan/install-the-vista-client-application-on-a-workstation/configure-the-vista-client-for-lan-connection).

- Obtain your server name
 from your system administrator or by viewing the
 client configuration on a workstation that is
 already connected to the server.

Note: We recommend that you do not modify the configuration
 settings once you have completed the initial setup unless
 directed to do so by your administrator.
To connect your workstation client to the server:

1. Open the Vista application.

1. Then select Sign in with Classic ID.

1. On the Vista log in screen, select the plus sign next to the Vista Server field and enter the URL provided by your administrator, starting with *https://*.

1. Select the check mark to validate your entry.

1. Complete the login process according to your normal procedure.For details, see [Log in to Vista Using
 Classic Credentials](/en/vista/vista/system-tools/user-interface-guide/log-in-to-vista/log-in-to-vista-using-classic-credentials).
Note: After a successful login,
 you can check the status of your connection to the
 server using the connection indicator in the
 bottom right corner of the Vista application window.
