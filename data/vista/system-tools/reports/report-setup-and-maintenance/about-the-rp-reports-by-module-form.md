# About the RP Reports by Module Form

Use this form to add your report to the Reports menu of selected modules.
This form can be accessed from the Main Menu or by
 double-clicking a record on the Assigned Modules tab in RP Report Titles. This form
 applies only to reports with an application type of Crystal or SQL Reporting Services
 (Application drop-down field in RP Report Titles).
Typically, you will assign reports to associated
 modules. For example, you would assign an AP Vendor report to the AP Reports menu; you
 would likely not assign it to the GL Reports menu.
