# Printing a Report

You can print a selected Crystal or SSRS report.
Note: When launching an SSRS report, the RP Report Launcher form displays only if Launch Parameters Form is selected in the SSRS Report Viewer Options drop down on the Info tab of the [VA User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) form.
To print a report:

1. Open the RP Report Launcher form by launching a report using a Work Center, the Reports folder in a module, or a form (Options > Reports).

1. Complete the necessary report parameter fields on the Input tab.

1. Click the Print button. The Print form displays once the report is generated.

1.  Use the Print form to set up how the report will print (print range, number of copies, etc.).

1. (Optional) Use the Properties button to change the printer settings of the selected printer.

1.  Click OK to print the report. Note: If you need to print a drilldown report (reports that drill down to several levels of information), you will need to print each level separately. Drilldowns provide several levels of information. At each level, double-clicking on specific data will drill you down to another level. If you wish to print the information provided on a specific level, you must open the report, then drill down to the level you wish to print.

Note: To change printer settings, see [Create, Change, or Delete Report Printer Settings](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/create-change-or-delete-report-printer-settings#task-7705--en__task-7705).
