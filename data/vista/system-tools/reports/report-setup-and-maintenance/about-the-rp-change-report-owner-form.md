# About the RP Change Report Owner Form

Use this form to change the owner of a report.
Only the owner of a report can modify or delete it.
Access this form by selecting File > Change
 Owner in the RP Report Titles form.
[Changing a Report Owner](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/changing-a-report-owner)
