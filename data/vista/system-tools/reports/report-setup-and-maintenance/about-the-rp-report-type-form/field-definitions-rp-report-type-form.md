# Field Definitions: RP Report Type Form

The following is a list of field descriptions for the RP
 Report Type form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Report Type

Enter a code, up to 10 characters, that
 identifies this type of report. Press F4 to see a list of available report
 types.

## Description

 Enter a description of this report type, up to 30 characters. This field is for informational purposes only.

Related information

- [About the RP Report Type Form](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-report-type-form)

## Active

Select this checkbox to activate this report type. When the checkbox is selected, this report type is available for selection when pressing F4 at the report type field on RP Report Titles.
Note: All report types, when added to the grid, automatically default to Active.
If the checkbox is cleared, this report type is inactive (i.e. is not activated when F4 is pressed at the report input).
