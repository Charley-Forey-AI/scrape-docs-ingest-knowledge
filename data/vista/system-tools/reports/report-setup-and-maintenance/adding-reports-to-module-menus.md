# Adding Reports to Module Menus

Custom reports can be added to any of the Report menus in the application.
The following procedure will guide you through the process.

1. Put the custom report on your server in the directory shown below. If this folder does not exist, you must create it first. Viewpoint Repository\Reports\Custom
Note: You can create additional folders in the Custom directory based on how you prefer to store reports (e.g. Viewpoint Repository\Reports\Custom\AP or Viewpoint Repository\Reports\Custom\Company A\AP).

1. Confirm that a location is set up in [RP Report Locations](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-report-locations-form). If a location does not already exist, add one using your company name (or an abbreviation of your company name) as the Location. The path name should be the same as designated above (i.e. \\Servername\Viewpoint Repository\Reports\Custom).

1. Add a new report title using [RP Report Titles](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-report-titles-form). Consider using the same prefix identifier for all user-created reports. This sorts customer created reports together. For more information on RP Report Titles, see the Related Topics section below.

1. Once you have entered
 all the appropriate information, click the Update Parameters button. The
 parameters for your report are now updated. For detailed information about
 Report Parameters, see Related Topics below.

1. Select the Show on
 Menu checkbox to add the report to any specified modules in RP
 Reports by Form.

1. Save the report title (by clicking the Save button in the toolbar, pressing Ctrl + S, or selecting the Save option from the Records menu).

1. Assign the report to specific modules and forms by using [RP Reports by Module](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-reports-by-module-form) and [RP Reports by Form](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-reports-by-form-form). The report should appear in the Reports menu for each specified module and be available in the Reports menu on specified forms.

1. If you are using security, assign security to the report in [VA Report Security](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-report-security-form).
