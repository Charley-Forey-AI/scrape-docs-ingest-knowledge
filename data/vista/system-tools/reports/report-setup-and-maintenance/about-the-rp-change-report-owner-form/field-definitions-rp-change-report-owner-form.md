# Field Definitions: RP Change Report Owner Form

The following is a list of field descriptions for the RP
 Change Report Owner form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  New Owner

 Enter the login name from [VA User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) that you want assigned as the new owner of
 this report, You can also press F4 to select a user from a list.

##  Update All Reports Assigned to Current Owner

 Check this box to update all reports assigned to the specified 'current owner' with the new owner.
 Do not check this box if you do not want all reports assigned to the specified 'current owner' to be updated with the new owner. Only the report specified above will be updated.
