# About the RP Report Type Form

Use this form to set up report types.
Report types are used to classify the different types of reports used by your company and can be used as a filter when assigning security in [VA Report Security](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-report-security-form). Some examples of report types are:

- Accounting

- Project Management

- Audit

- Forms (for pre-printed forms)

- Drill Down

- Error

The report types defined here are assigned to reports in RP Report Titles. All standard report types are stored in the RPTY table. Custom report types or overridden standard types are stored in the RPTYc table. The grid on this form displays both the RPTY and RPTYc tables in a single view, with types being labeled either “Custom” or “Standard.”
