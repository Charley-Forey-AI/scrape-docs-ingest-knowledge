# About the RP Report Layout Form

This form is used to update report layouts to the RP Report Objects (RPRF) table.
When run, the program reads the range of specified reports and then updates the table with the appropriate formulas and database fields. The Report Information report uses this data to provide information about each report.
In order to keep the RPRF table populated with the correct data, run this program each time changes are made to a report’s layout (in the application that you created the report in).
