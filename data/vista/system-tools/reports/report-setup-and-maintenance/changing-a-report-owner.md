# Changing a Report Owner

In order to edit or delete a report, you must be the report's owner. However, you can change the ownership when needed.
Note: You cannot change the ownership of reports owned by “viewpointcs”.
To change a report owner:

1. In the RP Report Titles
 form, enter the report whose owner you want to change in the Report
 ID field, or press F4 to select from a
 list.

1. Confirm that the value
 in the Owner field is not "viewpointcs".

1. Select File > Change Owner. The RP Change Report Owner form opens.

1. In the New
 Owner field, press F4 to select from a list of
 valid user logins.

1. If desired, update all
 reports belonging to the same current owner by selecting the Update All
 Reports Assigned to Current Owner checkbox.

1. Click Update.
