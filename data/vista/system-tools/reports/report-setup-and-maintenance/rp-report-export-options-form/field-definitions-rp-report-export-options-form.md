# Field Definitions: RP Report Export Options Form

The following is a list of field descriptions for the RP
 Report Export Options form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Export Type

The Export Type drop-down on the RP Report Export Options form.
Select the export format for the report. The following options are available:

- Crystal Report (.rpt)

- Rich Text File (.rtf)

- Word for Windows (.doc)

- Excel Format (.xls)

- Portable Document File (.pdf)

- HTML 4.0 file (.html)

- Excel Format Data Only (.xls)

- Text (.txt)

- Character Separated Value file (*.csv)

- Tab Separated (.txt)

- Editable Rich Text file (*.rtf)

- XML

- Excel Format Data Only (.xlsx) — available only for Crystal Reports

## Page Range

The Page Range field on the RP Report Export Options form.
This field displays for the following export options only:

- Rich Text File (.rtf)

-  Word for Windows (.doc)

- Excel Format (.xls)

- Portable Document File (.pdf)

- HTML 4.0 file (.html)

- Character Separated Value file (*.csv)

- Editable Rich Text file (*.rtf)

Select a page range option for exporting the report. The following options are available:

- All - Select this option to export the entire report.

- Page Range - Select this option to export a specific page range. When you select this, the system enables the From and To fields.

##  From

The From field on the RP Report Export Options form.
Enabled only when you select the Page Range option in the Page Range group box.
Note: The Page Range options are only available for selected report export options. For more information, see [Page Range](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/rp-report-export-options-form/field-definitions-rp-report-export-options-form#ID-0003d8df--en).

Enter the beginning page number in a range of pages to export for this report.

##  To

The To field on the RP Report Export Options form.
Enabled only when you select the Page Range option in the Page Range group box.
Note: The Page Range options are only available for selected report export options. For more information, see [Page Range](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/rp-report-export-options-form/field-definitions-rp-report-export-options-form#ID-0003d8df--en).

Enter the ending page number in a range of pages to export for this report.

## Column Width

The Column Width option on the RP Report Export Options form.
This option displays only when you select one of the Excel Format options.
Select a column width option for exporting to Microsoft Excel. The following options are available:

- Column width based on object in the - Select this option to base column widths on certain report sections. Use the drop-down to select the section that determines the column width.

- Constant column width - Select this option to set a standard column width. When you select this option, the system enables the points field.
