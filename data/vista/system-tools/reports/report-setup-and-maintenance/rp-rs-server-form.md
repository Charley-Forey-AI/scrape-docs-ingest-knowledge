# RP RS Server Form

This form displays the SSRS Report Servers that have been set up using the Viewpoint SSRS Reports installer.
These entries are created by the Viewpoint SSRS
 Reports installer, and they are associated with SSRS report locations using the
 RS Server
 Name field on the [RP Report Locations](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-report-locations-form) form.
Generally you should not have to make any changes to
 the records that display in this form. However, you may want to change the setting for
 the Sync
 Security checkbox, depending on whether you want to update SSRS reports
 when applying report security. For more information, see [Sync
 Security](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/rp-rs-server-form/field-definitions-rp-rs-server-form#ID-0003deae--en).
All of the servers set up in this form must point to the same Remote Helper and Vista database.
