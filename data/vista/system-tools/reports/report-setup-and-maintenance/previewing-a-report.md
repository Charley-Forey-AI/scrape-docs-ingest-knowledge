# Previewing a Report

You can preview a selected Crystal or SSRS report before printing or exporting.
Note: When launching an SSRS report, RP Report Launcher form displays only if Launch Parameters Form is selected in the SSRS Report Viewer Options drop down on the Info tab of the [VA User Profile](/en/vista/vista/administration/viewpoint-administration/setup-and-maintenance/user-profile-management/va-user-profile-form) form.

To preview a report:

1. Open the RP Report Launcher form by launching a report using a Work Center, the Reports folder in a module, or a form (Options > Reports).

1. Enter all applicable report parameters on the Input tab.

1. Click Preview.The report viewer displays. It may take a few minutes to load the information to the screen, depending on the size of the report and the ranges of information you have specified.
Note: If your deployment method is VRL Cloud, you may be presented with a screen indicating your position in the report queue. As other users close their report previews, your queue position updates and your report will begin processing automatically when your position reaches 0.

Once the preview screen displays, you can:

- refresh the report to include changed data

- move through the report pages or go to a specific page

- adjust the magnification of the report (zoom)

- search through report to locate specific data

- print or export the report.

If you are previewing a drilldown report, double-clicking on specific data on the current page accesses additional information by drilling down to another level. The levels of information available differ by report.
Note: When you first generate a report, the system saves your parameters so that if you refresh the report, the original parameters are still in effect. To set new parameters, close the Launcher and reopen.
