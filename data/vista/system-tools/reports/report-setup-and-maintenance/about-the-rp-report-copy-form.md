# About the RP Report Copy Form

Use this form to create a new custom report based on an existing Crystal or SQL Reporting Services report.
For example, you can copy a standard report and then make changes to the copy.
Open this form using the Programs folder in the
 Reports module or by clicking the Copy Report button on the [RP Report Titles](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-report-titles-form) form.
Note: If you have two SSRS Report Servers, you cannot use this form to copy a report from one server to another.Tasks
