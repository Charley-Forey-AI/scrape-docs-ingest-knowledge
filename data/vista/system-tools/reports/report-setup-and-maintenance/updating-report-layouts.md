# Updating Report Layouts

You can update report layouts to the RP Report Objects (RPRF)
 table for a single report or a range of reports.
When you update a report layout, the system updates
 the RPRF table with the appropriate formulas and database fields.
 The Report Information report uses this data to provide information
 about each report.
In order to keep the RPRF table populated with the correct data, you should follow these steps each time changes are made to a report's layout (in the application with which you created the report).

1. If desired, review any report layout using the Layout tab of the RP Report Titles form.

1. To
 update multiple report layouts:

1. Open the RP Report Layout form (Reports > Programs > RP
 Report Layout).

1. In the Beginning
 Report ID and Ending Report
 ID fields, specify the range of Report
 IDs to update to the RPRF file.

1. Click Update
 Layout.The
 update process begins and may take several
 minutes.

1. To
 update a single Crystal report layout:

1. Open the RP Report Titles form (Reports > Programs > RP
 Report Titles).

1. In the Report ID field,
 enter the desired report or press
 F4 to select from a list of
 reports.

1. Click Update
 Design.Note: This button does not apply to SQL Report
 Services reports.

The update process begins and may take
 several minutes.
