# About the RP Parameter Update Form

Use this form to update the report parameters, which display on the Parameters tab of RP Report Titles.
You can open this form by clicking the Update
 Parameters button on the Info tab of [RP Report
 Titles](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/about-the-rp-report-titles-form).
There are two options on this form:

- Get new parameters – Select this option to update
 only the new report parameters. For example you should use this option if
 you changed the report to include a new parameter but want all existing
 parameters to stay the same.

- Reset parameters – Select this option to delete all
 of the existing report parameters and then reload them from the report. For
 example select this option if you changed a report and want to change all of
 the existing parameters associated with it.
