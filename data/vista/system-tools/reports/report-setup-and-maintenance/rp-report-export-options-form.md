# RP Report Export Options Form

Use this form to export a Vista report to another format.
 Access this form by selecting the Export button on the RP Report Launcher form.
The format you select determines what additional fields display for entry. Once you enter the requested data, you can then select to do one following:

- Email - Opens the Send Email form, allowing you to specify the standard email information (To, CC, BCC, Subject, and Message). When you select Send, the system sends the email to the designated recipient(s) and attaches the report data in a file of the specified format.

- Open in Application - Opens the export file in the application associated with the selected export format. For example, if you select to export to a CSV format, the system opens the export file in an Excel spreadsheet.

- Save - Opens the Save Report window, allowing you to save the export file to a selected location.

- Attach to Form - This option is only available when you open the report from within a Vista form. If selected, the system automatically attaches the report, in the specified export format, to the active record on the form.

For more information about exporting reports, see [Exporting a Report](/en/vista/vista/system-tools/reports/report-setup-and-maintenance/exporting-a-report).
