# Exporting a Report

You can export any report in the system.
You can select the export type and add additional export parameters. Export parameters will differ based on the export type you select. Exporting a report also allows you to save the report to a specific format (e.g., Text, Excel, PDF, HTML, CSV, etc.).
Once you have finished configuring the export options, you can send the report via Email, open it in the associated application (e.g. PDF exports will open in Adobe Acrobat Reader), or save it to a file. If you opened the report from within the related form, you can attach the report to the selected record.
To export a report from the system:

1. Open the RP Report
 Launcher form by launching a report using a Work Center, the Reports folder in a
 module, or a form (Options > Reports).

1. Enter the report
 parameters for the report.

1. Click the Export button on the RP Report Launcher form. The RP Report
 Export Options form opens.

1. Select an export type
 from the Export Type drop-down. The system displays different export
 option fields, depending on the export type that you select.

1. Configure the export options fields, as applicable. See the F1 help for each field for more information.

1. Export the report using one of the following options:

1. Attach to Form - Select this option to attach the exported report to the current form. The system displays a confirmation dialog once it attaches the report. Note: The Attach to Form button is only available if you are running the report directly from the form. This is not available if you are running the report from a Programs folder.

1. Email - Select this option to email the exported report. The MS Outlook client displays a new message with the exported report attached. If you are not using MS Outlook, the Viewpoint email form displays with the exported report attached.

1. Open in Application - Select this option to open the report in the application associated with the export type (e.g. PDF export will open in Adobe Acrobat Reader). The associated application displays the exported report. Note: You must have the associated default application on your system to open the exported report.

1. Save - Use this option to save the exported report to your C: drive or other location. The system opens a browse window where you can select and save the exported report to the proper location.

1. When finished, click the Cancel button to close the RP Report Export Options form.
