# Additional Lookups

The Add’l Lookups tab in RP Report Parameters is used to assign additional lookups (standard or custom) to report inputs.
Note: Custom lookups are those created using the UD module.
 When multiple lookups are assigned to an input, each lookup appears as a radio button option in the lookup window; only one option may be selected at a time. The label that displays for each option is the title of the lookup. The lookup (including the datatype lookup) with the lowest load sequence number is the default lookup.
The grid on this tab is used to assign a lookup (or multiple lookups) to the selected input. The F4 at “Lookup” provides a list of all available lookups. Once a lookup is selected, the grid populates with the lookup name, status (Standard or Custom), and lookup query parameters. These fields cannot be edited.
The Active option allows you to toggle a lookup option on or off. All lookups, when added to the grid, automatically default to Active (checked). If the box is unchecked, the lookup does not activate when F4 is pressed at the report input. If multiple lookups are assigned, those that are marked as ‘inactive’ do not appear as radio button options in the lookup window.
