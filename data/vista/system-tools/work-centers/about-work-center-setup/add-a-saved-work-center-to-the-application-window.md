# Add a Saved Work Center to the Application Window

You can add saved work centers to your Vista application window to reduce setup time.
Pre-requisites: Work Center must exist in VA Work
 Center Library and be shared. For more information, see [Save a Work
 Center](/en/vista/vista/system-tools/work-centers/about-work-center-setup/save-a-work-center).

1. From the main
 application window, select Options > Work
 Centers.

1. Click Load.The Load From Work Center Library form
 displays.

1. Highlight a work center in the list and click Load.The Work Centers form displays, showing the
 work center you just added.

1. (Optional) If you wish to rename the work center or change the refresh interval, click Edit.The Work Center - Settings form displays. You can then rename the work center or change the refresh interval as needed.

1. (Optional) Use the
 Move
 Up/Move Down buttons to
 reposition the work center in the order you wish it to appear in the application
 window.

1. Click OK.The work center appears in your
 application window.
