# WF Notifier Viewpoint Tables

Use the WF Notifier Viewpoint Tables form to review all table/views associated with
 each module in the system.
Access this form by selecting Options > Viewpoint Views from WF Notifier Queries.
This form is for informational use only and does not perform any additional functions.
To see the tables/views for a module, select the module from the
 Module drop-down and then select the view from the
 View drop-down. The Columns text box
 displays all columns in the selected view. If any indexed columns exist in the view, the
 Indexed Columns text box indicates which columns are indexed.
