# Copying a Notifier Query

You cannot modify a standard Notifier query, but you
 can copy a standard query and then modify the copy - for example if you want to modify a
 standard HR module Notifier query to only include active employees.
Follow the steps below to copy a Notifier query.

1. Open the [WF Notifier Queries ](/en/vista/vista/system-tools/work-flow/about-automated-notifications/wf-notifier-queries) form.

1. Open the query that you want
 to copy.

1. Select File > Copy
 Query from the menu at the top of the form. This will open the Copy Query
 form.

1. Enter the name of the new
 query in the New Query Name field.

1. Click the Copy
 button.

1. Click Close.
 The system copies the query.

1. Once you have copied the query, you can modify it using the [WF Notifier Queries ](/en/vista/vista/system-tools/work-flow/about-automated-notifications/wf-notifier-queries) form, and then associate it with a Notifier Job using the [WF Notifier Job Manager ](/en/vista/vista/system-tools/work-flow/about-automated-notifications/wf-notifier-job-manager) form.
