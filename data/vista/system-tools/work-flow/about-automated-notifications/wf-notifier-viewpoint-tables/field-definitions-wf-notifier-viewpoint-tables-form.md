# Field Definitions: WF Notifier Viewpoint Tables Form

The following is a list of field descriptions for the WF
 Notifier Viewpoint Tables form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Module

 Select the module whose views you wish to review.

##  View

 Select the view you wish to review. Once you select a view, all columns for the view display in the Columns text box. If a column has an index, it displays in the Indexed Columns text box.
