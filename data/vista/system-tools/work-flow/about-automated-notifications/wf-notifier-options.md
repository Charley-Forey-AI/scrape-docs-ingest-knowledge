# WF Notifier Options

You can use the WF Notifier Options form to configure the From address for notifications.
You can access this form by selecting Options > Notifier Options from WF Notifier Job Manager.
Follow the steps below to configure the From address for notifications.

1. Enter an email address in the Notifier Address field. The address specified in this field will be used as the 'From' address for Notifier emails.Note: If this field is blank, the system uses the email address specified in the Email Address field on the VA Site Settings form, Email Settings tab. If an address is not available in VA Site Settings, the system uses yourname@yourdomain.com as the address.

1. Click OK. The form closes and you are returned to WF Notifier Job Manager.
