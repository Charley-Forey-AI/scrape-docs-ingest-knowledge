# Add Steps to Checklist Template Tasks

After adding a task to a checklist template in WF Checklist
 Template, you may need to break it up into steps.

1. Select the Steps tab.


1. Enter N
 in the Step field to create a new step.

1. Enter a brief
 description of the step in the Summary field.

1. Select the Required checkbox if someone must finish the step to complete
 the task.

1. Select the Use
 Notification checkbox to have a notification sent to the
 assignee when the step is ready.

1. Enter a more
 comprehensive description of the step in the Description field.

1. Select a step type from
 the Step
 Type drop-down field: Manual, Form, or Report.

1. If you selected Form or
 Report as the step type, specify the form or report in the Form/Report field, or press F4
 to select from a list of forms or reports.

1. Add additional steps as
 necessary.
