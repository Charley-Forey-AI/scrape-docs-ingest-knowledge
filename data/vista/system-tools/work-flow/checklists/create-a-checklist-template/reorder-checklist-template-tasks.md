# Reorder Checklist Template Tasks

After completing a checklist template in the WF Checklist
 Template form, you may need to modify the workflow and reorder the tasks.

1. Select File > Reorder Tasks. The Reorder Tasks window displays. All tasks associated with the
 checklist template display in the Tasks text box.

1. Select a task to move
 and click either the Up or Down button. You can also drag tasks for reordering.
 The task moves up or down one position in the list.

1. Rearrange tasks as
 necessary and click OK. The system displays a
 confirmation message.

1. Click Close.
