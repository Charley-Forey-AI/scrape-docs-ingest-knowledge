# Add Tasks to a Checklist Template

Once you create a checklist template using the WF Checklist
 Template form, you can add tasks to the checklist.

1. Create a checklist
 template. For more information, see [Creating a Checklist
 Template](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist-template) above.

1. Enter N
 in the Task field to create a new task record.

1. Enter a brief
 description of the task in the Summary field.

1. Enter a more
 comprehensive description of the task in the Description field.

1. Select the Required checkbox if someone must finish the task to complete
 the checklist.

1. Select the Use
 Notification checkbox to have a notification sent to the
 assignee when the task is ready.

1. Select a Task
 Type : Manual, Form, or Report.

1. If you selected Form or
 Report as the task type, specify the form or report in the Form/Report
 Name field, or press F4 to select from a list of
 forms or reports. Note: If you specify the form or report name,
 users can launch the form directly from the WF User Tasks and WF Checklist
 Tasks forms.

1. Add additional tasks and
 steps, as necessary. For more information on adding steps to a task, refer to
 [Adding Steps to Checklist Template Tasks](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist-template/add-steps-to-checklist-template-tasks) below.
