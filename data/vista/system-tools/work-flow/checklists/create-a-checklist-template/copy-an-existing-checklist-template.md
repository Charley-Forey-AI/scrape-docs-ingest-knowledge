# Copy an Existing Checklist Template

You can use the WF Checklist Template Copy form to make a
 copy of an existing checklist template. Once you copy a template, you can make any necessary
 changes to the new template.

1. Find the template to
 copy in WF Checklist Template.

1. Select File > Copy
 Template. The WF Checklist Template Copy form displays.

1. Enter a name for the new
 template in the New Template Name
 field.

1. Click Copy. The WF Checklist Template copy form closes and the system
 creates a copy of the original template with the new name.

1. Use WF Checklist
 Template to modify the new template as necessary.
