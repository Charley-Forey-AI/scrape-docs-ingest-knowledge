# Reorder Checklist Template Task Steps

After completing a checklist template in the WF Checklist
 Template form, you may need to modify the workflow and reorder task steps.

1. Select the Steps tab.


1. Select File > Reorder Steps. The Reorder Steps window displays. All steps associated with the
 task display in the Steps text box.

1. Select a step to move
 and click either the Up or Down button. The step moves up or down one position
 in the list. Note: You can also click and drag steps for
 reordering.

1. Rearrange steps as
 necessary and click OK. The system displays a
 confirmation message.

1. Click
 Close.

The system renumbers the steps
 appropriately.
