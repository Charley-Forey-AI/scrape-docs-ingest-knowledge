# Field Definitions: WF Checklist Template Form

The following is a list of field descriptions for the WF
 Checklist Template form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Template

Enter a name for the checklist template, up to 60 characters.

##  Description

 Enter a description for the checklist template, up to 2,000 characters.

##  Revised

 Enter a revision date for this checklist. This field initially defaults the current date. If you make a change to the checklist template, this field does not update, but the Changed By and Date display fields do. You might use this field to keep track of the revision/version number of the template.

## Enforce Tasks/Steps in Order

Select this checkbox to have the system enforce the checklist order. When you select this option, users cannot perform tasks and steps for this checklist out of order. You can override this setting when creating checklists based on this template. For more information on creating a checklist based on a template, refer to Related Topics below.

Related information

- [WF Checklist Initialization](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-initialization)

## Enable Notification

Select this checkbox to have the system send messages to assignees when a task or step is ready. You can override this setting for individual tasks and steps when creating checklists based on this template. If you do not select this checkbox, the system does not send notifications, even if notifications are set at the task and step level.
For more information on creating a checklist based on a template, refer to Related Topics below.

Related information

- [WF Checklist Initialization](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-initialization)

## Active

Select this checkbox to activate the checklist template. When active, you can create checklists from this template using WF Checklist Initialization. If For more information, refer to Related Topics below. If you do not select this checkbox, this template does not display on the WF Checklist Initialization form.

Related information

- [WF Checklist Initialization](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-initialization)

## Task

Enter N, New, or + to create a new task item for the
 checklist template. To view an existing task, enter the task number or press F4 to see a
 list of tasks.

## Summary

Enter a brief description of the task item, up to 256 characters.

## Required

Select this checkbox if someone must finish this task to complete the checklist. You can change this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

## Use Notification

Select this checkbox to have the system send messages to assignees when a task is ready. You can override this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

##  Description

 Enter a description for the task, up to 2,000 characters.

## Task Type

Select the type for this task.

- Manual – This indicates the task occurs outside of the software.

- Form – This indicates the task involves a form in the software.

- Report – This indicates the task involves a report in the software.

You can override this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

## Form/Report Name

This field is accessible when you select
 either Form or Report from the Task Type field. Enter the form or report
 that applies to the task or press F4 for a list.
Note: If you enter a form or report here, users can launch the form/report directly from the WF Checklist Tasks and WF User Tasks forms.
You can override this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

## Step

Enter N, New, or + to create a new step for the task item.

## Summary

Enter a brief description of the step, up to 256 characters.

## Required

Select this checkbox if someone must finish this step to complete the task. You can change this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

## Use Notification

Select this checkbox to have the system send messages to assignees when a step is ready. You can override this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

##  Description

 Enter a description for the step, up to 2000 characters.

## Step Type

Select the type for this step.

- Manual – This indicates the step occurs outside of the software.

- Form – This indicates the step involves a form in the software.

- Report – This indicates the step involves a report in the software.

You can override this setting for an individual checklist using WF Checklist Maintenance.

Related information

- [WF Checklist Maintenance](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-maintenance)

## Form/Report

This field is accessible when you select either Form or Report as the step type. Enter the form or report that applies to the step or press F4 for a list.
You can override this setting for an individual checklist using WF Checklist Maintenance.
