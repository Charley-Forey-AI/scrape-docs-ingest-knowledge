# Create a Checklist Template

You can use the WF Checklist Template form to create
 checklist templates. Templates are useful when you require a number of similar checklists,
 each with minor differences.
For example, you might create an AP processing
 checklist and create a number of instances based on the template (e.g., June AP
 Processing, July AP Processing, etc.). Then you can use [WF Checklist Initialization](/en/vista/vista/system-tools/work-flow/checklists/wf-checklist-initialization) to create specific
 checklists based on a template.

1. Enter a name for the
 template in the Template field.

1. Enter an optional
 description in the Description field.

1. Select the Enforce
 Tasks/Steps in Order checkbox to ensure that users can only
 complete checklist tasks and steps in order.

1. Select the Enable
 Notification checkbox to enable the system to send
 notifications to task/step assignees. Note: If you do not select this checkbox, the
 system does not send notifications, even if notifications are set at the
 task and step level.

1. Select the Active checkbox to allow users to make copies of this
 template. Note: If you do not select this option, this
 template is not available on the WF Checklist Initialization form.

1. Add steps and tasks to
 the template. For more information, see [Adding Tasks to a
 Checklist Template](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist-template/add-tasks-to-a-checklist-template) and [Adding Steps to
 Checklist Template Tasks](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist-template/add-steps-to-checklist-template-tasks) below.

1. Save the template as
 normal.
