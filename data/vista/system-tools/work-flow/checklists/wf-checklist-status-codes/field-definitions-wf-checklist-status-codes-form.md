# Field Definitions: WF Checklist Status Codes Form

The following is a list of field descriptions for the WF
 Checklist Status Codes form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Status Code

 Enter an ID number to create a new status code. Press F4 to see a list of existing status codes.

##  Description

 Enter a description for the status code, up to 30 characters. This description displays whenever the code is used.

## Status Type

Select a type for this status code.

- New

- In Progress

- Final

## Default

Select this checkbox if this status code is the default for the specified status type. Only one status code can be the default for each status type.
Note: If another code is set as the default for this status type, you must first deselect the Default checkbox for that code before setting the current code as the default.
