# Create a Checklist from a Template

You can create a new workflow checklist based on a template
 or an existing checklist. To create the checklist you can use either the WF Checklist
 Template or WF Checklist Maintenance form. You can access this form from the WF Programs
 folder, or by selecting File > Initialize
 Checklist from WF Checklist Template or WF Checklist Maintenance.
When you first access WF Checklist Template or WF
 Checklist Maintenance, active templates and any checklists with a status of ‘New’ or ‘In
 Progress’ display automatically. From here, you can perform a filter to help identify
 the checklist or template to copy and then initialize a new checklist.

1. Enter the company in the
 Company field.

1. Filter for the template you want to use. You can
 use the following methods to filter:OptionDescription
To filter by a checklist status code:Select any combination of the New, In
 Progress, and Final check
 boxes.
To filter by the checklist creation date:Select the Created radio button
 and specify a date range in the Beginning Date and
 Ending Date
 fields.
To filter by the checklist completion date:Select the Completed radio
 button and specify a date range in the Beginning Date and Ending Date
 fields.

1. Select the Init checkbox for each checklist or template to initialize
 from.

1. Enter a name for the new
 checklist on the New Name field.

1. Enter a new initial
 status for the checklist in the Init Status field. This field
 defaults to New status.

1. Enter a destination
 company in the Dest Co field. This field
 defaults to the current company.

1. Select the Set
 Private checkbox if you want to keep the new checklist from
 displaying for other users.

1. Select the Send
 Notification checkbox to enable automatic notifications for the
 checklist.

1. Select the Enforce
 Order checkbox if users must complete all checklist tasks and
 steps in order.

1. To assign a user to the
 checklist, enter the user name in the Assign To field.

1. Click Initialize. A validation message displays.

1. Click Close. The system creates new checklists based on selected
 templates or pre-existing checklists.
