# Add Steps to Checklist Tasks

After adding a task to a checklist in the WF Checklist
 Maintenance form, you may need to break it up into steps.

1. Select the Steps tab.


1. Enter N
 in the Step field to create a new step.

1. Enter a brief
 description of the step in the Summary field.

1. Select the Required checkbox if someone must finish the step to complete
 the task.

1. Select the Use
 Notification checkbox to have the system send messages to
 assignees when steps are ready.

1. Enter a status in the
 Status field, or press F4 to select from a list of
 valid status codes.

1. Enter a more
 comprehensive description of the step in the Description field.

1. Select a step type from
 the Step
 Type drop-down field: Manual, Form, or Report.

1. If you selected
 Form or Report as the Step
 Type, specify the form or report in the Form/Report field. Press F4 for a list of forms or
 reports.

1. Enter the assigned user
 in the Assigned To field, or press F4
 to select from a list of users.

1. Add additional steps as
 necessary.
