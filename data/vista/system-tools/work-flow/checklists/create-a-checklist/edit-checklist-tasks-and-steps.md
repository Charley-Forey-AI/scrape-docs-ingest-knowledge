# Edit Checklist Tasks and Steps

You can use the WF Checklist Tasks form to edit workflow
 checklist tasks and steps.
While you cannot reorder tasks or their associated
 steps, you can change all other settings.

1.  Filter for the
 appropriate checklist and its associated tasks. For information, see [Viewing Checklist
 Tasks](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist/view-checklist-tasks).

1. Change details for all
 appropriate tasks or steps and save the record. The system updates task lists
 for all assigned users (in WF Tasklist). For more information on individual
 fields, refer to the F1 help.
