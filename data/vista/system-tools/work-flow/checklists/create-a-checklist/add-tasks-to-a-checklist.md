# Add Tasks to a Checklist

You can use the WF Checklist Maintenance form to add tasks to
 workflow checklists.

1. Create a checklist. For
 more information, refer to [Creating a
 Checklist](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist) above.

1. Enter N
 in the Task field to create a new task record.

1. Enter a brief
 description of the task in the Summary field.

1. Select the Required checkbox if someone must finish the task to complete
 the checklist.

1. Select the Use
 Notification checkbox to have the system send messages to
 assignees when tasks are ready.

1. Enter a task status in
 the Status field, or press F4 to select from a list of
 available statuses.

1. Enter a more
 comprehensive description of the task in the Description field.

1. Select a task type:
 Manual, Form, or Report.

1. If you selected Form or
 Report as the task type, specify the form or report in the Form/Report
 Name field. Press F4 to select from a list of
 forms or reports.

1. Enter the user you are
 assigning the task to in the Assigned To field. The
 Assigned Date field defaults the current date.

1. Enter the date due in
 the Due
 Date field, if needed.

1. Add additional tasks and
 steps, as necessary. For more information on adding steps to a task, see [Adding Steps to Checklist Tasks](/en/vista/vista/system-tools/work-flow/checklists/create-a-checklist/add-steps-to-checklist-tasks).
