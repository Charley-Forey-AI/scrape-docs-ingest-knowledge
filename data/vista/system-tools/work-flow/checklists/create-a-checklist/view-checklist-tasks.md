# View Checklist Tasks

You can use the WF Checklist Tasks form to monitor checklist task progress.

1. Enter the checklist name in the Checklist field. Press F4 for a list of available checklists.

1. Select the statuses for filtering in the Checklist Item Status Type section. Note: The system automatically selects the New and In Progress checkboxes. If you clear all checkboxes in the Checklist Item Status section, the system reselects the New and In Progress checkboxes for the search when you click Refresh.

1. Select a date type to search by in the Search By Date field: None, Due, or Assigned.

1. If you selected a date type of Due or Assigned in step 3, enter a date range in the Beginning Date and Ending Date fields.

1. Click Refresh.

The system displays all tasks for the checklist that meet the filtering criteria.
