# Copy an Existing Workflow Process

You can create a copy of an existing workflow process using the WF Process Copy form.

1. Open the WF Workflow Process form.

1. Select the workflow process that you would like to copy.

1. From the toolbar, select Tasks > Copy. The WF Process Copy form displays, showing the source process and document type in the Copy From section.

1. In the Copy To section, use the Process field to enter the name of the new process.

1. In the Description field, enter a description of the new process.

1. From the Type drop down, select the type of document
to which the new process applies (PO or SL).Leave the drop-down blank if the new process is a generic process; that is, it is not restricted to a specific document type.

1. If you want to copy the approval steps to the new process, select the Approval
 Steps checkbox. If you do not select this checkbox, no approval steps will be copied to the new process.

1. Select Copy to create the new workflow process. A message displays indicating the copy was successful.

1. Close the WF Process Copy form.You are returned to the WF Workflow Process form.

1. Review the new process and edit as needed.
