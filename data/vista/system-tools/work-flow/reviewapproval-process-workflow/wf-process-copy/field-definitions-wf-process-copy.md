# Field Definitions: WF Process Copy

The following is a list of field descriptions for the WF
 Process Copy form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Copy From: Process

The Copy From: Process field on the WF Copy Process form.
This field is always disabled.
Display only, the process you are copying.
This field defaults the process selected in WF Workflow Process. If you want to copy a different approval process, return to the WF Workflow Process form, select the desired process, and then select Tasks > Copy.

Related information

- [Review/Approval Process Workflow](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow)

## Copy From: Type

The Copy From: Type field on the WF Process Copy form.
This field is always disabled.
Display only, the document type of the approval process being copied.
You can change the document type for the new approval/review process using the Type field in the Copy To section of this form.

## Copy To: Process

The Copy To: Process field on the WF Process Copy form.
Enter the name of the new process, up to 20 characters.

Related information

- [WF Process Copy](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow/wf-process-copy)

- [Review/Approval Process Workflow](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow)

## Copy To: Description

The Copy To: Description field on the WF Process Copy form.
Enter a description of the new approval process; up to 60 characters allowed.
By default this field populates with the description of the workflow process you are copying.

Related information

- [WF Process Copy](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow/wf-process-copy)

- [Review/Approval Process Workflow](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow)

## Copy To: Type

The Copy To: Type field on the WF Process Copy form.
From the drop-down, select the document type for the new process.

- PO - Purchase Order

- SL - Subcontracts

You can leave this field blank if the workflow process applies to all document types.

Related information

- [WF Process Copy](/en/vista/vista/system-tools/work-flow/reviewapproval-process-workflow/wf-process-copy)

## Approval Steps

The Approval Steps checkbox on the WF Process Copy form.
This checkbox defaults as selected.
Leave this checkbox selected to copy the source approval steps to the new process.
Deselect this checkbox if you are only copying the information on the info tab (in WF Workflow Process) and do not want to include the source approval steps. You can manually add approval steps once the copy process is complete.
