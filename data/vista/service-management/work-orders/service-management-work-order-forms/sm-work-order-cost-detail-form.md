# SM Work Order Cost Detail Form

Use the SM Work Order Cost Detail form to set up summarized cost estimates for a work order.
Access this form by clicking Details on the Info tab of the work order scopes section of SM Work Orders.
The labor, material, equipment, subcontract, and/or other cost estimates you set up here will be used to calculate the total estimated cost for the work order quote scope.
 For scopes with a Time and Materials pricing method or scopes on a job work order using the Markup costing method, you can also enter markup rates for your cost estimates to derive pricing estimates. The system will use the cost estimates and pricing estimates to determine the estimated profit for the work order scope.
Note: If you prefer to enter cost estimates at a more detailed level, use the Material, Equipment, and Labor tabs on the work order scope (detailed estimates are not available for miscellaneous expenses). You can use the detailed cost estimates to derive the Cost Total for the quote by selecting the Derived checkbox. If you have already entered summarized cost estimates, they will be left intact. For more information about cost estimate detail, see [Setting up Required Resources for a Work Order](/en/vista/vista/service-management/work-orders/about-work-order-scope-budgets/required-resources-for-work-orders).
