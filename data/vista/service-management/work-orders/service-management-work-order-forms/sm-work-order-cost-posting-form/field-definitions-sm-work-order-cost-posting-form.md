# Field Definitions: SM Work Order Cost Posting Form

The following is a list of field descriptions for the SM Work
 Order Cost Posting form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Process

Select this checkbox to include this work
 completed line when creating and posting a cost posting batch or click Check All to
 auto-select this checkbox for all work completed lines in the grid.
Leave this checkbox unselected if you do not want to include this work completed line when creating and posting a cost posting batch.

## Post Month

Post Month field on the SM Work Order Cost Posting form.
Use this field to change the post month. When you click the Change Post Month button, the system will update the post month for selected lines.
Note:The selected post month cannot be closed. Months are closed using the [GL Month End Close ](/en/vista/vista/accounting/general-ledger/gl-setup-and-maintenance/about-the-gl-month-end-close-form) form.
