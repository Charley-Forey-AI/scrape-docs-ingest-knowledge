# Work Order Summary Info

The Summary Info pane in SM Work Orders provides you with critical work order summary dashboard information so that you do not have to manually locate information in order to release a work order for billing.
Cost values are shown in exact values and are based on work completed entries. The Cost total shows the sum of actual and/or projected costs for work completed; if project costs are included in the total, an asterisk displays to the right. Tax and Margin values are shown with a link that you can select to toggle between dollar amount and percentage views. The Labor Hours section shows Budgeted hours vs. the Actual hours posted to work completed labor lines. Budgeted hours will be the combined total of hours posted on the Labor tab for scopes with the Derived checkbox selected and the hours posted in SM Work Order Cost Detail for scopes with the Derived checkbox unselected.

## Summary Notes

The Notes section provides access to notes entered for the customer or job, service site, and/or agreement (customer work orders only) associated with a work order. If the work order was created from a quote, notes entered for the quote header and scopes will display as Work order notes, along with any notes manually entered for the work order.
If notes exist for a listed item, the item appears as a link. Hovering over the link displays a pop-up window showing the notes; selecting the link opens a related notes form. For example, if Customer notes exist and you select the Customer link, the Customer Notes window displays. If agreement notes exist, the pop-up window shows agreement notes, term notes, and/or revision notes, each in their own section for easier viewing.
Notes are display only; any edits or additions to the notes must be handled in the source form. For work orders generated from a quote, you can edit notes using the Notes tab of the work order header and work order scope. Once you save the notes, selecting Refresh updates the notes so they are available to view from the work order.
