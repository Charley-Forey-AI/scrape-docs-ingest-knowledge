# About Work Orders Generated from Quotes

If you are using the Work Order Quotes feature, the system automatically generates an SM work order once you approve the work order quote.
When the work order is generated, the system creates a work order scope for each quote scope, and automatically sets up any tasks for the work order scope that are defined for the quote scope. You can edit most information on the work order as needed; however, some information, such as the service site/customer, price method and budget details, cannot be edited.
Work completed lines entered for these work orders will derive all billable rates/amounts from the override rates or rate template defined for the quote only; no other override rates/rate templates (i.e. service site or customer level) will be used.
For more information about work order quotes, see [SM Work Order Quotes Form](/en/vista/vista/service-management/work-order-quotes/work-order-quotes-forms/sm-work-order-quotes-form).
