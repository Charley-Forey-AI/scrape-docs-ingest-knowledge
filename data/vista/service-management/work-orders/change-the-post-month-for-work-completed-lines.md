# Change the Post Month for Work Completed Lines

You can change the post month for work completed lines in SM Work
 Order Cost Posting.

1. Open the SM Work Orders Cost Posting form.

1. For each work completed
 line you want to update, select the Process checkbox. If you want to select the Process checkbox for all lines, click Check All.Note: You can use the filter bar to restrict
 the records shown in the grid for easier selection. See [SM Work Order Cost Posting Form](/en/vista/vista/service-management/work-orders/service-management-work-order-forms/sm-work-order-cost-posting-form) for more
 information.

1. In the Post
 Month field, enter the new month and year (MM/YY) to which the
 selected work completed lines will be posted.

1. Click Change Post
 Month.

 The system updates the post month to the specified month and
 year in all selected lines.
