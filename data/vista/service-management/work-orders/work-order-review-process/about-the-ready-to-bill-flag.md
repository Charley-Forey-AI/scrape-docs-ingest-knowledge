# About the Ready To Bill Flag

You can mark SM work orders and/or work order scopes as ready-to-bill so that you can invoice the work order.

In order for you to select the Ready To Bill checkbox for a work order/scope, your login must be associated with a reviewer on the work order. If you are not a reviewer for the work order, the Ready To Bill checkbox does not display.
Reviewers may be authorized at the followings locations:

- SM Customers

- SM Service Centers

- SM Service Sites

- SM Work Orders

- SM Divisions

The Ready To Bill checkboxes appear in the following locations:

- Info tab – if selected, marks all work order scopes and their related work completed lines as ready-to-bill.

- Scopes – if selected, marks individual work order scopes and their related work completed lines as ready-to-bill. However, this checkbox only displays for Flat Price and Time & Material work order scopes.

If you select the Ready To Bill checkbox for a Flat Price scope, you are presented with a Partial Bill option, which allows you to specify what amount or percentage of the flat price you want to bill.
You can also mark individual work complete lines as ready-to-bill. If you mark all work completed lines as ready-to-bill for a work order scope, the work order scope's Ready to Bill checkbox displays in an "indeterminate" state (meaning that it is neither selected nor unselected).
If the Use Review Process checkbox in SM Company Parameters is selected and you are authorized as a reviewer on any level, when you select the Bill WO button on a work order, the system ask you whether you want to "Bill all work or only work marked Ready to Bill?"

- Select All to bill all Work Completed items

- Select Ready to Bill to bill only individual items marked as ready to bill
