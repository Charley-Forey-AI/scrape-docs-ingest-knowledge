# Create & Post a Batch for Provisional Work Completed Lines

If you have provisional work completed lines on a work order, you must post them using the SM Work Order Cost Posting
 form.

1. Open the SM Work Order Posting form.

1. For each work completed
 line you want to post, select the Process checkbox. To select
 the Process checkbox for all work completed lines in the grid,
 click Check All.Note: You can use the filter bar to restrict
 the records shown in the grid for easier selection. See [SM Work Order Cost Posting Form](/en/vista/vista/service-management/work-orders/service-management-work-order-forms/sm-work-order-cost-posting-form) for more
 information.

1. Click
 Process.

 The system creates and posts a single
 batch for all selected work completed lines.
