# Work Order Scope Cost Summary

You can view a summary of costs for a work order scope using the Summary tab in SM Work Orders.

The Summary tab (display only) provides a summary of costs, by cost type category and cost type, for the selected work order scope (based on work completed lines posted to the work order scope).
The following table describes the values in each grid column. Totals shown will be the sum of all work completed lines posted to the same cost type category/cost type.
Column NameDescription
Cost Type CategoryCost type category (L-Labor, E-Equipment, M-Material, O-Other, S-Subcontracts)
Cost TypeSM Cost Type assigned to work completed lines. The system combines lines with the same SM Cost Type
DescriptionDescription of the SM cost type
Is TaxableIndicates whether the cost type is taxable (checkbox in SM Cost Types)
Proj CostTotal projected cost for the cost type.
Actual CostTotal actual cost, including tax (for lines that include tax).
Act Pct of Fixed PriceTotal actual cost (including tax) divided by the scope's fixed price (flat prices scopes only). For Time and Material and Non-Billable scopes, this value will be 0.00%.
Act + Pct of Fixed PriceTotal remaining committed cost + total actual cost (including tax) divided by the scope's fixed price (flat price scopes only). For Time and Material and Non-Billable scopes, this value will be 0.00%.
Remain Proj CostTotal projected cost minus total actual cost (including tax)
Actual Tax AmtTotal actual cost tax
Proj Tax AmtTotal projected tax amount
Proj Tax BasisTotal projected tax basis
