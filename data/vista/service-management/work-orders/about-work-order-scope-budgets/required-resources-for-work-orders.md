# Required Resources for
 Work Orders

You can set up requirements for materials, labor, and equipment to capture estimated expenses for a work order.
Requirements are set up at the work order scope level using the Material, Labor, and Equipment tabs in SM Work Orders.
If you have set up tasks for the work order scope (Tasks tab) and the task references a standard task, any material, labor, equipment, or miscellaneous requirements defined for the task (in SM Standard Tasks) will be set up automatically on the corresponding tab for the scope. Likewise, removing a task that references a standard task will automatically remove all material, labor, equipment, or miscellaneous requirements associated with the standard task.
Note: Work orders generated from agreement services and work order quotes will be set up automatically with the resource requirements defined for the agreement service or quote sequence (respectively). If miscellaneous requirements were defined for an agreement service or quote sequence, they will be set up automatically as work completed miscellaneous lines (with a Provisional status) on the work order.
