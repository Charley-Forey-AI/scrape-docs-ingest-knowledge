# Edit Agreement/Service Pricing for Multiple Renewals At Once

You can update the pricing for multiple service agreements and agreement services using a markup percent.

1. Open the SM Agreement Renewal form (Service ManagementProgramsSM Agreement Renewal).

1. In the
 Grid
 Shown
 field, select Renewals.

1. Use the Type , Customer, and/or Only Show
 Agreements with Auto Renew fields to filter the renewals shown
 in the grid. As you tab to each field, the grid refreshes based on the values
 entered.For more information about each
 field, refer to the F1 help.

1. Click Refresh..The grid refreshes to show all agreement renewals
 meeting the selected criteria

1. Select the agreements to
 which you are applying the price change using one of the following
 methods.

- Select the Included checkbox in the
 grid for individual agreements.

- Select multiple agreements in the grid using the Shift
 or Ctrl key and then click the Include Selected button.

- Click the Include All button to select all
 agreements.Note: If you have agreements that are flagged for auto
 renewal and do not want to override the renewal markups defined for them
 in SM Agreements, make sure you do not select those agreements.

1. In the Markup Percent
 to Apply field, enter the percent by which to mark up agreement
 pricing.

1. Click Apply to
 Included.

 The system applies the specified markup percent to all selected
 agreements, as well as to their associated time-of service and periodic
 agreement services.
