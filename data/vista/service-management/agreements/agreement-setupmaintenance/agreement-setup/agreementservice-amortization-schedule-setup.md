# Agreement/Service Amortization Schedule
 Setup

If you elected to use the Revenue Recognition feature for an agreement, you must set up
 amortization schedules for the agreement and for any periodic service being billed
 separately.
You can set up amortization schedules manually or set them up
 automatically using the SM Automatic Scheduling form. This is useful when you are
 spreading revenue out equally over a period of time.
Click on the links below for information about setting up
 amortization schedules for agreements and agreement services.
