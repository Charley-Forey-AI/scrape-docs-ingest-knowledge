# Agreement/Service Billing
 Schedule Setup

All agreements require that you set up a billing schedule before you activate the agreement.
If you have added periodic services to the agreement
 that are flagged for separate billing, you must set up billing schedules for
 them as well.
 You can have the system set up agreement and service
 billing schedules automatically using the [SM Automatic Scheduling](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-automatic-scheduling-form) form. This is useful when you have
 multiple payments that are typically scheduled for the same day each billing
 cycle. If you prefer, you can also set up billing schedules manually in
 [SM Agreement Billing Schedule](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-agreement-billing-schedule-form) (agreements) or [SM Work Schedule Billings](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-work-schedule-billings-form) (agreement services). You might
 use this method when you only need to schedule one or two payments.
