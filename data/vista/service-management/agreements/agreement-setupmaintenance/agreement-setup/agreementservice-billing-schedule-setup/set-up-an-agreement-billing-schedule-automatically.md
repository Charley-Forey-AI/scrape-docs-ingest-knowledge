# Set Up an Agreement Billing Schedule
 Automatically

Set up an agreement billing schedule automatically.
Note: These steps assume you have already [set up the agreement quote](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/agreement-setup/set-up-an-agreement-in-sm-agreements). Also, if you set the
 agreement price to 0.00, you can only use automatic scheduling if you have set up
 periodic services that are billed with the agreement and have a price greater than
 0.00.

You will use the [SM
 Automatic Scheduling](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-automatic-scheduling-form) form to set up billing schedules automatically for
 agreements and period services that are flagged for separate billing.For information about manually setting
 up agreement and agreement service billing schedules, see [Setting up Agreement/Service Billing
 Schedules](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/agreement-setup/agreementservice-billing-schedule-setup).
To set up an
 agreement billing schedule automatically:

1. From the Vista main
 menu, select Service Management >
 Programs.

1. Double-click on the
 Agreements icon to open the SM Agreements form.

1. In the Agreement field, enter the agreement for which to set up the
 billing schedule. Agreement must be in Quote status. Press F4
 to select from a list of valid agreements.

1. Click on the Billing
 Schedule tab.

1. Click the Automatic
 Scheduling button (above the tab pages). The SM Automatic Scheduling form
 displays.

1. From the
 Frequency
 drop-down, select the frequency at which billings will occur. See
 the [F1](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-automatic-scheduling-form/field-definitions-sm-automatic-scheduling-form#ID-00041d88--en) help for more information.

1. In the
 Start Date
 field, enter the date with which to start the billing schedule.
 The system will use this date as the billing date for all billing sequences in
 the schedule. For more information, see the [F1](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-automatic-scheduling-form/field-definitions-sm-automatic-scheduling-form#ID-00041db4--en) help.

1. From the Tax
 Type drop-down, select 1-Sales, 2-Use, or 3-VAT to indicate the tax
 type for billing sequences. Leave blank if not applying taxes to billing
 sequences.

1. In the
 Tax
 Code
 field, enter the tax code to use for all billing sequences. Leave
 blank if not applying taxes to billing sequences.

1. In the
 Tax
 Basis Percent
 field, enter the percentage to use to determine the tax basis for
 billing sequences. Leave blank if not applying taxes to billing sequences. For
 more information, see the [F1](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/service-management-agreements-forms/sm-automatic-scheduling-form/field-definitions-sm-automatic-scheduling-form#ID-00041dd1--en) help.

1. Click Schedule. The
 system will auto-generate a billing schedule for the agreement based on the
 agreement term and the information you entered here. Once you generate a billing
 schedule, you can edit it as needed until you activate the
 agreement.
