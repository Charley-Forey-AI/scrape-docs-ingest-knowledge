# About Service Work Schedules

You can define the frequency at which to perform the services for an agreement using the Schedule tab in the SM Service form.
For each service on an agreement's work schedule, you must define the frequency at which that service is to be performed. For services using the Daily, Weekly, Monthly, or Yearly recurring patterns, service days / dates are defined on the the Schedule tab. For more information, see [Setting up Service Work Schedules](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/agreement-setup/agreement-service-setup/service-schedules/about-service-work-schedules/set-up-service-work-schedules).
For services using the Task Scheduled recurring pattern, service scheduling is handled in SM Agreement Task Schedule. For more information, see [Setting up an Agreement Task Schedule](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/set-up-equipment-tasking/about-tasking--task-scheduling/set-up-an-agreement-task-schedule).
Once you activate the agreement, the system generates service dates based on the defined work and task schedules and displays them on the Work Orders tab (in SM Agreements).
