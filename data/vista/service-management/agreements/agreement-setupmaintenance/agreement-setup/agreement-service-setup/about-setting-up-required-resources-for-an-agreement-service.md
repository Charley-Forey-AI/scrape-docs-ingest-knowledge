# About Setting up Required Resources for an Agreement Service

You can set up requirements for materials, labor, equipment, and miscellaneous expenses to capture estimated costs for an agreement service.
When you generate work orders for an agreement, the material, labor, and equipment requirements set up for an agreement service are copied to the corresponding tabs on the related work order scope. Miscellaneous cost requirements are set up as miscellaneous work completed lines (with a Provisional status) on the work order.
 If you have set up tasks for the agreement service (Tasks tab) and the task references a standard task, any material, labor, equipment, or miscellaneous requirements defined for the task (in SM Standard Tasks) will be set up automatically on the corresponding tab in SM Service. Likewise, removing a task that references a standard task will automatically remove all material, labor, equipment, or miscellaneous requirements associated with the standard task.
