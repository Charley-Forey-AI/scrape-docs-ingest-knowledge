# Agreement Service Setup

Agreement services define the work to be done on an agreement, where and when it will be done, and how you will bill for the completed work.
You can set up agreement services manually or automatically.
 With manual set up, you create each agreement service individually using the Work
 Schedule tab in SM Agreements or the SM Service form (accessed by double-clicking in the
 Work Schedule grid). With automatic setup, the system uses selected maintenance tasks
 that you created (in the SM Class Maintenance form) to auto-generate agreement
 services.
Once you activate the agreement, the services are used to auto-generate work orders based
 on a defined schedule.
