# Set up Task Groups

You can set up task groups for scheduling agreement tasks in the SM Task Group form.
You can set up task groups by PR Co/Craft/Class. This allows you to set up the same craft/class combination for multiple payroll companies. Once you set up task groups, you can assign them to task schedules (in SM Agreement Task Schedule) for each month in which the selected service is to be performed.

1. In the Task
 Group field, enter a code to represent the task group.

1. In the PR
 Co field, enter the Payroll company for the task group.

1. In the Craft field, enter the craft for the task group.

1. In the Class field, enter the craft class for the task group.
