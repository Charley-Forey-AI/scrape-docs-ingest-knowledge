# Set up Serviceable Item
 Classifications

You will use the Serviceable Item Class form to set up the various classifications of equipment (serviceable items) that are maintained at each of your service sites, along with the different types of equipment within class.
You can set up as many classifications as necessary; however, you must set up a minimum of one serviceable item class with one classification type.

To set up serviceable item classes:

1. Go to Service Management > Programs > SM Serviceable Item Class.The SM Serviceable Item Class form displays.

1. In the Class field, enter the serviceable item classification. For example, Heating Equip, Cooling Equip, etc.

1. In the Description field, enter a description of the serviceable item class.

1. Select the Active checkbox to activate the serviceable item class. This checkbox must be selected in order to assign this class to serviceable items for a service site.

1. Save the record.

1.  Set up classification types for the serviceable item class.Note: Classes require at least one type in order to be assigned to serviceable items.

1. Select the Types tab.

1. In the Type field, enter a class type. For example, if the class is "Heating Equip", you might enter a type of "Furnaces".

1. In the Description field, enter a description of the classification type.

1. Select the Active checkbox to activate the class type. This box must be selected in order to assign this type to serviceable items for a service site.

1. Save the record.

1. Repeat steps b-e to add additional classification types for this class.

1. Set up [maintenances](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/set-up-equipment-tasking/set-up-maintenance-tasks-for-serviceable-item-classes) for the serviceable item class.

You can now assign the classes you set up to serviceable items on service sites.
