# Field Definitions: SM Task Group Form

The following is a list of field descriptions for the SM Task
 Group form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Task Group

Enter a code to represent this task group, up to 3 characters.

## PR Co

Required.
Enter the Payroll company associated with this
 task group. Press F4 to select from a list of valid Payroll companies.

## Craft

Required.
Enter the craft associated with this task
 group. Press F4 to select from a list of valid crafts for the specified PR Co.

## Class

Required.
Enter the craft class associated with this
 task group. Press F4 to select from a list of valid classes for the specified craft.
