# SM Agreement Maintenance Form

Use the SM Agreement Maintenance form to auto-generate agreement services based on planned maintenance for selected serviceable items.
 Access this form by clicking
 Tasking in SM Agreements or SM Service.
Note: The use of this form is part of the Equipment Tasking feature. Before you select maintenance tasks for an agreement here, you must first set them up by class or class/type in SM Class Maintenance. In addition, there are several other setups that are required to use the Equipment Tasking feature. For a complete overview on setting up and using the Equipment Tasking feature, see [Equipment Tasking](/en/vista/vista/service-management/agreements/agreement-setupmaintenance/set-up-equipment-tasking).
When you first access this form for an agreement, the Task, Labor, Material, and Equipment grids will be blank. Once you begin to select serviceable items (via the Select Serviceable Items button), the Tasks grid will populate with all tasks associated with the selected serviceable items. If labor, material, and/or equipment requirements have been defined for a task, they will display on the corresponding Labor, Material, and/or Equipment tabs.
Tasks with the Recommended checkbox selected in SM Class Maintenance are added to the grid with the Include checkbox selected automatically. Tasks not flagged as Recommended are added with the Include checkbox unselected. You will need to manually select the checkbox in order to include it on the agreement.
Once you select the tasks to include and close this form, the system automatically creates an agreement service for each task you selected. Multiple serviceable items that share the service site and class maintenance task are set up as a single agreement service. You can modify the agreement services as needed (in SM Service), as well as add, delete, or modify the related labor, material, and equipment requirements.
