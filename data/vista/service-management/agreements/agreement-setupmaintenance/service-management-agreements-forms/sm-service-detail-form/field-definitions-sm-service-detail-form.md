# Field Definitions: SM Service Detail Form

The following is a list of field descriptions for the SM
 Service Detail form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Labor Hours

Enter the number of hours you estimate will be needed to complete the work associated with this agreement service.

## Craft

Craft field on the SM Service Detail form.
This field is only enabled for Daily, Weekly, Monthly, and Yearly scheduling options (defined on the Schedule tab in SM Service).
Enter
 the craft to which the budget hours apply or press F4 to select from a list of valid
 crafts.

## Class

Class field on the SM Service Detail form.
This field is only enabled for Daily, Weekly, Monthly, and Yearly scheduling options (defined on the Schedule tab in SM Service).
Enter
 the class to which the budget hours apply or press F4 to select from a list of valid craft
 classes.

## Common Markup Rate

This checkbox only displays for agreement services with a Time of Service, Rate Template pricing method.
Select this checkbox to use the same markup rate for all cost estimates. When you enter a markup rate for any of the cost estimates, the system will automatically set the markup rate for all other cost estimates equal to the value entered here. Likewise, changing the markup rate for any of the cost estimates will automatically update the remaining estimates accordingly.
Leave this checkbox unselected to enter individual markup rates for cost estimates.

## Labor: Cost Estimate

Enter the estimated labor costs for this agreement service.

## Labor: Markup

Enter the estimated markup for labor costs on this agreement service. Enter 0.00 if you are not applying a markup rate.
If you selected the Common Markup
 Rate checkbox, the system will automatically set the markup rate for the
 remaining cost estimates equal to what you enter here. Likewise, changes to the markup rate
 for any other cost estimate will automatically update this markup rate accordingly. Do not
 select the Common
 Markup Rate checkbox if the markup rate entered here should be different
 than the markup rate for other cost estimates.

## Labor: Pricing Estimate

This field displays the pricing estimate for labor, and is a calculation of the labor cost estimate x the specified markup rate. You may override the pricing estimate if needed; the system will automatically recalculate the markup rate.
Note: If you selected the Common Markup
 Rate checkbox, this field is disabled and cannot be edited. However, you
 can change the pricing estimates for all cost estimates by changing the markup rate.
