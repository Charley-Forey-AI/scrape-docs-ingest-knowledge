# Field Definitions: SM Agreement Adjustment Form

The following is a list of field descriptions for the SM
 Agreement Adjustment form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Agreement

Display only, the agreement for which you are creating the adjustment billing.

## Customer

Display only, the customer associated with the agreement for which you are creating the adjustment billing.

## Department

Enter the department to which this adjustment
 billing applies or press F4 to select from a list of valid
 departments associated with the agreement.
If you selected the Post Revenue to Service
 Center Department checkbox for one or more periodic services on the
 agreement (in SM Service), the F4 list will include the department associated with the
 agreement, as well as each department defined for the periodic services. This allows you to
 apply the adjustment billing to the appropriate department.

## Description

Enter a description for this adjustment billing. Allowance is virtually, unlimited, allowing you to provide a descriptive reason for the adjustment. Entry here defaults as the invoice description in SM Agreement Invoice Review.

## Amount

Enter the adjustment amount (negative or positive) for this agreement revision.

## Tax Type

Specify the tax type for this adjustment billing, if applicable.

- 1 - Sales

- 3 - VAT

## Tax Code

This field is enabled (and required) only if
 you selected a tax type (Sales, Use, or VAT) in the Tax Type field.
Enter the tax code to use for calculating the tax amount for this adjustment billing.

## Basis

This field is enabled only if you selected a
 tax type (Sales, Use, or VAT) in the Tax Type field.
Enter the basis on which to calculate the tax amount for this adjustment. This amount cannot exceed the adjustment amount. For example, if the adjustment amount is -5,000.00, the tax basis must be between 0 and -5,000.00.

## Amount

Display only, the tax amount for this adjustment billing. Defaults based on the tax basis and the tax rate for the specified tax code.
