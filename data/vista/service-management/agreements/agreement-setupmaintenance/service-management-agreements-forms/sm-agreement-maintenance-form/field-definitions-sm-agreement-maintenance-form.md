# Field Definitions: SM Agreement Maintenance Form

The following is a list of field descriptions for the SM
 Agreement Maintenance form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Include

Select this checkbox to include the maintenance task on the agreement. The system will use the maintenance task to auto-generate an agreement service for the agreement.
Note: Maintenance tasks with the Recommended
 checkbox selected in SM Class Maintenance will have this checkbox selected by default. If
 you uncheck a Recommended task here, the Include checkbox for the serviceable
 item (in SM Maintenance Item Selection) will display as filled in (neither selected or
 unselected), indicating that one or more recommended tasks have not been added to the
 agreement for the serviceable item.
Do not select this checkbox if you do not want to include the maintenance task on agreement.
You can use the Check All / Uncheck All buttons to select / deselect all items or a group of items in the grid.
