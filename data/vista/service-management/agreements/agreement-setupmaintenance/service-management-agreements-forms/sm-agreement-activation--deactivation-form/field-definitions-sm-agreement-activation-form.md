# Field Definitions: SM Agreement Activation Form

The following is a list of field descriptions for the SM
 Agreement Activation form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Agreement

Display only, the agreement number/code.

## Revision

Display only, the current revision number for this agreement.

## Type

Display only, the agreement type assigned to this agreement.
Note: If you need to change the agreement type, you must do so (via the SM Agreements form) prior to activation.

## Previous Revision

Display only, the previous revision number for this agreement (if applicable).
Note: This field will only display a number if the agreement has been amended or renewed.

## Customer

Display only, the customer associated with this agreement.

## Effective Date

Display only, the effective date for this agreement.
Note: If you need to change the effective date, you must do so (via the SM Agreements form) prior to activation.

## Delete Existing New Work Orders on Termination

This checkbox displays only when activating an agreement amendment.
Check this box to delete existing new work orders upon termination of the original agreement/revision. Once you activate an amendment, the system will delete new work orders for all revisions prior the amendment (i.e. all revisions prior to and including the revision terminated by the amendment).
Leave this box unchecked to leave new work orders intact upon termination of the original agreement/revision.
