# Edit and Process Multiple Pending Invoices

You can edit and process multiple pending invoices using the SM Invoices form.
Using the Search Criteria, you can filter the
 invoices so that only those invoices you need to work on are displayed in the grid.

1. Open the SM Invoices form.

1. In the Invoice Status
 section, select Pending Invoice.

1. Use the Search Criteria
 to filter based on Bill To Customer and/or Invoice Date range.

1. Click Search. The grid will refresh to show all invoices meeting the
 specified search criteria.

1. Select the invoices to
 edit. You can do this by checking the Select box for individual
 invoices in the grid or by clicking Select All.

1. Click the Open button
 below the grid. The system creates an invoice session and opens the related
 invoice form. If you selected both work order and agreement
 invoices, the system opens one invoice session for the agreement
 invoices and one invoice session for the work order invoices.

1. Edit the invoices as
 needed.Note: For agreement invoices, only the invoice
 header information can be changed, as there are no invoice lines. For work
 order invoices, you can edit the header information and the individual
 invoice lines. For more information, see [Editing Work Order Invoices](/en/vista/vista/service-management/sm-invoices/about-editingprocessing-invoices-via-sm-invoice/edit-a-work-order-invoice).

1. Click Process.  The AR
 Batch Process form opens.

1. Validate and post the
 batch.
