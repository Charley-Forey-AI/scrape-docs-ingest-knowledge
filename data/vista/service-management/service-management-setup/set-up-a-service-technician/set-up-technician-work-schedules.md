# Set up Technician Work Schedules

Once you have set up a service technician (in SM Technicians), you can use the Work Schedule tab to set up the technician's work schedule.
Work schedules allow for dispatchers to identify the times that a technician is available or unavailable for scheduling on a work order.
Note: You are not required to set up work schedules; however, if you do, the system warns you if you assign a technician to a trip outside of the defined work schedule.

Use the following steps to set up a technician's work schedule.

1. Select the technician to which the work schedule applies.

1. Click on the Work Schedule tab.

1. Select the checkbox next to the first working day of the technician's schedule.

1. Enter the starting and
 ending work times in the Work Hours Start and End
 fields.

1. Enter the starting and
 ending break times in the Break Start and End
 fields.

1. Repeat Steps 3-5 for each day the technician works.

1. Save the record.
