# Set up Material Requirements for a Standard
 Task

You will set up the material requirements for a standard task using the Material tab of SM Standard Tasks.
Materials defined here will be set up automatically on any work order quote, agreement service, or work order scope that references the standard task.
The following steps detail how to set up material requirements for a standard task.

1. Launch the SM Standard Tasks form

1. Select the task for which to set up material requirements.

1. Click on the Material tab.

1. In the
 Seq
 field, enter N,
 New,
 or +. The system will auto-generate a
 sequence number for the material requirements entry.

1. Enter a part type in the Part Type field. Press F4 for a list of valid part types.Leave this field blank to enter a material in the Material field instead (cannot enter both).
Note: You will typically only enter a value in this field when the task requires a type of material, but the material number differs based on the equipment being serviced. For example, you can specify a part type of "Filter" here, and then specify the appropriate filter number when you add the standard task to a work order.
If you entered a part type here, proceed to step 9.

1. If you did not specify a part type in the Part Type field, use the Material field to enter the material required for this standard task. Press F4 for a list of valid HQ materials.Note: You will typically enter a value in this field when the task requires the same material, regardless of the equipment being serviced.

1. If you specified a material (in Step 6), enter the unit of measure for the specified material in the UM field. Press F4 for a list of valid units of measure for the material.

1. If you specified a material (in Step 6), enter the quantity of the material required for the standard task in the Matl Qty field.

1. In the Notes field, enter any notes about the material requirement as applicable to the standard task.

1. Save the record.

1. Repeat steps 4-10 for any additional material requirements associated with this standard task.
