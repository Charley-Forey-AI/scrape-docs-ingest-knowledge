# Field Definitions: SM Add Effective Date Form

The following is a list of field descriptions for the SM Add
 Effective Date form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## New Effective Date

Enter the new effective date for equipment, labor, material, and standard item override rates. This rate will be assigned to the new template created during the copy process.

## Copy From

Select the rate template to copy.

- Basic Template - Select this option to copy the basic template (i.e. the rate template from which you accessed this form).

- Effective Date - Select this option to copy the rate template for a specific effective date (as defined for the basic rate template). Enter the template to copy in the field to the right.

## Effective Date

This field is enabled only when the Effective
 Date option is selected in the Copy From group box.
Enter the effective date template to copy.
 Press F4 for a list of Effective Date templates for the basic rate template.
