# SM Company Parameters Form

Use the SM Company Parameters form to set up Service Management companies, along with the various control options applicable to each company before you begin processing in SM.
Before you set up an SM company in this form, you must first set it up in the HQ Company Setup form.
Note: It is suggested that you restrict this program so that only the System Administrator has access to it.
