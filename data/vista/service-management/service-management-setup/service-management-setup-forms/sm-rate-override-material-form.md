# SM Rate Override Material Form

Use the SM Rate Override Material form to set up override discount or markup rates for materials/parts based on a set of defined break points.
You can also use this form to set up override markup or discount rates by category or material.
 Access this form by:

-  Clicking Advanced in the Material Rates section of SM Rate Templates

- Clicking Advanced in the Material Rates section of SM Rate Template Effect Dates

- Clicking Override Rates in SM Customers, SM Service Sites, or SM Work Order Quotes to access SM Rate Override Base, and then clicking Advanced in the Material Rates section

Note: You will only need to use this form if you discount or
 markup rates differently based on the total dollar or unit cost value of materials used
 during service work, or if discounts/markups are based on categories or
 materials.
You can set up Rate Break Points to define one or more levels of discount or markup rates
 (depending on the rate type specified) for materials. You will only need to set up rate
 break points if you want to override the standard discount or markup rate defined for
 the template. For more information, see [Rate Break
 Points](/en/vista/vista/service-management/service-management-setup/about-rate-templates-and-override-rates/about-setting-up-material-rate-overrides/about-material-rate-break-points).
If you want to set up override mark or discount rates at the material or material
 category level, use the Material & Category Overrides tab. For each entry, you must
 specify the category or material (cannot be both), the rate type, the calculation basis,
 and the markup or discount percent. Double-clicking in the grid opens the SM Override
 Category Material form, allowing you to define rate break points at
 the category or material level. For more information, see [SM Override Category Material](/en/vista/vista/service-management/service-management-setup/service-management-setup-forms/sm-override-category-material-form).Note: Due to the
 differing levels of material rate overrides that can be set up, the system uses a
 specific hierarchy to determine which markup/discount rates to use when calculating
 billable rates for work completed work completed inventory or work completed
 purchase lines. For more information, see [Material Rate Hierarchy](/en/vista/vista/service-management/service-management-setup/about-rate-templates-and-override-rates/about-setting-up-material-rate-overrides/material-rate-hierarchy).
