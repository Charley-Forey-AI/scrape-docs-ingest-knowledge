# SM Assignments Form

Use the SM Assignments form to create work order assignments that you can associate with trips on a work order scope.
You can access this form from the SM Dispatch Board form by right-clicking on a work order and selecting Open Assignment.
 Assignments allow you to link one or more trips together (even if they are not on consecutive days) or link concurrent trips for technicians who are working on the same assignment. You can also enter a description for the assignment to keep tabs on particular details. Assignments can be located by work order, scope, or trip, and you can add, reassign, or release an assignment on any tab.
