# SM Divisions Form

Use the SM Divisions form to create divisions for categorizing work performed at a service center.
Access this form by double-clicking in the Divisions grid of the SM Service Centers form.
You can set up divisions to represent any type of work that a service center performs, such as installations, maintenance, special projects, electrical work, and plumbing, to name a few.
For each division you set up, you have the option to assign an 'alternate' department. Departments define the cost, revenue, and WIP GL accounts updated when capturing work completed for a work order, and are assigned to each service center you set up. You will only need to assign a department to the division if you require separate accounting based on division of work.
Note: The system only uses the department assigned to a division if the division is referenced on a work order scope.
