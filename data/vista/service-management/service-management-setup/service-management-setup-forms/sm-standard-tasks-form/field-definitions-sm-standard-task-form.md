# Field Definitions: SM Standard Task Form

The following is a list of field descriptions for the SM
 Standard Task form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Standard Task

Enter a standard task, up to 15 characters (e.g. Replace Filters, Install HVAC, etc.).

## Name

Required.
Enter the name of the task, up to 60 characters.

## Description

Enter a detailed description of the task; character allowance is virtually unlimited.
