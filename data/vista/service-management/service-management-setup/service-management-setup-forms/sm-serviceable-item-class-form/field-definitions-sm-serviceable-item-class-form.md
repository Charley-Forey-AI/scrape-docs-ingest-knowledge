# Field Definitions: SM Serviceable Item Class Form

The following is a list of field descriptions for the SM
 Serviceable Item Class form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Class

Enter a serviceable item (equipment) class, up to 15 characters (e.g. Cooling Equip, Heating Equip, Water Treatment, Hydronics, etc.).

## Description

Enter a description of the serviceable item class, up to 60 characters.

## Active

Check this box to activate this serviceable item class. Once activated, it can be assigned to serviceable items for a service site (in SM Service Sites).
Uncheck this box to deactivate this serviceable item class. Once inactive, it can no longer be assigned to serviceable items for a service site.
Note: Deactivating a class after it has been assigned to one or more serviceable items will not affect the serviceable item.

## Type

Enter a class type, up to 15 characters (e.g. if Class is Heating Equip, types might be Boilers, Heat Pumps, etc.).

## Description

Enter a description of the class type, up to 60 characters.

## Active

Check this box to activate this class type. Once activated, it can be assigned to serviceable items for a service site (in SM Service Sites).
Uncheck this box to deactivate this class type. Once inactive, it can no longer be assigned to serviceable items for a service site.
Note: Deactivating a class type after it has been assigned to one or more serviceable items will not affect the serviceable item.
