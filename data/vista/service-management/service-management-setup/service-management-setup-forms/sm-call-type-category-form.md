# SM Call Type Category Form

Use the SM Call Type Category form to set up and maintain categories for your call types.
Call type categories allow you to categorize the types of service calls you receive. You can also assign colors to call types, so that trips associated with a call type are color-coded on the SM Dispatch Board to easily distinguish between types of calls. Note: Trip call-type color on the Dispatch Board is determined by the call type of the first open scope.

Call Type Categories can represent scopes or classifications of work, or work classified by accounting treatment. Some examples of call type categories might be:

- Maintenance

- Service

- Concrete

- Plumbing

- Warranty

- Furnace

- Electrical
