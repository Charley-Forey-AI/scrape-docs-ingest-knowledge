# Set Up a Standard Task

Set up a standard task from the SM Standard Tasks
 form.

1. From the SM Programs
 menu, double-click on the SM Standard Tasks icon.  The SM Standard Task form
 displays.

1. In the Standard
 Task field, enter a standard task (e.g. Replace Filters).

1. Enter a name for the
 task in the Name field (required).

1. Enter a description of
 the task in the Description field.

1. Use the Notes tab to
 enter any pertinent notes or instructions related to the standard task.
