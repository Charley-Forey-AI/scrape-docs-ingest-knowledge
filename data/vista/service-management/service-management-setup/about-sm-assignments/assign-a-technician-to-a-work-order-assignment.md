# Assign a Technician to a Work Order Assignment

You can assign technicians to a work order assignment using the SM Assignments form.
Assignments are used to link work order trips together or link concurrent trips for technicians who are working on the same assignment. Using the SM Assignments form, you can assign technicians to those work order assignments.

1. Open the
 SM Assignments form (SM > Programs > SM Assignments).

1. In the
 Assignment
 field on the Info tab, enter an assignment number or press

 F4
to select from a list of assignment
 numbers.

1. In the
 Lead
 Tech
 field, enter the technician to assign or press
 F4
to select from a list of valid technicians.

1. Press Save Record.
