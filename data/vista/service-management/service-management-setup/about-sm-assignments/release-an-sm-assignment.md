# Release an SM Assignment

You can use the SM Assignments form to release a technician from an assignment.

1. From the main menu, select Service Management > Programs > SM Assignments.The
 SM Assignments form displays.

1. In the
 Assignment
 field on the Info tab, enter an assignment number, or press

 F4
to select from a list of valid assignments.

1. Click
 Release. The system asks you to confirm that you want to
 release the current assignment, and that doing so will delete any Open trips.

1. Click
 Yes.

The system deletes all trips for the assignment that have an 'Open' status.
