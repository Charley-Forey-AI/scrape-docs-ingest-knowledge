# Set Up Call Type Categories

Use the SM Call Type Category form to set up and maintain categories for your call types (such as Maintenance, Service, Concrete, Plumbing, Warranty, etc.).
Call type categories allow you to apply broad categories to the [call types you set up](/en/vista/vista/service-management/service-management-setup/set-up-call-types) in SM Call Types to classify work performed on a work order in order.
You may select call-type colors to color-code trip types that appear on [The Dispatch Board](/en/vista/vista/service-management/call-handler--dispatch-board/sm-dispatch-board), in order for you to easily differentiate one trip type from another.
Note: Trip call-type color on The Dispatch Board is determined by the call type of the first open scope.
The following details how to set up service call Trip call-type color is determined by the call type of the first open scope on The Dispatch Board. type categories.

1. From the Programs menu of Service Management, double-click on the SM Call Type Category icon. This opens the SM Call Type Category form.

1. In the Category field on the Info tab, enter a name for your call type category, and press Enter/Return.

1. In the Description field, enter a description for the call type category.

1. Click the Choose Color button.

1. This opens the Color pop-up.

1. Select a color by doing one of the following:

- Select a color from the list of basic colors.

- Add a custom color by using the sampler and slider tools and then select Add to Custom Colors.

- Enter specific color coding into either the Hue, Sat, Lum or Red, Green, and Blue fields, and select Add to Custom Colors.

1. Click OK once you have selected an appropriate color for your call-type category. This closes the Color pop-up.
Note: Call-type categories colors can be edited by selecting Choose Color and selecting a color other than it currently has. You can also select Restore Default to return the call type category's color to its default color.
