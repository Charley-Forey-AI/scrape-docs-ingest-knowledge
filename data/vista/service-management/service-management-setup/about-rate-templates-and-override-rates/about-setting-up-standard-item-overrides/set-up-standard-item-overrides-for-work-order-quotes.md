# Set Up Standard Item Overrides for Work Order Quotes

Set up standard item overrides for work order
 quotes.

1. From the Vista main
 menu, select Service Management > Programs.

1. Double-click the SM Work
 Order Quotes icon to open the SM Work Order Quotes form.

1. In the
 Quote ID
 field, enter the work order quote for which to set up standard
 item overrides.

1. In the quote

 Seq
 field (lower section of form), enter the quote sequence for which
 to set up standard item overrides. Must be a quote sequence with a
 Price Method
 of Time and Material.

1. Click the Override Rates
 button to open the SM Rate Override Base form.

1. Click the Standard Item
 Overrides button to open the SM Standard Item Overrides form.

1. In the
 Standard Item
 field, enter the standard item for which you are setting up an
 override bill rate.

1. In the

 Billable Rate
 field, enter the override billable rate to use for the specified
 standard item.

1. Save the record.
