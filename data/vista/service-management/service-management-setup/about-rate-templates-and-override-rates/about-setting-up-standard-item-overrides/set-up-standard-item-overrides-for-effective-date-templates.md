# Set Up Standard Item Overrides for Effective Date Templates

Set up standard item overrides for effective date
 templates.

Note: This process assumes you have already added
 Effective Date templates for the rate template. If you have not, [you must add them](/en/vista/vista/service-management/service-management-setup/service-management-setup-forms/sm-add-effective-date-form) prior to setting up standard
 item overrides.

1. From the Vista main
 menu, select Service Management > Programs.

1. Double-click the SM Rate
 Template icon to open the SM Rate Template form.

1. In the Template field, enter the rate template for which to set up
 standard item overrides.

1. Click on the Effective
 Dates tab.

1. Double-click on the
 desired Effective Date to open the SM Rate Template Effect Dates form.

1. Click the Standard Item
 Overrides button to open the SM Standard Item Overrides form.

1. In the Standard
 Item field, enter the standard item for which you are setting up
 an override bill rate.

1. In the  Billable
 Rate field, enter the override billable rate to use for the
 specified standard item.

1. Save the record.
