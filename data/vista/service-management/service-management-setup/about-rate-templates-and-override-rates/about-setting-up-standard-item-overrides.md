# About Setting up Standard Item Overrides

You can set up override rates for the standard items used to define common charges on a work order that are not specific to labor, equipment, or materials.
 You can set up standard items for charges not specific to labor,
 equipment, or materials on a work order, such as fuel surcharges, trip charges, and waste disposal. If the billable rate for these charges differs based on the rate template, effective
 date, customer, service site, or work order quote, you can use the
 SM Standard Item Overrides form to set up override billing rates.
