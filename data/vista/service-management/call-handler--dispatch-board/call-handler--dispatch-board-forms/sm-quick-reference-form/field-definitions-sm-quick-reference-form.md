# Field Definitions: SM Quick Reference Form

The following is a list of field descriptions for the SM
 Quick Reference form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Customer

Customer field on the SM Quick Reference form.
Enter a
 customer number in the Customer field to filter by this
 customer.
Press
 F4
 for the SM Customer Info Lookup form from which to select a customer. Press F5 to connect
 to [SM
 Customers](/en/vista/vista/service-management/service-management-setup/service-management-setup-forms/sm-customers-form) in order to set up Service Management customers.

## Site ID

Site ID field on the SM Quick Reference form.
Enter a
 site ID in the Site
 ID field to filter by site ID.
Press
 F4
 for the SM Customer Service Site Lookup form from which to select a customer service site.
 Press F5 to connect to [SM Service Sites](/en/vista/vista/service-management/service-management-setup/service-management-setup-forms/sm-service-sites-form) in order to set up customer and job
 service sites.

## Work Order

Work Order field on the SM Quick Reference form.
Enter a
 work order number in the Work Order field to filter by work order.
Press
 F4
 for the SM Work Order Lookup form from which to select a work order. Press F5 to connect
 to [SM
 Work Orders](/en/vista/vista/service-management/work-orders/service-management-work-order-forms/sm-work-orders-form) in order to create customer and/or job work orders for service
 work.
