# Field Definitions: SM Dispatch Technician Sort Form

The following is a list of field descriptions for the SM
 Dispatch Technician Sort form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Dispatch Board

Dispatch Board field on the SM Dispatch Technician Sorting form.
Enter a
 dispatch board name, or press F4 for the SM Named Dispatch Board Lookup
 for a list of existing dispatch boards from which to choose.

## Technicians

Technicians field on the SM Dispatch Technician Sorting form.
Select a technician—or technicians—to make order changes to, by doing one of the following:

- Use your pointer to select an individual technician

- Use your pointer to select an individual technician, and select other technicians next to it by pressing the
 Shift
 key

- Use your pointer to select an individual technician and pressing the
 Cntrl
 key to select technicians other than the one highlighted

Once technician(s) have been selected, use the following buttons to sort their order on the Dispatch Board:

- Move To Top
 – to move the technician(s) to the top of the Technicians list

- Move Up
 – to move the technician(s) above the technician immediately preceding

- Move Down
 – to move the technician(s) below the technician immediately following

- Move To Bottom
 – to move the technician(s) to the bottom of the Technicians list

Once you have sorted technicians, you can perform the following three actions:

- Reset
 – to reset to the original order of technicians when they were first entered into the system

- Cancel
 – to cancel changes made to the order of technicians so the new order is not implemented

- Save
 – to save the changes made to the order of technicians, so they appear in this preferred order on [The Dispatch Board](/en/vista/vista/service-management/call-handler--dispatch-board/sm-dispatch-board).

## User ID

User ID field on the SM Dispatch Technician Sorting form.
The
 User
 ID field reflects the current username, and cannot be altered on the [SM Dispatch Technician Sorting](/en/vista/vista/service-management/call-handler--dispatch-board/call-handler--dispatch-board-forms/sm-dispatch-technician-sorting-form) form.
