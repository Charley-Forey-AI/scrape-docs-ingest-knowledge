# Field Definitions: SM Quick Reference Preference Form

The following is a list of field descriptions for the SM
 Quick Reference Preference form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Tab Order

Tab Order field on the SM Quick Reference Preference form.
Enter a number to alter the tab-order sequence in SM Quick Reference Preference.

## Tab Name

Tab Name field on the SM Quick Reference Preference form.
Enter a name for the particular tab to appear under in SM Quick Reference Preference.

## Query Name

Query Name field on the SM Quick Reference Preference form.
Enter a query name for a particular query in SM Quick Reference.

## Target Form

Target Form field on the SM Quick Reference Preference form.
Enter a target form to reference in SM Quick Reference, or press F4 for a list
 of available forms.
