# SM Quick Reference Form

Use the SM Quick Reference form in tandem with the SM Dispatch Board or SM Call Handler forms to obtain details related to a customer, site, or work order.
Details include:

- Alerts

- Work Order Notes

- Invoices

- Scheduled Maintenance

- Quotes

- Serviceable Items

- Call History

Enter information a customer number, site ID, or work order number in the appropriate field, and click Apply Filter for additional details related to a customer, site, or work order.
Click Clear Filter to clear all fields.
Note: Permissions for the tabs on this form are granted using the VA Inquiry Security form. Type "smquickreference" in the Query Name field in VA Inquiry Security to locate and edit each tab's permissions. For more information about inquiry security, see [Set Security for Inquiries](/en/vista/vista/administration/viewpoint-administration/inquiries--messages/set-security-for-inquiries).
