# SM Quick Reference Preference Form

Use the SM Quick Reference Preference form to customize the order in which data is shown in the SM Quick Reference form.
Access this form by clicking Preferences in the toolbar of the SM Quick Reference form (which is accessed by clicking Quick Reference in the SM Dispatch Board toolbar).
 Tabs default to the following left-to-right order:

- Info

- Call History

- Invoices

- Scheduled Maint

- Quotes

- Serviceable Items

- Work Orders

Each tab is assigned a Tab Order number. To change the tab order, just enter the desired tab order number next to each tab name in the list. Once you click Save, you must close and reopen the SM Quick Reference form to see your changes.
