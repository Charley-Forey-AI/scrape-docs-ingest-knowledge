# Change the Dispatch Board Font Size

You can change the font size of the Dispatch Board to meet your specific needs.

1. Open SM Dispatch Board and select the Dispatch
 Board to work with.

1. Click Board Options. The Options window displays.

1. In the Font Size/Scaling field, enter the font size (e.g. 8.00) or use
 the up/down arrows to select the appropriate font.Note: Font size must be equal to or greater than 3.00.

1. Click OK.

 The screen automatically refreshes to reflect the new
 font size.
