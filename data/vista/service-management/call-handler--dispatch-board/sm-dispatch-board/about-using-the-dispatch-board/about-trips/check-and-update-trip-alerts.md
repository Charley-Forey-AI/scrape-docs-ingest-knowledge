# Check and Update Trip Alerts

You can check for alerts about scheduled trips and update the alerts as needed using the Alerts notification window in the Dispatch Board.

1. Open the SM Dispatch Board and select the Dispatch Board to work with.

1. Click the Alerts box (upper right corner of the dispatch board). The SM Dispatch Board Alerts
 (Unfinished Trips) window displays, showing a list of incomplete trips.

1. If you want to reschedule the trip:

1. Select the new reschedule date in the date field (left of the
 Reschedule button) or leave as defaulted (today's date).

1. Select one or more trips to reschedule, using the SHIFT key for
 consecutive selection or the CTRL key for random selection.

1. Click Reschedule. The system automatically reassigns each selected trip to the
 specified date for the assigned technician, and then removes the trips
 from the Alerts list.

1. If you want to complete the trip:

1. Select one or more trips to complete, using the SHIFT key for
 consecutive selection or the CTRL key for random selection.

1. Click Mark Complete. The system automatically sets the status for each selected trip
 to Completed for the current date, and then removes the trips from the
 Alerts list.

Once you reschedule or complete all listed trips, the
 Alerts notification is cleared from the dispatch board. Trips that you mark as completed
 display in the Incomplete query (left pane) and are grayed out on the dispatch board for
 the originally scheduled date.
