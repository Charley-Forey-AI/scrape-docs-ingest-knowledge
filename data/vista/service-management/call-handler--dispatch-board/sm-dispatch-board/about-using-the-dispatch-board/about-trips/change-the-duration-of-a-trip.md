# Change the Duration of a Trip

You can change the duration of a scheduled trip using the SM
 Dispatch Board.
 You can change the duration of a trip using either
 the Hourly view or Date view of the Dispatch Board. If using the Hourly view and the
 trip is a multi-day trip, you cannot use the drag-and-drop function. You must manually
 change the trip duration in SM Trips. If you are using the Date view and the trip was
 originally hourly, resizing that trip will clear the time data and convert the trip to
 an all-day or multi-day trip. If this is not what you want, edit the trip duration in SM
 Trips.

1. Open SM Dispatch Board and select the Dispatch Board to work with.

1. In the calendar pane, select the trip to
 update.

1. Grab the left or right side of the trip and drag it to the new time slot
 (hourly view) or day (date view). The status of the trip determines how the
 drag-and-drop works:StatusDescription
Open or NotifiedYou can drag the left and right edge.
Accepted or In ProgressYou can drag only the right side of the trip, as dragging the left
 edge would change the start time.
En Route, Arrived, or IncompleteYou cannot use the drag-and-drop function.

 If using the Hourly view, the system updates the
 trip duration based on the total number of hours within the new time range. If using
 the Date view, the system updates the trip duration and schedule.
