# Reassign a Trip

You can reassign trips to other dates/times or other
 technicians using the Dispatch Board.
The following video illustrates how to reassign a
 trip on the Dispatch Board.Note: You must use a browser on your local computer to view
 this video, and not a browser in the cloud. If you are currently using a cloud-based
 browser, copy this page's URL and paste it into a web browser on your local
 computer.

The following details how to reassign a trip using the Dispatch
 Board.

1. Open SM Dispatch Board and select the Dispatch Board to work with.

1. In the calendar pane, select the trip to
 reassign.

1. Drag the trip from the current Technician and Date/Time (depending on selected
 view) and drop it on the new Date/Time or the new Technician and Date/Time.Note: If the trip status is In
 Progress, En Route, Arrived, or Incomplete, you cannot use the
 drag-and-drop function. You must manually change the technician, date,
 or time in SM Trips.If you are
 working with multiple boards, you can drag a trip from one board to
 another; however, both the source and destination boards must exist in
 the same SM company.

The
 system automatically updates the trip in SM Trips.
