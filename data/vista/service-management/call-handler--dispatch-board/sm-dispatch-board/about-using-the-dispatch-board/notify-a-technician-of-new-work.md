# Notify a Technician of New Work

You can use the Dispatch Board to notify technicians of new
 work that has been assigned to them.
Use this feature to send an email to the technician informing them they have been
 scheduled for a trip. The technician must have an email address specified in PR
 Employees in order to use this feature.

1. Open SM Dispatch Board and select the Dispatch Board to work with.

1. In the calendar pane, right-click on a trip for the selected
 technician and date.

1. Select Notify from the context
 menu.

 The system sends an email to the technician indicating the work order/description,
 customer number/name, service site/description, call handler, contact information, trip
 description and details, and scheduled date/time. Additionally, the trip status is set
 to Notified (indicated by a blue bar across the top of the trip).
