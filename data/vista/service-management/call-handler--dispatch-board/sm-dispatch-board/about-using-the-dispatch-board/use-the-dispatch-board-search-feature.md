# Use the Dispatch Board Search Feature

You can use Search field on the SM Dispatch Board to filter information technician, customer, site, customer PO, call type, trip status, work order, agreement, center, or division.
The Search field, when used in tandem with either the Today or Go To Date buttons, allows you to filter by technician, customer, site, customer PO, call type, trip status, work order, agreement, center, or division. Select filtered search's results, then click Search to have results simultaneously appear on the dispatch board. As you enter data in the Search bar, the results may contain multiple
 items with the same characters. To distinguish between different item types (customer,
 service site, work order, etc.), the results include icons that identify the item type so
 that you know what you are selecting. These icons are as follows:

1. Open SM Dispatch Board and select the Dispatch Board to work with.

1. Select the specific dispatch board from the dispatch board pull-down menu (or select All for all dispatch boards).

1. Enter search words and/or numbers into the Search field.

1. Select results on which to search (more than one result may be selected to search on).

1. Click Search.Results activity appears on the dispatch board

1. Enter additional search criteria in the Search field and click Search to view activity simultaneously.Additionally, you can:

- Click the red "x" next to any result in the Search field to remove activity from displaying.

- Click Clear Filter to entirely remove results from the Search field

You can also use the following keyboard shortcuts for easy searching:

- Ctrl+F – to go to the search bar

- Ctrl+C – to clear the search bar

- Enter – to search
