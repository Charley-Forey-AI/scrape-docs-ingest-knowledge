# About Using the Dispatch Board

The Service Management Dispatch Board allows you to schedule work orders, track technician availability, and monitor the status of work orders throughout their life cycle.
The most basic use for the dispatch board is to schedule work orders;
 that is, to schedule trips for the technicians to perform the work outline on a work
 order. The following diagram illustrates the basic workflow for using the dispatch
 board.
In addition to performing tasks on the
 Dispatch Board, you can:

- access information about a customer by clicking the Customers button in the
 toolbar, which opens the SM Customers form. If you wish to view information
 about the customer associated with a specific work order, right-click on the
 work order from the appropriate query and select Open Customer.

- access information about a service by clicking the Service Site button in the
 toolbar, which opens the SM Service Sites form. If you wish to view information
 about the service site associated with a specific work order, right-click on the
 work order from the appropriate query and select Open Service Site.

- access information about a service call by clicking the Call Handler button in
 the toolbar, which opens the SM Call Handler form. If you wish to view
 information about the service call associated with a specific work order,
 right-click on the work order from the appropriate query and select Open Service
 Site.

- access basic information about a work order (such as WO number, customer number
 and name, service site, etc.) by hovering over the work order in the query pane
 of the Dispatch Board. For more information, double-click the work order to open
 SM Work Orders.
