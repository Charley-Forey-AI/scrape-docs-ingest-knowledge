# Field Definitions: SM WO Quote Cost Detail Form

The following is a list of field descriptions for the SM WO Quote Cost Detail form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Labor Hours

The Labor Hours field on the SM WO Quote Cost Detail form.
Enter the number of hours you estimate will be needed to complete the work associated with this quote scope.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Craft

The Craft field on the SM WO Quote Cost Detail form.
Enter the craft to which the budget hours apply or press F4 to select from a list of valid crafts.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Class

The Class field on the SM WO Quote Cost Detail form.
Enter the class to which the budget hours apply or press F4 to select from a list of valid craft classes.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Common Markup Rate

The Common Markup Rate field on the SM WO Quote Cost Detail form.
This checkbox only displays for quote scopes with a Time and Material pricing method.
Select this checkbox to use the same markup rate for all cost estimates. When you enter a markup rate for any of the cost estimates, the system will automatically set the markup rate for all other cost estimates equal to the value entered here. Likewise, changing the markup rate for any of the cost estimates will automatically update the remaining estimates accordingly.
Leave this checkbox unselected to enter individual markup rates for cost estimates.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Labor: Cost Estimate

The Labor: Cost Estimate field on the SM WO Quote Cost Detail form.
Enter the estimated labor costs for this quote scope.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Labor: Markup

The Labor: Markup field on the SM WO Quote Cost Detail form.
Enter the estimated markup for labor costs on this quote scope. Enter 0.00 if you are not applying a markup rate.
If you selected the Common Markup Rate checkbox, the system will automatically set the markup rate for the remaining cost estimates equal to what you enter here. Likewise, changes to the markup rate for any other cost estimate will automatically update this markup rate accordingly. Do not select the Common Markup Rate checkbox if the markup rate entered here should be different than the markup rate for other cost estimates.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Labor: Pricing Estimate

The Labor: Pricing Estimate field on the SM WO Quote Cost Detail form.
This field displays the pricing estimate for labor, and is a calculation of the labor cost estimate x the specified markup rate. You may override the pricing estimate if needed; the system will automatically recalculate the markup rate.
Note: If you selected the Common Markup Rate checkbox, this field is disabled and cannot be edited. However, you can change the pricing estimates for all cost estimates by changing the markup rate.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Default All

The Default All checkbox on the SM WO Quote Cost Detail form.
This checkbox is disabled if the Matl Tax Override drop-down is set to No Tax or Use Tax. However, you can manually select the Default Tax Amount? checkbox (unlabeled) for all cost estimate lines with values except the material cost estimate line.
Select this checkbox to default sales tax amounts for all cost estimates. If selected, the system auto-selects the Default Tax Amount? checkbox (unlabeled) for each cost estimate. Once you enter a cost estimate value, the system automatically defaults the sales tax amount for that cost estimate. If you do not enter a cost estimate, the system disables the checkbox for that cost estimate and leaves the sale tax amount blank.
Do not select this checkbox if you want to individually select which cost estimates will default a sales tax amount.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Default Tax Amount? (unlabeled)

The Default Tax Amount? (unlabeled) checkbox on the SM WO Quote Cost Detail form.
This checkbox is enabled once you enter a cost estimate and markup rate. If you selected the Default All checkbox, this checkbox defaults as selected.
Select this checkbox to default a sales tax amount for the cost estimate. Once you enter the cost estimate values (cost estimate and markup), the system calculates the tax amount based on the resulting pricing estimate and the tax code assigned to the service site or service center (depending on the scope's tax source).
Note: If this is a material cost estimate line and you selected No Tax or Use Tax in the Matl Tax Override drop-down (for quote scope), you cannot select this checkbox. However, you can still manually enter a sales tax amount.
Leave this checkbox unselected if you do not want sales tax automatically calculated for this cost estimate. You can manually enter a sales tax amount if desired.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Labor: Sales Tax Amount

The Labor: Sales Tax Amount field on the SM WO Quote Cost Detail form.
Enter the sales tax amount for this labor cost estimate or accept the defaulted tax amount.
The system only defaults a tax amount if you selected the Default Tax Amount? checkbox (unlabeled) for the cost estimate. The default tax amount is based on the tax code assigned to the service site or service center (depending on the scope's tax source) and the cost estimate's pricing estimate. You may override the defaulted amount if applicable.
Note: Overriding the defaulted tax amount deselects the Default Tax Amount? checkbox. If you reselect the checkbox, the system resets the tax amount to the original defaulted amount.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.

## Material: Sales Tax Amount

The Material: Sales Tax Amount field on the SM WO Quote Cost Detail form.
Enter the sales tax amount for this material cost estimate or accept the defaulted tax amount.
The system only defaults a tax amount if you set the Matl Tax Override option to blank or Sales Tax Only and you selected the Default Tax Amount? checkbox (unlabeled) for the cost estimate. The default tax amount is based on the tax code assigned to the service site or service center (depending on the scope's tax source) and the pricing estimate.
 If you set the Matl Tax Override option to No Tax or Use Tax Only, you cannot select the Default Tax Amount? checkbox, nor will the system calculate a tax amount. However, you can manually enter a tax amount.
Note: Overriding the defaulted tax amount deselects the Default Tax Amount? checkbox. If you reselect the checkbox, the system resets the tax amount to the original defaulted amount.
Note: This field is disabled once you approve the work order quote. If you add a new sequence to the quote after it is approved, this field is enabled for the new sequence only.
