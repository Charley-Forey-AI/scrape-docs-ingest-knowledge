# SM Work Order Quote Cancel
 Form

Use the SM Work Order Quote Cancel form to cancel and reopen work order quotes.
You can only access this form from SM
 Work Order Quotes by clicking Cancel Quote (for open quotes) or
 Reopen Quote (for canceled quotes).
