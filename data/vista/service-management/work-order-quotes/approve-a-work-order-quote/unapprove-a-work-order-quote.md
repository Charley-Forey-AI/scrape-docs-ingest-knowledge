# Unapprove a Work Order Quote

Unapprove a work order quote from the SM Work Order Quotes
 form.

1. Open the SM Work
 Order Quotes form.

1. Select the work order
 quote to unapprove in the Quote ID field. Press F4 for a list of valid work order quotes.

1. Click the Unapprove
 button at the bottom of the form. The SM Work Order Quote Unapproval form
 displays for the selected work order quote/customer/service site.

1. Click the Unapprove
 button in SM Work Order Quote Unapproval. The system sets the quote status
 to New, deletes the related work order and its scopes, and re-enables the fields
 on the quote.
