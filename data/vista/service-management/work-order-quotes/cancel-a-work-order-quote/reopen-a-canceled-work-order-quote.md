# Reopen a Canceled Work Order Quote

You can reopen a canceled work order quote using the SM Work
 Order Quotes form.

1. Open the SM Work Order
 Quotes form.

1. In the Quote ID field, select the work order
 quote to reopen or press F4 to select from a list of valid work order quotes.

1. Click Reopen Quote.  The SM Work Order Quote Cancel form
 displays for the selected work order quote/customer/service site.

1. Click Reopen Quote.

 The system returns you to SM Work Order Quotes, sets the quote
 status to New, and re-enables all fields.
