# Approve a Work Order Quote

You can approve a work order quote to generate a work order using the SM Work Order Quotes form.

1. Open the SM Work Order Quotes form.

1. In the Quote ID field, enter the work order quote to approve or press F4 to select from a list of valid work order quotes.

1. Click Approve.  The SM Work Order Quote Approval form displays for the selected work order quote/customer/service site.

1. Click Approve.

The system generates a new work order and you are returned to the SM Work Order Quotes form. The quote status is set to Approved and all fields are disabled.Note: If you need to make changes to the quote, you must first unapprove it
