# Cloud Deployments

Information for Vista users who access the application in Trimble Viewpoint's cloud.
Note:This information refers to features associated with Vista's modern cloud infrastructure. If you do not see some of these options, you may be running on a legacy version of cloud. Contact your account representative for more information.

If your organization is deployed in the cloud, there are two methods for
 establishing a cloud connection to Vista.

- [Vista
 Remote Link (VRL)
 Cloud](/en/vista/vista/cloud-deployment/about-vrl-cloud) - connection method used by the great majority of all Vista users. This connection method is
 simpler to use.

- [Remote Desktop (RDP)](/en/vista/vista/cloud-deployment/about-remote-desktop-rdp) - connection method used
 by a relatively small numbers of users, usually out of necessity based on heavy
 usage. This connection method isn't as simple as VRL Cloud, but it provides
 a better overall experience for those requiring higher performance, such as when
 doing payroll or payment processing and/or generating certain reports.
