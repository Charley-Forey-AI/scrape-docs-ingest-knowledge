# AP Recurring Invoice Posting Form

Use the AP Recurring Invoice Posting form to automatically create invoices
 based on data set up in the AP Recurring Invoices form. Once created, you can post this
 batch of invoices just like a regular batch of invoices. If you need to make changes, close this form and open the same batch using the AP Transaction Entry form.
