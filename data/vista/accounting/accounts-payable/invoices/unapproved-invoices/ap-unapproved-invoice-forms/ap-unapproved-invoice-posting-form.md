# AP Unapproved Invoice Posting Form

Use the AP Unapproved Invoice Posting form to post invoices that have passed through the review and approval workflow.
Once you post the invoices using this form, the system adds them to a standard AP Transaction Entry batch, where you can process them in the normal manner.
For more information on using this form, see [Posting Approved Invoices](/en/vista/vista/accounting/accounts-payable/invoices/unapproved-invoices/posting-approved-invoices).
