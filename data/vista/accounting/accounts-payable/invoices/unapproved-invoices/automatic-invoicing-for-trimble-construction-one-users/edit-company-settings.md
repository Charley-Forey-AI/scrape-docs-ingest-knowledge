# Edit Company Settings

The Accounts Payable Admin can manage company-specific access
 settings for users.
With company
 settings you can assign users and inboxes to specific companies. Users will only have
 access to inboxes associated with their assigned companies. In addition, users will only
 be able to select one of the assigned companies when uploading a batch. Follow these
 steps to manage your users' company access settings.

1. From any tab in Automatic Invoicing, select the gear
 icon  in the upper right-hand corner.Note: If you do not see the gear
 icon, you have not been set up as an Accounts Payable Admin. Contact your
 System Administrator for assistance.

1. Select the Company Settings tab.

1. To set up company-specific access, toggle on Enable Company-Specific
 Access .Note: By default, Enable Company-Specific
 Access is toggled off.

1. To assign a user to companies:

1. Under Assign Users to
 Companies, select the user that you want to assign
 companies to. You can sort the list of users from A-Z or Z-A using the
 sort arrow at the top of the column.

1. Select Edit Companies.

1. In the Company list, select
 the companies that you want to assign to the selected user. You can sort
 the list of companies from A-Z or Z-A using the sort arrow at the top of
 the column. To select all companies, select the Company checkbox at
 the top of the list. To filter the list, start typing in the Search Companies
 field at the top of the list.

1. Select Save.

1. To remove a company / companies assigned to a user:

1. Under Assign Users to
 Companies, select the user that you want to unassign
 companies from.

1. Select Edit Companies.

1. In the Company list,
 deselect the companies that you want to unassign from the selected
 user.

1. Select Save.

1. To export a list of user-company and company-inbox assignments as a CSV file,
 select Export
 Assignments.

Users will only have access to [inboxes](/en/vista/vista/accounting/accounts-payable/invoices/unapproved-invoices/automatic-invoicing-for-trimble-construction-one-users/about-the-inbox/add-an-inbox)
 associated with their assigned companies and will only be able to select one of the
 assigned companies when uploading a batch.
