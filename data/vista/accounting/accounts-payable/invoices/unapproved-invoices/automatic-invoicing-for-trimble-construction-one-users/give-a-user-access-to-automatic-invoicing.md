# Give a User Access to Automatic Invoicing

An Enterprise Admin can provide access to Automatic Invoicing by setting the Accounts Payable role for a user in Trimble Construction One.
For instructions on enabling Automatic Invoicing for a user and editing other user settings, see [Edit a User](/en/vista/vista/accounting/accounts-payable/invoices/unapproved-invoices/automatic-invoicing-for-trimble-construction-one-users/give-a-user-access-to-automatic-invoicing/edit-a-user).
For more Trimble Construction One administrator instructions, see the [Administration](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:Team_000007:Team:Team) section of the Viewpoint Team Help.
