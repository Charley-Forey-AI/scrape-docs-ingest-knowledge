# Field Definitions: AP On Cost Groups Form

The following is a list of field descriptions for the AP On Cost Groups form.Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Group ID

Enter a group ID number between 1 - 255. Press
 F4
 for a list of group IDs.

## Description

Enter a description for the group. Once you
 save the record, this become the description for the Group ID field. If you change this
 description on another record associated with the group, this field will update for all
 records associated with the group.

## On-Cost ID

Specify an on-cost type to associate with the group. Press F4 to see a list of available on-cost types.
