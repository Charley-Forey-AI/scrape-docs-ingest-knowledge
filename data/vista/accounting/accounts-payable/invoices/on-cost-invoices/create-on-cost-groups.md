# Create On-Cost Groups

You can create on-cost groups that you can associate with
 subcontractors in the AP Vendors form.
To create on-cost groups:

1. In the Group ID field, enter an ID number for the on-cost group. Press F4 to see a list of previously created groups.

1.  In the Description field, enter a description of the on-cost group.

1.  Select the On-Cost Types tab.

1.  In the On-Cost ID field, enter an on-cost type to associate with the group or press F4 to select from a list of valid on-cost types.

1. (Optional) In the Notes field, enter any notes about the on-cost type.

1.  Add additional on-cost types as needed.

1. Save the record.

You can now associate the on-cost group to
 subcontractors in the AP Vendors form (On Cost tab).
