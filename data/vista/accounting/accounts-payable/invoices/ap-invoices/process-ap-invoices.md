# Process AP Invoices

A high-level overview of processing accounts payable invoices/transaction from start to finish, with links to articles containing more detailed information.
The entire process includes, but isn't limited to, entering and posting invoices of different types, managing holds, and printing checks.

1. Enter invoices in the AP Transaction Entry form and [process the batch](/en/vista/vista/accounting/accounts-payable/ap-general-maintenance/ap-maintenance-forms/ap-batch-process-form)

1. Process and post any recurring invoices. For more information, see [Recurring Invoices](/en/vista/vista/accounting/accounts-payable/invoices/recurring-invoices).

1. [Process and post any unapproved invoices](/en/vista/vista/accounting/accounts-payable/invoices/unapproved-invoices/ap-unapproved-invoice-forms/ap-unapproved-invoice-posting-form). For more information, see [Unapproved Invoices](/en/vista/vista/accounting/accounts-payable/invoices/unapproved-invoices/about-unapproved-ap-invoices).

1. Review the AP Open Payables report. This is a list of all open payables as of the last posting, or through a selected month, for a selection of vendors. You can use this report to verify a statement or to check the total amount owed to a vendor. In addition, you can use it at month-end to verify the reconciliation of AP to the GL module.

1. [Assign or release hold codes](/en/vista/vista/accounting/accounts-payable/payments/invoicetransaction-holds/about-placing-invoicestransactions-on-hold).

1. [Add invoices to a payment batch](/en/vista/vista/accounting/accounts-payable/payments/payment-posting/about-invoice-payment-batches).

1. [Review the AP Payment Preview with Compliance Report](/en/vista/vista/accounting/accounts-payable/payments/ap-payment-forms/ap-payment-preview-form). This report shows all payment information for the specified batch.

1. [Print checks](/en/vista/vista/accounting/accounts-payable/payments/ap-payment-forms/ap-check-print-form) using the AP Check Print form.

1. [Download EFT files](/en/vista/vista/accounting/accounts-payable/payments/ap-payment-forms/ap-eft-download-form) using the AP EFT Download form.

1. If you downloaded EFT files, run the AP EFT Remittance report. You can run this report by selecting Tasks >  EFT/ePayments Remittance Report from the AP Payment Posting form or by running the report from the AP Reports folder in the Main Menu. Use this report even if you are not using ePayments (U.S only).

1. [Post the payment batch](/en/vista/vista/accounting/accounts-payable/ap-general-maintenance/ap-maintenance-forms/ap-batch-process-form) using the AP Batch Process form.

1. [Email payment information to vendors](/en/vista/vista/accounting/accounts-payable/payments/ap-payment-forms/ap-email-pay-info-form) using the AP Email Pay Info form.Note: If you are processing invoices with value added tax (VAT), there are some additional steps you must take. For more information, see [Processing VAT Invoices](/en/vista/vista/accounting/accounts-payable/invoices/ap-invoices/process-ap-invoices/processing-vat-invoices).
