# AP EFT Download Form

Use the AP EFT Download form to create Electronic Funds Transfer (EFT) transmission files as part of
 the process of paying vendors.
Access this form from the AP Payment Posting form by selecting
 Tasks > Download EFTs.
Enter data in the fields as needed, then select Initialize to
 add the transactions to a batch. After you load EFT transactions into a batch but
 before you post the batch, you must create and save a data file that includes the
 payment information. Select Download to save the file. Once
 you create and save the data file, you can then transmit the data file to your
 bank.
For more information about this process, see [Generate the EFT File](/en/vista/vista/accounting/accounts-payable/payments/electronic-funds-transfers-efts/generate-the-eft-file).
