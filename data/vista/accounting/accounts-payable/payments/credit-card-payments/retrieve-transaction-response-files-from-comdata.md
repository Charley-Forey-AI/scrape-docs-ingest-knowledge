# Retrieve Transaction Response Files from Comdata

You can view response files for completed
 transactions from Comdata, and then retrieve the files to automatically update AP
 Payment History.
You must have entered information you received from Comdata into the VA Site Settings form. See [Setting up credit services with Comdata](/en/vista/vista/accounting/accounts-payable/payments/credit-card-payments/set-up-credit-services-with-comdata) for more information.

To receive response files from Comdata:

1. Open the AP FTP form.The
 Url, Port, User ID,
 and Password fields default the values defined in the VA Site Settings form (which
 were provided to you by Comdata). Accept the default values.

1. If the Connection Status field
 says "No connection", click Connect to connect
 Vista to
 Comdata's FTP server.

1. Click Refresh to display the latest
 response files from Comdata's system in the Remote Folder pane.

1. From the Remote
 Folder pane, select the response file and click Transfer.The selected response
 file automatically updates the record in AP Payment History.

If multiple response
 files exist, you must select and retrieve each response file separately.
