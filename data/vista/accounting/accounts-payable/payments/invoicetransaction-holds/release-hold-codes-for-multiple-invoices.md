# Release Hold Codes for Multiple Invoices

You can use the AP Hold and Release form to release
 non-retainage hold codes for a group of invoices.
You can release non-retainage hold codes from
 multiple transactions that are ready for payment.Note: To release
 retainage holds for a transaction, you must use AP Release Retainage. For more
 information, see [Release Retainage for an AP Invoice](/en/vista/vista/accounting/accounts-payable/payments/retainage/release-retainage-for-an-ap-invoice).
 The following instructions detail how to
 release non-retainage hold codes from transactions.

1. From the main menu, select Accounts Payable > Programs > AP Hold and Release.

1. Select the Release a Hold Code radio
 button.

1. Enter the hold code in the Hold Code field. Press F4 to see a list of hold codes.

1. If you want to restrict releasing by a job select the Restrict by Job checkbox.The system enables the JC Co# and Job fields.

1. Enter a company and job in the JC Co# and Job fields, respectively.The system enables the Restrict by
 Phase checkbox if you enter a value in the Job field.

1. If you want to restrict by phase select the Restrict by Phase checkbox and select the appropriate phase(s) from the selection
 box.

1. If you want to restrict by vendor, select the Restrict by Vendor checkbox and enter a vendor in the Vendor field.

1. If you want to restrict by payable type, select the Restrict by Payable Type box and select the appropriate payable type(s) from the selection
 box.

1. Click Fill Grid.The system displays all transaction detail in the grid on
 the Selection Display tab and displays the total release amount in the
 Total Amount to Release display field.

1. Review the transactions on the Selection Display tab.Tip: You can delete transactions from the grid by selecting a line
 and pressing the Delete key. You can shift-click to select multiple
 consecutive transaction or control-click to select multiple nonconsecutive
 transactions.

1. Click Update to release the hold codes from the transactions listed on the
 Selection Display tab.Note: If you close the AP Hold and Release form before pressing Update, the system displays a warning that it will clear all lines
 on the Selection Display tab. Click Yes to continue or click No to keep the AP Hold and Release form open.
