# Field Definitions: AR Release Retainage Detail Form

The following is a list of field descriptions for the AR Release Retainage Detail form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Release Pct

 This field indicates the release percent for this line item. Override percentage as necessary.

##  Release Amt

 This field indicates the release amount for this line item. Override amount as necessary.

##  Release Retg Tax Amt

 This field displays only if you checked the ‘Show in Grid’ option in F3 Properties.
 Indicates the amount of retainage tax released for this line item. Override amount as necessary.
