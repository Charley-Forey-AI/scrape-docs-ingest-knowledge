# Field Definitions: AR Add Transaction to Batch Form

The following is a list of field descriptions for the AR Add Transaction Batch form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  AR Transaction#

 Specify the AR transaction that you wish to add to this batch. Press F4 for a list of available transactions. The current batch month, and batch ID are displayed above.
