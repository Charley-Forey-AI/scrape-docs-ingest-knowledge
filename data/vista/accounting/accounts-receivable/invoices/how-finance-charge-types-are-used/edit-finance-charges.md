# Edit Finance Charges

You can edit system-calculated finance charges before posting the batch, or after posting as long as the GL period is still open.
To edit finance charges:

1. If the finance charges are already posted, bring the relevant records into a new batch using the AR Add Transaction to Batch form.

1. In your batch of records, make the necessary changes.

1. Post the batch.
