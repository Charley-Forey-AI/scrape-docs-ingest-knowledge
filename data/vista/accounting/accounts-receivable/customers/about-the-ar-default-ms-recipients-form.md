# About the AR Default MS Recipients Form

You can set up default recipients for invoice delivery by customer.
Use this form to set default recipients for MS invoice deliveries via the MS Invoice Deliver form. Access this form by clicking the Default Recipients option on the Material Sales tab of the AR Customers form.
Note: All listed default recipients will receive the same invoice(s).
