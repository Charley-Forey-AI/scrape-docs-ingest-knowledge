# Set Default Recipients for MS Invoice Deliveries

You can set up one or more default recipients for invoice delivery.

1. From the main menu, select Accounts Receivable > Programs > AR Customers.

1. Click on the Material Sales tab.

1. Click the Default Recipients button.The AR Default MS Recipients form
 appears.

1. In the Seq field, enter +, N, or New for the next sequential number.

1. In the Delivery Method field, select either E-Email  or P-Print for invoice delivery method.

1. In the Name field, enter the name of the invoice recipient.

1. Enter the additional email or address information in the appropriate fields for the default recipient.Note: If you select the E-Email delivery method, an
 Email address is required; if you select the P-Print delivery method, Address,
 City, State and ZIP Code information is required.

1. Save the record.

All listed default recipients hereafter receive all invoice(s) generated from MS Invoice Deliver.
