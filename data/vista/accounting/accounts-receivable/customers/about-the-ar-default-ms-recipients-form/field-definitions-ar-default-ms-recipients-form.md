# Field Definitions: AR Default MS Recipients Form

The following is a list of field descriptions for the AR Default MS Recipients form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Seq

Seq field on the AR Default MS Recipients form.
Enter +, N, or New. The system automatically assigns the next sequential number.

## Delivery Method

Delivery Method drop-down list on the AR Default MS Recipients form.
Select either E-Email or P-Print for invoice delivery method.

## Name

Name field on the AR Default MS Recipients form.
Enter the name of the recipient who should receive invoices generated from MS Invoice Deliver.

## Email

Email field on the AR Default MS Recipients form.
Enter the email address of the default recipient (required if you selected E-Email from Delivery Method).

## Address

Address field on the AR Default MS Recipients form.
Enter the address of the default recipient (required if you selected P-Print from Delivery Method).

## Add'l Address

Add'l Address field on the AR Default MS Recipients form.
Enter additional address information of the default recipient if applicable.

## City

City field on the AR Default MS Recipients form.
Enter the city of the default recipient (required if you selected P-Print as the Delivery Method).

## State

State field on the AR Default MS Recipients form.
Enter the state of the default recipient (required if you selected P-Print as the Delivery Method).

## Zip Code

Zip Code field on the AR Default MS Recipients form.
Enter the zip code of the default recipient (required if you selected P-Print as the Delivery Method).

## Country

Country field on the AR Default MS Recipients form.
Enter the country of the default recipient if other than United States.
