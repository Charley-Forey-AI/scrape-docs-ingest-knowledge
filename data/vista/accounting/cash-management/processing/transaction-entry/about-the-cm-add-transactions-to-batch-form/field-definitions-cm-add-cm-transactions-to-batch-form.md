# Field Definitions: CM Add CM Transactions to Batch Form

The following is a list of field descriptions for the CM Add CM Transactions to Batch form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  CM Transaction #

 Specify the CM transaction that you want to add to this batch. Press F4 for a list of valid transactions. Valid transactions will depend on the form to which you are adding the transaction.

- If adding to a CM Outstanding Entries batch, must be outstanding transaction posted from CM; transactions posted from AP, PR, or AR cannot be edited in an outstanding entries batch.

- If adding to a CM Transfers batch, must be a transfer transaction. Associated ‘to’ and ‘from’ transactions must exist as outstanding transactions.
