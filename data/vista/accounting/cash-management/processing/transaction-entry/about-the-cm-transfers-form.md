# About the CM Transfers Form

You can use the CM Transfers form to make transfers from one account to another, including transfers between
 companies.
Specify the source and destination accounts and the
 amount, and the system handles the rest. You do not need to specify any GL account code
 information when posting cash transfers. The system automatically pulls the account
 codes entered in the CM Accounts form.
Transfers generate an outstanding check in the From Account and a corresponding deposit in the To Account. Both of these entries must be cleared using the standard clearing procedure.
