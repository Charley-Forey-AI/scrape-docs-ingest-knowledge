# Field Definitions: CM Clear Initialize Form

The following is a list of field descriptions for the CM Clear Initialize form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Use Auto Clearing File

Use Auto-Clearing File checkbox on the CM Clear Initialize form.
Select this checkbox to upload entries from an auto-clearing file.
Do not select this checkbox if you are not initializing cleared entries using an auto-clearing file.

Related information

- [Clear CM Entries](/en/vista/vista/accounting/cash-management/processing/reconciliation/clear-cm-entries)

- [About the CM Clear Initialize Form](/en/vista/vista/accounting/cash-management/processing/reconciliation/about-the-cm-clear-initialize-form)

## Upload Date

Upload Date field on the CM Clear Initialize form.
This field is used when initializing cleared entries from a text file. Specify the upload date of the text file that you wish to initialize.

Related information

- [Clear CM Entries](/en/vista/vista/accounting/cash-management/processing/reconciliation/clear-cm-entries)

## Beginning Reference

Beginning Reference field on the CM Clear Initialize
 form.
Specify the CM reference number with which to begin the initialization. CM Reference numbers less than the number specified here will not be cleared. Leave blank for all.

Related information

- [Clear CM Entries](/en/vista/vista/accounting/cash-management/processing/reconciliation/clear-cm-entries)

- [About the CM Clear Initialize Form](/en/vista/vista/accounting/cash-management/processing/reconciliation/about-the-cm-clear-initialize-form)

## Ending Reference

Ending Reference field on the CM Clear Initialize
 form.
Specify the CM reference number with which to end the initialization. CM Reference numbers greater than the number specified here will not be cleared. Leave blank for all.

Related information

- [Clear CM Entries](/en/vista/vista/accounting/cash-management/processing/reconciliation/clear-cm-entries)

- [About the CM Clear Initialize Form](/en/vista/vista/accounting/cash-management/processing/reconciliation/about-the-cm-clear-initialize-form)

## Clear Outstanding Entries Dated On or Before

Clear Outstanding Entries Dated On or Before field on the CM Clear Initialize form.
Specify the date through which to automatically clear outstanding entries. All entries dated on or before this date will be marked as cleared.
