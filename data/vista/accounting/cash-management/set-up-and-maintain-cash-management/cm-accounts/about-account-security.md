# About Account Security

You can restrict access to bank accounts by turning on security for the CM Account datatype in VA Secure Tables and Datatypes, and by setting up restrictions in VA Data Security.
Once you have designated security for the datatype and for each occurrence (in this case, each account), only the users you grant access to the specific accounts may view or post to them.
If you want a specific user (or users) to have access to an account, set them up in the VA Security Groups form, and then grant the Security Group access to the CM Account Datatype with the “instance” being the specific account.
For more information on data security and security groups, see [VA Data Security Setup Form](/en/vista/vista/administration/viewpoint-administration/application-security/data-record-and-content-security/va-data-security-setup-form#ID-000489c2--en__ID-000489c2).
