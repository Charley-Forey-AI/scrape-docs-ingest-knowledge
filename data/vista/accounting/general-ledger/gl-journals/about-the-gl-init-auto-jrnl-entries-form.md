# About the GL Init Auto Jrnl Entries Form

Use the GL Init Auto Jrnl Entries form to create auto journal entries.
You can access this form in one of two ways:


- If you are creating a new batch of entries, this form will display automatically
 once you click OK in the Batch Selection form. You must initialize entries
 before you can access the GL Auto Journal Entry form.

- If you are already working with an open batch, you can access this form by
 selecting the Initialize Entries option from the File menu of GL Auto Journal
 Entry.

Automatic journal entries are initialized
 based on a journal and a frequency. Once you specify the journal, the selection box
 displays all available frequencies for the journal. You can then select the frequencies
 you want or select to initialize all frequencies. Once you click the Initialize button,
 the initialization process will create journal entries for each reference. When
 initialization is complete, a message displays indicating the number of entries
 initialized. Click the Close button to open the GL Auto Journal Entry form.
Note: Initialization will not skip references that have
 been previously initialized. Therefore, initializing a journal/reference more than once
 will cause duplicate entries for that journal/reference.
For details, see [About the GL Auto Journal Entry Form](/en/vista/vista/accounting/general-ledger/gl-journals/about-the-gl-auto-journal-entry-form).
