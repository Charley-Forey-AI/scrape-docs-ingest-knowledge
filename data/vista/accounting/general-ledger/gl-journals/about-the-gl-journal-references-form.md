# About the GL Journal References Form

Use the GL Journal References form to set up Journal References.
A reference provides a way to tie together related journal entries. All entries sharing the same reference within a journal and month must net to zero.
References are not validated, but are added as entries are posted to GL. You can add descriptions and notes at any time.
You cannot change the Journal and Reference fields on existing transactions. If you need to make changes, you must first delete the transaction and then re-enter it.
