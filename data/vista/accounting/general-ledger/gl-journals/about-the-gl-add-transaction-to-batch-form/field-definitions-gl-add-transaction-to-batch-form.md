# Field Definitions: GL Add Transaction to Batch Form

The following is a list of field descriptions for the GL Add
 Transaction to Batch form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## GL Transaction#

Specify the GL transaction you wish to add to this batch. Press F4 for a list of available GL detail transactions.
Note: You will typically enter a value in this field when adding transactions to the batch one at a time (e.g. you only want to add one transaction or, you are adding multiple transactions of differing journal/reference numbers).
If you need to add multiple transactions to
 the batch that use the same journal and reference number, skip this field and enter the
 applicable values in the Journal and Reference
 fields below.

## Journal

Enter the journal for which to add previously
 posted transactions to the batch. Entering a value in this field will require that you also
 enter a value in the Reference field. Once a reference number
 is specified, the Add to Batch button is enabled.
Note: You will typically use this field when you need to add
 multiple transactions to the batch that were posted to the same journal and reference
 number. If adding a single transaction to the batch, use the GL Transaction# field above.

## Reference

Enter the reference number of the transactions
 you are adding to the batch. You must enter a value in this field if you entered a value in
 the Journal field above. Once both fields are populated, the Add to Batch
 button is enabled.
Note: You will typically use this field when you need to add
 multiple transactions to the batch that were posted to the same journal and reference
 number. If adding a single transaction to the batch, use the GL Transaction# field above.
