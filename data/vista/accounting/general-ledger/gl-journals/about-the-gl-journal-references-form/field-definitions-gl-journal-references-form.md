# Field Definitions: GL Journal References Form

The following is a list of field descriptions for the GL
 Journal References form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Month

 Enter the month in which this reference is applicable.

##  Journal

 Enter the journal to which this reference is applicable.

##  Reference

 Enter a reference code (up to 10 characters) used to associate related journal entries. All entries with the same reference must balance. Once posted, the reference on a transaction cannot be changed. Reference codes must be unique only within the month and the journal.

##  Description

 Enter a description of this reference (up to 30 characters). This description displays for validation purposes when you specify this reference in a transaction.

##  Adjustment Entry

 Check this box to have entries associated with this GL reference posted as adjustment period entries. Adjustments can only be posted to the ending month of a fiscal year.
 Leave this box unchecked if entries with this reference will not be posted as adjustments.
Note: If journal entries exist for this company, month, and
 reference, this flag cannot be changed.
