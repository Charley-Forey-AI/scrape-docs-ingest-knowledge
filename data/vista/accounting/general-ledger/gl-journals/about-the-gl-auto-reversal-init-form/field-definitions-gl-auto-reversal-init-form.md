# Field Definitions: GL Auto Reversal Init Form

The following is a list of field descriptions for the GL Auto
 Reversal Init form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Journal

 Specify the journal for which to initialize reversals. Must be a reversing journal (one with the Auto Reverse option checked in GL Journals) with existing reversal entries.

##  Orig Month

 Specify the month in which the original entries were made.

##  Reversal Transaction Date

 Enter the date of the reversal entry.

## References

- All References—Select this option to initialize all auto reversal entries set up for this journal.

- Select References—Select this option to choose specific references within this journal for which to initialize reversing entries.
