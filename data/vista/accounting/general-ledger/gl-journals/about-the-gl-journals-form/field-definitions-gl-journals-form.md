# Field Definitions: GL Journals Form

The following is a list of field descriptions for the GL
 Journals form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

##  Journal

 Enter a two-character code for each journal. It is suggested that journals be set up for every subsidiary ledger (AP, AR, PR, etc.) as well as any specialized journals you desire. Because you can specify different journals within a module, you can set them up to be as specific as you desire.

##  Description

 Enter a description of this journal (up to 30 characters) as it should appear on all reports and printouts.

##  Auto Reverse

 Check this box if entries posted to this journal should be reversed in some future period.
 Leave this box unchecked if entries posted to this journal will not be set up to automatically be reversed.

##  Reversal Journal

 If you checked the Auto Reverse option for this journal, specify which journal will receive the reversing entries.This may be the same or a different journal.
