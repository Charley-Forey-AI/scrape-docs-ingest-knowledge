# Field Definitions: GL Init Auto Jrnl Entries Form

The following is a list of field descriptions for the GL Init
 Auto Jrnl Entries form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## Journal

 Indicate the journal to which these transactions will be posted.

## Transaction Date

 Enter the date in which the transaction is being made.

## Frequencies

- All Frequencies—Select this option to initialize all auto entries set up for this journal, regardless of the frequency assigned to each entry.

- Select Frequencies—Select this option to choose specific frequencies within this journal to initialize.
