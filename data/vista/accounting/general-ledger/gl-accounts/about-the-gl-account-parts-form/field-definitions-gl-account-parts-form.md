# Field Definitions: GL Account Parts Form

The following is a list of field descriptions for the GL
 Account Parts form. Many of the descriptions include links to other topics that provide additional information about or related to the topic.

## GL Account Part #

 Enter a unique numeric code identifying this Part.

## Description

 Enter a name for this group of accounts or part.

## Instance

 Enter a code to identify this account part. For example, if the account part is a profit center, such as Division, then Instance 01 might be the Commercial Division and Instance 02 might be the Residential Division.

## Instance Description

 Enter a description of this instance of the account code, such as asset or liability, or East or West branch.

## Major Category

Available only when the GL Account Part
 # field is set to 1.
Enter a major category (set up in [GL Report Categories](/en/vista/vista/accounting/general-ledger/gl-financial-reporting/about-the-gl-report-categories-form)), or press F4 for a list
 of existing major categories.
After you have associated the GL account part instance with a major category, you can then use the category to sort and group sections of financial statements.

## Minor Category

Available only when the
 GL Account Part #
 field is set to 1 and a value is present in the
 Major Category
 field.
Enter a minor category that is associated with the selected major category (set up in [GL Report Categories](/en/vista/vista/accounting/general-ledger/gl-financial-reporting/about-the-gl-report-categories-form)), or press
 F4
 for a list of existing minor categories.
After you have associated the GL account part instance with a major and minor category, you can then use them to sort and group sections of financial statements.
