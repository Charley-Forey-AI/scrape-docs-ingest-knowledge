# Level Of Detail Interfaced To GL

Every account code set up in GL Chart of Accounts allows you to specify whether that account will have all transactions interfaced to it from subsidiary ledgers (other modules), or whether the system will use the information specified in the subsidiary modules company file to determine what level of detail will interface.
If the Interface Detail checkbox is
 selected in GL Chart of Accounts, then data will always interface in the most detailed
 level possible, no matter what is specified in the subsidiary modules for that account.
 Each subsidiary ledger interface control is located in the Company Parameters form of that module.
