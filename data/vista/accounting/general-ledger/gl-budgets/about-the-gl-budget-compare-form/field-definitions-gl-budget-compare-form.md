# Field Definitions: GL Budget Compare Form

##  Compare To

Actuals – Select this option to use actual account balances from a selected year for comparison reporting.
Budget – Select this option to use the budget account balances for a certain budget code for comparison reporting.

##  Compare To: Fiscal Year Ending

 For comparison to actual or budget amounts, enter the calendar month of the fiscal year end for these budget amounts.

##  Compare To: GL Account

 Enter the GL account to use for budget comparison. Initially defaults to the GL selected in GL Budgets.

##  Compare To: Budget Code

 Enabled only if comparing to another budget.
 Specify the budget code to use for comparison.
