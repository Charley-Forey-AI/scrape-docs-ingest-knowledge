# Introduction

This section contains information to guide you through the
 installation and upgrade of your Vista™software.

The installation/upgrade process enables you to keep your Vista software up-to-date. For this release, the installation/upgrade process covers the followings applications:

- Database Server

- Application Server

## Audience

These installation instructions are intended for customers with
 self-hosted, on-premise versions of Vista™.
If your organization's instance of Vista is hosted in the cloud, Viewpoint also manages the installation and upgrade process. You
 should consider this content informational only.
