# Sign in to SSRS Report Manager with Trimble ID

Use your Trimble ID credentials to log in
 to the SSRS Report Manager portal.
Your organization must have SSRS reports set up and Trimble ID logins enabled.

1. Go to your SSRS Report Manager portal:
 https://<your_server_name>/reportsYou should see the following login screen:

1. Select Sign in with Trimble ID.The Trimble login screen opens.

1. Enter your Trimble ID credentials to [Sign in with Trimble ID](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:accountservices_002:accountservices:accountservices).If you don't have a Trimble ID, you can create one. For
 details, see [Create a Trimble ID](https://apim.viewpointplatform.com/help/v1/contents/urn:contentId:accountservices_001:accountservices:accountservices).

Once you finish signing in, you will be redirected to the SSRS Report Manager portal.
 If you experience issues signing in, contact your administrator.
