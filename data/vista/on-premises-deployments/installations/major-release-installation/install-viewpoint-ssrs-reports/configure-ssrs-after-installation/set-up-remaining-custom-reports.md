# Set Up Remaining Custom Reports

If there are reports on the SSRS Reports Server that are not set up in the RP Report Titles form, you must create those report titles manually.
In order to open these reports in Vista, you must set them up manually in the RP Report Titles form. This means that you may also have to manually create the report locations associated with those reports using the RP Report Locations form.
