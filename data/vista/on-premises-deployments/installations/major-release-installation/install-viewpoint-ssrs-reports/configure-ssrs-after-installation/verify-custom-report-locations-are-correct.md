# Verify Custom Report Locations Are Correct

Verify that the Viewpoint SSRS Reports installer has assigned the correct report locations to the custom SSRS reports.
This applies only to SSRS reports that are not part of the standard BI module library (custom reports).

1. Open the RP Report Titles form.

1. Select the Grid tab.

1. Click on the Application Type column to sort the reports by application type.

1. Verify that the Location and Path fields are correct on all custom SSRS reports.
