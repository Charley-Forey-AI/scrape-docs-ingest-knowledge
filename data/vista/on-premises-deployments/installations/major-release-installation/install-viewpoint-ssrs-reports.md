# Install Viewpoint SSRS Reports

The Viewpoint SSRS Reports installer loads a library of standard Vista SSRS reports onto your SSRS Report Server.
These reports are specifically designed to be added to Dashboard Work Centers in the Vista application, but you can also email them to a list of users at regular intervals.
Important: Please note the following:

- If you do not use SSRS Reports, you may skip this section.

- If you do use SSRS Reports, you are not required to run the SSRS installer each time there is a release; however, doing so will not cause any harm. If and when an SSRS reinstall is required, we will alert you in this installation guide.

- The versioning for the Viewpoint SSRS reports installer has been disassociated from other Vista products. Starting with Vista version 2023 R1, the correct version of the Viewpoint SSRS Reports installer is 42.0.0.x.
