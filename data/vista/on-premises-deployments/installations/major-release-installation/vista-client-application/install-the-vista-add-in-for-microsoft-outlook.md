# Install the Vista Add-in for Microsoft Outlook

You can install the Vista add-in for Microsoft Outlook to enable sending emails directly from Outlook to Vista.
If you choose to install the add-in, you will be able to index emails to Vista records, from within Outlook.
Note: The user login that will be using the Viewpoint Add-in must be the login to install it. In addition, the user login must have administrative rights to the workstation and must have previously configured Outlook. Outlook must be functional for the user login prior to installation. In the case of Citrix/Terminal Servers, the user permissions must be elevated for the period of time it takes to install the Add-in. Once complete, you can lower the permissions to normal. If needed, contact your system administrator.

 If you do not install the add-in, you cannot send emails directly from Outlook to Vista. You can, however, drag and drop emails to Vista data records as attachments.
If you want to disable the installed Viewpoint Add-in, you can do so in Microsoft Outlook (File > Options > Add-Ins).

1. Install the Vista add-in.​

1. Log in to the Vista client to ensure the latest Outlook Add-in version has been loaded to your computer.​

1. Open MS Outlook. If it is already open, close and restart it. If the system prompts it, restart your computer after restarting MS Outlook.Note: These steps are required for Microsoft Outlook to properly configure the Add-in.

Configure the Viewpoint Add-in for Microsoft Outlook. For instructions, see [Configure the Vista Add-In for Microsoft
