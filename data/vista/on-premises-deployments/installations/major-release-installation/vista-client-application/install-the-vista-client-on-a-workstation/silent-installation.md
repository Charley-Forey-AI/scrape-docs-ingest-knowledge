# Silent Installation

Silent installation applies only if you are not using the
 standard installation process. However, the prerequisites are the same.
You should use this method only if you are familiar with its use and if these
 instructions and the prerequisite information in [Install the Vista Client on a Workstation](/en/vista/vista/on-premises-deployments/installations/major-release-installation/vista-client-application/install-the-vista-client-on-a-workstation) provide you
 sufficient information to proceed.
To run the client installer silently, use this command:
 Vista_Client_2#.##.##.msi /qn
