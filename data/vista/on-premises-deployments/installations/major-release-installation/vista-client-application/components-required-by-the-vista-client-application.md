# Components Required by the Vista Client Application

Vista requires certain components installed on your computer in order to make full use of the application.
These components are not included in the Vista application installer, but they are required:

- Microsoft .NET Framework: [Download .NET Framework from
 Microsoft](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48)

- SAP Crystal Reports Runtime: Required version included in Vista
 Prerequisites download folder, available on the [Customer Portal](https://support.viewpoint.com/s/product-more-resources?product=Vista&type=Downloads&links=true&title=Downloads%20for%20Vista)

- ODBC Driver for SQL Server: [Download ODBC Driver 17 for SQL
 Server from Microsoft](https://learn.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server?view=sql-server-ver15#version-17). This is required for Vista version 2025.6 and
 later

If they're not already installed on your workstation, you must install them.
 For the required versions of all software components, see [Vista Software Requirements](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements/vista-software-requirements).
Tip: For best results, make sure each component is installed on the workstation in question before you install the Vista client. If the listed versions are already installed, there's no need to install them again.

If needed, download the required components from the [Customer Portal](https://support.viewpoint.com/s/product-more-resources?product=Vista&type=Downloads&links=true&title=Downloads%20for%20Vista). Components are
 located in the same version-specific folder as the Vista client installer.
Expect to spend 30-45 minutes downloading and installing them. You should follow any prompts to reboot the workstation after installing any component.

## Optional Components

You can opt to install a Vista add-in for MS Outlook, which allows you to create indexes and attach emails to Vista records, all from within Outlook. For more information, see [Install the Vista Add-in for Microsoft Outlook](/en/vista/vista/on-premises-deployments/installations/major-release-installation/vista-client-application/install-the-vista-add-in-for-microsoft-outlook).
Supported versions of Outlook are indicated in the Vista [Hardware / Software Requirements](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements).
