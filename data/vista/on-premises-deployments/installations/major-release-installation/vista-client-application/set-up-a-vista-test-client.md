# Set Up a Vista Test Client

When you are ready to install an new version of Vista, you can run both a test instance and your earlier-version production instance of the Vista client on the same workstation.
These instructions guide you through setting up a test client.

1. In the Vista Log In screen, click the ‘+’ sign next to the Vista Server field.

1. In the Vista Server field, enter the test server name and tab off the field.

1. Complete the remaining fields and log in.

You can now switch back and forth between your production and test instances using the drop-down arrow in the Vista Server field.
