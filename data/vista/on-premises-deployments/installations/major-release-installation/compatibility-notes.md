# Compatibility Notes

Review the compatibility requirements before you install
 Vista.

## Security Considerations Prohibit Use of Secondary Databases

Multiple databases on a single server is not supported. Please note:

- In order to connect to a secondary database, you will need to create a separate test environment for Vista, with a dedicated database and application server.

- If you would like support for an on-premises test environments, you will need to obtain a separate contract.

- For Cloud test environments, you may purchase a separate, full test
 environment at a greatly reduced cost.

## Changes to Microsoft Products Support

- Vista is compliant with 64-bit versions of Microsoft products.

- Windows Server 2012: In 2023, Viewpoint will discontinue support of Windows Server
 2012.

## Future Vista Development Impacts all LAN Connections

For complete information about compatibility, see the Vista[Hardware / Software Requirements](/en/vista/vista/whats-new/whats-new-in-vista/hardware--software-requirements).
