# Vista Installation Files Download

This section provides information about downloading the installation files from the Viewpoint Customer Portal.
Vista installer files are located on the Viewpoint Customer Portal. These enable you to install the latest software releases from Viewpoint.

## Prerequisites for Downloading Installation Files

Downloading the Vista installation file(s) from the Viewpoint Customer Portal requires the following:

- An active internet connection

- A valid Viewpoint Customer Portal username and password

## Download the Vista Installation File

This section describes how to download the Vista installation file.

1. To access the installation file, go to the [Downloads for Vista](https://support.viewpoint.com/s/product-more-resources?product=Vista&type=Downloads&links=true&title=Downloads%20for%20Vista) page.Note: Accessing this folder location requires logging in to the Viewpoint Customer Portal.

1. Select the link for the release to install.The selected release folder opens, displaying all applicable files.

1. Select the relevant release installer filename, and then select Download.

1. Save the install file to a location you can retrieve it from when it's time to install it.
