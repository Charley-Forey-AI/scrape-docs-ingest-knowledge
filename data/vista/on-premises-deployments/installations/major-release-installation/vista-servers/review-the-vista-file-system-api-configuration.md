# Review the Vista File System API Configuration

If you've installed the Vista File System API, we recommend you review and/or note the default settings in the Viewpoint Remote Services Configuration form, especially if this is the first time you are installing the File System API on this server.
Important: This section applies to you only if all the following apply:

- Your deployment method is VRL on-premises

- You have [installed the File System API](/en/vista/vista/on-premises-deployments/installations/major-release-installation/vista-servers/install-the-vista-file-system-api)

-  You have not already reviewed the default settings or want to review them again

Skip this section if it does not apply to you.

LAN deployments allow you to store reports, templates, and so on in multiple and disparate locations. For VRL on-premises, such items must be consolidated into defined locations so that providing user access and enforcing security is simpler.
To review the default Vista File System API settings:

1. On the application server, open Configure Viewpoint Remote Service.

1. Select the API Settings tab.

1. Review the settings.
