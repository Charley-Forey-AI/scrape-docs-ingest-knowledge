# Install on a Test System

If applicable, you can install a Vista release on both production and test systems.
A production system is one or more servers running a version of Vista that is actively supporting your daily processes. Also referred to as a live system, this is the option that most users choose when installing a release.
 A test system is one or more servers running Vista that you are using to test the current release, but not for supporting your active, daily processes. You might select this option if you perform extensive testing of a new Vista release before deploying it to your production system.
Important:Viewpoint does not support multiple Vista databases in a single SQL instance. Do not install a test database to your production instance - it will cause your production instance to fail.

For instructions on installing this release on a test system, see [Install the Vista Application on a Test Server](/en/vista/vista/on-premises-deployments/installations/major-release-installation/vista-servers/install-the-vista-application-on-a-test-server).
