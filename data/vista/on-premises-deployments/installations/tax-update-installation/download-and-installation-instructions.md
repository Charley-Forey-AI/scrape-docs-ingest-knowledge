# Download and Installation Instructions

The following is an overview of the installation process for Vista tax updates. Please read these instructions completely before downloading and installing the tax update.
Downloading the Vista installation file(s) from the Viewpoint Customer Portal requires the following:

- An active internet connection

- A valid Viewpoint Customer Portal username and password

Select the associated links for more information on completing a step. If you encounter problems or have questions, contact [Viewpoint Customer Support](https://support.viewpoint.com/).

1. Download tax updates from the Viewpoint Customer Portal. For more information, see [Download the Vista Installation File](/en/vista/vista/on-premises-deployments/installations/tax-update-installation/download-and-installation-instructions/download-the-vista-installation-file).

1. Install the tax update on your computer. For more information, see [Install the Tax Update](/en/vista/vista/on-premises-deployments/installations/tax-update-installation/download-and-installation-instructions/install-the-tax-update).

1. Activate the new tax routines in Vista. For more information, see [Activate the New Routines](/en/vista/vista/on-premises-deployments/installations/tax-update-installation/download-and-installation-instructions/activate-the-new-routines).
